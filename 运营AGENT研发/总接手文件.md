## 6. 目标驱动主循环

```pseudo
goal-driven-main(goal):
    set done = false
    set loop_count = 0
    set max_loops = 3

    while not done:
        loop_count += 1
        task_type = classify(goal)
        main_skill = route(task_type)
        gaps = inspect_missing_inputs(goal)

        if gaps affect conclusion:
            if can_derive_with_uncertainty(gaps):
                mark_uncertainty(gaps)
            else:
                request_or_collect_only_gap_data(gaps)

        result = execute(main_skill)
        qa = verify(result)

        if qa fails:
            patch_thin_points(result)
            continue

        if cross_task_needed(goal):
            split_and_merge_child_tasks()

        if loop_count >= max_loops and still_not_closed(goal):
            output_best_current_judgment_with_limits()
            done = true
        else:
            done = goal_is_closed(result, qa)
```
