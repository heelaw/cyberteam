# 产品经理入门课

## 第三周（上「功能训练」理解产品的构成 - 如何做产品功能点的设计

**课程内容：**

相比于已有功能的优化，从零到一设计一个完整的功能点对于产品经理来说无疑更具挑战。特别是初入行的产品经理，他们在设计功能时极容易踩到各种坑，最终导致做出来功能问题颇多，比如：

- 功能做出来了，发现没人用；
- 功能比较鸡肋，没有解决根本问题；
- 功能流程太复杂，用户用不明白。

这些问题大多源于产品经理早期对功能的设计目的不明确，流程不严谨。如果说已有功能点的优化是修理房屋，那么完整功能的设计无疑是重新建造一个房屋，这也要求产品经理对于功能需要有更深的理解。

那么接下来，就让布老师先来普及一下什么是功能点。
