# 产品经理进阶课

## 1.1 用户特征分析（用户画像）

**课程类型：** 视频

**课程时长：** 约12分钟

**课程内容：**

抽象客观的数据如何与具体感性的用户匹配呢？本小节会带大家了解profile（用户特征分析） 的概念，通过打标签的方式去去分析一个用户群体。

开始之前我们先来看两个场景：

**1. 当你在工作中遇到这样的情况，你会怎么做？**

- 老板说，我们搞个活动吧，一定要尽可能多地吸引产品的目标用户。
- 但产品的目标用户都去哪儿找？

- 公司要做个新产品，老板说你做个用户分析吧。
- 但怎么知道产品的目标用户是谁呢？

- 你在看产品的数据时，看到有大量老用户流失。
- 在拟定有针对性的策略之前，怎样找出哪些用户是可召回的？

**2. 小明一周前在淘宝收藏了一个吹风机，之后的每一天淘宝都在给他推荐各种各样的吹风机......**

上面场景中都藏着同一个关键词——用户特征。当我们进行用户分析时首先就需要找到产品的用户在哪里，而挖掘出用户的典型"特征"是找到用户的方法。

---

**用户特征分析的概念**

用户特征分析（User Profiling）是产品经理进行用户分析的基础方法，通过打标签的方式去分析用户群体。

**用户画像的应用场景**

1. **活动运营**：老板要求尽可能多地吸引产品的目标用户时，需要知道目标用户在哪里
2. **新产品开发**：做用户分析时，需要知道产品的目标用户是谁
3. **用户召回**：老用户流失时，需要找出哪些用户是可召回的
4. **个性化推荐**：如淘宝的推荐系统，基于用户特征进行精准推荐

---

**如何找到竞品用户画像**

产品经理很多时候在调研用户画像的过程中经常会有个疑惑，就是不知道如何去找到竞品的用户画像，导致猜测用户画像的过程中要不就是拍脑袋，要不就是以自己数据作为参考（"数据作参考"常发生在竞品调研过程），其实这样的都是不太准确的，所以此时我们要寻求第三方调研报告去参考找到竞品用户画像。

**课程总结：**

1. **用户特征分析（User Profiling）**：通过打标签的方式分析用户群体

2. **核心关键词**：用户特征 - 找到产品的用户在哪里，挖掘出典型特征

3. **应用场景**：
   - 活动运营找目标用户
   - 新产品用户分析
   - 用户流失与召回
   - 个性化推荐

4. **获取用户画像的方法**：
   - 寻求第三方调研报告
   - 不能只依靠拍脑袋或自己的数据

5. **B端产品也需要用户特征分析**：B端产品更要做用户特征分析，而且更复杂，因为使用者、决策者（付费者）分离
