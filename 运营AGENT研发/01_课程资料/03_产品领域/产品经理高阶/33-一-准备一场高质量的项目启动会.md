# [一] 准备一场高质量的项目启动会

众所周知，军队在出征以前，都要有个誓师大会。大会上统帅要向战士宣布作战的意义，表示作战的目的和决心，以鼓舞士气。

商场如战场，当我们启动一个新项目的时候，也需要一个类似誓师大会的场景，来向团队成员们鼓舞士气，表明决心，告知目的和意义等等。具体的表现呢，就是开一场项目启动会。

那么，如何开一场高质量的项目启动会呢？

在开项目启动会时，我们需要讲清楚：为什么要做，具体怎么做，有谁参与，协作方式以及马上要做什么。

一般项目有两种典型的场景，一种是在成熟业务中开展项目，一种是从零到一探索项目，这两种场景下，有关项目启动会上的侧重点和讲法策略是不一样的。

成熟业务中开展的项目，在启动会上宣讲的核心目的是让大家意识到问题的严重性，就是在业务中，问题已经严重到了不得不解决的状态，所以最关键的是让大家形成对问题的共识；

而从零到一，因为本来就是探索型项目，失败的概率很大，所以要让团队成员理解到我们做这件事情的目的和使命感，这件事非我们不可，同时要提前让大家做好心理准备，做好打硬仗的准备。

开展新的项目时，团队负责人有可能是空降过来的，或是担任临时成立的一个虚拟项目组leader。管理这样一个团队，首先必须达成互相之间的信任感，那刚开始怎么做到呢？布老师的建议是，做一场就职演说。

总之，启动会的目的是为了让团队成员对于项目，达成在目标上和方向上的共识，同时明确日常工作的协同和规则。
