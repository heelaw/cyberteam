# [本周总结]

恭喜你完成了第二周的学习！在本周，我们一起从解决问题的角度，跳出了项目视角，而从一个更长远的视角来分析业务与进行中长期产品规划。那么，我们第一模块的内容，即从接手项目到产品的中长期规划，也就到此告一段落了。

我们最后再来简单回顾一下项目分析画布提到的几块内容。

项目分析画布的前三块内容分别谈到了：如何给项目定位；确定项目目标与指标以及梳理上下游关系。

而在第二周，我们回归到了产品与业务，主要讲的是：建立业务驱动模型；制定中长期产品规划。

概括来看，我们第一周讲的三部分内容是以项目为中心；而在第二周讲的两块内容是以整个项目的存活与持续为中心。我们在做项目的过程中不断加深了对于业务的理解，以及下一阶段项目的重点把握；而通过更好建立业务驱动模型，以及更全面的规划产品中长期发展，我们也能够更好地负责起一个个公司级项目。
