# [三] 从MVP到PMF

在精益创业MVP这个词汇流行之前，马克·安德森就于2007年提出了PMF（Product-Market Fit）的概念

那么，MVP与PMF的关系又是什么呢？

全局的来看从零到一：

我们在提出idea之后，通过市场分析和竞品分析判断idea是否有价值；然后，进行典型用户的核心需求分析，找出关键功能、确定流程并完成demo的设计，即进行MVP设计的流程；接下来进行小范围的推广，收集种子用户进行用户访谈，验证MVP的可行性；在不断的循环中找到PMF。

MVP的关键是要找到PMF：

PMF与MVP之间的关系是相辅相成的，我们通过提出PMF的假设来设计MVP的方式，然后通过一系列MVP的过程去找到PMF的增长点。
