## 📖 课程概述

本课程由硅谷增长专家曲卉讲授，帮助学员建立系统的增长思维框架，掌握数据驱动增长和用户激活留存的实战方法。

**课程特色**：
- 🎯 **全局思维**：从业务负责人角度理解增长
- 📊 **数据驱动**：建立科学的增长实验流程
- 🔧 **实战方法**：可直接应用的增长策略
- 📈 **系统框架**：AARRR全链路增长方法论

## 📌 附录

**核心概念索引**：
- `PMF` (Product-Market Fit) - 产品市场匹配
- `AARRR` - 获取、激活、留存、变现、推荐
- `北极星指标` - 衡量增长的核心指标
- `ICE模型` - 实验优先级评估模型
- `A/B测试` - 增长实验方法
- `Hooked模型` - 用户习惯养成模型
- `LTV` (Life Time Value) - 用户生命周期价值

**学习建议**：
1. 按顺序学习，建立完整的增长知识体系
2. 每章结束后完成相应作业，巩固所学
3. 结合实际产品，思考如何应用所学方法
4. 第7章为选修内容，可根据实际需要学习
