## 🔑 核心概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| **PMF** | Product-Market Fit（产品市场契合度） | 有用户需要和喜欢你的产品，并在较长时间里重复使用 |
| **AARRR模型** | 用户增长漏斗模型 | Acquisition（获取）→ Activation（激活）→ Retention（留存）→ Revenue（变现）→ Referral（推荐） |
| **RARRR模型** | 调整后的增长模型 | Retention（留存）→ Activation（激活）→ Referral（推荐）→ Revenue（变现）→ Acquisition（获取） |

| **留存曲线** | PMF判断依据 | 当留存曲线趋于平稳时，表明已达到PMF |
| **全链漏斗型模型** | 增长模型类型 | 基于用户转化路径构建的模型 |
| **因子分解型模型** | 增长模型类型 | 基于数学公式拆解的模型 |
| **全定量模型** | 增长模型类型 | 完全数据驱动的Excel模型 |

## 📚 详细内容

### 主题一：产品增长准备度评估（PMF）

#### 1.1 你的产品，是否已准备好做大规模增长？

##### 两种常见的"死法"

很多产品失败的原因可以归结为两种：

**第一种：生于拉新，死于留存**

- 症状：大量投放广告获取用户，但用户使用一次后不再回来
- 根源：产品尚未达到PMF，就开始大规模拉新
- 结果：获客成本持续高于用户生命周期价值

**第二种：生于拉新，死于变现**

- 症状：用户活跃度不错，但无法转化为付费用户
- 根源：商业模式不清晰，或变现时机不对
- 结果：有流量但没有收入，持续亏损

##### PMF（Product-Market Fit）的定义

**PMF是指：产品与市场达到契合状态**

核心标志：
1. 有用户需要你的产品
2. 有用户喜欢你的产品
3. 用户在较长时间里重复使用

##### PMF前后的策略差异

| 阶段 | 核心任务 | 关键动作 | 避免事项 |
|------|----------|----------|----------|
| **PMF之前** | 探索与验证 | - 快速迭代产品<br>- 验证核心假设<br>- 寻找目标用户 | - 大规模投放<br>- 过早优化<br>- 固定商业模式 |
| **PMF之后** | 扩张与优化 | - 渠道扩张<br>- 产品优化<br>- 商业模式放大 | - 频繁转型<br>- 忽视用户体验<br>- 盲目多元化 |

##### RARRR模型：现代增长框架

传统AARRR模型的顺序是：Acquisition → Activation → Retention → Revenue → Referral

**现代RARRR模型调整为：**

```
Retention（留存）→ Activation（激活）→ Referral（推荐）→ Revenue（变现）→ Acquisition（获取）
```

**为什么把Retentio放在第一位？**

- 留存是验证PMF的核心指标
- 没有留存，其他环节都无意义
- 先确保用户会回来，再考虑扩大规模

##### PMF的判断标准

**标准一：肖恩·埃利斯的"40%法则"**

- 问题："如果你不能使用这个产品了，你会有多失望？"
- PMF标志：至少40%的用户表示"非常失望"

**标准二：留存曲线变平**

- 追踪用户留存率随时间的变化
- 当留存曲线在某一个水平趋于平稳
- 说明产品已经找到核心用户群，达到PMF

#### 1.2 通过4个问题，判断现阶段增长重点

##### 问题1：市场处于什么阶段？

**增量市场 vs 存量市场**

| 市场类型 | 特征 | 增长策略 | 案例 |
|----------|------|----------|------|
| **增量市场** | - 用户规模快速增长<br>- 竞争相对较小 | - 快速占领市场<br>- 扩大用户规模 | 早期短视频、早期电商 |
| **存量市场** | - 用户增长放缓<br>- 竞争激烈 | - 提升用户价值<br>- 抢占竞品用户 | 当前电商、当前社交 |

**判断方法：**
- 查看行业报告的用户增长率
- 分析目标用户群体的渗透率
- 观察竞争对手的数量和强度

##### 问题2：产品处于生命周期的哪个阶段？

**产品生命周期四阶段：**

```
探索期 → 成长期 → 成熟期 → 衰退期
```

**探索期特征：**
- 用户规模小，但增长率可能很高
- 产品频繁迭代
- 商业模式在验证中
- **增长重点：验证PMF**

**成长期特征：**
- 用户快速增长
- 竞争开始加剧
- 商业模式逐渐清晰
- **增长重点：快速扩张**

**成熟期特征：**
- 用户增长放缓
- 市场格局基本确定
- 利润率承压
- **增长重点：提升效率、挖掘深度价值**

**衰退期特征：**
- 用户流失
- 新技术/新产品替代
- **增长重点：转型或收割**

##### 问题3：产品属于什么品类？

**八大产品品类：**

| 品类 | 核心价值 | 增长重点 | 典型指标 |
|------|----------|----------|----------|
| **工具类** | 提升效率 | - 解决特定痛点<br>- 口碑传播 | 日活、使用频次 |
| **内容类** | 信息/娱乐 | - 内容质量<br>- 内容分发 | 阅读时长、内容消费量 |
| **游戏类** | 娱乐体验 | - 玩法设计<br>- 社交互动 | 留存、付费率 |
| **社交类** | 人际连接 | - 网络效应<br>- 用户互动 | 社交互动频次 |
| **电商类** | 商品交易 | - 商品供给<br>- 信任建立 | GMV、复购率 |
| **SaaS类** | 业务效率 | - 产品价值<br>- 客户成功 | 续费率、ARPU |
| **平台类** | 连接供需 | - 双边网络效应<br>- 供需平衡 | 交易量、双边活跃 |
| **杂交类** | 多重价值 | - 识别主价值<br>- 平衡多价值 | 组合指标 |

##### 问题4：商业模式有哪些独特因素？

**社交属性 vs 付费意愿四象限分析：**

```
高付费意愿
    │
    │  [象限1]            [象限2]
    │  社交属性弱         社交属性强
    │  付费意愿高         付费意愿高
    │  - B2B服务         - 高端社交
    │  - 专业工具         - 知识付费
    │
    │  [象限3]            [象限4]
    │  社交属性弱         社交属性强
    │  付费意愿低         付费意愿低
    │  - 工具产品         - 基础社交
    │  - 内容平台         - 早期产品
    │
    └─────────────────────────────→
    强社交属性                          弱社交属性
```

**各象限增长策略：**

- **象限1（高付费、弱社交）**：直接销售、客户成功
- **象限2（高付费、强社交）**：社群运营、口碑传播
- **象限3（低付费、弱社交）**：广告变现、规模效应
- **象限4（低付费、强社交）**：免费增值、增值服务

#### 1.3 快速挖掘竞品信息，拆解增长要素

##### 第一步：找到可借鉴的产品

**三类可借鉴对象：**

1. **直接竞品**
   - 相同目标用户
   - 相似产品功能
   - 相同商业模式

2. **非直接竞品（同目标用户）**
   - 目标用户相同
   - 产品形态不同
   - 可以借鉴获客方式

3. **非直接竞品（同品类）**
   - 产品品类相同
   - 目标用户不同
   - 可以借鉴产品逻辑

4. **同品牌调性产品**
   - 品牌定位相似
   - 可以借鉴品牌策略

##### 第二步：收集信息的渠道

**1. 应用商店分析**
- 产品介绍、更新日志
- 用户评价（高分和低分都要看）
- 排行榜位置变化

**2. 广告工具**
- Facebook Ads Library
- AppGrowing
- DataEye
- 观察广告素材、投放渠道、落地页

**3. 搜索引擎**
- 品牌+关键词搜索
- 查看SEO表现
- 分析SEM投放

**4. 微信生态**
- 公众号内容
- 小程序功能
- 社群运营
- 视频号内容

**5. 亲身体验**
- 注册流程
- 产品核心功能
- 用户引导
- 付费流程

##### 第三步：用AARRR模型还原增长要素

**AARRR拆解框架：**

| 环节 | 调研问题 | 关注点 |
|------|----------|--------|
| **Acquisition（获取）** | - 用户从哪里来？<br>- 看到什么广告/内容？ | - 渠道来源<br>- 获客素材<br>- 落地页设计 |
| **Activation（激活）** | - 新用户首先做什么？<br>- 如何引导首次使用？ | - 新手引导<br>- 首次体验设计<br>- 关键行为设置 |
| **Retention（留存）** | - 什么让用户回来？<br>- 有哪些召回机制？ | - 留存机制<br>- 推送策略<br>- 内容/功能更新 |
| **Revenue（变现）** | - 如何收费？<br>- 付费流程如何设计？ | - 付费模式<br>- 定价策略<br>- 付费节点 |
| **Referral（推荐）** | - 如何激励用户分享？<br>- 分享机制如何设计？ | - 分享激励<br>- 分享工具<br>- 病毒系数 |

**输出形式：竞品增长要素拆解表**

### 主题二：北极星指标

#### 2.1 找到增长方向标：北极星指标

##### 北极星指标的意义

**谷歌 vs YouTube的置顶结果差异**

- **Google搜索**：置顶结果是搜索量
- **YouTube**：置顶结果是观看时长

为什么同样都是Google的产品，北极星指标不同？

- Google的核心价值是"快速找到答案" → 搜索量
- YouTube的核心价值是"消磨时间" → 观看时长

**北极星指标的作用：**

1. 指明方向：让团队知道什么最重要
2. 聚焦资源：避免同时追求太多指标
3. 衡量价值：反映用户是否真正获得价值

##### 三种产品类型的北极星指标

**1. 消磨时间型产品**

核心价值：让用户愿意花时间使用

北极星指标：使用时长、使用频次

案例：
- YouTube：观看时长
- 抖音：用户时长
- 游戏类：在线时长

**2. 提升效率型产品**

核心价值：帮助用户更快完成任务

北极星指标：核心功能使用量、任务完成量

案例：
- 搜索引擎：搜索次数
- 工具软件：功能使用次数
- SaaS产品：关键功能使用量

**3. 完成交易型产品**

核心价值：促成交易

北极星指标：交易量、交易金额

案例：
- 电商：GMV、订单量
- O2O：交易单数
- 支付：交易笔数

##### YouTube案例：从点击率到观看时长

**早期YouTube的北极星指标：点击率**

问题：
- 标题党泛滥
- 用户点击后失望
- 长期价值受损

**转向"观看时长"后：**

- 推荐算法优化目标改变
- 优质内容获得更多推荐
- 用户满意度提升
- 平台长期价值增长

##### Facebook vs MySpace案例

**MySpace的指标：注册用户数**

问题：
- 追求数量忽略质量
- 大量僵尸账号
- 活跃度持续下降

**Facebook的指标：月活跃用户数**

优势：
- 关注真实活跃
- 质量优先于数量
- 可持续增长

##### 制定北极星指标的三步法

**步骤1：明确商业目标和用户价值**

商业目标 ≠ 用户价值，但好的北极星指标能同时反映两者

**步骤2：列出备选指标**

列出3-5个可能反映核心价值的指标

**步骤3：确认最终指标**

用六个标准检验

##### 北极星指标的六个标准

| 标准 | 说明 | 检验问题 |
|------|------|----------|
| **1. 反映用户获得价值** | 指标上升说明用户获得更多价值 | 指标增长 = 用户更满意？ |
| **2. 为长期商业目标奠定基础** | 指标增长有助于实现长期目标 | 指标增长 = 商业可持续？ |
| **3. 反映活跃程度** | 关注活跃用户而非总数 | 是活跃用户指标？ |
| **4. 提示公司向好发展** | 指标向好 = 公司向好 | 指标上升 = 公司健康？ |
| **5. 简单直观可拆解** | 易于理解和拆解 | 团队都能理解？ |
| **6. 先导指标** | 能预示未来趋势 | 变化早于结果？ |

#### 2.2 进一步理解北极星指标

##### 案例1：猎聘网

**北极星指标：个人投递简历日活**

**为什么选这个指标？**

1. 反映求职者的核心行为
2. 同时服务两端用户（求职者和企业）
3. 与平台交易强相关
4. 可拆解为可操作指标

##### 案例2：JIRA

**北极星指标：周活跃用户**

**为什么选周活而非日活？**

- JIRA是协作工具，不同用户使用频率不同
- 管理者可能每天看，普通成员每周看几次
- 周活能更好反映整体使用情况

**核心洞察：北极星指标要适配产品使用场景**

##### 案例3：小红书的北极星指标演化

**阶段1：早期 - 笔记数**

- 目标：快速积累内容
- 重点：鼓励用户发布内容

**阶段2：海淘期 - 海淘购物用户数**

- 目标：验证商业模式
- 重点：找到愿意付费的用户

**阶段3：成长期 - 活跃用户数**

- 目标：扩大用户规模
- 重点：提升整体活跃度

**阶段4：变现期 - 销售额**

- 目标：商业化
- 重点：实现营收目标

**核心洞察：北极星指标可以随阶段演化**

### 主题三：增长模型

#### 3.1 基础：构建和应用3种增长模型

##### 什么是增长模型？

**定义：把北极星指标拆分成可操作的细分指标**

**为什么要构建增长模型？**

```
输入变量（可控） → 中间变量 → 输出变量（北极星指标）
```

- 没有模型：北极星指标太抽象，不知道如何影响
- 有模型：找到可操作的输入变量，通过控制输入影响输出

##### 构建增长模型的三步法

**步骤1：分解北极星指标**

将北极星指标逐步拆解为更小的组成部分

**步骤2：绘制核心转化路径**

识别用户完成关键行为的主要路径

**步骤3：组装增长模型**

将拆解的指标组合成一个完整的数学模型

##### 两种基础增长模型类型

**1. 全链漏斗型**

基于用户转化路径构建

公式：`北极星指标 = 流量 × 转化率1 × 转化率2 × ...`

适用场景：
- 用户行为路径清晰
- 转化节点明确

**2. 因子分解型**

基于数学公式拆解

公式：`北极星指标 = 因子A × 因子B × 因子C`

适用场景：
- 指标可以用数学公式表达
- 因子之间的关系清晰

##### 增长模型的三个应用

**应用1：找杠杆**

- 目的：找到对北极星指标影响最大的因子
- 方法：对模型进行敏感性分析
- 价值：集中资源在高杠杆点

**应用2：摘线头**

- 目的：找到当前最弱的环节
- 方法：对比各环节的数据表现
- 价值：优先改善瓶颈

**应用3：搭乐高**

- 目的：通过组合不同策略达到目标
- 方法：调整多个输入变量的组合
- 价值：找到最优策略组合

#### 3.2-3.3 进阶：全定量增长模型

##### 四种增长模型对比

| 模型类型 | 特点 | 构建难度 | 应用场景 |
|----------|------|----------|----------|
| **AARRR定性模型** | 描述性，无量化 | 低 | 初步分析 |
| **全链漏斗型** | 基于转化路径 | 中 | 路径优化 |
| **因子分解型** | 基于数学拆解 | 中 | 指标拆解 |
| **全定量型** | 完全数据驱动 | 高 | 精准预测 |

##### 全定量模型的Excel实现

**基本结构：**

1. **输入层**：所有可控变量
2. **计算层**：中间变量和公式
3. **输出层**：北极星指标

**关键点：**

- 所有关系都量化为公式
- 可以实时调整输入看输出变化
- 支持多场景对比

##### 假设分析（What-if Analysis）

**目的：比较不同实验的效果**

**方法：**

1. 设定基准场景
2. 调整单个或多个变量
3. 观察北极星指标的变化
4. 比较不同场景的结果

**应用：**

- 实验优先级排序
- 目标设定
- 资源分配决策

#### 3.4-3.5 不是增长负责人，如何应用北极星指标和增长模型

##### 应用场景1：根据公司大指标制定个人指标和策略

**五步流程：**

**步骤1：理解公司目标**

- 清楚公司的北极星指标是什么
- 理解为什么选这个指标

**步骤2：确认个人指标**

- 找到个人工作与北极星指标的连接点
- 确定个人可控的指标

**步骤3：制定迷你增长模型**

- 将个人指标拆解为可操作的细分指标
- 构建个人工作的影响力模型

**步骤4：梳理资源和策略**

- 列出可用的资源
- 制定实现策略

**步骤5：制定OKR**

- 设定明确的目标
- 定义可衡量的关键结果

##### 应用场景2：重新给自己定指标和目标

**当公司没有明确北极星指标时：**

1. **观察行业标杆**

#### 3.6 快速回顾：运用北极星指标和增长模型驱动增长

##### Camera360案例：四步法应用

**步骤1：制定北极星指标**

- 分析产品类型：工具类产品
- 确定核心价值：帮助用户美化照片
- 选择北极星指标：照片编辑次数

**步骤2：绘制转化路径**

```
用户打开应用 → 选择照片 → 使用编辑功能 → 保存/分享照片
```

**步骤3：构建增长模型**

`照片编辑次数 = 活跃用户数 × 人均编辑次数`

进一步拆解：
- 活跃用户数 = 新用户 + 留存用户
- 新用户 = 获客 × 激活率
- 留存用户 = 上期活跃 × 留存率
- 人均编辑次数 = 编辑功能使用率 × 人均使用次数

**步骤4：推动指标增长**

- 发现低杠杆点：激活率低
- 优化新手引导
- 提升编辑功能易用性
- 增加社交分享激励

## 📌 本章要点

### 核心收获

**1. PMF是大规模增长的前提**

- 不要在PMF之前大规模拉新
- RARRR模型告诉我们：留存优先
- 用40%法则和留存曲线判断PMF

**2. 增长重点取决于多个因素**

- 市场阶段：增量vs存量
- 产品生命周期：探索期/成长期/成熟期/衰退期
- 产品品类：工具/内容/游戏/社交/电商/SaaS/平台/杂交
- 商业模式：社交属性×付费意愿

**3. 北极星指标是增长的方向标**

- 必须反映用户价值
- 必须为长期商业目标服务
- 可以随产品阶段演化
- 用六个标准检验

**4. 增长模型让北极星指标可操作**

- 把抽象指标拆解为可控变量
- 全链漏斗型 vs 因子分解型 vs 全定量型
- 三个应用：找杠杆、摘线头、搭乐高

**5. 个人如何应用**

- 理解公司北极星指标
- 找到个人与北极星指标的连接
- 构建个人迷你增长模型
- 用OKR落地执行
