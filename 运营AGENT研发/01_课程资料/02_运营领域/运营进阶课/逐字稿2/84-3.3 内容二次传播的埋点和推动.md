# 3.3 内容二次传播的埋点和推动

好 我们接下来再进入到本堂课程的第二个小节 内容二次传播的买点和推动 这个小节的内容没有那么的复杂 所以我们会讲的稍稍的快一点

首先我们来理解一下什么叫做二次传播 所谓的二次传播就是我们的内容已经出现在某个渠道或者平台后 通过我们人为的干预又引发的新一轮的内容传播 也就是说假如你的内容最初发在一个微信公号上了 这是第一轮传播 那事后已经发出来了以后 通过我们人为又做了一点点事情 又引发了新一轮的曝光和传播 那这个东西就叫做二次传播了

所以二次传播基本上是有助于我们的内容在某个渠道可以实现它的一个价值最大化的

那么二次传播可能又会有两种 一种叫做预先设计好的二次传播

我们先来看一下预先设计好的二次传播它的一个基础的逻辑是怎么样的

所以预先设计的二次传播 我们可能先要事先设计好一些潜在的二次传播节点 比如讲我们写了一篇文章 这篇文章首先发出来 它会引发第一轮的传播 那我们后边埋了一个传播节点 比如说是找了一个大V 比如说找了周鸿祎来给我们发声 力挺说我靠这篇文章写的牛逼 那这个东西就是一个潜在的二次传播节点 这个东西要事先设计好

假如这个东西设计好之后 你就要通过大量的沟通协调去确保它能够被执行到位 以及到这个二次传播节点执行到位了之后 ok那你要迅速的跟进 再做新一轮的放大 这就是叫做预先设计好的二次传播

你会发现他整个的逻辑并没有那么复杂 对吧

那在这儿我们举一个例子 是就是2015年6月份的时候 我在周博通招聘 然后做过的一个活动叫做613合伙人招募节

对它其实是一个高端的招聘会 这个招聘会当时整个的流程是这样的

2015年5月21 23号 我们首先发文宣布了我们要举办这个活动 后 5月26号的时候 经过我们的沟通协调 行业里边的几位大佬互联网行业里边几位大佬 包括暴风的CEO冯鑫 包括以前创业家的主编 后来罗辑思维的联合创始人申音 还有包括像果壳网和在行的这个CEO g 13 他们确认参与 对 这个是5月26号的事情

于是我们在5月28号就迅速的再次发声 然后对外宣布这几位大佬出席参加我们的这个活动了 然后于是带来了更大的一个传播力

这个活动其实也是在我们完成了二次传播的这个操作之后 他整个的影响力得到了迅速的放大

最终这个活动它的一个成果其实比屏幕上这个截图看到的还要好得多 最终差不多是有1300多个互联网公司的创始人报名参加 加上我们差不多应该是有1000五六百个这种合伙人宣布参加 总计累计申请参加的人数有接近3000人 差不多到了这样一个规模

所以还是一个比较成功的活动 而且这个活动在线上 单是他的这个传播量就近达到了三十多万次

所以这是一个例子 这个就叫做预先设计好的二次传播 预先买好一个点到点把它抛出来去放大它 基础逻辑就是这样的

那另外还有一种二次传播 叫做偶遇的二次传播

所以顾名思义 这种二次传播肯定就不是你事先设计好的 它可能是你正好赶巧赶上了 对吧

所以这样的二次传播我们基础的这个逻辑就是当你的内容发布以后 你还需要保持一定的敏感度 时刻要留心各种心境出现的关于这个内容的素材 比如讲用户的这种评论

或者是其他人对这个内容的一些见解 或者是围绕着这个内容周边 比如说有哪些号又转载了这样的一些信息 都可以作为你的二次传播素材而出现的

那以及如果你发现了一个很好的素材 就一定要第一时间跟进去 放大他的话题性 借此引发你这个内容的再次传播和围观

所以基础逻辑其实就是这样

我们也来看一个例子 上边的在这张截图上的上边这篇文章是我在4月22号发布了一篇文章

叫做趴皮酱2200万拍卖会的始末 思考罗振宇的作局与造势

这篇文章是我在当天中午临近12点的时候发布出来的 在发布完了之后 因为前一天这个帕皮酱的拍卖会刚刚结束 所以发布完了之后 很多人文章的本身的质量也不错 所以迅速他的阅读量就过了1万 很多人也纷纷的转发

但是到了当天晚上 理论上因为到当天晚上的时候 这个内容的传播可能就差不多了 基本上就没有太大规模的传播的可能性了 对吧

但是当天晚上又发生了一件什么事情呢 对 当天晚上的时候我们发现这个罗振宇本人亲自跑到这篇内容下面来 给这篇内容进行评论了

所以这件事情我们就非常敏感的意识到了 它是一个非常好的二次传播的素材

于是我们迅速的在微信公号后台回复了这个罗振宇的留言 罗振宇的评论 并且也迅速的找了一点人 把罗振宇的这个评论往上顶了一下

这个顶不用顶的太高 你差不多顶到几十的时候 因为其他人看到罗振宇本人来了 他们也会去点赞 对吧

往往把这个内容 把这个评论往上去推的 吹完了之后 ok那我们又在我们自己的朋友圈又纷纷去发 就说我考这篇文章 没想到罗振宇们都亲自跑来评论了 而且罗振宇还说这篇文章很牛逼

所以在这件事情的促使下 这篇文章就引发了新一轮的传播 在当天晚上最终这篇文章的传播量差不多是有个两万多次 差不多达到了这样一个状态 而且也引发了很多行业里边的大号去纷纷的转载

这种就叫做偶偶遇的二次传播

你对于这种偶遇的二次传播 一定要对于各种潜在的传播素材保持一定的敏感度 包括你在知乎发了一个评论 对吧 谁可能去去去跟你这个内容进行了互动 他比如像一些名人 一些行业里边的这种大牛 对吧 这些东西都是很好的 而且传播素材一定要对这个东西保持敏感

在这堂课程里面我们讲了两个方面 一方面是第一个小节我们讲了打通内容的对外传播通路 在这件事情上我们其实有这样几个工作步骤
