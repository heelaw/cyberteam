# 3.3 搭建转化场景：转化诱因和用户决策
> 本章节类型：视频课程
> 课程范围：运营进阶课 第63节

---

上一节的内容我们已经学习完了，你还记得提升营销推广效果的 3 个原则吗？我们来一起回顾下：

- 原则 1：要根据场景设计转化路径
- 原则 2：转化路径要尽量唯一，最短，可信
- 原则 3：要根据用户决策类型、决策成本搭建具体转化场景

在原则 3 里面有一个很重要的概念是"转化诱因"，接下来，我们来聊聊。

当你确定了目标用户和推广渠道，对如何搭建转化场景也有了足够的思考。之后要做的工作其实就是，出方案推进执行了，具体就是落实方案和物料，并协调资源落地执行了。

具体的推广方案各个公司要求各不相同。必备的要素有预算设置，竞品分析，产品定位，投放物料（设计稿，页面，文案），推广预算，推广目标，团队架构及分工等

具体可戳下文查看：[如何制定一份运营推广策划案](http://www.woshipm.com/operate/573343.html)
