# 第七章：推广营销 - 打通产品增长的外部引擎

> 所属章节：第七章
> 包含小节：认识推广营销、开拓推广入口、搭建转化场景、推广数据分析

---

## 本章导学

### 核心目标

本章讲解推广营销的核心方法论，帮助运营人员理解如何从外部渠道获取流量并完成用户转化。推广营销是产品增长的重要引擎，运营人员需要掌握如何选择渠道、搭建转化场景、设计转化路径。

### 学习重点

1. 理解推广营销的本质和核心逻辑
2. 掌握推广营销的三要素
3. 学会发现和开拓推广入口
4. 掌握转化场景的搭建方法
5. 了解推广数据分析的思路

---

## 详细内容

### 7.1 认识推广与营销

#### 推广营销的本质

推广营销的本质是从外部渠道找到用户，通过中间承接将用户转化为产品用户。

```
┌─────────────────────────────────────────────────────────────┐
│                   推广营销核心逻辑                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌──────────┐      ┌──────────┐      ┌──────────┐       │
│   │ 外部渠道  │  →   │  中间页   │  →   │  产品    │       │
│   │  找到用户  │      │  承接转化  │      │  用户    │       │
│   └──────────┘      └──────────┘      └──────────┘       │
│                                                             │
│   获取流量 ←────  转化流量  ──→ 完成目标                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 推广营销的两个核心问题

| 问题 | 说明 |
|------|------|
| 如何获取流量？ | 从哪些渠道获取用户 |
| 如何转化流量？ | 如何让用户完成目标行为 |

#### 获取流量的两种方式

| 方式 | 说明 | 示例 |
|------|------|------|
| 付费获取 | 花钱投广告 | SEM、信息流广告 |
| 免费获取 | 不花钱获取流量 | 软文、活动、事件营销 |

#### 流量转化的两个导向

| 导向 | 目标 | 适用产品 |
|------|------|---------|
| 用户导向 | 提升注册量、下载量 | 工具类产品 |
| 收入导向 | 提升销售额 | 电商类产品 |

### 7.3 发现及开拓匹配的推广入口

#### 渠道评估维度

```
┌─────────────────────────────────────────────────────────────┐
│                   渠道评估四维度                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 用户匹配度                                            │
│   ├─ 渠道的用户画像与目标用户是否匹配？                   │
│   └─ 用户的付费能力、年龄、地域等是否吻合？               │
│                                                             │
│   2. 流量规模                                              │
│   ├─ 渠道能提供多大的流量？                               │
│   └─ 流量的天花板在哪里？                                 │
│                                                             │
│   3. 竞争程度                                             │
│   ├─ 渠道上竞品多不多？                                   │
│   └─ 获取流量的成本和难度？                               │
│                                                             │
│   4. 转化效率                                             │
│   ├─ 渠道的用户质量如何？                                 │
│   └─ 转化率大概是什么水平？                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 常见推广渠道类型

| 渠道类型 | 特点 | 适用场景 |
|----------|------|---------|
| 搜索引擎 | 精准投放，按点击付费 | 明确需求的用户 |
| 信息流 | 兴趣推荐，融入内容 | 品牌曝光、新客获取 |
| 社交媒体 | 社交传播，KOL合作 | 品牌传播、裂变增长 |
| 应用商店 | 搜索+推荐 | APP推广 |
| 线下渠道 | 地域定向，O2O | 本地生活服务 |

#### 新人常见问题与解决

| 问题 | 解决思路 |
|------|---------|
| 找不到合适渠道 | 先调研用户画像，明确目标用户在哪里 |
| 不知道如何选择 | 先小规模测试，再根据数据优化 |
| 不会投放 | 学习渠道规则，从基础操作开始 |
| 效果不好 | 分析数据，找到问题环节进行优化 |

| 曝光量 | 广告被展示的次数 | - |
| 点击量 | 用户点击广告的次数 | - |

## 本章小结

1. **推广营销的本质**：从外部渠道找到用户，通过中间页承接，将用户转化为产品用户。核心关注两个问题：获取流量和转化流量。

2. **获取流量的两种方式**：付费获取（SEM、信息流）和免费获取（软文、活动）。两者各有优劣，需要根据业务阶段选择。

3. **推广营销三要素**：用户（目标是谁）、渠道（在哪儿推广）、场景（如何说服）。三要素必须界定清楚。

4. **渠道评估四维度**：用户匹配度、流量规模、竞争程度、转化效率。新人建议先小规模测试，再根据数据优化。

5. **转化场景搭建**：识别场景→设计信息→设置路径。在特定场景下，将说服用户的逻辑设计清楚。

6. **转化路径优化**：分析每个环节的转化率，找到流失最大的环节进行优化。

7. **推广数据分析**：关注曝光、点击、转化、成本、ROI等核心指标，通过数据发现问题并优化。
