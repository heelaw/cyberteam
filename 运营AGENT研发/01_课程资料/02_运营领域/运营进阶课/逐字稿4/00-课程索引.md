# 课程索引

> 课程名称：运营进阶课
> 课程章节：共8章
> 核心内容：文案进阶、数据分析、活动运营、推广营销、用户运营

| 序号 | 章节名称 | 核心要点 |
|------|---------|---------|
| 01 | 导论与底层逻辑 | 运营岗位相通逻辑、产品生命周期、精益运营 |
| 02 | 文案进阶（上） | 需求判断、转化策略、4P模型、AIDA模型 |
| 03 | 文案进阶（下） | 书写原则、用户感受、文案自检 |
| 04 | 数据分析 | 指标拆解、历史数据、增长发力点、异常分析 |
| 05 | 活动运营（上） | 活动本质、6大要素、主题策划、风险评估 |
| 06 | 活动运营（下） | 执行落地、微创新、案例分析 |
| 07 | 推广营销 | 入口发现、转化场景、4类营销方式 |
| 08 | 用户运营与工作管理 | 用户运营本质、分层运营、SOP管理、项目管理 |
