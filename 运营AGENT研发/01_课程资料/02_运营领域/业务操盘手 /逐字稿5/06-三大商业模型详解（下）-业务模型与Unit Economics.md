### 3.1 业务模型的核心本质

#### 什么是业务模型

```
┌─────────────────────────────────────────────────────────────┐
│                    业务模型定义                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  业务模型回答"如何运转"的问题：                             │
│                                                             │
│  1. 业务如何流转？                                          │
│     └─ 从输入到输出，中间经过哪些环节                       │
│                                                             │
│  2. 价值如何产生？                                          │
│     └─ 哪个环节创造了商业价值                               │
│                                                             │
│  3. 增长如何驱动？                                          │
│     └─ 如何让业务持续增长                                   │
│                                                             │
│  4. 各环节如何协作？                                        │
│     └─ 上下游之间的配合机制                                 │
│                                                             │
│  本质：描述业务如何从输入转化为输出的完整体系               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 业务模型的核心公式

```
┌─────────────────────────────────────────────────────────────┐
│                 业务模型核心公式                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│         输入(Input) ──→ 业务模型 ──→ 输出(Output)          │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  给定一个确定的A → 通过业务模型 → 得到确定的B        │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  关键特征：                                                 │
│  ├─ 输入是可控的（资金、用户、资源）                       │
│  ├─ 过程是可优化的（流程、机制、效率）                     │
│  └─ 输出是可预期的（收入、增长、价值）                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 全局业务模型与局部业务模型

#### 全局业务模型

```
┌─────────────────────────────────────────────────────────────┐
│                   全局业务模型定义                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  全局业务模型 = 公司整体业务的运转体系                        │
│                                                             │
│  包含三个层次：                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  第1层：整体业务增长模型（核心）                    │   │
│  │  ├─ 收入如何产生                                     │   │
│  │  ├─ 增长如何驱动                                     │   │
│  │  └─ 商业价值如何实现                                │   │
│  └─────────────────────────────────────────────────────┘   │
│                          ↓                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  第2层：生产供给交付保障体系                         │   │
│  │  ├─ 产品研发                                         │   │
│  │  ├─ 供应链管理                                       │   │
│  │  └─ 服务交付                                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                          ↓                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  第3层：基础工具与效率系统                          │   │
│  │  ├─ 管理系统                                         │   │
│  │  ├─ 数据工具                                         │   │
│  │  └─ 协作平台                                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 Unit Economics（单位经济模型）详解

#### 什么是Unit Economics

```
┌─────────────────────────────────────────────────────────────┐
│                   Unit Economics定义                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Unit Economics = 单位经济模型                               │
│                                                             │
│  定义：评估单个经营单元的盈利能力                            │
│                                                             │
│  核心问题：                                                 │
│  ├─ 每个订单/客户能赚多少钱？                               │
│  ├─ 每个门店/区域是否盈利？                                 │
│  ├─ 业务规模扩大后能否持续盈利？                            │
│                                                             │
│  关键要素：                                                 │
│  ├─ 固定成本：不随产量变化的成本                           │
│  ├─ 可变成本：随产量变化的成本                             │
│  ├─ 单位收入：每个单元产生的收入                           │
│  └─ 盈亏平衡点：盈利vs亏损的临界点                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 固定成本与可变成本

```
┌─────────────────────────────────────────────────────────────┐
│               固定成本 vs 可变成本                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【固定成本】                                               │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  定义：不随产量变化而变化的成本                       │   │
│  │                                                       │   │
│  │  示例：                                              │   │
│  │  ├─ 房租/物业                                        │   │
│  │  ├─ 设备采购                                         │   │
│  │  ├─ 人员基本薪资                                     │   │
│  │  ├─ 品牌宣传                                         │   │
│  │  └─ 行政费用                                         │   │
│  │                                                       │   │
│  │  特点：                                               │   │
│  │  ├─ 无论是否销售都需要支付                          │   │
│  │  └─ 产量增加时，单位固定成本降低                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【可变成本】                                               │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  定义：随产量变化而变化的成本                         │   │
│  │                                                       │   │
│  │  示例：                                              │   │
│  │  ├─ 原材料/商品采购                                  │   │
│  │  ├─ 销售提成                                         │   │
│  │  ├─ 营销推广费用                                     │   │
│  │  ├─ 配送费用                                         │   │
│  │  └─ 包装费用                                         │   │
│  │                                                       │   │
│  │  特点：                                               │   │
│  │  ├─ 每多卖一个产品，多一份成本                       │   │
│  │  └─ 通常与收入成比例                                 │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### Unit Economics分析图

```
┌─────────────────────────────────────────────────────────────┐
│                 盈亏平衡分析图                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│           收入/成本                                          │
│              ↑                                              │
│     200万 ─┤                      ╱ 收入线                 │
│              │                    ╱                        │
│     150万 ─┤                  ╱                            │
│              │              ╱                              │
│     100万 ─┤          ╱ ─────────────── 盈亏平衡点        │
│              │        ╱     ╱                               │
│      50万 ─┤      ╱    ╱  总成本线                         │
│              │    ╱   ╱                                   │
│        0 ──┼────┼───┼────→ 销量                            │
│              0   20   40   60   80   100                   │
│                                                             │
│  公式：                                                     │
│  ├─ 盈亏平衡销量 = 固定成本 / (单价 - 单位可变成本)        │
│  ├─ 超过平衡点后，每多卖一个产品，利润增加=单价-可变成本   │
│  └─ 规模效应：销量越大，单位固定成本越低                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.4 案例详解

#### 案例1：线下奶茶店单店模型

```
┌─────────────────────────────────────────────────────────────┐
│              奶茶店单店Unit Economics分析                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【单店收入】                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  单价：18元/杯                                       │   │
│  │  日均销量：100杯                                     │   │
│  │  月收入 = 18 × 100 × 30 = 54,000元                │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【可变成本】                                               │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  原材料成本：6元/杯                                  │   │
│  │  包装成本：2元/杯                                    │   │
│  │  营销成本：2元/杯（平均）                           │   │
│  │  单位可变成本合计：10元/杯                          │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【固定成本（每月）】                                       │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  房租：3,000元                                      │   │
│  │  人员薪资：15,000元（2-3人）                       │   │
│  │  合计：18,000元/月                                  │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【单店盈利计算】                                           │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  单位边际贡献 = 18 - 10 = 8元/杯                    │   │
│  │                                                       │   │
│  │  盈亏平衡点 = 18,000 / 8 = 2,250杯/月               │   │
│  │         = 2,250 / 30 = 75杯/天                      │   │
│  │                                                       │   │
│  │  当月销 > 2,250杯时盈利                             │   │
│  │  当月销 < 2,250杯时亏损                             │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【运营建议】                                               │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  1. 提升销量：日均100→150杯（月销4500）            │   │
│  │     利润 = (54,000×1.5) - 54,000 - 18,000         │   │
│  │          = 9,000元/月                              │   │
│  │                                                       │   │
│  │  2. 优化成本：降低营销成本至1.5元/杯                │   │
│  │     单位边际贡献 = 18 - 8.5 = 9.5元/杯             │   │
│  │     盈亏平衡点 = 18,000 / 9.5 = 1,895杯/月        │   │
│  │                                                       │   │
│  │  3. 提升客单价：推出22元新品                        │   │
│  │     单位边际贡献提升                                │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 核心概念总结

| 概念 | 定义 | 核心要点 |
|------|------|----------|
| 业务模型 | 业务如何运转 | 输入→过程→输出 |
| 全局业务模型 | 整体业务运转体系 | 增长模型+供给体系+效率系统 |
| 局部业务模型 | 某一环节的业务模型 | 明确的输入输出和上下游 |
| Unit Economics | 单位经济模型 | 单个经营单元的盈利能力 |
| 固定成本 | 不随产量变化的成本 | 房租、设备、人员基本薪资 |
| 可变成本 | 随产量变化的成本 | 原材料、销售提成、营销 |
| 盈亏平衡点 | 盈利vs亏损的临界点 | 销量达到即可盈利 |

## 本章小结

### 核心要点

1. **业务模型本质**
   - 稳定的输入→业务模型→稳定的输出
   - 是理解业务如何运转的核心框架

2. **全局vs局部业务模型**
   - 全局：整体业务的完整运转体系
   - 局部：某一环节单独梳理

3. **Unit Economics**
   - 评估单个经营单元的盈利能力
   - 固定成本+可变成本=总成本
   - 找到盈亏平衡点，优化盈利模型

4. **实战应用**
   - 分析自己业务的Unit Economics
   - 找到盈利优化的关键点

2. 计算你业务的Unit Economics
