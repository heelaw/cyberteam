# 三大商业模型详解（上）：商业模式与商业经营模型

> 所属章节：第一章第三小节（上）
> 核心内容：商业模式定义与类型、商业经营模型（财务模型）的核心概念

---

## 本章导学

### 核心目标

本章帮助学员建立对商业系统中三大核心模型的系统理解，掌握商业模式的基本类型，理解商业经营模型（财务模型）的核心概念和工具。

### 学习重点

- 理解商业模式的定义和常见类型
- 掌握商业经营模型的核心概念（毛利、业务利润、经营净利润）
- 理解核心经营单元的概念
- 学会用财务视角审视业务

### 2.1 三大模型概述

#### 商业系统的三大模型

```
┌─────────────────────────────────────────────────────────────┐
│                  商业系统三大核心模型                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────┐                                      │
│  │    商业模式      │  → 回答"如何赚钱"的基础问题          │
│  │  (Business Model)│  → 市场定位、产品服务、盈利方式     │
│  └────────┬────────┘                                      │
│           ↓                                                │
│  ┌─────────────────┐                                      │
│  │  商业经营模型    │  → 回答"经营效率"的问题             │
│  │(Business Logic) │  → 成本结构、利润产生、资源配置      │
│  └────────┬────────┘                                      │
│           ↓                                                │
│  ┌─────────────────┐                                      │
│  │    业务模型      │  → 回答"如何运转"的问题             │
│  │ (Operations Model)│ → 业务流程、协作机制、增长驱动     │
│  └─────────────────┘                                      │
│                                                             │
│  层层递进：商业模式 → 经营模型 → 业务模型                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 三大模型的关系

```
┌─────────────────────────────────────────────────────────────┐
│                    三大模型关系图                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  商业模式（顶层）                                           │
│  ├─ 定义：如何赚钱                                         │
│  ├─ 核心：市场选择 + 产品服务 + 盈利方式                   │
│  └─ 产出：收入公式                                         │
│           ↓                                                │
│  商业经营模型（中层）                                       │
│  ├─ 定义：如何高效赚钱                                     │
│  ├─ 核心：成本控制 + 利润优化 + 资源分配                   │
│  └─ 产出：财务指标（毛利、净利、人效）                     │
│           ↓                                                │
│  业务模型（底层）                                           │
│  ├─  定义：如何实现运转                                    │
│  ├─  核心：业务流程 + 协作机制 + 增长引擎                  │
│  └─  产出：业务成果 + 运营效率                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 商业模式详解

#### 什么是商业模式

```
┌─────────────────────────────────────────────────────────────┐
│                     商业模式定义                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  商业模式回答三个核心问题：                                 │
│                                                             │
│  1. 面向什么市场？                                         │
│     └─ 2C / 2B / 2G / 混合市场                           │
│                                                             │
│  2. 提供什么产品/服务？                                     │
│     └─ 实物产品 / 虚拟产品 / 服务 / 平台                   │
│                                                             │
│  3. 如何获利？                                             │
│     └─ 销售收入 / 佣金抽成 / 广告 / 增值服务              │
│                                                             │
│  本质：企业如何创造、传递、获取价值的基本框架              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 互联网时代的四大商业模式

```
┌─────────────────────────────────────────────────────────────┐
│               互联网时代四大商业模式                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【模式一】流量 + 广告变现                                   │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  逻辑：获取流量 → 展示广告 → 赚取广告费              │   │
│  │  公式：收入 = 广告展示量 × 点击率 × 单次点击价格      │   │
│  │  代表：搜索产品（百度）、社交产品（微信/FB）         │   │
│  │       媒体产品（今日头条、抖音）                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【模式二】平台 + 佣金抽成                                   │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  逻辑：搭建双边/多边平台 → 撮合交易 → 收取佣金        │   │
│  │  公式：收入 = GMV × 佣金率                           │   │
│  │  代表：电商平台（淘宝、天猫、京东）                   │   │
│  │       出行平台（滴滴）、外卖平台（美团）             │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【模式三】自营产供销                                        │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  逻辑：自建供应链 → 生产产品 → 自营销售 → 赚取差价    │   │
│  │  公式：收入 = 销量 × (单价 - 生产成本)               │   │
│  │  代表：制造业、零售业、在线教育（三节课）             │   │
│  │       亚马逊Kindle、苹果Apple Store                  │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【模式四】免费 + 增值服务                                   │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  逻辑：基础功能免费 → 增值服务收费                    │   │
│  │  公式：收入 = 免费用户×转化率 × 增值服务价格         │   │
│  │  代表：视频网站（爱奇艺、腾讯视频）                   │   │
│  │       游戏（王者荣耀）、社交（探探、Soul）            │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 不同商业模式的收入公式

```
┌─────────────────────────────────────────────────────────────┐
│               各商业模式收入公式对比                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  模式一：流量+广告                                          │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  收入 = DAU × 人均广告展示次数 × 广告点击率         │   │
│  │      × 单次点击价格                                  │   │
│  │                                                       │   │
│  │  优化方向：                                          │   │
│  │  ├─ 提升流量规模（DAU）                             │   │
│  │  ├─ 提升广告填充率                                  │   │
│  │  ├─ 优化点击率                                      │   │
│  │  └─ 提升单次点击价格                                │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  模式二：平台+抽成                                          │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  收入 = GMV × 佣金率                                │   │
│  │       = (用户数 × 客单价) × 佣金率                  │   │
│  │                                                       │   │
│  │  优化方向：                                          │   │
│  │  ├─ 提升平台用户数                                  │   │
│  │  ├─ 提升客单价                                      │   │
│  │  └─ 优化佣金率                                      │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  模式三：自营产供销                                         │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  收入 = 销量 × 客单价                               │   │
│  │       = (流量 × 转化率) × 客单价                    │   │
│  │                                                       │   │
│  │  优化方向：                                          │   │
│  │  ├─ 提升流量                                        │   │
│  │  ├─ 提升转化率                                      │   │
│  │  └─ 提升客单价                                      │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  模式四：免费+增值                                          │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  收入 = 免费用户数 × 转化率 × ARPU                  │   │
│  │                                                       │   │
│  │  优化方向：                                          │   │
│  │  ├─ 扩大免费用户池                                  │   │
│  │  ├─ 提升付费转化率                                  │   │
│  │  └─ 提升ARPU                                        │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 商业经营模型详解

#### 什么是商业经营模型

```
┌─────────────────────────────────────────────────────────────┐
│                   商业经营模型定义                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  商业经营模型 = 财务模型 = 如何高效经营企业                  │
│                                                             │
│  核心视角：                                                 │
│  ├─ 成本视角：钱花在哪里                                   │
│  ├─ 利润视角：如何赚钱                                     │
│  ├─ 效率视角：资源配置                                     │
│  └─ 评估视角：经营状况如何                                  │
│                                                             │
│  核心公式：                                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │           收入 - 成本 = 利润                          │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 财务三层次

```
┌─────────────────────────────────────────────────────────────┐
│                    财务三层次                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  第1层：毛利                                                │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  毛利 = 收入 - 产品/服务的直接成本                   │   │
│  │                                                       │   │
│  │  示例（餐饮店）：                                     │   │
│  │  毛利 = 菜品售价50元 - 食材成本15元 = 35元          │   │
│  │                                                       │   │
│  │  意义：评估产品盈利空间                              │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  第2层：业务利润                                            │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  业务利润 = 毛利 - 营销/销售/运营成本                │   │
│  │                                                       │   │
│  │  示例：                                              │   │
│  │  业务利润 = 35元 - 营销成本10元 - 销售提成5元       │   │
│  │         = 20元                                       │   │
│  │                                                       │   │
│  │  意义：评估业务盈利能力                              │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  第3层：经营净利润                                          │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  经营净利润 = 业务利润 - 固定成本                    │   │
│  │                                                       │   │
│  │  固定成本包括：                                      │   │
│  │  ├─ 房租/物业                                       │   │
│  │  ├─ 行政/管理                                       │   │
│  │  ├─ 研发/技术                                       │   │
│  │  └─ 税费                                             │   │
│  │                                                       │   │
│  │  示例：                                              │   │
│  │  经营净利润 = 20元 - 房租8元 - 管理5元 - 税费2元    │   │
│  │         = 5元                                         │   │
│  │                                                       │   │
│  │  意义：企业实际盈利水平                              │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 案例：教育公司财务模型

```
┌─────────────────────────────────────────────────────────────┐
│                教育公司财务模型详解                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【公司背景】                                               │
│  在线教育公司，提供职业技能培训课程                          │
│                                                             │
│  【收入端】                                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  收入 = 课程单价 × 付费用户数                        │   │
│  │       = 5000元 × 1000人 = 500万元                  │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【成本结构】                                               │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  1. 课酬成本（教学交付）                             │   │
│  │     = 1000元/人 × 1000人 = 100万元                  │   │
│  │     （每服务一个用户，支付讲师1000元）               │   │
│  │                                                       │   │
│  │  2. 研发成本（课程研发摊销）                        │   │
│  │     = 50万元（年度课程研发投入）                     │   │
│  │                                                       │   │
│  │  3. 营销成本（获客）                                │   │
│  │     = 2000元/人 × 1000人 = 200万元                  │   │
│  │     （获取一个付费用户平均花费2000元）               │   │
│  │                                                       │   │
│  │  4. 销售成本（转化团队）                            │   │
│  │     = 50万元（销售团队薪资+提成）                    │   │
│  │                                                       │   │
│  │  5. 固定成本（房租+管理+税费）                      │   │
│  │     = 80万元                                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【三层次利润计算】                                         │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  毛利 = 500万 - 100万 - 50万 = 350万               │   │
│  │        = 收入 - 课酬 - 研发                         │   │
│  │        = 70%  (行业标准≥65%)                        │   │
│  │                                                       │   │
│  │  业务利润 = 350万 - 200万 - 50万 = 100万           │   │
│  │           = 毛利 - 营销 - 销售                       │   │
│  │           = 20%                                      │   │
│  │                                                       │   │
│  │  经营净利润 = 100万 - 80万 = 20万                   │   │
│  │             = 业务利润 - 固定成本                     │   │
│  │             = 4%                                      │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.4 核心经营单元

#### 什么是核心经营单元

```
┌─────────────────────────────────────────────────────────────┐
│                  核心经营单元定义                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  核心经营单元 = 收入拆分的基本单位                          │
│                                                             │
│  定义：公司进行业务管理和收入拆解时的最小经营单位             │
│                                                             │
│  目的：                                                     │
│  ├─ 明确责任归属（谁来负责）                               │
│  ├─ 量化经营成果（做得怎样）                               │
│  ├─ 优化资源配置（资源给谁）                               │
│  └─ 评估投入产出（值不值得）                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 不同公司的核心经营单元

```
┌─────────────────────────────────────────────────────────────┐
│              不同类型公司的核心经营单元                       │
├─────────────────────────────────────────────────────────────┤

│  ┌─────────────────────────────────────────────────────┐   │

│  │  └─ 运营团队                                        │   │

│  │  考核指标：                                          │   │
│  │  ├─ 商家数                                           │   │
│  │  ├─ 订单量                                           │   │
│  │  ├─ 用户好评率                                       │   │
│  │  └─ GMV                                             │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【摩拜单车】                                               │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  经营单元：以城市/区域为单位                         │   │
│  │                                                       │   │
│  │  考核指标：                                          │   │

│  │  ├─ 订单量                                           │   │

│  │  团队配置：                                          │   │
│  │  ├─ 售前团队                                        │   │
│  │  ├─ 售后团队                                        │   │
│  │  └─ 客户成功团队                                     │   │
│  │                                                       │   │
│  │  考核指标：                                          │   │
│  │  ├─ 签约客户数                                       │   │
│  │  ├─ 续费率                                           │   │
│  │  └─ 收入                                             │   │

│                                                             │
│  【在线教育公司】                                            │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  经营单元：按课程/班期划分                           │   │

│  │  考核指标：                                          │   │
│  │  ├─ 付费人数                                         │   │
│  │  ├─ 完课率                                           │   │
│  │  ├─ 用户满意度                                       │   │
│  │  └─ 业务利润                                         │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【房地产中介】                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  经营单元：按门店划分                               │   │
│  │                                                       │   │
```

## 核心概念总结

| 概念 | 定义 | 核心要点 |
|------|------|----------|
| 商业模式 | 如何赚钱 | 市场选择+产品服务+盈利方式 |
| 商业经营模型 | 如何高效经营 | 成本控制+利润优化+资源配置 |
| 毛利 | 收入-直接成本 | 评估产品盈利空间 |
| 业务利润 | 毛利-营销销售成本 | 评估业务盈利能力 |
| 经营净利润 | 业务利润-固定成本 | 企业实际盈利水平 |
| 核心经营单元 | 收入拆分的基本单位 | 责任归属、量化成果 |
| 成本中心 | 不直接产生收入的部门 | 保障稳定+降本提效 |
| 利润中心 | 直接产生收入的部门 | 提升收入+优化ROI |

## 本章小结

### 核心要点

1. **四大商业模式**
   - 流量+广告、平台+抽成、自营产供销、免费+增值

2. **财务三层次**
   - 毛利（产品盈利空间）
   - 业务利润（业务盈利能力）
   - 经营净利润（实际盈利水平）

3. **核心经营单元**
   - 收入拆分的最小单位
   - 不同公司有不同的划分方式

4. **成本中心vs利润中心**
   - 成本中心：保障运转、降本提效
   - 利润中心：提升收入、优化ROI

2. 理解自己部门属于成本中心还是利润中心
