### 2.2 决策工具1：线性增长 vs 指数增长模型

#### 两种增长模型

```
┌─────────────────────────────────────────────────────────────┐
│               线性增长 vs 指数增长模型                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【线性增长模型】                                           │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                                                       │   │
│  │     收入                                              │   │
│  │       ↑                                              │   │
│  │       │          ╱ 线性增长                          │   │
│  │       │        ╱                                     │   │
│  │       │      ╱                                       │   │
│  │       │    ╱                                         │   │
│  │       │  ╱                                           │   │
│  │       └─────────────→ 投入/时间                      │   │
│  │                                                       │   │
│  │  特征：                                               │   │
│  │  ├─ 投入产出呈等比例关系                             │   │
│  │  ├─ 边际成本不变                                     │   │
│  │  └─ 增长依赖于资源堆砌                               │   │
│  │                                                       │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【指数增长模型】                                           │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                                                       │   │
│  │     收入                                              │   │
│  │       ↑                                              │   │
│  │       │                        ╲ 指数增长            │   │
│  │       │                      ╱                      │   │
│  │       │                    ╱                        │   │
│  │       │                  ╱                          │   │
│  │       │                ╱                             │   │
│  │       └────────────────→ 投入/时间                  │   │
│  │                                                       │   │
│  │  特征：                                               │   │
│  │  ├─ 投入产出呈指数关系                               │   │
│  │  ├─ 边际成本递减                                     │   │
│  │  └─ 增长来自模式创新或网络效应                       │   │
│  │                                                       │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 哪些业务是线性增长？

```
┌─────────────────────────────────────────────────────────────┐
│                   线性增长业务特征                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【典型线性增长业务】                                       │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  1. 2B销售驱动业务                                  │   │
│  │     ├─ 一个销售一年服务100个客户                    │   │
│  │     ├─ 10个销售=1000个客户                         │   │
│  │     └─ 增长依赖增加销售人数                         │   │
│  │                                                       │   │
│  │  2. 线下零售                                        │   │
│  │     ├─ 一家店支撑一定收入                           │   │
│  │     ├─ 10家店=10倍收入                              │   │
│  │     └─ 增长依赖增加门店数量                         │   │
│  │                                                       │   │
│  │  3. 线下教育                                        │   │
│  │     ├─ 一个老师服务一定学生                         │   │
│  │     ├─ 增加老师和教室=增加收入                      │   │
│  │     └─ 增长依赖资源堆砌                             │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  关键特征：                                                 │
│  ├─ 边际成本不为0                                        │
│  ├─ 无法实现边际成本递减                                  │
│  └─ 增长需要持续投入资源                                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 哪些业务是指数增长？

```
┌─────────────────────────────────────────────────────────────┐
│                   指数增长业务特征                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【典型指数增长业务】                                       │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  1. 社交产品                                        │   │
│  │     ├─ 用户邀请用户                                  │   │
│  │     ├─ 病毒式传播                                   │   │
│  │     └─ 边际成本趋近于0                              │   │
│  │                                                       │   │
│  │  2. 内容/工具产品                                   │   │
│  │     ├─ 一个内容/工具服务1人=服务1亿人               │   │
│  │     ├─ 研发成本固定                                 │   │
│  │     └─ 边际成本趋近于0                              │   │
│  │                                                       │   │
│  │  3. 平台业务                                        │   │
│  │     ├─ 双边网络效应                                 │   │
│  │     ├─ 供给越多，需求越多                           │   │
│  │     └─ 规模越大，效率越高                          │   │
│  │                                                       │   │
│  │  4. 电商平台                                        │   │
│  │     ├─ 品类越多，用户越多                           │   │
│  │     ├─ 用户越多，供给越多                           │   │
│  │     └─ 数据越多，推荐越准                          │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  关键特征：                                                 │
│  ├─ 边际成本趋近于0                                       │
│  ├─ 存在网络效应或规模效应                               │
│  └─ 增长来源于模式创新                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

│  │  分析：                                              │   │
│  │  ├─ 如果是线性增长 → 增长有上限                    │   │

│                                                             │
│  应用场景2：评估增长策略                                   │

│  │                                                       │   │
│  │  分析：                                              │   │
│  │  ├─ 线性增长策略 → 需要持续投入资源                │   │

│  │  分析：                                              │   │
│  │  ├─ 增长曲线是否趋于平缓？                         │   │
│  │  ├─ 边际成本是否开始上升？                         │   │
│  │  └─ 是否需要寻找新的增长引擎？                     │   │
│  └─────────────────────────────────────────────────────┘   │

└─────────────────────────────────────────────────────────────┘

```
┌─────────────────────────────────────────────────────────────┐
│                   成本收益模型核心                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  核心公式：                                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │         收益 - 成本 = 价值                          │   │
│  └─────────────────────────────────────────────────────┘   │

│  └─ 长期收益：未来可能带来的收益                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.4 决策工具3：头部效应模型

#### 头部效应的核心逻辑

```
┌─────────────────────────────────────────────────────────────┐
│                    头部效应模型                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  核心原理：                                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  在任何一个赛道，成为头部获得的收益                  │   │
│  │  远高于腰部、尾部                                  │   │
│  │                                                       │   │
│  │  差距可能是10倍、100倍、甚至1000倍                  │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  头部效应公式：                                            │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  头部收益 = 腰部收益 × N (N>>1)                   │   │
│  │                                                       │   │
│  │  头部集中度 = 第一名占据市场份额 ≥ 50%               │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 案例：彭小六的头部策略

```
┌─────────────────────────────────────────────────────────────┐
│              案例：彭小六如何利用头部效应                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【背景】                                                  │
│  2016年下半年开始做个人IP                                   │
│  微信公众号已是一片红海                                   │
│                                                             │
│  【分析】                                                  │
│  ├─ 微信公众号：已是成熟生态                               │
│  │   └─ 众多大V占据头部位置                            │
│  │                                                       │
│  ├─ 知乎：同样大V众多                                   │
│  │   └─ 竞争激烈                                        │
│  │                                                       │
│  └─ 简书：发展中的平台                                    │
│      └─ 在知识管理领域几乎无对手                          │
│                                                             │
│  【策略】                                                  │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  1. 选择简书作为主战场                              │   │
│  │     └─ 只有一个对手在该领域                         │   │
│  │                                                       │   │
│  │  2. 深度研究对手                                    │   │
│  │     └─ 每3天写1篇                                   │   │
│  │                                                       │   │
│  │  3. 制定超越策略                                    │   │
│  │     ├─ 对手3天写1篇 → 我1天写3篇                   │   │
│  │     ├─ 覆盖对手所有话题                             │   │
│  │     └─ 持续获取官方推荐                             │   │
│  │                                                       │   │
│  │  4. 持续巩固                                        │   │
│  │     └─ 超越后继续巩固3个月                          │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  【结果】                                                  │
│  ├─ 成为简书"一哥"                                        │
│  ├─ 获得"简书一哥"标签                                  │
│  ├─ 微信公众号涨粉速度飙升                               │
│  └─ 获得出版社、电视台等大量机会                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 头部策略的应用

┌─────────────────────────────────────────────────────────────┐
│                   头部策略的应用场景                          │

│  │  策略：                                             │   │
│  │  ├─ 策略1：退回二线战场，先成为二线头部             │   │
│  │  ├─ 策略2：换一个细分赛道                           │   │
│  │  └─ 策略3：等待机会，等待格局变化                  │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.5 决策工具4：杠铃策略与黑天鹅策略

#### 核心概念

```
┌─────────────────────────────────────────────────────────────┐
│               杠铃策略与黑天鹅策略                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【杠铃策略】                                               │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  核心理念：分散风险                                  │   │
│  │                                                       │   │
│  │  结构：                                              │   │
│  │  ├─ 90%资源 → 确定性高的事情（保底）              │   │
│  │  └─ 10%资源 → 高风险高回报的事情（博取上限）       │   │
│  │                                                       │   │
│  │  例子：                                              │   │
│  │  ├─ 投资：90%货币基金 + 10%高风险股票              │   │
