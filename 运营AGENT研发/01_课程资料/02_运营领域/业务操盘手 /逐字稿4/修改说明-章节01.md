#### 2. 补充详细的财务公式
**位置**：
- 1.2 如何评估和预测一家公司的价值（补充教育公司案例）
- 1.3.2 财务模型的核心公式示例（新增小节）

**中早期公司公式**：
```
收入 = 流量 × 转化率 × 客单价
成本 = 营销成本 + 人力成本 + 运营成本
利润 = 收入 - 成本
```

**成熟公司公式**：
```
每个板块价值 = 客户数量 × ARPU（单客平均收入）- 服务成本
增长 = 新客增长 + 老客复购 - 客户流失
总收入 = Σ（各板块收入）
毛利 = 收入 - 直接成本（生产/交付成本）
业务利润 = 毛利 - 营销/销售成本
经营净利润 = 业务利润 - 固定成本（房租/行政/税收）
```
