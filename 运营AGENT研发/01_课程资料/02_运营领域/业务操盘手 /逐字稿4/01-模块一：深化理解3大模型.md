## 本章导学

### 核心目标

本章是洞悉3大商业模型课程的入门篇章，核心目标是帮助学员深化理解业务模型、运营模型和组织模型这3大核心模型，以及它们之间的关系。掌握这些模型能够帮助运营人员更好地理解业务、制定策略、提升管理能力。

### 学习重点

本章的学习重点包括：理解3大模型的定义和关系；掌握业务模型的核心要素；学会梳理整体增长业务模型。

## 详细内容

### 1.1 深化理解3大模型及其关系

**3大模型概述**：

```
┌─────────────────────────────────────────────────────────────┐
│                   商业3大模型                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────┐       │
│   │  1. 业务模型                                     │       │
│   │     做什么：提供什么产品/服务                    │       │
│   │     卖给谁：目标客户是谁                         │       │
│   │     怎么卖：销售渠道和方式                       │       │
│   └─────────────────────────────────────────────────┘       │
│                         ↓                                    │
│   ┌─────────────────────────────────────────────────┐       │
│   │  2. 运营模型                                     │       │
│   │     怎么做：如何获取用户、留存、变现            │       │
│   │     核心：流量、转化、留存                      │       │
│   └─────────────────────────────────────────────────┘       │
│                         ↓                                    │
│   ┌─────────────────────────────────────────────────┐       │
│   │  3. 组织模型                                     │       │
│   │     谁来做：团队结构、职责分工                   │       │
│   │     支撑：流程、文化、激励机制                   │       │
│   └─────────────────────────────────────────────────┘       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**3大模型的关系**：

- 业务模型是基础：定义了做什么生意
- 运营模型是引擎：让业务模型运转起来
- 组织模型是支撑：提供人和流程的保障

### 1.2 业务模型的概念与全局业务模型的构成

**业务模型的构成要素**：

| 要素 | 问题 | 示例 |
|------|------|------|
| 客户细分 | 服务谁 | 职场人士、宝妈 |
| 价值主张 | 提供什么价值 | 省钱、省时间 |
| 渠道通路 | 如何触达 | APP、公众号 |
| 客户关系 | 如何维护 | 社群、客服 |
| 收入来源 | 如何赚钱 | 会员费、佣金 |
| 核心资源 | 需要什么 | 技术、内容 |
| 关键业务 | 做什么 | 研发、运营 |
| 成本结构 | 花多少钱 | 人力、推广 |

### 1.3 如何梳理整体增长业务模型

**增长业务模型梳理框架**：

```
┌─────────────────────────────────────────────────────────────┐
│                 增长业务模型框架                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   第一层：流量来源                                          │
│   ├─ 付费流量：广告投放                                     │
│   ├─ 免费流量：内容、SEO                                    │
│   └─ 私域流量：社群、会员                                   │
│           ↓                                               │
│   第二层：流量转化                                          │
│   ├─ 首次转化：注册、下单                                   │
│   └─ 持续转化：复购、升级                                   │
│           ↓                                               │
│   第三层：用户留存                                          │
│   ├─ 激活：新用户激活                                       │
│   ├─ 留存：防止流失                                         │
│   └─ 活跃：提升使用频率                                     │
│           ↓                                               │
│   第四层：价值变现                                          │
│   ├─ 直接变现：销售收入                                     │
│   └─ 间接变现：广告、数据                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 案例分析

### 案例：电商平台业务模型分析

**背景**：一家专注职场教育的电商平台

**业务模型分析**：

1. **客户细分**：职场人士、想要提升技能的人
2. **价值主张**：碎片化学习、实战课程
3. **渠道通路**：公众号、小程序、APP
4. **收入来源**：课程销售、会员费

**运营模型分析**：

1. **流量获取**：广告投放+内容营销
2. **转化**：低价引流课+正价课
3. **留存**：社群+会员服务
4. **变现**：课程收入+广告

**组织模型分析**：

1. **团队结构**：教研、运营、技术、销售
2. **激励机制**：绩效考核+奖金
3. **文化**：学习成长、用户第一

**分析要点**：

1. **模型要匹配**：业务、运营、组织模型要一致
2. **核心环节**：找到增长的关键环节
3. **持续优化**：根据数据不断调整模型
