# 案例3 - 某电商公司

我们再快速再来看最后一个例子。第三个业务模型是说一个电商公司的这种业务模型，这家电商公司它有线上和线下两个部分，线下它是有门店的，而在线上的部分它通常是说线上我首先有一些用户池，也有一些这种流量转化路径，同时我会把一部分线下的这样的用户，在线下门店里边的用户，我会引导转化到线上的场景里边去，让他在线上来去最终来获得付费，约是这么一个这种业务的逻辑。

然后在生产交付的部分方面，反正他在下边就构成了有几套的这种供应链管理体系和几套的这样的这种系统，在下面来去支持它，这是这家公司它的业务模型，它的年度目标可能说比如上级提了一个年度目标，是一个什么样的目标，是说我的线上业务要保持三倍增长，在它年度的战略目标要把线上这块业务要去做起来。

以上，同理我们也是简单查看它的收入公式和成本收益的公式。收入公式十分简单的一个逻辑基本说我线上收入假设我觉得我们把它看作极简单，流量成转化成APP，约是这么一个认为，无非我流量有各种不同的端，各种不同渠道来的

当然看模型，它每个渠道来的这种流量，它的核心的转化手段应该没有太大的区分。

至少图告诉我们是这样的，然后我的成本收益的公式说利润会等于销售额减去我们的销售成本，减去生产成本，减去供应链成本，再减去下门店成本，还有其他的成本，对本章是这么一个认为，因为我线上的我的订单有一部分来自于线下导给我的用户，但线下导给我的用户这部分他应该有一些线下门店成本会被摊销在里面，他应该是这么一个认为。

以上，如果是在这么一个背景下，我们就follow公式，线上业务怎么实现三倍增长，我们在增长导向的业务模型部分，也许可去思考的说扩大线上渠道，加强线下门店的导流，提升用户的转化，提高用户的复购，对也许都是可去着力的一些这种工作的发力点。

而在我们的下边的这种交付供应链保障的部分，也许可去关注的如何提升我们供应链的响应效率，如何降低仓储和运输的成本，如何做似乎选品以及如何做好我们线上线下的数据打通，数据打通最终指向的说我们例如面向用户的这种什么个性化的推荐，它能得到更加的精准，然后本质上它这种导向是这么一个认为，所以最终它指向的也是我们的像例如转化率的提升，本质上这么一个方面。

，所以这几个例子能看下来之后，希望各位对于我们的前面讲到的就这几个模型，一家公司的商业模式，业务模型，商业经营模型对到底会怎么跟我们要做的目标和工作规划结合起来，希望各位在部分就有更深刻的认知了。
