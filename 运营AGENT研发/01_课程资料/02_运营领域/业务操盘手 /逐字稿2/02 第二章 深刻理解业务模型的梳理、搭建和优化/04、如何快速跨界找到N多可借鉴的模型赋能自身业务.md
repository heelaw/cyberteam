# 04、如何快速跨界找到N多可借鉴的模型赋能自身业务

### &#x20;如何快速跨界找到可借鉴业务模型赋能自身业务.mp4

第四小节是如何快速跨界找到n多可借鉴的模型来赋能你自身的业务。第四小节核心讲的是个什么东西各位还有印象，我们在第三小节里边提到了，如何在业务当中从零搭建一个模型的部分，有一个重要的思考逻辑是说叫做先找到外部的模型，再看它能否帮助你获得稳定的 Input output，

然后约讲过这么一种思考的逻辑，在逻辑面前一定会很多人会存在一个问题，我们要在外部去找模型对到底怎么找？到底怎么从外部去找到可借鉴的模型，它对我而言会更高效对那本节我们要讲的重点告诉各位，尤其是如果你在一家公司里面，在负责一个独立的业务模块的时候，模块现在我们要建立一些模型，要解决一些重要的问题，但是我并不知道这样的问题该怎么解决会更因此，然后想在外边去找一些参照，这时候怎么高效的能从外部找到各种可参照的可借鉴的业务模型，这是本集要帮各位重点解决的问题，部分也十分重要，十分有价值。
