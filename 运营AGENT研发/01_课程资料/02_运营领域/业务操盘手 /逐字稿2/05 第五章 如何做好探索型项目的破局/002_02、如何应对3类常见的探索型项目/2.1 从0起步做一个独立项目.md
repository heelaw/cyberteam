# 2.1 从0起步做一个独立项目

首先先看第一个，从零起步做一个独立项目。，从零起步做一个独立项目，我们还是回到这张图。这张图里面假设我们把说我们要有的三类假设要分别归一个数字做代号，商业价值的构想一，商业产品用户价值和业务逻辑假设二，执行运作方式的构想三。

，再一个从零起步，我们独立来去探索一个项目的这样的过程当中，我们的工作的方法和我们思考的逻辑说我们要先基于二产生用户价值的假设，就像我们要做一款新的产品对我们一定要先想象说产品到底是按照一个什么逻辑来稳定地产生了用户的价值的。，我们要先想这么一件事儿。

，然后这件事儿我们可能有了一个基本的想象，就像我们刚才给各位举的例子，说操盘手计划这门课程对它应该是通过这么一种内容的结构，通过这样的一些这种训练的方式和教学的这种逻辑，对来帮助用户可习得说像对业务模型深刻的认知之类的，我们要先有假设，然后往下推演出三，形成我们的这种工作计划，对在2和3之间不断的可能去做验证，在2和3之间不断的去做思考校正，然后不断验证，直到二的模型稳定之后，我们我们让二变成定量，然后最后基于定量二再来去思考一。

，如果我们是从零起步，要探索一个独立项目，我们的思考逻辑它是这样的。

这儿可能给各位举一个例子，绝大部分创业公司都是这么一个过程。以三节课来看，三节课是从2014年就开始，然后前身是一个公益组织，2015年年底的时候我们正式出来创业对然后说我们在创业之前，然后或者在2015年2015年年底刚开始出来的时候，对当时我们并没有太多关于商业价值的想象。

当时我们就想做一件事儿，就想说我们如果就做线上课程，对到底能不能通过线上的课程带给我们的用户真实的能力提升，我们就先思考这样一件事儿。

，所以通常从2015年一直到2017年这段时间里边，我们完全没有想说三节课到底怎么才能赚钱，或三节课的商业价值它的上限在里，完全没有想，我们就完全只想到说我们的课程到底怎么能带给用户真实的能力的提升，而通常是从2017年之后，我们才有了关于商业价值的这样的想象，而开始有想象的前提，通常我们的产品的这种交付，产品的生产，然后它通常稳定了。

所以到时候我们有了稳定的产品价值，

有了稳定的产品层面，从产品的生产到我们的服务的业务逻辑之后，对随后我们才去想说这家公司它的商业价值，它的商业模式应该是怎样的，怎么去设计它，怎么让它更有想象力，

所以处理个是这么一个过程，绝大部分反正创业公司的发展的路径也是这么一个路径，或者很多例如在大厂里边独立拿出来去做的这种新的产品，新的项目，往往也会是这么一个这种逻辑，所以这是第一类的探索性项目。从零开始独立来做一个项目。
