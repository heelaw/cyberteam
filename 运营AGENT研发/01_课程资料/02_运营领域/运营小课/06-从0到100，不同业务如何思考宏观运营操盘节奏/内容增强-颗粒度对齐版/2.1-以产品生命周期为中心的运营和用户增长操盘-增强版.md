## 一、课程定位：从What/Why到How

在第一章中，我们讲解了产品生命周期的存在及其背后的逻辑（What和Why）。

在本章中，我们将深入探讨**How**——如何以产品生命周期为中心，做好运营和用户增长的操盘？

**核心问题：**
1. 在产品生命周期的每个阶段，适合重点依赖哪些运营手段和渠道？

## 二、产品生命周期四个阶段的运营策略

### 2.1 引入期（探索期）的运营策略

#### 核心特征

**阶段定位：** 产品刚推向市场，用户不了解、不信任，增长不可能快。

**运营现象：**
- 营销推广非常费劲
- 大众用户不感兴趣
- 增长缓慢，甚至停滞

**核心策略：**
1. **验证产品可行性**：确认产品能给用户带来价值
2. **锁定种子用户**：找到早期市场用户
3. **积累用户口碑**：在早期用户中建立口碑

---

#### 引入期的用户运营框架

**用户认知：小众化**
- **目标：** 不需要很多人知道，控制在特定圈层
- **渠道：** 垂直渠道、小众社区
- **传播：** 口碑传播，非大众传播

**用户构成：精英化**
- **特征：** 高容错性、高参与度、高传播意愿
- **类型：** 技术狂热者、高瞻远瞩者
- **策略：** 深度服务，建立情感连接

**用户预期：低预期**
- **设定：** 60分预期（不要过度承诺）
- **表达：** 诚实告知产品不完善，邀请共同成长
- **管理：** 控制预期，避免过度宣传

**用户体验：中高体验**
- **目标：** 70-80分体验（超出预期）
- **策略：** 超预期服务
- **结果：** 产生口碑

**口碑公式：**
```
口碑 = 用户体验 - 用户预期

当 用户体验 > 用户预期 → 产生口碑
当 用户体验 < 用户预期 → 产生负面评价
```

---

#### 引入期的运营手段与渠道

**核心渠道：**

**渠道1：垂直渠道**
- 线上：垂直论坛、专业社区、兴趣小组
- 线下：行业聚会、沙龙、展会
- **示例：** 科技产品上36氪、虎嗅，母婴产品上宝宝树

**渠道2：社会化媒体**
- 微信群、QQ群
- 知乎、豆瓣、小红书
- **策略：** 精准定位，深度互动

**渠道3：口碑推荐**
- 邀请制
- 早期用户邀请朋友
- **策略：** 给予邀请奖励，但控制规模

**核心手段：**

**手段1：种子用户运营**
- 建立用户群
- 一对一深度沟通
- 收集用户反馈
- **目标：** 让早期用户成为产品的"合伙人"

**手段2：内容传播**
- 讲述产品理念
- 分享创业故事
- 展示产品愿景
- **目标：** 吸引认同理念的用户

**手段3：垂直PR**
- 行业媒体报道
- KOL合作
- **目标：** 在垂直领域建立影响力

**手段4：小规模营销活动**
- 线下沙龙
- 闭门测试
- **目标：** 精准获取种子用户

---

#### 引入期经典案例：橙子电商的故事

**【案例背景】**

某生活购物网站刚上线，主要卖橙子。

**【事件经过】**

**第一步：用户购买**
我朋友爱人在网站刚上线时买了一箱橙子。

**第二步：发现问题**
吃了一周左右，吃了一半多，发现有一个橙子是坏的。

**第三步：用户投诉**
朋友爱人打电话给网站客服：
> "我在你们这买了一箱橙子，吃了一周多了，发现有一个是坏的。"

**第四步：网站回应**
客服沉默了几秒，然后说：
> "不然您看这样吧，之前那一箱橙子我们就免费了，不要钱了。同时，为了弥补给您带来的不好体验，我们免费再送您一箱。"

**【案例分析】**

**普通商家的逻辑：**
- 用户都吃了一周了，哪知道是在哪坏的
- 可能是无理取闹
- 拒绝或道歉即可

**这家网站的逻辑：**
- 超预期服务
- 把投诉用户变成忠实粉丝
- 通过超预期服务建立口碑

**【结果】**

朋友爱人从此成为网站的死忠粉：
- 在所有聚会上孜孜不倦地安利这个网站
- 主动向所有朋友推荐
- 成为网站的"口碑大使"

**【投入产出分析】**

**投入：**
- 免费送2箱橙子（成本约100元）

**产出：**
- 获得一个死忠粉
- 带来至少10-20个新用户（保守估计）
- 每个新用户LTV约500元
- 总价值：5000-10000元

**ROI：**
```
ROI = (产出 - 投入) / 投入
ROI = (7500 - 100) / 100 = 7400%
```

**【核心启示】**

在产品早期，最重要的是：
1. **锁定核心用户**：找到高价值、高容错的早期用户
2. **超预期服务**：用超出预期的体验建立口碑
3. **口碑驱动增长**：让早期用户成为传播者

**【可复制公式】**
```
早期口碑 = (超预期服务 × 核心用户数) × 传播系数

超预期服务 = 服务体验 - 用户预期
核心用户数 = 产品曝光 × 转化率 × 留存率
传播系数 = 每个用户平均带来的新用户数
```

---

#### 引入期种子用户的两大类型

**类型1：有巨大痛点的用户**

**特征：**
- 在某个问题上需求异于常人地强烈
- 愿意容忍产品早期的各种问题
- 愿意陪伴产品一起成长

**【案例：滴滴早期的天使用户】**

**背景：** 2012-2013年，滴滴刚上线，北京最早的乘客用户是谁？

**答案：** 中关村软件园、西二旗的程序员们

**场景还原：**
- **时间：** 晚上12点-2点
- **地点：** 中关村软件园、西二旗
- **环境：** 大雪纷飞，寒风刺骨
- **问题：** 加班结束，打不到车

**滴滴出现前：**
- 唯一选择：约黑车
- 黑车价格：贵（平时2-3倍，恶劣天气5-10倍）
- 黑车问题：不安全、无发票、无保障

**滴滴出现后：**
- 提前在办公室约车
- 价格明确，不超过黑车
- 安全、有发票、有保障

**用户心态：**
> "只要不比打黑车贵，我就愿意用！"
> "哪怕叫车成功率只有50%，也比在雪地里站半小时强！"

**结论：**
这帮程序员是滴滴早期的天使用户，因为他们在"深夜打车"这件事上有巨大痛点。

**【如何找到有巨大痛点的用户】**

**方法1：场景分析**
- 列出产品使用的典型场景
- 识别哪些场景下痛点最强
- 定位这些场景下的用户

**方法2：竞品差评分析**
- 分析竞品用户的差评
- 找出用户最不满意的地方
- 这些就是最大痛点

**方法3：用户访谈**
- 深度访谈10-20个目标用户
- 了解他们的真实痛点
- 识别痛点最强烈的用户

---

**类型2：对产品理念高度认同的用户**

**特征：**
- 认同产品的文化、理念、价值观
- 愿意与产品共同成长
- 愿意付出时间和精力参与

**【案例：小米早期的米粉】**

**背景：** 2010年，小米刚成立，MIUI系统刚发布。

**小米的理念：**
- "为发烧友而生"
- "做中国最好的安卓系统"
- "与用户一起做产品"

**米粉的行为：**
- 主动在论坛反馈Bug
- 提出产品改进建议
- 帮忙解答其他用户问题
- 主动向朋友推荐小米
- 参与小米线下活动

**米粉的价值：**
- 节省研发成本：用户帮忙测试
- 提升产品体验：用户提建议
- 扩大品牌影响：用户主动传播
- 降低获客成本：用户主动拉新

**【如何找到理念认同的用户】**

**方法1：明确并传播产品理念**
- 清晰定义产品理念
- 在各种渠道传播理念
- 吸引认同理念的用户

**方法2：建立社群**
- 建立用户社群（微信群、QQ群、论坛）
- 在社群中传播理念
- 识别高度认同的用户

**方法3：邀请制**
- 采用邀请制注册
- 让早期用户邀请同理念用户
- 形成同理念用户圈层

---

#### 引入期关键指标

**指标1：产品可行性指标**
```
核心功能使用率 = (使用核心功能的用户数 / 总用户数) × 100%
```
**标准：** > 60%

**指标2：用户留存指标**
```
次日留存 = (次日活跃用户 / 当日新增用户) × 100%
7日留存 = (第7日活跃用户 / 当日新增用户) × 100%
```
**标准：** 次留 > 40%，7留 > 30%

**指标3：用户满意度指标**
```
NPS = (推荐者占比 - 贬损者占比) × 100%
```
**标准：** NPS > 40

**指标4：用户参与度指标**
```
人均使用时长 = 总使用时长 / 活跃用户数
人均使用次数 = 总使用次数 / 活跃用户数
```
**标准：** 根据产品类型设定基准

### 2.2 成长期的运营策略

#### 核心特征

**阶段定位：** 产品初步验证，开始具备快速增长趋势。

**运营现象：**
- 部分用户开始自发口碑推荐
- 产品具备快速增长趋势
- 即使不做推广，增长曲线也有自然上扬趋势

**核心策略：**
1. **放量补贴**：如果资源充沛，放量补贴快速占领市场
2. **效果投放**：做效果广告投放，实现规模化增长
3. **爆发活动**：策划爆发性活动/事件，放大知名度

---

#### 成长期的用户运营框架

**用户认知：初步大众化**
- **目标：** 让更多大众用户知道产品
- **渠道：** 社会化媒体 + 效果媒体
- **传播：** 功能价值 + 口碑传播

**用户构成：两分法**
- **核心用户：** 重点维护，强化体验，让他们成为口碑来源
- **普通用户：** 保障基础体验，控制负面反馈

**用户预期：中等预期**
- **设定：** 80分预期（产品还不错，值得尝试）
- **表达：** 强调产品功能和价值，展示用户案例
- **管理：** 避免过度承诺，但可以适度宣传

**用户体验：分层体验**
- **核心用户：** 强化体验，超预期服务
- **普通用户：** 保障基础体验，避免负面反馈

**用户分层公式：**
```
用户分层 = 根据用户价值分层（高价值/中价值/低价值）

高价值用户（前20%）：重点投入，深度服务
中价值用户（中间30%）：适度服务，提升价值
低价值用户（后50%）：基础服务，控制成本
```

---

#### 成长期的运营手段与渠道

**核心渠道：**

**渠道1：社会化媒体**
- 微博、微信、抖音、快手等
- **策略：** 内容营销 + KOL合作
- **目标：** 扩大品牌知名度

**渠道2：效果类媒体**
- 信息流广告、搜索广告、应用商店广告
- **策略：** 数据驱动，优化ROI
- **目标：** 规模化获客

**渠道3：应用商店**
- App Store、Google Play、各大安卓市场
- **策略：** ASO优化，冲榜单
- **目标：** 自然流量增长

**核心手段：**

**手段1：补贴拉新**
- 新用户红包
- 邀请奖励
- **示例：** 滴滴早期红包拉新
- **公式：**
  ```
  拉新成本 = 补贴金额 / 新增用户数
  ROI = (用户LTV / 拉新成本)
  当 ROI > 1，可持续
  ```

**手段2：效果广告投放**
- 信息流广告
- 搜索广告（SEM）
- **策略：** A/B测试，持续优化
- **公式：**
  ```
  CPA = 广告费用 / 转化数
  ROI = (用户LTV / CPA)
  当 ROI > 1，可持续
  ```

**手段3：爆发性活动**
- 节日营销
- 事件营销
- **示例：** 支付宝"集五福"活动
- **目标：** 短期内大量曝光

**手段4：内容营销**
- 短视频
- 直播
- 公众号文章
- **目标：** 长期品牌建设

---

#### 成长期关键指标

**指标1：增长速度指标**
```
月增长率 = (本月新增用户 / 上月用户总数) × 100%
```
**标准：** > 30%（成长期高速增长）

**指标2：市场份额指标**
```
市场占有率 = (日活跃用户 / 目标市场总用户) × 100%
```
**标准：** 持续上升

**指标3：获客成本指标**
```
CAC = 营销费用 / 新增用户数
```
**标准：** CAC持续下降，或CAC < LTV/3

**指标4：品牌知名度指标**
```
品牌知名度 = (听说过品牌的人数 / 调查总人数) × 100%
```
**标准：** 持续上升

### 2.3 成熟期的运营策略

#### 核心特征

**阶段定位：** 产品已有大量用户，增长开始放缓。

**运营现象：**
- 用户量增长放缓
- 市场趋向饱和
- 竞争进一步加剧

**核心策略：**
1. **商业化探索**：加强变现能力
2. **精细化运营**：用户分层，提升LTV
3. **用户体系搭建**：会员体系、积分体系

---

#### 成熟期的用户运营框架

**用户认知：全面大众化**
- **目标：** 强化市场领导地位认知
- **渠道：** 全渠道覆盖
- **传播：** 品牌文化 + 行业权威

**用户构成：精细化分层**
- **分层维度：** 价值、活跃度、生命周期
- **运营策略：** 不同用户不同策略

**用户预期：高预期**
- **设定：** 90分预期（市场领导者，最专业）
- **表达：** 强调行业地位和权威性
- **管理：** 建立品牌壁垒

**用户体验：标准化体验**
- **目标：** 所有用户获得符合预期的体验
- **策略：** 标准化服务流程
- **保障：** 客服体系、应急机制

**用户分层公式：**
```
精细分层 = 价值维度 × 活跃度维度 × 生命周期维度

示例：
高价值高活跃：VIP服务，专属客服
高价值低活跃：召回策略，激活策略
低价值高活跃：提升价值策略
低价值低活跃：基础服务，控制成本
```

---

#### 成熟期的运营手段与渠道

**核心渠道：**

**渠道1：大众媒体**
- 电视广告
- 户外广告
- **策略：** 品牌广告，强化认知
- **目标：** 建立大众影响力

**渠道2：效果类媒体**
- 继续投放效果广告
- **策略：** 精准投放，优化ROI
- **目标：** 维持增长

**核心手段：**

**手段1：用户体系搭建**
- 会员体系
- 积分体系
- 等级体系
- **公式：**
  ```
  用户留存提升 = (会员用户留存 - 普通用户留存) × 会员用户占比
  ```

**手段2：精细化运营**
- 用户分层
- 个性化推荐
- 精准营销
- **公式：**
  ```
  精细化ROI = (精细化运营收入 - 精细化运营成本) / 精细化运营成本
  ```

**手段3：用户生命周期管理**
- 新用户：引导转化
- 活跃用户：提升价值
- 沉默用户：召回激活
- 流失用户：挽回
- **公式：**
  ```
  用户LTV = Σ(每月ARPU × 留存率)
  ```

**手段4：商业化探索**
- 增值服务
- 广告变现
- 电商变现
- **公式：**
  ```
  变现效率 = 收入 / 活跃用户数
  ```

---

#### 成熟期关键指标

**指标1：盈利能力指标**
```
利润率 = (收入 - 成本) / 收入 × 100%
```
**标准：** 持续提升

**指标2：用户价值指标**
```
ARPU = 收入 / 活跃用户数
LTV = Σ(每月ARPU × 留存率)
```
**标准：** 持续提升

**指标3：用户活跃度指标**
```
DAU/MAU = 日活跃用户 / 月活跃用户
```
**标准：** > 0.2（健康）

**指标4：用户流失率指标**
```
月流失率 = (流失用户 / 期初用户) × 100%
```
**标准：** < 5%（健康）

---

### 2.4 衰退期的运营策略

#### 核心特征

**阶段定位：** 新产品出现，用户开始流失。

**运营现象：**
- 核心用户开始转移到竞品
- 用户量持续下降
- 增长停滞或负增长

**核心策略：**
1. **用户生命周期管理**：加强维系和召回
2. **新业务探索**：寻找第二增长曲线
3. **产品转型/创新**：适应市场变化

---

#### 衰退期的运营策略

**策略1：用户生命周期管理**
- **新用户：** 获客成本高，谨慎投入
- **活跃用户：** 重点维系，防止流失
- **沉默用户：** 召回激活
- **流失用户：** 挽回调研

**策略2：新业务探索**
- **内部创新：** 孵化新业务
- **外部投资：** 投资相关领域
- **并购整合：** 收购创新公司
- **示例：** 腾讯从QQ到微信

**策略3：产品转型**
- **功能升级：** 适应新需求
- **定位调整：** 切换细分市场
- **商业模式：** 探索新变现方式
- **示例：** 诺基亚从功能机转向智能机（失败案例）

---

#### 衰退期关键指标

**指标1：流失率指标**
```
月流失率 = (流失用户 / 期初用户) × 100%
```
**标准：** < 10%（可控），> 20%（危险）

**指标2：剩余价值指标**
```
剩余价值 = (当前用户数 × 单用户价值) - 维持成本
```
**标准：** > 0（可维持），< 0（建议退出）

**指标3：新业务进展指标**
```
新业务收入占比 = (新业务收入 / 总收入) × 100%
```
**标准：** 持续上升

## 四、运营操盘的三大核心定律

### 4.1 运营的本质：管理用户的认知、预期、体验

**运营定义：**
运营是围绕产品/业务的有限生命周期，通过一系列与用户产生连接或互动的手段，通过管理用户的**认知、预期、构成、体验**，最终实现商业目的。

**核心公式：**
```
运营成功 = (用户认知管理 × 0.25) + (用户预期管理 × 0.25) + (用户构成管理 × 0.25) + (用户体验管理 × 0.25)
```

---

### 4.2 运营的节奏原则

**核心原则：**
不同阶段要有不同的节奏，不能错配。

**节奏错配公式：**
```
错配程度 = |当前生命周期阶段 - 运营策略阶段|

错配程度 = 0：完美匹配
错配程度 = 1：可以接受
错配程度 ≥ 2：危险，可能导致失败
```

**示例：**
- 引入期就做成长期的事（大规模投放）：错配程度 = 1，风险高
- 成长期还做引入期的事（小范围测试）：错配程度 = 1，错过机会
- 引入期就做成熟期的事（精细化运营）：错配程度 = 2，资源浪费

---

### 4.3 运营的资源分配原则

**核心原则：**
在产品早期，最重要的不是推广和增长，而是：
1. 锁定一类用户
2. 把推广预算拿回来补贴给他们
3. 让他们获得超预期体验
4. 从而产生口碑

**资源分配公式：**
```
早期阶段：
推广预算分配 = (种子用户补贴 × 0.7) + (精准渠道投放 × 0.3)

成长期：
推广预算分配 = (效果广告 × 0.6) + (补贴拉新 × 0.3) + (品牌建设 × 0.1)

成熟期：
推广预算分配 = (品牌广告 × 0.4) + (效果广告 × 0.3) + (用户维系 × 0.3)
```

## 五、本章核心要点总结

### 5.1 产品生命周期四个阶段的运营策略

**引入期：**
- 核心：验证产品，积累口碑
- 渠道：垂直渠道、社会化媒体
- 手段：种子用户运营、口碑传播

**成长期：**
- 核心：快速增长，抢占市场
- 渠道：社会化媒体 + 效果媒体
- 手段：补贴拉新、效果广告、爆发性活动

**成熟期：**
- 核心：商业化，精细化运营
- 渠道：效果媒体 + 大众媒体
- 手段：用户体系、精细化运营、用户生命周期管理

**衰退期：**
- 核心：控制流失，寻找第二曲线
- 渠道：精准渠道、召回渠道
- 手段：用户维系、新业务探索

### 5.2 用户运营的四维管理

**用户认知管理：**
- 引入期：小众化、精准
- 成长期：初步大众化
- 成熟期：全面大众化

**用户构成管理：**
- 引入期：精英化、高容错
- 成长期：两分法（核心用户 + 普通用户）
- 成熟期：精细化分层

**用户预期管理：**
- 引入期：低预期（60分）
- 成长期：中等预期（80分）
- 成熟期：高预期（90分）

**用户体验管理：**
- 引入期：超预期体验（70-80分）
- 成长期：分层体验（核心用户强化，普通用户基础）
- 成熟期：标准化体验

### 5.3 运营操盘的核心公式

```
口碑 = 用户体验 - 用户预期
```
当体验 > 预期，产生口碑

```
ROI = 用户LTV / 获客成本
```
当 ROI > 1，可持续增长

```
增长健康度 = (留存率 × 0.4) + (自然增长占比 × 0.3) + (NPS × 0.3)
```
健康度 > 0.6 为健康
