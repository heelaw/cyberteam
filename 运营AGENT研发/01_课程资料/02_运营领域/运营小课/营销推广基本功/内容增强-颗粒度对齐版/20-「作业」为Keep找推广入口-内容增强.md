## 一、作业背景

### 1.1 场景设定

**你的角色：**
Keep的运营推广负责人

**你的任务：**
推广自家APP，首先需要为Keep找到潜在的推广入口。

**当前进度：**
你现在在核心训练的如图位置（黄色框的部分是你现在的位置）

### 1.2 Keep产品分析

**【产品基本信息】**

| 项目 | 内容 |
|------|------|
| **产品名称** | Keep |
| **产品类型** | 健身APP |
| **核心功能** | 健身课程、运动记录、社交分享 |
| **目标用户** | 18-35岁，有健身需求的城市白领 |
| **产品阶段** | 成长期，需要快速获取用户 |

**【用户画像】**

**基础属性：**
- 年龄：18-35岁
- 性别：女性60%，男性40%
- 城市：一二线城市为主
- 收入：中等偏上

**行为特征：**
- 工作忙碌，时间碎片化
- 关注健康和身材
- 喜欢分享运动成果
- 有一定消费能力

**痛点需求：**
- 没时间去健身房
- 不知道怎么科学健身
- 缺乏健身伙伴
- 难以坚持

## 二、作业要求

### 2.1 核心任务

根据课程中提供的**寻找潜在推广入口的4个思考方向**，为Keep找到潜在的推广入口。

**【4个思考方向】**

1. **用户行为迁移方向**
2. **场景延伸方向**
3. **需求关联方向**
4. **技术变革方向**

### 2.2 提交要求

**每个方向至少找到1个潜在入口**

提交的作业必须包括：

1. **你的思考方向是什么？**
   - 选择哪个方向
   - 为什么选择这个方向

2. **在此方向下你找到的推广渠道是什么？**
   - 具体的平台或APP
   - 为什么选择这个渠道

3. **你准备在此渠道的具体什么位置（入口）上进行推广？**
   - 具体的广告位或内容形式
   - 为什么选择这个位置

4. **你会在这些入口上设置什么样的内容来吸引用户点击？**
   - 具体的文案、图片或视频
   - 为什么这样设计

## 三、作业示范

### 3.1 示范1：用户行为迁移方向

**【思考方向：用户行为迁移方向】**

**分析：**
关注用户在哪里消磨时间，用户的注意力在哪里。

当前趋势：
- 短视频用户时长快速增长
- 抖音日活用户超6亿
- 用户日均使用时长超100分钟

**【找到的推广渠道：抖音】**

**为什么选择抖音？**
1. 用户重合度高：抖音用户与Keep用户高度重合
2. 流量巨大：日活6亿，流量池大
3. 传播力强：短视频传播速度快
4. 成本可控：可以根据预算灵活投放

**【推广入口位置：抖音信息流广告】**

**为什么选择信息流广告？**
1. 原生性强：融入用户刷抖音的场景
2. 精准定向：可以根据年龄、性别、兴趣定向
3. 效果可衡量：可以监控曝光、点击、转化

**【内容设计】**

**视频创意（15秒）：**

```
画面1（0-3秒）：
场景：办公室，白领疲惫地坐在电脑前
文案："加班到深夜，腰酸背痛？"

画面2（3-8秒）：
场景：切换到家中，用户跟着Keep做拉伸
文案："Keep 5分钟办公健身，轻松缓解疲劳"

画面3（8-12秒）：
场景：用户做完拉伸，露出轻松的表情
文案："每天5分钟，告别腰酸背痛"

画面4（12-15秒）：
场景：Keep APP界面展示
文案："立即下载Keep，免费领取办公健身课程"
CTA按钮：点击下载
```

**设计理由：**
1. 痛点切入：加班疲劳是目标用户的真实痛点
2. 解决方案：5分钟办公健身，降低门槛
3. 效果展示：用户做完后轻松的表情
4. 行动召唤：免费领取课程，降低决策成本

**数据预期：**
- CTR：2-3%
- CVR：5-8%
- CPA：15-25元

### 3.2 示范2：场景延伸方向

**【思考方向：场景延伸方向】**

**分析：**
思考用户在什么场景下会用到Keep，找到新的使用场景。

现有场景：
- 家庭健身（主要）
- 健身房训练
- 户外运动

新场景洞察：
- 白领每天通勤时间1-2小时
- 地铁上无法做大幅动作
- 但可以做微运动

**【找到的推广渠道：地铁】**

**为什么选择地铁？**
1. 场景匹配：通勤是目标用户的日常场景
2. 触达精准：地铁用户主要是城市白领
3. 时间充裕：通勤时用户有无聊时间
4. 成本可控：地铁广告成本相对较低

**【推广入口位置：地铁车厢拉手广告】**

**为什么选择拉手广告？**
1. 视线必经：用户站立时必然会看到
2. 距离近：就在用户眼前
3. 可互动：用户可以拿出手机扫码
4. 成本低：比车厢海报便宜

**【内容设计】**

**广告创意：**

```
主标题：地铁上的隐形健身

副标题：3个动作，通勤也能瘦

配图：
简单的通勤动作示意图
（站立小腿拉伸、坐姿收腹等）

二维码：
扫码获取10个通勤健身动作

Slogan：
Keep，让每分钟都有价值
```

**设计理由：**
1. 场景化：直接关联地铁场景
2. 低门槛：隐形健身，不会尴尬
3. 即时性：扫码立即获取，满足即时需求
4. 价值感：让通勤时间变得有价值

**数据预期：**
- 扫码率：3-5%
- 转化率：10-15%
- 单次曝光成本：0.05元

### 3.3 示范3：需求关联方向

**【思考方向：需求关联方向】**

**分析：**
找到与Keep解决同样需求的其他入口。

核心需求：
- 用户想变瘦、变美
- 用户想健康
- 用户想提升身材

关联需求：
- 用户想减肥
- 用户想学健康饮食
- 用户想买运动装备

**【找到的推广渠道：小红书】**

**为什么选择小红书？**
1. 用户高度重合：小红书82%用户为女性，19-35岁占比75%
2. 需求关联：小红书用户关注减肥、健身、健康饮食
3. 信任度高：小红书用户相信种草推荐
4. 传播力强：优质内容容易获得推荐

**【推广入口位置：小红书笔记】**

**为什么选择笔记形式？**
1. 原生性强：符合小红书内容形态
2. 可信度高：真实分享比硬广更可信
3. 传播力强：优质笔记会被推荐
4. 成本低：可以自己创作，无需广告费

**【内容设计】**

**笔记标题：**
"30天瘦10斤，我是这样做到的（附Keep课程）"

**笔记内容：**

```

身高163，体重从65kg瘦到55kg，用了30天时间

【减肥方法】
1. 饮食控制：戒糖戒奶茶，多吃蔬菜蛋白质
2. 运动：每天跟着Keep做30分钟训练
3. 作息：23点前睡觉，7点起床

【Keep课程推荐】
☆ 燃脂训练（难度⭐⭐⭐）
☆ 腹肌撕裂者（难度⭐⭐⭐⭐）
☆ 拉伸放松（难度⭐⭐）

【减肥小贴士】
- 不要节食，要控制饮食
- 运动要循序渐进
- 坚持21天养成习惯

【我的成果】
30天后，不仅瘦了10斤，还养成了运动习惯

【Keep下载】
搜索"Keep"或点击链接下载
现在注册，免费领取7天会员

#减肥 #健身 #Keep #瘦身 #运动
```

**配图：**
1. 减肥前后对比图
2. Keep课程截图
3. 运动打卡图
4. 饮食搭配图

**设计理由：**
1. 真实感：真实经历分享，可信度高
2. 详细化：提供具体方法，有价值
3. 产品植入：自然植入Keep课程
4. 行动召唤：免费领取会员，降低门槛

**数据预期：**
- 笔记曝光：5-10万
- 点赞收藏：1000-2000
- 点击链接：500-1000
- 转化率：10-15%

### 3.4 示范4：技术变革方向

**【思考方向：技术变革方向】**

**分析：**
关注新技术带来的新流量平台。

技术趋势：
- AI技术快速发展
- AI数字人直播兴起
- 24小时不间断直播成为可能

**【找到的推广渠道：AI数字人直播】**

**为什么选择AI数字人直播？**
1. 成本低：仅为传统直播的1/5
2. 时长长：24小时不间断
3. 可复制：一个数字人可以在多个平台直播
4. 效果好：不疲劳，状态稳定

2. 转化率高：直播转化率高于视频
3. 互动性强：可以实时回答用户问题
4. 成交快：直播间可以直接下单

## 四、作业评分标准

### 4.1 完整性（40分）

- 是否覆盖了4个思考方向（10分×4）
- 每个方向是否找到了具体的推广入口
- 是否完成了所有4个必要内容的提交

### 4.2 合理性（30分）

- 推广渠道与目标用户的匹配度（10分）
- 推广位置选择的合理性（10分）
- 内容设计是否符合渠道特性（10分）

### 4.3 创意性（20分）

- 推广入口是否有新意（10分）
- 内容设计是否吸引人（10分）

### 4.4 可行性（10分）

- 方案是否可执行（5分）
- 预期效果是否合理（5分）
