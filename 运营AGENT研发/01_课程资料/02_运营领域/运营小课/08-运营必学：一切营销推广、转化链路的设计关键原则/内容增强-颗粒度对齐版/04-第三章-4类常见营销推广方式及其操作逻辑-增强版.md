## 课程导入

上一节我们介绍了营销推广工作背后的核心工作思路,本节将具体讲解4类常见的营销推广方式及其操作逻辑的差异。

市面上的营销推广方式种类繁多(信息流推广、搜索推广、事件营销、软文广告等),但从影响曝光、点击、转化的核心因素来看,可以归纳为4大类。

**核心公式**：
```
推广效果 = 曝光量 × 点击率 × 转化率
```

不同推广方式下,影响这三个核心要素的关键因素存在显著差异。

## 一、第一类:直接购买固定媒介资源进行推广

### 1.1 推广方式定义

**核心特征**:购买特定媒介的固定时间、固定位置的广告资源位

**典型形式**:
- 传统媒体:报纸杂志版面、电视广告时段、地铁灯箱
- 数字媒体:网站首页广告位、APP开屏广告、信息流广告
- 社交媒体:微信公众号推文、微博大号转发、抖音/小红书达人视频

### 1.2 操作流程与逻辑

#### **标准操作流程**

```
步骤1:需求分析
├─ 确定目标人群
├─ 明确推广目标(品牌曝光/效果转化)
└─ 确定预算范围

步骤2:渠道寻源
├─ 直接联系媒体/广告方业务部门
├─ 通过媒介公司代理
├─ 广告联盟/自媒体联盟/MCN机构
└─ 比较各渠道资源与报价

步骤3:获取刊例与报价
├─ 获取广告刊例手册
├─ 了解可售卖资源位
├─ 确认各位置报价
└─ 确认投放形式与规格要求

步骤4:商务洽谈
├─ 确定投放资源位
├─ 谈判价格与折扣
├─ 确定投放排期(日期/时长)
└─ 签订合同

步骤5:物料准备
├─ 设计制作广告图/视频
├─ 撰写文案
├─ 准备推广链接/二维码
└─ 准备后端承接页面

步骤6:上线投放
├─ 按排期上线广告
├─ 监控投放情况
└─ 收集数据反馈

步骤7:数据复盘
└─ 评估投放效果,总结经验
```

#### **典型案例:不同媒介的广告刊例**

**案例1:IT门户网站广告刊例**

| 广告位 | 位置 | 尺寸 | 报价(天) |
|--------|------|------|----------|
| 首页通栏 | 顶部 | 1000×90 | ¥15,000 |
| 首页画中画 | 中部 | 360×300 | ¥8,000 |
| 频道首页 | 顶部 | 960×80 | ¥5,000 |

**数据分析**:
- 日均UV: 50万
- 曝光量预估: 50万次/天
- 点击率行业基准: 0.5-1.5%
- 预估点击: 2,500-7,500次/天
- 转化率假设: 3%
- 预估转化: 75-225个/天

**案例2:抖音APP广告资源报价**

| 广告形式 | 展示位置 | 报价 | 曝光保证 |
|----------|----------|------|----------|
| 开屏广告 | 打开APP | ¥80万/天 | 2,000万+ |
| 信息流单页 | 推荐流 | ¥150/CPM | - |
| 搜索广告 | 搜索结果 | ¥120/CPM | - |

**计费公式**:
```
CPM费用 = (曝光量/1000) × CPM单价
例如: 100万曝光 × ¥150/CPM = ¥150,000
```

**案例3:微信公众号大号广告报价**

| 账号类型 | 粉丝数 | 头条报价 | 条次报价 | 内容定位 |
|----------|--------|----------|----------|----------|
| 头部大号 | 100万+ | ¥50,000 | ¥15,000 | 职场成长 |
| 中部大号 | 50-100万 | ¥25,000 | ¥8,000 | 母婴育儿 |
| 腰部大号 | 10-50万 | ¥8,000 | ¥3,000 | 生活方式 |

**ROI计算模型**:
```
CPA = 广告费用 / 转化人数
LTV = 用户平均生命周期价值
ROI = (LTV × 转化人数 - 广告费用) / 广告费用

示例:
- 广告费用: ¥25,000
- 转化人数: 100人
- CPA: ¥250/人
- LTV: ¥500/人
- ROI = (500×100 - 25000) / 25000 = 100%
```

### 1.3 推广特点分析

#### **曝光量特征**

**固定媒介曝光**:
```
曝光量 = 媒介受众基数 × 展现概率

特点:
- 相对稳定,波动较小
- 可通过历史数据预估
- 媒介受众规模决定天花板
```

**社会化媒体曝光**:
```
总曝光量 = 基础曝光 + 自然转发曝光

基础曝光 = 媒介粉丝数 × 展现率
转发曝光 = 基础曝光 × 转发率 × 平均曝光深度

影响因素:
- 内容质量与创意
- 文案吸引力
- 话题热度
- KOL影响力
```

#### **点击率影响因素**

**关键要素**:
1. **视觉吸引力**: 图片/视频设计质量
2. **文案质量**: 标题、描述的吸引力
3. **价值主张**: 用户利益点是否清晰
4. **行动召唤**: CTA是否明确有力

**行业基准数据**:
| 广告形式 | 平均点击率 | 优秀水平 |
|----------|------------|----------|
| Banner广告 | 0.5-1% | 2%+ |
| 信息流广告 | 1-3% | 5%+ |
| 微信软文 | 3-8% | 15%+ |
| 抖音视频 | 5-15% | 30%+ |

#### **转化率影响因素**

**核心公式**:
```
转化率 = (访问转化率 × 流量质量) × 页面转化能力

访问转化率 = 到达率 × 有效浏览率
页面转化能力 = 信任度 × 必要性 × 紧迫性
```

**优化策略**:
- 缩短转化路径
- 优化落地页设计
- 增强信任背书
- 提供紧迫性理由

## 二、第二类:利用流量平台的流量分发规则和免费资源

### 2.1 推广方式定义

**核心特征**:研究并利用流量平台的算法规则、排序机制、免费资源位进行推广

**典型形式**:
- ASO(应用商店优化)
- SEO(搜索引擎优化)
- 电商平台活动报名
- 社区内容运营
- 官方专题活动申报

### 2.2 应用商店SEO案例

#### **案例1:蹭热门搜索词**

**操作原理**:
```
用户搜索"微信" → 展示微信APP + 相关推荐APP

优化策略:
1. 分析热门搜索词
2. 在APP标题、描述、关键词中包含热词
3. 提升ASO排名因素(下载量、评分、评论)
4. 提高与热词的相关性得分
```

**数据效果**:
```
优化前:
- "微信"关键词下排名: 未进入前100
- 日均下载量: 50次

优化后:
- "微信"关键词下排名: 第3位
- 日均下载量: 800次
- 增长倍数: 16倍
```

#### **案例2:蹭官方推荐资源**

**操作流程**:
```
步骤1:监测官方专题
├─ 定期查看应用商店推荐专区
├─ 分析近期专题主题与要求
└─ 评估产品匹配度

步骤2:准备申报材料
├─ 准备高质量截图
├─ 撰写吸引人的产品描述
├─ 准备演示视频
└─ 整理用户评价与数据

步骤3:提交申报
├─ 按要求提交申请
├─ 跟进审核进度
└─ 根据反馈优化材料

步骤4:上线运营
├─ 监控下载量变化
├─ 收集用户反馈
└─ 持续优化产品评分
```

**效果对比**:
| 指标 | 日常运营 | 被推荐后 | 增幅 |
|------|----------|----------|------|
| 日下载量 | 200次 | 5,000次 | 25倍 |
| 搜索排名 | 15位 | 3位 | - |
| 用户评分 | 4.2 | 4.5 | - |

### 2.3 搜索引擎SEO案例

#### **操作原理**

```
SEO核心 = 技术优化 + 内容优化 + 权威度建设

技术优化要素:
├─ 网站代码结构
├─ 页面加载速度
├─ 移动端适配
├─ URL结构
└─ 内链外链布局

内容优化要素:
├─ 关键词布局
├─ 内容质量
├─ 内容原创性
├─ 内容更新频率
└─ 用户行为指标(停留时间、跳出率)

权威度建设:
├─ 高质量外链
├─ 品牌搜索量
├─ 社交媒体信号
└─ 用户口碑
```

#### **案例:劣评网SEO实践**

**目标关键词**: "财务找工作"、"会计求职"

**优化策略**:
```
1. 关键词研究
   - 核心词: "财务找工作"

   - 关键词密度: 2-3%

3. 内容建设
   - 字数: 2,000字+

   - 价值输出: 求职经验+避坑指南
   - 用户互动: 评论区引导

4. 外链建设
   - 行业论坛发布

**效果数据**:

- 自然搜索流量: 50/天 → 800/天

成本分析:
- SEO投入: ¥15,000(内容+外链)

- 节省成本: 83%
```

```

## 三、第三类:事件营销与传播型推广

### 3.1 推广方式定义

**核心特征**:通过策划有话题性、传播性的内容或事件,引发用户自发传播,实现品牌曝光

**典型形式**:
- 社会化媒体事件营销
- 线下行为艺术传播
- 病毒视频/内容
- 话题炒作

### 3.2 经典案例分析

#### **案例1:途牛网"只要心中有沙,哪里都是马尔代夫"**

**案例背景**:
```
目标: 提升途牛网品牌知名度与旅游产品销售
创意: 反差感+话题性+情感共鸣
时间: 2015年
```

**策划与执行**:

```
阶段1:创意策划
├─ 核心概念: "心中有沙,哪里都是马尔代夫"
├─ 传播载体: 街头行为艺术
├─ 情感触发: 对美好生活的向往
└─ 话题潜力: 具备病毒传播属性

阶段2:资源整合
├─ 找到"胖哥"演员
├─ 确定成都街景拍摄地

├─ 联合KOL资源(微博大号、微信公号)

阶段3:内容制作

### 4.4 计费方式详解

#### **CPM(按展示付费)**

```
定义:
Cost Per Mille
按广告展示1000次计费

计算公式:
费用 = (展示量 / 1000) × CPM单价

示例:
展示量: 50万次
CPM单价: ¥80
费用 = (500,000 / 1,000) × 80 = ¥40,000

适用场景:
✓ 品牌曝光为主
✓ 新品发布
✓ 大促活动
✓ 提升品牌知名度

优势:
✓ 保曝光量
✓ 预算可控
✓ 适合品牌广告

劣势:
✗ 不保证效果
✗ 点击率无保证
✗ 转化无保证
```

#### **CPC(按点击付费)**

```
定义:
Cost Per Click
按用户点击广告次数计费

计算公式:
费用 = 点击量 × CPC单价
平均点击成本 = 总费用 / 总点击量

示例:
点击量: 5,000次
CPC单价: ¥2.5
费用 = 5,000 × 2.5 = ¥12,500

适用场景:
✓ 搜索引擎广告
✓ 效果营销
✓ 流量获取
✓ 点击率较高的广告

优势:
✓ 按效果付费
✓ 只为点击买单
✓ 风险相对较低

劣势:
✗ 点击不保证转化
✗ 存在恶意点击风险
✗ 需要持续优化
```

#### **CPA(按行动付费)**

```
定义:
Cost Per Action
按用户完成特定行动计费

行动类型:
├─ 下载类: APP下载
├─ 注册类: 用户注册
├─ 留资类: 表单提交
├─ 到店类: 线下到店
└─ 试用类: 产品试用

计算公式:
费用 = 完成行动数 × CPA单价

示例:
行动: APP下载
完成量: 1,000次
CPA单价: ¥15
费用 = 1,000 × 15 = ¥15,000

适用场景:
✓ APP推广
✓ 用户获取
✓ 潜客收集
✓ 有明确转化目标

优势:
✓ 按结果付费
✓ 转化成本可控
✓ 风险最低

劣势:
✗ 单价较高
✗ 平台要求严格
└─ 需要技术对接
```

#### **CPS(按销售付费)**

```
定义:
Cost Per Sale
按销售成交金额或订单数计费

计费模式:
├─ 按订单数: 每单固定金额
├─ 按销售额: 销售额百分比
└─ 混合模式: 基础费+佣金

计算公式:
费用模式1 = 订单数 × 每单佣金
费用模式2 = 销售额 × 佣金比例

示例:
按订单数:
订单量: 100单
每单佣金: ¥30
费用 = 100 × 30 = ¥3,000

按销售额:
销售额: ¥100,000
佣金比例: 15%
费用 = 100,000 × 15% = ¥15,000

适用场景:
✓ 电商推广
✓ 分销佣金
✓ 联盟营销
✓ CPS广告联盟

优势:
✓ 按成交付费
✓ 风险最小
✓ ROI可控

劣势:
✗ 佣金比例高
└─ 商家要求高
```

#### **CPT(按时长付费)**

```
定义:
Cost Per Time
按用户有效停留时长计费

计费标准:
├─ 页面停留时长: 如超过10秒
├─ APP使用时长: 如超过30秒
├─ 视频观看时长: 如观看>50%
└─ 有效时长: 去除跳出访问

计算公式:
费用 = 有效用户数 × CPT单价

示例:
有效用户: 5,000人
(停留>10秒)
CPT单价: ¥0.5
费用 = 5,000 × 0.5 = ¥2,500

适用场景:
✓ APP推广
├─ 防刷量需求
├─ 质量要求高
└─ 品牌安全要求

优势:
✓ 防作弊
✓ 流量质量高
✓ 用户真实参与

劣势:
✗ 计费复杂
└─ 数据监测要求高
```

#### **计费方式对比**

| 计费方式 | 付费依据 | 风险程度 | 适用场景 | 平均价格 |
|----------|----------|----------|----------|----------|
| CPM | 展示量 | 高(广告主) | 品牌曝光 | ¥50-200/CPM |
| CPC | 点击量 | 中 | 流量获取 | ¥0.5-5/点击 |
| CPA | 行动量 | 低 | 效果转化 | ¥10-100/行动 |
| CPS | 销售额 | 最低 | 电商分销 | 5-30%佣金 |
| CPT | 时长 | 中低 | APP推广 | ¥0.3-2/人 |

**选择建议**:
```
品牌曝光期: CPM为主
流量积累期: CPC为主
效果转化期: CPA/CPS为主
防刷量要求: CPT优先
```

### 4.5 风险与注意事项

#### **常见风险点**

```
预算风险:
├─ 烧钱速度过快
│   ├─ 原因: 出价过高/定向过宽
│   ├─ 后果: 预算迅速耗尽
│   └─ 对策: 设置日预算上限
├─ ROI不达标
│   ├─ 原因: 转化率低/成本高
│   ├─ 后果: 持续亏损
│   └─ 对策: 及时暂停/优化

流量风险:
├─ 质量差
│   ├─ 表现: 高点击低转化
│   ├─ 原因: 定向不准/素材误导
│   └─ 对策: 收窄定向/优化素材
├─ 虚假流量
│   ├─ 表现: 异常点击模式
│   ├─ 原因: 流量作弊
│   └─ 对策: 反作弊机制/排查

平台风险:
├─ 政策变化
│   ├─ 表现: 规则突然调整
│   ├─ 影响: 投放效果下降
│   └─ 对策: 多平台分散
├─ 账户被封
│   ├─ 原因: 违反平台规则
│   ├─ 影响: 无法投放
│   └─ 对策: 合规经营

竞争风险:
├─ 成本上升
│   ├─ 原因: 竞争加剧
│   ├─ 表现: CPC/CPA上涨
│   └─ 对策: 提升转化率
└─ 流量减少
    ├─ 原因: 竞品抢占
    ├─ 表现: 展现量下降
    └─ 对策: 优化素材/出价
```

#### **最佳实践**

```
投放前:
✓ 明确目标与预算
✓ 准备多套素材
✓ 测试转化链路
✓ 设置预警机制

投放中:
✓ 实时监控数据
✓ 每日优化调整
✓ A/B测试素材
✓ 记录数据日志

投放后:
✓ 汇总分析数据
✓ 计算ROI
✓ 总结经验教训
✓ 优化后续策略

持续优化:
✓ 素材迭代
✓ 定向优化
✓ 页面优化
✓ 出价调整
```

## 本章小结

### 核心要点回顾

1. **4类推广方式各有特点**
   - 第一类:购买固定媒介,流量稳定成本高
   - 第二类:利用平台规则,成本低见效慢
   - 第三类:策划事件营销,爆发力强风险大
   - 第四类:精准效果投放,高效烧钱快

   - 产品类型与阶段
   - 目标用户特征
   - 预算与资源
   - 时间要求
   - 风险承受力

3. **组合使用效果更佳**
   - 不同方式互补
   - 分阶段实施
   - 持续优化调整
