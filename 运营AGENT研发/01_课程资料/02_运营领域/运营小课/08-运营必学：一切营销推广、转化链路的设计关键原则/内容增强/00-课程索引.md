# 课程索引：运营必学：一切营销推广、转化链路的设计关键原则

> 讲师：黄有璨
> 课程节数：10节
> 核心价值：掌握营销推广和转化链路的设计方法

---

## 课程定位

### 核心内容

1. **推广入口**：如何发现及开拓匹配的推广渠道
2. **推广方法**：营销推广工作的基本方法
3. **4类推广方式**：常见推广方式的理解和操作
4. **转化链路**：营销转化链路的设计原理和要素
5. **数据优化**：营销推广中的数据监测和分析

## 核心概念

### 推广入口

> 找到与业务匹配的推广渠道是营销推广的第一步。

### 4类常见推广方式

| 类型 | 说明 |
|-----|------|
| 付费推广 | 广告投放 |
| 内容推广 | 内容营销 |
| 社交推广 | 社交裂变 |
| 合作推广 | 资源合作 |

### 转化链路

> 理解用户消费行为，设计有效的转化路径。
