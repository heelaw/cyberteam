# 第二章：推广方法（详细版）

> 所属课程：运营必学：一切营销推广、转化链路的设计关键原则
> 讲师：黄有璨
> 本章重点：掌握营销推广的优化思路和方法

---

## 本章导学

### 核心目标

本章帮助学员理解营销推广优化的核心思路，当推广效果不好时知道如何分析和改进。核心目标是建立推广优化的系统思维框架，掌握常见的优化方向和方法。

### 学习重点

- 理解推广效果不好的常见原因
- 掌握推广优化的基本框架
- 学会从多个维度分析推广问题
- 理解持续优化迭代的重要性

## 详细内容

### 2.1 推广效果评估的基本框架

#### 推广效果评估的核心指标

```
┌─────────────────────────────────────────────────────────────────────┐
│                   推广效果评估指标体系                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  核心指标：                                                        │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │   │
│  │  1. ROI（投资回报率）                                      │   │
│  │     - 公式：收入 / 投入                                    │   │
│  │     - 衡量：每投入1元能赚回多少                           │   │
│  │     - 标准：ROI > 1 可持续                                 │   │
│  │                                                             │   │
│  │  2. 转化率（CVR）                                          │   │
│  │     - 公式：转化数 / 点击数                                │   │
│  │     - 衡量：流量的质量                                     │   │
│  │     - 标准：因产品而异                                     │   │
│  │                                                             │   │
│  │  3. CPA（单次获取成本）                                    │   │
│  │     - 公式：投入 / 转化数                                  │   │
│  │     - 衡量：获客效率                                       │   │
│  │     - 标准：低于产品客单价                                   │   │
│  │                                                             │   │
│  │  4. ROI = 1 的临界点                                      │   │
│  │     - ROI > 1：可持续推广                                 │   │
│  │     - ROI < 1：需要优化或止损                             │   │
│  │                                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

```
│                                                                     │

```

### 2.2 推广效果不好的常见原因

#### 五大常见原因

```
┌─────────────────────────────────────────────────────────────────────┐
│                推广效果不好的五大常见原因                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  原因1：流量不精准                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  表现：                                                      │   │
│  │  - 点击多但转化少                                           │   │
│  │  - 访客质量差                                               │   │
│  │  - 跳出率高                                                 │   │
│  │                                                              │   │
│  │  原因：                                                      │   │
│  │  - 渠道选择不当                                             │   │
│  │  - 人群定向不精准                                           │   │
│  │  - 素材与落地页不匹配                                       │   │
│  │                                                              │   │
│  │  解决方向：                                                 │   │
│  │  - 重新选择渠道                                             │   │
│  │  - 优化人群定向                                             │   │
│  │  - 优化素材和落地页匹配度                                   │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  原因2：转化路径过长                                              │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  表现：                                                      │   │
│  │  - 用户在转化过程中流失                                     │   │
│  │  - 转化周期长                                               │   │
│  │  - 转化率低                                                 │   │
│  │                                                              │   │
│  │  原因：                                                      │   │
│  │  - 转化步骤太多                                             │   │
│  │  - 需要多次跳转                                             │   │
│  │  - 用户操作复杂                                             │   │
│  │                                                              │   │
│  │  解决方向：                                                 │   │
│  │  - 简化转化路径                                             │   │
│  │  - 减少跳转步骤                                             │   │
│  │  - 优化用户操作体验                                         │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  原因3：素材吸引力不足                                            │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  表现：                                                      │   │
│  │  - 点击率低                                                 │   │
│  │  - 曝光量大但点击少                                         │   │
│  │                                                              │   │
│  │  原因：                                                      │   │
│  │  - 创意不突出                                               │   │
│  │  - 缺乏吸引力                                               │   │
│  │  - 与竞品相似                                               │   │
│  │                                                              │   │
│  │  解决方向：                                                 │   │
│  │  - 优化创意设计                                             │   │
│  │  - 测试不同创意版本                                         │   │
│  │  - 突出产品差异化                                           │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  原因4：落地页问题                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  表现：                                                      │   │
│  │  - 点击后跳出                                                │   │
│  │  - 访问时长短                                               │   │
│  │  - 不产生转化                                                │   │
│  │                                                              │   │
│  │  原因：                                                      │   │
│  │  - 页面加载慢                                               │   │
│  │  - 内容不吸引人                                             │   │
│  │  - 转化入口不明显                                           │   │
│  │  - 与素材承诺不符                                           │   │
│  │                                                              │   │
│  │  解决方向：                                                 │   │
│  │  - 优化页面加载速度                                         │   │
│  │  - 优化页面内容和布局                                       │   │
│  │  │  - 突出用户痛点                                           │   │
│  │  │  - 展示产品价值                                           │   │
│  │  │  - 明确的行动号召                                         │   │
│  │  └───────────────────────────────────────────────────────    │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  原因5：产品/价格问题                                             │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  表现：                                                      │   │
│  │  - 流量精准、落地页也好，但就是转化不了                     │   │
│  │  - 用户访问但不购买                                         │   │
│  │                                                              │   │
│  │  原因：                                                      │   │
│  │  - 产品不是用户真正需要的                                   │   │
│  │  - 价格太高                                                 │   │
│  │  - 购买信任不足                                             │   │
│  │  - 支付流程不顺畅                                           │   │
│  │                                                              │   │
│  │  解决方向：                                                 │   │
│  │  - 优化产品定位                                             │   │
│  │  - 调整价格策略                                             │   │
│  │  - 增加信任背书                                             │   │
│  │  - 优化支付流程                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 2.3 推广优化的基本流程

#### 优化流程六步法

```
┌─────────────────────────────────────────────────────────────────────┐
│                    推广优化流程六步法                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  Step 1: 数据收集与问题定位                                        │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  收集数据：                                                  │   │
│  │  - 各环节转化率                                              │   │
│  │  - 各渠道ROI                                                │   │
│  │  - 用户行为数据                                              │   │
│  │                                                              │   │
│  │  定位问题：                                                  │   │
│  │  - 找出转化率最低的环节                                     │   │
│  │  - 对比不同渠道的效果差异                                   │   │
│  │  - 分析问题环节的具体数据                                   │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                 ↓                                                  │
│  Step 2: 假设与归因                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  提出假设：                                                  │   │
│  │  - 为什么会产生这个问题？                                   │   │
│  │  - 可能的原因有哪些？                                       │   │
│  │                                                              │   │
│  │  归因分析：                                                  │   │
│  │  - 区分主因和次因                                           │   │
│  │  - 排出优先级                                                │   │
│  │  - 确定优化方向                                              │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                 ↓                                                  │
│  Step 3: 制定优化方案                                              │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  优化维度：                                                  │   │
│  │  - 渠道优化：更换或调整渠道                                 │   │
│  │  - 素材优化：测试新创意                                     │   │
│  │  - 落地页优化：优化页面内容                                 │   │
│  │  - 转化路径优化：简化转化流程                               │   │
│  │                                                              │   │
│  │  方案内容：                                                  │   │
│  │  - 具体的优化措施                                           │   │
│  │  - 预期效果                                                 │   │
│  │  - 测试计划                                                  │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                 ↓                                                  │
│  Step 4: A/B测试验证                                              │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  测试方法：                                                  │   │
│  │  - 保持其他变量不变，只改变一个因素                         │   │
│  │  - 划分流量到新旧两个版本                                   │   │
│  │  - 收集足够样本量后对比效果                                 │   │
│  │                                                              │   │
│  │  注意事项：                                                  │   │
│  │  - 测试周期要足够                                           │   │
│  │  - 样本量要足够                                             │   │
│  │  - 记录测试数据                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                 ↓                                                  │
│  Step 5: 效果评估与决策                                           │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  评估标准：                                                  │   │
│  │  - 新版本效果是否优于旧版本？                               │   │
│  │  - ROI是否提升？                                            │   │
│  │  - 转化率是否提升？                                         │   │
│  │                                                              │   │
│  │  决策：                                                      │   │
│  │  - 效果提升：全量上线                                      │   │
│  │  - 效果下降：保持原版本                                     │   │
│  │  - 效果持平：继续测试                                       │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                 ↓                                                  │
│  Step 6: 持续迭代优化                                             │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  优化逻辑：                                                  │   │
│  │  - 优化是无止境的                                           │   │
│  │  - 每次优化都可能带来效果提升                               │   │
│  │  - 持续测试，持续优化                                       │   │
│  │                                                              │   │
│  │  心态建设：                                                  │   │
│  │  - 不要期望一次优化解决所有问题                             │   │
│  │  - 接受不完美，持续改进                                     │   │
│  │  - 数据驱动决策                                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 2.4 常见优化方向详解

```

│  渠道测试策略：                                                    │

│  │  2. 组合测试法                                               │   │
│  │     - 主渠道+辅助渠道组合                                     │   │
│  │     - 分散风险                                               │   │
│  │                                                              │   │
│  │  3. 渠道生命周期管理                                         │   │
│  │     - 早期：探索测试                                         │   │
│  │     - 成长期：加大投入                                        │   │
│  │     - 成熟期：维持效率                                       │   │
│  │     - 衰退期：减少投入/替换                                 │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

```
│  │  测试流程：                                                  │   │
│  │  1. 每次只改变一个变量                                       │   │
│  │  2. 保持足够样本量                                          │   │
│  │  3. 记录每个版本的效果                                      │   │
│  │  4. 选择最优版本                                             │   │
```

## 本章小结

### 1. 推广效果评估框架
- 核心指标：ROI、CVR、CPA
- 推广漏斗：曝光→点击→访问→转化
- 找到转化率最低的环节，针对优化

### 2. 推广效果不好的五大原因
- **流量不精准**：渠道选择、人群定向问题
- **转化路径过长**：步骤太多，用户流失
- **素材吸引力不足**：点击率低
- **落地页问题**：跳出率高，不转化
- **产品/价格问题**：不是用户需要的

### 3. 优化流程六步法
1. 数据收集与问题定位
2. 假设与归因
3. 制定优化方案
4. A/B测试验证
5. 效果评估与决策
6. 持续迭代优化

### 4. 四大优化方向
- **渠道优化**：资源向高效渠道倾斜
- **素材优化**：测试迭代，持续改进
- **落地页优化**：匹配素材，突出价值
- **转化路径优化**：减少步骤，降低门槛

### 5. 优化心态
- 优化是无止境的
- 用数据驱动决策
- 持续测试，持续改进
