# 第三章：4类常见营销推广方式及其操作逻辑

> 所属课程：运营必学：一切营销推广、转化链路的设计关键原则
> 讲师：黄有璨
> 本章重点：掌握4类常见营销推广方式的底层逻辑和操作方法

---

## 本章导学（核心目标）

本章将系统讲解市面上常见的4类营销推广方式，帮助运营人员理解不同推广方式的核心逻辑、适用场景和操作要点。

**学习目标：**
1. 理解4类营销推广方式的本质区别
2. 掌握每类推广方式的操作流程和关键要素
3. 学会根据自身产品特点选择合适的推广方式
4. 了解效果广告的计费方式和投放策略

---

## 详细内容

### 第1类：直接购买固定媒介资源进行推广

**定义**：企业通过购买固定的媒介资源位进行推广，包括固定时间、固定位置的展示资源。

**常见形式：**
| 类型 | 示例 |
|------|------|
| 传统媒体 | 报纸版面、杂志封面、电视时段 |
| 户外媒体 | 地铁灯箱、公交广告、楼宇广告 |
| 数字媒体 | 网站/App开屏广告、首页横幅 |
| 社交媒体 | 公众号推文、微博大号、抖音达人 |

**操作流程：**

```
┌─────────────────────────────────────────────────────────────┐
│                  媒介资源购买推广流程                        │
├─────────────────────────────────────────────────────────────┤
│  1. 需求分析                                                │
│     └─ 确定目标用户画像、投放预算、预期效果                   │
│           ↓                                                 │
│  2. 渠道筛选                                                │
│     └─ 寻找匹配的媒介公司、广告联盟、MCN机构                  │
│           ↓                                                 │
│  3. 获取刊例                                                │
│     └─ 查阅资源报价单，了解各位置的价格和效果                │
│           ↓                                                 │
│  4. 商务洽谈                                                │
│     └─ 谈价格、排期、资源位置                               │
│           ↓                                                 │
│  5. 物料准备                                                │
│     └─ 制作广告文案、图片、视频等素材                       │
│           ↓                                                 │
│  6. 投放执行                                                │
│     └─ 正式上线广告，监控数据                               │
└─────────────────────────────────────────────────────────────┘
```

**核心特点：**
- 曝光量相对稳定（取决于媒介本身的用户基数）
- 投放效果高度依赖文案和物料质量
- 适合品牌曝光和规模化推广
- 社会化媒体投放时，内容质量会影响二次传播

### 第2类：利用流量平台规则进行免费/低成本推广

**定义**：通过研究并利用流量平台的分发规则，获取免费的自然流量或低成本推广资源。

**核心逻辑：**
```
┌──────────────────────────────────────────────────────────────┐
│               流量平台规则推广逻辑                            │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│   平台流量分发规则 ──────→  优化物料适配规则  ──────→  获 │
│                                                              │
│   （SEO/ASO优化）       （标题/标签/内容调整）      （自然流量）│
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

**常见应用场景：**

| 场景 | 案例说明 |
|------|----------|
| 应用商店优化(ASO) | 优化APP关键词、图标、描述，提升搜索排名 |
| 搜索引擎优化(SEO) | 优化网站内容，提升百度/Google排名 |
| 电商平台运营 | 参加淘宝/京东官方活动，获取免费推荐位 |
| 问答社区 | 在知乎回答热门问题，获取曝光 |

**案例：奥迪知乎营销**
- 问题："用汽车发动机煎牛排靠谱吗？"
- 奥迪官方精心回答，展示品牌专业度
- 获得3000+点赞，巨大品牌曝光
- 关键：找到与品牌相关的热门话题，提供有价值内容

**操作要点：**
1. 深入研究平台规则和算法
2. 持续优化内容和物料
3. 建立平台资源对接关系
4. 关注数据，持续迭代

### 第3类：事件营销/传播型推广

**定义**：通过策划具有话题性的事件或活动，引发用户主动传播，达到以小博大的推广效果。

**经典案例：**

| 案例 | 推广内容 | 传播效果 |
|------|----------|----------|
| "只要心中有沙，哪里都是马尔代夫" | 途牛网行为艺术营销 | 病毒式传播，全网皆知 |
| "人类高质量男性" | 某网红相亲视频 | 引发热议，成功带货付费社群 |

**事件营销策划流程：**

```
┌─────────────────────────────────────────────────────────────┐
│                   事件营销策划流程                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   创意构思  ──→  资源整合  ──→  执行落地  ──→  传播发酵    │
│                                                              │
│   • 洞察用户     • 媒介资源储备    • 选时间地点     • 观察舆情  │
│   • 找到话题点   • KOL/KOC联动     • 线下执行       • 调整优化  │
│   • 设计传播路径 • 物料准备        • 线上配合       • 二次传播  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

**特点分析：**
- **优点**：以小博大，传播势能大，品牌曝光效果好
- **缺点**：依赖创意和运气，转化效果不稳定
- **适用**：品牌曝光、话题造势

---

### 第4类：精准效果广告投放

**定义**：通过平台的精准定向能力，面向目标用户进行广告投放，按效果付费。

**主流平台：**
| 平台 | 广告形式 | 定向能力 |
|------|----------|----------|
| 百度 | 搜索广告 | 关键词定向 |
| 抖音/头条 | 信息流广告 | 兴趣/行为定向 |
| 腾讯(广点通) | 朋友圈/公众号广告 | 人群/地域定向 |
| 小红书 | 聚光平台 | 素人/达人定向 |

**计费方式：**

| 缩写 | 全称 | 说明 | 适用场景 |
|------|------|------|----------|
| CPM | Cost Per Mille | 按千次展示付费 | 品牌曝光 |
| CPC | Cost Per Click | 按点击付费 | 流量获取 |
| CPA | Cost Per Action | 按行动付费(下载/注册) | APP推广 |
| CPS | Cost Per Sale | 按销售分成 | 电商转化 |
| CPT | Cost Per Time | 按时长/停留时间付费 | 防作弊场景 |

**投放操作流程：**

```
┌─────────────────────────────────────────────────────────────┐
│                  效果广告投放流程                             │
├─────────────────────────────────────────────────────────────┤
│  1. 账户开通                                                  │
│     └─ 注册投放平台账户，了解规则                            │
│           ↓                                                 │
│  2. 策略制定                                                  │
│     └─ 确定投放关键词、人群、地域、时间等                     │
│           ↓                                                 │
│  3. 物料准备                                                  │
│     └─ 制作广告创意、图片/视频、落地页                       │
│           ↓                                                 │
│  4. 投放执行                                                  │
│     └─ 设置出价，定向人群，上线广告                           │
│           ↓                                                 │
│  5. 数据监控                                                  │
│     └─ 实时关注曝光、点击、转化、ROI数据                     │
│           ↓                                                 │
│  6. 优化迭代                                                  │
│     └─ 调整出价、创意、定向，持续优化                        │
└─────────────────────────────────────────────────────────────┘
```

**关键指标：ROI（投资回报率）**

```
ROI = 销售收入 / 广告投入

判断标准：
ROI > 1  ───→ 可持续投放（每投入1元能赚回超过1元）
ROI = 1  ───→ 盈亏平衡，谨慎放大
ROI < 1  ───→ 亏损，需要优化或暂停
```

**注意事项：**
- 效果广告烧钱速度快，需密切监控ROI
- 投放前需测试转化链路，确保能跑通
- 关键词和人群定向是核心优化点

## 案例分析

### 案例1：京东百度搜索推广

**背景**：京东在百度投放冰箱关键词广告

**转化链路分析：**

```
用户搜索"冰箱"
    ↓
看到京东/海尔广告（曝光）
    ↓
点击京东广告（点击）
    ↓
进入京东冰箱搜索结果页
    ↓
点击某品牌冰箱
    ↓
进入品牌详情页
    ↓
选择具体型号
    ↓
进入商品详情页
    ↓
加入购物车/立即购买
    ↓
登录/注册
    ↓
完成支付（转化）
```

**关键数据节点：**
- 搜索曝光量 → 点击转化率
- 点击量 → 详情页访问转化率
- 访问量 → 加购转化率
- 加购量 → 支付转化率

**优化思路：**
1. **优化流量**：扩展关键词覆盖，提升排名
2. **优化流程**：缩短转化路径，减少流失
3. **优化转化率**：优化落地页和详情页

## 核心概念表格

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 曝光 | 广告被用户看到的次数 | 取决于渠道流量和展示位置 |
| 点击 | 用户点击广告的行为 | 取决于文案和物料吸引力 |
| 转化 | 用户完成目标行为 | 取决于转化流程设计 |
| ROI | 投资回报率 | ROI=销售收入/投入，>1可持续 |
| CPM | 千次展示成本 | 品牌曝光类广告常用 |
| CPC | 单次点击成本 | 搜索广告常用 |
| CPS | 销售分成 | 电商转化常用 |
| SEO | 搜索引擎优化 | 提升自然排名获取流量 |
| ASO | 应用商店优化 | 提升APP搜索排名 |
| 事件营销 | 通过话题事件引发传播 | 以小博大，效果不稳定 |

## 本章小结

1. **4类营销推广方式**：
   - 直接购买媒介资源：稳定曝光，依赖物料质量
   - 利用平台规则：低成本，需要持续优化
   - 事件营销：以小博大，依赖创意
   - 效果广告：精准可控，需监控ROI

2. **核心工作思路**：关注曝光、点击、转化三个环节

3. **推广优化三板斧**：
   - 优化流量（扩展入口、提升排名）
   - 优化流程（缩短路径、减少流失）
   - 优化转化率（提升每个环节的转化）

4. **计费方式选择**：根据目标选择合适的计费模式，CPM适合品牌，CPC/CPA适合效果
