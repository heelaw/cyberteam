## 7. 落地3个重要原则

### 7.1 原则汇总

```
┌─────────────────────────────────────────────────────────────────────┐
│                      3个重要原则                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   原则1：锚定产品设计                                              │
│   • 通过对比引导选择目标产品                                       │
│   • 托品不一定要卖出                                               │
│   • 提升转化率和客单价                                             │
│                                                                     │
│   原则2：唯一最短可信                                              │
│   • 唯一：给少的选择                                               │
│   • 最短：不超过5步                                                │
│   • 可信：增强信任                                                 │
│                                                                     │
│   原则3：场景适配                                                  │
│   • 根据用户场景和决策成本设计                                      │
│   • 单场景/跨场景不同策略                                          │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 7.2 四象限场景策略

| 场景 | 特点 | 策略 |
|------|------|------|
| 短时间+冲动 | 快决策 | 强优惠+快速转化 |
| 长时间+冲动 | 详情页 | 详情展示+不断引导 |
| 短时间+理性 | 高客单 | 轻诱因+留资料 |
| 长时间+理性 | 深度 | 内容+信任+顾问 |

## 11. 常见问题解答

### Q1：如何选择推广渠道？

**答**：从4个角度思考：
1. 用户在哪里（渠道聚集目标用户）
2. 场景是什么（用户搜索/闲逛）
3. 成本多少（预算和ROI）
4. 能否测试（先小规模测试）

### Q2：转化效果不好怎么办？

**答**：三种优化思路：
1. 优化流量：扩展入口
2. 优化流程：缩短路径
3. 优化转化率：提升每步转化

### Q3：如何判断用户消费行为类型？

**答**：看3个维度：
1. 决策时间：短→冲动，长→理性
2. 产品价格：高→理性，低→冲动
3. 风险感知：高→理性，低→冲动
