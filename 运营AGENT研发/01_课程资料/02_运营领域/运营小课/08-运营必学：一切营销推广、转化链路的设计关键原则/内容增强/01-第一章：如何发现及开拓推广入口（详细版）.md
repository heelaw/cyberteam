# 第一章：如何发现及开拓推广入口（详细版）

> 所属课程：运营必学：一切营销推广、转化链路的设计关键原则
> 讲师：黄有璨
> 本章重点：掌握发现和开拓推广入口的方法论

---

## 本章导学

### 核心目标

本章帮助学员理解如何精准定位目标用户，并基于用户特征发现和开拓匹配的推广入口。核心目标是掌握四种发现推广入口的思考框架，并建立渠道运营的正确认知。

### 学习重点

- 理解精准定义用户的重要性
- 掌握精准定义用户的三种方法
- 掌握发现推广入口的四种思考方向
- 理解新渠道开拓的价值和策略

---

## 详细内容

### 1.3 发现推广入口的四种思考方向

#### 方向1：因A激发的消费欲望

**原理**：用户看到某些内容或信息后，会激起消费某种产品的欲望

```
┌─────────────────────────────────────────────────────────────────────┐
│               思考方向1：因A激发的消费欲望                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  核心问题：如果我的产品是B，用户看到什么A会激起消费B的欲望？        │
│                                                                     │
│  案例1：餐厅推荐类内容                                             │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  A（用户看到的）："帝都十大不容错过的小资饭店"             │   │
│  │  ↓                                                            │   │
│  │  B（激发的欲望）：想去这些餐厅吃饭                         │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  案例2：学习资料推荐                                               │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  A（用户看到的）："史上最全面的产品经理学习资料"           │   │
│  │  ↓ 其中提到：《运营之光》这本书                            │   │
│  │  B（激发的欲望）：想买《运营之光》                        │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  案例3：影单推荐                                                   │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  A（用户看到的）："史上评分最高的100部电影"               │   │
│  │  ↓                                                            │   │
│  │  B（激发的欲望）：想要下载/观看这些电影                   │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  应用场景：                                                         │
│  - 内容营销：创造能激发用户欲望的内容                               │
│  - 软文推广：通过推荐类内容引导消费                                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 1.5 新渠道开拓的价值与策略

#### 新渠道开拓的重要性

```
┌─────────────────────────────────────────────────────────────────────┐
│                   新渠道开拓的价值                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  红利期优势：                                                       │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  1. 流量成本低                                               │   │
│  │     - 2011年微博推广：几百元一条几十万粉丝的大号             │   │
│  │     - 现在：几千元到一万元一条                               │   │
│  │                                                              │   │
│  │  2. 推广效率高                                               │   │
│  │     - 2012年微博：几毛钱获取一个用户                         │   │
│  │     - 现在：几十元一个用户                                   │   │
│  │                                                              │   │
│  │  3. 竞争少                                                  │   │
│  │     - 早期知乎：付费找大V点赞的人少                         │   │
│  │     - 早期豆瓣：小组组长付费置顶                             │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  历史案例：                                                         │
│  - 2011年微博红利期：低价获取流量                                   │
│  - 2013年九宫格推广：几毛钱获取APP用户（现在几十元）              │
│  - 2016年快手红利期：早期进入者享受流量红利                       │
│                                                                     │
│  结论：                                                            │
│  新渠道开拓可能帮你节省10倍以上的推广成本                          │
│                                                                     │

#### 渠道深耕策略

```
┌─────────────────────────────────────────────────────────────────────┐
│                   渠道深耕策略                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  核心原则：                                                         │
│  与其铺100个渠道，不如把一个有效渠道挖透                            │
│                                                                     │
│  原因分析：                                                         │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  1. 每个渠道都有深度可挖                                     │   │
│  │     - 百度竞价：有人能做到1:10，有人只做到1:1                │   │
│  │     - 差距在于对渠道规则的深入理解和精细化运营               │   │
│  │                                                              │   │
│  │  2. 渠道效果差异大                                           │   │
│  │     - 不会做的人：每个渠道都浅尝辄止                         │   │
│  │     - 会做的人：花3-6个月摸透一个渠道                        │   │
│  │                                                              │   │
│  │  3. 资源分散导致每个渠道都做不好                             │   │
│  │     - 100个渠道 × 1%精力 = 每个渠道都做不好                  │   │

│  ┌─────────────────────────────────────────────────────────────┐   │

│  │  Step 3: 花3-6个月深度研究渠道规则和玩法                    │   │

│                                                                     │

## 案例分析

### 案例：在线教育课程的渠道发现

**背景**：
一家专注于职场技能培训的在线教育公司，需要找到有效的推广渠道。

**问题**：
如何精准定位目标用户，并找到合适的推广入口？

**解决方案**：

```
第一步：精准定义用户
┌─────────────────────────────────────────────────────────────┐
│  定义维度            │  具体描述                             │
├─────────────────────────────────────────────────────────────┤
│  职业状态           │  0-3年职场新人，或想转行者            │
│  年龄范围           │  22-30岁                              │
│  地域               │  一二线城市                           │
│  痛点               │  薪资低、晋升难、竞争力不足            │
│  需求               │  提升技能、升职加薪                   │
└─────────────────────────────────────────────────────────────┘

第二步：四个方向思考渠道
┌─────────────────────────────────────────────────────────────┐
│  方向1：激发的欲望                                               │
│  - 看到同龄人薪资翻倍的案例                                     │
│  - 看到别人转型成功的故事                                       │
│  → 渠道：公众号转型故事、知乎职场回答                           │
├─────────────────────────────────────────────────────────────┤
│  方向2：刚性需求                                                 │
│  - 找工作困难，投简历没回应                                     │
│  - 工作做不好被领导批评                                         │
│  → 渠道：招聘网站、职场社区                                     │
├─────────────────────────────────────────────────────────────┤
│  方向3：活跃场所                                                 │
│  - 活跃在知乎、B站、职场公众号                                   │
│  - 参加线下行业活动                                              │
│  → 渠道：知乎问题、垂直公众号、线下沙龙                         │
├─────────────────────────────────────────────────────────────┤
│  方向4：主动搜索                                                 │
│  - 搜索"如何提升职场竞争力"                                     │
│  - 搜索"零基础转行"                                             │
│  → 渠道：百度搜索、知乎搜索                                     │
└─────────────────────────────────────────────────────────────┘

第三步：选择主攻渠道
- 优先选择：知乎（精准匹配+搜索场景）
- 辅助渠道：垂直公众号
- 测试渠道：B站
```

**结果**：
- 通过知乎获取了大量精准用户
- 用户获取成本比百度投放低60%

## 核心概念表格

| 概念 | 定义 | 关键要点 |
|------|------|---------|
| 用户精准定义 | 清晰界定目标用户的特征 | 年龄、职业、地域、痛点、需求 |
| 推广入口 | 用户触达产品的渠道和场景 | 用户在哪里，入口就在哪里 |
| 四种思考方向 | 发现推广入口的方法论 | 欲望激发、刚性需求、活跃场所、主动搜索 |
| 渠道红利 | 新渠道早期的流量和价格优势 | 先入者享受低成本流量 |
| 渠道深耕 | 集中资源深度运营单个渠道 | 精而不滥 |

## 本章小结

### 1. 精准定义用户的重要性
- 用户定义越精准，推广效率越高
- 宽泛定义难以找到精准渠道
- 精准定义能快速锁定目标用户

### 2. 精准定义用户的三种方法
- 分析现有产品数据
- 用户调研和访谈
- 基于假设的小范围测试

### 3. 发现推广入口的四种思考方向
- **方向1**：因A激发的消费欲望 - 用户看到什么会想买
- **方向2**：因A产生的刚性需求 - 用户遇到什么问题必须解决
- **方向3**：用户活跃的场所 - 用户在哪里活跃
- **方向4**：用户主动搜索的场景 - 用户有需求会搜什么

### 4. 新渠道开拓的价值
- 红利期流量成本低
- 推广效率高
- 可能节省10倍以上成本

### 5. 渠道深耕策略
- 与其铺100个渠道，不如挖透1个渠道
- 花3-6个月深度研究一个渠道
- 找到有效打法后规模化复制
