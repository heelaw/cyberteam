# 第二章：4类常见营销推广方式及其操作逻辑

> 所属章节：第三章
> 包含小节：4类推广方式详解、操作逻辑、案例分析

---

## 本章导学

### 核心目标

本章的核心目标是帮助学员系统掌握市面上常见的四类营销推广方式。市面上的营销推广方式繁多，包括信息流推广、搜索推广、事件营销、SEM等，令人眼花缭乱。但万变不离其宗，这些推广方式都可以归纳为四类，而每类推广方式在落地时影响曝光、点击、转化的核心因素各不相同。通过本章的学习，学员将能够根据自身产品特点选择最适合的推广策略，并掌握各类推广方式的操作要点。

### 学习重点

本章的学习重点包括：理解四类营销推广方式的分类逻辑和各自特点；掌握每类推广方式的操作流程和关键要素；了解不同推广方式的适用场景和优劣势；能够根据产品特性选择合适的推广组合策略。

---

## 详细内容

### 2.1 四类营销推广方式总览

从影响曝光、点击、转化三个核心因素的角度，市面上的营销推广方式可以归纳为以下四类：

```
┌─────────────────────────────────────────────────────────────────┐
│                    营销推广方式分类体系                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1️⃣ 直接购买固定媒介资源                                      │
│     └─ 平面媒体、地铁灯箱、APP首页、公众号大号                  │
│                                                                 │
│  2️⃣ 利用流量平台规则和免费资源                               │
│     └─ ASO、SEO、电商平台专题、社区运营                         │
│                                                                 │
│  3️⃣ 事件营销/传播型推广                                       │
│     └─ 社交媒体事件、病毒式传播、话题营销                       │
│                                                                 │
│  4️⃣ 精准营销/效果广告投放                                     │
│     └─ SEM、信息流广告、网盟广告、朋友圈广告                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 第一类：直接购买固定媒介资源

**定义**：通过直接联系媒介公司或广告渠道，购买特定时间、特定位置的媒介资源进行推广。

**典型形式**：

| 推广形式 | 举例 |
|---------|------|
| 平面媒体 | 报纸杂志版面 |
| 户外媒体 | 地铁灯箱广告 |
| 数字媒体 | APP首页广告位、网站横幅 |
| 新媒体 | 公众号推文、微博大号转发、视频博主合作 |

**操作流程**：

```
┌─────────────────────────────────────────────────────────────────┐
│           直接购买媒介资源推广的完整流程                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  步骤1：找到媒介公司/广告渠道                                    │
│    ├─ 广告联盟                                                  │
│    ├─ 自媒体联盟                                                │
│    └─ MCN机构                                                  │
│           ↓                                                     │
│  步骤2：获取资源列表和报价（广告刊例）                          │
│           ↓                                                     │
│  步骤3：商务洽谈                                                │
│    ├─ 确定资源位                                                │
│    ├─ 谈判价格                                                  │
│    └─ 确定排期                                                  │
│           ↓                                                     │
│  步骤4：准备营销物料                                            │
│    ├─ 图片素材                                                  │
│    ├─ 文案内容                                                  │
│    └─ 后端承接准备                                              │
│           ↓                                                     │
│  步骤5：上线广告                                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**特点分析**：

| 维度 | 影响因素 | 说明 |
|------|---------|------|
| 曝光量 | 媒介基数 + 内容质量 | 媒介订阅人数固定，但社会化媒体可能因转发而放大 |
| 点击率 | 文案 + 创意素材 | 完全依赖文案和素材的吸引力 |
| 转化率 | 转化流程 + 文案物料 | 取决于整个营销转化链路设计 |

**关键洞察**：在社会化媒体投放时，内容本身的质量会直接影响转发量，从而影响最终曝光规模。

### 2.3 第二类：利用流量平台规则和免费资源

**定义**：通过研究流量平台的流量分发规则，或获取平台的免费资源位，以较低成本进行推广。

**典型形式**：

| 推广形式 | 平台举例 |
|---------|---------|
| ASO | App Store、应用商店搜索优化 |
| SEO | 百度、Google搜索引擎优化 |
| 电商推广 | 淘宝、京东类目和活动专题 |
| 社区运营 | 知乎、豆瓣内容营销 |

**核心逻辑**：这类推广方式的核心是"利用平台规则"，通过优化自身内容来获取平台的免费流量。

```
┌─────────────────────────────────────────────────────────────────┐
│               利用平台规则的推广逻辑                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌──────────────┐      ┌──────────────┐                       │
│  │  了解平台规则  │  →   │  优化自身内容  │  →   获取免费流量   │
│  └──────────────┘      └──────────────┘                       │
│                                                                 │
│  常见平台规则：                                                 │
│  ├─ 搜索排名规则（SEO/ASO）                                    │
│  ├─ 推荐算法规则（信息流）                                     │
│  ├─ 活动入选规则（专题推荐）                                   │
│  └─ 内容分发规则（社区）                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.4 第三类：事件营销/传播型推广

**定义**：通过策划具有话题性的事件或活动，引发用户自发传播，从而获得大规模曝光。

**典型特点**：

- 依赖内容的话题性和传播性
- 可能获得爆发式流量
- 难以规模化复制
- 效果难以预测

**常见形式**：

| 形式 | 举例 |
|------|------|
| 话题营销 | 冰桶挑战、炫富挑战 |
| 病毒内容 | 刷屏海报、H5小游戏 |
| 热点借势 | 节日营销、事件营销 |
| 用户共创 | UGC活动、挑战赛 |

### 2.5 第四类：精准营销/效果广告投放

**定义**：通过精准定向技术，将广告展示给目标用户群体，按效果付费的推广方式。

**典型形式**：

| 形式 | 平台举例 |
|------|---------|
| SEM | 百度搜索、谷歌搜索 |
| 信息流广告 | 抖音、微信朋友圈、今日头条 |
| 网盟广告 | 百度网盟、谷歌Display |
| 电商广告 | 淘宝直通车、京东快车 |

**核心优势**：

```
┌─────────────────────────────────────────────────────────────────┐
│                   效果广告的核心优势                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. 精准定向    →  只展示给目标用户                             │
│  2. 按效果付费  →  成本可控                                     │
│  3. 数据透明    →  效果可监测                                   │
│  4. 快速调整    →  优化灵活                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 案例分析

### 案例：奥迪知乎事件营销

**背景**：奥迪汽车品牌在知乎平台进行的一次成功的事件营销。

**操作方式**：
奥迪官方回答了"用汽车发动机煎牛排这事靠谱吗？"这个问题，为了回答这个问题，团队专门购买书籍研究，并录制了实验视频。

**效果**：
该回答获得了超过3000个赞，引发了大量用户关注和讨论。

**分析要点**：

1. **话题选择巧妙**：将汽车发动机与日常生活（煎牛排）结合，创造反差感和话题性。

2. **内容质量高**：不是简单的硬广，而是真正有价值的内容，用户愿意主动传播。

3. **平台调性匹配**：选择知乎这个知识分享平台，以专业态度做品牌传播。

## 核心概念表格

| 推广类型 | 核心特点 | 成本模式 | 适用场景 |
|---------|---------|---------|---------|
| 直接购买固定媒介 | 购买特定位置资源 | 按时间/按次 | 品牌曝光、大规模推广 |
| 利用平台规则 | 免费/低成本 | 人力投入 | 长期运营、精准获客 |
| 事件营销 | 话题性、传播性 | 创意成本 | 品牌声量、话题引爆 |
| 效果广告 | 精准定向、按效果付费 | CPC/CPM/CPA | 快速获客、效果导向 |

## 本章小结

本章介绍了四类常见的营销推广方式：

1. **直接购买固定媒介资源**：通过购买特定位置和时间的媒介资源进行推广，曝光量取决于媒介基数，内容质量影响转发效果。

2. **利用流量平台规则和免费资源**：通过研究平台规则获取免费流量，成本低但需要专业技能。

3. **事件营销/传播型推广**：通过策划话题性事件引发自发传播，可能获得爆发式流量但难以复制。

4. **精准营销/效果广告投放**：通过精准定向技术按效果付费，优势在于精准、可控、可优化。

选择推广方式时，需要根据产品特点、目标用户、预算规模等因素综合考虑，有时需要组合多种推广方式才能达到最佳效果。
