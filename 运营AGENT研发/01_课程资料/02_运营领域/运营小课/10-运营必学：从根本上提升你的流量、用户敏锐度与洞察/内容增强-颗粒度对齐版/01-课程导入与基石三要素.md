## 1.1 课程导入：为什么运营人必须掌握流量与用户洞察

### 1.1.1 运营工作的真相

在互联网公司里，无论你是做产品相关的工作、写营销文案，还是做促销活动、用户互动活动的策划，背后都贯穿着一个核心真相：**一切运营工作的底层都有基本规律，而这些规律中最根源的东西叫做"流量和用户洞察"**。

我们来看一个真实的故事。

在过去五六年时间里，为了让用户能够真正实践落地、更好地提升文案技巧，有看学社对比了近百门行业里的课程，花很大功夫做了一门非常精细落地的文案课程。这门课程的精细程度在行业里是领先的。

**行业里其他文案课程的教学方式**：告诉你"好的文案和不好的文案有什么区别"，总结出三点注意事项，一二三条款列完，你自己回去写。

**有看学社的精细做法**：让你写任何文案时，比如面前有一个水杯要卖出去——
- 首先用一套方法找到关键卖点
- 接着给出一套"如何面向用户完成说服"的具体方法
- 再到不同场景下的文案特性怎么精简（APP Push、地铁广告、公众号推文各有不同）

精细到这个颗粒度，确实非常细致了。

### 1.1.2 一个让所有人沉默的实战案例

然而，在实战作业中，让学员学完这门课后试着做一个任务——为一家鲜花品牌公司写一组海报文案，用来在圣诞节做产品促销。

**背景**：花点时间这家公司，圣诞节前后原价99元送四束花的活动，圣诞两天以59元特价发售，通过100个KOL朋友圈推广引爆活动。老板让你负责物料，给KOL们写朋友圈文案。

**学员们写出来的文案**（学完技巧后）："主题鲜花，送给女神的不二选择。"

品牌方看完沉默，用户看完也觉得"好像不太对"、"离用户有点远"。

**调整后的做法**：让学员重新研究用户——对花点时间的用户做定向调研。

**调研发现的两个基本结论**：
- 女性用户占比78.8%，男性用户只有20%出头
- 女性用户中超过一半是给自己买花，不是送别人

**用户画像**：高收入、年龄小、易于接受新鲜事物的女性，从事互联网、金融、广告、公关、媒体等高知的行业和领域。

这和之前假设的"买花送女朋友的男性用户"完全不同！

**进一步调研**：看淘宝和花点时间公众号下用户买花后的评价——

发现女性用户买花更多是为了：
- 装点自己的生活
- 留住美好
- 让自己开心

用户留言说："花开放的时候，真的美到极致，特别想把图片分享给你们"、"看到这些花心情都好起来了"、"让我留住了一些美好的瞬间"。

**调整后学员写出的文案**：
> "花点时间，犒劳自己。作为一个对生活拥有美好追求的女生，在圣诞节如此重要的节日里边，一边虽然说很忙碌仍然还在上班，另一方面也不能放弃自己的小缺心。你得花点时间，高颜值的鲜花，然后只要59块，你需要去满足一下自己的小缺心。"

品牌方再来看："这个文案确实有感觉多了，跟用户的调性更匹配了。"

### 1.1.3 问题的根源

**结论**：很多人学了不少套路和方法，但不管做活动还是写文案，效果总是差那么一点。问题的根源往往在于——**太关注技巧和套路，但很大程度上忽略了用户**。当你不了解用户时，一味学套路和方法，未必能有很好的效果。

### 1.1.4 运营工作的三层结构

一切运营工作从操作到做出结果，背后总是有三层东西：

```
┌─────────────────────────────────┐
│           效果层                │  ← 运营要负责的：流量、转化、用户活跃
├─────────────────────────────────┤
│           套路层                │  ← 方法、工具、模板、SOP
├─────────────────────────────────┤
│          底层规律层             │  ← 用户需求、场景洞察、传播规律
└─────────────────────────────────┘
```

我们在实际工作中往往都要对效果负责，要提升特定的业务指标。为了达成效果，会有各种方法和工具。但**在套路方法工具的下面，还有重要的基本规律——你得深刻理解用户需求、场景、传播、流量等背后的核心逻辑**。

所以，虽然一切运营工作都离不开传播、流量、用户维系、转化，但如果要真正做好这些工作，最根源的东西就是：**流量和用户的洞察**。

## 1.2 基石三要素：用户、场景、需求

### 1.2.1 什么是基石三要素

在互联网公司里，无论做什么工作，背后都有着三个最基本的要素：

1. **用户**——到底你的用户是一群什么样的人
2. **场景**——在什么场景下
3. **需求**——用户有什么样的需求需要被满足或解决

**用户**：不仅仅是"女性用户"这么笼统，而是要具体到：
- 一二线城市还是三四五线城市女性用户？
- 单身还是非单身？
- 二次元萌妹子还是中年宝妈？

你对用户的界定越清晰，在实际工作中找点来戳中他们、找点来互动时，也会变得更具体和具象。

**场景**：场景的背后有4组关键词——**时间、地点、人物、环境**。

你在什么时间、什么地点，身边有什么样的人，环境有什么特征？

比如早上7点到8点半通勤时，在地铁车厢里，身边是匆匆忙忙的陌生人，环境相对拥挤和局促。

人在不同的环境、不同的场景里，会产生的心理状态、个人感受可能是完全不一样的。

同样一个活动页面，一大早八九点发出来，和晚上零睡前十一二点发出去，效果可能完全不一样：
- 早上：活动页面要显得积极、阳光、充满能量
- 晚上：页面更多要表现出安静、温暖、小治愈的感觉

**需求**：典型用户身上的需求分为两种：
- **生理需求**（功能性的需求）
- **心理需求**（情感性的需求）

比如买一件羽绒服：
- 生理需求：冬天寒冷需要保暖
- 心理需求：是否够时尚、够潮、能否彰显身份和社会地位

### 1.2.2 三要素的无限细分空间

只要你的用户、场景、需求都足够清晰，作为一个运营，你一定可以在这三个要素上分别进行细分，延伸出运营工作中N多的思路。

**案例：外卖APP的运营思路拓展**

假设你在一家外卖APP公司工作，老板告诉你要定期面向用户搞点小主题促销活动。

**三要素基础版**：
- 用户：办公室白领
- 场景：工作日午间，大家在办公室忙碌
- 需求：生理需求是饿了需要饱腹，心理需求是懒得外出

**细分后**：

| 维度 | 可细分的方向 | 具体例子 |
|------|--------------|----------|
| 用户细分 | 性别 | 男性用户、女性用户 |
| | 口味偏好 | 吃辣爱好者、面食爱好者 |
| | 职级 | 中层管理者、基层员工 |
| 场景细分 | 天气 | 阳光明媚、小雨淅沥、大雪纷飞 |
| | 人物事件 | 有人临时约饭不知道吃什么、心情不好想约闺蜜聊聊天、刚拿下大单要跟大家High一下 |
| 需求细分 | 生理需求 | 饿、想吃饱 |
| | 心理需求 | 想激发同事干劲、想借吃饭跟同事混个脸熟 |

**从细分中产生的活动灵感**：
- 面向吃辣爱好者 + 大雪纷飞天气 → "大雪天，无火锅不欢"主题活动
- 面向中层管理者 + 拿下大单 → "庆功宴"主题活动

这就是三要素细分的价值——**每个要素对你来说，都有无限进一步细分的空间。这些信息深深映入你脑海中后，一定会成为你日常写文案、策划活动时手中的一些重要抓手**。

### 1.2.3 三要素的典型应用

基石三要素在写文案、做内容、做活动时，有两种最常见的应用：

**第一种应用：时刻装填三要素，与用户沟通互动，快速建立信任**

做内容或做活动时，脑海中装填着用户场景需求，时刻从这三个要素出发跟用户沟通互动，能帮你快速建立信任，快速唤起用户了解你的欲望。

**案例对比**：

A同学人设：专业方向是帮人解决人际关系问题

A同学的开篇：
> "我晚上看到一个挺逗的热搜，什么两口子吵架，丈夫20楼撒现金，给我什么怎么蒙了。今天就从这个角度，我跟你聊聊，恋爱人类的高质量吵架指南。"

B同学专业方向：教人做知识IP打造

B同学的开篇：
> "做知识IP，典型有两类人群：一类是天赋异禀、能力卓越，之前就已是某个平台有知名度的人；另外一类是大部分普通人，还没有形成自己的影响力，而且没有很大信心。这类人在做的过程中，通常都有一个很纠结心态，典型会遇到的场景——1234，人设定位总觉得不够清晰，该怎么办？有好几个领域内容都可以做，但不知道哪个最适合自己……"

**效果差异**：
- A同学开篇会导致读者一大半流失，无法快速抓住精准用户
- B同学开篇能牢牢圈住精准用户

这就是**对象感**的力量——你脑海中明确知道"我的用户就是一群什么样的人"，你的标题、开篇、中间的场景铺垫，都可以专门针对这一类人来强调："我这篇内容就是为你们而写"。

仅仅是"时刻在你的内容中不断的体现和强化用户场景需求"这一件事，就能够让文案和内容在流量获取和转化效率上提升很多倍。

**第二种应用：锁定用户需求 + 堆砌多个细分场景，戳中用户痛点**

在营销文案场景下，当你锁定了用户的一类需求（比如求职、获得更好发展），通过堆砌多个细分的场景来拼命戳中用户、唤起用户痛点，能带来更高的营销转化率。

**案例1：有看学社知识IP课程**
> "我做了一个调研，发现很多不是知识IP或刚刚起步的准知识IP们，想做这件事最大的问题是下面这几个：
> 1. 如何真的找到符合自己的真实人设、并且能最后跟变现打通的个人差异化定位？
> 2. 如何更好形成稳定的内容输出循环和习惯，不至于让自己是纯干货？
> 3. 如何在最初自己没谱、没信心的时候，还能够持续通过内容输出获得反馈，逐步建立用户认可度，完成自己第一个小的代表作？"

**案例2：课程制作人课程**
> "这门课能帮你解决问题包括但不限于：如何跟老师高效磨课？如何搭建特别靠谱的课程大纲和结构？如何做出对学习效果负责的课程？如何做好课程的交互设计和学前设计？再到如何讲好某一章某一节？我们会以付费类课程、轻训练仪式的课、重结果目标交付类课程为例，分别带你完整拆解行业顶级爆款课程从零到一生产的全流程。"

**案例3：商业操盘手课程**
> "你跟高层Leader思考不同频，经常陷入类似状态——老板在讲商业逻辑，你在讲执行，最后你跟老板之间沟通不同频，老板痛苦你也痛苦；见识少，没见过事情的全貌，所有解决方案只在一个小圈子里打转……"

这就是锁定用户需求后，堆砌多个典型场景来戳中用户的威力。

## 本章核心要点

1. **基石三要素是运营工作的底层能力**：用户、场景、需求，三者缺一不可
2. **三要素可以无限细分**：每个维度都可以细分出N多方向，成为运营抓手
3. **两种核心应用方式**：
   - 时刻装填三要素，与用户建立信任
   - 锁定需求+堆砌场景，戳中痛点提升转化
4. **对象感是关键**：永远要知道"我这篇内容/这个活动是为谁做的"
