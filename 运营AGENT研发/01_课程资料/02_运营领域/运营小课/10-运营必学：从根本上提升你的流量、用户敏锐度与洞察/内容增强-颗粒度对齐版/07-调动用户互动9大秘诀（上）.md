> 主讲人：黄有璨
> 课程范围：运营必学：从根本上提升你的流量、用户敏锐度与洞察 第6节

## 本节导读

运营与产品的关系：**产品提供长期价值，运营创造短期价值**。

即使课程内容精良，用户也未必能学完。运营需要创造短期价值，吸引并激励用户持续参与。

本节将分享撬动用户互动参与动机的9大秘诀中的前4个：
1. 有吸引力的奖励和概率事件
2. 创造稀缺感
3. 调动用户的竞争和攀比心理
4. 调动用户的炫耀和猎奇心理

---

## 一、运营与产品的关系

### 1.1 产品 vs 运营的价值维度

| 维度 | 产品 | 运营 |
|------|------|------|
| **价值类型** | 长期价值 | 短期价值 |
| **价值体现** | 功能解决问题 | 体验满足需求 |
| **作用时间** | 持续稳定 | 阶段性刺激 |
| **用户感知** | "这个东西有用" | "现在参与很值" |
| **典型案例** | 课程内容质量 | 打卡活动、抽奖激励 |

**核心认知**：
- 产品好 ≠ 用户会用完
- 好产品需要好运营激活
- 运营负责降低启动门槛、增强使用动机

### 1.2 运营创造短期价值的场景

| 场景 | 运营动作 | 短期价值 |
|------|----------|----------|
| **新用户注册** | 新人礼包、引导任务 | 即时反馈、降低门槛 |
| **促活留存** | 签到活动、限时福利 | 持续参与动机 |
| **内容消费** | 打卡挑战、进度激励 | 完成动力 |
| **社交传播** | 分享有奖、拼团优惠 | 社交货币 + 实际利益 |

## 二、秘诀一：有吸引力的奖励 + 概率事件

### 2.1 核心原理

**心理学机制**：

| 心理效应 | 说明 | 应用 |
|----------|------|------|
| **奖励预期** | 期待奖励时多巴胺分泌 | 提前展示奖励 |
| **概率性强化** | 不确定奖励比固定奖励更上瘾 | 抽奖机制 |
| **损失厌恶** | 得到后不想失去 | 限时使用优惠券 |
| **互惠原理** | 得到好处后想回报 | 转发、分享 |

**核心公式**：
```
用户参与动机 = 奖励吸引力 × 获得概率 × 即时性
```

### 2.2 案例分析

#### 案例1：支付宝中国锦鲤（2018）

**活动背景**：
- 时间：2018年9月29日
- 平台：支付宝微博
- 主题：寻找中国锦鲤

**活动规则**：
```
1. 转发支付宝官方微博的"中国锦鲤"活动微博
2. 抽取1位幸运儿成为"中国锦鲤"
3. 获得"全球免单大礼包"
```

**奖励清单（部分）**：

| 奖励类别 | 内容 | 价值估算 |
|----------|------|----------|
| **旅行** | 全球多地机票、酒店 | 约50万元 |
| **购物** | 各大品牌礼券、奢侈品 | 约80万元 |
| **餐饮** | 全球美食代金券 | 约20万元 |
| **娱乐** | 演出门票、主题乐园 | 约10万元 |
| **服务** | 体检、美容、教育 | 约20万元 |
| **总计** | - | **超过200万元** |

**传播数据**：
- 微博转发量：**300万+**
- 微博评论量：**100万+**
- 微博点赞量：**200万+**
- 话题阅读量：**25亿+**
- 品牌曝光：**千亿级**

**成功要素分析**：

| 要素 | 说明 |
|------|------|
| **超值奖励** | 总价值超200万，远超用户预期 |
| **单一大奖** | 只抽1人，稀缺性极强 |
| **清单式呈现** | 详细列出奖励清单，视觉冲击力强 |
| **低参与门槛** | 仅需转发微博，操作简单 |
| **品牌联合** | 数百品牌联合，增强可信度 |
| **话题性** | "锦鲤"成为年度热梗 |
| **社交货币** | 转发彰显自己"有运气" |
| **概率性** | 中奖概率低但期待值高 |

**运营节奏**：

```
Day 0: 发布活动微博（上午10点）
        ↓
Day 0-1: KOL转发造势（头部大V）
        ↓
Day 1-2: 自来水传播（用户自发）
        ↓
Day 2-3: 媒体报道（二次传播）
        ↓
Day 7: 公布中奖者（信小呆）
        ↓
Day 7+: 中奖者行程直播（持续热度）
```

**后续影响**：
- "锦鲤"成为年度热词
- 引发全网"锦鲤"梗狂欢
- 无数品牌跟风做"锦鲤"活动
- 中奖者信小呆成为网红

**ROI分析**：
- 投入：奖励成本200万 + 执行成本50万 = 250万
- 产出：品牌曝光价值>2亿（按CPM计算）
- ROI：**80倍+**

#### 案例2：苏宁818登录抽奖

**活动背景**：
- 时间：每年8月
- 平台：苏宁APP
- 目的：促活、引流、转化

**活动机制**：

```
用户登录APP → 跳出抽奖小游戏（老虎机） → 点击抽奖 → 获得优惠券 → 限时使用 → 促进消费
```

**活动界面元素**：
- 老虎机样式（视觉吸引力）
- 一键抽奖（低操作成本）
- 代金券/满减券（即时奖励）
- 限时提醒（损失厌恶）
- 再次抽奖机会（复购动机）

**用户心理路径**：

```
1. 看到抽奖 → 好奇（是什么？）
                ↓
2. 点击抽奖 → 期待（能抽到什么？）
                ↓
3. 获得优惠券 → 占便宜感（免费得到）
                ↓
4. 看到限时 → 损失厌恶（不用就亏了）
                ↓
5. 浏览商品 → 购物动机（券不能浪费）
                ↓
6. 完成购买 → 转化达成
```

**数据效果**（参考值）：
- APP日活提升：**30-50%**
- 抽奖参与率：**60%+**
- 优惠券核销率：**20-30%**
- 活动期间GMV提升：**15-25%**

**成功要素分析**：

| 要素 | 说明 | 心理机制 |
|------|------|----------|
| **概率性奖励** | 随机抽券，不确定性强 | 多巴胺分泌，上瘾机制 |
| **即时反馈** | 点抽即出结果 | 满足即时满足 |
| **零参与成本** | 仅需点击抽奖 | 降低行动门槛 |
| **损失厌恶** | 限时优惠券 | 不用就觉得亏 |
| **锚定效应** | 原价vs券后价 | 感知超值 |
| **复购动机** | 多次抽奖机会 | 持续参与 |

### 2.3 概率性奖励 vs 固定奖励

| 对比维度 | 概率性奖励 | 固定奖励 |
|----------|------------|----------|
| **用户期待** | 持续好奇 | 一次性满足 |
| **参与动机** | 可能性驱动 | 目标驱动 |
| **上瘾性** | 强（不确定奖赏效应） | 弱 |
| **成本控制** | 可控（设中奖率） | 刚性（每人必发） |
| **传播性** | 强（晒欧气） | 弱 |
| **适用场景** | 拉新、促活、裂变 | 留存、转化 |

**概率性奖励的设计公式**：

```
大奖价值 × 中奖概率 + 小奖价值 × 中奖率 = 用户期望值

优化方向：
- 提高大奖吸引力（话题性）
- 提高中小奖概率（参与感）
- 保证保底奖励（降低失望）
```

**应用场景对照表**：

| 场景 | 奖励类型 | 设计要点 |
|------|----------|----------|
| **新用户注册** | 固定奖励 | 新人礼包，确定性奖励 |
| **促活登录** | 概率奖励 | 每日抽奖，持续吸引 |
| **社交裂变** | 概率+固定 | 邀请得券（固定），抽奖升级（概率） |
| **消费激励** | 固定奖励 | 满减券，确定性利益 |
| **爆款活动** | 概率大奖 | 制造话题，吸引关注 |

### 2.4 秘诀一实操清单

**奖励设计检查清单**：

- [ ] 奖励有吸引力吗？（用户觉得值）
- [ ] 参与门槛够低吗？（3步内完成）
- [ ] 有即时反馈吗？（秒出结果）
- [ ] 有概率机制吗？（上瘾设计）
- [ ] 有损失厌恶吗？（限时/限量）
- [ ] 有传播动机吗？（社交货币）
- [ ] 成本可控吗？（概率/预算）

## 三、秘诀二：稀缺感

### 3.1 核心原理

**心理学机制**：

| 心理效应 | 说明 | 应用 |
|----------|------|------|
| **稀缺效应** | 越稀缺越想要 | 限量、限时、限资格 |
| **损失厌恶** | 害怕失去机会 | 紧迫感营造 |
| **从众心理** | 大家抢的一定好 | 显示剩余名额/销量 |
| **身份认同** | 稀有资格彰显身份 | 会员、认证 |

**核心公式**：
```
稀缺感 = 数量有限 + 时间紧迫 + 资格门槛 + 竞争感知
```

### 3.2 案例分析

#### 案例：500强销售总监课程（2012）

**背景**：
- 时间：2012年
- 平台：技能交易平台
- 产品：销售技能课程
- 讲师：500强销售总监

**讲师背景**：
- 某世界500强公司中国区销售总监
- 高中辍学 → 街边摆摊 → IBM → 销售总监
- 传奇经历 + 实战经验

**第一次尝试：失败**

| 维度 | 内容 |
|------|------|
| **课程名称** | 《500强销售总监的独门销售秘籍》 |
| **售卖方式** | 直接售卖课程 |
| **定价** | <1000元 |
| **推广渠道** | 10+个付费渠道 |
| **转化率** | <1% |
| **结果** | 非常惨淡 |

**复盘思考**：
- 课是好课吗？是
- 老师是好老师吗？是
- 价格贵吗？不算贵
- **问题出在哪？** 卖法有问题

**第二次尝试：成功**

| 维度 | 内容 |
|------|------|
| **课程名称** | 《世界500强中国区销售总监限量招收20位门徒，海选开始》 |
| **售卖方式** | 限额海选招募 |
| **定价** | <1000元（未变） |
| **推广渠道** | 相同（未变） |
| **转化率** | ~10% |
| **提升倍数** | **10倍+** |

**关键变化对比**：

| 对比维度 | 第一次 | 第二次 | 差异 |
|----------|--------|--------|------|
| **定位** | 我有课，你来买 | 我选人，你来申请 | 卖→买 vs 选→被选 |
| **角色** | 课程提供者 | 师徒关系 | 商业关系 vs 师徒关系 |
| **稀缺性** | 无限制 | 仅20位 | 无限 vs 稀缺 |
| **门槛** | 付费即可 | 需要申请筛选 | 无门槛 vs 有门槛 |
| **竞争** | 无竞争 | 海选竞争 | 无 vs 有 |
| **预期** | 买课学习 | 拜师学艺 | 消费 vs 身份 |

**心理学分析**：

| 心理机制 | 说明 |
|----------|------|
| **稀缺效应** | "仅20位" → 必须抢 |
| **资格门槛** | "海选" → 不是谁都能来 |
| **身份认同** | "门徒" → 师徒关系，身份升级 |
| **竞争心理** | "申请筛选" → 担心被拒绝 |
| **损失厌恶** | "错过这次就没机会了" |
| **权威背书** | "500强总监" → 顶级资源 |

**A/B测试数据对比**：

| 指标 | 方案A（直接售卖） | 方案B（限额海选） | 提升 |
|------|-------------------|-------------------|------|
| **点击率** | 2% | 5% | +150% |
| **转化率** | 0.8% | 9.5% | +1088% |
| **客单价** | 998元 | 998元 | - |
| **ROI** | 1:1.2 | 1:15 | +1150% |

**成本对比**：
- 方案A：获客成本 = 推广费 / 订单数
- 方案B：获客成本 = 推广费 / 订单数（相同推广费，更多订单）

### 3.3 稀缺感的6种应用场景

| 场景 | 稀缺类型 | 实操方法 | 案例 |
|------|----------|----------|------|
| **新品发售** | 数量稀缺 | "限量1000台" | 小米F码、茅台限量 |
| **课程招生** | 名额稀缺 | "仅招20人" | 门徒计划、训练营 |
| **时间促销** | 时间稀缺 | "限时24小时" | 双11秒杀、闪购 |
| **资格认证** | 身份稀缺 | "邀请制" | Clubhouse、知乎盐选 |
| **版本首发** | 时间稀缺 | "首发独家" | 游戏抢先体验 |
| **会员权益** | 等级稀缺 | "仅VIP可享" | 京东Plus、天猫88VIP |

**稀缺感设计公式**：

```
稀缺强度 = 稀缺维度（数量×时间×资格）× 可信度 × 紧迫感

维度说明：
- 稀缺维度：至少2个维度同时稀缺
- 可信度：说限量就真的限量（不能骗人）
- 紧迫感：倒计时、剩余名额实时显示
```

### 3.4 秘诀二实操清单

**稀缺感设计检查清单**：

- [ ] 有明确的稀缺维度吗？（数量/时间/资格）
- [ ] 稀缺信息清晰可见吗？（首页突出显示）
- [ ] 有紧迫感营造吗？（倒计时、实时数据）
- [ ] 稀缺是真实的吗？（说到做到）
- [ ] 有竞争感知吗？（显示抢购热度）
- [ ] 有损失厌恶吗？（错过不再有）
- [ ] 符合品牌调性吗？（不是所有品牌都适合）

**注意事项**：
- ⚠️ 过度稀缺会引发反感
- ⚠️ 虚假稀缺损害信任
- ⚠️ 高端品牌慎用"限时抢购"
- ⚠️ 需要平衡稀缺与用户体验

## 四、秘诀三：调动用户竞争和攀比心理

### 4.1 核心原理

**心理学机制**：

| 心理效应 | 说明 | 应用 |
|----------|------|------|
| **社会比较** | 通过与他人比较评估自己 | 排行榜、PK机制 |
| **竞争动机** | 想要赢过别人 | 排名、竞赛 |
| **从众心理** | 大家都做的事我也要做 | 显示参与人数 |
| **地位追求** | 想要获得认可和地位 | 等级、徽章、称号 |

**竞争类型**：

| 类型 | 驱动因素 | 设计要点 |
|------|----------|----------|
| **横向竞争** | 与他人比较 | 排行榜、PK |
| **纵向竞争** | 与自己比较 | 进度条、成就系统 |
| **团队竞争** | 团队间对抗 | 组队PK、战队赛 |
| **隐性竞争** | 不直接显示排名 | 角标、徽章收集 |

### 4.2 案例分析

#### 案例：613互联网合伙人招募节（2015）

**活动背景**：
- 时间：2015年6月
- 平台：周伯通招聘
- 背景：2015年是互联网创业高峰期
- 需求：大量创业公司寻找合伙人

**活动目标**：
- 线上传播量：30万+
- 线下参与CEO/合伙人：1800+
- 线下活动规模：100+人

**资源限制**：
- 时间：仅3周
- 人力：不到3人（2.5人）
- 预算：极低

**活动规则设计**：

```
1. 创始人/CEO报名需填写表单
2. 分享H5到朋友圈/微信群寻求支持
3. 点赞数量排名决定推荐优先级
4. 前20名获得线下活动参与资格
5. 线下活动有行业大佬（定向邀请）
```

**线下活动嘉宾**：
- 知名游戏公司创始人/CEO
- 知名VR公司创始人/CEO
- A股上市公司老大
- 各行业大佬

**参与数据**：

| 指标 | 数据 |
|------|------|
| **线上传播** | 30万+ |
| **报名CEO/合伙人** | 1800+ |
| **线下参与** | 100+人 |
| **创始人/CEO** | ~50人 |
| **合伙人候选人** | ~80人 |

**线下参与者质量（部分）**：

| 类型 | 人数 | 质量说明 |
|------|------|----------|
| **技术架构师** | 23人 | 2015年非常值钱 |
| **有创业经验** | 18人 | 有创业经历 |
| **百度系** | 26人 | 大厂背景 |
| **腾讯系** | 14人 | 大厂背景 |
| **阿里系** | 13人 | 大厂背景 |
| **PayPal中国区市场负责人** | 1人 | 高管 |
| **微软Bing商业产品负责人** | 1人 | 高管 |

**核心机制：排行榜驱动**

**用户行为路径**：

```
CEO报名活动
    ↓
分享H5到朋友圈/员工群
    ↓
请求点赞支持
    ↓
查看排行榜排名
    ↓
发现被竞争对手超越
    ↓
加大拉票力度（发到更多群）
    ↓
排名上升
    ↓
竞争加剧
    ↓
病毒式传播
```

**经典案例：富二代CEO**

- 背景：家里有工厂，数千员工
- 操作：发到工厂群，行政命令每天点赞
- 要求：完不成点赞任务受处罚
- 结果：排名进入前三，获得参与资格

**竞争行为观察**：

| 现象 | 说明 |
|------|------|
| **明争暗斗** | 同行业公司互相较劲 |
| **你追我赶** | A公司超B公司，B公司必反超 |
| **全员动员** | 动用一切可动用的资源 |
| **持续关注** | 每天查看排名变化 |

**成功要素分析**：

| 要素 | 说明 | 效果 |
|------|------|------|
| **排行榜机制** | 实时显示排名，激发竞争 | 核心驱动力 |
| **稀缺名额** | 仅前20名能参与线下 | 强化竞争动机 |
| **高价值奖励** | 线下活动有大佬、有合伙人 | 值得竞争 |
| **社交展示** | 分享到朋友圈，彰显地位 | 社交货币 |
| **实时反馈** | 点赞数实时更新 | 持续刺激 |
| **零参与成本** | 仅需分享和拉票 | 降低门槛 |

**数据效果**：

| 指标 | 数据 | 说明 |
|------|------|------|
| **预算** | 极低 | <5万 |
| **人力** | 2.5人 | 3周时间 |
| **传播量** | 30万+ | 主要靠CEO自发传播 |
| **ROI** | 极高 | 人力+预算 <10万，收获1800+高质量用户 |

**传播链路**：

```
发起活动（运营方）
    ↓
CEO报名（种子用户）
    ↓
CEO分享到公司群（一级传播）
    ↓
员工点赞+转发（二级传播）
    ↓
员工的社交圈（三级传播）
    ↓
竞争效应放大（更多CEO加入）
    ↓
病毒式扩散
```

### 4.3 竞争心理的3种类型

| 类型 | 心理机制 | 设计方法 | 应用场景 |
|------|----------|----------|----------|
| **直接竞争** | 想要赢过对手 | PK、对战、排行榜 | 游戏、竞赛 |
| **地位竞争** | 想要更高地位 | 等级、徽章、称号 | 社区、会员 |
| **从众竞争** | 不想落后于人 | 显示参与人数、热度 | 电商、活动 |

**排行榜设计最佳实践**：

| 设计要素 | 说明 | 示例 |
|----------|------|------|
| **排名可见性** | 实时显示排名变化 | 前100名实时更新 |
| **分层展示** | 不同区间展示策略 | 前三名特别展示 |
| **即时反馈** | 操作后排名立即变化 | 点赞后排名上升 |
| **竞争提醒** | 被超越时通知 | "您被XXX超越了" |
| **奖励梯度** | 不同排名不同奖励 | 第一名大奖，前十名有奖 |
| **公平性** | 明确规则，防作弊 | 每日限票、设备限制 |

### 4.4 秘诀三实操清单

**竞争机制设计检查清单**：

- [ ] 有明确的竞争维度吗？（排名、分数、等级）
- [ ] 有实时反馈吗？（即时显示排名变化）
- [ ] 有奖励梯度吗？（不同排名不同奖励）
- [ ] 有竞争提醒吗？（被超越通知）
- [ ] 有社交展示吗？（可分享成就）
- [ ] 规则公平透明吗？（防作弊机制）
- [ ] 符合用户调性吗？（不是所有用户都喜欢竞争）

**注意事项**：
- ⚠️ 过度竞争可能引发负面情绪
- ⚠️ 需要平衡竞争与合作
- ⚠️ 排行榜尾部用户容易流失
- ⚠️ 需要定期重置排行榜（新用户机会）

## 五、秘诀四：调动用户炫耀和猎奇心理

### 5.1 核心原理

**心理学机制**：

| 心理效应 | 说明 | 应用 |
|----------|------|------|
| **炫耀动机** | 展示自己的好，获得认可 | 分享成就、晒单 |
| **社交货币** | 分享内容彰显自己品味 | 优质内容、稀缺资源 |
| **好奇心** | 对未知、反常事物好奇 | 悬念、反差内容 |
| **从众心理** | 别人都在玩我也要玩 | 显示参与人数 |

**炫耀与猎奇的循环**：

```
用户A炫耀 → 朋友B好奇 → 点击查看 → B也想炫耀 → 分享 → 更多朋友好奇 → 病毒传播
```

### 5.2 案例分析

#### 案例1：一秒入职Facebook（2016愚人节）

**活动背景**：
- 时间：2016年4月1日（愚人节）
- 背景：传言Facebook即将入华
- 平台：周伯通招聘
- 形式：H5小游戏

**活动机制**：

```
1. 扫描二维码进入H5

3. 一键生成入职名片
4. 分享到朋友圈
```

**生成的名片内容**：
- 姓名：用户填写
- 岗位：用户选择
- 入职日期：自动生成（愚人节当天）
- Facebook logo
- "Welcome to Facebook"

**文案示例**：
- "我入职Facebook了"
- "世界最大社交网络，我来啦"
- "Menlo Park，我来了"

**数据效果**：
- 传播时间：1天
- 传播量：**50万+**
- 公众号新增粉丝：**小几万**
- 传播周期：短平快（愚人节当天）

**成功要素分析**：

| 要素 | 说明 |
|------|------|
| **热点借势** | Facebook入华传言 + 愚人节 |
| **炫耀心理** | "我入职Facebook了" → 装逼资本 |
| **猎奇心理** | 朋友看到后好奇 → 点开看 |
| **低参与成本** | 填个名字就能生成 |

### 5.3 炫耀心理的社交货币理论

**社交货币 = 用户分享后获得的"社交收益"**

**社交货币的5种类型**：

| 类型 | 说明 | 设计要点 | 案例 |
|------|------|----------|------|
| **成就炫耀** | 展示自己的成就 | 可视化成就、勋章 | 游戏段位、运动打卡 |
| **品味展示** | 彰显自己的品味 | 小众、高端、有趣 | 小众旅行地、独立音乐 |
| **知识分享** | 显示自己有见识 | 有用、深度内容 | 干货文章、行业洞察 |
| **幽默风趣** | 展示自己有趣 | 搞笑、反转内容 | 段子、表情包 |
| **从众参与** | 不落伍 | 显示参与人数 | 热门话题、全民活动 |

**炫耀设计公式**：

```
炫耀动机 = (成就高度 × 可见性) - (炫耀成本 × 社交风险)

优化方向：
- 提升成就高度（更有价值的成就）
- 提升可见性（更易被看到）
- 降低炫耀成本（更易分享）
- 降低社交风险（不显得尴尬）
```

### 5.4 猎奇心理的好奇心缺口理论

**好奇心缺口 = 已知与未知之间的差距**

**好奇心的触发机制**：

```
用户认知缺口 → 好奇心产生 → 寻求信息 → 缺口填补 → 满足/失望
```

**猎奇设计方法**：

| 方法 | 说明 | 案例 |
|------|------|------|
| **信息缺口** | 提示但不全说 | "你知道XX的秘密吗？" |
| **反差呈现** | 预期与现实的冲突 | "看似普通，实则..." |
| **悬念设置** | 延迟满足 | "结局你绝对想不到" |
| **反常理** | 打破常规认知 | "为什么越XX反而越YY？" |
| **神秘感** | 隐藏部分信息 | "神秘嘉宾" |

### 5.5 秘诀四实操清单

**炫耀机制设计检查清单**：

- [ ] 有炫耀的价值点吗？（成就、品味、知识）
- [ ] 可视化呈现吗？（图片、徽章、证书）
- [ ] 易于分享吗？（一键分享、精美模板）
- [ ] 社交风险低吗？（不显得尴尬）
- [ ] 有猎奇点吗？（反差、悬念、神秘）
- [ ] 时效性合适吗？（愚人节、特殊时期）
- [ ] 符合品牌调性吗？（不是所有品牌都适合恶搞）

**注意事项**：
- ⚠️ 过度炫耀可能引发反感
- ⚠️ 虚假炫耀损害信任
- ⚠️ 需要平衡真实与趣味
- ⚠️ 愚人节等特殊时节点更合适

## 六、本节核心要点总结

### 6.1 9大秘诀总览（上）

| 序号 | 秘诀 | 核心机制 | 关键要素 |
|------|------|----------|----------|

| 4 | 炫耀和猎奇心理 | 社交货币 + 好奇心缺口 | 成就展示、反差内容 |

### 6.3 心理学机制总表

|----------|------|----------|
