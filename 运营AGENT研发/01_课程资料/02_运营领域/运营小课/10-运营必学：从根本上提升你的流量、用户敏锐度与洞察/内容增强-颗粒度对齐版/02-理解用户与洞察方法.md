## 2.1 从三要素到用户洞察：为什么你需要更深刻的理解

### 2.1.1 在基石三要素基础上更进一步

基石三要素是运营工作的基础，但如果你要更进一步，就需要**更加深刻地去洞察用户身上潜藏着的重要情绪，以及用户身上的一些重要的心理敏感点**。

做到了这一点，会让你成为一个更加牛逼的运营。

### 2.1.2 两个天差地别的案例

**案例1：两个母婴公众号的差距**

左边这个母婴公众号现在是行业里非常顶级的大号，从零起步一年内就积累了大几百万到千万的用户。

右边是一个长期平平无奇的育儿知识号。

两篇内容主题相同——都是关于"宝宝吃夜奶"这件事。

**右边（普通号）的内容**：
- 偏理性简单的科普
- 讲得比较泛：如果宝宝夜间吃奶量和有效吸吮时间不多怎么办？如果宝宝已经形成吃奶和睡眠的规律大概又该怎么办？
- 泛泛提到就差不多了

**左边（顶级大号）的内容标题**：
> "一个夜奶妈妈的告白——我的夜是你不懂的黑"

**内容结构**：
- 以妈妈的身份讲述整个过程
- 花了几个月时间试着诊断和调整小孩的作息
- 整个过程太艰苦、太煎熬了，斗争了无数次
- 夹杂大量细微情绪的描述：
  > "母乳的好处，我比谁背得都熟；孩子对母亲的依恋，我也高度认可。但是这些精神层面的认可，完全无法源源不断给予一个妈妈永久的能量。作为我刚刚生产、还没有完全恢复的一个新妈妈，被一个小孩子每天经营的'深夜食堂'、每天晚上醒来好几次要喂他夜奶，这样的折磨其实我是非常崩溃的。"
  > "作为我们大人，总是说什么生命不是一场竞赛、孩子无需比较。可是如果听到'孩子需要完整的睡眠、孩子睡不好是家长的责任'这样的理论，你作为一个妈妈，你经历了这一切，你还淡定得了吗？"
  > "你遭遇了孩子睡眠倒退、翻身、长牙，夜间醒来无数次的夜晚；你顶着极度疲劳的双眼，却不得不苦撑着陪他耗。你还能淡定得了吗？"

**数据结果**：
- 右边：阅读量几百到小一千的普通内容
- 左边：第一篇10万+，从此这个大号开始写一系列戳宝妈情绪敏感点的内容
  - 喂夜奶
  - 妈妈一个人独自在家带娃的场景
  - 各种苦楚、孤独、苦闷

最终做成了粉丝几千万的行业大号。

**案例2：两场跨年晚会的口碑差异**

2019年底的两场跨年晚会：
- 湖南卫视：豆瓣评分5.5分，3000多人评价
- B站"2019最美的夜"：豆瓣评分9.1分，3万多人评价

**湖南卫视的配置**：
- 最强主持嘉宾团
- "全民一起摇，瓜分100亿大红包"
- 郭富城、韩红、黄子韬、大张伟、TFBOYS等众多明星

**用户评价**：
> "主持人众多导致很混乱"
> "感觉流量明星很多可压不了场"
> "全程没有重点"
> "毫无特色的浪费钱"

**B站晚会的做法**：
- 完全全心全意为年轻人（尤其是90后）服务
- 开头用《哈利波特》+ 魔术世界，回忆杀开场
- 方锦龙和虚拟偶像同台演奏，中西合璧
- 权力游戏主题曲，年轻人回忆中的存在
- 主持人请的是年轻人非常喜欢、有"段子手"著称的朱广权

**用户评价**：
> "感谢90年左右生的这些人们，爬上了行业的管理层，感觉我太幸福了，有了一个专属于我们的晚会，有被重视的感觉。"
> "如果B站的晚会年年都是这个水平，春晚都可以不用办了。"
> "今年最优秀的跨年晚会，去掉之一。"

**核心差异**：
- 湖南卫视：用户不清晰这台晚会到底是面向谁的、有什么特色
- B站：明确用户受众是90后往后的年轻人，主打互动、有情怀、有回忆杀、有专属感

### 2.1.3 结论

**每一类用户身上，都会有n多平常不被圈外人关注和理解的痛点和情绪**，包括他们专属的回忆、痛苦、憎恨、赞同等。

如果你能激发和理解这些重要的情绪，你才能够成为一个可以更好调动用户和流量的运营。

## 2.2 互联网圈层化与发烧友的力量

### 2.2.1 互联网的圈层化

互联网上用户越来越圈层化，存在着无数的小众圈层，**每一个小众圈层都会有他们专属的情绪敏感点和心理敏感地**。

90后年轻人群体里会有的小众爱好：
- 手办模型
- 潮玩盲盒
- 汉服
-Lolita
- 密室逃脱
- 剧本杀
- Cosplay

每一个小众爱好都能形成一个小众圈层。中年人、老年群体也都拥有着无数的圈层。

### 2.2.2 为什么要关注小众人群

**建议**：作为一个优秀的运营，你一定要保持着旺盛的好奇心，关注这些小众圈层人群，理解他们一般都在干些什么、他们做这些事背后的情绪和心理动机是什么。

如果你能形成这样的一些习惯和能力，会非常重要。

### 2.2.3 发烧友的威力

如果你能深刻体察一小部分小众用户，与他们形成深度的情感共振，大概率就可以抓住他们当中一部分狂热发烧友。**就这么一件事，也将会让你成为一个自带强大威力的运营**。

**为什么发烧友如此重要**：

1. **发烧友是早期最黄金的天使用户**
   - 只要认可你做这件事的初始动机，对你的容错性就会更高
   - 愿意陪你走更远、往前去走更长的路
   - 愿意给你很多反馈和帮助
   - 分享欲、表达欲往往比较旺盛

2. **发烧友可以影响大众，大众影响不了发烧友**
   - 你可以借助发烧友来影响其他大众用户
   - 但想借助大众用户影响发烧友，做不到

3. **影响发烧友的成本很低**
   - 只要你真的懂他们
   - 给他们讲一个他们认可的故事
   - 他们就愿意追随你一路往下走

### 2.2.4 经典案例：小米的米粉

2010年前后，虽然苹果已经开始推出智能手机，但安卓手机的UI界面非常不好用。

小米当时站出来说：要做不一样的安卓UI系统。

**寻找发烧友**：
- 在各种手机论坛找对苹果、三星手机使用上不满意、还能说出不满意理由和细节的用户
- 能用技术语言表达的
- 100多个用户愿意参加早期MIUI的内测和打造

**为什么感动这100多人**：
- 此前在别的手机论坛吐槽，没什么人搭理
- 现在有一家公司愿意这么尊重自己、重视自己
- 邀请他们加入到手机开发设计、UI系统开发设计
- 最后把他们的名字打到手机开屏界面上——"100个梦想的赞助商"

这100多人毫不犹豫、死心塌地成为忠实的米粉。

**结果**：
- 2013年之前，小米用于推广营销费用投入极低
- 5年左右时间，MIUI保持每周迭代，风雨无阻
- 5年迭代249个版本
- "为发烧而生"口号深入人心
- 粉丝从100多人积攒到50万、100万、大几百万

**核心启示**：搞定发烧友 = 自带传播能力 + 低成本获客

## 2.3 用户洞察画布：系统化理解用户的工具

### 2.3.1 什么是用户洞察画布

如果你是一个刚入行不久的运营，还没有能够把对用户的理解自然内化在脑海里面，建议先用用户洞察画布来慢慢积累和提升用户洞察。

**画布结构**（从左到右三个部分）：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        用户洞察画布                                      │
├─────────────┬─────────────────────────┬─────────────────────────────────┤
│   基础属性   │       重要情绪          │         喜好/痛点/生活方式      │
├─────────────┼─────────────────────────┼─────────────────────────────────┤
│ • 地域      │ • 人生态度理念          │ • 输引音（兴趣爱好）           │
│ • 行业      │ • 重要目标              │ • 关注争议/社会话题            │
│ • 公司      │ • 共同记忆              │ • 产品典型痛点和场景            │
│ • 岗位      │ • 拥护/反对对象         │ • 衣食住行偏好                 │
│ • 年龄性别  │ • 高频情绪场景          │ • 业余休闲活动                 │
│ • 教育背景  │                         │                                 │
│ • 收入资产  │                         │                                 │
│ • 家庭状况  │                         │                                 │
└─────────────┴─────────────────────────┴─────────────────────────────────┘
```

### 2.3.2 案例：有看学社的用户洞察画布

**基础属性**：
- 地域：一二线城市为主
- 行业：互联网行业
- 年龄：24-35岁
- 性别比例：男性40%，女性60%
- 学历：本科及以上70%
- 收入：月收入5-10K占50%，10-20K占30%，20K以上占20%
- 家庭状况：单身55%，非单身45%

**重要情绪**：

人生态度和理念：
- 不想过一眼看到头的生活
- 抛弃国企事业单位工作进入互联网
- 反对内卷、反对内卷现象

共同记忆：
- 深夜加班等上线
- 跟产品研发和设计撕逼
- 被用户反馈或留言感动
- 做成牛逼项目很骄傲
- 高考、大学毕业、恋爱

当前关注的争议和社会话题：
- 要不要去大厂
- 要不要回老家
- 要不要做副业
- 00后、05后在玩什么、关注什么

高频正负向情绪场景：
- 经历裁员和失业
- 经历公司业务剧烈变化
- 被老板PUA
- 间歇性对未来迷茫、焦虑
- 深陷办公室政治

**喜好/痛点/生活方式**：

产品典型痛点和场景：
- 焦虑：担心成长太慢、感觉没人带、没人引领、失去方向
- 面临业绩指标和工作压力

休闲活动：
- 喜欢尝鲜网红品类、潮流品类
- 好奇心强
- 喜欢打游戏、玩剧本杀、去周边游

**当你有了用户洞察画布，策划活动时有无穷灵感**：

**核心价值**：画布里所有的信息，就是典型用户最关注、最容易戳中他们的信息。这些信息都会成为你运营工作中的重要抓手。

## 2.4 如何填充用户洞察画布

### 2.4.1 两步心法

**第一步：近身观察反思**

如果你是刚入行不久的运营，面对新用户了解特别有限：

1. 找到目标用户特别扎堆的群或社区
2. 进到里面就近身观察
3. 重点观察：
   - 他们特别热衷的一些话题、事件
   - 收集过往这个群体身上曾经有过的一些热贴事件
   - 他们语言交流的一些风格
4. 在这个基础上，推测和理解他们背后的一些情绪、心理敏感点有哪些
5. 试着填充用户洞察画布

**第二步：以身试读**

如果感觉还不够，离用户还有点远：

1. 装扮成目标用户
2. 进入目标群体，跟对方去聊天
3. 直到成为目标群体里的小KOC或社群活跃分子，能被很多人信任和认可

**实操示例**：
- 如果目标用户是广场舞大妈，就把自己假装成广场舞大妈
- 进到用户群体里跟对方聊天
- 设定身份人设：设定要扮演的角色
- 加入社群做自我介绍
- 模仿用户的行为、语言风格，参与对方活动
- 描述清楚活动背后大家心里的乐趣
- 在不同场景下他们开心、悲伤时是什么感受
- 不断填充更新用户洞察画布
- 直到能在社群里分享张罗，获得大家认可和关注

### 2.4.2 重要注意事项

**用户洞察画布不是一成不变的，需要不断更新与时俱进**。

每隔一个阶段，用户身上那些重要的情绪点、关注的话题、热衷的休闲生活都会改变。要保持对用户洞察画布不断有更新。

**绝对不要以为画布填完就是正确答案**，日后所有东西都不变它就一定是对的——没有这样的道理。

## 本章核心要点

1. **情绪敏感点 > 基础需求**：每一类用户都有专属的情绪敏感点，理解这些才能成为牛逼的运营
2. **小众人群的价值**：关注小众圈层，理解发烧友，能让你自带流量和传播能力
3. **用户洞察画布**：系统化工具，三个维度——基础属性、重要情绪、喜好痛点生活方式
4. **画布的应用**：画布填完不是终点，而是运营工作的弹药库和抓手来源
5. **填充方法**：近身观察反思 + 以身试读
6. **持续更新**：用户洞察画布需要与时俱进，不断迭代
