### 6.2 提示词设计方法论

#### 什么是好的提示词

提示词（Prompt）是与AI工具对话时输入的指令，它直接决定了AI输出的质量和相关性。

```
┌─────────────────────────────────────────────────────────────┐
│                    提示词设计核心原则                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   原则1：明确具体                                           │
│   ├─ 错误：帮我写点东西                                     │
│   └─ 正确：帮我写一篇1500字的公众号文章，主题是职场沟通     │
│                                                             │
│   原则2：提供背景                                           │
│   ├─ 错误：推荐一款产品                                     │
│   └─ 正确：我是做少儿编程培训的，面向K12家长，帮我推荐     │
│           一款适合的引流产品                                │
│                                                             │
│   原则3：设定角色                                           │
│   ├─ 错误：解释一下这个概念                                 │
│   └─ 正确：你是一位有10年经验的运营专家，解释一下这个概念  │
│                                                             │
│   原则4：指定格式                                           │
│   ├─ 错误：给我一些建议                                     │
│   └─ 正确：给我5条提升用户活跃度的建议，用表格形式呈现      │
│                                                             │
│   原则5：迭代优化                                           │
│   ├─ 第一版：基础提问                                       │
│   ├─ 第二版：根据回答补充细节                               │
│   └─ 第三版：调整要求直到满意                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 使用说明

**提示词设计要点**：
- 使用方括号 `[...]` 表示需要根据实际情况填入的参数
- 可以根据具体业务需求调整提示词的细节
- 建议先使用英文提示词，再根据需要翻译调整

**应用场景**：
- 创业项目规划
- 商业模式设计
- 市场分析研究
- 竞品分析
- 运营策略制定

### 6.4 AI应用的局限性与注意事项

#### AI的局限性

```
┌─────────────────────────────────────────────────────────────┐
│                    AI的局限性                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   1. 知识时效性                                             │
│   ├─ ChatGPT知识截止2021年9月                              │
│   ├─ 不了解最新行业动态                                     │
│   └─ 解决：结合最新信息源使用                               │
│                                                             │

│   ├─ 可能生成看似合理但错误的内容                           │

│   3. 上下文理解                                             │
│   ├─ 复杂上下文可能理解偏差                                │
│   ├─ 长对话可能偏离主题                                     │

│                                                             │
│   4. 创意局限                                               │
│   ├─ 难以产生真正的创新突破                                 │
│   ├─ 内容模式化风险                                         │
│   └─ 解决：AI生成+人工创新                                  │
│                                                             │
│   5. 情感理解                                               │
│   ├─ 难以理解复杂的人类情感                                │
│   ├─ 缺乏真正的同理心                                       │
│   └─ 解决：重要沟通保留人际交流                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
