### 3.2 常见数据分析方法

#### 1. 趋势分析

**定义**：分析数据随时间变化的趋势

**应用场景**：
- 业务是增长还是下降
- 某指标的变化规律
- 季节性波动分析

**示例**：
```
月GMV趋势：
1月：50万
2月：45万 (-10%)
3月：60万 (+33%)
4月：70万 (+17%)
5月：75万 (+7%)
```

#### 2. 对比分析

**定义**：对比不同维度数据发现差异

**对比维度**：
- 同比（今年vs去年）
- 环比（本月vs上月）
- 竞品对比
- 渠道对比

**示例**：
```
各渠道转化率对比：
- 渠道A：5.2%
- 渠道B：3.8%
- 渠道C：2.1%

结论：渠道A转化率最高
```

#### 3. 漏斗分析

**定义**：分析用户转化路径中的流失情况

**漏斗模型**：
```
┌─────────────────────────────────────────────────────────────┐
│                       漏斗分析示例                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   步骤1：访问    10000人  (100%)                           │
│       │                                                      │
│       ▼ 流失 70%                                           │
│   步骤2：注册    3000人   (30%)                            │
│       │                                                      │
│       ▼ 流失 50%                                           │
│   步骤3：活跃    1500人   (15%)                            │
│       │                                                      │
│       ▼ 流失 60%                                           │
│   步骤4：付费    600人    (6%)                             │
│       │                                                      │
│       ▼ 流失 30%                                           │
│   步骤5：复购    420人    (4.2%)                           │
│                                                             │
│   关键洞察：                                                │
│   - 访问→注册流失最大（70%）                               │
│   - 重点优化注册环节                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 4. 留存分析

**定义**：分析用户持续使用产品的情况

**留存指标**：
- 次日留存
- 7日留存
- 30日留存
- 生命周期价值(LTV)

**示例**：
```
用户留存曲线：
- 次日留存：45%
- 7日留存：25%
- 30日留存：10%

结论：用户流失主要集中在前7天
```

#### 5. 分群分析

**定义**：将用户按特征分成不同群体进行分析

**分群维度**：
- 用户来源
- 消费能力
- 使用行为
- 生命周期阶段

**示例**：
```
用户分群：
- 高价值用户：复购3次以上，客单价500+
- 潜力用户：复购1-2次
- 待激活用户：注册但未付费
- 流失风险用户：30天未活跃
```

### 3.3 业务数据分析实战

#### 数据分析流程

```
┌─────────────────────────────────────────────────────────────┐
│                   数据分析6步法                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  第1步：明确问题                                           │
│          - 业务目标是什么                                   │
│          - 需要回答什么问题                                 │
│                                                             │
│  第2步：定义指标                                           │
│          - 选取哪些数据指标                                 │
│          - 如何量化问题                                     │
│                                                             │
│  第3步：收集数据                                           │
│          - 数据从哪里获取                                   │
│          - 数据是否可靠                                     │
│                                                             │
│  第4步：分析数据                                           │
│          - 使用合适的分析方法                               │
│          - 发现数据背后的规律                               │
│                                                             │
│  第5步：得出结论                                           │
│          - 数据说明了什么                                   │
│          - 有什么业务洞察                                   │
│                                                             │
│  第6步：制定行动                                           │
│          - 基于结论制定策略                                 │
│          - 设定实验验证                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 常用数据指标

| 指标类型 | 指标名称 | 定义 |
|----------|----------|------|
| 流量指标 | PV/UV | 页面访问量/独立访客数 |
| 转化指标 | CTR | 点击率 |
| 转化指标 | CVR | 转化率 |
| 营收指标 | ARPU | 每用户平均收入 |
| 营收指标 | LTV | 用户生命周期价值 |
| 活跃指标 | DAU/MAU | 日活/月活 |
| 留存指标 | Retention | 留存率 |

## 案例分析

### 案例：电商转化率下降分析

**背景**：
某电商APP近期转化率从5%下降到3%，需要分析原因。

**分析过程**：

```
1. 明确问题
   - 转化率从5%下降到3%
   - 下降幅度：40%

2. 拆分维度
   - 按渠道拆分
   - 按用户群体拆分
   - 按时间拆分

3. 分析结论
   - 渠道A转化率稳定
   - 渠道B转化率下降明显（5%→1%）
   - 渠道B带来大量低质量流量

4. 制定行动
   - 优化渠道B的投放策略
   - 提高渠道B的投放门槛
   - 预期转化率恢复至4%
```
