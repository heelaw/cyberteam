#### 方法二：漏斗分析

```
┌─────────────────────────────────────────────────────────────────┐
│                       漏斗分析方法                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   定义：分析用户从入口到最终转化各阶段转化情况                  │
│                                                                 │
│   漏斗构建步骤：                                               │
│                                                                 │
│   步骤1：确定转化目标                                          │
│      - 明确最终转化行为（如：购买、注册、付费）               │
│                                                                 │
│   步骤2：拆解转化路径                                          │
│      - 识别用户从入口到转化需要经过的关键步骤                  │
│                                                                 │
│   步骤3：设定漏斗层级                                          │
│      - 一般4-7层为宜，过多难以分析                             │
│      - 每层应该有明确的转化行为                               │
│                                                                 │
│   步骤4：数据埋点                                              │
│      - 确保每层转化点有数据埋点                               │
│      - 统一数据口径和定义                                     │
│                                                                 │
│   步骤5：分析优化                                              │
│      - 识别转化瓶颈                                           │
│      - 提出优化方案                                           │
│                                                                 │
│   漏斗分析案例：电商购买漏斗                                  │
│                                                                 │
│   ┌─────────────────────────────────────────────────────┐       │
│   │  阶段          │  用户数 │  转化率 │  流失率 │      │
│   │  ─────────────────────────────────────────────────  │       │
│   │  商品曝光      │ 100,000 │   100%  │    -   │      │
│   │  商品点击      │  35,000 │   35%   │   65%  │      │
│   │  加入购物车    │  12,000 │   34%   │   66%  │      │
│   │  提交订单      │   8,500 │   71%   │   29%  │      │
│   │  完成支付      │   5,500 │   65%   │   35%  │      │
│   └─────────────────────────────────────────────────────┘       │
│                                                                 │
│   分析结论：                                                    │
│   - 点击到加购转化率34%，低于行业平均40%，需优化商品详情页    │
│   - 加购到提交订单71%，表现良好                                │
│   - 提交到支付65%，支付流程可能存在问题，需排查支付环节      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────────┐
│                       相关分析方法                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │

│                                                                 │

│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐  │

│   │  0.5-0.8      │  中等正相关    │  A增加，B有一定增加  │  │
│   │  0.2-0.5      │  弱正相关      │  A增加，B略微增加   │  │
│   │  -0.2-0.2     │  无相关        │  A与B无明显关系     │  │
│   │  -0.5--0.2    │  弱负相关      │  A增加，B略微减少   │  │
│   │  -0.8--0.5    │  中等负相关    │  A增加，B有一定减少  │  │
│   │  -1.0--0.8    │  强负相关      │  A增加，B显著减少   │  │
│   └─────────────────────────────────────────────────────────┘  │
│                                                                 │
│   应用场景：                                                    │
│                                                                 │
│   1. 指标相关性分析                                            │
│      ┌─────────────────────────────────────────────────────┐   │
│      │  活跃天数与付费金额相关性：r=0.65（中等正相关）      │   │
│      │  → 提升用户活跃天数は提升付费的有效手段            │   │
│      └─────────────────────────────────────────────────────┘   │
│                                                                 │
│   2. 行为与结果相关性                                         │
│      ┌─────────────────────────────────────────────────────┐   │
│      │  首次消费时间与复购相关性：r=0.72（中等正相关）     │   │
│      │  → 引导用户尽早完成首次消费可提升复购              │   │
│      └─────────────────────────────────────────────────────┘   │
│                                                                 │
│   3. 运营动作与指标相关性                                     │
│      ┌─────────────────────────────────────────────────────┐   │
│      │  推送频次与留存相关性：r=-0.35（弱负相关）          │   │
│      │  → 推送过于频繁会导致留存下降，需控制频次          │   │
│      └─────────────────────────────────────────────────────┘   │
│                                                                 │
│   注意事项：                                                    │
│   - 相关不等于因果，需结合业务逻辑判断                          │
│   - 需要足够的样本量才能得出可靠结论                           │
│   - 注意异常值对结果的影响                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
