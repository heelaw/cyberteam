# 第四章：用户管理策略：用户激活和用户留存

> 所属章节：第四章 | 包含小节：8节

## 本章导学

本章聚焦用户激活和留存的核心策略。新用户从获取到贡献价值中间，还存在两个非常重要的环节：激活和留存。通过本章学习，你将掌握：

1. 激活、留存、活跃三者的关系和区别
2. HOOK模型：触发-行动-酬赏-投入的闭环设计
3. 如何为用户创造惊喜时刻
4. 三种留存方法：内容留存、功能留存、服务留存
5. 驱动用户主动分享的6大因素（疯传6原则）

---

## 4.1 导读：用户激活和留存的必要性

### 激活、活跃、留存的定义

**激活**是针对新用户来说的，引导用户去完成某些指定动作，目的是让用户体验到产品的价值，产生使用兴趣。

- **案例**：共享单车APP
  - 下载APP后能找到车 → 顺利骑车 → 体验到便捷性
  - 找不到车/车是坏的 → 无法体验价值 → 流失

- **惊喜时刻**：
  - 第一次用抖音发现倒放特效
  - 第一次用美颜软件一键磨皮
  - 抢票软件抢到票

**留存**是指用户在被激活后，持续使用产品的过程。

- 用户留存意愿递减：新奇感、需求强烈程度逐渐降低
- 用户分类：
  - **活跃用户**：使用频次高（如30天15天使用）
  - **一般留存用户**：使用频次中等（如30天7天使用）
  - **濒临流失用户**：使用频次低（如30天1天使用）

**活跃**是留存中的一种用户状态，活跃度是衡量用户的指标。用户越活跃，付费或分享的可能性越大。

### 留存的三种方法

1. **内容留存**：通过PUSH消息提醒用户，提高打开频次
2. **功能留存**：通过功能留存分析矩阵发现高留存功能
3. **服务留存**：通过用户分层（RFM模型）制定针对性策略

### 留存的目的

- 为产品带来营收贡献
- 为产品带来内容贡献
- **分享转发**（病毒式增长的原点）

## 4.2 打造激活和留存的闭环：HOOK模型

### HOOK模型四步

用户****第一步：触发 - 吸引用户的兴趣
- 裂变活动最终呈现给用户的是一张海报
- 海报里会有裂变活动的主题、分销奖励、大咖推荐、课程大纲等内容
- 这些内容构成了吸引用户听课或参与分销的元素

**第二步：让用户行动** - 考虑用户得到的是否超过付出
- 用户有没有付出的能力
- 例如：课卖19块钱，分销3个人可赚19块钱，用户有能力支付且容易赚到

**第三步：奖励** - 多变的酬赏
- 及时性奖励：分销1个人就获得分成
- 多样性奖励：通过分销数量排名获得额外奖励

**第四步：投入** - 持续的投入
- 用户在产品中付出的越多，用户就越容易留存
- 用户已投入的19块钱和让3个人购买就是投入
- 再邀请3个人可获得额外奖励，触发下一次HOOK循环

### 八角行为分析法

从游戏策划角度总结的心理驱动用户行为的维度：

| 维度 | 说明 |
|------|------|
| 使命感 | 用户感觉自己在做有意义的事 |
| 成就感 | 用户完成任务的满足感 |
| 拥有感 | 用户对物品的占有欲 |
| 社交性 | 与他人互动的需求 |
| 稀缺性 | 物品稀缺带来的紧迫感 |
| 未知性 | 探索未知的好奇心 |
| 失落感 | 害怕失去的心理 |
| 自我表达 | 展现自我的需求 |

### 需求三角模型

李叫兽的"需求三角模型"从以下三方面总结让用户付出行动的可能：

1. **缺乏感** - 需求
2. **目标物** - 奖励，奖励要满足需求
3. **能力** - 消费者6大成本（金钱、时间、精力等）

### 三级酬赏机制

| 类型 | 说明 |
|------|------|
| 自我酬赏 | 完成任务本身的满足感 |
| 猎人酬赏 | 获得实际物品或利益 |
| 社交酬赏 | 获得社交认可和互动 |

## 4.3 激活用户：为用户创造惊喜时刻

### 什么是激活？

激活是指用户体验到产品核心价值的过程，是对新用户来说的。激活率 = 新用户中使用某项产品功能的比率。

### 惊喜时刻的定义

惊喜时刻是指能够让用户感受到产品价值的功能。这个功能也就是我们所说的能给用户带来惊喜时刻的功能。

**FACEBOOK案例**：
- 惊喜时刻：添加10个好友
- 添加好友是用户进行的操作
- 10个是具体的操作行为

### 定义激活指标的要素

1. 定义用户产生了哪项操作
2. 定义用户的具体操作行为

### 案例思考：小米商城如何定义用户的惊喜时刻？

| 产品线 | 惊喜时刻定义 |
|--------|-------------|
| 小米商城（电商） | 购买过一次产品，体验到便捷性和优质配送服务 |
| 小米手机 | 使用手机体验到比其他手机更好（如玩吃鸡非常流畅） |
| 米粉社区 | 社区建议被官方人员认同 |

## 4.4 内容留存：通过PUSH消息提高用户活跃和留存

### PUSH消息的定义

PUSH消息是官方主动发送给用户的一切内容，包括：
- APP推送消息
- 营销广告邮件
- 营销短信
- 网站弹窗
- 公众号推送

**核心目的**：让用户点进来看，看完了在APP里打着滚的接着看

### PUSH效果气泡图分析

| 维度 | 说明 |
|------|------|
| 横轴 | 内容留存率 = 次日主动打开APP的人数 / 当日点击PUSH的人数 |
| 纵轴 | 留存标准差，反映相同维度数值之间的变化幅度 |

### PUSH转化关键点

1. **推送消息的吸引力**
2. **点击进入内容页后，内容是否符合用户的预期**
3. **内容页是否能够有效引导用户查看其它内容**

### 自动化PUSH推送策略

1. **推荐算法**：根据用户喜好推荐内容（如今日头条、腾讯新闻）
2. **手动推荐**：利用数据分析工具筛选用户特征
3. **数据记录**：手动记录PUSH打开率、留存率、跳转率等

## 4.5 功能留存：通过功能留存分析矩阵提高留存率

### 功能留存分析矩阵

通过矩阵分析产品中哪个功能对留存的价值最高：

| 维度 | 说明 |
|------|------|
| 使用用户的人数 | 活跃用户占比 |
| 连续使用功能的用户占比 | 功能留存率 |

### 计算方法

**功能留存率** = 当前周期继续使用该功能的用户数 / 上周期使用过该功能的用户数

- 不包含新用户

**活跃用户占比** = 当前周期使用该功能的用户数 / 当前周期使用产品的所有用户数

- 包含新用户

### 关键注意事项

1. 功能留存率的计算不包含新用户，活跃用户占比包含新用户
2. 两个数据维度的计算周期要相同

### 矩阵象限分析

| 象限 | 特征 | 策略 |
|------|------|------|
| 明星功能 | 高用户量、高留存率 | 重点维护 |
| 潜力功能 | 低用户量、高留存率 | 加大推广 |
| 问题功能 | 高用户量、低留存率 | 优化体验 |
| 淘汰功能 | 低用户量、低留存率 | 减少投入 |

## 4.6 服务留存：通过用户分层制定针对性留存策略

### 用户分层定义

用户分层是将用户按照某种特征分为不同群体，针对不同群体制定不同运营策略。

**知乎案例**：
- 10%回答问题的用户（核心创造者）
- 20%提问问题的用户
- 70%看内容的用户

### 为什么要做用户分层？

将有限的资源分配给最有价值的用户，来实现更好的留存。

### RFM模型

| 维度 | 说明 |
|------|------|
| R（Recency） | 最近一次消费时间 |
| F（Frequency） | 消费频率 |
| M（Monetary） | 消费金额 |

### 用户分层原则

1. 每一种用户分层的维度只有一个
2. 筛选的颗粒度越细，能够制定的策略就越有针对性

### RFM分层策略

| 用户类型 | 特征 | 策略 |
|----------|------|------|
| 高价值用户 | R低、F高、M高 | 重点维护，推荐高端商品 |
| 潜力用户 | R低、F低、M高 | 推荐竞品（品质好单价高） |
| 活跃用户 | R低、F高、M低 | 推荐爆品（销量大价格低） |

## 4.7 驱动留存用户主动分享的6大因素

### 疯传6原则（STEPS）

1. **Social Currency** - 社交货币
2. **Triggers** - 诱因
3. **Emotion** - 情绪
4. **Public** - 公众性
5. **Practical Value** - 使用价值
6. **Stories** - 故事

### 1. 社交货币（Social Currency）

**定义**：个人独有的谈资，加上一群人都感兴趣的共同话题

**人类心理基础**：
- 人们希望自己看起来更加聪慧、富有、时尚
- 人们喜欢共享他们的观点
- 人们进行口碑传播是为了保持良好印象

**如何创造社交货币**：
- 新奇的事物
- 显示特权（VIP、超级会员）
- 游戏化（徽章、积分卡、排行榜）
- 设置门槛限制
- 不要轻易付钱

**案例**：
- 星巴克：中杯/大杯/超大杯点单显示逼格
- 支付宝年度账单：炫目设计可以分享
- 网易云音乐：每周听歌最多的歌曲专辑封面拼图

### 2. 诱因（Triggers）

**定义**：用刺激瞬间激活人们的记忆，让他们想到品牌相关的内容

**常见诱因类型**：
- 时间诱因：午饭/晚饭时间触发外卖软件
- 空间诱因：到了特定地理位置触发推荐
- 声音诱因：定制推送提示音
- 需求诱因：根据用户行为猜测下一步行动

### 3. 情绪（Emotion）

**核心观点**：有感染力的内容经常能引发人们的即时情绪

**四种情绪分类**：

| 类型 | 举例 | 是否促进分享 |
|------|------|-------------|
| 积极高唤醒 | 敬畏、消遣、兴奋、幽默 | 是 |
| 积极低唤醒 | 满足 | 否 |
| 消极高唤醒 | 担忧、生气 | 是 |
| 消极低唤醒 | 悲伤 | 否 |

### 4. 公众性（Public）

**核心观点**：当人们看见别人使用产品时，会考虑是否符合自己的需要

**如何制造公共性**：
- 利用从众心理/羊群效应
- 强化差异：特色配色、大logo
- 行为剩余：产品纪念品、手提袋

### 5. 使用价值（Practical Value）

**核心观点**：证明产品能够节省时间或钱财

**行为经济学应用**：
- 心理参照点：1000元降10元无感，15元降10元很有感
- 人为制造稀缺性：限时/限量购买
- 折扣显示方式：低于100元用比例，超过100元用金额

### 6. 故事（Stories）

**核心观点**：故事是最原始的娱乐形式，人们对其他顾客的故事深信不疑

**关键要点**：
- 建立像特洛伊木马的传奇故事
- 将重要信息注入到情节中

**案例**：
- 快看漫画陈安妮：《对不起，我只过1%的生活》
  - 上线当天增加30万用户
  - 连续三天冲到免费榜第一位
