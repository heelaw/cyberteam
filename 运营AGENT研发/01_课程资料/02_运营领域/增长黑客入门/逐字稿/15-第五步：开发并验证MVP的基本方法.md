# 第五步：开发并验证MVP的基本方法

> 课程类型：视频（09:23）

---

经过第四步，梳理出来MVP的需求之后，就可以开始开发MVP产品了。

开发产品意味着我们需要学编程么？那太可怕了。事实上呢，已经有一些简单的开发工具，像做PPT一样就可以做出来一个网页或者APP了。

开发出MVP之后就要通过一些推广手段，找到用户来使用你的产品。这个阶段并不需要太大规模的用户量，可以找一些熟悉的朋友，通过一些论坛、QQ群推广，获得种子用户就可以了，用户使用的价值是给你提反馈意见。

那么如何验证MVP存在市场需求呢？范冰老师给出了2个方法。

这一节里，范冰老师讲了下面这些内容：
1.第三方开发MVP的工具；
2.验证MVP的关键指标观测；
3.验证MVP的肖恩埃利斯测试

好了，学完这一节，找到产品PMF的5个步骤你就学会了，下面开始学习吧~

---

**可视化开发工具**：非技术人员通过这些工具可以做出网页、小程序、APP

- [上线了](https://www.sxl.cn/) - 通过模板创建网页和小程序
- [unbounce](https://unbounce.com/) - 用来制作推广落地页的工具
- [有赞](https://www.youzan.com/) - 通过模板生成多个行业的电商商城，包括APP、小程序、网页等
- [APICloud](https://www.apicloud.com/) - 可以在线创建标准化的APP
- [变色龙云](https://www.bslyun.com/) - 基于模板在线创建APP和小程序

---

## 本节要点

### MVP开发

- 不需要编程，可使用可视化工具快速搭建
- 获得种子用户用于测试反馈

### 验证MVP方法

1. **关键指标观测**：用户使用数据、转化率、留存率
2. **肖恩埃利斯测试**：调研核心用户是否愿意付费

### MVP开发工具

| 工具 | 用途 |
|------|------|
| 上线了 | 网页、小程序 |
| unbounce | 推广落地页 |
| 有赞 | 电商商城 |
| APICloud | APP开发 |
| 变色龙云 | APP和小程序 |
