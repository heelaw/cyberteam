# 制定增长方案：七步构建获客增长计划

> 课程类型：视频（17:28）

---

增长黑客是被老板寄予厚望的岗位，一方面会有各种资源支持，另一方面也可能会在一个风和日丽的天气里遇见晴空霹雳般的增长任务。

例如，三节课的Boss找到我们增长团队的大佬说："增长黑客这门课的销量不错，每个班期再增加10万元的销售额吧"。

接到这个任务之后你会想到什么？找几个KOL推广一下这门课？通过SEM，信息流等渠道推广这门课？跑到互联网公司上门推销这门课？

不要慌，冷静一下，接到任务之后，你要和老板确定的是预算和时间，接下来我们需要做的是拆解任务指标。

怎么拆解？按照时间、渠道这两个维度拆解，然后你就会发现，其实也不是那么难。

这一节范冰老师讲解制定增长方案的7步法，通过这七步，帮你将一个增长指标拆解为可量化、可执行的增长方案，你将学到如下内容：

1. **定义指标**：将增长目的定义为包含时间和明确数量要求的增长指标，比如3个月内实现10万元销售额的增长；

2. **拆解指标**：将整体指标按照时间和渠道的维度拆解，比如3个月实现10万元增长，每个月实现3.3万元增长，拆分到3个渠道，每个渠道每个月实现1.1万元增长；

3. **评估渠道**：通过曝光量、转化率、获客成本等维度评估渠道的可行性，排列渠道投放的优先级，并预测转化效果是否能够达成增长目标；

4. **制定任务时间线**：确定渠道后，按照时间要求确定每个时间节点应该执行什么动作；

5. **追踪效果**：调整渠道投放策略，根据实际的转化数据调整渠道投放预算。

好了，在制定增长计划时关键的6个字是"可量化、可执行"，下面，让我们开始这一节的学习吧。

---

## 案例：增长实习生小A的指标拆解故事

**背景**：七夕活动，预期目标10万元销售额

**小A的错误**：过分注重创意（开发H5游戏），忽略指标拆解

**正确拆解过程**：

1. **定义目标**：5天内实现833个用户成交，平均下单120元
   - 老用户500人（贡献60%）
   - 新用户333人（贡献40%）

2. **计算所需流量**：
   - 新用户转化率2% → 需要16650名新用户访问
   - 老用户复购率10% → 需要向5000名老用户宣传

3. **最终活动目标**：
   > 5天活动时间内实现833个用户的成交，平均下单金额120元，需要获取新用户访问人数16650人，同时向5000名活跃老用户宣传活动信息。

## 制定增长方案的核心要点

| 步骤 | 关键问题 |
|------|----------|
| 1. 定义指标 | 目标数字和时间？ |
| 2. 拆解指标 | 这些用户从哪来？ |
| 3. 评估渠道 | 哪个渠道能带来目标用户？ |
| 4. 制定时间线 | 每个时间节点做什么？ |
| 5. 追踪调整 | 根据数据优化渠道？ |

**核心原则**：
- 先拆解指标，再想创意
- 创意服务于提高转化率
- 不断问"从哪来"直到"怎么做"
