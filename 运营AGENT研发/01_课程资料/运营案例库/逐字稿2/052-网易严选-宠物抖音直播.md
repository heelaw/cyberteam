## 一、行业背景

### 宠物经济崛起

近年来，随着**消费升级**和**社会老龄化**等因素，人们对于宠物陪伴的精神需求日益凸显，这也使得人们对宠物文化越来越重视。

随着养宠人群的不断扩大，宠物行业相关的商品和服务类型不断丰富，**国内宠物行业规模快速扩张**。

> 根据艾媒咨询发布的《2022-2023年中国宠物产业发展及消费者调研研究报告》：
> - 2022年中国宠物经济产业规模达**4936亿元**，同比增长25.2%
> - 预计2025年市场规模将高达**8114亿元**

## 二、网易严选宠物业务

### 业务定位

**网易严选宠物食品**是网易严选于2018年新推出的品类：
- 优选品质ODM代工
- 直接对接大牌制造商
- 走"**平价精品**"路线

### 竞争优势

虽然网易严选**并非宠物垂类品牌**，但：
- 产品性价比极高
- 激发了年轻消费者享受品质产品、低价淘货的庞大需求
- 网易严选涉猎赛道广泛，在各平台种草的阅读量有优势

## 三、抖音直播数据表现

### 渠道分布

**去年双11在抖音**：

| 渠道 | 销售占比 | 成交金额 |
|------|---------|---------|
| 直播 | 远高于视频 | 视频的**近2倍** |
| 视频 | 较低 | - |

### 核心成交来源

| 主播/渠道 | 占比 | 销售额 |
|----------|------|-------|
| 小杨哥直播带货 | 关联次数占23% | 单场**100w-250w** |
| 网易严选官方直播 | 9% | 最高**10w-25w** |
| 其他达人直播 | 次数多 | 销售额不高 |

**结论**：网易严选猫粮核心依靠**超头部主播+直播间带货**

### 账号布局

网易严选品牌宠物类目相关的直播间有**3个**：
- "网易严选宠物旗舰店"：每日直播
- "网易严选宠物官方旗舰"：每日直播
- 月均直播**34场**

### 用户画像问题

> 对比各个渠道的用户可以发现，视频种草的人群跟直播观看人群还是有差异，**种割结构不是很一致**

**建议**：应当加强对**18-23岁人群**的种草，可以参考阿飞和巴弟的种草视频

## 四、场景拆解

### 1. 账号主页设置

**亮点**：
- 预告了每天直播的时间
- 提供给用户的价值：咨询养宠问题
- 即使不购买也可以去体验咨询
- 简介上提到网易严选口碑优势，增加用户信任度

### 2. 直播间场景设置

**问题**：
- 一眼望过去**色调比较杂乱**
- **重点不够突出**
- 用户比价时关心单斤价格，但上方信息整体不够突出
- 斤数的信息容易被忽略

### 3. 团队配合及直播间氛围

**团队配置**：
- 一个**主播** + 一个**运营**
- 运营负责强调卖点，配合主播，控场

**氛围特点**：轻松且火热

**原因**：
1. 主攻**1个链接**，为3个爆款合集，弹窗上面有"热卖x几百"的标志，给人火热的感觉
2. 主播很擅长**互动**，从用户的需求点出发，结合自己养猫、使用猫粮的经历做分享
3. 氛围整体轻松，给人**信任感**

## 五、货品拆解

### 1. 商品（购物车）拆解

| 项目 | 具体内容 |
|------|---------|
| 货品数量 | 60+ |
| 排列顺序 | 主粮在前，零食在后；按销量排序，爆款前置 |
| 价格带 | 偏中高，与中产用户画像一致 |

**核心话术**："同样价格看品质，同样品质看价格"——强调性价比

### 2. 直播间过品节奏

- 大概**10分钟一个品**
- 比较多地和评论区互动
- 采取**单品循环策略**，只讲第一个链接，会搭配其他款讲

### 3. 单品循环策略详解

**为什么只主讲1个链接？**

主推品三个放在一个链接，包括鸡肉、兔肉粮和幼猫粮。这是**符合抖音推荐机制**的：
- 把三款爆款放在一起
- 一个链接的成交曝光高
- 得到流量推荐就更高
- 特别是对**商品卡曝光**有帮助

**三款粮的差异化**：

| 产品 | 卖点 | 目标用户 |
|------|------|---------|
| 幼猫粮 | 加了羊奶粉 | 2月以下的猫 |
| 鸡肉双拼 | 长胖发腮 | 需要长肉的猫，搭配主食罐 |
| 兔肉粮 | 美毛增肌不过度肥胖 | 成年猫，尤其绝育后 |

**如何讲出差异化？**

> 主播会结合用户的问题进行分析，再把卖点里面契合用户的部分给用户强调一遍，更加具有互动性。

- 不仅是年龄段的问题
- 什么时候绝育
- 平时喂猫习惯等等都会聊到
- 如果自己有相似经历也会结合讲

## 六、策略拆解

### 1. 直播间引流

| 引流方式 | 具体做法 |
|---------|---------|
| 直播预告 | 讲解接下来几天安排、直播时间、每月侧重点（7月尝鲜，8月囤货） |
| 社群预告 | 店铺社群会有近日活动预告 |
| 短视频 | 自营+达人，内容更新频率高 |

### 2. 直播间讲品

**讲品流程**：

```
在线cue人 → 引导关注直播间领优惠券 → 年龄段定位推荐粮
→ 配方讲解 → 卖点陈述 → 品牌背书 → 不吃可退包运费促单
```

**三大亮点**：

1. **主播分享自己养猫、使用自家猫粮的情况**，增强说服力

2. **结合场景和用户需求讲解产品**
   - 讲猫条不会只讲美味、补水
   - 还会讲怎么用猫条训练小猫黏人
   - 也会科普猫猫养育的知识，很加分

3. **互动性强**，从用户需求出发

### 3. 转化/逼单策略

**策略1：优惠券、运费险**

```
先领券，入会送15元优惠券宝宝，领券下单更便宜。

拍正装我们送120g试吃哈，猫猫不喜欢正装直接全部退，运费险我们来承担哈~

吃不吃只有带回去猫猫试过才知道哈~
```

**策略2：效果证明**

```
这是我家猫猫，给大家看看我家猫猫，吃的就是这款猫粮，两个月的变化，明显长肉肉哈~

我家猫粮的销量在各个平台都很高的哈，受到大家认可的。但是猫猫的口味都是不一样的，要把试吃先给猫猫感受过才能确定喜不喜欢哈~
```

**策略3：主播下播制造紧迫感**

```
主播还有10min下播哈，在场的宝宝们先领券后下单，抓紧时间下单哈~
```

## 七、可借鉴经验

### 1. 直播间的优化方向

**值得学习的地方**：

- 账号主页明确价值：提供养宠咨询服务
- 单品循环策略：符合抖音推荐机制
- 互动式讲品：结合用户问题分析
- 私域承接：活动预告、邀约进群

**需要改进的地方**：

> 直播间背景色调杂乱，重点不突出

**建议**：精简呈现信息，什么品、领券、到手几斤、多少钱

### 2. 宠物经济的启示

**市场机会**：
- 2022年规模4936亿，预计2025年达8114亿
- 增长空间大

**运营要点**：
- 专业知识很重要（宠物医生、养宠经验）
- 信任建立是关键（用自己猫演示、效果对比）
- 用户互动是核心竞争力

### 3. 抖音直播的方法论

**货品策略**：
- 爆款前置
- 单品循环
- 价格带定位

**内容策略**：
- 结合场景讲产品
- 科普知识加分
- 真实使用分享

**转化策略**：
- 优惠券引导
- 运费险降低顾虑
- 紧迫感逼单
