# 案例045：瑞幸咖啡9.9元策略背后的营销思路

> 日期：2023.06.29
> 类型：低价策略、品牌营销

---

## 案例背景

2023年，瑞幸咖啡推出大规模9.9元优惠活动，引发行业广泛关注和讨论。

```

├─────────────────────────────────────────────────────────────┤

│  - 抢占市场份额：通过低价获取更多用户                       │
│  - 对标星巴克：打破高端咖啡认知                             │
│  - 规模化扩张：快速增加门店数量和覆盖                       │
│                                                             │
│  【执行策略】                                               │
│  - 全覆盖：几乎所有基础款咖啡9.9元                          │
```

## 核心要点分析

### 1. 定价策略

```
┌─────────────────────────────────────────────────────────────┐
│                     瑞幸定价策略                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  常规期：27-35元/杯 (对标星巴克)                             │
│       ↓                                                     │
│  促销期：9.9元/杯 (基础款全参与)                             │
│       ↓                                                     │
│  恢复期：14.9-19.9元/杯 (逐步回升)                          │
│                                                             │
│  策略要点：                                                 │
│  - 用9.9元打破价格心理防线                                  │
│  - 用9.9元建立瑞幸=平价咖啡认知                             │
│  - 用9.9元培养用户消费习惯                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. 用户路径设计

```
曝光 → 领券/扫码 → 9.9元首单 → 正常价格复购 → 成为常客
  ↓      ↓           ↓              ↓              ↓
电梯/社  APP/小程序   第一次体验    价格适应      习惯养成
交/门店  优惠券       建立信任       品质认可       价值认同
```

### 3. 成本结构分析

| 成本项 | 常规成本 | 9.9元时成本 | 策略 |
|--------|----------|------------|------|
| 咖啡豆 | ~3元 | ~3元 | 批量采购降低 |
| 包材 | ~2元 | ~2元 | 规模效应 |
| 租金 | ~5元 | ~5元 | 快速摊薄 |
| 人工 | ~4元 | ~4元 | 效率提升 |
| **总成本** | **~14元** | **~14元** | **单杯亏损5元** |

**但考虑**：
- 新客LTV（生命周期价值）> 亏损
- 复购带来的规模效应
- 市场份额的长期价值

## 可借鉴经验

### 1. 低价策略的使用场景

```
✅ 适合场景：
- 市场进入期：快速获取用户
- 对标竞争期：打破对手优势
- 规模扩张期：快速铺设网络
- 品牌转型期：重塑品牌认知

❌ 不适合场景：
- 利润导向期：影响盈利
- 高端定位期：损害品牌
- 资金紧张期：难以持续
```

### 2. 低价策略的执行要点

1. **精选SKU**：不是所有产品都参与，选择高毛利、高周转产品
2. **限时优惠**：营造紧迫感，避免形成永久低价认知
3. **数据追踪**：监控ROI、LTV、复购率等关键指标
4. **退出机制**：设计好如何从低价回归正常价格

### 3. 瑞幸的差异化策略

- **产品创新**：生椰拿铁、椰云拿铁等爆款
- **数字化运营**：APP下单、精准推荐
- **私域运营**：企业微信社群运营
- **跨界联名**：与其他品牌合作扩大影响力

## 案例启示

### 对品牌方的启示

1. **价格是武器，不是目的**：低价是手段，市场份额和用户价值才是目标
2. **算好账再行动**：LTV > CAC才能持续
3. **要有退出策略**：如何从低价回归正常价格同样重要

### 对创业者的启示

1. **对标策略**：找到行业领导者，用差异化策略切入
2. **规模效应**：快速扩张才能降低单客成本
3. **品牌升级**：低价起家，但必须有品牌升级路径

## 延伸思考

1. **瑞幸能维持低价多久？**
   - 成本持续优化空间有限
   - 竞争对手跟进削弱优势
   - 品牌升级需要价格支撑

2. **星巴克如何应对？**
   - 保持高端定位不动摇
   - 通过会员体系和体验差异化
   - 推出自己的优惠活动

3. **其他行业可以借鉴吗？**
   - 新茶饮：蜜雪冰城已验证
   - 健身房：超级猩猩模式
   - 教育培训：大量低价引流课
