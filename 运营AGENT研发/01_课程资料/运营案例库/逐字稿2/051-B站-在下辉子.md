# 案例051：B站「在下辉子」开同学盲盒涨粉百万案例分析

> 日期：2023.9.21
> 类型：短视频内容、IP打造、情怀营销

---

## 一、「在下辉子」如何火起来的？

### 创意形式

拍老同学的故事也能涨粉百万？

**具体做法**：
- 抖音账号"在下辉子"将高中毕业照放大贴在墙上
- 每期随机点到一位高中同学
- 随后赶到对方所在的城市
- 探访对方现在的生活

### 数据表现

| 指标 | 数据 |
|------|------|
| 老同学系列视频 | 9条 |
| 涨粉 | 近340万 |
| 百万赞视频 | 3条 |
| 近一个月涨粉 | 超200万 |
| 目前粉丝 | 394万+ |

### 第一个爆款

**7月15日**发布的老同学系列第一期视频：
- 主人公是高考与人大失之交臂、本硕毕业于对外经贸大学的女同学
- 她也是辉子的初恋对象

### 为什么会火？

**核心吸引力**：网友很好奇尖子生毕业10年后的生活

> 辉子的高中同学毕业于辽宁省最好的中学之一本溪高中，全员考进"985"、"211"，还囊括省状元，自嘲"北大废物"的李雪琴也在这个班。

**网友想知道**：一群尖子生毕业10年后的生活是什么样？读书究竟有没有改变他们的命运？

## 二、同学故事中的典型人物

### 1. 魏新元

- 高考全省前十
- 毕业于香港大学
- 在私募基金公司工作
- 管过最大一笔资金近一个亿
- 现在自己开公司创业

### 2. 朱凌萱（初恋）

- 高考成绩本可上人大
- 后来去了对外经贸大学
- 本硕毕业后进入互联网公司
- 最近刚跳槽

### 3. 小睿（北大博士）

- 北大法学院本科直博
- 在象牙塔10年
- 看似顺风顺水，其实充满迷茫
- 不知道想做什么工作，想过什么生活
- 只能不停读书，逃避现实
- 一见到辉子便笑着说：我申请延毕了

**7月22日的视频**记录了小睿的现状：
- 在北大从本科、硕士读到博士
- 却在临近毕业时思考自己到底适不适合做老师
- 在30岁时迎来迟到的青春期
- 贴纹身、做美甲，去尝试各种新鲜事物
- 似乎在挣脱中规中矩"好学生"的滤镜

**这条获赞200万的视频**让"在下辉子"迎来了第一个涨粉高峰：
- 视频发布后，两天涨粉90万
- 粉丝量从十万突破百万大关

### 4. 阿舒（最治愈的同学）

> "在下辉子"的第二次涨粉转折点并不是因为明星同学，而是一位低调失联多年的女同学阿舒。

- 毕业于东财
- 成为大连事业编老师
- 已婚

**与学霸的区别**：
- 比起学霸考进北大、从网红成为脱口秀演员和综艺明星的"幸存者偏差"
- 阿舒似乎更贴近普通人生活

**评论区反馈**：
- "这期比李雪琴那期点赞还高，说明当下人们更与这种平淡和真实共情"
- "阿舒是看起来最治愈、过得最幸福的同学"

**数据**：
- 阿舒视频获赞量超出李雪琴216万，达355万
- 视频发布十天后，涨粉近150万

## 三、快速吸粉500万，他做对了什么？

### 1. 抓住了集体共鸣

**背景**：
- 辉子来自辽宁省本溪市高级中学
- 毕业于2013年
- 这一年辽宁省的文理科状元都是他的校友
- 在这所一本率常年保持在90%以上的重点高中
- 辉子的同学也多是学霸

> 作为中国学子的共同的回忆，**高考**往往能引起网友的共鸣，跟随辉子的镜头，网友窥见了当年的学霸现在到底过得怎么样。

### 2. 功底极强的叙事节奏和风格

**视频形式**：
- 大多时长5分钟
- 用纪实的对话画面上加入后期旁白
- 一般会和老同学边吃边聊
- 也会前往对方的工作或居住地点
- 近距离感受对方一天的生活

**文案风格**：
- 虽然辉子说话带着一股接地气的东北味
- 但他的文案里透露着一颗文艺的心

**前期积累**：
- 在拍摄老同学系列之前
- 他还曾拍摄过六条"散文式"内容
- 讲述鲁迅故居、国家博物馆和东北矿区等故事
- 不难看出辉子有着多年文科生积攒下的文案功底
- 形成了**朴实又文艺**的叙事风格

### 3. 拍摄对象的故事性和表现力

**拍摄对象的特点**：
- 无论是明星同学，还是：
  - 从全职妈妈重返职场从实习律师做起的北大毕业生
  - 被大厂裁员的制片人
  - 与自己和解的小镇女孩
- 除了"高考考得最好的一批人"的学霸标签
- 辉子的拍摄对象又自带不同的话题性
- 能从职场、生活和婚恋等不同层面引发网友的感慨

**表达能力**：
- 或许是因为出自文科尖子班
- 出现在视频中的对象无一例外地展现出流畅的表达能力
- 和对生活独到的见地

**运营难点**：
- 能否找到合适的对象
- 对方是否愿意配合
- 二人能否真诚地交流碰撞出火花

**网友评价**：
> 这个系列一般人做不了，因为大部分同学都已经"失联"了。能够主动联系十年未说过话的同学、并邀请对方出来见面，本身就需要很大的勇气。

### 4. 充分的时间和出行费投入

- 辉子每一期视频都会前往对方所在的城市
- 从北京、成都到大连宁波
- 横跨大半个中国去见分布在天南海北的同学们
- 这在起号初期是一笔不小的开支

**商业逻辑**：
- 只有当账号体量达到百万级以上、数据表现稳定后
- 才容易得到广告主的青睐
- 从而打破"入不敷出"实现变现

**当前报价**：
- 星图数据显示，目前"在下辉子"的广告报价为**10万元一条**
- 该账号已经接到了面霜和饮料品牌的广告
- 在辉子和同学的对话过程中进行植入

## 四、案例对比分析

### 前辈案例：「李程远不远」

**2019年底**拍摄过《三十而立》系列纪录片：
- 讲述了身边第一批年满30岁的90后的故事
- 30集短视频在抖音播放量超过1.4亿
- 助力"李程远不远"踏入百万博主的行列

**走红后**：
- 发力知识付费赛道
- 上线了价格1000元至3000元的自媒体创作课程

**现状**：
- 近期数据大不如前
- 去年3月宣布《三十而立》系列回归
- 视频反响平平，获赞大多在几万上下
- 与早期动辄获赞百万相差甚远

**原因分析**：
- 观众看多了"情怀杀"容易感到麻木
- 往往能猜到年少成名或是被生活磨平棱角的故事

### 两类账号的困境

在商业广告和知识付费之外，鲜少有情怀故事类账号进军直播带货：

**原因1**：
- 网友对视频的喜爱往往建立在每期视频的主人公上
- 创作者扮演着采访挖掘故事的角色
- 难以表现个人IP的性格特点
- 从而建立起与粉丝之间的信任

**原因2**：
- 这类账号的用户画像男粉居多
- 比起女粉为主的账号
- 带货日用快消和美妆护肤等直播间热门品类似乎不是最优解

**数据**：
- "在下辉子"和"李程远不远"男粉占比均超过60%
- 因此未涉足直播带货

## 五、对普通人做自媒体的借鉴

### 1. 选题角度

> 前同事、小镇同学、高考没考好的人怎么样了

**可借鉴的选题方向**：
- 同学系列
- 前同事系列
- 高考失利的同学
- 小镇青年
- 失联多年的人

### 2. 拍摄脚本

> 纪实的对话画面上加入后期旁白体现真实感

**形式要点**：
- 纪实对话
- 后期旁白
- 真实感

### 3. 挖掘表达欲

> 能从职场、生活和婚恋等不同层面展现出流畅的表达

**核心能力**：
- 找到有故事的人
- 挖掘对方表达欲
- 真诚交流碰撞火花

## 六、案例启示

### 对内容创作的启示

**1. 集体共鸣是流量密码**
- 高考是中国人共同的记忆
- 同学情谊是普遍的情感
- 抓住集体共鸣就能引发传播

**2. 真实比完美更重要**
- 阿舒比李雪琴更治愈
- 平淡和真实更能引发共情
- 普通人故事有时比学霸故事更有共鸣

**3. 叙事风格决定调性**
- 朴实又文艺的文案风格
- 纪实对话+后期旁白
- 形成独特的叙事节奏

### 对账号运营的启示

**1. 投入是必要的**
- 前期的资金投入（出差拍摄）
- 时间投入（积累内容）
- 只有坚持才能看到结果

**2. 变现需要等待**
- 百万级粉丝才能获得广告主青睐
- 在此之前需要持续投入

**3. 商业化有局限**
- 情怀类账号变现方式有限
- 广告+知识付费是主要路径
- 直播带货困难

### 对个人IP的启示

**1. 找到差异化定位**
- "同学盲盒"是独特的形式
- 既有话题性又有持续性

**2. 建立情感连接**
- 通过探访老同学建立情感深度
- 让观众产生代入感

**3. 内容需要有灵魂**
- 只是形式创新不够
- 需要有真实情感
- 需要有人生思考
