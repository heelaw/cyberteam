# 3.3 对标账号笔记那么多，重点看哪些？

大家好 我是Wusu 今天我们继续来学习关于找对标的第三节 对标账号笔记那么多 重点看哪些

在前面呢 我们主要是以账号维度来谈及对标账号的 但账号是由一篇篇笔记构成的 而且我们显然没有精力把每个账号的每篇笔记都仔仔细细的看完 所以这一节我们来聊一下重点要看哪些笔记

第一部分是早期笔记 早期笔记具有非常高的参考价值 因为它是这个账号最开始发的这些笔记 可以帮助我们去了解这个账号在一开始的时候做了哪些设计 我们可以通过观察早期笔记去看账号明确和数据增长的一些开始的节点 对于这些笔记的观察有助于我们冷启动阶段账号的运营

而且因为我们要看的笔记数量非常大 所以我拿到一个账号 我就会先拉到这个账号的最底部开始 从早期开始往现在翻

第二部分是高赞笔记 对于万赞以上的笔记肯定算是高赞了 那如果说是比较小规模的账号 或者说这是它比较早期的一个笔记 那么千赞级别的笔记也是值得留意一下的

第三类笔记是自媒体复盘笔记 有很多博主他都会发自媒体的复盘 比如说在多少天内有长多少粉 我做对了什么 或者说当某某某博主

这半年我赚了多少钱 这样的笔记可以为我们提供非常直观的关于这个账号的增长的情况和变现的情况 而且这些数据也方便我们横向比较不同账号的增长情况 可以去了解他们运营的策略和成功的一些因素 作为一个补充 就是说其他的笔记并不是不重要 但是在能启动阶段 我们可以重点关注以上这三类的笔记

如果说过了能启动阶段 我们的一个关注点其实会产生变化 比如说关于怎么做合作的笔记 以及关于引流的笔记等等 建议根据自己的运营阶段和需求 选择合适的笔记进行分析 接下来我们来看一些案例
