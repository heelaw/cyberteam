# 3.3 展示广告

我们接下来看一下另外一个在Google整个广告投放中投的比较多的这个展示广告。展示广告就是这个Google Display广告。它具体包括什么呢？

首先它的展示广告位置就是会展示到包括说像Google搜索，以及说Google联盟这个GDN的第三方网站上，以及包括是YouTube视频，以及包括Gmail邮箱——就是四个产品都会有展示广告这种类型。

主要的这个广告形式，它既可以是文字，也可以是图片，也可以是视频，也可以是动图。常见的这个营销目标，我们一般会选择要不就是以这个商品销售为导向，要不就是B2B企业以获取销售线索/留资为导向，要不就是提升品牌认知度Awareness，以及说再营销Retargeting。他是也可以去定向选择媒体，比如说我只投这个YouTube视频哪些版位也可以，或者说只投这个Gmail邮箱也可以——但是你就是选的越精准成本会越高。

我们来看这里面最常用的这个GDN广告。GDN广告它的全称叫Google Display Network，我前面也多次强调过，它就是一个谷歌合作的超过两百多万个它的第三方网站、视频、应用程序等等。大家来看这个图片会比较直观——就是谷歌的那个互联网第一代互联网的这种Banner广告对吧？这种大的这种通栏、横版、竖版、方形等等。他就是有两百多万个网站。

一般就是我们在投电商的话，电商会投得多一些，然后那种要实效比如说App这个互联网应用也会投，就是效果导向会比较明显。

GDN的一个广告特点就是说效果转化其实还可以，大家不要觉得这种是很古老的、这种互联网广告效果就不好，还不是——他对电商这个品类还真的就你的人群能够跑的比较精准化，转化率其实还是不错的。

另外一个就是Gmail广告。Gmail广告其实邮件营销在我们做出海中是一个非常重要的一个营销策略。为什么呢？因为我们以美国市场为例，美国用户他平均查看邮箱要花费时间高达4.1个小时。这个时长是很可怕的，每天工作时间也就8个小时，有一半的时间是在查看邮箱上。所以他们工作有一半的时间都要依赖于这个邮件去处理。

所以就是Gmail广告，他能利用什么方式去帮我们触达这个目标群体呢？

首先就是通过这种收件箱广告。这是最常见的，也是比较高效的这种广告形式，但它会标记一个Ad，然后你如果点击进去之后，就是你做的这个详情页。它是按照这个CPC点击付费的，所以你这个详情页信息就非常关键了。像示意图的这个，他可能是做了一个宠物类别的相关的这样的一个宠物食品广告，所以他就是会有一个单图图文的这样的一个广告形式。

那我们来看就是Gmail广告有提供四大广告模板：第一个就像你刚刚展示的那种图片模板，就是单图片的一个模板；另外一个就是这个单一推广模板，就是这个Single Promotion，它会有图片加文字，然后加一个点击按钮；还有一个就是叫这个多产品模板，就是你可以像示意图左下角这里可以同时展示6个、9个等等这种产品模板就比较适合电商了；还有一个叫目录模板，目录模板就是它有一个主打产品，后面再加上三个子产品。

我们来看一下目录产品，它就是会类似这样的一个展示方式——就有一个主图再加上三个创意的不同角度不同视角的这样一个图片。他可以在这个PC也可以在App上去展示。

Gmail广告就是看你的目标人群，就是你认为说你的目标人群如果说在Gmail上会有一定的渗透的话，那你就可以去把它添加上去。

我们来看几个行业案例。

比如说像这个零售行业，沃尔玛他就利用这个Dream Gmail广告，他怎么投的呢？先来学习他的标题跟文案。首先他的标题很明显，就是这个主要强调什么Saving，对吧？帮你省钱。这个文案直接就写什么——你进来就可以领取到10美金的这么一张优惠券，凸显他的这个价格优惠。然后一进来之后，这个详情就一张图，简单直观、主题明确：10美金的优惠券，而且会告诉你这个截止日期，制造这种紧迫感，对吧？然后你有单独的这么一个优惠码。这个就是零售行业借助于Gmail这个广告去触达他们要的这个C端人群的一个投放的案例。

我们来看这个教育他的一个广告文案，他的文案就是你可以申请这样的一个免费的一个学习名额——就是免费，免费是个由头，其实他可能就是一个试听课类似的。但是他的素材我认为是做的比较特色，它在突出他的学习场景——你可以跟其他同学一起来听这个教授的这么一个学习课程。然后通过免费主要是想吸引你点击进来。然后下一步就进入这个号召这个行动性的这种引导按钮了——就是点击你来这个申请。

另外还有一个广告案例，就是这个生鲜品类怎么投这个Gmail广告呢？他同理还是来看他的这个标题：可以获得30美金的减免。它的图就是非常简单直观的来突出——把这个食材配送到家，然后就是帮你搭配完善。其实海外他们做这种会员制的这种产品类目还挺多的，就不仅是在生鲜这个类目，就比如说服装、鞋服，都还是会有很多会员产品。我认为这个本质上其实海外用户其实对一些品牌忠诚度还是比较高的。中国你想去推这种会员制的这种产品销售，我认为会比较难。

像这个品牌它主推的就是说你购买它的会员之后，他每个月比如说每一周可能给你配送几餐到你家这个食材，然后你自己就去自行烹饪，帮你搭配好这个食材，你就降低了你的烹饪难度，然后帮你组建好这样的一个健康食谱类似的。所以它本质上跳转进去这个页面之后，是吸引你去注册为成为他的一个会员。这个就是生鲜行业他们在做会员制电商的这么一个案例。

还有一个案例就是B2B行业。B2B行业其实投放量还投得比较多。像这个案例，他是一个SEO的供应商，他投的这个素材你看到他在凸显什么——凸显出他是谷歌的合作伙伴，他的号召性引导按钮的点是一个电话，你可以电话去咨询他。因为这种To B的业务比较多，可能都需要这种销售一对一跟你做这个产品的详情的服务介绍跟单。所以他就跳转到这个Call的这么一个引导。这个是关于B2B在Gmail上广告它的一个呈现方式跟它的流量承载的一个方法。

好，我们再来看另外一个展示广告，就是YouTube上面的展示广告又是什么呢？

来看YouTube展示广告的主要版位有这么包括几个：

第一个就是这个Masthead这个标头广告，它是打大曝光的。

另外一个叫Bumper——叫导视广告，就是按CPM计费的。它的特点就是不可跳过，你必须强制你观看，然后最多就6秒，所以你要在6秒里面把这个广告内容去陈述清楚。

还有一个就是这个TrueView——就是按CPV计费的这么一个视频贴片广告，然后你观看30秒之后才需要计费。

还有一个叫Overlay——叫展示型广告，它也是按照这个CPV计费，你点击之后才需要收费。

还有一个叫Video Banner——就是这个影片内嵌的一个广告，这个是按照CPC点击才收费。

还有一个叫View Action——就是行动号召广告，它主要是进用户跳转到这个网站，或者说进用户填写表单、拨打电话、App下载等等。

这几个广告档位是对应不同的一个营销目标。

我具体来看：

第一个这个Masthead标头广告。大家来看这个图，它在手机端跟PC端就是这种非常霸道的，就是一打开首页，比如说你投这个国家——比如说我买这个国家按CPD我按天买下这个国家所有的用户全量，所有用户今天只要一登录就会看到你这个广告，就非常霸道，对吧？然后一般的这个广告它是适合那种大品牌投那种，比如说投跳转到这个品牌TV，比如说我们以前有一款新品上市，就基本上在大多数国家地区都会买下这个全天包段的这么一个广告位。他是以品牌导向为主的。后来他在有一些国家地区会推出说你可以按CPM去买，你按照CPM千次的曝光去买这个位置。素材主要就是以图片加视频为主，那也可以投跳转到直播，也可以同时投多条视频。

这个位置很明显就是非常引人注目，他的特点就是贵——比后面要介绍的其他常用的版位是比较贵。这种位置电商品类销售导向的品类一般少有，就是以品牌导向为主的。

另外这个叫Bumper Ads。Bumper有时候我们会跟前面那个大曝光的Masthead广告版位去做组合。这个Bumper它以千次CPM为计费的这种计费方式，所以它的特点就是说你的素材就是不能超过6秒，最多是6秒。它的特点就是不可跳过，你买了多少次这个曝光就有多少用户会看到，所以还是比较实在的作为一个广告，它会在视频钱中会出现。同时它除了出现在YouTube之外，还会出现在Google合作的其他第三方的多媒体广告平台上。这个是关于这个Bumper Ads，就是主要还是以视频播放为主的这么一个广告位。

还有一个叫TrueView Video，他的计费方式就按View——就是有多少用户观看了，而且是观看了至少要30秒之后才会计费。用户是可以选择跳过的，所以这个对你内容制作的这个素材要求是挺高的。他是在你看的，比如说你在看哪个博主的这个视频的一个前中会去插入这个广告，做一个贴片广告。那广告他会先播放5秒之后，用户可以选择略过广告，或者说是继续观看完。

还有一个广告叫TrueView Display，它也是按照CPV计价的。它的一个出现位置就是在这个搜索结果以及在视频的观看页去出现。用户只有点击后才可以看——那有点类似像这个推荐系统。它会根据你这个视频广告本身的一些关键词等等，去匹配给这个目标用户群体。他的展示形式就是你这个视频的缩略图跟文案。因为用户要点击后才可观看，所以你做的这个缩略图就非常关键，以及你的标题文案。

另外一个广告位是叫Video Banner，它主要是这个影片内嵌的这个广告，就在影片中会展示这么一个Banner条，然后它的计费方式是一个CPC，用户就是在看影片的时候，会看到这个广告，而且是需要点击了之后才会给我们计费的这么一个广告位。

还有一个广告就是这个View For Action了。这个主要比较适合比如说像App产品，比如说像B2B企业，比如说我们App下载时候——它的一个特点就是说你不需要离开YouTube，直接打开这个Mobile App就可以去完成你的应用的下载安装了。而且你也可以跳转到你自己的网站上，或者说让客户去填写这个表单，或者说引导客户拨打你的这个联系方式等等。这个主要就是以这个效果为导向的这么一个行动号召广告。

可能一下子介绍这么多的这种广告版位，同学可能没法一下子记住，认为这个都是非常正常的情况——尤其是Google它的广告生态又涉及到4款产品，然后四款产品它有不同的广告目标和不同的广告方式，以及不同的广告版位，所以一下子没法记住非常正常。这个不用担心，主要是我们对它的整个不同的广告形式有一定的了解，后面在实操中去应用起来，去加深你的记忆，这个会比较关键。
