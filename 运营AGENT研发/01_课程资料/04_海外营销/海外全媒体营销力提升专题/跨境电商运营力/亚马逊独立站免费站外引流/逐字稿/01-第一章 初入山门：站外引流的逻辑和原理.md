# 第一章 初入山门：站外引流的逻辑和原理

OK 那让我们开始我们第一章的课程内容 站外引流的逻辑和原理

说到站外引流的话 其实我们一定是需要有一个网站的 那这个网站不管是用自建站也好 或者用第三方的工具来搭建也好 一定是要需要有一个网站 那为什么要做网站呢

其实网站有这样几块优势 第一个就是风险会比较低 它不会像微信小程序或者是APP一样被应用市场或者被微信给封禁掉 那网站的所有的规则都是你自己去制定的

不会说像微信公众号每天只能发推送一次文章 那网站的话你每天更新几十上万篇文章都是可以的 那第二个是网站它是具有长期的复利效应的

就是你的内容发在微信里面或者发在小红书B站 这些地方的话你的内容是有一定的声明周期 尤其是微信和小红书头条这些算法推荐的平台 他们的内容声明周期都会相对较短 可能就三天五天一个星期

那你的内容发布之后可能一个星期之后就没有人看了 但是网站不一样 你把你的内容发布到你的网站之后 它会具有非常长期的一个声明周期 一个价值 能够在很长的一个周期内为你去带来流量

并且随着内容的积累流量越来越多 内容的质量或者是内容带来的流量也会越来越多 OK 所以网站的话它是具有一个积累效应 具有一个复利效应的

那第三个点是网站会足够灵活 相比APP小程序 不管是在PC端还是在手机端 在安卓IOS或是web端都能够访问网站 并且内容的形式更加多样

你可以制作音频视频文章图片都可以发布到你的网站上 非常方便 那第四个是变现很容易 一旦你的网站有了流量 我们可以通过广告可以接入谷歌AdSense来变现

也可以说我们来做一个卖货的独立站都是可以的 第五个点的话是成本比较低 现在的话网站的生态是非常成熟的 比APP我觉得会更成熟一些 那么有很多成熟的这种框架或者是脚手架可以去使用

就非常简单就能够搭建一个网站 而且成本会非常低 你只需要付出服务器和域名的成本 而且有很多第三方的平台 他们提供免费的这个服务器和域名

其实都可以做到零成本搭建一个网站 那最后一点呢就是方便为出海产品引流 不管我们做亚马逊也好 或者是做独立站也好 其实它的主战场还是在网站端

因为国外外国人他们的购物习惯很多都是在网站里面去进行一个购物 所以如果我们有一个自己的网站 并且流量还不错的话 我们可以很方便的为我们的亚马逊或者独立站去进行一个引流 OK 这是我们为什么要去做网站以及网站的一些优势

那么网站它的主要流量来源有哪些呢 其实主要分为三块 网站的流量第一块就是直接访问 比如说别人把你的网站加入了收藏夹 或者是加入了它的导航

那么他下次点击这个收藏夹里的网站就可以直接来访问了 或者是他记住了你的域名 通过你的域名 通过在浏览器输入你的域名 可只是可以直接访问

那第二个呢就是社交媒体 社交媒体的话是指有用户把你的内容分享到了像YouTube呀 Twitter呀 Facebook呀这些社交媒体 那么其他的用户看到了他们分享的这些链接

然后点击这些链接就可以进入到你的网站了 那第三块呢就是SEO SEO是指你在你的网站里面发布了内容 然后被搜索引擎给收录 然后用户呢通过搜在搜索引擎去进行一个搜索

通过某个关键词发现了你文章的内容 然后呢 通过如果你的文章或者你的网页排名比较靠前 那可能通过搜索结果直接就进入了你的网站 那么我们怎么样去提高这三块的一个访问量呢 其实直接访问的话是需要用户来认可你的品牌和价值

用户要能够记住你的网站 并且你的网站值得用户经常访问 值得他去加入收藏家或者加入他的导航 或者你的域名非常有个性 能够让用户在访问之后快速的记住你的域名

从而在有需要的时候直接去访问 那社交媒体的话需要有大量的这种分享的链接 这种可以是分享的文章 或者分享的视频 或者分享的图片都可以

比如说你在这个pinterest这个网站大量的分享图片 然后可以去去链接到你的这个网站 那么你分享的内容越多就越容易被算法推荐 那么也就越容易触达更多的用户 第三块SU的话呢

其实要想获取流量 我们需要研究关键词 并且以关键词排名作为一个目标去制作对应的内容 并且能获取关键词排名靠显的位置 从而才能获得一个长期稳定的搜索流量

但是国内的这种SU环境其实相对来说比较艰难 因为像百度啊 360等等这些国内的搜索引擎 他们在搜索结果里面其实大部分尤其是前两页大部分都是他们自己的产品 像独立的网站其实很难获取到排名和流量

所以做SU的话 其实主要是做谷歌SU以及必应的SU 其实相对来说从谷歌和必应可以通过SU获取到不错的流量 OK 这是站外引流的三种方式

那么我们这一这一整套课程呢 其实会重点来讲怎么样通过社交媒体或者是SU来获取流量 包含具体的策略和案例细节 那社交媒体其实获取流量的过程是有这样几步的 第一步呢

就是你要去发现有价值的这种话题 有争议的或者是能够引发传播的这种话题 那这种话题呢 其实就像鱼饵啊 你有很多个鱼饵

然后你围绕这些鱼饵呢 你去制作了一个属于你的鱼竿 也就是说你的内容生成了一篇内容围绕这个 你知道了这些鱼他们喜欢吃什么东西 根据热搜的这种话题

你知道了用户喜欢的内容 那么你根据这些内容再去制作相应的这种文章 那这些文章呢 在社交媒体上相应的就会比较容易获得算法的推荐 比如说你结合最近7天的热搜的话题 去针对性的写写一篇有传播力的这种文章

那这种文章你再去发到推特上 发到Facebook上 会比较容易获得算法推荐和传播

那你的内容其实相当于你的鱼竿 你的这个鱼竿能不能钓到鱼 其实取决于第一个是你的鱼饵够不够诱人 那第二个就是你的鱼竿质量到底好不好 你的鱼竿有没有放在合适的地方

如果你发了很多篇内容 有很多篇文章在社交媒体上为你去引流 那其实相当于你为自己编制了一张巨大的鱼网 那这个鱼网网到鱼的概率就更大了 那这是社交媒体的引流方式

那SVO其实话题对应的就是关键词 那你要知道用户他们热搜的关键词是什么 其实关键词你不用担心说被那些大的网站给垄断了 其实要整个速逻引擎这一块是有源源不断的新词的 就是用户的速逻需求其实一直在变

其实最近如果有什么新的动向 比如说最近发现的这个超导体 其实就会诞生很多大量的新的关键词 那这些新的关键词在速逻引擎上都会有很高的热度 那么我们围绕这些新的有热度的关键词再去做内容

这些内容的话就是我们的鱼竿鱼网 那么我们有了鱼饵鱼竿鱼网 我们就更容易在速逻引擎这个大海里面去捞到鱼 去网到鱼 那通过用户去搜索这些关键词呢

就能够在速逻引擎里面触达我们的整个网页或者文章这些内容 那最后我们把流量汇集到我们的网站之后呢 我们可以通过广告或者是电商来进行一个变现 那也就是说我们把把鱼打上来了 我们去卖鱼

我们通过这些鱼获取到我们的收入 那其实主要是广告 广告的话我们可以对接谷歌的AdSense 电商的话我们可以通过亚马逊或者独立站来进行一个变现 或者是接入亚马逊的affiliate

也就是CPS帮助其他人来买货进行一个变现 其实这样更简单 OK 说到这里的话 大家可能会想就是说付费广告相当于什么呢

其实付费广告 我的理解就是付费广告其实相当于直接去买鱼 然后把鱼买过来之后养大了再去卖 也就是说你直接花钱去买了一批用户 那这批用户到底能不能在你的产品里面留下来

到底能不能形成复购 其实是未知的 那如果你产品力足够强 你的产品非常OK 你的整个电商网站搭建的非常好

那其实用户是有可能去二次购买 去二次访问你的产品形成复购的 那这样的话你的你前面花了钱 可能可以从这个用户身上赚回来 但是说如果你的产品力本身不行

你去花了钱把用户买回来了 那可能用户流失了 他不会再来访问你的网站 会不会再次在你这边购买产品了 那其实相当于你前面的广告费就是打水漂了

所以就是说直接买鱼养大了再卖 这样的话也可以 但是说如果你养的好 鱼会越来越多 那你就会盈利

养的不好呢 那鱼就越来越少 你就会亏损 所以我的建议就是通过社交媒体和SEO这样免费的方式 我们先把鱼捞上来

先通过免费的方式去获得我们一定的这种基数 用户基数先养了一定的鱼 然后把整个商业模式证实可行之后 再通过付费的方式去为我们成立一个基数去扩大我们的规模 这样是可以的

所以必须要先通过SEO和社交媒体来免费获取一定的流量 然后再通过付费广告的方式去扩大我们的流量 扩大我们的整个收入 OK 这是战外引流整体的一个逻辑和思路
