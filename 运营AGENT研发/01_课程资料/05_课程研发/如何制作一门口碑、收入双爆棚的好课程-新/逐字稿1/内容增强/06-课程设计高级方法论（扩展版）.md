# 06-课程设计高级方法论（扩展版）

> 所属课程：如何制作一门口碑、收入双爆棚的好课程
> 讲师：黄有璨
> 本章重点：深入理解课程设计的底层规律和方法论

---

## 本章导学

### 核心目标

本章将深入探讨课程设计的底层规律和高级方法论，帮助学员从更专业的视角理解如何设计一门优质的课程产品。

### 学习目标

1. 掌握课程设计的底层规律
2. 理解用户学习的心理机制
3. 学会设计有效的学习路径
4. 掌握课程商业化的核心逻辑

### 2.2 用户学习心理机制

#### 用户学习的五个阶段

```
┌─────────────────────────────────────────────────────────────────────┐
│                      用户学习五阶段模型                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   阶段1：引起注意                                                  │
│   ├─ 问题：用户为什么要学？                                        │
│   ├─ 要点：激发兴趣、建立关联                                      │
│   └─ 方法：痛点共鸣、目标描绘、利益展示                             │
│                                                                     │
│   阶段2：激活旧知                                                  │
│   ├─ 问题：用户已经知道什么？                                      │

│   └─ 方法：复习、类比、对比                                        │
│                                                                     │
│   阶段3：呈现新知                                                  │
│   ├─ 问题：用户需要学什么？                                        │
│   ├─ 要点：清晰、准确、有序呈现                                    │
│   └─ 方法：讲解、示范、案例                                        │
│                                                                     │
│   阶段4：应用练习                                                  │
│   ├─ 问题：用户会了吗？                                            │
│   ├─ 要点：促进转化、发现问题                                      │
│   └─ 方法：练习、作业、反馈                                        │
│                                                                     │
│   阶段5：强化迁移                                                  │
│   ├─ 问题：用户能用吗？                                            │
│   ├─ 要点：巩固记忆、促进应用                                      │
│   └─ 方法：总结、复习、延伸                                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 本章小结

1. **课程价值公式**：课程价值由内容、服务、体验三个维度构成

2. **价值交付三层次**：知识传递 → 技能传授 → 思维塑造

3. **学习心理机制**：注意→激活→呈现→应用→迁移

4. **学习路径设计**：线性、分支、循环、网格、项目五种模式

5. **商业化逻辑**：定价策略、产品矩阵、口碑建设

6. **口碑核心要素**：超预期、情感连接、社交货币、传播便利
