## 本章导学（核心目标）

本章将讲解如何做好课程选题、明确课程定位，以及如何制定课程的学习交付目标。

**学习目标：**
1. 理解课程选题定位与售卖的关系
2. 掌握选题定位的核心流程
3. 学会制定清晰的学习交付目标
4. 理解三种售卖逻辑及对应策略

### 2.3 两种售卖逻辑

#### 逻辑1：爆款逻辑

**特征**：短期冲量，追求高销量

**适用场景**：
- 新兴话题，热度高
- 有强IP背书
- 流量充足

**策略**：
- 定价相对较低
- 集中资源推广
- 抓住时间窗口

---

#### 逻辑2：长销逻辑

**特征**：稳定销售，追求长期回报

**适用场景**：
- 经典主题，需求稳定
- 产品口碑好
- 渠道资源稳定

**策略**：
- 定价适中
- 持续渠道运营
- 口碑积累

---

### 2.4 课程交付目标的制定

#### 什么是交付目标

**定义**：用户学完课程后应该达成的状态

**重要性**：
- 是课程设计的核心锚点
- 直接影响课程定价
- 决定课程内容和服务的设计

#### 制定交付目标的方法

**SMART原则**：

| 原则 | 说明 | 示例 |
|------|------|------|
| Specific | 具体明确 | "掌握用户画像制作"而非"了解用户" |
| Measurable | 可衡量 | "能产出3份用户画像"而非"会做用户画像" |
| Achievable | 可达成 | 考虑用户基础和学习时间 |
| Relevant | 相关 | 与用户需求强相关 |
| Time-bound | 有时限 | "2周内掌握"而非"学会" |

## 案例分析

### 案例：运营课程选题定位

**背景**：做一门运营课程

**分析过程**：

```
1. 用户定位：
   - 0-3岁运营新人
   - 想转行做运营的小白
   - 需要系统学习

2. 话题方向：
   - 聚焦：新手运营入门
   - vs 宽泛：运营全面掌握

3. 课程模式选择：
   - 轻：知识付费型（99元）
   - 中：训练营型（999元）
   - 重：线下培训（9999元）

4. 售卖逻辑：
   - 爆款：低价引流，薄利多销
   - 长销：口碑积累，持续销售

5. 渠道选择：
   - 抖音/快手短视频
   - 微信社群
   - 知识付费平台
```

## 本章小结

1. **选题定位的本质**：回答5个核心问题，形成售卖逻辑和核心竞争力

2. **制作流程优化**：
   - 传统流程问题：易返工，成本高
   - 推荐流程：先打磨核心内容，验证大纲合理性

3. **两种售卖逻辑**：
   - 爆款逻辑：短期冲量，抓住时间窗口
   - 长销逻辑：稳定销售，口碑积累

4. **交付目标制定**：
   - 三个层次：最终目标、章节目标、知识点目标
   - SMART原则：具体、可衡量、可达成、相关、有时限

---

**思考题：**
1. 你的课程目标用户是谁？需求是什么？
2. 你计划采用哪种售卖逻辑？为什么？
3. 你的课程交付目标是什么？用SMART原则检验一下
