# 各阶段运营策略检查清单

## 探索期策略检查清单

### 核心目标验证

- [ ] **PMF验证**: 产品是否达到产品-市场匹配
  - 次日留存率是否>40%
  - 用户是否主动回来使用
  - 用户是否向他人推荐

- [ ] **核心功能验证**: 核心功能是否满足用户需求
  - 核心功能使用率是否稳定
  - 用户反馈是否指向核心功能
  - 是否存在高频使用的功能

### 策略执行检查

- [ ] **MVP策略**: 是否聚焦核心功能
  - 是否削减了非核心功能
  - 迭代周期是否保持快速（1-2周）

- [ ] **种子用户策略**: 是否建立核心用户群
  - 种子用户数量是否达到目标（100-500人）
  - 是否与种子用户保持深度互动
  - 种子用户反馈是否被有效收集

- [ ] **反馈驱动策略**: 是否建立反馈闭环
  - 是否有明确的反馈收集渠道
  - 反馈处理周期是否<1周
  - 产品迭代是否基于用户反馈

### 探索期禁止事项

- [ ] 不得追求DAU/MAU等规模指标
- [ ] 不得大量投入渠道获客
- [ ] 不得进行大规模功能扩展
- [ ] 不得投入品牌广告

---

## 成长期策略检查清单

### 核心目标验证

- [ ] **增长验证**: 是否实现规模化增长
  - 增速是否保持月增>10%
  - 获客成本是否在可接受范围
  - 自然增长占比是否提升

- [ ] **竞争验证**: 是否建立竞争优势
  - 市场份额是否提升
  - 用户选择我们的理由是否清晰
  - 竞争对手是否有跟随动作

### 策略执行检查

- [ ] **获客策略**: 是否实现高效获客
  - 各渠道ROI是否持续监测
  - 是否测试了10+获客渠道
  - 核心获客渠道是否已确定

- [ ] **爆品策略**: 是否打造了出圈卖点
  - 是否有用户主动传播的点
  - 是否有可裂变的功能/内容
  - 品牌是否开始有认知度

- [ ] **体验一致性**: 是否保障体验稳定
  - 用户体验是否保持一致
  - 服务端是否稳定
  - 客服响应是否及时

### 成长期禁止事项

- [ ] 不得忽视用户体验一致性
- [ ] 不得停止探索新的增长渠道
- [ ] 不得过早追求盈利
- [ ] 不得忽视竞争对手的跟进

---

## 成熟期策略检查清单

### 核心目标验证

- [ ] **价值验证**: 是否实现价值深耕
  - ARPU是否持续提升
  - 付费转化率是否稳定
  - 用户LTV是否健康

- [ ] **护城河验证**: 是否建立竞争壁垒
  - 用户迁移成本是否足够高
  - 品牌认知度是否领先
  - 是否有专利/独家资源

### 策略执行检查

- [ ] **用户分层策略**: 是否实现精细化运营
  - 是否有完整的用户分层体系
  - 各层用户是否有个性化运营
  - 高价值用户是否得到特殊照顾

- [ ] **ARPU提升策略**: 是否有效提升单用户价值
  - 是否有交叉销售策略
  - 会员体系是否完善
  - 是否有升级转化路径

- [ ] **新场景拓展**: 是否探索新增长点
  - 是否有新的用户场景
  - 新品类/新服务是否测试
  - 海外/下沉市场是否拓展

### 成熟期禁止事项

- [ ] 不得忽视高价值用户的流失
- [ ] 不得停止产品迭代
- [ ] 不得忽视新进入者挑战
- [ ] 不得过度收割用户价值

---

## 衰退期策略检查清单

### 核心目标验证

- [ ] **剩余价值验证**: 是否有效收割剩余价值
  - 剩余用户是否还有变现空间
  - 资产是否还有利用价值
  - 团队是否有其他出口

### 策略执行检查

- [ ] **成本优化策略**: 是否有效控制成本
  - 是否削减了非必要投入
  - ROI是否持续优化
  - 是否还有边际效益

- [ ] **收割策略**: 是否最大化剩余价值
  - 高意愿付费用户是否深耕
  - 是否有最后促销机会
  - 资产是否有序变现

- [ ] **迁移策略**: 用户是否有去处
  - 是否有新产品承接
  - 迁移方案是否制定
  - 用户通知是否有序

### 衰退期禁止事项

- [ ] 不得继续投入无回报的项目
- [ ] 不得伤害剩余用户体验
- [ ] 不得突然终止服务
- [ ] 不得忽视团队后续发展

---

## 通用阶段检查流程

```
1. 回顾本阶段核心目标
   └── 本阶段要达成什么

2. 检查关键指标达成
   ├── 核心指标表现
   └── 与预期对比

3. 识别下阶段信号
   ├── 阶段转型早期信号
   └── 需要警惕的问题

4. 评估策略执行情况
   ├── 执行了哪些关键动作
   └── 哪些没执行及原因

5. 制定下阶段调整
   └── 需要坚持/优化/停止什么
```
