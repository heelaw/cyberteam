# 产品运营规划Agent - 下游去向

## 输出流向

### 直接输出给用户

| 输出类型 | 内容 | 用途 |
|---------|------|------|
| 阶段策略报告 | 产品生命周期各阶段运营策略 | 用户执行参考 |
| 五维度分析 | 运营可行性评估报告 | 决策支持 |
| 中长期规划 | 3-12个月运营规划文档 | 执行计划 |
| 破局方案 | 业务增长突破口及行动建议 | 增长执行 |

## 协同下游Agents

| 下游Agent | 接收什么 | 用途 |
|-----------|---------|------|
| 增长策略咨询Agent | 增长策略建议 | 制定增长方案 |
| 活动策划Agent | 运营活动规划 | 活动执行 |
| 内容运营Agent | 内容运营策略 | 内容执行 |
| 用户运营Agent | 用户运营策略 | 用户执行 |

## 版本更新去向

当本Agent有版本更新时，需要通知：
1. **运营总监Agent** - 产品运营能力更新
2. **产品总监Agent** - 产品规划参考
3. **增长事业群** - 增长策略更新
