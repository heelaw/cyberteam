# 渠道评估打分表

## 使用说明

对每个候选渠道按以下维度打分（1-10分），最终加权计算总分。

## 基础信息

| 评估维度 | 权重 | 说明 |
|----------|------|------|
| 用户匹配度 | 25% | 渠道用户与目标用户的重合度 |
| 成本效率 | 25% | 获客成本和转化效率 |
| 规模潜力 | 20% | 渠道可承载的流量规模 |
| 竞争程度 | 15% | 渠道竞争激烈程度 |
| 门槛难度 | 15% | 渠道入驻和优化的难易程度 |

## 打分细则

### 用户匹配度（25%）

| 分数 | 描述 |
|------|------|
| 9-10 | 用户画像高度重合，渠道用户就是目标用户 |
| 7-8 | 用户重合度较高，可通过定向覆盖目标用户 |
| 5-6 | 用户有一定重合，需要内容或定向调整 |
| 3-4 | 用户重合度较低，需要较大人群拓展 |
| 1-2 | 用户画像差异大，不建议投入 |

### 成本效率（25%）

| 分数 | 描述 |
|------|------|
| 9-10 | CPA低于预期50%以上，ROI>3 |
| 7-8 | CPA低于预期20%左右，ROI 2-3 |
| 5-6 | CPA符合预期，ROI 1-2 |
| 3-4 | CPA高于预期50%，ROI<1 |
| 1-2 | CPA远高于预期，亏损严重 |

### 规模潜力（20%）

| 分数 | 描述 |
|------|------|
| 9-10 | 月均流量亿级以上，轻松消耗百万预算 |
| 7-8 | 月均千万级，百万预算需拓展多计划 |
| 5-6 | 月均百万级，百万预算需深挖优化 |
| 3-4 | 月均十万级，百万预算难以消耗 |
| 1-2 | 流量极小，不适合规模化投放 |

### 竞争程度（15%）

| 分数 | 描述 |
|------|------|
| 9-10 | 蓝海市场，竞争对手少 |
| 7-8 | 轻度竞争，有差异化空间 |
| 5-6 | 中度竞争，需要精细化运营 |
| 3-4 | 高度竞争，获客成本高 |
| 1-2 | 红海市场，头部垄断 |

### 门槛难度（15%）

| 分数 | 描述 |
|------|------|
| 9-10 | 开户简单，素材要求低，审核宽松 |
| 7-8 | 开户需资质，素材有特定要求 |
| 5-6 | 开户复杂，素材审核严格 |
| 3-4 | 需特批或白名单，素材审核周期长 |
| 1-2 | 几乎无法入驻或极难优化 |

## 决策建议

| 总分 | 建议 |
|------|------|
| 8.0+ | 主推渠道，重点投入 |
| 6.0-7.9 | 辅助渠道，适量投入 |
