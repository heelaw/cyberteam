## 渠道选择决策标准

### 主推渠道标准

- [ ] 四维评分≥8分
- [ ] 产品适配度≥9分
- [ ] 团队能力匹配度≥7分
- [ ] 预算能够支撑测试和放量

### 辅助渠道标准

- [ ] 四维评分6-8分
- [ ] 与主推渠道形成互补
- [ ] 能够在小预算下验证效果

### 测试渠道标准

- [ ] 四维评分5-6分
- [ ] 符合长期战略布局
- [ ] 测试预算≤总预算20%

## 常见错误清单

| 错误类型 | 错误描述 | 正确做法 |
|----------|----------|----------|
| 随大流 | 看别人投抖音我也投 | 根据产品特征选择 |
| 只看价格 | 哪个渠道便宜投哪个 | 综合考量ROI和用户质量 |
| 贪多求全 | 一次开10个渠道 | 聚焦2-3个核心渠道 |
| 忽视运营 | 以为开户就能出效果 | 预留优化时间和预算 |
| 缺乏测试 | 直接大批量投放 | 先小预算测试再放大 |
