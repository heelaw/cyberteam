# 复盘模板案例库

## 成功案例

### 案例1: 电商大促复盘
**背景**: 618活动效果未达预期，需分析原因
**方法**: KRT复盘法+数据对比分析
**发现**: 流量获取成本过高，转化率低于预期
**结果**: 制定双11优化策略
**适用场景**: 活动复盘、电商运营

### 案例2: 产品上线复盘
**背景**: 新功能上线后用户反馈不佳
**方法**: PDCA循环复盘+用户反馈分析
**发现**: 上线前测试不充分，优先级排序有误
**结果**: 建立更严格的上线检查清单
**适用场景**: 产品迭代、复盘文化

### 案例3: 团队协作问题复盘
**背景**: 跨部门项目出现沟通障碍和责任推诿
**方法**: 复盘四步法+责任矩阵明确
**发现**: 接口定义不清晰，缺乏沟通机制
**结果**: 建立每周跨部门同步会
**适用场景**: 团队管理、组织协作

### 案例4: 个人成长复盘
**背景**: 员工绩效未达标，需要分析改进
**方法**: 个人复盘模板+GROW模型
**发现**: 技能短板+工作方法问题
**结果**: 制定90天改进计划
**适用场景**: 绩效辅导、个人发展

### 案例5: 失败项目复盘
**背景**: 创新项目失败，需总结经验教训
**方法**: 根因分析+经验萃取
**发现**: 市场调研不足，过度技术导向
**结果**: 建立创新项目立项评估机制
**适用场景**: 创新管理、经验沉淀
