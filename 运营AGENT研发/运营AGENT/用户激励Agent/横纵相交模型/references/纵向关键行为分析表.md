# 纵向关键行为分析表

## 使用说明

在完成横向流程梳理后，针对每个流程节点，使用本表深入分析影响转化的关键行为。

### 通用影响因素

| **清晰度** | 信息展示清晰 | 理解率/误操作率 | 优化信息架构 |

| **即时性** | 即时反馈 | 响应时间/反馈率 | 优化反馈机制 |
| **激励性** | 正向激励 | 激励感知度/行动率 | 增加激励元素 |

## 表3: 影响程度评估标准

| 评分 | 影响程度 | 定义 | 行动建议 |

| 3 | 中等影响 | 对转化有一定影响 | 适度优化 |

## 表4: 流失关联度评估

### 流失关联度计算

```
流失关联度 = 节点流失率 × 行为影响程度
```

### 优先级判定

| 流失关联度 | 优先级 | 优化策略 |
|------------|--------|----------|
| >15 | P0 | 立即优化 |
| 10-15 | P1 | 尽快优化 |
| 5-10 | P2 | 计划优化 |
| <5 | P3 | 暂不优化 |
