# 激励方式选择决策表

## 一、激励方式对比

### 物质激励 vs 精神激励 vs 社交激励

| 对比维度 | 物质激励 | 精神激励 | 社交激励 |
|----------|----------|----------|----------|
| **即时性** | 强 | 中 | 弱 |
| **长期效果** | 弱 | 强 | 强 |
| **成本** | 高 | 低 | 低 |
| **可量化程度** | 高 | 中 | 低 |
| **适用行为** | 一次性/短期 | 持续性/长期 | 传播/裂变 |
| **用户疲劳度** | 高（易麻木） | 中 | 低 |

## 二、激励方式选择决策树

```
目标行为分析
    │
    ├── 行为类型判断
    │       │
    │       ├── 一次性行为（注册/首购/首次使用）
    │       │       │
    │       │       └── 推荐: 物质激励为主
    │       │
    │       ├── 持续性行为（每日使用/学习）
    │       │       │
    │       │       └── 推荐: 精神激励为主 + 物质激励辅助
    │       │
    │       ├── 传播性行为（分享/邀请）
    │       │       │
    │       │       └── 推荐: 社交激励 + 物质激励
    │       │
    │       └── 高价值行为（付费/大额购买）
    │               │
    │               └── 推荐: 物质激励 + 精神激励组合
    │
    └── 用户特征判断
            │
            ├── 价格敏感型用户
            │       │
            │       └── 推荐: 物质激励为主
            │
            ├── 成就感驱动型用户
            │       │
            │       └── 推荐: 精神激励为主
            │
            └── 社交驱动型用户
                    │
                    └── 推荐: 社交激励为主
```

## 四、激励力度设计指南

### 4.1 物质激励力度参考

| 行为价值 | 激励成本占比 | 激励力度 | 示例 |
|----------|--------------|----------|------|
| 高（>100元） | 5-10% | 中等 | 10-20元优惠券 |
| 中（50-100元） | 10-15% | 中上 | 5-10元优惠券 |
| 低（<50元） | 15-20% | 较高 | 积分加倍/小礼品 |

### 4.2 精神激励设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 稀缺性 | 高等级要有稀缺感 | 限量徽章/排名 |
| 可达性 | 努力就能获得 | 阶梯目标 |
| 展示性 | 可被他人看到 | 主页展示/排行榜 |
| 关联性 | 与业务目标一致 | 学习时长徽章 |

### 4.3 社交激励设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 信任背书 | 利用已有社交关系 | 好友推荐 |
| 双向激励 | 邀请人和被邀请人都有奖励 | 邀请各得X元 |
| 社交货币 | 分享能体现个人品味 | 专属内容分享 |

## 五、激励组合策略

### 5.1 组合原则

1. **主辅搭配**: 一种主要激励 + 一种辅助激励
2. **长短结合**: 即时激励 + 长期激励
3. **内外结合**: 物质激励 + 精神激励

### 5.2 组合示例

| 场景 | 组合策略 | 比例 |
|------|----------|------|
| 新用户激活 | 物质（首日奖励）+ 精神（新手指引成就） | 6:4 |
| 持续使用 | 精神（等级徽章）+ 物质（等级权益） | 7:3 |
| 传播裂变 | 社交（分享动机）+ 物质（邀请奖励） | 5:5 |
| 高价值转化 | 物质（大额优惠）+ 精神（专属身份） | 7:3 |

## 六、激励疲劳应对策略

### 6.1 疲劳信号识别

| 信号 | 定义 | 处理方式 |
|------|------|----------|
| 参与率下降 | 同一激励参与率持续下降>20% | 更换激励内容 |
| 成本上升 | 达到同样效果成本上升>30% | 优化激励结构 |
| 用户投诉 | 激励相关投诉上升 | 审视激励公平性 |

### 6.2 疲劳应对方法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| 激励多样化 | 增加不同类型的激励 | 单一激励疲劳 |
| 激励升级 | 提升激励价值或新鲜感 | 长期激励项目 |
| 激励限时化 | 限时限量增加紧迫感 | 促销类激励 |
| 激励个性化 | 基于用户偏好定制激励 | 用户差异大 |

## 七、激励效果评估表

### 7.1 短期效果指标

| 指标 | 计算方式 | 评估标准 |
|------|----------|----------|
| 参与率 | 参与人数/曝光人数 | >20%为佳 |
| 转化率 | 完成目标人数/参与人数 | 因场景而异 |
| 成本效率 | 激励成本/目标完成数 | 越低越好 |

### 7.2 长期效果指标

| 指标 | 计算方式 | 评估标准 |
|------|----------|----------|
| 用户留存率 | 激励后留存率 vs 激励前 | 提升>10%为佳 |
| 用户生命周期价值 | LTV变化 | 提升>15%为佳 |
| 行为内化程度 | 无激励时行为保持率 | >50%说明内化成功 |
