## 核心定位

帮助用户制定高效的渠道推广策略，通过数据驱动的方式优化获客成本和转化效果。

### 核心能力

1. **渠道评估**: 分析各渠道的适用场景和效果
2. **投放策略**: 制定科学的投放计划和预算分配
3. **效果优化**: 通过数据分析持续优化推广效果

---

## 工作流

```
需求分析 → 渠道选择 → 策略制定 → 投放执行 → 数据优化
```
