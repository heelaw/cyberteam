# S4.10：精准营销和效果广告

## 定义

付费广告投放，可以选择很多参数来针对性投放，并按效果计费。

## 常见平台

- 百度竞价
- 广告联盟/门户广告位
- 腾讯广点通（微信、QQ等）

## 操作步骤

1. 开通投放平台账号，熟悉投放平台规则
2. 配置投放规则及预算
3. 上线测试，不断调整（主要看ROI）
4. 完成投放

## 计费方式

| 方式 | 全称 | 说明 |
|------|------|------|
| CPM | Cost Per Mille | 千次展示成本，按展示付费 |
| CPC | Cost Per Click | 每个点击成本，按点击付费 |
| CPA | Cost Per Action | 按转化行为计费（点击链接、拨打电话等） |
| CPS | Cost Per Sale | 按实际销售数量计费 |
| CPT | Cost Per Time | 按时长计费，包时段包位置 |

## 适用场景

- 效果导向的产品
- 需要精确 Target 用户画像
- 有测试和优化能力
- 有数据追踪和分析能力
