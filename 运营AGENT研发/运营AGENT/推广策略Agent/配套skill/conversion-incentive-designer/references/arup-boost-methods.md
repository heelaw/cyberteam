# ARPU值提升方法

## ARPU值定义

ARPU（Average Revenue Per User）= 总收入 / 用户数

每用户平均收入是衡量产品变现能力的重要指标。

## 常见提升方法

### 1. 包邮

设置合理的包邮门槛，引导用户凑单。

**关键点：** 包邮门槛应根据客单价设置
- 低客单价（<50元）：门槛可设为满29元
- 中客单价（50-200元）：门槛可设为满99元
- 高客单价（>200元）：门槛可设为满199元或直接包邮

### 2. 相关推荐

在商品页/购物车页面推荐与主商品相关的配件或补充品。

**案例：**
- 买手机推荐手机壳、钢化膜、充电宝
- 买奶粉推荐奶瓶、保温杯

### 3. 满赠/满减/满返

达到一定消费金额后赠送商品、减免金额或返还积分。

**关键点：** 满减门槛应略高于平均客单价，引导提升消费

### 4. 产品对比

设计一个产品与主打产品作对比，帮助用户快速决策。

**技巧：**
- 用便宜来突出性价比
- 用贵来突出高端品质

**案例：**
```
普通版：¥99
豪华版：¥199（仅差4元，但多送X和Y）
→ 引导用户选择豪华版
```

### 5. 缺货引导

当普通款缺货时，引导用户选择利润更高的款式。

**案例：**
```
普通款：缺货，预计3天后到货
豪华款：有货，明天送达
→ 引导选择豪华款
```
