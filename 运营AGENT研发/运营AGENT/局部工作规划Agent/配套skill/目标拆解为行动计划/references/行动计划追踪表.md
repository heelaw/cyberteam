## 七、使用说明

### 追踪频率建议
| 任务类型 | 追踪频率 | 主要工具 |
|----------|----------|----------|
| 日常运营任务 | 每日站会 | 看板 |
| 项目型任务 | 每周 | 周报 |
| 长周期任务 | 每月 | 里程碑追踪 |
| 复杂项目 | 每两周 | 燃尽图 |

### 状态更新规范
- **每日**：更新当日完成情况
- **每周**：提交周报，更新整体进度
- **遇到问题**：立即更新状态（黄/红）
- **完成行动**：标记完成，记录实际完成日期
