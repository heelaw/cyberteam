# 团队状态管理模板

## 一、团队状态识别

### 状态评估维度
| 维度 | 指标 | 观察要点 |
|------|------|----------|
| 工作投入度 | 出勤、加班时长 | 是否正常、是否过度 |
| 协作意愿 | 沟通主动性、会议参与 | 是否积极、是否被动 |
| 产出质量 | 工作成果、返工率 | 是否稳定、是否有波动 |
| 情绪状态 | 面部表情、非正式沟通 | 是否积极、是否有怨气 |
| 离职信号 | 请假、社交媒体 | 是否有异常 |

### 状态分级
| 状态 | 表现 | 应对策略 |
|------|------|----------|
| 高效稳定 | 工作投入、产出高、质量好 | 保持现状，适度激励 |
| 压力过大 | 加班多、情绪紧张、易怒 | 减压、调整节奏 |
| 动力不足 | 被动执行、拖延、质量下降 | 激励、目标调整 |
| 疲惫涣散 | 出勤不稳定、沟通少、错误多 | 休整、团队建设 |
| 离职倾向 | 异常请假、社交消极、面试 | 深度沟通、了解原因 |

## 二、压力管理工具

### 压力信号识别清单
```
【个人层面】
身体信号：
- [ ] 经常头痛/胃痛
- [ ] 睡眠问题（失眠/早醒）
- [ ] 食欲变化
- [ ] 容易疲劳

行为信号：
- [ ] 迟到次数增加
- [ ] 会议经常缺席/早退
- [ ] 回避沟通
- [ ] 工作时间处理私事

情绪信号：
- [ ] 经常抱怨
- [ ] 情绪波动大
- [ ] 对工作失去热情
- [ ] 抱怨工作但又说不出具体不满

【团队层面】
- [ ] 整体加班时间明显增加
- [ ] 沟通中负面情绪多
- [ ] 团队活动参与率下降
- [ ] 冲突事件增加
```

### 减压措施库
| 措施 | 适用场景 | 操作方式 | 效果 |
|------|----------|----------|------|
| 调整工作量 | 工作量过饱和 | 重新分配任务、延期次要任务 | 快速见效 |
| 强制休整 | 长期高压 | 要求休假、强制周末不加班 | 恢复精力 |
| 授权赋能 | 过度管控 | 减少审批、让团队自己做主 | 提升自主感 |
| 资源补充 | 资源不足 | 申请增加人手/工具 | 提升效率 |
| 目标对齐 | 目标不清晰 | 重新解释目标和优先级 | 减少焦虑 |
| 团队活动 | 需要凝聚 | 团队建设、聚餐、户外 | 放松气氛 |

## 四、激励工具箱

### 即时激励（低成本/高频率）

| 激励方式 | 操作方式 | 适用场景 |
|----------|----------|----------|
| 口头表扬 | 当面/会议中具体表扬 | 做出成绩时 |
| 书面感谢 | 邮件/消息正式感谢 | 重要贡献 |
| 小礼物 | 咖啡/书籍/小礼品 | 特别感谢 |
| 弹性工作 | 允许早走/晚来 | 需要信任时 |
| 选择权 | 优先选择项目/任务 | 激励高潜 |

### 正式激励（高成本/低频率）

| 激励方式 | 操作方式 | 适用场景 |
|----------|----------|----------|
| 加薪 | 绩效评估后 | 年度激励 |
| 晋升 | 晋升周期 | 人才发展 |
| 奖金 | 项目完成后 | 成果激励 |
| 股权 | 长期绑定 | 核心人才 |
| 培训 | 外部课程/会议 | 发展激励 |

### 非物质激励技巧

1. **认可成就**
   - 具体说明什么做得好
   - 公开表扬（适合的人）
   - 感谢信/表彰公告

2. **提供发展机会**
   - 参与重要项目
   - 带领小团队
   - 外部学习和分享

3. **赋予自主权**
   - 让做决定
   - 减少监控
   - 信任和支持

4. **帮助成长**
   - 导师指导
   - 建设性反馈
   - 挑战性任务

## 七、常见问题处理

### Q1: 团队普遍加班严重怎么办？
**处理**：
1. 分析加班原因（工作量/效率/文化）
2. 与团队一起找解决方案
3. 设定加班上限，强制休整
4. 调整目标和范围
5. 申请增加资源

### Q2: 核心员工有离职倾向怎么办？
**处理**：
1. 尽快安排一对一深度沟通
2. 了解真实原因（薪酬/发展/环境）
3. 针对性提供解决方案
4. 必要时升级到HR
5. 提前准备备选人才

### Q3: 团队士气低落怎么提振？
**处理**：
1. 诊断根本原因（目标/领导/环境）
2. 短期：举办团队活动/小胜利
3. 中期：调整目标/改善环境
4. 长期：建立正向文化
