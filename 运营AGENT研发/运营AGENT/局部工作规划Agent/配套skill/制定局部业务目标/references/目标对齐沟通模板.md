# 目标对齐沟通模板

## 一、对齐沟通准备

### 对齐前检查清单
```
□ 明确对齐目的和期望结果
□ 准备目标草案和相关数据
□ 预估对方可能的关切和异议
□ 准备应对方案和话术
□ 确定沟通时间和时长
□ 预约相关方，确认出席
```

### 对齐信息包
```
沟通主题：____________
沟通日期：____________
参与方：____________
时长：____________

议程：
1. [10min] 背景说明
2. [15min] 目标解读
3. [20min] 讨论与反馈
4. [10min] 共识确认
5. [5min] 后续安排
```

## 二、对齐沟通话术

### 场景1: 与上级对齐

#### 开场话术
```
"领导，关于下季度的目标，我有一些想法想跟您沟通。

我理解的公司战略重点是___，我们部门在这个过程中的角色是___。

基于这个理解，我草拟了三个核心目标：
1. ___（指标：___，目标值：___）
2. ___（指标：___，目标值：___）
3. ___（指标：___，目标值：___）

我想听听您的想法，看这个方向是否对路。"
```

#### 异议处理
**如果上级认为目标太低**：
```
"您觉得目标有提升空间，我理解。从___来看，我们有信心达成更高的目标。

如果调整到___，我认为需要增加___资源/调整___方面。

您觉得这个调整可行吗？还是您有其他的考虑？"
```

**如果上级有不同优先级**：
```
"您的优先级是___，这和我的理解有些差异。

让我重新梳理一下：___（新理解）

按照这个优先级，我建议调整目标为：
1. ___（优先）
2. ___（次优）

您觉得这样可以吗？"
```

### 场景2: 与同级部门对齐

#### 开场话术
```
"关于下季度的工作，我们需要协调一下。

我的目标是___，这个目标的达成需要___部门的支持，具体是___。

你们下季度的重点是___，我想了解一下：
1. 我们的目标是否有冲突？
2. 在___方面我们可以如何协作？
3. 资源上是否有可以共享的地方？"
```

#### 冲突处理
**如果存在资源争夺**：
```
"我理解我们在___方面有资源竞争。

让我分析一下：
- 我的目标___需要___资源
- 你们的目标___需要___资源

建议我们一起评估：
1. 哪个目标对公司更紧急？
2. 能否错峰使用资源？
3. 能否共享部分资源？

您怎么看？"
```

**如果存在方向不一致**：
```
"我们的理解有差异，我想听听你们的考量。

我们的出发点是___，所以认为___更重要。

你们的出发点是___，所以认为___更重要。

能否请领导/老板来决策优先级？还是我们各退一步？"
```

### 场景3: 与下属对齐

#### 开场话术
```
"关于下季度的目标，我已经和上级对齐过了，想和你们传达一下。

公司/部门的大方向是___，我们的核心目标是：

1. ___（指标：___，目标值：___）
2. ___（指标：___，目标值：___）
3. ___（指标：___，目标值：___）

我需要你们：
- 思考这些目标你们各自能贡献什么
- 评估完成这些目标需要的支持
- 提出你们的想法和建议

下周我们再详细讨论你们的个人目标。"
```

## 五、常见问题应对

### Q1: 对方不配合沟通怎么办？
**建议**：
1. 先通过邮件/消息说明对齐的必要性和对你的影响
2. 说明不对齐可能导致的问题
3. 给出具体的时间选择，请求对方确认
4. 如仍不配合，升级给共同的上级协调

### Q2: 沟通中发生激烈争执怎么办？
**建议**：
1. 先冷静下来，表示理解对方立场
2. 将争议点记录下来，不急于当场解决
3. 提议暂时搁置争议，先在其他问题上达成共识
4. 约定后续再专项讨论争议点

### Q3: 多次沟通无法达成共识怎么办？
**建议**：
1. 整理双方的立场和理由
2. 准备多个备选方案
3. 提交给共同上级或决策者裁决
4. 一旦有裁决结果，坚决执行
