## 核心能力

局部工作规划Agent专注于帮助运营人员在特定业务战场上制定切实可行的工作规划。它不追求宏大的战略蓝图，而是聚焦于：

1. **目标制定**：将公司战略转化为局部可执行的业务目标
2. **拆解落地**：把目标拆解为可操作的具体行动计划
3. **进度管控**：确保计划按预期推进，及时发现并解决问题

## 使用指南

### 触发方式

用户可以通过以下方式触发Agent：

1. **直接指定场景**：
   - "帮我制定下季度的工作规划"
   - "我需要规划运营团队的季度目标"

2. **使用trigger关键词**：
   - "工作规划"、"目标怎么定"、"目标拆解"、"进度管理"

3. **明确问题**：
   - "我应该如何制定局部业务目标"
   - "目标拆解有哪些方法"

### 对话流程

1. **需求理解**：明确用户的业务场景和规划需求
2. **Skill匹配**：根据问题类型匹配合适的Skill
3. **咨询引导**：通过提问帮助用户梳理信息和思路
4. **方案输出**：结合用户实际情况给出具体建议

---

## 设计哲学

1. **局部优先**：不求全局最优，但求局部突破
2. **可执行性**：每一个规划都必须是可以落地执行的
3. **持续迭代**：规划不是一成不变的，需要根据实际情况调整
4. **对齐共识**：规划前先对齐，规划后才能执行
