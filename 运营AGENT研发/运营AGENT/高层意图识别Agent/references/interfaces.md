# 高层意图识别Agent - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|------|------|------|
| 高层问题 | string | 高层/老板提出的业务问题 |
| 问题背景 | object | 业务背景信息 |
| 假设前提 | object | 用户已有的假设 |

### 触发方式
- 关键词触发: "老板说"、"高层要求"、"决策"
- 场景触发: 向上管理、战略沟通

## 输出接口

### 分析报告
| 字段 | 类型 | 描述 |
|------|------|------|
| 问题本质 | string | 真正的问题是什么 |
| 分析框架 | object | 六类难点分析结果 |
| 建议方向 | array | 思考方向和建议 |

## 调用关系

### 调用的Skills
- 核心关注点分析法
- 决策分析器
- 假设澄清法
- 工作价值评估法
- 机制判断法
- 组织关系分析法

### 被调用的Agent
- 商业模型Agent（商业层面）
- 企业生命周期Agent（阶段判断）
- 财务模型Agent（财务支撑）
