# 常见业务假设清单

## 产品假设

- [ ] 用户需要这个功能
- [ ] 用户愿意为这个功能付费
- [ ] 产品体验足够好
- [ ] 产品有差异化

## 市场假设

- [ ] 市场规模足够大
- [ ] 市场正在增长
- [ ] 可以触达目标用户
- [ ] 用户愿意尝试新产品

## 商业模式假设

- [ ] 这个定价用户可以接受
- [ ] 获客成本可控
- [ ] LTV大于CAC
- [ ] 可以规模化

## 竞争假设

- [ ] 我们有竞争优势
- [ ] 竞争对手无法复制
- [ ] 可以建立壁垒

## 团队假设

- [ ] 团队有能力执行
- [ ] 可以招到需要的人
- [ ] 创始人愿意投入
