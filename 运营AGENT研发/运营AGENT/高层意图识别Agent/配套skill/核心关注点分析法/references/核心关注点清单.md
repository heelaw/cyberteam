# 各阶段核心关注点

## 初创期核心关注点

- 找到Product-Market Fit
- 验证核心假设
- 活下来
- 找到第一批用户
- 建立核心团队

---

## 成长期核心关注点

- 规模化增长
- 验证商业模式可复制
- 建立竞争壁垒
- 储备足够资金
- 打造核心团队

---

## 成熟期核心关注点

- 提升运营效率
- 保持稳定增长
- 探索新增长点
- 优化组织能力
- 准备退出路径

---

## 转型期核心关注点

- 识别新增长机会
- 评估新业务方向
- 保持现有业务
- 调整组织结构
- 获取转型资源
