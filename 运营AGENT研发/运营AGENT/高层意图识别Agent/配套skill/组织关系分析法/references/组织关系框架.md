# 组织关系分析框架

## 各层级关注点

### 高层（老板/VP）

- 公司战略方向
- 业务结果
- 竞争格局
- 团队整体
- 资源分配

### 中层（总监/经理）

- 任务执行
- 团队管理
- 效率提升
- 跨部门协调
- 向上汇报

### 基层（员工）

- 具体任务
- 工具和流程
- 成长机会
- 工作环境
- 收入稳定

---

## 常见组织问题

| 问题类型 | 表现 | 解决方法 |
|----------|------|----------|
| 沟通问题 | 信息不对称 | 建立沟通机制 |
| 期望错位 | 目标不一致 | 对齐KPI |
| 资源争夺 | 部门墙 | 协调机制 |
| 能力不足 | 胜任力问题 | 培训/招聘 |

---

## 协作原则

- 换位思考
- 目标对齐
- 责权对等
- 及时沟通
