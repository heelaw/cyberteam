# 考核标准

- 是否按可执行维度分层，而不是只写年龄职业。
- 是否识别主写对象、辅助对象和不该写的人群。
- 是否每一层都有行为或决策证据。
- 是否能把分层结果直接接到写作上。

## 量化门禁

- 必须识别购买者、使用者、拍板者、阻拦者、影响者中的至少 3 类。
- 必须说明为什么这个人是主写对象，而不是只写角色名称。
- 必须至少给出 1 条行为或决策证据。
- 必须明确产品属于单人产品、买用一体、买用分离或委员会决策中的哪一种。
- 必须说明主写对象和辅助对象如何分工。
- 不能只按年龄、性别、职业分人群。
- 结果必须能直接接到写作，不是停在画像层。

## 一票否决补充

- 把所有人都当主对象。
- 忽略决策权。
- 忽略认知差异。
- 输出只有标签，没有主次。
