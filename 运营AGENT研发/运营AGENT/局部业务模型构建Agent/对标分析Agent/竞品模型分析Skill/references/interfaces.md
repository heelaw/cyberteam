# 竞品模型分析Skill - 接口协议

## 输入接口

| 接口名称 | 类型 | 描述 | 必需 |
|----------|------|------|------|
| competitor_name | string | 竞品名称 | 是 |

## 输出接口

| 接口名称 | 类型 | 描述 |
|----------|------|------|
| competitor_profile | Object | 竞品概况 |
| business_model | Object | 商业模式分析 |
| operation_strategy | Object | 运营策略分析 |
| insights | Object | 洞察与建议 |
