# 行动计划输出Skill - 接口协议

## 输入接口

| 接口名称 | 类型 | 描述 | 必需 |
|----------|------|------|------|
| solution | Object | 解决方案 | 是 |
| timeline | Object | 时间约束 | 否 |

## 输出接口

| 接口名称 | 类型 | 描述 |
|----------|------|------|
| action_plan | Object | 行动计划 |
| milestones | Array | 里程碑列表 |
| task_breakdown | Object | 任务分解 |

## 错误码

| 错误码 | 描述 |
|--------|------|
| E001 | 解决方案为空 |
| E002 | 无法分解任务 |
