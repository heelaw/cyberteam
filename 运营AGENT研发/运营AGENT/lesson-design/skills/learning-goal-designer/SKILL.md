# learning-goal-designer

## 输入

- 课程背景
- 目标人群
- 学习目标
- 约束

# 学习目标定义

## SOP

1. 判断学员起点。
2. 判断交付终点。
3. 选择认知/理解/应用/掌握层级。
4. 输出可测量目标。


## 输出

- 结构化结果
- 下一步建议
