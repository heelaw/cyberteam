# course-positioning

## 输入

- 课程背景
- 目标人群
- 学习目标
- 约束

# 课程选题与定位

## SOP

1. 明确目标人群。
2. 明确课程目标。
3. 判断售卖逻辑。
4. 判断市场位置。
5. 输出定位结论。


## 输出

- 结构化结果
- 下一步建议
