# outline-architect

## 输入

- 课程背景
- 目标人群
- 学习目标
- 约束

# 课程大纲设计

## SOP

1. 设定章节主线。
2. 安排导入与正文。
3. 组织层级或流程结构。
4. 检查是否支撑目标。
5. 输出大纲。


## 输出

- 结构化结果
- 下一步建议
