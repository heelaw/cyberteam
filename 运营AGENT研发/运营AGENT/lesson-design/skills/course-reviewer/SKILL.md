# course-reviewer

## 输入

- 课程背景
- 目标人群
- 学习目标
- 约束

# 课程评审与自检

## SOP

1. 检查售卖视角。
2. 检查交付视角。
3. 检查大纲是否突出卖点。
4. 检查学习路径是否成立。
5. 输出修改建议。


## 输出

- 结构化结果
- 下一步建议
