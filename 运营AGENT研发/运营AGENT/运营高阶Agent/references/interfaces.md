# 运营高阶Agent - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|------|------|------|
| 问题类型 | string | 战略/框架/策略/执行 |
| 业务背景 | object | 市场、资源、目标 |
| 当前阶段 | string | 业务发展阶段 |

### 触发方式
- 关键词触发: "运营策略"、"框架思维"、"顶层设计"
- 场景触发: 战略规划、业务操盘

## 输出接口

### 战略方案
| 字段 | 类型 | 描述 |
|------|------|------|
| 方案类型 | string | 战略/框架/策略 |
| 核心内容 | object | 具体方案内容 |
| 执行计划 | array | 分阶段执行计划 |
| 风险预案 | array | 风险和对策 |

## 调用关系

### 调用的Skills
- 自上而下框架构建
- 自下而上框架构建
- 抽象问题拆解

### 被调用的Agent
- 框架思维Agent（框架构建）
- 策略执行Agent（执行落地）
- 数据驱动Agent（数据驱动）
