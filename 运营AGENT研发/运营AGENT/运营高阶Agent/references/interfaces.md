# 运营高阶Agent - 接口协议

## 输入接口

| 字段 | 类型 | 描述 |
|------|------|------|
| 问题类型 | string | 框架 / 指标 / 行动 / 规划 / 激励 / 执行 / 数据 |
| 业务背景 | object | 场景、资源、目标、约束 |
| 当前阶段 | string | 冷启动 / 增长期 / 成熟期 |
| 可用数据 | object | 现有指标、口径、样本 |

## 输出接口

| 字段 | 类型 | 描述 |
|------|------|------|
| 路由结果 | object | 命中的 Skill、理由、边界 |
| 结构化结论 | object | 框架、规划、指标或执行结论 |
| 行动计划 | array | 下一步动作与优先级 |
| 风险提示 | array | 缺口、前提和不可做事项 |
