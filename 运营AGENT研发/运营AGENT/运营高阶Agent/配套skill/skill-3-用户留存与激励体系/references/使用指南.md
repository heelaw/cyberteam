# 参考资料：用户留存与激励体系

## 核心概念

### 用户流失4种原因

1. **产品价值不明确**：用户不知道产品能带来什么
2. **价值获取成本高**：用户需要太多努力才能体验价值
3. **激励不足**：用户缺乏行动动力
4. **替代品吸引**：用户被竞品抢走

### 价值洼地公式

```
价值洼地 = 用户需求 - 产品供给
```

### 三层激励体系

1. **物质激励**：红包、优惠券、积分、会员权益
2. **精神激励**：徽章、称号、排行榜、成长体系
3. **社交激励**：关注、粉丝、社群、邀请

### 激励设计原则

- 激励必须与目标行为挂钩
- 短期价值引导长期价值
- 物质为基础，精神和社交为提升
