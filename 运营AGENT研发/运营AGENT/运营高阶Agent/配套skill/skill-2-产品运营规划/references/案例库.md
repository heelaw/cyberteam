# 参考资料：产品运营规划案例库

## 案例1：新品市场进入

**场景**：AI教育产品进入K12市场

**规划要点**：
- 阶段判断：探索期向成长期过渡
- 破局点：打造标杆案例，验证效果
- 策略重心：集中资源验证PMF，而非铺量

## 案例2：成熟工具产品增长

**场景**：工具类产品DAU增长停滞

**规划要点**：
- 阶段判断：成熟期
- 破局点1：深耕高价值用户，提升ARPU
- 破局点2：探索B端企业市场

## 案例3：社区产品转型

**场景**：内容社区从工具向社区转型

**规划要点**：
- 阶段判断：探索期（新方向）
- 破局点：内容生产者激励体系
- 策略重心：先建立内容生产-消费循环
