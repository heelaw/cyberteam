## 核心定位

把 `01_课程资料/02_运营领域/运营高阶课` 的内容拆成 7 条可独立路由的最小 Skill，并保证框架、规划、激励、执行、数据不串线。

## 工作流

框架认知 → 指标拆解 → 行动指导 → 产品规划 → 留存激励 → 策略执行 → 数据驱动

## 配套 Skills

1. 框架式思维与运营框架
2. 运营框架理解与指标拆解
3. 行动指导型框架
4. 产品运营规划
5. 用户留存与激励体系
6. 策略执行
7. 数据驱动运营
