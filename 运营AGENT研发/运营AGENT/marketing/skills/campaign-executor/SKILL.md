# 推广执行器

## Skill 概述

把方案变成可执行计划，关注准备、上线、监控、优化和收口。

## 不做的事

- 不替代用户洞察
- 不替代渠道选择
- 不替代转化设计

## SOP

1. 整理目标、资源、周期和负责人。
2. 拆出准备、上线、监控、优化、收口五段。
3. 列出物料、技术、预算和风控清单。
4. 定义验收标准和复盘条件。
5. 输出可执行排期。

## 输出

- 执行清单
- 排期表
- 负责人
- 风险清单
- 验收标准
