# 漏斗诊断器

## Skill 概述

识别广告投放、社群转化、留存或内容分享漏斗的瓶颈，并给出下一步验证动作。

## 不做的事

- 不替代渠道选择
- 不替代转化设计
- 不替代执行清单

## SOP

1. 判断漏斗类型。
2. 列出每个阶段和关键指标。
3. 找出流失最大的环节。
4. 提出最小可验证假设。
5. 输出下一轮测试建议。

## 输出

- 漏斗图
- 关键指标
- 最大流失点
- 测试假设
- 优化建议
