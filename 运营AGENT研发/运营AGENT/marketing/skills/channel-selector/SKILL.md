# 渠道选择器

## Skill 概述

判断媒介购买、平台流量、事件营销、效果广告哪一种最合适，并给出试跑顺序。

## 不做的事

- 不替代用户洞察
- 不替代转化设计
- 不替代投放执行

## SOP

1. 确认目标、预算、时效和受众匹配度。
2. 对照四类推广方式。
3. 比较可控性、成本和扩展性。
4. 输出首选渠道和备选渠道。
5. 给出试跑顺序和风险。

## 输出

- 渠道排序
- 选择理由
- 试跑计划
- 关键指标
- 风险提示
