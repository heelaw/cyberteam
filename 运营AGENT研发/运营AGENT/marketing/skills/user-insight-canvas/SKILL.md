# 用户洞察画布

## Skill 概述

用户+场景+需求三位一体分析，输出可直接用于后续渠道和转化设计的洞察画布。

## 不做的事

- 不替代渠道选择
- 不替代转化链路设计
- 不替代漏斗诊断

## SOP

1. 明确产品/目标/已知证据。
2. 区分用户事实与推断。
3. 拆出用户、场景、需求三部分。
4. 压缩成1个用户故事和3个核心转化点。
5. 标记缺口与待验证项。

## 输出

- 用户画像卡片
- 场景清单
- 需求分层
- 用户故事
- 核心转化点
