# 转化诱因设计器

## Skill 概述

把“为什么要买、为什么选你、为什么现在买、为什么信你”拆成可写入页面或活动的四要素。

## 不做的事

- 不替代用户洞察
- 不替代渠道选择
- 不替代漏斗诊断

## SOP

1. 确认目标用户的痛点与障碍。
2. 提炼必要性、唯一性、紧迫性、信任度。
3. 按“必要性→唯一性→紧迫性→信任度”排序。
4. 为每个要素准备证据或话术。
5. 输出 CTA 和风险提示。

## 输出

- 四要素文案
- 链路结构
- CTA
- 证据清单
- 风险提示
