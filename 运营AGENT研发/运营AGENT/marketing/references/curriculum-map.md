# 课程映射

## 来源课程

- 运营初阶课：导论与运营认知、文案写作、活动策划、社区运营、社群运营、新媒体运营、渠道推广
- 运营小课：用户洞察、流量密码、数据漏斗、互动原则、转化设计
- 运营高阶课：增长全景图、渠道分析、激活与留存、数据驱动优化

## Skill 对应

| Skill | 对应 SOP | 关键判断 |
|---|---|---|
| `user-insight-canvas` | 用户洞察画布 | 谁在什么场景下有什么需求 |
| `channel-selector` | 渠道类型判断 | 走哪条推广路最划算 |
| `conversion-trigger-designer` | 转化诱因设计 | 为什么买、为什么选你、为什么现在买、为什么信你 |
| `funnel-diagnoser` | 数据漏斗诊断 | 哪一段在漏、该怎么验证 |
| `campaign-executor` | 推广执行清单 | 怎么落地、怎么监控、怎么收口 |

## 边界说明

- 用户画像与场景判断先做。
- 渠道选择再做。
- 转化设计在渠道之后做。
- 漏斗诊断用于发现问题，不负责重写目标。
- 执行清单用于落地，不替代策略判断。

## 与其他 Agent 的关系

- 文案类问题可下发到 `营销转化Agent/转化文案写作法`。
- 活动创意和活动方案可进一步拆给活动类 Agent。
- 社群和私域执行可交给社群类 Agent。
