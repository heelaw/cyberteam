# 四类营销推广方法

## 1. 媒介资源购买占有型

### 特点
- **曝光量**：相对稳定（取决于媒体覆盖面）
- **点击量**：完全取决于文案和创意质量
- **转化率**：取决于流程设计和素材

### 流程
1. 寻找媒体公司或广告渠道
2. 获取资源列表和价格（媒体资料包）
3. 协商资源、价格和排期
4. 准备素材（图片、文案等）
5. 完成投放

### 常见资源
- **传统媒体**：报纸、杂志、电视、广播
- **户外媒体**：地铁灯箱、公交广告、广告牌
- **数字媒体**：网站横幅、应用广告位
- **社交媒体影响者**：微信公众号、微博账号、抖音创作者

### 适用场景
- 需要可预测的曝光量
- 有付费投放预算
- 建立品牌知名度
- 在已知渠道定向特定人群

## 2. 信息分发平台流量推广

### 特点
- **曝光量**：取决于优化效果或平台推广
- **点击量**：取决于文案质量和平台支持
- **转化率**：取决于流程和素材

### 两种方式

#### 方式一：规则优化
**流程：**
1. 研究平台的信息展示和排名规则
2. 准备和调整内容格式和细节
3. 持续优化（前5位获得最多流量）
4. 跟踪和迭代

**案例：**
- 应用商店：优化关键词、截图、描述以提升搜索排名
- SEO：优化网站结构、内容、反向链接以提升搜索排名
- 社交媒体：研究算法以最大化自然触达

#### 方式二：资源利用
**流程：**
1. 与平台管理员建立关系
2. 获取即将进行的官方活动信息
3. 准备素材并参与活动
4. 投放和优化
5. 活动结束

**案例：**
- 应用商店精选位、新产品推荐
- 电商平台主题活动
- 社交媒体官方活动参与

### 适用场景
- 预算有限，愿意投入时间
- 长期自然增长策略
- 平台有大量免费流量潜力

## 3. 传播事件型推广

### 特点
- **曝光量**：取决于创意和运气
- **点击量**：取决于创意和运气
- **转化率**：取决于转化逻辑设计和创意

### 流程
1. **创意策划**：创建引人注目的概念
2. **资源整合**：准备素材，聚集合作伙伴
3. **执行**：协调跨渠道投放
4. **观察调整**：监控响应，必要时调整

### 成功要素
- 与目标受众的情感共鸣
- 可分享性因素
- 时机和背景
- 与转化路径的整合

### 经典案例

#### 途牛"城市里的海滩"活动
- **概念**：城市街道上的人躺在沙滩上
- **文案**："心中有沙滩，哪里都是马尔代夫"
- **流程**：街拍照片 → 社交热议 → 途牛提供真实旅行
- **结果**：大规模社交分享和品牌知名度

#### "高质量男性"病毒现象
- **概念**：古怪的人设声称是富有的投资者
- **吸引点**：声称与外表之间的荒谬反差
- **变现**：售价75,000元/年的投资社区会员
- **结果**：病毒式传播，将好奇心转化为销售

### 适用场景
- 品牌建设和知名度
- 有创意资源
- 愿意接受不确定的结果
- 产品具有病毒传播潜力

### 风险考量
- 并非所有事件都会病毒式传播
- 可能会有负面反弹
- 可能无法直接推动转化
- 需要谨慎传递信息以避免品牌损害

## 4. 精准营销效果广告类

### 特点
- **曝光量**：可通过预算和定向配置
- **点击量**：可通过出价和创意优化
- **转化率**：可追踪和优化

### 常见付费模式

| 模式 | 定义 | 使用场景 |
|-------|------------|----------|
| **CPM** (Cost Per Mille) | 按千次展示付费 | 品牌知名度、高流量网站 |
| **CPC** (Cost Per Click) | 按点击付费 | 搜索广告、初期测试 |
| **CPA** (Cost Per Action) | 按特定行动付费（下载、注册） | 用户获取、应用安装 |
| **CPS** (Cost Per Sale) | 按销售/交易付费 | 电商、联盟营销 |
| **CPT** (Cost Per Time) | 按时长或参与度付费 | 视频观看、网站停留时间 |

### 流程
1. **账户设置**：开设广告平台账户
2. **平台熟悉**：了解定向选项、出价规则
3. **广告配置**：设置预算、定向、创意
4. **测试阶段**：小规模启动，监控表现
5. **优化**：根据数据调整定向、出价、创意
6. **扩展**：增加成功组合的预算

### 主要平台

#### 搜索广告
- **百度SEM**：基于关键词的文字广告
- **特点**：高意图、精准定向
- **最适合**：主动搜索解决方案的用户

#### 社交广告
- **微信广告**：朋友圈广告、公众号投放
- **抖音/TikTok**：视频广告、直播
- **微博**：信息流广告、热搜投放

#### 展示广告网络
- **广告网络**：跨网站广泛覆盖
- **重定向**：重新吸引过往访客
- **程序化**：自动购买和优化

### 适用场景
- 需要可扩展的增长
- 有明确的转化目标
- 有测试和优化预算
- ROI为正的客户获取

### 优化建议
- 从广泛定向开始，然后缩小范围
- 持续进行创意A/B测试
- 监控质量得分（影响成本）
- 设置频次上限以避免广告疲劳
- 使用重定向提高转化率

## 选择合适的方法

### 决策框架

| 因素 | 媒介购买 | 平台规则 | 事件营销 | 效果广告 |
|--------|---------------|----------------|-----------------|-----------------|
| **预算** | 高 | 低 | 中等 | 高（测试）→ 优化后 |
| **见效时间** | 快 | 慢 | 不确定 | 快（优化后） |
| **可控性** | 高 | 低 | 低 | 高 |
| **可扩展性** | 有限 | 有限 | 有限 | 高 |
| **品牌vs直接** | 品牌知名度 | 两者兼备 | 品牌 | 直接响应 |

### 常见组合
- **启动策略**：效果广告 + 平台规则（快速见效 + 自然基础）
- **品牌活动**：媒介购买 + 事件营销（保证触达 + 病毒潜力）
- **增长阶段**：平台规则 → 效果广告（扩展前验证）
