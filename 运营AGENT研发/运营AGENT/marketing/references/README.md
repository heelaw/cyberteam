# 参考索引

## 读取顺序

1. 先看 `curriculum-map.md`，确认这个营销问题属于哪一段课程 SOP。
2. 再看对应 Skill 的 `references/reference.md`，拿到最小判断框架。
3. 再看 `assessment.md`，检查输出是否够用。
4. 需要案例时，再进入 `assess/cases.md` 或具体案例文件。

## 文件作用

- `channel_types.md`：四类推广方式与适用场景。
- `conversion_triggers.md`：必要性、唯一性、紧迫性、信任度。
- `advanced/user_insights_canvas.md`：用户洞察画布。
- `advanced/data_funnels.md`：四种常见数据漏斗。
- `advanced/engagement_principles.md`：参与与互动原则。
- `advanced/traffic_codes_map.md`：流量密码与内容角度。
- `tools/checklists.md`：投放和执行检查清单。
- `assess/cases.md`：案例库索引与拆解提示。
- `evals/quality评估清单.json`：整体质量评分口径。
- `assets/templates/promotion_plan_template.md`：营销方案模板。
