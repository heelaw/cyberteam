# 营销推广执行检查清单

## 概述
本文档提供营销推广各阶段的执行检查清单，帮助确保不遗漏关键步骤。

---

## 📋 推广前准备清单

### 用户定义
- [ ] 明确目标用户画像（年龄、性别、地域、兴趣）
- [ ] 分析用户痛点和需求
- [ ] 确定用户决策成本（高/低）
- [ ] 识别用户活跃渠道

### 渠道评估
- [ ] 列出潜在推广渠道（至少5个）
- [ ] 评估每个渠道的用户匹配度
- [ ] 了解渠道推广规则和限制
- [ ] 确定渠道成本结构（CPM/CPC/CPA/CPS）
- [ ] 选择优先测试的渠道（1-2个）

### 转化设计
- [ ] 设计完整转化路径（5步以内）
- [ ] 定义各环节转化诱因：
  - [ ] 必要性诱因（痛点场景）
  - [ ] 唯一性诱因（差异化优势）
  - [ ] 紧迫性诱因（限时/限量）
  - [ ] 信任度诱因（背书/证言）
- [ ] 设计Landing页或承接页
- [ ] 准备追踪链接（UTM参数）

### 物料准备
- [ ] 准备文案（标题/正文/CTA）
- [ ] 准备视觉素材（图片/视频）
- [ ] 设置优惠活动（如有）
- [ ] 配置客服/售后资源

---

## 🚀 投放执行清单

### 账号设置
- [ ] 开启广告平台账号
- [ ] 完成企业认证（如需要）
- [ ] 绑定支付方式
- [ ] 设置投放预算（日预算/总预算）

### 受众定向
- [ ] 基础定向：年龄、性别、地域
- [ ] 兴趣定向：相关兴趣标签
- [ ] 行为定向：相关行为数据
- [ ] 自定义人群：上传种子用户（如有）
- [ ] 排除定向：排除不相关人群

### 创意配置
- [ ] 上传广告素材
- [ ] 设置广告标题和描述
- [ ] 配置落地页链接
- [ ] 设置转化追踪
- [ ] 配置A/B测试变量

### 出价策略
- [ ] 选择出价方式（CPC/oCPM）
- [ ] 设置初始出价
- [ ] 设置出价上限
- [ ] 配置投放时段
- [ ] 设置频次控制

## 💡 快速参考

### 常见转化率基准
- 详情页转化率：1-4%
- 微信图文点击：8-10%
- APP下载：1-2.5%
- 直播间下单：2-5%

### ROI计算公式
```
ROI = (收入 - 成本) / 成本
或
ROI = 收入 / 成本

目标：ROI > 1.5（可持续投放）
```

### 紧迫性词汇参考
- 限时：今日/本周/最后X天
- 限量：仅限XX份/售完即止
- 特价：直降XX/限时XX折
- 独享：新人专享/VIP专享
