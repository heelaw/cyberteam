# 案例库

## 1. 完美日记小红书种草

- **场景**：电商品牌新品上市。
- **关键动作**：KOL/KOC 金字塔、统一话题、品牌词承接、限时福利。
- **可复用判断**：先看用户是谁，再看内容怎么铺，再看品牌词和承接是否接得住。

## 2. 在线教育投放漏斗

- **场景**：信息流广告带来课程成交。
- **关键动作**：展示 → 点击 → 详情页 → 表单 → 支付 → 后续跟进。
- **可复用判断**：优先找最大流失段，不要平均优化。

## 3. 社群转化漏斗

- **场景**：前端流量进入社群后推动试驾/试用/咨询。
- **关键动作**：进群、互动、筛选高意向、关键体验、购买。
- **可复用判断**：大投入前先筛出高意向人群。
