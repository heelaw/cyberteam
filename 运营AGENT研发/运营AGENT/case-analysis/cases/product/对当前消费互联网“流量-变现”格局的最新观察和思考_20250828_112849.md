## 过去10年，整个互联网世界里，很多公司和创业者寻求的大机会，往往是对于“流量入口”的重构。

而今天，互联网流量格局已经相对奠定，重构“流量入口”的机会已经几乎不再存在（至少在元宇宙或其他新技术的发展带来重要变化之前是这样的）。
所以，今天往后，更多个人或创业者再想要寻求机会，更应该关注从“流量管道-交易成交场景-终端变现出口”方面存在的机会。
