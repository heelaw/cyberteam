# 1on1沟通模板

## 使用说明
用于管理者与团队成员一对一深度沟通，支持员工发展和关系维护。

## 一、沟通前准备

### 管理者准备清单

|--------|------|------|
| 回顾上次沟通action | | |

### 话题优先级判断

| 情况 | 建议话题 |
|------|----------|
| 新入职员工 | 适应情况、融入困难、资源需求 |
| 绩效下降 | 原因分析、支持需求 |
| 高潜力员工 | 职业发展、成长需求 |
| 稳定期员工 | 持续激励、新挑战 |
| 离职风险 | 保留谈话、真实原因 |

## 二、沟通框架

### 核心原则

1. **开放对话**: 创造安全的沟通氛围
2. **深度倾听**: 70%听，30%说
3. **具体反馈**: 事实+影响+建议
4. **共同成长**: 员工目标与团队目标结合

### 结构化话术

#### 开场（5分钟）
"今天我们1on1，主要是想听听你的想法和需要支持的地方。你可以聊聊最近的工作状态，或者有什么想讨论的话题。"

#### 话题深入（35-50分钟）

**必聊话题**:

1. **工作状态**
   - "最近工作感觉怎么样？"
   - "有什么让你特别有成就感的事？"
   - "有什么让你感到挑战或困扰的事？"

2. **目标进展**
   - "你的[目标/项目]进展如何？"
   - "有什么需要我协调的资源？"
   - "预期和实际有什么差距？"

3. **成长发展**
   - "最近学到了什么新东西？"
   - "希望下阶段在哪些方面提升？"
   - "有什么想尝试的新方向吗？"

4. **反馈与建议**
   - "我有什么可以帮到你的？"
   - "团队或公司有什么可以改进的？"
   - "希望我多关注哪些方面？"

#### 总结与Action（5-10分钟）

"我们来总结一下今天的讨论：
1. [讨论要点1]
2. [讨论要点2]

你这边确认的action是[xxx]，我这边确认的action是[yyy]。
下次1on1我们再回顾进展。"

## 四、反馈模型

### 反馈话术模板

**正向反馈**:
"我观察到[具体行为]，[这个行为带来了什么积极影响/结果]，[表达感谢/认可]。继续保持！"

**改进反馈**:
"我想和你聊一个话题，关于[具体事件/行为]。[描述观察到的行为，而不是评判]。这样做可能会导致[影响]。你对这个情况是怎么考虑的？[倾听回应后]我们一起来想想可以怎么改进。"

## 五、不同场景的话术

### 绩效问题

"我想和你聊聊最近[项目/任务]的情况。我注意到[具体表现]，这对[影响]产生了一定影响。你觉得主要原因是什么？我想听听你的分析。"

### 职业发展

"你在[领域]做得很好，我注意到[具体例子]。关于你的长期发展，你有什么想法？有没有想尝试的方向？"

### 团队协作

"我收到反馈说[协作问题]。我不是来追究责任的，是想了解情况，看看我们能做什么改进。你能说说你的角度吗？"

## 六、沟通频率建议

| 员工类型 | 建议频率 |
|----------|----------|
| 新入职员工（前3个月） | 每周1次 |
| 一般员工 | 每2周1次 |
| 高潜力/高绩效员工 | 每月1次 |
| 远程办公员工 | 每月2次 |

## 七、常见误区

| 误区 | 正确做法 |
|------|----------|
| 变成汇报会 | 以员工为主导，多倾听 |
| 只聊工作 | 也关注个人发展和状态 |
| 批评多过鼓励 | 正向反馈为主 |
| 不做记录 | 记录要点和action，跟踪执行 |
| 间隔太长 | 保持规律沟通节奏 |
