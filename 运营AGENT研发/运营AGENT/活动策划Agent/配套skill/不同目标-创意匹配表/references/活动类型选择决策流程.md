# 活动类型选择决策流程

## 决策流程说明

本流程用于帮助在多个活动类型之间做出选择，确保决策科学合理。

---

## 决策流程图

```
开始选择
  ↓
Step 1: 明确活动目标
  ↓

  ↓

  ↓

  ↓

  ↓

```

| PK排名 | 竞争激励 | 参与强 | 过度竞争 | 5-15万 |
| 竞猜预测 | 话题互动 | 话题强 | 依赖热点 | 3-8万 |
| 征集投稿 | UGC产出 | 内容多 | 质量难控 | 2-10万 |
| 收集兑换 | 稀缺驱动 | 传播强 | 规则复杂 | 5-20万 |
| 小游戏 | 沉浸有趣 | K值高 | 开发成本高 | 10-30万 |
