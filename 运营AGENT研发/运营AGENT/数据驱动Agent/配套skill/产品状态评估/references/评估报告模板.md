# 产品状态评估报告模板

## 使用说明

本模板适用于所有类型的产品状态评估（B2C、B2B）。评估人员必须逐项填写，不得省略任何章节。

| 评估周期 | 数据周期，格式：YYYY-MM 至 YYYY-MM | 2024-01 至 2024-03 |
| 评估目的 | 投入决策 / 问题排查 / 常规体检 | 投入决策 |

| 评估人员 | 执行评估的人或Agent | 产品运营Agent |

### 健康等级判定

**判定结果**：
```
□ 健康（≥85分，且无非红维度，无系统性风险）
□ 亚健康（70-84分，或有≤2个非核心维度红）
□ 风险（55-69分，或≥1个核心维度长期红）
□ 危机（<55分，或≥3个维度红，或系统性崩溃）
```
