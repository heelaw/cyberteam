# 活动复盘案例库

## 1. 这份文件解决什么

这份文件只解决一个问题：让新人知道什么样的活动算好，什么样的活动算坏，什么样的活动看起来热闹但其实不值得复用。

它不是概念补充，而是给 `活动效果评估` 提供可以直接对照的样本。

## 2. 先怎么读

拿到活动评估任务时，先按这个顺序看：

1. 活动目标是不是量化的。
2. 这次活动是拉新、促活、转化、品牌、复购，还是裂变。
3. 有没有目标基准、历史基准、行业基准。
4. 是哪个环节先掉的：曝光、点击、参与、完成，还是转化。
5. 结论是“可复用”，还是“必须停止”。

如果这五件事没看清，就不要急着写综合判断。

## 3. 样本分层总表

| 样本类型 | 典型表现 | 正确结论 | 不能犯的错 |
|---|---|---|---|
| 健康样本 | 目标完成、成本合理、传播/行为无明显短板 | 可复用，可扩张 | 不能只看 GMV |
| 亚健康样本 | 目标完成，但某一维明显拖后腿 | 继续做，但先修薄点 | 不能当健康样本 |
| 风险样本 | 关键环节掉得明显，ROI 或 CAC 失控 | 控损、收缩、修复 | 不能继续平均用力 |
| 危机样本 | 目标未完成且成本失衡 | 立即止损 | 不能用“有热度”掩盖亏损 |

## 4. 典型案例

### 4.1 拉新型活动：结果不错但要看质量

**信号**

- 新增用户数完成。
- 首单转化率不低。
- CAC 没明显超线。
- 30 日留存没有断崖。

**正确结论**

- 这类活动可以复用，但要保留承接页和后续运营动作。

**什么算好**

- 好不是“注册很多”，而是“注册后还能继续留下来”。

**什么绝对不能**

- 不能把注册数当成功。
- 不能用低价激励堆出假繁荣。

### 4.2 促活型活动：参与高但回访弱

**信号**

- 参与率高。
- 当天热闹，但 7 天后回访没起色。
- 核心功能使用没有提升。

**正确结论**

- 促活做到了“来一趟”，但没做到“留下来”。

**什么算好**

- 好是让用户回到核心功能，而不是只来领一次奖励。

**什么绝对不能**

- 不能只看活动当天 UV。
- 不能把短期热闹当长期留存。

### 4.3 转化型活动：入口好，承接差

**信号**

- CTR 高。
- 访问量大。
- 但下单率、支付率、成交率低。

**正确结论**

- 入口素材有效，承接页或转化链路有问题。

**什么算好**

- 好是“点进来以后还能继续往下走”。

**什么绝对不能**

- 不能只夸素材。
- 不能把点击当成转化。

### 4.4 品牌型活动：热度高但业务没接住

**信号**

- 曝光大。
- 话题热。
- 但实际转化不明显。

**正确结论**

- 品牌声量有了，但业务链路没有接住。

**什么算好**

- 好是品牌和转化两头都能接上。

**什么绝对不能**

- 不能只讲声量，不讲业务结果。

### 4.5 裂变型活动：传播强但质量差

**信号**

- 分享率高。
- 邀请数多。
- 但二次用户质量弱，后续转化差。

**正确结论**

- 传播机制成功了，但用户质量没有跟上。

**什么算好**

- 好是传播带来的新用户还能形成有效价值。

**什么绝对不能**

- 不能把 K 因子高直接等同于活动成功。

### 4.6 亏损样本：热闹但必须停

**信号**

- GMV 看起来大。
- 成本更大。
- ROI < 0。
- 补贴占比异常高。

**正确结论**

- 这是亏损活动，必须止损。

**什么算好**

- 好是能老老实实指出亏损，而不是用“声量不错”粉饰。

**什么绝对不能**

- 不能把亏损活动写成成功案例。
- 不能只看热度不看成本。

## 5. 反例库

| 反例 | 为什么错 | 正确说法 |
|---|---|---|
| 只报曝光 | 曝光不等于效果 | 要看点击、参与、转化 |
| 只报注册 | 注册不等于有效用户 | 要看首单和留存 |
| 只报 GMV | GMV 不等于赚钱 | 要看成本和 ROI |
| 只报分享 | 分享不等于传播价值 | 要看分享后的转化质量 |
| 只报“总体不错” | 没有边界和动作 | 要说哪里好、哪里坏、下一步做什么 |

## 6. 新人判断口诀

1. 先看目标。
2. 再看基准。
3. 再看漏斗。
4. 再看成本。
5. 再看能不能复用。

如果最后说不清“为什么好、为什么坏、哪里要停”，就说明还没看懂。
