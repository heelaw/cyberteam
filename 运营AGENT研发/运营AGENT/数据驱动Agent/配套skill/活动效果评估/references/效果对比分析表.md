# 效果对比分析表

## 使用说明

本模板用于横向对比多个活动的效果，支持：
1. 同类型活动横向对比
2. 环比分析（本期vs上期）
3. 同比分析（本期vs去年同期）
4. 渠道效果对比
5. 问题发现对比

**强制要求：**
- 对比时必须说明数据来源和时间范围
- 对比判断必须基于数据，不得主观臆断
- 异常数据必须标注并排查原因
