# 评估维度详解

## 活动效果四维度评估法

### 先定规则

活动效果评估不是“看热闹”，而是回答四个问题：

1. 这次活动有没有达成目标。
2. 价值来自哪一段链路。
3. 成本浪费在哪一段。
4. 哪些打法值得复用，哪些必须停止。

如果没有目标、基准和完整链路，先补数据，不要直接给好坏。

### 一、先判断活动类型

不同活动先看不同主目标：

| 活动类型 | 主目标 | 核心观察 |
|---|---|---|
| 拉新活动 | 新增、注册、首单 | 新客是否真新、是否有效 |
| 促活活动 | 参与、回访、留存 | 是否把沉默用户拉回来 |
| 转化活动 | 点击、访问、下单、支付 | 链路是否顺滑，承接是否有效 |
| 品牌活动 | 曝光、互动、分享、记忆 | 是否形成传播和认知 |
| 复购活动 | 复购、频次、LTV | 是否把一次成交变成多次成交 |

### 二、四维度评估的真正目的

#### 2.1 产出指标

产出指标的作用，是确认活动到底有没有把业务结果做出来。

必须拆成三层：

1. 拉新结果。
2. 转化结果。
3. 营收结果。

重点判断：

- 目标人群是不是对的。
- 活动结果是不是实的。
- 有没有把“看起来很多”变成“真正有价值”。

常见误判：

- 只看注册数，不看首单。
- 只看首单，不看用户质量。
- 只看 GMV，不看毛利和补贴。

#### 2.2 成本指标

成本指标的作用，是确认结果是不是买来的、买得值不值。

必须把成本拆开：

- 投放成本。
- 运营成本。
- 人力成本。
- 技术成本。
- 补贴成本。

判断时先看结构，再看总量：

- 如果投放占比过高，说明买量依赖强。
- 如果补贴占比过高，说明刺激重、可持续性弱。
- 如果人力或技术成本偏高，说明流程太重。

#### 2.3 传播指标

传播指标的作用，是看活动是否具备自增长能力。

传播漏斗固定看：

曝光 → 点击 → 访问 → 参与 → 分享 → 二次传播

其中最重要的不是总曝光，而是：

- CTR 是否匹配入口质量。
- 访问到参与是否有明显断点。
- 参与到分享是否说明活动有传播价值。

#### 2.4 用户行为指标

用户行为指标的作用，是看用户有没有真的参与，而不是只路过。

必须观察：

- 是否看懂规则。
- 是否愿意继续往下走。
- 是否在关键步骤掉得很厉害。
- 是否出现“参与了但没完成”。

### 三、判断链路

活动评估必须按这个顺序走：

1. 先看目标是否完成。
2. 再看成本是否失控。
3. 再看传播是否可复用。
4. 再看用户体验是否影响下一次复投。

不要先写结论，再回头找数据凑理由。

### 四、典型判断模板

#### 4.1 好活动的特征

- 目标完成。
- ROI 清楚。
- 传播链路顺。
- 用户愿意复用。
- 有明确的复投依据。

#### 4.2 差活动的特征

- 结果靠补贴堆出来。
- 成本高但链路弱。
- 曝光高，转化差。
- 用户只薅一次就走。
- 复用后容易失效。

#### 4.3 需要谨慎的情况

- 样本太小。
- 周期太短。
- 数据延迟。
- 成本口径不全。
- 指标只涨一个维度。

### 五、复用判断

复用不是“好看就复用”，而是要回答：

1. 哪个钩子有效。
2. 哪个渠道有效。
3. 哪个路径最短。
4. 哪个环节流失最高。
5. 哪些做法下次直接复用。
6. 哪些做法下次直接停止。

### 六、失败案例

#### 案例 1：只看 GMV

活动 GMV 上升，但补贴也大幅上升。结果看起来好，实际上 ROI 很差。正确做法是同时看成本结构和有效获客数。

#### 案例 2：只看曝光

活动曝光很高，但点击和参与都低，说明入口吸引人但承接差。正确做法是拆漏斗，不要只报曝光。

#### 案例 3：只看注册

注册量高，但首单转化很低，说明新客质量差。正确做法是看新客质量系数和首单转化。

### 七、输出要求

活动输出必须包含：

- 任务定义。
- 对比基准。
- 四维度判断。
- 综合评级。
- 复用经验。
- 停止项。
- 下一步动作。

### 八、绝对不能

- 不能只报喜不报忧。
- 不能只看最终 GMV 不看成本结构。
- 不能把估算值当成真实值。
- 不能没有对比基准就给评级。
- 不能只写“建议继续优化”。
- 不能把活动复盘写成故事会。

## 九、新人执行顺序

1. 先拿活动目标和活动类型。
2. 再确认成本口径和产出口径。
3. 再补传播与行为链路。
4. 再做目标 / 历史 / 行业对比。
5. 最后才写评级、复用项和停止项。

## 十、评估前 QA

- 是否至少有一个量化目标。
- 是否知道活动属于哪一类。
- 是否能计算完成率或 ROI。
- 是否知道缺哪一项数据会改变结论。
- 是否能给出一个必须停止的做法。
