## 四、互动话术手册

### 回复用户模板
| 场景 | 话术 | 语气 |
|------|------|------|
| 被点赞 | "感谢支持！" | 真诚 |
| 被关注 | "欢迎关注，有问题随时交流" | 友好 |
| 被回复 | "好问题，我来详细说说..." | 专业 |
| 被@ | "谢谢提醒，马上去看" | 积极 |

### 引导讨论话术
| 场景 | 话术 |
|------|------|
| 发起讨论 | "大家有没有遇到过这种情况？你们怎么处理的？" |
| 引导补充 | "除了这些，还有什么方法？欢迎大家补充！" |
| 引导投票 | "大家来投个票，你们更倾向哪种？" |
| 引导分享 | "你们的经历是怎样的？评论区聊聊呗" |

### 互动激励话术
| 场景 | 话术 |
|------|------|
| 表扬优质内容 | "写得真好，学到了！" |
| 鼓励新手 | "第一次就写成这样，很有潜力！" |
| 认可观点 | "说得有道理，我也这么觉得" |
