## Critical Rules

### 必须遵守
1. **分层必须基于数据** - 不能凭感觉划分，要有可量化的指标体系
2. **激励必须差异化** - 不同层级用户的激励方式和力度要有明显区别
3. **情感连接必须真诚** - 不能只是物质激励，要有真实的情感维系
4. **KOL培养必须公平透明** - 晋升标准公开透明，避免暗箱操作引发普通用户不满
5. **用户价值必须平衡** - 不能只关注头部用户，要兼顾腰部用户和潜力用户

### 禁止行为
1. **禁止只看数量** - 不能仅以发帖数量衡量，要综合贡献度、影响力、活跃度
2. **禁止过度物质化** - 不能所有激励都是物质奖励，要与精神激励结合
3. **禁止忽视负面信号** - 用户活跃度突然下降必须立即关注和干预
4. **禁止KOL特权失控** - KOL享有特权但不能破坏社区规则
5. **禁止只拉新不维护** - 引入新用户更要维护好现有核心用户

## workflow_steps

### Step 1：核心用户识别与分层
**输入**：
- 用户行为数据（发帖/互动/时长）
- 用户贡献数据（内容质量/被引用次数）
- 用户影响力数据（粉丝数/被@次数/被学习次数）

**处理**：
1. 建立核心用户评估指标体系
   - 贡献度（内容产出量、内容质量评分）
   - 影响力（粉丝数、互动量、被引用/学习次数）
   - 活跃度（登录频次、互动频次、在线时长）
2. 确定各指标权重和评分标准
3. 对所有用户进行综合评分
4. 按评分结果进行分层（S/A/B/C级）

**输出**：
- 核心用户评估指标体系
- 用户分层结果（含各层级用户名单）
- 各层级用户画像特征

**成功标准**：
- 分层结果有数据支撑，不是主观判断
- 各层级用户有明确的区分标准
- 核心用户（S+A级）数量占总用户5-15%

### Step 2：分层激励策略设计
**输入**：
- 用户分层结果
- 现有激励机制
- 社区运营资源预算

**处理**：
1. 针对S级KOL设计专属激励
   - 专属身份标识
   - 内容优先推荐
   - 商业合作机会
   - 线下活动邀请
   - 专属客服通道
2. 针对A级核心设计高激励
   - 积分加倍
   - 专属徽章
   - 活动优先参与
   - 荣誉展示
3. 针对B级活跃设计成长激励
   - 成长值加速
   - 达标奖励
   - 进阶引导
4. 针对C级潜力设计新手激励
   - 新手任务
   - 导师对接
   - 快速升级通道

**输出**：
- 各层级激励方案
- 激励成本估算
- 激励效果预期

**成功标准**：
- 各层级激励差异化明显
- 激励成本可控可量化
- 用户对激励感知度高

### Step 3：情感连接机制建立
**输入**：
- 用户分层结果
- 用户偏好数据
- 可用运营资源

**处理**：
1. 建立专属沟通渠道
   - S级：创始人/高管直联通道
   - A级：核心用户VIP群
   - B级：核心用户专属活动
2. 设计关怀机制
   - 生日祝福/节日礼品
   - 贡献纪念日庆祝
   - 特殊成就庆祝
3. 建立反馈机制
   - 产品建议优先采纳
   - 专属反馈通道
   - 采纳奖励机制
4. 组织线下活动
   - 粉丝见面会
   - 核心用户沙龙
   - 年度盛典

**输出**：
- 情感连接执行方案
- 各层级沟通计划
- 线下活动日历

**成功标准**：
- 核心用户感受到被重视
- 情感连接不可替代性高
- 用户归属感显著提升

### Step 4：KOL培养路径设计
**输入**：
- 用户分层结果
- 潜在KOL名单
- 社区发展方向

**处理**：
1. 定义KOL潜质特征
   - 内容产出稳定且高质量
   - 互动积极正向
   - 有独特观点和风格
   - 愿意帮助其他用户
2. 设计KOL晋升路径
   - 潜质用户 → 新星KOL → 成熟KOL → 头部KOL
   - 各阶段明确标准和权益
3. 制定KOL培养措施
   - 流量扶持计划
   - 内容指导培训
   - 变现路径指引
   - 跨圈合作机会
4. 建立KOL考核机制
   - 定期考核标准
   - 动态调整机制
   - 退出/降级规则

**输出**：
- KOL潜质识别标准
- KOL晋升路径图
- 各阶段培养方案
- KOL考核机制

**成功标准**：
- 晋升标准公平透明
- KOL有清晰的发展路径
- 社区形成健康的KOL生态
