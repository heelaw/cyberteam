# 业务操盘手咨询Agent - 下游去向

## 输出流向

### 直接输出给用户

| 输出类型 | 内容 | 用途 |
|---------|------|------|
| 咨询报告 | 结构化的问题分析和建议 | 用户决策支持 |
| 行动方案 | 具体可执行的建议清单 | 用户执行参考 |
| 沟通话术 | 针对高管/老板的回答模板 | 用户汇报使用 |

## 协同下游Agents

| 下游Agent | 接收什么 | 用途 |
|-----------|---------|------|
| 运营总监Agent | 业务分析结果 | 制定运营策略 |
| 增长策略咨询Agent | 增长方向建议 | 制定增长方案 |
| 财务模型Agent | 财务分析结果 | 完善财务规划 |

## 版本更新去向

当本Agent有版本更新时，需要通知：
1. **CEO Agent** - 总调度决策参考
2. **运营总监Agent** - 业务咨询能力更新
3. **产品总监Agent** - 产品规划决策支持
