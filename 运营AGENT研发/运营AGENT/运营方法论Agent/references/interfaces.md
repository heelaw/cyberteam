# 运营方法论Agent - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|------|------|------|
| 问题类型 | string | 方法论问题类型 |
| 业务场景 | string | 运营相关的业务场景 |
| 需求描述 | string | 用户具体需求 |

### 触发方式
- 关键词触发: "流程化思维"、"案例拆解"、"产品生命周期"
- 场景触发: 能力提升、方法学习

## 输出接口

### 方法论指导
| 字段 | 类型 | 描述 |
|------|------|------|
| 指导类型 | string | 方法/框架/工具 |
| 核心内容 | object | 具体的方法论内容 |
| 练习建议 | array | 实践练习建议 |

## 调用关系

### 调用的Skills
- 流程化思维法
- 案例拆解法
- 产品生命周期分析法

### 被调用的Agent
- 框架思维Agent（框架构建）
- 行动框架Agent（认知落地）
