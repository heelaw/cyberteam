# 内容案例五维度拆解框架

## 框架说明
内容案例拆解的核心框架，适用于文章、短视频、直播等多种内容形态。

---

## 维度一：选题策划

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| 热点追踪 | 如何借势热点 | 热点选择、切入角度 |
| 用户需求 | 用户想看什么 | 痛点、痒点、爽点 |
| 差异化角度 | 如何与众不同 | 独特视角、另辟蹊径 |
| 时机把控 | 什么时候发 | 发布时机、发酵周期 |

### 热点类型及价值

| 热点类型 | 举例 | 价值 | 时效性 |
|----------|------|------|--------|
| 突发热点 | 明星塌房、政策出台 | 流量爆发 | 极短 |
| 节点热点 | 节假日、行业峰会 | 确定性强 | 中等 |
| 常态热点 | 周一焦虑、周五憧憬 | 持续需求 | 长期 |
| 预测热点 | 高考、618 | 可准备 | 提前规划 |

### 选题策划检查

- [ ] 选题是否匹配目标用户需求
- [ ] 是否有独特切入角度
- [ ] 时机是否合适
- [ ] 是否有延展性（系列内容可能）

---

## 维度二：标题设计

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| 标题公式 | 使用了什么结构 | 数字型、疑问型、对比型 |
| 悬念设置 | 如何吸引点击 | 留白、冲突、未知 |
| 情绪调动 | 如何引起共鸣 | 焦虑、共情、愤怒 |
| 平台适配 | 是否符合平台调性 | 抖音/微信/小红书风格 |

### 爆款标题公式

| 公式类型 | 公式 | 示例 |
|----------|------|------|
| 数字型 | 数字+价值 | "3个技巧，让用户主动转介绍" |
| 疑问型 | 痛点+疑问 | "为什么你的活动没人参加？" |
| 对比型 | 冲突+反差 | "月薪3000和30000的运营差在哪" |
| 限时型 | 紧迫+稀缺 | "今晚12点涨价，错过等一年" |
| 共鸣型 | 情绪+认同 | "终于有人说实话了" |

### 标题设计检查

- [ ] 前3秒能抓住注意力
- [ ] 传递了核心价值
- [ ] 激发了点击欲望
- [ ] 符合平台风格

## 维度三：内容结构

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| 框架逻辑 | 如何组织信息 | 总分总、清单体、故事线 |
| 信息密度 | 内容充实程度 | 干货量、知识点 |
| 阅读节奏 | 读起来是否顺畅 | 段落长短、配图节奏 |
| 开篇设计 | 如何留住用户 | 钩子、前置价值 |

### 常见内容结构

| 结构类型 | 适用场景 | 特点 |
|----------|----------|------|
| 总分总 | 深度长文 | 结论先行，层层展开 |
| 清单体 | 干货分享 | 列表形式，便于阅读 |
| 故事线 | 案例分享 | 情节推进，引人入胜 |
| 对比型 | 测评攻略 | 正反对比，结论清晰 |
| 递进型 | 方法论 | 层层递进，逻辑严密 |

### 内容结构检查

- [ ] 开篇能否留住用户
- [ ] 框架逻辑是否清晰
- [ ] 信息密度是否适中
- [ ] 结尾是否有行动引导

---

## 维度四：转化设计

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| CTA位置 | 什么时候引导 | 开篇、中间、结尾 |
| 引导话术 | 如何让用户行动 | 行动指令清晰度 |
| 信任背书 | 如何增强可信度 | 数据、权威、案例 |
| 障碍消除 | 如何降低行动门槛 | 顾虑解答、简化操作 |

### CTA设计要点

| 要素 | 说明 | 示例 |
|------|------|------|
| 行动指令 | 明确告诉用户做什么 | "点击领取"、"立即报名" |
| 价值说明 | 行动能得到什么 | "免费获取运营地图" |
| 紧迫性 | 为什么要现在 | "限时免费"、"仅限前100名" |
| 便捷性 | 行动是否容易 | 一键领取、无需填写 |

### 转化设计检查

- [ ] CTA是否清晰醒目
- [ ] 价值说明是否有力
- [ ] 信任背书是否足够
- [ ] 行动门槛是否够低

---

## 维度五：数据表现

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| 曝光数据 | 内容被多少人看到 | 阅读量、播放量 |
| 互动数据 | 用户是否参与 | 点赞、评论、收藏 |
| 转化数据 | 用户是否行动 | 链接点击、私信、成交 |
| 传播数据 | 用户是否愿意分享 | 转发率、分享渠道 |

### 核心数据指标

| 指标类型 | 指标名称 | 行业参考值 |
|----------|----------|------------|
| 曝光 | 阅读量/播放量 | - |
| 互动 | 点赞率 | 3-5% |
| 互动 | 评论率 | 0.5-1% |
| 互动 | 收藏率 | 2-3% |
| 转化 | 链接点击率 | 1-3% |
| 传播 | 转发率 | 1-2% |

### 数据分析检查

- [ ] 数据是否达到预期
- [ ] 各环节转化是否正常
- [ ] 异常数据是否有解释
- [ ] 成功因素是否可归因

---

## 五维度综合检查清单

- [ ] 选题策划：热点把握准、角度独特、时机合适
- [ ] 标题设计：吸引力强、价值明确、平台适配
- [ ] 内容结构：框架清晰、节奏合理、开篇留人
- [ ] 转化设计：CTA明确、话术有力、背书足够
- [ ] 数据表现：指标达成、归因清晰、可复用
