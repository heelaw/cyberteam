# 行动计划模板

## 模板说明

## 使用说明
1. 先回顾目标，确保行动计划与目标一致
2. 列出所有可能的行动项
3. 用优先级矩阵排序
4. 识别资源需求和风险
5. 设置里程碑检查点
6. 定义成功指标
