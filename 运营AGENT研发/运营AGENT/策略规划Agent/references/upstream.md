# 策略规划Agent - 上游依赖

## 上游Agent（输入来源）

### 直接上游

| Agent名称 | 提供内容 | 说明 |
|-----------|----------|------|
| 全局业务模型构建器 | 业务模型结构、瓶颈环节 | 遇到增长瓶颈时 |
| 增长业务模型规划器 | 当前增长策略、目标、差距 | 需要新增长策略时 |
| 用户直接输入 | 业务问题描述、规划需求 | 用户主动发起时 |

### 间接上游

| Agent名称 | 提供内容 | 说明 |
|-----------|----------|------|
| 老板关注点分析器 | 老板关注的核心指标 | 通过全局业务模型构建器传递 |
| 业务模型Agent | 商业模式画布 | 通过全局业务模型构建器传递 |

## 依赖关系

```
老板关注点分析器 → 全局业务模型构建器 → 策略规划Agent
                                                  ↓
增长业务模型规划器 ──────────────────────────────→│
                                                  ↓
框架思维Agent ──────────────────────────────────→│
                                                  ↓
                                          策略执行Agent
```

## 依赖的外部资源

- 用户提供的业务背景信息
- 行业分析数据
- 竞争态势信息
- 历史规划方案（参考用）
