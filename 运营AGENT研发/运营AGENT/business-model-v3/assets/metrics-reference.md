# 常用运营指标参考

## 获客指标

### CAC (Customer Acquisition Cost)
**客户获取成本**

```
公式: CAC = 总获客成本 / 新增客户数

总获客成本包括:
- 营销广告费
- 销售团队佣金
- 营销工具费用
- 其他相关费用

基准:
- SaaS: $50-500
- 电商: $10-100
- O2O: $20-200
```

### 转化率
**Conversion Rate**

```
公式: 转化率 = 转化用户数 / 曝光用户数 × 100%

类型:
- 点击转化率 (CTR): 点击/曝光
- 注册转化率: 注册/访问
- 付费转化率: 付费/活跃

基准:
- 点击率: 1-3%
- 注册率: 5-15%
- 付费率: 2-10%
```

## 留存指标

### 留存率
**Retention Rate**

```
公式: 留存率 = 留存用户数 / 初始用户数 × 100%

类型:
- 次日留存 (D1)
- 7日留存 (D7)
- 30日留存 (D30)
- 90日留存 (D90)

基准:
- D1: > 40% (优秀), > 25% (良好)
- D7: > 20% (优秀), > 10% (良好)
- D30: > 10% (优秀), > 5% (良好)
- D90: > 5% (优秀), > 2% (良好)
```

### 流失率
**Churn Rate**

```
公式: 流失率 = 流失用户数 / 期初用户数 × 100%

类型:
- 月流失率
- 年流失率

基准:
- 月流失率: < 5% (优秀), < 10% (良好)
- 年流失率: < 20% (优秀), < 30% (良好)
```

## 收入指标

### ARPU (Average Revenue Per User)
**平均用户收入**

```
公式: ARPU = 总收入 / 活跃用户数

基准:
- 电商: $20-200/月
- SaaS: $10-100/月
- 内容平台: $1-10/月
```

### ARPPU (Average Revenue Per Paying User)
**平均付费用户收入**

```
公式: ARPPU = 总收入 / 付费用户数

基准:
- 电商: $50-500/月
- SaaS: $50-500/月
- 内容平台: $10-100/月
```

### LTV (Lifetime Value)
**客户生命周期价值**

```
简单公式: LTV = ARPU × 客户生命周期

精细公式: LTV = (ARPU × 毛利率) / 流失率

客户生命周期 = 1 / 流失率

基准:
- 电商: $100-1000
- SaaS: $500-5000
- O2O: $200-2000
```

## 效率指标

### LTV:CAC 比率
**客户价值与获客成本之比**

```
公式: LTV:CAC = LTV / CAC

基准:
- > 5:1 优秀，可以激进增长
- 3-5:1 健康，可持续增长
- 2-3:1 基本健康，增长受限
- < 2:1 不健康，需要优化

行业基准:
- SaaS: > 3:1
- 电商: > 3:1
- 移动应用: > 2:1
```

### 回本周期
**Payback Period**

```
公式: 回本周期 = CAC / (ARPU × 毛利率)

基准:
- < 6个月: 优秀
- 6-12个月: 良好
- 12-18个月: 需关注
- > 18个月: 不健康
```

### 毛利率
**Gross Margin**

```
公式: 毛利率 = (收入 - 销货成本) / 收入 × 100%

基准:
- SaaS: 70-90%
- 电商: 20-50%
- 数字产品: 80-95%
- 服务行业: 30-60%
```

## 增长指标

### 增长率
**Growth Rate**

```
公式: 增长率 = (本期数值 - 上期数值) / 上期数值 × 100%

类型:
- 用户增长率
- 收入增长率
- 流量增长率

基准:
- 早期阶段: > 20% 月增长
- 成长阶段: > 10% 月增长
- 成熟阶段: > 5% 月增长
```

### K因子
**病毒系数**

```
公式: K = 平均邀请数 × 邀请转化率

基准:
- K > 1: 病毒式增长
- K = 1: 自维持增长
- K < 1: 需要其他获客方式
```

## 活跃度指标

### DAU/MAU
**日活跃用户与月活跃用户比率**

```
公式: DAU/MAU = 日活跃用户 / 月活跃用户

基准:
- > 50%: 高粘性 (必需品)
- 20-50%: 中等粘性
- < 20%: 低粘性
```

### 使用频率
**Usage Frequency**

```
类型:
- 每日使用: 社交、通讯、新闻
- 每周使用: 娱乐、购物
- 每月使用: 旅游、教育
```

## 行业特定指标

### SaaS
- **MRR (Monthly Recurring Revenue)**: 月经常性收入
- **ARR (Annual Recurring Revenue)**: 年经常性收入
- **MRR Churn**: 月收入流失率
- **ARPU (Average Revenue Per Account)**: 平均账户收入

### 电商
- **AOV (Average Order Value)**: 平均订单金额
- **复购率**: 重复购买率
- **购物车放弃率**: Cart Abandonment Rate

### O2O
- **完播率**: 内容完成观看率
- **预约率**: 预约转化率
- **到店率**: 实际到店率
- **爽约率**: 预约未到店率

## 指标选择建议

### 根据业务阶段选择

**早期阶段**:
- 关注: 增长率、激活率、留存率
- 忽略: LTV:CAC、ROI

**成长阶段**:
- 关注: 增长率、LTV:CAC、留存率
- 监控: CAC、流失率

**成熟阶段**:
- 关注: LTV:CAC、毛利率、利润率
- 优化: 运营效率、成本控制

### 根据商业模式选择

**订阅制**:
- 核心: MRR/ARR、Churn Rate、LTV:CAC
- 次要: ARPU、活跃度

**交易制**:
- 核心: AOV、复购率、转化率
- 次要: CAC、LTV

**广告制**:
- 核心: DAU、使用时长、广告填充率
- 次要: 留存率、点击率
