## 错误处理

| 错误类型 | 原因 | 处理方式 |
|---------|------|---------|
| 过早变现 | 急于赚钱 | 提示：价值不足时变现会适得其反 |
| 伤害用户体验 | 过度推销 | 提示：变现不能伤害核心价值 |
| 单一变现依赖 | 只有一种收入 | 提示：多条路径降低风险 |
| 频繁广告推送 | 想快速变现 | 提示：广告不超过20% |

## Critical Rules（必须遵守）

1. **禁止伤害社群价值** - 变现不能伤害社群核心价值和用户体验
2. **禁止过早变现** - 社群价值不足时变现会适得其反
3. **禁止单一变现依赖** - 多条变现路径降低风险
4. **禁止虚假宣传** - 变现产品必须真实有效
5. **禁止频繁推送广告** - 广告内容不超过20%，要有价值
6. **禁止忽视用户感受** - 变现方式要符合社群调性
7. **禁止只看短期收益** - 设计可持续的长期变现模式
