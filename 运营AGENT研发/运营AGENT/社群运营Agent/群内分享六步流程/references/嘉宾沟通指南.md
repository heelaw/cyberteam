# 分享嘉宾沟通指南

## 邀请阶段

### 邀请话术

```
你好！我是[社群名称]的群主[昵称]

我们社群是一个[X人]的[定位]社群
主要成员是[目标用户]

看到你在[领域]有丰富经验
想邀请你来做一次分享

分享主题可以是：
- [建议主题1]
- [建议主题2]

时间大约[X]分钟
我们会提前宣传，也会有互动答疑

有兴趣的话可以进一步沟通～
```

### 拒绝应对

```
【嘉宾说没时间】
→ 确认其他可行时间
"理解您近期忙，方便的话可以选[备选时间]吗？"

【嘉宾说不太擅长】
→ 提供支持
"完全理解，我们可以帮您准备内容大纲，分享您的真实经验即可"

【嘉宾说收费】
→ 根据预算评估
"感谢您的坦诚，我们的预算在[X]以内，不知道是否合适？"

【嘉宾婉拒】
→ 保持关系
"完全理解，期待以后有机会合作"
```

## 确认阶段

### 确认话术

```
非常感谢您答应来分享！

【确认信息】
📅 日期：[具体日期]
🕐 时间：[具体时间]
⏱️ 时长：[X]分钟分享 + [X]分钟答疑

📍 地点：微信群（语音/文字）

【分享大纲】
[确认后的主题和大纲]

【准备工作】
- 分享形式：[语音/文字/PPT]
- 需要提前准备的材料：[清单]

【注意事项】
- 分享前我会发预告宣传
- 分享中我会主持流程
- 会有助理帮忙整理问题

请问我可以提前一天再和您确认一次吗？
```

## 准备阶段

### 准备清单

```
【嘉宾准备】
□ 确认分享内容大纲
□ 准备PPT（如需要）
□ 准备互动问题
□ 提前测试网络/设备

【群主准备】
□ 发布预告（提前3-5天）
□ 收集参与者问题（提前1天）
□ 确认嘉宾准备状态
□ 发送流程提醒（提前1小时）
□ 提前30分钟暖场准备
```

### 准备跟进话术

```
【提前1天】
@嘉宾 明天[时间]就是分享啦～
请问准备得怎么样？有什么需要我配合的吗？

【提前1小时】
@嘉宾 分享将在1小时后开始
请确保网络/设备正常
我会在[时间]@您开始

期待您的分享！
```

## 感谢阶段

### 感谢话术

```
【分享结束后】
@嘉宾 感谢您今天的精彩分享！[具体亮点]对我启发很大
辛苦了，好好休息～

【发送总结后】
@嘉宾 分享总结已发出，反响很好！
大家评价说"[精选反馈]"
感谢您的付出，下次有机会再合作！

【长期维护】
- 邀请成为常驻嘉宾
- 推荐其他嘉宾
- 节日问候
```
