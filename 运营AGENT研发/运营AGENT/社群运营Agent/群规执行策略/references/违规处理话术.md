# 违规处理话术手册

## 轻度违规处理话术

### 私下提醒（推荐）

```
【场景】：首次发现无关链接

@用户 打扰一下～ 群规禁止发[具体内容]哦
可能你没注意到，提醒一下，谢谢配合～

如果用户配合回复：
好的，谢谢理解～ 欢迎多分享[社群主题]相关内容
```

### 群内轻提示

```
【场景】：用户没注意到群规

@用户 温馨提醒：大家发[投票/问卷/链接]之前
可以先@群主确认一下是否合适～ 避免违规哦

或者：

建议大家[相关内容]可以先私信群主审核～ 谢谢配合
```

## 重度违规处理话术

### 正式警告（首次重度违规）

```
【场景】：发广告/推广链接

@用户 [用户昵称] 发广告违反了我们的群规
群规明确规定：禁止发推广链接/广告
这是第一次警告，下次将直接移出社群

请删除相关内容：________________
配合删除后本次警告生效
谢谢理解～
```

### 警告后跟进

```
【场景】：用户已删除内容

@用户 谢谢配合删除～
欢迎多分享[社群主题]相关内容

【场景】：用户不配合删除

@用户 请配合删除相关内容
否则将直接移出社群
```

### 私下警告（如果不想公开）

```
【私信】：你好，[违规行为]违反了群规
这是第一次警告，[具体后果]
请注意，谢谢配合

如不配合：
抱歉，多次违反群规，已被移出社群
如有疑问可联系群主
```

## 严重违规处理话术

### 直接移出（无需多言）

```
操作：直接移出，无需提前通知

移出后私下说明（可选）：
[用户昵称]，抱歉，因[具体严重违规行为]
你已被移出[社群名称]
[具体原因说明，如：发布虚假信息造成其他成员损失]

如有异议可联系群主申诉
```

### 群内说明

```
@所有人 处理提醒
[用户昵称] 因违反群规（[具体原因]）
已被移出社群
请大家共同维护社群秩序～

感谢理解
```

## 特殊场景话术

### 用户说"我不知道"

```
【用户反应】："我不知道这个不能做"

回复：
抱歉，群规在入群前已发送
也发布在群公告中
请先了解群规：[简要重复核心规则]

如需要可发送历史记录截图证明
```

### 用户求情

```
【用户反应】："能不能给一次机会"

回复（轻度违规）：
考虑到是第一次，这次给你机会
请删除相关内容，下次不会宽恕

回复（重度/严重违规）：
抱歉，规则面前人人平等
希望你能理解，下次入群请先了解群规
```

### 用户狡辩

```
【用户反应】："这个不算广告吧"

回复：
[具体判断依据]
根据群规[第X条]：[具体规则描述]
这个属于[违规类型]

请删除，谢谢配合
```

### 用户威胁退群

```
【用户反应】："我不发了还不行吗，大不了退群"

回复：
理解你的心情
如果你觉得社群不适合，可以随时退出
但请了解，群规是为了维护大家共同的交流环境

如果你愿意遵守群规，欢迎继续留在群里
如果你选择退出，祝你找到更适合的社群
```

### 群友起哄

```
【场景】：处理时有人起哄

不理会，继续正常运营

如需要：
@所有人 大家继续讨论[转移话题]
感谢理解～

如持续起哄，私下提醒：
[用户]，请不要讨论本次处理
让我们继续[原话题]
```

## 话术使用注意事项

1. **态度坚定**: 话术要明确表达规则立场
2. **语气友好**: 即使处理严重违规，也要保持基本尊重
3. **简洁明了**: 不要长篇大论，点到为止
4. **留有记录**: 重要处理要保留聊天记录
5. **一视同仁**: 无论谁违规，话术态度一致
