# 快速趟路行动计划模板

## 模板说明

此模板用于类型二项目（结构性机会快速趟路）的行动规划。核心方法是速度优先，在窗口期内完成验证和占领。

## 核心原则

1. **窗口意识** - 机会窗口是有限的，错过就是错过
2. **速度优先** - 天下武功唯快不破，过度准备是最大的浪费
3. **MVP先行** - 先做一个能跑的最小版本
4. **快速迭代** - 根据市场反馈快速调整

## 快速趟路项目检查清单

在每个关键节点检查：

**启动前**：
- [ ] 机会窗口是否明确？
- [ ] MVP是否足够小？
- [ ] 验证方案是否快速？

**验证中**：
- [ ] 数据是否在收集？
- [ ] 反馈是否在分析？
- [ ] 迭代是否在进行？

**决策点**：
- [ ] 是否达到成功标准？
- [ ] 是否应该加码/切换/终止？
- [ ] 决策依据是否充分？
