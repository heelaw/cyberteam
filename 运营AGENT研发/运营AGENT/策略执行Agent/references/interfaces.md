# 策略执行Agent - 接口协议

## 输入接口

### 用户输入

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| problem_description | string | 是 | 问题描述 |
| problem_type | string | 是 | 问题类型（A类/B类） |
| context | object | 否 | 背景上下文 |

### 来自其他Agent的输入

| 来源Agent | 传递内容 | 触发条件 |
|-----------|----------|----------|
| 策略规划Agent | 策略框架和执行要求 | 策略制定完成后 |
| 增长Agent | 增长目标和约束 | 增长项目启动时 |

## 输出接口

### 直接输出

| 字段 | 类型 | 说明 |
|------|------|------|
| diagnosis_report | object | 三要素诊断报告 |
| leverage_point | string | 识别的破局点 |
| execution_path | array | 执行路径规划 |
| action_plan | array | 详细行动计划 |

### 传递给其他Agent

| 目标Agent | 传递内容 | 传递条件 |
|-----------|----------|----------|
| 工作计划生成器 | 执行路径和行动计划 | 需要制定详细计划时 |
| 进度追踪器 | 里程碑和时间节点 | 需要跟踪进度时 |
