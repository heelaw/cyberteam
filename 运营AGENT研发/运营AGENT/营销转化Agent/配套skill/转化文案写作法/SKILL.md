## Critical Rules（必须遵守）

1. **禁止标题党** - 标题必须与内容一致，虚假标题伤害信任
2. **禁止空洞承诺** - 功效描述必须具体可感知
3. **禁止只讲产品** - 要讲用户能得到什么好处
4. **禁止复杂难懂** - 文案要通俗易懂，不要过度包装
5. **禁止模糊CTA** - 行动号召必须明确具体
6. **禁止忽视阅读场景** - 不同平台需要不同风格的文案
7. **禁止不测试** - 必须进行A/B测试优化文案
