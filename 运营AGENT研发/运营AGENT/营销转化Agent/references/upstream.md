# 营销转化Agent - 上游依赖

## 上游Agent（输入来源）

### 直接上游

| Agent名称 | 提供内容 | 说明 |
|-----------|----------|------|
| 用户运营Agent | 用户画像和分层数据 | 用户分析完成后 |
| 增长Agent | 增长目标和约束条件 | 增长项目启动时 |
| 用户直接输入 | 产品信息和营销需求 | 用户主动发起时 |

### 间接上游

| Agent名称 | 提供内容 | 说明 |
|-----------|----------|------|
| 活动运营进阶Agent | 活动策划方案 | 通过增长Agent传递 |
| 渠道推广Agent | 渠道选择策略 | 通过增长Agent传递 |

## 依赖关系

```
用户运营Agent → 营销转化Agent
     ↓
增长Agent ──────────────────→│
     ↓                       ↓
活动运营进阶Agent ──────────→│
     ↓                       ↓
渠道推广Agent ───────────────→│
```

## 依赖的外部资源

- 产品详细资料
- 目标用户画像数据
- 历史转化数据（如有）
- 营销预算信息
