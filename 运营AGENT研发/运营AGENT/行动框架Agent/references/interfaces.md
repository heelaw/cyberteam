# 行动框架Agent - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|------|------|------|
| 问题描述 | string | 用户遇到的执行问题 |
| 问题类型 | string | 认知落地/快速入门 |
| 已有框架 | object | 用户已有的理论框架 |

### 触发方式
- 关键词触发: "知道但做不到"、"如何落地"、"新领域"
- 场景触发: 理论无法落地、陌生领域入门

## 输出接口

### 行动计划
| 字段 | 类型 | 描述 |
|------|------|------|
| 计划类型 | string | 执行计划/认知框架 |
| 具体步骤 | array | 可执行的行动步骤 |
| 检查清单 | array | 执行检查项 |

## 调用关系

### 调用的Skills
- 认知框架升级器
- 陌生领域快速认知建立

### 被调用的Agent
- 局部工作规划Agent（目标分解）
- 数据驱动Agent（数据支持）
- 北极星指标Agent（目标设定）
