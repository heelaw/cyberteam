# 用户画像模板

## 基础信息

| 字段 | 内容 | 填写示例 |
|------|------|---------|
| 画像名称 | 便于识别的代号 | 小王 |
| 性别比例 | 男性/女性占比 | 60%女/40%男 |
| 年龄区间 | 主要年龄段 | 25-30岁 |
| 所在地区 | 地域分布 | 一线城市 |
| 职业背景 | 从事行业 | 互联网运营 |
| 收入水平 | 月薪范围 | 10-20K |

---

## 行为特征

| 字段 | 内容 | 填写示例 |
|------|------|---------|
| 活跃时间 | 什么时段活跃 | 21:00-23:00 |
| 内容偏好 | 常看的内容类型 | 干货、案例 |
| 常逛平台 | 使用的APP | 抖音、小红书 |
| 信息来源 | 了解新产品的渠道 | 朋友推荐、算法推荐 |
| 决策周期 | 从了解到购买时间 | 7-14天 |
| 价格敏感度 | 对价格的敏感程度 | 中等 |
| 购买渠道 | 在哪购买 | 京东、天猫 |

---

## 需求痛点

| 字段 | 内容 | 填写示例 |
|------|------|---------|
| 核心需求 | 最想解决的问题 | 快速提升运营能力 |
| 表面需求 | 用户自己说的需求 | 学更多运营技巧 |
| 深层需求 | 用户没说的需求 | 晋升加薪、职场竞争力 |
| 担心问题 | 购买前的顾虑 | 课程太理论、不实战 |
| 使用场景 | 什么情况会用 | 通勤地铁上、晚上回家后 |

---

## 心理特征

| 字段 | 内容 | 填写示例 |
|------|------|---------|
| 付费动机 | 为什么要付费 | 焦虑驱动、追求成长 |
| 决策顾虑 | 购买时担心什么 | 怕被割韭菜、怕没用 |
| 追求风格 | 喜欢什么风格 | 简洁、实用 |
| 从众心理 | 是否容易受他人影响 | 会看销量和评价 |
| 品牌偏好 | 对品牌的看法 | 迷信大牌/性价比优先 |

---

## 典型场景

### 场景1：发现产品

```
什么场景下接触到产品？
→ 在抖音刷到课程推荐视频
→ 视频讲的是同事做运营3年月薪3万的案例
```

### 场景2：了解产品

```
会做什么来了解产品？
→ 搜索课程评价
→ 看免费试听片段
→ 对比其他类似课程
```

### 场景3：决策购买

```
购买的关键决策点？
→ 试听后觉得老师讲得实战
→ 有学员案例证明效果
→ 价格在预算内
→ 有优惠活动
```

| 转化策略 | 限时优惠+班主任 |
