# 投放常见问题解答

## Q1: 第一次投放多少钱合适？

**建议**：
- 测试期：3000-5000元
- 验证后再加量

**原则**：
- 小预算测试
- 测试OK再放大

---

## Q2: 投放效果不好怎么办？

**排查顺序**：
1. 先看CTR → 素材问题？
2. 再看CVR → 落地页问题？
3. 最后看CPA → 成本问题？

**调整策略**：
| 问题 | 解决方案 |
|------|---------|
| CTR低 | 换素材开头、优化定向 |
| CVR低 | 优化落地页、调整价格 |
| CPA高 | 收定向、提高转化 |

---

## Q3: 哪些渠道适合新手？

**推荐新手渠道**：
1. 抖音 - 量大、门槛低
2. 微信朋友圈 - 精准、转化高

**谨慎选择**：
- 百度搜索 - 需要关键词优化经验
- 信息流以外形式 - 成本高

---

## Q4: 素材怎么做才能有效果？

**开头3秒原则**：
- 抓住注意力（痛点/悬念/反差）
- 展现价值（产品能解决什么问题）

**结构**：
```
开头（3秒）→ 痛点/悬念
中间（5秒）→ 产品展示/案例
结尾（2秒）→ CTA引导
```

---

## Q5: 定向怎么设置最合理？

**原则**：
- 先窄后宽：先精准测试，再放宽

**参考设置**：
```
第一轮测试：
- 年龄：核心年龄段
- 地域：一二线城市
- 兴趣：核心兴趣

第二轮优化：
- 保留转化好的
- 扩大相似人群
```

---

## Q6: 投放需要多长时间见效？

**时间节奏**：
- Day1-2：学习期，数据不稳定
- Day3-7：稳定期，可判断效果
- Day7后：优化期，持续调整

**耐心等待**：至少测试3天再判断

---

## Q7: 转化追踪怎么设置？

**必设事件**：
- 表单提交
- 按钮点击
- 购买成功

**设置方式**：
- 字节：巨量引擎后台
- 腾讯：广告后台+SDK
- 百度：百度统计

---

## Q8: 如何控制投放成本？

**方法**：
1. 设置预算上限
2. 控制出价
3. 精准定向
4. 优化素材

**红线**：CPA超过产品毛利必须停
