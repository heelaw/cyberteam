### 4.2 增长阶段判断

```
当前增长阶段：
[ ] 0→1 冷启动期：解决有没有的问题
[ ] 1→10 验证期：验证模式是否可行
[ ] 10→100 加速期：快速规模化
[ ] 100→N 成熟期：效率优化

阶段特征对照：
- 冷启动期：用户留存差，但这是正常的
- 验证期：留存改善，寻找可复制的增长模式
- 加速期：快速获客，牺牲短期利润
- 成熟期：精细化运营，提升效率
```
