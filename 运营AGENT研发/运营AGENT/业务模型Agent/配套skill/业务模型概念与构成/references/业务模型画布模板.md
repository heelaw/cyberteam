## 填写说明

### 1. 价值主张（Value Proposition）

**问题清单**：
- 我们为用户解决什么问题？
- 我们提供什么核心价值？
- 我们与竞争对手有什么不同？

**填写格式**：
```
核心价值主张：
1.
2.
3.

差异化优势：
1.
2.

用户收益：
1.
2.
```

---

### 2. 渠道触达（Channels）

**问题清单**：
- 如何触达我们的用户？
- 获客渠道有哪些？
- 转化路径是什么？

**填写格式**：
```
获客渠道：
1. 渠道名称：
   - 触达方式：
   - 成本：
   - 效果：

转化渠道：
1.
2.

留存渠道：
1.
2.
```

---

### 3. 用户群体（Customer Segments）

**问题清单**：
- 我们的核心用户是谁？
- 用户有什么特征？
- 用户需求是什么？

**填写格式**：
```
核心用户：
- 用户特征：
- 核心需求：
- 付费意愿：

潜在用户：
- 用户特征：
- 拓展难度：

用户分层：
| 层级 | 名称 | 占比 | 特征 |
|------|------|------|------|
| L1 |      |      |      |
| L2 |      |      |      |
| L3 |      |      |      |
```

### 4. 核心资源（Key Resources）

**问题清单**：

- 我们的核心能力是什么？

```

   - 作用：

2. 知识资源：
   - 资源名称：
   - 作用：

3. 人力资源：
   - 资源名称：
   - 作用：

核心能力：
1.
2.

资源壁垒：
1.
2.
```

### 5. 客户关系（Customer Relations）

**问题清单**：
- 如何与用户建立关系？
- 用户服务方式是什么？
- 如何提升用户忠诚度？

**填写格式**：
```
客户关系类型：
[ ] 专属客服    [ ] 自助服务    [ ] 社区共建
[ ] 会员体系    [ ] 社群运营    [ ] 内容运营

服务方式：
1.
2.

忠诚度策略：
1.
2.
```

**问题清单**：

```
|----------|----------|----------|----------|
|          |          |          |          |
|          |          |          |          |

```

### 7. 成本结构（Cost Structure）

**问题清单**：
- 主要成本是什么？
- 成本结构如何？
- 如何优化成本？

**填写格式**：
```
固定成本：

   - 金额/月：
   - 占比：

变动成本：
1. 成本项：
   - 金额/单：
   - 占比：

成本优化空间：
1.
2.
```

### 8. 收入模型（Revenue Model）

**问题清单**：
- 如何获得收入？
- 收入来源有哪些？
- 定价策略是什么？

**填写格式**：
```
收入来源：
| 来源 | 占比 | 金额 | 增长率 |
|------|------|------|--------|
|      |      |      |        |

- 策略类型：
- 价格区间：

盈利预测：
- 毛利率：
- 净利率：
- 盈亏平衡点：
```

## 填写示例：在线教育平台

```
┌───────────────────────┬─────────────────────┬───────────────────────┐
│ 价值主张              │ 渠道触达            │ 用户群体              │
│ - 名师授课            │ - 抖音/快手投放     │ - 25-35岁职场人群     │
│ - 随时学习           │ - 公众号引流        │ - 有技能提升需求       │
│ - 实战为主           │ - 社群裂变          │ - 付费意愿中等        │
├───────────────────────┼─────────────────────┼───────────────────────┤
│ 核心资源              │                     │ 客户关系              │
│ - 师资团队            │   价值定位          │ - 班主任全程服务      │
│ - 课程内容            │   "技能实战专家"    │ - 学员社群            │
│ - 技术平台            │                     │ - 作业点评服务        │
├───────────────────────┴─────────────────────┼───────────────────────┤
│ 合作伙伴                                     │ 成本结构              │
│ - 第三方平台                                 │ - 师资成本 40%        │
│ - 内容合作方                                 │ - 流量成本 30%        │
│ - 支付服务商                                 │ - 技术成本 20%        │
├─────────────────────────────────────────────┴───────────────────────┤
│ 收入模型                                                             │
│ - 课程销售收入 70%                                                  │
│ - 会员订阅收入 20%                                                  │
│ - 企业培训收入 10%                                                  │
└─────────────────────────────────────────────────────────────────────┘
```
