# IDP计划案例库

## 成功案例

### 案例1: 管理培训生IDP制定
**背景**: 管理培训生需要清晰的成长路径
**方法**: SMART目标+70-20-10法则
**发现**: 明确第1年达到独立项目管理能力
**结果**: 12个月后成功晋升项目主管
**适用场景**: 管培生、新员工发展

### 案例2: 技术专家职业发展规划
**背景**: 高级工程师缺乏成长方向
**方法**: 技术能力矩阵+职业通道设计
**发现**: 确定走技术专家路线
**结果**: 2年内成为首席工程师
**适用场景**: 技术人才、职业发展

### 案例3: 跨部门转岗IDP
**背景**: 员工希望从运营转产品
**方法**: 能力差距分析+学习路径规划
**发现**: 需补充用户研究、数据分析能力
**结果**: 6个月成功转岗产品经理
**适用场景**: 人才流动、跨职能发展

### 案例4: 中层管理者领导力发展
**背景**: 晋升为管理者后角色转变困难
**方法**: 领导力胜任模型+IDP制定
**发现**: 团队管理和战略思维是短板
**结果**: 参加领导力培训后成功转型
**适用场景**: 领导力发展、管理培训

### 案例5: 创业公司人才保留
**背景**: 核心员工有离职风险
**方法**: 个人IDP+职业通道+激励方案
**发现**: 成长空间和股权激励需求
**结果**: 核心员工保留率提升
**适用场景**: 人才保留、激励设计
