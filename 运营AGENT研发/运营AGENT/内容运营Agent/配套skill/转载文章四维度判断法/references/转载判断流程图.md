# 转载判断四维度流程图

## 快速判断决策树

```
收到待判断文章
        ↓
┌───────────────────────────────┐
│ Step 1: 原创性评估              │
│  ↓                             │
│ 是否标注"原创"？                 │
│   ↓是          ↓否             │
│ 需授权      ┌────────────┐     │
│   ↓        │ 通过原创性  │     │
│ 作者是否    │ 评估环节    │     │
│ 明确授权？  └────────────┘     │
└───────────────────────────────┘
        ↓
┌───────────────────────────────┐
│ Step 2: 相关性评估              │
│  ↓                             │
│ 文章主题与账号定位匹配度？        │
│  ↓                             │
│ 1-2分 → 淘汰                  │
│ 3-5分 → 进入下一步             │
└───────────────────────────────┘
        ↓
┌───────────────────────────────┐
│ Step 3: 质量评估               │
│  ↓                             │
│ 内容是否有干货价值？             │
│ 数据/案例是否可靠？             │
│  ↓                             │
│ 1-2分 → 淘汰                  │
│ 3-5分 → 进入下一步             │
└───────────────────────────────┘
        ↓
┌───────────────────────────────┐
│ Step 4: 版权评估               │
│  ↓                             │
│ 是否可获得授权？                │
│ 是否有图片/字体风险？           │
│  ↓                             │
│ 不可授权/有风险 → 淘汰         │
│ 可授权/无风险 → 可转载         │
└───────────────────────────────┘
        ↓
    输出最终判断结论
```

## 各维度合格线

| 维度 | 合格线 | 说明 |
|------|--------|------|
| 原创性 | 通过 | 无原创标或有明确授权 |
| 相关性 | ≥3分 | 与账号定位匹配 |
| 质量 | ≥3分 | 有干货价值 |
| 版权 | 合规 | 可授权且无风险 |

## 综合判断规则

| Step1 | Step2 | Step3 | Step4 | 最终结果 |
|-------|-------|-------|-------|----------|
| 通过 | ≥3 | ≥3 | 合规 | ✅ 可转载 |
| 通过 | ≥3 | ≥3 | 需申请 | ⚠️ 获授权后转载 |
| 存疑 | - | - | - | ❌ 需进一步核实 |
| 不通过 | - | - | - | ❌ 不建议转载 |
| 通过 | <3 | - | - | ❌ 调性不匹配 |
| 通过 | ≥3 | <3 | - | ❌ 质量不达标 |

## 快速自检清单

### 原创性自检
- [ ] 文章无"原创"标
- [ ] 不是洗稿/拼接内容
- [ ] 作者未明确禁止转载

### 相关性自检
- [ ] 粉丝会关心这个话题
- [ ] 能解决粉丝的真实需求
- [ ] 与账号调性一致

### 质量自检
- [ ] 有干货/实用信息
- [ ] 数据/案例可靠
- [ ] 逻辑清晰可读
- [ ] 值得转发分享

### 版权自检
- [ ] 获得明确授权
- [ ] 标注来源和作者
- [ ] 图片字体无风险
