# 选题方法论

## 核心概念定义

选题是内容生产的第一步，决定了内容是否会受到用户关注。好的选题能够激发用户好奇心、引发共鸣、并促使用户主动传播。

选题策划的核心是在**用户需求**、**自身定位**、**时效性**三个维度找到交集。

## 8个常见选题方向

### 方向1：对知名对象的吐槽
- **特征**：争议性强，引发好奇心
- **适用场景**：需要快速获取关注时
- **要点**：吐槽要有理有据，不能纯情绪发泄

### 方向2：深度分析解读
- **特征**：专业深度，正面严肃
- **适用场景**：建立专业权威时
- **要点**：视角独特，结论有洞见

### 方向3：颠覆式认知
- **特征**：反常识观点，引发讨论
- **适用场景**：需要差异化时
- **要点**：观点要经得起推敲，反驳成本高

### 方向4：热点差异化解读
- **特征**：速度快，角度独特
- **适用场景**：追热点时
- **要点**：比普通解读深一层

### 方向5：数据盘点预言
- **特征**：年度盘点，趋势预测
- **适用场景**：时间节点时（年终、季度）
- **要点**：数据要权威，预测要有依据

### 方向6：共鸣性问题解读
- **特征**：触及痛点，疑问式切入
- **适用场景**：建立用户信任时
- **要点**：问题要真实存在，解决方案要可行

### 方向7：娱乐性话题关联
- **特征**：大众喜闻乐见，轻松有趣
- **适用场景**：降低用户阅读门槛时
- **要点**：娱乐性要与专业性平衡

### 方向8：精彩故事段子
- **特征**：情节反转，不可思议
- **适用场景**：提升内容可读性时
- **要点**：故事要服务于核心观点

## 关键成功因素

1. **用户视角**：选题要从用户需求出发，而非自我表达
2. **差异化角度**：同一话题要有独特的切入角度
3. **时效把控**：热点选题要快，深度选题要稳
4. **系列化思维**：好选题要能形成系列，持续输出

## 常见误区

- **自嗨式选题**：基于个人兴趣而非用户需求
- **泛泛而谈**：选题太大没有焦点
- **过度追热点**：所有选题都追热点，失去独特性
- **忽视历史数据**：不了解用户对什么选题更感兴趣
- **选题孤立**：每次选题独立，没有系列化规划
