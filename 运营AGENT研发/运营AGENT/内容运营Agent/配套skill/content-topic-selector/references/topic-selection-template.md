## 填写说明

### 用户价值评估
- 高：直接解决用户痛点
- 中：提供娱乐或信息价值
- 低：仅为追热点价值有限

### 差异化评估
- 高：有独特视角，竞品少
- 中：有差异化但不是独家
- 低：与竞品高度重合

### 时效性评估
- 高：需要立即响应（热点相关）
- 中：有较长生命周期
- 低：几乎无时效性

### 可执行性评估
- 高：团队有相关积累
- 中：需要一定研究
- 低：需要全新领域探索
