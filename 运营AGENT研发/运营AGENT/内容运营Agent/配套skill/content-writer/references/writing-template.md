# 写作模板

## 一、短文案模板

### 朋友圈/微博文案

```
【 Hook标题 】
第一行：引发好奇（问题/数字/对比）
第二行：价值说明（用户收益）
第三行：行动号召（评论/转发/点击）

#话题标签1 #话题标签2 #话题标签3
```

### 广告文案（信息流）

```
主标题（≤30字，制造好奇）：
副标题（≤50字，说明收益）：
正文（≤150字，强化价值）：
行动按钮：
```

### 社群推广文案

```
️ 提醒：
[具体场景/痛点描述]
↓
[解决方案]
↓
[收益说明]
↓
[行动号召+限时福利]

立即参与：[链接]
```

## 二、中长转化文案模板

### 通用结构

```
【开头：痛点共鸣】
描述用户正在经历的问题（1-2段）
制造"这就是我"的共鸣感

【主体：解决方案】
介绍方法/理念/产品（3-4段）
包含具体案例或数据支撑

【结尾：行动号召】
总结核心价值
明确下一步动作
限时/稀缺性促进行动
```

### 故事型转化文案

```
【第一幕：场景】（建立熟悉感）
描述目标用户最常见的工作/生活场景

【第二幕：冲突】（揭示痛点）
在这个场景中遇到了什么问题

【第三幕：解决】（引出方案）
通过什么方法/产品解决了问题

【第四幕：行动】（引导转化】
你也面临同样问题？点击了解...
```

**示例**：

### 短文案示例

```
【为什么你定的目标总是完不成？】

不是因为你不够努力
而是用错了目标管理方法

✅ 只定"结果目标" → 永远焦虑
✅ 改用"过程目标" → 步步为营

试试这个目标管理表，3步设定让结果自然达成

#职场成长 #目标管理
```

### 中长文案开头示例

```
你是不是也这样：

年初定下"要读书50本"的目标，到年底发现只看了5本...

不是你不努力，而是你用错了方法。

我见过太多人，用"结果目标"折磨自己：
- 目标：今年要瘦20斤 → 坚持1个月放弃
- 目标：英语要流利 → 背了3天单词表放弃

真正有效的方式，是把"结果目标"翻译成"过程目标"...
```
