## 关键词管理注意事项

### 设置原则

1. **覆盖面广**：覆盖用户80%以上的常见需求
2. **简洁易记**：关键词短好输入，如「01」「试听」
3. **定期更新**：每月检查一次，删除过期内容
4. **避免冲突**：确保关键词不与其他规则冲突

### 常见问题

1. **关键词失效**
   - 定期检查链接有效性
   - 及时更新过期内容

2. **匹配混乱**
   - 精确匹配优先
   - 模糊匹配注意范围

3. **回复过多**
   - 控制关键词数量
   - 删除低触发率关键词
