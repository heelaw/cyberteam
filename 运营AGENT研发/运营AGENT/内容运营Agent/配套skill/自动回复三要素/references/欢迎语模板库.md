**适用场景**：教育培训、知识付费、职场技能类账号

```

我是[身份]，专注于[领域]。

这里你能获得：
- [核心价值1]
- [核心价值2]
- [核心价值3]

回复「[关键词]」领取[免费资源]
回复「[关键词]」加入[社群名称]

点击底部菜单，发现更多！
```

## 欢迎语优化 checklist

### 基础要素检查

- [ ] 包含账号名称/定位
- [ ] 包含你能提供的核心价值
- [ ] 有明确的行动指引
- [ ] 有吸引回复的「诱饵」

### 进阶要素检查

- [ ] 开头有「钩子」吸引继续阅读
- [ ] 数字具体（「3个方法」而非「很多方法」）
- [ ] 口语化、自然、不像机器人
- [ ] 有差异化竞争优势的简短表达

### 转化要素检查

- [ ] 有微信/二维码等即时联系方式
- [ ] 有关键词回复示例
- [ ] 有菜单栏引导
- [ ] 有社群入口

## 避坑提示

### 常见错误

1. **太官方**
   - 错误：「感谢您的关注，我们将为您提供优质服务」
   - 正确：「hi~终于等到你了！」

2. **太长**
   - 错误：一大段没有分段，用户看不下去
   - 正确：分成3-4个短段落，关键信息加粗

3. **只有感谢**
   - 错误：「感谢您的关注，祝您生活愉快」
   - 正确：「感谢关注！这里有[具体价值]等着你...」

4. **上来就营销**
   - 错误：「点击购买课程，享受5折优惠」
   - 正确：「先送你一份[免费资源]，觉得有用再...」

5. **没有下一步**
   - 错误：「感谢关注」结束
   - 正确：「回复「01」领取[福利]，开始我们的第一课」
