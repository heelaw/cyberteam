# 相关概念

## 常用数据分析框架

### 5W2H分析法
| 维度 | 问题 |
|------|------|
| What | 发生了什么问题？ |
| Why | 为什么会发生？ |
| When | 什么时候发生的？ |
| Where | 在哪里发生的？ |
| Who | 涉及到谁？ |
| How | 怎么发生的？ |
| How much | 影响了多少？ |

### 对比分析法
| 类型 | 说明 | 应用场景 |
|------|------|---------|
| 环比 | 与上一周期 | 短期趋势 |
| 同比 | 与去年同期 | 季节性分析 |
| 横比 | 与竞品/行业 | 竞争力评估 |

## 常见指标定义

### 拉新指标
| 指标 | 定义 |
|------|------|
| CAC | 获取一个用户的成本 |
| 新增用户数 | 一定时间内新增注册用户 |
| 渠道转化率 | 渠道流量→注册用户的比例 |

### 活跃指标
| 指标 | 定义 |
|------|------|
| DAU | 日活跃用户数 |
| MAU | 月活跃用户数 |
| 活跃率 | 活跃用户/总用户 |

### 变现指标
| 指标 | 定义 |
|------|------|
| LTV | 用户生命周期价值 |
| ARPU | 每用户平均收入 |
| 付费率 | 付费用户/总用户 |

### 留存指标
| 指标 | 定义 |
|------|------|
| 次日留存 | 首日次日留存率 |
| 7日留存 | 首日7日后留存率 |
| 流失率 | 流失用户/总用户 |
