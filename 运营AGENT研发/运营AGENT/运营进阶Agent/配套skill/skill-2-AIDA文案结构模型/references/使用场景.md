# 使用场景参考

## 场景1：朋友圈广告

**特点**：
- 用户状态：刷朋友圈，注意力分散
- 阅读时间：3-5秒
- 文案要求：短平快，一眼看懂

**AIDA建议**：
- Attention：制造悬念或冲突
- Interest：快速关联用户痛点
- Desire：1-2个核心证据
- Action：明确CTA

---

## 场景2：应用商店详情页

**特点**：
- 用户状态：主动搜索，货比三家
- 阅读时间：中等
- 文案要求：建立信任，展示差异化

**AIDA建议**：
- Attention：差异化标题
- Interest：3-5个核心卖点
- Desire：用户评价+数据证明
- Action：立即下载

---

## 场景3：落地页

**特点**：
- 用户状态：被广告吸引，想了解更多
- 阅读时间：较长
- 文案要求：完整展示，说服转化

**AIDA建议**：
- Attention：吸引人的首图+标题
- Interest：详细的产品介绍
- Desire：大量证据和社会证明
- Action：明确的下单指引

---

## 场景4：App推送

**特点**：
- 用户状态：正在做其他事
- 阅读时间：极短
- 文案要求：引发好奇，提供价值

**AIDA建议**：
- Attention：制造好奇或紧迫
- Interest：暗示能获得什么
- Desire：省略或简短
- Action：点击查看
