# 常见问题与解答

## Q1：分层太细导致运营不过来怎么办？

**问题**：分了10层，每层策略都不同，执行不过来
**解答**：
- 建议分层不超过5层
- 相邻层级可以合并策略
- 优先运营高价值用户
- 其他层级用自动化策略

---

## Q2：沉默用户召回无效怎么办？

**问题**：发了很多召回消息，但没人回来
**解答**：
- 确认沉默原因：是产品问题还是需求消失？
- 召回时机是否合适
- 激励是否有吸引力
- 触达渠道用户是否常用
- 可能是"伪沉默"：用户已流失，无法召回

---

## Q3：分层阈值怎么定？

**问题**：不知道各层阈值设多少合理
**解答**：
- 看数据分布：找到自然分界线
- 参考业务经验：行业通用标准
- A/B测试：用数据验证阈值合理性
- 动态调整：阈值不是一成不变

---

## Q4：不同层级用户策略冲突怎么办？

**问题**：给新用户的优惠导致老用户不满
**解答**：
- 新用户专享不对老用户开放
- 老用户给予"专属"替代权益
- 设计升级路径：让新用户看到成为老用户的好处
- 避免直接对比
