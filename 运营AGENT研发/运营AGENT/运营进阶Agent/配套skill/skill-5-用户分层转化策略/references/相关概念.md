# 相关概念

## RFM模型

| 维度 | 说明 | 应用 |
|------|------|------|
| R (Recency) | 最近一次消费 | 越近越有价值 |
| F (Frequency) | 消费频率 | 越频繁越有价值 |
| M (Monetary) | 消费金额 | 越多越有价值 |

**RFM分层**：
- 高价值用户：R低+F高+M高
- 潜力用户：R低+F低+M高
- 流失风险：R高+F高+M高（需挽留）
- 流失用户：R高+F低+M低

## 用户生命周期价值（LTV）

```
LTV = 用户平均收入 × 用户生命周期 × 毛利率

用户生命周期 = 1 / 流失率

影响因素：
- 留存率
- 付费率
- 客单价
- 复购频率
```

## 用户分层方法对比

| 方法 | 维度 | 适用场景 |
|------|------|---------|
| 价值分层 | 付费金额、活跃度 | 电商、SaaS |
| 生命周期分层 | 新/活/沉/流失 | 通用 |
| 行为分层 | 核心行为完成度 | 内容平台 |
| 需求分层 | 用户需求类型 | 教育、医疗 |
