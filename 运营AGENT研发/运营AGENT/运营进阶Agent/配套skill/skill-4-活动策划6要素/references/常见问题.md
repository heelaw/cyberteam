# 常见问题与解答

## Q1：活动目标设定多少合适？

**问题**：不知道目标定多少合理
**解答**：
- 参考历史数据：去年同期的XX%
- 参考行业水平：行业平均的XX倍
- 资源倒推法：按预算能获取多少用户
- 保守+进取双目标：保底目标和挑战目标

---

## Q2：活动玩法没有新意怎么办？

**问题**：想不出有创意的玩法
**解答**：
- 组合创新：把两个常见玩法组合
- 门槛创新：改变参与门槛
- 奖励创新：改变奖励形式和发放方式
- 场景创新：改变活动主题场景
- 参考竞品：看竞品做过什么，找差异化

---

## Q3：活动预算如何分配？

**问题**：不知道怎么分配预算
**解答**：
- 激励成本：60-70%（用户奖励）
- 传播成本：20-30%（渠道投放）
- 执行成本：10%（技术、设计、客服）
- 预留备用：5%（应急）

---

## Q4：活动效果如何预测？

**问题**：如何预估活动效果
**解答**：
- 参考历史活动数据
- 按渠道预估转化率
- 考虑活动规模和激励力度
- 设置保守、正常、乐观三种预估
