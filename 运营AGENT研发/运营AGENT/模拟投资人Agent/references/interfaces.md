# 模拟投资人Agent - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|------|------|------|
| 问题描述 | string | 用户提出的业务评估问题 |
| 业务信息 | object | 用户提供的业务相关数据 |
| 评估目标 | string | 期望的评估类型（估值/增长/尽调等） |

### 触发方式
- 关键词触发: "投资人视角"、"投资评估"、"项目评估"、"估值"
- 主动调用: 用户明确请求投资人角度分析

## 输出接口

### 评估报告
| 字段 | 类型 | 描述 |
|------|------|------|
| 报告类型 | string | 评估报告/增长逻辑/估值分析 |
| 核心结论 | string | 投资人视角的核心判断 |
| 详细分析 | object | 各项评估维度的详细分析 |
| 建议 | array | 改进方向和建议清单 |

## 调用关系

### 调用的Skills
- 投资人评估框架法
- 业务价值评估法
- 增长逻辑梳理法
- 财务尽职调查法

### 被调用的Agent
- 财务模型Agent（提供财务数据支持）
- 商业模型Agent（提供商业模式分析）
- 企业生命周期Agent（提供阶段判断）
