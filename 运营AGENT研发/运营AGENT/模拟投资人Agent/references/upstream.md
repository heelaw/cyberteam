# 模拟投资人Agent - 上游依赖

## 数据上游

| 上游来源 | 数据类型 | 说明 |
|----------|----------|------|
| 用户输入 | 业务描述 | 用户主动提供的业务信息 |
| 财务模型Agent | 财务数据 | UE模型、利润结构等 |
| 商业模型Agent | 商业模式 | 业务类型、盈利模式 |
| 企业生命周期Agent | 企业阶段 | 所处发展阶段判断 |

## 能力上游

| 上游来源 | 能力支持 | 说明 |
|----------|----------|------|
| 投资人评估框架法 | 评估标准 | 提供投资人评估维度 |
| 业务价值评估法 | 价值评估 | 业务价值分析方法 |
| 增长逻辑梳理法 | 增长分析 | 增长逻辑梳理工具 |
| 财务尽职调查法 | 财务分析 | 投资人关注财务点 |
