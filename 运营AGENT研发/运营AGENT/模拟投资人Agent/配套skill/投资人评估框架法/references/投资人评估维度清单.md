# 投资人评估维度清单

## 一、市场评估维度

### 市场规模

| 维度 | 说明 | 评估标准 |
|------|------|----------|
| TAM | 总市场容量 | 越大越好 |
| SAM | 可服务市场 | 可触达比例 |
| SOM | 可获得市场 | 实际可获取 |

### 市场特征

| 维度 | 说明 |
|------|------|
| 增长率 | 行业增速 > GDP增速 |
| 周期 | 成长期/成熟期/衰退期 |
| 集中度 | 碎片化/分散/集中 |

### 单位经济

| 指标 | 计算 | 优秀标准 |

## 五、评分标准

### 综合评分（1-5分）

| 分数 | 评价 |
|------|------|
| 5分 | 优秀 |
| 4分 | 良好 |
| 3分 | 一般 |
| 2分 | 较差 |
| 1分 | 差 |

### 投资建议

| 分数 | 建议 |
|------|------|
| >4.5 | 强烈推荐 |
| 4.0-4.5 | 推荐 |
| 3.5-4.0 | 可考虑 |
| 3.0-3.5 | 谨慎 |
| <3.0 | 不推荐 |
