# 短文案模板库

## 一、痛点开头模板

### 模板结构
```
痛点描述 → 情绪强化 → 解决方案
```

### 常用句式

**职场类**
- 还在为XX发愁？这招帮你搞定
- XX困扰？3个方法破解
- XX的你，需要这个

**生活类**
- 每天XX的你，试试这个
- XX怎么解决？这一篇就够了
- XX怎么办？教你几招

**学习类**
- XX学不会？问题出在这
- 想XX？这篇帮你少走弯路
- XX太难？看完这篇就会了

---

## 二、数字钩子模板

### 模板结构
```
具体数字 → 解释说明 → 引导行动
```

### 常用句式

**销量数据**
- XX人已购买/已学习/已使用
- 月销XX件，好评XX%
- XX万人都在用的XX

**效果数据**
- XX%用户反馈XX
- 用了XX天，效果看得见
- XX天改变XX

**时间数据**
- XX分钟学会XX
- 每天XX分钟，XX周见效
- XX年经验分享

## 三、场景代入模板

### 模板结构
```
场景描述 → 情绪感受 → 产品解决
```

### 常用句式

**时间场景**
- 早上起不来？XX帮你多睡XX分钟
- 晚上睡不着？试试这个方法

**地点场景**
- 出门前XX？带上这个
- 在家XX？用这个就够了

**状态场景**
- 熬夜后皮肤差？急救神器来了
- XX压力大？试试这招

## 五、综合模板

### 产品推荐类
```
场景痛点 + 数字钩子 + 解决方案
```
示例：熬夜脸垮？7天急救，这瓶精华液卖爆10万瓶

### 课程推广类
```
身份共鸣 + 效果承诺 + 稀缺紧迫
```
示例：想转行做XX？3个月从小白到入门，限XX个名额

### 服务推广类
```
权威背书 + 案例效果 + 行动引导
```
示例：服务过XX大厂的专家，帮你解决XX问题，限时免费诊断
