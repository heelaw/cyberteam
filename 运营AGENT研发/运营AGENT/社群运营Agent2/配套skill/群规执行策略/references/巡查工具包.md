## 系统辅助配置建议

### 自动助手功能

| 功能 | 配置建议 |
|-----|---------|
| 新人入群提醒 | 发送入群须知 |
| 敏感词警告 | 拦截并提醒管理员 |
| 刷屏警告 | 5分钟内超过10条警告 |
| 链接审核 | 非白名单链接审核后放行 |

### 白名单设置

```
【链接白名单】
- 社群官方链接
- 经过审批的合作方链接
- 知名平台链接（知乎、微信公众号等）

【内容白名单】
- 每周活动报名链接
- 官方公告
- 管理员发布的合作信息
```
