# 群内分享六步流程话术模板

## 使用说明
本模板提供分享全流程的标准话术，可直接使用或根据社群特点修改。

---

## 一、预热阶段话术

### 预告发布话术

```
🎙️ 【重磅嘉宾分享】

📌 分享主题：
[具体主题]

👤 主讲嘉宾：
[姓名] | [职位/Title]
[一句话背景介绍]

🎯 你将获得：
• [价值点1]
• [价值点2]
• [价值点3]

📅 分享时间：
[日期] [时间]
⏰ 预计时长：[X]分钟

📝 报名方式：
回复"报名"+你最想问的一个问题

👥 已报名：XX人

⚠️ 名额有限，先到先得！
```

### 二次预热话术

```
🔥 还有[X]个名额！

【[主题]】分享

[嘉宾名字]要来啦！

已有[X]位伙伴报名~

📝 报名请回复：
"报名" + "你最想问的问题"

⏰ 报名截止：[时间]
📅 分享时间：[日期] [时间]
```

---

## 二、提醒阶段话术

### 当日上午提醒

```
📢 今日分享提醒

今晚[时间]，[嘉宾]将带来分享：

【[主题]】

📍 地点：本群
⏰ 时间：[时间]-[时间]
👤 主讲：[嘉宾]

💡 先睹为快：
[1-2句话剧透精华]

记得准时来听哦~
回复"1"占座！
```

### 分享前1小时提醒

```
⏰ 1小时后开讲！

【[主题]】
👤 主讲：[嘉宾]

📍 地点：本群
⏰ 时间：[时间]-[时间]

🎯 你将获得：
• [价值点1]
• [价值点2]

❓ 你最想问的问题准备好了吗？
快在留言区写下来~

马上开始！
```

### 分享前5分钟提醒

```

【[主题]】

📍 地点：本群
⏰ 时间：[时间]-[时间]

```

## 三、执行阶段话术

### 正式开场

```
🎙️ 分享正式开始！

各位小伙伴们晚上好！

今晚我们非常荣幸邀请到[嘉宾]带来分享~

━━━━━━━━━━━━━━━

【嘉宾介绍】
👤 [姓名]
📍 [职位]
[3句话背景介绍]

【分享主题】
[主题]

【你将获得】
• [价值1]
• [价值2]
• [价值3]

━━━━━━━━━━━━━━━

📌 温馨提示：
• 分享中有互动环节
• 大家可以随时留言问题
• 精华内容稍后整理发布

预计分享时长[X]分钟，
分享结束后有[答疑/互动]环节。

🎤 现在有请[嘉宾]！
```

### 嘉宾入场

```
🎤 让我们欢迎[嘉宾]~

@嘉宾 请开始你的分享！
```

### 互动环节A（分享中途）

```
💬 互动时间！

刚才[嘉宾]分享了[内容]，

我想问一下大家：

[互动问题]

A. [选项1]
B. [选项2]
C. [选项3]

有想法的同学可以直接回复~

[等待回复，选取优质回复]
```

### 互动环节B（分享中途）

```
🤔 思考题！

[嘉宾]刚才提到[内容]，

大家觉得[问题]？

在留言区说说你的看法~

[等待回复]
```

### 嘉宾结束语

```
🎤 以上就是[嘉宾]今天的分享！

感谢[嘉宾]的精彩内容~

总结3个核心要点：
1. [要点1]
2. [要点2]
3. [要点3]

━━━━━━━━━━━━━━━

❓ 现在进入答疑环节！

有问题的小伙伴请在留言区提问~
[嘉宾]会挑选问题回答~

💬 提问格式：
【问题】：[你的问题]
```

### 答疑结束后

```
🎉 感谢[嘉宾]的耐心解答！

以及感谢大家的积极提问~

━━━━━━━━━━━━━━━

📝 后续安排：
• 精华内容整理后发布
• 完整资料稍后发送给参与者
• [如有后续活动]

🌟 特别感谢[嘉宾]！
期待下次再来~

👉 有进一步交流需求的朋友，
可以联系管理员~
```
