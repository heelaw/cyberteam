# 嘉宾邀请与管理指南

## 一、嘉宾选择标准

### 必备条件

| 条件 | 说明 |
|-----|------|
| 专业度 | 在分享领域有实战经验 |
| 表达力 | 能把复杂问题讲清楚 |
| 配合度 | 愿意配合社群流程 |
| 时间可控 | 能按时完成分享 |

### 加分条件

| 条件 | 说明 |
|-----|------|
| 名气背书 | 有知名公司/项目背书 |
| 粉丝基础 | 自带流量 |
| 合作意愿 | 愿意长期合作 |
| 资源支持 | 能提供额外资源 |

---

## 二、嘉宾分级

| 等级 | 标准 | 权益 |
|-----|------|------|
| S级 | 行业顶级大佬 | 全方位包装+长期合作 |
| A级 | 知名公司高管 | 重点预热+资料支持 |
| B级 | 实战专家 | 标准流程+适当曝光 |
| C级 | 社群成员 | 内部分享+荣誉证书 |

---

## 三、邀请话术

### 初次邀请

```
[名字]，您好！

我是[社群名称]的管理员[名字]。

我们社群是一个[社群定位]的社群，
目前有[X]位成员，主要是[目标用户]。

看到您在[领域]有很多实战经验，
想邀请您来群里做一次分享。

【分享主题】
[初步主题]

【分享形式】
• 时长：[X]分钟分享 + [X]分钟答疑
• 形式：语音/文字/录屏
• 时间：您方便的时间

【社群支持】
• 我们会提前预热宣传
• 整理分享内容沉淀
• 给予嘉宾适当曝光
• [其他支持]

有意向的话，我们可以详细聊聊分享主题~

期待您的回复！
```

### 确认合作

```
[名字]，很高兴您愿意来分享！

【确认信息】
📌 分享主题：[主题]
👤 主讲嘉宾：[名字]
📅 分享时间：[日期] [时间]
⏰ 预计时长：[X]分钟

【分享前】
• 我们会提前3天发布预告
• 需要您提供：个人介绍 + 分享大纲
• 预告发布后我们会与您确认

【分享中】
• 管理员会协助主持
• 会有2-3次互动环节
• 答疑时间请预留[X]分钟

【分享后】
• 我们会整理精华内容
• 资料发给参与者
• 视情况可长期合作

请问以上信息确认吗？
```

## 四、嘉宾注意事项

### 分享前提醒

```

【确认信息】
📌 主题：[主题]
⏰ 时间：[时间]-[时间]
📍 地点：本群

【准备提醒】
• 请准备[语音/文字/录屏]格式
• 大纲准备好了吗？
• 需要管理员协助的内容？

【注意事项】
• 时长控制在[X]分钟以内
• 建议预留答疑时间
• 互动问题可以提前准备

期待明天的精彩分享！
```

## 五、嘉宾维护

### 荣誉体系

| 等级 | 条件 | 权益 |
|-----|------|------|
| 分享达人 | 分享1次 | 电子证书 |
| 金牌讲师 | 分享3次 | 实体证书+周边 |
| 首席顾问 | 分享5次+ | 专属头衔+优先合作 |
