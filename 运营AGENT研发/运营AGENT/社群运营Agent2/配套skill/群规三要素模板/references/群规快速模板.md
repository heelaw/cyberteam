# 群规快速模板

## 使用说明
根据社群类型选择合适的模板，直接使用或微调后发布。

---

## 一、五类社群群规模板

### 1. 学习教育类社群

```
📚 [学习社群名称]群规

【我们鼓励】
✅ 分享学习心得和实战经验
✅ 提出问题，帮助他人进步
✅ 推荐优质学习资源
✅ 鼓励式交流，友善讨论

【我们禁止】
❌ 发广告、推广链接
❌ 频繁发送与学习无关的内容
❌ 人身攻击、恶意争吵
❌ 搬运未经授权的盗版内容

【奖惩机制】
🌟 优质分享 → 置顶推荐+收录精华库
⚠️ 违规处理：
   私聊警告 → 群内警告 → 移出社群

💬 有问题找管理员 @xxx
```

## 二、规则制定检查清单

### 必检项

| 检查项 | 合格标准 |
|-------|---------|
| 字数控制 | 不超过200字 |
| 层级清晰 | 允许/禁止/奖惩三段式 |
| 禁止明确 | 至少5条禁止行为 |
| 奖惩对等 | 奖励措施 ≥ 惩罚措施 |
| 可执行性 | 每条规则都能实际执行 |

### 优化项

| 检查项 | 建议 |
|-------|------|
| 可记性 | 用户看完能记住3条 |
| 友好度 | 语气友善不生硬 |
| 争议性 | 无模糊地带，不会产生争议 |

## 三、群规发布流程

```
Step 1: 草拟群规
   ↓
Step 2: 小范围测试（老成员反馈）
   ↓
Step 3: 修改完善
   ↓
Step 4: 正式发布
   ↓
Step 5: 入群即视为同意
```

### 正式发布话术

```
📢 群规更新通知

各位小伙伴，[社群名称]的群规进行了更新，
主要调整了以下内容：

[列出主要变化]

📌 完整版群规请查看群公告。

⚠️ 进群即视为同意遵守群规。
如有疑问请联系 @管理员

感谢大家的理解与支持！
```
