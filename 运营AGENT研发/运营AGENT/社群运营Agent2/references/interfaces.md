# 社群运营Agent2 - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|------|------|------|
| 问题描述 | string | 社群运营相关问题 |
| 社群阶段 | string | 搭建/成熟/衰退 |
| 社群类型 | string | 付费/免费/产品/粉丝 |

### 触发方式
- 关键词触发: "社群运营"、"微信群"、"社群门槛"、"群规"
- 场景触发: 社群搭建、活跃度提升、变现

## 输出接口

### 运营指导
| 字段 | 类型 | 描述 |
|------|------|------|
| 指导类型 | string | 搭建/规则/内容/活动 |
| 核心方案 | object | 具体的运营方案 |
| 执行清单 | array | 可执行的操作清单 |

## 调用关系

### 调用的Skills
- 社群三门槛分类法
- 用户进群引导三步法
- 群规三要素模板
- 群规执行策略
- 早晚资讯三步法
- 群内分享六步流程
- 社群价值提供三法

### 被调用的Agent
- 用户激励Agent（用户激励支持）
- 增长Agent（裂变增长）
- 内容运营Agent（内容支持）
