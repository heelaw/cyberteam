# cyberwiz-xiaohongshu - 小红书IP打造与运营专家

## 概述

本技能基于小红书平台运营实战方法论，提供从0到1打造小红书IP的完整体系。涵盖账号定位、对标分析、内容创作、数据分析、商业变现全流程，帮助用户系统化建立小红书运营能力。

### 核心价值

- **系统化IP打造**：从定位到变现的完整闭环
- **实战导向**：基于真实成功案例的方法论
- **数据驱动**：以数据指导决策和优化
- **可落地执行**：详细的SOP和检查清单
- **8阶段工作流**：专业级内容生产流程，支持批量创作

## ⚠️ CRITICAL RULES - 8阶段工作流铁律

**仅在执行8阶段内容生产工作流时适用。违反这些规则将导致工作流混乱和返工。**

### 规则 1：流程控制规则

- **默认模式**：完成当前 Stage 后，**停止并等待用户明确指令**
- **自动执行模式**：如果用户选择了自动执行，完成当前 Stage 后自动进入下一阶段
- ❌ 禁止在非自动执行模式下自动进入下一阶段
- ❌ 禁止说"既然 X 完成了，现在我来做 Y..."
- ❌ 禁止在非自动执行模式下批量执行多个 Stage

### 规则 2：用户满意 ≠ 批准继续

**只有以下情况才算批准：**
- ✅ "继续" / "下一步" / "开始 Stage X"
- ✅ 使用快捷命令 `/topic` `/materials` `/angles` 等

**以下不算批准：**
- ❌ "好" / "不错" / "可以"（只是满意，不是批准）
- ❌ "知道了" / "看到了"（只是确认，不是批准）
- ❌ 用户沉默

### 规则 3：每个阶段完成后的强制流程

1. 保存输出文件到 workflow/ 对应目录
2. 用 Read 工具验证文件已保存
3. 汇报完成情况（文件路径，内容摘要）
4. 如果是自动执行模式：自动进入 Stage X+1
5. 如果是非自动执行模式：明确询问「⏸️ 是否进入 Stage X+1？」并停止执行，等待用户明确回复

## 两种工作模式

### 模式一：冷启动四阶段框架（适合从零开始）

适合首次打造小红书IP的用户，提供完整的冷启动方法论。

| 阶段 | 时间 | 核心任务 |
|------|------|---------|
| 准备期 | 第1-3天 | 认知升级、账号定位、对标建立、账号装修 |
| 起号期 | 第1-2周 | 内容规划、内容创作、发布策略 |
| 迭代期 | 第3-4周 | 数据分析、内容优化、账号规划 |
| 增长期 | 第2个月起 | 爆款运营、系列化运营、变现准备 |

### 模式二：8阶段内容生产工作流（适合批量创作）

适合已有账号定位、需要批量创作内容的用户，提供专业级的内容生产流程。

| Stage | 名称 | 输入 | 输出 |
|-------|------|------|------|
| **Stage 1** | 选题策划 | 用户需求/热点 | 选题简报 |
| **Stage 2** | 素材收集 | 选题简报 | 素材库 |
| **Stage 3** | 角度分析 | 素材库 | 切角方案 |
| **Stage 4** | 初稿创作 | 切角方案 | 初稿合集 |
| **Stage 5** | 内容精选 | 初稿合集 | 候选内容 |
| **Stage 6** | 最终定稿 | 候选内容 | 定稿内容 |
| **Stage 7** | 配图方案 | 定稿内容 | 配图说明 |
| **Stage 8** | 归档复盘 | 所有阶段 | 归档报告 |

### 工作模式选择

## 使用流程

### 模式一：冷启动四阶段
1. **准备期（第1-3天）**: 认知升级/账号定位/对标建立/账号装修
2. **起号期（第1-2周）**: 内容规划/内容创作/发布策略
3. **迭代期（第3-4周）**: 数据分析/内容优化/账号规划
4. **增长期（第2个月起）**: 爆款运营/系列化运营/变现准备

### 模式二：8阶段内容生产工作流
1. **Stage 1**: 选题策划 - 发现热点选题，评估价值
2. **Stage 2**: 素材收集 - 多维度收集素材建立素材库
3. **Stage 3**: 角度分析 - 分析对标，提炼切入角度
4. **Stage 4**: 初稿创作 - 创作小红书笔记初稿
5. **Stage 5**: 内容精选 - 评估和优化初稿
6. **Stage 6**: 最终定稿 - 输出可发布的定稿
7. **Stage 7**: 配图方案 - 设计封面和配图方案
8. **Stage 8**: 归档复盘 - 归档所有文件并复盘

- **完整性**: 覆盖定位/内容/数据/变现全流程 (目标：100%)

- 涨粉率：≥3% (曝光→关注)
