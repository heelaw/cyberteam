## 协同机制

### 马甲号之间协同
| 场景 | 协同方式 |
|------|----------|
| 话题配合 | A发内容，B评论，C补充 |
| 活动配合 | A预热，B带动，C收尾 |
| 应急支援 | 一个马甲号出问题，另一个补位 |

### 与官方账号协同
| 场景 | 协同方式 |
|------|----------|
| 内容配合 | 马甲号配合官方内容互动 |
| 活动配合 | 马甲号响应官方活动节奏 |
| 舆情应对 | 马甲号协助正面引导 |

## 风险管理

### 身份泄露风险

| 风险点 | 预防措施 | 应急方案 |
|--------|----------|----------|
| 用户怀疑 | 人设统一、内容真实 | 准备解释话术 |
| 数据异常 | 控制互动比例 | 降低互动频率 |
| 恶意揭穿 | 保持低调，不争辩 | 淡化处理 |

### 效果不佳风险

| 风险点 | 预防措施 | 应急方案 |
|--------|----------|----------|
| 用户不参与 | 内容更优质、激励更大 | 调整话题方向 |
| 互动数据差 | 优化话术、把握时间 | 增加马甲号参与 |
| 负面反馈 | 真诚沟通、快速响应 | 准备道歉话术 |
