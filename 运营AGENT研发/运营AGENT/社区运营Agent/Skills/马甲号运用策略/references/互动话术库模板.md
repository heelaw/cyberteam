## 话术质量标准

### 好的话术特征
1. **真诚**：像真实用户说话，不生硬
2. **有价值**：能引发进一步互动或解决问题
3. **符合人设**：每个话术符合马甲号人设
4. **适度**：不过分，不过度互动

### 差的话术特征
1. **模板化**：机械重复，没有针对性
2. **过度热情**：显得假，不真实
3. **信息量少**：只有表情或无意义回复
4. **破坏人设**：与马甲号人设矛盾
