# work-planning-agent 案例库

## 案例分类

### 1. 目标拆解案例

#### 案例1：年度目标拆解
- **目标**：年度GMV 1亿
- **拆解维度**：
  - 季度分解：Q1 2000万、Q2 2500万、Q3 2500万、Q4 3000万
  - 月度分解：逐月递增
  - 品类分解：核心品类+新兴品类
- **关键动作**：明确责任人和时间节点

#### 案例2：项目目标拆解
- **目标**：产品上线
- **拆解框架**：
  - 阶段：设计→开发→测试→上线
  - 里程碑：Alpha→Beta→Release
  - 任务：具体可执行任务
- **关键动作**：识别依赖关系和风险

### 2. 优先级排序案例

#### 案例3：OKR优先级排序
- **O目标**：提升用户留存
- **KR关键结果**：
  - KR1：DAU提升至10万
  - KR2：次留提升至45%
  - KR3：7留提升至25%
- **优先级**：KR2>KR1>KR3

#### 案例4：任务优先级矩阵
- **四象限**：
  - 重要紧急：立即处理
  - 重要不紧急：计划安排
  - 不重要紧急：委托他人
  - 不重要不紧急：删除
- **应用**：每日任务排序

### 3. 时间规划案例

#### 案例5：周计划制定
- **框架**：
  - 周目标：3个核心任务
  - 日分配：上午深度工作，下午沟通协作
  - Buffer：20%缓冲时间
- **关键动作**：每日复盘+次日计划

#### 案例6：项目时间规划
- **工具**：甘特图
- **要素**：
  - 任务名称
  - 开始/结束时间
  - 依赖关系
  - 负责人
- **关键动作**：关键路径识别

## 案例使用方法

1. **拆解方法**：学习目标拆解的维度和方法
2. **排序方法**：参考优先级排序的决策框架
3. **规划工具**：使用案例中的工具进行时间规划

## 案例来源

- 10+真实工作规划案例
- 涵盖电商、产品、管理等多个领域
