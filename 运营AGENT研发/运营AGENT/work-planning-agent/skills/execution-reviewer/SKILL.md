# 执行复盘

## 适用场景

- 项目结束，需要做阶段复盘
- 需要判断目标达成情况
- 需要沉淀可复用经验

## 不做的事

- 不替代进度跟踪
- 不替代实时纠偏
- 不替代目标重新定义

## SOP

1. 对照原目标。
2. 看完成度。
3. 找偏差原因。
4. 总结可复用经验。
5. 输出下轮优化建议。

## 输出

- 目标对照
- 完成度结论
- 偏差原因
- 可复用经验
- 下轮优化建议
