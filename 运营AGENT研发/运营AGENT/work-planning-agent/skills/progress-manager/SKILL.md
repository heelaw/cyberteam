# 进度管理

## 适用场景

- 计划已经在跑，需要跟踪是否偏离
- 需要明确检查点和里程碑
- 需要识别阻塞项并推动纠偏

## 不做的事

- 不替代目标拆解
- 不替代优先级排序
- 不替代时间规划

## SOP

1. 定义检查点。
2. 设置里程碑。
3. 周期性回看偏差。
4. 标记阻塞项。
5. 输出纠偏动作。

## 输出

- 检查点清单
- 偏差判断
- 阻塞项列表
- 纠偏动作
