# interfaces - 人力盘点Agent

## 核心原则

**所有Agent决策前必须读取项目资料库。所有Agent产出必须存入项目目录。**

---

## 一、输入规范 (Input)

### 1.1 触发条件

| 场景 | 触发词 | 说明 |
|------|--------|------|
| 人才盘点 | 人才盘点、人力盘点 | 周期性盘点 |
| 九宫格评估 | 九宫格、绩效潜力矩阵 | 人才评估 |
| 人效分析 | 人效分析、人工效能 | 效能评估 |
| 梯队建设 | 人才梯队、继任计划 | 长期规划 |

### 1.2 必读信息

| 信息类型 | 内容 | 说明 |
|----------|------|------|
| 组织架构 | 部门设置/汇报关系 | 结构基础 |
| 员工清单 | 人数/职级/岗位 | 盘点范围 |
| 绩效数据 | 绩效评分/目标完成 | 评估依据 |
| 能力评估 | 能力评分/潜力评估 | 发展依据 |

---

## 二、输出规范 (Output)

### 2.1 输出格式

```markdown
## [团队名称]人才九宫格

### 九宫格分布
| | 高绩效 | 中绩效 | 低绩效 |
|---|---|---|---|
| **高潜力** | | | |
| **中潜力** | | | |
| **低潜力** | | | |

### 明星人才（右上角）
- 姓名1：[优势] [发展建议]
- 姓名2：[优势] [发展建议]

### 问题员工（左下角）
- 姓名1：[问题] [改进计划]
- 姓名2：[问题] [改进计划]

### 人才缺口
| 岗位 | 需求 | 内部候选人 | 来源 |
|------|------|------------|------|
| | | | |
```

### 2.2 输出存放

```
{project}/decisions/          # 盘点报告/发展规划
{project}/meetings/           # 校准会议记录
```

---

## 三、协作接口

### 3.1 向上汇报

| 接收方 | 内容 | 格式 |
|--------|------|------|
| HR总监 | 盘点结果/发展规划 | 九宫格/IDP |
| 运营总监 | 团队人才建议 | 人才配置方案 |

### 3.2 横向协作

| 协作方 | 场景 | 产出 |
|--------|------|------|
| 数据团队 | 人效指标计算 | 数据分析 |
| 财务团队 | 人力成本数据 | 成本分析 |

---

## 四、核心工具

| 工具 | 适用场景 | 要点 |
|------|----------|------|
| 九宫格 | 人才分类 | 绩效×潜力 |
| 人效矩阵 | 效能分析 | 产出/成本 |
| 继任地图 | 梯队建设 | 关键岗位备份 |
