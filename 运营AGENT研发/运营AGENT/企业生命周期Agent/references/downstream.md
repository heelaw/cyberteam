# 企业生命周期Agent - 下游去向

## 输出流向

### 直接输出给用户

| 输出类型 | 内容 | 用途 |
|---------|------|------|
| 阶段判断报告 | 企业当前发展阶段 | 用户决策参考 |
| 特征分析报告 | 该阶段典型特征 | 用户理解参考 |
| 经营重点清单 | 当前阶段核心任务 | 执行优先级参考 |
| 阶段策略建议 | 与阶段匹配的策略 | 策略制定参考 |

## 协同下游Agents

| 下游Agent | 接收什么 | 用途 |
|-----------|---------|------|
| 商业模型Agent | 阶段 → 商业模式选择 | 商业模式制定 |
| 模拟投资人Agent | 阶段 → 项目评估 | 投资判断 |
| 财务模型Agent | 阶段 → 财务重点 | 财务管理 |
| 增长策略咨询Agent | 阶段 → 增长策略 | 增长规划 |

## 版本更新去向

当本Agent有版本更新时，需要通知：
1. **运营总监Agent** - 阶段判断能力更新
2. **CEO Agent** - 战略决策参考
3. **产品总监Agent** - 产品规划参考
