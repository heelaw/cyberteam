# 财务模型Agent - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|------|------|------|
| 分析目标 | string | UE模型/利润结构/财务指标 |
| 财务数据 | object | 用户提供的财务相关数据 |
| 业务背景 | object | 业务类型、规模等上下文 |

### 触发方式
- 关键词触发: "UE模型"、"LTV"、"CAC"、"利润结构"、"财务报表"
- 场景触发: 财务分析需求、融资准备

## 输出接口

### 财务分析
| 字段 | 类型 | 描述 |
|------|------|------|
| 分析类型 | string | UE/利润/报表/指标 |
| 核心结论 | string | 财务分析的核心发现 |
| 详细数据 | object | 各项财务指标的计算结果 |
| 建议 | array | 优化建议清单 |

## 调用关系

### 调用的Skills
- UE模型拆解法
- 利润结构分析法
- 财务三表解读法
- 财务指标诊断法

### 被调用的Agent
- 模拟投资人Agent（投资评估）
- 商业模型Agent（商业逻辑）
- 业务模型Agent（业务支撑）
