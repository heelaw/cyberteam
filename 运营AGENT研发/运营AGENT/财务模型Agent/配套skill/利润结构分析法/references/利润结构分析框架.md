# 利润结构分析框架

## 收入结构分析

| 类型 | 说明 | 占比参考 |
|------|------|----------|
| 主营业务收入 | 核心业务收入 | >70% |
| 其他业务收入 | 辅助业务 | <20% |
| 营业外收入 | 非经常 | <10% |

## 成本结构分析

| 类型 | 说明 | 占比参考 |
|------|------|----------|
| 直接成本 | 产品/服务直接成本 | 30-60% |
| 间接成本 | 运营管理成本 | 20-30% |
| 期间费用 | 销售/管理/财务 | 20-40% |

## 毛利分析

| 行业 | 高毛利 | 中毛利 | 低毛利 |
|------|--------|--------|--------|
| SaaS | >70% | 50-70% | <50% |
| 电商 | >20% | 10-20% | <10% |
| 制造 | >30% | 15-30% | <15% |

## 净利分析

| 水平 | 净利率 | 说明 |
|------|--------|------|
| 优秀 | >20% | 盈利能力强 |
| 良好 | 10-20% | 盈利正常 |
| 一般 | 5-10% | 盈利较弱 |
| 较差 | <5% | 盈利困难 |
