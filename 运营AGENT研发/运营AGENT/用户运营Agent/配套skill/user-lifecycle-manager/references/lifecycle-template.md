## 二、生命周期阶段定义

### 2.1 阶段划分标准

| 阶段 | 定义标准（活跃天数） | 用户特征 | 占比目标 |
|------|---------------------|---------|---------|
| 拉新期 | 注册0-7天 | 新用户，认知不足 | 20% |
| 成长期 | 8-30天 | 开始使用核心功能 | 30% |
| 成熟期 | 31-180天 | 稳定活跃，贡献价值 | 35% |
| 沉默期 | 181-365天 | 活跃下降，有流失风险 | 10% |
| 流失期 | >365天无活跃 | 已流失或已离开 | 5% |

### 2.2 关键转化节点

| 转化路径 | 转化条件 | 转化率目标 |
|---------|---------|-----------|
| 拉新→成长 | 完成首次核心行为 | >40% |
| 成长→成熟 | 连续7天活跃 | >50% |
| 成熟→沉默 | 连续7天不活跃 | <20% |
| 沉默→流失 | 连续30天不活跃 | <30% |
