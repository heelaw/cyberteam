# 商业模式类型图谱

## 一、交易模式

### 平台型

| 特征 | 案例 |
|------|------|
| 撮合双边或多边用户 | 淘宝、滴滴、Airbnb |
| 不直接拥有产品/服务 | 美团、大众点评 |
| 收入来自佣金/服务费 | 抖音、小红书 |

### 自营型

| 特征 | 案例 |
|------|------|
| 自建产品/服务 | 京东自营、网易严选 |
| 自主定价销售 | 苹果官网、 Nike官网 |
| 承担库存风险 | 无印良品 |

### 混合型

| 特征 | 案例 |
|------|------|
| 平台+自营共存 | 天猫（淘宝商城）、京东 |
| 第三方+自有品牌 | 网易考拉 |

---

## 二、客户类型

### B2B（企业对企业）

| 特征 | 案例 |
|------|------|
| 客户是企业 | 阿里巴巴1688、企业微信 |
| 决策链条长 | Salesforce、钉钉 |
| 客单价高 | 各种企业服务 |

### B2C（企业对消费者）

| 特征 | 案例 |
|------|------|
| 客户是个人 | 天猫、京东 |
| 决策快 | 各种电商 |
| 营销重要 | 完美日记 |

### C2C（消费者对消费者）

| 特征 | 案例 |
|------|------|
| 买卖方都是个人 | 闲鱼、咸鱼 |
| 门槛低 | 抖音小店 |
| 信任重要 |  |

---

## 三、付费模式

### 订阅制

| 特征 | 案例 |
|------|------|
| 定期付费 | Netflix、Spotify |
| 收入可预测 | 爱优腾会员 |
| 注重续费率 | SaaS产品 |

### 一次性付费

| 特征 | 案例 |
|------|------|
| 单次购买 | 传统零售 |
| 不需要续费 | 单机游戏 |
| 营销获客重要 | 知识付费课程 |

### 交易抽成型

| 特征 | 案例 |
|------|------|
| 按交易额收费 | 淘宝、滴滴 |
| 收入与GMV挂钩 | 美团外卖 |
| 需要规模效应 | 抖音电商 |

### 免费增值（Freemium）

| 特征 | 案例 |
|------|------|
| 基础功能免费 | 印象笔记、石墨文档 |
| 增值功能付费 | 各种工具类SaaS |
| 转化率关键 | 飞书、钉钉 |

### 广告收入

| 特征 | 案例 |
|------|------|
| 靠广告变现 | 抖音、微信 |
| 需要流量规模 | 今日头条 |
| 效果广告为主 | 百度 |

---

## 四、价值链位置

### 生产端

- 自有产品制造
- OEM/ODM
- 案例：苹果、耐克

### 流通端

- 分销/代理
- 零售终端
- 案例：7-11、屈臣氏

### 服务端

- 提供服务
- 不涉及实物产品
- 案例：咨询公司、律所

### 平台端

- 撮合交易
- 不直接参与服务
- 案例：淘宝、滴滴

---

## 五、商业模式组合矩阵

| 交易模式 | 客户类型 | 付费模式 | 典型案例 |
|----------|----------|----------|----------|
| 平台型 | B2C | 抽成型 | 淘宝、滴滴 |
| 平台型 | C2C | 抽成型 | 闲鱼 |
| 自营型 | B2C | 订阅制 | Netflix |
| 自营型 | B2C | 一次性 | Apple Store |
| 自营型 | B2B | 订阅制 | Salesforce |
| 混合型 | B2C | 混合 | 京东、天猫 |
