## 错误处理

| 错误类型 | 典型表现 | 处理方式 |
|---------|---------|---------|
| 虚假因子 | 用与业务无关的因子 | 每个因子必须能回答"我们能做什么影响它" |
| 无效分解 | 把不可控因素作为因子 | 只能用"业务能干预"的因子 |
| 遗漏关键 | 核心驱动因素遗漏 | 用行业标准公式检查完整性 |
| 循环论证 | 因子之间有逻辑循环 | 确保因子独立可调 |
| 数学错误 | 乘积不等于总指标 | 逐项验证，交叉检验 |

## 核心定义

乘法式指标拆解是将一个总体指标按照**乘法关系**拆解为多个影响因子的方法。每个因子都是总指标的驱动要素，通过提升各因子可以实现总指标的增长。

## 核心公式

```
总指标 = 因子1 × 因子2 × 因子3 × ... × 因子N
```

## 适用场景

| 场景类型 | 示例 |
|---------|------|
| 增长类问题 | 销售额 = 流量 × 转化率 × 客单价 × 复购率 |
| 转化类问题 | 注册转化 = 曝光 × 点击率 × 注册率 |
| 产出类问题 | GMV = UV × 转化率 × 客单价 |
| 效率类问题 | 人效 = 人均产出 × 人数 |

## 与并列式拆解的区别

| 维度 | 并列式拆解 | 乘法式拆解 |
|------|----------|-----------|
| 关系 | 加法关系 | 乘法关系 |
| 目标 | 理解构成 | 找到杠杆点 |
| 适用 | 结构分析 | 增长分析 |
| 公式 | A = B + C + D | A = B × C × D |

## Critical Rules

### 必须遵守（5条）

1. **因子有效性** - 每个因子必须对总指标有正向影响
2. **独立可控** - 每个因子必须能够独立调整和控制
3. **数据可测** - 每个因子必须能够被准确测量和追踪
4. **逻辑完整** - 因子相乘必须等于总指标，无数学错误
5. **行业验证** - 因子选择必须符合行业最佳实践

### 禁止行为（5条）

1. **禁止虚假因子** - 不能使用与总指标无因果关系的因子
2. **禁止无效分解** - 不能将不可控因素作为因子
3. **禁止遗漏关键因子** - 核心驱动因素不能遗漏
4. **禁止循环论证** - 因子之间不能有逻辑循环
5. **禁止脱离业务** - 不能使用业务无法干预的外部因素
