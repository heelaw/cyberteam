---

## 核心能力

1. **并列式拆解** - 将大指标按并列关系拆解为多个子指标（加法关系）
2. **乘法式拆解** - 将指标按乘法关系拆解，找到影响因子
3. **指标量化** - 将模糊目标转化为可衡量的具体指标
4. **执行路径规划** - 根据拆解结果输出可执行的行动方案
