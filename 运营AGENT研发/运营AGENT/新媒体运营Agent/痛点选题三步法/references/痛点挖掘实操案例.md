## 第二步：分析挖掘痛点

### 痛点深挖过程

**关键词：时间不够用**
- Q1：为什么觉得时间不够用？
- A：工作太忙，占用太多时间
- Q2：为什么工作会占用这么多时间？
- A：效率低，不会合理安排
- Q3：效率低的原因是什么？
- A：总是被打断、不会做计划、拖延症

**深层痛点提炼**：
- 表面：没时间
- 本质：效率问题 → 缺乏时间管理能力
