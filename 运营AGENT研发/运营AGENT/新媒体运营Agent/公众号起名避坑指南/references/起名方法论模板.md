## 二、候选名称发散（Step 2产出）

### 方法一：定位词直接组合
格式：人群 + 领域/价值

| 组合 | 候选名称 |
|------|----------|
| 人群+领域 | 职场新人口袋书 |
| 人群+价值 | 新人职场锦囊 |
| 价值+领域 | 沟通有方法 |
| 特色+领域 | 老黄聊职场 |

### 方法二：人物+领域
格式：人物称呼 + 具体领域

| 组合 | 候选名称 |
|------|----------|
| 昵称+领域 | 职场小黄说 |
| 身份+领域 | HR美妈的职场经 |
| 特点+领域 | 老江湖聊职场 |

### 方法三：场景/问题+方案
格式：用户处境/痛点 + 解决方案暗示

| 组合 | 候选名称 |
|------|----------|
| 问题+方案 | 告别沟通障碍 |
| 场景+暗示 | 开会怎么办 |
| 痛点+方案 | 职场人际急救包 |

### 方法四：动词+价值
格式：动作词 + 价值承诺

| 组合 | 候选名称 |
|------|----------|
| 动词+名词 | 学点职场 |
| 动词+形容词 | 进化你的职场 |
| 动宾短语 | 提升职场力 |
