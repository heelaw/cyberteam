## 七、起名模式总结

### 高分名称公式

| 公式 | 示例 | 适用领域 |
|------|------|----------|
| 人群+价值 | 职场进化论、考研帮 | 职场、教育 |
| 人物+领域 | 老黄聊商业、吴晓波频道 | 职场、财经 |
| 具体承诺 | 每天学做一道菜 | 美食、生活 |
| 场景+情绪 | 深夜食堂、职场夜读 | 情感、职场 |
| 领域+形式 | 财新网、钛媒体 | 新闻、科技 |
| 对比/反差 | 婚与不婚、职场与生活 | 情感、职场 |

### 避坑警示

| 类型 | 避免 | 建议 |
|------|------|------|
| 生僻字 | 梓、轩、璟 | 用常见字组合 |
| 过长 | XX指南针、XX手册 | 控制在8字内 |
| 模糊 | 心灵港湾、品质生活 | 名称即内容承诺 |
| 中英混搭 | 韩式XX、艾瑞XX | 全中文或全英文 |
