## 一、被关注自动回复

### 模板结构
```
[欢迎语] + [自我介绍/价值说明] + [行动引导]
```

### 模板示例

#### 职场类
```
👋 欢迎来到[账号名]！

我是你的职场成长导师，专注分享：
✅ 职场沟通技巧
✅ 晋升加薪方法
✅ 效率提升干货

回复关键词领取资料：
📌 回复"沟通"→ 领取沟通话术模板
📌 回复"简历"→ 获得简历优化指南
📌 回复"晋升"→ 解锁晋升攻略
```

#### 美食类
```
🍳 欢迎来到美食小窝！

这里有：
• 零基础也能学的家常菜谱
• 每周免费美食直播
• 美食爱好者交流群

点击菜单[我要学做菜]查看全部教程
回复"入群"加入美食大家庭～
```

#### 情感类
```
💕 感谢关注[账号名]

在这里你会找到：
• 感情困惑的解答
• 两性相处的智慧
• 自我成长的力量

后台留言你的故事，Jojo陪你聊聊
```

## 二、关键词自动回复模板

### 模板结构
```
[共情/确认] + [核心信息] + [行动引导]
```

### 模板示例

#### 资料领取类
```
✅ 收到！

[资料名称]马上发给你～
📂 包含：内容1、内容2、内容3

点击链接领取：[URL]
⚠️ 链接7天内有效，抓紧哦～
```

#### 问题解答类
```
📚 关于[问题关键词]：

[简短解答，建议30字以内]

想深入了解？推荐阅读：
👉 [文章标题]（链接）

还有问题？留言告诉我～
```

#### 活动引导类
```
🎉 恭喜你发现了隐藏福利！

[活动名称]正在火热进行中：
• 活动时间：XX-XX
• 参与方式：XXX
• 奖励：XXX

立即参与：[链接]
```

## 三、常见问题回复模板

### 模板结构
```
[确认问题] + [直接回答] + [延伸价值/引导]
```

### 回复模板库


## 四、自动回复禁止用语

| 类型 | 禁止示例 | 问题分析 |
|------|----------|----------|
| 强制分享 | "必须转发朋友圈才能领取" | 引起反感 |
| 信息过载 | 一口气发5条消息 | 轰炸用户 |
| 模糊回答 | "这个你可以自己查一下" | 没有价值 |
| 过度承诺 | "保证你月入十万" | 不真实 |
| 态度冷淡 | "嗯"、"知道了" | 无互动感 |
