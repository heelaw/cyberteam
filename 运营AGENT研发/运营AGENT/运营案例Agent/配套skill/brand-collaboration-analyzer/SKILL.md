> 技能名称：品牌联名营销分析
> 版本: v1.0
> 难度等级: simple
> 关联Agent: 运营案例Agent
> 来源案例: 瑞幸x茅台、喜茶xFENDI、奈雪x魔卡少女樱等

---

## 1. 技能概述

本技能提供品牌联名营销的分析框架，帮助运营从业者评估联名价值、分析成功因素、指导联名策划。

**核心能力**:
- 分析品牌联名的互补性
- 评估联名产品的创新度
- 设计联名传播策略
- 控制联名活动风险

### Step 1: 评估品牌互补性

**输入**: 联名合作的两个品牌

**处理**:
```
┌─────────────────────────────────────────────────────────────┐
│ 品牌互补性评估模型                                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                     品牌A                                    │
│                        ↕                                    │
│              互补性 > 竞争性                                │
│                        ↕                                    │
```

## 5. Critical Rules

### 5.1 必须遵守

1. **互补性优先** - 选择联名品牌时，互补性大于相似性
2. **产品要创新** - 联名不是LOGO叠加，是产品融合
3. **用户无负担** - 高端品牌联名要让大众能消费
4. **传播有节奏** - 预热-爆发-持续三阶段清晰

### 5.2 禁止行为

1. **不选相似品牌** - 相似品牌联名话题性不足
2. **不纯贴标** - 没有产品创新的联名会损害品牌
3. **不脱离群众** - 高端品牌联名要接地气
4. **不忽视供应** - 断货会严重影响口碑
5. **不一次性消耗** - 联名要有后续延续

---

## 6. 评估维度

| 维度 | 说明 | 最低要求 |
|------|------|----------|
| 互补性 | 品牌选择是否互补 | 弱互补以上 |
| 创新度 | 产品是否有创新 | 中创新以上 |
| 传播性 | 话题是否足够 | 有社媒热度 |
| 风险控制 | 预案是否完备 | P0有预案 |
