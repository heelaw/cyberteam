## 填写示例

### 示例: 在线教育平台

| 字段 | 内容 |
|------|------|
| **指标名称** | 核心课程作业提交率 |
| **指标英文名** | Core Course Assignment Submission Rate |
| **指标简称** | 作业提交率 |

### 精确定义

| 字段 | 内容 |
|------|------|
| **指标定义** | 每周期学员完成并提交的作业数占应提交作业数的比例 |
| **统计口径** | 统计周期内，核心课程布置的作业中学员实际提交的作业数 |
| **衡量单位** | 百分比（%） |
| **时间维度** | 按周统计，按月汇总 |

### 计算公式

```markdown
## 计算公式

主公式:
作业提交率 = (实际提交作业数 / 应提交作业数) × 100%

## 分子
- 实际提交作业数: 学员在规定时间内成功提交的作业数
- 判定标准: 已提交且通过格式检查
- 数据来源: 学习平台作业系统

## 分母
- 应提交作业数: 核心课程布置的作业总数 × 实际上课学员数
- 排除情况: 请假学员、体验课学员
- 数据来源: 学习平台排课系统
```

### 拆解公式

```markdown
## 指标拆解

作业提交率 = 完课率 × 作业完成率 × 主动提交率

| 因子 | 定义 | 数据来源 |
|------|------|---------|
| 完课率 | 学完课程视频的学员比例 | 视频观看系统 |
| 作业完成率 | 学完课后完成作业的学员比例 | 作业系统 |
| 主动提交率 | 不需要催交主动提交的作业比例 | 作业系统 |
```
