# A/B测试基础概念

## 什么是A/B测试

### 定义
A/B测试（也称分裂测试）是一种对照实验方法，将用户随机分为两组（实验组和对照组），分别展示不同版本，通过对比两组的表现来判定哪个版本更优。

### 核心原理
1. **随机分组**: 确保实验组和对照组用户特征分布一致
2. **对照比较**: 控制单一变量，排除其他干扰因素
3. **统计推断**: 用样本数据推断总体差异是否真实存在

### A/B测试 vs A/B/n测试
- A/B测试: 2个版本对比
- A/B/n测试: 2个以上版本对比（需多重比较校正）

## 统计基础概念

### 1. 假设检验

#### 零假设（H₀）
- 实验组与对照组没有差异
- 差异是随机波动造成的

#### 备择假设（H₁）
- 实验组与对照组存在差异
- 差异是真实存在的

#### 假设检验的逻辑
```
反证法：
假设H₀为真 → 计算出现当前差异的概率 → 
如果概率很小（小于α），则拒绝H₀，认为差异显著
```

### 2. 显著性水平（α）

| 水平 | 常用场景 |
|------|----------|
| α=0.05 | 常规业务场景（95%置信度） |
| α=0.01 | 高风险决策（99%置信度） |
| α=0.10 | 探索性分析、早期测试 |

### 3. p值

**定义**: 在零假设为真的条件下，观察到当前或更极端结果的概率

**解读**:
- p < α → 拒绝零假设 → 差异显著
- p ≥ α → 无法拒绝零假设 → 差异不显著

**注意事项**:
- p值不是"假设为真的概率"
- p值不能衡量效应大小
- p值受样本量影响（样本量越大，p值越小）

### 4. 置信区间

**定义**: 总体参数真值落在某个区间的概率

**95%置信区间解读**:
- 如果重复实验100次，有95次的总体差异会落在置信区间内
- 不等于"有95%概率包含真值"

**与p值的关系**:
- p < 0.05 当且仅当95%置信区间不包含0
- 置信区间提供更多信息（效应方向和大小）

### 5. 统计功效（Power）

**定义**: 当总体确实存在差异时，正确检测到差异的概率

**组成**:
- 1-β = 统计功效
- β = 第二类错误（漏报错误）的概率

**标准**:
- 统计功效 ≥ 80% 为可接受
- 统计功效 ≥ 90% 为理想

### 6. 效应量（Effect Size）

**定义**: 实际差异的大小，与样本量无关

**常见指标**:
| 效应量 | 适用场景 | 小 | 中 | 大 |
|--------|----------|----|----|-----|
| Cohen's d | 连续变量 | 0.2 | 0.5 | 0.8 |
| Hedges' g | 小样本校正 | 0.2 | 0.5 | 0.8 |
| Cramér's V | 分类变量 | 0.1 | 0.3 | 0.5 |

## A/B测试核心指标

### 1. 转化率指标（二项分布）

**适用场景**:
- 登录转化率
- 注册转化率
- 购买转化率
- 点击率

**计算公式**:
```
转化率 = 转化人数 / 访问人数
提升率 = (实验组转化率 - 对照组转化率) / 对照组转化率
```

### 2. 连续指标

**适用场景**:
- 人均停留时长
- 人均订单金额
- 人均页面浏览量

**计算公式**:
```
均值 = 总和 / 人数
提升率 = (实验组均值 - 对照组均值) / 对照组均值
```

### 3. 比率指标

**适用场景**:
- 每用户平均收入（ARPU）
- 每用户平均订单数

## 常见统计错误

### 1. 新奇效应（Novelty Effect）
- 用户因新功能好奇而过度使用
- 长期可能回落

### 2. 幸存者偏差
- 只分析留存用户，忽略流失用户
- 导致高估效果

### 3. p-hacking
- 反复检验直到p<0.05
- 增大假阳性风险

### 4. 忽视多重比较
- 同时检验多个指标
- 未做校正，增大假阳性率

### 5. 提前停止
- 看到显著结果就停止测试
- 未考虑假阳性风险

## 重要术语速查

| 术语 | 英文 | 定义 |
|------|------|------|
| 实验组 | Treatment Group | 接受新版本的用户组 |
| 对照组 | Control Group | 保持原版本的用户组 |
| 随机化 | Randomization | 将用户随机分配到各组 |
| 样本量 | Sample Size | 参与实验的用户数量 |
| 显著性水平 | Significance Level | 判定显著性的阈值α |
| p值 | p-value | 在H₀为真时观察到结果的概率 |
| 置信区间 | Confidence Interval | 总体参数的估计区间 |
| 统计功效 | Statistical Power | 正确检测到真实差异的概率 |
| 效应量 | Effect Size | 实际差异的大小 |
| 第一类错误 | Type I Error | 假阳性（α） |
| 第二类错误 | Type II Error | 假阴性（β） |
