# A/B测试方法学

## 统计检验方法

### 1. 两比例z检验（z-test for two proportions）

**适用场景**: 比较两个比例/转化率

**适用条件**:
- 样本来自二项分布
- 样本量足够大（np > 5 且 n(1-p) > 5）

**检验公式**:
```
p_pooled = (x1 + x2) / (n1 + n2)

SE = sqrt(p_pooled * (1 - p_pooled) * (1/n1 + 1/n2))

z = (p1 - p2) / SE

p值 = 2 * (1 - Φ(|z|))
```

**简化公式**（当样本量足够大时）:
```
SE = sqrt(p1*(1-p1)/n1 + p2*(1-p2)/n2)
z = (p1 - p2) / SE
```

**Python实现**:
```python
from scipy import stats
import numpy as np

def z_test_two_proportions(p1, n1, p2, n2):
    """两比例z检验"""
    # 合并比例
    p_pooled = (p1*n1 + p2*n2) / (n1 + n2)
    
    # 标准误
    se = np.sqrt(p_pooled * (1 - p_pooled) * (1/n1 + 1/n2))
    
    # z统计量
    z = (p1 - p2) / se
    
    # 双侧p值
    p_value = 2 * (1 - stats.norm.cdf(abs(z)))
    
    # 置信区间
    se_diff = np.sqrt(p1*(1-p1)/n1 + p2*(1-p2)/n2)
    ci_lower = (p1 - p2) - 1.96 * se_diff
    ci_upper = (p1 - p2) + 1.96 * se_diff
    
    return {
        'z': z,
        'p_value': p_value,
        'ci_95': (ci_lower, ci_upper)
    }
```

### 2. 卡方检验（Chi-square Test）

**适用场景**: 检验两个分类变量之间的独立性

**适用条件**:
- 样本随机分配
- 每个期望频数 ≥ 5

**检验公式**:
```
χ² = Σ (O - E)² / E

其中:
O = 观察频数
E = 期望频数 = (行合计 × 列合计) / 总计

自由度 df = (行数 - 1) × (列数 - 1)
```

**2x2列联表**:
```
            转化    未转化    合计
实验组       a       b       a+b
对照组       c       d       c+d
合计        a+c     b+d       N
```

**Python实现**:
```python
from scipy import stats

def chi_square_test(contingency_table):
    """卡方检验"""
    chi2, p_value, dof, expected = stats.chi2_contingency(contingency_table)
    
    return {
        'chi2': chi2,
        'p_value': p_value,
        'dof': dof,
        'expected': expected
    }
```

### 3. 独立样本t检验（Independent t-test）

**适用场景**: 比较两个连续变量的均值

**适用条件**:
- 数据近似正态分布
- 两组方差齐性（可用Levene检验）

**检验公式**:
```
t = (x̄₁ - x̄₂) / SE_pooled

SE_pooled = sqrt(s₁²/n₁ + s₂²/n₂)

自由度 df = n₁ + n₂ - 2（等方差假设）
```

**Welch's t检验**（方差不齐时）:
```
t = (x̄₁ - x̄₂) / sqrt(s₁²/n₁ + s₂²/n₂)

df ≈ (s₁²/n₁ + s₂²/n₂)² / 
     ((s₁²/n₁)²/(n₁-1) + (s₂²/n₂)²/(n₂-1))
```

**Python实现**:
```python
from scipy import stats

def t_test_independent(group1, group2, equal_var=True):
    """独立样本t检验"""
    t_stat, p_value = stats.ttest_ind(group1, group2, equal_var=equal_var)
    
    # 计算效应量 Cohen's d
    pooled_std = np.sqrt(((len(group1)-1)*np.std(group1, ddof=1)**2 + 
                          (len(group2)-1)*np.std(group2, ddof=1)**2) / 
                         (len(group1)+len(group2)-2))
    cohens_d = (np.mean(group1) - np.mean(group2)) / pooled_std
    
    return {
        't': t_stat,
        'p_value': p_value,
        'cohens_d': cohens_d
    }
```

## 样本量计算

### 1. 转化率样本量计算

**公式**（两比例z检验）:
```
n = 2 * (Z_α/2 + Z_β)² * p̄(1-p̄) / (p₁ - p₂)²

其中:
p̄ = (p₁ + p₂) / 2
Z_α/2 = 1.96（α=0.05时）
Z_β = 0.84（β=0.20，功效80%时）
```

**Python实现**:
```python
from scipy import stats
import numpy as np

def sample_size_proportions(p1, p2, alpha=0.05, power=0.8):
    """计算两比例比较所需样本量"""
    z_alpha = stats.norm.ppf(1 - alpha/2)  # 双侧检验
    z_beta = stats.norm.ppf(power)
    
    p_bar = (p1 + p2) / 2
    
    n = 2 * (z_alpha + z_beta)**2 * p_bar * (1 - p_bar) / (p1 - p2)**2
    
    return np.ceil(n)

# 示例: 转化率从10%提升到12%
n = sample_size_proportions(0.10, 0.12)
print(f"每组需要样本量: {n}")
```

### 2. 均值比较样本量计算

**公式**:
```
n = 2 * (Z_α/2 + Z_β)² * σ² / δ²

其中:
δ = |μ₁ - μ₂|（期望差异）
σ = 标准差
```

**Python实现**:
```python
def sample_size_means(mean1, mean2, std, alpha=0.05, power=0.8):
    """计算均值比较所需样本量"""
    z_alpha = stats.norm.ppf(1 - alpha/2)
    z_beta = stats.norm.ppf(power)
    
    delta = abs(mean1 - mean2)
    
    n = 2 * (z_alpha + z_beta)**2 * std**2 / delta**2
    
    return np.ceil(n)

# 示例: 时长从5分钟提升到5.5分钟，标准差3分钟
n = sample_size_means(5.5, 5, 3)
print(f"每组需要样本量: {n}")
```

## 功效分析

### 1. 事后功效分析

**目的**: 评估已完成的实验是否有足够的功效检测到真实效应

**Python实现**:
```python
from scipy import stats

def power_analysis(n1, n2, p1, p2, alpha=0.05):
    """计算统计功效"""
    # 合并比例
    p_pooled = (p1*n1 + p2*n2) / (n1 + n2)
    
    # 效应量 h (Cohen's h)
    h = 2 * (np.arcsin(np.sqrt(p1)) - np.arcsin(np.sqrt(p2)))
    
    # 标准误
    se = np.sqrt(1/n1 + 1/n2)
    
    # z值
    z = h / se
    
    # 功效
    power = 1 - stats.norm.cdf(z - stats.norm.ppf(1 - alpha/2))
    
    return power

# 示例
power = power_analysis(5000, 5000, 0.12, 0.10, alpha=0.05)
print(f"统计功效: {power:.2%}")
```

### 2. 功效与样本量关系

| 样本量(每组) | 功效(检测10%相对提升) |
|--------------|----------------------|
| 1,000        | ~30%                 |
| 3,000        | ~60%                 |
| 5,000        | ~80%                 |
| 10,000       | ~95%                 |

## 多重比较校正

### 1. Bonferroni校正

**原理**: 将显著性阈值除以比较次数

**公式**:
```
α_adj = α / k

其中 k = 比较次数
```

**Python实现**:
```python
def bonferroni_correction(p_values, alpha=0.05):
    """Bonferroni校正"""
    k = len(p_values)
    alpha_adj = alpha / k
    
    significant = [p < alpha_adj for p in p_values]
    
    return {
        'alpha_adj': alpha_adj,
        'significant': significant
    }
```

### 2. Benjamini-Hochberg（BH）校正

**原理**: 控制假发现率（FDR）

**步骤**:
1. 将p值从小到大排序
2. 找到最大的满足 p(i) ≤ i/k * α 的i
3. 该i及之前的所有检验显著

**Python实现**:
```python
from scipy import stats

def bh_correction(p_values, alpha=0.05):
    """Benjamini-Hochberg校正"""
    n = len(p_values)
    sorted_p = sorted(enumerate(p_values), key=lambda x: x[1])
    
    significant = [False] * n
    for i, (idx, p) in enumerate(sorted_p):
        threshold = (i + 1) / n * alpha
        if p <= threshold:
            significant[idx] = True
        else:
            break
    
    return significant
```

## 效应量计算

### 1. Cohen's d（连续变量）

**公式**:
```
d = (M₁ - M₂) / SD_pooled

SD_pooled = sqrt((SD₁² + SD₂²) / 2)
```

**解读**:
| d值 | 效应大小 |
|-----|----------|
| 0.2 | 小 |
| 0.5 | 中 |
| 0.8 | 大 |

### 2. Cohen's h（比例）

**公式**:
```
h = 2 * (arcsin(√p₁) - arcsin(√p₂))
```

### 3. 相对提升率

**公式**:
```
提升率 = (实验组指标 - 对照组指标) / 对照组指标 × 100%
```

## 方法选择决策树

```
数据是什么类型？
│
├─ 二项数据（转化/未转化）
│   │
│   ├─ 样本量 > 30/组 → z检验
│   └─ 样本量 ≤ 30/组 → 精确二项检验
│
├─ 计数数据
│   └─ 泊松检验
│
└─ 连续数据
    │
    ├─ 正态分布 → t检验
    └─ 非正态分布 → Mann-Whitney U检验
```

## 常见问题

### Q1: p值不显著怎么办？
1. 检查样本量是否足够
2. 评估效应量是否有实际意义
3. 考虑延长测试时间
4. 分析是否有其他变量影响

### Q2: 样本量如何估算？
1. 明确最小可检测效应（MDE）
2. 确定显著性水平（通常0.05）
3. 确定统计功效（通常80%）
4. 使用公式或在线工具计算

### Q3: 多个指标如何处理？
1. 区分核心指标和辅助指标
2. 对核心指标使用原始α
3. 对辅助指标使用校正后的α
4. 明确报告每个指标的显著性
