# 课程映射

## 章节 -> 岗位

- 第 1 章：`增长全景与业务视角`
- 第 2 章：`PMF与增长重点`
- 第 3 章：`增长实验设计`
- 第 4 章：`数据驱动增长`
- 第 5 章：`新用户激活`
- 第 6 章：`留存与变现`
- 第 7 章：`留存与变现` + `数据驱动增长`

## 工作流

- 先判断是否适合增长，再谈 PMF 和北极星。
- 先拆变量和分群，再谈实验。
- 先看数据和线索，再谈激活与留存。
- 变现和渠道判断必须结合 LTV/CAC。

## 课程边界

- 高阶课覆盖策略、模型、实验和数据驱动方法。
- 不重复入门课的基础概念解释，只保留进阶与实战。
