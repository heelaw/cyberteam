# 增长黑客高阶Agent - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|---|---|---|
| 问题类型 | string | 全景/PMF/指标/模型/实验/数据/激活/留存/变现 |
| 业务背景 | object | 产品、阶段、数据、渠道 |
| 当前阶段 | string | 探索/成长期/成熟期 |

## 输出接口

### 增长方案
| 字段 | 类型 | 描述 |
|---|---|---|
| 方案类型 | string | 认知/策略/实验/数据 |
| 核心内容 | object | 具体方案内容 |
| 执行计划 | array | 分阶段执行计划 |
| 风险预案 | array | 风险和对策 |

## 调用关系

### 调用的 Skills
- 增长全景与北极星
- PMF与增长重点
- 增长模型与分群
- 增长实验设计
- 数据驱动增长
- 新用户激活
- 留存与变现
