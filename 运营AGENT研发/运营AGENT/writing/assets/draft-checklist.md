# 初稿检查清单 (Draft Checklist)

## 🎯 核心策略检查
- [ ] 核心卖点突出
- [ ] 痛点匹配准确
- [ ] 符合目标人群
- [ ] 符合平台特性
- [ ] 明确的行动召唤 (CTA)

## 📝 内容结构检查
- [ ] 标题吸引人
- [ ] 开头有钩子
- [ ] 痛点描述清晰
- [ ] 解决方案具体
- [ ] 证据支撑充分
- [ ] 逻辑流程顺畅
- [ ] 结尾有力

## 💬 语言风格检查
- [ ] 口语化表达
- [ ] 简洁有力
- [ ] 避免专业术语
- [ ] 有情感共鸣
- [ ] 无冗余信息

## 📊 数据与证据检查
- [ ] 数据真实可靠
- [ ] 案例具体可信
- [ ] 引用来源明确
- [ ] 避免夸大宣传

## 🔒 合规性检查
- [ ] 符合广告法
- [ ] 无虚假宣传
- [ ] 无敏感内容
- [ ] 版权合规

## 🎨 格式与排版检查
- [ ] 段落清晰
- [ ] 重点突出
- [ ] 易读性强
- [ ] 符合平台格式要求

## 🎯 转化率优化检查
- [ ] 有明确的目标
- [ ] 有紧迫感
- [ ] 降低决策门槛
- [ ] 增加信任度

## 📱 平台适配检查
- [ ] 符合小红书特性 (如果投放小红书)
- [ ] 符合公众号特性 (如果投放公众号)
- [ ] 符合电商平台特性 (如果投放电商平台)
- [ ] 符合短视频特性 (如果投放短视频平台)

## ✅ 最终检查
- [ ] 符合品牌调性
- [ ] 差异化明显
- [ ] 有记忆点
- [ ] 易于传播
