# 产品资料库使用指南

## 目录结构

```
products/
├── README.md                   # 本文件
├── templates/                  # 模板文件
│   ├── product_profile.md     # 产品档案模板
│   └── user_research.md       # 人群研究模板
├── tech/                       # 科技产品类
├── beauty/                     # 美妆护肤类
├── food/                       # 食品饮料类
├── education/                  # 教育培训类
├── service/                    # 服务类
└── other/                      # 其他类别
```

## 如何使用

### 1. 创建新产品档案

**步骤：**

1. 复制 `templates/product_profile.md` 模板
2. 放到对应的产品类别目录下
3. 重命名为 `[产品名称].md`
4. 填写产品信息（标注 **[必填]** 的为必填项）

**示例：**
```bash
# 复制模板
cp templates/product_profile.md tech/smartphone_x.md

# 编辑文件，填写产品信息
```

### 2. 创建人群研究档案

**步骤：**

1. 复制 `templates/user_research.md` 模板
2. 放到对应的目录下
3. 重命名为 `[人群名称].md`
4. 填写人群研究信息

**示例：**
```bash
# 复制模板
cp templates/user_research.md tech/urban_white_collar_women.md

# 编辑文件，填写人群研究信息
```

### 3. 按产品分类组织

建议按产品类别分类存放：

| 类别 | 目录 | 适用产品 |
|-----|------|---------|
| 科技产品 | `tech/` | 手机、电脑、家电、数码配件 |
| 美妆护肤 | `beauty/` | 护肤品、彩妆、香水、个护 |
| 食品饮料 | `food/` | 零食、饮料、保健品、酒类 |
| 教育培训 | `education/` | 在线课程、培训机构、书籍 |
| 服装服饰 | `clothing/` | 男装、女装、童装、运动装 |
| 家居生活 | `home/` | 家具、家纺、家居用品 |
| 服务类 | `service/` | 软件服务、咨询服务、会员服务 |
| 其他 | `other/` | 不属于以上类别的产品 |

### 4. 文件命名规范

**产品档案命名：**
- 推荐格式：`[品牌]_[产品名].md`
- 示例：
  - `apple_iphone15.md`
  - `nike_air_max.md`
  - `lancome_advanced_genifique.md`

**人群研究命名：**
- 推荐格式：`[人群特征].md`
- 示例：
  - `urban_white_collar_women.md`
  - `post_90s_fitness_lovers.md`
  - `new_parents.md`

## 文档维护

### 更新频率建议

| 文档类型 | 更新频率 | 说明 |
|---------|---------|------|
| 产品档案 | 按需更新 | 产品更新、新功能、新评价时 |
| 人群研究 | 季度更新 | 用户需求变化、市场趋势变化时 |

### 版本记录

在文档底部记录更新：

```markdown
---

## 更新记录

- **2024-01-15**：添加用户评价新增10条
- **2024-01-10**：更新竞品信息
- **2024-01-05**：创建文档
```

## 最佳实践

1. **及时更新**：产品功能、价格、评价变化时及时更新档案
2. **真实数据**：使用真实的用户评价和数据，不编造
3. **分类清晰**：按产品类别分类，便于查找
4. **命名规范**：统一命名规范，便于搜索
5. **版本管理**：重要更新时记录版本变化

## 常见问题

**Q: 一个产品应该创建多少个文档？**
A: 每个产品一个档案文档（product_profile.md）。如果针对不同人群，可以创建多个人群研究文档。

**Q: 用户评价从哪里获取？**
A: 可以从电商平台（京东、天猫）、社交媒体（小红书、微博）、应用商店等渠道收集。

**Q: 是否需要填写所有字段？**
A: 标注 **[必填]** 的字段必须填写，其他字段根据实际情况填写。信息越完整，文案效果越好。

**Q: 如何处理竞品信息？**
A: 在产品档案的"竞品分析"部分填写。如果竞品信息复杂，可以为竞品单独创建档案。
