## Step 3: 差异化评估

### 评估矩阵

| 卖点候选 | 差异化程度(高/中/低) | 用户感知度(高/中/低) | 竞品有没有 |
|----------|---------------------|---------------------|-----------|
|          |                     |                     |           |
|          |                     |                     |           |
|          |                     |                     |           |

### 筛选标准

**差异化程度判断**：
- 高：竞品完全没有
- 中：竞品有但不如我们
- 低：竞品都一样

**用户感知度判断**：
- 高：用户直接能感受到
- 中：需要解释才能理解
- 低：用户根本不关心

## 输出示例

### 示例：在线课程

**Step 1: 基础信息**
```
产品名称：Excel数据分析课
核心功能：7天学会数据分析
目标用户：职场新人
价格：99元
```

**Step 2: 7方向发散（节选）**
```
方向2-效果承诺：
- 7天学会做数据分析报表
- 工资涨500
- 能独立完成项目

方向3-解决问题：
- 不会做表被领导骂
- 加班整理数据
- 同事都会只有我不会
```

**Step 3: 差异化评估**
| 卖点候选 | 差异化 | 感知度 | 竞品有？ |
|----------|--------|--------|----------|
| 7天学会 | 高 | 高 | 无 |
| 工资涨500 | 高 | 高 | 无 |
| 99元 | 中 | 中 | 有 |

**Step 4: 最终选择**
```
主卖点：7天学会数据分析汇报
选择理由：时间明确、效果可见、解决核心痛点
文案方向：用完能独立做数据分析报表
```
