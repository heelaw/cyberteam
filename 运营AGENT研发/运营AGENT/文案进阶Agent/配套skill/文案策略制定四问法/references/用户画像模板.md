# 用户画像模板

## 用户画像构成

### 基础信息

| 维度 | 问题 | 答案 |
|------|------|------|
| 人口统计 | 年龄、性别、学历 | |
| 职业背景 | 行业、职位、收入 | |
| 地域分布 | 一二线/三四线 | |
| 消费能力 | 高/中/低 | |

### 行为特征

| 维度 | 问题 | 答案 |
|------|------|------|
| 触媒习惯 | 平时用什么平台 | |
| 信息获取 | 怎么了解新产品 | |
| 购买决策 | 考虑多久、看重什么 | |
| 使用习惯 | 什么场景使用 | |

### 心理特征

| 维度 | 问题 | 答案 |
|------|------|------|
| 痛点 | 最困扰的问题 | |
| 痒点 | 想要但还没实现 | |
| 需求 | 真正想要的是什么 | |
| 顾虑 | 购买前担心什么 | |

## 常见用户画像模板

### 模板1：职场转型者

```
用户画像：想转行的职场人

基础信息：
- 年龄：25-35岁
- 学历：大专以上
- 收入：5-15K
- 痛点：对现工作不满意，想改变

行为特征：
- 信息获取：知乎、公众号、求职网站
- 决策周期：1-2周
- 看重：效果、成功率

心理特征：
- 焦虑：怕学不会
- 期待：改变现状
- 顾虑：怕投入时间金钱后学不出来

→ 核心关切：能不能学会？学完能不能找到工作？
```

### 模板2：健康追求者

```
用户画像：注重健康的人群

基础信息：
- 年龄：25-45岁
- 收入：10K+
- 有一定健康意识

行为特征：
- 信息获取：小红书、公众号
- 决策周期：冲动消费
- 看重：成分、效果、口碑

心理特征：
- 痛点：想要健康但没时间/坚持不了
- 期待：轻松变健康
- 顾虑：是不是真的有用

→ 核心关切：有没有效果？安不安全？
```

### 模板3：企业决策者

```
用户画像：企业老板/高管

基础信息：
- 年龄：30-50岁
- 企业规模：50-500人
- 关注：效率、成本

行为特征：
- 信息获取：行业媒体、销售推荐
- 决策周期：1-3个月
- 看重：效果、稳定性、服务

心理特征：
- 痛点：现有工具不好用、员工抱怨
- 期待：提升效率、降低成本
- 顾虑：实施风险、员工不会用

→ 核心关切：能不能解决问题？多少钱？多难实施？
```

## 用户画像检查清单

### 必填项

- [ ] 目标用户是谁？（一句话描述）
- [ ] 年龄/职业大概是什么？
- [ ] 最大的痛点是什么？
- [ ] 最关心什么？
- [ ] 在哪里能看到这条文案？

### 选填项

- [ ] 收入水平？
- [ ] 决策周期？
- [ ] 购买顾虑？
- [ ] 竞品用户？

## 用户画像示例

### 示例1：在线课程

```
用户画像：职场技能提升者

一句话：25-32岁，想提升职场竞争力的职场人

基础特征：
- 年龄：25-32岁
- 学历：本科
- 收入：8-20K
- 痛点：晋升难、技能不够

行为特征：
- 信息获取：知乎、公众号、朋友圈
- 决策周期：1-2周
- 看重：效果、时间投入

心理特征：
- 焦虑：怕学完没用
- 期待：快速提升职场竞争力
- 顾虑：怕花了钱没效果

→ 文案要点：强调效果、时间短、有案例
```

### 示例2：消费品

```
用户画像：精致妈妈

一句话：28-38岁，有孩子的都市女性

基础特征：
- 年龄：28-38岁
- 学历：本科以上
- 收入：10K+
- 痛点：没时间、想给孩子最好的

行为特征：
- 信息获取：小红书、妈妈群、电商平台
- 决策周期：冲动+比较
- 看重：品质、口碑、安全

心理特征：
- 焦虑：担心产品不安全
- 期待：给孩子最好的
- 顾虑：是不是真的好用

→ 文案要点：强调安全、品质、其他妈妈推荐
```
