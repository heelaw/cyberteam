# 场景化表达库

## 为什么要场景化

用户不买"产品"，买"产品在特定场景下帮他解决的问题"。
场景化表达 = 告诉用户在什么情况下、用这个产品、得到什么结果。

---

## 常用场景模板

### 1. 痛点场景

```
当你______（痛点场景）时，
我们的产品______（解决方案），
让你______（得到的好处）

示例：
当你为写不出文案熬夜时，
我们的AI工具3分钟生成10个标题，
让你早点睡个好觉
```

### 2. 荣耀场景

```
以前你需要______（麻烦的事），
现在只需要______（简单的事），
就能______（得到的结果）

示例：
以前你需要找设计师排版2小时，
现在上传图片自动生成海报，
3分钟就能发朋友圈
```

### 3. 身份场景

```
像______（用户熟悉的身份）一样______（行为），
你也可以______（获得的结果）

示例：
像专业营养师一样搭配每日三餐，
你也可以轻松管理全家人的健康
```

### 4. 对比场景

```
不用______（竞品的麻烦），
只要______（我们的简单），
就能______（同样的结果）

示例：
不用花几万块请团队，
只要用这个工具，
就能做出专业的品牌设计
```

## 场景词汇库

### 时间场景词

| 场景 | 口语表达 |
|------|----------|
| 快速 | 3分钟、立刻、马上、一眨眼 |
| 长期 | 5年、一直、以后、从此 |
| 日常 | 每天、随手、没事的时候 |
| 关键时刻 | 关键时刻、紧急时候、需要时 |

### 空间场景词

| 场景 | 口语表达 |
|------|----------|
| 办公室 | 在公司、开会间隙、办公桌前 |
| 家里 | 床上、沙发上、做饭时 |
| 路上 | 通勤、等电梯、排队时 |
| 外出 | 出差、旅行、见客户路上 |

### 情绪场景词

| 场景 | 口语表达 |
|------|----------|

| 嫌麻烦 | 不想动、嫌累、麻烦 |

## 场景化案例库

### 案例1：在线课程

**原抽象表达**：高效学习，快速提升

**场景化改写**：

- 通勤场景：地铁上打开APP，20分钟后到站时已经学完今天的内容
- 睡前场景：躺床上听15分钟，听完就能睡着，知识也学了
- 等人场景：等咖啡的时间听一节课，等到了课也上完了

### 案例2：健身App

**原抽象表达**：科学健身，塑造完美身材

**场景化改写**：

- 早起场景：早上6点起床，15分钟完成晨练，洗个澡去上班
- 办公室场景：久坐腰酸，站起来活动5分钟，缓解疲劳
- 出差场景：酒店没健身房，在房间跟着APP练，20分钟出出汗

### 案例3：外卖App

**原抽象表达**：30分钟送达，新鲜热乎

**场景化改写**：

- 加班场景：加班到9点，饿着肚子改方案，外卖到了才有力气干活
- 居家场景：周末宅家不想动，手机点一下，饭就送到家门口
- 带娃场景：孩子哭闹没空做饭，点个外卖，20分钟开饭

### 案例4：理财服务

**原抽象表达**：专业理财，稳健收益

**场景化改写**：

- 养老场景：现在开始每月存一点，30年后每月能多领5000块退休金
- 买房场景：攒了5年首付差一点，理财收益补上这块就能上车
- 看病场景：万一得大病，理财收益够付医药费，不用动老本

## 场景提炼方法

### Step 1: 列出用户一天

```
用户的典型一天是什么样的？
- 早上起床做什么
- 上班路上做什么
- 工作时间做什么
- 下班后做什么
- 周末做什么
```

### Step 2: 找到产品介入点

```
在用户的典型场景中，
产品能在哪个环节帮上忙？
```

### Step 3: 描述具体画面

```
在______（场景），
______（用户行为），
______（产品介入），
______（得到结果）
```

## 检查清单

你的文案是否场景化？

- [ ] 有具体的时间点吗？（几点、几分钟、多久）
- [ ] 有具体的地点吗？（家里、公司、路上）
- [ ] 有具体的动作吗？（打开、点击、输入）
- [ ] 有具体的结果吗？（学会、做完、得到）
- [ ] 用户能想象出画面吗？
