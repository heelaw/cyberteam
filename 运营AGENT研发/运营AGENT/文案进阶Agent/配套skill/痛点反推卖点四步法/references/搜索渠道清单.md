## 问卷调查

### 调查方式

| 方式 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 线上问卷 | 大规模 | 成本低、覆盖广 | 样本可能不精准 |
| 电话回访 | 重点用户 | 深度 | 成本高 |
| 线下访谈 | 深度体验 | 真实 | 样本少 |
| 社群调研 | 活跃用户 | 快速 | 样本偏差 |

### 问卷问题设计

```
必问问题：
1. 您使用中最不满意的一点是？
2. 您为什么没有继续使用？
3. 您希望我们改进的是？
4. 您会推荐给朋友吗？为什么？
```
