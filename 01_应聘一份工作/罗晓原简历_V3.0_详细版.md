**我的职责**：
- 利用AI技术评估与验证各种假设
- 完善商业计划书（BP）
- 验证LTV/CAC模型

**验证内容（详细版）**：

1. **现金流模型验证**
   - 北美本土运营各种手段的成本核算
   - 不同城市、不同模式的收益预测
   - 盈亏平衡点分析

2. **市场假设验证**
   - 目标用户规模估算
   - 市场需求强度评估
   - 竞争格局分析
   - 各指标落地的可行性分析

3. **业务模式验证**
   - O2O模式（线上预约+线下体验）vs DTC模式（直接面向消费者）
   - 两种模式对营业额、成本、用户体验的影响对比
   - 最终建议：O2O模式更适合北美市场

4. **核心指标验证**
   - LTV（客户生命周期价值）：估算单客户长期贡献
   - CAC（获客成本）：估算获取客户的成本
   - LTV/CAC比率：验证是否满足投资人预期（通常要求3:1以上）
   - 回本周期：验证投资人资金安全边际

**项目结局**：
- 投资人要求：成立美国公司 → 组团队 → MVP验证
- 董事会分歧：对风险评估、股份分配存在分歧
- 项目解散：公司资金链断裂
