## 使用方法

### 添加新 Skill

1. 在 `00_基础研发/02_Skill 研发/Skills/` 中开发
2. 测试通过后：
   - 复制到 `~/.claude/skills/My Skill/`（运行位置）
   - 复制到本目录（配置位置，便于 Git 管理）
   - 在 `02 领域/01_专业能力建设/01_Skills/` 创建文档索引

### 删除 Skill

1. 从 `~/.claude/skills/` 删除（运行位置）
2. 从本目录删除（配置位置）
3. 在 `02 领域/01_专业能力建设/01_Skills/` 更新索引

### 修改 Skill

1. 在 `00_基础研发/02_Skill 研发/Skills/` 中修改
2. 测试通过后同步到：
   - `~/.claude/skills/`
   - 本目录
