## 一、定价策略设计

### 1.1 定价模型选择

经过对 Freemium + 订阅制、 按使用量计费、 一次性买断、 企业定制 四种模型的深入分析，推荐采用 **"Freemium + 分层订阅 + 使用量叠加"** 的复合模型。

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                        CyberTeam v2.1 定价模型架构                            │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  ┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐│
│  │   Free      │ ──▶ │   Pro       │ ──▶ │   Team      │ ──▶ │  Enterprise││
│  │   (免费)     │     │   (订阅)    │     │   (团队)    │     │  (定制)    ││
│  └─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘│
│         │                   │                   │                   │      │
│         ▼                   ▼                   ▼                   ▼      │
│   入门门槛              核心价值              协作增强             深度定制   │
│   用户培育              变现主力              ARPU提升             客单价突破   │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 各层级定价详情

#### 层级一：Free（免费）

**定位**: 获客漏斗入口，用户培育

| 维度 | 内容 |
|------|------|
| **定价** | 0元/月 |
| **目标用户** | 独立创业者、尝鲜用户、学生 |
| **核心功能** | 3个核心Agent、5个基础Skill、每月10次任务 |
| **限制条件** | 输出长度限制、无优先支持、无高级分析 |
| **升级钩子** | 任务完成后提示"升级解锁完整报告" |

**心理锚定策略**:
- 明确标注"价值XX元的功能现已免费"
- 展示Pro版本的对比优势（但不标价）

#### 层级二：Pro（个人专业版）

**定位**: 核心变现层级，ARPU主力

| 维度 | 内容 |
|------|------|
| **定价** | 99元/月 或 799元/年（年付享83折） |
| **目标用户** | 独立创业者、产品经理、投资人 |
| **核心功能** | 全部12个Agent、15个Skill、无限任务、完整报告 |
| **增值服务** | 优先队列、邮件支持、模板库访问 |
| **升级钩子** | "团队协作功能敬请期待" |

**定价心理学设计**:

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          Pro 定价心理学                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  1. 锚定效应                                                                │
│     ├── 原价标注: ¥199/月                                                  │
│     ├── 优惠价: ¥99/月  ← 用户感知"省了100元"                               │
│     └── 视觉呈现: 划线原价 + 醒目的优惠价                                    │
│                                                                              │
│  2. 损失规避                                                                │
│     ├── 年付选项: "省¥389"（相比月付）                                     │
│     ├── 月付压力: "随时可取消，无锁定"                                      │
│     └── 免费试用: "7天无理由退款"                                           │
│                                                                              │
│  3. 社会认同                                                                │
│     ├── 页面展示: "已有 5,280 位创业者选择 Pro"                             │
│     ├── 评价摘录: 展示3条用户好评摘要                                       │
│     └── 数据背书: "平均节省 4 小时/周"                                     │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 层级三：Team（团队协作版）

**定位**: 提升ARPU，支持协作场景

| 维度 | 内容 |
|------|------|
| **定价** | 299元/用户/月 或 2,399元/用户/年 |
| **最低购买** | 3个席位起 |
| **目标用户** | 创业团队、投资机构、营销团队 |
| **核心功能** | 全部Agent+Skill、团队工作区、任务分配、权限管理 |
| **增值服务** | Slack/钉钉集成、API访问、专属客服 |
| **升级钩子** | "需要SSO/SLA？请咨询企业版" |

**团队定价策略**:
- **席位梯度优惠**: 3-5席无折扣、6-10席9折、11-20席85折、21+席8折
- **追加销售**: 单用户升级Pro仅需49元/月（原99元）

#### 层级四：Enterprise（企业定制版）

**定位**: 突破客单价天花板，建立竞争壁垒

| 维度 | 内容 |
|------|------|
| **定价** | 5万/年 起（基于需求定制） |
| **目标用户** | 中大型企业、上市公司、投资集团 |
| **核心能力** | 私有化部署、定制Agent开发、专属模型训练 |
| **服务内容** | SLA保障、7x24支持、客户成功经理 |
| **合同形式** | 年框协议，季度里程碑验收 |

**企业版定价分层**:

| 版本 | 定价区间 | 功能差异 | 目标客户 |
|------|----------|----------|----------|
| Standard | 5-15万/年 | 基础定制+标准SLA | 中型企业 |
| Professional | 15-50万/年 | 深度定制+优先支持 | 大型企业部门 |
| Unlimited | 50万+/年 | 无限使用+专属团队 | 集团客户 |

### 1.3 竞品定价对比

| 产品 | 免费版 | Pro版 | Team版 | Enterprise |
|------|--------|-------|--------|------------|
| **CyberTeam v2.1** | 10次/月 | ¥99/月 | ¥299/用户/月 | 5万+/年 |
| **Notion AI** | 有限 | $10/月 | $15/用户/月 | 定制 |
| **Cursor** | 有限 | $20/月 | $40/用户/月 | 定制 |
| **Perplexity Pro** | 有限 | $20/月 | - | - |
| **Gamma.app** | 有限 | $15/月 | $20/用户/月 | 定制 |

**竞争优势**:
- 免费额度比竞品更慷慨（10次 vs 3-5次）
- Pro定价低于行业平均水平（¥99 vs $10-20）
- 企业版起点合理，门槛适中
