# 思考天团重组版

这是思考天团的可发布重构版。

- 入口用 Skill，负责接收问题与路由。
- 中枢用主持人 Agent，负责调度和整合。
- 核心能力保留为 12 个独立 Agent。
- 长尾思维模型沉入模型库，按需调用。

这个版本的目标不是堆更多模型，而是让每次输出都能落到结论和行动。

## 结构

- `SKILL.md`：入口与路由规则。
- `AGENT.md`：主持人工作准则。
- `WORKFLOW.md`：协作流程。
- `MAPPING.md`：旧名到新归属的映射。
- `agents/`：核心专家。
- `models/`：长尾模型库。

## 使用方式

1. 先判断问题类型。
2. 选 1 个主专家，必要时补 1-4 个辅助专家。
3. 用并行、辩论或链式模式处理。
4. 最后只交付结论、分歧、行动建议。
