# 思考天团异常检测机制设计

**问题**: 如何设计思考天团（14个思维专家 + 运营Agent）的自动异常检测与恢复机制？

## 一、异常检测体系设计原则

### 1.1 设计目标

| 目标 | 说明 |
|------|------|
| **及时发现** | 异常发生后快速检测，不遗漏 |
| **准确定位** | 精确定位异常类型和位置 |
| **自动恢复** | 具备自愈能力，减少人工干预 |
| **告警分级** | 不同级别异常采取不同响应 |
| **可追溯** | 异常历史可查，支持复盘 |

## 二、异常类型定义

### 2.1 异常分类矩阵

| 类别 | 异常类型 | 严重程度 | 检测难度 |
|------|----------|---------|---------|
| **Agent异常** | Agent无响应 | P1 | 低 |
| | Agent输出异常 | P2 | 中 |
| | Agent崩溃 | P0 | 低 |
| **任务异常** | 任务卡死 | P0 | 低 |
| | 任务超时 | P1 | 低 |
| | 任务失败 | P2 | 低 |
| **协作异常** | 消息丢失 | P1 | 中 |
| | 消息延迟 | P2 | 低 |
| | 辩论僵局 | P2 | 中 |
| **系统异常** | 队列积压 | P1 | 低 |
| | 资源耗尽 | P0 | 中 |
| | 存储溢出 | P1 | 低 |

### 2.2 Agent类异常

#### 2.2.1 Agent无响应（AgentNotResponding）

**定义**: Agent在规定时间内没有产生任何输出

**检测特征**:
| 特征 | 阈值 | 说明 |
|------|------|------|
| 首次输出时间 | > 60秒 | 从分配到首次输出的时间 |
| 输出间隔 | > 300秒 | 连续两次输出的最大间隔 |
| 心跳 | 缺失 | 超过5分钟无心跳 |

**数据来源**:
```python
# 从 workspace/tasks/{question_id}/tasks/{expert_id}.json 检测
task = load_task(expert_id, question_id)

# 特征1：首次输出超时
if task.started_at and not task.first_output_at:
    elapsed = now() - task.started_at
    if elapsed > 60:  # 60秒
        return "AGENT_NOT_RESPONDING"

# 特征2：输出间隔超时
if task.last_output_at and task.second_last_output_at:
    gap = task.last_output_at - task.second_last_output_at
    if gap > 300:  # 5分钟
        return "AGENT_OUTPUT_STALLED"
```

#### 2.2.2 Agent输出异常（AgentOutputAnomaly）

**定义**: Agent输出不符合预期格式或内容异常

**检测特征**:
| 特征 | 类型 | 说明 |
|------|------|------|
| 输出为空 | hard | output.content 为空 |
| 输出过短 | soft | 输出长度 < 100字符 |
| 输出格式错误 | soft | 不符合预期的JSON/markdown |
| 包含错误标记 | soft | 包含"error"、"failed"等关键词 |
| 重复输出 | soft | 与前3次输出相似度 > 90% |

**数据来源**:
```python
# 从 workspace/tasks/{question_id}/tasks/{expert_id}.json 检测
task = load_task(expert_id, question_id)

# 特征1：输出为空
if not task.output or not task.output.get("content"):
    return "AGENT_OUTPUT_EMPTY"

# 特征2：输出过短
content = task.output.get("content", "")
if len(content) < 100:
    return "AGENT_OUTPUT_TOO_SHORT"

# 特征3：格式错误
if task.output_type == "json":
    try:
        json.loads(task.output.content)
    except:
        return "AGENT_OUTPUT_INVALID_JSON"

# 特征4：包含错误
error_keywords = ["error", "failed", "exception", "无法", "失败"]
if any(kw in content.lower() for kw in error_keywords):
    return "AGENT_OUTPUT_HAS_ERROR"

# 特征5：重复检测
if is_similar_to_recent_outputs(content, last_n=3, threshold=0.9):
    return "AGENT_OUTPUT_REPETITIVE"
```

#### 2.2.3 Agent崩溃（AgentCrashed）

**定义**: Agent进程异常终止

**检测特征**:
| 特征 | 说明 |
|------|------|
| 状态为failed | status = "failed" |
| 无错误信息 | error_message 为空 |
| 异常退出码 | exit_code != 0 |
| 进程消失 | 进程ID不存在 |

**数据来源**:
```python
# 从 workspace/tasks/{question_id}/tasks/{expert_id}.json 检测
task = load_task(expert_id, question_id)

# 特征：任务失败但无有效输出
if task.status == "failed":
    if not task.error_message:
        return "AGENT_CRASHED_NO_ERROR"

    if task.error_message in ["SIGKILL", "SIGTERM", "OOM"]:
        return "AGENT_CRASHED_SIGNAL"
```

### 2.3 任务类异常

#### 2.3.1 任务卡死（TaskStalled）

**定义**: 任务进度在规定时间内无任何进展

**检测特征**:
| 特征 | 阈值 | 说明 |
|------|------|------|
| 进度停滞 | 30分钟 | progress字段无变化 |
| 状态不变 | 30分钟 | status字段无变化 |
| 无日志更新 | 30分钟 | logs无新条目 |

**数据来源**:
```python
# 从 workspace/tasks/{question_id}/board.json 检测
board = load_board(question_id)

# 特征：进度停滞
if board.updated_at:
    time_since_update = now() - board.updated_at
    if time_since_update > 30 * 60:  # 30分钟
        # 检查所有expert状态
        for expert in board.experts:
            if expert.status == "running":
                # 该expert是否也在停滞
                if now() - expert.updated_at > 30 * 60:
                    return "TASK_STALLED"
```

#### 2.3.2 任务超时（TaskTimeout）

**定义**: 任务执行时间超过预估时间

**检测特征**:
| 特征 | 阈值 | 说明 |
|------|------|------|
| 执行时间 | > 预估时间 × 2 | 超过预估2倍 |
| 辩论轮次 | > 20轮 | 辩论超过20轮 |
| 等待时间 | > 600秒 | 等待其他任务超过10分钟 |

**数据来源**:
```python
# 从 workspace/tasks/{question_id}/metadata.json 检测
metadata = load_metadata(question_id)

# 特征：执行超时
if metadata.status == "running":
    elapsed = now() - metadata.started_at
    estimated = metadata.estimated_duration
    if elapsed > estimated * 2:
        return "TASK_TIMEOUT"

# 特征：辩论超时
if board.phase == "debate":
    debate_duration = now() - board.debate_started_at
    if debate_duration > 60 * 60:  # 60分钟
        return "DEBATE_TIMEOUT"
```

#### 2.3.3 任务失败（TaskFailed）

**定义**: 任务执行失败

**检测特征**:
| 特征 | 说明 |
|------|------|
| 状态为failed | status = "failed" |
| 失败原因 | error_type 不为空 |
| 重试次数 | retry_count > 3 |

**数据来源**:
```python
# 从 workspace/tasks/{question_id}/board.json 检测
board = load_board(question_id)

# 统计失败
failed_experts = [e for e in board.experts if e.status == "failed"]
failed_count = len(failed_experts)
total_count = len(board.experts)

# 特征：过多专家失败
if failed_count / total_count > 0.5:
    return "TASK_FAILED_TOO_MANY_EXPERTS"

# 特征：关键专家失败
critical_experts = ["kahneman", "first-principle", "inversion"]
failed_critical = [e for e in failed_experts if e.id in critical_experts]
if failed_critical:
    return "TASK_FAILED_CRITICAL_EXPERT"
```

### 2.4 协作类异常

#### 2.4.1 消息丢失（MessageLost）

**定义**: Agent间消息未能送达

**检测特征**:
| 特征 | 阈值 | 说明 |
|------|------|------|
| 发送成功 | status != "delivered" | 消息未送达 |
| 投递延迟 | > 30秒 | 投递超过30秒 |
| 死信 | > 3次重试 | 重试超过3次 |

**数据来源**:
```python
# 从 workspace/messages/{question_id}/ 检测
messages = load_messages(question_id)

# 特征：消息未送达
for msg in messages:
    if msg.status == "failed":
        return "MESSAGE_LOST"

    if msg.delivered_at and msg.sent_at:
        delay = msg.delivered_at - msg.sent_at
        if delay > 30:
            return "MESSAGE_DELAYED"

# 特征：消息堆积（发送多但确认少）
sent_count = len([m for m in messages if m.status in ["sent", "delivered"]])
acknowledged_count = len([m for m in messages if m.status == "acknowledged"])
if sent_count - acknowledged_count > 50:
    return "MESSAGE_BACKLOG"
```

#### 2.4.2 辩论僵局（DebateDeadlock）

**定义**: 辩论过程中观点僵持，无法达成共识

**检测特征**:
| 特征 | 说明 |
|------|------|
| 轮次过多 | > 15轮无共识 |
| 观点重复 | 连续3轮观点相似 |
| 无新观点 | 连续3轮无新信息 |

**数据来源**:
```python
# 从 workspace/tasks/{question_id}/debate/ 检测
debate = load_debate(question_id)

# 特征：轮次过多
if len(debate.rounds) > 15 and not debate.consensus_reached:
    return "DEBATE_DEADLOCK_ROUNDS"

# 特征：观点重复
recent_rounds = debate.rounds[-3:]
if all_same_viewpoint(recent_rounds):
    return "DEBATE_DEADLOCK_REPETITION"

# 特征：无新观点
recent_info = sum(r.new_information for r in recent_rounds)
if recent_info == 0:
    return "DEBATE_DEADLOCK_NO_PROGRESS"
```

## 三、异常检测算法

### 3.1 检测方法对比

| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| **阈值检测** | 超时、积压 | 简单直接 | 需要预设阈值 |
| **状态机** | 任务生命周期 | 清晰可控 | 状态复杂时难维护 |
| **模式识别** | 输出异常 | 灵活 | 需要训练数据 |
| **对比分析** | 异常检测 | 准确 | 需要基准数据 |
| **趋势预测** | 资源预警 | 提前发现 | 实现复杂 |

### 3.2 阈值检测算法

```python
# detector/threshold.py

from typing import Dict, List, Optional
from dataclasses import dataclass
from datetime import datetime, timedelta
import json
from pathlib import Path

@dataclass
class ThresholdRule:
    """阈值规则"""
    metric_id: str
    operator: str  # >, <, >=, <=, ==
    value: float
    duration_seconds: Optional[int] = None  # 持续时间

@dataclass
class Anomaly:
    """异常记录"""
    anomaly_id: str
    type: str
    severity: str  # critical, warning, info
    source: str
    details: Dict
    timestamp: datetime
    recommended_action: str

class ThresholdDetector:
    """阈值检测器"""

    def __init__(self, config_path: str):
        self.rules = self._load_rules(config_path)

    def detect(self, metrics: Dict) -> List[Anomaly]:
        """检测异常"""
        anomalies = []

        for rule in self.rules:
            value = self._get_metric_value(metrics, rule.metric_id)

            if self._evaluate(rule, value):
                anomaly = self._create_anomaly(rule, value, metrics)
                anomalies.append(anomaly)

        return anomalies

    def _evaluate(self, rule: ThresholdRule, value: float) -> bool:
        """评估规则"""
        if rule.operator == ">":
            return value > rule.value
        elif rule.operator == "<":
            return value < rule.value
        elif rule.operator == ">=":
            return value >= rule.value
        elif rule.operator == "<=":
            return value <= rule.value
        elif rule.operator == "==":
            return value == rule.value
        return False

# 配置示例
# thresholds.json
{

      "id": "agent_response_timeout",
```

### 3.3 状态机检测算法

```python
# detector/state_machine.py

from enum import Enum
from typing import Dict, Set

class TaskState(Enum):
    """任务状态"""
    PENDING = "pending"
    INITIALIZING = "initializing"
    RUNNING = "running"
    DEBATing = "debating"
    INTEGRATING = "integrating"
    COMPLETED = "completed"
    FAILED = "failed"
    STALLED = "stalled"
    TIMEOUT = "timeout"

# 状态转换图
VALID_TRANSITIONS = {
    TaskState.PENDING: {TaskState.INITIALIZING, TaskState.FAILED},
    TaskState.INITIALIZING: {TaskState.RUNNING, TaskState.FAILED, TaskState.TIMEOUT},
    TaskState.RUNNING: {TaskState.DEBATING, TaskState.INTEGRATING, TaskState.FAILED, TaskState.STALLED, TaskState.TIMEOUT},
    TaskState.DEBATING: {TaskState.INTEGRATING, TaskState.FAILED, TaskState.STALLED, TaskState.TIMEOUT},
    TaskState.INTEGRATING: {TaskState.COMPLETED, TaskState.FAILED, TaskState.TIMEOUT},
}

class StateMachineDetector:
    """状态机检测器"""

    def __init__(self):
        self.state_history: Dict[str, List[TaskState]] = {}

    def check_transition(self, task_id: str, from_state: TaskState, to_state: TaskState) -> Optional[str]:
        """检查状态转换是否合法"""

        # 记录历史
        if task_id not in self.state_history:
            self.state_history[task_id] = []
        self.state_history[task_id].append(to_state)

        # 检查转换合法性
        if to_state not in VALID_TRANSITIONS.get(from_state, set()):
            return f"INVALID_TRANSITION: {from_state} -> {to_state}"

        # 检查异常状态
        if to_state == TaskState.STALLED:
            return "TASK_STALLED_DETECTED"

        if to_state == TaskState.TIMEOUT:
            return "TASK_TIMEOUT_DETECTED"

        # 检查状态循环（卡死）
        if self._is_stuck_in_loop(task_id):
            return "TASK_IN_STATE_LOOP"

        return None

    def _is_stuck_in_loop(self, task_id: str) -> bool:
        """检查是否陷入状态循环"""
        history = self.state_history.get(task_id, [])
        if len(history) < 6:
            return False

        # 检查最后6个状态是否在来回切换
        recent = history[-6:]
        if recent[0] == recent[2] == recent[4] and recent[1] == recent[3] == recent[5]:
            return True
        return False
```

### 3.4 模式识别算法

```python
# detector/pattern.py

import re
from typing import Dict, List
from collections import Counter

class PatternDetector:
    """模式识别检测器"""

    def __init__(self):
        self.output_patterns = self._load_patterns()

    def detect_output_anomaly(self, output: str) -> List[Dict]:
        """检测输出异常模式"""
        anomalies = []

        # 模式1：空输出
        if not output or not output.strip():
            anomalies.append({
                "type": "EMPTY_OUTPUT",
                "confidence": 1.0,
                "details": "输出为空"
            })

        # 模式2：过短输出
        elif len(output.strip()) < 100:
            anomalies.append({
                "type": "SHORT_OUTPUT",
                "confidence": 0.8,
                "details": f"输出长度仅{len(output)}字符"
            })

        # 模式3：重复内容
        repeats = self._find_repeats(output)
        if repeats:
            anomalies.append({
                "type": "REPETITIVE_OUTPUT",
                "confidence": 0.9,
                "details": f"发现{repeats}处重复"
            })

        # 模式4：错误关键词
        error_keywords = [
            "error", "failed", "exception", "crashed",
            "无法完成", "失败", "异常", "崩溃"
        ]
        found_errors = [kw for kw in error_keywords if kw.lower() in output.lower()]
        if found_errors:
            anomalies.append({
                "type": "ERROR_KEYWORDS",
                "confidence": 0.7,
                "details": f"发现错误关键词: {found_errors}"
            })

        # 模式5：格式异常
        if self._is_invalid_format(output):
            anomalies.append({
                "type": "INVALID_FORMAT",
                "confidence": 0.8,
                "details": "输出格式不符合预期"
            })

        return anomalies

    def _find_repeats(self, text: str, min_length: int = 50) -> int:
        """查找重复段落"""
        words = text.split()
        word_counts = Counter(words)

        # 找出出现频率过高的词
        total_words = len(words)
        repeats = sum(1 for count in word_counts.values()
                     if count > total_words * 0.1 and count > 3)
        return repeats

    def _is_invalid_format(self, output: str) -> bool:
        """检查格式是否无效"""
        # 检查JSON格式
        if output.strip().startswith("{"):
            try:
                import json
                json.loads(output)
                return False
            except:
                return True

        # 检查markdown格式
        if output.strip().startswith("#"):
            return False

        # 如果既不是有效JSON也不是有效markdown，可能是异常
        return len(output.strip()) > 0
```

```python
# detector/comparison.py

import numpy as np
from typing import Dict, List
from datetime import datetime, timedelta

class ComparisonDetector:
    """对比分析检测器"""

    def __init__(self, history_path: str):
        self.baseline_metrics = self._load_baseline(history_path)

    def detect_deviation(self, current_metrics: Dict) -> List[Dict]:
        """检测偏离基线的异常"""
        anomalies = []

        for metric_id, baseline in self.baseline_metrics.items():
            current_value = current_metrics.get(metric_id)
            if current_value is None:
                continue

            # 计算偏离程度
            deviation = self._calculate_deviation(
                current_value,
                baseline["mean"],
                baseline["std"]
            )

            # 超过2个标准差视为异常
            if abs(deviation) > 2:
                anomalies.append({
                    "type": "METRIC_DEVIATION",
                    "metric_id": metric_id,
                    "current_value": current_value,
                    "baseline_mean": baseline["mean"],
                    "deviation_sigma": deviation,
                    "severity": "critical" if abs(deviation) > 3 else "warning"
                })

        return anomalies

    def _calculate_deviation(self, value: float, mean: float, std: float) -> float:
        """计算偏离程度（标准差倍数）"""
        if std == 0:
            return 0 if value == mean else float('inf')
        return (value - mean) / std

    def compare_between_agents(self, question_id: str) -> List[Dict]:
        """对比同一任务中不同Agent的表现"""
        board = load_board(question_id)
        anomalies = []

        # 获取所有expert的执行时间
        durations = {}
        for expert in board.experts:
            if expert.completed_at and expert.started_at:
                durations[expert.id] = (expert.completed_at - expert.started_at).total_seconds()

        if not durations:
            return anomalies

        # 计算统计量
        values = list(durations.values())
        mean_time = np.mean(values)
        std_time = np.std(values)

        # 找出异常慢的expert
        for expert_id, duration in durations.items():
            if std_time > 0 and (duration - mean_time) / std_time > 2:
                anomalies.append({
                    "type": "AGENT_SLOW",
                    "expert_id": expert_id,
                    "duration": duration,
                    "mean_duration": mean_time,
                    "deviation": (duration - mean_time) / std_time
                })

        return anomalies
```

### 3.6 趋势预测算法

```python
# detector/trend.py

import numpy as np
from typing import List, Dict
from datetime import datetime, timedelta

class TrendDetector:
    """趋势预测检测器"""

    def __init__(self):
        self.metrics_history = {}

    def detect_trend_anomaly(self, metric_id: str, window_size: int = 10) -> Dict:
        """检测趋势异常"""
        history = self.metrics_history.get(metric_id, [])

        if len(history) < window_size:
            return {"anomaly": False, "reason": "数据不足"}

        # 使用滑动窗口计算趋势
        recent = history[-window_size:]

        # 线性回归
        x = np.arange(len(recent))
        y = np.array(recent)
        slope, intercept = np.polyfit(x, y, 1)

        # 计算加速度（二阶导数近似）
        velocities = np.diff(recent)
        accelerations = np.diff(velocities)
        avg_acceleration = np.mean(accelerations)

        # 检测异常趋势
        anomalies = []

        # 趋势1：持续上升
        if slope > 0.1:  # 阈值需根据具体指标调整
            anomalies.append({
                "type": "TREND_UP",
                "slope": slope,
                "description": f"{metric_id}持续上升"
            })

        # 趋势2：急剧上升（加速度过大）
        if avg_acceleration > np.std(accelerations) * 2:
            anomalies.append({
                "type": "TREND_ACCELERATING",
                "acceleration": avg_acceleration,
                "description": f"{metric_id}上升加速"
            })

        # 趋势3：即将达到阈值
        if len(recent) > 0:
            latest = recent[-1]
            threshold = self._get_threshold(metric_id)
            if threshold and latest > threshold * 0.9:
                anomalies.append({
                    "type": "TREND_THRESHOLD_NEAR",
                    "current": latest,
                    "threshold": threshold,
                    "eta_to_threshold": self._estimate_time_to_threshold(
                        slope, latest, threshold
                    )
                })

        return {
            "anomaly": len(anomalies) > 0,
            "details": anomalies,
            "trend_slope": slope
        }

    def _estimate_time_to_threshold(self, slope: float, current: float, threshold: float) -> str:
        """估算到达阈值的时间"""
        if slope <= 0:
            return "不会到达"

        remaining = threshold - current
        units = remaining / slope

        if units < 1:
            return f"{int(units * 60)}分钟"
        elif units < 60:
            return f"{int(units)}小时"
        else:
            return f"{int(units / 24)}天"
```

```python
# recovery/agent_recovery.py

from typing import Dict, Optional
from dataclasses import dataclass
from datetime import datetime
import json

@dataclass
class RecoveryAction:
    """恢复动作"""
    action_id: str
    action_type: str  # restart, retry, skip, fallback, abort
    target: str
    parameters: Dict
    max_attempts: int = 3

class AgentRecovery:
    """Agent异常恢复"""

    def __init__(self, config: Dict):
        self.config = config
        self.recovery_history = {}

    def handle_anomaly(self, anomaly: Dict) -> Optional[RecoveryAction]:
        """处理异常，返回恢复动作"""
        anomaly_type = anomaly["type"]

        # 根据异常类型选择恢复策略
        handlers = {
            "AGENT_NOT_RESPONDING": self._handle_not_responding,
            "AGENT_OUTPUT_EMPTY": self._handle_output_empty,
            "AGENT_CRASHED": self._handle_crashed,
            "TASK_STALLED": self._handle_stalled,
            "TASK_TIMEOUT": self._handle_timeout,
            "MESSAGE_LOST": self._handle_message_lost,
            "DEBATE_DEADLOCK": self._handle_deadlock,
            "QUEUE_OVERFLOW": self._handle_queue_overflow,
        }

        handler = handlers.get(anomaly_type)
        if handler:
            return handler(anomaly)

        return None

    def _handle_not_responding(self, anomaly: Dict) -> RecoveryAction:
        """处理Agent无响应"""
        expert_id = anomaly["source"]["expert_id"]
        question_id = anomaly["source"]["question_id"]

        # 获取重试次数
        retry_count = self._get_retry_count(expert_id, question_id)

        if retry_count < self.config.max_retries:
            return RecoveryAction(
                action_id=f"restart_{expert_id}_{question_id}",
                action_type="restart",
                target=expert_id,
                parameters={
                    "question_id": question_id,
                    "reason": "无响应",
                    "retry_count": retry_count + 1
                },
                max_attempts=self.config.max_retries
            )
        else:
            # 超过重试次数，跳过该Agent
            return RecoveryAction(
                action_id=f"skip_{expert_id}_{question_id}",
                action_type="skip",
                target=expert_id,
                parameters={
                    "question_id": question_id,
                    "reason": "多次无响应"
                }
            )

    def _handle_output_empty(self, anomaly: Dict) -> RecoveryAction:
        """处理输出为空"""
        expert_id = anomaly["source"]["expert_id"]
        question_id = anomaly["source"]["question_id"]

        return RecoveryAction(
            action_id=f"retry_{expert_id}_{question_id}",
            action_type="retry",
            target=expert_id,
            parameters={
                "question_id": question_id,
                "reason": "输出为空",
                "clear_context": True
            }
        )

    def _handle_crashed(self, anomaly: Dict) -> RecoveryAction:
        """处理Agent崩溃"""
        expert_id = anomaly["source"]["expert_id"]
        question_id = anomaly["source"]["question_id"]

        return RecoveryAction(
            action_id=f"restart_crashed_{expert_id}",
            action_type="restart",
            target=expert_id,
            parameters={
                "question_id": question_id,
                "reason": "Agent崩溃",
                "preserve_state": False,
                "retry_count": 0
            }
        )

    def _handle_stalled(self, anomaly: Dict) -> RecoveryAction:
        """处理任务卡死"""
        question_id = anomaly["source"]["question_id"]

        return RecoveryAction(
            action_id=f"restart_task_{question_id}",
            action_type="restart",
            target="task",
            parameters={
                "question_id": question_id,
                "reason": "任务卡死",
                "resume_from_checkpoint": True
            }
        )

    def _handle_timeout(self, anomaly: Dict) -> RecoveryAction:
        """处理任务超时"""
        question_id = anomaly["source"]["question_id"]

        # 获取已完成的expert
        completed = anomaly["details"].get("completed_experts", [])

        return RecoveryAction(
            action_id=f"retry_task_{question_id}",
            action_type="retry",
            target="task",
            parameters={
                "question_id": question_id,
                "reason": "任务超时",
                "skip_completed": completed,
                "increase_timeout": True
            }
        )

    def _handle_message_lost(self, anomaly: Dict) -> RecoveryAction:
        """处理消息丢失"""
        message_id = anomaly["source"]["message_id"]

        return RecoveryAction(
            action_id=f"resend_{message_id}",
            action_type="resend",
            target="message",
            parameters={
                "message_id": message_id,
                "priority": "high"
            }
        )

    def _handle_deadlock(self, anomaly: Dict) -> RecoveryAction:
        """处理辩论僵局"""
        question_id = anomaly["source"]["question_id"]

        return RecoveryAction(
            action_id=f"end_debate_{question_id}",
            action_type="end_debate",
            target="debate",
            parameters={
                "question_id": question_id,
                "reason": "辩论僵局",
                "force_consensus": True
            }
        )

    def _handle_queue_overflow(self, anomaly: Dict) -> RecoveryAction:
        """处理队列积压"""
        return RecoveryAction(
            action_id="rate_limit",
            action_type="rate_limit",
            target="system",
            parameters={
                "limit_new_tasks": True,
                "duration_minutes": 30
            }
        )

    def _get_retry_count(self, expert_id: str, question_id: str) -> int:
        """获取重试次数"""
        key = f"{expert_id}_{question_id}"
        return self.recovery_history.get(key, {}).get("retry_count", 0)

    def record_recovery(self, action: RecoveryAction, result: str):
        """记录恢复结果"""
        key = f"{action.target}_{action.parameters.get('question_id', '')}"

        if key not in self.recovery_history:
            self.recovery_history[key] = {
                "retry_count": 0,
                "last_action": None,
                "history": []
            }

        self.recovery_history[key]["history"].append({
            "action": action.action_type,
            "result": result,
            "timestamp": datetime.now().isoformat()
        })

        if result == "success":
            self.recovery_history[key]["retry_count"] = 0
        else:
            self.recovery_history[key]["retry_count"] += 1
```

### 9.1 异常统计指标

```python
def generate_anomaly_report(date: str) -> Dict:
    """生成异常分析报告"""

    anomaly_types = {}
    for alert in get_alerts(date):
        anomaly_type = alert.anomaly_type
        anomaly_types[anomaly_type] = anomaly_types.get(anomaly_type, 0) + 1

    # 统计恢复情况
    recovery_stats = {
        "auto_recovered": 0,
        "manual_resolved": 0,
        "pending": 0
    }

    # 统计恢复时间
    recovery_times = []

    # 统计影响任务
    affected_tasks = set()

    return {
        "date": date,
        "summary": {
            "total_anomalies": len(alerts),
            "anomaly_rate": len(alerts) / get_total_tasks(date),
            "auto_recovery_rate": recovery_stats["auto_recovered"] / len(alerts)
        },
        "by_type": anomaly_types,
        "recovery": recovery_stats,
        "recovery_time_avg": avg(recovery_times),
```
