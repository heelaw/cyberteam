# 思考天团系统监控指标设计

**问题**: 如何设计思考天团（14个思维专家 + 运营Agent）的系统监控指标体系？

---

## 一、监控体系设计原则

### 1.1 设计目标

| 目标 | 说明 |
|------|------|
| **实时可见** | 任务进度、Agent状态实时可查 |
| **问题预警** | 异常情况及时告警，快速定位问题 |
| **性能优化** | 发现性能瓶颈，持续优化 |
| **质量评估** | 专家表现可量化，持续改进 |
| **容量规划** | 资源使用可预测，支持扩展 |

### 1.2 监控层级

```
┌─────────────────────────────────────────────────────────────────┐
│                        监控层级                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ L1. 系统级监控 - 整体健康度                              │   │
│  │   任务成功率、响应时间、吞吐量、错误率                    │   │
│  └─────────────────────────────────────────────────────────┘   │
│                              │                                   │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │ L2. Agent级监控 - 专家与运营Agent表现                    │   │

│  └─────────────────────────────────────────────────────────┘   │
```

### 3.1 任务成功率（Task Success Rate）

| 属性 | 值 |
|------|-----|
| **指标名称** | 任务成功率 |
| **英文名** | Task Success Rate |
| **指标ID** | SYS_TASK_SUCCESS |
| **计算公式** | success_count / total_count |

**度量方式**:

```python
# 统计周期：每小时/每天
# 数据来源：workspace/tasks/{question_id}/board.json

def calculate_task_success(date: str) -> dict:
    tasks = load_all_tasks(date)

    total = len(tasks)
    success = sum(1 for t in tasks if t.status == "completed")
    failed = sum(1 for t in tasks if t.status == "failed")
    abandoned = sum(1 for t in tasks if t.status == "abandoned")

    return {
        "total": total,
        "success": success,
        "failed": failed,
        "abandoned": abandoned,
        "rate": success / total if total > 0 else 0
    }
```

**正常范围与告警阈值**:

| 级别 | 阈值 | 说明 |
|------|------|------|
| 绿色 | >= 95% | 正常运行 |
| 黄色 | 80% - 95% | 需要关注 |
| 橙色 | 60% - 80% | 预警 |
| 红色 | < 60% | 紧急告警 |

## 四、Agent级监控指标

### 4.1 Agent活跃度（Agent Activity）

| 属性 | 值 |
|------|-----|
| **指标名称** | Agent活跃度 |
| **英文名** | Agent Activity |
| **指标ID** | AGT_ACTIVITY |
| **计算公式** | tasks_assigned / time_window |

**度量方式**:

```python
def calculate_agent_activity(expert_id: str, date: str) -> dict:
    tasks = load_expert_tasks(expert_id, date)

    assigned = len(tasks)
    completed = sum(1 for t in tasks if t.status == "completed")
    failed = sum(1 for t in tasks if t.status == "failed")

    # 利用率：完成任务时间 / 总可用时间
    total_time = sum(t.duration for t in tasks if t.status == "completed")
    available_time = 8 * 3600  # 假设每天8小时
    utilization = total_time / available_time

    return {
        "tasks_assigned": assigned,
        "tasks_completed": completed,
        "tasks_failed": failed,
        "success_rate": completed / assigned if assigned > 0 else 0,
        "utilization": min(utilization, 1.0)
    }
```

**正常范围与告警阈值**:

| 级别 | 利用率 | 任务完成率 | 失败率 |
|------|-------|-----------|--------|
| 绿色 | >= 60% | >= 90% | < 5% |
| 黄色 | 40-60% | 75-90% | 5-10% |
| 橙色 | 20-40% | 50-75% | 10-20% |
| 红色 | < 20% | < 50% | > 20% |

### 4.2 Agent响应延迟（Agent Latency）

| 属性 | 值 |
|------|-----|
| **指标名称** | Agent响应延迟 |
| **英文名** | Agent Latency |
| **指标ID** | AGT_LATENCY |
| **计算公式** | task.started_at - task.assigned_at |

**度量方式**:

```python
def calculate_agent_latency(expert_id: str, date: str) -> dict:
    tasks = load_expert_tasks(expert_id, date)

    startup_times = []
    execution_times = []

    for task in tasks:
        # 启动延迟：从分配到开始执行
        if task.started_at and task.assigned_at:
            startup = (task.started_at - task.assigned_at).total_seconds()
            startup_times.append(startup)

        # 执行时间：从开始到完成
        if task.completed_at and task.started_at:
            execution = (task.completed_at - task.started_at).total_seconds()
            execution_times.append(execution)

    return {
        "avg_startup_seconds": np.mean(startup_times) if startup_times else 0,
        "avg_execution_seconds": np.mean(execution_times) if execution_times else 0,
        "p90_startup": np.percentile(startup_times, 90) if startup_times else 0,
        "p90_execution": np.percentile(execution_times, 90) if execution_times else 0
    }
```

**正常范围与告警阈值**（单位：秒）:

| 级别 | 启动延迟 | 执行时间(P90) |
|------|---------|--------------|
| 绿色 | < 10 | < 180 |
| 黄色 | 10-30 | 180-300 |
| 橙色 | 30-60 | 300-480 |
| 红色 | > 60 | > 480 |

```python
def calculate_expert_participation(expert_id: str, date: str) -> dict:
    debates = load_debates(date)

    participated = sum(1 for d in debates if expert_id in d.participants)
```

### 5.2 任务耗时（Task Duration）

| 属性 | 值 |
|------|-----|
| **指标名称** | 任务耗时 |
| **英文名** | Task Duration |
| **指标ID** | TSK_DURATION |
| **计算公式** | completed_at - created_at |

**度量方式**:

```python
def calculate_task_duration(question_id: str) -> dict:
    task = load_task(question_id)

    if task.status != "completed":
        elapsed = (datetime.now() - task.created_at).total_seconds()
        return {
            "question_id": question_id,
            "status": task.status,
            "elapsed_seconds": elapsed,
            "estimated_total": task.estimated_duration
        }

    total_duration = (task.completed_at - task.created_at).total_seconds()

    expert_durations = {}
    for expert in task.experts:
        if expert.completed_at and expert.started_at:
            expert_durations[expert.id] = (
                expert.completed_at - expert.started_at
            ).total_seconds()

    return {
        "question_id": question_id,
        "total_duration_seconds": total_duration,
        "by_expert": expert_durations,
        "estimated_vs_actual": {
            "estimated": task.estimated_duration,
            "actual": total_duration,
            "variance": (total_duration - task.estimated_duration) / task.estimated_duration
        }
    }
```

**正常范围与告警阈值**（单位：分钟）:

| 级别 | 预估偏差 | 说明 |
|------|---------|------|
| 绿色 | < 20% | 正常完成 |
| 黄色 | 20-50% | 轻微超时 |
| 橙色 | 50-100% | 中度超时 |
| 红色 | > 100% | 严重超时 |

### 6.1 消息队列健康度（Message Queue Health）

| 属性 | 值 |
|------|-----|
| **指标名称** | 消息队列健康度 |
| **英文名** | Message Queue Health |
| **指标ID** | COL_MESSAGE_QUEUE |
| **计算公式** | 成功投递 / 总投递数 |

**度量方式**:

```python
def calculate_message_queue_health(question_id: str) -> dict:
    messages = load_messages(question_id)

    total = len(messages.messages)
    delivered = sum(1 for m in messages if m.status == "delivered")
    failed = sum(1 for m in messages if m.status == "failed")
    pending = sum(1 for m in messages if m.status == "pending")

    # 平均投递延迟
    delivery_latencies = [
        (m.delivered_at - m.sent_at).total_seconds()
        for m in messages if m.delivered_at
    ]

    return {
        "total_messages": total,
        "delivered": delivered,
        "failed": failed,
        "pending": pending,
        "delivery_rate": delivered / total if total > 0 else 0,
        "avg_delivery_latency": np.mean(delivery_latencies) if delivery_latencies else 0
    }
```

**正常范围与告警阈值**:

| 级别 | 投递率 | 失败率 | 平均延迟 |

|------|-----|

**度量方式**:

```python
def calculate_debate_quality(question_id: str) -> dict:
    debate = load_debate(question_id)

    # 参与度
    participation = len(debate.participants) / debate.expected_participants

    # 轮次充分性
    rounds = len(debate.rounds)
    expected_rounds = debate.expected_rounds
    rounds_adequacy = min(rounds / expected_rounds, 1.0) if expected_rounds > 0 else 0

    # 共识达成
    consensus_reached = 1.0 if debate.consensus_reached else 0.0

    # 观点多样性
    unique_views = len(set(debate.get_all_viewpoints()))
    view_diversity = unique_views / len(debate.participants)

    # 综合评分
    quality_score = (
        participation * 0.25 +
        rounds_adequacy * 0.25 +
        consensus_reached * 0.3 +
        view_diversity * 0.2
    )

    return {
        "participation_rate": participation,
        "rounds_adequacy": rounds_adequacy,
        "consensus_reached": consensus_reached,
        "view_diversity": view_diversity,
        "quality_score": quality_score,
        "total_rounds": rounds
    }
```

| 级别 | 质量分数 | 参与率 | 共识达成率 |

| 绿色 | >= 0.8 | >= 80% | >= 70% |
| 黄色 | 0.6-0.8 | 60-80% | 50-70% |
| 橙色 | 0.4-0.6 | 40-60% | 30-50% |
| 红色 | < 0.4 | < 40% | < 30% |

```python
def calculate_output_quality(question_id: str) -> dict:
    report = load_report(question_id)

    # 完整性
    required_sections = ["问题定义", "专家分析", "辩论过程", "决策建议", "风险提示"]
    missing_sections = [s for s in required_sections if s not in report.sections]
    completeness = 1.0 - len(missing_sections) / len(required_sections)

    # 专家覆盖率
    expert_coverage = len(report.referenced_experts) / 14.0

    # 决策建议可执行性
    actionability = rate_actionability(report.decisions)

    # 用户满意度（如有反馈）
    user_satisfaction = report.user_feedback.score if report.user_feedback else None

    return {
        "completeness": completeness,
        "expert_coverage": expert_coverage,
        "actionability": actionability,
        "user_satisfaction": user_satisfaction,
        "quality_score": (
            completeness * 0.3 +
            expert_coverage * 0.2 +
            actionability * 0.3 +
            (user_satisfaction or 0.2) * 0.2
        )
    }
```

### 8.2 数据采集脚本

```python
# scripts/collector.py
import json
import os
from datetime import datetime, timedelta
from pathlib import Path

class MetricsCollector:
    def __init__(self, base_path: str):
        self.base_path = Path(base_path)
        self.monitoring_path = self.base_path / "monitoring"

    def collect_system_metrics(self) -> dict:
        """采集系统级指标"""
        date = datetime.now().strftime("%Y-%m-%d")
        hour = datetime.now().hour

        # 任务成功率
        task_success = self._calculate_task_success()

        # 响应时间
        response_time = self._calculate_response_time()

        # 吞吐量
        throughput = self._calculate_throughput()

        # 错误率
        error_rate = self._calculate_error_rate()

        # 队列深度
        queue_depth = self._calculate_queue_depth()

        metrics = {
            "date": date,
            "hour": hour,
            "timestamp": datetime.now().isoformat(),
            "metrics": {
                "task_success": task_success,
                "response_time": response_time,
                "throughput": throughput,
                "error_rate": error_rate,
                "queue_depth": queue_depth
            }
        }

        # 保存到文件
        self._save_metrics(metrics, "system", date, hour)

        return metrics

    def collect_agent_metrics(self, expert_id: str) -> dict:
        """采集Agent级指标"""
        date = datetime.now().strftime("%Y-%m-%d")

        activity = self._calculate_agent_activity(expert_id)
        latency = self._calculate_agent_latency(expert_id)
        quality = self._calculate_agent_quality(expert_id)

        metrics = {
            "expert_id": expert_id,
            "date": date,
            "timestamp": datetime.now().isoformat(),
            "metrics": {
                "activity": activity,
                "latency": latency,
                "quality": quality
            }
        }

        # 保存到文件
        self._save_agent_metrics(metrics, expert_id, date)

        return metrics

    def collect_task_metrics(self, question_id: str) -> dict:
        """采集任务级指标"""
        progress = self._calculate_task_progress(question_id)
        duration = self._calculate_task_duration(question_id)

        metrics = {
            "question_id": question_id,
            "timestamp": datetime.now().isoformat(),
            "metrics": {
                "progress": progress,
                "duration": duration
            }
        }

        return metrics

    def _save_metrics(self, metrics: dict, category: str, date: str, hour: int):
        """保存指标到文件"""
        file_path = self.monitoring_path / "metrics" / category / date / f"hour_{hour:02d}.json"
        file_path.parent.mkdir(parents=True, exist_ok=True)

        with open(file_path, "w") as f:
            json.dump(metrics, f, indent=2, ensure_ascii=False)

    def _save_agent_metrics(self, metrics: dict, expert_id: str, date: str):
        """保存Agent指标到文件"""
        file_path = self.monitoring_path / "metrics" / "agent" / expert_id / f"{date}.json"
        file_path.parent.mkdir(parents=True, exist_ok=True)

        with open(file_path, "w") as f:
            json.dump(metrics, f, indent=2, ensure_ascii=False)
```
