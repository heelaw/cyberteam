## 二、Skill定义模式分析

### 2.1 标准Skill目录结构

```
skill-name/
├── SKILL.md (必需)
│   ├── YAML frontmatter metadata (必需)
│   └── Markdown instructions (必需)
└── Bundled Resources (可选)
    ├── scripts/          - 可执行代码
    ├── references/       - 按需加载的文档
    └── assets/          - 模板、图标等
```

### 2.2 Frontmatter格式

**必需字段**:
```yaml
---
name: skill-name                    # 技能名称（英文、kebab-case）
description: 技能描述...            # 触发条件描述
---
```

**扩展字段（claude-skills使用）**:
```yaml
---
name: code-reviewer
description: Analyzes code diffs and files to identify bugs...
metadata:
  author: https://github.com/Jeffallan
  version: "1.1.0"
  domain: quality                     # 领域分类
  triggers: code review, PR review... # 触发关键词
  role: specialist                   # 角色类型
  scope: review                      # 作用域
  output-format: report              # 输出格式
  related-skills: security-reviewer... # 相关技能
---
```

### 2.3 SKILL.md正文结构

```markdown
## When to Use This Skill
- 场景1
- 场景2

## Core Workflow
1. **Step Name** — 描述
   > Checkpoint: 检查点

2. **Step Name** — 描述

## Reference Guide
| Topic | Reference | Load When |
|-------|-----------|-----------|
| 主题 | `references/file.md` | 加载条件 |

## Constraints
### MUST DO
- 规则1
- 规则2

### MUST NOT DO
- 禁止1
- 禁止2

## Output Templates
输出模板定义

## Knowledge Reference
技术栈、知识域列表
```

### 2.4 三级加载系统

```
Level 1: 元数据（name + description）- 始终在上下文（约100词）
Level 2: SKILL.md正文 - 技能触发时加载（<5k词）
Level 3: 捆绑资源 - 按需加载（脚本可执行不占用上下文）
```

### 2.5 整合建议

| 维度 | 当前思考天团 | 建议改进 |
|------|-------------|----------|
| 触发机制 | description + 关键词 | 增加triggers字段 |
| 工作流 | 简要描述 | 增加Checkpoint检查点 |
| 约束条件 | Critical Rules | 统一为MUST DO/MUST NOT DO |
| 输出格式 | 未明确 | 增加output-format |

## 五、技术栈实现分析

### 5.1 PUA机制（pua-main）

**核心架构**:
```
┌─────────────────────────────────────────────────┐
│              PUA 压力升级系统                     │
├─────────────────────────────────────────────────┤
│  L1 (第2次失败): 温和失望 → 换本质不同方案      │
│  L2 (第3次失败): 灵魂拷问 → 搜索+源码+列假设    │
│  L3 (第4次失败): 361考核 → 完成7项检查清单     │
│  L4 (第5次+):   毕业警告 → 拼命模式            │
└─────────────────────────────────────────────────┘
```

**Hook系统集成**:
```json
// PreCompact Hook: 上下文压缩前自动保存状态
{
  "PreCompact": [{
    "matcher": "*",
    "hooks": [{
      "type": "prompt",
      "prompt": "[PUA v2 PreCompact — State Checkpoint]\n\n写 ~/.pua/builder-journal.md..."
    }]
  }]
}
```

```
                              └── CLAUDE.md files --规则统计
                              │

```
