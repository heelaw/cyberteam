## 路径1：营业指标增长

### 触发关键词
- "帮我增长下个月营业指标"
- "提升本月收入"
- "下个季度GMV目标"
- "营业指标增长方案"

### 涉及部门/专家
| 角色 | 部门 | 职责 |
|------|------|------|
| 策略执行Agent | 运营部（中书省） | 主责：找到杠杆破局点 |
| 局部工作规划Agent | 运营部（中书省） | 目标拆解为行动计划 |
| 模拟投资人Agent | 财务部（协同） | 评估增长逻辑 |
| 活动运营进阶Agent | 运营部（执行） | 具体活动策划 |
| 内容运营专家 | 运营部（执行） | 内容发布执行 |
| 产品部 | 尚书省 | 产品侧优化 |
| 设计部 | 尚书省 | 物料支持 |

### 详细执行步骤

#### Step 1：太子分拣（CEO解析）
```
输入："帮我增长下个月营业指标"
↓
收入公式拆解：
  营业指标 = 流量 × 转化率 × 客单价
           = (曝光量 × 点击率) × (访客转化 × 下单转化) × (均单价 × 复购频次)
↓
识别核心杠杆点：
  - 流量侧：曝光/渠道/内容
  - 转化侧：产品/定价/促销
  - 客单侧：组合销售/会员体系
```

#### Step 2：中书省规划（策略执行Agent）
```
→ 技能调用：杠杆破局点识别
→ 技能调用：目标拆解为行动计划

破局点排序（按ROI/见效速度）：
  1. [高ROI] 转化率优化（落地页/话术/促销）
  2. [中ROI] 渠道曝光提升（内容投放）
  3. [长周期] 客单价提升（产品组合）

生成行动计划：
  - 本周：转化率诊断 + 落地页优化
  - 第2周：内容矩阵发布 + 渠道投放
  - 第3-4周：促销活动 + 数据复盘
```

#### Step 3：门下省审议（框架思维Agent）
```
→ 技能调用：自上而下框架构建法

方案框架评审：
  ✓ 目标对齐：月指标→周目标→日任务
  ✓ 资源匹配：人力/预算/渠道
  ✓ 风险预案：转化率不达标的备选方案
  ✓ 执行链路：清晰的责任人+时间节点

输出：评审通过/需调整反馈
```

#### Step 4：尚书省派发（六部接单）
```
派发任务：
  ┌─────────────────────────────────────────────┐
  │ 运营部：活动运营进阶Agent                    │
  │   - 转化率优化落地页设计                     │
  │   - 促销活动策划（限时折扣/满减/买赠）      │
  ├─────────────────────────────────────────────┤
  │ 产品部：                                      │
  │   - 产品组合优化（高毛利+高周转搭配）        │
  │   - 会员体系权益设计                         │
  ├─────────────────────────────────────────────┤
  │ 设计部：                                      │
  │   - 活动主视觉设计                           │
  │   - 落地页UI/UX优化                          │
  ├─────────────────────────────────────────────┤
  │ 内容运营专家：                                │
  │   - 朋友圈/社群推广素材                      │
  │   - 短视频脚本 + 发布                         │
  └─────────────────────────────────────────────┘
```

#### Step 5：六部执行（baoyu-skills调用）
```
内容运营专家执行：

Step 5.1 素材生成
  /baoyu-cover-image 促销活动.md --aspect 16:9
  /baoyu-article-illustrator 促销活动.md --type flowchart

Step 5.2 推广物料
  /baoyu-xhs-images 活动预告.md --style bold --count 4
  /baoyu-slide-deck 月度活动方案.md --slides 8

Step 5.3 渠道发布
  /baoyu-post-to-weibo --text "【限时活动】..." --images 活动图1.jpg
  /baoyu-post-to-wechat --title "本月活动" --content 活动文章.html

Step 5.4 数据追踪
  → 记录：曝光量/点击率/转化率/GMV
  → 每日汇报：核心指标达成率
```
