### 路径1: 增长营业指标

#### 用户场景
```
用户: "帮我增长下个月营业指标"
```

#### 触发关键词
```
增长 / 营业指标 / 提升收入 / 增加业绩 / 拉升GMV / 营收目标
```

#### CEO解析流程
```
┌────────────────────────────────────────────────────────────┐
│ 旨意解析                                                    │
├────────────────────────────────────────────────────────────┤
│ 输入: "帮我增长下个月营业指标"                                │
│                                                            │
│ 营收公式拆解:                                               │
│   营业指标 = 流量 × 转化率 × 客单价 × 复购率                  │
│                                                            │
│ 增长路径分析:                                               │
│   路径A: 流量提升 (曝光/渠道)                                │
│   路径B: 转化率提升 (内容/话术)                              │
│   路径C: 客单价提升 (套餐/促销)                              │
│   路径D: 复购率提升 (会员/激励)                              │
│                                                            │
│ → 选择杠杆最高的破局点                                        │
└────────────────────────────────────────────────────────────┘
```

#### 部门分配

| 部门 | Agent | 职责 |
|------|-------|------|
| **尚书省·户部** | 策略执行Agent | 主责 - 杠杆破局点识别 |
| **尚书省·礼部** | 用户激励Agent | 辅责 - 复购激励设计 |
| **尚书省·礼部** | 活动运营Agent | 辅责 - 增长活动策划 |
| **门下省** | 局部工作规划Agent | 辅责 - 目标拆解 |
| **中书省** | 框架思维Agent | 辅责 - 公式拆解 |
