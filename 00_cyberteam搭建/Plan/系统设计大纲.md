## 一、系统概述

### 1.1 产品定位

**CyberTeam** 是一款 AI 模拟公司协作系统，核心理念：

> **"一句话交代，整个团队干活"**

用户只需输入一个业务目标，AI 模拟公司即可自动完成：目标澄清 → 方案制定 → 部门执行 → 结果交付的全流程。

## 五、执行流程

### 5.1 思考天团辩论流程（5轮）

```
第1轮: 各专家独立陈述
    │
    ▼
第2轮: 交叉质疑
    │
    ▼
第3轮: 辩护与反驳
    │
    ▼
第4轮: 收敛检查（Jaccard相似度）
    │
    ├─ 收敛 → 进入第5轮
    └─ 不收敛 → 回到第2轮（最多3次）
    │
    ▼
第5轮: 综合结论 + 打分
    │
    ▼
输出: 方案 + 分数
    │
    ├─ ≥50分 → 提交总指挥
    └─ <50分 → 返回用户重新描述问题
```

### 5.2 收敛判断算法

```python
def check_convergence(round_outputs: List[str], threshold: float = 0.3) -> bool:
    """
    收敛条件: 变化比例 ≤ 30%
    即: changed_count / len(keywords) ≤ 0.3
    """
    keywords_sets = [set(extract_keywords(out)) for out in round_outputs]
    avg_similarity = sum(
        len(a & b) / len(a | b) if a | b else 0
        for i, a in enumerate(keywords_sets)
        for j, b in enumerate(keywords_sets) if i < j
    ) / len(keywords_sets)
    return avg_similarity >= threshold
```
