### 2.2 P0问题详解

#### C1: Q64/Q65任务状态模型不兼容

**Q64定义的状态**:
```python
class TaskStatus(Enum):
    CREATED    # 已创建
    QUEUED     # 排队中
    RUNNING    # 运行中
    WAITING_DEPS  # 等待依赖 ← Q65没有
    PAUSED     # 已暂停 ← Q65没有
    COMPLETED  # 已完成
    FAILED     # 失败
    CANCELLED  # 已取消
```

**Q65使用的状态**:
```python
# QueuedTask.status: "queued", "running", "completed", "failed", "cancelled"
# ResultStatus: PENDING, RUNNING, COMPLETED, FAILED, TIMEOUT, CANCELLED
# TIMEOUT是Q65特有，Q64没有
```

**影响**: 字符串vs枚举不统一，无法互操作

#### C2: TaskStore从未被使用

```python
# Q64定义了完整的TaskStore
class TaskStore:
    def save_task(self, task_id, task_data): ...
    def get_task(self, task_id): ...
    def update_status(self, task_id, status): ...

# 但Q65 ParallelExecutor从未导入或使用它
class ParallelExecutor:
    def __init__(self):
        self.task_queue = TaskQueue()      # 使用自己的队列
        self.scheduler = TaskScheduler(...)  # 使用自己的调度器
        # 没有 task_store !
```

#### C5: Q66与Q70"统一"是假的

```python
# Q70定义的统一评分器
class UnifiedQualityScorer:
    def score(self, output, stage="result_review") -> QualityScore:
        # 完整的六维评分实现

# 但Q66 L3 ResultReviewer完全绕过了它
class ResultReviewer:
    def review(self, outputs):
        # 直接内部实现，不调用 UnifiedQualityScorer
        dimension_scores["completeness"] = self._score_completeness(outputs)
        # ...自己实现了一套四维评分
```

## 五、改进建议详解

### 5.1 C2解决方案: TaskStore集成

```python
# 方案A: Q65直接使用Q64的TaskStore (推荐)
from task_store import TaskStore

class ParallelExecutor:
    def __init__(self, task_store: TaskStore = None):
        self.task_store = task_store or TaskStore()
        self.queue = TaskQueue()  # 仅用于调度

# 方案B: TaskStoreAdapter适配层
class TaskStoreAdapter:
    def __init__(self, queue: TaskQueue):
        self.queue = queue

    def save_task(self, task_id, task_data):
        qtask = QueuedTask(task_id=task_id, **task_data)
        self.queue._task_map[task_id] = qtask
```

### 5.2 C5解决方案: Q66调用Q70

```python
# Q66 ResultReviewer应改为:
class ResultReviewer:
    def __init__(self):
        self.unified_scorer = UnifiedQualityScorer()

    def review(self, outputs: Dict[str, Dict]) -> ReviewResult:
        # 调用统一评分器
        result = self.unified_scorer.score(outputs, stage="result_review")

        # 转换为Q66格式
        return ReviewResult(
            status=ReviewStatus.PASS if result.total >= 70 else ReviewStatus.NEEDS_REVISION,
            score=result.total,
            dimensions=result.breakdown,
            feedback=self._generate_feedback(result),
            issues=result.issues
        )
```
