# uncertainty

## 定位

不确定性与概率库。用于概率判断、风险评估和证据更新。

## 收录内容

- probabilistic
- bayesian
- pre-mortem
- counterfactual
- inversion
- reversal
- prospect-theory
- regression-mean
- ambiguity-aversion
- base-rate-fallacy

## 适用场景

- 风险评估
- 预测判断
- 证据更新
- 预演失败
- 处理信息不足时的决策

## 用法

当问题不是“能不能”，而是“多大概率、代价多高、证据够不够”时使用。
