# decision

## 定位

决策取舍库。用于判断该不该做、要不要继续、何时止损、哪个选项更值。

## 收录内容

- opportunity-cost
- sunk-cost
- decision-tree
- 10/10/10
- loss-aversion
- surrogate-decision
- local/global-optimum
- priority-ordering
- long-term-thinking
- time-machine-theory
- decision-loss
- counterfactual-thinking
- prospect-theory
- correct-and-non-consensus

## 适用场景

- 多个选项难取舍
- 继续还是停止
- 值不值得投入
- 先做什么最划算
- 未来收益和当前代价冲突

## 用法

当问题本质上是“值不值、停不停、选哪个”时，先查这里。若核心问题变成“多大概率、风险多高、证据够不够”，再转 `uncertainty`。
