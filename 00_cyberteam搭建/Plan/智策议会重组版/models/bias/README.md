# bias

## 定位

认知偏差与判断失真库。用于提醒主持人和专家：人的直觉不等于事实，熟悉不等于准确。

## 收录内容

- confirmation-bias
- availability-heuristic
- representativeness-heuristic
- base-rate-fallacy
- conjunction-fallacy
- gamblers-fallacy
- disposition-effect
- ambiguity-aversion
- zero-sum-thinking
- sunk-cost-fallacy
- dunning-kruger

## 适用场景

- 决策前自检
- 风险提示
- 结果反证
- 观点冲突时找偏差来源

## 用法

作为路由辅助和分析检查清单使用，不单独承载主流程。
