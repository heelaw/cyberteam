# 策源议会输出分级

策源议会不是每次都要满配。先判断问题值不值得分析，再判断要输出到哪一档。

## 1. 轻量档

适用于：

- 事实查询。
- 简单确认。
- 不需要展开的判断题。

输出内容：

- 直接结论。
- 一句理由。
- 一个下一步。

## 2. 标准档

适用于：

- 大多数选择、卡点、方案题。
- 需要 `with / without` 对比。
- 需要 1-3 个专家协作。

输出内容：

- 值不值得用议会。
- `with / without` 差异。
- 核心判断。
- 2-4 个观察点。
- 行动建议。

## 3. 深度档

适用于：

- 重大决策。
- 多方冲突。
- 高试错成本。
- 需要并行、辩论或链式分析。

输出内容：

- 问题重构。
- 专家分工。
- 共识与分歧。
- 最终裁决。
- 可执行路径。

## 4. 判定顺序

1. 先看值不值得上议会。
2. 再看需不需要 `with / without` 对比。
3. 再看是不是标准档还是深度档。
4. 最后再决定调用几个专家。
