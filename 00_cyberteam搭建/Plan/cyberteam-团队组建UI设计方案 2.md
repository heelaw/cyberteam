# CyberTeam 团队组建 UI 设计方案

> 基于 Accio Work Teams 灵感 + ClawTeam 底层能力
> 创建日期：2026-03-27

---

## 一、核心交互理念

### 1.1 用户心智模型

```
用户就像一家公司的CEO：
  1. 我要组建一个临时项目团队
  2. 我需要选择管理层（谁来管）
  3. 我需要选择员工（谁来干）
  4. 我要看到团队结构图
  5. 我要一键启动团队
```

### 1.2 与 Accio Work 的差异

| 维度 | Accio Work | CyberTeam |
|------|------------|-----------|
| **团队结构** | 扁平化 Agent 列表 | **层级化企业结构**（管理层+员工层） |
| **角色定位** | 平等协作 | **明确上下级关系**（CEO→总监→专家） |
| **组建方式** | 添加成员到团队 | **先选管理层，再选员工层** |
| **可视化** | 成员列表 | **组织架构图** |

## 三、交互流程设计

### 3.1 完整用户旅程

```
1. 用户输入目标
   ↓
2. AI 分析需求，推荐管理层配置
   ↓
3. 用户确认/调整管理层
   ↓
4. AI 根据管理层推荐员工层
   ↓
5. 用户确认/调整员工层
   ↓
6. 生成团队结构图
   ↓
7. 用户确认，一键启动
   ↓
8. 实时监控执行进度
```

### 3.2 智能推荐逻辑

```python
# 伪代码：智能推荐管理层
def recommend_management(user_goal):
    """根据用户目标推荐管理层配置"""
    goal_keywords = analyze(user_goal)

    # 核心管理层（必选）
    management = ["CEO", "COO"]

    # 专业总监（按需推荐）
    if has_keywords(goal_keywords, ["文案", "内容", "运营"]):
        management.append("运营总监")
    if has_keywords(goal_keywords, ["营销", "推广", "品牌"]):
        management.append("营销总监")
    if has_keywords(goal_keywords, ["设计", "UI", "UX"]):
        management.append("产品总监")

    return management

# 伪代码：智能推荐员工层
def recommend_employees(management_layer, user_goal):
    """根据管理层和目标推荐员工"""
    employees = {}

    for director in management_layer:
        if director == "运营总监":
            employees["运营"] = [
                "用户运营专家",
                "内容运营专家",
                "活动运营专家"
            ]
        elif director == "营销总监":
            employees["营销"] = [
                "小红书专家",
                "抖音专家",
                "微信专家"
            ]
        # ... 其他部门

    return employees
```

### 4.1 数据结构映射

| 前端概念 | ClawTeam 对应 | 数据结构 |
|----------|---------------|----------|
| **团队** | `Team` | `team_id`, `name`, `goal` |
