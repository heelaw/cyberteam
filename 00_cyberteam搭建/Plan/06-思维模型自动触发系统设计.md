## 一、系统概述

### 1.1 设计目标

```yaml
设计目标:
  核心: 根据问题类型自动选择最合适的思维模型组合
  原则: 最小化人工干预，最大化思维覆盖
  约束: 响应时间 < 1秒，准确性 > 90%

  关键指标:
    - 触发准确率: > 85%
    - 平均触发数量: 3-7个模型/任务
    - 漏检率: < 5%
    - 误触发率: < 10%
```

### 1.2 触发流程

```
用户输入
    ↓
┌─────────────────────────────┐
│     问题分类器 (L0)          │
│                             │
│  解析输入 → 识别关键词       │
│  判断复杂度 → 识别行业       │
│  输出: 问题类型 + 置信度      │
└──────────────┬──────────────┘
               ↓
┌─────────────────────────────┐
│     思维模型路由器 (L1)      │
│                             │
│  根据问题类型 → 查表匹配      │
│  根据关键词 → 补充触发        │
│  根据复杂度 → 调整数量       │
│  输出: 排序后的思维模型列表   │
└──────────────┬──────────────┘
               ↓
┌─────────────────────────────┐
│     CEO执行层 (L2)           │
│                             │
│  按顺序调用思维模型           │
│  聚合分析结果                │
│  输出: 结构化问题定义         │
└─────────────────────────────┘
```

### 2.2 问题分类算法

```python
# 问题分类器伪代码
class ProblemClassifier:
    def classify(self, user_input: str) -> dict:
        # 1. 关键词提取
        keywords = self.extract_keywords(user_input)

        # 2. 复杂度判断
        complexity = self.assess_complexity(
            input_length=len(user_input),
            has_context=bool(user_input.get('context')),
            has_constraints=bool(user_input.get('constraints'))
        )

        # 3. 问题类型匹配
        type_scores = {}
        for problem_type, config in PROBLEM_TYPES.items():
            score = self.match_score(keywords, config['keywords'])
            if score > THRESHOLD:
                type_scores[problem_type] = score

        # 4. 返回结果
        primary_type = max(type_scores, key=type_scores.get)
        confidence = type_scores[primary_type] / sum(type_scores.values())

        return {
            'primary_type': primary_type,
            'secondary_types': self.get_secondary_types(type_scores),
            'confidence': confidence,
            'complexity': complexity,
            'keywords': keywords,
            'trigger_count': self.calculate_trigger_count(complexity)
        }
```

## 四、权重计算算法

### 4.1 动态权重计算

```python
class ThinkingModelRouter:
    def calculate_weight(self, model: dict, context: dict) -> float:
        """
        计算思维模型的最终权重
        """
        # 1. 基础权重
        base_weight = model.get('weight', 0.5)

        # 2. 上下文加成
        context_bonus = 0.0

        # 行业加成
        industry = context.get('industry', '')
        if model.get('best_for_industries', []):
            if industry in model['best_for_industries']:
                context_bonus += 0.1

        # 规模加成
        scale = context.get('scale', 'medium')
        scale_map = {'small': 0, 'medium': 0.05, 'large': 0.1, 'enterprise': 0.15}
        context_bonus += scale_map.get(scale, 0)

        # 紧急度加成
        urgency = context.get('urgency', 'normal')
        if urgency == 'high':
            # 高紧急度：优先快速决策类模型
            if model['id'] in ['01-kahneman', '40-decision_tree']:
                context_bonus += 0.15

        # 3. 互斥惩罚
        exclusion_penalty = 0.0
        for excluded in model.get('excludes', []):
            if excluded in context.get('selected_models', []):
                exclusion_penalty += 0.2

        # 4. 最终权重
        final_weight = base_weight + context_bonus - exclusion_penalty
        return max(0.1, min(1.0, final_weight))

    def select_models(self, problem_type: str, context: dict, limit: int = 7) -> list:
        """
        选择最合适的思维模型组合
        """
        rules = ROUTING_RULES.get(problem_type, {})
        base_models = rules.get('base_models', [])

        # 计算每个模型的权重
        scored_models = []
        for model in base_models:
            weight = self.calculate_weight(model, context)
            scored_models.append((model['id'], weight, model.get('reason', '')))

        # 添加可选模型
        optional_models = rules.get('optional_models', [])
        for model in optional_models:
            # 检查条件是否满足
            condition = model.get('condition', '')
            if self.check_condition(condition, context):
                weight = self.calculate_weight(model, context)
                scored_models.append((model['id'], weight, model.get('reason', '')))

        # 排序并选择
        scored_models.sort(key=lambda x: x[1], reverse=True)
        selected = scored_models[:limit]

        return [{
            'id': m[0],
            'weight': m[1],
            'reason': m[2]
        } for m in selected]
```

### 5.2 权重自动调整

```python
class AdaptiveLearner:
    def update_weights(self, problem_type: str, triggered_models: list,
                       effectiveness: float):
        """
        根据反馈更新模型权重
        """
        learning_rate = 0.1

        for model_info in triggered_models:
            model_id = model_info['id']

            # 获取当前权重
            current_weight = self.get_current_weight(problem_type, model_id)

            # 计算新权重
            # 效果好 -> 权重增加
            # 效果差 -> 权重减少
            delta = learning_rate * (effectiveness - 0.5)

            new_weight = current_weight + delta
            new_weight = max(0.1, min(1.0, new_weight))

            self.update_weight(problem_type, model_id, new_weight)
```

## 六、异常处理

### 6.1 边界情况处理

```yaml
edge_cases:
  关键词缺失:
    处理: 触发默认基础模型组合
    默认: [01-kahneman, 02-first_principle, 21-5w1h1y]

  多类型混合:
    处理: 选择置信度最高的类型 + 补充其他类型的关键模型
    逻辑: 取Top 2类型，各取前3-4个模型

  超长输入:
    处理: 截取关键段落后分析
    策略: 取首段 + 含关键词的段落

  纯闲聊输入:
    处理: 识别为"general"类型
    触发: 仅基础逻辑思维模型

  专业术语密集:
    处理: 识别专业领域，调用领域特定模型
    领域映射:
      技术: [02-first_principle, 19-systems_thinking, 86-fmea]
      金融: [15-opportunity_cost, 82-monte_carlo, 83-sensitivity]
      医疗: [18-critical_thinking, 33-assumption, 86-fmea]
```
