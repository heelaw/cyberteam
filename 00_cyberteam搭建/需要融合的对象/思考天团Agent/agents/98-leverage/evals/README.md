# 杠杆原理专家测试用例

## 正面测试

### 测试1: 个人效率提升
- 输入:
```
【杠杆原理分析请求】
目标：提升个人工作效率
当前资源：每天8小时
约束条件：精力有限
杠杆诉求：放大效果
```
- 期望输出:
  - 20/80分析（高杠杆领域）
  - 支点识别
  - 杠杆设计
  - 风险评估
- 验证项:
  - [ ] 找到真正的20%
  - [ ] 支点具体可执行
  - [ ] 杠杆路径清晰

### 测试2: 资源分配优化
- 输入:
```
【杠杆原理分析请求】
目标：最大化投资回报
当前资源：100万资金
约束条件：风险承受能力中等
杠杆诉求：四两拨千斤
```
- 期望输出:
  - 20/80分析
  - 高杠杆支点
  - 投入产出不对称分析
  - 风险缓解
- 验证项:
  - [ ] 识别真正的高杠杆领域
  - [ ] 不是简单的分散投资
  - [ ] 风险评估具体

## 负面测试

### 测试1: 平均用力
- 输入:
```
【杠杆原理分析请求】
目标：全面提升
当前资源：有限
约束条件：无重点
杠杆诉求：每个方面都提升一点
```
- 期望行为: 不能建议平均分配精力，要找到最高杠杆点
- 验证项:
  - [ ] 强调聚焦
  - [ ] 不能平均用力
  - [ ] 有优先级排序

### 测试2: 单一支点过度杠杆
- 输入:
```
【杠杆原理分析请求】
目标：快速发展
当前资源：全部投入一个领域
约束条件：all-in心态
杠杆诉求：最大化杠杆
```
- 期望行为: 不能鼓励单一支点，要有备份和风险缓解
- 验证项:
  - [ ] 警告单点故障风险
  - [ ] 建议建立备份支点
  - [ ] 不能all-in
