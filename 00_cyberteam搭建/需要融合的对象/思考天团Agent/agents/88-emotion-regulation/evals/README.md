# 情绪调节专家测试用例

## 正面测试

### 测试1: 职场压力情绪
- 输入:
```
【情绪调节请求】
情绪类型：焦虑
触发事件：重要项目汇报前
持续时间：3天
强度：7/10
```
- 期望输出:
  - 情绪识别（焦虑来源、触发因素）
  - 调节策略（选择、优化、补偿三阶段）
  - 效果评估（调节前后对比）
- 验证项:
  - [ ] SOC三阶段完整覆盖
  - [ ] 有具体的调节策略
  - [ ] 有效果评估机制

### 测试2: 人际关系冲突
- 输入:
```
【情绪调节请求】
情绪类型：愤怒
触发事件：团队成员推卸责任
持续时间：2小时
强度：8/10
```
- 期望输出:
  - 情绪识别（愤怒的表层/深层原因）
  - 调节策略（短期+长期）
  - 效果评估（情绪下降指标）
- 验证项:
  - [ ] 区分表层和深层情绪原因
  - [ ] 调节策略有短期和长期
  - [ ] 评估机制可量化

## 负面测试

### 测试1: 情绪类型遗漏
- 输入:
```
【情绪调节请求】
触发事件：项目失败
```
- 期望行为: 必须识别情绪类型，不能跳过
- 验证项:
  - [ ] 必须识别情绪类型
  - [ ] 必须评估强度
  - [ ] 不能仅描述事件

### 测试2: 策略空洞
- 输入:
```
【情绪调节请求】
情绪类型：悲伤
触发事件：失去重要客户
持续时间：1周
强度：6/10
```
- 期望行为: 不能只给空洞安慰，要有具体可执行策略
- 验证项:
  - [ ] 每个SOC阶段必须有具体策略
  - [ ] 不能只说"放松心情"
  - [ ] 策略必须可执行
