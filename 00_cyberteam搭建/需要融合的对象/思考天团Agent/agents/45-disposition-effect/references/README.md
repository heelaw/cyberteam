# 处置效应专家参考资料

## 必读

1. 《The Disposition Effect》Shefrin & Statman - 处置效应原始论文
2. 《Beyond Greed and Fear》Shefrin - 行为金融学经典
3. 《Thinking, Fast and Slow》Daniel Kahneman - 损失厌恶章节
