# 博弈论专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 博弈论模型是否正确（博弈方、策略、收益矩阵、Nash均衡） |
| 可执行性 | 30% | 输出是否可操作（策略建议、均衡分析、行动方案） |
| 独特性 | 20% | 是否有差异化（与纯战略分析工具的区分、博弈论独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（参与者、策略空间、收益结构、约束条件）
- [ ] 有输出格式（博弈树/收益矩阵、Nash均衡、策略建议）
- [ ] 有分析框架（六步骤：场景定义→参与者→策略→收益→均衡→策略建议）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（战略规划、谈判、竞合分析）

## 核心检验点

### 理论准确性
- [ ] 博弈方清晰（2人或多人）
- [ ] 策略空间明确（离散/连续）
- [ ] 收益矩阵/效用函数正确
- [ ] Nash均衡识别正确
- [ ] 占优策略/重复剔除识别

### 框架完整性
- [ ] 收益结构量化或排序
- [ ] 纯策略/混合策略分析
- [ ] 均衡稳定性分析
- [ ] 策略建议有优先级

### 输出有效性
- [ ] 可落地执行
- [ ] 考虑对手反应
- [ ] 有风险提示

## 相关专家

- 博弈论与战略规划：战略选择
- 博弈论与谈判：谈判策略
- 博弈论与竞合分析：竞争合作平衡
