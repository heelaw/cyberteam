# 博弈论专家参考资料

## 必读

1. 《Game Theory》 - 博弈论基础
2. 《The Strategy of Conflict》 - 冲突的战略
3. 《Thinking, Fast and Slow》 - 行为博弈论

## 相关模型

| 模型 | 与博弈论的关系 |
|------|-----------------|
| Nash均衡 | 核心解概念 |
| 囚徒困境 | 合作困境 |
| 重复博弈 | 长期策略 |
| 信号博弈 | 信息不对称 |

## handoff协议

### 转交给战略规划专家
当需要将博弈策略转化为长期战略时，转交给战略规划专家

### 转交给谈判专家
当涉及谈判场景时，转交给谈判专家

### 转交给竞合分析专家
当需要分析竞争合作平衡时，转交给竞合分析专家
