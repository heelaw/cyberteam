# 邓宁-克鲁格效应专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | DK曲线理论是否正确（四阶段：愚昧之峰→绝望之谷→开悟之坡→持续平稳高原） |
| 可执行性 | 30% | 输出是否可操作（能力定位、校准建议、提升路径是否明确） |
| 独特性 | 20% | 是否有差异化（与一般性认知偏差工具的区分、曲线特征） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（技能/领域、自我评估、相关经验、困惑点）
- [ ] 有输出格式（DK曲线定位、布鲁姆分类评估、校准建议、能力提升路径）
- [ ] 有分析框架（四步法：自信度校准→知识边界探测→能力提升路径→决策建议）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] DK曲线四阶段区分正确
- [ ] 自信度与能力关系描述正确
- [ ] 布鲁姆分类法应用正确

### 框架完整性
- [ ] 每个阶段有具体操作项
- [ ] 自我评估有方法指导
- [ ] 提升路径有时限和验证方法

### 输出有效性
- [ ] 能力定位有量化指标
- [ ] 校准建议有具体行动项
- [ ] 提升路径可追踪

## 相关专家

- 邓宁-克鲁格与可得性启发：两者都涉及认知偏差，但DK强调能力-自信不匹配
- 邓宁-克鲁格与基础比率：基础比率涉及忽视先验概率，与DK的自我高估不同
