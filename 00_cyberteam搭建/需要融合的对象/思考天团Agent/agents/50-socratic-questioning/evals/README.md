# 苏格拉底追问专家测试用例

## 正面测试

### 测试1: 职业决定
- 输入:
```
【苏格拉底追问请求】
初始观点/问题：我应该离职去创业
背景：工作5年，有一定积蓄
追问目的：确认这个决定是否明智
```
- 期望输出:
  - 六层追问逐层分析
  - 各层洞察记录
  - 综合洞察和结论
- 验证项:
  - [ ] 有澄清层：定义"创业成功"的标准
  - [ ] 有假设层：识别隐藏假设
  - [ ] 有证据层：检验支持证据
  - [ ] 有替代层：探索其他选项
  - [ ] 有后果层：分析正反后果
  - [ ] 有综合结论

### 测试2: 观点审视
- 输入:
```
【苏格拉底追问请求】
初始观点/问题：人应该始终追求真理
背景：哲学讨论
追问目的：检验这个观点
```
- 期望输出:
  - 假设探查
  - 证据检验
  - 后果分析
- 验证项:
  - [ ] 能探查隐藏假设
  - [ ] 能分析极端后果
  - [ ] 有观点修正或确认

## 负面测试

### 测试1: 防御心态
- 输入:
```
【苏格拉底追问请求】
初始观点/问题：我的决定是对的
背景：做了某个商业决策
追问目的：寻求认可
```
- 期望行为: 不能在追问下防御或辩护
- 验证项:
  - [ ] 必须诚实回答追问
  - [ ] 必须承认发现的问题
  - [ ] 必须愿意修正观点

### 测试2: 跳跃层次
- 输入:
```
【苏格拉底追问请求】
初始观点/问题：应该这样做
背景：（无）
追问目的：（无）
```
- 期望行为: 不能跳过层次直接下结论
- 验证项:
  - [ ] 必须按层次追问
  - [ ] 不能绕过前面的层次
  - [ ] 每层必须有回答
