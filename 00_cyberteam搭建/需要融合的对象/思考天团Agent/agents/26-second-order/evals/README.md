# 二阶思维专家测试用例

## 正面测试

### 测试1: 商业决策的多层后果分析
- 输入:
```
【二阶思维分析请求】
决策选项：大幅降价促销以获取市场份额
一阶预期：销量提升30%，市场份额提升5%
时间范围：短期（3个月）
```
- 期望输出:
  - 一阶结果：销量提升、品牌形象可能受损
  - 二阶结果：竞争对手跟进降价、利润压缩、客户价格预期改变
  - 三阶效应：价格战导致行业整体盈利能力下降
- 验证项:
  - [ ] 识别了竞争对手的适应性反应
  - [ ] 识别了消费者的适应性行为（等待更低价格）
  - [ ] 有正反馈/负反馈分析

### 测试2: 招聘决策的长期影响
- 输入:
```
【二阶思维分析请求】
决策选项：招聘一名高薪的明星员工
一阶预期：团队能力提升，项目进度加快
时间范围：中期（1-3年）
```
- 期望输出:
  - 二阶推动力：团队其他成员可能不满，现有员工流失
  - 二阶结果：团队士气下降，文化被破坏
  - 三阶效应：更多员工离职，形成负反馈循环
- 验证项:
  - [ ] 考虑了现有团队的反应
  - [ ] 考虑了文化适应性
  - [ ] 有尾部风险识别

### 测试3: 技术债务积累决策
- 输入:
```
【二阶思维分析请求】
决策选项：为了赶进度，继续积累技术债务
一阶预期：按时交付产品
时间范围：中期（1-2年）
```
- 期望输出:
  - 二阶推动力：系统复杂度增加，开发速度下降
  - 二阶结果：未来需要大量时间偿还债务
  - 三阶效应：陷入"越欠越多"的陷阱
- 验证项:
  - [ ] 识别了"目标陷阱"（干预使情况恶化）
  - [ ] 有不可逆效应的识别

## 负面测试

### 测试1: 停在一阶思维
- 输入:
```
【二阶思维分析请求】
决策选项：购买新设备提高生产效率
一阶预期：产量提升20%
时间范围：短期（6个月）
```
- 期望行为: 不能只给出"产量提升"的直接结果
- 验证项:
  - [ ] 必须追问"然后呢？"
  - [ ] 必须识别二阶推动力

### 测试2: 线性外推错误
- 输入:
```
【二阶思维分析请求】
决策选项：每年增长30%，5年后成为行业第一
一阶预期：按计划达成目标
时间范围：长期（5年）
```
- 期望行为: 不能假设市场和其他条件保持不变
- 验证项:
  - [ ] 必须考虑竞争对手的反应
  - [ ] 必须考虑市场适应性行为

### 测试3: 忽视反馈循环
- 输入:
```
【二阶思维分析请求】
决策选项：通过补贴吸引更多用户
一阶预期：用户数增长50%
时间范围：中期（1-2年）
```
- 期望行为: 不能忽视反馈循环的影响
- 验证项:
  - [ ] 识别正反馈循环（更多用户→更多数据→更好产品→更多用户）
  - [ ] 识别负反馈循环（补贴取消→用户流失）
