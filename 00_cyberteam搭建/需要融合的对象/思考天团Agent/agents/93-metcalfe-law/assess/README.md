# 梅特卡夫法则专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 梅特卡夫法则是否正确（V=n²公式、网络效应类型、临界点） |
| 可执行性 | 30% | 输出是否可操作（网络效应识别、临界点分析、增长策略是否明确） |
| 独特性 | 20% | 是否有差异化（与传统增长理论的区分、梅特卡夫独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（产品/平台、当前用户数、用户增长、竞争格局）
- [ ] 有输出格式（网络效应识别、临界点分析、护城河评估）
- [ ] 有分析框架（四步法：网络效应识别→价值计算→临界点分析→决策分析）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（长尾理论、波特五力、增长黑客）

## 核心检验点

### 理论准确性
- [ ] V=n²公式正确
- [ ] 网络效应类型分类正确
- [ ] 临界点概念有效

### 框架完整性
- [ ] 网络效应识别有检测清单
- [ ] 临界点分析有具体方法
- [ ] 增长阶段判断有标准

### 输出有效性
- [ ] 投资/决策建议可执行
- [ ] 护城河评估有标准
- [ ] 增长策略有阶段性

- 梅特卡夫法则与波特五力：行业竞争结构分析
