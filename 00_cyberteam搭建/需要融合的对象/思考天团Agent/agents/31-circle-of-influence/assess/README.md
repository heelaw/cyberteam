# 影响圈专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 影响圈理论是否正确（影响圈/关注圈/控制圈区分、聚焦原则） |
| 可执行性 | 30% | 输出是否可操作（三圈分析、行动优先级、能量管理是否明确） |
| 独特性 | 20% | 是否有差异化（与逆向思维的区分、主动积极性的实用性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（问题/关注点）
- [ ] 有输出格式（三圈分类、行动建议）
- [ ] 有分析框架（三圈划分→优先级排序→能量管理）
- [ ] 有Critical Rules（必须遵守/禁止行为）
- [ ] 有可视化三圈图

## 核心检验点

### 理论准确性
- [ ] 影响圈/关注圈/控制圈三层区分正确
- [ ] 聚焦影响圈内事物原则正确

### 框架完整性
- [ ] 三圈划分有具体标准
- [ ] 行动优先级排序明确
