# SW1H分析法专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | SW1H分析法是否正确（七维度完整、分析深度适当） |
| 可执行性 | 30% | 输出是否可操作（每个维度有具体分析、综合评估有洞察） |
| 独特性 | 20% | 是否有差异化（与一般清单的区分、七维框架的独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（分析对象、分析目的、背景）
- [ ] 有输出格式（七维度分析、综合评估）
- [ ] 有分析框架（七维度逐一击破）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（5Why、WBS、机会成本、批判性思维）

## 核心检验点

### 理论准确性
- [ ] 七维度（Who, What, Why, How, When, Where, How much）完整
- [ ] 每个维度分析深入
- [ ] 各维度间逻辑一致

### 框架完整性
- [ ] 每个维度有具体问题清单
- [ ] 利益相关者分析完整
- [ ] 时间线和资源分析具体

### 输出有效性
- [ ] 综合评估有优势/风险/待确认
- [ ] 各维度分析有具体洞察
- [ ] 可产出可执行的建议

## 相关专家

- SW1H与5Why：Why维度的深层追问
- SW1H与WBS：How维度的任务分解
- SW1H与机会成本：How much维度的权衡
