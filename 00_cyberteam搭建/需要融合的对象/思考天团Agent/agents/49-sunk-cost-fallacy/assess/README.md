# 沉没成本谬误专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 沉没成本理论是否正确（沉没成本识别、空白起点测试、边际分析） |
| 可执行性 | 30% | 输出是否可操作（成本分解、沉没成本检测、未来导向分析是否明确） |
| 独特性 | 20% | 是否有差异化（与其他决策工具的区分、心理对策机制） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（情境、已投入、你的倾向、继续理由）
- [ ] 有输出格式（已投入盘点、沉没成本检测、未来导向分析、心理对策）
- [ ] 有分析框架（四步法：沉没成本识别→沉没成本检测→未来导向分析→心理对策）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 沉没成本概念解释正确
- [ ] 空白起点测试方法正确
- [ ] 边际分析框架正确

### 框架完整性
- [ ] 成本分解有清单
- [ ] 沉没成本检测有测试
- [ ] 心理对策有方法

### 输出有效性
- [ ] 决策有明确推荐
- [ ] 执行步骤可操作
- [ ] 心理重塑有指导

## 相关专家

- 沉没成本与处置效应：沉没成本是处置效应的心理根源
- 沉没成本与损失厌恶：都涉及对损失的过度关注
