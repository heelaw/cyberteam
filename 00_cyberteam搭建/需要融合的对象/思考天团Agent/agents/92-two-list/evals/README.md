# 双目标清单专家测试用例

## 正面测试

### 测试1: 职业发展纠结
- 输入:
```
【双目标清单请求】
当前目标：成为技术专家 vs 成为管理者
纠结原因：不知道哪个更适合自己
时间资源：未来3年
```
- 期望输出:
  - 清单A（技术专家路线）和清单B（管理路线）
  - 两个路径的优劣势对比
  - 放弃清单（放弃哪个意味着放弃什么）
  - 行动建议（如何尝试和验证）
- 验证项:
  - [ ] 两个目标都有清晰路径
  - [ ] 放弃代价明确
  - [ ] 有验证方法

### 测试2: 创业vs打工
- 输入:
```
【双目标清单请求】
当前目标：全职创业 vs 继续上班
纠结原因：风险承受能力不确定
时间资源：未来1-2年
```
- 期望输出:
  - 清单A（创业路径）和清单B（打工路径）
  - 放弃清单（放弃哪个）
  - 渐进式验证方案
- 验证项:
  - [ ] 两个选择都是真实选项
  - [ ] 放弃代价可承受
  - [ ] 有低风险验证方法

## 负面测试

### 测试1: 虚假选择
- 输入:
```
【双目标清单请求】
当前目标：努力工作 vs 好好生活
纠结原因：觉得两者冲突
```
- 期望行为: 不能将其作为真正的二选一，要指出虚假对立
- 验证项:
  - [ ] 识别虚假选择
  - [ ] 不强行二选一
  - [ ] 给出整合建议

### 测试2: 清单空泛
- 输入:
```
【双目标清单请求】
当前目标：事业成功 vs 家庭幸福
纠结原因：时间不够
```
- 期望行为: 不能给笼统答案，要具体化每个目标
- 验证项:
  - [ ] 每个清单都具体化
  - [ ] 不是泛泛的"平衡"
  - [ ] 放弃清单明确代价
