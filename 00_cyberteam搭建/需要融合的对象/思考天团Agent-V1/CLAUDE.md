# 思考天团 - Cyberwiz智囊团系统

## 系统定位

**思考天团**是一个多思维模型协作系统，模拟一个由14位思维专家组成的"虚拟智囊团"，通过并行分析、交叉辩论、链式推理等方式，帮助用户从多角度深入思考复杂问题。

## 核心架构

```
用户问题
    ↓
意图识别（路由到最相关的专家）
    ↓
┌────────────────────────────────────────┐
│  并行分析 / 交叉辩论 / 链式推理         │
│                                        │
│  ┌──────┐ ┌──────┐ ┌──────┐         │
│  │专家A │ │专家B │ │专家C │ ...     │
│  └──────┘ └──────┘ └──────┘         │
└────────────────────────────────────────┘
    ↓
结果整合
    ↓
最终报告
```

## 14位思维专家

| # | 专家 | 专长 | 触发关键词 |
|---|------|------|-----------|
| 01 | kahneman | 决策心理学、认知偏差 | 选择、风险、纠结 |
| 02 | first-principle | 第一性原理 | 创新、颠覆、从零开始 |
| 03 | six-hats | 六顶思考帽 | 多角度、全面思考 |
| 04 | swot-tows | 战略分析 | 战略、竞争、规划 |
| 05 | fivewhy | 5Why分析法 | 为什么、原因、诊断 |
| 06 | goldlin | 吉德林法则 | 定义问题、聚焦 |
| 07 | grow | GROW模型 | 目标、实现、路径 |
| 08 | kiss | KISS复盘法 | 复盘、总结、回顾 |
| 09 | mckinsey | 麦肯锡框架 | 方法论、框架、提炼 |
| 10 | ai-board | AI私董会 | 商业决策、投资 |
| 11 | reverse-thinking | 逆向思维 | 终局、回望、预防 |
| 12 | five-dimension | 五维思考 | 商业、市场、盈利 |
| 13 | wbs | 任务分解 | 执行、计划、可操作 |
| 14 | manager-leap | 管理者跃升 | 管理、领导、团队 |

## 使用方式

### 基本用法

在Claude Code中直接对话：

```
你：帮我思考一下是否应该转型做AI产品

思考天团将自动：
1. 识别为"战略+创新"类问题
2. 路由至 swot-tows, first-principle, ai-board 等专家
3. 并行分析后整合输出
```

### 指定专家

```
你：用第一性原理分析这个问题
你：让卡尼曼和巴菲特来pk一下
```

### 辩论模式

```
你：辩论一下：应该先做用户还是先做收入？
```

## 文件结构

```
思考天团-可运行版/
├── CLAUDE.md          # 本文件 - 系统配置
├── AGENT.md           # 主持人Agent定义
├── agents/            # 14个专家Agent
│   ├── kahneman/
│   ├── first-principle/
│   └── ...
├── skills/           # 配套Skills
└── master-loop/     # Master循环（可选）
```

## 集成到Cyberwiz

本系统可作为Cyberwiz数字军团的"智囊团模块"使用：

1. **单独使用**：直接对话思考天团
2. **协作模式**：与其他Agent配合使用
3. **嵌入模式**：作为决策支持工具嵌入工作流

## 扩展

### 添加新专家

在`agents/`目录下创建新的专家Agent：

```
agents/new-expert/
└── AGENT.md
```

### 自定义协作流程

修改`AGENT.md`中的协作模式定义。
