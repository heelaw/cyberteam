## 技术实现

### 1. Master Agent (goal_driven_master.py)

核心循环:
```python
while True:
    if check_criteria():  # 检查目标是否达成
        mark_task_complete()
        break

    if failure_count >= max_retries:
        break

    status = check_executor_status()

    if status == "inactive" or status == "unknown":
        apply_pua_pressure()  # 应用PUA压力
        restart_executor()    # 重启执行器
        failure_count += 1

    await asyncio.sleep(check_interval)
```

### 2. PUA压力升级机制

| 失败次数 | 压力等级 | 动作 |
|---------|---------|------|
| 1次 | L1 | 换一种根本不同的方法 |
| 2次 | L2 | WebSearch + 读源码 |
| 3次 | L3 | 执行7步检查清单 |
| 4次+ | L4 | 绝望模式 + 重启 |

### 3. ClawTeam集成

- 使用 `cyberteam spawn` 启动Executor
- Executor在tmux中运行
- Master通过 `cyberteam board show` 检查状态
- 通过bash alias调用: `source ~/.cyberteam-venv/bin/activate && cyberteam ...`
