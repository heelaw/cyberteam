## 五、关键机制

### 5.1 Goal-Driven循环

```python
while (not goal_achieved):
    if check_criteria():
        goal_achieved = True
        break

    status = check_executor()
    if status == INACTIVE:
        apply_pressure()
        restart_executor()

    wait(check_interval)
```

### 5.2 AutoResearch循环

```python
while (attempts < max_attempts):
    result = try_execute()

    if not result.success:
        pattern = analyze_failure()
        hypotheses = generate_hypotheses()

        if consecutive_failures >= threshold:
            upgrade_pressure()

        try_alternative_method()

    attempts += 1
```

### 5.3 PUA压力等级

| 等级 | 连续失败次数 | 动作 |
|------|-------------|------|
| L0 | 0 | 正常 |
| L1 | ≥2 | 换方法 |
| L2 | ≥4 | 搜索+源码 |
| L3 | ≥6 | 检查清单 |
| L4 | ≥8 | 绝望+重启 |
