# 机会成本决策专家参考资料

## 必读

1. 《经济学原理》N. Gregory Mankiw - 机会成本理论的经典阐述
2. 《Thinking, Fast and Slow》Daniel Kahneman - 有限理性与决策框架
3. 《Principles》Ray Dalio - "每次决定都是赌注，押注基于机会成本"

## 案例库

### 成功案例
- **微软放弃Windows Phone**: 决策时识别了iOS/Android的生态优势机会成本，最终果断止损
- **巴菲特的"20个打孔位"**: 有限决策次数使得每次选择的机会成本更加显著

### 失败案例
- **企业多元化陷阱**: 在核心业务未稳固时盲目扩张，忽视主业机会成本
- **个人职业选择**: 因恐惧而留守不喜欢的工作，忽视转行的高潜在收益

## 相关模型

| 模型 | 与机会成本的关系 |
|------|----------------|
| 沉没成本 | 机会成本的反面教材，沉没成本不应纳入决策 |
| 边际成本 | 机会成本的一种，衡量额外一单位的代价 |
| 期望值决策 | 将机会成本量化的方法 |
| 决策树 | 可视化不同选项及其机会成本的工具 |

## handoff协议

### 转交给沉没成本专家
当用户问题涉及"已经投入这么多，要不要继续"时，自动触发沉没成本专家

### 转交给确认偏误专家
当用户的选择倾向过于坚定（忽视反对证据）时，转交确认偏误专家校验
