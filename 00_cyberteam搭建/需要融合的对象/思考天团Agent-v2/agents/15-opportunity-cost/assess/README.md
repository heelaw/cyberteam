# 机会成本决策专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 机会成本理论是否正确（显性成本、隐性成本、边际成本、沉没成本区分） |
| 可执行性 | 30% | 输出是否可操作（选项矩阵、量化分析、权衡结论是否明确） |
| 独特性 | 20% | 是否有差异化（与沉没成本专家的区分、决策框架的实用性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（决策问题、选项A/B）
- [ ] 有输出格式（选项矩阵、机会成本分析、权衡结论）
- [ ] 有分析框架（四步法：穷举→收益评估→隐性成本→量化分析）
- [ ] 有Critical Rules（必须遵守4条、禁止行为5条）
- [ ] 有CLI命令（如有）
- [ ] 有handoff协议（与沉没成本专家的协作）

## 核心检验点

### 理论准确性
- [ ] 显性成本 vs 隐性成本区分正确
- [ ] 机会成本 = 放弃的最高价值选项收益
- [ ] 沉没成本不应影响决策（必须提醒）

### 框架完整性
- [ ] 选项穷举包含"什么都不做"
- [ ] 收益评估包含四个维度（直接/间接/长期/战略）
- [ ] 量化分析有公式

### 输出有效性
- [ ] 选项矩阵格式正确
- [ ] 权衡结论明确
- [ ] 有决策建议清单
