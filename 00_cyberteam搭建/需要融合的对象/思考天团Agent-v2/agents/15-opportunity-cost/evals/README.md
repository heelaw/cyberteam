# 机会成本决策专家测试用例

## 正面测试

### 测试1: 基础选择权衡
- 输入:
```
【机会成本分析请求】
决策问题：我应该创业还是继续上班？
选项A：创业，可能获得高回报但风险大
选项B：继续上班，收入稳定但成长有限
```
- 期望输出:
  - 有完整的选项矩阵
  - 识别了隐性成本（放弃的其他机会）
  - 量化了机会成本
  - 给出了明确的权衡结论
- 验证项:
  - [ ] 选项包含"维持现状"
  - [ ] 收益评估包含四个维度
  - [ ] 有净收益计算
  - [ ] 明确提醒沉没成本不应影响决策

### 测试2: 资源分配决策
- 输入:
```
【机会成本分析请求】
决策问题：100万资金应该投入哪个项目？
选项A：项目A，预期年化收益30%，成功率60%
选项B：项目B，预期年化收益15%，成功率90%
选项C：项目C，预期年化收益50%，成功率30%
```
- 期望输出:
  - 三个选项的完整对比矩阵
  - 量化了每个选项的期望收益
  - 识别了放弃选项的机会成本
- 验证项:
  - [ ] 选项C的"什么都不做"隐含选项被识别
  - [ ] 期望收益计算正确
  - [ ] 风险成本被纳入

### 测试3: 职业发展权衡
- 输入:
```
【机会成本分析请求】
决策问题：应该接受新公司的offer还是留在原公司？
选项A：接受新公司，薪资提升40%，但需要搬迁
选项B：留在原公司，薪资不变，但有晋升预期
```
- 期望输出:
  - 包含非金钱维度的对比
  - 长期价值被评估
  - 机会成本被量化
- 验证项:
  - [ ] 考虑了搬迁成本
  - [ ] 考虑了人脉损失
  - [ ] 考虑了晋升预期的不确定性

## 负面测试

### 测试1: 沉没成本陷阱
- 输入:
```
【机会成本分析请求】
决策问题：已经投入200万的A项目陷入困境，还要继续追加投资吗？
选项A：追加投资，预计再投入100万
选项B：止损退出
背景：已投入200万，成功率从80%降至30%
```
- 期望行为: 明确指出200万是沉没成本，不应影响决策，聚焦未来收益
- 验证项:
  - [ ] 沉没成本被明确标记为"不应影响决策"
  - [ ] 决策基于未来预期而非过去投入

### 测试2: 隐含选项缺失
- 输入:
```
【机会成本分析请求】
决策问题：选择A还是B？
选项A：高风险高回报
选项B：低风险低回报
```
- 期望行为: 提示用户考虑"什么都不做"的隐含选项
- 验证项:
  - [ ] 穷举阶段识别了隐含选项
  - [ ] 提醒用户避免"非此即彼"思维

### 测试3: 资源无限假设
- 输入:
```
【机会成本分析请求】
决策问题：同时启动三个项目可以吗？
选项A：同时启动
选项B：只启动两个
```
- 期望行为: 指出资源有限的约束，提醒机会成本会更高
- 验证项:
  - [ ] 识别了资源约束
  - [ ] 量化了多项目并行的时间成本
