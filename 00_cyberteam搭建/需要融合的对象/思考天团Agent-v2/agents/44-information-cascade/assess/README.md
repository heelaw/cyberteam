# 信息级联专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 信息级联理论是否正确（级联三要素、信息链、独立判断） |
| 可执行性 | 30% | 输出是否可操作（级联诊断、私有信息清单、决策树是否明确） |
| 独特性 | 20% | 是否有差异化（与群体思维的区分、独立决策机制） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（现象、群体行为、你的判断、专家意见）
- [ ] 有输出格式（级联诊断、私有信息、决策建议、行动计划）
- [ ] 有分析框架（四步法：级联识别→信息链分析→独立判断练习→级联风险评估）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 级联三要素描述正确
- [ ] 信息链分析正确
- [ ] 独立判断检验方法正确

### 框架完整性
- [ ] 级联诊断清单完整
- [ ] 私有信息评估有方法
- [ ] 决策树有指导

### 输出有效性
- [ ] 级联风险评估有量化
- [ ] 独立思考清单可操作
- [ ] 决策建议有依据

## 相关专家

- 信息级联与可得性启发：信息级联涉及忽视私有信息，AH涉及记忆偏差
- 信息级联与批判性思维：都涉及独立思考
