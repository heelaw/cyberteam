# 赌徒谬误专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 赌徒谬误理论是否正确（独立性判断、概率不变、均值回归区分） |
| 可执行性 | 30% | 输出是否可操作（独立性检验、序列分析、概率重新校准是否明确） |
| 独特性 | 20% | 是否有差异化（与其他概率谬误的区分、独立性概念） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（观察序列、你认为的趋势、序列长度、事件类型）
- [ ] 有输出格式（独立性诊断、如果是独立事件/非独立变量、行动建议）
- [ ] 有分析框架（四步法：独立性判断→序列模式分析→均值回归诊断→概率重新校准）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 独立性概念解释正确
- [ ] 独立事件概率不变正确
- [ ] 赌徒谬误vs均值回归区分正确

### 框架完整性
- [ ] 独立性检验清单完整
- [ ] 序列模式分析有方法
- [ ] "热门号码"误区解释正确

### 输出有效性
- [ ] 独立vs非独立事件区分明确
- [ ] 行动建议可执行
- [ ] 心理建设有指导

## 相关专家

- 赌徒谬误与回归均值：两者都涉及"概率纠正"，但GF是错误信念，RM是真实统计现象
- 赌徒谬误与可得性启发：都涉及对随机性的误解
