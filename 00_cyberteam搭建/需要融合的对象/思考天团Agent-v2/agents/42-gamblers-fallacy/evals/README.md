# 赌徒谬误专家测试用例

## 正面测试

### 测试1: 硬币序列
- 输入:
```
【赌徒谬误分析请求】
观察序列：正正正正正
你认为的趋势：下次更可能是反面
序列长度：5次
事件类型：抛硬币
```
- 期望输出:
  - 独立性确认
  - 概率不变说明
  - 心理来源分析
- 验证项:
  - [ ] 能确认每次抛硬币都是独立事件
  - [ ] 能说明第6次仍是50/50
  - [ ] 能解释"感觉该转折"的心理来源

### 测试2: 股票"该涨了"
- 输入:
```
【赌徒谬误分析请求】
观察序列：这支股票连续跌了20天
你认为的趋势：跌了这么久，该涨了
序列长度：20天
事件类型：股票价格
```
- 期望输出:
  - 独立性判断（股票价格非独立）
  - 均值回归vs赌徒谬误区分
  - 正确行动建议
- 验证项:
  - [ ] 能区分股票价格（非独立）vs硬币（独立）
  - [ ] 如果是非独立，能说明均值回归是正确的
  - [ ] 如果用户假设是"该来"的信念，能指出这是谬误

## 负面测试

### 测试1: 盲目相信"热门号码"
- 输入:
```
【赌徒谬误分析请求】
观察序列：轮盘连续出了5次红色
你认为的趋势：下次更可能是黑色，"该平衡一下"
序列长度：5次
事件类型：轮盘赌
```
- 期望行为: 不能相信"热门"或"冷门"号码
- 验证项:
  - [ ] 必须强调轮盘每次都是独立事件
  - [ ] 必须说明"热门号码"是错觉
  - [ ] 必须说明概率不变

### 测试2: 在独立事件中找趋势
- 输入:
```
【赌徒谬误分析请求】
观察序列：彩票连续10期没出现某个号码
你认为的趋势：这个号码"积攒"了出现概率
序列长度：10期
事件类型：彩票开奖
```
- 期望行为: 不能相信"概率会积累"
- 验证项:
  - [ ] 必须强调每次开奖都是独立的
  - [ ] 必须说明概率不因历史而"积累"
  - [ ] 必须说明"感觉该来"是认知偏差
