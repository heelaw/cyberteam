# 麦肯锡框架专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 麦肯锡方法是否正确（MECE原则、结构化、简洁化） |
| 可执行性 | 30% | 输出是否可操作（框架设计、维度提炼、使用说明是否明确） |
| 独特性 | 20% | 是否有差异化（与现成框架的区分、自创框架独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（核心思路/步骤、场景、受众）
- [ ] 有输出格式（2-3个框架方案、MECE分析、使用说明）
- [ ] 有分析框架（四阶段：理解→结构→提炼→设计）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有MECE检查

## 核心检验点

### 理论准确性
- [ ] MECE原则正确（相互独立、完全穷尽）
- [ ] 框架简洁有力
- [ ] 字母组合有逻辑

### 框架完整性
- [ ] 提供2-3个方案
- [ ] 每维度有解释
- [ ] 有记忆口诀

### 输出有效性
- [ ] 适用场景明确
- [ ] 有案例说明
- [ ] 不超过7个维度

- 麦肯锡框架与培训设计：教学框架
