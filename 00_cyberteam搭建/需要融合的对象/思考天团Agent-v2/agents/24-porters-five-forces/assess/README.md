# 长尾理论评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 头部分析+长尾评估+聚合效应+成本结构四部分齐全 |
| 准确性 | 30% | 理论应用是否正确，头尾划分是否合理 |
| 可执行性 | 30% | 策略建议是否具体可落地，头尾比例是否合理 |
| 独特性 | 20% | 是否基于具体业务特征分析，非通用模板 |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词
- [ ] 有输入格式
- [ ] 有输出格式（含头部分析表+长尾评估表+聚合分析表+成本结构表）
- [ ] 有五步分析框架
- [ ] 有Critical Rules（5条禁止+5条必须）
- [ ] 有策略建议（纯长尾/头部+长尾/动态平衡/平台策略）
- [ ] 有handoff协议

## 评估检查点

### 头部市场分析
- [ ] 识别了头部产品和品牌
- [ ] 分析了头部特征和壁垒
- [ ] 评估了头部饱和度

### 长尾市场评估
- [ ] 评估了长尾长度（细分需求数量）
- [ ] 分析了需求分散度（帕累托 vs 均匀）
- [ ] 识别了需求真实性（非伪需求）
- [ ] 评估了需求频率

### 聚合效应分析
- [ ] 定义了最小可行市场
- [ ] 计算了聚合临界点
- [ ] 评估了聚合成本
- [ ] 有案例参照

### 成本结构分析
- [ ] 分析了存储成本
- [ ] 分析了分销成本
- [ ] 分析了个性化成本
- [ ] 分析了推荐成本

### 策略建议
- [ ] 推荐策略与成本结构匹配
- [ ] 头尾比例建议合理
- [ ] 核心建议具体可执行
- [ ] 风险提示有应对方案
