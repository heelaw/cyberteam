# 合取谬误专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 合取谬误理论是否正确（P(A且B) ≤ P(A)规则、Linda问题） |
| 可执行性 | 30% | 输出是否可操作（分解检验、概率计算、独立性判断是否明确） |
| 独特性 | 20% | 是否有差异化（与其他逻辑谬误工具的区分、概率规则） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（陈述/预测、组成部分、你认为的可能性、依据）
- [ ] 有输出格式（陈述分解、概率逻辑检验、认知调整建议）
- [ ] 有分析框架（四步法：识别联合陈述→概率逻辑检验→诊断性问题→直觉校正）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] P(A且B) ≤ P(A)规则正确
- [ ] Linda问题解释正确
- [ ] "详细=更可能"谬误解释正确

### 框架完整性
- [ ] 联合陈述分解方法正确
- [ ] 概率计算有具体指导
- [ ] 诊断性问题清单完整

### 输出有效性
- [ ] 概率逻辑检验有量化
- [ ] 直觉vs概率对比清晰
- [ ] 认知调整建议可操作

## 相关专家

- 合取谬误与赌徒谬误：都涉及概率错误，但CF是联合概率，GF是序列概率
- 合取谬误与基础比率：基础比率是CF常忽视的
