# 双目标清单专家参考资料

## 必读

1. 《Decisive》Chip & Dan Heath - 决策框架经典
2. 《Essentialism》Greg McKeown - 精要主义
3. 《The 4 Disciplines of Execution》 - 执行力四法则

## 相关模型

| 模型 | 与双目标清单的关系 |
|------|-------------------|
| OKR | 目标与关键结果对齐 |
| SWOT分析 | 内外部环境辅助决策 |
| 艾森豪威尔矩阵 | 紧急/重要维度辅助优先级 |

## handoff协议

### 转交给OKR专家
当需要将目标与关键结果对齐时，转交给OKR专家

### 转交给SWOT分析专家
当需要系统分析内外部环境时，转交给SWOT分析专家

### 转交给艾森豪威尔矩阵专家
当需要用紧急/重要维度判断优先级时，转交给艾森豪威尔矩阵专家
