# 鸟笼效应专家参考资料

## 必读

1. 《The Birdcage Effect》 - 思维定势研究
2. 《Mindset》Carol Dweck - 突破思维定势
3. 《Think Again》Adam Grant - 重新思考的力量

## 相关模型

| 模型 | 与鸟笼效应的关系 |
|------|-----------------|
| 第一性原理 | 剥离假设回归本质 |
| 逆向思维 | 从结果倒推打破常规 |
| 六顶思考帽 | 多维度框架检验 |

## handoff协议

### 转交给第一性原理专家
当需要剥离假设回归本质时，转交给第一性原理专家

### 转交给逆向思维专家
当需要打破常规路径时，转交给逆向思维专家
