# 沉没成本决策专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 沉没成本理论是否正确（成本分类、损失评估、止损信号） |
| 可执行性 | 30% | 输出是否可操作（决策树、止损建议、行动清单是否明确） |
| 独特性 | 20% | 是否有差异化（与机会成本专家的区分、止损框架的实用性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（困境描述、已投入、坚持理由）
- [ ] 有输出格式（成本分类、损失预估、决策树、止损建议）
- [ ] 有分析框架（四步法：识别→损失评估→决策树→止损时机）
- [ ] 有Critical Rules（必须遵守4条、禁止行为5条）
- [ ] 有CLI命令（如有）
- [ ] 有handoff协议（与机会成本专家的协作）

## 核心检验点

### 理论准确性
- [ ] 沉没成本 vs 可变成本区分正确
- [ ] 决策基于未来收益而非过去投入
- [ ] 损失厌恶和损失加速被正确理解

### 框架完整性
- [ ] 成本分类表完整（沉没/可变/机会/边际）
- [ ] 决策树清晰（继续 vs 止损两条路）
- [ ] 止损信号具体可执行

### 输出有效性
- [ ] 损失量化明确（有具体数字或范围）
- [ ] 三种建议分类清晰（立即止损/观察等待/继续坚持）
- [ ] 心理建设部分有帮助性
