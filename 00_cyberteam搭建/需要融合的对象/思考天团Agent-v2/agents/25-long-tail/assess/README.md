# 长尾理论专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 长尾理论是否正确（头部/长尾区分、聚合效应、成本结构） |
| 可执行性 | 30% | 输出是否可操作（策略建议具体、头尾比例明确） |
| 独特性 | 20% | 是否有差异化（与头部市场策略的区分、与复利效应的区分） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（细分市场、利基产品、个性化、长尾效应等）
- [ ] 有输入格式（分析对象、市场结构）
- [ ] 有输出格式（头部分析表、长尾评估表、聚合效应分析、策略建议）
- [ ] 有分析框架（五步法：头部识别→长尾评估→聚合效应→成本结构→策略建议）
- [ ] 有Critical Rules（必须遵守4条、禁止行为6条）
- [ ] 有头尾分离原则
- [ ] 有聚合效应分析

## 核心检验点

### 理论准确性
- [ ] 头部 vs 长尾区分正确
- [ ] 聚合效应的前提条件明确
- [ ] 边际成本是长尾经济性的关键

### 框架完整性
- [ ] 五步分析框架完整
- [ ] 成本结构分析包含存储/分销/个性化/推荐成本
- [ ] 策略建议区分纯长尾/头尾结合/平台策略

### 输出有效性
- [ ] 头部长尾评估表格格式正确
- [ ] 聚合临界点有量化
- [ ] 策略建议具体可执行
