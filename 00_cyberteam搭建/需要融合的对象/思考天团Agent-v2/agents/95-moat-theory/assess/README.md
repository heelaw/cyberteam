# 护城河理论专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 护城河理论是否正确（四大来源、宽度评估、侵蚀分析） |
| 可执行性 | 30% | 输出是否可操作（护城河识别、宽度评估、构建策略是否明确） |
| 独特性 | 20% | 是否有差异化（与传统竞争优势的区分、护城河独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（分析对象、竞争对手、行业特点、时间范围）
- [ ] 有输出格式（护城河识别、宽度评估、侵蚀分析、战略建议）
- [ ] 有分析框架（四步法：来源识别→宽度评估→侵蚀分析→构建策略）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（波特五力、SWOT分析、巴菲特思维）

## 核心检验点

### 理论准确性
- [ ] 四大护城河来源正确（无形资产、转换成本、网络效应、成本优势）
- [ ] 宽度评估标准合理
- [ ] 侵蚀分析有效

### 框架完整性
- [ ] 护城河识别有清单
- [ ] 宽度评估有矩阵
- [ ] 构建策略有路径

### 输出有效性
- [ ] 竞争优势可对比
- [ ] 侵蚀信号可监测
- [ ] 战略建议可执行

## 相关专家

- 护城河理论与波特五力：行业结构分析
- 护城河理论与SWOT分析：战略匹配分析
- 护城河理论与巴菲特思维：价值投资视角
