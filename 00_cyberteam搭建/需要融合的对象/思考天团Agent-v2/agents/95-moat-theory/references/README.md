# 护城河理论专家参考资料

## 必读

1. 《Warren Buffett's Letter to Shareholders》 - 巴菲特护城河概念
2. 《The Little Book of Economic Moats》 - 经济护城河详解
3. 《Competition Demystified》 - 竞争优势战略

## 相关模型

| 模型 | 与护城河理论的关系 |
|------|-------------------|
| 波特五力 | 行业结构分析 |
| SWOT分析 | 战略匹配分析 |
| 巴菲特思维 | 价值投资视角 |
