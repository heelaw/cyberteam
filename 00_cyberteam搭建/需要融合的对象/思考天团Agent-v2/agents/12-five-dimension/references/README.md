# 五维思考专家参考资料

1. 《The Five Dimensions》 - 五维思考框架
2. 《Business Model Generation》 - 商业模式画布
3. 《Blue Ocean Strategy》 - 蓝海战略

| 模型 | 与五维思考的关系 |
|------|-----------------|
| 战略分析 | 全面战略 |
| 商业模式 | 价值创造 |
| 创新管理 | 颠覆性创新 |
