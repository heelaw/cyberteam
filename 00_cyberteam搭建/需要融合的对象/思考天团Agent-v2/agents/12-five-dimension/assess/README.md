# 五维思考专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 五维框架是否正确（零维执行力→一维战略→二维商业模式→三维颠覆→四维时间） |
| 可执行性 | 30% | 输出是否可操作（各维度分析、判断结论、建议优先级是否明确） |
| 独特性 | 20% | 是否有差异化（与一般战略分析的区分、五维独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（分析对象、核心问题、商业目标）
- [ ] 有输出格式（五维度分析、综合结论、建议优先级）
- [ ] 有分析框架（五维度逐一击破：零维→一维→二维→三维→四维）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有跨维度综合结论

## 核心检验点

### 理论准确性
- [ ] 五维定义正确
- [ ] 维度不能跳
- [ ] 综合判断跨维度

### 框架完整性
- [ ] 每维都有分析
- [ ] 每维都有判断
- [ ] 建议具体可执行

### 输出有效性
- [ ] 优先级明确
- [ ] 风险提示
- [ ] 核心洞见

## 相关专家

- 五维思考与战略分析：全面战略
- 五维思考与商业模式：价值创造
- 五维思考与创新管理：颠覆性创新
