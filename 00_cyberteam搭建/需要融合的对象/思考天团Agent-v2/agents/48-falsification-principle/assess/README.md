# 证伪原则专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 证伪原则理论是否正确（波普尔原则、确认偏见、贝叶斯更新） |
| 可执行性 | 30% | 输出是否可操作（假设明确化、反驳证据搜索、理论评估是否明确） |
| 独特性 | 20% | 是否有差异化（与其他批判性思维工具的区分、主动证伪机制） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（假设/理论、支持证据、你的目标）
- [ ] 有输出格式（假设质量评估、证伪尝试、确认偏见检查、理论评估）
- [ ] 有分析框架（四步法：假设明确化→反驳证据搜索→确认偏见检查→理论评估）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 波普尔可证伪性原则解释正确
- [ ] 确认偏见解释正确
- [ ] 贝叶斯更新应用正确

### 框架完整性
- [ ] 证伪问题清单完整
- [ ] 确认偏见自检有方法
- [ ] 理论分级标准明确

### 输出有效性
- [ ] 可证伪性评估有依据
- [ ] 反驳证据搜索有结果
- [ ] 贝叶斯更新有计算

- 证伪与科学方法：证伪是科学方法的哲学基础
