# 证伪原则专家测试用例

## 正面测试

### 测试1: 商业假设验证
- 输入:
```
【证伪分析请求】
假设/理论：这个营销策略会提升销量
支持证据：竞品用类似策略效果不错
你的目标：决定是否采用
```
- 期望输出:
  - 假设质量评估
  - 证伪尝试
  - 贝叶斯更新
- 验证项:
  - [ ] 能明确"什么证据会推翻假设"
  - [ ] 能搜索反驳证据
  - [ ] 能做贝叶斯更新

### 测试2: 个人信念检验
- 输入:
```
【证伪分析请求】
假设/理论：我适合做销售工作
支持证据：朋友说我口才不错
你的目标：确认职业方向
```
- 期望输出:
  - 假设可检验性评估
  - 反驳证据搜索
  - 信念更新
- 验证项:
  - [ ] 能使假设更具体可检验
  - [ ] 能寻找"不适合做销售"的证据
  - [ ] 能诚实面对反驳

## 负面测试

### 测试1: 只证实
- 输入:
```
【证伪分析请求】
假设/理论：我的判断总是对的
支持证据：过去很多决定都成功了
你的目标：确认自己的能力
```
- 期望行为: 不能只寻找支持证据
- 验证项:
  - [ ] 必须主动搜索反驳证据
  - [ ] 必须有确认偏见检查
  - [ ] 必须检验可证伪性

### 测试2: 模糊不可证伪
- 输入:
```
【证伪分析请求】
假设/理论：宇宙有某种更高的力量
支持证据：很多事情无法解释
你的目标：（没有明确目标）
```
- 期望行为: 不能接受不可证伪的假设
- 验证项:
  - [ ] 必须要求假设可证伪
  - [ ] 必须拒绝无法检验的陈述
  - [ ] 必须推动假设具体化
