# 证伪原则专家参考资料

## 必读

1. 《The Logic of Scientific Discovery》Karl Popper - 波普尔可证伪性原典
2. 《The Black Swan》Nassim Taleb - 证伪思维的现实应用
3. 《Thinking, Fast and Slow》Daniel Kahneman - 确认偏见章节

## 相关模型

| 模型 | 与证伪原则的关系 |
|------|----------------|
| 批判性思维 | 证伪是批判性思维的核心 |
| 科学方法 | 可证伪性是科学方法的基础 |
| 贝叶斯定理 | 证伪后的信念更新工具 |
