# 批判性思维专家评估标准

## 评估维度
| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全 |
| 准确性 | 30% | 理论是否正确 |
| 可执行性 | 30% | 输出是否可操作 |
| 独特性 | 20% | 是否有差异化 |

## 及格线
- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单
- [ ] 有触发词
- [ ] 有输入格式
- [ ] 有输出格式
- [ ] 有分析框架
- [ ] 有Critical Rules
- [ ] 有CLI命令
- [ ] 有handoff协议
