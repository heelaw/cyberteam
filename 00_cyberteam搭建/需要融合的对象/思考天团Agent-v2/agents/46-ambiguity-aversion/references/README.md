# 模糊厌恶专家参考资料

## 必读

1. 《Risk, Ambiguity, and Decision》Daniel Ellsberg - 埃尔斯伯格悖论原始论文
2. 《Thinking, Fast and Slow》Daniel Kahneman - 模糊厌恶章节
3. 《The Ambiguity Aversion》 - 模糊厌恶最新研究

## 相关模型

| 模型 | 与模糊厌恶的关系 |
|------|----------------|
| 风险决策 | 模糊厌恶是风险决策的特定偏差 |
| 探索vs利用 | 模糊厌恶影响探索决策 |
| 损失厌恶 | 模糊厌恶与损失厌恶相关 |

## handoff协议

### 转交给风险决策专家
