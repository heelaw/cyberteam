# 模糊厌恶专家测试用例

## 正面测试

### 测试1: 职业选择
- 输入:
```
【模糊厌恶分析请求】
选项A：稳定工作，月薪1万，从不变化
选项B：新机会，可能月薪2万也可能5千
你的倾向：选A，更稳定
犹豫原因：不知道B到底怎么样
```
- 期望输出:
  - 模糊厌恶诊断
  - 未知量化分析
  - 探索价值评估
- 验证项:
  - [ ] 能识别这是典型的模糊厌恶
  - [ ] 能量化B的期望值
  - [ ] 能评估探索B的价值

### 测试2: 投资决策
- 输入:
```
【模糊厌恶分析请求】
选项A：国债，年化4%，确定
选项B：股票基金，可能赚30%可能亏20%
你的倾向：选A，因为确定
犹豫原因：股票太不确定了
```
- 期望输出:
  - 确定性价格分析
  - 探索价值评估
  - 四象限决策
- 验证项:
  - [ ] 能说明为确定性付出的代价
  - [ ] 能评估B的期望值是否值得
  - [ ] 有具体的决策建议

## 负面测试

### 测试1: 纯确定性偏好
- 输入:
```
【模糊厌恶分析请求】
选项A：确定赚100
选项B：50%赚200，50%赚0
你的倾向：选A
犹豫原因：（没有犹豫）
```
- 期望行为: 不能只因为"确定"就选A
- 验证项:
  - [ ] 必须量化期望值对比
  - [ ] 必须评估B的探索价值
  - [ ] 必须给出确定性价格

### 测试2: 盲目拒绝未知
- 输入:
```
【模糊厌恶分析请求】
选项A：继续当前工作
选项B：创业，可能大成功可能大失败
你的倾向：选A
犹豫原因：创业太不确定了，不敢
```
- 期望行为: 不能因为"不知道"就放弃未知选项
- 验证项:
  - [ ] 必须量化B的期望值
  - [ ] 必须评估探索成本和收益
  - [ ] 必须有时机判断
