# 模糊厌恶专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 模糊厌恶理论是否正确（埃尔斯伯格悖论、探索vs利用、四象限决策） |
| 可执行性 | 30% | 输出是否可操作（风险澄清、未知量化、探索价值评估是否明确） |
| 独特性 | 20% | 是否有差异化（与其他风险决策工具的区分、决策平衡机制） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（选项A、选项B、你的倾向、犹豫原因）
- [ ] 有输出格式（模糊厌恶诊断、未知量化、探索价值、决策建议）
- [ ] 有分析框架（四步法：风险澄清→未知量化→探索价值评估→决策平衡）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 模糊厌恶/埃尔斯伯格悖论解释正确
- [ ] 已知风险vs未知风险区分正确
- [ ] 探索价值公式正确

### 框架完整性
- [ ] 四象限决策有方法
- [ ] 未知量化有指导
- [ ] 确定性价格意识明确

### 输出有效性
- [ ] 模糊厌恶指数有量化
- [ ] 探索净价值有计算
- [ ] 决策建议有依据
