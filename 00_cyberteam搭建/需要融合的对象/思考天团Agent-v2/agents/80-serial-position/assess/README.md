# 系列位置效应专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 系列位置效应理论是否正确（首因效应、近因效应、中间遗忘、U型曲线） |
| 可执行性 | 30% | 输出是否可操作（信息结构分析、序列优化设计、呈现技巧是否明确） |
| 独特性 | 20% | 是否有差异化（与一般记忆技巧的区分、首因近因应用） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（场景类型、信息内容、时间限制、目标受众）
- [ ] 有输出格式（信息结构分析、序列优化设计、呈现技巧）
- [ ] 有分析框架（三步法：信息结构分析→序列优化设计→呈现技巧）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（金字塔原理、六顶思考帽、记忆宫殿）

## 核心检验点

### 理论准确性
- [ ] 首因效应和近因效应解释正确
- [ ] 中间遗忘现象解释正确
- [ ] U型曲线描述正确

### 框架完整性
- [ ] 信息分类有方法
- [ ] 位置分配有依据
- [ ] 时间分配建议合理

### 输出有效性
- [ ] 推荐顺序可执行
- [ ] 呈现技巧实用
- [ ] 记忆优化建议具体

## 相关专家

- 系列位置效应与金字塔原理：信息结构化组织
- 系列位置效应与六顶思考帽：多维度分析信息
