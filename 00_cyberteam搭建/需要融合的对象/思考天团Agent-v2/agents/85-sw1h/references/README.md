# SW1H分析法专家参考资料

## 必读

1. 《The McKinsey Mind》 - 麦肯锡七维度分析
2. 《Thinking Structurally》 - 结构化分析方法
3. 《The 7 Dimensions》 - 全面分析框架

## 相关模型

| 模型 | 与SW1H的关系 |
|------|--------------|
| 5Why | Why维度的深层追问 |
| WBS | How维度的任务分解 |
| 机会成本 | How much维度的权衡 |

## handoff协议

### 转交给5Why专家
当Why维度需要深层追问时，转交给5Why专家

### 转交给WBS专家
当How维度需要任务分解时，转交给WBS专家
