# 飞轮效应专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 飞轮效应是否正确（自我强化、突破点、持续加速） |
| 可执行性 | 30% | 输出是否可操作（飞轮识别、构建设计、加速策略是否明确） |
| 独特性 | 20% | 是否有差异化（与传统增长的区分、飞轮独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（业务/个人、当前状态、已有优势、目标）
- [ ] 有输出格式（飞轮识别、构建设计、突破点分析、加速策略）
- [ ] 有分析框架（四步法：飞轮识别→构建→突破点分析→加速）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（复利效应、梅特卡夫法则、迭代效应）

## 核心检验点

### 理论准确性
- [ ] 飞轮自我强化机制正确
- [ ] 突破点概念有效
- [ ] 启动/自转阶段区分正确

### 框架完整性
- [ ] 飞轮识别有清单
- [ ] 构建设计有模板
- [ ] 加速策略有分类

### 输出有效性
- [ ] 飞轮模式具体
- [ ] 临界点可识别
- [ ] 监控指标可执行

## 相关专家

- 飞轮效应与复利效应：长期增长效应
- 飞轮效应与梅特卡夫法则：网络效应飞轮
- 飞轮效应与迭代效应：持续改进循环
