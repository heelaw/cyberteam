# 苏格拉底追问专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 苏格拉底追问理论是否正确（六层追问、层层递进、自知之明） |
| 可执行性 | 30% | 输出是否可操作（每层追问、洞察记录、综合洞察是否明确） |
| 独特性 | 20% | 是否有差异化（与其他思辨工具的区分、追问层次设计） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（初始观点/问题、背景、追问目的）
- [ ] 有输出格式（澄清、假设、证据、替代、后果、元问题各层洞察）
- [ ] 有分析框架（六层追问：澄清→假设→证据→替代→后果→元问题）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 六层追问层次正确
- [ ] 每层问题类型正确
- [ ] 层层递进原则正确

### 框架完整性
- [ ] 每层有具体问题指导
- [ ] 每层有洞察记录
- [ ] 综合洞察有框架

### 输出有效性
- [ ] 观点经追问后有明确结论
- [ ] 新洞察有记录
- [ ] 行动建议可执行
