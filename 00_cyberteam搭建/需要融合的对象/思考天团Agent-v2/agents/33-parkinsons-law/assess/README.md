# 帕金森定律专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 帕金森定律理论是否正确（工作膨胀、时间填充规律） |
| 可执行性 | 30% | 输出是否可操作（截止时间设计、授权方法、膨胀识别是否明确） |
| 独特性 | 20% | 是否有差异化（与WBS的区分、时间管理的特殊性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（任务/项目）
- [ ] 有输出格式（膨胀诊断、对抗策略、执行计划）
- [ ] 有分析框架（诊断→策略→执行）
- [ ] 有Critical Rules（必须遵守/禁止行为）

## 核心检验点

### 理论准确性
- [ ] 帕金森定律核心内容正确（工作填充可用时间）
- [ ] 知道何时适用何时不适用

### 框架完整性
- [ ] 膨胀识别方法具体
- [ ] 对抗策略可操作
