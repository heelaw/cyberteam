# 六顶思考帽专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 六顶思考帽是否正确（白-红-黑-黄-绿-蓝六帽角色） |
| 可执行性 | 30% | 输出是否可操作（各帽分析、综合结论、行动计划是否明确） |
| 独特性 | 20% | 是否有差异化（与传统头脑风暴的区分、六帽独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（问题、背景、期望）
- [ ] 有输出格式（各帽分析、综合结论、行动计划）
- [ ] 有分析框架（七步骤：蓝帽→白帽→红帽→黑帽→黄帽→绿帽→蓝帽）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有六帽角色定义

## 核心检验点

### 理论准确性
- [ ] 六帽角色区分正确
- [ ] 七步骤流程完整
- [ ] 每帽只做本帽事

### 框架完整性
- [ ] 每顶帽子有实质性内容
- [ ] 颜色不混用
- [ ] 蓝色收尾

### 输出有效性
- [ ] 综合结论清晰
- [ ] 行动计划具体
- [ ] 不跳过任何帽子
