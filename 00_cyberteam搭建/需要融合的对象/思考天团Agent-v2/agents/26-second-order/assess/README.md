# 二阶思维专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 二阶思维理论是否正确（一阶/二阶/三阶区分、反馈循环识别） |
| 可执行性 | 30% | 输出是否可操作（因果链、尾部风险、监控指标是否明确） |
| 独特性 | 20% | 是否有差异化（与逆向思维的区分、多层次推理的实用性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（决策选项、一阶预期、时间范围）
- [ ] 有输出格式（一阶结果、二阶链、三阶效应、尾部风险）
- [ ] 有分析框架（五步法：一阶→二阶→三阶→时间→尾部）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有CLI命令（如有）
- [ ] 有handoff协议

## 核心检验点

### 理论准确性
- [ ] 一阶/二阶/三阶思维区分正确
- [ ] 反馈循环（正反馈/负反馈）识别正确
- [ ] 干预陷阱识别方法正确

### 框架完整性
- [ ] 因果链包含推动力和适应性行为
- [ ] 时间线分短期/中期/长期
- [ ] 尾部风险包含最坏情况、不可逆效应、黑天鹅

### 输出有效性
- [ ] 二阶结果链格式正确
- [ ] 有具体的战略建议
- [ ] 有可监控的指标
