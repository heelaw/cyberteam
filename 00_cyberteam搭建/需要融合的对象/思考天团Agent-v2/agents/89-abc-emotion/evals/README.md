# 情绪ABC理论专家测试用例

## 正面测试

### 测试1: 职场挫折分析
- 输入:
```
【情绪ABC分析请求】
触发事件A：晋升失败
情绪结果C：极度沮丧和愤怒
```
- 期望输出:
  - 识别潜在信念B（"我必须每次都晋升"、"领导不认可我"）
  - 反驳D（这些信念的证据？逻辑是否合理？）
  - 认知重建E（新的理性信念、更健康的情绪）
- 验证项:
  - [ ] ABC三要素完整
  - [ ] 信念识别具体非笼统
  - [ ] 有反驳和重建

### 测试2: 人际冲突分析
- 输入:
```
【情绪ABC分析请求】
触发事件A：朋友没有回复消息
情绪结果C：焦虑、猜疑
```
- 期望输出:
  - 识别潜在信念（"朋友不在乎我"、"我做错了什么"）
  - 反驳（有没有其他解释？）
  - 认知重建（新的理解角度）
- 验证项:
  - [ ] 识别多种可能信念
  - [ ] 反驳有证据支持
  - [ ] 新旧情绪对比明确

## 负面测试

### 测试1: 跳过信念识别
- 输入:
```
【情绪ABC分析请求】
触发事件A：被客户批评
情绪结果C：生气
```
- 期望行为: 不能跳过信念B的识别
- 验证项:
  - [ ] 必须识别信念B
  - [ ] 不能直接跳到解决方案
  - [ ] 信念必须具体化

### 测试2: 非理性信念未反驳
- 输入:
```
【情绪ABC分析请求】
触发事件A：求职被拒
情绪结果C：自我否定
```
- 期望行为: 必须对非理性信念进行有效反驳
- 验证项:
  - [ ] 识别非理性信念
  - [ ] 提供反驳证据
  - [ ] 不能仅安慰要重建认知
