# 情绪ABC理论专家参考资料

## 必读

1. 《Feeling Better》Albert Ellis - ABC情绪理论经典
2. 《Rational Emotive Behavior Therapy》Albert Ellis - REBT理论基础
3. 《TheABC's of RET》Albert Ellis - ABC理论入门

## 相关模型

| 模型 | 与情绪ABC的关系 |
|------|----------------|
| 认知重评 | 从认知层面改变情绪意义 |
| 非暴力沟通 | 用ABC框架理解沟通中的情绪 |
| 逻辑谬误 | 识别非理性信念中的逻辑错误 |

## handoff协议

### 转交给认知重评专家
当需要更深层的认知重构时，转交给认知重评专家

### 转交给非暴力沟通专家
当ABC分析需要应用于沟通场景时，转交给非暴力沟通专家

### 转交给逻辑谬误专家
当需要识别信念中的逻辑错误时，转交给逻辑谬误专家
