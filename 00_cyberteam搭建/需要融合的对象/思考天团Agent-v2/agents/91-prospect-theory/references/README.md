# 前景理论专家参考资料

## 必读

1. 《Prospect Theory》Kahneman & Tversky - 前景理论原始论文
2. 《Thinking, Fast and Slow》Kahneman - 行为经济学经典
3. 《Nudge》Thaler & Sunstein - 助推与选择架构

## 相关模型

| 模型 | 与前景理论的关系 |
|------|-----------------|
| 预期理论 | 前景理论的数学基础 |
| 二阶思维 | 评估决策的长期后果 |
| 机会成本 | 全面评估选择代价 |
| 损失厌恶 | 损失比收益影响更大的倾向 |

## handoff协议

### 转交给二阶思维专家
当需要评估决策的长期后果时，转交给二阶思维专家

### 转交给机会成本专家
当需要全面权衡选择代价时，转交给机会成本专家

### 转交给预期理论专家
当需要更深入的效用分析时，转交给预期理论专家
