# 前景理论专家测试用例

## 正面测试

### 测试1: 投资决策分析
- 输入:
```
【前景理论分析请求】
决策情境：股票亏损20%，是否割肉
当前框架：损失框架（想着"亏了"）
潜在收益/损失：可能反弹vs继续下跌
```
- 期望输出:
  - 参照点分析（当前参照点是否合理）
  - 框架诊断（损失框架导致的偏差）
  - 理性校准建议（如何重新框架）
- 验证项:
  - [ ] 正确识别损失厌恶
  - [ ] 提供框架转换方法
  - [ ] 校准建议可执行

### 测试2: 职业选择
- 输入:
```
【前景理论分析请求】
决策情境：现有工作稳定vs新工作高风险高回报
当前框架：损失框架（"放弃稳定"）
潜在收益/损失：稳定vs成长
```
- 期望输出:
  - 参照点重设（稳定是否是真正的参照点）
  - 偏差识别（损失厌恶程度）
  - 理性校准（基于概率和效用的分析）
- 验证项:
  - [ ] 参照点分析深入
  - [ ] 偏差量化
  - [ ] 校准后决策更理性

## 负面测试

### 测试1: 框架固化
- 输入:
```
【前景理论分析请求】
决策情境：项目失败后的选择
当前框架：极度损失厌恶
潜在收益/损失：激进方案vs保守方案
```
- 期望行为: 不能简单说"要理性"，要给出具体框架转换步骤
- 验证项:
  - [ ] 有具体框架转换方法
  - [ ] 不是简单的"要乐观"
  - [ ] 有实践检验步骤

### 测试2: 概率忽视
- 输入:
```
【前景理论分析请求】
决策情境：是否买彩票
当前框架：小概率高收益
潜在收益/损失：极小概率大奖
```
- 期望行为: 必须识别概率权重偏差，不能鼓励赌博心态
- 验证项:
  - [ ] 识别概率权重偏差
  - [ ] 不合理化赌博行为
  - [ ] 给出理性决策标准
