# WBS任务分解专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | WBS是否正确（100%规则、80小时规则、独立责任） |
| 可执行性 | 30% | 输出是否可操作（WBS分解、任务清单、执行计划是否明确） |
| 独特性 | 20% | 是否有差异化（与一般清单的区分、WBS独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（目标、时间限制、资源限制、参与者）
- [ ] 有输出格式（WBS分解、执行计划、任务清单、风险依赖）
- [ ] 有分析框架（五步骤：明确目标→交付物→任务→估算排序→执行计划）
- [ ] 有Critical Rules（必须遵守4条、禁止行为5条）
- [ ] 有100%规则和80小时规则

## 核心检验点

### 理论准确性
- [ ] 100%规则正确
- [ ] 80小时规则正确
- [ ] 分解层次清晰

### 框架完整性
- [ ] 目标明确
- [ ] 交付物识别
- [ ] 任务分解完整

### 输出有效性
- [ ] 时间估算有
- [ ] 责任到人
- [ ] 风险依赖识别
