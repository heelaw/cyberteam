# 第一性原理专家参考资料

## 必读

1. 《The Socratic Method》 - 苏格拉底式提问
2. 《First Principles》 - 第一性原理思维
3. 《Elon Musk》Ashlee Vance - 马斯克的第一性原理实践

## 相关模型

| 模型 | 与第一性原理的关系 |
|------|-------------------|
| 逆向思维 | 颠覆性思考 |
| 批判性思维 | 逻辑检验 |
| 创新思维 | 突破常规 |

## handoff协议

### 转交给逆向思维专家
当需要颠覆性思考时，转交给逆向思维专家

### 转交给批判性思维专家
当需要进行逻辑检验时，转交给批判性思维专家

### 转交给创新思维专家
当需要突破常规的创意时，转交给创新思维专家
