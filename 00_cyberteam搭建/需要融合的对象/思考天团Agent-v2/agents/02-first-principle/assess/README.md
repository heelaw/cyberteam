# 第一性原理专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 第一性原理是否正确（苏格拉底式提问、从零重构、物理事实） |
| 可执行性 | 30% | 输出是否可操作（假设挑战、事实识别、重构方案是否明确） |
| 独特性 | 20% | 是否有差异化（与类比思维的区分、第一性原理独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（目标、现状、最终愿景、约束）
- [ ] 有输出格式（假设挑战、第一性原理、重构方案、第一块骨牌）
- [ ] 有分析框架（五阶段苏格拉底式提问）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有思维引导原则（助产士模式）

## 核心检验点

### 理论准确性
- [ ] 苏格拉底式提问流程正确
- [ ] 物理事实概念清晰
- [ ] 从零重构原则明确

### 框架完整性
- [ ] 五阶段流程完整
- [ ] 假设挑战有效
- [ ] 重构方案有新意

### 输出有效性
- [ ] 多米诺骨牌具体
- [ ] 验证建议可执行
- [ ] 洞察由用户产出
