# 回归均值专家测试用例

## 正面测试

### 测试1: 孩子成绩波动
- 输入:
```
【回归均值分析请求】
极端事件：这次考试得了100分
正常表现：平时都是80分左右
观察时间：一直关注每次考试
归因解释：最近学习特别努力
```
- 期望输出:
  - 极端程度评估
  - 统计回归vs因果分析
  - 归因检验
- 验证项:
  - [ ] 能识别100分可能是极端值
  - [ ] 能说明下次可能趋向80分
  - [ ] 能区分"努力"干预是否在极端前

### 测试2: 冠军魔咒
- 输入:
```
【回归均值分析请求】
极端事件：去年得了冠军，今年成绩下滑
正常表现：前年第3名，去年第1名，今年第8名
观察时间：3年
归因解释：夺冠后心态变了
```
- 期望输出:
  - 极端程度分析
  - "冠军魔咒"归因检验
  - 统计回归解释
- 验证项:
  - [ ] 能识别夺冠可能是极端表现
  - [ ] 能质疑"心态变了"的因果归因
  - [ ] 能用统计回归解释成绩下滑

## 负面测试

### 测试1: 把相关当因果
- 输入:
```
【回归均值分析请求】
极端事件：上次批评后成绩提高了
正常表现：平时80分，批评后90分
观察时间：1年
归因解释：批评有效
```
- 期望行为: 不能把时间继发当作因果
- 验证项:
  - [ ] 必须质疑批评发生在极端值（高分）之后
  - [ ] 必须说明不批评也可能回归
  - [ ] 必须要求对照组

### 测试2: 急于归因
- 输入:
```
【回归均值分析请求】
极端事件：这次项目成功了
正常表现：以前成功率只有30%
观察时间：只看这一次
归因解释：新方法有效
```
- 期望行为: 不能在观察期不足时归因
- 验证项:
  - [ ] 必须要求多次观察
  - [ ] 必须识别这是单次极端观察
  - [ ] 必须等待回归完成后再评估
