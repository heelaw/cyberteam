# 回归均值专家参考资料

## 必读

1. 《Regression Toward Fraternity》Francis Galton - 回归均值原始发现
2. 《Thinking, Fast and Slow》Daniel Kahneman - 关于回归均值的章节
3. 《The Triumph of Uncertainty》 - 统计思维经典

## 相关模型

| 模型 | 与回归均值的关系 |
|------|----------------|
| 赌徒谬误 | GF错误地认为概率会纠正，RM是真实统计现象 |
| 批判性思维 | 都涉及避免错误归因 |
| 实验设计 | 对照组是避免错误归因的关键 |

## handoff协议

### 转交给批判性思维专家
当需要更深层的归因分析时，转交给批判性思维专家

### 转交给实验设计专家
当需要设计对照实验验证因果时，转交给实验设计专家
