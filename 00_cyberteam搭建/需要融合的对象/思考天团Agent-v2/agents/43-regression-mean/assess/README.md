# 回归均值专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 回归均值理论是否正确（统计回归vs因果回归、极端值识别、归因分析） |
| 可执行性 | 30% | 输出是否可操作（极端程度评估、均值计算、干预决策是否明确） |
| 独特性 | 20% | 是否有差异化（与赌徒谬误的区分、统计思维） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（极端事件、正常表现、观察时间、归因解释）
- [ ] 有输出格式（极端程度分析、归因分析、决策建议）
- [ ] 有分析框架（四步法：识别极端值→区分两种回归→避免错误归因→均值计算）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 统计回归（不可避免）解释正确
- [ ] 因果回归（真实变化）解释正确
- [ ] 两者区分方法正确

### 框架完整性
- [ ] 极端程度评估有量化方法
- [ ] 错误归因案例有说明
- [ ] 正确归因要求明确

### 输出有效性
- [ ] 均值计算有公式指导
- [ ] 干预决策有时序考虑
- [ ] 心理预期有建设

## 相关专家

- 回归均值与赌徒谬误：RM是真实统计现象，GF是错误信念
- 回归均值与批判性思维：都涉及避免错误归因
