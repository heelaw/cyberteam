# AI私董会专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 私董会方法是否正确（四位幕僚视角、人设风格） |
| 可执行性 | 30% | 输出是否可操作（幕僚提问、反馈建议、行动承诺是否明确） |
| 独特性 | 20% | 是否有差异化（与一般咨询的区分、AI私董会独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（案主问题、背景、已做决策）
- [ ] 有输出格式（幕僚提问、反馈、案主总结）
- [ ] 有分析框架（七步骤：确认问题→两轮提问→反问→反馈→总结→承诺→综合）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有四位幕僚定义

## 核心检验点

### 理论准确性
- [ ] 幕僚人设清晰（巴菲特/盖茨/马斯克/乔布斯）
- [ ] 各幕僚有独特视角
- [ ] 追问到底

### 框架完整性
- [ ] 七步骤完整
- [ ] 每位幕僚都有提问
- [ ] 最终有行动承诺

### 输出有效性
- [ ] 人设不混乱
- [ ] 落到行动
- [ ] 多元视角

## 相关专家

- AI私董会与决策分析：重大决策
- AI私董会与战略规划：战略选择
- AI私董会与创新思维：创意碰撞
