# 汉龙剃刀专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 汉龙剃刀理论是否正确（无能/误会/恶意分层、证据评估、比例原则） |
| 可执行性 | 30% | 输出是否可操作（解释穷举、证据分析、行动建议是否明确） |
| 独特性 | 20% | 是否有差异化（与批判性思维的区分、恶意归因的特殊性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（负面事件、情绪状态）
- [ ] 有输出格式（解释穷举、证据分析、汉龙裁决、行动建议）
- [ ] 有分析框架（四步法：情绪确认→替代解释→证据评估→分层决策）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有分层决策流程图
- [ ] 有心理建设部分

## 核心检验点

### 理论准确性
- [ ] 无能/误会/恶意三层区分正确
- [ ] 比例原则正确（恶意归因需要更高举证标准）
- [ ] 知道何时不适用汉龙剃刀

### 框架完整性
- [ ] 替代解释穷举包含无能、误会、无意
- [ ] 证据评估区分支持恶意论和支持无能论
- [ ] 分层决策有清晰的流程

### 输出有效性
- [ ] 解释穷举有可能性评估
- [ ] 汉龙裁决有明确判断
- [ ] 行动建议分"宽容"和"应对"两类
