# 汉龙剃刀专家测试用例

## 正面测试

### 测试1: 同事忽视你的意见
- 输入:
```
【汉龙剃刀分析请求】
负面事件：开会时提了个建议，没人回应，领导直接跳过了
当事人：领导和同事
你的解读：他们故意忽视我，不重视我的意见
情绪状态：愤怒、受伤
```
- 期望输出:
  - 有情绪确认（愤怒是合理的）
  - 穷举了替代解释（领导不知道、信息缺失、优先级冲突）
  - 评估了证据（是只针对你还是对所有人都这样）
  - 给出分层裁决
- 验证项:
  - [ ] 不要求用户压抑情绪
  - [ ] 提供"无能"的替代解释
  - [ ] 有心理建设部分

### 测试2: 朋友放鸽子
- 输入:
```
【汉龙剃刀分析请求】
负面事件：约好聚会，朋友临时说来不了
当事人：多年的朋友
你的解读：他根本不在乎我
情绪状态：失望、被忽视
```
- 期望输出:
  - 穷举了替代解释（真的有急事、沟通误会、优先级冲突）
  - 评估了证据（是不是只对你这样）
  - 给出裁决和行动建议
- 验证项:
  - [ ] 不是要求"要大度"
  - [ ] 有具体的沟通建议

## 负面测试

### 测试1: 确实存在恶意
- 输入:
```
【汉龙剃刀分析请求】
负面事件：有人在背后散布谣言损害你名誉
当事人：同事
你的解读：对方是故意的
情绪状态：愤怒
证据：有证人听到、多次发生、知道会伤害你
```
- 期望行为: 当恶意证据充分时，不回避汉龙裁决
- 验证项:
  - [ ] 有明确的恶意裁决
  - [ ] 有设立边界和反击的建议

### 测试2: 道德说教倾向
- 输入:
```
【汉龙剃刀分析请求】
负面事件：合作伙伴没有履行承诺
当事人：合作方
你的解读：他故意欺骗我
情绪状态：愤怒
```
- 期望行为: 不能要求用户"要大度"或"理解对方"
- 验证项:
  - [ ] 基于证据做裁决
  - [ ] 行动建议是设立边界，不是道德说教

### 测试3: 阴谋论倾向
- 输入:
```
【汉龙剃刀分析请求】
负面事件：公司宣布裁员10%
你的解读：高层故意的，要搞垮我们
情绪状态：愤怒、不信任
```
- 期望行为: 帮助识别过度解读
- 验证项:
  - [ ] 提供"无能"的替代解释（公司真的亏损、管理失误）
  - [ ] 不是否定用户情绪，而是提供更合理的归因
