# 反事实思维专家测试用例

## 正面测试

### 测试1: 投资失败复盘
- 输入:
```
【反事实思维分析请求】
事件/决策：投资某股票亏损50%
实际结果：亏损50%
预期结果：预期盈利20%
可控程度：选股决策可控，宏观经济不可控
```
- 期望输出:
  - 事件解剖
  - 上行反事实分析
  - 经验提取
- 验证项:
  - [ ] 有时间线梳理
  - [ ] 有"如果做了不同选择"的反事实假设
  - [ ] 有具体教训和检查清单

### 测试2: 职业决策
- 输入:
```
【反事实思维分析请求】
事件/决策：拒绝了某工作邀请
实际结果：留在原公司，错过新机会
预期结果：（希望有更好发展）
可控程度：可以联系猎头重新争取
```
- 期望输出:
  - 关键决策点
  - 上行反事实（如果接受会怎样）
  - 下行反事实（如果情况更糟会怎样）
- 验证项:
  - [ ] 有聚焦可控因素的分析
  - [ ] 有经验提取
  - [ ] 有决策检查清单

## 负面测试

### 测试1: 沉溺于遗憾
- 输入:
```
【反事实思维分析请求】
事件/决策：3年前没买房
实际结果：现在房价涨了
预期结果：（后悔不已）
可控程度：（已无法改变）
```
- 期望行为: 不能沉溺于不可控的遗憾
- 验证项:
  - [ ] 必须聚焦可控因素
  - [ ] 必须提取可操作教训
  - [ ] 不能停留在感慨

### 测试2: 只关注不可控
- 输入:
```
【反事实思维分析请求】
事件/决策：项目失败
实际结果：项目取消
预期结果：（本来指望这个项目）
可控程度：市场竞争不可控
```
- 期望行为: 不能只关注不可控因素
- 验证项:
  - [ ] 必须聚焦可控因素
  - [ ] 必须有具体改进措施
  - [ ] 不能为失败找借口
