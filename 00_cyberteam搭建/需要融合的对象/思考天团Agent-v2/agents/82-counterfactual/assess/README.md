# 反事实思维专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 反事实思维理论是否正确（上行反事实、下行反事实、沉淀效应、可控性聚焦） |
| 可执行性 | 30% | 输出是否可操作（事件解剖、上行/下行反事实构建、经验提取是否明确） |
| 独特性 | 20% | 是否有差异化（与一般复盘的区分、假设性推理） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（事件/决策、实际结果、预期结果、可控程度）
- [ ] 有输出格式（事件解剖、上行反事实分析、下行反事实分析、经验提取）
- [ ] 有分析框架（四步法：事件解剖→上行反事实构建→下行反事实构建→经验提取）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（KISS复盘、5Why、二阶思维）

## 核心检验点

### 理论准确性
- [ ] 上行反事实概念正确
- [ ] 下行反事实概念正确
- [ ] 可控性聚焦原则正确

### 框架完整性
- [ ] 事件解剖有时间线
- [ ] 反事实构建有具体方法
- [ ] 经验提取有框架

### 输出有效性
- [ ] 决策检查清单可操作
- [ ] 未来应用建议具体
- [ ] 教训可落实

## 相关专家

- 反事实思维与KISS复盘：复盘框架整合
- 反事实思维与5Why：深层原因挖掘
- 反事实思维与二阶思维：后果链延伸
