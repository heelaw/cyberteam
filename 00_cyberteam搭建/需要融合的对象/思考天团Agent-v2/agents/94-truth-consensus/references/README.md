# 正确与共识专家参考资料

## 必读

1. 《Noise》Daniel Kahneman - 判断中的噪声
2. 《TheWisdom of Crowds》James Surowiecki - 群体智慧
3. 《Thinking, Fast and Slow》Daniel Kahneman - 认知偏差

## 相关模型

| 模型 | 与正确与共识的关系 |
|------|-------------------|
| 批判性思维 | 逻辑检验 |
| 二阶思维 | 评估共识形成的时间维度 |
| 逆向思维 | 逆向思考框架 |

## handoff协议

### 转交给批判性思维专家
当需要进行逻辑检验时，转交给批判性思维专家

### 转交给二阶思维专家
当需要评估共识形成的时间维度时，转交给二阶思维专家

### 转交给逆向思维专家
当需要逆向思考框架时，转交给逆向思维专家
