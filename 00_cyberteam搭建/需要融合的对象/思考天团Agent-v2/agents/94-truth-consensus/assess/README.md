# 正确与共识专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 正确与共识理论是否正确（共识形成机制、正确性检验、信号识别） |
| 可执行性 | 30% | 输出是否可操作（共识识别、裂缝检测、决策策略是否明确） |
| 独特性 | 20% | 是否有差异化（与传统决策理论的区分、独立思考独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（情境、主流观点、你的判断、信息来源）
- [ ] 有输出格式（共识识别、正确性评估、信号识别、决策建议）
- [ ] 有分析框架（四步法：识别共识→正确性检验→信号识别→决策策略）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（批判性思维、二阶思维、逆向思维）

## 核心检验点

### 理论准确性
- [ ] 共识形成机制正确（社会认同、信息瀑布、沉默螺旋）
- [ ] 正确性维度完整
- [ ] 裂缝信号分类有效

### 框架完整性
- [ ] 共识识别有清单
- [ ] 正确性检验有标准
- [ ] 决策策略有分类

### 输出有效性
- [ ] 仓位建议可量化
- [ ] 监控指标具体
- [ ] 行动清单可执行

- 正确与共识与二阶思维：评估共识形成的时间维度
- 正确与共识与逆向思维：逆向思考框架
