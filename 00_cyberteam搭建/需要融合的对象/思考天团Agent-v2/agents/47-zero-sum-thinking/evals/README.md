# 零和思维专家测试用例

## 正面测试

### 测试1: 商业竞争
- 输入:
```
【零和思维分析请求】
情境：和竞争对手争夺同一个客户
你的假设：这是一个零和博弈，我赢他就输
相关方：我方、竞争对手、客户
利益结构：我方要赢得合同，竞争对手也要
```
- 期望输出:
  - 博弈类型诊断
  - 正和可能性分析
  - 整合式谈判策略
- 验证项:
  - [ ] 能质疑"零和"假设
  - [ ] 能识别合作可能性（如竞合）
  - [ ] 有做大蛋糕的方案

### 测试2: 薪酬争议
- 输入:
```
【零和思维分析请求】
情境：团队加薪总额固定，如何分配
你的假设：给你多了给我的就少了
相关方：团队成员、管理层
利益结构：每个人都想要更高薪酬
```
- 期望输出:
  - 负和诊断
  - 正和机会分析
  - 整合式分配方案
- 验证项:
  - [ ] 能识别固定总额的分配是零和
  - [ ] 能提出扩大薪酬池的方案
  - [ ] 有信任建设机制

## 负面测试

### 测试1: 零和预设
- 输入:
```
【零和思维分析请求】
情境：和供应商谈判价格
你的假设：商业就是零和，他赚就是我亏
相关方：我方、供应商
利益结构：都想争取有利价格
```
- 期望行为: 不能预设所有商业都是零和
- 验证项:
  - [ ] 必须检验零和假设
  - [ ] 必须寻找正和可能性
  - [ ] 必须考虑长期关系价值

### 测试2: 对抗心态
- 输入:
```
【零和思维分析请求】
情境：离婚财产分割
你的假设：必须争取更多，否则我吃亏
相关方：双方
利益结构：都想最大化自己份额
```
- 期望行为: 不能陷入"必须赢"的思维
- 验证项:
  - [ ] 必须评估合作障碍
  - [ ] 必须有保护机制
  - [ ] 必须考虑长期成本
