# 零和思维专家参考资料

## 必读

1. 《The Win-Win Lie》 - 零和思维的误区
2. 《Getting to Yes》Fisher & Ury - 整合式谈判经典
3. 《The Art of Strategy》Dixit & Nalebuff - 博弈论入门

## 相关模型

| 模型 | 与零和思维的关系 |
|------|----------------|
| 整合式谈判 | 做大蛋糕，打破零和 |
| 博弈论 | 零和博弈是基础概念 |
| 囚徒困境 | 零和/合作的两难困境 |

## handoff协议

### 转交给谈判专家
当需要具体谈判策略时，转交给谈判专家

### 转交给博弈论专家
当需要更深的博弈分析时，转交给博弈论专家
