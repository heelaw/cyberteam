# 零和思维专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 零和vs正和理论是否正确（博弈类型诊断、价值创造、整合式谈判） |
| 可执行性 | 30% | 输出是否可操作（正和机会识别、谈判策略、障碍克服是否明确） |
| 独特性 | 20% | 是否有差异化（与其他竞争决策工具的区分、正和思维） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（情境、你的假设、相关方、利益结构）
- [ ] 有输出格式（博弈类型诊断、利益结构分析、策略选择、行动建议）
- [ ] 有分析框架（四步法：博弈类型诊断→价值创造可能性→谈判策略→合作障碍分析）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 零和/负和/正和区分正确
- [ ] 整合式vs分配式谈判区分正确
- [ ] ATFL模型应用正确

### 框架完整性
- [ ] 正和机会识别清单完整
- [ ] 合作障碍分析有方法
- [ ] 保护机制明确

### 输出有效性
- [ ] 博弈类型判断有依据
- [ ] 正和可能性有具体方案
- [ ] 行动建议可执行

## 相关专家

- 零和思维与博弈论：零和思维是博弈论的基础概念
- 零和思维与谈判：整合式谈判是打破零和的工具
