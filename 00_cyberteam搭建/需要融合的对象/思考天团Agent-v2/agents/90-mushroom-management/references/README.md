# 蘑菇管理专家参考资料

## 必读

1. 《The Mushroom Principle》 - 蘑菇管理法则经典
2. 《Upward Management》 - 向上管理的艺术
3. 《Taking Charge》 - 职场主动权指南

## 相关模型

| 模型 | 与蘑菇管理的关系 |
|------|-----------------|
| 向上管理 | 蘑菇期如何获得上级支持 |
| 主动领导 | 不等待主动争取机会 |
| SWOT分析 | 内外部环境分析找到突破口 |

## handoff协议

### 转交给向上管理专家
当需要学习如何与上级有效沟通获得支持时，转交给向上管理专家

### 转交给主动领导专家
当需要发展主动承担责任的能力时，转交给主动领导专家

### 转交给SWOT分析专家
当需要系统分析内外部环境时，转交给SWOT分析专家
