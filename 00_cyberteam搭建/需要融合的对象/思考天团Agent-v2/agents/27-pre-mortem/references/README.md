# 事前尸检专家参考资料

## 必读

1. 《Thinking, Fast and Slow》Daniel Kahneman - 事前尸检的心理学基础
2. 《The Black Swan》Nassim Nicholas Taleb - 不可预见的极端事件
3. 《Project Retrospectives》Norm Kerth - 事后复盘与事前预防的对比

## 案例库

### 成功案例
- **NASA的飞行准备检查**: 通过事前尸检识别了挑战者号发射的潜在风险（虽然最终未能阻止）
- **微软Windows Vista开发**: 运用事前尸检识别了项目复杂度的风险
- **医院手术前核查**: 医疗行业的"Time-out"制度本质上是事前尸检的应用

### 失败案例
- **挑战者号航天飞机**: 忽视了工程师对O形环的警告，未进行充分的事前风险分析
- **铱星计划**: 过度乐观的市场预测，未能识别技术替代风险
- **新产品发布失败**: 缺乏充分的用户测试和风险验证

## 相关模型

| 模型 | 与事前尸检的关系 |
|------|----------------|
| KISS复盘 | 事前尸检的"反向"，事后复盘vs事前预防 |
| 5Why分析 | 事前尸检的根因追问部分 |
| SWOT分析 | 事前尸检的内外部分析可以借鉴 |
| 决策树 | 事前尸检的风险量化可以借助决策树 |

## handoff协议

### 转交给KISS复盘专家
当项目失败需要复盘时，转交给KISS复盘专家进行事后分析

### 转交给第一性原理专家
当需要从零分析失败根因时，转交给第一性原理专家进行深度追问

### 转交给系统思维专家
当失败涉及复杂的反馈回路和系统性因素时，转交给系统思维专家分析
