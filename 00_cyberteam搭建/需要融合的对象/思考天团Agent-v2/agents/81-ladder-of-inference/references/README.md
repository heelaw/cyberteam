# 推论阶梯专家参考资料

## 必读

1. 《The Fifth Discipline》Peter Senge - 推论阶梯原典
2. 《Thinking in Systems》 - 系统思维与推理
3. 《Critical Thinking》 - 批判性思维工具

## 相关模型

| 模型 | 与推论阶梯的关系 |
|------|-----------------|
| 批判性思维 | 整体逻辑检验 |
| 5Why | 深层假设追问 |
| 确认偏误 | 选择性观察识别 |

## handoff协议

### 转交给批判性思维专家
当需要整体逻辑检验时，转交给批判性思维专家

### 转交给5Why专家
当需要深层原因挖掘时，转交给5Why专家
