# 复利效应专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 复利理论是否正确（指数增长/雪球效应/临界点/正负复利区分） |
| 可执行性 | 30% | 输出是否可操作（时间价值表/增长评估/策略建议是否明确） |
| 独特性 | 20% | 是否有差异化（与机会成本/成长思维专家的区分） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（情境描述/初始投入/预期行动/时间维度）
- [ ] 有输出格式（复利识别表格/增长评估/时间价值表/策略建议）
- [ ] 有分析框架（五步法：复利识别→增长率评估→时间价值→触发条件→策略建议）
- [ ] 有Critical Rules（必须遵守4条、禁止行为5条）
- [ ] 有CLI命令（如有）
- [ ] 有handoff协议

## 核心检验点

### 理论准确性
- [ ] 指数增长 vs 线性增长区分正确
- [ ] 复利公式 F = P × (1 + r)^n 正确使用
- [ ] 临界点概念被正确解释

### 框架完整性
- [ ] 复利识别四维度完整（自我强化/边际成本递减/网络效应/知识累积）
- [ ] 增长率评估包含年化增长率/复利周期/临界点/上限分析
- [ ] 时间价值表有多时间节点

### 输出有效性
- [ ] 复利识别表格格式正确
- [ ] 增长评估有量化数字
- [ ] 有明确的策略建议清单
- [ ] 风险提示完整（负复利/中断代价/虚假复利）

### 独特性检验
- [ ] 与机会成本专家区分清晰（复利强调长期累积，机会成本强调选择权衡）
- [ ] 与成长思维专家区分清晰（复利强调指数效应，成长思维强调进步曲线）
