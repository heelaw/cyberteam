# 复利效应专家参考资料

## 必读

1. 《爱因斯坦传》- "复利是世界第八大奇迹"
2. 《穷查理的普通常识》- 查理·芒格的"多元思维模型"强调复利效应
3. 《原则》Ray Dalio - "理解复利的力量是理解增长的关键"

## 案例库

### 成功案例
- **巴菲特的滚雪球**: 50年年化20%增长，从10万到数百亿
- **习惯的复利**: 每天进步1%，一年后提升37倍（1.01^365 ≈ 37.78）
- **知识的复利**: 芒格通过跨学科知识叠加形成独特判断力

### 失败案例
- **负复利积累**: 每天熬夜2小时，半年后健康指数断崖式下跌
- **虚假复利**: 简单数量叠加而非指数累积，团队扩张≠复利增长
- **复利中断**: 投资中断导致雪球效应消失，前期积累大幅缩水

## 相关模型

| 模型 | 与复利效应的关系 |
|------|----------------|
| 指数增长 | 复利的数学表达 |
| 边际成本递减 | 复利的成本特征 |
| 网络效应 | 复利的一种形式（用户越多价值越高） |
| 习惯养成 | 复利在个人成长中的应用 |
| 临界点理论 | 解释复利何时从线性转为指数 |

## 延伸阅读

- 复利思维与长期主义
- 投资中的复利与风险
- 个人成长中的复利效应
- 负复利：坏习惯的指数级危害
