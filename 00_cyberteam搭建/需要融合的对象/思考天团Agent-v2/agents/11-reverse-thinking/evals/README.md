# 逆向思维专家测试用例

## 正面测试

### 测试1: 人生目标规划
- 输入:
```
【逆向思维分析请求】
终极目标：10年后财务自由
当前状态：普通工薪层
时间框架：10年
```
- 期望输出:
  - 终局描绘（生动具体）
  - 逆向路径（里程碑倒推）
  - 北极星原则（核心价值观）
  - 第一基石（24小时可启动）
- 验证项:
  - [ ] 终局具体可感知
  - [ ] 里程碑从后往前
  - [ ] 有具体行动

### 测试2: 企业愿景
- 输入:
```
【逆向思维分析请求】
终极目标：5年后成为行业领导者
当前状态：行业第三
时间框架：5年
```
- 期望输出:
  - 终局状态
  - 逆向里程碑
  - 核心驱动力
  - 第一步
- 验证项:
  - [ ] 倒推有序
  - [ ] 原则抽象
  - [ ] 行动具体

## 负面测试

### 测试1: 跳过终局
- 输入:
```
【逆向思维分析请求】
终极目标：成功
当前状态：一般
```
- 期望行为: 不能在终局不清晰时就开始倒推
- 验证项:
  - [ ] 先描绘具体终局
  - [ ] 不能模糊的"成功"
  - [ ] 终局是起点

### 测试2: 正向思维
- 输入:
```
【逆向思维分析请求】
终极目标：成为专家
当前状态：入门
```
- 期望行为: 不能用"先做A再做B"的正向逻辑
- 验证项:
  - [ ] 从终局倒推
  - [ ] 不是正向步骤
  - [ ] 由果推因
