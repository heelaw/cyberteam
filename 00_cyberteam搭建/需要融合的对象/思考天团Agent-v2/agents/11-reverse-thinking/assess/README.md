# 逆向思维专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 逆向思维是否正确（由果推因、终局定义、回溯里程碑） |
| 可执行性 | 30% | 输出是否可操作（终局描绘、逆向路径、第一基石是否明确） |
| 独特性 | 20% | 是否有差异化（与正向思维的区分、逆向路径独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（终极目标、当前状态、时间框架）
- [ ] 有输出格式（终局描绘、逆向路径、北极星原则、第一基石）
- [ ] 有分析框架（四步骤：定义终局→回溯里程碑→挖掘核心驱动力→锚定第一基石）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有终局优先原则

## 核心检验点

### 理论准确性
- [ ] 由果推因正确
- [ ] 终局描绘生动
- [ ] 逆向路径有序

### 框架完整性
- [ ] 里程碑从终局倒推
- [ ] 北极星原则抽象到价值观
- [ ] 第一基石具体可执行

### 输出有效性
- [ ] 终局具体可感知
- [ ] 24小时可启动
- [ ] 里程碑可达

## 相关专家

- 逆向思维与战略规划：长期战略
- 逆向思维与目标管理：目标设定
- 逆向思维与创新思维：颠覆性创新
