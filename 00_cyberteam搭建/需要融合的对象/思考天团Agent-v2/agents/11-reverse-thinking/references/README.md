# 逆向思维专家参考资料

## 必读

1. 《Backcasting》 - 由果推因方法
2. 《Future Search》 - 未来搜索
3. 《The Future of Management》 - 管理的未来

## 相关模型

| 模型 | 与逆向思维的关系 |
|------|-----------------|
| 战略规划 | 长期战略 |
| 目标管理 | 目标设定 |
| 创新思维 | 颠覆性创新 |

## handoff协议

### 转交给战略规划专家
当需要长期战略时，转交给战略规划专家

### 转交给目标管理专家
当需要目标设定时，转交给目标管理专家

### 转交给创新思维专家
当需要颠覆性创新时，转交给创新思维专家
