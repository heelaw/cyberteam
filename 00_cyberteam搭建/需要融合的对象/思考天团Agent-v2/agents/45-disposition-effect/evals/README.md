# 处置效应专家测试用例

## 正面测试

### 测试1: 盈利卖出焦虑
- 输入:
```
【处置效应分析请求】
持仓情况：持有某股票已涨30%
盈利持仓：涨了30%的股票A
亏损持仓：跌了20%的股票B
持有原因：A涨了这么多应该卖了，B等回本再说
```
- 期望输出:
  - 处置效应识别
  - 成本锚定诊断
  - 逆向测试结果
- 验证项:
  - [ ] 能识别"A涨了卖"是处置效应倾向
  - [ ] 能诊断"B等回本"是成本锚定
  - [ ] 有"如果没买会买吗"的检验

### 测试2: 亏损持有
- 输入:
```
【处置效应分析请求】
持仓情况：持有3年亏了50%的股票
盈利持仓：（无）
亏损持仓：亏了50%的股票C
持有原因：已经亏这么多了，卖了就是真的亏了
```
- 期望输出:
  - 处置效应诊断
  - 沉没成本警告
  - 理性决策检验
- 验证项:
  - [ ] 能识别这是典型的处置效应
  - [ ] 能强调成本价是沉没成本
  - [ ] 能给出"如果没买会买吗"的检验

## 负面测试

### 测试1: 回本锚定
- 输入:
```
【处置效应分析请求】
持仓情况：持有某ETF亏了30%
盈利持仓：（无）
亏损持仓：亏了30%的ETF
持有原因：等回本了就卖
```
- 期望行为: 不能以"回本"作为卖出依据
- 验证项:
  - [ ] 必须强调"回本"是心理锚定
  - [ ] 必须要求用未来信息做决策
  - [ ] 必须给出卖出规则

### 测试2: 利润焦虑
- 输入:
```
【处置效应分析请求】
持仓情况：持有某基金赚了20%
盈利持仓：赚了20%的基金D
亏损持仓：（无）
持有原因：涨了这么多万一跌回去呢，先落袋为安
```
- 期望行为: 不能因为赚钱了就急着卖出
- 验证项:
  - [ ] 必须要求评估是否应该持有
  - [ ] 必须用逆向测试检验
  - [ ] 必须设定止盈规则而不是情绪决策
