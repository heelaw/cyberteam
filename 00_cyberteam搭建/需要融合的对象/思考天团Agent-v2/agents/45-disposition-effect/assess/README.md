# 处置效应专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 处置效应理论是否正确（成本锚定、损失厌恶、逆向测试） |
| 可执行性 | 30% | 输出是否可操作（持仓分类、成本锚定诊断、规则设定是否明确） |
| 独特性 | 20% | 是否有差异化（与一般投资偏差的区分、规则执行机制） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（持仓情况、盈利持仓、亏损持仓、持有原因）
- [ ] 有输出格式（持仓概览、处置效应识别、成本锚定诊断、行动计划、规则建议）
- [ ] 有分析框架（四步法：持仓分类→成本锚定诊断→理性决策检验→规则设定）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 成本锚定解释正确
- [ ] 损失厌恶与处置效应的关系正确
- [ ] "回本再卖"谬误解释正确

### 框架完整性
- [ ] 持仓分类有量化指标
- [ ] 成本锚定诊断清单完整
- [ ] 逆向测试方法正确

### 输出有效性
- [ ] 处置效应指数有量化
- [ ] 卖出/持有建议有依据
- [ ] 防处置效应规则可执行

## 相关专家

- 处置效应与损失厌恶：损失厌恶是处置效应的心理根源
- 处置效应与沉没成本：都涉及对过去的非理性依赖
