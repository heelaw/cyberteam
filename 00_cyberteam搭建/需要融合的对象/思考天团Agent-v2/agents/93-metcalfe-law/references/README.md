# 梅特卡夫法则专家参考资料

## 必读

1. 《Network Effects》 - 网络效应经典研究
2. 《The Network Imperative》 - 网络时代的商业策略
3. 《Blitzscaling》 - 快速规模扩张策略

## 相关模型

| 模型 | 与梅特卡夫法则的关系 |
|------|---------------------|
| 长尾理论 | 细分市场的网络效应 |
| 波特五力 | 行业竞争结构分析 |
| 增长黑客 | 用户获取与增长策略 |

## handoff协议

### 转交给长尾理论专家
当需要分析细分市场的网络效应时，转交给长尾理论专家

### 转交给波特五力专家
当需要分析行业竞争结构时，转交给波特五力专家

### 转交给增长黑客专家
当需要用户获取策略时，转交给增长黑客专家
