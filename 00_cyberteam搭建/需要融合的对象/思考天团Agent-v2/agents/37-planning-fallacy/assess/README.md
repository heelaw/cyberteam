# 规划谬误专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 规划谬误理论是否正确（内部视角vs外部视角、三点估算、缓冲机制） |
| 可执行性 | 30% | 输出是否可操作（基准数据、缓冲系数、里程碑是否明确） |
| 独特性 | 20% | 是否有差异化（与一般估算工具的区分、外部校准机制） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（计划/项目、初步估算、类似经验）
- [ ] 有输出格式（内部估算、外部校准、缓冲建议、里程碑）
- [ ] 有分析框架（五步法：内部估算→外部校准→偏差识别→区间估算→缓冲风险金）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 内部视角vs外部视角区分正确
- [ ] 三点估算（乐观/最可能/悲观）方法正确
- [ ] 缓冲系数建议合理（1.5-2.0倍）

### 框架完整性
- [ ] 每个阶段有具体操作项
- [ ] 有同类项目基准数据查询指导
- [ ] 有乐观偏差识别清单

### 输出有效性
- [ ] 最终估算有区间而非点估计
- [ ] 里程碑设置可追踪
- [ ] 风险金分配明确

## 相关专家

- 规划谬误与邓宁-克鲁格：两者都涉及过度自信，但PF专注规划和估算
- 规划谬误与帕金森定律：都涉及时间管理，但PF强调外部校准
