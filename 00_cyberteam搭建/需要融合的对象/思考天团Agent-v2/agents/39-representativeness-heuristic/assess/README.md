# 代表性启发专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 代表性启发理论是否正确（相似性判断、基础比率忽视、合取谬误、赌徒谬误） |
| 可执行性 | 30% | 输出是否可操作（相似性检查、贝叶斯评估、合取检验是否明确） |
| 独特性 | 20% | 是否有差异化（与其他认知偏差工具的区分、贝叶斯方法） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（观察对象、初步判断、判断依据、基础数据）
- [ ] 有输出格式（代表性分析、基础比率忽视诊断、概率校正、认知调整）
- [ ] 有分析框架（四步法：代表性评估→基础比率忽视→合取谬误识别→赌徒谬误识别）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 相似性判断机制描述正确
- [ ] 基础比率忽视问题识别正确
- [ ] Linda问题（合取谬误）解释正确
- [ ] 赌徒谬误（独立性错误）解释正确

### 框架完整性
- [ ] 贝叶斯思维有具体应用
- [ ] 合取谬误有检验方法
- [ ] 独立性判断有指导

### 输出有效性
- [ ] 概率校正有贝叶斯计算
- [ ] 判断清单可操作
- [ ] 认知调整建议具体

## 相关专家

- 代表性启发与可得性启发：两者都是判断启发法，但RH用相似性，AH用记忆获取容易度
- 代表性启发与基础比率：RH常忽视BR，BR是RH的纠正工具
