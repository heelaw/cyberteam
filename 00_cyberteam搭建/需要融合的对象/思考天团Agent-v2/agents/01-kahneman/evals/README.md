# 卡尼曼决策专家测试用例

## 正面测试

### 测试1: 投资决策分析
- 输入:
```
【卡尼曼决策分析请求】
问题描述：股票投资选择
选项A：稳健型，年收益5%
选项B：进取型，可能年收益15%也可能亏损10%
```
- 期望输出:
  - 问题重构
  - 偏差扫描（损失厌恶、过度自信等）
  - 噪声评估
  - 决策建议
- 验证项:
  - [ ] 偏差识别至少4种
  - [ ] 选项对比清晰
  - [ ] 建议明确

### 测试2: 职业选择
- 输入:
```
【卡尼曼决策分析请求】
问题描述：职业发展方向
选项A：大公司稳定但成长慢
选项B：创业公司高风险高成长
```
- 期望输出:
  - 问题重构
  - 偏差扫描
  - 噪声评估
  - 决策建议
- 验证项:
  - [ ] 现状偏见识别
  - [ ] 锚定效应检查
  - [ ] 有量化分析

## 负面测试

### 测试1: 只说因人而异
- 输入:
```
【卡尼曼决策分析请求】
问题描述：要不要转行
选项A：继续当前工作
选项B：转行到新领域
```
- 期望行为: 不能只说"因人而异"，必须给出具体分析
- 验证项:
  - [ ] 有具体偏差分析
  - [ ] 不是模糊回答
  - [ ] 有明确建议倾向

### 测试2: 忽视噪声
- 输入:
```
【卡尼曼决策分析请求】
问题描述：明天股市预测
选项A：上涨
选项B：下跌
```
- 期望行为: 必须评估噪声影响，不能忽视判断的不确定性
- 验证项:
  - [ ] 噪声评估包含
  - [ ] 区分认知偏差和噪声
  - [ ] 不夸大确定性
