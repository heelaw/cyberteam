# 卡尼曼决策专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 卡尼曼理论是否正确（系统1/系统2、认知偏差、噪声） |
| 可执行性 | 30% | 输出是否可操作（偏差扫描、噪声评估、决策建议是否明确） |
| 独特性 | 20% | 是否有差异化（与传统决策的区分、双系统独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（问题描述、选项A、选项B）
- [ ] 有输出格式（偏差风险分析、噪声提示、决策建议）
- [ ] 有分析框架（四步法：问题重构→偏差扫描→噪声评估→建议生成）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（行为经济学、认知心理学、决策理论）

## 核心检验点

### 理论准确性
- [ ] 系统1/系统2区分正确
- [ ] 主要认知偏差覆盖完整
- [ ] 噪声概念有效

### 框架完整性
- [ ] 偏差扫描有清单
- [ ] 噪声评估有方法
- [ ] 决策建议具体

### 输出有效性
- [ ] 选项对比清晰
- [ ] 偏差风险可量化
- [ ] 辅助工具推荐有效

## 相关专家

- 卡尼曼决策与行为经济学：决策行为的系统分析
- 卡尼曼决策与认知心理学：人类判断的局限
- 卡尼曼决策与决策理论：理性与有限理性
