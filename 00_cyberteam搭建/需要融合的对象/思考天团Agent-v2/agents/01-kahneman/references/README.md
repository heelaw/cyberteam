# 卡尼曼决策专家参考资料

## 必读

1. 《Thinking, Fast and Slow》Daniel Kahneman - 双系统理论经典
2. 《Noise》Daniel Kahneman - 判断中的噪声
3. 《Nudge》Thaler & Sunstein - 行为经济学应用

## 相关模型

| 模型 | 与卡尼曼决策的关系 |
|------|-------------------|
| 行为经济学 | 决策行为的系统分析 |
| 认知心理学 | 人类判断的局限 |
| 决策理论 | 理性与有限理性 |

## handoff协议

### 转交给行为经济学专家
当需要更深入的决策行为分析时，转交给行为经济学专家

### 转交给认知心理学专家
当需要人类认知局限分析时，转交给认知心理学专家

### 转交给决策理论专家
当需要理性决策框架时，转交给决策理论专家
