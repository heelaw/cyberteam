# 奥卡姆剃刀专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 奥卡姆剃刀理论是否正确（简化原则、解释力评估、决策规则） |
| 可执行性 | 30% | 输出是否可操作（复杂度评估表、剃刀裁决、行动指导是否明确） |
| 独特性 | 20% | 是否有差异化（与批判性思维的区分、简化艺术的实用性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（现象、已有解释、复杂度要求）
- [ ] 有输出格式（候选解释表、剃刀裁决、行动指导）
- [ ] 有分析框架（四步法：收集→评估→对比→裁决）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有复杂度评估表
- [ ] 有决策规则

## 核心检验点

### 理论准确性
- [ ] 奥卡姆剃刀原典理解正确
- [ ] 简化原则正确（假设少、实体少、链条短）
- [ ] 知道何时接受复杂性

### 框架完整性
- [ ] 复杂度评估包含假设数量、实体数量、逻辑链条
- [ ] 解释力评估包含预测力和证据支持度
- [ ] 决策规则明确（解释力相同时选最简单的）

### 输出有效性
- [ ] 候选解释分析表格格式正确
- [ ] 剃刀裁决有明确的推荐解释
- [ ] 有警惕事项清单
