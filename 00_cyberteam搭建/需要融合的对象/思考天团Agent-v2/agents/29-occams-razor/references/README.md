# 奥卡姆剃刀专家参考资料

## 必读

1. 《Occam's Razor》William of Ockham - 原典思想
2. 《The Rational Optimist》Matt Ridley - 复杂性的演化视角
3. 《Thinking, Fast and Slow》Daniel Kahneman - 认知偏误与简化思维

## 案例库

### 成功案例
- **达尔文的自然选择理论**: 用最简单的"适者生存"解释复杂的生命演化
- **爱因斯坦的相对论**: E=mc²的简洁公式解释了大量现象
- **牛顿的万有引力**: 一个公式解释天上地下所有物体的运动

### 失败案例
- **地心说**: 为解释行星逆行引入复杂的本轮-均轮系统
- **以太假说**: 为解释光速恒定引入不必要的实体

## 相关模型

| 模型 | 与奥卡姆剃刀的关系 |
|------|----------------|
| 批判性思维 | 奥卡姆剃刀是批判性思维的工具之一 |
| Hanlon's Razor | 另一个剃刀原则，与奥卡姆互补 |
| KISS原则 | 工程实践中的简化原则 |

## 延伸阅读

- 剃刀家族：奥卡姆、Hanlon、Hickam
- 简单与复杂的平衡艺术
- 科学理论的选择标准

## handoff协议

### 转交给系统思维专家
