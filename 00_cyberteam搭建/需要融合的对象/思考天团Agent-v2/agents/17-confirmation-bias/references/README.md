# 确认偏误认知专家参考资料

## 必读

1. 《Thinking, Fast and Slow》Daniel Kahneman - 确认偏误的系统性阐述
2. 《The Psychology of Judgment》Scott Plous - 判断与决策中的偏误全面综述
3. 《Advantage》Thomas Bayes - 贝叶斯思维与置信度校准

## 案例库

### 成功案例
- **桥水基金的原则**: 达利欧强调"可信度加权"和"极度透明"来对抗确认偏误
- **科学研究的同行评审**: 通过故意质疑和反驳来减少确认偏误

### 失败案例
- **2008年金融危机**: 华尔街分析师忽视反对证据，坚信房价不会下跌
- **扁鹊医术**: 蔡桓公忽视早期警告信号，证实了自己不想看到的

## 相关模型

## 延伸阅读

- 动机性推理：为什么我们相信想信的
- 群体思维：团队中的确认偏误
- 算法确认偏误：推荐系统如何强化偏见

## handoff协议

### 转交给机会成本专家
当确认后的决策需要权衡时，转交机会成本专家分析选项

### 转交给沉没成本专家
当用户的"坚持"基于不愿承认错误时，转交沉没成本专家分析

### 转交给批判性思维专家
当需要更深入的逻辑谬误分析时，转交批判性思维专家
