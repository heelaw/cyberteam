# 确认偏误认知专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 确认偏误理论是否正确（四种类型、反向搜索、魔鬼代言人） |
| 可执行性 | 30% | 输出是否可操作（证据平衡表、偏误风险评估、置信度校准） |
| 独特性 | 20% | 是否有差异化（质疑方法论、置信度量化框架） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（核心观点、支持/反对证据）
- [ ] 有输出格式（假设检验、证据平衡表、偏误评估、置信度校准）
- [ ] 有分析框架（四步法：观点解剖→反向搜索→魔鬼代言人→校正建议）
- [ ] 有Critical Rules（必须遵守4条、禁止行为5条）
- [ ] 有CLI命令（如有）
- [ ] 有handoff协议

## 核心检验点

### 理论准确性
- [ ] 四种确认偏误类型正确（选择性搜索/解读/记忆/动机性推理）
- [ ] 魔鬼代言人方法论正确
- [ ] 置信度量化框架合理

### 框架完整性
- [ ] 观点解剖包含核心假设识别
- [ ] 反向搜索覆盖四种偏误类型
- [ ] 魔鬼代言人涵盖证据/逻辑/替代/利益质疑

### 输出有效性
- [ ] 证据平衡表格式正确
- [ ] 偏误风险评估有具体维度
- [ ] 置信度校准有明确的新证据要求
