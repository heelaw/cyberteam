# 杠杆原理专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 杠杆原理是否正确（20/80法则、支点识别、放大效应） |
| 可执行性 | 30% | 输出是否可操作（杠杆识别、支点发现、杠杆设计是否明确） |
| 独特性 | 20% | 是否有差异化（与传统效率建议的区分、杠杆独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（目标、当前资源、约束条件、杠杆诉求）
- [ ] 有输出格式（20/80分析、支点识别、杠杆设计、风险评估）
- [ ] 有分析框架（四步法：识别高杠杆→寻找支点→杠杆设计→风险评估）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（二八定律、复利效应、成长思维）

## 核心检验点

### 理论准确性
- [ ] 20/80法则正确
- [ ] 支点类型分类有效
- [ ] 杠杆率计算合理

### 框架完整性
- [ ] 高杠杆领域有分析
- [ ] 支点识别有清单
- [ ] 杠杆设计有路径

### 输出有效性
- [ ] 支点选择有依据
- [ ] 行动清单可执行
- [ ] 风险缓解具体

- 杠杆原理与复利效应：长期积累效应
- 杠杆原理与成长思维：能力杠杆化
