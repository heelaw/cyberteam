# 博弈论分析专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全：博弈识别、玩家分析、策略空间、均衡分析、策略建议 |
| 准确性 | 30% | 理论是否正确：博弈类型判断、均衡求解、策略推荐 |
| 可执行性 | 30% | 输出是否可操作：是否有具体的策略建议、承诺机制、行动路线图 |
| 独特性 | 20% | 是否有差异化：是否体现博弈论的策略互动本质，而非简单利弊分析 |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（8个场景覆盖）
- [ ] 有输入格式（博弈论分析请求模板）
- [ ] 有输出格式（完整策略互动分析报告）
- [ ] 有分析框架（5步框架：博弈识别→玩家分析→策略空间→均衡分析→策略建议）
- [ ] 有Critical Rules（5条必须+6条禁止）
- [ ] 有支付矩阵分析
- [ ] 有均衡类型判断（纳什均衡/占优策略/子博弈完美/贝叶斯）
- [ ] 有行动路线图（短期/中期/长期）

## 评估检查项

### 完整性检查 (20%)

| 检查项 | 要求 |
|--------|------|
| 博弈识别 | 包含类型判断：零和vs非零和/合作vs非合作/一次vs重复 |
| 玩家分析 | 包含利益函数、优先级、理性程度、风险态度 |
| 策略空间 | 包含纯策略/混合策略/优势策略/劣势策略分析 |
| 均衡分析 | 包含纳什均衡点识别和稳定性判断 |
| 策略建议 | 包含承诺/威胁/惩罚/奖励机制建议 |

### 准确性检查 (30%)

| 检查项 | 要求 |
|--------|------|
| 博弈类型判断 | 正确区分零和vs非零和、一次vs重复博弈 |
| 均衡求解 | 均衡点满足"给定对方不变，我不愿改" |
| 玩家理性假设 | 考虑有限理性，而非假设完全理性 |
| 换位思考 | 真正从对方角度分析最优反应 |

### 可执行性检查 (30%)

| 检查项 | 要求 |
|--------|------|
| 支付矩阵 | 有具体的收益值/效用（即使定性也要明确） |
| 策略建议 | 有具体操作步骤，不是泛泛而谈 |
| 承诺机制 | 有让威胁/承诺可置信的具体方法 |
| 行动路线图 | 有短期（单次互动）/中期（建立声誉）/长期（改变结构）的区分 |

### 独特性检查 (20%)

| 检查项 | 要求 |
|--------|------|
| 策略依赖性 | 体现"你的最优选择取决于他人的选择" |
| 换位分析 | 必须分析对方的最优反应 |
| 均衡多重性 | 识别可能存在多个均衡的情况 |
| 时间维度 | 重复博弈策略与一次性博弈要有区分 |

## 常见失分点

1. **单向思维** - 只分析自己的最优，忽视对方反应
2. **假设信息完全** - 忽视信息不对称在博弈中的关键作用
3. **零和泛化** - 将所有博弈都当作零和处理
4. **忽视时间维度** - 重复博弈用了一次性博弈的策略
5. **承诺不可信** - 威胁或许诺缺乏可信度分析
6. **忽视均衡多重性** - 只找到一个均衡就认为问题解决

## 博弈类型判断标准

| 类型 | 判断标准 | 策略要点 |
|------|----------|----------|
| 零和博弈 | 一方所得即另一方所失 | 最大化相对收益 |
| 非零和博弈 | 总收益可以变化 | 寻找双赢可能 |
| 合作博弈 | 可达成有约束力协议 | 建立可信承诺 |
| 非合作博弈 | 无约束力协议 | 依赖自我执行 |
| 一次性博弈 | 无未来影响 | 一次性策略 |
| 重复博弈 | 未来有互动 | 建立声誉 |

## 均衡类型适用场景

| 均衡类型 | 适用场景 |
|----------|----------|
| 纳什均衡 | 一般情况下的策略稳定点 |
| 占优策略均衡 | 存在无论对方如何都最优的策略 |
| 子博弈完美均衡 | 多阶段动态博弈 |
| 贝叶斯均衡 | 不完全信息博弈 |
| 核 | 合作博弈中的可行分配 |
