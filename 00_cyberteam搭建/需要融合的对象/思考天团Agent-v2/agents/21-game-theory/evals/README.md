# 博弈论专家测试用例

## 正面测试

### 测试1: 双寡头竞争
- 输入:
```
【博弈论请求】
场景：双寡头市场竞争
参与者：企业A、企业B
企业A策略：[扩张、维持、收缩]，收益：[30,15,5]
企业B策略：[扩张、维持、收缩]，收益：[25,20,8]
约束条件：双方均无法通信
```
- 期望输出:
  - 收益矩阵完整
  - Nash均衡识别
  - 策略建议
- 验证项:
  - [ ] 矩阵结构正确
  - [ ] 均衡计算正确
  - [ ] 有最优策略

### 测试2: 囚徒困境
- 输入:
```
【博弈论请求】
场景：两嫌疑人审讯博弈
参与者：嫌疑人1、嫌疑人2
策略空间：[坦白、沉默]
收益结构：-5/-10/-1/-3
```
- 期望输出:
  - 收益矩阵
  - 纳什均衡分析
  - 合作可能性
- 验证项:
  - [ ] 矩阵对称性
  - [ ] 均衡为(坦白,坦白)
  - [ ] 社会福利分析

## 负面测试

### 测试1: 信息不完整
- 输入:
```
【博弈论请求】
场景：谈判博弈
参与者：A、B
```
- 期望行为: 指出信息不足，要求补充参与者、策略、收益结构
- 验证项:
  - [ ] 不硬凑矩阵
  - [ ] 明确要求补充信息
  - [ ] 不给出误导性均衡

### 测试2: 无均衡情形
- 输入:
```
【博弈论请求】
场景：石头剪刀布
参与者：玩家1、玩家2
策略：[石头、布、剪刀]
```
- 期望行为: 识别混合策略均衡，不强行给出确定均衡
- 验证项:
  - [ ] 识别循环博弈
  - [ ] 给出混合策略
  - [ ] 验证随机性价值
