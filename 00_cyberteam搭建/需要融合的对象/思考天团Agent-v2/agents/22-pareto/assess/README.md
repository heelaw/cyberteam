# 二八定律专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 二八法则是否正确（80/20识别、原因分析、重点聚焦） |
| 可执行性 | 30% | 输出是否可操作（优先级清单、资源重配、行动方案） |
| 独特性 | 20% | 是否有差异化（与一般优先级分析区分、二八独特视角） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（现象描述、数据范围、时间维度）
- [ ] 有输出格式（关键少数清单、资源配置建议、预期效果）
- [ ] 有分析框架（六步骤：现象→数据→80/20识别→原因→聚焦→重配）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（优先级排序、资源配置、绩效管理）

## 核心检验点

### 理论准确性
- [ ] 正确识别80/20分布
- [ ] 区分关键少数与长尾多数
- [ ] 避免伪二八（数字游戏）
- [ ] 因果关系验证

### 框架完整性
- [ ] 数据支撑或合理推断
- [ ] 关键少数≤20%
- [ ] 资源配置方案具体
- [ ] 效果可量化

### 输出有效性
- [ ] 可立即行动
- [ ] 资源重配可行
- [ ] 效果可衡量

## 相关专家

- 二八定律与目标管理：重点目标
- 二八定律与资源配置：效率优化
- 二八定律与客户分析：关键客户
