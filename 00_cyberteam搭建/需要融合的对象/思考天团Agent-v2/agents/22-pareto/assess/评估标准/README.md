# 二八定律专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全：头部识别、贡献分布、效率分析、长尾评估、策略建议 |
| 准确性 | 30% | 理论是否正确：80/20只是参考、动态边界、边际思维 |
| 可执行性 | 30% | 输出是否可操作：是否有具体的头部识别、杠杆指数计算、行动建议 |
| 独特性 | 20% | 是否有差异化：是否超越机械的"抓大放小"，包含动态策略和长尾评估 |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景覆盖）
- [ ] 有输入格式（二八定律分析请求模板）
- [ ] 有输出格式（完整关键少数分析报告）
- [ ] 有分析框架（5步框架：头部识别→量化贡献→尾巴分析→杠杆点定位→策略建议）
- [ ] 有Critical Rules（4条必须+5条禁止）
- [ ] 有贡献分布表（头部/颈部/尾部）
- [ ] 有杠杆指数计算
- [ ] 有长尾评估（投入/不投入/条件投入）

## 评估检查项

### 完整性检查 (20%)

| 检查项 | 要求 |
|--------|------|
| 头部识别 | 识别贡献80%价值的20%因素 |
| 量化贡献 | 计算各因素的累计贡献占比 |
| 效率分析 | 杠杆指数 = 贡献占比 / 资源占比 |
| 长尾评估 | 评估长尾总量价值、边际覆盖成本、战略价值 |
| 策略建议 | 头部策略 + 均衡策略 + 动态策略 |

### 准确性检查 (30%)

| 检查项 | 要求 |
|--------|------|
| 比例弹性 | 不死守80/20，允许70/30或90/10 |
| 动态边界 | 认识到头部和长尾边界可能随时间变化 |
| 边际思维 | 关注边际收益而非平均收益 |
| 因果区分 | 不将相关性等同于因果性 |

### 可执行性检查 (30%)

| 检查项 | 要求 |
|--------|------|
| 头部明确 | 给出具体的关键少数因素名单 |
| 杠杆指数 | 有具体的杠杆指数计算和解释 |
| 长尾建议 | 有明确的"投入/不投入/条件投入"建议 |
| 行动建议 | 有3-5条具体可执行的行动建议 |

### 独特性检查 (20%)

| 检查项 | 要求 |
|--------|------|
| 长尾价值 | 不盲目抛弃长尾，评估其战略价值 |
| 动态调整 | 什么条件下应该调整头部 |
| 条件策略 | 包含"条件投入"而非简单二分 |
| 边界模糊 | 承认80/20只是经验法则，非精确定律 |

## 常见失分点

1. **机械80/20** - 死守80/20比例，忽视实际情况可能是70/30或90/10
2. **忽视长尾** - 只聚焦头部而忽视长尾的战略价值
3. **静态思维** - 假设头部因素永久有效，忽视市场和技术变化
4. **相关性问题** - 将相关性当作因果性，头部因素可能是结果而非原因
5. **资源机械分配** - 只按比例分配，忽视可行性、风险等因素
6. **缺乏动态视角** - 没有考虑什么条件下应该调整策略

## 贡献分布模型

| 区间 | 数量占比 | 价值贡献 | 策略定位 |
|------|----------|----------|----------|
| 头部 | 20% | 目标80%+ | 全力投入 |
| 颈部 | 20% | 视情况而定 | 评估投入 |
| 尾部 | 60% | 通常较少 | 战略性评估 |

## 杠杆指数解读

| 杠杆指数 | 含义 | 策略建议 |
|----------|------|----------|
| > 1 | 高效投入区 | 加大投入 |
| = 1 | 均衡区 | 维持现状 |
| < 1 | 低效投入区 | 减少投入 |

## 长尾评估框架

| 评估项 | 问题 |
|--------|------|
| 长尾总量价值 | 长尾部分的绝对价值有多大？ |
| 边际覆盖成本 | 覆盖长尾需要额外多少资源？ |
| 战略价值 | 长尾是否有品牌、护城河等战略意义？ |
| 聚集效应 | 长尾是否能转化为头部？ |
| 建议 | 投入 / 不投入 / 条件投入 |
