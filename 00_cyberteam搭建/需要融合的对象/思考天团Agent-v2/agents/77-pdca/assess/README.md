# PDCA循环专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | PDCA循环理论是否正确（Plan→Do→Check→Act四阶段完整、闭环管理） |
| 可执行性 | 30% | 输出是否可操作（目标设定、数据收集、效果评估、标准化是否明确） |
| 独特性 | 20% | 是否有差异化（与KISS复盘的区分、持续改进的独特性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（问题/目标、当前状态、改进意愿）
- [ ] 有输出格式（P/D/C/A四阶段、改进成效表）
- [ ] 有分析框架（四步法：计划→执行→检查→行动）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（协作专家、使用顺序）

## 核心检验点

### 理论准确性
- [ ] PDCA四阶段区分正确
- [ ] 闭环管理原则正确

### 框架完整性
- [ ] 每个阶段有具体操作项
- [ ] 根因分析有方法指导

### 输出有效性
- [ ] 行动计划有明确的负责人和截止日期
- [ ] 效果评估有量化指标
