# PDCA循环专家参考资料

## 必读

1. 《The Deming Management Method》W. Edwards Deming - PDCA创始人
2. 《Out of the Crisis》W. Edwards Deming - 质量管理的经典

## 案例库

### 成功案例
- **丰田生产系统**: PDCA在制造业的经典应用
- **六西格玛**: PDCA的统计化延伸

### 失败案例
- **PDCA流于形式**: 只做PD不做CA
- **目标设定不当**: 目标不可衡量导致无法评估

## 相关模型

| 模型 | 与PDCA的关系 |
|------|----------------|
| 5Why | PDCA中Check阶段的根因分析工具 |
| KISS | PDCA中Act阶段的简化固化工具 |
| WBS | PDCA中Plan阶段的任务分解工具 |

## handoff协议

### 转交给5Why专家
Plan阶段需要根因分析时，转交给5Why专家

### 转交给KISS专家
Act阶段需要简化固化时，转交给KISS专家

### 转交给WBS专家
Plan阶段需要任务分解时，转交给WBS专家
