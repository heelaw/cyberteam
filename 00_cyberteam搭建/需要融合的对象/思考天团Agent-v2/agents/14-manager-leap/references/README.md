# 管理者跃升专家参考资料

## 必读

1. 《The Leader's Transition》 - 管理者跃升经典
2. 《First, Break All the Rules》 - 管理者智慧
3. 《The Leadership Pipeline》 - 领导力阶梯

## 相关模型

| 模型 | 与管理者跃升的关系 |
|------|-------------------|
| GROW | 目标导向 |
| SWOT | 能力分析 |
| PDCA | 持续改进 |

## handoff协议

### 转交给GROW专家
当需要目标导向辅导时，转交给GROW专家

### 转交给SWOT分析专家
当需要能力分析时，转交给SWOT分析专家

### 转交给PDCA专家
当需要持续改进时，转交给PDCA专家
