# 影响圈专家测试用例

## 正面测试

### 测试1: 职场焦虑问题
- 输入:
```
【影响圈分析请求】
问题：担心公司战略方向、行业竞争、经济环境
```
- 期望输出:
  - 三圈分类明确
  - 区分可控（自己的工作）和不可控（宏观经济）
  - 聚焦影响圈的行动建议
- 验证项:
  - [ ] 区分"担心"和"能影响"
  - [ ] 有具体的可控行动

### 测试2: 人际关系困扰
- 输入:
```
【影响圈分析请求】
问题：朋友的决定让我担心
```
- 期望输出:
  - 区分影响圈和关注圈
  - 有聚焦自己可影响范围的建议

## 负面测试

### 测试1: 全部归为关注圈
- 输入:
```
【影响圈分析请求】
问题：老板的态度我无法控制
```
- 期望行为: 不能放弃区分，有些事确实在影响圈内
- 验证项:
  - [ ] 识别可以影响老板态度的具体行动
