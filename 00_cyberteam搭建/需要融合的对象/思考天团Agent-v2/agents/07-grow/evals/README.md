# GROW模型专家测试用例

## 正面测试

### 测试1: 职业发展
- 输入:
```
【GROW模型教练对话】
对话主题：未来3年职业发展
当前状态：工作3年，做技术
期望成果：成为技术总监
```
- 期望输出:
  - G目标设定（SMART目标）
  - R现状分析
  - O方案选择
  - W行动承诺
- 验证项:
  - [ ] 目标SMART
  - [ ] 现状客观
  - [ ] 方案充分
  - [ ] 承诺具体

### 测试2: 绩效改进
- 输入:
```
【GROW模型教练对话】
对话主题：提高销售业绩
当前状态：季度目标完成60%
期望成果：完成100%
```
- 期望输出:
  - 四阶段分析
  - 障碍识别
  - 行动方案
  - 承诺书
- 验证项:
  - [ ] 障碍具体
  - [ ] 方案有选项
  - [ ] 承诺可衡量

## 负面测试

### 测试1: 给建议
- 输入:
```
【GROW模型教练对话】
对话主题：团队管理问题
```
- 期望行为: 不能说"你应该..."，要用提问引导
- 验证项:
  - [ ] 用提问而非建议
  - [ ] 答案在用户心中
  - [ ] 教练是镜子不是顾问

### 测试2: 跳过R阶段
- 输入:
```
【GROW模型教练对话】
对话主题：想换个工作
```
- 期望行为: 不能跳过R直接到O，要先分析现状
- 验证项:
  - [ ] R阶段有实质内容
  - [ ] 现状分析完整
  - [ ] 不能直接给方案
