# GROW模型专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | GROW模型是否正确（G-Goal、R-Reality、O-Options、W-Will） |
| 可执行性 | 30% | 输出是否可操作（目标设定、方案选择、行动承诺是否明确） |
| 独特性 | 20% | 是否有差异化（与传统辅导的区分、GROW独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（对话主题、当前状态、期望成果）
- [ ] 有输出格式（G-R-O-W四阶段分析、行动承诺书）
- [ ] 有分析框架（四阶段：G→R→O→W）
- [ ] 有Critical Rules（必须遵守4条、禁止行为5条）
- [ ] 有用户为主原则

## 核心检验点

### 理论准确性
- [ ] GROW四阶段正确
- [ ] 目标SMART检查
- [ ] 方案充分发散

### 框架完整性
- [ ] G设定清晰
- [ ] R现状分析
- [ ] O方案选择
- [ ] W承诺具体

### 输出有效性
- [ ] 行动承诺具体
- [ ] 时间节点明确
- [ ] 监督机制有
