# GROW模型专家参考资料

1. 《Coaching for Performance》John Whitmore - GROW模型原版
2. 《The GROW Model》 - GROW教练技术
3. 《Internal Coaching》 - 内部教练技术

| 模型 | 与GROW的关系 |
|------|-------------|
| 吉德林法则 | 目标澄清 |
