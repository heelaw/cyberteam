# 断点理论专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 断点理论是否正确（断点类型、前兆检测、预防策略） |
| 可执行性 | 30% | 输出是否可操作（断点识别、距离评估、预防策略是否明确） |
| 独特性 | 20% | 是否有差异化（与传统风险管理的区分、断点预警独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（系统/关系、当前状态、历史数据、关注维度）
- [ ] 有输出格式（断点识别、预警信号、距离分析、预防策略）
- [ ] 有分析框架（四步法：系统扫描→断点识别→距离分析→预防策略）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（系统思维、风险管理、PDCA）

## 核心检验点

### 理论准确性
- [ ] 断点类型分类正确
- [ ] 前兆信号有效
- [ ] 非线性特性正确

### 框架完整性
- [ ] 系统扫描有清单
- [ ] 断点识别有类型
- [ ] 预防策略有方法

### 输出有效性
- [ ] 距离评估可量化
- [ ] 预警信号具体
- [ ] 干预时机可执行

## 相关专家

- 断点理论与系统思维：整体系统分析
- 断点理论与风险管理：风险识别与应对
- 断点理论与PDCA：持续监控改进
