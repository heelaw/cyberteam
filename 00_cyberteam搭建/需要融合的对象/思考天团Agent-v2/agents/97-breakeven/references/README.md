# 断点理论专家参考资料

## 必读

1. 《The Breakpoint》 - 临界点与系统崩溃研究
2. 《Thinking in Systems》Donella Meadows - 系统思维经典
3. 《The Black Swan》Nassim Taleb - 极端事件与断点

## 相关模型

| 模型 | 与断点理论的关系 |
|------|-----------------|
| 系统思维 | 整体系统分析 |
| 风险管理 | 风险识别与应对 |
| PDCA | 持续监控改进 |
