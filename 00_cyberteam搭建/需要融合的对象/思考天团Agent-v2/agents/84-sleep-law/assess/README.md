# 睡眠定律专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 睡眠定律理论是否正确（睡眠惯性、睡眠整合、决策时机、疲劳保护） |
| 可执行性 | 30% | 输出是否可操作（决策紧迫性评估、睡眠准备优化、决策时机选择是否明确） |
| 独特性 | 20% | 是否有差异化（与一般休息建议的区分、睡眠认知科学） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（决策问题、决策时间、决策重要性、当前状态）
- [ ] 有输出格式（紧迫性评估、睡眠准备、决策时机推荐、疲劳保护）
- [ ] 有分析框架（四步法：决策紧迫性评估→睡眠准备优化→决策时机选择→疲劳决策保护）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（机会成本、二阶思维、批判性思维）

## 核心检验点

### 理论准确性
- [ ] 睡眠惯性概念正确
- [ ] 睡眠整合概念正确
- [ ] 最佳决策时机正确

### 框架完整性
- [ ] 时间vs质量权衡有方法
- [ ] 睡眠准备有具体步骤
- [ ] 疲劳信号识别清单完整

### 输出有效性
- [ ] 决策时机推荐具体
- [ ] 睡眠准备可执行
- [ ] 疲劳保护措施有效
