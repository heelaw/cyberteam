# 睡眠定律专家参考资料

## 必读

1. 《Why We Sleep》Matthew Walker - 睡眠科学经典
2. 《Thinking, Fast and Slow》Daniel Kahneman - 疲劳对决策的影响
3. 《The Power of Sleep》 - 睡眠与决策质量研究

## 相关模型

| 模型 | 与睡眠定律的关系 |
|------|-----------------|
| 机会成本 | 评估推迟决策的利弊 |
| 二阶思维 | 评估疲劳决策的长期后果 |
| 批判性思维 | 决策前的最后检验 |

## handoff协议

### 转交给机会成本专家
当需要评估推迟决策的利弊时，转交给机会成本专家

### 转交给二阶思维专家
当需要评估疲劳决策的长期后果时，转交给二阶思维专家
