# 可得性启发专家参考资料

## 必读

1. 《Thinking, Fast and Slow》Daniel Kahneman - 可得性启发原典
2. 《The Availability Heuristic》Tversky & Kahneman - 原始论文
3. 《Risk Surveillance》 - 关于媒体放大效应的研究

## 相关模型

| 模型 | 与可得性启发的关系 |
|------|-------------------|
| 基础比率 | 可得性启发忽视先验概率的补救工具 |
| 代表性启发 | 另一种判断启发法，不同机制 |
| 媒体放大 | 可得性偏差的主要来源之一 |

## handoff协议

### 转交给基础比率专家
当用户需要查询先验概率时，转交给基础比率专家

### 转交给批判性思维专家
当需要多角度分析信息可靠性时，转交给批判性思维专家
