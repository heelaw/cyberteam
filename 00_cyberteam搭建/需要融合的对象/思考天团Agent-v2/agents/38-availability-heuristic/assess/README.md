# 可得性启发专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 可得性启发理论是否正确（媒体放大、亲身经历、鲜活性、近期效应） |
| 可执行性 | 30% | 输出是否可操作（可得性诊断、校正系数、基础比率查询是否明确） |
| 独特性 | 20% | 是否有差异化（与其他认知偏差工具的区分、鲜活性校正方法） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（事件、可能性判断、判断依据、相关信息）
- [ ] 有输出格式（可得性诊断、基础比率、概率校正、认知调整）
- [ ] 有分析框架（四步法：可得性来源诊断→基础比率查询→鲜活性校正→替代世界分析）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 可得性偏差来源识别正确（媒体报道、亲身经历、近期效应、情感强度）
- [ ] 鲜活性校正系数合理
- [ ] 基础比率概念应用正确

### 框架完整性
- [ ] 每个偏差来源有对应检查项
- [ ] 有量化校正方法
- [ ] 有反事实分析指导

### 输出有效性
- [ ] 概率校正有具体计算
- [ ] 校正后有概率区间
- [ ] 认知调整建议可操作

## 相关专家

- 可得性启发与基础比率：两者都涉及概率判断，但AH强调记忆偏差，BR强调先验概率
- 可得性启发与代表性启发：两者都是判断启发法，但机制不同
