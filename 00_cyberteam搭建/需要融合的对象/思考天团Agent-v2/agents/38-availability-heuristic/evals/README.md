# 可得性启发专家测试用例

## 正面测试

### 测试1: 航空恐惧症
- 输入:
```
【可得性启发分析请求】
事件：坐飞机失事
可能性判断：飞机很危险，很容易出事
判断依据：新闻里经常看到飞机事故
相关信息：其实没坐过几次飞机
```
- 期望输出:
  - 可得性污染来源诊断
  - 真实统计数据对比
  - 鲜活性校正后概率
- 验证项:
  - [ ] 能识别媒体报道是主要偏差来源
  - [ ] 有飞机事故真实概率数据
  - [ ] 有校正系数计算

### 测试2: 近期创伤后高估风险
- 输入:
```
【可得性启发分析请求】
事件：再次创业失败
可能性判断：很可能失败
判断依据：最近认识的人都创业失败了
相关信息：我的项目有一些不同
```
- 期望输出:
  - 近期效应诊断
  - 基础比率对比
  - 独立事件分析
- 验证项:
  - [ ] 能识别"最近认识的人"是近期偏差
  - [ ] 有创业成功率基础比率
  - [ ] 能区分相关性vs因果性

## 负面测试

### 测试1: 以偏概全
- 输入:
```
【可得性启发分析请求】
事件：这个投资产品是骗局
可能性判断：肯定是骗局
判断依据：我朋友去年被这种产品骗过
相关信息：没有其他证据
```
- 期望行为: 不能用个人经历代替统计数据
- 验证项:
  - [ ] 必须查询基础比率
  - [ ] 不能仅凭一个案例下结论

### 测试2: 忽视鲜活性
- 输入:
```
【可得性启发分析请求】
事件：游泳很危险
可能性判断：每年很多人溺水
判断依据：电视上经常看到溺水新闻
相关信息：那是极端案例
```
- 期望行为: 不能忽视情感的鲜活性效应
- 验证项:
  - [ ] 必须进行鲜活性校正
  - [ ] 必须区分极端案例和普通情况
