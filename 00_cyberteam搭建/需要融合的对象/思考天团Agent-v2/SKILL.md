# 思考天团 Skill

## 基本信息

| 属性 | 内容 |
|------|------|
| **Skill名称** | 思考天团 |
| **定位** | 多思维模型协作的入口Skill |
| **触发词** | 思考天团、分析问题、多角度思考、全面分析 |
| **版本** | v1.0 |

---

## 核心能力

### 1. 问题理解与路由

**输入**: 用户的自然语言问题

**处理**:
- 解析问题关键词
- 判断问题类型
- 识别隐含约束
- 选择最匹配的专家Agent组合

**输出**: 路由决策 + 专家选择理由

---

### 2. 并行独立分析

**触发条件**: 问题需要多角度分析

**处理**:
- 向选中的专家Agent并行发送分析请求
- 收集各专家的独立分析结果
- 整理成结构化的观点汇编

**输出**: 多专家观点并列的中间报告

---

### 3. 交叉辩论

**触发条件**: 重大决策需要权衡

**处理**:
- 确定辩论议题
- 安排专家依次发言
- 组织质疑和辩护环节
- 生成综合裁决

**输出**: 辩论纪要 + 裁决结论

---

### 4. 链式分析

**触发条件**: 问题解决型任务

**处理**:
- 确定分析链顺序
- 依次调用专家Agent
- 每步输出作为下一步输入
- 最终整合全链输出

**输出**: 完整的问题解决路径

---

### 5. 结果整合

**处理**:
- 提炼各专家核心洞察（≤3点/专家）
- 识别共识与分歧
- 生成优先级建议
- 输出可执行的操作指南

**输出**: 结构化的最终分析报告

---

## 工作流程

```
用户输入 → 问题理解 → 意图分类 → 路由决策
                                        ↓
                              ┌─────────┼─────────┐
                              ↓         ↓         ↓
                          并行模式   辩论模式   链式模式
                              ↓         ↓         ↓
                              └─────────┼─────────┘
                                        ↓
                                  结果整合
                                        ↓
                                  最终报告
```

## 输入格式

### 用户触发方式

```
/思考天团

[你的问题描述]

背景：[可选的背景信息]
期望角度：[可选的特定思考角度]
```

### 快捷触发词

| 快捷词 | 触发的专家组合 |
|--------|---------------|
| 分析一下这个问题 | 自动选择最相关的3个专家 |
| 多角度思考 | 所有相关维度专家并行 |
| 辩论 | 交叉辩论模式 |
| 系统分析 | 完整14专家轮询 |
