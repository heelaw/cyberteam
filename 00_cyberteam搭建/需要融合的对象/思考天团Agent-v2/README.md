> 让AI拥有19位顶级思维导师，为你提供多角度、深层次的分析建议

**版本**: v4.0 | **状态**: P0扩展完成（19专家）| **质量评级**: A级（NEXUS 88/110）

## 一、项目概述

### 1.1 是什么

**思考天团**是一个多思维模型协作Agent系统。它不是单一的AI助手，而是由**14位思维导师**组成的"智囊团"，通过不同的思维框架和视角，为复杂问题提供多维度的分析。

### 1.2 解决什么问题

| 传统方式 | 思考天团 |
|---------|---------|
| 单一视角分析 | 多达14种思维视角并行 |
| 浅层建议 | 深度洞察（≤3点/专家） |
| 主观判断 | 结构化辩论 + 证据支撑 |
| 模糊结论 | 含指标/时间线/风险的可执行方案 |

### 1.3 核心优势

```
┌─────────────────────────────────────────────────────────────┐
│                    14位思维导师智囊团                         │
│                                                              │
│   卡尼曼 ──→ 行为经济学视角      识别认知偏差              │
│   第一性原理 ──→ 本质还原视角      从零构建方案              │
│   六顶思考帽 ──→ 全面分析视角      6维度无死角审视          │
│   SWOT ──→ 战略竞争视角      优势/劣势/机会/威胁           │
│   ...                                                      │
│   (共14位专家，覆盖决策、创新、诊断、规划全场景)            │
│                                                              │
│   三种协作模式：并行分析 / 交叉辩论 / 链式推理               │
│   ≤3点整合策略：避免堆砌，提炼核心                          │
└─────────────────────────────────────────────────────────────┘
```

## 三、三种协作模式

### 3.1 并行独立分析（最常用）

**适用**: 需要多角度快速获取洞察

```
用户问题
    ↓
意图识别（选择N个专家）
    ↓
┌─────────────────────────────────────┐
│  专家A ──→ 专家B ──→ 专家C ──→ ...  │  （并行分析）
└─────────────────────────────────────┘
    ↓
结果整合（≤3点提炼 + 共识/分歧）
    ↓
最终报告
```

**时间**: 3-8分钟（取决于专家数量）

---

### 3.2 交叉辩论模式

**适用**: 重大决策需要权衡利弊

```
辩题 + 方案选项
    ↓
正方（A专家）vs 反方（B专家）
    ↓
正方论证 ←──质疑──→ 反方论证
    ↓
第三方补充（C专家视角）
    ↓
主持人综合裁决
    ↓
最终结论
```

**规则**:
- 每方发言不超过3轮
- 质疑必须基于事实和逻辑
- 主持人保持中立，综合裁决

**时间**: 8-15分钟

---

### 3.3 链式分析模式

**适用**: 问题解决型任务，需要多步骤推导

| 预设链 | 适用问题 | 分析顺序 |
|--------|---------|---------|
| **链A** | 问题诊断 | Goldlin → 5Why → 第一性原理 |
| **链B** | 目标实现 | GROW → WBS → KISS |
| **链C** | 战略规划 | SWOT → 逆向思维 → 五维度 |
| **链D** | 决策分析 | 卡尼曼 → 六顶思考帽 → AI-Board |
| **链E** | 创新突破 | 第一性原理 → 逆向思维 → 六顶思考帽 |

```
定义终局（逆向思维）
        ↓
分解问题（5Why/Goldlin）
        ↓
重构方案（第一性原理）
        ↓
验证方案（多视角）
        ↓
行动计划（WBS）
        ↓
整合输出
```

**时间**: 5-12分钟（取决于链长度）

| `帮我规划` | 链B（目标链） |

### 5.3 示例对话

**用户**:
```
/思考天团

公司用户增长停滞，注册转化率从15%下降到8%，
尝试了多种促销方式效果不佳，如何重新激活增长？
```

**思考天团**:
```
═══════════════════════════════════════════
       【思考天团】分析报告
═══════════════════════════════════════════

📋 问题理解
───────────────────────────────────
某在线教育平台增长停滞，核心问题是"促销免疫"，
实际瓶颈可能在价值感知层面...

🧠 专家观点（3位并行）
───────────────────────────────────
【卡尼曼】识别认知偏差
- 锚定效应：团队以15%为基准，但8%才是现实
- 促销疲劳：损失框架已失效
- 核心洞察：重构价值主张，从"损失"转为"获得"

【第一性原理】本质还原
- 质疑假设：促销不创造需求，只是转移时间窗口
- 核心洞察：增长本质是用户价值实现，不是注册动作

【六顶思考帽】全面审视
- 白帽：转化率-47%，促销全面失效
- 黑帽：促销加速品牌贬值
- 黄帽：8%仍高于行业均值5%，基本盘未失
- 核心洞察：增长引擎需从"价格刺激"切换为"价值感知"

⚖️ 整合结论
───────────────────────────────────
【共识】
✓ 促销已全面失效，继续促销加速贬值
✓ 核心问题是价值感知断裂
✓ 需要先诊断再行动

【分歧】
△ 诊断后优先做什么：重构框架 vs 建立信任锚点 vs 多路径并行

【决策树】
→ 若断点在注册前：优先建立信任锚点
→ 若断点在注册后：优先重构价值主张

✅ 行动建议
───────────────────────────────────
1. 立即暂停促销（2周），验证影响可控
2. 启动用户深度访谈（20+人），定位价值感知断点
3. 重构产品价值主张，切换增长引擎

📊 成功指标
───────────────────────────────────
- 转化率目标：8% → 12%（6个月）
- 完课率目标：12% → 20%
- NPS目标：≥40

⚠️ 风险信号
───────────────────────────────────
- 红色警戒：转化率48小时下跌超15%
- 撤退条件：连续3天下跌超10%

📝 下一步（72小时内）
───────────────────────────────────
1. 停止所有付费促销活动
2. 建立监控仪表盘
3. 启动用户访谈招募
```

## 八、使用指南

### 8.1 最佳实践

1. **问题描述要具体**
   - ✅ "用户增长停滞3个月，转化率从15%降到8%"
   - ❌ "增长有问题"

2. **提供背景信息**
   - 行业、竞品、团队资源等

3. **明确期望角度**（可选）
   - "我特别关注用户行为角度"

### 8.2 局限性与注意事项

| 局限 | 说明 | 缓解措施 |
|------|------|---------|
| Token消耗 | 14专家并行可能消耗较高 | 建议3-5专家并行 |
| 实时性 | 知识有截止日期 | 重要决策需补充最新信息 |
| 责任边界 | AI建议仅供参考 | 重大决策需人工审核 |
