## 五、检查清单使用说明

### 使用步骤

1. **执行前**：完成"执行前检查"
2. **执行中**：每个关键步骤后完成"执行中检查"
3. **执行后**：完成后完成"执行后检查"
4. **复盘**：记录问题到"问题发现记录表"

### 检查原则

- **逐项检查**：每一项都要检查，不能跳过
- **实事求是**：不符合就是不符合，不要凑合
- **及时记录**：发现问题立即记录
- **闭环处理**：问题必须跟进解决
