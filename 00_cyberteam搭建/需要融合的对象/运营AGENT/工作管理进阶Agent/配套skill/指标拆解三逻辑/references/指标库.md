# 指标库

## 一、通用运营指标

### 用户指标

| 指标名称 | 定义 | 计算公式 | 业务含义 |
|----------|------|----------|----------|
| DAU | 日活跃用户数 | 当日活跃的用户数 | 产品日活跃度 |
| WAU | 周活跃用户数 | 当周活跃的用户数 | 产品周活跃度 |
| MAU | 月活跃用户数 | 当月活跃的用户数 | 产品月活跃度 |
| 新增用户 | 每日新增注册用户 | 成功注册的用户数 | 用户获取能力 |
| 留存率 | 用户持续使用比例 | 期末活跃/期初活跃 | 用户粘性 |
| 流失率 | 用户流失比例 | 流失用户/总用户 | 用户流失情况 |
| 活跃率 | 用户活跃比例 | 活跃用户/总用户 | 整体活跃水平 |

### 转化指标

| 指标名称 | 定义 | 计算公式 | 业务含义 |
|----------|------|----------|----------|
| 曝光量 | 内容展示次数 | 内容被展示的次数 | 触达能力 |
| 点击量 | 内容点击次数 | 用户点击的次数 | 吸引程度 |
| 点击率 | 点击/曝光比例 | 点击数/曝光数 | 素材效果 |
| 转化率 | 目标行为比例 | 转化数/点击数 | 转化效率 |
| 成交率 | 成交/点击比例 | 成交数/点击数 | 购买转化 |

### 收入指标

| 指标名称 | 定义 | 计算公式 | 业务含义 |
|----------|------|----------|----------|
| GMV | 成交总额 | 拍下金额总和 | 交易规模 |
| 营收 | 实际收入 | 确认收入金额 | 真实收入 |
| ARPU | 用户平均收入 | 收入/用户数 | 单用户价值 |
| LTV | 生命周期价值 | 用户全周期贡献收入 | 用户价值 |
| CAC | 获取成本 | 营销费用/新增用户 | 获客效率 |
| ROI | 投资回报率 | 收入/投入 | 投入产出比 |

## 五、指标选择决策树

```
目标是什么？
│
├─ 提升收入
│   ├─ 电商 → 销售额=流量×转化率×客单价
│   ├─ SaaS → MRR=用户数×付费率×ARPU
│   └─ 付费社群 → 收入=人数×付费率×ARPU
│
├─ 提升活跃
│   ├─ 社群 → 活跃率=发言人数/总人数
│   ├─ 产品 → DAU/MAU
│   └─ 内容 → 互动率=互动数/阅读数
│
├─ 提升留存
│   ├─ 次日留存 → 次日活跃/新增
│   ├─ 7日留存 → 7日活跃/新增
│   └─ 30日留存 → 30日活跃/新增
│
└─ 提升获客
    ├─ 渠道获客 → 各渠道新用户数
    ├─ 裂变获客 → K因子
    └─ 付费获客 → CAC
```

## 六、指标命名规范

### 命名规则

1. **动宾结构**：转化率、留存率、活跃度
2. **明确分子分母**：不要用模糊的"效果""规模"
3. **统一时间维度**：DAU、WAU、MAU
4. **注明计算方式**：XX率=XX/XX

### 常用修饰词

| 修饰词 | 含义 | 示例 |
|--------|------|------|
| 新增 | 新增用户 | 新增用户数 |
| 活跃 | 活跃用户 | DAU/WAU/MAU |
| 留存 | 留存用户 | 次日/7日/30日留存 |
| 付费 | 付费用户 | 付费转化率 |
| 核心 | 核心用户 | 核心功能使用率 |
