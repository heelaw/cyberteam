# 目标设定标准

## 一、SMART原则

所有目标必须符合SMART原则：

| 原则 | 要求 | 示例 |
|------|------|------|
| S-Specific | 具体明确 | "提升转化率" → "提升首页到详情页转化率" |
| M-Measurable | 可衡量 | 有明确的数值目标 |
| A-Achievable | 可达成 | 基于现有数据有提升空间 |
| R-Relevant | 相关性 | 与业务目标相关 |
| T-Time-bound | 有时间限制 | 明确截止时间 |

## 二、目标值设定方法

### 1. 基于历史数据

| 方法 | 公式 | 适用场景 |
|------|------|----------|
| 环比增长 | 目标值 = 当前值 × (1 + 增长率) | 稳定增长业务 |
| 同比增长 | 目标值 = 去年同值 × (1 + 增长率) | 季节性业务 |
| 移动平均 | 目标值 = 近N期平均值 | 波动业务 |

### 2. 基于行业benchmark

```
目标值 = 行业均值 × (1 + 调整系数)

调整系数根据：
- 团队能力
- 资源投入
- 竞争态势
```

### 3. 基于资源投入

```
目标值 = 投入资源 ÷ 单位成本

示例：
- 预算10万，单用户获取成本50元
- 目标用户数 = 10万 ÷ 50 = 2000人
```

## 三、目标层级设定

### 1. 北极星指标（最终目标）

| 维度 | 要求 |
|------|------|
| 定义 | 反映业务核心价值的唯一指标 |
| 数量 | 1个，最多不超过3个 |
| 时间 | 季度/年度目标 |
| 负责人 | 业务负责人/COO |

### 2. 过程指标（阶段性目标）

| 维度 | 要求 |
|------|------|
| 定义 | 达成北极星指标的过程指标 |
| 数量 | 3-7个 |
| 时间 | 月度目标 |
| 负责人 | 部门负责人 |

### 3. 执行指标（操作目标）

| 维度 | 要求 |
|------|------|
| 定义 | 执行层面的具体动作指标 |
| 数量 | 10-20个 |
| 时间 | 周/日目标 |
| 负责人 | 执行人员 |

## 四、预警值设定

### 1. 预警值计算方法

| 方法 | 公式 | 说明 |
|------|------|------|
| 百分比法 | 预警值 = 目标值 × 80% | 保守设定 |
| 差距法 | 预警值 = 目标值 - 容差值 | 允许一定差距 |
| 趋势法 | 预警值 = 当前值 + 预期增量 × 50% | 基于增长预期 |

### 2. 预警等级

| 等级 | 触发条件 | 处理方式 |
|------|----------|----------|
| 绿色 | >=目标值90% | 正常执行 |
| 黄色 | 目标值70-90% | 关注，必要时调整 |
| 橙色 | 目标值50-70% | 预警，分析原因 |
| 红色 | <目标值50% | 紧急干预 |

## 七、常见问题

### Q1: 目标设定过高怎么办？

1. 拆解为阶段性目标
2. 识别关键瓶颈
3. 调整资源配置
4. 与上级沟通预期

### Q2: 目标达成但业务没起色？

1. 检查指标定义是否正确
2. 检查数据来源是否准确
3. 检查是否关注了正确的指标
4. 可能需要重新定义北极星指标

### Q3: 目标频繁变化？

1. 明确业务节奏和周期
2. 建立目标稳定性机制（季度目标不变，月度可调）
3. 设定清晰的触发条件
