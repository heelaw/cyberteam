# 群规执行SOP

## 执行SOP总览

```
┌─────────────────────────────────────────────────────────┐
│                   群规执行SOP流程                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Step 1: 违规发现                                       │
│  ↓                                                      │
│  Step 2: 违规判定（轻度/重度/严重）                        │
│  ↓                                                      │
│  Step 3: 选择处理方式                                    │
│  ↓                                                      │
│  ┌─────────────────────────────────────────┐            │
│  │ 轻度 → 私下提醒 → 记录存档               │            │
│  │ 重度 → 正式警告 → 记录存档               │            │
│  │ 严重 → 直接移出 → 记录存档 → 群内说明     │            │
│  └─────────────────────────────────────────┘            │
│  ↓                                                      │
│  Step 4: 跟进观察                                        │
│  ↓                                                      │
│  Step 5: 定期复盘                                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## Step 1: 违规发现

### 发现渠道

| 渠道 | 说明 |
|------|------|
| 主动巡查 | 管理员定期查看群消息 |
| 用户举报 | 用户主动举报违规行为 |
| 系统提示 | 微信风控提示 |

### 发现后动作

```
1. 截图保存违规证据
2. 确认违规内容和时间
3. 初步判断违规类型
```

## Step 2: 违规判定

### 违规分级标准

| 级别 | 行为类型 | 判定依据 |
|------|---------|---------|
| 轻度 | 无关链接、偶尔刷屏 | 首次、非恶意、无影响 |
| 重度 | 广告推广、私加好友、重复违规 | 有目的性、重复、骚扰 |
| 严重 | 诈骗、网暴、泄露隐私 | 违法、伤害性、破坏性 |

### 判定检查清单

- [ ] 违规行为的具体内容是什么？
- [ ] 这是第几次违规？（查记录）
- [ ] 违规的意图和影响？
- [ ] 应该判定为哪个级别？

## Step 3: 执行处理

### 轻度违规处理流程

```
处理方式：私下提醒
话术：[用户昵称]，温馨提醒[具体规则]，谢谢配合～

处理后：
1. 确认用户是否配合
2. 如配合，本次结束
3. 如不配合，提升为重度处理
4. 记录存档
```

### 重度违规处理流程

```
处理方式：正式警告
话术：@用户 [违规行为]违反了群规[第X条]
这是第[X]次违规，下次将直接移出社群
请配合删除相关内容

处理后：
1. 确认用户是否配合
2. 如配合，本次结束但记录在案
3. 如不配合，直接移出
4. 记录存档
```

### 严重违规处理流程

```
处理方式：直接移出
话术：无需多言，直接移出

移出后（如需要）：
[用户昵称]，抱歉因[原因]已被移出社群
如有疑问可联系群主

群内说明：
@所有人 [用户]因违反群规已被移出
请大家共同维护社群秩序～

记录存档
```

## Step 4: 跟进观察

### 跟进时间线

| 时间点 | 观察内容 |
|--------|---------|
| 处理后1小时 | 群内反应，是否有人议论 |
| 处理后24小时 | 是否有模仿违规者 |
| 处理后7天 | 违规率是否下降 |

## Step 5: 定期复盘

### 复盘周期

| 周期 | 复盘内容 |
|------|---------|
| 每周 | 本周违规统计、处理是否得当 |
| 每月 | 违规趋势、群规是否需要调整 |
| 每季度 | 整体执行效果、策略优化 |

### 复盘指标

```
指标1: 违规处理次数
指标2: 重复违规率
指标3: 用户反馈满意度
指标4: 社群活跃度变化
```
