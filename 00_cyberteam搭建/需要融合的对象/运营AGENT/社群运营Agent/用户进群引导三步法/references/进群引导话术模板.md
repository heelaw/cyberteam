# 进群引导话术模板

## 模板一：标准付费社群引导

```
═══════════════════════════════════════════
        【付费社群】进群引导标准流程
═══════════════════════════════════════════

【Step 1：告知定位】

[用户昵称]，感谢你的信任与支持！🎉

在拉你入群前，先给你介绍一下我们社群：

🌟 社群定位：
帮助[目标用户]实现[核心价值]

📚 社群内容：
- 每周[具体内容，如：一节直播课]
- 每月[具体内容，如：一次实战案例分析]
- 日常[具体内容，如：随时答疑]

👥 成员构成：
目前已有[人数]位[目标用户]加入

⏰ 交流方式：
[具体方式，如：微信群内文字/语音交流]

【Step 2：说明规则】

在入群前，先说一下我们的群规：

✅ 欢迎行为：
- 鼓励提问，有问必答
- 鼓励分享你的经验和案例
- 鼓励参与每周[具体活动]

❌ 禁止行为：
- 禁止发广告/拉人/私加好友
- 禁止发无关链接/二维码
- 禁止讨论[敏感话题]

【Step 3：获得确认】

以上就是我们社群的基本情况~

请确认以下两点：
1️⃣ 你是否认可我们的社群定位？
2️⃣ 你是否愿意遵守我们的群规？

确认无误后，回复"确认入群"，我即刻拉你！

【Step 4：入群后欢迎】

🎉 欢迎[用户昵称]加入【XXX社群】！

我是群主[群主昵称]，负责社群的日常运营~

给你介绍一下：
- 我们的核心成员[简介]
- 社群的日常安排是[简介]
- 建议你先看看群公告，了解[具体内容]

有任何问题随时@我，群里见~

═══════════════════════════════════════════
```

## 模板二：免费学习型社群引导

```
═══════════════════════════════════════════
        【免费学习社群】进群引导流程
═══════════════════════════════════════════

【Step 1：告知定位】

[用户昵称]，欢迎加入！🙌

先了解一下我们社群：

🌟 我们是：
一个帮助[目标用户]实现[价值]的学习型社群

📚 我们提供：
- 每日[具体内容，如：早报分享]
- 每周[具体内容，如：一次学习打卡]
- 不定期[具体内容，如：经验分享]

【Step 2：入群任务】

为了让大家更好地融入，我们准备了一个小任务：

请选择完成以下任一任务：
1️⃣ 自我介绍：我是[姓名]，从事[职业]，想学习[方向]
2️⃣ 回答问题：你最想在[领域]提升哪方面？

完成后回复"已完成"，我拉你入群~

【Step 3：确认入群】

太好了！期待你在群里的分享~

回复"确认"或"已完成"，我即刻拉你！

【Step 4：入群后欢迎】

🎉 欢迎[用户昵称]加入！

请完成你的自我介绍，让大家认识你~
建议格式：
姓名 | 职业 | 学习目标

完成自我介绍后，你将获得[新人福利，如：资料包一份]

═══════════════════════════════════════════
```
