## 练习4：混合型需求

### 题目
用户说"效果不知道怎么样，而且公司规模小万一倒闭了数据怎么办"，应该如何应对？

### 练习模板

```
1. 需求判断：
   主要需求：____（类型）
   次要需求：____（类型）

2. 策略选择（分主次）：
   主要需求策略：____
   次要需求策略：____

3. 文案产出：
   功能文案：____
   社会文案：____
```

---

## 参考答案示例

### 练习1参考答案

```
1. 需求判断：
   需求类型：功能型
   核心痛点：担心学会没效果

2. 策略选择：
   适用策略：效果量化 + 证据证明
   具体方法：给具体学习路径+学员案例

3. 文案产出：
   主文案：21天学会Python，独立完成项目
   辅助文案：10万学员验证，平均涨薪50%
```

### 练习2参考答案

```
1. 需求判断：
   需求类型：情感型
   核心痛点：担心身份不匹配

2. 策略选择：
   适用策略：身份认同 + 场景共鸣
   具体方法：展示目标用户画像+使用场景

3. 文案产出：
   主文案：每个精致女人，都值得拥有
   辅助文案：背出自信，活出态度
```

### 练习3参考答案

```
1. 需求判断：
   需求类型：社会型
   核心痛点：担心品牌不靠谱

2. 策略选择：
   适用策略：权威背书 + 销量证明
   具体方法：展示合作客户+销售数据

3. 文案产出：
   主文案：XX上市公司都在用的系统
   辅助文案：服务企业超过500家
```

### 练习4参考答案

```
1. 需求判断：
   主要需求：功能型（效果、数据安全）
   次要需求：社会型（公司规模）

2. 策略选择（分主次）：
   主要需求策略：证据证明（稳定性数据）+ 风险保障（数据导出）
   次要需求策略：背书（大客户案例）+ 成立时间

3. 文案产出：
   功能文案：99.99%可用性，数据每日自动备份，可随时导出
   社会文案：服务XX上市公司5年，行业深耕10年
```
