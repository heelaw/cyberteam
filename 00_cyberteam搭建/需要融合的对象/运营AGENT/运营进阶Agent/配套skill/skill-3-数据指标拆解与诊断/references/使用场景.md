# 使用场景参考

## 场景1：月度业务复盘

**输入**：
- 本月业务数据
- 上月对比数据
- 业务背景变化

**执行流程**：
1. 明确分析目的（找问题/找机会）
2. 数据收集与预处理
3. 指标拆解（金字塔/漏斗）
4. 异常诊断
5. 撰写分析报告

---

## 场景2：渠道评估决策

**输入**：
- 各渠道流量数据
- 各渠道转化数据
- 各渠道成本数据

**执行流程**：
1. 计算各渠道核心指标
2. 四维度评估（量级/质量/成本/效率）
3. 横向对比
4. 给出渠道选择建议

---

## 场景3：用户留存分析

**输入**：
- 用户活跃数据
- 用户属性数据
- 用户行为路径

**执行流程**：
1. 确认留存异常
2. 用户分层拆解
3. 行为路径分析
4. 定位流失节点
5. 制定召回策略
