# interfaces - IDP计划Agent

## 核心原则

**所有Agent决策前必须读取项目资料库。所有Agent产出必须存入项目目录。**

---

## 一、输入规范 (Input)

### 1.1 触发条件

| 场景 | 触发词 | 说明 |
|------|--------|------|
| IDP制定 | IDP、个人发展计划 | 新周期规划 |
| 职业规划 | 职业发展规划、职业通道 | 长期发展 |
| 学习路径 | 学习路径、能力提升 | 学习规划 |
| 导师匹配 | 导师、辅导 | 发展资源 |

### 1.2 必读信息

| 信息类型 | 内容 | 说明 |
|----------|------|------|
| 职业目标 | 长期/中期/短期目标 | 发展方向 |
| 能力评估 | 当前能力/目标能力 | 差距分析 |
| 发展资源 | 预算/导师/课程 | 资源条件 |
| 时间约束 | 计划周期/里程碑 | 时间框架 |

---

## 二、输出规范 (Output)

### 2.1 输出格式

```markdown
## 个人发展计划（IDP）

### 基本信息
- 姓名：
- 部门：
- 职位：
- 制定日期：

### 职业发展目标
#### 长期（3年）
成为[方向]领域的[目标职位]

#### 中期（1年）
达成[具体目标]

#### 短期（Q1/Q2/Q3/Q4）
| 目标 | 衡量标准 | 时间 | 状态 |
|------|----------|------|------|

### 能力发展计划
#### 当前能力评估
| 能力项 | 当前水平 | 目标水平 | 差距 |
|--------|----------|----------|------|
| | | | |

#### 学习计划（70-20-10）
- 工作实践（70%）：
- 向人学习（20%）：
- 正式培训（10%）：

### 所需资源
- 预算：
- 导师：
- 其他：

### 检视机制
- 月度自检：
- 季度回顾：
- 年度复盘：
```

### 2.2 输出存放

```
{project}/decisions/          # IDP文档
{project}/meetings/           # 沟通记录
```

## 三、协作接口

### 3.1 向上协作

| 接收方 | 内容 | 格式 |
|--------|------|------|
| 直属上级 | IDP审批/季度回顾 | IDP文档/回顾记录 |
| HR | IDP存档/资源协调 | IDP文档 |

### 3.2 横向协作

| 协作方 | 场景 | 产出 |
|--------|------|------|
| 导师 | 发展辅导 | 辅导记录 |
| 培训部 | 课程推荐 | 培训计划 |

---

## 四、核心方法论

| 方法 | 适用场景 | 要点 |
|------|----------|------|
| SMART目标 | 目标设定 | 具体可衡量 |
| 70-20-10法则 | 学习规划 | 实践/学习/培训 |
| 能力差距分析 | 发展重点 | 当前vs目标 |
