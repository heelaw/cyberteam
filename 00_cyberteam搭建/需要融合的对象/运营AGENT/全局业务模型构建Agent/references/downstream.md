# 全局业务模型构建Agent - 下游去向

## 输出流向

### 直接输出给用户

| 输出类型 | 内容 | 用途 |
|---------|------|------|
| 商业模式识别报告 | 业务类型和特征 | 用户决策参考 |
| 收支结构报告 | 收入和成本结构 | 财务分析 |
| 流程指标报告 | 业务流程和KPI | 执行参考 |
| 完整业务模型 | 整体业务模型图 | 战略规划 |
| 验证报告 | 模型有效性评估 | 优化参考 |

## 协同下游Agents

| 下游Agent | 接收什么 | 用途 |
|-----------|---------|------|
| 指标拆解Agent | 业务模型 → 指标体系 | 指标细化 |
| 北极星指标Agent | 业务模型 → 目标设定 | 目标确定 |
| 局部业务模型构建Agent | 全局模型 → 局部模型 | 深入分析 |
| 增长策略咨询Agent | 业务模型 → 增长方案 | 增长规划 |

## 版本更新去向

当本Agent有版本更新时，需要通知：
1. **运营总监Agent** - 业务模型能力更新
2. **CEO Agent** - 战略决策参考
3. **产品总监Agent** - 产品规划参考
