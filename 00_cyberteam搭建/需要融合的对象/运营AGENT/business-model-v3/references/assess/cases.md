# business-model-v3 案例库

## 案例分类

### 1. 商业模式分析案例

#### 案例1：O2O预定模式 - 美团外卖
- **背景**：某二线城市外卖平台创业项目
- **业务模型**：线上获客 + 线下服务 + 预定收款
- **核心指标**：
  - CAC：45元
  - LTV：180元
  - 转化率：12%
- **分析要点**：收入公式拆解、转化漏斗优化

#### 案例2：SaaS订阅模式 - 在线办公软件
- **背景**：针对中小企业的在线文档协作工具
- **业务模型**：基础免费 + 高级订阅
- **核心指标**：
  - 月续费率：85%
  - CAC：120元
  - LTV：960元
- **分析要点**：订阅经济模型、留存驱动增长

#### 案例3：电商DTC模式 - 新消费品牌
- **背景**：某新锐护肤品品牌
- **业务模型**：社交电商 + 私域运营 + 会员体系
- **核心指标**：
  - GMV增长率：200%
  - 复购率：45%
  - 私域占比：60%
- **分析要点**：DTC增长飞轮、私域流量运营

### 2. 北极星指标案例

#### 案例4：工具类产品 - 任务管理App
- **北极星指标**：周活跃完成任务数
- **拆解逻辑**：DAU × 任务完成率 × 人均完成任务数
- **价值树映射**：功能使用 → 任务完成 → 续费意愿

#### 案例5：内容平台 - 短视频App
- **北极星指标**：人均观看时长
- **拆解逻辑**：DAU × 日均观看时长
- **关键杠杆**：推荐算法优化、内容质量提升

### 3. 探索型项目破局案例

#### 案例6：从0到1的新业务探索
- **背景**：某教育公司探索素质教育新业务
- **四步验证**：
  1. 价值假设：小提琴培训有市场需求
  2. 风险假设：家长愿意付费
  3. MVP测试：20人试课
  4. 验证结果：转化率60%，继续推进
- **关键洞察**：小切口验证，快速迭代

### 4. 业务目标拆解案例

#### 案例7：年度GMV目标拆解
- **目标**：年度GMV 1亿
- **拆解维度**：用户数 × 转化率 × 客单价 × 复购次数
- **执行策略**：四保原则（保核心、保增长、保现金流、保团队）

## 案例使用方法

1. **匹配场景**：根据用户行业和业务类型选择相似案例
2. **提取方法论**：从案例中提炼可复用的分析框架
3. **应用落地**：结合用户实际情况调整应用

## 案例来源

- 85+真实运营案例（2022-2024）
- 涵盖电商、SaaS、教育、内容、O2O等多个行业
