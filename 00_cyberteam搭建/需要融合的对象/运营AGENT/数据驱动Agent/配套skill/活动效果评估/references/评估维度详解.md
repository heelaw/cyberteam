# 评估维度详解

## 活动效果四维度评估法

### 维度一：产出指标

产出指标是衡量活动直接业务价值的核心维度，包括拉新效果、转化效果和营收效果三个子维度。

#### 1.1 拉新效果指标

| 指标 | 定义 | 计算方式 | 评估标准 |
|-----|------|---------|---------|
| 新用户注册数 | 活动期间新注册的用户数量 | COUNT(DISTINCT user_id WHERE注册时间in活动期AND新用户标记) | 目标达成率>100%优秀 |
| 渠道拉新量 | 各渠道新用户数 | 按渠道拆分注册用户 | 渠道效果对比 |
| 新用户质量系数 | 首单转化用户/注册用户 | 首单转化数/注册数 | >40%为高质量 |

#### 1.2 转化效果指标

| 指标 | 定义 | 计算方式 | 评估标准 |
|-----|------|---------|---------|
| 注册转化率 | 曝光到注册的转化 | 注册数/曝光量×100% | >3%为优秀 |
| 付费转化率 | 注册到付费的转化 | 付费用户数/注册数×100% | >10%为优秀 |
| 活动参与率 | 访问到参与的转化 | 参与用户数/访问UV×100% | >50%为优秀 |
| 人均订单数 | 付费用户平均订单数 | 总订单数/付费用户数 | >1.5为高价值 |

#### 1.3 营收效果指标

| 指标 | 定义 | 计算方式 | 评估标准 |
|-----|------|---------|---------|
| GMV | 活动期间成交总额 | SUM(订单金额) | 目标达成率>100%优秀 |
| ARPU | 平均每用户收入 | GMV/付费用户数 | 行业对比 |
| 活动利润率 | 营收减去成本 | (GMV-成本)/GMV×100% | >30%为健康 |

### 维度二：成本指标

成本指标是评估活动投入产出效率的关键维度，包括成本构成、获客成本和ROI分析。

#### 2.1 成本构成分析

| 成本类型 | 包含项目 | 占比建议 | 异常判断 |
|---------|---------|---------|---------|
| 投放成本 | 广告投放、渠道采购 | 50-70% | >80%需优化 |
| 运营成本 | 奖品、红包、优惠券 | 20-30% | <10%或>40%异常 |
| 人力成本 | 人员投入折算 | 5-10% | >15%需评估 |
| 技术成本 | 系统开发、运维 | 3-5% | >10%需评估 |

#### 2.2 获客成本(CAC)分析

```
CAC = 总活动成本 / 有效获客数

有效获客数 = 新注册用户数 × 质量系数

质量系数等级：
- >50%: 极高价值用户
- 30-50%: 高价值用户
- 10-30%: 普通价值用户
- <10%: 低价值用户
```

**CAC评估标准**：
| CAC区间 | 评估 | 行动建议 |
|--------|-----|---------|
| CAC < 目标值80% | 优秀 | 可适当增加投放 |
| CAC = 目标值 | 达标 | 维持现状 |
| CAC > 目标值 但 < 目标值1.5倍 | 勉强接受 | 需优化 |
| CAC > 目标值1.5倍 | 失败 | 必须优化或停止 |

#### 2.3 ROI分析

```
ROI = (活动营收 - 活动成本) / 活动成本 × 100%

活动营收 = 直接营收 + 后续营收预估(乘以生命周期系数)
```

**ROI评估标准**：
| ROI区间 | 评估 | 行动建议 |
|--------|-----|---------|
| ROI > 300% | 极其优秀 | 大力复制 |
| ROI 200-300% | 优秀 | 保持规模 |
| ROI 100-200% | 良好 | 可持续 |
| ROI 50-100% | 一般 | 需优化 |
| ROI < 50% | 亏损 | 重新评估 |

### 维度三：传播指标

传播指标衡量活动的传播效果和自增长能力。

#### 3.1 传播漏斗

```
曝光 → 点击 → 访问 → 参与 → 分享 → 二次传播
```

| 漏斗阶段 | 核心指标 | 计算方式 | 健康标准 |
|---------|---------|---------|---------|
| 曝光→点击 | CTR点击率 | 点击数/曝光量×100% | >2% |
| 点击→访问 | 访问率 | 访问UV/点击数×100% | >60% |
| 访问→参与 | 参与率 | 参与用户/访问UV×100% | >50% |
| 参与→分享 | 分享率 | 分享数/参与用户×100% | >5% |
| 分享→二次曝光 | 传播倍数 | 二次曝光/一次曝光 | >1.5 |

#### 3.2 传播渠道对比

| 渠道类型 | 特点 | 适用场景 | 成本效率 |
|---------|-----|---------|---------|
| 付费投放 | 见效快、可控 | 冲刺目标 | 取决于ROI |
| 自然流量 | 成本低、可信 | 品牌活动 | 效率高 |
| 社交裂变 | 传播快、成本低 | 拉新活动 | 爆发力强 |
| 私域传播 | 精准、成本低 | 促活活动 | 转化率高 |

### 维度四：用户行为指标

用户行为指标评估用户参与深度和体验质量。

#### 4.1 参与漏斗分析

| 阶段 | 用户数 | 流失率 | 流失原因 |
|-----|-------|-------|---------|
| 活动页访问 | | - | |
| 了解活动规则 | | | |
| 产生兴趣 | | | |
| 参与活动 | | | |
| 完整体验 | | | |
| 达成目标 | | | |

#### 4.2 用户体验评分

| 体验维度 | 评估指标 | 数据来源 | 评分标准 |
|---------|---------|---------|---------|
| 操作便捷性 | 操作步骤数 | 产品数据 | 步骤越少越好 |
| 奖励感知度 | 奖励领取率 | 活动数据 | >80%为优秀 |
| 流程顺畅度 | 页面跳出率 | 埋点数据 | <30%为优秀 |
| 整体满意度 | 用户评分 | 反馈数据 | >4.0为优秀 |

## 综合评级计算

### 权重分配

| 维度 | 权重 | 说明 |
|-----|-----|-----|
| 产出指标 | 30% | 核心业务价值 |
| 成本指标 | 25% | 投入产出效率 |
| 传播指标 | 20% | 品牌价值 |
| 用户行为 | 25% | 用户体验 |

### 评级标准

| 综合得分 | 评级 | 行动建议 |
|---------|-----|---------|
| 90-100分 | 优秀 | 大力推广 |
| 75-89分 | 良好 | 保持优化 |
| 60-74分 | 合格 | 需改进 |
| <60分 | 不合格 | 重新策划 |
