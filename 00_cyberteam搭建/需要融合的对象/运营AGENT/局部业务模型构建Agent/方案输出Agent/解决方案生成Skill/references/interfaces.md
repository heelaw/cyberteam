# 解决方案生成Skill - 接口协议

## 输入接口

| 接口名称 | 类型 | 描述 | 必需 |
|----------|------|------|------|
| problem_analysis | Object | 问题分析结果 | 是 |
| context | Object | 业务上下文 | 否 |

## 输出接口

| 接口名称 | 类型 | 描述 |
|----------|------|------|
| solutions | Array | 解决方案列表 |
| summary | Object | 方案摘要 |

## 错误码

| 错误码 | 描述 |
|--------|------|
| E001 | 问题分析结果为空 |
| E002 | 无法生成有效方案 |
