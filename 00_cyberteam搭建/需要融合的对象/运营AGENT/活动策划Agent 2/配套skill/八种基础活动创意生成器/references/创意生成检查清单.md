# 创意生成检查清单

## 一、创意生成前检查

### 1.1 需求确认
- [ ] 活动目标类型已明确（拉新/促活/转化/品牌/留存）
- [ ] 目标受众画像已确定
- [ ] 预算范围已确定
- [ ] 时间窗口已明确

### 1.2 参考素材
- [ ] 已分析竞品近期活动
- [ ] 已了解行业常见玩法
- [ ] 已收集用户反馈数据

---

## 二、创意生成中检查

### 2.1 目标匹配度
- [ ] 创意能直接贡献核心KPI
- [ ] 创意类型与目标类型一致
- [ ] 目标受众会被这个创意吸引

### 2.2 差异化检查
- [ ] 有独特的创意切入点
- [ ] 不是简单复制竞品
- [ ] 有明确的记忆点

### 2.3 可行性检查
- [ ] 技术方案可行
- [ ] 预算在范围内
- [ ] 时间窗口足够

---

## 三、创意输出检查

### 3.1 必含要素
- [ ] 创意类型标注
- [ ] 核心创意描述（一句话）
- [ ] 创意亮点（差异化竞争优势）
- [ ] 详细活动形式
- [ ] 参与流程
- [ ] 激励机制
- [ ] 目标匹配度说明
- [ ] 风险提示

### 3.2 格式要求
- [ ] 使用标准输出格式
- [ ] 描述清晰易懂
- [ ] 数据有据可查

---

## 四、创意筛选检查

### 4.1 自检清单审核
- [ ] 通过目标匹配度审核
- [ ] 通过创意差异化审核
- [ ] 通过用户体验审核
- [ ] 通过可执行性审核
- [ ] 通过合规安全性审核

### 4.2 综合评估
- [ ] 综合评分达到及格线以上
- [ ] 无否决条件触发
- [ ] 风险可控

---

## 五、创意优化检查

### 5.1 如评分不足80分，需优化
- [ ] 针对低分项进行优化
- [ ] 优化后重新自检
- [ ] 提交二审

### 5.2 优化方向
| 维度 | 常见问题 | 优化方向 |
|-----|---------|---------|
| 目标匹配 | 贡献路径不清晰 | 明确创意如何贡献KPI |
| 差异化 | 创意雷同 | 寻找独特切入点 |
| 用户体验 | 门槛太高 | 降低参与门槛 |
| 可执行性 | 资源不足 | 调整资源需求 |
| 合规安全 | 违规风险 | 调整措辞和形式 |

---

## 六、创意定稿检查

### 6.1 最终确认
- [ ] 创意已通过全部审核
- [ ] 创意已获得相关方确认
- [ ] 创意已进入方案设计阶段

### 6.2 交付物清单
- [ ] 创意方案文档
- [ ] 自检报告
- [ ] 评审纪要（如有）

---

## 七、常见问题排查

### 7.1 创意与目标不匹配
**问题**: 创意很新颖但不知道如何贡献KPI
**解决**: 回溯到目标，确保创意服务目标

### 7.2 创意过于保守
**问题**: 创意缺乏差异化，和竞品没区别
**解决**: 寻找独特的创意切入角度

### 7.3 创意难以执行
**问题**: 创意很好但现有资源无法实现
**解决**: 简化创意或申请更多资源

### 7.4 创意存在风险
**问题**: 创意可能有合规或舆情风险
**解决**: 调整创意消除风险或准备预案
