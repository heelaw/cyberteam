## 六、角度执行检查

### 6.1 角度选择检查
- [ ] 角度与品牌相关
- [ ] 角度不牵强
- [ ] 角度积极正面
- [ ] 角度有传播性

### 6.2 内容执行检查
- [ ] 文案自然流畅
- [ ] 不生硬贴热点
- [ ] 品牌价值自然融入
- [ ] 用户能产生共鸣

### 6.3 风险预防检查
- [ ] 无法律合规问题
- [ ] 无版权侵权问题
- [ ] 无舆情风险
- [ ] 有备选方案
