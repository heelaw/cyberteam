# 热点风险管控指南

## 一、风险分类体系

### 1.1 风险类型矩阵

| 风险类型 | 描述 | 发生概率 | 影响程度 |
|---------|------|---------|---------|
| 反转风险 | 热点事件出现反转 | 中 | 高 |
| 舆情风险 | 引发负面舆论 | 中 | 高 |
| 法律风险 | 违反广告法或法规 | 低 | 极高 |
| 执行风险 | 响应速度不够快 | 高 | 中 |
| 关联风险 | 强行蹭热点引发反感 | 中 | 中 |

---

## 二、反转风险管控

### 2.1 反转预警信号

**早期预警**:
- [ ] 事件信息不完整
- [ ] 官方尚未确认
- [ ] 存在多种解读可能
- [ ] 涉及敏感群体

**中期预警**:
- [ ] 开始有质疑声音
- [ ] 媒体开始深挖
- [ ] 当事人态度模糊
- [ ] 出现不同版本

### 2.2 应对策略

**预防阶段**:
- 不抢第一时间
- 等待信息确认
- 准备备选方案

**监测阶段**:
- 实时监测舆情
- 设置预警阈值
- 准备应急响应

**撤退阶段**:
- 立即停止借势
- 撤下相关物料
- 发布澄清声明

### 2.3 撤退决策树

```
热点事件发展
    ↓
是否出现反转信号？
    ↓是
是否已发布借势内容？
    ↓是
立即撤下内容 + 发布声明
    ↓否
停止借势计划
    ↓否
继续监测...
```

## 三、舆情风险管控

### 3.1 常见舆情风险点

| 风险点 | 场景 | 应对措施 |
|--------|------|---------|
| 强行关联 | 热点与品牌无关硬蹭 | 寻找真实关联点 |
| 消费悲剧 | 蹭社会悲剧热点 | 绝对禁止 |
| 夸大宣传 | 承诺无法兑现 | 避免虚假宣传 |
| 侵犯权益 | 使用未授权内容 | 确保版权合规 |
| 低俗营销 | 使用低俗内容 | 保持品牌调性 |

### 3.2 舆情监测方案

**监测渠道**:
- 微博评论
- 抖音评论
- 微信生态
- 新闻媒体

**监测指标**:
| 指标 | 正常范围 | 预警阈值 |
|-----|---------|---------|
| 负面评论占比 | <5% | >10% |
| 情感倾向 | 正面为主 | 负面为主 |
| 传播声量 | 预期范围内 | 超出预期10倍 |

### 3.3 舆情应对话术

**危机公关原则**:
- 快速响应
- 真诚道歉
- 承担责任
- 给出解决方案

**示例话术**:
```
如引发反感：
"感谢大家的反馈，我们第一时间反思了这次借势的方式，
确实考虑不周。我们深感抱歉，后续会更加谨慎。
再次感谢大家的监督和批评。"
```

## 四、法律合规风险

### 4.1 广告法红线

**绝对禁止**:
- [ ] 使用国家级、最高级、最佳等用语
- [ ] 虚假宣传、夸大优惠
- [ ] 侵犯他人知识产权
- [ ] 未经授权使用他人肖像
- [ ] 诱导分享、欺骗用户

### 4.2 借势营销合规要点

| 类型 | 合规要求 | 违规风险 |
|-----|---------|---------|
| 明星热点 | 不得丑化、歪曲 | 侵犯肖像权 |
| 影视综艺 | 不得直接使用素材 | 版权侵权 |
| 事件营销 | 不得虚假编造 | 欺诈用户 |
| 价格宣传 | 不得虚构原价 | 虚假宣传 |

### 4.3 合规审查清单

- [ ] 借势文案已法务审核
- [ ] 无国家级、最高级等用语
- [ ] 价格优惠有真实依据
- [ ] 使用素材已获授权
- [ ] 无侵犯用户隐私内容

## 五、执行风险管控

### 5.1 时间风险

**常见问题**:
- 审批流程过长，错过热点
- 技术开发时间不足
- 物料制作延误

**解决方案**:
- 建立绿色审批通道
- 提前准备借势模板
- 预留足够准备时间

### 5.2 资源风险

**常见问题**:
- KOL档期无法协调
- 推广预算不足
- 执行人员不足

**解决方案**:
- 提前锁定KOL资源
- 预留应急预算
- 建立备用执行团队

## 七、风险复盘机制

### 7.1 复盘要点

每次热点借势后，都应进行复盘：

- [ ] 风险识别是否准确
- [ ] 预案执行是否到位
- [ ] 撤退决策是否及时
