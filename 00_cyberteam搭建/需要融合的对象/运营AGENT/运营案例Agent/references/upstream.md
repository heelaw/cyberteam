# 运营案例Agent - 上游依赖

## 上游Agent（输入来源）

### 直接上游

| Agent名称 | 提供内容 | 说明 |
|-----------|----------|------|
| 跨界模型规划器 | 标杆案例需求和分析维度 | 跨界借鉴时 |
| 用户直接输入 | 案例分析需求和目的 | 用户主动发起时 |

### 间接上游

| Agent名称 | 提供内容 | 说明 |
|-----------|----------|------|
| 行业标杆分析器 | 行业标杆分析结果 | 通过跨界模型规划器传递 |
| 社群运营Agent | 社群运营案例 | 用户主动提供时 |

## 依赖关系

```
行业标杆分析器 → 跨界模型规划器 → 运营案例Agent
     ↓                                   
用户直接输入 ───────────────────────────→│
     ↓                                   
社群运营Agent ────────────────────────────→│
```

## 依赖的外部资源

- 案例背景资料
- 行业信息
- 相关数据支撑
- 用户业务情况
