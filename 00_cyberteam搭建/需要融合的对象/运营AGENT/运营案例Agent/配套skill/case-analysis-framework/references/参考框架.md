# case-analysis-framework 参考资料

## 1. 单一案例拆解框架

### 三步拆解法

```
Step 1: 还原成果
├── 案例基本信息（品牌/时间/领域）
├── 核心数据指标
└── 成功程度评估

Step 2: 归纳业务目的
├── A. 流量与传播
├── B. 营销与转化
└── C. 留存与活跃

Step 3: 提炼执行细节
├── 流程设计亮点
├── 互动传播机制
├── 营销转化策略
└── 留存促活机制
```

## 2. 批量拆解四维度

```
维度1: 主题 - 活动名称的吸引力
维度2: 形式 - 活动玩法的机制设计
维度3: 流程 - 用户参与路径与体验
维度4: 类型 - 裂变型/传播型/留存型
```

## 3. 弹药库结构

```
弹药库/
├── 活动主题弹药库/
│   ├── 裂变类主题
│   ├── 传播类主题
│   └── 留存类主题
├── 活动形式弹药库/
│   ├── 红包类玩法
│   ├── 任务类玩法
│   └── 游戏类玩法
├── 活动流程弹药库/
│   ├── 拉新流程
│   ├── 促活流程
│   └── 转化流程
└── 内容素材弹药库/
    ├── 标题公式
    ├── 开篇模板
    └── 金句素材
```

## 4. 经典案例分类

- 拼多多天天领现金

### 传播型活动案例

### 留存型活动案例
