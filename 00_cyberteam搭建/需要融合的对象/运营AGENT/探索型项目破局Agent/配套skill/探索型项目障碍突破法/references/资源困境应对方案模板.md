# 资源困境应对方案模板

## 模板说明

此模板用于类型二障碍（资源困境）的应对规划。当你的项目遇到预算、人手或时间不足时，使用此模板。

## 核心原则

1. **聚焦核心** - 资源有限时，只做最核心的事
2. **优化配置** - 重新评估资源配置优先级
3. **借力打力** - 想办法借用外部资源
4. **降低标准** - 用更低的成本完成同样的目标

## 常见问题

**Q: 什么都不能砍怎么办？**
A: 那就延长项目周期，用时间换资源。

**Q: 降标准会不会影响验证结果？**
A: 关键标准的降低会影响结论，要非常谨慎。可以降低非关键标准。

**Q: 借力会不会欠人情？**
A: 会的。要评估借力的长期成本，考虑是否有互惠的可能。
