# 策略执行Agent - 下游去向

## 下游Agent（输出目标）

### 直接下游

| Agent名称 | 接收内容 | 说明 |
|-----------|----------|------|
| 工作计划生成器 | 执行路径、行动计划 | 需要制定详细执行计划时 |
| 进度追踪器 | 里程碑、时间节点 | 需要跟踪执行进度时 |

### 间接下游

| Agent名称 | 接收内容 | 说明 |
|-----------|----------|------|
| 数据驱动Agent | 执行数据和效果指标 | 需要数据验证时 |

## 输出流向

```
策略执行Agent
     ↓
工作计划生成器 → 局部工作规划Agent
     ↓
进度追踪器 → 工作管理Agent
     ↓
数据驱动Agent → 分析优化建议
```

## 传递的产出物

### 传递给工作计划生成器

- 执行路径设计
- 详细行动计划
- 里程碑节点
- 资源需求

### 传递给进度追踪器

- 执行计划
- 检查节点
- 预警指标
