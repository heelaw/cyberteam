## 问题诊断表

| 诊断维度 | 问题 | 严重程度 | 原因分析 |
|----------|------|---------|---------|
| 曝光→点击 | CTR偏低 | 高 | 素材不够吸引/定向不精准/渠道不匹配 |
| 点击→转化 | CVR偏低 | 高 | 落地页问题/价格过高/信任不足 |
| 整体 | 都偏低 | 极高 | 产品竞争力/竞品/市场需求 |

### 卡点定位口诀

```
CTR低 → 素材、定向、渠道问题
CVR低 → 落地页、价格、信任问题
都偏低 → 产品、竞品、需求问题
```

## 优化建议表

| 优先级 | 问题 | 优化方向 | 具体建议 | 预期效果 |
|--------|------|---------|---------|---------|
| P0 | CTR低 | 素材优化 | A/B测试不同素材开头 | CTR提升30-50% |
| P1 | CTR低 | 定向优化 | 缩小定向范围，提升精准度 | CTR提升20-30% |
| P2 | CVR低 | 落地页优化 | 优化首屏卖点展示 | CVR提升15-25% |
| P3 | CVR低 | 价格策略 | 设置首单优惠/限时折扣 | CVR提升20-40% |
