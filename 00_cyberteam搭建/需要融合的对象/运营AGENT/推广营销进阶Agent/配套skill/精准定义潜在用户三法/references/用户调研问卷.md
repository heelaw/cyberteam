## 调研注意事项

1. **样本量**：定量问卷至少50份，深度访谈至少5人
2. **筛选条件**：确保调研对象符合目标画像
3. **奖励机制**：可设置小礼品激励完成问卷
4. **数据清洗**：剔除无效问卷（如全部选同一选项）
