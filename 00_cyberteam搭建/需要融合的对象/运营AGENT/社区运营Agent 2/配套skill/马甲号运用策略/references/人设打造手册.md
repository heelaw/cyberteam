# 人设打造手册

## 一、人设类型库

### 专家型
| 特征 | 描述 |
|------|------|
| 头像风格 | 专业形象照或职业场景 |
| 昵称风格 | 带专业头衔或昵称 |
| 简介 | 突出专业背景和经验 |
| 性格 | 严谨、专业、有深度 |
| 语言风格 | 学术化、数据支撑 |
| 擅长领域 | 垂直细分领域专家 |
| 适合话题 | 专业教程、深度分析 |

### 达大型
| 特征 | 描述 |
|------|------|
| 头像风格 | 生活化、有亲和力 |
| 昵称风格 | 接地气、易记 |
| 简介 | 强调实战经验 |
| 性格 | 热情、乐于分享 |
| 语言风格 | 口语化、接地气 |
| 擅长领域 | 经验分享、实战技巧 |
| 适合话题 | 经验分享、干货教程 |

### 新手型
| 特征 | 描述 |
|------|------|
| 头像风格 | 年轻化、清新 |
| 昵称风格 | 可爱或简洁 |
| 简介 | 表明新手身份 |
| 性格 | 好学、积极 |
| 语言风格 | 疑问句多、谦虚 |
| 擅长领域 | 提出问题、引导讨论 |
| 适合话题 | 新手问题、经验求助 |

### 旁观型
| 特征 | 描述 |
|------|------|
| 头像风格 | 低调、不突出 |
| 昵称风格 | 中性、不带倾向 |
| 简介 | 简短或空白 |
| 性格 | 理性、客观 |
| 语言风格 | 观望、引导 |
| 擅长领域 | 客观评价、理性分析 |
| 适合话题 | 争议话题、观点讨论 |

## 三、语言风格指南

### 专家型话术
```
"从专业角度来看，这个问题需要从以下几个方面分析..."
"根据我的经验，建议大家注意以下几点..."
"数据显示，XXX趋势正在形成..."
```

### 达大型话术
```
"作为一个过来人，我踩过这个坑，给大家分享下..."
"亲测有效！这个方法真的好用..."
"不多说，直接上干货！..."
```

### 新手型话术
```
"萌新提问，这个具体怎么操作啊？"
"求助！有没有大佬遇到过同样的问题..."
"请问各位前辈，这个情况正常吗？"
```

### 旁观型话术
```
"这个话题挺有意思，我围观一下..."
"两边说的都有道理，大家怎么看？"
"客观来说，这个问题需要多方面考虑..."
```

- [ ] 是否避免了禁止行为
