# 情感连接执行手册

| 渠道类型 | 具体形式 | 负责人 | 频次 |
|----------|----------|--------|------|
| 直联通道 | 创始人微信直联 | 创始人 | 按需 |
| 高层群 | 5-10人精品群 | 运营总监 | 每周 |
| 线下拜访 | 1v1深度交流 | BD/运营 | 按需 |

| 渠道类型 | 具体形式 | 负责人 | 频次 |

| 定期活动 | 月度核心用户活动 | 运营 | 每月 |

### B级活跃用户沟通渠道
| 渠道类型 | 具体形式 | 负责人 | 频次 |

| 活动群 | 活动专属群 | 活动运营 | 活动时 |
| 私信 | 问题反馈处理 | 客服 | 即时 |
| 官方账号 | 社区官方账号 | 运营 | 日常 |

## 三、反馈机制

### 建议收集
- **S级**：每季度1v1访谈，记录建议
- **A级**：月度问卷+VIP群收集
- **B级**：季度问卷

### 采纳奖励
| 采纳建议类型 | 奖励 |
|-------------|------|
| 产品功能建议 | 500积分+功能冠名 |
| 内容方向建议 | 200积分 |
| 活动建议 | 100积分 |
| 其他建议 | 50积分 |

### 反馈处理时效
| 层级 | 反馈响应 | 反馈处理 |
|------|----------|----------|
| S级 | 即时 | 3个工作日内 |
| A级 | 24小时内 | 5个工作日内 |
| B级 | 48小时内 | 7个工作日内 |
