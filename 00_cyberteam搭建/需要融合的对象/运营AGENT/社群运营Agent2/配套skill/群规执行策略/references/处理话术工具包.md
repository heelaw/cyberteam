# 分级处理话术工具包

## 一、严重违规处理话术

### 1. 发广告/推广

**群内通知（匿名）**:
```
⛔ 违规处理通知

@[违规者昵称] 因在群内发布广告/推广信息，
已被移出[社群名称]。

本群禁止一切形式的广告和推广。
感谢理解。
```

**移出后私聊**:
```
📝 移出通知

[昵称]，你好！

您因在[社群名称]发布广告/推广信息，
已被移出社群。

[社群名称]是一个[社群定位]的社群，
为了维护社群质量，我们禁止一切广告和推广。

如有优质内容想分享，欢迎通过以下方式：
- 联系管理员审核
- 优秀内容可免费发布

感谢理解。
```

### 2. 人身攻击/辱骂

**群内通知**:
```
⚠️ 群规警告

@[违规者昵称] 因[人身攻击行为]，
已被移出社群。

请所有成员引以为戒，
尊重他人，友善交流。

如有问题请私下沟通解决。
```

### 3. 拉人进其他群

**群内通知**:
```
⚠️ 违规处理

@[违规者昵称] 因未经同意拉人进其他群，
已被移出[社群名称]。

本群禁止未经管理员同意拉人进其他群。
如有合作需求请联系管理员。
```

### 4. 诈骗/虚假信息

**处理后通知**:
```
🚨 重要通知

@[违规者昵称] 因发布虚假/诈骗信息，
已被移出[社群名称]，相关信息已举报。

请所有成员警惕网络诈骗，
不要轻信群内任何涉及金钱交易的信息。

如有疑问请联系管理员。
```

## 二、一般违规处理话术

### 1. 第一次私聊警告

```
📝 群规提醒

[昵称]，你好！

我是[社群名称]的管理员[名字]。

我们注意到您刚才在群内发布了[内容描述]，
这个内容[不太适合/违反了]群规[具体条款]。

[温和说明原因]

请注意：
- 这是您的第1次此类提醒
- 下次再犯将升级为正式警告
- 如有疑问可以回复本消息

感谢理解，让我们一起维护好社群氛围！
```

### 2. 第二次群内警告

```
⚠️ 群规警告

@[违规者昵称] 提醒：

您的[违规行为]已违反群规[具体条款]。

这是您的第2次违规。
根据群规，将进行[下一步处理]。

请遵守群规，共同维护社群秩序。

👉 群规全文：[链接]
```

### 3. 第三次禁言处理

```
🔇 禁言通知

@[违规者昵称] 因[违规原因]，
即日起被限制发言权限3天。

限制期间：
- 不可发送消息
- 可查看群内容
- 期限：2024年[X]月[X]日

如有疑问请联系 @管理员
```

### 4. 第四次移出

```
⛔ 移出通知

@[违规者昵称] 因[多次违规行为]，
已被移出[社群名称]。

违规次数：4次
处理方式：永久移除

感谢您之前的参与。
祝顺利。
```

## 三、轻微违规处理话术

### 1. 友善刷屏提醒

```
💬 温馨提示

@[昵称] 建议：

您今天在群内发言比较频繁，
可能打扰到其他成员。

建议：
- 可以考虑整理后一次性分享
- 重要内容可以发长消息
- 交流时注意节奏

感谢配合~
```

### 2. 格式不当提醒

```
💬 小提示

@[昵称] 建议：

您的发言格式可能不太方便阅读。

建议格式：
```
内容标题
内容正文
```

这样大家更容易理解和互动哦~
```

### 3. 话题偏离提醒

```
💬 温馨提示

@[昵称] 建议：

您的话题与本期主题[具体主题]略有偏离。

本期重点讨论：[核心问题]

当然，日常交流也欢迎，
只是希望主题讨论时更聚焦~

感谢理解！
```

## 四、特殊情况处理话术

### 1. 误判道歉

```
📝 道歉声明

@[昵称] 您好！

非常抱歉，之前对您的[行为判定有误]。

经复核，[说明原因]。

我们已撤销[处理措施]。
欢迎您继续参与社群！

感谢您的理解和支持。
```

### 2. 申诉回复-成立

```
✅ 申诉处理结果

您的申诉已处理。

复核结果：**申诉成立**

[详细说明原因]

我们已撤销[原处理措施]，
对给您带来的不便深感抱歉。

感谢您的理解。
```

### 3. 申诉回复-不成立

```
📝 申诉处理结果

您的申诉已处理。

复核结果：**申诉不成立**

[详细说明维持原判的原因]

如果您仍有异议，
可在[时间]前联系[上级联系方式]。

感谢理解。
```
