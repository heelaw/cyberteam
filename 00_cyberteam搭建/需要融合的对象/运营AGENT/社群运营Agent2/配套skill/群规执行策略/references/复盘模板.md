## 复盘会议议程

### 周会（30分钟）

```
1. 数据通报（5分钟）
   - 本周违规数据

2. 典型案例讨论（10分钟）
   - 1-2个典型案例

3. 问题发现（10分钟）
   - 本周遇到的问题

4. 解决方案（5分钟）
   - 讨论解决方案
```

### 月会（60分钟）

```
1. 数据通报（10分钟）
   - 全月数据概览

2. 趋势分析（15分钟）
   - 违规趋势分析

3. 典型案例复盘（15分钟）
   - 重点案例深度复盘

4. 规则迭代讨论（10分钟）
   - 规则调整提案

5. 下月计划（10分钟）
   - 重点工作计划
```
