# 用户进群引导三步法话术模板

## 模板说明
本模板提供三步引导的标准话术，可直接使用或根据社群特点修改。

---

## 第一步：即时欢迎话术

### 基础版

```
@新成员 👋 欢迎[昵称]加入[社群名称]！

📌 社群简介：[一句话定位]
👥 成员规模：[人数]位[用户类型]
🎯 本群目标：[社群核心目标]

📋 入群必做：
1️⃣ 修改群昵称：姓名-公司/行业-城市
2️⃣ 查看群公告
3️⃣ 介绍你自己

有问题找管理员 @xxx ~
```

### 进阶版（带老成员欢迎）

```
🎉 热烈欢迎 [昵称] 加入[社群名称]！

@xxx @xxx @xxx 快来欢迎新伙伴~

📌 [昵称] 请先：
1️⃣ 修改群昵称
2️⃣ 看看群公告
3️⃣ 做个小介绍

[社群名称]期待你的分享！
```

### 高端版（带价值预告）

```
@新成员 [昵称] 你好！

欢迎加入[社群名称]——[一句话定位]。

🎁 入群即享：
- [权益1]
- [权益2]
- [权益3]

📅 近期活动：
[日期] [活动名称] - 报名请回复[暗号]

💡 第一步：
点击群公告，了解[社群名称]的玩法~
```

【每日早报】⏰ 每天8:00

[内容类型]，深度学习[主题]

[日期1] [活动1]
[日期2] [活动2]

## 第三步：互动引导话术

### 自我介绍引导型

```
@新成员 [昵称] 入群满24小时啦！

👤 请用这个格式做自我介绍：

📛 姓名：
🏢 公司/行业：
💼 职位/角色：
💡 能提供：
🤝 需要：

👋 老规矩：
[呼唤老成员] 来欢迎新伙伴~

📋 新人小任务（任选其一）：
□ 在群里做自我介绍
□ 点赞今日内容
□ 回答一个问题

🎁 完成任务可得：
[小福利/资料/工具]
```

### 任务引导型

```
@新成员 [昵称] 欢迎入群！

🎯 你的第一个小任务：

□ 任务1：[描述]，完成后获得[奖励]
□ 任务2：[描述]，完成后获得[奖励]
□ 任务3：[描述]，完成后获得[奖励]

📅 完成时间：入群后7天内

👥 老成员帮忙：
@xxx @xxx 欢迎新伙伴~

💬 有问题找 @管理员
```

## 自动回复话术

### 欢迎语（机器人自动发送）

```
👋 [昵称] 欢迎加入[社群名称]！

📖 了解我们：
点击这里 → [群公告链接]

📝 加入后必做：
1. 修改群昵称
2. 看群公告
3. 做自我介绍

期待你的精彩分享！
```

### 常见问题自动回复

| 问题 | 自动回复 |
|-----|---------|
| 怎么改昵称 | 请点击群成员列表，找到自己，点击修改昵称为：姓名-公司/行业-城市 |
| 活动在哪报名 | 回复 [暗号] 即可报名本月活动，或联系管理员 @xxx |
| 群规是什么 | 请查看群公告，群规包含：[简要说明] |
| 怎么联系管理员 | 请 @管理员xxx，或私信 [微信号] |
</parameter>
