| 错误类型 | 触发条件 | 处理方式 |

## 常见问题

### Q1: 初创公司适合哪些渠道？

**A**: 建议从低成本渠道开始：
1. **SEO + 内容营销** - 长期投入，积累自然流量
2. **社交媒体运营** - 微博/抖音/小红书免费内容分发
3. **KOC合作** - 找小规模KOC合作，成本低效果真实
4. **信息流广告测试** - 小预算测试，找到有效渠道后再放大

### Q2: 如何选择合适的KOL？

**A**: 评估维度：
1. **粉丝画像** - 与目标用户的重合度
2. **内容质量** - 内容的专业性和可信度
3. **互动率** - 点赞/评论/分享的比例
4. **带货历史** - 是否有类似产品带货经验
5. **报价合理性** - 性价比是否合适

### Q3: 多渠道如何分配预算？

**A**: 建议分配方式：
1. **测试期（1-2周）**: 80%测试 + 20%优化
2. **放量期**: 70%主力渠道 + 20%辅助渠道 + 10%测试新渠道
3. **稳定期**: 60%主力 + 30%辅助 + 10%探索
