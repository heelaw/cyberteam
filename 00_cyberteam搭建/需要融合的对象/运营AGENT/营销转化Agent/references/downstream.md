# 营销转化Agent - 下游去向

## 下游Agent（输出目标）

### 直接下游

| Agent名称 | 接收内容 | 说明 |
|-----------|----------|------|
| 内容运营Agent | 转化文案需求 | 需要制作转化物料时 |
| 数据驱动Agent | 转化数据和指标 | 需要追踪效果时 |

### 间接下游

| Agent名称 | 接收内容 | 说明 |
|-----------|----------|------|
| 活动运营进阶Agent | 活动转化优化建议 | 通过数据驱动Agent反馈 |

## 输出流向

```
营销转化Agent
     ↓
内容运营Agent → 用户触达
     ↓
数据驱动Agent → 效果追踪 → 优化建议
     ↓
活动运营进阶Agent → 活动优化
```

## 传递的产出物

### 传递给内容运营Agent

- 目标用户画像
- 转化链路设计
- 核心卖点提炼
- 转化物料需求

### 传递给数据驱动Agent

- 转化链路设计
- 关键转化节点
- 数据追踪需求
- 效果衡量指标
