# 飞轮设计工作表

## 什么是飞轮

飞轮是指能形成自我强化循环的战略机制。一旦飞轮转动起来，就会持续加速，为业务提供持续的增长动力。

## 飞轮的特征

| 特征 | 说明 | 自检问题 |
|------|------|----------|
| **正向循环** | 每个环节都强化下一个环节 | 是正向还是负向？ |
| **自我驱动** | 不需要外部持续输血 | 需要多少外部干预？ |
| **持续加速** | 越转越快 | 增速是否递减？ |
| **可壁垒化** | 竞争对手难以复制 | 壁垒在哪里？ |

### 双边平台飞轮示例

**案例: 滴滴的飞轮**

```
司机供给密度
    ↓
乘客等待时间减少
    ↓
乘客满意度提升
    ↓
乘客自然增长
    ↓
单司机订单量提升
    ↓
司机收入提升
    ↓
更多司机加入
    ↓
【回到司机供给密度】
```

**启动方案**: 早期通过司机补贴建立供给密度

## 飞轮自检清单

### 基础检查

- [ ] 飞轮有明确的起点
- [ ] 每个环节都是正向强化
- [ ] 飞轮能够闭环
- [ ] 飞轮越转越快（而非减速）

### 商业检查

- [ ] 飞轮有清晰的商业模式支撑
- [ ] 各环节的数据可监测
- [ ] 飞轮各环节的指标可优化
- [ ] 飞轮有可构建的壁垒

### 竞争检查

- [ ] 竞争对手难以复制整个飞轮
- [ ] 即使复制，也难以复制全部环节
- [ ] 飞轮的先发优势明显

## 飞轮常见问题

### 问题1: 伪飞轮

**特征**: 看似是循环，实际某环节会衰减

**案例**: 补贴驱动的增长 → 停止补贴就停止增长

**诊断**: 检查每个环节是否依赖外部输入

### 问题2: 断点飞轮

**特征**: 某个环节会中断循环

**案例**: 内容多了但用户不增长

**诊断**: 画出完整链条，检查断点

### 问题3: 减速飞轮

**特征**: 初期有效，但逐渐减速

**案例**: 单一获客渠道边际效益递减

**诊断**: 检查是否有规模带来的成本优势
