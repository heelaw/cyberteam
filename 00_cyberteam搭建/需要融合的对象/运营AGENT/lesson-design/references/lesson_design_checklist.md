# 课程设计清单

## 课程设计要素检查

基于五要素设计法，逐项检查你的课程设计：

### 1. 教学目标设计

- [ ] **目标明确性**
  - [ ] 使用行为动词描述（如"能够运用"、"能够分析"）
  - [ ] 避免模糊表述（如"了解"、"掌握"）
  - [ ] 符合SMART原则

- [ ] **目标层次性**
  - [ ] 有核心目标（必须达成）
  - [ ] 有次要目标（争取达成）
  - [ ] 有拓展目标（锦上添花）

- [ ] **目标可检验**
  - [ ] 设计了评估方式
  - [ ] 有明确的完成标准
  - [ ] 可观察学员行为变化

---

### 2. 学习内容设计

- [ ] **内容组织逻辑**
  - [ ] 选择了合适的设计逻辑（□内容消费型 □学习交互型 □行动动机改变型）
  - [ ] 内容模块划分清晰
  - [ ] 模块之间有逻辑递进

- [ ] **内容覆盖度**
  - [ ] 覆盖所有教学目标
  - [ ] 理论与实践结合
  - [ ] 有实际案例支撑

- [ ] **内容难度匹配**
  - [ ] 符合学员基础水平
  - [ ] 有难度梯度设计
  - [ ] 避免跳跃过大

- [ ] **知识完整性**
  - [ ] 核心概念讲解充分
  - [ ] 原理机制说明清晰
  - [ ] 方法步骤可操作

---

### 3. 教学活动设计

- [ ] **活动多样性**
  - [ ] 讲授与互动结合
  - [ ] 个人练习与小组活动结合
  - [ ] 线上与线下结合（如适用）

- [ ] **活动有效性**
  - [ ] 每个活动有明确目的
  - [ ] 活动时长合理
  - [ ] 有操作指南

- [ ] **互动设计**
  - [ ] 设计了提问环节
  - [ ] 有讨论/交流机制
  - [ ] 鼓励学员参与

---

### 4. 学习资源设计

- [ ] **资源完备性**
  - [ ] 提供必要的学习材料
  - [ ] 有参考资料清单
  - [ ] 工具/模板准备充分

- [ ] **资源可用性**
  - [ ] 资源易于获取
  - [ ] 格式便于使用
  - [ ] 质量有保障

- [ ] **资源多样性**
  - [ ] 文字、图片、视频结合
  - [ ] 有多种呈现形式
  - [ ] 适应不同学习偏好

---

### 5. 学习评估设计

- [ ] **评估时机**
  - [ ] 课前有前置测评
  - [ ] 课中有形成性评估
  - [ ] 课后有总结性评估

- [ ] **评估方式**
  - [ ] 知识测试（测验、考试）
  - [ ] 技能考核（实操、项目）
  - [ ] 综合评估（作品、方案）

- [ ] **反馈机制**
  - [ ] 评估结果及时反馈
  - [ ] 有改进建议
  - [ ] 支持再次评估

---

## 学习体验设计检查

### 学习体验曲线

- [ ] **开场设计（第一印象）**
  - [ ] 建立学习期待
  - [ ] 降低学习焦虑
  - [ ] 激发学习兴趣
  - [ ] 说明学习价值

- [ ] **过程设计（心流状态）**
  - [ ] 难度适中（有挑战但可实现）
  - [ ] 节奏紧凑（不拖沓）
  - [ ] 频繁互动（保持参与）
  - [ ] 及时反馈（确认进度）

- [ ] **结尾设计（完课体验）**
  - [ ] 总结回顾核心内容
  - [ ] 强调学习成果
  - [ ] 指明后续路径
  - [ ] 创造分享动机

### 学习难度曲线

- [ ] **难度梯度**
  - [ ] 从易到难渐进
  - [ ] 有坡度但不过陡
  - [ ] 关键节点有支撑

- [ ] **难度平衡**
  - [ ] 有简单内容建立信心
  - [ ] 有挑战内容激发思考
  - [ ] 避免难度忽高忽低

- [ ] **难度调节**
  - [ ] 提供可选的挑战任务
  - [ ] 支持个性化学习路径
  - [ ] 有帮助/提示机制

---

## 在线学习障碍应对

### 障碍1：注意力分散

- [ ] **内容设计**
  - [ ] 单个知识点时长≤15分钟
  - [ ] 核心内容前置
  - [ ] 避免冗余信息

- [ ] **互动设计**
  - [ ] 每5-10分钟一次互动
  - [ ] 多种互动形式
  - [ ] 及时响应反馈

- [ ] **视觉设计**
  - [ ] 页面简洁清晰
  - [ ] 重点突出
  - [ ] 动画适度使用

### 障碍2：学习孤独感

- [ ] **社群设计**
  - [ ] 建立学习社群
  - [ ] 助教/辅导员支持
  - [ ] 学员互动机制

- [ ] **反馈设计**
  - [ ] 作业批改反馈
  - [ ] 答疑机制
  - [ ] 进度确认

- [ ] **同伴设计**
  - [ ] 小组学习
  - [ ] 互评互改
  - [ ] 经验分享

### 障碍3：学习认知负荷

- [ ] **内容拆分**
  - [ ] 知识点碎片化
  - [ ] 多个短课时
  - [ ] 逐步累积

- [ ] **认知卸载**
  - [ ] 提供笔记模板
  - [ ] 关键信息总结
  - [ ] 思维导图

- [ ] **记忆辅助**
  - [ ] 重要信息重复
  - [ ] 联想记忆法
  - [ ] 口诀/公式

### 障障4：学习落地难

- [ ] **行动设计**
  - [ ] 课后作业/练习
  - [ ] 实践项目
  - [ ] 行动计划

- [ ] **工具设计**
  - [ ] 提供操作模板
  - [ ] 检查清单
  - [ ] 决策工具

- [ ] **追踪设计**
  - [ ] 学习进度追踪
  - [ ] 实践打卡
  - [ ] 成果展示

## 课程结构检查

### 单节课程结构

- [ ] **开场（5-10%）**
  - [ ] 引入/破冰
  - [ ] 激发兴趣
  - [ ] 说明目标

- [ ] **主体（70-80%）**
  - [ ] 知识讲解
  - [ ] 案例分析
  - [ ] 实践练习
  - [ ] 互动答疑

- [ ] **收尾（10-15%）**
  - [ ] 总结回顾
  - [ ] 评估反馈
  - [ ] 行动指引

### 整体课程结构

- [ ] **模块划分**
  - [ ] 模块主题明确
  - [ ] 模块数量合理（3-7个）
  - [ ] 模块之间有逻辑递进

- [ ] **课时安排**
  - [ ] 单节时长合理（20-60分钟）
  - [ ] 总时长与内容匹配
  - [ ] 有休息安排（如适用）

- [ ] **难度递进**
  - [ ] 基础→进阶→高级
  - [ ] 理论→实践→应用
  - [ ] 知识→技能→能力

---
