# learning-experience-designer

## 输入

- 课程背景
- 目标人群
- 学习目标
- 约束

# 学习体验设计

## SOP

1. 分学前、学中、学后。
2. 识别障碍。
3. 设计正反馈。
4. 控制难度曲线。
5. 输出体验方案。


## 输出

- 结构化结果
- 下一步建议
