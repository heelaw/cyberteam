# 文案初稿生成器

## 作用

生成可用初稿，不伪装成最终稿。

## 不做的事

- 不替代前置判断。
- 不把缺失信息硬补成内容。

## SOP

1. 确认目标用户、卖点和场景已经定下来。
2. 按选定结构填充模块，不乱序。
3. 先写核心结论，再写理由和证据。
4. 最后补行动号召和风险提示。
5. 把缺口单独列出来，避免假装完整。

## 输出

- 文案初稿
- 可删改段落
- 待确认信息
- 结构说明
