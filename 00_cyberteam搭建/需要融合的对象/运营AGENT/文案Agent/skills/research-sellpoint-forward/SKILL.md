# 正向卖点提炼器

## 作用

把产品事实转成可写入文案的卖点材料。

## 不做的事

- 不直接编造卖点。
- 不把功能、收益和证据混在一起。

## SOP

1. 先列产品事实和可用证据。
2. 按七个方向逐一发散候选卖点。
3. 每个方向保留至少三个候选。
4. 按差异化、感知度和可信度筛选。
5. 输出主卖点、辅助卖点和证据缺口。

## 输出

- 产品画像
- 7方向候选
- 主卖点
- 辅助卖点
- 证据缺口
