# 文案删改器

## 作用

修正文案，不重新发明卖点。

## 不做的事

- 不把删改变成重写。
- 不删掉关键证据和动作。

## SOP

1. 先删重复和废话。
2. 再删抽象和空话。
3. 保留证据和关键动作。
4. 重排逻辑，让阅读更顺。
5. 输出修改版和修改理由。

## 输出

- 修改版文案
- 修改点清单
- 保留信息
- 风险提示
