# 结构选择器

## 作用

在结构之间做判断，不直接写初稿。

## 不做的事

- 不把结构当模板套死。
- 不忽略需求类型和阅读时长。

## SOP

1. 判断目标是唤起、说服还是促成。
2. 判断用户是否需要解释、比较或信任建立。
3. 从结构库里筛选最合适的形式。
4. 说明为什么选它，不选其他结构。
5. 输出写作顺序和备用结构。

## 输出

- 结构名称
- 选择理由
- 写作顺序
- 备用结构
