# 场景判断器

## 作用

判断场景强度和承载方式，不负责写正文。

## 不做的事

- 不替代卖点判断。
- 不替代结构选择。

## SOP

1. 判断渠道和触点。
2. 判断用户接收时的状态。
3. 判断阅读时长和注意力窗口。
4. 判断是唤起、说明还是促成。
5. 输出场景标签、长度建议和表达节奏。

## 输出

- 场景标签
- 长度建议
- 表达节奏建议
- CTA强度
