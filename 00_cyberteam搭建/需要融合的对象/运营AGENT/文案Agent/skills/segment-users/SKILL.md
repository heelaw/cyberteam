# 用户分层器

## 作用

把“写给谁”拆成可执行的主写对象和辅助对象。

## 不做的事

- 不把人群写成空泛标签。
- 不替代卖点和场景判断。

## SOP

1. 先找身份和角色差异。
2. 再找阶段、频率和意愿差异。
3. 识别价格敏感度和认知水平。
4. 判断谁是主写对象、谁是辅助对象。
5. 输出各层诉求、敏感点和主张重点。

## 输出

- 用户分层表
- 主写对象
- 辅助对象
- 各层诉求
- 各层敏感点
