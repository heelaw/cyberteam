# 文案自检清单器

## 作用

交付前复核闭环，不负责新增卖点。

## 不做的事

- 不跳过检查直接交付。
- 不把风险藏起来。

## SOP

1. 检查写给谁。
2. 检查说了什么卖点。
3. 检查场景是否匹配。
4. 检查结构是否完整。
5. 检查动作是否明确。
6. 检查证据和风险是否交代清楚。

## 输出

- 自检结果
- 风险点
- 修改建议
- 是否通过
