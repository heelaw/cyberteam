# 内容定位方法论

## 核心概念定义

内容定位是指为内容型产品确定独特的价值主张和差异化方向，使用户能够清晰感知该内容与其他竞品的区别。核心要素包括：**核心定位**（你是谁）、**价值主张**（用户为何而来）、**差异化点**（你与竞品的区别）、**语调风格**（你如何说话）。

内容定位不是简单的"写什么领域"，而是回答"为什么用户要看你而不是别人"这一根本问题。

## 方法论步骤

### 步骤1：竞品分析
- 列出同领域TOP5内容源
- 分析其内容风格、话题选择、表达方式
- 识别已占据的用户心智位置

### 步骤2：差异化识别
- 列出自身可能的差异化方向
- 评估每个方向的可行性（团队能力、资源匹配）
- 选择最能持续坚持的方向

### 步骤3：具体化体现
- 将抽象定位转化为可执行的内容规范
- 明确话题边界（写什么+不写什么）
- 设定风格指南（语调、篇幅、呈现形式）

### 步骤4：规范化输出
- 制定《内容风格指南》
- 设定内容禁区（绝对不碰的话题/角度）
- 建立内容审核标准

### 步骤5：持续执行
- 保持发布频率和内容质量一致性
- 定期回顾数据检验定位有效性
- 根据反馈迭代调整

## 关键成功因素

1. **差异化可感知**：用户能在3秒内说出你的特点
2. **定位可持续**：能坚持产出至少100期内容
3. **用户有共鸣**：定位解决的是真实用户痛点
4. **竞争有空间**：该定位在市场中尚未被占据

## 常见误区

- **定位过宽**：试图服务所有用户，结果无人记住
- **定位模糊**：差异化不明确，与竞品高度重叠
- **定位漂移**：内容风格前后不一致
- **定位自嗨**：基于团队偏好而非用户需求
- **定位空泛**：口号式定位无法指导实际内容生产
