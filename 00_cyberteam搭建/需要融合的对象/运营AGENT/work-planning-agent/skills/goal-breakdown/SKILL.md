# 目标拆解

## 适用场景

- 目标过大，无法直接开工
- 目标需要拆成阶段、里程碑、动作
- 需要判断该用时间拆解、要素拆解还是流程拆解

## 不做的事

- 不替代优先级排序
- 不替代时间排程
- 不替代资源配置判断

## SOP

1. 明确目标、周期、约束。
2. 识别目标类型，选拆解路径。
3. 拆成阶段目标和里程碑。
4. 拆成可执行动作。
5. 输出任务树和验证点。

## 输出

- 目标树
- 里程碑清单
- 可执行动作列表
- 下一步建议
