# 时间规划

## 适用场景

- 任务顺序已经明确，但不知道怎么排进日程
- 需要判断哪些时间段适合做高价值任务
- 需要给计划留出缓冲和机动空间

## 不做的事

- 不替代目标拆解
- 不替代优先级排序
- 不替代进度管理

## SOP

1. 确定时间窗口。
2. 识别固定占用。
3. 估算耗时。
4. 预留缓冲。
5. 输出日程表。

## 输出

- 时间块安排
- 缓冲安排
- 日程表
- 下一步建议
