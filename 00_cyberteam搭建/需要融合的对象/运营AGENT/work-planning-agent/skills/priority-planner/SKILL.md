# 优先级排序

## 适用场景

- 任务很多，先做什么不清楚
- 需要在有限资源下做取舍
- 需要解释为什么这件事排前面

## 不做的事

- 不替代目标拆解
- 不替代时间规划
- 不替代执行过程中的动态调整

## SOP

1. 列出待办。
2. 判断价值。
3. 判断紧迫性。
4. 判断依赖关系。
5. 输出排序结果和取舍理由。

## 输出

- 排序后的待办列表
- 每项的排序理由
- 下一步建议
