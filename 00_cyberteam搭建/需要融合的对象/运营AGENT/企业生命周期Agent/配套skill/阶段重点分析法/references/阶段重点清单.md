# 各阶段重点清单

## 初创期（0-1）

### 核心任务
- 找到PMF（产品市场契合）

### 必须做
- [ ] 找到核心用户
- [ ] 验证真实需求
- [ ] 验证付费意愿
- [ ] 快速迭代产品

### 不该做
- [ ] 大量投放广告
- [ ] 过早扩张团队
- [ ] 追求收入规模

---

## 成长期（1-10）

### 核心任务
- 规模化增长

### 必须做
- [ ] 优化获客渠道
- [ ] 提升转化效率
- [ ] 扩张团队能力
- [ ] 储备融资

### 不该做
- [ ] 忽视单位经济
- [ ] 过早追求盈利
- [ ] 盲目多元化

---

## 成熟期（10-100）

### 核心任务
- 效率与盈利

### 必须做
- [ ] 优化成本结构
- [ ] 提升客户价值
- [ ] 探索新增长点
- [ ] 组织能力升级

### 不该做
- [ ] 激进扩张
- [ ] 忽视新趋势
- [ ] 躺平不作为

---

## 转型期

### 核心任务
- 找到新增长

### 必须做
- [ ] 评估新机会
- [ ] 小步快跑验证
- [ ] 调整组织结构
- [ ] 争取转型资源

### 不该做
- [ ] 盲目all in新业务
- [ ] 放弃现有现金流
- [ ] 激进裁员
