## 使用建议

1. **定期更新**：每周/每拆解完一个案例就更新
2. **统一格式**：保持表格结构一致，便于检索
3. **标注来源**：记录案例来源，便于追溯
4. **定期回顾**：每月/每季度回顾一次，加深印象
5. **实际应用**：策划活动时先查阅弹药库
