# 赞助 ECC

ECC 作为跨 Claude Code、Cursor、OpenCode 和 Codex 应用程序/CLI 的开源代理利用性能系统进行维护。

## 为什么赞助

直接赞助资金：

- 更快的错误修复和发布周期
- 跨线束的跨平台奇偶校验工作
- 对社区免费的公共文档、技能和可靠性工具

## 赞助级别

这些都是实际的起点，可以根据合作范围进行调整。

|等级 |价格|最适合 |包括 |
|------|------|----------|----------|
|试点合作伙伴| $200/月 |首次赞助商参与|每月指标更新、路线图预览、优先维护者反馈 |
|成长伙伴| 500 美元/月 |团队积极采用 ECC |试点福利+每月办公时间同步+工作流程整合指导|
|战略合作伙伴| $1,000+/月 |平台/生态系统合作伙伴关系 |增长收益 + 协调的发布支持 + 更深入的维护者协作 |

## 赞助商报告

每月共享的指标可以包括：

- npm 下载（`ecc-universal`、`ecc-agentshield`）
- 存储库采用（明星、分叉、贡献者）
- GitHub App安装趋势
- 发布节奏和可靠性里程碑

有关确切的命令片段和可重复的拉取过程，请参阅 [`docs/business/metrics-and-sponsorship.md`](docs/business/metrics-and-sponsorship.md)。

## 期望和范围

- 赞助支持维护和加速；它不转移项目所有权。
- 根据赞助商级别、生态系统影响和维护风险对功能请求进行优先级排序。
- 安全性和可靠性修复优先于全新功能。
