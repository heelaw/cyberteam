# PHP 编码风格

> 该文件使用 PHP 特定内容扩展了通用编码风格规则。

## 标准

- 遵循 **PSR-12** 格式和命名约定。
- 在应用程序代码中首选“declare(strict_types=1);”。
- 在新代码允许的地方使用标量类型提示、返回类型和类型化属性。

## 不变性

- 对于跨服务边界的数据，首选不可变的 DTO 和值对象。
- 尽可能对请求/响应有效负载使用“只读”属性或不可变构造函数。
- 为简单地图保留数组；将关键业务结构提升为显式类。

## 格式化

- 使用 **PHP-CS-Fixer** 或 **Laravel Pint** 进行格式化。
- 使用 **PHPStan** 或 **Psalm** 进行静态分析。
