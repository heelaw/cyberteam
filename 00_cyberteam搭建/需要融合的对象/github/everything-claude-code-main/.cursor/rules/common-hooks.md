# 钩子系统

## 钩子类型

- **PreToolUse**：工具执行之前（验证、参数修改）
- **PostToolUse**：工具执行后（自动格式化、检查）
- **停止**：会话结束时（最终验证）

## 自动接受权限

谨慎使用：
- 支持可信、明确的计划
- 禁用探索性工作
- 切勿使用危险的跳过权限标志
- 在`~/.claude.json`中配置`allowedTools`

## TodoWrite 最佳实践

使用 TodoWrite 工具可以：
- 跟踪多步骤任务的进度
- 验证对说明的理解
- 启用实时转向
- 显示细化的实施步骤

待办事项列表揭示：
- 无序步骤
- 丢失物品
- 额外不必要的物品
- 粒度错误
- 误解的要求
