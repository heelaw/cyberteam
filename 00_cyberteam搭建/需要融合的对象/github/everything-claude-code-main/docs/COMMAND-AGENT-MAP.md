## 命令引用的技能

- **持续学习**、**持续学习-v2**：`/learn`、`/learn-eval`、`/instinct-*`、`/evolve`、`/promote`、`/projects`
- **验证循环**：`/检查点`，`/验证`
- **评估线束**：`/eval`
- **安全扫描**：`/安全扫描`（运行 AgentShield）
- **战略紧凑**：建议在压实点（钩子）

## 如何使用此地图

- **可发现性：** 查找哪个命令触发哪个代理（例如“使用`/code-review`作为代码审查器”）。
- **重构：** 重命名或删除代理时，搜索此文档和命令文件以获取参考。
- **CI/docs:** 目录脚本 (`node script/ci/catalog.js`) 输出代理/命令/技能计数；这张地图通过指挥-特工关系来补充它。
