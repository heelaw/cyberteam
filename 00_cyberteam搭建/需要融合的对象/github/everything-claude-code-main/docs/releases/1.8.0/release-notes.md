# ECC v1.8.0 发行说明

## 定位

ECC v1.8.0 将该项目定位为代理利用性能系统，而不仅仅是配置包。

## 主要改进

- 稳定的挂钩和生命周期行为。
- 扩展了评估和循环操作界面。
- 升级 NanoClaw 以供操作使用。
- 改进了跨线束奇偶校验（Claude Code、Cursor、OpenCode、Codex）。

## 升级焦点

1. 验证您环境中的挂钩配置文件默认值。
2. 运行“/harness-audit”来确定项目的基线。
3. 使用“/quality-gate”和更新的评估工作流程来强制一致性。
4. 查看引用的生态系统的归属和许可说明：[reference-attribution.md](./reference-attribution.md)。
5. 对于合作伙伴/赞助商光学，使用实时分发指标和谈话要点：[../business/metrics-and-sponsorship.md](../../business/metrics-and-sponsorship.md)。
