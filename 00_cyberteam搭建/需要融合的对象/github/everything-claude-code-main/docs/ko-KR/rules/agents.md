#韩国

## 사용 实例

`~/.claude/agents/`에위치:

| 에전트 | 용도 | 사용 시점 |
|---------|------|----------|
|策划师| 구현 계획 | 복잡한 기능, 리팩토링 |
|建筑师 | 시스템 설계 | 아키텍처 의사결정 |
| TDD 指南 | 테스트 주도 개발 | 새 기능, 버그 수정 |
|代码审查员 | 코드 리뷰 | 코드 작성 후 |
|安全审查员 | 보안 분석 | 커밋 전 |
|构建错误解析器 | 빌드 에러 수정 | 빌드 실패 시 |
| e2e-跑步者 | E2E 테스팅 | 핵심 사용자 흐름 |
|数据库审阅者 | 스키마/쿼리 리뷰 | 스키마 설계, 쿼리 최적화 |
|去评论家 |去 코드 리뷰 | Go 코드 작성 또는 수정 후 |
|去构建解析器 |去吧 빌드 에러 수정 | `go build` 또는 `go vet` 실패 시 |
|重构清理器 | 사용하지 않는 코드 정리 | 코드 유지보수 |
|文档更新程序 | 문서 관리 | 문서 업데֊ |

## 즉시 에 Been 사용

사용자 프롬프트 불필요:
1. 복잡한 기능 요청 - **策划者** 에 Been 사용
2. 코드 작성/수정 직후 - **代码审阅者** 에 Shoulder전트 사용
3. 버그 수정 또는 새 기능 - **tdd-guide** 에 Been 사용
4. 아키텍처 의사결정 - **建筑师** 에 Been 사용

## 병렬 任务 실행

任务任务：```markdown
# 좋음: 병렬 실행
3개 에이전트를 병렬로 실행:
1. 에이전트 1: 인증 모듈 보안 분석
2. 에이전트 2: 캐시 시스템 성능 리뷰
3. 에이전트 3: 유틸리티 타입 검사

# 나쁨: 불필요하게 순차 실행
먼저 에이전트 1, 그다음 에이전트 2, 그다음 에이전트 3
```## 다중 관점 분석

重要信息：
- 사실 검증 리뷰어
- 시니어 엔지니어
- 보안 전문ν
- 일관성 검토자
- 중복 검사자
