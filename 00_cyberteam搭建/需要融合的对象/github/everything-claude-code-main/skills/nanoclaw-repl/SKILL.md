# 纳米爪 REPL

在运行或扩展 `scripts/claw.js` 时使用此技能。

## 能力

- 持久的降价支持会话
- 使用`/model`进行模型切换
- 使用“/load”动态加载技能
- 使用 `/branch` 进行会话分支
- 使用“/search”进行跨会话搜索
- 使用“/compact”进行历史压缩
- 使用 `/export` 导出到 md/json/txt
- 使用“/metrics”的会话指标

## 操作指导

1. 保持会议以任务为中心。
2. 在高风险变更之前进行分支。
3. 在重大里程碑之后进行紧凑。
4. 在共享或存档之前导出。

## 扩展规则

- 保持零外部运行时依赖
- 保留降价作为数据库的兼容性
- 保持命令处理程序的确定性和本地性
