# 市场研究

进行支持决策的研究，而不是研究剧场。

## 何时激活

- 研究市场、类别、公司、投资者或技术趋势
- 建立 TAM/SAM/SOM 估算
- 比较竞争对手或相邻产品
- 在外展之前准备投资者档案
- 在构建、资助或进入市场之前对论文进行压力测试

## 研究标准

1. 每一个重要的主张都需要有来源。
2. 优先选择最新数据并剔除陈旧数据。
3. 包括逆向证据和不利案例。
4. 将调查结果转化为决策，而不仅仅是总结。
5. 清楚地区分事实、推论和建议。

## 常见研究模式

### 投资者/基金尽职调查
收集：
- 基金规模、阶段和典型支票规模
- 相关投资组合公司
- 公开论文和近期活动
- 基金适合或不适合的原因
- 任何明显的危险信号或不匹配

### 竞争分析
收集：
- 产品真实性，而非营销文案
- 资金和投资者历史（如果公开）
- 牵引力指标（如果公开）
- 分销和定价线索
- 优势、劣势和定位差距

### 市场规模
用途：
- 根据报告或公共数据集进行自上而下的估计
- 根据现实的客户获取假设进行自下而上的健全性检查
- 每一次逻辑飞跃的明确假设

### 技术/供应商研究
收集：
- 它是如何运作的
- 权衡和采用信号
- 集成复杂性
- 锁定、安全、合规和运营风险

## 输出格式

默认结构：
1. 执行摘要
2. 主要发现
3. 影响
4. 风险和注意事项
5、推荐
6. 来源

## 质量门

发货前：
- 所有数字均来源于或标记为估计值
- 旧数据被标记
- 建议源自证据
- 包括风险和反驳
- 输出使决策更容易
