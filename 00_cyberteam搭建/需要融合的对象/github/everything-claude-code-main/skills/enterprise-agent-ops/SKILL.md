# 企业代理操作

对于需要单个 CLI 会话之外的操作控制的云托管或持续运行的代理系统，请使用此技能。

## 操作域

1.运行时生命周期（启动、暂停、停止、重启）
2.可观察性（日志、指标、痕迹）
3.安全控制（范围、权限、终止开关）
4.变更管理（推出、回滚、审核）

## 基线控制

- 不可变的部署工件
- 最低权限凭证
- 环境级秘密注入
- 硬超时和重试预算
- 高风险行为的审核日志

## 要跟踪的指标

- 成功率
- 每个任务的平均重试次数
- 恢复时间
- 每个成功任务的成本
- 故障类别分布

## 事件模式

当故障激增时：
1. 冻结新的推出
2. 捕捉代表性痕迹
3. 隔离故障路由
4. 安全变更最小的补丁
5.运行回归+安全检查
6. 逐步恢复

## 部署集成

该技能与：
- PM2 工作流程
- 系统服务
- 容器编排器
- CI/CD 门
