# 投资者资料

构建一致、可信且易于辩护的面向投资者的材料。

## 何时激活

- 创建或修改宣传材料
- 撰写投资者备忘录或一页纸
- 建立财务模型、里程碑计划或资金使用表
- 回答加速器或孵化器申请问题
- 围绕一个事实来源调整多个筹款文档

## 黄金法则

所有投资者材料必须相互一致。

在编写之前创建或确认单一事实来源：
- 牵引力指标
- 定价和收入假设
- 提高尺寸和仪器
- 资金使用
- 团队简介和头衔
- 里程碑和时间表

如果出现冲突的数字，请在起草之前停止并解决它们。

## 核心工作流程

1. 盘点规范事实
2. 识别缺失的假设
3.选择资产类型
4. 起草具有明确逻辑的资产
5. 根据事实来源交叉核对每个数字

## 资产指引

### 宣传材料
推荐流程：
1.公司+楔子
2、问题
3、解决方案
4.产品/演示
5、市场
6、商业模式
7.牵引力
8. 团队
9. 竞争/差异化
10.问
11. 资金使用/里程碑
12. 附录

如果用户想要一个网络原生的平台，请将此技能与“前端幻灯片”配对。

### 单页/备忘录
- 用简洁的句子说明公司的业务
- 现在说明原因
- 尽早包括牵引力和证明点
- 使询问准确
- 使声明易于验证

### 财务模型
包括：
- 明确的假设
- 有用时的熊市/基本情况/牛市情况
- 清晰的逐层收益逻辑
- 与里程碑相关的支出
- 敏感性分析，其中决策取决于假设

### 加速器应用
- 回答确切的问题
- 优先考虑牵引力、洞察力和团队优势
- 避免浮肿
- 保持内部指标与牌组和模型一致

## 要避免的危险信号

- 无法证实的说法
- 没有假设的模糊市场规模
- 团队角色或头衔不一致
- 收入数学求和不清晰
- 在假设脆弱的情况下夸大确定性

## 质量门

发货前：
- 每个数字都符合当前的事实来源
- 资金使用和收入层正确相加
- 假设是可见的，而不是被隐藏的
- 故事清晰，没有炒作语言
- 最终资产在合作伙伴会议上是可以辩护的
