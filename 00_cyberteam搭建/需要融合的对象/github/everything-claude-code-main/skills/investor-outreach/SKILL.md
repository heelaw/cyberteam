# 投资者外展

撰写简短、个性化且易于采取行动的投资者沟通信息。

## 何时激活

- 给投资者写一封冷冰冰的电子邮件
- 起草热情的介绍请求
- 会议结束后或没有回复时发送后续信息
- 在过程中撰写投资者最新动态
- 根据基金主题或合作伙伴契合度定制推广活动

## 核心规则

1. 个性化每条出站消息。
2. 保持询价低摩擦。
3. 使用证据，而不是形容词。
4. 保持简洁。
5. 切勿向任何投资者发送可能发送给任何投资者的通用副本。

## 冷邮件结构

1. 主题行：简短而具体
2.开场白：为什么这位投资者特别
3. 推介：公司做什么，为什么现在，什么证据很重要
4.问：具体的下一步
5. 签字：姓名、角色、一名可信度锚（如果需要）

## 个性化来源

参考以下一项或多项：
- 相关投资组合公司
- 公开论文、演讲、帖子或文章
- 相互联系
- 符合投资者关注重点的明确市场或产品

如果缺少该上下文，请询问或声明草稿是等待个性化的模板。

## 后续节奏

默认值：
- 第 0 天：初始出境
- 第 4-5 天：使用一个新数据点进行简短跟进
- 第 10-12 天：最后的后续行动，干净利落地结束

之后不要继续轻推，除非用户想要更长的序列。

## 热情的介绍请求

让连接器的生活变得轻松：
- 解释为什么介绍适合
- 包括可转发的简介
- 将可转发的简介控制在 100 个字以内

## 会后更新

包括：
- 讨论的具体事情
- 承诺的答案或更新
- 一个新的证据点（如果有）
- 下一步

## 质量门

发货前：
- 消息是个性化的
- 询问是明确的
- 没有任何废话或乞求语言
- 证明点是具体的
- 字数保持紧张
