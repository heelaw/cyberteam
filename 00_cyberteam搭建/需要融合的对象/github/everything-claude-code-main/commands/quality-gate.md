# 质量门命令

根据文件或项目范围的需要运行 ECC 质量管道。

## 用法

`/quality-gate [路径|.] [--fix] [--strict]`

- 默认目标：当前目录（`.`）
- `--fix`：允许在配置时自动格式化/修复
- `--strict`：在支持的情况下警告失败

## 管道

1. 检测目标语言/工具。
2. 运行格式化程序检查。
3. 在可用时运行 lint/类型检查。
4. 制定一份简明的补救措施清单。

## 注释

该命令反映了钩子行为，但由操作员调用。

## 参数

$参数：
- `[path|.]` 可选目标路径
- `--fix` 可选
- `--strict` 可选
