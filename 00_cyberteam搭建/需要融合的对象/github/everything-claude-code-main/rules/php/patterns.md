# PHP 模式

> 此文件使用 PHP 特定内容扩展了 [common/patterns.md](../common/patterns.md)。

## 瘦控制器、显式服务

- 让控制器专注于传输：身份验证、验证、序列化、状态代码。
- 将业务规则移至应用程序/域服务中，无需 HTTP 引导即可轻松测试。

## DTO 和值对象

- 将请求、命令和外部 API 有效负载的形状繁重的关联数组替换为 DTO。
- 使用货币、标识符、日期范围和其他受限概念的值对象。

## 依赖注入

- 依赖于接口或狭窄的服务契约，而不是框架全局变量。
- 通过构造函数传递协作者，以便无需查找服务定位器即可测试服务。

## 边界

- 当模型层不仅仅做持久性工作时，将 ORM 模型与领域决策隔离。
- 将第三方 SDK 封装在小型适配器后面，以便代码库的其余部分取决于您的合同，而不是他们的合同。
