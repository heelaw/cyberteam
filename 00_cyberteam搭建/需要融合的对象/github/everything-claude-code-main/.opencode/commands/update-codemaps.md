# 更新代码映射命令

更新代码映射以反映当前代码库结构：$ARGUMENTS

## 你的任务

在 `docs/CODEMAPS/` 目录中生成或更新代码映射：

1. **分析代码库结构**
2. **生成组件图**
3. **文档关系**
4. **更新导航指南**

## 代码映射类型

### 架构图```
docs/CODEMAPS/ARCHITECTURE.md
```- 高级系统概述
- 组件关系
- 数据流程图

### 模块图```
docs/CODEMAPS/MODULES.md
```- 模块描述
- 公共API
- 依赖关系

### 文件映射```
docs/CODEMAPS/FILES.md
```- 目录结构
- 文件用途
- 关键文件

## 代码图格式

### [模块名称]

**目的**：【简要说明】

**位置**：`src/[路径]/`

**关键文件**：
- `file1.ts` - [目的]
- `file2.ts` - [目的]

**依赖关系**：
- [模块A]
- [模块B]

**出口**：
- `函数名()` - [描述]
- `类名` - [描述]

**使用示例**：```typescript
import { functionName } from '@/module'
```## 生成过程

1.扫描目录结构
2. 解析导入/导出
3. 构建依赖图
4.生成markdown图
5. 验证链接

---

**提示**：添加新模块或重大重构时保持代码图更新。
