# 验证命令

运行验证循环来验证实施：$ARGUMENTS

## 你的任务

执行全面验证：

1. **类型检查**：`npx tsc --noEmit`
2. **Lint**：`npm run lint`
3. **单元测试**：`npm test`
4. **集成测试**：`npm run test:integration`（如果可用）
5. **构建**：`npm run build`
6. **覆盖率检查**：验证 80% 以上的覆盖率
