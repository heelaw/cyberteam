# 学习命令

从当前会话中提取模式、学习内容和可重用的见解：$ARGUMENTS

## 你的任务

分析对话和代码变化来提取：

1. **发现的模式** - 重复出现的解决方案或方法
2. **应用的最佳实践** - 效果良好的技术
3. **要避免的错误** - 遇到的问题和解决方案
4. **可重用的片段** - 值得保存的代码模式

## 输出格式

### 发现的模式

**图案：[名称]**
- 上下文：何时使用此模式
- 实施：如何应用
- 示例：代码片段

### 应用最佳实践

1. [执业名称]
   - 为什么它有效
   - 何时申请

### 要避免的错误

1.【错误描述】
   - 出了什么问题
   - 如何预防

### 建议的技能更新

如果模式很重要，建议更新：
- `技能/编码标准/SKILL.md`
- `技能/[域]/SKILL.md`
- `规则/[类别].md`

{
  "trigger": "[situation that triggers this learning]",
  "action": "[what to do]",
  "confidence": 0.7,

}
```---
