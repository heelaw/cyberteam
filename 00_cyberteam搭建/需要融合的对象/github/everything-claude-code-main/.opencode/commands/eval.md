# 评估命令

根据验收标准评估实施情况：$ARGUMENTS

## 你的任务

运行结构化评估以验证实施是否满足要求。

## 评估框架

### 分级机类型

1. **二元分级机** - 通过/失败
   - 有效果吗？是/否
   - 适合：功能完成、错误修复

2. **标量分级机** - 分数 0-100
   - 效果如何？
   - 适合：性能、质量指标

3. **Rubric Grader** - 类别分数
   - 多维度评估
   - 适合：全面复习

## 评估流程

### 第 1 步：定义标准```
Acceptance Criteria:
1. [Criterion 1] - [weight]
2. [Criterion 2] - [weight]
3. [Criterion 3] - [weight]
```### 第 2 步：运行测试

对于每个标准：
- 执行相关测试
- 收集证据
- 得分结果

### 第 3 步：计算分数```
Final Score = Σ (criterion_score × weight) / total_weight
```### 第 4 步：报告
