# 循环启动命令

使用安全默认值启动托管自主循环模式。

## 用法

`/loop-start [模式] [--模式安全|快速]`

- `模式`：`顺序`、`连续-pr`、`rfc-dag`、`无限`
-`--模式`：
  - `safe`（默认）：严格的质量门和检查点
  - “快速”：减少速度门

## 流量

1. 确认存储库状态和分支策略。
2. 选择循环模式和模型层策略。
3. 启用所选模式所需的挂钩/配置文件。
4. 创建循环计划并在`.claude/plans/`下编写Runbook。
5. 打印命令以启动和监视循环。

## 必要的安全检查

- 在第一次循环迭代之前验证测试是否通过。
- 确保“ECC_HOOK_PROFILE”未全局禁用。
- 确保循环具有明确的停止条件。

## 参数

$参数：
- `<pattern>` 可选（`顺序|连续-pr|rfc-dag|无限`）
- `--mode safe|fast` 可选
