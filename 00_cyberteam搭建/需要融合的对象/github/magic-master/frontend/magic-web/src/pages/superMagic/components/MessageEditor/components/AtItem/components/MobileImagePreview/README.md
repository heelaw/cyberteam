# Mobile Image Preview Component

基于 Figma 设计稿实现的移动端全屏图片预览组件，支持函数式调用。使用原生样式实现，无依赖外部Modal组件，支持完整的触摸手势交互。

## 特性

- ✅ 基于 Figma 设计稿的移动端样式
- ✅ 全屏黑色背景展示，支持透明度渐变
- ✅ 白色圆形关闭按钮，支持点击和触摸
- ✅ 支持函数式调用，无需手动管理组件状态
- ✅ 仅在移动端显示，桌面端自动隐藏
- ✅ 背景点击关闭功能
- ✅ 响应式图片适配
- ✅ 原生样式实现，无Modal依赖
- ✅ 精美的入场出场动画效果
- ✅ 自动滚动锁定和ESC键支持
- ✅ 背景模糊效果和透明度渐变
- ✅ **双指缩放功能 (0.5x - 5x)**
- ✅ **拖拽移动支持**
- ✅ **双击缩放切换**
- ✅ **智能边界约束**
- ✅ **实时缩放比例显示**

## 🤏 触摸手势功能

### 双指缩放（Pinch Zoom）
- **操作方式**：两根手指捏合/分开
- **缩放范围**：0.5x 到 5x
- **缩放中心**：跟随双指中心点
- **实时反馈**：显示缩放百分比指示器
- **平滑操作**：无延迟响应，流畅缩放

### 拖拽移动（Pan）
- **触发条件**：仅在图片放大时（缩放比例 > 1x）
- **操作方式**：单指拖拽移动图片
- **边界约束**：自动限制在合理范围内
- **智能回弹**：触摸结束后自动调整到边界内

### 双击缩放（Double Tap Zoom）
- **操作方式**：快速双击图片
- **识别时间**：300ms内的连续点击
- **缩放切换**：在1x和2.5x之间自动切换
- **平滑动画**：0.3秒缓动动画过渡

### 智能交互
- **背景关闭**：仅在1x缩放时点击背景关闭
- **缩放保护**：放大状态下点击背景不关闭预览
- **手势优先级**：双指操作优先于单指操作
- **动画协调**：入场动画完成后才启用手势交互

## 动画效果详解

### 🎬 入场动画 (0.3秒)
1. **背景层**：
   - 透明度：0 → 1
   - 背景色：透明 → rgba(0,0,0,0.95)
   - 背景模糊：0px → 8px
   - 动画时长：0.25秒
   - 动画曲线：cubic-bezier(0.4, 0, 0.2, 1)

2. **图片**：
   - 缩放：0.7 → 1.0
   - 位移：向上30px → 0px
   - 透明度：0 → 1
   - 模糊：4px → 0px
   - 延迟：0.05秒
   - 动画时长：0.3秒
   - 动画曲线：cubic-bezier(0.34, 1.56, 0.64, 1) (弹性效果)

3. **关闭按钮**：
   - 缩放：0.5 → 1.0
   - 位移：向上50px → 0px
   - 透明度：0 → 1
   - 背景：透明 → 半透明白色
   - 边框：透明 → 白色边框
   - 图标旋转：0° → 180°
   - 延迟：0.15秒
   - 动画时长：0.35秒
   - 动画曲线：cubic-bezier(0.34, 1.56, 0.64, 1) (弹性效果)

4. **交互启用**：
   - 延迟：0.35秒后启用触摸手势
   - 确保入场动画完成后才能操作

### 🎬 出场动画 (0.25秒)
- 所有元素反向播放入场动画
- 背景透明度和模糊同步消失
- 图片缩小并向下移动，同时添加模糊
- 关闭按钮缩小并向下移动
- 自动重置缩放状态到1x

### 🎯 交互动画
- **按钮悬停**：缩放1.1倍，背景变亮
- **按钮按下**：缩放0.95倍，背景更亮
- **图标旋转**：显示时180°旋转动画 (0.2秒)
- **缩放指示器**：操作时显示1秒后淡出 (0.2秒)
- **双击缩放**：0.2秒缓动动画
- **边界回弹**：0.2秒平滑动画

## 使用方法

### 1. 函数式调用（推荐）

```typescript
import { showMobileImagePreview, hideMobileImagePreview } from '@/opensource/pages/superMagic/components/MessageEditor/components/AtItem/components'

// 显示图片预览
const handleImageClick = () => {
  showMobileImagePreview({
    src: 'https://example.com/image.jpg',
    alt: 'Image description'
  })
}

// 手动关闭预览（可选，用户可以通过关闭按钮或点击背景关闭）
const handleClose = () => {
  hideMobileImagePreview()
}
```

### 2. 组件方式使用

```tsx
import { MobileImagePreview } from '@/opensource/pages/superMagic/components/MessageEditor/components/AtItem/components'

function YourComponent() {
  const [visible, setVisible] = useState(false)
  const [imageSrc, setImageSrc] = useState('')

  return (
    <>
      <img 
        src={imageSrc} 
        onClick={() => setVisible(true)} 
      />
      
      <MobileImagePreview
        src={imageSrc}
        alt="Preview image"
        visible={visible}
        onClose={() => setVisible(false)}
      />
    </>
  )
}
```

## 设计规范

组件严格按照 Figma 设计稿实现：

- **背景**：黑色半透明 (rgba(0,0,0,0.95))
- **背景模糊**：8px 高斯模糊
- **图片尺寸**：最大 269.707px × 480px (移动端)
- **缩放范围**：0.5x 到 5x，支持无级缩放
- **关闭按钮**：
  - 尺寸：40px × 40px
  - 位置：底部居中偏右
  - 样式：白色边框圆形，半透明背景
  - 图标：白色 X 关闭图标，180°旋转动画
- **缩放指示器**：
  - 位置：顶部居中
  - 样式：黑色半透明背景，白色文字
  - 显示：操作时出现，1秒后自动消失

## 技术实现

- **原生样式实现**：使用 `position: fixed` 全屏覆盖，无需依赖 Ant Design Modal
- **Portal渲染**：使用 `createPortal` 渲染到 document.body，确保正确的层级管理
- **高级动画效果**：
  - 弹性动画曲线：cubic-bezier(0.34, 1.56, 0.64, 1)
  - 缓入缓出：cubic-bezier(0.4, 0, 0.2, 1)
  - 多层级动画延迟
  - 背景模糊和透明度渐变
- **触摸手势引擎**：
  - 原生Touch Events处理
  - 实时计算双指距离和中心点
  - GPU加速的transform变换
  - 智能边界检测和约束
  - 防抖动和误触优化
- **性能优化**：
  - 直接操作CSS transform，避免重绘
  - 使用requestAnimationFrame优化动画
  - 事件节流和防抖处理
  - 条件渲染和内存管理
- **滚动锁定**：预览时自动锁定页面滚动
- **键盘支持**：ESC键关闭预览
- **响应式设计**：适配不同移动设备尺寸
- **TypeScript类型安全**：完整的类型定义

## 交互功能

1. **打开预览**：点击图片或调用函数式API
2. **关闭预览**：
   - 点击关闭按钮
   - 点击背景区域（仅在1x缩放时）
   - 按ESC键（键盘设备）
3. **缩放操作**：
   - 双指捏合：实时缩放 (0.5x - 5x)
   - 双击：在1x和2.5x之间切换
   - 缩放指示器：显示当前百分比
4. **移动操作**：
   - 单指拖拽：仅在放大状态下可用
   - 智能边界：自动约束在合理范围
   - 回弹动画：触摸结束后自动调整
5. **动画效果**：
   - 入场：背景渐入 + 图片弹性放大 + 按钮延迟出现
   - 退出：所有元素同步淡出缩小 + 重置缩放
   - 按钮交互：悬停放大 + 按下缩小
   - 手势动画：平滑的缩放和移动响应
6. **滚动锁定**：预览时防止页面滚动

## 注意事项

1. 需要在应用根组件中添加 `MobileImagePreviewProvider`（已自动添加）
2. 仅在移动端生效，桌面端不显示
3. 使用 `createPortal` 渲染，确保在所有内容之上显示
4. 自动处理图片加载状态和错误
5. 支持动态视口高度（100dvh）以适配移动端地址栏变化
6. 动画时长：入场0.3秒，出场0.25秒，确保快速响应的用户体验
7. 手势交互在入场动画完成后启用，避免冲突
8. 缩放状态下的背景点击不会关闭预览，提供更好的用户体验
9. 图片切换时自动重置缩放状态，避免状态混乱
