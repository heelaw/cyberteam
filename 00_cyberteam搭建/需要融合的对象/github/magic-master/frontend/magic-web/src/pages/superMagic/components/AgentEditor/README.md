# AgentEditor 组件

## 功能说明

AgentEditor 是一个基于 TipTap 的富文本编辑器组件，支持以下功能：

### 斜杠菜单功能

按下 `/` 键可以打开斜杠菜单，支持快速插入以下格式：

- **Text** (📝) - 普通文本段落
- **Heading 1** (H1) - 一级标题  
- **Heading 2** (H2) - 二级标题
- **Heading 3** (H3) - 三级标题
- **Bullet List** (•) - 无序列表
- **Numbered List** (1.) - 有序列表
- **Quote** (") - 引用块
- **Code Block** (<>) - 代码块
- **Divider** (—) - 分割线

### 使用方法

1. 在编辑器中输入 `/` 字符
2. 斜杠菜单会自动弹出
3. 使用上下箭头键选择格式
4. 按 Enter 键插入选中的格式
5. 或者直接点击菜单项

### 搜索功能

在斜杠菜单中，你可以继续输入来过滤菜单项：
- `/head` - 显示所有标题选项
- `/list` - 显示列表选项
- `/code` - 显示代码块选项

### 键盘快捷键

- `↑` / `↓` - 在菜单中导航
- `Enter` - 选择当前项
- `Escape` - 关闭菜单

## 技术实现

- 基于 `@tiptap/suggestion` 扩展
- 支持键盘和鼠标交互
- 智能定位系统，防止菜单超出视口
- 优化的滚动条样式，避免双滚动条问题
- 流畅的动画效果和视觉反馈
- 支持亮色和暗色主题
- 响应式设计，适配不同屏幕尺寸
