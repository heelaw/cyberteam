## Article 2: Definitions

### 2.1 AI-Generated Content
Content created or synthesized using artificial intelligence technology, including text, images, audio, video, virtual scenes, and other information.

### 2.2 Watermark-Free Content
AI-generated content from which explicit identifiers (such as text prompts, graphic logos, audio cues, etc.) have been removed, while implicit identifiers (identification information in file metadata) remain.

### 2.3 Explicit Identifier
An identifier added to generated or synthesized content or interactive interface, presented in text, sound, graphics, or other forms that are readily perceptible to users.

### 2.4 Implicit Identifier
An identifier added to the file data of generated or synthesized content through technical means, not readily perceptible to users, including attributes of the generated or synthesized content, service provider name or code, content number, and other production elements.
