## 第二条 定义与说明

### 2.1 AI生成内容
指利用人工智能技术生成、合成的文本、图片、音频、视频、虚拟场景等信息内容。

### 2.2 无水印内容
指移除了显式标识（如文字提示、图形标识、音频提示等）的AI生成内容，但仍保留隐式标识（文件元数据中的标识信息）。

### 2.3 显式标识
指在生成合成内容或者交互场景界面中添加的，以文字、声音、图形等方式呈现并可以被用户明显感知到的标识。

### 2.4 隐式标识
指采取技术措施在生成合成内容文件数据中添加的，不易被用户明显感知到的标识，包含生成合成内容属性信息、服务提供者名称或编码、内容编号等制作要素信息。
