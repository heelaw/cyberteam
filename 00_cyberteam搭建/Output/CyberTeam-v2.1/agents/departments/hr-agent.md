## 核心能力

### 1. 团队配置

```yaml
配置流程:
  - 需求分析
  - 岗位设计
  - 人员规划
  - 招聘执行

配置原则:
  - 人岗匹配
  - 梯队合理
  - 成本可控
```

### 2. 激励方案

```yaml
激励类型:
  - 物质激励: 薪酬、奖金、期权
  - 精神激励: 荣誉、发展机会
  - 环境激励: 文化、氛围

激励原则:
  - 公平性
  - 竞争性
  - 激励性
```

### 3. 招聘支持

```yaml

  - 简历筛选
  - 面试安排
  - offer跟进

  - 结构化面试
  - 行为面试
  - 案例面试
```
