## 核心框架

### 框架一：目标-漏斗-发力点-驱动力模型（升级版）

```
                    ┌───────────────┐
                    │   活动目标    │  ← 最终要达成的结果（北极星指标）
                    │   (北极星)    │
                    └───────┬───────┘
                            │
                    ┌───────┴───────┐
                    │   转化漏斗    │  ← 达成目标的路径（AARRR各环节）
                    │   (各环节)    │
                    └───────┬───────┘
                            │
                    ┌───────┴───────┐
                    │   发力点      │  ← 需要重点优化的环节（杠杆支点）
                    │   (杠杆支点)  │
                    └───────┬───────┘
                            │
                    ┌───────┴───────┐
                    │   驱动力      │  ← 针对发力点的驱动力强化组合
                    │  (强化组合)   │
                    └───────────────┘
```

**核心逻辑**：
- 目标拆解为漏斗各环节
- 各环节存在转化率
- 优化转化率最低的环节（瓶颈）= 最高ROI
- 发力点 = 杠杆效应最大的环节
- 发力点 × 驱动力组合 = 最优效果

### 框架二：发力点识别四步法（融合版）

```
Step 1: 结构化目标 → 目标是什么？（公式化）
    ↓
Step 2: 拆解变量因子 → 受哪些因素影响？
    ↓
Step 3: 分析当前数据 → 哪个环节最薄弱？（绘制转化漏斗图）
    ↓
Step 4: 确定发力优先级 → 优化哪里最有效？+ 用什么驱动力强化？
```

## 执行步骤（融合版）

### Step 1: 结构化活动目标

**目的**：将模糊的活动目标转化为可量化、可拆解的结构

**操作流程**：

1. **明确最终目标**

   ```
   目标模板：
   在 [时间周期] 内，通过 [活动名称]，
   达成 [核心KPI] [数值]，环比 [基准] [变化幅度]。

   示例：
   在618活动期间（6月15-18日），
   达成GMV 500万，环比日常均值增长50%。
   ```

2. **拆解目标公式**

   ```
   GMV = UV × 点击率 × 注册率 × 首付率 × 客单价
   GMV = 访客数 × 下单率 × 平均订单金额

   新付费用户 = 曝光 × 点击率 × 注册率 × 首付率
   ```

3. **列出所有变量和目标值**

   | 变量 | 定义 | 基准值 | 目标值 | 提升空间 |
   |------|------|--------|--------|----------|
   | UV | 独立访客 | 50万 | 80万 | +60% |
   | 点击率 | 点击/曝光 | 15% | 20% | +33% |
   | 注册率 | 注册/点击 | 60% | 65% | +8% |
   | 首付率 | 首单/注册 | 25% | 30% | +20% |
   | 客单价 | GMV/订单 | 200元 | 220元 | +10% |

---

### Step 2: 拆解关键影响因子

**目的**：理解每个变量受什么因素影响，为驱动力匹配做准备

**拆解模板**：

```
UV = 站内流量 + 站外流量

站内流量 = 首页流量 + 频道流量 + 搜索流量 + 推送流量
站外流量 = 广告投放 + 社交传播 + 合作渠道 + 私域唤醒

转化率 = f(优惠力度#1, 商品吸引力#1, 用户匹配度#4, 页面体验#8)

客单价 = f(品类结构#1, 满减门槛#2, 推荐精准度#5, 用户购买力#6)
```

**关键因子与驱动力映射表**：

| 目标变量 | 直接因子 | 对应驱动力 | 强化方式 |
|----------|----------|------------|----------|
| UV | 渠道数量, 投放预算 | #1利益,#4社交 | 分享裂变、邀请奖励 |
| 点击率 | 入口位置, 素材吸引力 | #9好奇心,#2稀缺 | 悬念预热、限时领取 |
| 首付率 | 优惠力度, 信任度 | #1利益,#7恐惧 | 限时折扣、即将失效 |
| 客单价 | 满减设计, SKU深度 | #2稀缺,#3竞争 | 阶梯满减、实时排行 |

---

### Step 3: 分析当前数据瓶颈（融合驱动力诊断）

**目的**：识别最薄弱、最需要优化的环节，并确定对应的驱动力强化方案

**操作流程**：

1. **绘制转化漏斗图**

   ```
   曝光(1000万) ════════════════════════════════ 100%
       ↓ CTR 10% [发力P2: 素材优化 #9好奇心]
   点击(100万)   ════════════════════════════════ 10%
       ↓ 注册率 50% [发力P3: 门槛降低 #1利益]
   注册(50万)    ════════════════════════════════ 5%
       ↓ 首付率 20% [发力P1: 激励强化 #1+#7]
   首单(10万)    ════════════════════════════════ 1%
       ↓ 客单价 200元 [发力P3: SKU调整 #2稀缺]
   GMV(2000万)   ════════════════════════════════ 0.2%
   ```

2. **计算各环节转化效率和发力优先级**

   | 环节 | 当前转化 | 目标 | 提升空间 | 优化难度 | 发力优先级 | 对应驱动力 |
   |------|----------|------|----------|----------|------------|------------|
   | 点击率 | 10% | 15% | +50% | 低 | **P1** | #9好奇心+#2稀缺 |
   | 注册率 | 50% | 55% | +10% | 低 | P2 | #1利益 |
   | 首付率 | 20% | 30% | +50% | 中 | **P1** | #1利益+#7恐惧 |
   | 客单价 | 200元 | 250元 | +25% | 高 | P3 | #2稀缺+#3竞争 |

3. **瓶颈环节驱动力诊断**

   ```
   瓶颈识别：
   ├─ 绝对瓶颈：首付率20%（转化最低）
   ├─ 相对瓶颈：首付率20%→30%（提升空间50%）
   └─ 杠杆瓶颈：首付率优化 / 优化难度 = 高ROI

   驱动力强化方案：
   → 核心发力点：首付率提升
   → 对应驱动力：#1利益（优惠券）+ #7恐惧（限时）
   → 协同放大：#2稀缺（限量优惠）= 1.5倍效果
   ```

---

### Step 4: 确定发力方向和驱动力强化策略

**目的**：制定针对性的优化方案，精准匹配驱动力

**操作流程**：

1. **计算各环节发力价值和驱动力需求**

   ```
   发力价值公式：
   价值 = 当前值 × (目标值 - 当前值) / 优化难度

   示例计算（GMV=1000万基准）：

   ├─ 首付率优化（#1利益+#7恐惧）：
   │  20%→30%，贡献GMV+500万，优先级P1
   │
   ├─ 点击率优化（#9好奇心+#2稀缺）：
   │  10%→15%，贡献GMV+250万，优先级P1
   │
   └─ 客单价优化（#2稀缺+#3竞争）：
      200→250元，贡献GMV+250万，优先级P2
   ```

2. **制定发力策略和驱动力配置**

   | 发力点 | 当前问题 | 驱动力组合 | 优化策略 | 预期效果 | 资源投入 |
   |--------|----------|------------|----------|----------|----------|
   | 首付率 | 优惠门槛高 | #1利益+#7恐惧 | 降低门槛+限时专享 | +30%首付 | 低 |
   | 点击率 | 素材创意弱 | #9好奇心+#2稀缺 | 悬念预热+A/B测试 | +50%点击 | 中 |
   | 客单价 | SKU结构差 | #2稀缺+#3竞争 | 满减阶梯+实时排行 | +15%客单 | 高 |

3. **驱动力强化执行方案**

   ```
   【首付率发力方案】

   驱动力配置：
   ├─ #1利益：99减25专属首单券（原价199）
   ├─ #7恐惧：24小时限时（倒计时）
   └─ #2稀缺：首批限量500份

   文案：
   "【限时24小时】首单专属优惠，仅剩XX份！
    99元即可获得价值199元商品，不下单就亏了！"

   效果预估：
   首付率：20% → 28%（+40%提升）
   GMV贡献：+300万
   ```

## 使用示例

### 场景：618活动目标发力点分析+驱动力配置

**背景**：
- 活动目标：GMV 500万（日常150万的333%）
- 现有资源：预算50万，团队10人

**Step 1: 结构化目标**

```
GMV = UV × 点击率 × 注册率 × 首付率 × 客单价
500万 = UV × 15% × 60% × 20% × 220元

倒推各环节目标：
- UV需达到：126万
- 点击率：维持15%
- 注册率：维持60%
- 首付率：需提升至25%（当前20%）← 核心发力点
- 客单价：维持220元
```

**Step 2: 拆解因子+驱动力映射**

```
首付率 = f(优惠券力度#1, 首单商品#1, 页面体验#8, 信任度#6)

首付率问题诊断：
├─ 优惠券：当前满100减15，门槛偏高 → 强化#1利益
├─ 首商品：缺乏高性价比引流款 → #1利益
├─ 紧迫感：缺乏时间压力 → 强化#7恐惧
└─ 信任度：老用户基础OK
```

**Step 3: 瓶颈识别+驱动力诊断**

```
漏斗分析：
曝光100万 → 点击15万(15%) → 注册9万(60%) → 首单1.8万(20%) → GMV400万

当前问题：
├─ 首付率20%是最大瓶颈（转化最低）
├─ 低于行业均值25%
└─ 提升空间67%（20%→25%）

驱动力诊断：
→ 核心发力点：提升首付转化
→ 驱动力强化：#1利益（利益激励）+ #7恐惧（损失厌恶）
→ 协同驱动：#2稀缺（限量优惠）= 1.5倍紧迫感
```

**Step 4: 确定发力方案+驱动力配置**

```
发力价值计算：
首付率+5% = GMV+100万（贡献67%差距）
点击率+3% = GMV+20万（贡献13%差距）

发力策略：
1. 核心发力：首付率提升
   ├─ #1利益：99减25专属首单券
   ├─ #7恐惧：24小时限时
   ├─ #2稀缺：首批限量1000份
   └─ 预期首付率：20%→28%

2. 辅助发力：点击率提升
   ├─ #9好奇心：悬念预热
   └─ 预期点击率：15%→18%

驱动力配置：
- #1利益：25元优惠（占预算15万）
- #7恐惧：24小时倒计时强化
- #2稀缺：实时剩余库存播报
```
