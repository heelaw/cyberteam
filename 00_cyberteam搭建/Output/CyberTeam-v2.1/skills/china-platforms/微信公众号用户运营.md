# 微信公众号用户运营

## 概述

公众号运营的核心不是"发内容"，而是"经营用户关系"。本skill帮助运营者通过精细化用户运营，提升用户粘性、活跃度和转化。

## 用户分层管理

### 分层模型

```markdown
## 按关注时间分层
- 新关注用户（0-30天）
- 活跃用户（30-90天）
- 沉默用户（90天以上未互动）
- 流失用户（取关前兆）

## 按价值分层
- 高价值用户（频繁互动、高转化）
- 中价值用户（偶尔互动）
- 低价值用户（仅偶尔阅读）
- 潜在流失用户（互动下降）
```

### 分层运营策略

```markdown
## 新关注用户
- 欢迎语：价值预告+引导互动
- 3天内推送：精华内容合集
- 7天内：引导首次互动

## 活跃用户
- 推送频率略高
- 优先体验新功能/活动
- 邀请参与内测

## 沉默用户
- 减少推送频率
- 推送唤醒内容
- 私信关怀

## 高价值用户
- 专属福利
- 1v1沟通机会
- 荣誉激励
```

## 自动回复配置

### 关键词回复

```markdown
## 高频关键词回复配置

常见问题类：
- 回复"1"：获取产品介绍
- 回复"2"：查看价格
- 回复"3"：联系客服
- 回复"4"：获取使用教程

内容分类类：
- 回复"干货"：获取历史干货合集
- 回复"活动"：查看最新活动
- 回复"案例"：查看成功案例
```

## 用户互动策略

### 评论互动

```markdown
## 评论管理原则

1. 及时回复
   - 工作时间内2小时内回复
   - 重要问题优先处理

2. 回复技巧
   - 先认同，再回应
   - 简洁明了
   - 引导进一步互动

3. 精选留言
   - 精选正面评论置顶
   - 引导用户参考已有评论
```

### 投票与问答

```markdown
## 互动活动设计

1. 投票类
   - 话题投票：了解用户偏好
   - 评选类：用户参与感

2. 问答类
   - 定期问答：专家答疑
   - 征集问题：UGC内容来源

3. 抽奖类
   - 评论抽奖：提升互动
   - 转发抽奖：提升传播
```

## 用户增长策略

### 裂变增长

```markdown
## 裂变模式

1. 关注裂变
   - 任务：邀请N人关注
   - 奖励：资料/课程/产品

2. 朋友圈裂变
   - 任务：分享文章到朋友圈
   - 奖励：解锁更多内容

3. 社群裂变
   - 公众号引流到社群
   - 社群内二次传播
```

### 增长指标

```
新增关注数 = 内容传播带来的新增 + 裂变活动带来的新增
取关率 = 取关数 / 总关注数（健康值<5%/月）
净增关注 = 新增 - 取关
```
