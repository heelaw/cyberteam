# 微信公众号内容策划

## 概述

微信公众号是中国最重要的商业内容平台之一。与其他平台不同，公众号更注重深度内容和用户关系。本skill帮助运营者策划有价值、能转化的公众号内容。

## 内容策略框架

### 60/30/10内容法则

```
60% 价值内容
- 行业干货、知识科普
- 解决问题、提供价值
- 目的：建立信任，建立专业形象

30% 互动内容
- 话题讨论、用户征集
- 投票活动、粉丝福利
- 目的：增强粘性，提高互动

10% 推广内容
- 产品介绍、促销活动
- 品牌故事、合作案例
- 目的：商业转化，但不过度营销
```

## 内容类型规划

### 深度图文

```markdown
## 适合场景
- 行业洞察与分析
- 产品深度测评
- 品牌故事讲述
- 专业知识和教程

## 内容结构
1. 引人入胜的标题
2. 价值预告（让用户知道读完能得到什么）
3. 核心内容（分点论述，有数据支撑）
4. 总结升华
5. 行动号召（关注、转发、点击）
```

### 短内容推送

```markdown
## 适合场景
- 日常更新
- 热点借势
- 轻松话题

## 内容结构
1. 一个核心观点或一件事
2. 简洁有力的表达
3. 引导互动
```

### 系列专题

```markdown
## 适合场景
- 长期品牌建设
- 专业领域深耕
- 用户教育

## 规划要点
1. 确定系列主题和方向
2. 规划系列篇数（建议8-12篇）
3. 保持更新节奏
4. 建立系列品牌
```

## 标题策划

### 高打开率标题公式

**数字型**
```
"3个技巧，让XXX效率翻倍"
"99%的人都不知道的XXX"
```

**疑问型**
```
"XXX是真的吗？"
"如何XXX？这是我见过最好的答案"
```

**对比型**
```
"XXX和XXX，差距竟然这么大"
"别人XXX，你却XXX，差在哪"
```

**情绪型**
```
"XXX，让我彻底改变了"
"终于有人把XXX说清楚了"
```

### 标题撰写要点

```
1. 控制在20字以内
2. 包含关键词（利于搜索）
3. 制造悬念或承诺价值
4. 引发情绪共鸣
5. 避免标题党
```

## 热点策划

### 热点分类

```markdown
## 可预测热点
- 节日：春节、情人节、618、双11
- 季节：春夏秋冬、换季
- 行业：展会、行业峰会
- 固定：月末、年初、毕业季

## 突发热点
- 社会热点事件
- 行业突发新闻
- 竞品动态
```

### 热点借势原则

```
1. 快：热点生命周期短，30分钟内出稿
2. 准：与品牌/产品有自然关联
3. 稳：不蹭敏感话题和政治热点
4. 巧：借势角度要巧妙，不能硬蹭
```

## SEO优化

```markdown
## 公众号搜索优化

1. 标题关键词
   - 包含用户搜索的关键词
   - 控制在20字以内

2. 正文关键词布局
   - 首段出现核心关键词
   - 正文自然出现2-3次
   - 结尾再次强调

3. 标签设置
   - 设置3-5个相关标签

4. 链接优化
   - 相关阅读引导
   - 历史文章互链
```
