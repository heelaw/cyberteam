## 核心框架

### 增长飞轮模型（升级版）

```
增长飞轮 = 驱动力 × 转化率 × 留存率 × 传播系数

驱动力: 产品价值 / 价格优势 / 品牌拉力 / #1利益 #7恐惧
转化率: 流量 → 用户 → 付费（各环节转化率）
留存率: 付费 → 复购 → 忠诚（AARRR核心）
传播系数: 口碑 / 裂变 / 生态（K因子）

飞轮核：留存率 × 传播系数 = 复利基础
```

### 生命周期阶段判断（核心决策）

| 阶段 | 核心特征 | 增长重点 | 北极星指标 | 飞轮状态 |
|------|----------|----------|------------|----------|
| **探索期(0→1)** | 验证PMF | 留存、NPS | 激活率、Aha时刻 | 种子积累 |
| **增长期(1→N)** | 规模化增长 | CAC、LTV、K因子 | 新增用户、GMV | 飞轮加速 |
| **成熟期(N→稳)** | 内涵增长 | LTV、ARPU | 续费率、ARPU | 飞轮稳固 |
| **衰退期(下滑)** | 价值收割/转型 | 召回率、转型验证 | 流失率、转型成功率 | 飞轮修复 |

### 阶段判断决策树

```
当前指标状态？
    │
    ├── 留存率<20% → 探索期（尚未PMF）
    │   └→ 核心：产品打磨，不是增长
    │
    ├── CAC/LTV > 1/3 → 增长期（模型未验证）
    │   └→ 核心：验证单位经济模型
    │
    ├── K因子 > 0.5 → 增长期（自增长基础）
    │   └→ 核心：饱和攻击+裂变
    │
    └── 增速<10%/月 → 成熟期/衰退期
        ├─ 流失率<5%/月 → 成熟期（内涵增长）
        │   └→ 核心：深耕LTV
        └─ 流失率>5%/月 → 衰退期
            └→ 核心：召回+转型
```

## 执行步骤（升级版）

### Step 1: 商业模式画布梳理

```
九要素画布:
  □ 价值主张: 解决什么问题（用户痛点）
  □ 客户细分: 谁是核心用户（画像）
  □ 渠道通路: 如何触达用户（获客路径）
  □ 客户关系: 如何维护关系（留存体系）
  □ 收入来源: 如何赚钱（变现模式）
  □ 核心资源: 关键资产是什么（护城河）
  □ 关键业务: 必须做什么（增长杠杆）
  □ 重要合作: 依赖谁（生态伙伴）
  □ 成本结构: 花在哪里（CAC构成）

输出: 商业模式画布图 + 价值链条
```

### Step 2: 关键指标拆解

```
北极星指标 (唯一关键指标):
  - 唯一指标：聚焦唯一，不贪多
  - 反映用户价值：用户真正在乎的
  - 可驱动增长：能牵引全公司行动

辅助指标:
  - 先行指标 (预测北极星)：提前预警
  - 滞后指标 (反映北极星结果)：事后验证

指标层级树:
  北极星指标
    ├── 先行指标A
    │   ├── 指标A1
    │   └── 指标A2
    └── 先行指标B
        ├── 指标B1
        └── 指标B2

示例 (电商平台):
  北极星: GMV
    ├── 流量
    │   ├── 付费流量
    │   └── 免费流量
    └── 转化
        ├── 访问→加购
        └── 加购→付款

示例 (SaaS产品):
  北极星: MRR (月度经常性收入)
    ├── 新增MRR
    │   ├── 新增客户数
    │   └── 平均客单价
    └── 留存MRR
        ├── 续费率
        └── 扩增率
```

### Step 3: 增长阶段诊断

```
阶段判断流程：

1. 数据输入
   ├─ 留存率: 次日/7日/30日
   ├─ CAC/LTV比值
   ├─ K因子（裂变系数）
   └─ 增速/月环比

2. 阶段判断
   │
   ├── 留存率<20%未达PMF → 探索期
   │   └→ 不追求增长，先验证产品
   │
   ├── CAC/LTV > 1/3 → 增长期（模型未跑通）
   │   └→ 先优化单位经济，再投入增长
   │
   ├── K因子>0.5 + CAC/LTV<1/3 → 增长期（自增长启动）
   │   └→ 饱和攻击，规模化增长
   │
   ├── 增速趋缓 + 留存稳定 → 成熟期
   │   └→ 深耕LTV，提高ARPU
   │
   └── 流失率上升 + 增速下滑 → 衰退期
       └→ 召回优先，或转型探索
```

### Step 4: 增长飞轮构建

```
复利公式: 增长 = 基础量 × (1+增长率)^时间

飞轮构建四要素：
1. 驱动力（引擎）
   ├─ 产品价值：核心解决用户什么问题
   ├─ 价格优势：凭什么选你不选别人
   └─ 品牌拉力：心智占领程度

2. 转化率（齿轮组）
   ├─ 流量→用户：获客效率
   ├─ 用户→付费：激活效率
   └─ 单次→复购：留存效率

3. 留存率（飞轮核）
   ├─ 复购率：>40%健康
   ├─ 粘性指标：DAU/MAU>30%
   └─ NPS：>50优秀

4. 传播系数（加速器）
   ├─ K因子：>1自增长
   ├─ 口碑指数：推荐率
   └─ 内容传播：UGC/PGC

复利来源分析：
  □ 口碑传播: K因子 = 平均邀请人数 × 邀请转化率
    - K>1 → 自增长
    - K<1 → 需外部输血

  □ 留存复购: 复购率提升 → LTV提升 → 可投入更多获客

  □ 规模效应: 规模增长 → 成本下降 → 价格优势 → 更多用户

  □ 网络效应: 用户增长 → 价值增长 → 更多用户

识别复利点:
  - 找出增长飞轮的"飞轮核"
  - 找到让飞轮转速加快的节点
  - 设计让飞轮自我加速的机制
```

### Step 5: 增长阶段性规划（按阶段配置策略）

```
阶段一: 0→1 (Product-Market Fit)
  目标: 找到PMF
  指标: 留存率>20%、NPS>50
  重点: 产品打磨、核心用户
  动作:
    ├─ 种子用户筛选（邀请制+审核制）
    ├─ 深度服务（1对1沟通+入群）
    ├─ 最小化实验（A/B测试）
    └─ 超预期体验（口碑=体验-预期）

阶段二: 1→N (Growth)
  目标: 规模化增长
  指标: CAC/LTV<1/3、K因子>0.5
  重点: 渠道优化、单位经济模型
  动作:
    ├─ 渠道ROI排序+饱和攻击
    ├─ 裂变机制设计（K因子>1）
    ├─ 激活优化（Aha时刻引导）
    └─ 留存红线（同步监控）

阶段三: N→品牌 (Defense)
  目标: 建立护城河
  指标: 市场份额、品牌认知度
  重点: 品牌建设、生态扩展
  动作:
    ├─ 品牌营销（占领心智）
    ├─ 会员体系（提升LTV）
    └─ 生态扩展（平台化）

阶段四: 衰退/转型
  目标: 价值收割或战略转型
  指标: 流失率、转型成功率
  重点: 召回策略、新曲线探索
  动作:
    ├─ 召回专案（#7恐惧+#1利益）
    ├─ 产品创新（第二曲线）
    └─ 战略收缩（聚焦核心）
```

## 增长逻辑检验清单（升级版）

```
□ 阶段判断：当前处于哪个生命周期阶段？
□ 逻辑自洽：因果关系清晰？
□ 数据可测：指标可量化？
□ 路径可行：有明确的增长路径？
□ 飞轮存在：有自我强化的机制？
□ 壁垒匹配：增长能建立壁垒？
□ 复利可期：有复利增长的可能？
□ 时间合理：增长所需时间可接受？

□ 阶段匹配：
  ├─ 探索期：是否还在打磨产品？
  ├─ 增长期：单位经济模型是否跑通？
  ├─ 成熟期：LTV是否在提升？
  └─ 衰退期：是否有转型计划？

□ 关键比率健康度：
  ├─ LTV/CAC > 3？（增长期门槛）
  ├─ K因子 > 0.5？（自增长门槛）
  └─ 复购率 > 40%？（留存健康）
```

## 使用示例（升级版）

```
输入: 某知识付费产品

Step 1: 画布梳理
  价值主张: 提升职场竞争力
  收入来源: 会员订阅 + 课程付费
  核心资源: 名师+内容

Step 2: 指标拆解
  北极星: GMV
    ├── 新增
    │   ├── 新增付费用户
    │   └── 平均客单价
    └── 留存
        ├── 续费率
        └── 扩购率

Step 3: 阶段诊断
  数据：
  - 次日留存: 35%
  - 30日留存: 15%
  - CAC/LTV: 1/4
  - K因子: 0.3

  判断：增长期早期
  - 留存达标（>20%）→ PMF已验证
  - CAC/LTV<1/3 → 单位经济OK
  - K因子<0.5 → 裂变待加强

Step 4: 飞轮构建
  当前飞轮：
  产品好用 → 用户学习 → 能力提升 → 口碑推荐 → 更多用户

  飞轮核分析：
  - 留存率：15%（需提升）
  - K因子：0.3（需提升至>0.5）

  优化重点：
  1. 提升留存：课程完课率+作业互动
  2. 提升K因子：邀请裂变+奖学金机制

Step 5: 阶段规划
  当前阶段：增长期早期
  Q1目标：K因子提升至0.5
  Q2目标：规模化增长，付费用户10万

结论: 增长逻辑清晰，核心在提升K因子
```
