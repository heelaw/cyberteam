# 价值洼地诊断

## 身份定位

```
┌─────────────────────────────────────────────────────────────┐
│  💎 价值洼地诊断                                           │
│  用途: 找投入产出比最高的优化点                             │
│  配套Agent: 用户激励Agent                                    │
└─────────────────────────────────────────────────────────────┘
```

---

## 核心概念

```
价值洼地 = 高收益 × 低投入 × 高确定性

高收益: 优化后效果显著
低投入: 所需资源少
高确定性: 成功概率高

价值洼地 = 钻石点 → 优先去做
```

---

## 执行步骤

### Step 1: 全局扫描

```
扫描范围:
  □ 产品体验
    - 功能使用率
    - 操作效率
    - 体验痛点

  □ 用户价值
    - 用户感知价值
    - 价值与成本比
    - 价值提升空间

  □ 转化效率
    - 各环节转化率
    - 瓶颈位置
    - 优化潜力

  □ 运营效率
    - 人均产出
    - 资源利用率
    - 流程效率
```

### Step 2: 价值计算

```
每个机会的价值计算:

  价值 = (优化前 - 优化后) × 用户量 × 单位价值

  投入 = 开发成本 + 运营成本 + 时间成本

  ROI = 价值 / 投入

排序:
  ROI从高到低 → 优先做ROI最高的
```

### Step 3: 确定性评估

```
确定性等级:
  □ 已知可行
    - 有案例证明
    - 数据支持
    - 风险低

  □ 推断可行
    - 逻辑成立
    - 缺乏数据
    - 风险中

  □ 假设可行
    - 需要验证
    - 风险高
```

### Step 4: 洼地识别

```
洼地识别矩阵:
  ┌──────────┬─────────────┬──────────────┐
  │          │ 高确定性    │ 低确定性     │
  ├──────────┼─────────────┼──────────────┤
  │ 高ROI   │ A类洼地    │ B类洼地     │
  │         │ 立即执行   │ 小规模测试  │
  ├──────────┼─────────────┼──────────────┤
  │ 低ROI   │ C类洼地    │ D类洼地     │
  │         │ 排期做     │ 放弃        │
  └──────────┴─────────────┴──────────────┘

A类洼地 (价值洼地):
  - ROI高
  - 确定性高
  - 立即执行
```
