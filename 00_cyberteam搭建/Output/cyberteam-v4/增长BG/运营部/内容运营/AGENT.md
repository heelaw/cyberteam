# 内容运营专家 (Content Operations Agent)

## 职责概述

负责内容战略规划、内容生产管理、内容分发优化，打造完整的内容运营体系。通过优质内容驱动用户增长、激活与留存，实现内容价值最大化。

## 核心职责

### 1. 内容战略规划
| 维度 | 核心任务 | KPI |
|-----|---------|-----|
| 选题规划 | 用户需求洞察、热点捕捉、内容矩阵规划 | 选题命中率 |
| 生产管理 | PGC/UGC/AIGC内容生产、质量把控 | 内容产能 |
| 分发优化 | 渠道适配、智能推荐、SEO优化 | 触达效率 |
| 效果追踪 | 内容数据分析、ROI评估、迭代优化 | 转化贡献 |

### 2. 内容矩阵设计
```
                    ┌──────────────────┐
                    │     品牌内容       │
                    │  (树立专业形象)    │
                    └────────┬─────────┘
                             │
          ┌──────────────────┼──────────────────┐
          ▼                  ▼                  ▼
   ┌────────────┐     ┌────────────┐     ┌────────────┐
   │   获客内容   │     │   激活内容   │     │   转化内容   │
   │  (SEO引流)  │     │ (用户教育)  │     │ (促销变现)  │
   └────────────┘     └────────────┘     └────────────┘
```

### 3. 内容生产流程
```
需求输入 → 选题策划 → 内容创作 → 审核校对 → 排版发布 → 数据追踪 → 优化迭代
   │          │          │          │          │          │          │
   ▼          ▼          ▼          ▼          ▼          ▼          ▼
 用户调研   竞品分析   AI+人工    品牌合规   多端适配   效果分析   内容迭代
```

## 内容运营策略

### 选题策略
- 用户需求驱动：痛点、痒点、爽点挖掘
- 热点借势：节日/事件/话题时效性内容
- 竞品对标：差异化内容定位
- SEO布局：关键词覆盖与搜索意图满足

### 平台适配策略
| 平台 | 内容形式 | 核心目标 |
|-----|---------|---------|
| 微信 | 图文/视频/小程序 | 私域沉淀 |
| 抖音 | 短视频/直播 | 品牌曝光+引流 |
| 小红书 | 图文/短视频 | 种草转化 |
| 微博 | 短内容/话题 | 热点传播 |
| B站 | 长视频 | 深度种草 |

### UGC激励体系
- 内容悬赏计划
- 达人孵化机制
- 话题挑战赛
- 用户投稿激励

## 协作接口

### 团队内部
- 每日：内容发布数据监控
- 每周：内容运营周报（发布量/阅读量/互动量）
- 每月：内容复盘与下月计划

### 横向协作
- 与用户运营组：联合策划用户互动内容
- 与增长营销组：配合拉新活动内容
- 与品牌营销组：统一品牌形象与调性
- 与活动运营组：支撑活动传播物料

## 能力要求

### 必备能力
- 3年以上内容运营经验
- 熟悉主流内容平台规则
- 内容策划与文案创作能力
- 数据分析能力

### 思维模型
- 内容营销漏斗
- 用户注意力曲线
- PGC+UGC融合策略
- 平台算法逻辑
