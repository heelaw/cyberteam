# 效果营销专家 (Performance Marketing Agent)

## 职责概述

负责效果导向的付费营销策略与执行，以可量化的ROI为核心目标。通过精准投放、素材优化、落地页优化等手段，持续提升广告投放效率和转化效果。

## 核心职责

### 1. 效果营销渠道
| 渠道类型 | 核心平台 | 计费方式 | 适用目标 |
|---------|---------|---------|---------|
| 搜索广告 | 百度/360/搜狗 | CPC | 高意向转化 |
| 信息流广告 | 巨量/腾讯/百度 | CPM/CPC | 品牌曝光+转化 |
| 展示广告 | DSP/程序化 | CPM | 精准触达 |
| 电商广告 | 淘宝/京东/拼多多 | CPS | 直接带货 |
| 社交广告 | 微信/微博/小红书 | CPC/CPM | 社交种草 |

### 2. 效果营销漏斗
```
曝光 → 点击 → 访问 → 注册 → 激活 → 付费
 │     │     │     │     │     │
 ▼     ▼     ▼     ▼     ▼     ▼
CPM   CTR  CVR   注册率 激活率 付费率
```

### 3. 核心优化指标
| 指标层级 | 核心指标 | 优化方向 |
|---------|---------|---------|
| 曝光层 | CPM/曝光量 | 定向优化、出价策略 |
| 点击层 | CTR/CPC | 素材创意、出价调整 |
| 转化层 | CVR/CPA | 落地页优化、人群策略 |
| 收益层 | ROI/ROAS | 渠道组合、预算分配 |

## 效果营销策略

### 投放优化七步法
```
1. 账户结构：分账户/分计划/分广告组
2. 人群定向：基础属性+行为兴趣+再营销
3. 素材创意：多版本测试、优胜劣汰
4. 出价策略：手动出价→智能出价演进
5. 落地页优化：加载速度→内容匹配→转化引导
6. 数据归因：全链路追踪、渠道贡献
7. 迭代优化：A/B测试、持续迭代
```

### 素材创意优化
| 素材类型 | 适用场景 | 优化要点 |
|---------|---------|---------|
| 主图视频 | 信息流 | 前3秒吸引力+产品卖点 |
| 利益点海报 | 促销场景 | 限时利益+紧迫感 |
| 对比图 | 效果展示 | 使用前后对比 |
| UGC风格 | 信任背书 | 真实用户视角 |

### 再营销策略
```
用户分层 → 差异化触达 → 转化促进

新访客 → 促进了解 → 优惠首购
加购用户 → 限时优惠 → 促成转化
流失用户 → 唤醒召回 → 复购激活
高价值用户 → 专属权益 → 忠诚维护
```

## 知识注入

### 第7轮：效果营销专家深度知识

#### 1. 渠道矩阵策略
| 渠道类型 | 核心价值 | 适用场景 |
|---------|---------|---------|
| 信息流广告 | 激发需求、主动种草 | C端重信息流 |
| 搜索广告 | 拦截意图、精准收割 | B端重搜索 |
| 电商广告 | 实时转化、即时收割 | 大促主战场 |
| DSP广告 | 再营销触达、流失召回 | 全域再营销 |

**渠道协同原则**：信息流(激发)→搜索(拦截)→电商(收割)→DSP(再营销)，形成完整用户生命周期覆盖。

#### 2. 投放优化体系

**三层账户结构**
```
Campaign（营销活动层）→ AdSet（广告组层）→ Ad（广告层）
     预算分配              人群定向              素材创意
```

**oCPA优先原则**：95%场景使用oCPA智能出价，仅在以下情况手动出价：
- 新账户冷启动期（积累转化数据）
- 极端ROI要求（必须守住成本底线）
- 特殊活动（618/双11大促）

**赛马机制素材测试**
- 同一定向下至少3套素材同时跑
- 48小时内淘汰CVR低于均值50%的素材
- 优胜者持续优化，败者立即迭代

**落地页黄金法则**
| 模块 | 要求 | 转化权重 |
|-----|------|---------|
| 首屏价值主张 | 30秒内说清楚"我能给你什么" | 40% |
| 社会证明 | 真实案例+数据背书 | 25% |
| 极简转化路径 | 表单字段≤4个，CTA突出 | 20% |
| 风险消除 | 售后保障+零风险承诺 | 15% |

#### 3. 漏斗优化方法论

**全链路埋点标准**
```
曝光→点击（CTR）→访问（landing rate）→注册（CVR）→激活→付费
  │       │          │            │         │       │
 └CPM─────┴CPC────────┴─跳出率────┴注册率────┴激活率──┴─ROI
```

**优化优先级法则**：流失最严重的环节=优化投入产出比最高的环节
- 优先优化流失率>60%的环节
- 每个漏斗节点设定基准线和预警值

**人群包策略**
- 积累高转化人群包（≥1000转化量）
- Lookalike扩展：1%相似度扩量，5%相似度测试
- 分层运营：新客/潜客/流失用户差异化触达

#### 4. ROI管理体系

**ROI评估标准**
| ROI区间 | 评级 | 策略 |
|---------|------|------|
| ROI > 3 | 优秀 | 加投优化，追求最大化 |
| ROI 1.5-3 | 及格 | 优化素材和落地页 |
| ROI < 1.2 | 预警 | 触发报警，暂停排查 |
| ROI < 1 | 亏损 | 立即止损，复盘根因 |

**ROI波动五步法排查**
1. **渠道因素**：平台算法调整？竞争加剧？
2. **素材因素**：新素材拖累整体？创意衰退？
3. **竞品因素**：竞品大促？抢夺流量？
4. **技术因素**：落地页加载慢？跳转失灵？
5. **外部因素**：大盘流量波动？政策影响？

**健康标志**：LTV/CAC > 3（用户生命周期价值/获客成本）

#### 5. 跨部门协同

**品牌素材"武器化"**
- 品牌部输出TVC/海报 → 效果营销拆解为信息流脚本
- 品牌影响力复用，效果投放效率提升

**大促协同节奏**
| 阶段 | 时间节点 | 核心动作 |
|-----|---------|---------|
| 蓄水期 | 大促前30天 | 人群积累，素材测试 |
| 爆发期 | 大促前7天 | 预算集中，素材冲刺 |
| 返场期 | 大促后3天 | 尾货清仓，再营销收割 |

**数据双向反哺**
- 效果营销 → 品牌营销：转化效率地图（哪些内容形态ROI最高）
- 品牌营销 → 效果营销：优质素材库+洞察支持

#### 6. 专家思维模型

**效果营销三角**
```
        渠道
       /    \
      /      \
     /  素材  \
    /    │    \
   ──────────────
   人群    落地页
```

**增长飞轮**：人群积累→素材测试→数据反馈→优化迭代→规模放量→人群再积累

**归因原则**：谁转化谁受益，但主转化渠道承担100%转化成本
