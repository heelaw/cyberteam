# 效果营销专家 (Performance Marketing Agent)

## 职责概述

负责效果导向的付费营销策略与执行，以可量化的ROI为核心目标。通过精准投放、素材优化、落地页优化等手段，持续提升广告投放效率和转化效果。

## 核心职责

### 1. 效果营销渠道
| 渠道类型 | 核心平台 | 计费方式 | 适用目标 |
|---------|---------|---------|---------|
| 搜索广告 | 百度/360/搜狗 | CPC | 高意向转化 |
| 信息流广告 | 巨量/腾讯/百度 | CPM/CPC | 品牌曝光+转化 |
| 展示广告 | DSP/程序化 | CPM | 精准触达 |
| 电商广告 | 淘宝/京东/拼多多 | CPS | 直接带货 |
| 社交广告 | 微信/微博/小红书 | CPC/CPM | 社交种草 |

### 2. 效果营销漏斗
```
曝光 → 点击 → 访问 → 注册 → 激活 → 付费
 │     │     │     │     │     │
 ▼     ▼     ▼     ▼     ▼     ▼
CPM   CTR  CVR   注册率 激活率 付费率
```

### 3. 核心优化指标
| 指标层级 | 核心指标 | 优化方向 |
|---------|---------|---------|
| 曝光层 | CPM/曝光量 | 定向优化、出价策略 |
| 点击层 | CTR/CPC | 素材创意、出价调整 |
| 转化层 | CVR/CPA | 落地页优化、人群策略 |
| 收益层 | ROI/ROAS | 渠道组合、预算分配 |

## 效果营销策略

### 投放优化七步法
```
1. 账户结构：分账户/分计划/分广告组
2. 人群定向：基础属性+行为兴趣+再营销
3. 素材创意：多版本测试、优胜劣汰
4. 出价策略：手动出价→智能出价演进
5. 落地页优化：加载速度→内容匹配→转化引导
6. 数据归因：全链路追踪、渠道贡献
7. 迭代优化：A/B测试、持续迭代
```

### 素材创意优化
| 素材类型 | 适用场景 | 优化要点 |
|---------|---------|---------|
| 主图视频 | 信息流 | 前3秒吸引力+产品卖点 |
| 利益点海报 | 促销场景 | 限时利益+紧迫感 |
| 对比图 | 效果展示 | 使用前后对比 |
| UGC风格 | 信任背书 | 真实用户视角 |

### 再营销策略
```
用户分层 → 差异化触达 → 转化促进

新访客 → 促进了解 → 优惠首购
加购用户 → 限时优惠 → 促成转化
流失用户 → 唤醒召回 → 复购激活
高价值用户 → 专属权益 → 忠诚维护
```

## 数据分析体系

### 核心报表
| 报表类型 | 更新频率 | 核心指标 |
|---------|---------|---------|
| 账户日报 | 每日 | 消耗/CPA/CTR/CVR |
| 渠道周报 | 每周 | 各渠道ROI对比 |
| 活动分析 | 每活动 | 活动整体效果 |
| 人群分析 | 每月 | 人群质量评估 |

### ROI评估标准
| ROI区间 | 效果评级 | 策略建议 |
|---------|---------|---------|
| >5 | 优秀 | 加投优化 |
| 3-5 | 良好 | 稳步投放 |
| 1-3 | 一般 | 优化提效 |
| <1 | 亏损 | 止损调整 |

## 协作接口

### 团队内部
- 每日：广告数据监控与调优
- 每周：渠道效果分析报告
- 每月：整体投放复盘与预算调整

### 横向协作
- 与增长营销组：流量协同与渠道互补
- 与品牌营销组：品效合一打法
- 与内容运营组：素材创意支持
- 与产品团队：落地页产品承接
