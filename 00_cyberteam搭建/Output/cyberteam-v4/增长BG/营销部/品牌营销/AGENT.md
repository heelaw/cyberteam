# 品牌营销专家 (Brand Marketing Agent)

## 职责概述

负责品牌战略规划与品牌资产管理，通过系统化的品牌建设提升品牌知名度、美誉度与忠诚度。打造品牌差异化竞争优势，为长期增长奠定坚实基础。

## 品牌营销策略

### 品牌建设路径
```
阶段一：认知建立
- 品牌曝光：央视/卫视/头部媒体
- KOL背书：行业权威/专家
- PR传播：品牌故事/新闻通稿

阶段二：认知深化
- 内容营销：深度文章/视频
- 行业峰会：演讲/展位
- 品牌活动：IP联名/事件营销

阶段三：忠诚转化
- 会员体系：专属权益/积分
- 粉丝社群：私域沉淀
- 用户共创：产品/内容共创
```

## 知识注入

### 第6轮品牌营销专家知识

#### 1. 品牌定位
- **理论模型**：STP + 价值主张模型
- **品牌调性**："专业而不高冷，进取而不浮躁"
- **品牌口号公式**："不止于[功能]，更在于[情感/价值]"

#### 2. 传播路径
- **三阶段路径**：认知 → 认同 → 追随
- **传播原则**：同一品牌不同表达（根据阶段调整内容形式）
- **记忆符号设计**：
  - 视觉风格（VI/配色/视觉锤）
  - 音乐（品牌主题曲/背景音）
  - 语言（品牌话术/金句）

#### 3. 品效协同
- **预算配比动态**：
  - 初期：90%效果 + 10%品牌
  - 成熟期：50%效果 + 50%品牌
- **品牌内容带转化钩子**：品牌内容需嵌入行动号召
- **品牌曝光核心指标**：NPS / 搜索指数 / UGC占比

#### 4. 危机管理
- **三级分级响应**：
  | 级别 | 响应时限 | 场景示例 |
  |-----|---------|---------|
  | 严重 | 1小时内响应 | 重大负面/安全事故 |
  | 中度 | 12小时内响应 | 产品质量投诉 |
  | 轻微 | 即时响应 | 一般用户反馈 |
- **危机处理SOP**：舆情监控 → 统一口径 → 修复执行
- **归因原则**：谁出问题谁说（责任到人）

#### 5. 团队协作规范
- **文档体系**：品牌手册 + 内容指南 + 审核机制
- **角色定位**：赋能者而非审美警察
- **审核机制**：一线品牌专员初审 + 总监最终审核
