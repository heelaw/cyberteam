# Skill - 增长营销专家

## 基本信息
- **名称**: growth-marketing
- **版本**: v1.0.0
- **类型**: 营销Skill
- **归属**: 增长BG/营销部

## 功能概述

用户增长营销Skill，覆盖渠道拓展、裂变活动、获客优化。

## 触发条件

- 获客成本上升
- 需要新渠道拓展
- 裂变系数下降
- 增长目标未达成

## 工作流程

### Phase 1: 增长诊断
```
1. 渠道分析 → 各渠道ROI
2. 漏斗分析 → 转化瓶颈
3. 竞品调研 → 增长策略
4. 目标设定 → KPI分解
```

### Phase 2: 增长策略
```
1. 渠道优化 → 高ROI渠道
2. 裂变设计 → 老带新机制
3. 内容营销 → 自然增长
4. 合作拓展 → 异业合作
```

### Phase 3: 增长执行
```
1. 小范围测试 → MVP验证
2. 数据分析 → 效果评估
3. 大规模推广 → 放量
4. 持续优化 → 迭代改进
```

## 核心方法

### 增长飞轮
```
获客 → 激活 → 留存 → 变现 → 推荐 → 获客
  ↑                                      ↓
  └──────────── 循环放大 ◄──────────────┘
```

### ICE优先级
- **I (Impact)**: 影响程度
- **C (Confidence)**: 信心程度
- **E (Ease)**: 难易程度
