# Skill - 效果营销专家

## 基本信息
- **名称**: performance-marketing
- **版本**: v1.0.0
- **类型**: 营销Skill
- **归属**: 增长BG/营销部

## 功能概述

效果广告投放Skill，覆盖投放策略、定向优化、ROI提升。

## 触发条件

- 新广告投放
- 投放效果下降
- 需要新渠道测试
- 预算调整
- 转化成本高

## 工作流程

### Phase 1: 投放策略
```
1. 目标设定 → CPA/ROI目标
2. 渠道选择 → 平台选择
3. 人群定向 → 目标人群
4. 预算分配 → 渠道预算
```

### Phase 2: 投放执行
```
1. 素材制作 → 广告创意
2. 计划搭建 → 投放设置
3. AB测试 → 素材迭代
4. 实时监控 → 效果跟踪
```

### Phase 3: 效果优化
```
1. 数据分析 → 效果归因
2. 定向优化 → 人群调整
3. 素材迭代 → 创意更新
4. 出价优化 → 成本控制
```

## 核心方法

### 归因模型
- **首触归因**: 第一个渠道
- **末触归因**: 最后一个渠道
- **线性归因**: 平均分配
- **时间衰减**: 越近权重越高

### ROI优化
```
CPA = 消耗 / 转化数
ROI = 收入 / 消耗
LTV > CPA * 3 → 可持续
```
