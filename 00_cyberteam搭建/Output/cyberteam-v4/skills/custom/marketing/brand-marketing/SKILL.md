# Skill - 品牌营销专家

## 基本信息
- **名称**: brand-marketing
- **版本**: v1.0.0
- **类型**: 营销Skill
- **归属**: 增长BG/营销部

## 功能概述

品牌建设与传播Skill，覆盖品牌定位、视觉规范、PR传播。

## 触发条件

- 新品牌建设
- 品牌升级
- PR危机
- 品牌声量下降
- KOL合作需求

## 工作流程

### Phase 1: 品牌定位
```
1. 品牌审计 → 当前认知
2. 竞品分析 → 差异化
3. 用户调研 → 期望形象
4. 定位确定 → 品牌策略
```

### Phase 2: 品牌建设
```
1. 视觉规范 → VI系统
2. 品牌故事 → 理念传递
3. 调性统一 → 品牌一致性
4. 品牌手册 → 规范文档
```

### Phase 3: 品牌传播
```
1. PR规划 → 媒体关系
2. KOL合作 → 达人种草
3. 活动传播 → 品牌事件
4. 舆情监控 → 声誉管理
```

## 核心方法

### 品牌金字塔
- **核心价值**: 为什么选择我们
- **品牌个性**: 我们是什么样
- **情感利益**: 用户感受什么
- **功能利益**: 产品解决什么问题
