# Skill - 内容运营专家

## 基本信息
- **名称**: content-operations
- **版本**: v1.0.0
- **类型**: 运营Skill
- **归属**: 增长BG/运营部

## 功能概述

内容生态建设Skill，覆盖内容策划、生产、分发全链路。

## 触发条件

- 内容消费下降
- 需要热点运营
- UGC内容少
- 内容质量不稳定
- 需要内容策略规划

## 工作流程

### Phase 1: 内容规划
```
1. 用户调研 → 内容偏好
2. 热点追踪 → 实时热点
3. 竞品分析 → 内容方向
4. 内容规划 → 选题日历
```

### Phase 2: 内容生产
```
1. PGC生产 → 官方内容
2. UGC激励 → 用户创作
3. 内容审核 → 质量把控
4. 内容标签 → 便于分发
```

### Phase 3: 内容分发
```
1. 标签匹配 → 精准推送
2. 推荐算法 → 个性化分发
3. 效果追踪 → 数据反馈
4. 内容优化 → 迭代改进
```

## 核心方法

### 内容矩阵
| 类型 | 目的 | 频率 |
|------|------|------|
| 干货内容 | 价值输出 | 60% |
| 热点内容 | 流量获取 | 20% |
| 互动内容 | 活跃提升 | 10% |
| 品牌内容 | 品牌建设 | 10% |

### 传播模型
- AIDA模型：注意→兴趣→欲望→行动
- PAS模型：问题→放大→方案
- 故事化：起承转合
