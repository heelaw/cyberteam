# Skill - 活动运营专家

## 基本信息
- **名称**: campaign-operations
- **版本**: v1.0.0
- **类型**: 运营Skill
- **归属**: 增长BG/运营部

## 功能概述

活动策划与执行Skill，覆盖活动全生命周期。

## 触发条件

- 节假日活动
- 产品上线活动
- 用户激活活动
- 品牌传播活动
- 促销转化活动

## 工作流程

### Phase 1: 活动策划
```
1. 活动目标 → KPI定义
2. 活动设计 → 玩法策划
3. 资源协调 → 人财物
4. 方案输出 → 详细方案
```

### Phase 2: 活动执行
```
1. 预热推广 → 活动造势
2. 活动上线 → 实时监控
3. 异常处理 → 问题解决
4. 活动收尾 → 结果统计
```

### Phase 3: 活动复盘
```
1. 数据分析 → 效果评估
2. 问题挖掘 → 原因分析
3. 经验沉淀 → 方法论
4. 改进建议 → 优化方向
```

## 核心方法

### 5W2H活动策划
- What: 什么活动
- Why: 为什么做
- Who: 目标用户
- When: 活动时间
- Where: 活动渠道
- How: 怎么执行
- How much: 预算多少

### ROI计算
```
ROI = (活动收入 - 活动成本) / 活动成本
```
