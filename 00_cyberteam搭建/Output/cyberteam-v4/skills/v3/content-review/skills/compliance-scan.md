# 《合规风险扫描 Skill》

版本：v3.1

---

## 1. 功能

### 1.1 广告法红线检测

- 绝对化用语识别
- 虚假宣传检测
- 贬低竞品识别
- 违禁词汇过滤

### 1.2 平台规则检测

- 小红书社区规范
- 微信公众号规范
- 微博社区公约
- 抖音内容规范

### 1.3 知识产权检测

- 版权内容识别
- 引用合规性检查
- 图片授权验证
- 字体版权检查

---

## 2. 合规检查矩阵

### 2.1 广告法红线

| 类别 | 典型词汇/行为 | 风险等级 |
|------|--------------|----------|
| 绝对化用语 | 最好、第一、顶级、唯一、国家级 | P0 |
| 虚假宣传 | 虚构数据、伪造认证、假冒资质 | P0 |
| 贬低竞品 | 「垃圾」「比不上」「不如」 | P1 |
| 违禁词 | 特效、根治、100%治愈 | P0 |
| 医疗功效 | 治病、疗效、治疗疾病 | P0 |
| 投资承诺 | 稳赚、保本、高收益 | P0 |

### 2.2 平台规则

#### 小红书（XHS）

| 规则类型 | 具体要求 | 风险等级 |
|----------|----------|----------|
| 医疗健康 | 不可发布医疗建议，需资质认证 | P0 |
| 化妆品功效 | 需使用「功效宣称合规词」 | P1 |
| 虚假种草 | 不得伪装真实用户体验 | P0 |
| 导流行为 | 禁止私信导流、外部链接 | P1 |
| 侵权行为 | 不得未经授权使用他人内容 | P0 |

#### 微信公众号

| 规则类型 | 具体要求 | 风险等级 |
|----------|----------|----------|
| 原创声明 | 需确认内容为原创或已获授权 | P1 |
| 商业广告 | 需明确标注「广告」 | P1 |
| 诱导行为 | 禁止诱导关注、分享 | P1 |
| 谣言传播 | 禁止传播未经核实的信息 | P0 |

#### 微博

| 规则类型 | 具体要求 | 风险等级 |
|----------|----------|----------|
| 时政内容 | 敏感话题需谨慎，避免违规 | P0 |
| 虚假信息 | 不得发布传播谣言 | P0 |
| 侵权举报 | 被侵权可被投诉下架 | P1 |
| 营销内容 | 超过比例需报备 | P1 |

#### 抖音

| 规则类型 | 具体要求 | 风险等级 |
|----------|----------|----------|
| 内容分级 | 需符合内容分级标准 | P1 |
| 直播规范 | 禁止虚假宣传、极限词 | P0 |
| 带货规范 | 需真实体验，不得虚假宣传 | P0 |
| 未成年人 | 需保护未成年人权益 | P0 |

### 2.3 知识产权

| 类型 | 检查项 | 风险等级 |
|------|--------|----------|
| 文字版权 | 引用是否超过合理使用范围 | P1 |
| 图片版权 | 是否有商业授权 | P0 |
| 字体版权 | 是否使用免费/已授权字体 | P1 |
| 音乐版权 | 背景音乐是否有授权 | P1 |
| 视频素材 | 素材是否有商业授权 | P1 |

---

## 3. 风险等级分类

| 等级 | 定义 | 处理方式 |
|------|------|----------|
| **P0** | 必须修复 | 严重违规，直接影响账号安全，发布前必须修改 |
| **P1** | 应该修复 | 合规风险，可能导致限流/封禁，建议修改后发布 |
| **P2** | 建议修复 | 优化项，不影响发布，但可提升内容质量 |

## 5. 合规自检清单

发布前必检项：

- [ ] 无绝对化用语（最好、第一、唯一等）
- [ ] 无虚假数据（所有数据有据可查）
- [ ] 无贬低竞品内容
- [ ] 无医疗功效承诺
- [ ] 无投资回报承诺
- [ ] 引用内容已获授权
- [ ] 配图有商业使用授权
- [ ] 符合目标平台规范
