# 《品牌调性审核 Skill》

版本：v3.1

---

## 1. 功能

### 1.1 语气检查

- 评估整体语调是否符合品牌人设
- 检测语气一致性
- 识别语气突变或不协调

### 1.2 用词规范检查

- 敏感词过滤
- 专业术语使用规范性
- 网络用语和流行语的适切性

### 1.3 竞品描述检查

- 竞品提及的客观性
- 避免贬低性描述
- 竞品对比的公正性

---

## 2. 检查维度

### 2.1 绝对化用语

| 禁用级别 | 典型词汇 | 示例 |
|----------|----------|------|
| P0 | 最XX、第一、唯一、顶级 | 「最好用」「行业第一」 |
| P1 | 绝对、完全、完美、极致 | 「绝对安全」「完美解决方案」 |
| P2 | 超级、非常、特别、十分 | 「超级好用」（需结合语境） |

### 2.2 夸张用语

| 类型 | 示例 | 判断标准 |
|------|------|----------|
| 效果夸大 | 「三天瘦十斤」「一周学会」 | 是否无法证实或明显不可能 |
| 规模夸大 | 「全球销量第一」「所有人都」 | 是否缺乏数据支撑 |
| 感受夸张 | 「惊艳全场」「太绝了」 | 是否脱离实际感受 |

### 2.3 品牌禁用词

| 类别 | 词汇 | 说明 |
|------|------|------|
| 医疗相关 | 根治、治愈、疗效 | 无资质不可使用 |
| 投资相关 | 稳赚、保本、收益率 | 涉及金融合规 |
| 功效保证 | 无效退款、100%有效 | 需实际条款支撑 |
| 虚假代言 | 「XX推荐」「明星都在用」 | 无授权不可使用 |

## 4. 调性评估维度

| 维度 | 评估标准 |
|------|----------|
| 专业度 | 是否使用规范表达，无低级错误 |
| 一致性 | 语气、用词是否前后统一 |
| 适切性 | 是否符合目标受众预期 |
| 差异性 | 是否体现品牌独特性 |
| 合规性 | 是否遵守广告法及平台规则 |

---

## 5. 常见调性问题

| 问题 | 表现 | 建议 |
|------|------|------|
| 语气过高 | 「必须」「一定要」等命令式过多 | 增加建议性表达 |
| 语气过低 | 过于谦卑，缺乏专业感 | 适度加强肯定性表达 |
| 中英混杂 | 无意义的中英混用 | 统一语言风格 |
| 口语化 | 网络用语过多 | 根据平台调整 |
| emoji滥用 | 表情符号过多 | 适度使用或统一风格 |
