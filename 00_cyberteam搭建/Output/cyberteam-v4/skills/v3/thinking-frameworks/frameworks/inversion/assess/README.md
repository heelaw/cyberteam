# 反演思维专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 反演思维理论是否正确（避免失败→实现成功的逻辑） |
| 可执行性 | 30% | 输出是否可操作（失败模式识别、避免策略、行动清单是否明确） |
| 独特性 | 20% | 是否有差异化（与逆向思维的区分、主动预防的实用性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（目标/问题）
- [ ] 有输出格式（失败模式、避免策略）
- [ ] 有分析框架（识别失败→分析原因→制定避免策略）
- [ ] 有Critical Rules（必须遵守/禁止行为）
- [ ] 有行动清单

## 核心检验点

### 理论准确性
- [ ] 反演思维与逆向思维区分正确
- [ ] "避免失败"转化为"实现成功"的逻辑正确

### 框架完整性
- [ ] 失败模式识别系统化
- [ ] 避免策略具体可执行
