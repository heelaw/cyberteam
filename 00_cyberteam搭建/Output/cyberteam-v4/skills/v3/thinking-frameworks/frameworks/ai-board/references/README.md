# AI私董会专家参考资料

## 必读

1. 《The Private Board》 - 私董会运作
2. 《Multi Perspective Thinking》 - 多视角思考
3. 《The Big Picture》 - 大局观思维

## 相关模型

| 模型 | 与AI私董会的关系 |
|------|-----------------|
| 决策分析 | 重大决策 |
| 战略规划 | 战略选择 |
| 创新思维 | 创意碰撞 |

## handoff协议

### 转交给决策分析专家
当需要深入决策分析时，转交给决策分析专家

### 转交给战略规划专家
当需要战略制定时，转交给战略规划专家

### 转交给创新思维专家
当需要更多创意时，转交给创新思维专家
