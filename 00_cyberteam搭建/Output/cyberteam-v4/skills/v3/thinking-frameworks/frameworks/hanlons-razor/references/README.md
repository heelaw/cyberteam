# 汉龙剃刀专家参考资料

## 必读

1. 《Hanlon's Razor》Robert Hanlon - 原创思想
2. 《The 48 Laws of Power》Robert Greene - 权力游戏中识别恶意
3. 《Thinking, Fast and Slow》Daniel Kahneman - 基本归因错误

## 案例库

### 成功案例
- **历史上的冤假错案**: 因过度归因于恶意导致的悲剧
- **办公室政治**: 大多数是沟通不畅和能力问题，不是真正的恶意

### 失败案例
- **过度信任**: 汉龙剃刀不能用于已经证实恶意的情境
- **自我欺骗**: 用"无能"来回避必须设立的边界

## 相关模型

| 模型 | 与汉龙剃刀的关系 |
|------|----------------|
| 奥卡姆剃刀 | 都是简化复杂情况的工具 |
| 批判性思维 | 汉龙剃刀是批判性思维在归因中的应用 |
| 博弈论 | 恶意确实存在时的分析工具 |

## 延伸阅读

- 基本归因错误
- 人际归因的心理学
- 何时可以假设恶意

## handoff协议
