1. 《The Base Rate Fallacy》 - 基础比率谬误经典论文
2. 《Thinking, Fast and Slow》Daniel Kahneman - 基础比率章节
3. 《Bayes' Theorem》 - 贝叶斯定理基础

| 代表性启发 | 常忽视基础比率的认知偏差 |
