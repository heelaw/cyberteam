# 煤气灯效应专家测试用例

## 正面测试

### 测试1: 人际关系困惑
- 输入:
```
【煤气灯效应分析请求】
情况描述：与上司关系紧张
具体事件：上次会议上我提出的方案被说成是他的想法
你的感受：困惑，觉得好像是自己记错了
对方反应：坚称我记错了，是我理解有误
```
- 期望输出:
  - 操控手段识别
  - 影响程度评估
  - 应对策略
- 验证项:
  - [ ] 能识别"否认事实"操控手段
  - [ ] 有影响程度评估
  - [ ] 有具体应对建议

### 测试2: 亲密关系
- 输入:
```
【煤气灯效应分析请求】
情况描述：与配偶的沟通问题
具体事件：每次争吵后对方都说我反应过度
你的感受：开始怀疑自己是不是真的有问题
对方反应：说我是"神经病"，需要看医生
```
- 期望输出:
  - 操控手段检测
  - 影响等级
  - 长期策略
- 验证项:
  - [ ] 能识别"质疑心智"操控
  - [ ] 有影响等级评估
  - [ ] 有边界设定建议

## 负面测试

### 测试1: 自我怀疑
- 输入:
```
【煤气灯效应分析请求】
情况描述：朋友说我记性不好
具体事件：他坚称上次见面我没提到某事
你的感受：可能是自己真的记错了
对方反应：（正常讨论）
```
- 期望行为: 不能在没有操控证据时下结论
- 验证项:
  - [ ] 必须确认操控行为的存在
  - [ ] 必须有具体的行为模式识别
  - [ ] 不能仅凭一次事件下结论

### 测试2: 急于指控
- 输入:
```
【煤气灯效应分析请求】
情况描述：同事不同意我的观点
具体事件：讨论中有不同意见
你的感受：他在操控我吗？
对方反应：（正常不同意）
```
- 期望行为: 不能把正常分歧当作操控
- 验证项:
  - [ ] 必须区分正常分歧和操控行为
  - [ ] 必须有系统性的操控手段检测
  - [ ] 必须评估影响程度
