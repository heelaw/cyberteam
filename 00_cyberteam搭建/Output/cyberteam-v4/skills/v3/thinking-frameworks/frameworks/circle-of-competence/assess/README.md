# 能力圈专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 能力圈理论是否正确（圈内/圈边/圈外区分、能力评级） |
| 可执行性 | 30% | 输出是否可操作（能力盘点、匹配分析、建设路径是否明确） |
| 独特性 | 20% | 是否有差异化（与职业规划专家的区分、自我认知的实用性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（情境、自我评估、困惑点）
- [ ] 有输出格式（能力自画像、匹配分析、决策建议、能力建设计划）
- [ ] 有分析框架（四步法：盘点→绘制→匹配→建设）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有能力圈可视化图
- [ ] 有handoff协议

## 核心检验点

### 理论准确性
- [ ] 圈内/圈边/圈外三层区分正确
- [ ] 能力评级方法正确
- [ ] 学习曲线评估正确

### 框架完整性
- [ ] 能力盘点包含硬技能、软技能、资源
- [ ] 情境匹配分析有匹配度百分比
- [ ] 能力建设路径有优先级排序

### 输出有效性
- [ ] 能力圈自画像格式正确
- [ ] 决策建议明确
- [ ] 有警示清单
