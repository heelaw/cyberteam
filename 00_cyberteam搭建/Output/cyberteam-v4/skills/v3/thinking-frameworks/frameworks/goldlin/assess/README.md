# 吉德林法则专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 吉德林法则是否正确（倾倒问题、分离事实/情绪/假设/期望） |
| 可执行性 | 30% | 输出是否可操作（问题陈述、核心问题重述、解决方向是否明确） |
| 独特性 | 20% | 是否有差异化（与一般问题分析的区分、吉德林独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（初步困扰、主观感受、背景）
- [ ] 有输出格式（问题解构、影响评估、核心问题陈述、解决方向）
- [ ] 有分析框架（六步：倾倒→分离→影响→期望→重述→启发）
- [ ] 有Critical Rules（必须遵守4条、禁止行为5条）
- [ ] 有用户为主原则

## 核心检验点

### 理论准确性
- [ ] 六步流程完整
- [ ] 分离清单正确（事实/情绪/假设/期望）
- [ ] 核心问题重述格式正确

### 框架完整性
- [ ] 倾倒问题充分
- [ ] 四类分离完整
- [ ] 启发由用户产出

### 输出有效性
- [ ] 问题清晰度评估
- [ ] 核心问题陈述具体
- [ ] 行动第一步明确
