# A/B测试案例库

## 案例1: 电商落地页优化（简单）

### 背景
- 产品: 某跨境电商APP
- 实验: 优化商品详情页的"加入购物车"按钮
- 目标: 提升购买转化率

### 实验设计
| 项目 | 内容 |
|------|------|
| 实验组 | 改版按钮（更大、更醒目、颜色从灰色改为橙色） |
| 对照组 | 原版按钮 |
| 核心指标 | 点击转化率（点击人数/访问人数） |
| 实验周期 | 14天 |

### 数据摘要

```
实验组数据:
- 样本量: 12,500
- 点击人数: 1,500
- 转化率: 12.00%

对照组数据:
- 样本量: 12,500
- 点击人数: 1,250
- 转化率: 10.00%
```

### 统计检验

**检验方法**: 两比例z检验

**计算过程**:
```
p1 = 0.12, n1 = 12,500
p2 = 0.10, n2 = 12,500

合并比例 p̄ = (1500+1250)/(12500+12500) = 0.11

标准误 SE = √(0.11×0.89×(1/12500+1/12500)) = 0.004

z = (0.12 - 0.10) / 0.004 = 5.0

p值 = 2 × (1 - Φ(5.0)) ≈ 0.00001
```

### 结果分析

| 检验项 | 结果 | 判定 |
|--------|------|------|
| p值 | < 0.0001 | 显著 |
| 置信区间(95%) | [1.2%, 2.8%] | 包含正数 |
| 相对提升 | +20% | 有实际价值 |
| 统计功效 | > 99% | 充足 |

### 结论

- **统计显著**: p < 0.0001，差异非常显著
- **业务显著**: 20%提升具有显著商业价值
- **行动建议**: 推广实验组版本

### 经验教训
1. 按钮视觉优化对转化有显著影响
2. 大样本下即使小差异也能检测到
3. 20%提升在电商领域属于较大提升

## 案例2: 订阅付费流程优化（中等）

### 背景
- 产品: 在线视频会员订阅
- 实验: 优化订阅流程的支付页面
- 目标: 提升付费转化率

### 实验设计
| 项目 | 内容 |
|------|------|
| 实验组 | 精简版支付页面（去除多余信息，突出价格） |
| 对照组 | 原版支付页面 |
| 核心指标 | 付费转化率 |
| 辅助指标 | 页面停留时长、支付成功率 |
| 实验周期 | 21天 |

### 数据摘要

```
实验组数据:
- 样本量: 8,000
- 付费人数: 320
- 转化率: 4.00%
- 平均支付时长: 45秒

对照组数据:
- 样本量: 8,000
- 付费人数: 280
- 转化率: 3.50%
- 平均支付时长: 68秒
```

### 统计检验

**检验方法**: 两比例z检验

**计算过程**:
```
p1 = 0.040, n1 = 8,000
p2 = 0.035, n2 = 8,000

z = (0.040 - 0.035) / √(0.0375×0.9625×2/8000) = 2.18

p值 = 2 × (1 - Φ(2.18)) = 0.029

置信区间: [0.05%, 1.45%]
```

### 功效分析

```
当前检测到提升: 14.3%相对提升
所需最小样本量(检测14%提升): 每组约6,800
实际样本量: 每组8,000
统计功效: ~72%
```

### 结果分析

| 检验项 | 结果 | 判定 |
|--------|------|------|
| p值 | 0.029 | 显著（边界） |
| 置信区间 | [0.05%, 1.45%] | 显著 |
| 相对提升 | +14.3% | 有价值 |
| 统计功效 | 72% | 略不足 |

### 结论

- **统计显著**: p = 0.029 < 0.05，但接近边界
- **业务显著**: 14.3%提升具有商业价值
- **功效问题**: 72%功效略低于80%标准
- **行动建议**: 
  - 可推广，但需监控长期效果
  - 建议继续观察2周确认效果稳定

### 注意事项
1. p值接近0.05边界，存在假阳性风险
2. 建议延长测试或增加样本量确认
3. 需关注新奇效应（用户对新页面好奇）

## 案例3: 推荐算法调优（复杂）

### 背景
- 产品: 内容推荐平台
- 实验: 优化推荐算法A/B测试
- 目标: 提升用户留存和内容消费

### 实验设计

**多层指标体系**:
| 指标类型 | 指标名称 | 目标 |
|----------|----------|------|
| 核心指标 | 7日留存率 | 提升 |
| 核心指标 | 人均内容消费量 | 提升 |
| 辅助指标 | 点赞率 | 提升 |
| 辅助指标 | 分享率 | 提升 |
| 负面指标 | 负反馈率 | 下降 |

**实验组/对照组**:
- 实验组: 新算法（基于深度学习）
- 对照组: 旧算法（基于协同过滤）
- 流量分配: 50%/50%

### 数据摘要

```
实验组 (n=50,000):
- 7日留存率: 42.5%
- 人均内容消费: 28.3条/周
- 点赞率: 8.2%
- 分享率: 3.1%
- 负反馈率: 1.2%

对照组 (n=50,000):
- 7日留存率: 40.0%
- 人均内容消费: 25.7条/周
- 点赞率: 7.8%
- 分享率: 2.9%
- 负反馈率: 1.1%
```

### 统计检验

#### 7日留存率检验
```
p1 = 0.425, n1 = 50,000
p2 = 0.400, n2 = 50,000

z = (0.425 - 0.400) / √(0.4125×0.5875×2/50000) = 8.25

p值 < 0.0001

相对提升: +6.25%
```

#### 人均内容消费检验（t检验）
```
实验组均值: 28.3, SD: 15.2
对照组均值: 25.7, SD: 14.8

t = (28.3 - 25.7) / √(15.2²/50000 + 14.8²/50000) = 28.6

p值 < 0.0001

效应量 Cohen's d = 0.18（小效应）
```

#### 多重比较校正
```
原始p值:
- 留存率: <0.0001
- 人均消费: <0.0001
- 点赞率: 0.014
- 分享率: 0.043
- 负反馈率: 0.210

Bonferroni校正后 (α/5 = 0.01):
- 留存率: 显著
- 人均消费: 显著
- 点赞率: 显著
- 分享率: 不显著
- 负反馈率: 不显著
```

### 功效分析

```
目标检测最小效应: 2%绝对提升
所需样本量: 每组约 150,000
实际样本量: 每组 50,000
统计功效: 约 85%
```

### 复杂因素分析

**时间因素**:
- 测试周期: 4周
- 第1周: 新奇效应明显，提升较大
- 第2-4周: 效果趋于稳定

**用户分群差异**:
| 用户类型 | 实验组提升 | 显著性 |
|----------|-----------|--------|
| 新用户(注册<7天) | +8.2% | 显著 |
| 活跃用户 | +4.1% | 显著 |
| 沉默用户 | +1.2% | 不显著 |

**长期效果评估**:
- 30日留存: 实验组 35.2% vs 对照组 34.1%（+3.2%，显著）

### 结果分析

| 检验项 | 结果 | 判定 |
|--------|------|------|
| 留存率p值 | < 0.0001 | 显著 |
| 人均消费p值 | < 0.0001 | 显著 |
| 效应量 | 0.18 | 小 |
| 多重比较 | 4/5显著 | 可接受 |
| 统计功效 | 85% | 充足 |

### 结论

- **核心指标显著**: 留存率和消费均显著提升
- **效应量较小**: Cohen's d = 0.18，属于小效应
- **用户差异**: 新用户提升更明显
- **长期稳定**: 30日留存也有提升
- **行动建议**: 逐步推广（先10%用户，再50%，最后100%）

### 经验教训
1. 多指标体系需做多重比较校正
2. 小效应量也有商业价值（用户基数大）
3. 需分析用户分群差异
4. 建议分阶段推广，持续监控

## 案例4: 搜索结果排序优化（中等）

### 背景
- 产品: 电商搜索
- 实验: 优化搜索结果排序算法
- 目标: 提升搜索转化率

### 实验设计

**实验变量**:
- 实验组: 新排序算法（加入销量权重和实时性权重）
- 对照组: 原排序算法（主要基于相关性）

**关键指标**:
- 核心: 搜索转化率（点击后购买/点击）
- 辅助: 点击率、页面停留时长

### 数据摘要

```
实验组 (n=25,000搜索会话):
- 转化率: 12.8%
- 点击率: 45.2%
- 停留时长: 32秒

对照组 (n=25,000搜索会话):
- 转化率: 11.5%
- 点击率: 44.8%
- 停留时长: 31秒
```

### 统计检验

```
p1 = 0.128, n1 = 25,000
p2 = 0.115, n2 = 25,000

z = (0.128 - 0.115) / √(0.1215×0.8785×2/25000) = 4.12

p值 < 0.0001

置信区间: [0.7%, 1.9%]
```

### 结论

- 统计显著: p < 0.0001
- 业务显著: 11.3%相对提升
- 行动建议: 推广

## 案例5: 新手引导流程优化（复杂）

### 背景
- 产品: SaaS工具
- 实验: 优化新用户新手引导
- 目标: 提升首次使用完成率和后续活跃

### 挑战

1. **多阶段指标**: 引导包含5个步骤
2. **长周期**: 需要跟踪30天活跃
3. **季节性**: B端产品受假期影响
4. **数据稀疏**: 某些细分用户群样本少

### 数据表现

| 指标 | 实验组 | 对照组 | 提升 | p值 |
|------|--------|--------|------|-----|
| 首次完成率 | 68% | 62% | +9.7% | <0.001 |
| 7日活跃 | 45% | 42% | +7.1% | 0.008 |
| 30日活跃 | 32% | 31% | +3.2% | 0.21 |
| 付费转化 | 8.5% | 7.8% | +9.0% | 0.12 |

### 结论

- 短期指标显著，长期效果不明显
- 建议: 继续观察，做用户分群分析

## 案例总结

### 统计显著≠业务显著

| 案例 | p值 | 统计显著 | 业务显著 | 决策 |
|------|-----|----------|----------|------|
| 案例1 | <0.0001 | 是 | 是(+20%) | 推广 |
| 案例2 | 0.029 | 是(边界) | 是(+14%) | 推广+观察 |
| 案例3 | <0.0001 | 是 | 是(小效应) | 逐步推广 |
| 案例4 | <0.0001 | 是 | 是(+11%) | 推广 |

### 关键教训

1. **样本量是基础**: 足够的样本量是统计检验的前提
2. **关注效应量**: 统计显著不等于实际有价值
3. **多重比较**: 多指标测试需要校正
4. **功效分析**: 事前计算样本量，事后评估功效
5. **长期观察**: 新奇效应可能导致短期虚高
