## 二、封面图规范

### 2.1 封面图规格

| 项目 | 规格 |
|------|------|
| **比例** | 3:4 (竖版) |
| **尺寸** | 1242x1660 px |
| **格式** | JPG/PNG |
| **文件大小** | ≤5MB |
| **DPI** | 72 DPI |

### 2.2 封面图设计

**构图原则**:
- 标题醒目
- 留白适当
- 色彩对比
- 信息清晰

**标题位置**:
```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│           【醒目标题文字】                │
│           副标题/补充说明                │
│                                         │
│                                         │
│              [图片内容]                  │
│                                         │
└─────────────────────────────────────────┘
```

### 2.3 封面图风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **现代简约** | 大字报、简洁背景 | 干货、教程 |
| **生活场景** | 真实场景、温馨感 | 分享、好物 |
| **精致美图** | 高质量摄影作品 | 时尚、旅行 |
| **对比冲击** | 使用前后对比 | 效果展示 |

## 三、标题规范

### 3.1 标题长度

| 长度 | 字数 | 效果 |
|------|------|------|
| **最佳** | 15-20字 | 显示完整，有吸引力 |
| **简短** | < 15字 | 需配合封面 |
| **较长** | > 20字 | 可能显示不全 |

### 3.2 高点击率标题

| 类型 | 公式 | 示例 |
|------|------|------|
| **痛点型** | "[数字]个方法解决[痛点]" | "3个方法解决夏天油皮问题" |
| **数字型** | "[数字]个[关键词]，太[形容词]了" | "5个让皮肤变好的习惯" |
| **对比型** | "用了[产品]，才知道[差距]" | "用了这款粉底液，才知道差距" |
| **悬念型** | "[秘密/方法]大公开" | "月瘦10斤的减肥方法大公开" |
| **情绪型** | "[情绪词]！[事件]" | "后悔！现在才知道这个好东西" |
| **身份型** | "[人群]必看！" | "油皮姐妹必看！控油攻略" |

### 3.3 标题禁忌

```
❌ 标题党
  "史上最全..." "不看后悔..."

❌ 虚假夸大
  "一天瘦10斤" "用完白一个度"

❌ 负面引导
  "千万别..." "后悔没早..."
```

## 四、正文规范

### 4.1 正文长度

| 类型 | 字数 | 适用场景 |
|------|------|----------|
| **短笔记** | < 500字 | 快分享、简单种草 |
| **标准笔记** | 500-1000字 | 完整种草 |
| **长笔记** | 1000-2000字 | 详细教程 |

### 4.2 正文结构

```
┌─────────────────────────────────────────────────────┐
│ 【开头】痛点/场景引入                               │
│ "夏天对于油皮来说真的太难了..."                      │
├─────────────────────────────────────────────────────┤
│ 【主体】核心内容                                    │
│ - 产品/方法介绍                                    │
│ - 使用体验                                          │
│ - 效果展示                                          │
├─────────────────────────────────────────────────────┤
│ 【结尾】总结 + 互动引导                             │
│ "以上就是我的分享，你们有什么..."                    │
├─────────────────────────────────────────────────────┤
│ 【标签】#关键词1 #关键词2 #关键词3                   │
└─────────────────────────────────────────────────────┘
```

### 4.3 排版规范

- 段落之间空一行
- 每段 2-4 行
- 适当使用 emoji (但不过度)
- 列表简洁明了

## 六、标签规范

### 6.1 标签数量

| 数量 | 效果 |
|------|------|
| **< 3个** | 可能曝光不足 |
| **3-5个** | 最佳平衡 |
| **6-10个** | 曝光最大化 |
| **> 10个** | 可能被视为堆砌 |

### 6.2 标签选择

```
标签组合公式:

[核心词] + [细分词] + [热点词] + [品牌词]

示例 (护肤品):
  - #护肤 → 核心词
  - #油皮护肤 → 细分词
  - #夏天护肤 → 热点词
  - #SKII → 品牌词

示例 (穿搭):
  - #穿搭 → 核心词
  - #小个子穿搭 → 细分词
  - #显高穿搭 → 痛点词
  - #ootd → 平台热词
```

### 6.3 标签来源

| 来源 | 说明 |
|------|------|
| **搜索建议** | 发布时输入关键词的联想 |
| **热门话题** | 平台当前热门话题 |
| **竞品标签** | 竞品笔记使用的标签 |
| **长期词** | 长期有搜索量的词 |

## 七、互动引导

### 7.1 评论互动

```
互动引导话术:

1. 提问式
   "你们觉得哪个更好用？评论区告诉我！"
   "你们还有什么问题？评论区见~"

2. 分享式
   "你们有类似的经历吗？评论区分享一下吧！"
   "你们还知道哪些好用的产品？评论区补充！"

3. 参与式
   "想要教程的举手！人多就出详细版！"
   "评论区抽3个宝子送小样！"
```

### 7.2 避免的互动

```
❌ 强制要求
   "点赞过100更新..."

❌ 道德绑架
   "不点赞的不是姐妹"

❌ 虚假承诺
   "明天公布中奖名单" (但不会公布)
```

## 九、合规规范

### 9.1 违规行为

```
❌ 禁止内容:
  - 虚假宣传
  - 夸大功效
  - 医疗效果承诺
  - 色情低俗
  - 政治敏感
  - 违法犯罪

❌ 禁止行为:
  - 诱导站外交易
  - 私信导流
  - 虚假数据
  - 抄袭搬运
```

### 9.2 广告规范

| 类型 | 规范 |
|------|------|
| **软广** | 必须标注"赞助"或"合作" |
| **品牌合作** | 使用蒲公英平台 |
| **产品对比** | 不能恶意贬低竞品 |
| **极限词** | 禁止"最"、"第一" |

## 十、数据指标

### 10.1 核心指标

| 指标 | 说明 | 优质值 |
|------|------|--------|
| **曝光量** | 笔记被看到的次数 | > 1000 |
| **阅读量** | 笔记被打开的次数 | > 曝光量 10% |
| **点赞** | 用户点赞数 | > 阅读量 5% |
| **收藏** | 用户收藏数 | > 阅读量 5% |
| **评论** | 用户评论数 | > 阅读量 2% |
| **关注** | 新增关注数 | > 阅读量 1% |

### 10.2 数据优化

```
提升曝光:
  - 优化标签选择
  - 提高内容质量
  - 选择最佳发布时间

提升收藏:
  - 内容有干货价值
  - 提供实用信息
  - 结尾引导收藏

提升评论:
  - 内容引发共鸣
  - 主动提问互动
  - 及时回复评论
```
