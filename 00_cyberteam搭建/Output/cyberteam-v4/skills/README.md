# CyberTeam V4 Skills 目录

## 目录结构

```
skills/
├── third-party/     # 第三方 Skills
└── custom/         # 自定义 Skills
```

## 第三方 Skills (third-party/)

| Skill | 说明 | 来源 |
|-------|------|------|
| **baoyu** | 宝珠工具集（18个） | 第三方 |
| **gstack** | 工程技能 | 第三方 |
| **superpowers** | 超级能力 | 第三方 |
| **agency-agents** | Agency Agents (15类) | 第三方 |
| **pua-skill** | PUA 激励系统 | 第三方 |

## 自定义 Skills (custom/)

自定义 Skills 目录，用于存放 CyberTeam V4 特有的扩展 Skills。

## Skills 使用指南

### 调用第三方 Skill

```bash
# 使用 gstack
/gstack review

# 使用 baoyu
/baoyu-image-gen

# 使用 pua
/pua
```

### 查看所有可用 Skills

```bash
ls skills/third-party/
```
