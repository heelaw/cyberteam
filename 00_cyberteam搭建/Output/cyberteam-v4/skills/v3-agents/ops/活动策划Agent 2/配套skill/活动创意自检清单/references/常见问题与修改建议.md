# 常见问题与修改建议

## 一、目标匹配度问题

### 问题1: KPI贡献路径不清晰

**表现**:
- 创意很好但说不清如何贡献KPI
- KPI与创意关联牵强
- 贡献量级无法估算

**修改建议**:
```
步骤1: 明确核心KPI
步骤2: 逆向推导，找到创意与KPI的直接关联
步骤3: 量化贡献路径（曝光→点击→转化→KPI）
步骤4: 如无法找到直接关联，重新设计创意
```

### 问题2: 目标受众不匹配

**表现**:
- 创意吸引的不是目标用户
- 目标用户在活动中被忽略
- 触达渠道无法覆盖目标人群

**修改建议**:
```
步骤1: 重新明确目标受众画像
步骤2: 分析目标受众的偏好和触点
步骤3: 调整创意形式或激励机制
步骤4: 更换或增加触达渠道
```

## 二、差异化问题

### 问题3: 创意高度同质化

**表现**:
- 与竞品或行业常见玩法高度相似
- 用户看到没有惊喜感
- 难以在众多活动中脱颖而出

**修改建议**:
```
步骤1: 分析竞品和行业常见玩法
步骤2: 找出差异化的切入角度
步骤3: 在创意/形式/激励/传播上创新
步骤4: 确保差异化点用户能感知
```

### 问题4: 缺乏记忆点

**表现**:
- 用户看完就忘
- 无法用一句话描述活动亮点
- 缺乏传播性

**修改建议**:
```
步骤1: 提炼活动最吸引人的一点
步骤2: 将这一点做到极致
步骤3: 设计传播记忆符号（slogan/视觉/玩法）
步骤4: 让用户愿意主动传播
```

---

## 三、用户体验问题

### 问题5: 参与门槛过高

**表现**:
- 参与步骤超过3步
- 需要复杂操作或输入
- 门槛与奖励不成正比

**修改建议**:
```
步骤1: 梳理用户参与路径
步骤2: 识别冗余步骤并删除
步骤3: 简化操作，减少输入
步骤4: 如必须，增加激励补偿
```

### 问题6: 规则理解困难

**表现**:
- 用户看完不明白怎么玩
- 规则描述模糊或有歧义
- 需要客服大量解释

**修改建议**:
```
步骤1: 用一句话描述活动核心
步骤2: 规则描述简化为3条以内
步骤3: 增加示例或图示说明
步骤4: 准备FAQ应对疑问
```

### 问题7: 激励感知价值低

**表现**:
- 用户觉得奖励不值得
- 奖励与付出不成正比
- 奖励同质化没有吸引力

**修改建议**:
```
步骤1: 重新审视奖励对用户的价值
步骤2: 提高奖励感知价值（稀缺/专属/高价值）
步骤3: 调整激励结构（阶梯/保底/惊喜）
步骤4: 优化展示方式（突出价值感）
```

## 四、可执行性问题

### 问题8: 技术实现困难

**表现**:
- 需要大量新增开发
- 技术方案复杂风险高
- 现有系统无法支持

**修改建议**:
```
步骤1: 评估技术可行性
步骤2: 简化技术方案
步骤3: 分期实现，MVP先行
步骤4: 如无法简化，考虑替代方案
```

### 问题9: 资源严重不足

**表现**:
- 预算远高于预算范围
- 人力无法支持执行
- 时间窗口不够

**修改建议**:
```
步骤1: 梳理资源需求清单
步骤2: 识别可削减的资源需求
步骤3: 调整活动规模或周期
步骤4: 申请追加资源或放弃
```

---

## 五、合规安全问题

### 问题10: 广告法违规风险

**表现**:
- 使用了"最高级"、"国家级"等禁用词
- 价格宣传无依据
- 存在虚假宣传嫌疑

**修改建议**:
```
步骤1: 排查所有文案中的禁用词
步骤2: 价格宣传必须有据可查
步骤3: 避免绝对化表述
步骤4: 法务审核全流程
```

### 问题11: 版权或隐私风险

**表现**:
- 使用了未授权素材
- 收集了非必要用户数据
- 存在侵权隐患

**修改建议**:
```
步骤1: 审核所有使用素材的授权
步骤2: 梳理数据收集范围
步骤3: 获取必要的用户授权
步骤4: 如无法解决，更换素材或取消功能
```

## 六、否决后的处理

### 6.1 否决原因分析

收到否决后，首先分析否决原因：

```
否决类型:
- [ ] 目标匹配度否决
- [ ] 用户体验否决
- [ ] 差异化否决
- [ ] 可执行性否决
- [ ] 合规安全性否决
```

### 6.2 重新设计流程

```
步骤1: 确定需要修改的维度
步骤2: 针对否决原因制定修改方案
步骤3: 修改后重新自检
步骤4: 提交二审
```

### 6.3 二审准备

二审时需提供：
- [ ] 修改说明（针对哪些问题做了修改）
- [ ] 修改后的自检报告
- [ ] 修改点的详细说明

## 七、修改案例

### 案例1: 从不及格到良好

**原创意**: 复杂的大转盘抽奖
- 问题: 步骤5步，中奖率模糊，规则复杂

**修改后**: 简化的抽奖机制
- 修改: 1步参与，公示中奖概率，规则简化到1句话
- 结果: 评级从不及格到良好

### 案例2: 从及格到优秀

**原创意**: 普通的满减活动
- 问题: 有差异化但记忆点不突出

**修改后**: "1元买100"的极限优惠
- 修改: 将优惠力度做到极致，创造话题性
- 结果: 评级从及格到优秀
