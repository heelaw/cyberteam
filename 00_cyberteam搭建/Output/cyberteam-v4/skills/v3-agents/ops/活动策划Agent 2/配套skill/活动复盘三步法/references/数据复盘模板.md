# 数据复盘模板

## 一、数据复盘框架

### 1.1 数据复盘流程

```
数据收集 → 数据清洗 → 对比分析 → 漏斗分析 → 归因分析 → 输出报告
```

### 1.2 数据来源清单

| 数据类型 | 数据来源 | 采集方式 |
|---------|---------|---------|
| 曝光数据 | 投放后台、统计工具 | API/手动导出 |
| 流量数据 | 网站统计、APP统计 | 埋点 |
| 行为数据 | 业务系统、埋点 | 埋点 |
| 转化数据 | 业务系统 | 数据库 |
| 成本数据 | 财务系统 | 人工统计 |
