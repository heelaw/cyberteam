## 使用指南

### 1. 选择合适的模板

根据当前阶段选择对应模板：
- **Stage 1**: 使用 `business-model-template.md`
- **Stage 2**: 使用 `revenue-formula-template.md`
- **Stage 3**: 使用 `conversion-funnel-template.md`
- **Stage 4**: 使用 `operational-metrics-template.md`
- **Stage 5**: 使用 `breakthrough-points.md`
- **Stage 6**: 使用 `final-report-template.md`

### 2. 填充模板

按照模板中的占位符和指导填充内容：
- `{{变量名}}` 格式的占位符需要替换为实际值
- `- **项目**` 格式的列表需要填写具体内容
- 表格中的空行需要填写数据

### 3. 自定义模板

可以根据项目需求自定义模板：
- 添加特定行业的章节
- 调整指标定义
- 修改报告结构

## 最佳实践

### 模板使用

1. **阅读完整模板**: 开始前通读整个模板，了解结构
2. **收集数据**: 准备好所需的数据和信息
3. **逐项填写**: 按照模板顺序逐项填写
4. **保持一致**: 使用一致的术语和格式
5. **验证完整**: 确保所有占位符都已填写

### 命名规范

输出文件命名格式：
```
{项目名称}-{阶段}-{日期}.{扩展名}

示例：
- lacecode-业务模型分析-20250119.md
- lacecode-收入公式-20250119.md
- lacecode-转化链路-20250119.md
```
