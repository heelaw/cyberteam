# 用户运营指标体系

## 一、指标体系概述

### 指标分类

**按用户生命周期**：
- 获取指标
- 激活指标
- 留存指标
- 活跃指标
- 收入指标
- 推荐指标

**按业务类型**：
- 用户规模指标
- 用户质量指标
- 用户价值指标
- 用户行为指标

## 二、用户获取指标

### 获客数量

**新增用户数**
- 定义：一定时期内新注册/下载用户数
- 单位：人
- 监测频率：日、周、月

**渠道新增用户数**
- 定义：各渠道新增用户数
- 用途：对比渠道效果

### 获客成本

**CAC (Customer Acquisition Cost)**
```
CAC = 总营销费用 / 新增用户数
```

**渠道CAC**
- 各渠道单独计算
- 用于优化渠道投放

### 获取质量

**注册转化率**
```
注册转化率 = 注册用户数 / 访问用户数
```

**获客到激活转化率**
```
激活转化率 = 激活用户数 / 新增用户数
```

**渠道质量评估**
| 指标 | 计算 | 用途 |
|------|------|------|
| 渠道留存率 | 各渠道用户留存率 | 评估用户质量 |
| 渠道LTV | 各渠道用户LTV | 评估长期价值 |
| 渠道CAC | 各渠道获客成本 | 评估成本效率 |

## 三、用户激活指标

### 激活定义

**激活用户**：完成首次核心行为的用户

### 激活指标

**激活率**
```
激活率 = 激活用户数 / 注册用户数
```

**首日激活率**
```
首日激活率 = 注册当天激活用户数 / 注册用户数
```

**7日激活率**
```
7日激活率 = 7天内激活用户数 / 注册用户数
```

**激活时长**
- 注册到激活的平均时长
- 越短越好

### 核心行为指标

**首次核心行为完成率**
```
完成率 = 完成首次核心行为用户数 / 注册用户数
```

**功能渗透率**
```
功能渗透率 = 使用过某功能用户数 / 活跃用户数
```

## 四、用户留存指标

### 留存率

**留存率定义**：一定时期后仍活跃的用户占比

**次日留存率**
```
次日留存 = Day 2活跃用户 / Day 1新增用户
```

**7日留存率**
```
7日留存 = Day 8活跃用户 / Day 1新增用户
```

**30日留存率**
```
30日留存 = Day 31活跃用户 / Day 1新增用户
```

### 留存曲线

**留存曲线形状**：
- 优秀：次日>40%, 7日>20%, 30日>10%
- 良好：次日>30%, 7日>15%, 30日>5%
- 一般：次日>20%, 7日>10%, 30日>3%
- 较差：次日<20%, 7日<10%, 30日<3%

### 留存分析

**Cohort分析**：
- 按获取时间分组
- 对比不同cohort的留存
- 观察留存趋势

**留存分布**：
- 短期留存（次日、7日）
- 中期留存（30日）
- 长期留存（90日、180日）

## 五、用户活跃指标

### 活跃规模

**DAU (Daily Active Users)**
- 日活跃用户数
- 定义：当日有核心行为的用户

**WAU (Weekly Active Users)**
- 周活跃用户数

**MAU (Monthly Active Users)**
- 月活跃用户数

### 活跃质量

**DAU/MAU比值**
```
DAU/MAU = DAU / MAU
```
- 衡量用户粘性
- >50%：优秀
- 30-50%：良好
- <30%：一般

**使用频次**
```
人均使用频次 = 总使用次数 / 活跃用户数
```

**使用时长**
```
人均使用时长 = 总使用时长 / 活跃用户数
```

**使用深度**
```
人均使用功能数 = 总功能使用数 / 活跃用户数
```

### 活跃趋势

**活跃增长率**
```
DAU增长率 = (今日DAU - 昨日DAU) / 昨日DAU
```

**活跃用户构成**
- 新增活跃用户
- 老用户活跃
- 回流用户

## 六、用户价值指标

### LTV相关

**LTV (Life Time Value)**
```
LTV = (月购买频次 × 客单价 × 毛利率) / 月流失率
```

**ARPU (Average Revenue Per User)**
```
ARPU = 总收入 / 用户数
```

**ARPPU (Average Revenue Per Paying User)**
```
ARPPU = 总收入 / 付费用户数
```

### 付费相关

**付费转化率**
```
付费转化率 = 付费用户数 / 活跃用户数
```

**复购率**
```
复购率 = 复购用户数 / 付费用户数
```

**客单价**
```
客单价 = 总收入 / 订单数
```

### 用户价值分层

**高价值用户**
- LTV前20%用户
- 贡献80%收入

**中价值用户**
- LTV中间60%用户

**低价值用户**
- LTV后20%用户

## 七、用户质量指标

### 满意度

**NPS (Net Promoter Score)**
```
NPS = 推荐者% - 贬损者%
```
- 9-10分：推荐者
- 7-8分：中立者
- 0-6分：贬损者

**用户满意度**
- 满意用户占比
- 非常满意用户占比

### 流失相关

**流失率**
```
流失率 = 流失用户数 / 总用户数
```

**流失预警**
- 流失风险用户数
- 流失风险用户占比

### 社交相关

**社交连接数**
```
人均好友数 = 总好友关系数 / 用户数
```

**社交互动率**
```
互动率 = 有互动用户数 / 活跃用户数
```

**内容生产率**
```
内容生产率 = 发布内容用户数 / 活跃用户数
```

## 八、用户行为指标

### 使用行为

**功能使用率**
```
功能使用率 = 使用某功能用户数 / 活跃用户数
```

**页面浏览量**
```
PV (Page Views)
UV (Unique Visitors)
```

**点击率**
```
CTR = 点击次数 / 展示次数
```

### 内容行为

**内容消费率**
```
消费率 = 消费内容用户数 / 活跃用户数
```

**内容生产率**
```
生产率 = 生产内容用户数 / 活跃用户数
```

**内容互动率**
```
互动率 = (点赞+评论+分享) / 内容数
```

### 转化行为

**转化率**
```
转化率 = 完成转化用户数 / 进入用户数
```

**漏斗转化**
```
漏斗转化率 = 本步骤用户数 / 上一步骤用户数
```

## 九、指标监测体系

### 监测层级

**一级指标（核心指标）**：
- DAU
- 留存率（次日、7日、30日）
- LTV
- CAC

**二级指标（过程指标）**：
- 新增用户数
- 激活率
- 付费转化率
- ARPU

**三级指标（细节指标）**：
- 各功能使用率
- 各渠道指标
- 各行为指标

### 监测频率

**实时监测**：
- DAU
- 在线用户数
- 核心行为完成数

**日度监测**：
- 新增用户
- 留存率
- 活跃度

**周度监测**：
- 周活跃用户
- 周留存
- 周增长率

**月度监测**：
- MAU
- 月留存
- LTV
- 财务指标

### 数据看板

**核心看板**：
1. 用户规模看板（DAU、MAU、新增）
2. 用户质量看板（留存、活跃）
3. 用户价值看板（LTV、ARPU、付费）
4. 用户行为看板（功能使用、行为路径）

**看板设计原则**：
- 关键指标突出
- 趋势对比明显
- 异常预警清晰
- 下钻分析便捷

## 十、指标分析框架

### 同比环比分析

**同比**
```
同比增长率 = (本期 - 去年同期) / 去年同期
```

**环比**
```
环比增长率 = (本期 - 上期) / 上期
```

### 漏斗分析

**转化漏斗**：
- 注册 → 激活 → 留存 → 付费 → 复购

**关键点**：
- 各步骤转化率
- 转化瓶颈
- 优化空间

### 分层分析

**用户分层**：
- 新老用户
- 高中低价值用户
- 不同生命周期用户

**分层对比**：
- 各层级指标差异
- 各层级变化趋势

## 十一、指标优化目标

### 常见目标值

**产品类型指标参考**：

| 指标 | 工具型 | 内容型 | 社交型 | 电商型 | 游戏型 |
|------|--------|--------|--------|--------|--------|
| 次日留存 | 30-40% | 35-45% | 40-50% | 35-45% | 40-50% |
| 7日留存 | 15-20% | 20-25% | 25-35% | 15-20% | 25-35% |
| 30日留存 | 5-10% | 10-15% | 15-25% | 5-10% | 15-25% |
| DAU/MAU | 20-30% | 30-40% | 40-60% | 20-30% | 40-60% |
| 付费率 | 5-10% | 3-8% | 2-5% | 10-20% | 5-15% |

### 优化优先级

**第一阶段**：
- 提升次日留存（>30%）
- 提升激活率（>40%）

**第二阶段**：
- 提升DAU
- 提升付费转化率

**第三阶段**：
- 提升LTV
- 优化CAC
