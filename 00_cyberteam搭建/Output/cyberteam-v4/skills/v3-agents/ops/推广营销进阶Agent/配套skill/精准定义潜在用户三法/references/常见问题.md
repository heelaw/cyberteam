# 精准定义用户常见问题

## Q1: 完全没有用户数据怎么办？

**解决方案**：
1. 竞品分析：研究竞品的用户画像
2. 行业报告：参考艾瑞、QuestMobile等报告
3. 用户访谈：找目标用户深聊（5-10人）
4. 社交媒体：观察目标人群的讨论

---

## Q2: 画像做出来后怎么验证？

**验证方法**：
1. 小规模投放测试（1000-2000元）
2. 用户访谈确认
3. 转化数据对比

**判断标准**：
- 投放CTR>1% → 画像较准
- 用户访谈符合度>80% → 画像较准

---

## Q3: 用户画像需要多久更新一次？

**更新频率**：
- 产品重大迭代后
- 投放效果明显变化时
- 季度/年度定期复盘
- 至少每半年更新一次

---

## Q4: 画像和投放定向有什么区别？

| 画像 | 定向 |
|------|------|
| 完整的人 | 投放系统标签 |
| 多维度描述 | 有限的标签组合 |
| 指导策略 | 落地执行 |
| 定性分析 | 定量筛选 |

**关系**：画像指导定向，但定向不等于画像

---

## Q5: 用户画像要画几个才合适？

**数量建议**：
- 核心用户：1-2个（重点服务）
- 潜力用户：1个（可拓展）
- 边缘用户：0-1个（不重点）

**原则**：不超过3个，聚焦核心用户

---

## Q6: 画像中的数据从哪来？

**数据来源**：
- 已有用户：订单数据、行为数据
- 竞品：公开资料、用户调研
- 行业：艾瑞、QuestMobile、券商报告
- 用户：问卷、访谈

---

## Q7: 画像和用户分层有什么区别？

| 画像 | 用户分层 |
|------|---------|
| 还没成为用户的人 | 已经是用户的人 |
| 获客用 | 运营用 |
| 指导投放定向 | 指导精细化运营 |

---

## Q8: 画像不准投放效果差怎么办？

**排查步骤**：
1. 检查投放定向是否按画像设置
2. 检查素材是否匹配画像
3. 用小预算测试不同画像组合
4. 根据投放数据反向修正画像
