# 群规模板集

## 模板一：学习型社群群规

```
═══════════════════════════════════════════
        【学习型社群】群规模板
═══════════════════════════════════════════

【关于我们】

🌟 社群定位：
帮助[目标用户]实现[学习目标]的学习型社群

👥 成员构成：
已有[人数]位[目标用户]加入

💬 交流氛围：
[积极/专业/互助]，鼓励分享、共同成长

【鼓励行为】

✅ 欢迎以下行为：
1. 积极提问，分享工作中的疑惑
2. 分享学习心得、读书笔记、课程推荐
3. 回答其他成员的提问，互相帮助
4. 参与社群组织的学习活动

🎁 社群服务：
- 每周[具体学习活动]
- 每月[具体活动]
- 日常答疑[具体形式]

【禁止行为】

🚫 严格禁止：
1. 发广告/推广链接 → 直接移出
2. 发无关投票/问卷 → 警告后移出
3. 人身攻击/争吵 → 直接移出
4. 讨论敏感政治话题 → 警告后移出

【违规处理】

遵守规则，违规必究
第一次警告，第二次移出
欢迎大家监督举报～

【入群即视为认同以上规则】

═══════════════════════════════════════════
```
