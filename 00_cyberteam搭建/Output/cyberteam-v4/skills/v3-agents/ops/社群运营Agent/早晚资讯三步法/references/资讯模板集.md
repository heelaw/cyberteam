## 模板使用说明

1. **保持固定结构**: 让用户形成阅读习惯
2. **控制长度**: 内容不宜过长，关键信息突出
3. **加入图片**: 适当配图增加可读性
4. **互动引导**: 每期内容都要有互动话题
5. **个人IP**: 融入个人见解和风格

---

*模板版本: 1.0 | 适用场景: 各类社群资讯内容*
