# 内容运营工具清单

## 信息收集工具

### 资讯聚合平台

| 工具名称 | 适用场景 | 使用方式 |
|---------|---------|---------|
| 即刻 | 行业资讯订阅 | 关注相关话题/用户 |
| 微信公众号 | 深度文章 | 置顶星标优质公众号 |
| 雪球 | 投资/财经资讯 | 关注相关股票/话题 |
| 知乎 | 深度讨论 | 关注相关话题/专栏 |

### 内容筛选标准

```
【必选标准】
✓ 与社群定位相关
✓ 时效性（24小时内）
✓ 有实用价值或讨论空间
✓ 信息来源可靠

【排除标准】
✗ 纯广告/软文
✗ 过时信息
✗ 与社群无关
✗ 重复内容
```

### 发布检查清单

```
【发布前检查】
□ 内容是否符合社群定位
□ 信息来源是否可靠
□ 是否有敏感内容
□ 字数是否合适（不过长/不过短）
□ 是否有配图/emoji
□ 互动话题是否明确
□ 发送时间是否合适
□ 是否需要@某些人
```
