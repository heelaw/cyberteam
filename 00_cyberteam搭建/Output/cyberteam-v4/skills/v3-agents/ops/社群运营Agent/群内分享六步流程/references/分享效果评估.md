# 分享效果评估指南

## 评估指标体系

### 核心指标

| 指标 | 计算方式 | 健康值 | 说明 |
|------|---------|--------|------|
| 参与率 | 实际参与/预告后报名 | >70% | 预告转化能力 |
| 完听率 | 听完全程/总参与 | >60% | 内容吸引力 |
| 互动率 | 提问人数/总参与 | >30% | 用户参与度 |
| 满意度 | 满意回复/总反馈 | >80% | 整体满意度 |

### 数据采集

```
【参与数据】

【反馈数据】

```

## 评估流程

### 分享后24小时

```
1. 发布满意度调查
2. 整理分享内容
3. 感谢嘉宾
4. 收集数据
```

### 满意度调查模板

```
大家觉得今天的分享怎么样？

⭐️⭐️⭐️⭐️⭐️ 非常满意，干货满满

```
