# 命题作文优化方案模板

## 模板说明

此模板用于类型三项目（命题作文找解法）的方案规划。核心方法是在既定约束条件下找到最优解。

## 核心原则

1. **接受约束** - 约束是既定的，不能无视它重新设计
2. **界定边界** - 明确什么是必须完成的，什么是可以调整的
3. **最优解而非完美解** - 在约束内做到最好，而不是追求完美
4. **管理期望** - 让各方对结果有合理预期

## 命题构成要素

| 要素 | 描述 | 是否可变 |
|------|------|---------|
| 预算 | 可用资金额度 | 有限可变 |
| 时间 | 交付时间节点 | 部分可变 |
| KPI | 考核指标要求 | 不可变 |
| 范围 | 项目涉及的范围 | 可协商 |
| 质量 | 必须达到的标准 | 底线不可低 |
