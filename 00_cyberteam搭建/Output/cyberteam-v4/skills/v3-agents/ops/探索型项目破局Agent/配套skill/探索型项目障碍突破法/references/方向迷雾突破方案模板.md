# 方向迷雾突破方案模板

## 模板说明

此模板用于类型一障碍（方向迷雾）的突破规划。当你不确定方向是否正确时，使用此模板。

## 核心原则

1. **回到价值假设** - 方向迷雾的本质是价值假设不清晰
2. **用验证代替猜测** - 不要猜，用数据验证
3. **设置决策点** - 预设什么情况下坚持，什么情况下放弃

## 常见问题

**Q: 如果假设无法验证怎么办？**
A: 尝试不同的验证方法，或者降低验证标准，或者主动寻找反例。

**Q: 如果老板不同意放弃方向怎么办？**
A: 用数据和验证结果说话，而不是用感觉说话。如果数据支持放弃，应该说服老板接受。

**Q: 如果方向正确但进展慢怎么办？**
A: 方向正确不代表方法正确。进展慢可能是因为执行效率问题，而不是方向问题。
