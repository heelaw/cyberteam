# 项目类型诊断表

## 模板说明

此模板用于快速判断项目属于三类探索型项目中的哪一类，是匹配正确破局策略的前提。

## 诊断问题清单

回答以下5个问题，根据多数答案判断项目类型：

| # | 问题 | 选项A | 选项B | 选项C |
|---|------|-------|-------|-------|
| 1 | 项目与现有业务的关系？ | 完全独立，从零构建 | 利用现有能力抓住新机会 | 在现有框架内优化 |
| 2 | 核心挑战是什么？ | 找到正确的方向 | 抓住时间窗口 | 在约束内最优解 |
| 3 | 成功的关键因素？ | 方法正确，方向正确 | 速度快，执行力强 | 平衡各方，精准交付 |
| 4 | 时间周期？ | 长周期探索 | 短周期窗口 | 按既定周期交付 |
| 5 | 资源策略？ | 逐步加码 | 快速到位 | 精准投放 |

## 诊断规则

- **选项A居多** → 类型一：从0起步独立项目
- **选项B居多** → 类型二：结构性机会快速趟路
- **选项C居多** → 类型三：命题作文找解法

## 三类项目特征对比

### 类型一：从0起步独立项目

| 特征 | 描述 |
|------|------|
| 业务关系 | 与现有业务完全独立 |
| 方向确定性 | 方向未验证，需要探索 |
| 时间预期 | 长周期，不确定多久能成 |
| 常见心态 | 焦虑、迷茫、容易放弃 |
| 核心能力 | 耐心 + 方法论 |
| 典型错误 | 过于急躁、方向摇摆 |

### 类型二：结构性机会快速趟路

| 特征 | 描述 |
|------|------|
| 业务关系 | 利用现有能力或资源 |
| 方向确定性 | 方向基本确定，考验执行 |
| 时间预期 | 短周期，窗口期有限 |
| 常见心态 | 焦虑、担心错过 |
| 核心能力 | 速度 + 执行力 |
| 典型错误 | 过度准备、贻误战机 |

### 类型三：命题作文找解法

| 特征 | 描述 |
|------|------|
| 业务关系 | 在既定框架内 |
| 方向确定性 | 目标明确，路径待定 |
| 时间预期 | 按既定周期 |
| 常见心态 | 受限感、难以施展 |
| 核心能力 | 平衡 + 期望管理 |
| 典型错误 | 无视约束、超出范围 |

## 注意事项

1. **混合类型**：如果答案比较分散，可能项目是混合类型，需要识别主要矛盾
2. **类型转换**：项目可能随着发展从一种类型转换为另一种类型，需要定期重新诊断
3. **禁忌混用**：一旦确定类型，必须采用对应的策略，混用会导致失败
