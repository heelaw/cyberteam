# 团队稳定化行动清单

## 清单说明

此清单用于类型四障碍（团队动摇）的稳定化行动。当团队士气低落、有人想放弃时，使用此清单。

## 团队动摇的信号

**早期信号**：
- 会议参与度下降
- 私下小团体讨论增加
- 对项目进展的抱怨增加
- 开始私下看机会

**严重信号**：
- 有人明确提出离职
- 团队整体士气低落
- 对项目失去信心

## 稳定化行动清单

### 第一步：诊断根本原因

在采取行动之前，先找到团队动摇的根本原因：

| 常见原因 | 诊断问题 |
|---------|---------|
| 方向不清晰 | 团队是否清楚项目的目标和方向？ |
| 进展不理想 | 项目是否取得了预期的进展？ |
| 回报不匹配 | 付出与回报是否成正比？ |
| 沟通不充分 | 项目信息是否透明？ |
| 个人原因 | 是否有个人发展诉求？ |

**诊断方法**：
- 一对一沟通，了解真实想法
- 匿名问卷，了解整体情况
- 回顾历史，找到触发点

### 第二步：重塑愿景和意义

如果动摇的原因是失去方向感和意义感：

**愿景重塑三问**：
1. **用户层面**：这个项目为用户解决了什么问题？
2. **团队层面**：做成这个项目对团队意味着什么？
3. **个人层面**：参与这个项目对个人有什么价值？

**愿景重塑话术**：
> "我知道最近进展不顺，但我们要解决的是[用户问题]，这个价值是真实存在的。只要方向对了，进展慢一点是正常的。"

### 第三步：设置小步胜利

如果团队因为进展慢而动摇，设置一些容易达成的小目标：

| 小目标 | 设置理由 | 完成标准 | 时间 |
|-------|---------|---------|------|
| [目标1] | 让团队有成就感 | [标准] | [日期] |
| [目标2] | 验证某个关键点 | [标准] | [日期] |
| [目标3] | 获得某个里程碑 | [标准] | [日期] |

**小步胜利的原则**：
- 目标必须能在2周内完成
- 目标必须有明确的完成标志
- 完成后必须公开庆祝

### 第四步：公开进展和透明度

如果动摇的原因是信息不透明：

**透明度检查**：
- [ ] 团队是否知道项目的整体进展？
- [ ] 团队是否知道下一步的计划？
- [ ] 团队是否知道决策的依据？
- [ ] 团队是否有渠道提出问题？

**改进措施**：
- 每周项目进展同步
- 重要决策开会说明原因
- 设立匿名反馈渠道

### 第五步：解决核心人员顾虑

对于核心人员（可能离职的人），单独沟通：

**一对一沟通模板**：

| 环节 | 内容 |
|------|------|
| 开场 | "我注意到最近你对这个项目有些疑虑，我想听听你的真实想法" |
| 倾听 | 认真听，不要辩解 |
| 理解 | 复述对方的核心顾虑，表示理解 |
| 回应 | 针对顾虑给出具体解决方案 |
| 共识 | 明确下一步和期望 |

### 第六步：调整激励机制

如果动摇的原因是回报不匹配：

**短期激励**：
- 项目奖金与里程碑挂钩
- 完成关键阶段给予奖励
- 认可和表彰

**长期激励**：
- 成功后的职位发展机会
- 参与核心项目经历背书
- 利益分享

### 第七步：注入新鲜血液（慎用）

如果团队已经精疲力竭，考虑补充新鲜血液：

**注意事项**：
- 新人需要融入时间，短期可能加剧问题
- 优先考虑从内部调配
- 新人最好有类似项目经验

## 团队稳定化行动检查清单

### 紧急处理（1-3天内）

- [ ] 完成一对一沟通，了解真实情况
- [ ] 识别最核心的动摇人员
- [ ] 评估是否需要立即采取行动

### 短期措施（1-2周）

- [ ] 设置小步胜利目标
- [ ] 召开全员会，重塑愿景
- [ ] 改善信息透明度
- [ ] 解决最紧迫的个人顾虑

### 中期措施（1个月）

- [ ] 调整激励机制
- [ ] 优化工作方式
- [ ] 建立持续反馈机制

## 团队稳定化的禁忌

| 禁忌行为 | 后果 |
|---------|------|
| 施压逼迫 | 只会加速离职 |
| 空喊口号 | 失去信任 |
| 视而不见 | 问题恶化 |
| 轻诺寡信 | 失去最后信任 |
| 简单粗暴 | 伤害团队感情 |

---

## 何时应该考虑重组团队

如果以下情况严重，即使稳定了军心也要考虑重组：

1. **核心人员已经离开** - 团队失去主心骨
2. **技能不匹配** - 团队能力无法胜任
3. **文化冲突** - 团队文化有问题
4. **规模需要调整** - 项目规模变化需要不同团队

**重组的注意事项**：
- 提前规划，避免青黄不接
- 做好知识传承
- 妥善处理离职人员
