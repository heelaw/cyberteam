## Critical Rules（必须遵守）

1. **禁止资源摊大饼** - 平均分配等于没有重点
2. **禁止只看短期效益** - 要平衡短期和长期投入
3. **禁止忽视风险储备** - 必须留有应急资源
4. **禁止一次性分配完** - 保留动态调整空间
5. **禁止只看投入产出比** - 还要考虑战略价值和协同效应
6. **禁止忽视人效** - 人力资源是核心资源
7. **禁止脱离业务目标** - 资源配置必须服务业务目标

## Workflow Steps

### Step 1: 资源盘点

**输入**:
- 可用预算总额
- 人力资源情况
- 时间资源
- 外部资源支持

**处理**:
1. 盘点各类资源总量
2. 评估资源质量
3. 识别资源约束条件
4. 明确资源的灵活性和刚性

**输出**:
- 资源清单表
- 资源约束说明
- 资源灵活性评估

**成功标准**: 资源种类完整，数量准确

---

### Step 2: 目标分解

**输入**:
- 业务目标
- 资源清单
- 历史数据

**处理**:
1. 分解年度/季度目标到具体任务
2. 评估每个任务对资源的需求
3. 识别任务间的依赖关系
4. 确定目标优先级

**输出**:
- 目标分解表
- 任务资源需求表
- 目标优先级排序
- 任务依赖图

**成功标准**: 目标分解到可执行级别，优先级明确

---

### Step 3: 效益评估与分配

**输入**:
- 资源清单
- 目标优先级
- 任务需求

**处理**:
1. 建立效益评估维度
2. 计算各任务的投入产出比
3. 评估战略价值和协同价值
4. 制定初步分配方案
5. 校验资源约束

**输出**:
- 效益评估矩阵
- 资源分配方案
- 资源配置说明

**成功标准**: 分配方案有明确依据，符合资源约束

---

### Step 4: 动态调整机制

**输入**:
- 分配方案
- 执行数据
- 市场变化

**处理**:
1. 建立资源使用监测机制
2. 设置调整触发条件
3. 制定应急储备使用规则
4. 定期回顾和调整

**输出**:
- 监测指标体系
- 调整触发条件
- 储备金使用规则
- 定期回顾机制

**成功标准**: 有机制能及时调整资源分配
