# SMART目标模板

## 模板说明
用于将模糊的目标转化为SMART化的具体目标。

## 模板内容

```
## 目标名称
[给这个目标起一个清晰的名字]

## SMART化分析

### S - 具体性（Specific）
- **目标内容**：我要达成的具体结果是什么？
- **答案**：[填写]

### M - 可衡量性（Measurable）
- **衡量指标**：如何衡量目标是否达成？
- **指标值**：[具体的数字或标准]
- **数据来源**：[从哪里获取数据]

### A - 可实现性（Achievable）
- **资源条件**：实现这个目标需要什么资源？
- 现有资源：[列出已有资源]
- 缺口资源：[列出需要补充的资源]
- **可行性评估**：[高/中/低 及理由]

### R - 相关性（Relevant）
- **战略关联**：这个目标与公司/团队的什么战略相关？
- [填写]
- **优先级**：与其他目标相比，这个目标的优先级？
- [高/中/低]

### T - 时限性（Time-bound）
- **截止日期**：[具体日期]
- **里程碑**：
  - 里程碑1：[日期] - [目标]
  - 里程碑2：[日期] - [目标]
- **时间约束说明**：[如有特殊时间要求]

## SMART目标最终表述

> [将以上要素整合成1-2句话的SMART目标]

## 验收标准
- [ ] 标准1：[具体描述]
- [ ] 标准2：[具体描述]
- [ ] 标准3：[具体描述]
```

## 使用说明
1. 先填写目标名称
2. 逐项分析SMART的每个要素
3. 在"SMART目标最终表述"中整合成完整句子
4. 最后制定验收标准
