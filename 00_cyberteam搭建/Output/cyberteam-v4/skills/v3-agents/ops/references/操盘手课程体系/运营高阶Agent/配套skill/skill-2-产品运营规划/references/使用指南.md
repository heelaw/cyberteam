# 参考资料：产品运营规划

## 核心概念

### 产品生命周期四阶段

1. **探索期**：找Product-Market Fit，用户缓慢增长
2. **成长期**：快速扩大用户规模，竞争加剧
3. **成熟期**：增长放缓，深挖用户价值和商业化
4. **衰退期**：用户流失，考虑转型或收缩

### 5种核心因素

1. **产品阶段**：当前处于哪个生命周期阶段
2. **用户特征**：核心用户是谁，需求是什么
3. **竞争环境**：直接竞品有哪些，差异化在哪
4. **资源条件**：人、钱、资源有多少
5. **团队能力**：擅长什么，执行力如何

### 运营破局点选择原则

- 探索期：验证产品价值，打造PMF
- 成长期：快速放大，差异化突破
- 成熟期：深耕价值，探索新增长
- 衰退期：转型收缩，挖掘剩余价值
