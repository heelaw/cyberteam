## 案例一：爆款文章转载判断

### 案例背景
- **文章标题**：《数字化转型的三大误区，90%的企业都踩了》
- **原创平台**：36氪
- **发布时间**：2024年3月10日
- **阅读量**：58000
- **账号平均阅读量**：12000

### 四维度分析

#### 1. 阅读量对比
- 对比指数：(58000 ÷ 12000) × 100% = 483%
- **判断**：远高于平均，爆款文章

#### 2. 用户定位匹配度
- 账号定位：企业服务、数字化转型
- 文章主题：企业数字化转型误区
- **评分**：5/5分，完全匹配

#### 3. 用户消费程度
- 结构：总分总结构，条理清晰
- 信息：三大误区，每个有案例说明
- 可读性：专业但易懂
- **评级**：高

#### 4. 是否被埋没好文
- 数据表现：58000阅读量已证明传播效果
- **判断**：非埋没好文，数据说明一切

### 综合结论
**转载建议：强烈推荐**
- 理由：数据表现优秀，定位完全匹配，内容质量高

## 案例二：小众优质文章识别

### 案例背景
- **文章标题**：《如何用OKR在一周内完成年度规划》
- **原创平台**：某职场博主（粉丝5000）
- **发布时间**：2024年2月25日
- **阅读量**：980
- **账号平均阅读量**：1500

### 四维度分析

#### 1. 阅读量对比
- 对比指数：(980 ÷ 1500) × 100% = 65%
- **判断**：低于平均

#### 2. 用户定位匹配度
- 账号定位：职场成长、效率提升
- 文章主题：OKR年度规划方法
- **评分**：4/5分，高度匹配

#### 3. 用户消费程度
- 结构：小号主精心打磨的内容
- 信息：实操性强，有模板有案例
- 可读性：通俗易懂，干货满满
- **评级**：高

#### 4. 是否被埋没好文
- 分析：小号主影响力有限，但内容质量远超平均
- 证据：高收藏转发比（收藏180，转发67）
- **判断**：是被埋没的优质内容

### 综合结论
**转载建议：推荐**
- 理由：虽然阅读量低，但内容质量高，定位匹配，属于被埋没好文，值得转载

## 案例三：谨慎判断的场景

### 案例背景
- **文章标题**：《震惊！这个行业即将颠覆》
- **原创平台**：某行业自媒体
- **发布时间**：2024年3月12日
- **阅读量**：45000
- **账号平均阅读量**：8000

### 四维度分析

#### 1. 阅读量对比
- 对比指数：(45000 ÷ 8000) × 100% = 562%
- **判断**：数据异常高

#### 2. 用户定位匹配度
- 账号定位：行业深度分析
- 文章主题：行业趋势预判
- **评分**：3/5分，中度匹配

#### 3. 用户消费程度
- 结构：标题党痕迹明显
- 信息：缺乏数据支撑，观点激进
- 可读性：煽动性强但空洞
- **评级**：低

#### 4. 是否被埋没好文
- 分析：阅读量高但疑似标题党
- **判断**：非埋没好文，数据可能有水分

### 综合结论
**转载建议：观望/不推荐**
- 理由：虽然数据好看，但标题党嫌疑大，消费程度低，需谨慎

## 案例四：时效性判断案例

### 案例背景
- **文章标题**：《2023年最新社保政策解读》
- **原创平台**：政策解读类大号
- **发布时间**：2023年1月
- **阅读量**：32000
- **账号平均阅读量**：5000

### 判断要点

1. **时效性风险**：文章发布于2023年1月，政策可能已有更新
2. **执行判断**：需核实政策是否仍在有效期内
3. **最终建议**：若政策已更新则不推荐；若内容为长期有效政策可考虑推荐

### 经验总结

- **教训**：数据好的文章不一定值得转载，要考虑时效性
- **改进**：增加第五维度"时效性评估"作为常规检查项
