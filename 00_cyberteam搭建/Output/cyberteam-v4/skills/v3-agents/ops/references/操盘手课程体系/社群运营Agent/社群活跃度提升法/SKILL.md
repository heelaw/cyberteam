## Critical Rules（必须遵守）

1. **禁止只发广告** - 社群价值在于内容，广告只能占20%以下
2. **禁止机器人刷屏** - 虚假活跃伤害用户信任，得不偿失
3. **禁止忽视沉默用户** - 80%的沉默用户才是最大增长空间
4. **禁止无节奏运营** - 发布时间和频率需要固定，形成用户预期
5. **禁止忽视新人** - 新人是社群活力重要来源，必须有欢迎机制
6. **禁止群主一言堂** - 培养KOC，让用户影响用户
7. **禁止只看数量** - 活跃度核心是质量而非数量

## Workflow Steps

**输入**:

**处理**:
1. 分析活跃用户占比和层级分布
2. 识别沉默用户比例和原因

4. 找出活跃度低下的核心问题

**输出**:
- 社群活跃度诊断报告
