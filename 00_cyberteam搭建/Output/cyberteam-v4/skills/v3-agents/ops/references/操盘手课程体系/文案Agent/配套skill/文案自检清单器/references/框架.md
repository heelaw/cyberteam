# 文案自检清单法 - 方法论

## 核心框架

文案自检是发布前的最后一道防线，通过系统化的检查清单，确保文案达到发布标准。

## 原理说明

### 四维检查体系

#### 1. 结构检查
**核心**: 确保文案结构完整清晰

**检查点**:
- 标题：是否完整、是否有吸引力
- 开头：是否能抓住注意力
- 主体：逻辑是否清晰、层次是否分明
- 结尾：是否有CTA、是否有力

#### 2. 内容检查
**核心**: 确保内容准确无误

**检查点**:
- 事实：信息是否准确
- 数据：数据是否真实可查证
- 承诺：承诺是否合理可实现
- 表述：是否清晰无歧义

#### 3. 格式检查
**核心**: 确保格式规范统一

**检查点**:
- 错别字：逐字检查
- 标点：使用正确
- 段落：长度适中
- 字数：符合要求
- 敏感词：检查敏感词库

#### 4. 合规检查
**核心**: 确保符合法律法规

**检查点**:
- 广告法：不使用违禁词
- 知识产权：不侵犯版权
- 竞品提及：合规
- 隐私政策：符合规定
- 行业规定：符合监管要求

## 自检流程

### 自检步骤
1. 打印/调出文案
2. 逐项对照检查
3. 发现问题记录
4. 修改后复查
5. 确认通过后发布

### 常见问题

#### 结构问题
- 标题平淡无吸引力
- 开头冗长抓不住人
- 主体逻辑混乱
- 结尾无CTA或CTA软弱

#### 内容问题
- 事实性错误
- 数据夸大或虚假
- 承诺无法兑现
- 表述有歧义

#### 格式问题
- 错别字
- 标点错误
- 段落过长
- 格式不统一

#### 合规问题
- 使用违禁词（最好、最低、第一等）
- 侵犯知识产权
- 竞品对比不当
- 隐私数据使用不当

## 自检技巧

### 错别字检查
- 从头到尾逐字读
- 谐音字检查
- 形近字检查
- 反复确认不确定的字

### 数据检查
- 数据来源可查证
- 数据时间最新
- 数据对比合理
- 百分比计算正确

### 合规检查
- 对照广告法违禁词
- 确认版权授权
- 竞品只描述不贬低
- 隐私数据脱敏处理
