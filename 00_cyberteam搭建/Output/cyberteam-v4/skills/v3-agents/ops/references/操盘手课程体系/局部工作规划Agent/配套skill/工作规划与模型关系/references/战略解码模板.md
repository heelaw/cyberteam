## 四、冲突识别与解决

### 常见冲突类型
1. **资源冲突**：多目标争夺有限资源
2. **优先级冲突**：目标重要性排序不一致
3. **方法冲突**：达成路径有不同的看法
4. **指标冲突**：不同指标此消彼长

### 冲突解决流程
```
Step 1: 识别冲突
  - 冲突描述：____________
  - 涉及方：____________
  - 影响程度：□ 高 □ 中 □ 低

Step 2: 分析原因
  - 根本原因：____________
  - 双方立场：____________

Step 3: 寻求方案
  - 方案A：____________
  - 方案B：____________
  - 方案C：____________

Step 4: 决策执行
  - 选择方案：____________
  - 决策依据：____________
  - 执行责任：____________

Step 5: 跟踪反馈
  - 效果评估：____________
  - 调整需求：____________
```

## 五、模板使用说明

### 使用场景
- 年度战略解码
- 季度目标制定
- 新业务规划
- 组织调整后的目标重设

### 注意事项
1. 每一步都要认真完成，不要跳过
2. 对齐沟通要留下记录
3. 冲突解决要有闭环
4. 定期检视目标映射的有效性
