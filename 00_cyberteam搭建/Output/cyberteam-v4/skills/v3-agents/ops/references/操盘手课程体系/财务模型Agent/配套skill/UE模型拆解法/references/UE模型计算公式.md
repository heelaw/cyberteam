# UE模型计算公式汇总

## 核心公式

### LTV（用户生命周期价值）

**基本公式**：
```
LTV = 客单价 × 复购次数 × 毛利率
```

**月度公式**：
```
LTV = ARPU × 毛利率 × 用户生命周期(月)
```

**精确公式**：
```
LTV = Σ (月ARPU × 留存率 × 毛利率) / (1+月折现率)^n
```

### CAC（用户获取成本）

**基本公式**：
```
CAC = 总营销费用 ÷ 新增用户数
```

**完整成本公式**：
```
CAC = (广告费用 + 销售成本 + 营销人员成本 + 工具成本) ÷ 新增用户数
```

### ROI（投资回报率）

**基本公式**：
```
ROI = LTV ÷ CAC
```

### 回本周期

**公式**：
```
回本周期 = CAC ÷ (月ARPU × 毛利率 - 月运营成本)
```

## 行业参考值

### 电商

| 指标 | 优秀 | 良好 | 一般 |
|------|------|------|------|
| LTV/CAC | >3 | 1-3 | <1 |
| 回本周期 | <6个月 | 6-12个月 | >12个月 |

### SaaS

| 指标 | 优秀 | 良好 | 一般 |
|------|------|------|------|
| LTV/CAC | >5 | 3-5 | <3 |
| 月流失率 | <5% | 5-10% | >10% |

### 在线教育

| 指标 | 优秀 | 良好 | 一般 |
|------|------|------|------|
| LTV/CAC | >3 | 1-3 | <1 |
| 续费率 | >40% | 20-40% | <20% |

## 常见错误

1. 只算广告费，忽略销售成本
2. 用收入而非毛利计算LTV
3. 忽略用户生命周期的不确定性
4. 只看短期ROI，忽略长期价值
