# 业务模型三层次分析卡

## 概述

业务模型可分为三个层次，从宏观到微观，从战略到执行。

```
┌─────────────────────────────────────────────┐
│              战略层（What）                  │
│         我们做什么？方向与定位                │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│              运营层（How）                   │
│         我们怎么做？流程与方法                │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│              收入层（Profit）                │
│         我们怎么赚？盈利与财务                │
└─────────────────────────────────────────────┘
```

---

## 层次一：战略层（What）

### 核心问题

| 问题 | 说明 |
|------|------|
| 我们做什么业务？ | 业务边界和范围 |
| 我们的用户是谁？ | 目标用户定位 |
| 我们提供什么价值？ | 价值主张定位 |
| 我们去哪里？ | 战略目标方向 |

### 战略层要素

#### 1. 业务定位

```
业务定位公式：
"为 [目标用户] 提供 [价值主张] 的 [业务类型]"
示例： 为25-35岁职场人提供技能提升的在线教育平台
```

#### 2. 战略目标

| 时间维度 | 目标类型 | 示例 |
|----------|----------|------|
| 长期(3-5年) | 愿景目标 | 成为行业头部 |
| 中期(1-3年) | 发展阶段 | 达到百万用户 |
| 短期(当年) | 运营目标 | 用户增长100% |

#### 3. 竞争策略

| 策略类型 | 说明 | 适用场景 |
|----------|------|----------|
| 成本领先 | 以低价取胜 | 规模化市场 |
| 差异化 | 独特价值取胜 | 细分市场 |
| 集中化 | 聚焦特定群体 | 垂直市场 |

### 战略层检核

```
[ ] 业务边界清晰
[ ] 目标用户明确
[ ] 价值主张独特
[ ] 战略目标可行
[ ] 竞争策略匹配
```

## 层次二：运营层（How）

### 核心问题

| 问题 | 说明 |
|------|------|
| 我们怎么做业务？ | 运营模式 |
| 关键流程是什么？ | 业务流程 |
| 渠道有哪些？ | 渠道策略 |
| 资源如何配置？ | 资源配置 |

### 运营层要素

#### 1. 运营模式

| 模式类型 | 特征 | 代表案例 |
|----------|------|----------|
| 自营模式 | 全链路自己完成 | 京东自营 |
| 平台模式 | 撮合双边 | 淘宝、滴滴 |
| 混合模式 | 自营+平台结合 | 美团外卖 |

#### 2. 业务流程

```
获客流程：
触达 → 认知 → 兴趣 → 激活 → 成交

转化流程：
曝光 → 点击 → 注册 → 首次使用 → 复购

交付流程：
下单 → 支付 → 履约 → 评价 → 复购
```

#### 3. 渠道策略

| 渠道类型 | 作用 | 策略重点 |
|----------|------|----------|
| 获客渠道 | 获取新用户 | 精准投放 |
| 转化渠道 | 促进成交 | 促销活动 |
| 留存渠道 | 维护老用户 | 会员体系 |

#### 4. 资源配置

| 资源维度 | 配置原则 | 优化方向 |
|----------|----------|----------|
| 人力 | 按业务优先级 | 核心岗位 |
| 资金 | 按阶段重点 | ROI导向 |
| 技术 | 支撑业务发展 | 中台能力 |

### 运营层检核

```
[ ] 运营模式清晰
[ ] 业务流程顺畅
[ ] 渠道策略有效
[ ] 资源配置合理
[ ] 运营指标可追踪
```

## 层次三：收入层（Profit）

### 核心问题

| 问题 | 说明 |
|------|------|
| 我们怎么赚钱？ | 盈利模式 |
| 收入从哪里来？ | 收入结构 |
| 成本有哪些？ | 成本结构 |
| 利润率如何？ | 盈利能力 |

### 收入层要素

#### 1. 盈利模式

| 盈利模式 | 说明 | 关键指标 |
|----------|------|----------|
| 卖产品 | 一次性销售 | 销量、毛利率 |
| 卖服务 | 周期性服务 | 复购率、续费率 |
| 卖资源 | 流量/数据变现 | CPM、ARPU |
| 卖模式 | 加盟/代理 | 加盟费、分成 |

#### 2. 收入结构

```
收入结构分析：
├── 主营业务收入
│   ├── 产品/服务A：占比X%，增长率Y%
│   ├── 产品/服务B：占比X%，增长率Y%
│   └── 产品/服务C：占比X%，增长率Y%
├── 其他业务收入
│   ├── 增值服务：占比X%
│   └── 广告收入：占比X%
└── 收入合计：XX万元/月
```

#### 3. 成本结构

| 成本类型 | 占比 | 优化空间 |
|----------|------|----------|
| 固定成本 | 40% | 规模化后降低 |
| 变动成本 | 50% | 优化效率 |
| 营销成本 | 10% | ROI优化 |

#### 4. 盈利能力

| 指标 | 计算公式 | 健康值 |
|------|----------|--------|
| 毛利率 | (收入-成本)/收入 | >30% |
| 净利率 | 净利润/收入 | >10% |
| LTV | 用户生命周期价值 | >3倍CAC |
| CAC | 获客成本 | <1/3 LTV |

### 收入层检核

```
[ ] 盈利模式可行
[ ] 收入结构健康
[ ] 成本控制合理
[ ] 单位经济模型正向
[ ] 盈亏平衡可预期
```

## 三层次关联

### 战略→运营→收入传导

```
战略层（做什么）
    ↓ 决定
运营层（怎么做）
    ↓ 支撑
收入层（赚多少）
```

### 层级对话

| 层级 | 问上层 | 问下层 |
|------|--------|--------|
| 战略层 | "为什么做这个业务？" | "这样做能实现战略吗？" |
| 运营层 | "战略需要什么样的运营？" | "运营能带来多少收入？" |
| 收入层 | "运营需要多少资源？" | "收入能覆盖成本吗？" |

## 实战：三层次诊断

### 案例：某在线教育平台

#### 战略层诊断
- 业务定位：为职场人提供技能提升课程
- 问题：课程同质化严重，差异化不足

#### 运营层诊断
- 运营模式：平台+自营混合
- 问题：获客成本过高，转化率偏低

#### 收入层诊断
- 收入结构：课程收入为主，续费率低
- 问题：过于依赖单一收入来源

#### 综合建议
1. 战略层：聚焦垂直领域，建立差异化
2. 运营层：优化获客渠道，提升转化
3. 收入层：发展会员订阅，提高续费率
