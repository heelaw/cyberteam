## 情绪标签词典

### 正面情绪

| 标签 | 描述 | 常见表达 |
|------|------|---------|
| 开心 | 愉悦的心情 | "太好了"、"很满意" |
| 兴奋 | 强烈的愉悦 | "太棒了"、"超出预期" |
| 惊喜 | 意外的正向 | "没想到"、"居然" |
| 安心 | 安全感和信任 | "放心了"、"靠谱" |
| 自豪 | 成就感 | "炫耀"、"分享" |

### 中性情绪

| 标签 | 描述 | 常见表达 |
|------|------|---------|
| 平淡 | 无明显情绪 | "还行"、"一般" |
| 困惑 | 不理解 | "怎么用"、"不懂" |
| 犹豫 | 决策困难 | "要不要"、"纠结" |

### 负面情绪

| 标签 | 描述 | 常见表达 |
|------|------|---------|
| 不满 | 不满意 | "不好"、"失望" |
| 愤怒 | 强烈不满 | "太差了"、"垃圾" |
| 焦虑 | 不安的担心 | "怎么办"、"担心" |
| 挫败 | 失败感 | "太难了"、"用不了" |
| 无奈 | 无力感 | "没办法"、"只能这样" |
