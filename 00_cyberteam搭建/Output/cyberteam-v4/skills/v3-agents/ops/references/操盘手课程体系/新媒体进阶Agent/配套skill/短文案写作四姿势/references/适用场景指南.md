# 适用场景指南

## 一、姿势选择原则

### 根据产品选择

| 产品类型 | 推荐姿势 | 说明 |
|----------|----------|------|
| 新产品 | 痛点开头 | 先解决认知问题 |
| 爆款产品 | 数字钩子 | 用数据增加可信度 |
| 高客单价 | 场景代入 | 强调价值 |
| 礼品类 | 社交货币 | 强调面子 |

### 根据用户选择

| 用户特征 | 推荐姿势 | 说明 |
|----------|----------|------|
| 痛点明确 | 痛点开头 | 直接回应需求 |
| 理性消费者 | 数字钩子 | 需要数据说服 |
| 女性用户 | 场景代入 | 感性代入 |
| 年轻用户 | 社交货币 | 喜欢分享 |

### 根据场景选择

| 使用场景 | 推荐姿势 | 说明 |
|----------|----------|------|
| 短视频 | 痛点开头 | 前3秒抓住注意力 |
| 海报 | 数字钩子 | 一眼看到重点 |
| 朋友圈 | 场景代入 | 引发共鸣 |
| 社群 | 社交货币 | 引发讨论 |

## 二、平台特性适配

### 抖音/快手

**特点**：时长短、节奏快、娱乐性强

**建议**：
- 痛点开头（前3秒）
- 数字钩子（强调效果）
- 场景代入（画面感强）
- 语速快、情绪足

### 小红书

**特点**：种草属性强、女性用户多

**建议**：
- 场景代入（画面感）
- 社交货币（晒单感）
- 真实感、生活化
- 适合"测评"形式

### 微信朋友圈

**特点**：私密性强、信任度高

**建议**：
- 场景代入（日常感）
- 案例分享（真实感）
- 不宜太营销
- 注重互动

### 淘宝/京东

**特点**：购买意向明确

**建议**：
- 数字钩子（销量、好评）
- 痛点开头（需求匹配）
- 效果承诺
- 促销信息

## 三、行业应用

### 美妆护肤

**首选姿势**：痛点开头 + 场景代入

**文案示例**：
```
皮肤暗沉？这款精华用完第二天就见效
```

### 职场教育

**首选姿势**：数字钩子 + 痛点开头

**文案示例**：
```
3个月从小白到入门，已有5万人学会
```

### 食品餐饮

**首选姿势**：场景代入 + 社交货币

**文案示例**：
```
朋友圈都在打卡的店，你还没来过？
```

### 母婴用品

**首选姿势**：场景代入 + 数字钩子

**文案示例**：
```
10个妈妈9个推荐，温和不刺激
```

### 金融理财

**首选姿势**：数字钩子 + 权威背书

**文案示例**：
```
年化收益XX%，已服务XX万用户
```

## 四、注意事项

### 避免过度

- 数字要真实：不要夸大
- 场景要合理：不要夸张
- 社交要适度：不要炫耀

### 持续优化

- 测试不同姿势
- 数据反馈调整
- 保持更新迭代

### 合规要求

- 不虚假宣传
- 不违规承诺
- 遵守平台规则
