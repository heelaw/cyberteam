# 文案自检清单

## 标题自检

| 检查项 | 是 | 否 | 说明 |
|--------|----|----|------|
| 有明确的好处承诺 | □ | □ | |
| 目标用户一眼看得懂 | □ | □ | |
| 区别于其他同类内容 | □ | □ | |
| 引发好奇心或共鸣 | □ | □ | |
| 没有虚假夸大 | □ | □ | |

### 优质标题公式

```
[人群] + [痛点] + [解决方案] + [效果]
例子：转行产品经理必看 | 0基础3个月拿offer的秘诀
```

## 开头自检

| 检查项 | 是 | 否 | 说明 |
|--------|----|----|------|
| 开头能抓住注意力 | □ | □ | |
| 和目标用户相关 | □ | □ | |
| 引发继续阅读的欲望 | □ | □ | |
| 没有自嗨 | □ | □ | |

### 优质开头类型

1. **痛点型**：描述用户痛苦场景
2. **故事型**：分享真实经历
3. **数据型**：用惊人数据吸引
4. **疑问型**：提出用户关心的问题
5. **对比型**：展示改变前后的差异

### 痛点描述技巧

- 用具体数字而非模糊表述
- 描述具体场景而非抽象概念
- 挖掘深层心理需求

### 卖点vs买点

```
❌ 卖点思维：我们的产品是316不锈钢
✅ 买点思维：食品级材质，喝水更安心

❌ 卖点思维：课程有20个章节
✅ 买点思维：系统学习2个月成为产品经理
```

## CTA（行动号召）自检

| 检查项 | 是 | 否 | 说明 |
|--------|----|----|------|
| CTA明确具体 | □ | □ | |
| 告诉用户下一步做什么 | □ | □ | |
| 有紧迫感或合理性 | □ | □ | |
| 放在显眼位置 | □ | □ | |
| 只有一个主要CTA | □ | □ | |

### CTA文案技巧

```
❌ 不明确：了解更多 → ✅ 具体：免费领取课程大纲
❌ 没紧迫感：立即购买 → ✅ 有理由：前100名送1v1咨询
❌ 太复杂：点击这里购买 → ✅ 简单：立即0元试听
```
