# promotion-channel-selector

## 简介

根据产品特点和目标用户，选择最匹配的推广渠道。本Skill覆盖四种主流推广方式：媒介资源购买占有型、信息分发平台流量推广、传播事件型推广、精准营销效果广告。

## 触发关键词

```
推广渠道、渠道选择、媒介资源、信息分发平台、传播事件、精准营销、效果广告、投放平台、推广方式
```

## 触发场景描述

```
当需要做出以下决策时触发本Skill：
- 选择哪个推广渠道/平台
- 4种推广方式中哪种最适合
- 如何在预算有限时选择最高效的渠道
- 如何判断一个陌生平台的推广规则
```

## 输入

### 必需输入

| 字段 | 类型 | 说明 |
|------|------|------|
| user_persona | object | 目标用户画像（来自user-targeting-analyzer） |
| product_info | string | 产品/服务基本信息 |
| product_stage | string | 产品阶段（初创/成长/成熟） |

### 可选输入

| 字段 | 类型 | 说明 |
|------|------|------|
| budget | string | 推广预算范围 |
| existing_channels | array | 已尝试过的推广渠道 |
| competitor_channels | array | 竞品常用渠道 |

## 方法论：四类推广方式

### 1. 媒介资源购买占有型推广

**定义：** 购买特定的广告位或媒介资源

**操作流程：**
1. 找到广告渠道或媒介公司
2. 获取资源List及报价
3. 洽谈资源、价格和排期
4. 准备物料：图片、文案等
5. 完成投放

**特点：**
- 曝光量基本恒定
- 点击基本靠文案和推广物料
- 转化取决于流程和文案、物料

**适用场景：** 有明确预算、大品牌转化

### 2. 信息分发平台流量推广

**定义：** 在信息流平台（如今日头条、百度、应用商店）推广

**操作步骤：**

**方式一：研究既定规则和持续优化**
1. 研究对应渠道信息展示排序规则
2. 准备物料，调整信息发布形式&细节
3. 持续优化：排在前五位流量才会很大
4. 持续跟踪优化

**方式二：主动抱大腿，蹭流量**
1. 找到渠道负责人
2. 获取近期官方推广活动等信息
3. 准备物料，对接渠道，加入活动
4. 活动上线，优化调整
5. 推广完成

**特点：**
- 参与活动式推广曝光量基本恒定
- 信息内容优化式推广曝光量取决于优化效果
- 点击基本靠文案

**适用场景：** 需要持续优化、预算有限

### 3. 传播事件型推广

**定义：** 通过策划事件/话题引发传播

**操作流程：**
1. 策划、构思
2. 资源整合，物料准备
3. 执行落地
4. 观察效果，后期调整

**特点：**
- 曝光量基本靠策划能力和运气
- 点击量基本靠策划能力和运气
- 转化取决于转化逻辑和流程

**适用场景：** 品牌传播、话题营销

### 4. 精准营销效果广告

**定义：** 付费广告投放，可精确选择投放参数

**操作步骤：**
1. 开通投放平台账号，熟悉投放平台规则
2. 配置投放规则及预算
3. 上线测试，不断调整（主要看ROI）
4. 完成投放

**计费方式：**
- CPM：千次展示成本
- CPC：每个点击成本
- CPA：按转化行为计费
- CPS：按实际销售数量计费
- CPT：按时长计费

**适用场景：** 效果导向、需要精确 Targeting

## 工作流步骤

### Step 1: 分析目标用户渠道偏好

- **输入:** user_persona
- **处理:** 根据用户画像识别用户聚集的平台和搜索行为
- **输出:** 用户渠道偏好清单
- **成功标准:** 列出3-5个用户最可能出现的渠道

### Step 2: 评估四种推广方式

- **输入:** user_persona, product_info, product_stage, budget
- **处理:** 根据四类推广方式的特点匹配产品阶段和预算
- **输出:** 四种方式的适用性评分
- **成功标准:** 识别最适合的1-2种方式

### Step 3: 选择具体渠道

- **输入:** 用户渠道偏好, 方式评估结果, competitor_channels
- **处理:** 结合用户偏好、竞品策略、产品特点选择具体渠道
- **输出:** 优先级排序的渠道列表
- **成功标准:** 明确主推渠道和辅助渠道

### Step 4: 制定渠道规则

- **输入:** 选定的渠道列表
- **处理:** 梳理每个渠道的投放规则、计费方式、优化要点
- **输出:** 渠道规则速查表
- **成功标准:** 可直接用于投放执行

## Critical Rules

### 必须遵守

1. 先有用户画像，再选渠道
2. 预算有限时聚焦单一渠道做透
3. 了解渠道规则后再开始投放
4. 4种方式可以组合使用，但要有主次

### 禁止行为

1. 禁止不看用户画像就选渠道
2. 禁止同时铺太多渠道导致资源分散
3. 禁止不了解渠道规则就投放
4. 禁止只看曝光量忽视转化率
5. 禁止盲目复制竞品渠道策略

## 评估清单

### 评估维度

| 维度 | 标准 | 权重 |
|------|------|------|
| 用户匹配度 | 渠道是否与目标用户匹配 | 30% |
| 预算匹配度 | 成本是否在预算范围内 | 20% |
| 产品阶段匹配 | 方式是否与产品阶段匹配 | 15% |
| 执行可行性 | 团队是否有能力操作该渠道 | 20% |
| 数据可追踪性 | 能否有效监控推广效果 | 15% |

### 评估标准

- **90-100分：** 五个维度全部达标，渠道选择精准
- **75-89分：** 四个维度达标，渠道选择合理
- **60-74分：** 三个维度达标，需要调整
- **<60分：** 渠道选择有问题，需要重新分析
