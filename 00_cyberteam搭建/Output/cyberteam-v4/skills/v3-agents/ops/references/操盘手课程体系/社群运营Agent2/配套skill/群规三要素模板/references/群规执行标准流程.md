# 群规执行标准操作流程

## 执行流程总览

```
发现违规行为
    ↓
判断违规等级（严重/一般/轻微）
    ↓
根据等级执行对应处理流程
    ↓
记录处理结果
    ↓
定期复盘优化规则
```

---

## 一、违规行为判定标准

### 严重违规（立即处理）

| 行为类型 | 判断标准 | 处理方式 |
|---------|---------|---------|
| 发广告 | 包含联系方式、购买链接、二维码 | 立即移出 |
| 拉人 | 主动邀请他人加入其他群 | 立即移出 |
| 诈骗 | 涉及金钱诈骗、虚假信息 | 立即移出+举报 |
| 人身攻击 | 直接辱骂、威胁他人 | 立即移出 |
| 泄露隐私 | 未经授权公开群内信息 | 立即移出 |

### 一般违规（警告处理）

| 行为类型 | 判断标准 | 处理方式 |
|---------|---------|---------|
| 软广 | 看似分享实为推广 | 私聊警告+删除内容 |
| 重复提问 | 频繁发送相同内容 | 私聊提醒 |
| 无关内容 | 频繁发布与主题无关内容 | 私聊警告 |
| 负面情绪 | 频繁抱怨、传播负能量 | 私聊引导 |

### 轻微违规（提醒处理）

| 行为类型 | 判断标准 | 处理方式 |
|---------|---------|---------|
| 轻微刷屏 | 短时间内发送多条消息 | 私聊提醒 |
| 格式不当 | 发布内容格式混乱 | 善意提醒 |

## 二、执行操作流程

### Step 1: 发现违规

**来源**:
- 管理员日常巡查发现
- 群内成员举报
- 系统自动监测

**动作**:
1. 截图保存证据
2. 记录违规时间、行为、内容
3. 初步判断违规等级

### Step 2: 判断等级

```
判断树：

是严重违规吗？
├── 是 → 直接移出（走Step 3.1）
└── 否 ↓
是 一般违规 吗？
├── 是 → 警告处理（走Step 3.2）
└── 否 ↓
轻微违规 → 提醒处理（走Step 3.3）
```

### Step 3: 执行处理

#### 3.1 严重违规处理

```
⛔ 执行操作：
1. 管理员确认截图证据
2. 移出社群
3. 群内发送处理通知（匿名）
4. 加入黑名单

📝 通知文案：
该成员因违反群规已被移出社群。
感谢理解。

⚠️ 注意事项：
- 处理要迅速但不要冲动
- 保留证据以备申诉
- 不要在群内引发骂战
```

#### 3.2 一般违规处理

```
⚠️ 执行操作：
1. 私聊发送警告通知
2. 说明违规内容和后果
3. 告知整改要求
4. 记录处理结果

📝 私聊话术：
您好，[昵称]！

我们在群内监测到您在[时间]发布了[内容描述]，
该行为已违反群规[具体条款]。

请注意：
- 这是您的第[X]次违规
- 下次违规将[后果]
- 如有异议可在24小时内申诉

请配合遵守群规，共同维护社群秩序。
```

#### 3.3 轻微违规处理

```
💬 执行操作：
1. 私聊或@友善提醒
2. 简短说明原因
3. 给出正确做法示例

📝 提醒话术：
@[昵称] 温馨提醒：
[简短说明情况]
建议：[正确做法]
```

## 三、申诉处理流程

### 申诉条件
- 用户对违规判定有异议
- 收到申诉后48小时内处理

### 申诉流程

```
收到申诉
    ↓
复核证据
    ↓
判断申诉是否成立
    ↓
┌── 成立 → 撤销处理+道歉
└── 不成立 → 维持原判+说明原因
```

### 申诉回复模板

```
【申诉回复】

您的申诉已处理。

复核结果：[成立/不成立]

[详细说明原因]

如有疑问可联系：[上级联系方式]
```

## 五、常见问题处理

### Q1: 误判了怎么办？

**处理方式**:
1. 立即道歉
2. 撤销处理
3. 恢复成员资格
4. 优化判断标准

### Q2: 成员威胁要曝光群聊记录？

**处理方式**:
1. 保持冷静，不要激化
2. 说明群规条款
3. 如果涉及隐私保护，可以法律警告
4. 记录证据，必要时法律维权

### Q3: 违规者是VIP/付费用户怎么办？

**处理方式**:
1. 规则面前人人平等
2. 违规处理与身份无关
3. 可以酌情考虑但不能免于处理
4. 做好沟通，避免退款纠纷
