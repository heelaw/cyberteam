## 进度预警规则

| 预警等级 | 条件 | 响应措施 |
|----------|------|----------|
| 红灯 | 进度落后>20%或关键路径延期 | 立即升级汇报，重新评估资源 |
| 黄灯 | 进度落后10-20% | 增加监控频率，制定补救计划 |
| 绿灯 | 进度正常（±10%） | 正常推进 |
