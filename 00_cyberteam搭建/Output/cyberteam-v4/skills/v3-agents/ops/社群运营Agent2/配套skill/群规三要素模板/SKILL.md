## Skill描述（PUSHY）

社群变成广告群、死群、吵架群？

问题根源：群规设计不完善！

90%的社群问题都是因为群规不明确。"群规三要素模板"从"允许行为→禁止行为→奖惩机制"三个维度，设计出清晰、可执行、有约束力的社群规则。

## Critical Rules（必须遵守）

1. **规则必须简洁**: 群规全文不超过5条核心规则，用户才能记住
2. **禁止行为必须有**: 每条规则必须有对应的违规处理方式
3. **奖惩必须对等**: 奖励优秀行为，惩罚违规行为，且惩罚要适度
4. **规则必须可执行**: 设计时就要考虑执行可行性，不能有无法执行的规则
5. **管理员必须受约束**: 管理员同样需要遵守规则，不能有特权
6. **规则必须可量化**: 禁止行为必须定义清晰，能判断是否违规

---

## Workflow Steps（标准化步骤）

### Step 1: 梳理允许行为

**输入**: 社群定位和核心价值

**处理**:
1. 明确社群鼓励什么行为
2. 确定与社群价值匹配的行为
3. 区分必须行为、鼓励行为、允许行为

**输出**: 允许行为清单

**成功标准**:
- 至少包含3条鼓励行为
- 每条行为与社群价值相关
- 行为描述具体可执行

---

### Step 2: 梳理禁止行为

**输入**: 允许行为清单

**处理**:
1. 列出所有可能损害社群价值的行为
2. 按严重程度分级（严重违规、一般违规、轻微违规）
3. 为每类违规定义处理方式

**输出**: 禁止行为清单及处理方式

**成功标准**:
- 至少包含5条禁止行为
- 分级合理，标准清晰
- 处理方式明确可执行

---

### Step 3: 设计奖惩机制

**输入**: 禁止行为清单

**处理**:
1. 设计奖励机制（鼓励正向行为）
2. 设计惩罚机制（约束违规行为）
3. 明确奖惩的执行流程

**输出**: 奖惩机制方案

**成功标准**:
- 奖励机制能激励正向行为
- 惩罚机制有梯度不过度
- 执行流程清晰

---

### Step 4: 整合群规文案

**输入**: 允许行为清单、禁止行为清单、奖惩机制方案

**处理**:
1. 按"允许→禁止→奖惩"结构组织
2. 精简文案，每条规则不超过20字
3. 配上执行话术和示例

**输出**: 完整群规文案

**成功标准**:
- 全文不超过200字
- 结构清晰，用户能快速理解
- 有可直接使用的执行话术

## 群规三要素框架

### 要素一：允许行为（Do）

| 类型 | 描述 | 示例 |
|-----|------|------|
| 必须行为 | 入群必须做的 | 修改群昵称、遵守群规 |
| 鼓励行为 | 社群赞赏的 | 分享经验、帮助他人 |
| 允许行为 | 社群容许的 | 提问、围观、偶尔广告（需审批） |

### 要素二：禁止行为（Don't）

| 等级 | 行为 | 处理方式 |
|-----|------|---------|
| 严重违规 | 发广告、拉人、诈骗 | 直接移出，永不录用 |
| 一般违规 | 人身攻击、负面信息 | 警告一次，第二次移出 |
| 轻微违规 | 频繁发无关内容 | 私聊提醒，连续3次警告 |

### 要素三：奖惩机制

| 奖励类型 | 具体措施 | 执行条件 |
|---------|---------|---------|
| 精神奖励 | 群内表扬、置顶发言 | 贡献优质内容 |
| 物质奖励 | 资料优先权、活动优先权 | 活跃贡献者 |
| 惩罚类型 | 具体措施 | 执行条件 |
| 轻度惩罚 | 私聊提醒 | 轻微违规 |
| 中度惩罚 | 群内警告 | 重复违规 |
| 重度惩罚 | 移出社群 | 严重违规 |

## 群规模板

### 模板一：简洁版（3条规则）

```
📌 [社群名称]群规

1️⃣ 【允许】可以分享[主题相关内容]、
    帮助他人解答问题、参与讨论

2️⃣ 【禁止】禁止发广告、拉人、吵架、
    人身攻击、发布不实信息

3️⃣ 【奖惩】优质内容会被置顶推荐，
    违规者会被警告或移出社群

⚠️ 详细规则请看群公告
```

### 模板二：标准版（5条规则）

```
📌 [社群名称]群规

【入群必读】
进群即视为同意遵守群规

【允许行为】
✅ 分享与[主题]相关的经验和资源
✅ 帮助他人解答问题
✅ 发布经过审批的活动信息
✅ 提出建设性意见

【禁止行为】
❌ 发广告、推广链接（硬广软广都不行）
❌ 拉人进其他群
❌ 人身攻击、恶意揣测
❌ 发布谣言、不实信息
❌ 频繁发与主题无关的内容

【奖惩机制】
🌟 优质内容：置顶推荐、全员表彰
⚠️ 违规处理：
   第一次：私聊警告
   第二次：群内警告
   第三次：移出社群

【联系我们】
违规举报：@管理员
问题咨询：@管理员
```

### 模板三：严格版（7条规则）

```
📌 [社群名称]社群公约（V2.0）

本群采用实名制，进群视为同意以下公约：

【身份规范】
1. 必须使用真实姓名
2. 必须标注公司/行业
3. 禁止冒充他人身份

【内容规范】
4. 只发布与[主题]相关的内容
5. 禁止一切商业广告和推广
6. 禁止转发未经核实的信息

【互动规范】
7. 尊重他人，友善交流
8. 禁止人身攻击和网络暴力
9. 禁止拉人进其他社群

【违规处理】
⚠️ 严重违规（1次即移出）：
- 发广告、推广
- 冒充他人
- 诈骗行为

⚠️ 一般违规（2次机会）：
- 人身攻击
- 发布不实信息
- 频繁发无关内容

【奖励机制】
🌟 月度之星：贡献最多者获得专属福利
🌟 优质分享：优秀内容被永久收录精华区

【执法标准】
管理员同样遵守本公约，接受全员监督
```

## 执行话术模板

### 欢迎语（附带群规）

```
欢迎加入[社群名称]！

📌 社群公约：
[使用以上任一模板]

点击群公告了解更多规则。
进群即视为同意遵守群规。
```

### 轻微违规提醒

```
@成员[昵称] 温馨提醒：
您刚才发布的内容可能不太适合社群氛围。
[简要说明原因]

📌 请查阅群规：[规则链接]
如有疑问请联系 @管理员
```

### 严重违规警告

```
⚠️ 群内警告

@成员[昵称] 你的行为已违反群规[具体条款]。

[违规行为描述]

根据群规，将对你进行[处理方式]。
如有异议可在24小时内申诉。
```

### 移出社群通知

```
⛔ 移出社群通知

@成员[昵称] 因违反群规[具体条款]，
已被移出[社群名称]。

[违规行为描述]

移出原因：[简要说明]
申诉通道：[邮箱/联系方式]
申诉有效期：收到通知后48小时内

感谢你的理解。
```
