## Skill描述（PUSHY）

规则定了一推就废？执行打折扣？

问题根源：缺乏系统的执行策略！

再好的群规，执行不力等于没定。"群规执行策略"提供从预防→监测→处理→复盘的全链路执行方案，让规则真正落地。

## Critical Rules（必须遵守）

1. **预防优先**: 把问题扼杀在摇篮里，比事后处理更有效
2. **证据先行**: 处理前必须先收集证据，截图存档
3. **先礼后兵**: 先友善提醒，再警告，最后处理
4. **一视同仁**: 规则面前人人平等，管理员也不例外
5. **留有记录**: 所有处理都要记录，可追溯
6. **定期复盘**: 每周复盘执行情况，持续优化

---

## Workflow Steps（标准化步骤）

### Step 1: 预防机制建立

**输入**: 群规全文

**处理**:
1. 设计入群门槛，减少问题用户入群
2. 设置入群须知，新人必须阅读
3. 建立内容规范，减少违规内容
4. 设置提醒机制，提前预防违规

**输出**: 预防执行方案

**成功标准**:
- 入群门槛能过滤大部分问题用户
- 新人阅读群规后才能发言
- 有清晰的内容发布规范

---

### Step 2: 监测机制建立

**输入**: 预防执行方案

**处理**:
1. 设定监测时间表（定时巡查）
2. 建立举报机制（成员协助）
3. 设置敏感词预警（系统辅助）
4. 明确监测责任分工

**输出**: 监测执行方案

**成功标准**:
- 有固定的巡查时间表
- 成员知道如何举报
- 系统能自动识别明显违规

---

### Step 3: 处理流程执行

**输入**: 监测到的违规行为

**处理**:
1. 判断违规等级（严重/一般/轻微）
2. 根据等级执行对应处理
3. 记录处理结果
4. 必要时进行申诉处理

**输出**: 处理执行报告

**成功标准**:
- 处理有记录可追溯
- 处理速度符合SLA
- 申诉机制畅通

---

### Step 4: 复盘优化机制

**输入**: 一段时间内的处理记录

**处理**:
1. 统计违规类型分布
2. 分析处理效果
3. 识别规则漏洞
4. 优化执行策略

**输出**: 复盘优化报告

**成功标准**:
- 每周有数据统计
- 每月有复盘报告
- 规则根据实际执行情况迭代

## 执行策略框架

### 一、预防执行方案

#### 1. 入群门槛设计

| 门槛类型 | 具体措施 | 目的 |
|---------|---------|------|
| 身份门槛 | 验证背景信息 | 确保目标用户 |
| 行为门槛 | 完成入群任务 | 筛选有意愿者 |
| 认知门槛 | 阅读并承诺遵守 | 减少无知违规 |

#### 2. 入群须知设计

```
📖 入群须知

请在发言前阅读以下内容：

1. 群规全文：[链接]
2. 内容发布规范：[链接]
3. 常见问题FAQ：[链接]

✅ 完成以下动作才能发言：
□ 我已阅读群规
□ 我承诺遵守群规
□ 我了解违规后果
```

#### 3. 敏感词/链接过滤

| 类型 | 示例 | 处理方式 |
|-----|------|---------|
| 广告链接 | t.cn/xxx, bit.ly/xxx | 自动警告+删除 |
| 联系方式 | 微信号、手机号 | 审核后放行 |
| 二维码 | 图片二维码 | 自动警告+删除 |
| 竞品名称 | [竞品名] | 提醒但不禁止 |

---

### 二、监测执行方案

#### 1. 定时巡查机制

| 时间段 | 巡查频率 | 负责人 | 重点 |
|-------|---------|-------|-----|
| 早间 8:00-9:00 | 1次/天 | 管理员A | 夜间违规内容 |
| 午间 12:00-13:00 | 1次/天 | 管理员B | 午休刷屏 |
| 晚间 20:00-22:00 | 1次/30分钟 | 管理员A | 黄金时段 |
| 周末/节假日 | 1次/小时 | 轮值 | 综合巡查 |

#### 2. 成员举报机制

```
🚨 举报流程

1. 发现违规内容
2. 点击举报/截图保存
3. @管理员 说明情况
4. 管理员确认处理
5. 结果反馈给举报人

举报奖励：
- 有效举报 +5积分
- 协助调查 +10积分
```

#### 3. 系统预警设置

| 预警类型 | 触发条件 | 提醒方式 |
|---------|---------|---------|
| 广告链接 | 包含短链接/二维码 | @管理员 |
| 刷屏 | 5分钟内发10条以上 | @管理员 |
| 新人 | 入群1小时内发广告 | @管理员 |
| 敏感词 | 命中敏感词库 | 延迟发送+审核 |

### 三、分级处理方案

#### 严重违规（立即处理）

| 行为 | 处理方式 | 附加动作 |
|-----|---------|---------|
| 发广告/推广 | 立即移出 | 加入黑名单 |
| 拉人进其他群 | 立即移出 | 通知被拉者 |
| 诈骗/虚假信息 | 立即移出+举报 | 保留证据 |
| 人身攻击/辱骂 | 立即移出 | 禁止入群 |
| 泄露隐私 | 立即移出 | 法律途径 |

**处理话术**:
```
⛔ 违规处理通知

@[违规者昵称] 因[违规行为]，已被移出社群。

违规类型：严重违规
处理方式：永久移除

请所有成员引以为戒。
```

#### 一般违规（警告处理）

| 违规次数 | 处理方式 | 等待时间 |
|---------|---------|---------|
| 第1次 | 私聊警告 | - |
| 第2次 | 群内警告 | 7天内 |
| 第3次 | 禁言3天 | 7天内 |
| 第4次 | 移出社群 | - |

**私聊警告话术**:
```
📝 群规提醒

[昵称]，你好！

我是[社群名称]管理员[名字]。

我们注意到您刚才在群内发布了[内容描述]，
根据群规[条款]，这个行为[不被允许/需要注意]。

[温和说明原因]

请注意：
- 这是您的第[X]次此类违规
- 下次再犯将[升级处理]
- 如有疑问可以回复本消息

感谢理解！
```

**群内警告话术**:
```
⚠️ 群规警告

@[违规者昵称] 提醒：

您的[违规行为]已违反群规[条款]。

这是您的第[X]次违规。
请遵守群规，共同维护社群秩序。

👉 群规全文：[链接]
```

#### 轻微违规（提醒处理）

| 行为 | 处理方式 |
|-----|---------|
| 轻微刷屏 | @提醒 |
| 格式不当 | 友善提醒 |
| 小广告倾向 | 私信提醒 |

**提醒话术**:
```
💬 温馨提醒

@[昵称] 建议：

[简短建议]

谢谢配合~
```

## 特殊情况处理

### 情况1：管理员自己违规

**处理流程**:
1. 其他管理员或成员指出
2. 管理员主动承认并道歉
3. 按规则接受同等处理
4. 公开说明处理结果

**话术**:
```
📢 管理员声明

@[管理员昵称] 刚才因[违规原因]违反了群规[条款]，
已接受[处理方式]。

管理员同样遵守群规，接受大家监督。
感谢指出！
```

### 情况2：成员是VIP/付费用户

**处理原则**: 规则面前人人平等，但可酌情沟通

**处理流程**:
1. 按规则执行处理
2. 私聊说明原因，表达遗憾
3. 提供替代方案（如有）
4. 做好后续服务，避免退款

### 情况3：违规者是KOL/活跃用户

**处理原则**: 从重处理，因为影响更大

**处理流程**:
1. 警告更明确
2. 公开说明（以儆效尤）
3. 考虑其影响力决定去留
4. 做好群内舆论引导
