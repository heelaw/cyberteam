# 互动引导话术模板

## 使用说明
本模板提供早晚咨讯中互动引导的标准话术，帮助提升社群活跃度。

---

## 一、早间互动话术

### 1. 选择型互动

```
💬 今日互动：

[话题描述]

请直接回复选项：
A. [选项1]
B. [选项2]
C. [选项3]

⏰ 下午6点公布统计结果~
```

### 2. 开放型互动

```
💬 今日话题：

[一个开放性问题]

留言格式：
【我的观点】：[你的看法]

优秀留言将收录精华区~
```

### 3. 参与型互动

```
💬 今日任务：

今天给大家一个小挑战：
[具体任务]

完成后截图发群里~
前10名完成者获得[奖励]！

⏰ 截止时间：今日24:00
```

### 4. 问答型互动

```
❓ 今日问答：

[一个实用问题]

知道答案的伙伴请留言~

[设置小奖励鼓励回答]
```

## 二、晚间互动话术

### 1. 今日总结互动

```
💬 今日话题互动：

今天的话题收到了XX条回复！

最受欢迎的观点是：
[最多人选的选项]

[精选2-3条优质留言]

感谢参与，明天继续~
```

### 2. 感谢参与互动

```
🌟 今日贡献榜：

感谢今天的活跃分子：
@[昵称1]
@[昵称2]
@[昵称3]

🌟 特别表扬：
@[昵称] - 今日最佳[分享/回答]

明天见~
```

### 3. 明日预告互动

```
📅 明日预告：

明天的话题与[主题]相关，先透露一点：

[预告内容]

大家可以提前想想~
明天一起来讨论！
```

### 4. 晒单/成果展示

```
🎉 今日成果：

[晒单/成果展示]

@[昵称1] 的[成果]
@[昵称2] 的[成果]
@[昵称3] 的[成果]

👍 恭喜以上伙伴！
其他人也快来展示吧~
```

## 四、互动话术场景库

### 提升参与率

```
💬 等你来发言：

这个话题可能有XX种答案，
你的答案是哪一种？

[话题]

👇 留言区说说你的想法~
```

### 引导深度讨论

```
💬 深度讨论：

关于[话题]，你遇到过类似的情况吗？
当时你是怎么处理的？

[话题描述]

期待你的故事~
格式：【我的经历】：[内容]
```

### 鼓励新人

```
👋 新人专属互动：

本周新入群的小伙伴，
快来做个自我介绍吧~

格式：
📛 姓名：
🏢 公司/行业：
💡 能提供：
🤝 需要：

完成自我介绍的同学，
可以获得[小奖励]~
```

### 激活沉默用户

```
🎯 激活任务：

好久没看到你发言了~
今天的话题很适合你哦：

[话题]

30秒留言，一起参与吧~
```

### 互动率计算

```
互动率 = 参与人数 / 社群总人数 * 100%

合格标准：
- >10% 优秀
- 5-10% 合格
- <5% 待改进
```

### 优化方向

| 互动率 | 原因分析 | 优化方向 |
|-------|---------|---------|
| 持续<5% | 话题不匹配 | 换话题类型 |
| 持续<5% | 门槛太高 | 降低参与难度 |
| 持续<5% | 时间不合适 | 调整发送时间 |
| 某天突然低 | 内容质量 | 提升内容质量 |
