## Skill描述（PUSHY）

社群每天发内容，但用户还是觉得没价值？

问题根源：缺乏节奏感和价值感知！

早晚资讯三步法是一套"早上提供价值感知→晚上提供归属感"的内容节奏体系，让用户每天都有理由打开社群。

## Critical Rules（必须遵守）

1. **固定时间发送**: 每天同一时间发，培养用户习惯
2. **固定格式**: 统一格式让用户一眼识别
3. **价值前置**: 最有价值的内容放最前面
4. **互动引导**: 每条内容都要有互动钩子
5. **不要太长**: 用户时间宝贵，内容要精炼
6. **周末例外**: 周末可以调整节奏或休息

---

## Workflow Steps（标准化步骤）

### Step 1: 早间内容准备（6:00-8:00）

**输入**: 当日资讯素材、群定位

**处理**:
1. 收集当日热点资讯（行业/目标用户相关）
2. 筛选3-5条最有价值的内容
3. 撰写早报文案（固定格式）
4. 添加互动引导

**输出**: 早间资讯文案

**成功标准**:
- 内容与社群定位相关
- 总字数控制在300字以内
- 有明确的互动引导
- 发送时间误差不超过30分钟

---

### Step 2: 晚间内容准备（19:00-21:00）

**输入**: 当日社群内容回顾、用户反馈

**处理**:
1. 回顾当日社群重要讨论
2. 整理用户提出的好问题
3. 撰写晚报文案
4. 预告次日内容

**输出**: 晚间资讯文案

**成功标准**:
- 包含今日精华回顾
- 包含明日内容预告
- 有社群氛围建设内容
- 发送时间误差不超过30分钟

---

### Step 3: 内容优化与归档

**输入**: 当日早晚资讯数据反馈

**处理**:
1. 记录发布后的互动数据
2. 收集用户反馈
3. 分析内容效果
4. 优化次日内容选择

**输出**: 内容优化建议

**成功标准**:
- 有发布后的互动记录
- 有数据对比分析
- 有优化迭代记录

## 资讯模板框架

### 早报标准模板

```
📮 早安/[社群名称]

⏰ [时间] | 星期[X]

【今日头条】
[1条最重要的资讯，50字以内]

【行业动态】
[2-3条行业资讯，每条30字以内]

【今日话题】
[一个讨论话题，引导互动]

【今日活动】（可选）
[当日活动预告]

💬 今日互动：
[一个问题，引导用户回复]

[固定签名]
```

### 晚报标准模板

```
🌙 晚安/[社群名称]

[日期]精彩回顾

【今日高光】
[今日最精彩的讨论或分享，50字]

【问答精选】
Q: [一个好问题]
A: [精选回答]

【明日预告】
[明日内容预告]

📊 今日数据：
- 今日消息：XX条
- 新增成员：X人
- 活跃成员：X人

🌟 感谢今日贡献者：
@[昵称1] @[昵称2] @[昵称3]

💬 明日话题预告：
[预告明日话题]

晚安，明天见！

[固定签名]
```

## 内容来源矩阵

### 早间内容来源

| 内容类型 | 来源渠道 | 筛选标准 |
|---------|---------|---------|
| 行业热点 | 行业媒体、公众号 | 与社群主题相关 |
| 政策解读 | 政府官网、行业报告 | 对目标用户有影响 |
| 数据报告 | 研究机构报告 | 结论对业务有指导 |
| 工具推荐 | 产品测评、用户反馈 | 实操性强 |
| 经验分享 | 优质自媒体 | 有借鉴意义 |

### 晚间内容来源

| 内容类型 | 来源 | 说明 |
|---------|------|------|
| 今日精华 | 社群聊天记录 | 人工筛选 |
| 好问题 | 群内提问 | 有代表性的 |
| 精彩回答 | 群内回答 | 有价值的 |
| 用户反馈 | 私聊收集 | 正负都要 |
| 明日选题 | 编辑策划 | 提前准备 |
