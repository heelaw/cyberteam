# 业务价值评估参考框架

## 一、收入价值评估标准

### 收入规模评分

| 评分 | 年收入 | 说明 |
|------|--------|------|
| 1分 | <100万 | 收入极低 |
| 2分 | 100-1000万 | 收入较低 |
| 3分 | 1000-5000万 | 收入中等 |
| 4分 | 5000万-2亿 | 收入较高 |
| 5分 | >2亿 | 收入很高 |

### 收入质量指标

- **可持续性**：收入是否依赖一次性客户
- **集中度**：前10客户占比是否过高
- **复购率**：老客户收入占比

---

## 二、增长价值评估标准

### 增长速度评分

| 评分 | 年增长率 | 说明 |
|------|----------|------|
| 1分 | <10% | 增长缓慢 |
| 2分 | 10-30% | 增长一般 |
| 3分 | 30-50% | 增长良好 |
| 4分 | 50-100% | 增长较快 |
| 5分 | >100% | 增长很快 |

### 增长质量指标

- **自然增长vs补贴增长**：补贴驱动增长不可持续
- **获客成本**：CAC是否合理
- **用户质量**：新增用户是否有价值

---

## 三、成本价值评估标准

### 成本效率指标

- **毛利率**：越高越好（>60%为优）
- **变动成本占比**：越低越好
- **规模效应**：收入增长10%时成本增速

### 边际成本评估

- 每增加一个客户的边际成本
- 每增加一元收入的边际成本

---

## 四、护城河价值评估

### 护城河类型

1. **网络效应**：用户越多越有价值
2. **转换成本**：用户迁移成本高
3. **无形资产：品牌、专利、牌照
4. **有效规模：成本优势形成的壁垒
5. **低成本优势**：规模带来的成本优势

### 护城河强度评分

| 评分 | 护城河类型 | 说明 |
|------|------------|------|
| 1分 | 无护城河 | 充分竞争 |
| 2分 | 单一护城河 | 有一定壁垒 |
| 3分 | 2-3种护城河 | 壁垒较强 |
| 4分 | 多种护城河 | 壁垒很强 |
| 5分 | 生态级护城河 | 几乎无法撼动 |

## 五、综合评分方法

### 权重分配示例

| 维度 | 成长期公司 | 成熟期公司 |
|------|------------|------------|
| 收入价值 | 20% | 30% |
| 增长价值 | 30% | 15% |
| 成本价值 | 20% | 25% |
| 护城河价值 | 25% | 25% |
| 可选价值 | 5% | 5% |

### 计算公式

```
综合得分 = Σ(维度得分 × 权重)
```

## 六、价值判断标准

| 综合得分 | 价值判断 | 投资建议 |
|----------|----------|----------|
| 4.5-5.0 | 价值很高 | 强烈推荐 |
| 4.0-4.4 | 价值较高 | 推荐 |
| 3.5-3.9 | 价值中等 | 可考虑 |
| 3.0-3.4 | 价值有限 | 谨慎 |
| <3.0 | 价值较低 | 不推荐 |
