## 六、A/B测试注意事项

### 常见错误

1. **样本量不足**：未达统计显著就下结论
2. **测试时间过短**：未覆盖完整周期
3. **多变量同时测**：无法区分变量影响
4. **忽视辅助指标**：只看核心指标
5. **外部因素干扰**：促销活动等影响

### 最佳实践

1. **一次只测一个变量**
2. **保证足够的样本量和时间**
3. **提前定义成功标准**
4. **同时关注核心和辅助指标**
5. **记录所有测试结果**
