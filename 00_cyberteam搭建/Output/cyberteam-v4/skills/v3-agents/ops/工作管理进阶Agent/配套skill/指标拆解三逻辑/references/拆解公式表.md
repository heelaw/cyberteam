## 二、AARRR拆解模板

### Acquisition（获取）

```
新用户数 = 渠道A流量 × 渠道A转化率 + 渠道B流量 × 渠道B转化率 + ...

关键指标：
- 渠道流量
- 渠道转化率
- 注册转化率
- 获取成本（CAC）
```

### Activation（激活）

```
激活用户 = 新用户 × 激活率

关键指标：
- 首次使用时间
- 核心功能使用率
- 新手引导完成率
- 激活率
```

### Retention（留存）

```
留存用户 = 期末用户 × 留存率

关键指标：
- 次日留存
- 7日留存
- 30日留存
- 流失率
```

### Revenue（收入）

```
收入 = 活跃用户 × 付费率 × ARPU

关键指标：
- 付费转化率
- ARPU（用户平均收入）
- LTV（生命周期价值）
- 续费率
```

### Referral（推荐）

```
推荐用户 = 活跃用户 × 推荐率

关键指标：
- 推荐率
- 裂变系数（K因子）
- 邀请转化率
```

## 三、公式推导示例

### 示例：从"提升收入"推导

**起点**：提升收入

**第一层分解**：
收入 = 用户数 × 付费率 × ARPU

**第二层分解**：
- 用户数 = 新增用户 + 留存用户
- 付费率 = 试用转化 + 续费转化
- ARPU = 基础套餐 + 增值服务

**第三层分解**：
- 新增用户 = 渠道A + 渠道B + 渠道C
- 留存用户 = 核心用户 + 普通用户
- 续费转化 = 活跃续费 + 沉默唤醒

**最终形成指标树**：
```
收入
├── 用户数
│   ├── 新增用户（渠道维度）
│   │   ├── 渠道A新用户
│   │   ├── 渠道B新用户
│   │   └── 渠道C新用户
│   └── 留存用户（时间维度）
│       ├── 核心用户留存
│       └── 普通用户留存
├── 付费率
│   ├── 试用转化率
│   └── 续费率
└── ARPU
    ├── 基础套餐收入
    └── 增值服务收入
```
