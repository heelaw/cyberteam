## 菜单命名避坑指南

### 禁止使用

| 模糊命名 | 问题 | 建议修改 |
|----------|------|----------|
| 更多 | 用户不知道里面有什么 | 改为具体功能如「工具箱」「资料库」 |
| 其他 | 同样模糊 | 合并到其他菜单或删除 |
| 关于我们 | 太泛 | 改为「公司介绍」「团队故事」 |
| 历史消息 | 价值感低 | 改为「全部文章」「内容精选」 |
| 联系我们 | 容易和有赞等冲突 | 改为「商务合作」「联系客服」 |

### 注意事项

1. **字数限制**：主菜单最多4-5个字，子菜单最多6个字
2. **避免生僻字**：不要用用户不认识的字
3. **统一风格**：三个主菜单风格要一致
4. **体现价值**：让用户看到菜单就知道能得到什么
5. **动词开头**：转化类菜单用动词更有行动号召力
