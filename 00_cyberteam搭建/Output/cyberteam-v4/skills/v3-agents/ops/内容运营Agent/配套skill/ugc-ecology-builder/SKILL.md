# ugc-ecology-builder

## 简介

搭建UGC类型的内容生态，通过用户生产内容形成可持续的内容供给。

## 触发关键词

```
UGC生态、用户生成内容、内容生态、用户激励、UGC运营
```

## 触发场景描述

```
当需要搭建UGC内容生态时触发：
- 需要让用户产生内容
- 现有内容供给不足
- 需要用户参与互动
- 需要长期维系UGC氛围
```

## 输入

### 必需输入

| 字段 | 类型 | 说明 |
|------|------|------|
| platform_type | string | 平台类型 |
| target_users | string | 目标用户 |

### 可选输入

| 字段 | 类型 | 说明 |
|------|------|------|
| existing_content | array | 现有内容 |

## 方法论：UGC生态搭建5步

### 步骤1：内容初始化

- 氛围塑造
- 话题挑选
- 初始内容填充

### 步骤2：种子用户引入

- 邀请核心用户
- 发现KOL
- 建立强联系

### 步骤3：激励持续生产

- 及时回复互动
- 增加曝光关注
- 物质奖励

### 步骤4：规模化扩张

- 优质内容传播
- 拉新推广引入

### 步骤5：生态长期维系

- 话题标杆更新
- 激励持续优化
- 社区文化沉淀

### 禁止行为

1. 禁止忽视内容审核导致生态恶化
2. 禁止激励过度导致用户为奖励而行动
3. 禁止对劣质UGC不处理打击创作者积极性
4. 禁止只有物质激励忽视精神激励
5. 禁止生态运营短视只顾眼前数据
