# 痛点选题三步法

## 触发关键词

**主关键词**: 选题、帮我选题、不知道写什么、没灵感、选题思路

**次关键词**: 写什么主题、内容方向、文章题材、话题选择

**场景触发**: 用户表达不知道该写什么、缺乏选题灵感、需要确定内容方向时自动触发

## 描述PUSHY

还在为写什么发愁？用痛点选题三步法，30分钟内锁定高转化选题。不懂用户需求的选题都是自嗨！跟着走，让你每次推送都能戳中粉丝G点。

## 核心定义

通过「痛点挖掘 → 选题判断 → 优先级排序」三步闭环，从用户真实需求出发，筛选出既能解决用户问题、又能达成账号目标的高价值选题。

## workflow_steps

### Step 1: 痛点挖掘

**输入**:
- 目标用户画像（年龄、职业、痛点）
- 账号定位（能提供什么价值）
- 近期热点（可选）

**处理**:
1. 列出目标用户最关心的3个生活场景
2. 每个场景挖掘2个深层痛点
3. 区分「表面需求」和「真实动机」

**输出**:
- 痛点清单（6个痛点）
- 痛点优先级标注

**成功标准**:
- 每个痛点都能对应一个具体场景
- 痛点描述使用用户语言（非行业术语）

### Step 2: 选题判断

**输入**:
- Step 1输出的痛点清单
- 账号内容类型（知识科普/情感共鸣/产品带货）

**处理**:
1. 将每个痛点转化为「选题候选」
2. 用四维打分评估每个选题：
   - 痛点强度（1-5分）：用户看到这个标题会不会「扎心」？
   - 覆盖广度（1-5分）：有多少人有这个痛点？
   - 账号匹配度（1-5分）：账号能提供可信解决方案吗？
   - 传播潜力（1-5分）：读者会不会转发给朋友？
3. 得分≥16分的进入下一轮

**输出**:
- 选题评分表
- 推荐选题TOP3

**成功标准**:
- 每个入选选题至少一个维度≥4分
- 总分必须≥16分

### Step 3: 优先级排序

**输入**:
- Step 2输出的TOP3选题
- 本月内容计划
- 近期资源状况

**处理**:
1. 评估每个选题的「执行难度」和「预期效果」
2. 结合热点时效性做时间排序
3. 确定本周/本月推送节奏

**输出**:
- 最终选题执行顺序
- 每篇预计推送日期
- 配套素材清单

**成功标准**:
- 每周至少1篇高优先级选题
- 冷门选题不超过总内容的30%

## Critical Rules

1. **禁止自嗨式选题** - 没有经过痛点验证的选题一律不得执行，必须用用户语言描述
2. **禁止纯追热点** - 热点只是加分项，脱离痛点的追热点等于无效曝光
3. **禁止抄袭同行** - 选题可以借鉴角度，但必须有自己的独特洞察
4. **禁止只看阅读量** - 选题成功标准是「解决用户问题的数量」，而非单纯阅读量
5. **禁止忽略时效** - 季节性/节日性选题必须提前4周规划，不得临时抱佛脚
6. **禁止忽视资源** - 评估选题时必须考虑团队是否有能力执行

## 质量评估标准

### 合格选题特征

- 有明确的目标用户画像
- 标题能让用户「有代入感」
- 内容能实际解决一个问题
- 符合账号一贯风格和调性

### 淘汰选题特征

- 泛泛而谈，无具体场景
- 标题党，无实质内容
- 与账号定位不符
- 执行难度超出团队能力

## 注意事项

1. 痛点挖掘时优先选择「高频高需」而非「低频刚需」
2. 选题判断时不要太相信自己的直觉，要用数据验证假设
3. 如果某个选题连续3次都是高分，警惕「信息茧房」效应
