## 菜单配置检查清单

### 设置前检查

- [ ] 每个跳转链接已验证有效
- [ ] 跳转页面在移动端体验良好
- [ ] 小程序已配置业务域名
- [ ] 图文消息已创建并审核通过

### 设置后检查

- [ ] 24小时内菜单是否生效
- [ ] 点击每个菜单测试是否正常跳转
- [ ] 图片/小程序是否正常显示
- [ ] 页面加载速度是否可接受

### 定期检查（月度）

- [ ] 检查是否有失效链接
- [ ] 分析菜单点击数据
- [ ] 优化低点击菜单
- [ ] 更新过季/过期内容
