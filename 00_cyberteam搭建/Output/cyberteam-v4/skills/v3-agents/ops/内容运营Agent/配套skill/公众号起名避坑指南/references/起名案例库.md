# 起名案例库

## 成功案例

### 案例一：职场成长类

**需求**：帮助职场人提升核心竞争力

**禁用词排查**：
- 排除：最职场、第一职场
- 风险：精英（太泛）
- 安全词：职场、成长、技能、进阶

**候选方案**：
1. 职场进化论
2. 技能进化论
3. 进阶职场
4. 竞争力研究所
5. 职场人成长记

**最终选择**：职场进化论

**选择理由**：
- 「进化」暗示成长变化，有过程感
- 4字简洁好记
- 「论」有专业感和体系感
- 无明显竞品重名

**注册结果**：一次通过

### 案例二：母婴育儿类

**需求**：科学育儿，减轻新手妈妈焦虑

**禁用词排查**：
- 排除：最棒妈妈、第一育儿
- 风险：权威、专家（虚假宣传风险）
- 安全词：妈妈、宝宝、育儿、科学、新手

**候选方案**：
1. 新手妈妈研究所
2. 科学育儿说
3. 妈妈喂养指南
4. 宝贝成长记
5. 妈咪育儿经

**最终选择**：新手妈妈研究所

**选择理由**：
- 「新手妈妈」精准定位目标用户
- 「研究所」暗示专业科学感
- 与「科学育儿」定位契合
- 差异化于「宝宝树」等老牌账号

**注册结果**：一次通过

### 案例三：美食制作类

**需求**：家常菜教学，实用接地气

**禁用词排查**：
- 排除：最简单、独家
- 风险：秘方、秘笈
- 安全词：菜谱、厨房、简单、好吃、家常

**候选方案**：
1. 厨房新手村
2. 家常菜教程
3. 好吃不贵
4. 懒人菜谱
5. 下饭菜

**最终选择**：厨房新手村

**选择理由**：
- 「新手」定位清晰，用户知道是教新手的
- 「村」有社群感和亲切感
- 差异化于「美食杰」等平台
- 口语传播自然

**注册结果**：需查重，「新手村」被占用，最终改为「厨房新手屋」

## 失败案例（避坑参考）

### 案例四：踩坑之过度承诺

**初始方案**：月薪三万

**问题分析**：
- 「三万」属于具体数字承诺
- 暗示转载即能月薪三万
- 过于夸大，易引发虚假宣传投诉

**修改方案**：升职加薪记

**理由**：保留目标感，但不过度承诺

---

### 案例五：踩坑之侵权风险

**初始方案**：阿里管理智慧

**问题分析**：
- 「阿里」为知名企业品牌
- 使用其名称涉嫌不正当竞争
- 可能收到阿里法务警告

**修改方案**：名企管理课

**理由**：保留「大企业参考」的价值感，但去除品牌侵权风险

---

### 案例六：踩坑之生僻字

**初始方案**：职场进阶誌

**问题分析**：
- 「誌」为生僻字
- 输入法难打出
- 用户搜索和传播成本高
- 违背命名基本原则

**修改方案**：职场进阶志

**理由**：改用「志」替代，虽然意思略有不同，但降低了使用门槛

## 起名检查清单

### 通过微信审核

- [ ] 无政治敏感词
- [ ] 无绝对化用语（最、第一等）
- [ ] 无知名品牌/名人
- [ ] 无色情低俗
- [ ] 字符数在6-20之间

### 便于用户记忆

- [ ] 读一遍能记住
- [ ] 听一遍能复述
- [ ] 输入法容易打出
- [ ] 无生僻字/特殊符号

### 符合账号定位

- [ ] 能准确传达账号价值
- [ ] 目标用户一眼看懂
- [ ] 与内容调性一致
- [ ] 有一定差异化

### 利于长期发展

- [ ] 商标可注册
- [ ] 多平台名称可统一
- [ ] 账号易主后仍可用
- [ ] 不容易被仿冒
