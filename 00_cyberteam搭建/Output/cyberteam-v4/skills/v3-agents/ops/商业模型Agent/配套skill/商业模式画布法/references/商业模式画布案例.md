## 商业模式画布填写要点

### 价值主张

- 聚焦核心价值
- 差异化明显
- 可感知可衡量

### 客户细分

- 画像具体
- 规模可估
- 痛点清晰

### 渠道通路

- 触达高效
- 成本可控
- 体验一致

### 收入来源

- 模式清晰
- 定价合理
- 可持续
