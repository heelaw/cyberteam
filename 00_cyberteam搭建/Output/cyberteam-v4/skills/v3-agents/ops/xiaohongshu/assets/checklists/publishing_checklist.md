## 一、封面检查（5项）

### 视觉效果
- [ ] **尺寸正确**：3:4竖幅（1242x1648像素）
- [ ] **主体清晰**：无模糊、无遮挡
- [ ] **光线明亮**：光线充足，不昏暗
- [ ] **构图合理**：主体突出，不拥挤
- [ ] **风格统一**：与账号风格一致

### 文字设计（如有文字）
- [ ] **文字可读**：暗底浅字/亮底深字
- [ ] **对比度足够**：文字与背景对比明显
- [ ] **字体统一**：使用2-3种字体
- [ ] **文字无错别字**：检查文字内容
- [ ] **文字位置合适**：不遮挡主体

### 设计质量
- [ ] **符合小红书审美**：简洁、美观
- [ ] **色彩协调**：配色方案和谐
- [ ] **有视觉冲击力**：能吸引用户注意
