# 工作价值评估清单

## 价值类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 战略价值 | 长期、方向性 | 战略规划 |
| 业务价值 | 直接业务产出 | 产品开发 |
| 运营价值 | 日常运营支持 | 汇报材料 |
| 风险价值 | 避免损失 | 合规 |

---

## 优先级矩阵

| 紧急\重要 | 重要 | 不重要 |
|-----------|------|--------|
| 紧急 | P0：立即做 | P2：授权做 |
| 不紧急 | P1：计划做 | P3：不做 |

---

## 成本评估

- 时间成本
- 机会成本（做这件事就不能做那件）
- 资源成本
- 团队注意力成本
