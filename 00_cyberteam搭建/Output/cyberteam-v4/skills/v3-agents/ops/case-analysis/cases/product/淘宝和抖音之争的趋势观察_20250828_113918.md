# 淘宝和抖音之争的趋势观察_20250828_113918.docx

---

本文，重点解读下淘宝和抖音之争。
淘宝和抖音之间，我们有两个判断——
1）抖音，将会是未来2年里对淘系最大的敌人。

## 高举高打的开始。

电商平台之争，有4个要素——流量、商家&货品供应链、交易保障能力、交易撮合能力。

#### 腾讯的流量围剿下杀出一条血路，让自己成为了国内电商流量的绝对入口，乃是一段佳话。

但此后十几年里，淘宝的绝对优势和壁垒始终都在于供应链、交易保障能力。但在流量、交易撮合能力上，并未有太多进化。
恰好相反，今天的抖音，正是在流量能力、交易撮合能力上取得了远胜于淘宝的成果，于是有了和淘宝正面对战的逻辑和契机。
抖音的流量，靠算法和短视频，抖音的交易撮合，靠直播。当前，短视频-直播带货已经公认是全网效率最高的流量变现模式，没有之一。
这两点，淘宝望尘莫及。

### 2020年，抖音GMV5000亿，而淘宝7万亿。


### 但，抖音的GMV正处在超高速增长期，而淘宝的增速，已经见顶。

而且，还有更致命的事——整个淘系生态里，多年以来，从来就没有过大体量流量的生产能力，这可以视作是整个淘系的阿喀琉斯之踵。
即便过去几年淘宝的直播电商发展得还可以，也绑定了李佳琦、薇娅这样的头部主播，但直播电商本质上也是在消耗存量流量，无法带来增量流量。
双方在流量能力上的差距，可以这么感受下——

### 抖音日活6-7亿，淘宝APP日活2-3亿，都是官方数据。

但抖音的用户的日均在线时长，可能是淘宝的10倍。
所以，抖音和淘宝之争，可以理解为是抖音有了无敌的流量能力，在“直播带货”上也完全跑通了撮合交易的能力，接下来，抖音势必要发起进攻，去打淘系的商家&货品供应链和交易保障能力。
而今年双十一，对抖音就是一个绝佳的时机——可以预计，抖音很可能会借今年双十一为契机，全面对淘系发起进攻，独立自建商城，大量拓展自己的商家&货品供应链，并开始强化交易保障能力。
这二者，是淘系的底牌，一旦被击破，影响将极为深远。
所以淘系会如何应对，将会是重要的看点，我们也很期待看到今年的淘系双十一方案。
但迄今为止，淘系在应对抖音上，我们能看到的牌，真的是少得可怜。
已可见淘宝打出的牌，是独立做了一个“点淘”APP，且最近在大力推广之——李佳琦和薇娅最近在直播间进行双十一活动预热时，也会不停的对“点淘”进行推介。相关负责人介绍，今年整个双十一第一波预售场，淘宝直播（点淘）也将作为主场去承接。
但其实点淘在今年 1 月份就已经上线了，它采取的是种草短视频+直播带货的双核模式，我们刷下来感觉优质原创内容较少，能明显感受到视频内容依然处在初创期，发展较为缓慢，在流量生产上应该说完全不具备跟抖音相提并论的基础。
但毫无疑问，今年双十一，肯定会是一个重要的战场和看点。
期待在双十一过后，看到更多新的战局变化，也再给你分享更多的思考。
