# cyberwiz-caseanalysis

## 概述

本Skill提供了一套系统化的运营和营销案例拆解方法论，基于85+真实案例的深度分析提炼而成，帮助用户深入分析成功案例背后的逻辑和执行细节，建立个人运营弹药库。

## 核心特点

- **九步拆解法**：从案例背景到可复用方法的完整框架
- **七大类型**：针对私域运营、用户增长、品牌营销等不同类型的专业化拆解
- **85+案例库**：涵盖2022-2024年真实运营案例
- **实战模板**：可直接使用的拆解模板和工具

## 工作流程决策树

```
用户提出案例拆解需求
    │
    ├─→ 需要拆解具体案例？
    │   │
    │   ├─ 是 → 使用【九步拆解法】深度分析
    │   │
    │   └─ 否 → 需要查找参考案例？
    │       │
    │       ├─ 是 → 查看【案例索引】选择相关案例
    │       │
    │       └─ 否 → 明确拆解目标
```

## 九步拆解法（核心方法论）

基于85+真实案例分析提炼的通用拆解框架：

### 第一步：案例背景
- 品牌/产品介绍（定位、核心价值、市场地位）
- 行业背景与市场环境
- 面临的挑战与痛点
- 案例时间节点

### 第二步：核心目标
- 业务目标（GMV、DAU/MAU、转化率）
- 运营目标（拉新、留存、复购）
- 品牌目标（知名度、美誉度、用户心智）

### 第三步：目标用户
- 用户画像分析（年龄、性别、地域、收入）
- 用户需求洞察（核心需求、痛点、痒点）
- 用户行为特征
- 用户决策路径

### 第四步：运营策略
- 核心策略打法
- 渠道策略（公域+私域）
- 内容策略
- 产品策略
- 活动策略

### 第五步：执行方案
- 具体执行步骤
- 时间节奏安排
- 资源配置
- 团队分工

### 第六步：数据指标
- 北极星指标
- 关键过程指标
- 结果数据展示
- 数据分析与洞察

### 第七步：亮点创新
- 创新点总结
- 差异化竞争
- 破圈关键

### 第八步：效果复盘
- 目标达成情况
- 成功经验总结
- 不足与反思
- 优化建议

### 第九步：可复用方法
- 核心方法论提炼
- 可复制的关键动作
- 适配场景分析
- 注意事项与避坑

## 输入规范

### 必需输入
- **案例**: 需要拆解的运营/营销案例
- **拆解目标**: 案例拆解的目的和学习目标

### 可选输入
- **案例类型**: 案例的类型（私域运营、用户增长、品牌营销等）
- **行业领域**: 案例所属的行业领域
- **数据资料**: 案例相关的数据或资料

## 输出规范

### 标准输出
- **案例拆解报告**: 基于九步拆解法的完整分析报告
- **可复用方法**: 从案例中提炼的可复用方法论

### 可选输出
- **对比分析**: 与相关案例的对比分析
- **应用建议**: 方法论应用到实际工作的建议
- **学习笔记**: 案例学习的关键要点

## 使用流程

1. **明确拆解目标**: 确定案例拆解的目的和学习目标
2. **收集案例信息**: 通过网络搜索、网页抓取等方式收集案例资料
3. **按照九步法拆解**: 从案例背景到可复用方法进行系统分析
4. **提炼方法论**: 总结案例中的核心方法和可复用技巧
5. **形成应用建议**: 提出将方法论应用到实际工作的建议

## 使用建议

1. **先明确目标** - 是深度学习还是快速参考？
2. **善用案例索引** - 从85个案例中找到相关案例
3. **按九步法拆解** - 确保分析的完整性和深度
4. **关注可复用性** - 提炼可应用到实际工作的方法论
5. **建立个人弹药库** - 使用 `assets/case_library_template.md` 积累案例
