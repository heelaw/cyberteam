# KOL培养路径模板

## 一、KOL潜质识别

### 潜质特征
| 特征维度 | 具体表现 | 识别指标 |
|----------|----------|----------|
| 内容能力 | 输出稳定且高质量 | 每月≥5篇，互动率≥5% |
| 互动态度 | 积极回复他人 | 回复率≥80% |
| 风格独特 | 有自己的观点和角度 | 内容差异化评分≥4 |
| 乐于助人 | 主动帮助新手 | 帮助回复数≥10/月 |
| 社区认同 | 认同社区价值观 | 无违规记录≥6月 |

### 潜质评估表
| 评估项 | 分值 | 达标标准 |
|--------|------|----------|
| 内容产出 | 0-25分 | ≥5篇/月 |
| 内容质量 | 0-25分 | 互动率≥5% |
| 互动活跃 | 0-20分 | 回复率≥80% |
| 风格独特 | 0-15分 | 差异化评分≥4 |
| 社区贡献 | 0-15分 | 帮助他人≥10次 |

> **潜质KOL标准**：总分≥60分，且无违规记录

## 二、KOL晋升路径

### 四阶段路径
```
潜质用户 → 新星KOL → 成熟KOL → 头部KOL
```

### 各阶段标准与权益

#### 阶段1：潜质用户
| 项目 | 内容 |
|------|------|
| 晋升标准 | 潜质评估≥60分 |
| 观察期 | 3个月 |
| 晋升要求 | 3个月持续达标 |

**权益**：
- 流量扶持（每篇内容+100曝光）
- 新星标识
- 优先参与活动

#### 阶段2：新星KOL
| 项目 | 内容 |
|------|------|
| 晋升标准 | 粉丝≥1000，内容质量评分≥60 |
| 观察期 | 6个月 |
| 晋升要求 | 持续产出+月度评估达标 |

**权益**：
- 流量扶持（每篇内容+500曝光）
- 新星标识+徽章
- 官方推荐机会
- 月度礼品

#### 阶段3：成熟KOL
| 项目 | 内容 |
|------|------|
| 晋升标准 | 粉丝≥10000，形成个人品牌 |
| 观察期 | 12个月 |
| 晋升要求 | 持续影响力+月度评估达标 |

**权益**：
- 流量扶持（每篇内容+2000曝光）
- 专属内容标签
- 商业合作机会
- 线下活动邀请
- 年度分成计划

#### 阶段4：头部KOL
| 项目 | 内容 |
|------|------|
| 晋升标准 | 粉丝≥100000，行业影响力 |
| 特殊通道 | 运营直接提名 |

**权益**：
- 首页推荐位
- 官方账号转发
- 专属经纪人服务
- 年度盛典演讲机会
- 股权/分红激励

## 三、KOL考核机制

### 月度考核指标
| 指标 | 新星KOL | 成熟KOL | 头部KOL |
|------|---------|---------|---------|
| 内容产出 | ≥4篇 | ≥8篇 | ≥12篇 |
| 互动率 | ≥3% | ≥5% | ≥8% |
| 违规记录 | 0次 | 0次 | 0次 |
| 活动参与 | ≥1次 | ≥2次 | ≥3次 |

### 降级规则
| 降级条件 | 降级处理 |
|----------|----------|
| 连续2月不达标 | 降一级 |
| 严重违规 | 直接取消KOL资格 |
| 用户投诉≥3次 | 警告+观察 |

### 退出规则
- KOL主动申请退出
- 连续3个月未登录
- 违反社区规则被封号
