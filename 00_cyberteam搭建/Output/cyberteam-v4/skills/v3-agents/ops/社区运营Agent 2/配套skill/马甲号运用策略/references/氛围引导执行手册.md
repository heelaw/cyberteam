# 氛围引导执行手册

## 一、正面引导机制

### 优质内容发现与互动
| 时机 | 操作 | 话术 |
|------|------|------|
| 优质UGC出现 | 点赞+收藏+评论 | "写得真好，收藏了！" |
| 新手发优质 | 点赞+鼓励 | "第一次就这么赞，未来可期！" |
| 专业内容出现 | 点赞+转发 | "专业！给大佬点赞" |
| 有趣内容出现 | 点赞+评论 | "笑死我了，真实！" |

### 引导讨论方向
| 场景 | 目标 | 操作 |
|------|------|------|
| 讨论偏离主题 |拉回主题 | "话说回来，这个问题是不是..." |
| 讨论氛围负面 | 转向正面 | "虽然有问题，但也有好的方面..." |
| 用户发生争执 | 缓和气氛 | "大家角度不同，都很有道理" |
| 好话题没人理 | 重新激活 | "这个话题很有意思，怎么没人聊？" |

## 二、危机应对机制

### 负面内容监控
| 监控对象 | 监控方式 | 响应时间 |
|----------|----------|----------|
| 新发内容 | 实时监控 | 5分钟内 |
| 评论/回复 | 关键词提醒 | 10分钟内 |
| 用户投诉 | 私信提醒 | 30分钟内 |

### 危机分级
| 等级 | 情况 | 处理方式 | 负责人 |
|------|------|----------|--------|
| P1 | 大面积负面 | 立即处理+官方声明 | 总监 |
| P2 | 局部负面 | 控评+引导 | 运营 |
| P3 | 少量负面 | 正常回复 | 客服 |

### 危机话术
| 场景 | 话术 |
|------|------|
| 用户抱怨 | "感谢反馈，我们会改进..." |
| 虚假信息 | "这个信息有误，正确是..." |
| 竞品攻击 | "我们专注做好自己，欢迎监督" |
| 恶意攻击 | "请理性讨论，违规内容会被处理" |

| 精华回复 | 有深度+有帮助 | 置顶 |
