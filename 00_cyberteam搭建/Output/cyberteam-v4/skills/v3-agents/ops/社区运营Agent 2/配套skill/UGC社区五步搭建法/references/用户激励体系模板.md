# 用户激励体系模板

## 一、用户成长体系

### 等级定义
| 等级 | 称号 | 经验要求 | 用户占比 |
|------|------|----------|----------|
| Lv1 | 潜水新人 | 0-99 | 50% |
| Lv2 | 活跃新手 | 100-499 | 25% |
| Lv3 | 小有名气 | 500-1999 | 15% |
| Lv4 | 圈内达人 | 2000-4999 | 7% |
| Lv5 | 意见领袖 | 5000-9999 | 2% |
| Lv6 | 核心元老 | 10000+ | 1% |

### 升级规则
- 经验值来源：
  - 发布内容：+10/条
  - 被点赞：+2/次
  - 被收藏：+3/次
  - 被回复：+5/次
  - 优质内容加精：+50/次

### 等级权益
| 等级 | 解锁特权 |
|------|----------|
| Lv1 | 基础发言 |
| Lv2 | +发表情包 |
| Lv3 | +发长文权限 |
| Lv4 | +专属标识 |
| Lv5 | +推荐优先 |
| Lv6 | +参与决策 |

## 二、内容激励规则

### 优质内容标准
| 标准 | 说明 | 奖励 |
|------|------|------|
| 原创性 | 首发/独到见解 | +20分 |
| 实用性 | 可操作/可落地 | +15分 |
| 完整性 | 图文并茂/逻辑清晰 | +10分 |
| 互动性 | 引发讨论/回答问题 | +10分 |

### 每周精选
- 每周精选3篇优质内容
- 入选奖励：+100积分 + 首页推荐

## 三、互动激励规则

### 行为积分
| 行为 | 积分 | 每日上限 |
|------|------|----------|
| 点赞 | +1 | 20 |
| 收藏 | +2 | 10 |
| 回复 | +3 | 15 |
| 分享 | +5 | 10 |

## 四、荣誉体系

### 徽章系统
| 徽章 | 获取条件 | 稀有度 |
|------|----------|--------|
| 初来乍到 | 完成注册 | 普通 |
| 第一篇内容 | 发布第一篇 | 普通 |
| 突破100 | 获得100赞 | 稀有 |
| 日更达人 | 连续7天发布 | 稀有 |
| 月度之星 | 每月top10 | 史诗 |

### 排行榜
- 周榜：本周积分TOP10
- 月榜：本月积分TOP10
- 热榜：内容热度TOP10
