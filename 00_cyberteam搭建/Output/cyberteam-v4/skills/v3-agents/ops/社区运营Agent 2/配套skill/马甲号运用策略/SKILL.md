## 核心定位
通过系统化的马甲号运营策略，帮助运营者在社区冷启动期和日常运营中，通过账号矩阵建设、人设打造、内容填充、氛围引导等方式，正向引导社区氛围，激活用户参与，构建健康的社区生态。

## Critical Rules

### 必须遵守
1. **马甲号必须与真人区分** - 不能伪装成真实用户进行欺骗，人设要透明或半透明
2. **内容必须有价值** - 马甲号发布的内容必须对用户有价值，不能只是灌水
3. **引导必须正向** - 只能正向引导，不能操纵舆论或打压用户
4. **数量必须合理** - 马甲号与真实用户比例要合理，过多会破坏社区生态
5. **合规必须遵守** - 必须遵守各平台规则，不能使用违规手段

### 禁止行为
1. **禁止伪装真实用户** - 不能让马甲号伪装成普通用户欺骗他人
2. **禁止恶意打压竞品** - 不能用马甲号恶意攻击竞品或传播虚假信息
3. **禁止刷量作弊** - 不能用马甲号刷点赞/评论/粉丝等数据
4. **禁止操纵舆论** - 不能用马甲号制造虚假舆论或压制真实用户声音
5. **禁止违规内容** - 马甲号不能发布违规/虚假/低俗内容

## workflow_steps

### Step 1：账号矩阵设计
**输入**：
- 社区规模和发展阶段
- 可用资源（账号数量/预算）
- 运营目标

**处理**：
1. 确定矩阵结构
   - 金字塔型：1个头部KOL号+多个腰部账号+大量基础账号
   - 网状型：多个同等量级账号互相联动
   - 单一型：只运营1-2个核心马甲号
2. 确定账号数量
   - 基础账号：总用户量的5-10%
   - 活跃马甲：基础账号的20-30%
   - 核心马甲：5-10个
3. 规划账号分工
   - 内容型：发布高质量内容
   - 互动型：回复用户、引导讨论
   - 氛围型：营造正向氛围
   - 应急型：应对突发情况

**输出**：
- 账号矩阵结构图
- 账号分工表
- 资源配置方案

**成功标准**：
- 账号数量与社区规模匹配
- 各账号有明确分工
- 资源配置合理高效

### Step 2：人设打造
**输入**：
- 账号分工
- 社区用户画像
- 社区调性

**处理**：
1. 设计人设类型
   - 专家型：垂直领域专业人士
   - 达大型：分享经验和建议
   - 新手型：初学者提问
   - 旁观型：理性客观评价
2. 设定人设特征
   - 头像/昵称/简介
   - 性格特征
   - 擅长领域
   - 语言风格
3. 确保人设一致性
   - 统一的行为模式
   - 一致的观点立场
   - 符合人设的内容输出

**输出**：
- 各账号人设档案（含头像/昵称/简介/性格/风格）
- 人设一致性手册
- 语言风格指南

**成功标准**：
- 人设真实可信
- 各账号人设有区分度
- 人设行为一致

### Step 3：内容发布计划
**输入**：
- 账号人设档案
- 社区内容需求
- 运营节奏

**处理**：
1. 制定发布节奏
   - 发布频率（每日/每周）
   - 发布时间（考虑用户活跃时间）
   - 发布平台选择
2. 设计内容类型
   - 经验分享型
   - 问题提问型
   - 讨论引导型
   - 资讯分享型
   - 互动活动型
3. 规划内容来源
   - 原创内容
   - 转载内容
   - 用户UGC二次加工
4. 制定互动策略
   - 如何回复用户
   - 如何引导讨论
   - 如何激发用户参与

**输出**：
- 内容发布日历
- 内容模板库
- 互动话术手册

**成功标准**：
- 发布节奏稳定
- 内容质量符合人设
- 互动自然不生硬

### Step 4：氛围引导策略
**输入**：
- 账号矩阵和人设
- 当前社区氛围
- 运营目标

**处理**：
1. 正面引导机制
   - 及时点赞优质内容
   - 真诚回复用户问题
   - 引导正向讨论
   - 树立内容标杆
2. 危机应对机制
   - 负面内容监控
   - 快速响应流程
   - 危机公关话术
   - 必要时官方澄清
3. 标杆塑造机制
   - 挖掘优质UGC
   - 内容加精推荐
   - 用户荣誉激励
4. 社区文化塑造
   - 社区专属词汇
   - 互动仪式设计
   - 情感连接建立

**输出**：
- 氛围引导执行方案
- 危机应对预案
- 社区文化手册

**成功标准**：
- 社区氛围正向
- 危机处理及时
- 用户感知自然

### Step 5：效果评估与迭代
**输入**：
- 运营数据
- 用户反馈
- 社区氛围变化

**处理**：
1. 效果评估指标
   - 内容互动率
   - 用户参与度
   - 社区氛围评分
   - 马甲号被识别率
2. 评估方法
   - 数据分析
   - 用户调研
   - 舆情分析
3. 迭代优化
   - 人设调整
   - 内容策略优化
   - 互动方式改进

**输出**：
- 效果评估报告
- 优化建议
- 迭代计划

**成功标准**：
- 运营效果达标
- 用户感知正向
- 马甲号未被发现
