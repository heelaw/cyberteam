# 热点筛选评估表

## 一、评估维度

### 1. 相关度评估（权重：40%）
| 评估项 | 评估内容 | 分值 |
|--------|----------|------|
| 主题相关 | 与社区主题的关联程度 | 0-40分 |
| 用户匹配 | 与目标用户兴趣的匹配程度 | 0-30分 |
| 目标契合 | 与当前运营目标的契合程度 | 0-30分 |

### 2. 热度评估（权重：30%）
| 评估项 | 评估内容 | 分值 |
|--------|----------|------|
| 当前热度 | 当前热度值排名 | 0-25分 |
| 生命周期 | 所处生命周期阶段 | 0-25分 |
| 延展性 | 话题可多角度延展程度 | 0-25分 |
| 持续性 | 话题持续发酵潜力 | 0-25分 |

### 3. 风险评估（权重：30%）
| 评估项 | 评估内容 | 分值 |
|--------|----------|------|
| 政治敏感 | 政治敏感风险程度 | 0-100分（越低越好） |
| 舆论反转 | 舆论反转风险程度 | 0-100分（越低越好） |
| 平台规则 | 违反平台规则风险 | 0-100分（越低越好） |

## 二、综合评分公式

```
综合评分 = 相关度得分×0.4 + 热度得分×0.3 + (100-风险得分)×0.3
```

## 三、优先级划分

| 优先级 | 综合评分 | 操作 |
|--------|----------|------|
| P0 | 75-100分 | 集中资源引爆 |
| P1 | 60-74分 | 常规跟热点 |
| P2 | 45-59分 | 观察等待 |
| P3 | <45分 | 放弃 |

## 四、评估表示例

| 热点话题 | 相关度(40%) | 热度(30%) | 风险(30%) | 综合评分 | 优先级 | 决策理由 |
|----------|-------------|------------|-----------|----------|--------|----------|
| #健身打卡挑战# | 38 | 25 | 90 | 74.7 | P1 | 高相关度，热度稳定 |
| #职场35岁危机# | 35 | 22 | 85 | 72.1 | P1 | 用户匹配，可延展 |
| #某明星离婚# | 5 | 28 | 30 | 30.1 | P3 | 与社区无关 |
| #AI技术突破# | 20 | 26 | 95 | 49.3 | P2 | 风险较高 |
