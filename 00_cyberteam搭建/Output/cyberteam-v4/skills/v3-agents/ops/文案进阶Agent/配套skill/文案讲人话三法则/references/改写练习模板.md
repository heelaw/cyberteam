## 参考答案示例

### 练习1参考答案（题目1 SaaS产品）

```
术语识别：分布式架构、高并发、弹性扩容、可用性

改写方案：
分布式架构 → 分布在多台服务器上，一台出问题不影响使用
高并发 → 同时支持100万人同时使用不卡
弹性扩容 → 用户多了自动增加服务器，不用手动操作
可用性 → 一年365天几乎不会宕机
```

### 练习4参考答案（题目1 企业服务）

```
问题诊断：
- 术语问题：云原生架构、微服务设计理念、AI智能分析、数字化转型、高可用、高性能、高扩展性、千万级数据处理
- 抽象问题：企业复杂业务需求
- 主角问题：本系统采用...，系统具备...

改写方案：
云原生架构 → 像水电一样随用随取，不用自己建机房
微服务设计理念 → 一个功能模块出问题，不会导致整个系统崩溃
AI智能分析 → 像有个军师帮你分析业务数据
高可用 → 系统几乎不会出问题
数字化转型 → 用电脑代替手工，更快更准
```
