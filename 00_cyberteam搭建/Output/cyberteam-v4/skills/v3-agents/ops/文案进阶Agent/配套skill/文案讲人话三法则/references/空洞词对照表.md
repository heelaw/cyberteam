# 空洞词对照表

## 常用空洞词及改写方案

### 品质类空洞词

| 空洞词 | 问题 | 改写方案 |
|--------|------|----------|
| 高效 | 太抽象 | 具体说明"效率提升多少" |
| 优质 | 无感知 | 描述具体品质细节 |
| 专业 | 无标准 | 说明专业在哪里 |
| 安全 | 抽象承诺 | 给出安全案例或数据 |
| 可靠 | 主观评价 | 说明可靠的具体表现 |
| 便捷 | 抽象感受 | 描述操作步骤简化多少 |
| 智能 | 术语化 | 说明智能做了什么 |
| 先进 | 无概念 | 说明先进在哪里 |

### 效果类空洞词

| 空洞词 | 问题 | 改写方案 |
|--------|------|----------|
| 快速提升 | 无时间概念 | "X天/周达到X效果" |
| 显著改善 | 无数据 | "改善前后对比" |
| 快速增长 | 无比例 | "增长X%" |
| 彻底解决 | 绝对化 | 具体解决什么问题 |
| 全面覆盖 | 无边界 | 覆盖哪些方面 |
| 深度融合 | 抽象 | 融合后产生什么效果 |

### 体验类空洞词

| 空洞词 | 问题 | 改写方案 |
|--------|------|----------|
| 极致体验 | 夸张过度 | 具体什么体验 |
| 完美体验 | 不可信 | 哪些环节做得好 |
| 流畅体验 | 无感知 | 描述操作感受 |
| 沉浸式体验 | 术语化 | 什么样的沉浸感 |
| 极致享受 | 空洞 | 享受什么具体内容 |

## 改写公式

### 空洞词 → 具体化

```
公式：空洞词 + 具体证据/数字/场景

示例：
高效 → 高效：处理速度提升3倍
专业 → 专业：10年行业经验
安全 → 安全：服务100万用户0事故
```

### 抽象 → 具象

```
公式：抽象概念 + 用户能看到的画面

示例：
流畅体验 → 点一下就能用，不用等待
沉浸式 → 好像置身其中的感觉
智能推荐 → 像懂你的朋友推荐
```

### 产品视角 → 用户视角

```
公式：产品特点 → 用户获得的好处

示例：
采用X技术 → 你将拥有X能力
支持X功能 → 你可以用X功能做Y事
符合X标准 → 你使用时会感受到X
```
