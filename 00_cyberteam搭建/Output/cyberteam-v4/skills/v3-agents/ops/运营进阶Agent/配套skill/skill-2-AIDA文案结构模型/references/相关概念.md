# 相关概念

## AIDA vs PAS

| 模型 | 适用场景 | 结构 |
|------|---------|------|
| AIDA | 销售导向 | Attention→Interest→Desire→Action |
| PAS | 问题导向 | Problem→Agitate→Solution |

**PAS适合**：
- 用户已经有痛点感知
- 需要先唤醒需求

---

## AIDA vs 4P

| 模型 | 关注点 | 适用产品 |
|------|--------|---------|
| AIDA | 用户心理转化 | 单一产品、具体解决方案 |
| 4P | 营销组合 | 多产品线、品牌建设 |

## Attention的10种写法

| 类型 | 示例 |
|------|------|
| 反常识 | "月入3万的人都在用这个方法" |
| 悬念 | "为什么你总是做不好运营？" |
| 冲突 | "努力工作却得不到晋升？" |
| 数字 | "3个技巧让转化率翻倍" |
| 权威 | "连XX都在用的方法" |
| 提问 | "你想知道XX的秘密吗？" |
| 恐惧 | "错过这次再等一年" |
| 利益 | "免费领取XX" |
| 故事 | "从前有个人..." |
| 场景 | "想象你..." |
