# 使用场景参考

## 场景1：会员体系设计

**输入**：
- 产品：在线教育平台
- 已有用户数据：注册时间、付费金额、课程完成数

**执行流程**：
1. 确定分层维度（价值+活跃度）
2. 提取用户数据
3. 设定分层阈值
4. 制定各层级策略
5. 设计升级路径
6. 建立监测指标

---

## 场景2：精准营销

**输入**：
- 业务：电商大促
- 目标：提升整体转化率

**执行流程**：
1. 用户分层（新用户/老用户/沉默用户）
2. 各层用户需求分析
3. 制定差异化触达策略
4. 设计差异化激励
5. 执行与监测

---

## 场景3：流失召回

**输入**：
- 沉默用户定义：30天未登录
- 用户数量：10000人

**执行流程**：
1. 分析沉默原因
2. 用户细分（轻度/中度/重度沉默）
3. 制定差异化召回策略
4. 选择触达渠道
5. 设计召回激励
6. 监测召回效果
