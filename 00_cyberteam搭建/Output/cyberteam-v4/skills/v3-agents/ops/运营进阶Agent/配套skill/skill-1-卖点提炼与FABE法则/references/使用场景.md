# 使用场景参考

## 场景1：写产品详情页文案

**输入**：
- 产品：一款在线笔记App
- 目标用户：职场办公人群
- 使用场景：应用商店详情页

**执行流程**：
1. 调用本Skill，提炼产品卖点
2. 结合skill-2（AIDA文案结构）组织内容
3. 输出完整的详情页文案

---

## 场景2：做竞品对比

**输入**：
- 己方产品：在线课程A
- 竞品：在线课程B
- 目标用户：想转行的职场人

**执行流程**：
1. 分别对两个产品应用本Skill
2. 对比双方的FABE卖点卡片
3. 找出差异化优势

---

## 场景3：制作销售话术

**输入**：
- 产品：企业SaaS软件
- 目标用户：中小企业主

**执行流程**：
1. 应用本Skill，提炼企业级卖点
2. 将FABE转化为销售话术
