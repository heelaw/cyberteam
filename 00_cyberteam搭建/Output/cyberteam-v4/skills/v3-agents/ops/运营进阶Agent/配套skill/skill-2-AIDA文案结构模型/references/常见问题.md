# 常见问题与解答

## Q1：Attention写不好怎么办？

**问题**：标题总是平淡无奇
**解答**：
- 反常识法："月入3万的人都在用这个方法"
- 悬念法："为什么你总是做不好运营？"
- 数字法："3个技巧让转化率翻倍"
- 权威法："连XX都在用的方法"

---

## Q2：Interest和Desire有什么区别？

**问题**：两个阶段容易混淆
**解答**：
- Interest：我对这个产品有兴趣了
- Desire：我现在就要这个产品
- Interest是理性认知，Desire是情感冲动

---

## Q3：短文案如何压缩AIDA？

**问题**：只有50字怎么用AIDA？
**解答**：
- Attention：标题（10字）
- Interest：1句话关联痛点（15字）
- Desire：1个核心利益（15字）
- Action：CTA（10字）

---

## Q4：Action总被忽略怎么办？

**问题**：用户看完不行动
**解答**：
- 检查欲望是否足够强
- CTA是否够明确（不是"了解更多"）
- 是否有紧迫感（限时、限量）
- 是否有信任保障（退款、售后）
