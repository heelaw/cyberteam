# user-lifecycle-manager

## 简介

管理用户生命周期，在不同阶段采取不同运营策略，提升用户全生命周期价值。

## 触发关键词

```
用户生命周期、LTV、生命周期管理、用户成长、用户留存
```

## 触发场景描述

```
当需要提升用户生命周期价值时触发：
- 需要提升用户留存
- 需要延长用户生命周期
- 需要在各阶段采取不同策略
- 需要监控生命周期健康度
```

## 方法论：用户生命周期5阶段

### 1. 拉新期

- 目标：获取新用户
- 策略：渠道投放、裂变活动

### 2. 成长期

- 目标：激活用户
- 策略：新用户引导、核心功能体验

### 3. 成熟期

- 目标：提升活跃和留存
- 策略：积分激励、内容运营

### 4. 沉默期

- 目标：唤醒沉睡用户
- 策略：定向推送、激励召回

### 5. 流失期

- 目标：延缓流失
- 策略：流失预警、专属关怀

### 禁止行为

1. 禁止只关注拉新忽视用户留存
2. 禁止各生命周期阶段策略断层
3. 禁止用统一策略对待所有用户
4. 禁止发现用户流失才进行干预
5. 禁止不监控生命周期健康指标
