### KR（Key Results）- 关键结果

**KR类型说明**:
- **基线型**: 有明确终点目标
- **范围型**: 在范围内波动即可
- **正向型**: 越高越好
- **负向型**: 越低越好（如负面反馈）

## 四、SMART目标设定

检查清单：
- [ ] **S**pecific（具体）: 目标描述是否清晰？
- [ ] **M**easurable（可衡量）: 能否用数据验证？
- [ ] **A**chievable（可实现）: 有资源支持吗？
- [ ] **R**elevant（相关）: 与上级目标对齐吗？
- [ ] **T**ime-bound（有时限）: 截止时间明确吗？

## 五、目标书写示例

### 不好的目标
"提升用户满意度"

### 好的目标（SMART）
"O: 打造让用户惊喜的服务体验
KR1: NPS净推荐值从35提升至45
KR2: 用户好评率从60%提升至75%
KR3: 投诉响应时间从24小时缩短至4小时"

## 六、目标对齐检查

| 检查项 | 说明 |
|--------|------|
| 垂直对齐 | 个人目标是否承接上级目标 |
| 水平对齐 | 跨部门目标是否相互配合 |
| 挑战程度 | 目标是否有挑战性但可实现 |
| 资源匹配 | 是否有足够资源支持目标达成 |
