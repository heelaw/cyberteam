# 产品分类指南

## 三类产品类型与北极星指标

### 1. 消磨时间产品（Time-Killing Products）

**定义**: 用户使用产品的主要目的是消磨空闲时间，获得娱乐体验

**典型特征**:
- 用户无明确目标，浏览为主
- 内容消费型
- 沉浸式体验

**北极星指标方向**:
| 指标 | 说明 |
|------|------|
| 观看时长 | 用户在产品上花费的总时间 |
| 内容消耗量 | 用户消费的内容数量 |
| 互动时长 | 用户参与互动的总时长 |

**案例**:
- YouTube: 每日总观看时长
- 抖音: 用户平均停留时长
-  Spotify: 总听歌时长
- 知乎: 内容消费数

---

### 2. 效率提升产品（Productivity Products）

**定义**: 用户使用产品的主要目的是提升工作效率或完成任务

**典型特征**:
- 有明确的任务目标
- 工具属性强
- 付费意愿高

**北极星指标方向**:
| 指标 | 说明 |
|------|------|
| 功能使用量 | 关键功能的使用次数 |
| 付费转化 | 用户付费情况 |
| 任务完成率 | 核心任务的完成比例 |

**案例**:
- 百度网盘: 文件操作数
- 钉钉: 发送消息数量
- Notion: 页面创建数
- Zoom: 会议完成数

---

### 3. 交易类产品（Transaction Products）

**定义**: 用户使用产品的主要目的是完成交易

**典型特征**:
- 有明确的交易目标
- 平台属性强
- 供需双边市场

**北极星指标方向**:
| 指标 | 说明 |
|------|------|
| 交易量 | 完成的交易数量 |
| 交易额 | GMV总额 |
| 付费用户数 | 成功付费的用户数 |

**案例**:
- Airbnb: 总预定天数
- 淘宝: 销售额
- 美团: 订单数
- 滴滴: 出行订单数

---

## 产品分类决策树

```
用户使用产品的目的是？
│
├─ 消磨时间 → 关注：时长、消耗量、沉浸度
│   └─ 指标：观看时长、日活、内容消费量
│
├─ 效率提升 → 关注：任务完成、效率提升、付费
│   └─ 指标：功能使用量、付费转化率
│
└─ 完成交易 → 关注：交易量、交易额、供需匹配
    └─ 指标：GMV、订单数、付费用户数
```

---

## 分类注意事项

### 避免简单归类
- 很多产品兼具多种属性，需根据核心定位判断
- 例如：微信（通讯工具→社交→生活方式）

### 考虑产品阶段
- 早期产品可能定位模糊
- 成熟产品可能多属性并存

### 阶段性调整
- 产品生命周期不同阶段，分类可能变化
- 例如：滴滴早期是效率工具，后期是交易平台
