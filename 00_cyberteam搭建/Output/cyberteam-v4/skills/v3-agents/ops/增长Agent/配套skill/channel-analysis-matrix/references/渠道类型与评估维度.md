# 渠道类型与评估维度

## 一、五大渠道类型

### 1. 付费渠道（Paid Channels）

**定义**: 需要投入资金获取流量的渠道

**典型渠道**:
| 渠道 | 特点 | 适用场景 |
|------|------|----------|
| SEM（搜索广告） | 高意向、精准 | B2B、高客单价 |
| 信息流广告 | 量大、创意驱动 | 消费、娱乐 |
| 展示广告 | 品牌曝光 | 品牌建设期 |
| 应用商店广告 | 精准安装 | App获客 |
| 短视频广告 | 创意强、转化好 | 消费品、服务 |

**优点**:
- 见效快
- 可规模化
- 精准定向

**缺点**:
- 成本持续上涨
- 依赖预算
- 停止投放效果即降

---

### 2. 有机渠道（Organic Channels）

**定义**: 不需要付费但需要持续投入运营的渠道

**典型渠道**:
| 渠道 | 特点 | 适用场景 |
|------|------|----------|
| SEO | 长期稳定、成本低 | B2B、内容产品 |
| 应用商店优化 | 自然流量 | App产品 |
| 口碑传播 | 信任度高 | 高客单价、服务 |
| 官网内容 | 品牌展示 | 所有产品 |

**优点**:
- 成本可控
- 长期价值高
- 可积累

**缺点**:
- 见效慢
- 需要专业能力
- 流量天花板明显

---

### 3. 社交渠道（Social Channels）

**定义**: 通过社交关系网络获取用户的渠道

**典型渠道**:
| 渠道 | 特点 | 适用场景 |
|------|------|----------|
| 社交媒体（微博、小红书） | 品牌传播、种草 | 消费品、生活服务 |
| KOL/KOC合作 | 信任背书 | 品牌曝光、转化 |
| 社群运营 | 私域沉淀 | 复购、高净值用户 |
| UGC传播 | 内容裂变 | 社区、内容产品 |
| 裂变活动 | 病毒传播 | 增长初期 |

**优点**:
- 传播力强
- 信任度高
- 可低成本获取

**缺点**:
- 难以规模化
- 质量不稳定
- 需要持续运营

---

### 4. 转化渠道（Conversion Channels）

**定义**: 直接触达已有用户促进转化的渠道

**典型渠道**:
| 渠道 | 特点 | 适用场景 |
|------|------|----------|
| EDM邮件 | 成本低、精准 | B2B、跨境电商 |
| 短信营销 | 打开率高 | 电商、O2O |
| Push推送 | 实时触达 | App产品 |
| 小程序消息 | 轻量触达 | 微信生态产品 |
| 公众号模板 | 私域触达 | 微信生态产品 |

**优点**:
- 成本极低
- 精准度高
- 可自动化

**缺点**:
- 依赖已有用户池
- 容易过度打扰
- 触达有限制

---

### 5. 战略渠道（Strategic Channels）

**定义**: 长期战略性布局的获客渠道

**典型渠道**:
| 渠道 | 特点 | 适用场景 |
|------|------|----------|
| 战略合作 | 资源互换 | 平台型产品 |
| PR/品牌建设 | 长期价值 | 品牌类产品 |
| 线下渠道 | 体验感强 | O2O、高客单价 |
| 渠道分销 | 借助外力 | B2B、平台 |
| 收购并购 | 快速获取 | 规模化扩张 |

**优点**:
- 竞争壁垒
- 规模效应
- 长期价值大

**缺点**:
- 投入大
- 周期长
- 不确定性高

---

## 二、六大评估维度详解

### 维度1: 规模（Scale）

**定义**: 该渠道能够触达的潜在用户数量

**评估问题**:
- 渠道的总用户量级是多少？
- 目标用户在渠道中的占比是多少？
- 渠道的增长趋势如何？

**数据来源**:
- 平台官方数据
- 行业报告
- 第三方监测

**评分标准**:
| 等级 | 评分 | 说明 |
|------|------|------|
| A | 9-10 | 千万级用户池 |
| B | 6-8 | 百万级用户池 |
| C | 3-5 | 十万级用户池 |
| D | 1-2 | 万级以下用户池 |

---

### 维度2: 效率 - CAC（Customer Acquisition Cost）

**定义**: 获取单个付费用户的平均成本

**计算公式**:
```
CAC = 渠道总花费 / 渠道付费用户数
```

**评估问题**:
- 该渠道的CAC是多少？
- 与目标LTV相比是否合理？（LTV/CAC > 3）
- CAC的趋势是上升还是下降？

**健康标准**:
| LTV/CAC | 渠道健康度 |
|---------|-----------|
| > 3 | 优质渠道 |
| 1-3 | 需优化 |
| < 1 | 亏损渠道 |

---

### 维度3: 质量 - LTV（Life Time Value）

**定义**: 用户在整个生命周期内贡献的价值

**计算公式**:
```
LTV = 平均订单金额 × 购买频次 × 用户生命周期
```

**评估问题**:
- 该渠道获取用户的LTV是多少？
- 用户质量是否稳定？
- 长期留存率如何？

**数据来源**:
- 用户分析报表
- 留存数据
- 复购数据

---

### 维度4: ROI（Return on Investment）

**定义**: 渠道投资回报率

**计算公式**:
```
ROI = (渠道收入 - 渠道成本) / 渠道成本 × 100%
```

**评估问题**:
- 该渠道的ROI是多少？
- ROI能否覆盖产品毛利？
- ROI的持续性如何？

**健康标准**:
| ROI | 渠道价值 |
|-----|---------|
| > 300% | 明星渠道 |
| 100%-300% | 现金牛渠道 |
| 0-100% | 潜力渠道 |
| < 0 | 亏损渠道 |

---

### 维度5: 速度（Speed）

**定义**: 从开始投放到看到效果的时间

**评估问题**:
- 渠道效果启动需要多长时间？
- 优化迭代的周期是多久？
- 能否快速扩量？

**速度等级**:
| 等级 | 时间 | 渠道类型 |
|------|------|----------|
| 极快 | <1周 | 信息流、SEM |
| 快 | 1-4周 | 应用商店、短视频 |
| 中 | 1-3个月 | SEO、内容营销 |
| 慢 | >3个月 | PR、品牌建设 |

---

### 维度6: 协同效应（Synergy）

**定义**: 该渠道与其他渠道的协同配合能力

**评估问题**:
- 能否为其他渠道导流？
- 用户是否会在多渠道间流转？
- 是否有品牌溢出效应？

**协同类型**:
| 类型 | 说明 | 案例 |
|------|------|------|
| 流量协同 | 为其他渠道导流 | 小红书种草→电商转化 |
| 数据协同 | 跨渠道数据打通 | 投放数据→网站分析 |
| 品牌协同 | 品牌建设协同 | PR→口碑传播 |

## 三、渠道选择决策树

```
产品阶段？
│
├─ 启动期（0-1年）
│   └─ 重点：验证获客模型
│       ├─ 首选：低成本有机渠道（SEO、内容）
│       ├─ 测试：付费渠道（小规模测试）
│       └─ 探索：社交渠道（KOL、裂变）
│
├─ 成长期（1-3年）
│   └─ 重点：规模化获客
│       ├─ 主力：付费渠道（放大）
│       ├─ 扩展：社交渠道（规模化）
│       └─ 维护：有机渠道（持续）
│
└─ 成熟期（3年+）
    └─ 重点：优化与防御
        ├─ 稳定：所有渠道组合优化
        ├─ 探索：新渠道测试
        └─ 品牌：战略渠道投入
```

## 四、产品类型与渠道适配

| 产品类型 | 首选渠道 | 次选渠道 | 注意事项 |
|----------|----------|----------|----------|
| B2B产品 | SEM、内容营销 | LinkedIn、线下活动 | 销售线索质量优先 |
| 消费品 | 信息流、小红书 | 短视频、电商平台 | 种草+转化联动 |
| 工具产品 | SEM、应用商店 | 内容营销、社群 | 功能使用引导重要 |
| 电商产品 | 信息流、搜索 | 直播、KOL | 转化路径要清晰 |
| 内容产品 | 内容平台、社交 | 搜索、推荐 | 关注用户时长 |
