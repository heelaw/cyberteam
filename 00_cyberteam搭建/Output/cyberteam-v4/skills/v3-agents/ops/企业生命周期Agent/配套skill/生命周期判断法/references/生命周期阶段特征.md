# 生命周期阶段判断标准

## 初创期特征

| 维度 | 特征 |
|------|------|
| 融资 | 天使轮或未融资 |
| 团队 | 10人以下 |
| 收入 | 几乎无收入 |
| 业务 | 探索PMF |
| 风险 | 极高 |

---

## 成长期特征

| 维度 | 特征 |
|------|------|
| 融资 | A轮-B轮 |
| 团队 | 10-100人 |
| 收入 | 快速增长中 |
| 业务 | 验证模式 |
| 风险 | 较高 |

### 成长期细分

| 阶段 | 融资 | 团队 | 收入 |
|------|------|------|------|
| 早期成长期 | A轮 | 10-30人 | 快速增长 |
| 中期成长期 | B轮 | 30-100人 | 规模化增长 |
| 后期成长期 | C轮 | 100人以上 | 准备上市/并购 |

---

## 成熟期特征

| 维度 | 特征 |
|------|------|
| 融资 | C轮后或已盈利 |
| 团队 | 100人以上 |
| 收入 | 稳定增长 |
| 业务 | 优化放大 |
| 风险 | 中等 |

---

## 转型期特征

| 维度 | 特征 |
|------|------|
| 融资 | 可能需要新融资 |
| 团队 | 可能调整 |
| 收入 | 增长放缓或下降 |
| 业务 | 寻找新增长点 |
| 风险 | 较高 |

---

## 判断检查清单

### 初创期信号

- [ ] 商业模式不清晰
- [ ] 团队10人以下
- [ ] 收入很少或没有
- [ ] 融资在天使轮及以前

### 成长期信号

- [ ] 商业模式初步验证
- [ ] 收入快速增长
- [ ] 团队在扩张
- [ ] 正在进行A-B轮融资

### 成熟期信号

- [ ] 商业模式成熟
- [ ] 收入稳定
- [ ] 组织架构完善
- [ ] 开始考虑上市/退出

### 转型期信号

- [ ] 原有业务增长放缓
- [ ] 需要寻找新方向
- [ ] 可能需要重新融资
- [ ] 组织可能需要调整
