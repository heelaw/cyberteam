# 文案自检清单 - 完整版

## 一、结构检查

### 标题
- [ ] 标题完整，不缺少主语或谓语
- [ ] 标题有吸引力，能引起好奇
- [ ] 标题传达用户收益，不是产品功能
- [ ] 标题长度适中（10-20字）
- [ ] 无错别字

### 开头
- [ ] 开头能抓住用户注意力
- [ ] 开头与目标用户相关
- [ ] 开头引出主题
- [ ] 开头不要太长（1-3句话）

### 主体
- [ ] 结构清晰，层次分明
- [ ] 逻辑连贯，无跳跃
- [ ] 重点突出，非重点不喧宾夺主
- [ ] 每段有核心句

### 结尾
- [ ] 有明确的CTA
- [ ] CTA清晰可执行
- [ ] CTA有吸引力
- [ ] 无虎头蛇尾

---

## 二、内容检查

### 事实准确性
- [ ] 所有事实陈述准确
- [ ] 无虚假信息
- [ ] 无夸大宣传

### 数据真实性
- [ ] 数据来源可查证
- [ ] 数据时间最新
- [ ] 数据对比合理公平
- [ ] 百分比计算正确
- [ ] 引用他人数据已注明来源

### 承诺合理性
- [ ] 承诺可实现
- [ ] 无绝对化用语（100%、第一、最好等）
- [ ] 承诺有明确条件

### 表述清晰度
- [ ] 无歧义表达
- [ ] 专业术语有解释
- [ ] 逻辑清晰

---

## 三、格式检查

### 错别字
- [ ] 逐字检查无错别字
- [ ] 形近字无误用
- [ ] 谐音字无误用

### 标点符号
- [ ] 标点使用正确
- [ ] 无中英文标点混用
- [ ] 标点位置正确

### 段落格式
- [ ] 段落长度适中（不超过150字）
- [ ] 段落之间有空行
- [ ] 无孤立的单字行

### 字数
- [ ] 字数符合要求
- [ ] 不太长或太短

### 敏感词
- [ ] 无广告法违禁词
- [ ] 无政治敏感词
- [ ] 无低俗词汇

---

## 四，合规检查

### 广告法合规
- [ ] 无"最"、"第一"、"国家级"等极限词
- [ ] 无虚假销量/排名
- [ ] 无未经证实的数据

### 知识产权
- [ ] 图片已获授权或有版权
- [ ] 文案内容无抄袭
- [ ] 引用内容已注明来源

### 竞品提及
- [ ] 无贬低竞品内容
- [ ] 竞品对比客观公正
- [ ] 无混淆视听的内容

### 隐私合规
- [ ] 无用户隐私信息
- [ ] 脱敏处理合格
- [ ] 符合GDPR/个保法要求

### 整体质量
- [ ] 达到发布标准
- [ ] 用户体验良好
- [ ] 品牌形象正面

### 风险评估
- [ ] 无法律风险
- [ ] 无舆情风险
- [ ] 无品牌风险
