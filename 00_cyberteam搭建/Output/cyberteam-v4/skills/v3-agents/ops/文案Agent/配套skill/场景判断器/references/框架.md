# 场景判断法 - 方法论

## 核心框架

场景判断是内容策略的基础。不同场景下用户的阅读状态、心理准备、时间投入完全不同，需要匹配不同的内容策略。

## 原理说明

### 三种场景类型

#### 1. 短场景（1-3分钟）
**典型场景**: 朋友圈、微博、信息流广告

**用户状态**:
- 时间碎片化
- 注意力分散
- 环境干扰多

**内容策略**:
- 标题必须抓眼球
- 内容精简有力
- 视觉冲击优先

#### 2. 中场景（3-10分钟）
**典型场景**: 订阅号文章、知乎回答、小红书笔记

**用户状态**:
- 有一定整块时间
- 主动点开内容
- 有一定耐心

**内容策略**:
- 标题引发兴趣
- 结构清晰分层
- 提供完整价值

#### 3. 长场景（10分钟+）
**典型场景**: 官网文章、深度报告、白皮书

**用户状态**:
- 主动搜索/深度阅读
- 带着问题来
- 期望深度价值

**内容策略**:
- 标题精准匹配
- 深度专业内容
- 完整解决方案

## 判断维度

### 渠道维度
- 社交媒体 → 短场景
- 内容平台 → 中场景
- 官方网站 → 长场景

### 用户维度
- 无意刷到 → 短场景
- 主动点击 → 中场景
- 主动搜索 → 长场景

### 内容维度
- 产品介绍 → 中场景
- 使用教程 → 长场景
- 促销信息 → 短场景

## 使用技巧

### 场景匹配原则
- 短场景：标题大于内容
- 中场景：结构大于深度
- 长场景：深度大于形式
