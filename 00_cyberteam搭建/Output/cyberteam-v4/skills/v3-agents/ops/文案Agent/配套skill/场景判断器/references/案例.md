# 场景判断法 - 成功案例库

## 案例1: 同一产品的多场景文案

**产品**: 在线文档协作工具

### 短场景 - 朋友圈广告
**场景**: 朋友圈信息流
**阅读时间**: 1-2分钟
**文案策略**:
- 标题：别再邮件传文件了，这个工具好用到哭
- 内容：3句话 + GIF演示
- CTA：免费用

### 中场景 - 公众号文章
**场景**: 订阅号推送
**阅读时间**: 5分钟
**文案策略**:
- 标题：团队文档协作的5个坑，你踩过几个？
- 结构：痛点共鸣 → 解决方案 → 功能介绍 → 案例证明
- 长度：800-1200字

### 长场景 - 官网介绍页
**场景**: 官网落地页
**阅读时间**: 10分钟+
**文案策略**:
- 标题：企业文档协作解决方案
- 结构：问题分析 → 方案介绍 → 功能详解 → 案例展示 → 价格方案
- 长度：3000字+

## 案例2: 教育产品多场景覆盖

**产品**: 编程培训课程

### 短场景 - 抖音短视频
**场景**: 抖音信息流
**阅读时间**: 15-60秒
**文案策略**:
- 开头3秒：抓住注意力
- 内容：痛点 + 解决方案 + 效果展示
- 形式：真人出镜 + 代码演示

### 中场景 - 小红书笔记
**场景**: 小红书搜索
**阅读时间**: 3-5分钟
**文案策略**:
- 标题：从月薪5000到25000，我学编程的那一年
- 结构：个人故事 → 学习经历 → 成果展示 → 课程推荐
- 长度：600-1000字

### 长场景 - 官网课程页
**场景**: 官网浏览
**阅读时间**: 15分钟+
**文案策略**:
- 结构：课程体系 → 师资介绍 → 学员评价 → 就业数据 → 报名方式
- 长度：完整课程介绍

## 案例3: B2B产品多场景策略

**产品**: 企业协作SaaS

### 短场景 - 信息流广告
**场景**: 百度/头条信息流
**文案**: "团队文档总是乱？3分钟了解正解方案"
**特点**: 痛点开头 + 解决方案 + 点击引导

### 中场景 - 知乎回答
**场景**: 知乎相关问题
**文案**: 《如何提升团队文档协作效率？5年经验分享》
**特点**: 专业背书 + 实战经验 + 工具推荐

### 长场景 - 官网案例页
**场景**: 官网案例中心
**文案**: 完整客户案例故事
**特点**: 背景 → 挑战 → 方案 → 效果 → 数据
