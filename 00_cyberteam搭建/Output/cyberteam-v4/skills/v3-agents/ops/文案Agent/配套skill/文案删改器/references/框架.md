# 文案删改法 - 方法论

## 核心框架

文案删改三步法：删（删减冗余）、改（改写不佳）、优（优化整体）。

## 原理说明

### 删的原则
**核心**: 删到只剩核心信息

**删什么**:
- 重复表达（说两遍同样的意思）
- 显而易见（用户本来就知道）
- 干扰信息（分散注意力的内容）
- 自我炫耀（显水平但无价值的）
- 过渡内容（连接词和缓冲语）

**删多少**:
- 初稿删20-30%
- 优秀文案删30-50%
- 精简是常态，不是例外

### 改的原则
**核心**: 每句都有存在理由

**改什么**:
- 表意不清的句子
- 过于复杂的句式
- 语气不当的表达
- 软弱的肯定句
- 空洞的形容词

**怎么改**:
- 长句拆短句
- 复杂句简化
- 抽象改具体
- 被动改主动

### 优的原则
**核心**: 让整体效果最大化

**优什么**:
- 开头是否够抓人
- 结尾是否够有力
- 节奏是否流畅
- 关键信息是否突出
- 整体调性是否一致

## 删改技巧

### 删的技巧
- 从后往前读，每句问"删掉会怎样"
- 站在用户角度问"这对我有什么用"
- 砍掉所有修饰词看核心信息是否完整

### 改的技巧
- 动词要具体（"取得" → "获得"）
- 名词要明确（"好东西" → 具体名称）
- 删掉"非常"、"十分"、"特别"等修饰

### 优的技巧
- 开头一句话必须抓住注意力
- 每段第一句必须是精华
- 结尾必须有行动召唤
- 关键信息要重复强调

## 常见错误

### 删的常见错误
- 删过头，核心信息丢失
- 删了重要细节
- 破坏逻辑连贯性

### 改的常见错误
- 改后不如原文
- 改变原意
- 产生歧义

### 优的常见错误
- 优化后风格不一致
- 加强了错误的信息
- 节奏被打乱
