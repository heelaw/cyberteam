# 文案四原则 - 检查清单

## 讲清楚检查

### 信息准确
- [ ] 数据真实可查证
- [ ] 承诺有依据
- [ ] 描述与实际相符
- [ ] 无夸大宣传

### 结构清晰
- [ ] 标题清晰突出
- [ ] 段落分层明确
- [ ] 重点信息前置
- [ ] 逻辑递进清晰

### 逻辑通顺
- [ ] 前后逻辑一致
- [ ] 论据支撑论点
- [ ] 无逻辑跳跃
- [ ] 因果关系清晰

### 无歧义
- [ ] 用词明确
- [ ] 表达清晰
- [ ] 不会产生误解
- [ ] 用户理解路径唯一

## 讲人话检查

### 用户语言
- [ ] 用词用户能理解
- [ ] 无陌生术语
- [ ] 口语化表达
- [ ] 日常用语优先

### 表达自然
- [ ] 句式口语化
- [ ] 无翻译腔
- [ ] 无生硬表达
- [ ] 读起来顺口

### 通俗易懂
- [ ] 复杂概念简化
- [ ] 专业术语有解释
- [ ] 初中水平能读懂
- [ ] 无理解门槛

## 有感检查

### 情感触发
- [ ] 有情感词
- [ ] 能引发情绪
- [ ] 不是冷冰冰的说明
- [ ] 语气真诚

### 共鸣建立
- [ ] 用户能代入
- [ ] 痛点真实
- [ ] 情境熟悉
- [ ] 能说"这就是我"

### 故事元素
- [ ] 有案例故事
- [ ] 有人物
- [ ] 有情节
- [ ] 有结果

## 增强感受检查

### 感官描述
- [ ] 有视觉描述
- [ ] 有听觉/嗅觉等
- [ ] 细节丰富
- [ ] 可感知体验

### 具体细节
- [ ] 有具体数字
- [ ] 有具体场景
- [ ] 有具体人物
- [ ] 避免模糊词

### 画面感
- [ ] 用户能想象
- [ ] 场景清晰
- [ ] 如临其境
- [ ] 代入感强
