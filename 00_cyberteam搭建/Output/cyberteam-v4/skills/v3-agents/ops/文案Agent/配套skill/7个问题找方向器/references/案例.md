# 7个问题找方向法 - 成功案例库

## 案例1: 在线教育平台

**背景**: 某编程学习平台，需要提炼卖点

**7个问题分析**:

**问题1 - 用户是谁**
- 主要用户：想要转行的在职人员，25-35岁
- 用户特征：有工作压力，想提升竞争力
- 场景：通勤时间学习，晚上练习

**问题2 - 用户的问题**
- 表面问题：想学编程但不知道从何学起
- 深层问题：担心学完找不到工作，时间不够用

**问题3 - 产品功能**
- 核心功能：体系化课程 + 实战项目
- 支持功能：代码评测 + 就业指导

**问题4 - 如何解决**
- 学习路径：视频学习 → 动手练习 → 项目实战 → 就业指导

**问题5 - 与竞品差异**
- 竞品：自学网站（内容杂，无指导）
- 差异：体系化学习 + 实战项目 + 就业保障

**问题6 - 用户收益**
- 短期：掌握编程技能
- 长期：获得一份程序员工作
- 量化：70%学员成功就业

**问题7 - 证明**
- 数据：70%就业率
- 案例：学员就业案例集
- 证言：学员 testimonial

**最终卖点方向**: "学完能就业的编程培训"

## 案例2: 健身App

**背景**: 某健身App，需要差异化定位

**7个问题分析**:

**问题1 - 用户是谁**
- 主要用户：想健身但没时间去健身房的人
- 用户特征：时间碎片化，意志力薄弱

**问题2 - 用户的问题**
- 表面问题：不知道该怎么练
- 深层问题：坚持不下去，没人监督

**问题3 - 产品功能**
- 核心功能：15分钟碎片化训练
- 支持功能：AI教练指导，社群打卡

**问题4 - 如何解决**
- 解决方案：随时随地可用，AI监督，社群激励

**问题5 - 与竞品差异**
- 竞品：健身房（时间地点受限）
- 差异：碎片化时间 + AI监督 + 社群陪伴

**问题6 - 用户收益**
- 量化：每天15分钟 = 每月锻炼20+次
- 感受：有人陪练的感觉

**问题7 - 证明**
- 数据：平均每月锻炼次数
- 证言：用户坚持打卡案例

**最终卖点方向**: "15分钟碎片化健身，AI教练陪你坚持"

## 案例3: 企业服务SaaS

**背景**: 某企业协作SaaS，需要B端定位

**7个问题分析**:

**问题1 - 用户是谁**
- 主要用户：中小企业管理者，IT负责人
- 用户特征：预算有限，技术能力弱

**问题2 - 用户的问题**
- 表面问题：团队协作效率低
- 深层问题：工具太复杂推不动员工用

**问题3 - 产品功能**
- 核心功能：简单易用的协作工具
- 支持功能：部署快，无需培训

**问题4 - 如何解决**
- 解决方案：5分钟上手，不用培训

**问题5 - 与竞品差异**
- 竞品：大型协作平台（功能强但复杂）
- 差异：简单 + 快速部署 + 低成本

**问题6 - 用户收益**
- 量化：团队采纳率从30%提升到80%
- 感受：管理者轻松推，员工愿意用

**问题7 - 证明**
- 案例：中小企业快速部署案例
- 数据：5分钟上手统计

**最终卖点方向**: "5分钟上手的企业协作工具，不用培训全员会用"

## 失败案例警示

### 案例4: 某智能硬件

**问题**: 卖点定义为"高品质智能硬件"

**错误分析**:
- 问题1回答太宽泛："所有需要智能硬件的人"
- 问题2没有挖掘深层问题
- 问题5没有明确差异化
- 问题7空洞无证明

**教训**: 卖点需要具体、差异化、有证明
