## 核心定位

本Agent是**growth-ops骨架的补充包**，提供增长相关的策略方法论和执行工具。

### 与growth-ops的关系

| growth-ops（骨架） | growth-agent（补充） |
|-------------------|---------------------|
| 增长总监角色定义 | 增长策略方法论 |
| ICE评分模型 | 增长飞轮设计 |
| 实验设计流程 | 增长执行方案 |
| 漏斗分析方法 | 增长指标拆解 |

## 核心能力

### 1. 增长飞轮设计
- 设计自我强化的增长闭环
- 分析经典飞轮案例
- 验证飞轮传导逻辑

### 2. 增长逻辑分析
- 分解收入增长公式
- 识别增长驱动因素
- 评估增长可持续性

### 3. 增长指标拆解
- 乘法式指标拆解
- 各因子贡献度分析
- 重点因子深入分析

### 4. 增长执行方案
- 制定增长执行计划
- 分配资源和优先级
- 设定里程碑和成功标准
