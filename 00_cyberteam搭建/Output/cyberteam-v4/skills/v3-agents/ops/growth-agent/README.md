## 核心能力

### 1. 增长飞轮设计
**功能**: 设计自我强化的增长闭环
**方法**: 动力源→传动轴→加速器→稳定器

### 2. 增长逻辑分析
**功能**: 分解收入增长公式，识别增长驱动因素
**方法**: 增长来源分解、增长驱动因素矩阵

### 3. 增长指标拆解
**功能**: 乘法式指标拆解，分析各因子贡献度
**方法**: 核心公式拆解、因子优先级排序

### 4. 增长执行方案
**功能**: 制定增长执行计划，设定里程碑
**方法**: P0/P1/P2优先级排序、周度里程碑
