# 业务模型五要素定义卡

## 要素一：价值主张（Value Proposition）

### 定义
企业向用户承诺的核心价值，是业务存在的根本理由

### 核心问题
- 我们给用户承诺的核心价值是什么？
- 我们与竞争对手有什么不同？
- 我们优先交付哪些价值？

### 价值主张类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 新颖型 | 全新体验 | iPhone重新定义手机 |
| 性能型 | 更好性能 | 特斯拉高性能电动车 |
| 定制型 | 个性化满足 | 定制服装 |
| 品牌型 | 身份象征 | 奢侈品品牌 |
| 便利型 | 方便省时 | 外卖服务 |

### 价值主张画布要素
```
目标用户 → 用户的痛点 → 用户的收益
    ↓            ↓             ↓
我们的    →  我们的    →  我们的
产品/服务    痛点解决方案   价值主张
```

## 要素二：用户群体（Customer Segments）

### 定义
企业服务的目标用户，包括核心用户和潜在用户

### 用户分层模型

| 层级 | 名称 | 特征 | 运营策略 |
|------|------|------|----------|
| L1 | 核心用户 | 贡献80%收入 | 重点维护 |
| L2 | 活跃用户 | 频繁使用 | 持续激活 |
| L3 | 普通用户 | 偶尔使用 | 培养转化 |
| L4 | 沉默用户 | 注册未使用 | 唤醒策略 |

### 用户画像要素
```
人口统计：年龄、性别、地域、职业
行为特征：消费习惯、活跃度、偏好
心理特征：价值观、动机、痛点
生命周期：新手期、成长期、成熟期、流失期
```

## 要素三：渠道触达（Channels）

### 定义
企业与用户接触的路径和方式

### 渠道类型

| 类型 | 作用 | 举例 |
|------|------|------|
| 获客渠道 | 获取新用户 | 广告投放、内容营销 |
|转化渠道 | 促进首次成交 | 新用户礼包、首单优惠 |
| 留存渠道 | 维护老用户 | 会员体系、积分体系 |
| 传播渠道 | 口碑裂变 | 分享有礼、邀请好友 |

### 渠道漏斗
```
触达 → 认知 → 兴趣 → 试用 → 成交 → 复购 → 推荐
```

## 要素四：核心资源（Key Resources）

### 定义
企业拥有的关键资源和能力

### 资源类型

| 类型 | 说明 | 举例 |
|------|------|------|
| 实物资源 | 实体资产 | 仓库、物流、生产设备 |
| 知识资源 | 无形资产 | 专利、品牌、数据 |
| 人力资源 | 核心人才 | 技术团队、运营团队 |
| 财务资源 | 资金能力 | 现金流、融资能力 |

### 核心能力识别
```
1. 价值性：能否为用户创造价值？
2. 稀缺性：竞争对手是否难以复制？
3. 不可替代性：是否有替代方案？
4. 难以模仿性：需要多长时间能复制？
```

## 要素五：收入模型（Revenue Model）

### 定义
企业获取收入的方式和模式

### 收入模式类型

| 模式 | 说明 | 代表产品 | 关键指标 |
|------|------|----------|----------|
| 交易收入 | 基于交易次数 | 电商、外卖 | GMV、订单量 |
| 订阅收入 | 周期性收费 | Netflix、会员 | 订阅数、续费率 |
| 增值收入 | 附加功能收费 | 游戏、在线课程 | 付费率、ARPU |
| 广告收入 | 流量变现 | 抖音、公众号 | CPM、CPC |
| 佣金收入 | 撮合收费 | 滴滴、贝壳 | 抽成比例 |

### 定价策略

| 策略 | 适用场景 | 示例 |
|------|----------|------|
| 成本加成 | 制造业 | 产品价格=成本×(1+利润率) |
| 竞争定价 | 同质化市场 | 对标竞争对手 |
| 价值定价 | 差异化产品 | 按用户价值定价 |
| 渗透定价 | 抢占市场 | 低价切入 |
| 撇脂定价 | 新品上市 | 高价短期 |
