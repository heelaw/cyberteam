# 参考资料：框架式思维

## 核心概念

### 框架式思维三层次

1. **结构化拆分**：将复杂问题拆解为若干个子问题
2. **关系梳理**：明确各子问题之间的关系（并列/递进/因果）
3. **动态演化**：考虑问题的变化和发展

### 元素类型识别

| 类型 | 特征 | 拆解策略 |
|------|------|----------|
| 概念性名词 | 模糊不可量化 | 转化为可量化指标 |
| 非具体动词 | 抽象无执行路径 | 明确为具体动作 |
| 具体动词 | 已有执行路径 | 评估路径可行性 |

### 三种拆解思路

1. **并列式**：总指标 = A + B + C（按来源分解）
2. **相关式**：总指标 = A × B × C（按因果/流程分解）
3. **典型代表式**：通过典型案例推广到整体
