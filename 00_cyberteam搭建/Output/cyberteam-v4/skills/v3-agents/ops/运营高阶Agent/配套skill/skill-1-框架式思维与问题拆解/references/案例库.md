# 参考资料：框架式思维案例库

## 案例1：电商转化率拆解

**场景**：某电商App转化率下降分析

**框架应用**：
```
GMV = UV × 点击率 × 加购率 × 支付转化率 × 客单价

层级拆解：
Level 1: GMV
Level 2: UV, 点击率, 加购率, 支付转化率, 客单价
Level 3: 各指标进一步拆解
```

## 案例2：用户活跃度拆解

**场景**：社群活跃度提升

**框架应用**：
```
活跃度 = 发言人数 × 人均发言次数

发言人数 = 核心用户 + 普通用户
人均发言次数 = 激励驱动 + 内容驱动
```

## 案例3：内容产品留存拆解

**场景**：资讯App用户留存分析

**框架应用**：
```
留存率 = 功能体验 × 内容匹配 × 激励反馈

功能体验 = 核心功能使用率 × 使用深度
内容匹配 = 推荐准确度 × 内容质量
激励反馈 = 即时激励 × 持续激励
```
