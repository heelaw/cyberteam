# 过往文案参考库

这个目录存放我过去写的优秀文案，供 AI 学习我的写作风格。

## 使用方法

1. **选择 3-5 篇最能代表你风格的文案**
2. **保存为 markdown 格式**
3. **命名规则**：`{platform}-{topic}-{date}.md`
   - 例如：`ecommerce-skincare-product-2024-10-15.md`
   - 例如：`social-xiaohongshu-dress-2024-09-20.md`

## 文案类型建议

建议至少包含以下几类文案各一篇：

### 1. 电商产品描述
- **示例**：`ecommerce-laptop-2024-08.md`
- **作用**：展示你如何描述产品功能和优势

### 2. 广告投放文案
- **示例**：`ad-facebook-shoes-2024-07.md`
- **作用**：展示你如何设计吸引人的广告文案

### 3. 社交媒体推广
- **示例**：`social-wechat-cosmetics-2024-06.md`
- **作用**：展示你如何在社交媒体上推广产品

### 4. 销售页面文案
- **示例**：`landingpage-course-2024-05.md`
- **作用**：展示你如何构建完整的销售页面

### 5. 品牌故事文案
- **示例**：`brand-story-coffee-2024-04.md`
- **作用**：展示你如何讲述品牌故事

## 文案标注建议

在每篇文案的开头添加元数据：
```markdown
---
platform: 电商平台
topic: 护肤品
product: 保湿面霜
date: 2024-10-15
conversion_rate: 8.5%
what-worked: 痛点描述准确，解决方案清晰，CTA 明确
what-could-be-better: 可以增加更多用户证言
key-phrases: 
  - "24小时持续保湿"
  - "适合敏感肌"
  - "无添加"
  - "立即购买"
writing_style: 专业、信任、亲切
---
```

## 重要提示

- **质量 > 数量**：3-5 篇精选文案足够
- **多样性**：覆盖不同类型和平台
- **代表性**：选择最能体现你风格的
- **定期更新**：随着你的风格演进，更新参考文案

## 如何让 AI 学习这些文案

在与 AI 对话时，可以说：

> "请先阅读 `persona/past-articles/` 目录下的文案，学习我的写作风格，然后帮我写..."

或在工作流中引用：

> "参考 `persona/past-articles/` 中的文案，学习我的语气和转化逻辑"
