## 二、阅读量对比参照表

| 对比指数 | 判断结果 | 转载建议 |
|----------|----------|----------|
| ≥200% | 远高于平均 | 强烈推荐，优先转载 |
| 150%-199% | 高于平均 | 推荐，可考虑转载 |
| 100%-149% | 接近平均 | 观望，需结合其他维度 |
| 50%-99% | 低于平均 | 谨慎，除非其他维度突出 |
| <50% | 远低于平均 | 不推荐 |

---

## 三、定位匹配度评分表

| 评分 | 匹配程度 | 说明 |
|------|----------|------|
| 5分 | 完全匹配 | 受众一致，需求精准对接 |
| 4分 | 高度匹配 | 主体相关，部分内容延伸 |
| 3分 | 中度匹配 | 有一定关联，但非核心用户 |
| 2分 | 低度匹配 | 受众有交集，但不精准 |
| 1分 | 不匹配 | 受众差异大，不建议转载 |

---

## 四、消费程度评估表

| 等级 | 评估标准 |
|------|----------|
| 高 | 结构清晰（800字以上有分段）、有案例/数据支撑、有可操作方法 |
| 中 | 内容有价值但结构一般，有一定可读性 |
| 低 | 标题党、信息空洞、逻辑混乱 |

---

## 五、综合判断决策树

```
开始判断
↓
获取文章基础数据
↓
计算阅读量对比指数
↓
评估定位匹配度 ≥ 3分？
├─ 否 → 不推荐
└─ 是 → 继续评估
↓
评估消费程度 = 高？
├─ 否 → 观望
└─ 是 → 继续评估
↓
判断是否被埋没的好文？
├─ 是 → 强烈推荐
└─ 否 → 阅读量对比指数 ≥ 150%？
    ├─ 否 → 推荐
    └─ 是 → 强烈推荐
```

---

## 六、注意事项

1. **数据真实性核实**：对于数据异常高的文章，需核实是否刷量
2. **时效性判断**：判断内容是否过时，避免转载过时文章
3. **授权确认**：转载前必须确认授权情况
4. **品牌调性**：评估文章风格是否符合账号调性
