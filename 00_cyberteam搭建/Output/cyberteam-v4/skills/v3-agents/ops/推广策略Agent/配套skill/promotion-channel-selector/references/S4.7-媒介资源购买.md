# S4.7：媒介资源购买占有型推广

## 定义

购买特定的广告位或媒介资源，如电视台广告位、户外广告牌、杂志广告位等。

## 操作流程

1. 找到广告渠道或媒介公司
2. 获取资源List及报价
3. 洽谈资源、价格和排期
4. 准备物料：图片、文案等
5. 完成投放

## 特点

- **曝光量基本恒定**，变化不会太大
- 社会化媒体中的媒介资源可能曝光量还取决于内容和文案
- **点击**基本靠**文案和其他推广物料**
- **转化**取决于**流程和文案、物料等**

## 适用场景

- 有明确预算的大品牌转化
- 需要快速提升知名度的产品
- 有成熟物料准备团队的情况
