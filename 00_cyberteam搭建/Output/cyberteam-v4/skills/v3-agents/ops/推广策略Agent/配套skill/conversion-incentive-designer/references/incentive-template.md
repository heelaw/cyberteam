# 转化诱因设计模板

## 模板说明

本模板用于系统性设计转化诱因，确保诱因组合有效且符合产品特性。

---

## 第一步：分析转化场景

### 场景基本信息

| 字段 | 内容 |
|------|------|
| 产品类型 | □实物商品 □虚拟商品 □服务 □课程 □App |
| 客单价范围 | |
| 目标用户 | |
| 转化发生的位置 | |

### 决策复杂度评估

| 评估维度 | 评估结果 |
|---------|---------|
| 客单价高低 | □高(>1000) □中(100-1000) □低(<100) |
| 决策周期 | □长(>7天) □中(3-7天) □短(<3天) |
| 需要的信息量 | □多 □中 □少 |
| 信任需求 | □高 □中 □低 |

### 场景分类

根据评估结果，确定属于哪种场景：
- □ 轻决策+轻场景（速决型）
- □ 轻决策+重场景（浏览型）
- □ 重决策+轻场景（引导型）
- □ 重决策+重场景（深耕型）

- [ ] 主诱因在首屏显眼位置
