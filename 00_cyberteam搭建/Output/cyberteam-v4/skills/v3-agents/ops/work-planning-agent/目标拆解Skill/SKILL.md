## 目标拆解方法论

### 1. 剥洋葱法（时间维度）

```
年目标 → 季度目标 → 月目标 → 周目标 → 日目标
```

**操作步骤**：
1. 确认年度目标（如年营收1000万）
2. 拆解到季度（250万/季）
3. 拆解到月（83万/月）
4. 拆解到周（20万/周）
5. 拆解到日（3万/日）

### 2. 要素拆解法（构成维度）

```
总目标 = 要素1 + 要素2 + 要素3
```

**示例：营收拆解**
```
营收 = 客户数 × 客单价
     = (新客 + 老客) × 客单价
     = (渠道A + 渠道B + 渠道C) × 客单价
```

### 3. 流程拆解法（链路维度）

```
总目标 = 环节1产出 × 环节2转化 × ...
```

**示例：销售漏斗**
```
营收 = 曝光量 × 点击率 × 转化率 × 客单价 × 复购次数
```

## 分析流程

### Step 1: 明确总目标

**处理**：
- 确认目标的具体数值
- 确认时间周期
- 确认目标性质（增长/维持/转型）

**输出**: 清晰的总目标描述

### Step 2: 选择拆解方法

**处理**：
- 时间连续目标 → 用剥洋葱法
- 可分解要素目标 → 用要素拆解法
- 流程转化目标 → 用流程拆解法

**输出**: 选择的拆解方法

### Step 3: 执行拆解

**处理**：
- 按时间拆：年→季→月→周→日
- 按要素拆：识别构成要素，计算各要素目标
- 按流程拆：识别转化漏斗，计算各环节目标

**输出**: 拆解后的子目标清单

### Step 4: 验证可行性

**处理**：
- 计算每个子目标需要的资源
- 评估可行性
- 调整不合理的子目标

**输出**: 调整后的目标体系

## 目标拆解示例

### 示例1：年营收1000万

```
总目标：年营收1000万

时间拆解：
- Q1: 200万（20%）
- Q2: 250万（25%）
- Q3: 250万（25%）
- Q4: 300万（30%）

要素拆解：
- 现有客户续费：400万（40%）
- 新客户获取：600万（60%）

渠道拆解：
- 渠道A：400万
- 渠道B：300万
- 渠道C：200万
- 渠道D：100万
```

### 示例2：用户增长100万

```
总目标：新增100万用户

漏斗拆解：
- 曝光：5000万
- 点击：250万（5%）
- 注册：50万（20%）
- 激活：35万（70%）

渠道拆解：
- 投放：60万
- 裂变：20万
- 私域：15万
- ASO：5万
```

## Critical Rules

1. **目标必须SMART**: 具体、可衡量、可达成、相关、有时限

2. **拆解必须有依据**: 不能随意拆分，要基于业务逻辑

3. **子目标必须可执行**: 每个子目标都要有明确的执行路径

4. **必须验证可行性**: 拆解后要评估是否能完成

5. **必须留有余地**: 目标要略高于能力，激发潜能

---

## 禁止行为

1. **禁止随意分配数字**: 不能拍脑袋给每个渠道分配目标

2. **禁止忽略可行性**: 不考虑资源能力就定目标

3. **禁止只拆不解**: 只拆分目标，不给执行路径

4. **禁止忽视关联**: 忽略子目标之间的关联和依赖

5. **禁止一拆了之**: 拆解后不跟踪执行
