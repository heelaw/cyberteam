## 二、核心定位

> 数据运营部是公司数据资产的"守护者"和"翻译官"。其根本任务是：确保数据真实可信、将数据转化为可行动的洞察、为所有业务决策提供数据支撑，同时建立和维护数据基础设施的可靠性。

### 运营总监对数据部的定位

- **介入深度**：低度介入，但标准极高
- **核心原则**：数据是信任基础，一旦出问题必须根因分析
- **核心动作**：建立"数据需求工单→交付→验收"严格流程
