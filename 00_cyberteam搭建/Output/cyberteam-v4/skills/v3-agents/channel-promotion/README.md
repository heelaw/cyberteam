# 渠道推广Agent

## 目录结构

```
channel-promotion/
├── SOUL.md              # Agent核心定义
├── README.md            # 本文件
└── skills/              # 配套Skills
    ├── 渠道推广四大避坑点/
    ├── 第三方渠道类型清单/
    ├── 渠道效果分析法/
    └── 渠道选择决策表/
```

## Agent概述

| 属性 | 内容 |
|------|------|
| **Agent名称** | 渠道推广Agent |
| **Agent角色** | 渠道选择与推广策略专家 |
| **定位** | 帮助用户选择合适渠道并制定推广策略 |
| **适用模型** | sonnet |
| **版本** | v1.0 |

## 核心Skills

### 1. 渠道推广四大避坑点 (P1)
**触发**: 渠道推广、推广避坑
**功能**: 规避常见渠道推广陷阱

### 2. 第三方渠道类型清单 (P1)
**触发**: 有哪些渠道、渠道类型
**功能**: 识别和了解主流推广渠道

### 3. 渠道效果分析法 (P2)
**触发**: 渠道效果、效果分析
**功能**: 分析各渠道的投放效果

### 4. 渠道选择决策表 (P2)
**触发**: 渠道选择、如何选
**功能**: 辅助渠道选择决策

## 工作流程

```
用户输入需求
    ↓
分析推广场景和目标
    ↓
调用合适Skill提供服务
    ↓
输出渠道选择建议/推广策略
    ↓
根据反馈优化方案
```

## 协同说明

本Agent可与以下Agent协同工作：

| 协同Agent | 协作场景 |
|-----------|----------|
| 增长策略Agent | 增长策略制定 |
| 用户分析Agent | 目标用户画像匹配 |
| 数据分析Agent | 投放效果监测 |

## 使用限制

1. 本Agent专注于渠道选择和推广策略，不涉及具体广告投放操作
2. 渠道规则可能随平台政策变化，建议参考最新官方文档
3. 投放效果受多因素影响，本Agent提供策略指导，不保证具体ROI
