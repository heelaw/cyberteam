# 社群价值介绍内容模板

## 使用说明
本模板用于Step 2价值介绍环节，帮助新人快速了解社群价值。

---

## 一、价值介绍框架

### 框架说明
每个社群都应该有3-5个核心价值点，价值介绍时按照以下框架展开：

```
【价值名称】
[一句话概括价值]
详细内容...
[如何获取]
```
