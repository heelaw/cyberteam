# 不同类型社群门槛参考

## 一、付费课程类社群

### 门槛设计原则
- 以付费记录作为主要身份验证
- 以学习目标作为认知门槛
- 以自我介绍作为行为门槛

### 参考模板

| 门槛类型 | 具体要求 |
|---------|---------|
| 身份门槛 | 提供课程订单截图或报名手机号 |
| 行为门槛 | 填写学习目标问卷（5道题） |
| 认知门槛 | 阅读群规并承诺遵守 |

### 标准话术

```
欢迎来到[课程名称]学员社群！

🎯 加入条件：
已购买[课程名称]的学员

📝 加入流程：
1. 提供购课手机号末4位或订单截图
2. 填写学习目标问卷（2分钟）
3. 承诺遵守群规

💎 社群权益：
- 每周[2次]直播答疑
- 专属学员资料下载
- 优秀学员资源对接
```

## 二、行业交流类社群

### 门槛设计原则
- 以行业背景作为身份验证
- 以问答测试作为行为门槛
- 以价值认知作为认知门槛

### 参考模板

| 门槛类型 | 具体要求 |
|---------|---------|
| 身份门槛 | 提供公司名片或工牌 |
| 行为门槛 | 正确回答3道行业测试题 |
| 认知门槛 | 了解社群以交流为主非广告群 |

### 标准话术

```
欢迎加入[行业名称]交流群！

🎯 加入条件：
[行业名称]从业者

📝 加入流程：
1. 验证从业身份（名片/工牌/公司邮箱）
2. 回答3道入门测试题
3. 阅读社群公约并承诺遵守

💎 社群公约：
- 禁止发布广告和推广链接
- 禁止拉人进群发广告
- 鼓励分享行业经验和资源
- 尊重他人，友善交流

❌ 以下人员会被移出社群：
- 微商/保险从业者
- 发广告者
- 长期潜水无贡献者
```

## 三、高端人脉类社群

### 门槛设计原则
- 以社会身份作为主要验证
- 以资源对接意愿作为行为门槛
- 以私密性承诺作为认知门槛

### 参考模板

| 门槛类型 | 具体要求 |
|---------|---------|
| 身份门槛 | 创始人/高管/投资人等特定身份 |
| 行为门槛 | 提供个人简介用于自我介绍 |
| 认知门槛 | 签署隐私保护承诺书 |

### 标准话术

```
[社群名称]入群申请

尊敬的[先生/女士]：

本社群采用严格审核制，请配合完成以下入群流程：

【身份审核】
请提供以下任一证明：
□ 创始人证明（营业执照/工商信息）
□ 高管证明（名片/LinkedIn）
□ 投资人证明（基金从业资格证）

【资源对接】
您希望在社群中获得：
□ 融资对接 □ 业务合作 □ 人才招聘 □ 其他____

【隐私承诺】
入群后您承诺：
□ 不在社群外传播社群内讨论
□ 不私自添加成员进行推销
□ 尊重成员隐私

审核通过后，我们将在3个工作日内发送入群邀请。
```

## 四、同城生活类社群

### 门槛设计原则
- 以地域验证作为主要门槛
- 以基本信息作为行为门槛
- 以群定位认知作为认知门槛

### 参考模板

| 门槛类型 | 具体要求 |
|---------|---------|
| 身份门槛 | 验证同城居民身份 |
| 行为门槛 | 提供所在商圈或小区信息 |
| 认知门槛 | 了解社群非广告群定位 |

### 标准话术

```
欢迎加入[城市名称]-[区域名称]邻里社群！

🏠 加入条件：
本社区/商圈常住居民

📝 验证方式：
1. 提供[小区名称]或[写字楼名称]
2. 回答：您在本区域居住/工作多久？
3. 了解社群定位

📌 社群定位：
- 本社群是[区域]邻居互助群
- 禁止发布广告、拼车、拼房等信息
- 欢迎发布邻里互助、闲置转让、活动组织等信息

💡 特别说明：
如发现微商、中介等账号，将被立即移出社群。
```

## 五、兴趣垂直类社群

### 门槛设计原则
- 以兴趣证明作为身份验证
- 以内容测试作为行为门槛
- 以社区规则作为认知门槛

### 参考模板

| 门槛类型 | 具体要求 |
|---------|---------|
| 身份门槛 | 提供兴趣相关证明（作品/证书/经历） |
| 行为门槛 | 发布一条与主题相关的内容 |
| 认知门槛 | 阅读并理解社区内容规范 |

### 标准话术

```
欢迎加入[兴趣主题]爱好者社群！

🎯 我们欢迎这样的你：
- [具体兴趣]的热爱者
- 有[具体要求]经验或作品
- 愿意与同好交流分享

📝 加入方式：
1. 证明你的兴趣背景（作品/证书/经历截图）
2. 在入群前先发布一条与主题相关的内容
3. 阅读社区内容规范并承诺遵守

💎 内容规范：
- 每周至少贡献[1条]优质内容
- 禁止发布与主题无关的内容
- 禁止转发其他平台的商业广告
- 尊重他人创作，友善交流
```
