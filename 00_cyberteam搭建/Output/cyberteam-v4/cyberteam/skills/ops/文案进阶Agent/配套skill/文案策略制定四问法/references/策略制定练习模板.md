## 参考答案示例

### 练习1参考答案

```
问题一：写给谁看？
- 目标用户：想转行做程序员的零基础小白
- 用户特征：非计算机专业、25-35岁、对现工作不满意
- 用户痛点：不知道能不能学会、学完能不能找到工作
- 用户关心：效果、时间投入、性价比

问题二：让他们做什么？
- 文案目标：欲望/行动
- 理由：用户已有学习意愿，需要的是选择理由

问题三：为什么买我不买竞品？
- 核心卖点：21天学会Python，能做项目
- 差异化：时间明确，效果可量化

问题四：用户现在在哪里？
- 用户状态：考虑中
- 文案风格：案例型、证据型

策略建议：
- 标题方向：21天学会Python，转行做程序员
- 正文要点：学员案例、学习路径、效果证明
- 行动号召：限时优惠，7天无理由退款
```

### 练习2参考答案

```
问题一：写给谁看？
- 目标用户：通勤族、音乐爱好者
- 用户特征：25-40岁、注重生活品质、愿意为体验付费
- 用户痛点：噪音干扰、耳机线缠绕
- 用户关心：音质、降噪效果、续航、舒适度

问题二：让他们做什么？
- 文案目标：欲望/行动
- 理由：品类认知已有，需要激发购买

问题三：为什么买我不买竞品？
- 核心卖点：主动降噪+长续航
- 差异化：续航30小时，降噪效果行业领先

问题四：用户现在在哪里？
- 用户状态：知道/考虑中
- 文案风格：场景型+证据型

策略建议：
- 标题方向：隔绝噪音，还你音乐世界
- 正文要点：降噪效果、续航时间、使用场景
- 行动号召：限时优惠，下单送耳机盒
```
