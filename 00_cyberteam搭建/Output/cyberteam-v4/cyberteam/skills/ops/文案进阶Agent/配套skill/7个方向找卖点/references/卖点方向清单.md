# 卖点方向清单

## 7大方向检查表

### 方向1：功能独特性

**核心问题**：这个产品有什么别人没有的功能？

| 维度 | 检查项 |
|------|--------|
| 唯一功能 | 只有这个产品有的功能 |
| 首发功能 | 行业内首先推出的功能 |
| 改进功能 | 比竞品做得更好的功能 |
| 组合功能 | 功能组合产生的独特价值 |

**常见卖点表达**：
- 行业首创/独家功能
- 唯一支持XXX的产品
- 首家实现XXX

---

### 方向2：效果承诺

**核心问题**：用户使用后能得到什么具体结果？

| 维度 | 检查项 |
|------|--------|
| 量化效果 | 效果可以数字化的 |
| 时间效果 | 效果有明确时间的 |
| 对比效果 | 和不用的对比 |

**常见卖点表达**：
- X天/周/月达到X效果
- 效率提升X%
- 从X变成X

---

### 方向3：解决问题

**核心问题**：能帮用户解决什么具体问题？

| 维度 | 检查项 |
|------|--------|
| 痛点问题 | 用户最痛苦的问题 |
| 痒点问题 | 用户想要但还没实现的 |
| 隐性需求 | 用户自己都没意识到的 |

**常见卖点表达**：
- 再也不用XXX
- 解决XXX的烦恼
- 专门针对XXX问题

---

### 方向4：用户身份

**核心问题**：用这个产品代表什么身份？

| 维度 | 检查项 |
|------|--------|
| 身份认同 | 用这个产品说明是什么人 |
| 圈层归属 | 代表进入某个圈子 |
| 形象塑造 | 帮用户成为什么人 |

**常见卖点表达**：
- XXX人都在用
- 身份的象征
- 圈子标配

---

### 方向5：价格优势

**核心问题**：价格上有什么优势？

| 维度 | 检查项 |
|------|--------|
| 绝对低价 | 同样功能最低价 |
| 相对低价 | 同样价格功能多 |
| 性价比 | 功能/价格比值高 |

**常见卖点表达**：
- 不到X分之一的价格
- 同样的品质一半价格
- 一天只要X块钱

---

### 方向6：服务保障

**核心问题**：有什么额外的保障和服务？

| 维度 | 检查项 |
|------|--------|
| 售后保障 | 退换货、质保 |
| 增值服务 | 额外提供的服务 |
| 会员权益 | VIP专属权益 |

**常见卖点表达**：
- X天无理由退换
- 终身免费更新
- 专属客服服务

---

### 方向7：情感价值

**核心问题**：能带来什么情感满足？

| 维度 | 检查项 |
|------|--------|
| 情感共鸣 | 引发什么情感 |
| 情怀价值 | 有什么故事/理念 |
| 自我实现 | 满足什么自我认知 |

**常见卖点表达**：
- 活出XXX的态度
- XXX的精神
- 给自己XXX的礼物

---

## 卖点筛选矩阵

### 差异化程度评估

| 等级 | 定义 | 标准 |
|------|------|------|
| 高 | 竞品完全没有 | 竞品做不到/没做过 |
| 中 | 竞品有但不如我们 | 竞品也有但我们更好 |
| 低 | 竞品都一样 | 大家都在说 |

### 用户感知度评估

| 等级 | 定义 | 标准 |
|------|------|------|
| 高 | 用户直接能感受到 | 肉眼可见/直接体验 |
| 中 | 需要解释才能理解 | 说清楚才能懂 |
| 低 | 用户根本不关心 | 说出来也没感觉 |

### 推荐选择

| 差异化程度 | 用户感知度 | 推荐度 |
|------------|------------|--------|
| 高 | 高 | ★★★★★ 必选 |
| 高 | 中 | ★★★★☆ 重点选 |
| 中 | 高 | ★★★★☆ 重点选 |
| 中 | 中 | ★★★☆☆ 可选 |
| 低 | 高/中 | ★★☆☆☆ 不推荐 |
| 高/中/低 | 低 | ★☆☆☆☆ 放弃 |

## 卖点写作公式

### 效果承诺型

```
公式：用完/拥有后 → 具体效果/改变

示例：
- 用完皮肤年轻10岁
- 喝完体重下降2斤
- 学完能独立做项目
```

### 解决问题型

```
公式：不用/再也不用 → 痛点问题

示例：
- 再也不用手动整理数据
- 不用学编程也能做网站
- 再也不怕写不出文案
```

### 身份认同型

```
公式：XXX人/圈子都在用

示例：
- 产品经理都在用的工具
- 网红都在推的护肤
- 程序员首选的电脑
```

### 情感价值型

```
公式：给自己/家人 → 什么礼物/态度

示例：
- 给自己一个改变的机会
- 给家人更好的生活
- 活出精致人生
