## 竞品清单

### 1. LangChain
- **仓库**: langchain-ai/langchain
- **定位**: LLM 应用开发框架
- **核心特性**: 链式调用、工具集成、提示词管理

### 2. AutoGen
- **仓库**: microsoft/autogen
- **定位**: 多智能体对话框架
- **核心特性**: 多 Agent 协作、代码执行、人类介入

### 3. CrewAI
- **仓库**: crewaiinc/crewai
- **定位**: 角色扮演 Agent 团队
- **核心特性**: 角色定义、任务分配、协作流程

### 4. MetaGPT
- **仓库**: FoundationAgents/MetaGPT
- **定位**: 软件公司多 Agent 系统
- **核心特性**: SOP、产品经理/架构师/工程师角色

### 5. ChatDev
- **仓库**: OpenBMB/ChatDev
- **定位**: 软件开发团队
- **核心特性**: 编程、聊天、定制化

### 6. LangGraph
- **仓库**: langchain-ai/langgraph
- **定位**: 有状态的 Agent 应用
- **核心特性**: 图结构、循环、持久化

### 7. AgentUniverse
- **仓库**: agentuniverse-ai/agentUniverse
- **定位**: 企业级 Agent 平台
- **核心特性**: 标准化、可扩展、生产就绪
