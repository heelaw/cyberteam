# 商业分析模型参考

本文档收录了常用的商业分析模型，可在需要时查阅使用。

---

## 1. PEST分析模型（宏观环境分析）

用于分析宏观环境对行业的影响。

| 维度 | 分析要点 |
|------|----------|
| **P**olitical（政治） | 政府政策、政治稳定性、贸易限制、税收政策、劳动法 |
| **E**conomic（经济） | 经济增长、利率、汇率、通胀、消费者可支配收入、失业率 |
| **S**ocial（社会） | 人口结构变化、生活方式变化、文化态度、教育水平、健康意识 |
| **T**echnological（技术） | 新技术、研发活动、自动化、技术激励机制、技术迭代速度 |

---

## 2. 波特五力模型（行业竞争分析）

用于评估行业竞争强度和盈利潜力。

| 力量 | 说明 | 关键问题 |
|------|------|----------|
| **现有竞争者竞争** | 行业内现有竞争对手之间的竞争强度 | 市场是否增长缓慢？竞争对手是否势均力敌？退出壁垒是否高？ |
| **潜在进入者威胁** | 新进入者对行业构成的威胁 | 进入壁垒有多高？资本需求多少？品牌忠诚度如何？ |
| **替代品威胁** | 替代产品或服务的威胁 | 有多少替代品？替代品的价格/性能如何？转换成本如何？ |
| **供应商议价能力** | 供应商对价格和条件的影响 | 供应商是否集中？转换成本是否高？前向一体化威胁？ |
| **买方议价能力** | 买方对价格和条件的影响 | 买方是否集中？产品是否标准化？后向一体化威胁？ |

## 3. SWOT分析（战略态势分析）

| | 有利（内部） | 不利（内部） |
|---|---|---|
| **内部** | **Strengths（优势）** | **Weaknesses（劣势）** |
| **外部** | **Opportunities（机会）** | **Threats（威胁）** |

---

## 4. STP分析（市场战略定位）

1. **Segmentation（市场细分）**：将市场划分为不同的客户群体
   - 地理细分、人口细分、心理细分、行为细分

2. **Targeting（目标市场选择）**：选择要服务哪些细分市场
   - 评估细分市场的吸引力
   - 选择目标细分市场

3. **Positioning（市场定位）**：在目标客户心中建立独特形象
   - 价值主张设计
   - 差异化策略

## 6. 价值链分析

### 基本活动
- **进货物流**：接收、存储、分配
- **生产作业**：加工、包装、装配
- **出货物流**：仓储、配送、订单处理
- **市场营销**：定价、促销、渠道
- **服务**：安装、维修、培训

### 支持活动
- **采购**：原材料、设备采购
- **技术开发**：产品工艺改进
- **人力资源管理**：招聘、培训、薪酬
- **企业基础设施**：财务、计划、法律

---

## 7. 市场规模估算模型

```
TAM (Total Addressable Market) = 总体可服务市场
SAM (Serviceable Addressable Market) = 可服务可达市场
SOM (Serviceable Obtainable Market) = 可获得市场
```

### 自上而下法
从宏观数据开始，逐步缩小范围

### 自下而上法
从单位客户价值开始，乘以潜在客户数量

## 8. BCG矩阵（业务组合分析）

| | 高市场份额 | 低市场份额 |
|---|---|---|
| **高增长** | **明星**（Star） | **问题**（Question Mark） |
| **低增长** | **现金牛**（Cash Cow） | **瘦狗**（Dog） |

---

## 9. ANSOFF矩阵（增长战略）

| | 现有产品 | 新产品 |
|---|---|---|
| **现有市场** | 市场渗透 | 产品开发 |
| **新市场** | 市场开发 | 多元化 |

---

## 使用建议

- 不同模型适用于不同分析场景
- 可以组合使用多个模型获得更全面的视角
- 模型是工具，关键在于正确的数据输入和深入思考
