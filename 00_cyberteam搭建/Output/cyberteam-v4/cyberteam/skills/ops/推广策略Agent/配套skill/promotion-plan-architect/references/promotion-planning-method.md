# 推广方案规划方法论

## 核心概念

推广方案规划是将推广目标、渠道选择、转化设计、执行排期等要素整合成一份可落地执行的整体方案。一个好的推广方案应该清晰、具体、可执行、有时间节点。

---

## 六要素框架

### 要素一：渠道核心规则和推广逻辑

**核心问题：** 这个渠道是怎么运作的？

**分析维度：**
| 维度 | 分析内容 |
|------|---------|
| 流量分发机制 | 平台如何分配流量？排序逻辑是什么？ |
| 定向规则 | 支持哪些定向维度？能精准到什么程度？ |
| 计费方式 | CPM/CPC/CPA/CPS？各有什么特点？ |
| 审核规则 | 什么内容不能投？审核标准是什么？ |

**操作步骤：**
1. 深入学习渠道官方文档
2. 研究竞品在该渠道的投放方式
3. 小规模测试验证理解
4. 形成渠道操作规范

---

### 要素二：用户转化路径和场景设计

**核心问题：** 用户从接触到转化经过哪些步骤？

**路径设计原则：**
1. **步骤少** - 一般不超过5步
2. **逻辑清** - 每步之间有因果关系
3. **时间短** - 整个路径控制在合理时间内
4. **体验顺** - 页面切换流畅，无断裂感

**场景设计要点：**

| 场景类型 | 设计要点 |
|---------|---------|
| 主动需求场景 | 突出产品优势，直接引导转化 |
| 被动触达场景 | 先吸引注意力，再引导兴趣，最后转化 |
| 高决策门槛 | 多步引导，先建立信任再要求决策 |

---

### 要素三：转化诱因设计

**核心问题：** 凭什么让用户现在就行动？

**诱因设计框架：**
1. **主诱因** - 最核心的转化动力（1个）
2. **辅助诱因** - 增强主诱因的效果（2-3个）
3. **信任元素** - 消除用户顾虑（1-2个）

**匹配场景选择：**

| 场景 | 推荐诱因类型 |
|------|------------|
| 轻决策+轻场景 | 折扣、福利、限量 |
| 轻决策+重场景 | 评价、案例、销量 |
| 重决策+轻场景 | 资料包、咨询服务 |
| 重决策+重场景 | 顾问、多重背书、案例 |

---

### 要素四：推广落地物料

**核心问题：** 需要准备哪些素材？

**物料清单：**

| 物料类型 | 具体内容 | 规格要求 |
|---------|---------|---------|
| 视觉物料 | Banner图、落地页主图 | 按渠道要求尺寸 |
| 文案物料 | 标题、副标题、描述文案 | 符合渠道字符限制 |
| 信任物料 | 资质证书、媒体报道 | 高清版本 |
| 转化物料 | 表单、按钮、咨询入口 | 位置和样式明确 |

**物料准备流程：**
```
需求确认 → 设计初稿 → 内部审核 → 渠道审核 → 正式上线
```

---

### 要素五：推广落地排期表

**核心问题：** 什么时候做什么？

**排期设计原则：**
1. **阶段清晰** - 测试期、优化期、规模化期分明
2. **节点明确** - 每个阶段的里程碑要清晰
3. **预留缓冲** - 考虑审核和调整时间
4. **可衡量** - 每个节点有可量化的指标

**标准阶段划分：**

| 阶段 | 时长 | 目标 | 关键里程碑 |
|------|------|------|-----------|
| 测试期 | 1-2周 | 验证方案可行性 | 确定核心物料和定向 |
| 优化期 | 2-4周 | 优化各项指标 | CTR/CVR达标 |
| 放量期 | 持续 | 规模化获客 | CPA稳定，持续ROI>1 |

---

### 要素六：数据监控指标

**核心问题：** 如何衡量推广效果？

**指标体系：**

| 指标层级 | 指标名称 | 定义 | 监控频率 |
|---------|---------|------|---------|
| 曝光层 | 曝光量 | 广告展示次数 | 日 |
| 曝光层 | CPM | 千次曝光成本 | 日 |
| 点击层 | 点击量 | 广告被点击次数 | 日 |
| 点击层 | CTR | 点击率 | 日 |
| 点击层 | CPC | 每点击成本 | 日 |
| 转化层 | 转化量 | 完成目标行为数 | 日 |
| 转化层 | CVR | 转化率 | 日 |
| 转化层 | CPA | 每转化成本 | 日 |
| 综合层 | ROI | 投资回报率 | 周 |

**预警机制：**

| 指标状态 | 定义 | 行动 |
|---------|------|------|
| 正常 | 达标80%-120% | 持续执行 |
| 预警 | 低于80% | 排查原因 |
| 优化 | 低于50% | 暂停调整 |
| 暂停 | 持续3天低于30% | 暂停排查 |

## 方案制定流程

### Step 1：信息收集

1. 产品分析：产品定位、核心卖点、目标用户
2. 资源盘点：预算、人力、已有物料
3. 竞品分析：主要竞品的推广策略
4. 渠道调研：可选渠道的特点和成本

### Step 2：方案设计

1. 确定推广目标和KPI
2. 选择核心渠道（不超过3个）
3. 设计转化路径和诱因
4. 制定排期计划
5. 确定数据监控指标

### Step 3：方案评审

1. 逻辑完整性检查
2. 资源可行性评估
3. 风险识别和预案
4. ROI预估

### Step 4：执行准备

1. 物料制作和审核
2. 账户和投放配置
3. 数据监控看板搭建
4. 团队分工明确

---

## 关键成功因素

1. **目标清晰** - KPI要具体可量化
2. **逻辑完整** - 六要素缺一不可
3. **数据驱动** - 每个决策要有数据支撑
4. **灵活应变** - 根据数据及时调整
5. **团队协同** - 市场、产品、运营密切配合

---

## 常见误区

| 误区 | 错误做法 | 正确做法 |
|------|---------|---------|
| 目标模糊 | "提升品牌影响力" | "获取10000新用户，CPA<50元" |
| 贪多求全 | 10个渠道同时投 | 聚焦2-3个核心渠道 |
| 忽视测试 | 直接大预算上线 | 先测试验证再放量 |
| 重投放轻转化 | 只管引流不管承接 | 投放和转化路径一体化 |
| 虎头蛇尾 | 上线后不管优化 | 持续监控和优化 |
