# 文案书写四原则 - 方法论

## 核心框架

文案书写四原则是文案质量的基础检查标准：讲清楚、讲人话、有感、增强感受。

## 原理说明

### 原则1: 讲清楚
**核心**: 信息准确、结构清晰、易于理解

**检查点**:
- 信息是否准确无误
- 结构是否清晰分层
- 逻辑是否通顺
- 是否有歧义表达

**常见问题**:
- 专业术语未解释
- 句子过长（超过30字）
- 段落过长（超过5行）
- 逻辑跳跃

### 原则2: 讲人话
**核心**: 用用户语言，不是产品语言

**检查点**:
- 是否用了用户熟悉的词
- 句式是否口语化
- 是否有翻译腔
- 是否生硬晦涩

**常见问题**:
- 使用专业术语
- 句子过于复杂
- 表达过于书面化
- 用词不够日常

### 原则3: 有感
**核心**: 触发情感，建立共鸣

**检查点**:
- 是否有情感触发
- 是否引发用户共鸣
- 语气是否恰当
- 是否有故事元素

**常见问题**:
- 过于理性没有温度
- 空洞的形容词堆砌
- 没有情感连接
- 语气过于生硬

### 原则4: 增强感受
**核心**: 感官描述让体验更具体

**检查点**:
- 是否有感官描述
- 细节是否具体
- 是否有画面感
- 数据是否有场景感

**常见问题**:
- 描述过于抽象
- 没有具体数字
- 缺少场景细节
- 没有画面感

## 优化技巧

### 讲清楚技巧
- 长句拆短句
- 复杂句简化
- 术语加解释
- 重点要突出

### 讲人话技巧
- 产品语言→用户语言
- 书面语→口语
- 抽象→具体
- 专业→通俗

### 有感技巧
- 加入情感词
- 讲述真实故事
- 引发共鸣
- 语气真诚

### 增强感受技巧
- 加入数字
- 描述具体场景
- 使用感官词
- 创造画面感
