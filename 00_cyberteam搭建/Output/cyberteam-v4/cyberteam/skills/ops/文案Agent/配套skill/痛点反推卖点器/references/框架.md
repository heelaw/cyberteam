# 痛点反推卖点法 - 方法论

## 核心框架

痛点反推法是一种逆向思维的产品卖点提炼方法。通过从用户的痛点和问题出发，反向推导产品的核心卖点。

## 原理说明

### 痛点层次模型
- **表面痛点**：用户直接描述的问题
- **深层痛点**：问题背后的原因
- **本质痛点**：用户真正关心和担心的

### 痛点驱动购买决策
用户购买产品往往是为了：
- 摆脱痛苦（逃避动机）
- 获得快乐（追求动机）
- 逃避痛苦的动力 > 追求快乐的动力

### 情绪放大效应
痛点 + 情绪 = 强烈共鸣
共鸣越强，决策越快

## 四步推导法

### 第一步：痛点场景挖掘
- 收集用户原声
- 识别高频痛点
- 挖掘3层深度

### 第二步：情绪放大
- 识别负面情绪
- 量化情绪强度
- 找到共鸣点

### 第三步：解决方案对应
- 建立痛点-功能映射
- 突出核心功能
- 量化效果

### 第四步：卖点输出
- 组合痛点+方案+收益
- 提炼有穿透力的文案

## 使用技巧

### 痛点挖掘技巧
- 多收集用户原话
- 分析评论和反馈
- 关注"但是"、"就是"、"总是"等转折词

### 情绪放大技巧
- 用具体场景承载情绪
- 避免过度夸张
- 找到多数人的共鸣点
