# 策略规划Agent - 接口协议

## 输入接口

### 用户输入

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| business_problem | string | 是 | 业务问题描述 |
| target_dimension | string | 是 | 规划维度（增长/变现/运营等） |
| industry | string | 是 | 所在行业 |
| constraints | string | 否 | 资源约束或其他限制 |

### 来自其他Agent的输入

| 来源Agent | 传递内容 | 触发条件 |
|-----------|----------|----------|
| 全局业务模型构建器 | 业务模型结构、瓶颈环节 | 遇到增长瓶颈时 |
| 增长业务模型规划器 | 当前增长策略、目标、差距 | 需要新增长策略时 |

## 输出接口

### 直接输出

| 字段 | 类型 | 说明 |
|------|------|------|
| framework | object | 策略框架图 |
| action_items | array | 行动项清单 |
| risk_points | array | 关键风险点 |
| recommendations | array | 执行建议和优先级 |

### 传递给其他Agent

| 目标Agent | 传递内容 | 传递条件 |
|-----------|----------|----------|
| 策略执行Agent | 策略框架、执行要求 | 策略制定完成后 |
| 模型迁移设计器 | 借鉴方案、迁移要素 | 需要细化落地时 |
| 工作价值评估器 | 规划方案的投资回报 | 需要评估价值时 |
