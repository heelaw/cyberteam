# 全局洞察提炼模板

## 模板说明

## 使用说明
1. 从流程层开始逐层向上扩展
2. 分析每个层级对原始问题的影响
3. 基于扩展分析提炼洞察
4. 对比局部视角和全局视角的差异
5. 检查是否有需要避免的局部优化
6. 优先级排序洞察
