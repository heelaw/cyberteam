# 同理心地图模板

## 标准四象限模板

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           同理心地图                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│ 用户群体:                                                                   │
│ 分析目标:                                                                   │
├────────────────────────────┬────────────────────────────────────────────────┤
│                            │                                                 │
│      看到 / 听到           │            想到 / 感受到                        │
│      (See)                 │            (Think/Feel)                       │
│                            │                                                 │
│ 用户在环境中看到什么？      │ 用户内心在想什么？                              │
│ 用户听到什么？              │ 用户有什么感受？                                │
│                            │                                                 │
│                            │                                                 │
├────────────────────────────┼────────────────────────────────────────────────┤
│                            │                                                 │

│      (Say/Do)              │            (Pain/Gain)                       │
│                            │                                                 │
│ 用户说什么？                │ 用户的痛苦是什么？                              │
│ 用户实际做什么？            │ 用户的期望是什么？                              │
│                            │                                                 │
│                            │                                                 │
└────────────────────────────┴────────────────────────────────────────────────┘
```

### 象限一：看到/听到 (See)

**引导问题**:
- 用户在什么环境中？
- 用户在日常生活中看到什么？
- 用户的社交圈在讨论什么？
- 用户接触哪些媒体和信息源？
- 用户周围的人都在做什么？

**填充格式**:

```markdown
### 看到/听到

**物理环境**
- 

**信息环境**
- 

**社交环境**
- 

**媒体接触**
- 
```

---

### 象限二：想到/感受到 (Think/Feel)

**引导问题**:
- 用户在心里真正想什么？
- 用户关心什么？
- 用户担心什么？
- 用户的梦想是什么？
- 用户有什么情绪？为什么？

**填充格式**:

```markdown
### 想到/感受到

**内心想法** (用引号表示用户内心独白)
- "

**情绪感受**
| 情绪 | 触发场景 |
|------|---------|
| | |

**深层动机**
- 
```

---

### 象限三：说什么/做什么 (Say/Do)

**引导问题**:
- 用户说什么？（原话引用）
- 用户告诉别人什么？
- 用户实际做什么？
- 用户的行为有什么特点？
- 用户的典型行为模式是什么？

**填充格式**:

```markdown
### 说什么

**用户原话** (引用)
- "
- "
- "

### 做什么

**典型行为**
- 

**行为模式**
- 
```

---

### 象限四：痛点/期望 (Pain/Gain)

**引导问题**:
- 用户的主要挫折是什么？
- 用户面临哪些障碍？
- 用户冒什么风险？
- 用户想要达成什么？
- 成功是什么样的？

**填充格式**:

```markdown
### 痛点

| 痛点 | 程度 | 证据 |
|------|------|------|
| | 高/中/低 | |

### 期望

| 期望 | 价值 | 可行性 |
|------|------|--------|
| | 高/中/低 | 高/中/低 |
```
