# 情绪评估量表

## 情绪量化原则

在用户旅程图中量化情绪，需要遵循以下原则：
1. **相对性** - 情绪是相对于用户期望而言的
2. **动态性** - 情绪会随体验过程变化
3. **多维性** - 考虑情绪的效价和唤醒度
4. **证据性** - 情绪判断必须有行为数据或反馈支撑

---

## 5级情绪量表

### 评分标准

| 分数 | 情绪状态 | 效价 | 描述 | 典型表现 |
|------|---------|------|------|---------|
| 5 | 非常愉悦 | 正向 | 超预期，有惊喜 | 主动推荐、社交分享 |
| 4 | 愉悦 | 正向 | 满意，符合期望 | 完成任务、给予好评 |
| 3 | 中性 | 中性 | 无感，预期一般 | 按流程完成、无反馈 |
| 2 | 不悦 | 负向 | 不满意，低于预期 | 抱怨、犹豫、放弃 |
| 1 | 非常不悦 | 负向 | 沮丧，完全失望 | 投诉、卸载、负面传播 |

## 各阶段情绪判断指南

### 认知阶段（发现、吸引）

**期望来源**: 广告承诺、品牌印象、他人推荐

| 情绪 | 触发条件 | 行为证据 |
|------|---------|---------|
| 5 | 超出预期的惊喜感 | 主动收藏、分享 |
| 4 | 符合预期，有兴趣 | 浏览时长增加 |
| 3 | 无感 | 快速划过 |
| 2 | 不感兴趣 | 跳出 |
| 1 | 完全不感兴趣 | 负面评论 |

### 评估阶段（比较、决策）

**期望来源**: 产品功能、价格、评价

| 情绪 | 触发条件 | 行为证据 |
|------|---------|---------|
| 5 | 超预期价值感 | 立即购买 |
| 4 | 物有所值 | 收藏、加购 |
| 3 | 犹豫不决 | 反复查看 |
| 2 | 觉得不值 | 离开 |
| 1 | 觉得被骗 | 投诉 |

### 行动阶段（购买、使用）

**期望来源**: 流程顺畅度、服务态度

| 情绪 | 触发条件 | 行为证据 |
|------|---------|---------|
| 5 | 流程超顺畅，有惊喜 | 主动好评 |
| 4 | 顺利完成 | 正常完成 |
| 3 | 有小障碍但能完成 | 需帮助完成 |
| 2 | 遇到明显障碍 | 放弃、投诉 |
| 1 | 完全无法完成 | 退款、投诉 |

### 完成阶段（售后、复购）

**期望来源**: 售后响应、问题解决

| 情绪 | 触发条件 | 行为证据 |
|------|---------|---------|
| 5 | 问题快速解决，有补偿 | 主动复购、推荐 |
| 4 | 问题解决满意 | 复购 |
| 3 | 勉强解决 | 无反馈 |
| 2 | 解决不满意 | 差评 |
| 1 | 未解决或更糟 | 投诉、曝光 |

## 情绪判断的证据类型

### 直接证据（首选）

| 证据类型 | 示例 | 可信度 |
|---------|------|--------|
| 用户访谈反馈 | "我觉得..." | 高 |
| 问卷调查评分 | NPS/CES评分 | 高 |
| 用户投诉/表扬 | 客服记录 | 高 |

### 间接证据（参考）

| 证据类型 | 示例 | 可信度 |
|---------|------|--------|
| 行为转化率 | 各环节流失率 | 中 |
| 页面停留时长 | 页面热力图 | 中 |
| 社交媒体情绪 | 舆情分析 | 中 |
| 完成率 | 任务完成比例 | 中 |

## 情绪校准方法

### 方法1: 对比分析

对比完成用户和流失用户的情绪差异

```
完成用户: 情绪平均4.2分
流失用户: 情绪平均2.1分
结论: 流失主要发生在情绪<3的阶段
```

### 方法2: 关键节点追踪

追踪情绪转折点的前后变化

```
阶段3前: 情绪4分
阶段3中遇到问题: 情绪降至2分
阶段4情绪回升: 3分（问题解决）
结论: 阶段3是情绪低点，需重点优化
```

### 方法3: 期望对比

对比用户期望和实际感知的差距

```
用户期望: 支付1分钟完成
实际感知: 支付3分钟完成
情绪影响: 期望落差导致-1分
```

## 情绪曲线绘制指南

### 坐标系设置

```
Y轴: 情绪分数 (1-5)
X轴: 旅程阶段

     5 │ 😊
       │   😊
     4 │ 😊
       │
     3 │ 😐 ─────────────────────
       │
     2 │     😟
       │ 😟
     1 │ 😟
       └──────┬──────┬──────┬──────┬──────→ X轴
            阶段1   阶段2   阶段3   阶段4
```

### 高低点标注

```
情绪高点: 🌟 阶段名称 - 原因
情绪低点: ⚠️ 阶段名称 - 原因
```

---

## 注意事项

1. **不要主观臆断** - 必须有证据支撑
2. **不要过度细分** - 每个阶段一个情绪值即可
3. **考虑个体差异** - 不同用户群体可能有不同情绪曲线
4. **动态审视** - 情绪会随时间变化，需要定期更新
