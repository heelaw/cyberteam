# 选题案例

## 成功案例

### 案例1：「十点读书」年度选题规划

**背景**：读书类账号，需要持续输出内容，保持用户活跃。

**选题策略**：
- 方向规划：每季度确定3个核心选题方向
- 系列化操作：将选题做成系列，如"21天读书计划"
- 用户参与：让用户投票选择下一期读什么书

**执行要点**：
- 年度选题日历提前规划，确保内容节奏稳定
- 每个系列第一期做"为什么读这本书"吸引用户
- 结尾固定引导互动"下一本你想读什么"

**成果**：
- 系列内容打开率比单篇高40%
- 用户投票参与率超过60%
- 形成稳定的内容预期和粉丝粘性

**关键教训**：选题系列化可以提升用户期待感，提高长期关注率。

### 案例2：「半佛仙人」热点选题差异化

**背景**：股市剧烈波动，多个财经账号都在分析。

**普通做法**：
- "今天股市为什么跌"
- "专家解读股市走势"
- "后市展望"

**半佛仙人做法**：
- "为什么你一买股票就跌"
- "股市里的那些骗局"
- "我为什么劝年轻人远离股市"

**成果**：财经赛道中脱颖而出，阅读量远超同期财经内容。

**关键教训**：热点选题要找到自己的独特角度，不能做信息的二道贩子。

## 失败案例

### 案例：某知识类账号选题失焦

**背景**：账号定位"职场成长"，初期选题聚焦。

**问题出现**：
- 看到情感类内容流量好，开始写"如何处理职场关系"
- 看到科技内容有热度，开始蹭"互联网大佬动态"
- 看到娱乐内容互动高，开始写"某某明星的职场启示"

**结果**：
- 粉丝困惑：这不是一个职场账号吗？
- 互动率持续下降
- 账号被用户评价为"四不像"

**关键教训**：
1. 选题必须服务于账号定位，不能什么火写什么
2. 热点选题要与账号定位有强关联才能蹭
3. 偶尔跨界是调味，不能成为主要内容

---

## 关键教训总结

| 教训 | 说明 |
|------|------|
| 系列化选题优于单篇孤零选题 | 用户有期待，团队有节奏 |
| 热点选题要有独特角度 | 不能做信息的搬运工 |
| 选题要服务于账号定位 | 偶尔跨界是调味剂，不是主菜 |
| 用户参与提升选题命中率 | 让用户参与选题决策 |
| 历史数据指导选题优化 | 复盘哪些选题表现好，为什么 |
