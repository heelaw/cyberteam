# 微信起名禁用词清单

## 一、绝对禁用词（触碰即封号）

### 政治敏感类
- 任何涉及国家领导人姓名/绰号
- 党史/军史/革命史相关敏感词
- 国际争议地区/组织名称
- 邪教/迷信相关词汇

### 色情低俗类
- 含有性暗示的谐音词
- 身体部位低俗描述
- 约X/一夜情相关暗示

### 违法犯罪类
- 毒品/赌博相关词汇
- 诈骗/灰产相关词汇
- 黑客/破解/外挂相关

### 虚假夸大类
- 最、第一、顶级、极致、顶尖
- 全球/世界级
- 绝对/100%

---

## 二、高频驳回词（尽量避免）

### 平台专属词
- 微信、微商、微店
- 公众号、订阅号、服务号
- 钱包、支付、刷卡

### 金融相关
- 贷款、借钱、信用卡
- 投资、理财、炒股
- 保险、社保代缴

### 医疗健康
- 医院、门诊、处方药
- 狐臭/狐臭/私密护理
- 试管婴儿/性别选择

### 教育培训
- 执业医师/药师/教师资格证
- 代报名、包过、免考
- 考研/四六级/考公代考

---

## 三、行业通用词（已被大量占用）

| 类别 | 已被占用的通用词 |
|------|------------------|
| 职场 | 职场、职场记、职场说、职场人 |
| 情感 | 情感、爱情、婚姻、恋爱 |
| 美食 | 美食、吃货、厨房、菜谱 |
| 母婴 | 妈妈、宝宝、育儿、亲子 |
| 科技 | 科技、智能、AI、机器人 |
| 财经 | 财经、金融、股票、投资 |

**建议**：避免直接使用通用词，添加差异化前缀/后缀

---

## 四、商标品牌风险词

### 知名企业品牌
- 苹果、华为、小米、腾讯、阿里
- 字节、抖音、快手、百度
- 奔驰、宝马、奥迪、大众

### 知名人物姓名
- 企业大佬（马云、马化腾、刘强东等）
- 明星艺人（除非是本人账号）
- 网红KOL（有粉丝基础的真名）

### 知名IP名称
- 漫威、迪士尼、哈利波特
- 游戏/动漫IP名称

---

## 五、字符限制

| 项目 | 要求 |
|------|------|
| 总字符数 | 6-20个字符 |
| 中文 | 1个汉字=1字符 |
| 英文 | 1个字母=1字符 |
| 数字 | 1个数字=1字符 |

**建议**：4-10个汉字为最佳，过短无辨识度，过长难记忆

---

## 六、自检清单

### 起名前必查

- [ ] 是否包含政治敏感词？
- [ ] 是否包含绝对化用语？
- [ ] 是否包含知名品牌/名人？
- [ ] 是否是纯数字/纯符号？
- [ ] 字符数是否在6-20之间？
- [ ] 微信搜索是否已被占用？
- [ ] 微博/小红书是否保持一致？

### 起名后验证

- [ ] 读一遍：是否顺口？
- [ ] 写一遍：输入法是否方便？
- [ ] 问一遍：别人听到能拼写出来吗？
- [ ] 想一遍：能准确传达账号价值吗？
