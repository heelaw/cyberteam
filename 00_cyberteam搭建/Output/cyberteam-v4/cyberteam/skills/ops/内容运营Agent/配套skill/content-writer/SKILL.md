# content-writer

## 简介

根据选题撰写符合调性的内容，包括短文案和中长转化文案。

## 触发关键词

```
内容写作、文案撰写、转化文案、写作方法、内容创作
```

## 触发场景描述

```
当需要撰写内容时触发：
- 需要撰写具体文章
- 需要写转化文案
- 需要写短文案
- 需要写活动推广文
```

## 输入

### 必需输入

| 字段 | 类型 | 说明 |
|------|------|------|
| topic | string | 选题 |
| positioning | object | 内容定位 |
| content_type | string | 内容类型（短/中/长） |

### 可选输入

| 字段 | 类型 | 说明 |
|------|------|------|
| word_limit | number | 字数限制 |
| cta | string | 行动号召 |

## 输出

```json
{
  "content": {
    "title": "标题",
    "outline": ["文章大纲"],
    "body": "正文内容",
    "cta": "行动号召"
  },
  "writing_notes": {
    "key_points": ["要点"],
    "style_check": "风格检查"
  }
}
```

## 方法论：内容写作类型

### 短文案4P原则

1. **有吸引力** (Pull)
2. **有价值** (Provide)
3. **有说服力** (Persuasive)
4. **有行动号召** (Push)

### 中长转化文案3方法

1. **讲故事**
   - 建立场景
   - 制造冲突
   - 提供解决方案

2. **设场景**
   - 描述用户痛点
   - 提供希望
   - 引导行动

3. **立人设**
   - 专家角度
   - 真实案例
   - 信任建立

### 禁止行为

1. 禁止标题党博眼球
2. 禁止内容与定位不符
3. 禁止抄袭或洗稿
4. 禁止出现错别字和语病
5. 禁止文案过长没有重点
