# 公众号起名避坑指南

## 触发关键词

**主关键词**: 公众号起名、公众号名字、公众号名称、取名字、账号命名

**次关键词**: 名字怎么取、名字推荐、名字大全、好名字、名称建议

**场景触发**: 用户新注册公众号需要命名，或现有公众号需要改名优化时自动触发

## 描述PUSHY

名字起错毁一年！公众号起名不是随便想想就行，一个好名字=免费流量+自动信任+过目不忘。踩中这5个坑，粉丝还没涨就开始掉。跟着走，让你一次取对名字。

## 核心定义

通过系统性的命名禁忌排查 + 创意方向匹配 + 命名方案输出，帮助企业/个人找到既有辨识度、又能传递价值、还能被目标用户记住的公众号名称。

## workflow_steps

### Step 1: 命名禁忌排查

**输入**:
- 账号定位（做什么）
- 目标用户（给谁看）
- 行业/领域

**处理**:
1. 排查通用禁忌：
   - 政治敏感词
   - 色情低俗词
   - 违法犯罪词
   - 虚假夸大词（最、极、第一等）
   - 侵权风险词（名人/品牌名）
2. 排查平台违规词：
   - 微信公众平台禁止注册词汇
   - 近30天高频驳回词汇
3. 排查竞争类比词：
   - 与头部账号重名/高度相似
   - 行业通用词被占情况

**输出**:
- 禁用词清单（明确不能用的）
- 高风险词清单（尽量避免的）
- 安全词清单（可用词汇）

**成功标准**:
- 无平台违规风险
- 无明显竞争劣势
- 至少保留5个可用基础词汇

### Step 2: 创意方向匹配

**输入**:
- Step 1的安全词清单
- 账号核心价值
- 目标用户画像
- 个性化需求（个人IP/品牌名/功能描述）

**处理**:
1. 确定命名风格：
   - 个人IP型：如「Spenser」「秋叶大叔」
   - 功能描述型：如「职场充电宝」「好用的数据」
   - 价值承诺型：如「增长黑盒」「运营人加油」
   - 场景情绪型：如「人间日记」「职场显微镜」
2. 确定命名结构：
   - 纯名字：2-4字中文名
   - 名前缀：XX说、XX记、XX姐
   - 名后缀：课堂、学院、研究所
3. 列出10个候选名称

**输出**:
- 确定命名风格和结构
- 10个候选名称清单
- 每个候选名称的核心卖点

**成功标准**:
- 风格与账号定位一致
- 结构简洁易记
- 覆盖不同创意方向

### Step 3: 最终方案输出

**输入**:
- 10个候选名称
- 账号定位和目标用户

**处理**:
1. 逐一验证可用性：
   - 微信搜索是否被占用
   - 微博/小红书是否有一致性
   - 域名/商标是否可注册
2. 评估记忆度和传播性：
   - 读一遍能记住吗？
   - 口语传播时会不会变形？
   - 输入法是否方便打字？
3. 给出最终推荐TOP3

**输出**:
- 最终推荐TOP3（带优先级）
- 每个方案的核心卖点
- 落地执行建议

**成功标准**:
- TOP3中至少有1个可立即注册
- 给用户提供明确的决策建议

## Critical Rules

1. **禁止政治敏感词** - 包含任何政治隐喻、领导人名字、时事热点的词汇一律不得使用
2. **禁止虚假夸大** - 不得使用"最、第一、顶级、极致"等绝对化用语
3. **禁止侵权名称** - 不得使用名人姓名、知名品牌、行业Top排名等易引发侵权的词汇
4. **禁止纯数字/符号** - 名称必须包含可读中文或英文单词，纯数字/符号组合无法通过审核
5. **禁止过长名称** - 名称控制在4-10个字为佳，过长难以记忆和传播
6. **禁止混淆用户** - 名称必须与账号实际内容高度相关，不得误导用户期待

## 质量评估标准

### 好名字的特征

- 一眼看懂：用户能立刻知道账号做什么
- 过目不忘：独特有记忆点，不易混淆
- 方便传播：口语/文字输入都方便
- 利于SEO：包含用户常搜索的关键词
- 合规安全：平台审核无风险

### 坏名字的特征

- 过于抽象，不知所云
- 与头部账号高度相似
- 生僻字多，难读难输入
- 名称与内容不符
- 触碰平台红线

## 注意事项

1. 起名后记得查重，避免与现有大号混淆
2. 个人IP号建议用真实姓名+定位组合
3. 企业号建议用品牌名+行业词组合
4. 名称确定前先查商标，避免后续纠纷
5. 一旦确定不要频繁改名，会损失已有粉丝认知
