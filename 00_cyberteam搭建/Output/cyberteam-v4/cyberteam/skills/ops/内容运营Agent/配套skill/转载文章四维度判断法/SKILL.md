# 转载文章四维度判断法

## 触发关键词

**主关键词**: 转载、要转载吗、能不能转载、这篇文章能用吗、转载判断

**次关键词**: 授权、版权、原创判断、侵权风险、内容合规

**场景触发**: 用户发来一篇文章询问是否适合转载，或需要评估某篇文章的转载价值时自动触发

## 描述PUSHY

转载不是复制粘贴！用四维度判断法，5分钟判断一篇文章值不值得转载。选错内容不仅掉粉还侵权，选对内容躺增1000+阅读。跟着走，让你的转载既安全又高效。

## 核心定义

通过「原创性评估 × 相关性评估 × 质量评估 × 版权评估」四个维度，系统性判断一篇文章是否值得转载，确保内容安全合规且对粉丝有价值。

## workflow_steps

### Step 1: 原创性评估

**输入**:
- 待判断文章全文
- 原文出处/作者信息

**处理**:
1. 核查文章是否标注"原创"
2. 搜索是否存在洗稿/拼接痕迹
3. 确认作者是否明确授权转载
4. 检查文章是否在平台首发规则保护期内

**输出**:
- 原创性判断结论：通过/存疑/不通过
- 如有风险点，列出具体问题

**成功标准**:
- 无"原创"标
- 无洗稿痕迹
- 有明确授权或属于公版内容

### Step 2: 相关性评估

**输入**:
- 待判断文章主题/角度
- 账号粉丝画像
- 账号内容定位

**处理**:
1. 分析文章主题与账号定位的匹配度
2. 评估粉丝是否会关心这个话题
3. 判断文章能否解决粉丝的真实需求
4. 检查是否存在与账号调性冲突的内容

**输出**:
- 相关性评分（1-5分）
- 相关点：匹配的原因
- 冲突点：潜在的不匹配因素

**成功标准**:
- 评分≥3分
- 无明显调性冲突

### Step 3: 质量评估

**输入**:
- 待判断文章内容质量
- 文章结构/逻辑/数据/案例

**处理**:
1. 评估文章是否有干货价值
2. 检查数据/案例是否可靠
3. 判断文章可读性（排版/逻辑/字数）
4. 评估转发分享价值（读者会不会转发）
5. 给 出质量综合评分

**输出**:
- 质量评分（1-5分）
- 优点清单
- 缺点清单
- 改进建议（如需要补充的内容）

**成功标准**:
- 评分≥3分
- 无事实性错误

### Step 4: 版权评估

**输入**:
- 文章作者信息
- 原文发布时间
- 原文平台
- 授权情况

**处理**:
1. 确认作者是否在文章中明确允许转载
2. 如无明确说明，通过正规渠道申请授权
3. 评估是否存在图片/字体版权风险
4. 检查是否需要标注来源/作者

**输出**:
- 版权合规性结论：可直接转载/需申请授权/不可转载
- 转载必须要素：来源标注、作者署名、授权截图

**成功标准**:
- 必须获得明确授权
- 必须保留原文来源信息
- 图片字体版权无风险

## Critical Rules

1. **禁止转载原创声明文章** - 凡标注"原创"的文章，一律不得未经授权转载
2. **禁止转载洗稿内容** - 经核实属于洗稿/拼接的文章，即使原链接有效也不得转载
3. **禁止无授权转载** - 必须获得作者明确授权，截图留证，口头承诺不算
4. **禁止忽略调性匹配** - 即使文章质量高，若与账号调性不符也不得转载
5. **禁止只图阅读量** - 转载内容必须对粉丝有实际价值，拒绝标题党内容
6. **禁止忽视版权细节** - 图片使用、数据引用、案例人物都必须确保无版权风险

## 质量评估标准

### 可转载文章特征

- 原文无"原创"标
- 作者明确允许转载
- 主题与账号定位高度相关
- 内容有干货价值
- 无事实性错误

### 不可转载文章特征

- 标注"原创"且无授权
- 洗稿/拼接痕迹明显
- 与账号调性严重冲突
- 纯标题党无实质内容
- 存在版权争议

## 注意事项

1. 转载前务必核实作者真实意图，部分作者虽未标注原创但也不希望被转载
2. 企业号转载需格外注意商业授权范围
3. 转载后的二次编辑必须保留原文核心观点，不得歪曲原意
4. 定期回顾已转载内容，避免因原文状态变化产生风险
