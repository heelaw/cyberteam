# 预算分配指南

## 一、预算分配原则

### 1.1 基本原则

| 原则 | 说明 |
|-----|------|
| 聚焦目标 | 预算向核心目标倾斜 |
| 效率优先 | ROI最高的渠道/形式优先 |
| 弹性预留 | 留出应急备用金 |
| 透明分配 | 各项目分配清晰可追溯 |

### 1.2 预算结构

```
总预算
├── 奖品/激励预算 (50-70%)
├── 推广预算 (20-30%)
├── 运营人力预算 (10-15%)
├── 技术/物料预算 (5-10%)
└── 应急备用金 (5%)
```

## 四、奖品/激励预算设计

### 4.1 激励结构设计

**人人有奖型**:
| 奖励层级 | 奖励内容 | 占比 | 成本 |
|---------|---------|-----|-----|
| 参与奖 | 小额优惠券 | 70% | 低 |
| 中等奖 | 中额优惠券/礼品 | 25% | 中 |
| 大奖 | 实物/高额礼品 | 5% | 高 |

**阶梯型**:
| 阶梯 | 条件 | 奖励内容 | 占比 |
|-----|------|---------|-----|
| 完成基础任务 | 100%完成 | 基础奖励 | 60% |
| 完成进阶任务 | 30%完成 | 进阶奖励 | 30% |
| 完成终极任务 | 10%完成 | 大奖 | 10% |

**排行型**:
| 排名 | 占比 | 奖励内容 |
|-----|------|---------|
| 第1名 | 1% | 大奖 |
| 前10名 | 9% | 中等奖 |
| 前100名 | 30% | 小奖 |
| 参与者 | 60% | 参与奖 |

### 4.2 成本核算模板

```
总奖品预算: X万元
目标参与人数: Y人
人均成本: X/Y万元

激励结构:
- 参与奖: Z元 × Y人 = Z×Y万元
- 中等奖: Z元 × Y人 = Z×Y万元
- 大奖: Z元 × Y人 = Z×Y万元
合计: Z×Y+Z×Y+Z×Y = Z×3Y万元
```

## 五、预算执行监控

### 5.1 预算监控指标

| 指标 | 计算公式 | 预警阈值 |
|-----|---------|---------|
| 预算消耗率 | 已消耗/总预算 | >80%预警 |
| 人均成本 | 已消耗/已参与人数 | 超出目标20%预警 |
| ROI | 产出/投入 | 低于预期50%预警 |

### 5.2 预算调整机制

**调整触发条件**:
- [ ] 预算消耗过快
- [ ] ROI明显低于预期
- [ ] 目标人群反馈不及预期

**调整流程**:
```
触发预警 → 原因分析 → 制定调整方案 → 审批 → 执行
```

### 5.3 预算复盘检查清单

- [ ] 预算实际消耗与计划对比
- [ ] 各项目实际花费明细
- [ ] ROI达成情况分析
- [ ] 预算使用效率评估
- [ ] 下次活动预算建议
