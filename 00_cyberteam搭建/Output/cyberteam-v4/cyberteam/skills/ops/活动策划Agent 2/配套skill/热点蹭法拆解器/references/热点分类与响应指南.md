# 热点分类与响应指南

## 一、热点分类体系

### 1.1 五大热点类型

| 类型 | 特征 | 可预测性 | 生命周期 | 响应要求 |
|-----|------|---------|---------|---------|
| 突发热点 | 突发事件、新闻 | 不可预测 | 短(24-72h) | 快速响应 |
| 固定热点 | 节假日、纪念日 | 可预测 | 长(1-4周) | 提前准备 |
| 娱乐热点 | 影视剧、综艺 | 周期性 | 中(3-7天) | 及时跟进 |
| 行业热点 | 报告、政策 | 可预测 | 中(1-2周) | 专业深度 |
| 节点热点 | 大促、周年 | 可预测 | 长(1-4周) | 系统规划 |

---

## 二、突发热点应对

### 2.1 识别与评估

**判断标准**:
- [ ] 是否与品牌相关
- [ ] 热度是否足够高
- [ ] 生命周期是否足够长
- [ ] 是否有负面风险

**评估矩阵**:
| 维度 | 高分标准 | 权重 |
|-----|---------|-----|
| 相关性 | 与品牌/产品直接相关 | 30% |
| 热度 | 微博热搜前10 / 抖音热榜 | 25% |
| 生命周期 | >48小时 | 20% |
| 正面性 | 无负面争议 | 25% |

### 2.2 响应机制

**响应等级**:
| 等级 | 触发条件 | 响应时间 | 审批流程 |
|-----|---------|---------|---------|
| P0 | 重大突发，强烈相关 | 2小时内 | 绿色通道 |
| P1 | 重大热点，弱相关 | 4小时内 | 快速审批 |
| P2 | 一般热点，相关 | 12小时内 | 常规审批 |

**必须满足的条件**:
- [ ] 热度达到一定阈值
- [ ] 与品牌有合理关联点
- [ ] 内容积极正面
- [ ] 无法律合规风险

### 2.3 禁忌事项

**绝对禁止**:
- 社会悲剧（地震、疫情、事故）
- 政治敏感话题
- 争议性社会事件
- 低俗内容

**可能禁止**:
- 蹭热度超过48小时
- 强行关联品牌
- 可能引发舆论争议

## 四、娱乐热点把握

### 4.1 娱乐热点来源

| 来源 | 内容类型 | 时效性 |
|-----|---------|--------|
| 微博热搜 | 娱乐新闻、明星八卦 | 高 |
| 抖音热榜 | 短视频挑战、流行梗 | 极高 |
| 知乎热榜 | 深度讨论、观点争议 | 中 |
| B站热门 | UP主创作、弹幕文化 | 中 |

### 4.2 借势角度

**热门影视**:
- 剧情讨论
- 角色分析
- 金句传播
- CP炒作

**热门综艺**:
- 环节模仿
- 金句传播
- 达人参与

**明星热点**:
- 结婚/离婚
- 获奖/塌房
- 新作品发布

### 4.3 注意事项

- [ ] 版权问题（不要直接使用影视素材）
- [ ] 粉丝情绪（明星相关要谨慎）
- [ ] 话题反转（娱乐热点容易反转）

## 五、行业热点策略

### 5.1 行业热点类型

| 类型 | 内容 | 适合形式 |
|-----|------|---------|
| 政策解读 | 监管政策分析 | 深度文章 |
| 报告发布 | 行业数据报告 | 专业内容 |
| 技术突破 | 新技术应用 | 科普文章 |
| 榜单排名 | 品牌/产品排名 | 对比分析 |

### 5.2 借势策略

**专业深度**:
- 发布专业解读文章
- 提供数据支撑
- 建立行业权威

**差异化角度**:
- 独特视角分析
- 细分领域深耕
- 观点鲜明有立场

## 六、热点响应检查清单

### 6.1 是否值得蹭

- [ ] 热点热度足够高
- [ ] 与品牌有合理关联
- [ ] 内容积极正面
- [ ] 无法律合规风险
- [ ] 在热点生命周期内

### 6.2 借势角度

- [ ] 有独特的切入点
- [ ] 不是简单贴热点标签
- [ ] 品牌价值能自然融入

### 6.3 执行准备

- [ ] 内容已准备完毕
- [ ] 审批流程已完成
- [ ] 物料已准备齐全
- [ ] 监测机制已建立

### 6.4 风险预案

- [ ] 有反转风险预案
- [ ] 有撤出条件
- [ ] 有舆情应对口径

---

## 七、热点生命周期管理

### 7.1 阶段策略

| 阶段 | 特征 | 策略 |
|-----|------|------|
| 萌发期 | 热度上升 | 快速跟进，抢占先机 |
| 爆发期 | 热度峰值 | 集中资源，制造声量 |
| 长尾期 | 热度下降 | 持续输出，保持热度 |
| 衰退期 | 热度消失 | 适时退出，转移到新热点 |

### 7.2 撤出机制

**撤出条件**:
- 热点热度下降到一定水平
- 出现明显反转或负面
- 品牌关联显得牵强
- 已达成传播目标

**撤出方式**:
- 停止新内容发布
- 撤下相关物料
- 发布正向收尾内容
