# 四类留存问题快速诊断卡

## 诊断流程图

```
用户留存问题
    │
    ├── 流失时机分析
    │       │
    │       ├── 首日/首周流失 → 类型2（短期价值缺失）
    │       │
    │       ├── 长期使用后流失 → 类型1（长期价值不足）
    │       │
    │       ├── 低频需求领域 → 类型3（天然低频）
    │       │
    │       └── 用户属性不符 → 类型4（用户不匹配）
    │
    └── 交叉验证
            │
            ├── 数据支撑
            ├── 用户反馈印证
            └── 竞品对比
```

---

## 类型1: 产品长期价值不足

### 核心特征

| 特征 | 表现 |
|------|------|
| 流失时机 | 使用一段时间后（通常1-4周） |
| 流失模式 | 缓慢稳定下滑 |
| 用户反馈 | "用几次就没意思了" |
| 典型场景 | 工具类、游戏类、内容类产品 |

### 快速自检

- [ ] 用户初期使用频率高
- [ ] 核心功能使用率持续下降
- [ ] 产品没有成长体系
- [ ] 没有持续的价值交付机制

**3项以上符合** → 类型1高度可能

### 改善方向

1. **建立成长体系**
   - 用户等级/积分/成就系统
   - 持续解锁新功能/内容
   - 成长可视化

2. **持续价值交付**
   - 定期更新核心内容
   - 保持功能新鲜感
   - 持续优化用户体验

3. **建立长期目标**
   - 帮用户设定长期目标
   - 阶段性里程碑设计
   - 长期价值可视化

---

## 类型2: 短期价值缺失

### 核心特征

| 特征 | 表现 |
|------|------|
| 流失时机 | 首次使用后（次日/首周） |
| 流失模式 | 快速断崖式下跌 |
| 用户反馈 | "不知道怎么用"、"没什么用" |
| 典型场景 | 新用户引导差、价值展示不清晰 |

### 快速自检

- [ ] 次日/首周流失率异常高
- [ ] 新用户引导流程复杂
- [ ] 首次体验没有即时正反馈
- [ ] 产品核心价值不清晰

**3项以上符合** → 类型2高度可能

### 改善方向

1. **优化新用户引导**
   - 简化引导流程
   - 突出核心价值
   - 个性化引导

2. **即时正反馈机制**
   - 首日体验奖励
   - 快速上手任务
   - 早期成就解锁

3. **清晰价值展示**
   - 首屏核心功能
   - 3秒理解产品价值
   - 明确使用收益

---

## 类型3: 天然低频场景

### 核心特征

| 特征 | 表现 |
|------|------|
| 需求本质 | 本身就是低频需求 |
| 流失特点 | 各家产品流失率都高 |
| 用户行为 | 有需求才来，无需求就走 |
| 典型场景 | 婚庆、搬家、装修等 |

### 快速自检

- [ ] 产品解决的是低频刚需
- [ ] 竞品留存率也普遍较低
- [ ] 用户使用频率极低
- [ ] 行业整体用户活跃度不高

**3项以上符合** → 类型3高度可能

### 改善方向

1. **创造关联使用场景**
   - 拓展高频关联需求
   - 日常辅助功能
   - 内容消费场景

2. **建立召回机制**
   - 定期push通知
   - 内容更新提醒
   - 优惠活动推送

3. **延伸服务链**
   - 从单次服务到长期关系
   - 增值服务拓展
   - 会员制锁客

---

## 类型4: 用户不匹配

### 核心特征

| 特征 | 表现 |
|------|------|
| 流失特点 | 用户属性与目标不符 |
| 用户反馈 | "不适合我"、"太贵了" |
| 获客问题 | 渠道带来非目标用户 |
| 定位问题 | 产品定位与营销不一致 |

### 快速自检

- [ ] 大量用户不是目标群体
- [ ] 用户期望与产品不符
- [ ] 获客渠道与目标用户不匹配
- [ ] 营销承诺与实际体验差异大

**3项以上符合** → 类型4高度可能

### 改善方向

1. **调整获客渠道**
   - 聚焦目标用户渠道
   - 优化广告投放定向
   - 渠道效果监测

2. **强化产品定位**
   - 明确目标用户画像
   - 营销与产品一致性
   - 拒绝无效流量

3. **用户分层运营**
   - 识别高价值用户
   - 针对性激励策略
   - 精准服务匹配

## 快速决策表

| 现象 | 最可能类型 | 优先级 | 改善周期 |
|------|------------|--------|----------|
| 首日流失严重 | 类型2 | 最高 | 1-2周 |
| 用几周后流失 | 类型1 | 高 | 1-3月 |
| 低频需求产品 | 类型3 | 中 | 持续优化 |
| 用户属性不匹配 | 类型4 | 高 | 调整渠道 |

---

## 常见误区

| 误区 | 正确认识 |
|------|----------|
| 留存低一定是产品问题 | 也可能是获客问题（类型4） |
| 低频产品留存低很正常 | 可以通过关联场景改善 |
| 用户流失无法挽回 | 找到原因就有改善空间 |
| 留存问题只能靠产品改版 | 有时调整获客渠道更有效 |
