# 用户留存自检清单

## 使用说明

当您不确定自己的产品是否存在留存问题时，可使用此清单进行自检。每个问题请根据实际情况填写"是/否/不确定"，最终可得出初步结论。

## 六、常见问题与解答

**Q: 如果符合多个类型怎么办？**
A: 多数产品会同时存在多个问题，按优先级排序，优先解决最严重的问题。

**Q: 行业基准从哪里获取？**
A: 可以从竞品分析报告、行业研究报告、或第三方数据平台获取。

**Q: 自检结果可靠吗？**
A: 这只是初步判断，具体原因需要更深入的数据分析才能确认。
