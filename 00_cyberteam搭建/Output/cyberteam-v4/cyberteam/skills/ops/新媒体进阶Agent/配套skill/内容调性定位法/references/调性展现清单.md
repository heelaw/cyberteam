# 调性展现清单

## 一、语言风格展现

### 1. 词汇选择

**温暖亲切型**
- 常用词：我们、你呀、咱们、亲爱的
- 避免词：您、必须、一定、强制

**专业权威型**
- 常用词：数据、研究、表明、显示
- 避免词：我觉得、应该是、可能

**轻松幽默型**
- 常用词：哈哈、绝了、太搞笑、笑死
- 避免词：严肃、认真、庄重

### 2. 句式结构

**陈述句型**
- 温暖型：今天想和你聊聊...
- 专业型：数据显示...
- 幽默型：你绝对想不到...

**疑问句型**
- 引发思考：难道你不想...
- 制造好奇：你知道吗...
- 互动引导：你会怎么做...

### 3. 语气词使用

| 类型 | 适用 | 示例 |
|------|------|------|
| 感叹型 | 强调情绪 | 真的太香了！ |
| 疑问型 | 引发好奇 | 竟然是这样？ |
| 柔和型 | 亲切感 | 呀、呢、哦 |
| 果断型 | 增加可信度 | 绝对、肯定 |

## 二、视觉风格展现

### 1. 封面设计

**统一要素**
- 固定版式
- 固定元素
- 固定字体
- 固定配色

**不同调性示例**
- 温暖型：圆角、暖色、手写体
- 专业型：直角、深色、衬线体
- 活泼型：撞色、emoji、表情包

### 2. 图片风格

| 调性 | 图片风格 |
|------|----------|
| 高端 | 精致、摆拍、高清 |
| 亲和 | 生活化、真实、有氛围感 |
| 潮流 | 时尚、滤镜感、创意 |

### 3. 排版规范

- 字号层级
- 行间距
- 留白比例
- 重点标注方式

## 三、内容风格展现

### 1. 开场方式

| 调性 | 开场方式 |
|------|----------|
| 干货型 | "今天分享..."、"3个方法..." |
| 情感型 | "想起一件事..."、"最近..." |
| 故事型 | "话说..."、"还记得..." |
| 互动型 | "评论区告诉我..."、"你有过..." |

### 2. 内容结构

| 结构 | 特点 | 适用调性 |
|------|------|----------|
| 清单式 | 1、2、3、4 | 干货型 |
| 故事型 | 起承转合 | 情感型 |
| 对比型 | 前后对比 | 种草型 |
| 问答型 | 问题+解答 | 专业型 |

### 3. 结尾方式

| 类型 | 示例 |
|------|------|
| 互动型 | "你怎么看？"、"评论区聊聊" |
| 总结型 | "总结一下..."、"划重点..." |
| 引导型 | "下期想看..."、"点个赞..." |
| 悬念型 | "下期告诉你..." |
