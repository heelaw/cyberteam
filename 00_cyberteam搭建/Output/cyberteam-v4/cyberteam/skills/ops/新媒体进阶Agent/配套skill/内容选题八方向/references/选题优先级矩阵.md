# 选题优先级矩阵

## 一、选题价值评估

### 流量潜力

| 等级 | 指标 | 说明 |
|------|------|------|
| A | 10万+ | 热点话题、刚需问题 |
| B | 1-10万 | 大众需求、有一定热度 |
| C | 5000-1万 | 小众需求、垂直用户 |
| D | <5000 | 冷门话题 |

### 转化价值

| 等级 | 指标 | 说明 |
|------|------|------|
| A | 高转化 | 直接关联产品、能引发购买 |
| B | 中转化 | 相关需求、间接引导 |
| C | 低转化 | 品牌建设、用户信任 |
| D | 无转化 | 纯内容分享 |

### 制作成本

| 等级 | 指标 | 说明 |
|------|------|------|
| A | 低成本 | 现有素材、简单整理 |
| B | 中成本 | 需要一定准备 |
| C | 高成本 | 需要大量调研/制作 |
| D | 超高成本 | 定制内容、专业制作 |

## 二、优先级排序

### 第一优先级：A+A类选题

特征：流量大 + 转化高 + 成本低

示例：
- 热点+产品相关
- 刚需问题+解决方案
- 用户最关心的话题

### 第二优先级：A+B或B+A类选题

特征：流量大+转化中 或 流量中+转化高

示例：
- 热点话题（流量大+转化中）
- 垂直干货（流量中+转化高）

### 第三优先级：B+B类选题

特征：流量中+转化中

示例：
- 常规内容更新
- 知识科普类

### 第四优先级：C类选题

特征：流量小或转化低

示例：
- 品牌故事
- 幕后花絮
- 互动话题

## 四、不同阶段账号的选题策略

### 冷启动期（0-1000粉）

目标：涨粉、提升曝光

优先级：
1. 痛点型（解决用户问题）
2. 知识型（提供价值）
3. 热点型（获取流量）

### 成长期（1000-1万粉）

目标：建立人设、提升互动

优先级：
1. 案例型（建立专业性）
2. 互动型（提升互动）
3. 痛点型（持续提供价值）

### 成熟期（1万粉+）

目标：转化、变现

优先级：
1. 转化型（引导购买）
2. 案例型（建立信任）
3. 品牌型（强化人设）

## 五、选题自检清单

### 检查项
- [ ] 是否符合账号定位
- [ ] 是否对目标用户有价值
- [ ] 是否有独特角度
- [ ] 是否有素材支撑
- [ ] 是否有时效性
- [ ] 是否有传播点

### 优化问题
- 标题是否吸引人
- 封面是否有亮点
- 内容是否有干货
- 结尾是否有互动引导
