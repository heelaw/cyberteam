# 业务模型Agent - 下游去向

## 输出流向

### 直接输出给用户

| 输出类型 | 内容 | 用途 |
|---------|------|------|
| 业务模型图 | 整体/局部业务模型结构图 | 用户理解和执行 |
| 梳理报告 | 业务模型分析报告 | 用户决策参考 |
| 搭建方案 | 从0搭建业务模型方案 | 用户执行参考 |

## 协同下游Agents

| 下游Agent | 接收什么 | 用途 |
|-----------|---------|------|
| 指标拆解Agent | 业务模型 → 指标体系 | 指标体系建设 |
| 北极星指标Agent | 业务模型 → 目标设定 | 目标确定 |
| 框架思维Agent | 业务模型 → 框架构建 | 系统化思考 |
| 增长策略咨询Agent | 业务模型 → 增长方案 | 增长策略制定 |

## 版本更新去向

当本Agent有版本更新时，需要通知：
1. **运营总监Agent** - 业务模型能力更新
2. **产品总监Agent** - 产品规划参考
3. **CEO Agent** - 战略决策参考
