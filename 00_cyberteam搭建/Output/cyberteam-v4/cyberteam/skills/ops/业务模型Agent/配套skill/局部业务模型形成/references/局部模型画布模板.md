## 局部模型检核清单

### 定位检核
- [ ] 定位清晰，不与其他模块重叠
- [ ] 与整体业务目标对齐
- [ ] 边界明确

### 目标检核
- [ ] 目标具体可量化
- [ ] 目标可实现但有挑战
- [ ] 与整体目标协同

### 策略检核
- [ ] 策略能支撑目标达成
- [ ] 关键动作清晰可执行
- [ ] 有备选方案

### 指标检核
- [ ] 指标可追踪可衡量
- [ ] 与目标对应
- [ ] 有预警机制

### 资源检核
- [ ] 资源配置能支撑策略执行
- [ ] 资源获取可行
- [ ] 有资源协调机制
