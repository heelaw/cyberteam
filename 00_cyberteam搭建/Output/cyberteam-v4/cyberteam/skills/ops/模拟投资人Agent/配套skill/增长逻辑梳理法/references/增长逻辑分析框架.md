# 增长逻辑分析参考框架

## 一、增长来源分解

### 收入增长公式

```
收入 = 用户数 × 客单价
用户数 = 新增用户 - 流失用户 + 留存用户
```

### 增长来源类型

| 类型 | 说明 | 占比参考 |
|------|------|----------|
| 获客增长 | 新用户获取 | 40-60% |
| 留存增长 | 老用户持续使用 | 20-40% |
| 变现增长 | 客单价提升 | 10-20% |
| 扩展增长 | 新产品/新市场 | 5-15% |

---

## 二、增长驱动因素矩阵

### 产品驱动型增长

- 产品体验优秀
- 自然推荐率高
- 不依赖投放

**指标特征**：自然增长占比>60%

### 渠道驱动型增长

- 依赖广告投放
- 渠道成本占比高
- 可规模化

**指标特征**：CAC占比高，但可控

### 品牌驱动型增长

- 品牌知名度高
- 自然流量大
- 转化率高

**指标特征**：品牌词搜索占比高

### 口碑驱动型增长

- 用户推荐意愿强
- K因子>1
- 病毒传播

**指标特征**：NPS>50，K因子>1

### 资本驱动型增长

- 大量补贴获客
- 增长快但不可持续
- 依赖融资

**指标特征**：补贴占比>30%

## 三、增长效率指标标准

### LTV/CAC 评估标准

| LTV/CAC | 评价 | 投资建议 |
|---------|------|----------|
| >3 | 优秀 | 强烈推荐 |
| 2-3 | 良好 | 推荐 |
| 1-2 | 一般 | 谨慎 |
| <1 | 较差 | 不推荐 |

### 回收周期标准

| 回收周期 | 评价 |
|----------|------|
| <6个月 | 优秀 |
| 6-12个月 | 良好 |
| 12-18个月 | 一般 |
| >18个月 | 较差 |

### 获客成本（CAC）行业基准

| 行业 | 合理CAC |
|------|---------|
| SaaS | 客单价的1/12 |
| 电商 | 客单价的20-30% |
| 金融 | 客单价的30-50% |
| 游戏 | 客单价的10-20% |

## 四、增长可持续性评估

### 自然增长占比

| 占比 | 可持续性 |
|------|----------|
| >70% | 高度可持续 |
| 50-70% | 较可持续 |
| 30-50% | 一般 |
| <30% | 不可持续 |

### 复购率标准

| 行业 | 优秀 | 良好 | 一般 |
|------|------|------|------|
| 电商 | >40% | 25-40% | <25% |
| SaaS | >90% | 80-90% | <80% |
| 订阅 | >60% | 40-60% | <40% |

### 流失率预警

| 流失率 | 状态 | 风险 |
|--------|------|------|
| <5%/月 | 健康 | 低 |
| 5-10%/月 | 预警 | 中 |
| >10%/月 | 危险 | 高 |

---

## 五、增长逻辑风险识别

### 常见增长陷阱

1. **补贴泡沫**：补贴停止后增长归零
2. **流量依赖**：渠道政策变化导致增长停滞
3. **虚假繁荣**：羊毛党造成的虚假增长
4. **规模不经济**：增长越快亏损越大

### 风险识别清单

- [ ] 自然增长占比是否>50%？
- [ ] LTV/CAC是否>2？
- [ ] 获客成本是否在上升？
- [ ] 用户留存是否稳定？
- [ ] 增长是否依赖单一渠道？
