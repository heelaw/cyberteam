# 小红书笔记创作模板

## 一、产品推荐类笔记

### 封面设计

**类型选择**：
- [ ] IP主题型（产品实拍）
- [ ] 前后对比型（使用效果）
- [ ] 拼图+文字型（多款对比）

**封面尺寸**：3:4竖幅（推荐）

**标题模板**：
```
[标题]：10字以内，突出核心卖点
[副标题]：补充说明（可选）
```

### 正文结构（400-600字）

```markdown
【开头/痛点】
描述用户痛点或使用场景（50-80字）

【产品介绍】
产品名称+价格+购买渠道（80-100字）

【亮点/特点】
亮点1：具体描述
亮点2：具体描述
亮点3：具体描述

【使用效果/体验】
真实使用感受+对比（100-150字）

【总结】
适合人群+使用建议+回购意愿（50-80字）

【互动引导】
记得点赞收藏哦，有问题评论区见
```

### 关键词布局

**核心关键词**（必选）：
```
#[品类] #好物推荐 #[品牌] #[场景]
```

**长尾关键词**（可选）：
```
#[具体功效] #适用人群 #价格区间
```

## 二、干货分享类笔记

### 封面设计

**类型选择**：
- [ ] IP主题型（书籍/工具实拍）
- [ ] IP主题+关键词（突出主题）

**封面文字**：
- 主标题：方法/技巧名称
- 副标题：时间/效果

### 正文结构（500-700字）

```markdown
【引入】
为什么这个方法/技巧重要（50-80字）

【方法/技巧详解】
步骤1：具体操作
步骤2：具体操作
步骤3：具体操作
...（200-300字）

【案例/论证】
真实案例+数据支撑（100-150字）

【注意事项】
避坑指南+常见问题（80-100字）

【总结】
适用场景+预期效果（50-80字）

【互动引导】
你还有什么好方法？欢迎分享
记得收藏起来慢慢看
```

### 关键词布局

```
#[领域] #干货分享 #[具体主题] #技巧 #教程 #[场景]
```

## 三、观点表达类笔记

### 封面设计

**类型选择**：
- [ ] IP主题型（人物+文字）
- [ ] 对比型（观点VS观点）

**封面文字**：
- 核心观点（简短有力）
- 反问句/惊叹句

### 正文结构（500-700字）

```markdown
【现象描述】
描述一个普遍现象或热点话题（80-100字）

【观点表达】
明确表达你的观点（50-80字）

【论证/案例】
论证1：案例或数据
论证2：案例或数据
论证3：案例或数据
（200-300字）

【不同视角】
考虑反面观点或注意事项（80-100字）

【结论/建议】
总结观点+给出建议（50-80字）

【互动引导】
你怎么看？评论区讨论
点赞+关注，下次分享不迷路
```

### 关键词布局

```
#[领域] #观点 #[话题] #思考 #[相关标签]
```

## 四、生活记录类笔记

### 封面设计

**类型选择**：
- [ ] 拼图+文字型（Photo log）
- [ ] IP主题型（真实场景）

**封面要求**：
- 色调统一
- 排版整齐
- 文字简洁

### 正文结构（300-500字）

```markdown
【时间/地点/事件】
记录基本信息（30-50字）

【过程描述】
详细描述过程或体验（150-200字）

【感受/收获】
个人感受或收获（80-100字）

【Tips】
相关建议或注意事项（50-80字）

【互动引导】
你也有类似的经历吗？
欢迎评论区分享
```

### 关键词布局

```
#[领域] #生活记录 #[场景] #日常 #[相关标签]
```

## 五、通用检查清单

### 封面检查
- [ ] 尺寸正确（3:4竖幅推荐）
- [ ] 主体清晰
- [ ] 文字可读（暗底浅字/亮底深字）
- [ ] 风格统一

### 标题检查
- [ ] 使用了11种标题方法之一
- [ ] 字数适中（10-20字）
- [ ] 包含关键词
- [ ] 有吸引力

### 内容检查
- [ ] 字数符合要求（400-600字）
- [ ] 分段清晰（回车法）
- [ ] 无敏感词
- [ ] emoji使用恰当

### 关键词检查
- [ ] 话题标签5-8个
- [ ] 核心关键词在前
- [ ] 包含长尾关键词
- [ ] 标签相关性强

### 互动检查
- [ ] 有明确的互动引导
- [ ] 文案有投票区或提问
- [ ] 结尾有收藏/关注引导

---

**使用说明**：选择对应的笔记类型，按照模板创作，完成后对照检查清单逐项检查。
