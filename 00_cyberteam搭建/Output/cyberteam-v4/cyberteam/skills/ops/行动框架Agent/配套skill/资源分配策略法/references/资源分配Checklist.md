| 资源盘点完整 | □ | 不遗漏任何资源类型 |

| 约束条件明确 | □ | 不超出资源上限 |

---

## 分配原则检查

### 必须满足

- [ ] 不超过总资源量
- [ ] 有明确的优先级排序
- [ ] 战略方向有足够支撑
- [ ] 留有应急储备

### 应该避免

- [ ] 没有平均分配（摊大饼）
- [ ] 没有忽视长期投入
- [ ] 没有一次性分配完
- [ ] 没有只看短期回报

## 常见错误清单

| 错误类型 | 典型表现 | 正确做法 |
|----------|----------|----------|
| 平均主义 | 每个方向都差不多 | 突出重点，有所为有所不为 |
| 短视主义 | 只看眼前收益 | 平衡短期和长期 |
| 饱和攻击 | 资源一次投完 | 保留调整空间 |
| 忽视风险 | 没留储备金 | 预留应急资源 |
| 主观臆断 | 拍脑袋分配 | 基于数据和评估 |
| 脱离目标 | 分配和目标脱节 | 资源服务目标 |
