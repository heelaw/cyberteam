# 用户留存与激励体系 - 评估清单

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 诊断准确性 | 25% | 价值洼地识别是否准确 |
| 路径合理性 | 20% | 用户引导路径是否顺畅 |
| 激励有效性 | 20% | 激励是否能驱动目标行为 |
| 激励平衡性 | 15% | 三层激励是否平衡配合 |
| 成本可控性 | 10% | 激励成本是否可持续 |
| 风险可控性 | 10% | 是否有防薅羊毛机制 |

## 评分等级

| 等级 | 分数 | 说明 |
|------|------|------|
| 优秀 | 90-100 | 全面满足所有维度，激励体系完整有效 |
| 良好 | 75-89 | 满足大部分维度，体系可用 |
| 合格 | 60-74 | 基本满足维度要求，需补充 |
| 不合格 | <60 | 存在明显缺陷，需重新设计 |

## 逐项检查清单

### 诊断准确性 (25分)

- [ ] 用户流失原因有数据支撑
- [ ] 价值洼地识别基于用户需求分析
- [ ] 不是主观猜测，有调研或数据依据
- [ ] 区分了主次原因

### 路径合理性 (20分)

- [ ] 用户典型路径清晰
- [ ] 关键节点识别准确
- [ ] 路径长度适中（不过长也不过短）
- [ ] 考虑了不同用户类型的路径差异

### 激励有效性 (20分)

- [ ] 激励与目标行为直接挂钩
- [ ] 激励力度足够驱动行为
- [ ] 激励形式用户认可
- [ ] 有即时反馈机制

### 激励平衡性 (15分)

- [ ] 物质激励作为基础保障
- [ ] 精神激励满足高级需求
- [ ] 社交激励增强用户粘性
- [ ] 三者相互配合而非孤立

### 成本可控性 (10分)

- [ ] 激励成本有预算规划
- [ ] 单用户激励成本在可接受范围
- [ ] 激励带来的收益大于成本
- [ ] 有成本监控和调整机制

### 风险可控性 (10分)

- [ ] 有防薅羊毛机制
- [ ] 有异常监控
- [ ] 有止损预案
- [ ] 激励不被滥用

## 常见问题

### 问题1：激励与价值脱钩
**表现**：发红包留用户，但用户只领红包不用产品
**修复**：激励必须与产品核心价值体验挂钩

### 问题2：激励过于复杂
**表现**：设计了几十种激励规则，用户看不懂
**修复**：简化激励规则，核心激励不超过5种

### 问题3：纯物质激励
**表现**：只靠发优惠券、红包维持用户
**修复**：增加精神激励和社交激励的比重
