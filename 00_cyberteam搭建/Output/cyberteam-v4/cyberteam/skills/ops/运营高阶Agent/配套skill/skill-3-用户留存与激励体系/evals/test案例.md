# 用户留存与激励体系 - 测试案例

## 测试案例1

**难度**：medium
**场景**：社区产品用户激励设计

**输入**：
```
背景：一款内容社区产品，定位是帮助用户发现好内容、连接同好。用户可以通过发布内容、点赞、评论、收藏等行为互动。近一个月数据：DAU 10万，但次日留存率只有35%，7日留存率仅15%。产品团队希望通过用户激励体系改善留存。
核心观察：大部分用户是内容消费者，很少主动发布内容。优质内容生产者流失严重。
```

**预期输出**：
```
1. 流失原因诊断：
   - 产品价值不明确（用户不知为什么要留下来）
   - 价值获取成本高（需要主动发现内容）
   - 缺乏行动动力（没有即时反馈）
   核心原因：缺乏正向反馈循环

2. 价值洼地诊断：
   用户需求：被认可、被关注、发现价值
   产品供给：内容丰富，但缺乏引导用户发现价值的机制
   核心洼地：缺乏对内容生产者的认可和反馈机制

3. 用户引导路径设计：
   [新用户进入]
       ↓
   [内容消费] → 完成首刷
       ↓
   [互动行为] → 点赞/收藏
       ↓
   [激励反馈] → 获得认可
       ↓
   [生产内容] → 开始发布
       ↓
   [持续激励] → 获得更多反馈
       ↓
   [成为核心用户]

4. 激励体系设计：
   ┌─────────────┬──────────────────┬────────────────┐
   │   类型      │   具体措施        │   触发条件      │
   ├─────────────┼──────────────────┼────────────────┤
   │ 物质激励    │ 发布内容得积分    │ 每次发布内容    │
   │            │ 优质内容奖励      │ 被精选置顶      │
   ├─────────────┼──────────────────┼────────────────┤
   │ 精神激励    │ 成就徽章系统      │ 达成里程碑      │
   │            │ 创作者认证        │ 发布10/50/100篇 │
   ├─────────────┼──────────────────┼────────────────┤
   │ 社交激励    │ 粉丝系统          │ 有人关注        │
   │            │ 评论回复通知      │ 被回复          │
   │            │ 精选评论置顶      │ 被精选          │
   └─────────────┴──────────────────┴────────────────┘

5. 防风险机制：
   - 发布内容需审核，防止水化
   - 积分有上限，防止刷分
   - 异常行为监控
```

**验收标准**：
- [ ] 流失原因诊断有数据支撑
- [ ] 激励覆盖三类且相互配合
- [ ] 激励与内容生产行为挂钩
- [ ] 有防薅羊毛机制

## 测试案例2

**难度**：simple
**场景**：电商产品新手引导激励

**输入**：
```
背景：某垂直电商App，主打年轻女性时尚品类。上线半年，用户注册量50万，但新用户首单转化率仅8%。复购率也不理想，30天复购率25%。希望通过新手激励提升转化和复购。
用户特征：18-25岁年轻女性，价格敏感度高，喜欢跟风种草。
```

**预期输出**：
```
1. 流失原因诊断：
   - 价值获取成本高（首次购物决策成本高）
   - 缺乏信任（对商品质量不确定）
   - 激励不足（没有足够动力完成首单）
   核心原因：新用户缺乏完成首单的信心和动力

2. 价值洼地诊断：
   用户需求：便宜、放心、知道买什么
   产品供给：商品丰富，但缺乏引导
   核心洼地：新用户不知道买什么+不信任

3. 引导路径：
   [下载注册]
       ↓
   [完善资料] → 获取偏好标签
       ↓
   [新人专区] → 专属低价商品
       ↓
   [下单完成] → 首单激励
       ↓
   [复购引导] → 品类券

4. 激励体系：
   ┌─────────────┬──────────────────┬────────────────┐
   │   类型      │   具体措施        │   触发条件      │
   ├─────────────┼──────────────────┼────────────────┤
   │ 物质激励    │ 新人专享价（10%off）│ 进入新人专区    │
   │            │ 首单立减20元      │ 完成首单        │
   │            │ 复购券包          │ 首单后发放      │
   ├─────────────┼──────────────────┼────────────────┤
   │ 精神激励    │ 新人专属标识      │ 完成注册        │
   │            │ 购物成就徽章      │ 完成首单        │
   ├─────────────┼──────────────────┼────────────────┤
   │ 社交激励    │ 分享得优惠券      │ 分享商品        │
   │            │ 闺蜜拼团          │ 邀请好友拼团     │
   └─────────────┴──────────────────┴────────────────┘

5. 激励成本控制：
   - 首单激励成本由商家承担
   - 优惠券设置使用门槛确保转化
   - 设置有效期促进及时转化
```

**验收标准**：
- [ ] 诊断逻辑清晰
- [ ] 激励针对首单转化
- [ ] 考虑了用户价格敏感特点
- [ ] 成本由多方分担
