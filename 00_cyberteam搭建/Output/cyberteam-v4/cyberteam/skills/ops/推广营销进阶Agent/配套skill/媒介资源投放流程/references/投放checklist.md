# 投放Checklist

## 上线前检查

### 1. 目标确认
- [ ] 投放目标明确（品牌/转化/互动）
- [ ] 核心指标已设定（CTR/CVR/CPA）
- [ ] 预算已审批

### 2. 素材检查
- [ ] 素材已制作完成
- [ ] 素材符合平台规范
- [ ] 至少2套素材可测试
- [ ] 落地页已准备
- [ ] 落地页与素材匹配

### 3. 定向检查
- [ ] 定向已设置（年龄/地域/兴趣）
- [ ] 排除人群已设置
- [ ] 竞品词已排除（如需要）

### 4. 追踪检查
- [ ] 转化追踪已配置
- [ ] 事件追踪已验证
- [ ] URL参数已添加

### 5. 账户检查
- [ ] 账户余额充足
- [ ] 预算设置正确
- [ ] 出价策略已确认

---

## 投放中检查

### 每日检查
- [ ] 数据是否正常跑通
- [ ] 消耗是否正常
- [ ] 转化是否正常

### 异常情况
- [ ] 无消耗 → 检查出价/定向/素材
- [ ] 消耗过快 → 调整预算/出价
- [ ] 无转化 → 检查落地页/定向

---

## 投放后复盘

### 数据记录
- [ ] 总曝光量
- [ ] 总点击量
- [ ] CTR
- [ ] 转化量
- [ ] CVR
- [ ] 总花费
- [ ] CPA
- [ ] ROI

### 效果分析
- [ ] 是否达到目标
- [ ] 哪个渠道效果最好
- [ ] 哪个素材效果最好
- [ ] 哪些定向效果最好

### 优化建议
- [ ] 下一轮投放优化方向
- [ ] 需要调整的策略
