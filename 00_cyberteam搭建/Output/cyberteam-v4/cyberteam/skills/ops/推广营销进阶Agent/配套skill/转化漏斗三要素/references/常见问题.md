# 转化漏斗常见问题解答

## Q1: 转化率低但不知道问题在哪？

**诊断步骤**：
1. 先算CTR判断是"曝光→点击"还是"点击→转化"的问题
2. CTR低 → 优化素材和定向
3. CVR低 → 优化落地页和价格
4. 都低 → 综合诊断产品竞争力

---

## Q2: 行业参考值怎么选？

**选择原则**：
1. 先按行业选（电商/教育/SaaS等）
2. 再按客单价区间校正
3. 最后按渠道调整（搜索>信息流）

**优先级**：客单价 > 行业 > 渠道

---

## Q3: 数据量太少怎么分析？

**数据量标准**：
- 最少1000曝光才可分析CTR
- 最少100点击才可分析CVR
- 数据量不足时建议先积累数据

**小流量建议**：
- 延长投放周期积累数据
- 多渠道对比看趋势
- 用同比环比判断变化

---

## Q4: 不同渠道数据能直接对比吗？

**不建议直接对比**：
- 抖音CTR 1%和百度CTR 1%意义不同
- 渠道特性差异大

**正确方式**：
- 各渠道分别对标行业该渠道均值
- 跨渠道对比ROI而非CTR/CVR

---

## Q5: 转化率突然下降怎么办？

**排查步骤**：
1. 检查是否改了素材/定向/出价
2. 检查竞品是否有大动作
3. 检查产品是否有负面口碑
4. 检查流量质量是否下降

**应急方案**：
- 暂停表现差的计划
- 启用备用素材
- 缩小定向范围

---

## Q6: 转化率多高才算好？

**没有统一标准**，影响因素：
- 行业：游戏10% vs 医疗0.5%
- 客单价：99元5% vs 9999元0.5%
- 渠道：搜索5% vs 信息流1%

**正确做法**：
- 和自己历史数据比（环比）
- 和同渠道同行业比（对标）
- 关注趋势变化而非绝对值

---

## Q7: 优化后效果不明显？

**可能原因**：
1. 优化方向错误（没诊断对卡点）
2. 测试周期不够（建议至少7天）
3. 样本量不足（需要更多流量验证）
4. 多个问题同时存在（需要系统优化）

**建议**：
- 用A/B测试验证每个假设
- 一次只改一个变量
- 记录每次测试结果

---

## Q8: 如何提升整体转化效率？

**系统优化思路**：
```
1. 精准定向 → 提升CTR
2. 优化素材 → 提升CTR
3. 优化落地页 → 提升CVR
4. 优化价格 → 提升CVR
5. 优化流程 → 减少流失
6. 组合销售 → 提升客单价
```

**优先级**：先CTR后CVR，先渠道后落地页
