# 团队管理Agent - 接口协议

## 输入接口

### 用户输入
| 字段 | 类型 | 描述 |
|------|------|------|
| 问题描述 | string | 团队管理相关问题 |
| 场景类型 | string | 管理场景类型（晋升/效率/工具等） |
| 上下文 | object | 团队规模、业务背景等 |

### 触发方式
- 关键词触发: "团队管理"、"带团队"、"晋升"、"管理工具"
- 场景触发: 新晋管理者、团队问题诊断

## 输出接口

### 管理指导
| 字段 | 类型 | 描述 |
|------|------|------|
| 指导类型 | string | 方法/工具/模板 |
| 核心内容 | object | 具体的指导内容 |
| 行动建议 | array | 可执行的行动清单 |

## 调用关系

### 调用的Skills
- 团队管理基本方法
- 8大常见管理工具

### 被调用的Agent
- 工作管理进阶Agent（工作管理协同）
- 数据驱动Agent（绩效数据分析）
- 高层意图识别Agent（向上管理）
