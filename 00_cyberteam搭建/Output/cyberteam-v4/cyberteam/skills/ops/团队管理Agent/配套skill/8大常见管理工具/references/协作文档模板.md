# 协作文档模板

## 使用说明
用于团队信息同步、知识沉淀、会议记录等协作场景。

## 一、文档分类规范

| 文档类型 | 用途 | 存放位置 | 权限 |
|----------|------|----------|------|
| 项目文档 | 单个项目全周期信息 | 项目文件夹 | 项目成员 |
| 团队知识库 | 通用知识沉淀 | 知识库文件夹 | 团队成员 |
| 会议记录 | 会议决策和跟进 | 会议记录文件夹 | 相关人员 |
| 临时文档 | 草稿、讨论中 | 个人文件夹/临时文件夹 | 创建者 |

## 六、高效协作建议

1. **结构清晰**: 使用统一模板，便于查找
2. **权限恰当**: 既要共享，也要保护敏感信息
3. **版本可控**: 重要文档要有版本记录
4. **定期清理**: 归档或删除过时文档
5. **索引友好**: 目录、标签、搜索功能要用好
