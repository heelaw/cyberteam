# 策略执行Agent - 上游依赖

## 上游Agent（输入来源）

### 直接上游

| Agent名称 | 提供内容 | 说明 |
|-----------|----------|------|
| 策略规划Agent | 策略框架、目标、约束条件 | 策略制定完成后的执行阶段 |
| 增长Agent | 增长目标、业务假设 | 增长项目启动时 |
| 业务操盘手 | 业务背景、问题描述 | 用户直接发起时 |

### 间接上游

| Agent名称 | 提供内容 | 说明 |
|-----------|----------|------|
| 老板关注点分析器 | 老板关注的核心指标 | 通过策略规划Agent传递 |
| 业务模型Agent | 业务模型结构 | 通过策略规划Agent传递 |

## 依赖关系

```
老板关注点分析器 → 策略规划Agent → 策略执行Agent
                                           ↓
业务模型Agent ──────────────────────────────→│
                                           ↓
增长Agent ─────────────────────────────────→│
                                           ↓
                                    工作计划生成器
```

## 依赖的外部资源

- 用户提供的业务背景信息
- 业务假设验证结果
- 可用的资源清单（人力、资金、时间）
