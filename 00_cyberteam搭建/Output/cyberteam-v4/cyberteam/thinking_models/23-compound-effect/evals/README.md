# 复利效应专家测试用例

## 正面测试

### 测试1: 习惯养成分析
- 输入:
```
【复利效应分析请求】
情境描述：每天坚持学习英语1小时
初始投入：每天1小时
预期行动：持续学习1年
时间维度：1年
```
- 期望输出:
  - 复利识别表格完整
  - 增长评估包含年化增长率
  - 时间价值表展示累积效果
  - 复利触发条件检查
  - 策略建议涵盖习惯复利
- 验证项:
  - [ ] 复利特征判断准确（自我强化/边际成本递减/知识累积）
  - [ ] 有具体的临界点时间预测
  - [ ] 时间价值表包含多时间节点
  - [ ] 策略建议有可执行性

### 测试2: 投资复利分析
- 输入:
```
【复利效应分析请求】
情境描述：每年投资10万元，年化收益8%
初始投入：10万
预期行动：每年再投入10万
时间维度：30年
```
- 期望输出:
  - 复利公式正确应用
  - 时间价值表展示复利增长
  - 临界点分析
  - 策略建议涵盖资本复利
- 验证项:
  - [ ] 复利计算正确
  - [ ] 区分了线性增长和指数增长阶段
  - [ ] 有风险提示（负复利陷阱）
  - [ ] 中断代价被量化

### 测试3: 职业发展复利
- 输入:
```
【复利效应分析请求】
情境描述：在专业领域持续深耕
初始投入：3年专业经验
预期行动：每年技能提升20%
时间维度：10年
```
- 期望输出:
  - 知识复利识别
  - 职业发展曲线分析
  - 临界点预测
  - 策略建议涵盖技能复利
- 验证项:
  - [ ] 自我强化特征被识别
  - [ ] 边际收益递增被分析
  - [ ] 有清晰的阶段划分
  - [ ] 风险提示包含技能贬值

## 负面测试

### 测试1: 虚假复利识别
- 输入:
```
【复利效应分析请求】
情境描述：销售团队扩张，每个月多招1个人
初始投入：10人团队
预期行动：每月增加1人
时间维度：12个月
```
- 期望行为: 识别这不是真正的复利（人员增加是线性叠加而非自我强化）
- 验证项:
  - [ ] 正确识别虚假复利
  - [ ] 明确说明为什么不是复利
  - [ ] 给出真正的复利条件

### 测试2: 负复利陷阱
- 输入:
```
【复利效应分析请求】
情境描述：每天加班2小时，短期业绩提升
初始投入：身体健康
预期行动：持续加班6个月
时间维度：6个月后
```
- 期望行为: 识别负复利效应（健康损耗的指数级累积）
- 验证项:
  - [ ] 负复利被明确指出
  - [ ] 长期危害被量化
  - [ ] 有止损建议

### 测试3: 中断代价分析
- 输入:
```
【复利效应分析请求】
情境描述：已经坚持学习90天，偶尔中断几天
初始投入：90天学习基础
预期行动：继续学习，但可能偶有中断
时间维度：1年
```
- 期望行为: 量化中断对复利效果的影响
- 验证项:
  - [ ] 中断代价被量化
  - [ ] 复利中断的指数级损失被说明
  - [ ] 有保持持续性的建议
