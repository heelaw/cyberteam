# 情绪ABC理论专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | ABC理论是否正确（Activating event, Belief, Consequence五阶段） |
| 可执行性 | 30% | 输出是否可操作（信念识别、反驳策略、认知重建是否明确） |
| 独特性 | 20% | 是否有差异化（与一般情绪调节的区分、ABC独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（触发事件A、情绪结果C、潜在信念）
- [ ] 有输出格式（信念识别、反驳D、认知重建E）
- [ ] 有分析框架（五阶段：A→B→C→D→E）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（非暴力沟通、认知重评）

## 核心检验点

### 理论准确性
- [ ] ABC三要素完整（A事件、B信念、C结果）
- [ ] 五阶段流程正确（A→B→C→D→E）
- [ ] D（Disputing）反驳机制有效

### 框架完整性
- [ ] 信念识别有具体问题清单
- [ ] 反驳策略有方法指导
- [ ] 认知重建有操作步骤

### 输出有效性
- [ ] 信念识别可落地
- [ ] 反驳有具体路径
- [ ] 新的情绪结果可评估

## 相关专家

- 情绪ABC与认知重评：认知层面改变情绪体验
- 情绪ABC与非暴力沟通：用ABC框架理解沟通中的情绪
- 情绪ABC与逻辑谬误：识别导致负面情绪的非理性信念
