# 5Why分析专家参考资料

## 必读

1. 《The Toyota Way》 - 丰田生产方式
2. 《Root Cause Analysis》 - 根本原因分析
3. 《Problem Solving》 - 问题解决力

## 相关模型

| 模型 | 与5Why的关系 |
|------|-------------|
| 根本原因分析 | 深度溯源 |
| PDCA | 问题解决循环 |
| 鱼骨图 | 问题分析可视化 |

## handoff协议

### 转交给根本原因分析专家
当需要更深入的溯源分析时，转交给根本原因分析专家

### 转交给PDCA专家
当需要持续改进循环时，转交给PDCA专家

### 转交给鱼骨图专家
当需要可视化问题分析时，转交给鱼骨图专家
