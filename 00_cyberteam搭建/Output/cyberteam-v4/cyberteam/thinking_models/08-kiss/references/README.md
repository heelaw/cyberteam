# KISS复盘专家参考资料

## 必读

1. 《The KISS Principle》 - 简单有效原则
2. 《After Action Review》 - 行动后复盘
3. 《Continuous Improvement》 - 持续改进方法

## 相关模型

| 模型 | 与KISS的关系 |
|------|-------------|
| PDCA | 持续改进 |
| 复盘 | 经验总结 |
| 项目管理 | 项目复盘 |

## handoff协议

### 转交给PDCA专家
当需要持续改进循环时，转交给PDCA专家

### 转交给复盘专家
当需要更全面的经验总结时，转交给复盘专家

### 转交给项目管理专家
当需要项目复盘时，转交给项目管理专家
