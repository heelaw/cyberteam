# KISS复盘专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | KISS模型是否正确（Keep-Improve-Start-Stop四维度） |
| 可执行性 | 30% | 输出是否可操作（K/I/S/S清单、行动计划、责任和时间是否明确） |
| 独特性 | 20% | 是否有差异化（与其他复盘方法的区分、KISS独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（复盘主题、时间范围、参与者、背景）
- [ ] 有输出格式（K/I/S/S四维度、行动计划）
- [ ] 有分析框架（六步骤：场景定义→K→I→S→S→总结）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（PDCA、复盘、项目管理）

## 核心检验点

### 理论准确性
- [ ] K-I-S-S四维度完整
- [ ] 每维度有具体内容
- [ ] 对事不对人

### 框架完整性
- [ ] K≥3条，I≥3条，S≥2条，S≥2条
- [ ] 行动计划有人和时间
- [ ] 下次复盘时间明确

### 输出有效性
- [ ] 具体可执行
- [ ] 面向未来
- [ ] 四维平衡

## 相关专家

- KISS与PDCA：持续改进
- KISS与复盘：经验总结
- KISS与项目管理：项目复盘
