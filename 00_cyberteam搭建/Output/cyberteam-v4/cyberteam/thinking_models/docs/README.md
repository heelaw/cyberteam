## 快速参考

### 风格速查（3条核心规则）

```
1. 禁止：═══、┌┬┐、###、长表格
2. 必须：emoji锚点、→ 逻辑流、核心判断第一句
3. 目标：读者"扫一眼就抓住重点"
```

### 门禁速查（3级Gate）

```
Gate 1 完整性：问题识别→框架正确→结论明确
Gate 2 风格：长度/格式/人设声音/逻辑流
Gate 3 价值：反直觉洞察+可执行建议+啊哈时刻
```

### 层级速查（4级）

```
L1 微洞察：50-100字，1个观点
L2 短分析：150-300字，判断+逻辑+行动
L3 标准输出：400-600字，完整分析+洞察
L4 完整报告：800-1200字，深度+风险预案
```

---

## 使用流程

```
1. 收到用户问题
2. 选择对应层级（L1-L4）
3. 按风格指南输出
4. 过Gate 1 → Gate 2 → Gate 3
5. 如有失败，按Gate失败处理返回修改
6. 通过后交付
```

## 责任矩阵

| 角色 | 职责 |
|------|------|
| **专家Agent** | 按风格指南输出，过Gate自检 |
| **COE/CEO** | 审核Gate通过状态，决定是否打回 |
| **系统** | 自动检查格式规范（可配置） |
