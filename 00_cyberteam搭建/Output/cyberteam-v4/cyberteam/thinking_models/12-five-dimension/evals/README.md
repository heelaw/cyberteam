# 五维思考专家测试用例

## 正面测试

### 测试1: 商业机会分析
- 输入:
```
【五维思考分析请求】
分析对象：某创业公司
核心问题：是否值得投资
商业目标：获得回报
```
- 期望输出:
  - 零维执行力分析
  - 一维战略分析
  - 二维商业模式分析
  - 三维颠覆性分析
  - 四维时间价值分析
  - 综合结论和建议
- 验证项:
  - [ ] 五维都有
  - [ ] 每维有判断
  - [ ] 有优先级

### 测试2: 个人发展分析
- 输入:
```
【五维思考分析请求】
分析对象：个人职业
核心问题：是否转型
商业目标：长期发展
```
- 期望输出:
  - 各维度分析
  - 判断结论
  - 建议
- 验证项:
  - [ ] 五维应用于个人
  - [ ] 每维有洞见
  - [ ] 具体可执行

## 负面测试

### 测试1: 跳维
- 输入:
```
【五维思考分析请求】
分析对象：某战略决策
只关注竞争优势
```
- 期望行为: 不能跳过任何一维
- 验证项:
  - [ ] 五维都分析
  - [ ] 不能只做某一维
  - [ ] 完整覆盖

### 测试2: 无判断
- 输入:
```
【五维思考分析请求】
分析对象：某决策
```
- 期望行为: 每个维度必须有明确判断结论
- 验证项:
  - [ ] 每维有💡判断
  - [ ] 不能只描述不判断
  - [ ] 最终有综合结论
