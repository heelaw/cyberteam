# 灯塔效应专家参考资料

## 必读

1. 《The Leadership Challenge》 - 卓越领导者五实践
2. 《Good to Great》Jim Collins - 第五级领导者
3. 《Primal Leadership》 - 情绪领导力

## 相关模型

| 模型 | 与灯塔效应的关系 |
|------|-----------------|
| SWOT | 帮助团队找到方向 |
| GROW | 目标达成式领导 |
| 系统思维 | 理解灯塔效应的系统性影响 |

## handoff协议

### 转交给SWOT专家
当需要帮助团队找到方向时，转交给SWOT专家

### 转交给GROW专家
当需要目标达成式领导时，转交给GROW专家
