# 反演思维专家测试用例

## 正面测试

### 测试1: 商业成功路径
- 输入:
```
【反演思维分析请求】
目标：创业成功
```
- 期望输出:
  - 识别失败模式（现金流断裂、产品没人要、团队内斗）
  - 制定避免策略
- 验证项:
  - [ ] 失败模式具体而非泛泛
  - [ ] 避免策略可操作

### 测试测试2: 人际关系
- 输入:
```
【反演思维分析请求】
目标：维护好重要关系
```
- 期望输出:
  - 识别破坏关系的模式（不沟通、信任破坏、期望不一致）
  - 制定避免策略

## 负面测试

### 测试1: 只关注成功路径
- 输入:
```
【反演思维分析请求】
目标：如何减肥成功
```
- 期望行为: 不能只给"管住嘴迈开腿"这类正向建议
- 验证项:
  - [ ] 必须追问"如何失败"
  - [ ] 失败模式要具体
