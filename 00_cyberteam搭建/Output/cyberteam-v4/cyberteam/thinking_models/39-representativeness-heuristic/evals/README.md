# 代表性启发专家测试用例

## 正面测试

### 测试1: 职业判断
- 输入:
```
【代表性启发分析请求】
观察对象：一个人穿着格子衫，戴眼镜，说话逻辑性强
你的判断：这个人是程序员
判断依据：他看起来像典型的程序员
基础数据：某高校计算机系学生只占全校5%
```
- 期望输出:
  - 相似性分析
  - 基础比率忽视诊断
  - 贝叶斯概率校正
- 验证项:
  - [ ] 能识别"看起来像"不构成充分证据
  - [ ] 有P(程序员|像程序员)的贝叶斯计算
  - [ ] 能区分诊断性和非诊断性特征

### 测试2: 合取谬误识别
- 输入:
```
【代表性启发分析请求】
观察对象：一个MBA毕业生，曾在投行工作，喜欢分析财报
你的判断：这个人在高盛工作的概率很高，而且是VP级别
判断依据：背景太符合了
基础数据：每年MBA毕业生中有1%能进高盛，VP需要10年经验
```
- 期望输出:
  - 合取谬误识别
  - P(A且B) <= P(A)检验
  - 概率校正
- 验证项:
  - [ ] 能识别"而且是VP"违反合取规则
  - [ ] 有正确的概率上限分析
  - [ ] 有分层估算

## 负面测试

### 测试1: 相似即判断
- 输入:
```
【代表性启发分析请求】
观察对象：候选人面试时表现得很自信，说话有条理
你的判断：他能胜任高管职位
判断依据：他的气场和专业度像高管
基础数据：高管职位任职成功率约20%
```
- 期望行为: 不能只凭相似性下结论
- 验证项:
  - [ ] 必须有诊断性评估
  - [ ] 必须考虑基础比率
  - [ ] 必须有独立证据检验

### 测试2: 赌徒谬误
- 输入:
```
【代表性启发分析请求】
观察对象：彩票分析
你的判断：连续5次没中了，下次肯定中
判断依据：概率应该回归了
基础数据：每次中奖概率独立，都是1/1000
```
- 期望行为: 不能认为概率会"纠正"
- 验证项:
  - [ ] 必须强调独立性
  - [ ] 必须识别赌徒谬误
  - [ ] 必须说明每次概率不变
