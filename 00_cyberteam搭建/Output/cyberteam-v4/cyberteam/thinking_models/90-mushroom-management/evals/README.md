# 蘑菇管理专家测试用例

## 正面测试

### 测试1: 新人迷茫期
- 输入:
```
【蘑菇管理分析请求】
当前处境：入职3个月，感觉被忽视
工作类型：技术岗位
领导风格：任务导向型
```
- 期望输出:
  - 阶段识别（蘑菇期哪一阶段）
  - 策略建议（如何破局）
  - 行动清单（本周可做的事）
- 验证项:
  - [ ] 阶段判断准确
  - [ ] 策略针对性强
  - [ ] 行动清单具体可执行

### 测试2: 职业停滞期
- 输入:
```
【蘑菇管理分析请求】
当前处境：工作2年，感觉没有成长机会
工作类型：销售岗位
领导风格：放权型
```
- 期望输出:
  - 阶段识别（是否处于蘑菇期）
  - 策略建议（主动破局还是等待时机）
  - 行动清单（具体提升计划）
- 验证项:
  - [ ] 正确判断是否真的在蘑菇期
  - [ ] 策略因人而异
  - [ ] 有长期和短期规划

## 负面测试

### 测试1: 阶段误判
- 输入:
```
【蘑菇管理分析请求】
当前处境：工作5年，已经是部门主管
工作类型：管理岗位
领导风格：教练型
```
- 期望行为: 不能将非蘑菇期情况误判为蘑菇期
- 验证项:
  - [ ] 阶段判断与实际情况匹配
  - [ ] 不强行套用蘑菇框架
  - [ ] 策略与职级匹配

### 测试2: 被动等待
- 输入:
```
【蘑菇管理分析请求】
当前处境：入职1个月，感觉没学到东西
工作类型：市场岗位
领导风格：不明确
```
- 期望行为: 不能给被动等待的建议，要给出主动行动方案
- 验证项:
  - [ ] 策略强调主动出击
  - [ ] 不给"等待时机"的消极建议
  - [ ] 行动清单由用户主导
