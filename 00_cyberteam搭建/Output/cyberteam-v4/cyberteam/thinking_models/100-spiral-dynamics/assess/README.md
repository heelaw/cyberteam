# 螺旋动力学专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 螺旋动力学是否正确（8个阶段、过渡模式、演化规律） |
| 可执行性 | 30% | 输出是否可操作（阶段诊断、过渡分析、变革引导是否明确） |
| 独特性 | 20% | 是否有差异化（与传统阶段理论的区分、螺旋独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（分析对象、当前阶段、现象、分析目的）
- [ ] 有输出格式（阶段诊断、过渡分析、变革引导、实践建议）
- [ ] 有分析框架（四步：阶段识别→主导诊断→过渡分析→变革引导）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（系统思维、成长思维、变革管理）

## 核心检验点

### 理论准确性
- [ ] 8个阶段正确（Beige-Purple-Red-Blue-Orange-Green-Yellow-Turquoise）
- [ ] 过渡模式正确
- [ ] 驱动力和核心价值正确

### 框架完整性
- [ ] 阶段识别有特征检测
- [ ] 主导诊断有多个层面
- [ ] 变革引导有路径

### 输出有效性
- [ ] 阶段诊断具体
- [ ] 过渡挑战识别
- [ ] 实践建议可执行
