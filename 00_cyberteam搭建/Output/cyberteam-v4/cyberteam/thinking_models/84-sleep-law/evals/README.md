# 睡眠定律专家测试用例

## 正面测试

### 测试1: 重大商业决策
- 输入:
```
【睡眠定律分析请求】
决策问题：是否收购某公司
决策时间：明天上午10点
决策重要性：关键
当前状态：轻度疲劳（昨晚只睡了5小时）
```
- 期望输出:
  - 紧迫性评估
  - 睡眠准备建议
  - 决策时机推荐
- 验证项:
  - [ ] 有时间vs质量权衡分析
  - [ ] 有睡眠准备步骤
  - [ ] 有避开睡眠惯性的建议

### 测试2: 疲劳状态判断
- 输入:
```
【睡眠定律分析请求】
决策问题：是否接受新工作机会
决策时间：需要现在做决定
决策重要性：重要
当前状态：极度疲劳（连续加班1周）
```
- 期望输出:
  - 疲劳决策警告
  - 保护措施
  - 推迟决策建议
- 验证项:
  - [ ] 有疲劳决策严重警告
  - [ ] 有强制休息建议
  - [ ] 有保护措施

## 负面测试

### 测试1: 疲劳时坚持决策
- 输入:
```
【睡眠定律分析请求】
决策问题：重要合同条款谈判
决策时间：必须现在决定
决策重要性：关键
当前状态：已经连续工作16小时
```
- 期望行为: 绝对不能在疲劳时做关键决策
- 验证项:
  - [ ] 必须强制建议休息
  - [ ] 不能同意"疲劳时坚持决策"
  - [ ] 必须说明疲劳决策的风险

### 测试2: 忽视睡眠惯性
- 输入:
```
【睡眠定律分析请求】
决策问题：醒来后立刻做投资决定
决策时间：刚刚醒来
决策重要性：重要
当前状态：刚睡醒（睡眠惯性高峰期）
```
- 期望行为: 刚睡醒不能立即做重要决策
- 验证项:
  - [ ] 必须说明睡眠惯性
  - [ ] 必须建议等待1小时
  - [ ] 必须有最佳决策时机建议
