# 二阶思维专家参考资料

## 必读

1. 《The Art of Thinking Clearly》Rolf Dobelli - 二阶思维的经典阐述
2. 《Thinking in Systems》Donella Meadows - 系统思考与反馈循环
3. 《Superforecasting》Philip Tetlock - 长期预测中的多层次推理

## 案例库

### 成功案例
- **查理·芒格的"避免嫉妒"**: 识别了嫉妒作为动机对他人的二阶效应
- **亚马逊的价格策略**: 考虑了一阶（降价吸引）二阶（竞争对手反应、市场格局）的多层效应
- **巴菲特的投资决策**: 总是考虑"然后呢？"至少3次

### 失败案例
- **罗斯福新政的部分措施**: 短期帮助但长期产生了依赖效应
- **禁酒令**: 一阶减少饮酒，二阶导致有组织犯罪增长
- **次贷危机的蔓延**: 华尔街的短期利润忽视了对系统的长期破坏

## 相关模型

| 模型 | 与二阶思维的关系 |
|------|----------------|
| 逆向思维 | 二阶思维的补充，先确定目标再反推 |
| 系统思维 | 二阶思维的系统层面，关注反馈循环 |
| 临界点思维 | 二阶思维在临界效应中的应用 |

## 延伸阅读

- 二阶思维与长期主义
- 投资中的多层次推理
- 政策制定的长期效应评估

## handoff协议

### 转交给逆向思维专家
当用户需要"从目标倒推路径"时，转交给逆向思维专家

### 转交给系统思维专家
当问题涉及复杂的反馈回路时，转交给系统思维专家进行深度分析

### 转交给机会成本专家
当决策涉及多个选项的权衡时，转交给机会成本专家进行量化分析
