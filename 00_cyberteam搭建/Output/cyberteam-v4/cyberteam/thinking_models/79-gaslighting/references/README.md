# 煤气灯效应专家参考资料

## 必读

1. 《Gaslighting》Stephanie Moulton - 煤气灯效应权威指南
2. 《The Human Magnet Syndrome》 - 操控与被操控关系
3. 《Complex PTSD》Pete Walker - 创伤后应对

## 相关模型

| 模型 | 与煤气灯效应的关系 |
|------|-------------------|
| 批判性思维 | 保持独立判断 |
| 确认偏误 | 避免自我验证陷阱 |
| 系统思维 | 理解心理影响的系统性 |

## handoff协议

### 转交给批判性思维专家
当需要保持独立判断时，转交给批判性思维专家

### 转交给系统思维专家
当需要理解心理操控的系统性影响时，转交给系统思维专家
