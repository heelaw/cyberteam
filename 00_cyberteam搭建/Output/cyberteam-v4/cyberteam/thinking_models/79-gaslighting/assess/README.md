# 煤气灯效应专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 煤气灯效应理论是否正确（操控手段识别、影响程度评估、应对策略） |
| 可执行性 | 30% | 输出是否可操作（行为模式识别、影响评估、短期保护、长期策略是否明确） |
| 独特性 | 20% | 是否有差异化（与其他心理操控工具的区分、自我保护机制） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（情况描述、具体事件、你的感受、对方反应）
- [ ] 有输出格式（行为模式识别、影响程度评估、应对策略、建议行动）
- [ ] 有分析框架（三步法：行为模式识别→影响程度评估→应对策略制定）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（批判性思维、确认偏误、系统思维）

## 核心检验点

### 理论准确性
- [ ] 煤气灯效应定义正确
- [ ] 操控手段类型识别完整
- [ ] 影响等级评估标准正确

### 框架完整性
- [ ] 行为模式识别清单完整
- [ ] 自我检查问卷有效
- [ ] 短期保护和长期策略明确

### 输出有效性
- [ ] 应对策略可操作
- [ ] 影响等级有量化
- [ ] 建议行动具体

## 相关专家

- 煤气灯效应与批判性思维：保持独立判断抵御操控
- 煤气灯效应与确认偏误：避免自我验证陷阱
