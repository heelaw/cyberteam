# 长尾理论专家测试用例

## 正面测试

### 测试1: 利基市场分析
- 输入:
```
【长尾理论分析请求】

分析对象：在线教育平台
市场结构：
- 头部产品：K12应试课程、语言培训
- 现有长尾：兴趣爱好、技能提升、小众语言
```
- 期望输出:
  - 头部分析：头部集中度、进入壁垒
  - 长尾评估：利基产品数量、需求分散度
  - 聚合效应：品类聚合潜力
  - 策略建议：头尾比例、推荐策略
- 验证项:
  - [ ] 正确区分头部和长尾市场
  - [ ] 聚合效应分析具体
  - [ ] 成本结构评估完整

### 测试2: 电商平台SKU策略
- 输入:
```
【长尾理论分析请求】

分析对象：综合电商平台
市场结构：
- 头部产品：热销爆品
- 现有长尾：长尾SKU占比80%，但销量仅占30%
```
- 期望输出:
  - 评估长尾SKU的经济性
  - 给出是否扩展长尾的建议
  - 提出聚合长尾需求的方案
- 验证项:
  - [ ] 边际成本分析准确
  - [ ] 推荐算法价值被强调
  - [ ] 区分了盲目扩展SKU vs 精准长尾

### 测试3: 内容平台UGC策略
- 输入:
```
【长尾理论分析请求】

分析对象：短视频内容平台
市场结构：
- 头部内容：明星/头部创作者
- 现有长尾：普通用户UGC
```
- 期望输出:
  - 分析UGC长尾的聚合效应
  - 评估用户聚合vs内容聚合
  - 给出平台策略建议
- 验证项:
  - [ ] 识别内容平台的聚合特性
  - [ ] 讨论头部引流+长尾变现模式
  - [ ] 给出具体的头尾比例建议

## 负面测试

### 测试1: 伪长尾识别
- 输入:
```
市场中有10000个小众产品，但每个产品只有1-2个用户，需求零散且不稳定
```
- 期望行为: 指出这不是有价值的长尾，是无效长尾
- 验证项:
  - [ ] 需求真实性和稳定性是长尾前提
  - [ ] 量化长尾需要最小可行市场规模

### 测试2: 成本失控长尾
- 输入:
```
实体商品长尾，每增加一个SKU需要增加仓储和物流成本
```
- 期望行为: 分析边际成本，说明长尾经济性不成立
- 验证项:
  - [ ] 数字产品 vs 实体产品的成本差异
  - [ ] 边际成本是长尾策略的关键

### 测试3: 忽视头部依赖
- 输入:
```
长尾平台只关注利基产品，忽视头部产品引流
```
- 期望行为: 指出长尾平台往往仍依赖头部引流
- 验证项:
  - [ ] 头尾结合是常见成功模式
  - [ ] 不能完全忽视头部产品
