# SWOT+TOWS专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | SWOT+TOWS是否正确（SWOT四要素、TOWS四组合） |
| 可执行性 | 30% | 输出是否可操作（SWOT条目、TOWS战略、行动计划是否明确） |
| 独特性 | 20% | 是否有差异化（与传统SWOT的区分、TOWS战略匹配独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（分析主体、核心目标、时间范围）
- [ ] 有输出格式（SWOT矩阵、TOWS战略、行动计划）
- [ ] 有分析框架（五阶段：目标界定→内部扫描→外部扫描→TOWS生成→总结行动）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有TOWS战略矩阵

## 核心检验点

### 理论准确性
- [ ] SWOT四要素正确
- [ ] TOWS四组合正确
- [ ] 战略匹配逻辑正确

### 框架完整性
- [ ] SWOT条目具体
- [ ] TOWS每个组合都有战略
- [ ] 优先级明确

### 输出有效性
- [ ] 战略建议可执行
- [ ] 行动计划有节点
- [ ] 不空洞具体化

## 相关专家

- SWOT+TOWS与战略规划：战略制定
- SWOT+TOWS与竞争分析：差异化战略
- SWOT+TOWS与职业生涯：个人战略
