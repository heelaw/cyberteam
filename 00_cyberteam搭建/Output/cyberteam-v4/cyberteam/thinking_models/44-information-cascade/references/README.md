# 信息级联专家参考资料

## 必读

1. 《Information Cascades》Sushil Bikhchandani - 信息级联原始论文
2. 《The Wisdom of Crowds》James Surowiecki - 群体智慧与群体愚昧
3. 《Investments and Markets》- 金融泡沫中的信息级联案例

## 相关模型

| 模型 | 与信息级联的关系 |
|------|----------------|
| 羊群效应 | 信息级联的一种形式 |
| 群体极化 | 信息级联导致极端观点 |
| 批判性思维 | 抵御信息级联的工具 |

## handoff协议

当需要更深层的独立思考训练时，转交给批判性思维专家
