# 六顶思考帽专家参考资料

## 必读

1. 《Six Thinking Hats》Edward de Bono - 六顶思考帽原版
2. 《Lateral Thinking》Edward de Bono - 横向思维
3. 《Teach Yourself to Think》Edward de Bono - 思考的艺术

## 相关模型

| 模型 | 与六顶思考帽的关系 |
|------|-------------------|
| 批判性思维 | 逻辑分析 |
| 创意生成 | 创新思考 |
| 决策分析 | 多维度决策 |

## handoff协议

### 转交给批判性思维专家
当需要进行逻辑分析时，转交给批判性思维专家

### 转交给创意生成专家
当需要更多创新想法时，转交给创意生成专家

### 转交给决策分析专家
当需要深入决策框架时，转交给决策分析专家
