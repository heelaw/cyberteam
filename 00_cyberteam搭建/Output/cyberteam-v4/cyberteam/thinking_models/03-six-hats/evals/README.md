# 六顶思考帽专家测试用例

## 正面测试

### 测试1: 商业决策分析
- 输入:
```
【六顶思考帽分析请求】
问题：是否进入新市场
背景：公司有一定资源
期望：做出全面决策
```
- 期望输出:
  - 💙蓝色帽（问题澄清）
  - 🤍白色帽（信息事实）
  - ❤️红色帽（直觉反应）
  - 🖤黑色帽（风险分析）
  - 💛黄色帽（价值发现）
  - 💚绿色帽（创意生成）
  - 💙蓝色帽（综合结论）
- 验证项:
  - [ ] 七步骤完整
  - [ ] 每帽有实质内容
  - [ ] 蓝色收尾

### 测试2: 团队问题解决
- 输入:
```
【六顶思考帽分析请求】
问题：团队沟通不畅
背景：跨部门项目
期望：找到解决方案
```
- 期望输出:
  - 各帽完整分析
  - 综合结论
  - 行动计划
- 验证项:
  - [ ] 不跳帽
  - [ ] 不混色
  - [ ] 有具体行动

## 负面测试

### 测试1: 跳帽
- 输入:
```
【六顶思考帽分析请求】
问题：快速决策
期望：简化流程
```
- 期望行为: 不能跳过任何一顶帽子
- 验证项:
  - [ ] 六顶帽子都经过
  - [ ] 每帽都有内容
  - [ ] 不简化流程

### 测试2: 混色
- 输入:
```
【六顶思考帽分析请求】
问题：市场分析
```
- 期望行为: 不能在一个帽子中混入其他帽子的视角
- 验证项:
  - [ ] 白帽只说事实
  - [ ] 红帽只说直觉
  - [ ] 黑帽只说风险
