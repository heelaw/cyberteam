# 管理者跃升专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 跃升理论是否正确（做事→协调、专才→通才、战术→战略、内在→外在） |
| 可执行性 | 30% | 输出是否可操作（跃升诊断、障碍识别、行动计划是否明确） |
| 独特性 | 20% | 是否有差异化（与一般管理建议的区分、跃升独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（当前角色、目标角色、跃升挑战、时间节点）
- [ ] 有输出格式（跃升诊断图谱、障碍分析、行动计划、立即行动）
- [ ] 有分析框架（三步：诊断→障碍识别→跃升路径）
- [ ] 有Critical Rules（必须遵守4条、禁止行为5条）
- [ ] 有四维跃升模型

## 核心检验点

### 理论准确性
- [ ] 四大跃升正确
- [ ] 障碍类型完整
- [ ] 行动策略具体

### 框架完整性
- [ ] 四维自评完整
- [ ] 核心障碍识别
- [ ] 立即行动有

### 输出有效性
- [ ] 目标行为具体
- [ ] 验证标准明确
- [ ] 30天检查点
