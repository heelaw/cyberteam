# 基础比率谬误专家测试用例

## 正面测试

### 测试1: 医学诊断
- 输入:
```
【基础比率分析请求】
目标事件：某种罕见病
具体描述：患者有头痛和视力模糊症状
相关数据：这个症状组合在该罕见病中出现率约70%
忽略项：这种罕见病在人群中的基础比率只有0.01%
```
- 期望输出:
  - 先验概率明确
  - 证据诊断性评估
  - 贝叶斯校正后概率
- 验证项:
  - [ ] 能强调0.01%的基础比率
  - [ ] 有症状诊断性分析
  - [ ] 后验概率计算正确

### 测试2: 创业成功判断
- 输入:
```
【基础比率分析请求】
目标事件：创业成功
具体描述：创始人斯坦福MBA，有10年经验
相关数据：创始人背景看起来很优秀
忽略项：其实95%的创业公司活不过3年
```
- 期望输出:
  - 基础比率查询
  - 证据校正
  - 决策建议
- 验证项:
  - [ ] 不能忽视95%失败的基础比率
  - [ ] 有"你的情况有多独特"的评估
  - [ ] 建议贴近基础比率估计

## 负面测试

### 测试1: 跳过硬基础
- 输入:
```
【基础比率分析请求】
目标事件：这个人是好人
具体描述：他对人很友善
相关数据：他的朋友都说他好
忽略项：（没有考虑基础比率）
```
- 期望行为: 不能跳过基础比率步骤
- 验证项:
  - [ ] 必须先确定"好人"在人群中的基础比率
  - [ ] 必须有先验概率设定步骤

### 测试2: 过度自信
- 输入:
```
【基础比率分析请求】
目标事件：我一定能成功
具体描述：因为我比别人都努力
相关数据：我的技能很强
忽略项：其实竞争对手也很强
```
- 期望行为: 不能忽视"我的情况一定不同"的过度自信
- 验证项:
  - [ ] 必须先接受基础比率
  - [ ] 必须有证据证明"确实不同"
  - [ ] 不能仅凭努力和技能就忽视基础概率
