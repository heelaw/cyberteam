# 基础比率谬误专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 基础比率理论是否正确（先验概率、证据诊断性、贝叶斯校正） |
| 可执行性 | 30% | 输出是否可操作（基础比率查询、证据评估、贝叶斯计算是否明确） |
| 独特性 | 20% | 是否有差异化（与其他概率工具的区分、贝叶斯方法论） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（目标事件、具体描述、相关数据、忽略项）
- [ ] 有输出格式（先验概率、证据评估、贝叶斯校正、决策建议）
- [ ] 有分析框架（四步法：基础比率查询→证据评估→贝叶斯校正→决策建议）
- [ ] 有Critical Rules（必须遵守5条、禁止行为5条）
- [ ] 有协作配置（相关专家）

## 核心检验点

### 理论准确性
- [ ] 先验概率概念描述正确
- [ ] 证据诊断性评估方法正确
- [ ] 贝叶斯公式应用正确

### 框架完整性
- [ ] 基础比率查询有数据来源指导
- [ ] 证据整合有量化方法
- [ ] 有常见领域应用示例

### 输出有效性
- [ ] 后验概率计算正确
- [ ] 概率区间合理
- [ ] 决策建议可操作

- 基础比率与可得性启发：BR是AH忽视的关键信息
- 基础比率与代表性启发：BR是纠正RH的重要工具
- 基础比率与贝叶斯定理：贝叶斯是BR的应用框架
