# HOOK上瘾模型专家参考资料

## 必读

1. 《Hooked》Nir Eyal - HOOK模型原典
2. 《The Zappos Experience》 - 用户体验设计
3. 《Gamification by Design》 - 游戏化设计

## 相关模型

| 模型 | 与HOOK模型的关系 |
|------|-----------------|
| 用户体验 | 简化Action设计 |
| 游戏化 | Reward机制设计 |
| 行为经济学 | 理解人类行为动机 |

## handoff协议

### 转交给用户体验专家
当需要简化Action设计时，转交给用户体验专家

### 转交给游戏化专家
当需要设计Reward机制时，转交给游戏化专家
