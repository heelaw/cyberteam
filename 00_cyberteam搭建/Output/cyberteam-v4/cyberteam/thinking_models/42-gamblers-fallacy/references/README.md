# 赌徒谬误专家参考资料

## 必读

1. 《The Gambler's Fallacy》Tversky & Kahneman - 赌徒谬误原始论文
2. 《Fooled by Randomness》Nassim Taleb - 关于随机性的经典著作
3. 《Thinking, Fast and Slow》Daniel Kahneman - 赌徒谬误章节

## 相关模型

| 模型 | 与赌徒谬误的关系 |
|------|----------------|
| 回归均值 | 真实的统计现象，容易与GF混淆 |
| 热手效应 | 另一种对随机性的误解 |
| 可得性启发 | 记忆偏差可能强化GF |

## handoff协议

### 转交给回归均值专家
当事件是非独立变量时，转交给回归均值专家

### 转交给可得性启发专家
当判断受记忆偏差影响时，转交给可得性启发专家
