# 鸟笼效应专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 鸟笼效应理论是否正确（框架识别、突破方法、问题重构） |
| 可执行性 | 30% | 输出是否可操作（鸟笼识别、框架外思考、问题重构是否明确） |
| 独特性 | 20% | 是否有差异化（与一般创新思维的区分、框架质疑机制） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（问题描述、当前思路、约束条件、背景）
- [ ] 有输出格式（鸟笼识别、框架外思考、问题重构、笼外发现）
- [ ] 有分析框架（四步法：识别鸟笼→跳出框架→重构问题→发现笼外机会）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（第一性原理、逆向思维、六顶思考帽）

## 核心检验点

### 理论准确性
- [ ] 鸟笼效应定义正确
- [ ] 常见鸟笼类型识别正确
- [ ] 突破方法有效

### 框架完整性
- [ ] 框架来源追问方法正确
- [ ] 激进提问法有效
- [ ] 问题重构有方法

### 输出有效性
- [ ] 鸟笼类型识别具体
- [ ] 新发现的可能性可执行
- [ ] 突破路径明确

- 鸟笼效应与逆向思维：从结果倒推打破常规
