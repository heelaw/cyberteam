# 沉没成本决策专家参考资料

## 必读

1. 《Thinking, Fast and Slow》Daniel Kahneman - 损失厌恶与沉没成本效应
2. 《Contagious》Jonah Berger - 为什么人们不愿承认错误
3. 《止损》Toby Cordwell - 专业投资者的损失规避策略

## 案例库

### 成功案例
- **微软放弃Windows Phone**: 果断承认失败，及时止损转投移动战略
- **巴菲特"认错"**: 卖掉亏损股票而非等待回本，专注于优质资产

### 失败案例
- **泰坦尼克号救生艇不足**: 为面子不承认需要救援，沉没成本导致更大损失
- **企业维持亏损业务**: 通用电气多年维持亏损业务拖累整体业绩

## 相关模型

| 模型 | 与沉没成本的关系 |
|------|----------------|
| 机会成本 | 沉没成本决策的反面，止损后资源的机会成本 |
| 损失厌恶 | 沉没成本谬误的心理根源 |
| 损失加速 | 越晚止损，损失越大 |
| 决策树 | 可视化止损决策的工具 |

## 延伸阅读

- 沉没成本谬误的神经科学解释
- 为什么企业难以关闭亏损业务
- 个人投资中的止损心理
