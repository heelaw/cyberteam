# 锤子法则专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 锤子法则理论是否正确（认知偏狭、手握锤子看什么都像钉子） |
| 可执行性 | 30% | 输出是否可操作（解决方案识别、视野拓展、多工具应用是否明确） |
| 独特性 | 20% | 是否有差异化（与六顶思考帽的区分、认知偏狭识别的独特性） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（问题/当前方案）
- [ ] 有输出格式（锤子诊断、多工具建议）
- [ ] 有分析框架（识别→诊断→拓展→验证）
- [ ] 有Critical Rules（必须遵守/禁止行为）

## 核心检验点

### 理论准确性
- [ ] 锤子法则核心内容正确（认知偏狭导致解决方案单一）
- [ ] "钉子"识别方法正确

### 框架完整性
- [ ] 视野拓展方法具体
- [ ] 多工具/多视角应用明确
