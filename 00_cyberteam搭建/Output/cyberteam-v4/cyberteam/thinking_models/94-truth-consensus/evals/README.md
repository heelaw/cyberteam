# 正确与共识专家测试用例

## 正面测试

### 测试1: 投资决策分析
- 输入:
```
【正确与共识分析请求】
情境：是否投资某热门赛道
主流观点：所有人都看好，股价已经涨了3倍
你的判断：觉得估值太高
信息来源：行业分析报告、财务数据
```
- 期望输出:
  - 共识识别（共识内容、形成时间、强度）
  - 正确性评估（可验证性、逻辑一致性、预测效力）
  - 信号识别（裂缝信号、早期异见者）
  - 决策建议（仓位建议、监控指标）
- 验证项:
  - [ ] 区分共识与正确
  - [ ] 裂缝信号具体
  - [ ] 仓位建议合理

### 测试2: 职业方向选择
- 输入:
```
【正确与共识分析请求】
情境：是否转行到AI领域
主流观点：AI是未来，所有人都在学
你的判断：不确定自己是否适合
信息来源：行业趋势、个人能力评估
```
- 期望输出:
  - 共识分析（AI领域的共识强度）
  - 正确性检验（AI是否真的是未来）
  - 信号识别（是否有反向信号）
  - 决策建议（如何验证这个选择）
- 验证项:
  - [ ] 独立于从众压力
  - [ ] 有具体验证方法
  - [ ] 不为逆而逆

## 负面测试

### 测试1: 为逆而逆
- 输入:
```
【正确与共识分析请求】
情境：是否否定当前热门观点
主流观点：某技术方向被普遍看好
你的判断：想要唱反调
```
- 期望行为: 不能为逆而逆，要真正分析共识和正确性
- 验证项:
  - [ ] 不是简单的唱反调
  - [ ] 有具体的正确性分析
  - [ ] 逆行有充分理由

### 测试2: 忽视共识
- 输入:
```
【正确与共识分析请求】
情境：某投资机会
主流观点：市场普遍看好
你的判断：少数派观点
```
- 期望行为: 不能完全忽视共识，共识通常有一定道理
- 验证项:
  - [ ] 承认共识的合理性
  - [ ] 逆行仓位要控制
  - [ ] 有风险缓解措施
