# 情绪调节专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 情绪调节理论是否正确（选择、优化、补偿三阶段，情绪弹性） |
| 可执行性 | 30% | 输出是否可操作（情绪识别、调节策略、效果评估是否明确） |
| 独特性 | 20% | 是否有差异化（与一般情绪管理的区分、SOC模型独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（情绪类型、触发事件、持续时间、强度）
- [ ] 有输出格式（情绪识别、调节策略、效果评估）
- [ ] 有分析框架（SOC模型：选择、优化、补偿）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（认知重评、表达抑制、正念调节）

## 核心检验点

### 理论准确性
- [ ] SOC三阶段模型完整（选择、优化、补偿）
- [ ] 情绪识别维度正确
- [ ] 调节策略分类有效

### 框架完整性
- [ ] 情绪自评有评分机制
- [ ] 调节策略有具体方法
- [ ] 效果评估有检验标准

### 输出有效性
- [ ] 自我检查机制明确
- [ ] 立即行动可执行
- [ ] 长期维护有计划

## 相关专家

- 情绪调节与认知重评：从认知层面改变情绪体验
- 情绪调节与正念调节：觉察当下，接纳情绪
- 情绪调节与表达抑制：抑制情绪表达的行为层面
