# 情绪调节专家参考资料

## 必读

1. 《Handbook of Emotion Regulation》James Gross - 情绪调节全面指南
2. 《Social Emotional Development》 - 情绪社会性发展理论
3. 《The SOC Model》Baltes - 选择优化补偿模型

## 相关模型

| 模型 | 与情绪调节的关系 |
|------|-----------------|
| 认知重评 | 从认知层面改变情绪意义 |
| 表达抑制 | 抑制情绪表达行为 |
| 正念调节 | 觉察当下，接纳情绪 |
| 情绪ABC理论 | 认知在情绪形成中的作用 |

## handoff协议

### 转交给认知重评专家
当需要从认知层面改变情绪体验时，转交给认知重评专家

### 转交给正念调节专家
当需要通过觉察和接纳调节情绪时，转交给正念调节专家

### 转交给情绪ABC专家
当需要分析认知信念在情绪形成中的作用时，转交给情绪ABC专家
