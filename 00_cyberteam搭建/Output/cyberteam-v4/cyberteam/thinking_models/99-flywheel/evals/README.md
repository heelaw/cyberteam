# 飞轮效应专家测试用例

## 正面测试

### 测试1: 业务增长飞轮
- 输入:
```
【飞轮效应分析请求】
业务/个人：电商平台
当前状态：成长期
已有优势：用户口碑好
目标：持续快速增长
```
- 期望输出:
  - 飞轮识别（现有循环）
  - 飞轮构建（推荐模式）
  - 突破点分析
  - 加速策略
- 验证项:
  - [ ] 飞轮循环具体
  - [ ] 突破点条件明确
  - [ ] 加速策略分阶段

### 测试2: 个人成长飞轮
- 输入:
```
【飞轮效应分析请求】
业务/个人：个人职业发展
当前状态：瓶颈期
已有优势：学习能力强
目标：建立成长正循环
```
- 期望输出:
  - 个人成长飞轮设计
  - 突破点分析
  - 启动策略
- 验证项:
  - [ ] 飞轮模式适合个人
  - [ ] 有具体行动建议
  - [ ] 有监控指标

## 负面测试

### 测试1: 过早放弃
- 输入:
```
【飞轮效应分析请求】
业务/个人：新业务
当前状态：启动期，进展缓慢
已有优势：有核心产品
目标：快速增长
```
- 期望行为: 不能在启动期放弃，要耐心推动飞轮
- 验证项:
  - [ ] 强调耐心启动
  - [ ] 不是进展缓慢就换方向
  - [ ] 有启动期策略

### 测试2: 同时推多个飞轮
- 输入:
```
【飞轮效应分析请求】
业务/个人：多元化企业
当前状态：多个业务线
已有优势：资源充沛
目标：所有业务都增长
```
- 期望行为: 不能同时推动多个飞轮，要聚焦单一
- 验证项:
  - [ ] 警告分散精力
  - [ ] 建议聚焦一个飞轮
  - [ ] 有优先级建议
