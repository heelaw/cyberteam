# 沉没成本谬误专家测试用例

## 正面测试

### 测试1: 项目继续决策
- 输入:
```
【沉没成本分析请求】
情境：项目已投入100万，还需再投50万才能完成
已投入：100万
你的倾向：继续，因为已经投了这么多
继续理由：不能半途而废
```
- 期望输出:
  - 沉没成本识别
  - 空白起点测试
  - 边际分析
- 验证项:
  - [ ] 能识别100万已是沉没成本
  - [ ] 能做空白起点测试
  - [ ] 能基于边际收益做决策

### 测试2: 电影继续看
- 输入:
```
【沉没成本分析请求】
情境：电影很无聊但票已经买了
已投入：票价100元，已经看了30分钟
你的倾向：继续看完，因为不能浪费票
继续理由：已经买票了
```
- 期望输出:
  - 沉没成本诊断
  - 未来导向分析
  - 心理对策
- 验证项:
  - [ ] 能识别票价已是沉没成本
  - [ ] 能问"如果没买票还会看吗"
  - [ ] 能建议立刻止损离开

## 负面测试

### 测试1: 损失挽回心理
- 输入:
```
【沉没成本分析请求】
情境：股票已经亏了50%
已投入：10万，现在值5万
你的倾向：持有，等回本
继续理由：已经亏了，不能卖
```
- 期望行为: 不能把继续当作"挽回"损失
- 验证项:
  - [ ] 必须强调过去已是损失
  - [ ] 必须用边际分析做决策
  - [ ] 必须破除"回本"锚定

### 测试2: 关系坚持
- 输入:
```
【沉没成本分析请求】
情境：感情已经破裂但不想分手
已投入：3年感情
你的倾向：继续，因为已经这么久
继续理由：不能浪费这3年
```
- 期望行为: 不能用"已经投入"作为继续理由
- 验证项:
  - [ ] 必须做空白起点测试
  - [ ] 必须问"如果从头来还会选择吗"
  - [ ] 必须基于未来收益做决策
