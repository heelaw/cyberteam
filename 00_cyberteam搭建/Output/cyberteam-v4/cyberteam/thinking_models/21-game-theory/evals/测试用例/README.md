# 博弈论分析专家测试用例

## 正面测试

### 测试1: 竞争对手降价策略分析

- **输入**:
```
【博弈论分析请求】

博弈情境：在线教育平台市场，我的平台市场份额35%，竞争对手45%
参与者：我的平台（A）、竞争对手（B）
历史行为：竞争对手多次在我推出新产品后降价应对
```
- **期望输出**:
  - 识别为零和博弈还是非零和博弈
  - 支付矩阵分析
  - 纳什均衡点识别
  - 策略建议：是否跟进降价、如何建立承诺
- **验证项**:
  - [ ] 博弈类型判断合理
  - [ ] 包含支付矩阵分析
  - [ ] 识别纳什均衡点
  - [ ] 分析对方最优反应
  - [ ] 有具体策略建议

---

### 测试2: 长期合作协议博弈

- **输入**:
```
【博弈论分析请求】

博弈情境：两家科技公司谈判长期技术授权合作
参与者：公司A（技术提供方）、公司B（技术使用方）
约束：双方都希望建立长期稳定关系，但担心对方机会主义行为
```
- **期望输出**:
  - 识别为合作博弈、重复博弈
  - 囚徒困境分析
  - 惩罚机制设计（触发策略/胡萝卜加大棒）
  - 声誉机制建议
- **验证项**:
  - [ ] 博弈类型判断为合作+重复博弈
  - [ ] 识别囚徒困境
  - [ ] 设计具体的惩罚机制
  - [ ] 有声誉管理建议
  - [ ] 行动路线图分短期/中期/长期

---

### 测试3: 招聘市场薪资博弈

- **输入**:
```
【博弈论分析请求】

博弈情境：高端人才招聘市场
参与者：应聘者、招聘方
应聘者策略：如实陈述能力 / 夸大能力
招聘方策略：信任应聘者陈述 / 严格背调
```
- **期望输出**:
  - 识别不完全信息博弈
  - 逆向选择分析
  - 信号传递机制设计
  - 筛选机制建议
- **验证项**:
  - [ ] 识别信息不对称问题
  - [ ] 分析逆向选择风险
  - [ ] 给出信号传递建议（如学历/证书作为信号）
  - [ ] 设计筛选机制

## 负面测试

### 测试1: 单向策略建议

- **输入**:
```
【博弈论分析请求】

博弈情境：与供应商的价格谈判
参与者：我方、供应商
目标：要求最低价
```
- **期望行为**: 输出应避免单纯的"压价"思维，而应分析对方的博弈策略和反应
- **验证项**:
  - [ ] 分析供应商的最优反应
  - [ ] 考虑供应商可能采取的报复措施
  - [ ] 寻找非零和解（创造价值而非仅分割利益）
  - [ ] 不建议会导致长期关系恶化的策略

---

### 测试2: 忽视承诺可信度

- **输入**:
```
【博弈论分析请求】

博弈情境：价格战
参与者：市场中的两个竞争者
场景：我方宣布"永不降价"以建立低价形象
```
- **期望行为**: 输出应分析这个承诺是否可置信，以及如何让承诺变得可置信
- **验证项**:
  - [ ] 指出"永不降价"承诺可能不可信
  - [ ] 分析什么因素能让承诺可置信（沉没成本、合同约束等）
  - [ ] 建议更可信的承诺方式

---

### 测试3: 零和思维泛化

- **输入**:
```
【博弈论分析请求】

博弈情境：公司与员工的关系
参与者：公司、员工
问题：如何分配利益才能最大化公司收益？
```
- **期望行为**: 输出应指出这不是零和博弈，员工和公司的利益有一定一致性
- **验证项**:
  - [ ] 指出这是非零和博弈
  - [ ] 分析合作剩余（双方合作产生的额外价值）
  - [ ] 建议激励机制设计而非单纯的利益分割

## 边界测试

### 测试1: 多重均衡识别

- **输入**:
```
【博弈论分析请求】

博弈情境：社交平台市场竞争
参与者：平台A、平台B
场景：双方都可以选择"开放生态"或"封闭生态"
```
- **期望行为**: 识别可能存在多个纳什均衡，帮助用户理解均衡选择的难点
- **验证项**:
  - [ ] 识别出多个可能的均衡点
  - [ ] 分析每个均衡的稳定性
  - [ ] 讨论如何从一个均衡转向另一个均衡
  - [ ] 建议协调机制

---

### 测试2: 动态博弈中的策略

- **输入**:
```
【博弈论分析请求】

博弈情境：先发制人的投资决策
参与者：企业A（先行）、企业B（后发）
场景：A面临是否首先进入新市场的决策
```
- **期望行为**: 分析先动优势与后动优势的权衡
- **验证项**:
  - [ ] 分析先动vs后动的博弈结构
  - [ ] 给出先发制人或等待的策略判断
  - [ ] 设计如何改变博弈结构（如建立承诺）

---

### 测试3: 有限理性假设

- **输入**:
```
【博弈论分析请求】

博弈情境：小商贩定价
参与者：小商贩A、小商贩B（同一市场）
场景：双方都面临"高价高利润低销量"或"低价低利润高销量"的选择
```
- **期望行为**: 认识到小商贩可能不是完全理性人，考虑有限理性下的策略
- **验证项**:
  - [ ] 指出假设完全理性的局限
  - [ ] 分析有限理性对博弈结果的影响
  - [ ] 建议更现实的策略（重复博弈中的学习机制）
