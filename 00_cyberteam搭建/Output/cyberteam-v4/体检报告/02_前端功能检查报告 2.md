## 🔴 新发现严重问题 (P0 - 必须立即修复)

### 问题 0: API 响应格式与前端期望严重不匹配

**位置**:
- 前端 `src/pages/Dashboard.tsx:19-27`
- 前端 `src/pages/Departments.tsx:33-43`
- 前端 `src/pages/Agents.tsx:44-60`
- 后端 `backend/app/api/departments.py:152-164`
- 后端 `backend/app/api/agents.py:214-256`

**问题描述**:

前后端 API 响应格式存在严重不匹配：

```typescript
// 前端 Dashboard.tsx 第19-27行
useEffect(() => {
  Promise.all([api('/departments'), api('/agents')])
    .then(([d, a]) => {
      setDepts(d);   // 期望 d 是 Dept[]
      setAgents(a);  // 期望 a 是 Agent[]
    })
    ...
}, []);

// 后端 departments.py 返回
return {"departments": departments, "total": len(departments)}
// 返回的是对象 {departments: [...], total: N}，不是数组！

// 后端 agents.py 返回
return {"agents": agents, "total": len(agents)}
// 返回的是对象 {agents: [...], total: N}，不是数组！
```

**影响**:
- Dashboard 页面加载时 `setDepts(d)` 会把整个对象赋值给数组状态
- Department 和 Agent 列表页面的 `map()` 调用会失败或显示错误数据
- 应用启动后页面显示空白或错误

**修复建议**:
```typescript
// 前端 api 调用后需要提取正确的字段
const [deptRes, agentRes] = await Promise.all([
  api('/departments'),
  api('/agents'),
]);
setDepts(deptRes.departments || []);
setAgents(agentRes.agents || []);
```

### 问题 0-b: Agent 数据模型字段名完全不匹配

**位置**:
- 前端 `src/pages/Agents.tsx:6-10`
- 后端 `backend/app/api/agents.py:202-209`

**问题描述**:

```typescript
// 前端期望的 Agent 接口
interface Agent {
  id: string;           // 后端没有
  name: string;
  role: string;         // 后端用 description

class AgentOut(BaseModel):

    type: str            # 前端没有
```

### 问题 2: 后端返回字段与前端期望不匹配 (Agent)

**位置**:
- 后端 `server/main.py:952` 返回 `level: "L3"` (字符串)
- 前端 `src/pages/Agents.tsx:8` 期望 `level: number`
- 前端 `src/pages/Dashboard.tsx:6` 期望 `level: number`

**问题描述**:
后端 API 返回的 Agent 数据中 `level` 字段是字符串格式（如 `"L3"`），但前端 TypeScript 接口定义为数字类型，且前端代码直接使用 `L{a.level}` 渲染时会显示 `"LL3"`（重复 L）。

**修复建议**:
前端应处理后端返回的字符串 level：
```typescript
// Agents.tsx
<span className="text-xs px-2 py-1 bg-slate-700 text-gray-300 rounded">
  {String(a.level).replace('L', '')}级
</span>
```

或在 API client 中统一转换：
```typescript
const transformAgent = (agent: any) => ({
  ...agent,
  level: typeof agent.level === 'string'
    ? parseInt(agent.level.replace('L', ''))
    : agent.level
});
```

**位置**:
- `src/stores/skillStore.ts` - finally 中设置

**位置**: 每个页面组件

每个页面都实现了类似的：
- `api()` 函数
- `showToast()` 函数

```typescript
// src/hooks/useApi.ts
export function useApi() {
  const [loading, setLoading] = useState(false);
  const [toast, setToast] = useState<{msg: string; type: 'success'|'error'} | null>(null);

  const showToast = (msg: string, type: 'success'|'error') => {
    setToast({msg, type});
    setTimeout(() => setToast(null), 3000);
  };

  const api = async (path: string, opts?: RequestInit) => {
    setLoading(true);
    try {
      return await fetch(`${API}${path}`, opts);
    } finally {
      setLoading(false);
    }
  };

  return { loading, toast, showToast, api };
}
```

## 补充检查：Dashboard 路由问题

### 问题 15: Dashboard 使用原生 `<a>` 标签而非 React Router `<Link>`

**位置**: `src/pages/Dashboard.tsx` 第 82-84, 125, 153, 167 行

**问题代码**:
```tsx
// 第 82-84 行
<a href="/departments" className="px-4 py-2 bg-blue-600 text-white rounded-lg hover:bg-blue-700 text-sm font-medium">部门管理</a>
<a href="/agents" className="px-4 py-2 bg-purple-600 text-white rounded-lg hover:bg-purple-700 text-sm font-medium">Agent 管理</a>
<a href="/skills" className="px-4 py-2 bg-green-600 text-white rounded-lg hover:bg-green-700 text-sm font-medium">技能配置</a>

// 第 125 行
<a href="/departments" className="text-blue-400">点击添加</a>

// 第 153 行
<a href="/agents" className="text-blue-400">点击添加</a>

// 第 167 行
<a key={i} href={item.href}
  className={`${item.color} rounded-lg p-4 hover:opacity-90 transition-opacity cursor-pointer`}>
```

**影响**:
- 点击链接会触发浏览器整页刷新（full page reload）
- 破坏 React 应用的用户体验（页面闪烁、状态丢失）
- 增加服务器负载
- 无法利用 React Router 的客户端路由功能

**修复建议**:
```tsx
import { Link } from 'react-router-dom';

// 替换所有 <a href= 为 <Link to=
<Link to="/departments" className="px-4 py-2 bg-blue-600 text-white rounded-lg hover:bg-blue-700 text-sm font-medium">部门管理</Link>
<Link to="/agents" className="px-4 py-2 bg-purple-600 text-white rounded-lg hover:bg-purple-700 text-sm font-medium">Agent 管理</Link>
<Link to="/skills" className="px-4 py-2 bg-green-600 text-white rounded-lg hover:bg-green-700 text-sm font-medium">技能配置</Link>

// 快捷操作卡片
{[
  { to: '/departments', label: '添加部门', icon: '🏢', desc: '创建新部门', color: 'bg-blue-600' },
  { to: '/agents', label: '添加 Agent', icon: '🤖', desc: '新增角色', color: 'bg-purple-600' },
  { to: '/skills', label: '配置技能', icon: '⚡', desc: '管理技能库', color: 'bg-green-600' },
  { to: '/teams', label: '组建团队', icon: '👥', desc: '创建团队', color: 'bg-orange-600' },
].map((item, i) => (
  <Link key={i} to={item.to}
    className={`${item.color} rounded-lg p-4 hover:opacity-90 transition-opacity cursor-pointer`}>
    <div className="text-2xl mb-1">{item.icon}</div>
    <div className="text-white font-medium">{item.label}</div>
    <div className="text-white/70 text-xs">{item.desc}</div>
  </Link>
))}
```
