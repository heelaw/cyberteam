## 详细修复内容

### 1. Dashboard.tsx

**问题**: API 响应格式不匹配 + Base URL 硬编码

**修复前**:
```typescript
const API = 'http://localhost:8000/api';
// ...
Promise.all([api('/departments'), api('/agents')])
  .then(([d, a]) => {
    setDepts(d);   // 错误：d 是 {departments: [...], total: N}
    setAgents(a);  // 错误：a 是 {agents: [...], total: N}
  })
```

**修复后**:
```typescript
const API = '/api';
// ...
Promise.all([api('/departments'), api('/agents')])
  .then(([d, a]) => {
    const deptList = d.departments || d || [];
    const agentList = (a.agents || a || []).map((ag: any) => ({
      ...ag,
      id: ag.agent_id || ag.id,
      departmentId: ag.department_id || ag.departmentId,
      level: typeof ag.level === 'string' ? parseInt(ag.level.replace('L', '')) || 3 : (ag.level || 3),
    }));
    setDepts(deptList);
    setAgents(agentList);
  })
```

### 2. Departments.tsx

**问题**: Base URL 硬编码 + `department_id` -> `id` 映射缺失

**修复前**:
```typescript
const API = 'http://localhost:8000/api';
// ...
const fetchDepts = async () => {
  const data = await api('/departments');
  setDepts(data);  // 错误：data.departments 未提取
};
```

**修复后**:
```typescript
const API = '/api';
// ...
const fetchDepts = async () => {
  const data = await api('/departments');
  const deptList = data.departments || data || [];
  const mappedDepts = deptList.map((d: any) => ({
    ...d,
    id: d.department_id || d.id,
    parentId: d.parentId || null,
    level: d.level || 1,
  }));
  setDepts(mappedDepts);
};
```

## 兼容性处理策略

1. **API 响应兼容**: 前端同时支持 `Array` 和 `{data: Array, total: N}` 两种格式
2. **字段映射**: `agent_id` -> `id`, `department_id` -> `departmentId`
3. **类型转换**: `level` 可能是字符串 "L3" 或数字，统一转换为数字
4. **相对路径**: 所有 API 调用使用 `/api` 相对路径，适配代理配置
