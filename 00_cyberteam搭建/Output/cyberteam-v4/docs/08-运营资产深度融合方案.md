# CyberTeam v4 运营资产深度融合方案

> **目标**：将 v3 的 60+ 运营 Agent 直接集成到 v4 公司架构中，并梳理完整的决策链、关系链和 Skill 调用体系

---

## 📋 融合原则

### 1. 直接集成 vs 引用关系

| 方式 | 说明 | 采用场景 |
|------|------|----------|
| **直接集成** | 把 v3 Agent 的能力完全融入 v4 Agent | 核心能力、高频使用 |
| **调用关系** | v4 Agent 可以调用 v3 Agent | 独立功能模块、专项任务 |
| **Skill 工具化** | v3 Agent 的 Skill 提取为独立工具 | 通用方法论、标准化流程 |

### 2. Skill 分类体系

```
┌─────────────────────────────────────────────────────────┐
│                    Skill 分类体系                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  通用型 Skill（全公司可用）                              │
│  ├── 决策类：汇报框架、时间管理、进度管理、复盘模板      │
│  ├── 分析类：数据分析、A/B测试、ROI计算                 │
│  └── 沟通类：1v1沟通、团队激励、高层意图识别            │
│                                                         │
│  专家型 Skill（特定专家专用）                            │
│  ├── 用户运营：RFM分析、生命周期运营、召回策略          │
│  ├── 内容运营：选题策划、内容生产、多平台分发           │
│  ├── 活动运营：活动创意、方案模板、活动复盘             │
│  ├── 增长营销：渠道ROI、病毒系数、归因分析              │
│  ├── 品牌营销：品牌定位、危机管理、健康度监测           │
│  └── 效果营销：账户结构、出价策略、转化优化             │
│                                                         │
│  协作型 Skill（跨专家协同）                              │
│  ├── 增长战役协同：战役策划、资源分配、效果复盘         │
│  ├── 数据共享：用户标签、渠道数据、内容数据             │
│  └── 决策流程：问题分拣→专家匹配→Skill调用             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```
