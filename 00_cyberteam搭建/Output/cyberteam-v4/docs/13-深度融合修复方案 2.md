### 2.2 执行层：Python Agent类设计

每个Python Agent类需要：
1. **继承BaseAgent**
2. **引用对应Skill**（通过SkillLoader）
3. **实现think()和execute()方法**
4. **注入灵魂**（8轮Q&A的业务逻辑）

### 2.3 知识层：Skill引用机制

```python
class GrowthBGAgent(BaseAgent):
    """增长BG总指挥"""

    def __init__(self):
        metadata = AgentMetadata(
            name="GrowthBGAgent",
            description="增长事业群总指挥",
            capabilities=[
                "战略规划",
                "团队管理",
                "跨部门协调"
            ],
            tags=["growth", "leader", "bg"]
        )
        super().__init__(metadata)

        # 引用知识层的Skill
        self.skills = {
            "汇报框架": SkillLoader.load("skills/v3-agents/ops/01_汇报框架Agent"),
            "时间管理": SkillLoader.load("skills/v3-agents/ops/02_时间管理Agent"),
            "进度管理": SkillLoader.load("skills/v3-agents/ops/03_进度管理Agent"),
            "复盘模板": SkillLoader.load("skills/v3-agents/ops/04_复盘模板Agent"),
            "团队管理": SkillLoader.load("skills/v3-agents/ops/团队管理Agent"),
            "策略规划": SkillLoader.load("skills/v3-agents/ops/策略规划Agent"),
            "策略执行": SkillLoader.load("skills/v3-agents/ops/策略执行Agent"),
        }

    async def think(self, input_data):
        # 注入灵魂：增长理念
        # 1. 先留存再拉新
        # 2. 数据+实验驱动
        # 3. 模型驱动
        # 4. 80/20聚焦
        pass

    async def execute(self, task):
        # 调用Skill执行
        pass
```

### 2.4 灵魂层：业务智慧注入
