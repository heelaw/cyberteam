## 核心目录详解

### 1. `cyberteam/` ⭐ 核心引擎包

**这是CyberTeam v4的核心代码库，包含所有运行时加载的Python模块。**

```
cyberteam/
├── __init__.py           # 包初始化，导出主要API
├── __main__.py          # 入口：python -m cyberteam
├── _version.py          # 版本信息
├── agents/              # Agent模块 ⭐
├── board/               # 仪表盘/看板
├── cli/                 # 命令行接口
├── config.py            # 全局配置
├── constants.py         # 常量定义
├── dependencies.py       # 依赖注入
├── identity.py          # 身份管理
├── mcp/                 # MCP协议支持
├── skills/              # Skills模块 ⭐
├── spawn/               # Agent生成器
├── team/                # 团队管理器
├── templates/           # Agent模板
├── thinking_models/     # 100个思维模型 ⭐
├── timefmt.py          # 时间格式化
└── workspace/          # 工作区管理
```
