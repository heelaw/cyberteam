# 电商评论情感分析功能 - 详细技术规格

## 1. 项目概述

**项目名称**: E-commerce Review Sentiment Analysis
**项目目标**: 对电商网站用户评论进行情感分析，自动识别正面、负面和中性评价
**核心价值**: 帮助商家快速了解用户反馈，优化产品和服务

---

## 2. 技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | FastAPI + Python 3.10+ |
| 情感分析模型 | transformers (bert-base-chinese-sentiment) |
| 数据库 | SQLite (开发) / PostgreSQL (生产) |
| 前端框架 | React + TypeScript + Vite |
| API 文档 | OpenAPI/Swagger (自动生成) |

---

## 3. 数据模型

### 3.1 Comment (评论)

```python
class Comment:
    id: str              # UUID
    product_id: str      # 商品ID
    user_id: str         # 用户ID
    content: str         # 评论内容
    rating: int          # 评分 1-5星
    sentiment: str       # positive/negative/neutral
    sentiment_score: float  # -1.0 到 1.0
    positive_keywords: List[str]   # 正面关键词
    negative_keywords: List[str]    # 负面关键词
    created_at: datetime # 创建时间
```

### 3.2 SentimentStats (统计)

```python
class SentimentStats:
    product_id: str      # 商品ID
    total_comments: int   # 总评论数
    positive_count: int   # 正面评论数
    negative_count: int   # 负面评论数
    neutral_count: int    # 中性评论数
    avg_sentiment_score: float  # 平均情感分数
    avg_rating: float     # 平均评分
    period: str           # 统计周期
```

## 4. API 端点

### 4.1 评论情感分析

| 方法 | 路径 | 描述 |
|------|------|------|
| POST | `/api/v1/comments/analyze` | 分析单条评论 |
| POST | `/api/v1/comments/batch-analyze` | 批量分析评论 |
| GET | `/api/v1/comments/{comment_id}` | 获取评论详情 |
| DELETE | `/api/v1/comments/{comment_id}` | 删除评论 |

### 4.2 统计分析

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | `/api/v1/comments/stats` | 获取整体统计 |
| GET | `/api/v1/comments/stats/{product_id}` | 获取指定商品统计 |

### 4.3 健康检查

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | `/health` | 服务健康检查 |
| GET | `/api/v1/model/status` | 情感分析模型状态 |

## 5. API 请求/响应格式

### 5.1 POST /api/v1/comments/analyze

**请求体**:
```json
{
  "product_id": "prod_12345",
  "user_id": "user_67890",
  "content": "这个产品质量很好，物流也很快，推荐购买！",
  "rating": 5
}
```

**响应体**:
```json
{
  "id": "uuid-xxx",
  "sentiment": "positive",
  "sentiment_score": 0.95,
  "positive_keywords": ["好", "快", "推荐"],
  "negative_keywords": [],
  "confidence": 0.92
}
```

### 5.2 POST /api/v1/comments/batch-analyze

**请求体**:
```json
{
  "comments": [
    {
      "product_id": "prod_12345",
      "user_id": "user_001",
      "content": "产品质量一般",
      "rating": 3
    },
    {
      "product_id": "prod_12345",
      "user_id": "user_002",
      "content": "非常差，等了一周才收到",
      "rating": 1
    }
  ]
}
```

**响应体**:
```json
{
  "results": [
    {
      "index": 0,
      "id": "uuid-xxx",
      "sentiment": "neutral",
      "sentiment_score": 0.1
    },
    {
      "index": 1,
      "id": "uuid-yyy",
      "sentiment": "negative",
      "sentiment_score": -0.85
    }
  ],
  "total": 2,
  "success_count": 2,
  "failed_count": 0
}
```

### 5.3 GET /api/v1/comments/stats

**响应体**:
```json
{
  "total_comments": 1500,
  "positive_count": 850,
  "negative_count": 200,
  "neutral_count": 450,
  "positive_ratio": 0.567,
  "negative_ratio": 0.133,
  "neutral_ratio": 0.300,
  "avg_sentiment_score": 0.42,
  "avg_rating": 3.85
}
```
