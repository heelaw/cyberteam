## Critical Rules（必须遵守）

1. **禁止盲目追热门渠道** - 热门渠道不一定适合你的产品，必须先评估匹配度
2. **禁止忽视渠道规则** - 投放前必须熟悉渠道政策，违规将导致账户被封
3. **禁止无转化场景投放** - 先搭建好转化承接页面，再开始投放
4. **禁止无数据监测投放** - 必须设置转化追踪，否则无法评估效果
5. **禁止只看短期数据** - 渠道效果需要持续观察，至少等待3-7天再判断
6. **禁止单一渠道依赖** - 建议至少2-3个渠道组合投放，降低风险
7. **禁止忽视竞争对手** - 投放前分析竞争对手的渠道策略

## 常见问题

### Q1: 如何判断渠道匹配度？

**A**: 从三个维度评估：
1. **用户重叠度** - 渠道用户与你目标用户的重合程度
2. **内容形式契合度** - 渠道的内容形式是否适合展示你的产品
3. **购买决策路径** - 用户在该渠道的购买决策路径是否顺畅

### Q2: 渠道规则在哪里查看？

**A**: 各渠道官方文档：
- 广点通：腾讯广告投放平台官方文档
- 巨量引擎：字节跳动营销服务中心
- 百度SEM：百度营销中心
- 小红书：蒲公英平台官方规则

### Q3: 转化场景搭建有什么常见错误？

**A**: 常见错误：
1. 落地页与投放素材信息不一致
2. CTA按钮不够醒目或位置不当
3. 转化路径过长（超过3步）
4. 未配置转化追踪事件
