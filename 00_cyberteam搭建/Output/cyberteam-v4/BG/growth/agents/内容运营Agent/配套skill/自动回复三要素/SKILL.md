# 自动回复三要素

## 触发关键词

**主关键词**: 自动回复、回复设置、回复语、关键词回复、消息回复

**次关键词**: 怎么回复、回复内容、欢迎语、关注回复、菜单回复

**场景触发**: 用户询问如何设置公众号自动回复，或需要优化现有回复内容时自动触发

## 描述PUSHY

自动回复不是随便写句"您好"！好的自动回复=即时价值传递+粉丝留存钩子+转化路径设计。90%的新手只做到了第一步。跟着走，让你的自动回复从"摆设"变成"印钞机"。

## 核心定义

通过「欢迎语设计 → 关键词回复配置 → 常见问题自动回复」三个核心要素，构建完整的自动回复体系，让粉丝在第一时间获得价值、留下来并产生下一步行动。

## workflow_steps

### Step 1: 欢迎语设计

**输入**:
- 账号定位（做什么的）
- 目标用户（给谁看的）
- 核心价值（能提供什么）
- 转化目标（希望用户做什么）

**处理**:
1. 设计欢迎语结构：
   - 第一句：打招呼+自我介绍+价值承诺
   - 第二句：告诉用户「我是谁」「你能得到什么」
   - 第三句：引导下一步行动（阅读/回复/菜单）
2. 控制在100字以内（语音条30秒内）
3. 设计「诱饵」：提供免费资源吸引回复
4. 避免常见错误：
   - 太官方像机器人
   - 太随意显得不专业
   - 只有感谢没有价值
   - 没有引导下一步

**输出**:
- 关注自动回复文案
- 关键词触发示例
- 引导话术设计

**成功标准**:
- 用户看完知道账号是做什么的
- 有明确的行动指引
- 有吸引用户继续互动的钩子

### Step 2: 关键词回复配置

**输入**:
- 用户常见问题清单
- 高价值内容清单
- 转化路径设计

**处理**:
1. 梳理用户可能输入的关键词：
   - 问题类：「怎么」「如何」「是什么」「为什么」
   - 需求类：「资料」「课程」「案例」「模板」
   - 数字类：「01」「1」「干货」
2. 配置关键词回复规则：
   - 精确匹配 vs 模糊匹配
   - 回复类型：文字/图片/音频/视频/小程序
   - 回复内容：直接答案 vs 引导跳转
3. 设置关键词跳转：
   - 关键词 → 自动回复 → 底部菜单
   - 形成「需求→满足→转化」闭环

**输出**:
- 关键词列表（10-20个）
- 关键词回复对应表
- 回复内容模板

**成功标准**:
- 覆盖80%以上用户常见需求
- 关键词回复有实际价值
- 有转化导向的关键词设计

### Step 3: 常见问题自动回复

**输入**:
- 历史用户问题数据
- 常见问题清单（FAQ）
- 人工客服工作时间

**处理**:
1. 梳理TOP10常见问题：
   - 入门问题：「怎么开始」「从哪里看」
   - 内容问题：「历史文章在哪」「怎么搜索」
   - 商务问题：「怎么合作」「多少钱」
   - 技术问题：「打不开」「收不到」
2. 设计问题识别机制：
   - 关键词匹配：「合作」「价格」→ 触发商务回复
   - 模糊匹配：「谢谢」「收到」→ 触发礼貌结束语
3. 配置智能分流：
   - 简单问题 → 自动回复解决
   - 复杂问题 → 引导联系人工
4. 设置服务时间提醒：
   - 工作时间：立即回复
   - 非工作时间：告知等待时间

**输出**:
- FAQ问题分类表
- 问题→回复映射表
- 人工客服入口提示

**成功标准**:
- 80%常见问题能自动解答
- 用户知道何时能找到人工客服
- 有明确的问题解决路径

## Critical Rules

1. **禁止无价值回复** - 欢迎语必须有实质内容，不能只是「感谢关注」
2. **禁止虚假承诺** - 不要承诺你做不到的事，如「24小时回复」
3. **禁止过度营销** - 关注即推销会引发取关，要先给价值再谈转化
4. **禁止无关键词** - 不能只有欢迎语没有关键词回复，那等于一次性互动
5. **禁止回复过长** - 单条回复不超过200字，用户没耐心看完
6. **禁止缺少人工入口** - 必须保留人工客服通道，不能让用户感到被「机器人」困住

## 质量评估标准

### 合格自动回复特征

- 欢迎语简洁有力，有价值承诺
- 关键词覆盖常见需求
- 回复内容真正能解决问题
- 有清晰的下一步行动指引
- 有人工客服入口

### 不合格自动回复特征

- 只有「感谢关注，请等待客服联系」
- 关键词回复答非所问
- 回复内容多年未更新
- 全部是营销推广
- 用户遇到问题找不到人工

## 注意事项

1. 自动回复需要定期更新，根据用户反馈优化
2. 关键词设置要定期检查，避免失效
3. 大促/节假日需要临时调整回复内容
4. 自动回复不是万能的，复杂问题必须引导人工
5. 数据很重要，定期看关键词回复的触发数据来优化
