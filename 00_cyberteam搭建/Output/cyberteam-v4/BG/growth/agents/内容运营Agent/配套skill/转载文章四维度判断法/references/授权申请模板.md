# 转载授权申请模板

## 模板一：微信公众号文章授权申请

### 申请方式
通过公众号后台「转载授权」功能申请，或私信作者

### 申请话术模板

```
您好，我是【账号名称】的运营者。

非常欣赏您的文章《文章标题》，文章中的【具体观点/数据】让我印象深刻。

我们公众号专注于【账号定位】，粉丝主要是【目标用户画像】，与您的读者群体高度吻合。

希望能够获得您的授权进行转载，转载后会：
1. 标注作者和原文来源
2. 在文首注明"本文已获授权转载"
3. 附上您的个人介绍和原文链接

期待您的回复，谢谢！
```

## 模板二：微博/小红书文章授权申请

### 申请话术模板

```
您好，我是【账号名称】的小编。

您的笔记《标题》内容非常实用，特别是【具体内容】，我们的粉丝也很需要这类信息。

我们是一个【账号定位】账号，目前拥有【粉丝量】粉丝。

希望获得您的转载授权，转载时会：
1. @您的账号
2. 标注原文来源
3. 附上您的个人主页链接

方便的话请回复"允许转载"，我截图保存，感谢！
```

## 模板三：网站/博客文章授权申请

### 申请话术模板

```
尊敬的老师/作者：

您好！

我是【公司/账号名称】的【职位】。

拜读了您的文章《文章标题》（链接：xxx），受益匪浅。贵文关于【核心观点】的阐述非常精辟，对我们的读者很有价值。

我们公众号主要面向【目标用户】，长期输出【内容方向】相关的干货内容。

希望能获得您的转载授权。我们承诺：
1. 保留原文完整内容和核心观点
2. 明确标注作者姓名和原文链接
3. 在文章开头注明转载来源
4. 如有需要，配合推广您的其他内容

请知悉是否允许转载，以及是否有其他特别要求。

期待您的回复，祝好！

【申请人】
【联系方式】
【日期】
```

## 授权确认回复模板

### 作者回复"允许转载"后

```
截图保存授权证明，包含：
1. 对话中明确表达"允许转载"的时间
2. 作者账号信息
3. 文章标题

保存为图片文件，命名格式：授权截图_文章标题_日期.png
```
