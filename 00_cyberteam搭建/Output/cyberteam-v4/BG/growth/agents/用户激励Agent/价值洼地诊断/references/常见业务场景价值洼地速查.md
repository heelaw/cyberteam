# 常见业务场景价值洼地速查

## 一、电商类

### 典型转化路径

```
访问 → 浏览 → 加购 → 结算 → 支付 → 完成
```

### 高价值洼地特征

| 洼地位置 | 识别特征 | 常见原因 | 推荐优化方向 |
|----------|----------|----------|--------------|
| 加购环节 | 加购率<30% | 商品展示/价格/评价 | 优化商品详情页 |
| 结算环节 | 结算率<60% | 优惠不明显/流程复杂 | 简化结算流程 |
| 支付环节 | 支付率<70% | 支付方式少/信任不足 | 优化支付体验 |

### 价值指数参考

| 优化点 | 典型提升 | 难度系数 | 价值指数 |
|--------|----------|----------|----------|
| 加购按钮优化 | 5-10% | 0.5 | 10-20 |
| 购物车提醒 | 3-5% | 0.5 | 6-10 |
| 支付流程简化 | 5-8% | 1.0 | 5-8 |
| 支付方式增加 | 3-5% | 1.5 | 2-3 |

## 七、决策速查表

### 问题导向速查

| 用户问题 | 最可能的洼地 | 优先验证指标 | 推荐优先行动 |
|----------|--------------|--------------|--------------|
| 转化率低 | 决策前环节 | 各节点转化率 | 检查流失最大的节点 |
| 新用户流失 | 激活环节 | 激活率 | 优化新手指引 |
| 老用户流失 | 留存环节 | 留存曲线 | 分析流失时间点 |
| 付费转化低 | 付费环节 | 付费率 | 优化付费流程 |

### 资源约束速查

| 资源约束 | 适合的洼地类型 | 不适合的洼地类型 |
|----------|----------------|------------------|
| 时间紧 | 运营优化、文案优化 | 技术重构、算法优化 |
| 人力少 | 激励设计、文案优化 | 需要大量测试的优化 |
| 技术弱 | 流程优化、激励优化 | 需要复杂技术的优化 |
| 预算少 | push优化、文案优化 | 需要付费推广的优化 |

## 八、洼地识别检查清单

### 自检问题

- [ ] 是否梳理了完整的用户路径？
- [ ] 是否识别了每个节点的转化率？
- [ ] 是否找到了流失率>20%的节点？
- [ ] 是否分析了流失原因？
- [ ] 是否评估了各节点的优化空间？
- [ ] 是否计算了价值指数？
- [ ] 是否在资源约束下做了选择？

### 常见遗漏

1. 遗漏了路径中的某个节点
2. 高估了某个节点的优化空间
3. 低估了实施难度
4. 忽视了资源约束
5. 没有考虑节点间的关联
