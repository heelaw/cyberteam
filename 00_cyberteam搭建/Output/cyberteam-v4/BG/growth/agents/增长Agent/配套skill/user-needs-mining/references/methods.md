# 用户需求挖掘6大渠道方法论

## 渠道详解

### 1. 社交媒体挖掘
**平台**: 微博、小红书、知乎、豆瓣、抖音
**方法**: 关键词搜索、热议话题分析
**适用**: 发现新趋势、用户痛点
**样本**: 大规模、代表性强

### 2. 竞品评论分析
**来源**: App Store、各大应用商店、论坛
**方法**: 评论抓取、情感分析、主题聚类
**适用**: 发现用户重视的功能和痛点
**优点**: 成本低、信息真实

### 3. 深度访谈
**形式**: 1对1、60-90分钟
**方法**: 追问、场景还原、任务观察
**适用**: 挖掘深层动机和需求
**关键**: 找到正确的用户

### 4. 问卷调查
**形式**: 在线问卷
**方法**: 定量分析、交叉分析
**适用**: 验证假设、量化需求
**要点**: 样本量、问卷设计

### 5. 行业报告
**来源**: 艾瑞、QuestMobile、易观、CNNIC
**方法**: 数据提取、趋势分析、对标
**适用**: 了解宏观趋势和市场规模
**注意**: 数据时效性

### 6. 搜索趋势
**工具**: 百度指数、Google Trends、微信指数
**方法**: 搜索量变化、趋势分析
**适用**: 发现新兴需求、季节性波动
**价值**: 预测未来需求

## 整合方法

1. **定性先行**: 先用社交媒体、访谈发现方向
2. **定量验证**: 用问卷、搜索数据验证规模
3. **竞品对标**: 参考竞品的用户反馈
4. **行业对照**: 用行业报告验证假设
