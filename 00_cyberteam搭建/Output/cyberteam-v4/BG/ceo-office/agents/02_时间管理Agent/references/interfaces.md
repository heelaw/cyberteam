# interfaces - 时间管理Agent

## 核心原则

**所有Agent决策前必须读取项目资料库。所有Agent产出必须存入项目目录。**

---

## 一、输入规范 (Input)

### 1.1 触发条件

| 场景 | 触发词 | 说明 |
|------|--------|------|
| 时间不够 | 时间不够用、事情太多 | 资源不足感 |
| 优先级困惑 | 不知道怎么安排、优先级 | 决策困难 |
| 效率低下 | 忙却没成果、拖延症 | 产出问题 |
| 计划制定 | 周计划、日程计划 | 规划需求 |

### 1.2 必读信息

| 信息类型 | 内容 | 说明 |
|----------|------|------|
| 任务清单 | 所有待办任务 | 原始材料 |
| 目标分解 | 上级目标/OKR | 方向指引 |
| 时间约束 | 截止日期/会议 | 硬性限制 |
| 精力时段 | 高效时段/低谷时段 | 适配安排 |

---

## 二、输出规范 (Output)

### 2.1 输出格式

```markdown
## 今日计划 - [日期]

### 今日3件要事
- [ ] 1. [任务] - 预计[时间]
- [ ] 2. [任务] - 预计[时间]
- [ ] 3. [任务] - 预计[时间]

### 艾森豪威尔分类
- Q1（立即做）：
- Q2（计划做）：
- Q3（委托做）：
- Q4（删除）：

### 时间块
| 时间段 | 任务 | 状态 |
|--------|------|------|
| 9:00-11:00 | | |
| 11:00-12:00 | | |
| 14:00-16:00 | | |
| 16:00-18:00 | | |
```

### 2.2 输出存放

```
{project}/decisions/          # 计划类产出
{project}/meetings/           # 复盘记录
```

## 三、协作接口

### 3.1 向上协作

| 接收方 | 内容 | 格式 |
|--------|------|------|
| 运营总监 | 计划同步/冲突协调 | 周计划/日计划 |
| 增长总监 | 目标对齐/资源请求 | OKR对齐表 |

### 3.2 横向协作

| 协作方 | 场景 | 产出 |
|--------|------|------|
| 产品部 | 任务依赖协调 | 任务协调记录 |
| 数据团队 | 数据获取时间 | 时间预估 |

---

## 四、核心工具

| 工具 | 适用场景 | 要点 |
|------|----------|------|
| 艾森豪威尔矩阵 | 优先级排序 | Q2优先 |
| 番茄工作法 | 深度工作 | 25+5 |
| 时间日志 | 问题诊断 | 连续3-5天 |
| 要事第一 | 每日规划 | 3件要事 |
