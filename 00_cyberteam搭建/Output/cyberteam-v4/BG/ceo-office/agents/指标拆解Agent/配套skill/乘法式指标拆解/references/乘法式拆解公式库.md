# 乘法式拆解公式库

## 公式结构

```
目标指标 = 因子1 × 因子2 × 因子3 × ... × 因子N
```

## 验证方法

| 验证步骤 | 操作 | 判定标准 |
|---------|------|----------|
| 1 | 因子相乘验证 | ∏因子 = 目标指标 |
| 2 | 因子数量检查 | 通常3-5个为宜 |
| 3 | 因子有效性验证 | 每个因子与目标有因果关系 |
| 4 | 提升空间评估 | 每个因子有明确的可提升空间 |

---

## 电商行业公式

### GMV拆解（标准版）

```
GMV = 访客数(UV) × 转化率 × 客单价
```

**案例**:
- 当前: 10万 × 5% × 200元 = 100万
- 目标: 12万 × 6% × 220元 = 158.4万

### GMV拆解（详细版）

```
GMV = 流量 × 曝光率 × 点击率 × 注册率 × 转化率 × 客单价
```

**适用场景**: 需要分析完整漏斗时使用

### 收入拆解

```
收入 = 付费用户数 × 付费率 × 平均客单价 × 平均付费周期
```

---

## 用户运营行业公式

### DAU拆解

```
DAU = 新增用户 + 活跃用户 × 留存率 + 召回用户 × 召回率
```

### 用户生命周期价值(LTV)拆解

```
LTV = 日活跃天数 × 日均ARPU × 付费转化率 × 平均订单金额
```

### 留存率拆解

```
N日留存率 = 功能完成率 × 内容消费深度 × 社交互动指数
```

---

## 内容平台行业公式

### 内容消费量拆解

```
总消费时长 = 用户数 × 人均消费时长
          = DAU × 人均VV × 单VV时长
```

### 创作者收益拆解

```
创作者收益 = 创作者数 × 活跃率 × 人均产出 × 单价
```

---

## B2B行业公式

### 营收拆解

```
营收 = 客户数 × 客单价 × 续费率 + 新增客户 × 客单价
```

### 签约率拆解

```
签约率 = 线索数 × 有效率 × 跟进率 × 成交率
```

### 年度营收目标拆解

```
目标营收 = 存量客户 × 续费率 × 续费客单价
         + 增量客户 × 新签客单价
```

---

## 线下门店行业公式

### 销售额拆解

```
销售额 = 客流量 × 入店率 × 转化率 × 客单价 × 复购次数
```

### 客流拆解

```
日客流 = 自然客流 + 引导客流 + 活动客流
```

---

## 增长黑客公式

### AARRR拆解

```
Acquisition获取 = 曝光量 × 转化率
Activation激活 = 访问用户 × 激活率
Retention留存 = 活跃用户 × 留存率
Referral传播 = 活跃用户 × 分享率 × 分享带来的新用户
Revenue变现 = 付费用户 × 付费率 × 客单价
```

### 增长飞轮

```
增长 = 存量用户 × 激活率 × 留存率 × 传播系数(K因子)
```

## 公式选择指南

| 问题类型 | 推荐公式 | 使用场景 |
|---------|---------|---------|
| GMV提升 | UV×转化率×客单价 | 电商基础分析 |
| 收入增长 | 流量×转化×客单×复购 | 全链路分析 |
| 用户增长 | 新增+留存+召回 | 用户运营 |
| 留存优化 | 功能×内容×社交 | 体验优化 |
| 线下提升 | 客流×入店×转化×客单 | 门店分析 |
| B2B营收 | 存量续费+新增 | 企业销售 |
