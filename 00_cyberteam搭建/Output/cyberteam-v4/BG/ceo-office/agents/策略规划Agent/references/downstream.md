# 策略规划Agent - 下游去向

## 下游Agent（输出目标）

### 直接下游

| Agent名称 | 接收内容 | 说明 |
|-----------|----------|------|
| 策略执行Agent | 策略框架、执行要求 | 策略制定完成后 |
| 模型迁移设计器 | 借鉴方案、迁移要素 | 需要细化跨界借鉴方案时 |
| 工作价值评估器 | 规划方案的投资回报 | 需要评估方案价值时 |

### 间接下游

| Agent名称 | 接收内容 | 说明 |
|-----------|----------|------|
| 工作计划生成器 | 行动项清单 | 通过策略执行Agent传递 |

## 输出流向

```
策略规划Agent
     ↓
策略执行Agent → 工作计划生成器
     ↓
模型迁移设计器 → 局部业务模型构建器
     ↓
工作价值评估器 → 资源放大器
```

## 传递的产出物

### 传递给策略执行Agent

- 完整的策略框架
- 执行优先级排序
- 关键成功要素
- 风险预警

### 传递给模型迁移设计器

- 跨界借鉴方案
- 可迁移要素清单
- 本地化调整建议

### 传递给工作价值评估器

- 规划方案描述
- 预期投入资源
- 预期产出价值
