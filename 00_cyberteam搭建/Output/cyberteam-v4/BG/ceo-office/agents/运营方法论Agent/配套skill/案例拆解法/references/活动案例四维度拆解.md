# 活动案例四维度拆解框架

## 框架说明
活动案例拆解的核心框架，从四个维度系统分析一个活动的完整逻辑。

---

## 维度一：目标用户

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| 用户画像 | 活动面向谁 | 年龄、职业、收入、消费习惯 |
| 需求特征 | 用户为什么参与 | 实用性、社交性、情感需求 |
| 选择原因 | 为什么选这个活动 | 差异化价值、时机、渠道 |
| 规模预估 | 目标用户池有多大 | 潜在用户数量、转化可行性 |

### 分析示例

```
案例：某知识付费产品拉新活动

目标用户分析：
- 用户画像：25-40岁职场人群，有提升技能需求
- 需求特征：职场晋升焦虑、渴望被认可、时间碎片化
- 选择原因：免费+名师+实用技能
- 规模预估：目标人群约2亿，转化目标10万
```

### 常见错误

1. **画像太宽泛** - "面向所有职场人群"是无效画像
2. **忽视用户动机** - 不理解用户为什么选择这个活动
3. **规模估算缺失** - 不知道目标用户池有多大

## 维度二：核心策略

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| 差异化价值 | 活动凭什么吸引用户 | 独特卖点、竞争优势 |
| 钩子设计 | 如何让用户行动 | 奖励机制、紧迫感、稀缺性 |
| 转化路径 | 用户如何完成目标 | 路径设计、关键节点 |
| 传播机制 | 如何让用户分享 | 社交货币、邀请奖励 |

### 常见钩子类型

| 钩子类型 | 描述 | 适用场景 |
|----------|------|----------|
| 免费钩 | 免费获取/体验 | 拉新引流 |
| 限量钩 | 限时限量 | 提升转化 |
| 专属钩 | 身份特权 | 高端用户 |
| 社交钩 | 邀请得奖励 | 裂变传播 |
| 成长钩 | 完成任务升级 | 提升粘性 |

### 分析示例

```
案例：某电商双11活动

核心策略分析：
- 差异化价值：全年最低价+独家赠品
- 钩子设计：前100名免单+整点秒杀
- 转化路径：浏览→加购→付定金→尾款
- 传播机制：分享得购物金+组队瓜分
```

## 维度三：执行细节

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| 资源投入 | 人力物力财力 | 团队配置、预算、技术 |
| 时间节点 | 什么时候做什么 | 预热、启动、高潮、收尾 |
| 关键动作 | 落地最重要的是什么 | 执行要点 |
| 风险控制 | 可能出什么错 | 预案准备 |

### 时间节点规划

```
活动时间轴：
T-30天  方案确定
T-21天  资源准备（设计、开发、物料）
T-14天  预热启动（内测、小范围传播）
T-7天   全面推广（多渠道曝光）
T-3天   最后冲刺（提醒、紧迫感）
T       活动高潮（双11当天）
T+7天   收尾期（售后、复盘）
```

### 常见风险及预案

| 风险类型 | 常见问题 | 预案建议 |
|----------|----------|----------|
| 技术风险 | 服务器崩溃 | 限流、分流预案 |
| 运营风险 | 库存不足 | 超卖处理方案 |
| 客服风险 | 咨询量暴增 | 客服扩容、FAQ |
| 舆情风险 | 负面评价 | 应急预案、话术准备 |

## 维度四：数据结果

### 分析要点

| 要点 | 问题 | 分析内容 |
|------|------|----------|
| 核心指标 | 活动的北极星指标 | 拉新量、销售额、DAU |
| 完成度 | 目标完成情况 | 实际 vs 目标 |
| 异常分析 | 有没有数据异常 | 异常节点的解释 |
| 归因分析 | 成功的关键因素是什么 | 数据验证 |

### 核心指标体系

| 指标类型 | 指标名称 | 计算方式 |
|----------|----------|----------|
| 拉新指标 | 新用户数 | 完成注册的用户数 |
| 拉新指标 | 获客成本 | 预算/新用户数 |
| 转化指标 | 转化率 | 成交数/访客数 |
| 转化指标 | 客单价 | 销售额/订单数 |
| 活跃指标 | DAU增量 | 活动期间DAU-日常DAU |
| 口碑指标 | 分享率 | 分享人数/参与人数 |

### 分析示例

```
案例数据结果：
- 拉新目标：10万，实际完成：12万（120%）
- 获客成本：目标8元，实际4.2元（节省47%）
- 转化率：35%（行业平均15%）
- 异常分析：活动第二天分享率下降，因为朋友圈刷屏被限制
```

---

## 四维度综合检查清单

- [ ] 目标用户：画像清晰、动机明确、规模可估
- [ ] 核心策略：差异化明显、钩子有效、路径顺畅
- [ ] 执行细节：资源充足、时间合理、风险可控
- [ ] 数据结果：指标完整、完成度高、归因清晰
