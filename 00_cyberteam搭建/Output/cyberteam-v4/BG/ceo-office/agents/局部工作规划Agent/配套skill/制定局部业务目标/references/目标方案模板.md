## 评分标准

| 评分 | 说明 |
|------|------|
| 1.0 | 几乎不可能完成（dream goal）|
| 0.7 | 有难度但可完成（stretch goal）|
| 0.5 | 保底目标，必须完成（commitment）|
| 0.3 | 低于预期，需要改进 |
| 0.0 | 完全未达成 |

---

## 周期结束评分

| KR | 实际完成 | 评分 | 说明 |
|----|----------|------|------|
| KR1 |          |      |      |
| KR2 |          |      |      |
| KR3 |          |      |      |

**本周期综合评分**：____
**下周期改进计划**：____________
