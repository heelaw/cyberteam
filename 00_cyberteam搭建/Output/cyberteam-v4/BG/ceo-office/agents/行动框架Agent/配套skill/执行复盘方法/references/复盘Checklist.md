## 复盘过程检查

### 目标回顾

- [ ] 目标是否清晰可量化
- [ ] 目标是否与实际执行目标一致
- [ ] 是否所有目标都被评估
- [ ] 偏差是否有数据支撑

### 过程分析

- [ ] 关键时间节点是否完整
- [ ] 关键决策点是否被识别
- [ ] 得失分析是否客观
- [ ] 是否区分了运气和实力

### 规律提炼

- [ ] 成功因素是否可复用
- [ ] 失败原因是否深入根因
- [ ] 是否有明显的幸存者偏差
- [ ] 规律是否可验证

### 行动制定

- [ ] 行动是否具体可执行
- [ ] 是否有明确的责任人
- [ ] 是否有明确的完成时间
- [ ] 是否有明确的验证标准

## 常见复盘问题

| 问题类型 | 典型表现 | 正确做法 |
|----------|----------|----------|
| 流于形式 | 复盘只是走过场 | 深入追问为什么 |
| 只谈外因 | 都是别人的问题 | 先从自身找原因 |
| 否定一切 | 成功都是运气 | 肯定优点和问题并存 |
| 没有行动 | 复盘完就结束 | 制定具体改进行动 |
| 归咎个人 | 都是个人责任 | 关注系统和流程问题 |
