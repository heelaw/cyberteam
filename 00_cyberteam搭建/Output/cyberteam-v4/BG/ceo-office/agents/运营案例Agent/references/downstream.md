# 运营案例Agent - 下游去向

## 下游Agent（输出目标）

### 直接下游

| Agent名称 | 接收内容 | 说明 |
|-----------|----------|------|
| 跨界模型规划器 | 案例分析结果 | 跨界借鉴完成时 |
| 模型迁移设计器 | 可迁移要素和方法论 | 需要设计迁移方案时 |

### 间接下游

| Agent名称 | 接收内容 | 说明 |
|-----------|----------|------|
| 局部业务模型构建器 | 案例方法论 | 业务模型构建时 |
| 运营方法论Agent | 萃取的方法论 | 方法论沉淀时 |

## 输出流向

```
运营案例Agent
     ↓
跨界模型规划器 → 新业务策略
     ↓
模型迁移设计器 → 迁移方案 → 局部业务模型构建器
     ↓
运营方法论Agent → 方法论库
```

## 传递的产出物

### 传递给跨界模型规划器

- 标杆案例分析报告
- 成功要素清单
- 可借鉴要素

### 传递给模型迁移设计器

- 可迁移要素清单
- 方法论框架
- 适用边界条件

### 传递给运营方法论Agent

- 萃取的方法论
- 案例支撑材料
- 应用场景定义
