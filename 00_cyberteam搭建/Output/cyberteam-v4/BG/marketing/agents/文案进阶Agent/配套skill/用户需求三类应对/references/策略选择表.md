# 策略选择表

## 功能型需求策略

### 用户特征
- 理性决策
- 关注效果
- 需要证据

### 策略矩阵

| 痛点 | 适用策略 | 具体方法 | 案例 |
|------|----------|----------|------|
| 担心没效果 | 效果量化 | 给出具体数字 | "7天学会" |
| 担心不安全 | 试用体验 | 免费试用 | "免费试用30天" |
| 担心不适合 | 证据证明 | 案例展示 | "10万用户验证" |
| 担心价格高 | 对比展示 | 算账对比 | "请一次家教要500" |

### 文案公式

```
功能需求文案公式1：
XX人使用后____（效果）
平均____（数据）

功能需求文案公式2：
使用前____ → 使用后____
____（对比）

功能需求文案公式3：
每天/次 ____（低投入）
获得/达到 ____（高回报）
```

## 情感型需求策略

### 用户特征
- 感性决策
- 关注感受
- 需要认同

### 策略矩阵

| 痛点 | 适用策略 | 具体方法 | 案例 |
|------|----------|----------|------|
| 担心不适合 | 身份认同 | 目标用户画像 | "XX人都在用" |
| 担心不好看 | 场景共鸣 | 使用场景描述 | "清晨从XX开始" |
| 担心没品质 | 情怀故事 | 品牌故事 | "XX年的坚持" |
| 担心没面子 | 社交货币 | 分享属性 | "朋友圈都在晒" |

### 文案公式

```
情感需求文案公式1：
像____（目标用户）一样____（行为）
你也值得____（情感满足）

情感需求文案公式2：
____（使用场景）就是你的____（身份标签）

情感需求文案公式3：
给自己一个____（情感价值）
而不是一个____（功能价值）
```

## 社会型需求策略

### 用户特征
- 群体意识
- 关注认可
- 从众心理

### 策略矩阵

| 痛点 | 适用策略 | 具体方法 | 案例 |
|------|----------|----------|------|
| 担心品牌差 | 权威背书 | 名人/专家推荐 | "XX都在用" |
| 担心没人买 | 销量证明 | 销售数据 | "热销100万" |
| 担心口碑差 | 口碑展示 | 用户评价 | "好评如潮" |
| 担心被孤立 | 圈层效应 | 特定人群 | "XX圈子的选择" |

### 文案公式

```
社会需求文案公式1：
____（数字）人都在用
____（权威）都在推荐

社会需求文案公式2：
你值得拥有____（被认可）
____（群体）的一致选择

社会需求文案公式3：
当____（名人/专家）都在用
你也应该拥有
```

├─ 功能型（关心效果）

├─ 情感型（关心感受）

│  ├─ 担心没品质 → 情怀故事

└─ 社会型（关心认可）
