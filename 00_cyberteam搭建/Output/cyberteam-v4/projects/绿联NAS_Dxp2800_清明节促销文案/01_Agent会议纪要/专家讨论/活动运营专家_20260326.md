## 四、活动玩法设计（第三步：设计活动玩法）

### 4.1 驱动力选择与组合

根据九大驱动力框架，本次活动选择以下组合：

| 驱动力 | 编号 | 选择理由 | 应用场景 |
|--------|------|----------|----------|
| **利益驱动** | #1 | 核心转化动力 | 新人专享券、满减优惠、以旧换新补贴 |
| **稀缺驱动** | #2 | 提升紧迫感 | 限量特价名额、限时48小时抢购 |
| **恐惧驱动** | #7 | 唤醒数据安全焦虑 | "你的数据还在裸奔吗？" |
| **社交驱动** | #4 | 裂变传播 | 邀请好友得奖励、晒单分享 |
| **荣誉驱动** | #6 | 增强参与感 | 晒单达人榜、种草官认证 |
| **好奇心驱动** | #9 | 引流吸睛 | 悬念式预告、盲盒抽奖 |

```
驱动力协同公式：
活动吸引力 = #1×0.3 + #2×0.25 + #7×0.2 + #4×0.15 + #6×0.1
            + 协同放大(0.2)

协同效应：
- #2稀缺 + #7恐惧 = 1.5倍紧迫感（"仅剩XX台，错过再等一年"）
- #4社交 + #6荣誉 = 1.3倍传播效果（"晒单赢取种草官徽章"）
- #1利益 + #9好奇心 = 1.4倍参与度（"猜价格赢免单"）
```

#### 机制一：「清明守护价」限时抢购

```
活动规则：
1. 4月4日-4月6日，每日10:00/16:00限量释放特价名额

```

```
1. 新用户（未购买过绿联NAS）可领取500元新人礼包
```

```
活动规则：

- "旧硬盘坏了，数据全没了？NAS给您数据保底"
```

```
1. 购买用户在小红书/抖音发布NAS使用场景（晒单/种草）
```
