# 西北发面包子品牌定位报告

**项目**：西北发面包子品牌策划
**业态**：中式快餐/面点
**位置**：深圳海岸城
**面积**：200平方米
**版本**：v1.0
**日期**：2026-03-27

---

## 一、品牌定位

### 1.1 品牌核心价值主张

**主定位语**：

> **"一笼好包，馅满情足"**

**价值主张三层拆解**：

| 层级 | 价值点 | 用户感知 |
|------|--------|----------|
| **物质层** | 馅料实在、品种丰富 | 吃得到的满足感 |
| **情感层** | 温暖、用心、实在 | 被认真对待的归属感 |
| **精神层** | 匠心传承、现做新鲜 | 对品质的认同 |

**核心价值支撑**：

1. **大馅担当**：拒绝"皮多馅少"，每款包子馅料占比≥60%
2. **品种多样**：招牌系列+地方特色+季节限定，满足多元口味
3. **现包现蒸**：透明厨房，看得见的新鲜制作
4. **便捷高效**：3分钟出餐，午餐不等待

---

### 1.2 品牌差异化定位

**品类定位**：发面包子专门店（而非传统包子铺）

| 维度 | 传统包子铺 | 本品牌 |
|------|------------|--------|
| **产品策略** | 单品走量 | 多SKU精品路线 |
| **馅料风格** | 地域局限 | 融合创新，不拘一格 |
| **消费场景** | 早餐为主 | 全时段运营 |
| **空间体验** | 档口即买即走 | 堂食+外带双场景 |
| **品牌调性** | 早餐铺子 | 品质简餐 |

**差异化核心**：

> **"不是所有的包子都叫发面包子，不是所有的发面包子都有这么足的馅"**

**竞争对手对比**：

| 竞品类型 | 代表品牌 | 本品牌差异点 |
|----------|----------|--------------|
| 传统包子铺 | 巴比馒头、庆丰包子 | 品质感更强，SKU更精 |
| 快餐连锁 | 真功夫、72街 | 产品形态更轻，场景更灵活 |
| 网红面馆 | 和府捞面、陈香贵 | 切入更细分的"包子"赛道 |
| 新中式点心 | 墨茉点心局、虎头局 | 主食属性更强，刚需性更高 |

---

### 1.3 品牌调性与性格

**品牌个性（Brand Personality）**：

| 维度 | 描述 | 体现场景 |
|------|------|----------|
| **实在** | 不花哨，有实力 | 大馅足料、明码标价 |
| **温暖** | 不冷漠，有温度 | 服务话术、环境氛围 |
| **品质** | 不将就，有追求 | 食材溯源、现做工艺 |
| **活力** | 不老气，有创新 | 季节菜单、联名限定 |

**品牌人设**：

> **"一个认真做包子的手艺人"**
> - 对外表达：朴实务实、亲切真诚
> - 行动准则：用好料、做好包、讲实在话
> - 禁忌：过度营销、虚假宣传、偷工减料

**视觉调性关键词**：

```
温暖金 → 麦色、蒸汽、金黄
实在感 → 清晰、饱满、圆润
品质感 → 留白、精致、有序
年轻化 → 简约、活力、国际化
```

**品牌色彩体系**：

| 色彩 | 色值 | 应用场景 |
|------|-----|----------|
| 麦金 | #D4A853 | 主色调、品牌标识 |
| 蒸白 | #F5F0E8 | 背景色、留白 |
| 馅红 | #C53D43 | 点缀色、强调 |
| 箸青 | #4A7C59 | 辅助色、健康感 |

## 二、人群定位

### 2.1 核心目标人群

**人群画像**：都市打拼的"饱腹刚需族"

| 属性 | 描述 |
|------|------|
| **年龄** | 25-40岁 |
| **职业** | 写字楼白领、商务人士 |
| **收入** | 月薪8,000-25,000元 |
| **场景** | 工作日午餐、加班餐 |
| **心理特征** | 务实高效、注重性价比、不愿在吃饭上浪费太多时间 |

**用户故事**：

> *"中午休息时间就1小时，不想吃太油腻的，也不想等太久。最好是能吃饱、吃好、价格别太贵。包子方便快捷，馅料丰富还能换着吃，比天天吃同一家盖浇饭强。"*
> — 深圳海岸城某写字楼白领

**消费动机**：

1. **刚需驱动**：工作日必须解决午餐问题
2. **效率优先**：3分钟内出餐，不耽误休息时间
3. **多样化需求**：一周吃5天不重样
4. **品质稳定**：不希望踩雷，追求稳定预期

**触达渠道**：

- 写字楼电梯广告
- 午间外卖平台
- 商场动线引导
- 口碑推荐（同事间的"这家包子不错"）

---

---

### 2.3 人群优先级

| 优先级 | 人群 | 策略定位 | 贡献占比 |
|--------|------|----------|----------|
| **P0 核心** | 都市白领（午餐刚需） | 主打刚需、高频、午市峰值 | 60% |
| **P1 重要** | 家庭消费者 | 周末增量、场景延伸 | 20% |
| **P2 补充** | 年轻尝鲜族 | 口碑传播、社交裂变 | 15% |
| **P3 边缘** | 银发一族 | 早市补充、公益形象 | 5% |

**人群策略核心逻辑**：

> **先把午餐场景的白领用户服务透，再逐步延伸至其他场景和人群。**

## 三、场景定位

### 3.1 早餐场景

**场景特征**：

| 要素 | 描述 |
|------|------|
| 时段 | 7:00-9:30 |
| 客流量 | 约占全天15-20% |
| 需求 | 快速打包、即买即走 |
| 决策因素 | 便捷性>品质>价格 |

**产品策略**：

- **精简SKU**：主打3-4款招牌包子，缩短决策时间
- **打包组合**：包子+豆浆/粥品，套餐优惠
- **外带优先**：减少堂食压力，提高翻台

**价格带**：

| 组合 | 定价 | 定位 |
|------|------|------|
| 单品 | 6-10元/个 | 品质感 |
| 套餐A | 15-18元 | 饱腹型 |
| 套餐B | 20-25元 | 丰富型 |

**触达方式**：

- 地铁口/写字楼派发传单
- 外卖平台早餐时段的精准推送
- 社群营销（写字楼福利群）

---

### 3.2 午餐场景

**场景特征**：

| 要素 | 描述 |
|------|------|
| 时段 | 11:00-14:00 |
| 客流量 | 约占全天50-60%（黄金时段） |
| 需求 | 吃饱吃好、效率适中 |
| 决策因素** | 品质>效率>价格 |

**产品策略**：

- **全SKU展示**：招牌+特色+限定，让用户有选择空间
- **小份制**：包子按个卖，单人可选择2-3个不同口味
- **加料选项**：可加卤蛋、肠、饮料等，提升客单价
- **堂食优化**：提供餐盘，托盘动线设计

**核心卖点输出**：

```
现包现蒸 → 新鲜看得见
大馅足料 → 实实在在
品种丰富 → 一周不重样
3分钟出餐 → 不用干等
```

**价格带**：

| 组合 | 定价 | 定位 |
|------|------|------|
| 单人简餐 | 25-35元 | 吃饱不贵 |
| 单人丰盛版 | 40-50元 | 品质工作餐 |
| 商务套餐 | 50-60元 | 接待/加班 |

---

### 3.3 下午茶/晚餐场景

**场景特征**：

| 要素 | 下午茶 | 晚餐 |
|------|--------|------|
| 时段 | 14:00-17:00 | 17:00-21:00 |
| 客流占比 | 10-15% | 15-20% |
| 需求 | 轻食、社交、休憩 | 家庭餐、简餐 |
| 决策因素 | 体验>价格 | 品质>便捷 |

**下午茶场景**：

- **产品**：小份装包子（2-3个）+ 茶饮/豆浆
- **场景**：闺蜜聊天、商务洽谈、独自休憩
- **包装**：精致纸盒+品牌手提袋（可拍照分享）
- **氛围**：背景音乐、舒适座椅、充电插座

**晚餐场景**：

- **产品**：招牌包子 + 地方特色 + 季节限定
- **场景**：加班餐、附近居民、外卖
- **策略**：延长营业至21:00，承接外卖晚高峰

**价格带**：

| 场景 | 组合 | 定价 |
|------|------|------|
| 下午茶 | 包子2个+饮品 | 22-30元 |
| 晚餐 | 包子3-4个+小菜 | 35-50元 |
| 外卖 | 套餐制 | 30-45元 |

## 四、品牌故事与Slogan建议

### 4.1 品牌故事框架

**故事主线**：传承与创新

**故事大纲**：

```
【缘起】
张记包子铺，源自西北小镇
三代人专注发面包子
坚持"皮薄馅大"的祖训

【挑战】
传统手艺如何走进都市？
深圳海岸城的年轻食客会接受吗？

【破局】
保留古法发酵的技艺
融入时尚多元的口味
现包现蒸，看得见的新鲜
每一笼都是热气腾腾的诚意

【理念】
不是所有的包子都叫发面包子
不是所有的大馅都叫实在馅
我们做的不仅是包子
更是一份"被认真对待"的温暖
```

**品牌溯源表达**：

| 维度 | 内容 |
|------|------|
| **起源地** | 西北某小镇（可虚构，利于传播） |
| **传承** | 三代技艺，专注发面 |
| **匠心** | 18道工序，老面发酵 |
| **创新** | 融合南北口味，不拘一格 |
