## 一、项目理解

| 项目要素 | 描述 |
|---------|------|
| 业态 | 西北发面包子为主的餐厅 |
| 面积 | 200平方米 |
| 位置 | 深圳海岸城 |
| 核心特点 | 馅料丰富、品种多，不局限于西北风味，以发面包子为主 |

## 二、北极星指标

**核心指标**: 品牌认知度 + 开业后前3个月日均营收

| 指标 | 目标值 | 优先级 |
|------|--------|--------|
| 日均营收 | ¥8,000-12,000/天 | P0 |
| 客单价 | ¥25-35 | P1 |
| 翻台率 | 3-4次/天 | P1 |
| 复购率 | 40%+ | P2 |

## 三、约束条件

| 约束类型 | 具体限制 |
|---------|----------|
| 预算 | 未知，预留空间 |
| 面积 | 200平，约40-50个座位 |
| 定位 | 中高端品质感，不能是街边店档次 |
| 位置 | 海岸城商圈，面向都市白领 |

## 四、风险偏好

**定位：平衡型（不是激进也不是保守）**

- 允许一定的新品类试错
- 但核心产品必须有确定性
- 不做纯网红模式，要有复购支撑

| 问题 | CEO判断 |
|------|--------|
| 200平能否撑起纯包子业态？ | **需要专家给出可行性分析** |
| 是否要做堂食+外卖双驱动？ | **是，外卖必须做** |
| 是否要做全时段（早餐+午餐+晚餐）？ | **是，早午餐为主，晚餐为辅** |
| 价格定位 | 中高端，客单价25-35元 |
