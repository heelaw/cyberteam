# 对话记录模板 v2.0

> 版本更新 (v2.0): 明确CEO-COO-专家的三层对话体系

---

## 一、对话层级体系

```
第一层：CEO → COO（战略对齐）
  └── CEO与COO对齐目标、约束、风险偏好、资源

第二层：COO → 专家团队（策略制定）
  └── 卖点方向、用户场景、渠道策略、转化策略

第三层：专家团队内部分歧讨论（深度碰撞）
  └── 风险预案、保底措施、多轮迭代

第四层：COO汇总上报（向上汇报）
  └── 向CEO汇报策略方案、风险预案
```

## 六、使用说明

### 对话记录命名规范

```bash
对话_[序号]_[类型]_[日期].md

示例：
对话_01_CEO对齐_20260326.md    # CEO-COO对齐
对话_02_策略讨论_20260326.md   # COO-专家策略讨论
对话_03_风险预案_20260326.md   # 风险预案讨论
对话_04_CEO汇报_20260326.md    # COO向CEO汇报
```

### 强制对话节点

| 节点 | 必须对话 | 产出物 |
|------|----------|--------|
| **启动时** | CEO→COO战略对齐 | 对齐记录 |
| **策略制定** | COO→专家团队讨论 | 策略讨论记录 |
| **方案确定前** | 风险预案讨论 | 风险预案记录 |
| **执行前** | COO→CEO汇报 | 汇报记录 |
| **执行后** | 复盘讨论 | 复盘报告 |
