# business_data - 标准业务数据模板

**版本**: v1.0
**创建**: 2026-03-25
**用途**: 所有业务数据文件必须遵循此模板

---

## 一、产品信息

```yaml
product:
  name: 产品名称
  type: 产品类型
  launch_date: 上线日期
  current_stage: 探索期/成长期/成熟期/衰退期
```

### 1.1 产品类型分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 交易平台 | 电商、OTA、外卖 | 淘宝、美团 |
| SaaS工具 | 订阅制软件 | Slack、Figma |
| 内容平台 | 社交、内容、UGC | 抖音、B站 |
| 双边市场 | 供需撮合平台 | 滴滴、Airbnb |
| 混合型 | 多种模式混合 | 微信、抖音电商 |

## 二、核心指标定义

### 2.1 北极星指标 (NSM)

```yaml
nsm:
  name: [指标名称]
  definition: [指标定义]
  calculation: [计算公式]
  target_value: [目标值]
  current_value: [当前值]
  unit: [单位]
  frequency: [更新频率]
```

### 2.2 AARRR指标

| 层级 | 指标 | 计算公式 | 行业基准 |
|------|------|----------|----------|
| **Acquisition** | | | |
| | 新增用户数 | 期间新增注册用户 | 行业差异大 |
| | 获客成本 (CAC) | 营销费用 ÷ 新增付费用户 | <100元 |
| | 渠道ROI | 渠道收入 ÷ 渠道成本 | >1 |
| **Activation** | | | |
| | 激活率 | 完成激活用户 ÷ 新增用户 | >30% |
| | Aha时刻达成率 | 达成Aha时刻用户 ÷ 新增用户 | >20% |
| **Retention** | | | |
| | 次日留存率 | 次日回访 ÷ 新增 | >40% |
| | 7日留存率 | 7日回访 ÷ 新增 | >20% |
| | 30日留存率 | 30日回访 ÷ 新增 | >10% |
| **Revenue** | | | |
| | 付费转化率 | 付费用户 ÷ DAU | >5% |
| | ARPU | 总收入 ÷ DAU | 行业差异 |
| | ARPPU | 总收入 ÷ 付费用户 | 行业差异 |
| | LTV | 用户生命周期总价值 | LTV > 3×CAC |
| **Referral** | | | |
| | 病毒系数 (K) | 邀请人数 ÷ 被邀请转化人数 | >1 |
| | NPS | 推荐者% - 贬损者% | >30 |

## 三、用户分层标准

### 3.1 RFM分层

| 分层 | R(最近) | F(频率) | M(金额) | 运营策略 |
|------|---------|---------|---------|----------|
| 高价值用户 | R<7天 | F>5次 | M>1000元 | 专属服务 |
| 成长用户 | R<30天 | F>2次 | M>100元 | 培育转化 |
| 沉默用户 | R>30天 | F<2次 | M<100元 | 召回 |
| 流失用户 | R>90天 | F=0 | M=0 | 放弃/清理 |

### 3.2 AARRR分层

| 阶段 | 定义 | 核心指标 |
|------|------|----------|
| 新用户 | 注册<7天 | 激活率 |
| 活跃用户 | DAU/MAU>20% | 使用时长 |
| 高价值用户 | 付费>3次 | LTV |
| 流失风险用户 | 连续3天未访问 | 留存率 |

## 四、数据格式标准

### 4.1 时间格式

```
ISO 8601: 2026-03-25T09:00:00+08:00
日期: YYYY-MM-DD
月份: YYYY-MM
周: YYYY-WXX
```

### 4.2 数值格式

| 类型 | 格式 | 示例 |
|------|------|------|
| 百分比 | X.X% | 35.5% |
| 金额 | X万/亿 | 1000万 |
| 用户数 | X万/X千万 | 500万 |
| 比率 | X.XX | 3.25 |

### 4.3 趋势标识

```
up: ↑ (较上期增长)
down: ↓ (较上期下降)
stable: → (较上期持平)
warning: ⚠️ (接近阈值)
critical: 🔴 (突破阈值)
```
