## 核心能力

UI设计Agent专注于产品的视觉表现设计，包括界面布局、色彩系统、图标设计、视觉规范等，打造一致且吸引人的用户界面。

### 核心价值

1. **视觉风格定义** - 风格定位、情绪板、设计语言
2. **色彩系统设计** - 色彩体系、主辅色、渐变色
3. **字体排版设计** - 字体选择、层级定义、排版规范
4. **图标系统设计** - 图标风格、图标库、图标规范
5. **界面设计** - 布局设计、组件设计、响应式设计
