# 绩效管理专家 接口协议

## 输入接口 (Upstream)

### 来自HR总监
| 输入类型 | 格式 | 说明 |
|---------|------|-----|
| 考核方案 | 考核周期/指标模板/流程 | 明确考核要求 |
| 校准比例 | 分布比例/晋升名额 | 控制结果分布 |
| 晋升政策 | 晋升标准/评审流程 | 规范晋升决策 |
| 预算信息 | 调薪预算/激励总额 | 结果应用约束 |

### 来自业务部门
| 输入类型 | 格式 | 说明 |
|---------|------|-----|
| 目标调整 | OKR变更/特殊情况 | 目标调整申请 |
| 考核反馈 | 自评/他评/特殊情况 | 评估依据 |
| 晋升提名 | 候选人材料/推荐理由 | 晋升评审输入 |

## 输出接口 (Downstream)

### 流向HR总监
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 考核进度 | 周报/月报 | 进度追踪 |
| 考核结果 | 汇总分析报告 | 数据汇报 |
| 校准建议 | 分布分析/异常预警 | 决策支持 |
| 晋升建议 | 候选人排名/方案 | 晋升决策 |

### 流向业务部门
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 考核模板 | OKR/KPI模板/填写指南 | 工具支持 |
| 考核通知 | 时间节点/流程说明 | 宣导通知 |
| 结果反馈 | 个人绩效报告/改进建议 | 反馈面谈 |
| 校准会议 | 会议安排/数据准备 | 组织协调 |

### 流向员工
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 目标确认 | 绩效合同/确认签字 | 目标对齐 |
| 考核通知 | 考核开始/截止时间 | 流程提醒 |
| 结果通知 | 绩效等级/改进方向 | 反馈面谈 |
| 晋升通知 | 晋升决定/发展建议 | 成长肯定 |

## 反馈回路

### 向HR总监汇报
- 月报：考核进度/数据汇总
- 季报：中期检视/目标调整
- 年报：年度考核总结/体系优化

### 向业务部门反馈
- 实时：考核流程支持
- 月同步：考核进度/异常情况
- 季回顾：绩效数据/改进建议

## 调用协议

### 触发条件
1. 新考核周期启动
2. 目标调整需求
3. 考核结果校准
4. 晋升评审启动

### 协作流程
```
年度方案制定 → 目标设定 → 过程追踪 → 绩效考核
    → 结果校准 → 结果应用 → 复盘优化
```
