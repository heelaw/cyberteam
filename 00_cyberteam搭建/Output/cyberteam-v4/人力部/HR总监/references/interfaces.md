# HR总监 接口协议

## 输入接口 (Upstream)

### 来自CEO
| 输入类型 | 格式 | 说明 |
|---------|------|-----|
| 战略目标 | 战略文档/会议纪要 | 公司级战略方向 |
| 高管需求 | 需求清单/优先级 | 关键岗位招聘/激励 |
| 组织决策 | 决策备忘录 | 架构调整/编制变化 |

### 来自各部门
| 输入类型 | 格式 | 说明 |
|---------|------|-----|
| HC申请 | 申请表+JD | 人员需求申请 |
| 绩效结果 | 考核汇总表 | 绩效评估输出 |
| 激励申请 | 申请表+方案 | 专项激励需求 |

## 输出接口 (Downstream)

### 流向招聘Agent
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 招聘需求 | 岗位说明书+优先级 | 明确招聘要求 |
| 预算批复 | 预算审批单 | HC和薪酬预算 |
| 招聘策略 | 策略文档 | 招聘渠道/方向 |

### 流向绩效管理Agent
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 考核方案 | 考核指标体系 | KPI/OKR标准 |
| 校准决策 | 校准会议纪要 | 晋升/调薪决策 |
| 改进计划 | PIP文档 | 绩效改进要求 |

### 流向团队激励Agent
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 激励预算 | 预算分配方案 | 激励总额控制 |
| 激励方向 | 激励策略文档 | 重点激励对象 |
| 氛围要求 | 文化建设方向 | 文化落地要求 |

## 反馈回路

### 向CEO汇报
- 每月：人力月报（编制/成本/关键指标）
- 每季：人才盘点报告/组织健康度
- 按需：重大风险预警/政策调整

### 横向协作
- 与增长BG：HC协同/人才共享
- 与产品BG：考核联动/激励对齐
- 与技术BG：技术序列/职级体系

## 调用协议

### 触发条件
1. 新战略周期启动（年度/季度）
2. 重大组织调整
3. 核心人才流失风险
4. 跨部门协作需求

### 协作流程
```
需求接收 → 评估分析 → 方案制定 → 资源协调 → 执行落地 → 效果追踪
```
