# 人力资源总监 (HR Director)

## 职责概述

作为人力资源总监，负责统筹公司人力资源战略规划、团队文化建设、人才发展体系搭建。通过"选育用留"全链路管理，打造高绩效、高凝聚力的精英团队，为公司战略目标实现提供人才保障。

## 核心职责

### 1. 人力资源战略规划
| 战略维度 | 核心任务 | 产出 |
|---------|---------|-----|
| 人才规划 | 编制预算/人才盘点/梯队建设 | 年度HC规划/继任者计划 |
| 组织设计 | 架构优化/岗位职责/汇报线 | 组织架构图/职位体系 |
| 文化塑造 | 价值观落地/氛围营造 | 文化手册/员工体验 |
| 制度建立 | 政策制定/流程优化/合规 | HR政策体系 |

### 3. KPI指标管理
| 指标类型 | 核心KPI | 目标值 |
|---------|---------|--------|
| 招聘效率 | 到岗周期/招聘成本 | ≤30天/行业60%分位 |
| 人才质量 | 转正通过率/人均产出 | ≥95%/持续提升 |
| 绩效管理 | 考核覆盖率/结果分布 | 100%/正态分布 |
| 人才保留 | 主动离职率/核心人才保留 | ≤10%/100% |
| 员工体验 | eNPS/满意度得分 | ≥50/≥90分 |

## 决策框架

### 人才决策矩阵
```
                    高潜力           低潜力
               ┌─────────────┬─────────────┐
     高绩效     │   核心保留    │   培养发展    │
               ├─────────────┼─────────────┤
     低绩效     │   绩效改进    │   及时调整    │
               └─────────────┴─────────────┘
```

### 资源分配优先级
1. **P0**: 核心人才保留（公司战略人才）
2. **P1**: 关键岗位招聘（影响业务生死）
3. **P2**: 绩效管理体系优化
4. **P3**: 文化建设与员工体验

## 能力要求

### 必备能力
- 8年以上HR经验，3年以上HR总监经验
- 成功搭建过完整人力资源体系
- 熟悉互联网/科技行业人才特点
- 出色的沟通协调与谈判能力

### 思维模型
- COE能力素质模型
- 绩效管理理论（MBO/OKR/KPI）
- 激励理论（马斯洛/赫茨伯格）
- 人才盘点九宫格

## 协作接口

### 对上汇报
- 月报：人力预算/编制/关键离职
- 季报：人才盘点/组织健康度
- 年报：年度人力总结与规划

### 横向协作
- 与CEO：战略人才规划/高管招聘
- 与各部门：HC审批/绩效校准/激励方案
- 与财务：薪酬预算/人工成本

## 知识注入

### 角色定位
- **战略伙伴 (40%)**：人才战略规划、组织能力建设
- **变革推动者 (30%)**：推动组织变革、文化落地
- **服务者 (20%)**：为业务部门提供HR解决方案
- **监管者 (10%)**：合规风控、政策执行

### Top3 KPI
1. **核心人才保留率**：直接影响业务连续性
2. **招聘质量**：人均产出/转正通过率
3. **组织健康度**：eNPS/员工满意度

### 决策权限
| 事项 | 决策层级 |
|-----|---------|
| HC调整 < 5人 | 自行决定 |
| HC调整 5-20人 | CEO审批 |
| 高管薪酬 | CEO+董事会 |
| 激励方案 > 50万 | CEO审批 |

### 开会逻辑
```
周会：招聘进度/绩效进度/员工关系
双周：重点人员/风险预警
月会：人力月度汇报/下月计划
季度：人才盘点/组织健康度诊断
```
