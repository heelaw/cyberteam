# 招聘专家 接口协议

## 输入接口 (Upstream)

### 来自HR总监
| 输入类型 | 格式 | 说明 |
|---------|------|-----|
| 招聘需求 | 申请表+JD+优先级 | 明确岗位要求 |
| 预算信息 | 薪酬范围/HC数量 | 成本控制 |
| 渠道策略 | 渠道优先级/重点平台 | 资源分配 |
| 特殊要求 | 紧急程度/隐藏偏好 | 风险提示 |

### 来自业务部门
| 输入类型 | 格式 | 说明 |
|---------|------|-----|
| 补充需求 | 能力补充/文化偏好 | 细化人才画像 |
| 面试反馈 | 评估表/录用建议 | 决策支持 |
| 调整需求 | JD变更/暂停/取消 | 需求变更 |

## 输出接口 (Downstream)

### 流向HR总监
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 周报 | 招聘进度/到岗情况 | 数据汇报 |
| 月报 | 渠道效果/成本分析 | 复盘优化 |
| 风险预警 | 紧急事项/难招聘岗位 | 预警提示 |

### 流向业务部门
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 候选人推荐 | 简历+评估报告 | 决策参考 |
| 面试安排 | 日程+面试官+地点 | 协调确认 |
| Offer通知 | 薪酬方案+入职时间 | 最终确认 |

### 流向候选人
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 面试邀请 | 时间+流程+准备材料 | 体验优化 |
| 结果通知 | 录用/待定/不合适 | 及时反馈 |
| 入职指引 | 准备事项+联系人 | 顺利入职 |

## 反馈回路

### 向HR总监汇报
- 周报：本周招聘进度/下周计划
- 月报：月度招聘数据复盘
- 专项：紧急岗位/重要进展

### 向业务部门反馈
- 实时：候选人推荐/面试安排
- 周同步：招聘进度/预计到岗
- 月回顾：招聘质量/满意度

## 调用协议

### 触发条件
1. 收到招聘需求申请
2. 业务部门补充需求
3. 候选人面试安排
4. Offer谈判与发放

### 协作流程
```
需求接收 → JD沟通 → 人才寻访 → 简历筛选 → 面试安排
    → 评估反馈 → Offer谈判 → 入职跟进 → 试用期关怀
```
