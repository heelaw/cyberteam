# 团队激励专家 接口协议

## 输入接口 (Upstream)

### 来自HR总监
| 输入类型 | 格式 | 说明 |
|---------|------|-----|
| 激励预算 | 预算总额/分配比例 | 成本控制 |
| 策略方向 | 激励重点/倾斜方向 | 策略指引 |
| 重点对象 | 核心人才名单/激励需求 | 重点关注 |
| 政策调整 | 薪酬政策/激励规则变更 | 方案调整依据 |

### 来自业务部门
| 输入类型 | 格式 | 说明 |
|---------|------|-----|
| 激励需求 | 申请表/目标/理由 | 需求输入 |
| 奖励提名 | 候选人/事跡/建议 | 评选依据 |
| 活动申请 | 团建需求/预算/方案 | 氛围建设 |

## 输出接口 (Downstream)

### 流向HR总监
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 激励方案 | 薪酬/奖金/荣誉方案 | 决策审批 |
| 预算报告 | 成本分析/效果预估 | 资源配置 |
| 效果评估 | 激励效果/满意度分析 | 复盘优化 |
| 风险预警 | 核心流失/不满风险 | 预警提示 |

### 流向业务部门
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 激励通知 | 获奖名单/奖励内容 | 即时反馈 |
| 薪酬通知 | 调薪通知/生效时间 | 结果告知 |
| 活动通知 | 团建安排/参与方式 | 组织协调 |
| 荣誉证书 | 获奖证明/表彰决定 | 仪式呈现 |

### 流向员工
| 输出类型 | 格式 | 说明 |
|---------|------|-----|
| 奖励发放 | 奖金/奖品/证书 | 即时激励 |
| 荣誉公示 | 获奖公告/展示 | 认可宣传 |
| 发展机会 | 培训/晋升/项目机会 | 成长激励 |
| 关怀沟通 | 节日问候/困难帮扶 | 情感连接 |

## 反馈回路

### 向HR总监汇报
- 月报：激励发放/成本分析
- 季报：激励效果/满意度调研
- 年报：年度激励总结/体系优化

### 向业务部门反馈
- 实时：激励申请结果/活动安排
- 月同步：激励预算使用/效果跟踪
- 季回顾：激励满意度/改进建议

## 调用协议

### 触发条件
1. 绩效考核结果应用
2. 业务部门激励申请
3. 重大荣誉评选
4. 员工关怀需求

### 协作流程
```
需求接收 → 方案设计 → 预算审批 → 执行发放
    → 效果追踪 → 持续优化
```
