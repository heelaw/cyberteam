# downstream - 技术BG Agent

## 直接下游

| 下游Agent | 关系 | 说明 |
|-----------|------|------|
| 监控中台 | 监控依赖 | 系统监控需求 |
| 通信中台 | 任务交接 | 需要多Agent协作时 |
| 代码仓库 | 代码存储 | 代码提交和版本管理 |
| CI/CD系统 | 持续集成 | 自动化构建部署 |

## 输出物

| 输出物 | 下游使用方 | 用途 |
|--------|------------|------|
| 代码产物 | 代码仓库 | 存储和管理 |
| API文档 | 调用方 | 接口说明 |
| 架构文档 | 团队共享 | 技术参考 |
| 测试报告 | QA团队 | 质量评估 |
| 部署配置 | DevOps | 自动化部署 |

## 产出分类

### 功能开发产出

| 产出物 | 格式 | 下游使用 |
|--------|------|----------|
| 源代码 | 多语言 | 仓库存储 |
| 测试用例 | pytest/unittest | 自动化测试 |
| API文档 | OpenAPI/Swagger | 接口调用 |
| 部署配置 | YAML/JSON | CI/CD |

### 架构设计产出

| 产出物 | 格式 | 下游使用 |
|--------|------|----------|
| 架构文档 | markdown | 技术团队 |
| 组件图 | markdown/图片 | 架构可视化 |
| 接口协议 | markdown | 团队协作 |
| 数据库设计 | SQL/markdown | 数据库开发 |

### 运维支持产出

| 产出物 | 格式 | 下游使用 |
|--------|------|----------|
| 监控配置 | YAML | 监控中台 |
| 日志规范 | markdown | 日志系统 |
| 告警规则 | YAML | 监控中台 |
| 性能报告 | markdown | 决策支持 |

## 下游数据流

```
技术BG执行
    ↓
    ├── 源代码 → 代码仓库 → CI/CD → 生产环境
    ├── API文档 → 调用方（前端/移动端）
    ├── 测试报告 → QA团队 → 质量门禁
    ├── 监控配置 → 监控中台 → 系统监控
    └── 架构文档 → 技术团队共享
```
