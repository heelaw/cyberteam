# downstream - 产品BG Agent

## 直接下游

| 下游Agent | 关系 | 说明 |
|-----------|------|------|
| 技术BG | 技术依赖 | 产品功能开发需求 |
| 设计BG | 设计依赖 | UI/UX设计需求 |
| 资产中台 | 资产调用 | 调用设计类baoyu技能 |
| 通信中台 | 任务交接 | 需要多Agent协作时 |

## 输出物

| 输出物 | 下游使用方 | 用途 |
|--------|------------|------|
| 产品规格说明书 | 技术BG | 开发依据 |
| 功能优先级列表 | 技术BG | 开发排期 |
| UX优化方案 | 设计BG | 体验改进 |
| 原型描述 | 设计BG | 设计参考 |
| 需求文档 | 技术BG | 详细需求 |

## 产出分类

### 产品设计产出

| 产出物 | 格式 | 下游使用 |
|--------|------|----------|

| 功能清单 | JSON | 排期开发 |
| 流程图 | markdown | 理解实现 |
| 原型描述 | markdown | UI设计 |

### 需求分析产出

| 产出物 | 格式 | 下游使用 |
|--------|------|----------|

| 优先级排序 | JSON | 排期 |

### UX优化产出

| 产出物 | 格式 | 下游使用 |
|--------|------|----------|
| 体验优化方案 | markdown | 设计 |
| 用户旅程图 | markdown | 理解用户 |
| 优化建议清单 | JSON | 排期 |

## 下游数据流

```
产品BG执行
    ↓
    ├── 产品规格 → 技术BG → 开发实现
    ├── 功能清单 → 技术BG → 排期开发
    ├── UX方案 → 设计BG → UI/UX设计
    └── 需求优先级 → PM协调器 → 项目管理
```
