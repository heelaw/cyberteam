# upstream - 产品BG Agent

## 直接上游

| 上游Agent | 关系 | 说明 |
|-----------|------|------|
| CEO | 任务路由 | 产品BG的唯一上游，接收CEO的L3A路由任务 |
| PM协调器 | 任务协同 | 高复杂度任务时的PM协调 |
| 记忆中台 | 数据依赖 | 获取历史产品数据和用户反馈 |

## 数据依赖

| 数据源 | 格式 | 刷新频率 | 说明 |
|--------|------|----------|------|
| 用户任务描述 | text | 每次任务 | CEO路由传递 |
| 用户研究数据 | JSON | 按需 | 用户研究员支持 |
| 竞品分析 | JSON | 每月更新 | 竞品功能对比 |
| 用户反馈 | JSON | 实时 | 产品反馈收集 |

## 产品专业上游

| 上游资源 | 接口类型 | 说明 |
|----------|----------|------|
| 产品思维大师Agent | internal | 用户为中心的产品思维 |
| 需求分析师Agent | internal | KANO模型、需求优先级 |
| 体验设计师Agent | internal | 用户体验设计 |
| 数据分析师Agent | internal | 数据驱动决策 |
| A/B测试师Agent | internal | 实验思维验证 |

## 上游数据流

```
用户输入 → CEO路由 → 产品BG
                      ↑
                  记忆中台（历史产品数据）
                  产品专家Agent团（专业能力）
                  用户研究员（用户洞察）
```
