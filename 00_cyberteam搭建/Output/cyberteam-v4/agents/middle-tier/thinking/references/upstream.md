# upstream - 思维注入中台

## 直接上游

| 上游Agent | 关系 | 说明 |
|-----------|------|------|
| CEO | 调用依赖 | CEO路由决策后可调用思维注入 |
| PM协调器 | 调用依赖 | 任务拆解时需要专家思维 |
| 部门BG | 调用依赖 | 各BG执行任务时需要专家注入 |

## 数据依赖

| 数据源 | 格式 | 刷新频率 | 说明 |
|--------|------|----------|------|
| 专家注册表 | JSON | 实时 | 100+专家的完整信息 |
| 场景矩阵 | JSON | 定期更新 | 场景-专家匹配规则 |
| 历史效果数据 | JSON | 每次注入后更新 | 效果追踪数据 |
| 用户反馈 | JSON | 按需 | 满意度评分 |

## 专家库构成

| 类别 | 数量 | 来源 |
|------|------|------|
| 战略类专家 | 15 | 内置+外部框架 |
| 产品类专家 | 12 | 产品方法论 |
| 技术类专家 | 15 | 工程最佳实践 |
| 运营类专家 | 20 | 操盘手课程 |
| 营销类专家 | 15 | 营销体系 |
| 设计类专家 | 10 | 设计思维 |
| 财务类专家 | 8 | 财务分析 |
| 人力资源类专家 | 8 | 管理咨询 |
| 市场研究类专家 | 5 | 调研方法 |
| 决策类专家 | 7 | 决策科学 |

## 匹配算法上游

| 上游计算 | 输入 | 输出 |
|----------|------|------|
| 场景识别 | 用户Query | 场景类型 |
| 关键词提取 | 用户Query | 关键词列表 |
| 专家检索 | 场景类型 | 候选专家列表 |
| 相关度评分 | 专家+Query | 相关度分数 |
| 协同度计算 | 专家组合 | 协同度分数 |

## 上游数据流

```
用户Query → CEO路由 → 思维中台
                        ↑
                    专家库（100+专家）
                    场景矩阵（匹配规则）
                    历史效果数据（优化匹配）
```
