# upstream - 监控中台

## 直接上游

| 上游Agent | 关系 | 说明 |
|-----------|------|------|
| 所有执行Agent | 监控上报 | 各Agent上报操作历史 |
| 通信中台 | 消息传递 | 接收Agent间消息 |
| 任务系统 | 任务数据 | 获取任务状态数据 |

## 数据依赖

| 数据源 | 格式 | 刷新频率 | 说明 |
|--------|------|----------|------|
| Agent操作历史 | JSON | 实时 | 各Agent上报 |
| 任务状态数据 | JSON | 实时 | 任务系统推送 |
| 系统指标 | JSON | 5秒 | CPU/内存/网络 |
| 日志数据 | 日志流 | 实时 | Loki/ELK |

## 监控指标上游

| 指标类型 | 数据源 | 采集频率 |
|----------|--------|----------|
| Agent活跃度 | Agent心跳 | 10秒 |
| 任务进度 | 任务系统 | 实时 |
| 系统资源 | Prometheus | 5秒 |
| 日志流 | Loki | 实时 |
| 调用链 | Jaeger | 实时 |

## 外部系统依赖

| 外部系统 | 接口类型 | 说明 |
|----------|----------|------|
| Prometheus | metrics | 指标收集存储 |
| Loki | logs | 日志聚合检索 |
| Alertmanager | alerts | 告警管理 |
| Grafana | dashboards | 监控大屏 |
| Jaeger | traces | 分布式追踪 |

## 上游数据流

```
执行Agent → 操作历史 → 监控中台
    ↓                   ↑
任务系统 → 任务状态 ────┘
    ↓
Prometheus/Loki/Jaeger → 系统指标/日志/追踪
                         │
                     监控中台聚合
```
