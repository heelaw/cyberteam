# downstream - 记忆存储中台

## 直接下游

| 下游Agent | 关系 | 说明 |
|-----------|------|------|
| CEO | 上下文提供 | 路由决策时的历史上下文 |
| 所有Agent | 记忆召回 | 各Agent获取相关记忆 |
| 通信中台 | 日志存储 | 通信日志记录 |
| 监控中台 | 历史数据 | 任务历史分析 |

## 输出物

| 输出物 | 下游使用方 | 用途 |
|--------|------------|------|
| 历史上下文 | CEO/Agent | 决策参考 |
| 召回记忆 | 调用Agent | 执行参考 |
| 知识片段 | 各Agent | 知识复用 |
| 分析报告 | 监控中台 | 趋势分析 |

## 召回报告格式

```yaml
记忆召回报告:
  查询: {用户查询}
  召回数量: {数量}条
  融合方式: RRF (向量{权重} + 关键词{权重} + 关系{权重})
  Top-3结果:
    - [L{级别}] {标题} (相似度: {分数})
    - [L{级别}] {标题} (相似度: {分数})
    - [L{级别}] {标题} (相似度: {分数})
  上下文增强: +{百分比} (时间衰减+任务相关)
  召回置信度: {分数}
```

## 输出分类

### 实时召回输出

| 输出物 | 格式 | 下游使用 |
|--------|------|----------|
| 工作记忆 (L1) | MemoryItem[] | 当前任务 |
| 短期记忆 (L2) | MemoryItem[] | 项目上下文 |
| 中期记忆 (L3) | MemoryItem[] | 经验参考 |

### 知识查询输出

| 输出物 | 格式 | 下游使用 |
|--------|------|----------|
| 长期记忆 (L4) | MemoryItem[] | 知识沉淀 |
| 知识片段 | MemoryItem[] | 专业参考 |
| 最佳实践 | MemoryItem[] | 执行指导 |

### 分析报告输出

| 输出物 | 格式 | 下游使用 |
|--------|------|----------|
| 记忆使用统计 | JSON | 系统分析 |
| 召回质量报告 | JSON | 效果评估 |
| 知识图谱 | JSON | 关系分析 |

## 记忆流转下游

```
记忆中台存储:
    ↓
    ├── L1工作记忆 → 当前会话 → 会话结束 → 评估升级
    ├── L2短期记忆 → 7天保留 → 访问≥3次 → 升级L3
    ├── L3中期记忆 → 90天保留 → 重要性>0.8 → 升级L4
    └── L4长期记忆 → 永久保留 → 知识库索引

记忆中台召回:
    ↓
    ├── L1 → 实时任务上下文
    ├── L2 → 项目进行中参考
    ├── L3 → 经验积累复用
    └── L4 → 知识沉淀查询
```
