# upstream - 通信协作中台

## 直接上游

| 上游Agent | 关系 | 说明 |
|-----------|------|------|
| 所有Agent | 消息发送 | 各Agent发送消息的通道 |
| CEO | 跨部门协调 | 重大任务需要通信支持 |
| PM协调器 | 任务协同 | 多部门协作的通信需求 |

## 数据依赖

| 数据源 | 格式 | 刷新频率 | 说明 |
|--------|------|----------|------|
| Agent注册表 | JSON | 实时 | Agent可达性状态 |
| 消息队列 | 队列 | 实时 | 消息缓冲 |
| Handoff状态 | JSON | 实时 | 当前所有Handoff状态 |
| 协作模板 | YAML | 定期 | 预设协作流程 |

## Agent通信协议上游

| 协议 | 说明 | 参考 |
|------|------|------|
| CyberTeam Handoff | 任务交接协议 | CyberTeam官方 |
| FIPA ACL | Agent通信语言 | FIPA标准 |
| WebSocket | 实时通信 | W3C标准 |

## 传输层上游

| 组件 | 说明 |
|------|------|
| WebSocket | 实时双向通信 |
| Message Queue | 异步消息 |
| Redis Pub/Sub | 发布订阅 |

## 上游数据流

```
执行Agent → 发送消息 → 通信中台
    ↓                   ↑
任务系统 → 协作请求 ────┘
    ↓
消息队列/Redis → 消息缓冲
                    │
                通信中台处理
```
