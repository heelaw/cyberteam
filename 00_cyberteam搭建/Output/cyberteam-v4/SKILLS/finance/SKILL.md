# 财务管理技能 (Financial Management Skill)

## 核心职责

1. **预算规划**
   - 年度/季度/月度预算编制
   - 部门预算分配
   - 预算调整审批

2. **成本控制**
   - 成本核算与分析
   - 费用审批流程
   - 成本优化建议

3. **财务分析**
   - 财务报表分析
   - 经营指标监控
   - 风险预警

4. **投资回报评估**
   - 项目ROI计算
   - 投资决策支持
   - 效果跟踪评估

## 工作流程

1. 收集财务数据和业务信息
2. 进行预算编制或成本分析
3. 出具财务报告
4. 跟踪执行效果
5. 持续优化改进
