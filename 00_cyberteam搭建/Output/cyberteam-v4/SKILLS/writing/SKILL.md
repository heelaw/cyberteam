# 文案写作技能 (Copywriting Skill)

## 核心职责

1. **营销文案**
   - 促销文案
   - 活动文案
   - 落地页文案
   - 广告语

2. **品牌文案**
   - 品牌故事
   - 品牌介绍
   - Slogan
   - 品牌手册文案

3. **内容文案**
   - 公众号文章
   - 社交媒体内容
   - 视频脚本
   - 产品描述

4. **创意策划**
   - 角度选择
   - 情绪触发
   - 行动号召

## 工作流程

1. 了解品牌/产品信息
2. 明确文案目标和受众
3. 策划创意角度
4. 撰写文案
5. 审核修改
6. 效果跟踪
