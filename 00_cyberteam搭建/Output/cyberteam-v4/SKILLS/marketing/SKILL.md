# 市场营销技能 (Marketing Strategy Skill)

## 核心职责

1. **品牌建设**
   - 品牌定位
   - 品牌故事
   - 视觉规范

2. **渠道推广**
   - 渠道选择与评估
   - 投放策略
   - 效果追踪

3. **内容营销**
   - 内容规划
   - 素材创作
   - 分发策略

4. **营销活动**
   - 活动策划
   - 预热引爆
   - 效果复盘

## 工作流程

1. 明确营销目标
2. 分析市场和用户
3. 制定营销策略
4. 执行落地
5. 效果评估与优化
