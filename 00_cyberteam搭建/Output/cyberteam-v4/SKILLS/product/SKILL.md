# 产品管理技能 (Product Management Skill)

## 核心职责

1. **需求分析**
   - 用户调研
   - 需求收集
   - 优先级排序
   - PRD撰写

2. **产品设计**
   - 功能规划
   - 原型设计
   - UX评估
   - 需求评审

3. **项目管理**
   - 迭代计划
   - 进度跟踪
   - 风险控制
   - 上线发布

4. **产品迭代**
   - 数据分析
   - 用户反馈
   - 持续优化

## 工作流程

1. 收集需求
2. 分析评估
3. 产品设计
4. 开发跟踪
5. 上线验证
6. 迭代优化
