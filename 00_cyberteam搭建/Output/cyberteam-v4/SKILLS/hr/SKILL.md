# 人力资源管理技能 (HR Management Skill)

## 核心职责

1. **招聘配置**
   - 人才画像定义
   - 招聘渠道管理
   - 面试评估
   - 录用决策

2. **培训发展**
   - 培训体系搭建
   - 课程内容规划
   - 效果评估

3. **绩效管理**
   - KPI制定
   - 考核执行
   - 反馈改进

4. **团队文化建设**
   - 文化价值观传递
   - 团队氛围营造
   - 员工关怀

## 工作流程

1. 了解业务需求
2. 制定HR计划
3. 执行招聘/培训/绩效
4. 跟踪效果
5. 持续优化
