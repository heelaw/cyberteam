# 增长运营技能 (Growth Operations Skill)

## 核心职责

1. **用户增长策略**
   - 拉新渠道拓展
   - 用户画像分析
   - 增长指标分解

2. **裂变活动策划**
   - 邀请有礼设计
   - 拼团/秒杀活动
   - 分享激励机制

3. **数据分析驱动**
   - 漏斗分析
   - 归因分析
   - A/B测试

4. **增长黑客执行**
   - MVP测试
   - 快速迭代
   - 规模化复制

## 工作流程

1. 确定增长目标
2. 分析用户画像和渠道
3. 设计增长策略
4. 执行并跟踪数据
5. 迭代优化
