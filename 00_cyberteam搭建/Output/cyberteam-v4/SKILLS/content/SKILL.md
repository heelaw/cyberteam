# 内容运营技能 (Content Operations Skill)

## 核心职责

1. **内容配比规划**
   - PGC(专业生成内容): 60%
   - UGC(用户生成内容): 30%
   - AIGC(AI生成内容): 10%

2. **选题三叉模型**
   - 热点话题: 30%
   - 用户需求: 30%
   - 数据驱动: 40%

3. **内容分类运营**
   - 流量型: 引流获客
   - 粘性型: 促活留存
   - 转化型: 付费变现

## 工作流程

1. 分析业务阶段和目标
2. 制定内容配比策略
3. 执行选题策划
4. 创作/审核内容
5. 发布并跟踪数据
6. 复盘优化
