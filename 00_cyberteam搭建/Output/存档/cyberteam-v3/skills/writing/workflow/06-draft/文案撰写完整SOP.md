## 1. 准备输入材料

### 1.1 必读输入文档

| 文档 | 来源Stage | 核心内容 | 必读要点 |
|------|-----------|----------|----------|
| **核心信息屋** | Stage 4 | WHO+WHAT+HOW策略 | 目标用户画像、核心卖点、说服模式 |
| **痛点-卖点匹配矩阵** | Stage 3 | 痛点与卖点对应关系 | 优先级排序、说服逻辑、支撑证据 |
| **内容结构决策卡** | Stage 5 | 渠道策略、结构模型 | 推荐结构、字数要求、平台规范 |

### 1.2 输入材料核对清单

```
□ 《核心信息屋》是否完整？
  ├─ 目标用户画像清晰
  ├─ 核心卖点明确（P0/P1/P2）
  ├─ 需求强度判断正确
  └─ 说服模式已选定

□ 《痛点-卖点匹配矩阵》是否完整？
  ├─ 痛点优先级已排序
  ├─ 匹配卖点已对应
  ├─ 说服逻辑已撰写
  └─ 支撑证据已准备

□ 《内容结构决策卡》是否完整？
  ├─ 目标渠道已确定
  ├─ 推荐结构已选定
  ├─ 字数要求已明确
  └─ 平台规范已了解
```

### 1.3 信息整合表

| 维度 | 信息要点 | 在文案中的位置 |
|------|----------|----------------|
| 用户 | 目标人群/痛点/期望 | 开头痛点/共鸣段落 |
| 产品 | 核心卖点/差异化/证据 | 主体内容/解决方案 |
| 说服 | 选择的说服模式 | 整体逻辑框架 |
| 结构 | 选定的内容结构 | 文案组织方式 |
| 渠道 | 平台特点/字数要求 | 表达风格/长度控制 |

## 2. 写作流程与时间分配（1000-1500字为例）

### 2.1 黄金时间分配法

| 阶段 | 时间 | 占比 | 任务 | 产出物 |
|------|------|------|------|--------|
| **1. 定结构列大纲** | 5分钟 | 5% | 搭框架、定要点 | 大纲草稿 |
| **2. 写标题与开头** | 15分钟 | 15% | 吸引注意、激发兴趣 | 标题+开头 |
| **3. 填充主体内容** | 60分钟 | 60% | 展开论证、呈现价值 | 主体段落 |
| **4. 撰写结尾与CTA** | 10分钟 | 10% | 总结收尾、推动行动 | 结尾+CTA |
| **5. 初稿通读与微调** | 10分钟 | 10% | 检查逻辑、润色表达 | 定稿初稿 |
| **总计** | **100分钟** | 100% | - | - |

### 2.2 各阶段执行要点

#### Stage 1: 定结构列大纲（5分钟）

**执行动作**:
1. 回顾选定结构模板（AIDA/4P/情景问题方案等）
2. 根据模板列出段落大纲
3. 标注每段的核心任务

**大纲模板**:
```
【标题】
    ↓
【开头钩子】- 吸引注意
    ↓
【痛点共鸣】- 戳中痛点
    ↓
【解决方案】- 展示价值
    ↓
【证据支撑】- 建立信任
    ↓
【差异化说明】- 消除疑虑
    ↓
【结尾总结】- 强化记忆
    ↓
【CTA】- 推动行动
```

#### Stage 2: 写标题与开头（15分钟）

**执行动作**:
1. 先写5个标题备选
2. 选择最优标题
3. 撰写开头钩子

**质量标准**:
- 标题：3秒内抓住注意力
- 开头：5秒内激发继续阅读欲望

#### Stage 3: 填充主体内容（60分钟）

**执行动作**:
1. 按照大纲逐段撰写
2. 每段聚焦一个核心信息
3. 保持段落间的逻辑连贯

**质量标准**:
- 每个卖点有FAB支撑
- 每个痛点有共鸣表达
- 证据有数据/案例背书

#### Stage 4: 撰写结尾与CTA（10分钟）

**执行动作**:
1. 总结核心价值主张
2. 选择合适CTA类型
3. 撰写引导性结尾

**质量标准**:
- 结尾强化记忆点
- CTA明确具体可操作

#### Stage 5: 初稿通读与微调（10分钟）

**执行动作**:
1. 通读全文，检查流畅度
2. 检查逻辑连贯性
3. 润色表达、删减冗余
4. 最终定稿

**检查清单**:
```
□ 阅读是否流畅？
□ 逻辑是否连贯？
□ 卖点是否突出？
□ 证据是否充分？
□ CTA是否明确？
□ 字数是否达标？
```

## 4. 标题撰写六公式

### 4.1 痛点型

**公式**: [痛点人群] + [痛点描述] + [解决方案暗示]

**示例**:
| 痛点人群 | 痛点描述 | 解决方案暗示 | 完整标题 |
|----------|----------|--------------|----------|
| 皮肤暗沉的女生 | 卸妆后不敢照镜子 | 用了XX之后 | "皮肤暗沉的女生，卸妆后敢照镜子了" |
| 手残党 | 画眼线手抖 | 一招搞定 | "手残党画眼线，一招搞定" |
| 减肥人 | 饿得难受还瘦不了 | 用这个方法 | "减肥饿得难受？用这个方法吃饱也能瘦" |

### 4.2 数字型

**公式**: [数字] + [核心价值] + [目标人群/场景]

**示例**:
| 数字 | 核心价值 | 目标人群 | 完整标题 |
|------|----------|----------|----------|
| 3天 | 皮肤变好 | 爱美女生 | "3天皮肤变好，她是怎么做到的？" |
| 50万+ | 用户选择 | 妈妈们 | "50万+妈妈的选择，这个辅食真的好吗？" |
| 1招 | 解决XX问题 | 所有人 | "1招解决90%的XX问题" |

### 4.3 悬念型

**公式**: [反常识陈述] + [悬念词] + [悬念点]

**示例**:
| 反常识陈述 | 悬念词 | 悬念点 | 完整标题 |
|------------|--------|--------|----------|
| XX竟然是错的 | 很多人还在 | 继续犯错 | "护肤这么多年，XX竟然是错的，很多人还在继续" |
| 不是XX而是XX | 真相 | 颠覆认知 | "皮肤不好，不是护肤品的问题，而是..." |
| XX的秘密 | 终于 | 被发现了 | "大S美白的秘密，终于被发现了" |

### 4.4 情绪型

**公式**: [情绪词] + [场景/事件] + [情感共鸣]

**示例**:
| 情绪词 | 场景/事件 | 情感共鸣 | 完整标题 |
|--------|-----------|----------|----------|
| 太后悔了 | 没早点用这个 | - | "太后悔了，没早点用这个精华" |
| 哭死 | 终于找到 | 平价替代 | "哭死！终于找到XX大牌平价替代" |
| 救命 | 皮肤问题 | 终于解决 | "救命！困扰我3年的皮肤问题终于解决了" |

### 4.5 身份型

**公式**: [目标身份] + [需求/痛点] + [解决方案]

**示例**:
| 目标身份 | 需求/痛点 | 解决方案 | 完整标题 |
|----------|-----------|----------|----------|
| 职场妈妈 | 早起没时间化妆 | 一招搞定 | "职场妈妈早起没时间化妆？一招搞定" |
| 油皮星人 | 夏天脱妆 | 必入 | "油皮星人夏天脱妆？这款粉底必入" |
| 30+女性 | 抗老 | 平价好物 | "30+女性抗老，这款平价好物值得入" |

### 4.6 对比型

**公式**: [A] vs [B] + [差异点] + [结论]

**示例**:
| A | B | 差异点 | 结论 | 完整标题 |
|---|---|--------|------|----------|
| 普通XX | 用了XX之后 | 差距 | 太大了 | "普通XX vs 用了XX之后，差距太大了" |
| XX年前 | XX年后 | 对比照 | 逆生长 | "XX岁 vs XX岁，她的逆生长秘诀是..." |
| 别人 | 你 | 差距 | 在哪里 | "别人XX，你却XX，差距在哪里？" |

## 5. 四种开头钩子

### 5.1 痛点钩

**公式**: 描述痛点场景 + 引发共鸣

**模板**:
```
[具体场景描述] + [痛点表达] + [情绪共鸣]

示例：
"周一早上7点，闹钟响了第三遍才勉强爬起来。
看着镜子里那张暗沉的脸，叹了口气：
'又是一个没脸见人的早晨。'
```

**适用场景**: 种草内容、情感营销

### 5.2 数字钩

**公式**: 惊人数字 + 引发好奇

**模板**:
```
[惊人数字/统计] + [核心发现/结论]

示例：
"90%的女生都不知道：
护肤最重要的不是用什么，
而是在什么时间用。"
```

**适用场景**: 知识科普、决策辅助

### 5.3 故事钩

**公式**: 真实故事 + 制造冲突

**模板**:
```
[身份] + [经历] + [冲突/转折]

示例：
"我叫小雨，曾经是个月薪3000的普通上班族。
用了3年时间，我终于实现了月入5万。
但最让我骄傲的，不是收入的变化，
而是终于找到了自己喜欢的生活方式。"
```

**适用场景**: 品牌故事、个人IP内容

### 5.4 冲突钩

**公式**: 反常识/违背直觉 + 引发好奇

**模板**:
```
[反常识陈述] + [制造悬念]

示例：
"护肤品不是越贵越好。
恰恰相反，我用过的最有效的护肤品，
价格还不到大牌的1/3。"
```

**适用场景**: 爆款内容、社交媒体

## 6. 四种CTA类型

### 6.1 直接CTA

**适用场景**: 高意向用户、促销场景

**公式**: 明确行动 + 简化路径

**示例**:
| 类型 | 示例 |
|------|------|
| 购买类 | "立即下单" / "点击购买" |
| 咨询类 | "立即咨询" / "了解更多" |
| 关注类 | "立即关注" / "点击关注" |

**优化技巧**:
- 按钮文案具体化：❌"点击这里" → ✅"立即领取30元优惠券"
- 降低行动门槛：❌"立即购买" → ✅"先加购再考虑"

### 6.2 场景CTA

**适用场景**: 场景化内容、种草内容

**公式**: 场景植入 + 行动引导

**示例**:
| 场景 | CTA示例 |
|------|---------|
| 睡前场景 | "睡前记得点开链接，给自己一个惊喜" |
| 早晨场景 | "明天早上洗漱时，记得来试试这个方法" |
| 约会前场景 | "约会前2小时用它，皮肤状态好到哭" |

### 6.3 风险逆转CTA

**适用场景**: 高决策成本、信任建立期

**公式**: 消除顾虑 + 保障承诺

**示例**:
| 类型 | CTA示例 |
|------|---------|
| 无条件退换 | "不满意30天无理由退款，零风险尝试" |
| 效果保证 | "7天无效全额退款，我们说到做到" |
| 运费险 | "下单即送运费险，不满意免费退" |

### 6.4 稀缺紧迫CTA

**适用场景**: 促销节点、限时活动

**公式**: 限时/限量 + 利益点

**示例**:
| 类型 | CTA示例 |
|------|---------|
| 限时优惠 | "仅剩48小时，错过再等一年" |
| 限量抢购 | "今日仅剩50份，手慢无" |
| 早鸟价 | "前100名下单，立减100元" |

## 7. 各阶段质量标准

### 7.1 标题质量标准

| 检查项 | 优秀 | 合格 | 不合格 |
|--------|------|------|--------|
| 注意力 | 3秒内被抓 | 5秒内被抓 | 超过5秒 |
| 目标人群 | 精准命中 | 基本命中 | 模糊宽泛 |
| 价值承诺 | 明确具体 | 有所表达 | 模糊不清 |
| 独特性 | 差异化明显 | 有一定差异 | 同质化严重 |

### 7.2 开头质量标准

| 检查项 | 优秀 | 合格 | 不合格 |
|--------|------|------|--------|
| 注意力延续 | 吸引继续读 | 能读下去 | 直接跳出 |
| 痛点共鸣 | 强烈共鸣 | 基本共鸣 | 无共鸣 |
| 逻辑衔接 | 自然过渡 | 基本衔接 | 割裂感 |
| 字数控制 | 恰到好处 | 可接受 | 过长/过短 |

### 7.3 主体质量标准

| 检查项 | 优秀 | 合格 | 不合格 |
|--------|------|------|--------|
| 卖点呈现 | 突出有力 | 基本呈现 | 不突出 |
| 证据支撑 | 充分有力 | 有一定支撑 | 空洞宣称 |
| 逻辑连贯 | 层层递进 | 基本连贯 | 逻辑混乱 |
| 可读性 | 流畅易懂 | 可读 | 晦涩难懂 |

### 7.4 CTA质量标准

| 检查项 | 优秀 | 合格 | 不合格 |
|--------|------|------|--------|
| 明确性 | 具体清晰 | 基本明确 | 模糊笼统 |
| 行动性 | 立即想动 | 可以行动 | 不知如何行动 |
| 合理性 | 恰到好处 | 可接受 | 过于强硬 |
| 紧迫感 | 适度合理 | 基本适度 | 过度紧迫 |
