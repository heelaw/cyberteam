# 社交媒体平台写作指南

> **版本**: v3.1
> **更新日期**: 2026-03-24

---

## 1. 小红书平台

### 核心特点

- **种草平台**: 用户主动寻找消费建议
- **视觉主导**: 图片/视频质量决定点击率
- **女性为主**: 20-35岁女性占比高
- **内容标签化**: 话题标签增加曝光

### 内容策略

**标题**: 数字+痛点+解决方案
> "3个方法解决熬夜暗沉肌"

**结构**:
1. 痛点/问题
2. 解决方案/产品介绍
3. 使用效果/测评
4. 总结/购买建议

**标签**: 10-15个，包括热门标签和精准标签

### 发布规范

| 项目 | 规范 |
|------|------|
| **封面** | 3:4 (1242x1660), JPG/PNG |
| **正文** | ≤2000字, Markdown |
| **标签** | 3-10个, #话题#格式 |
| **标题** | ≤20字, 有吸引力 |

## 2. 微信公众号

### 核心特点

- **深度阅读**: 用户有耐心阅读长文
- **粉丝忠诚度高**: 关注即认可
- **内容形式多样**: 图文、音频、视频

### 内容策略

**标题**: 悬念式、干货式、数字式

**结构**: 清晰段落、小标题、图文结合

**长度**: 800-2000字，深度内容可更长

### 发布规范

| 项目 | 规范 |
|------|------|
| **封面** | 2.35:1 (900x383), JPG/PNG |
| **标题** | ≤64字, ≤36字符 |
| **正文** | Markdown, 无字数限制 |
| **图片** | ≤5MB, 建议≤2MB |

---

## 3. 微博平台

### 核心特点

- **信息快速传播**: 热点发酵地
- **用户广泛**: 年轻用户多
- **互动性强**: 评论、转发、点赞

### 内容策略

**标题/开头**: 直接、有冲击力

**长度**: 140-280字最佳

**话题标签**: 2-5个，包括热门话题

### 发布规范

| 项目 | 规范 |
|------|------|
| **正文** | ≤140字 (长文除外) |
| **图片** | 1:1/16:9, ≤9张 |
| **话题** | #话题#, 2-5个 |
| **发布时间** | 7-9点, 12-13点, 18-21点 |

---

## 4. 抖音/快手平台

### 核心特点

- **短视频主导**: 15秒-3分钟视频
- **算法推荐**: 内容质量决定曝光
- **即时性**: 热点内容快速传播

### 内容策略

**开头3秒**: 必须抓住注意力
> - 痛点: "夏天出油脱妆的姐妹看过来！"
> - 悬念: "原来我们都做错了！"
> - 反差: "同一个产品，她用就是奶油肌！"

**内容节奏**: 快节奏，信息密度高

**文案辅助**:
- 标题: 简短有力（10-20字）
- 字幕: 关键信息用字幕突出
- 评论区: 引导互动，补充信息

### 发布规范

| 项目 | 规范 |
|------|------|
| **时长** | 15-60秒 |
| **字幕** | 必须有 |
| **BGM** | 热门音乐 |
| **CTA** | 直接、明显 |

## 5. LinkedIn平台

### 核心特点

- **职业社交**: 专业、商务
- **用户**: 职场人士、企业家
- **风格**: 正式、专业、有深度

### 内容策略

**内容类型**:
- 行业洞察
- 经验分享
- 专业知识
- 职业建议

**风格**: 理性、客观、有深度

### 发布规范

| 项目 | 规范 |
|------|------|
| **正文** | 1500-3000字 |
| **结构** | 小标题、要点列表 |
| **CTA** | 关注、连接、访问网站 |

## 6. 内容类型模板

### 6.1 痛点解决型

```
【痛点】: [描述用户的问题]
【原因】: [分析问题产生的原因]
【解决方案】: [介绍产品/方法]
【效果】: [展示效果、数据]
【CTA】: [引导行动]
```

### 6.2 产品测评型

```
【产品介绍】: [产品基本信息]
【外观设计】: [产品外观、包装]
【使用体验】: [质地、效果、感受]
【优点】: [产品优势]
【不足】: [客观指出不足]
【总结】: [是否推荐]
【CTA】: [引导行动]
```

### 6.3 故事型

```
【场景】: [描述场景]
【冲突】: [遇到的问题]
【转折】: [发现解决方案]
【结果】: [使用效果]
【感悟】: [从中获得的启发]
【CTA】: [引导行动]
```

## 7. 写作检查清单

- [ ] 标题/开头是否吸引人？
- [ ] 是否用"你"称呼用户？
- [ ] 是否有明确的痛点？
- [ ] 内容是否简洁有力？
- [ ] 是否有明确的CTA？
- [ ] 是否适合平台风格？
- [ ] 是否有互动引导？
- [ ] 是否有视觉元素配合？
