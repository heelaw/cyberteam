# 《数据驱动 Skill 体系》

版本：v3.1
更新日期：2026-03-25

---

## 1. 定位

为数据驱动决策提供系统化的产品评估、问题诊断和效果评估能力，支撑精细化运营。

---

## 2. 三个子技能

### 2.1 产品状态评估

- **功能**：产品核心指标评估、生命周期阶段判断、竞争力分析
- **输出**：产品状态评估报告
- **路径**：`产品状态评估/SKILL.md`

### 2.2 产品问题诊断

- **功能**：问题根因分析、关键因素识别、优化方向建议
- **输出**：问题诊断报告及解决方案
- **路径**：`产品问题诊断/SKILL.md`

### 2.3 活动效果评估

- **功能**：活动数据追踪、效果指标分析、ROI计算、改进建议
- **输出**：活动效果评估报告
- **路径**：`活动效果评估/SKILL.md`

---

## 3. 调用方式

```
/data eval-product <产品信息>
/data diagnose <问题描述>
/data eval-campaign <活动数据>
```

---

## 4. 评估维度

| 维度 | 说明 |
|------|------|
| 用户指标 | DAU/MAU、留存率、活跃度 |
| 商业指标 | 转化率、ARPU、LTV |
| 产品指标 | 功能使用率、用户满意度 |
