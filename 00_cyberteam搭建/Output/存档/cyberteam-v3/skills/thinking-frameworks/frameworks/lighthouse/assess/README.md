# 灯塔效应专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式，分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 灯塔效应理论是否正确（清晰性、稳定性、可及性、持久性、利他性） |
| 可执行性 | 30% | 输出是否可操作（灯塔特征自检、迷茫来源识别、灯塔建设计划是否明确） |
| 独特性 | 20% | 是否有差异化（与一般领导力模型的区分、灯塔特征体系） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（情境、角色、团队状态、迷茫程度）
- [ ] 有输出格式（灯塔自检、迷茫来源识别、灯塔建设计划、长期维护）
- [ ] 有分析框架（四步法：灯塔特征自检→识别迷茫来源→建立灯塔式领导力→长期灯塔维护）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（SWOT、GROW、系统思维）

## 核心检验点

### 理论准确性
- [ ] 灯塔四特征（清晰性、稳定性、可及性、持久性）正确
- [ ] 迷茫类型识别正确
- [ ] 灯塔失效警示信号有效

### 框架完整性
- [ ] 灯塔自检有评分
- [ ] 迷茫来源识别有清单
- [ ] 建设计划有具体行动

### 输出有效性
- [ ] 自我检查机制明确
- [ ] 立即行动可执行
- [ ] 灯塔失效信号可识别

## 相关专家

- 灯塔效应与SWOT：帮助团队找到方向
- 灯塔效应与GROW：目标达成式领导
