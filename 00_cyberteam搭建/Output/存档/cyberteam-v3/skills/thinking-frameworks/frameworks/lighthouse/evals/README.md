# 灯塔效应专家测试用例

## 正面测试

### 测试1: 新任经理
- 输入:
```
【灯塔效应分析请求】
情境：刚升职为团队经理
角色：新任管理者
团队状态：迷茫，不知道方向
迷茫程度：中度
```
- 期望输出:
  - 灯塔自检评分
  - 迷茫来源识别
  - 灯塔建设计划
- 验证项:
  - [ ] 有灯塔特征自检
  - [ ] 有迷茫来源分析
  - [ ] 有具体建设计划

### 测试2: 危机领导
- 输入:
```
【灯塔效应分析请求】
情境：公司面临危机，员工士气低落
角色：CEO
团队状态：焦虑，迷茫，不知道公司还能撑多久
迷茫程度：重度
```
- 期望输出:
  - 灯塔自检
  - 危机应对策略
  - 稳定军心计划
- 验证项:
  - [ ] 有清晰性建设建议
  - [ ] 有稳定性建设建议
  - [ ] 有立即可执行行动

## 负面测试

### 测试1: 模糊表达
- 输入:
```
【灯塔效应分析请求】
情境：需要给团队指引方向
角色：管理者
团队状态：等待指示
迷茫程度：轻度
```
- 期望行为: 指引必须清晰明确
- 验证项:
  - [ ] 禁止"大概"、"可能"类模糊表达
  - [ ] 必须给出清晰的指引
  - [ ] 必须有可执行的具体指示

### 测试2: 情绪波动
- 输入:
```
【灯塔效应分析请求】
情境：压力下情绪不稳定
角色：管理者
团队状态：观察管理者情绪
迷茫程度：（观察中）
```
- 期望行为: 管理者必须情绪稳定
- 验证项:
  - [ ] 禁止情绪波动
  - [ ] 必须保持稳定
  - [ ] 必须有情绪管理建议
