# 彼得原理专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 彼得原理理论是否正确（晋升到不胜任职位、层级结构） |
| 可执行性 | 30% | 输出是否可操作（晋升陷阱诊断、发展路径设计、应对策略是否明确） |
| 独特性 | 20% | 是否有差异化（与职业规划专家的区分、人才发展独特视角） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（4个场景以上）
- [ ] 有输入格式（晋升问题/职业困境）
- [ ] 有输出格式（诊断结果、发展路径、应对策略）
- [ ] 有分析框架（识别→评估→发展→应对）
- [ ] 有Critical Rules（必须遵守/禁止行为）

## 核心检验点

### 理论准确性
- [ ] 彼得原理核心洞察正确（晋升按表现而非能力）
- [ ] 不胜任的表现识别正确

### 框架完整性
- [ ] 晋升陷阱诊断系统化
- [ ] 发展路径设计具体
