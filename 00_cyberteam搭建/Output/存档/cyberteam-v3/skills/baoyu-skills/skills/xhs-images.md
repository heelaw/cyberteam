# baoyu-skills - 小红书系列图生成

> **版本**: v3.1
> **分类**: 图像生成
> **技能**: baoyu-xhs-images
> **触发词**: 小红书系列图, xhs-images, 小红书图片, 系列图, 种草图

---

## ⚠️ 铁律

1. **叙事连贯**: 系列图必须有完整的叙事逻辑
2. **风格统一**: 所有图片必须保持风格一致
3. **平台规范**: 必须符合小红书平台规范
4. **版权合规**: 确保图像和字体不侵犯版权

---

## 一、技能说明

### 1.1 功能概述

生成小红书系列图（1-9张），用于笔记内容的配图，支持完整的叙事逻辑和风格统一。

### 1.2 与 baoyu-cover-image 的区别

| 技能 | 数量 | 用途 | 特点 |
|------|------|------|------|
| **baoyu-cover-image** | 1张 | 单一封面 | 重点突出 |
| **baoyu-xhs-images** | 1-9张 | 系列配图 | 叙事完整 |

### 1.3 推荐组合

| 数量 | 适用场景 | 叙事结构 |
|------|----------|----------|
| **1张** | 强调型内容 | 单点突出 |
| **3张** | 快速种草 | 问题→方案→结果 |
| **5张** | 完整种草 | 起承转合 |
| **9张** | 详细教程/攻略 | 完整展开 |

## 二、调用参数

### 2.1 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `title` | string | 是 | 系列图主题 |
| `--count` | number | 否 | 数量 (1-9, 默认3) |
| `--aspect` | string | 否 | 比例: 3:4/16:9/1:1 |
| `--style` | string | 否 | 风格: modern/minimal/vibrant/lifestyle |
| `--publish` | boolean | 否 | 是否直接发布 |
| `--narrative` | string | 否 | 叙事逻辑描述 |
| `--output` | string | 否 | 输出路径 |

### 2.2 参数示例

```bash
# 生成5张系列图
baoyu-xhs-images "AI写作神器使用教程" --count 5

# 生成种草型系列图
baoyu-xhs-images "用了这款粉底液，我后悔了" --count 3 --style lifestyle

# 生成并发布
baoyu-xhs-images "新手化妆教程" --count 9 --publish

# 指定叙事逻辑
baoyu-xhs-images "我的护肤routine" --count 5 --narrative "晨间-产品介绍-使用手法-效果展示-总结"
```

## 三、执行流程

```
1. 明确内容主题
   └── 确定系列图要表达的核心内容

2. 规划叙事结构
   └── 设计故事的起承转合

3. 确定图片数量
   └── 根据内容复杂度选择 3/5/9 张

4. 设计风格统一
   └── 确定整体视觉风格

5. 生成系列图
   └── 调用 baoyu-xhs-images

6. 审核质量
   ├── 单图质量
   ├── 系列连贯性
   └── 文字清晰度

7. 发布 (可选)
   └── 直接发布到小红书
```

## 五、设计规范

### 5.1 小红书平台规范

| 项目 | 规范 |
|------|------|
| **比例** | 3:4 (竖版) |
| **尺寸** | 1242x1660 px |
| **格式** | JPG/PNG |
| **大小** | ≤5MB |
| **分辨率** | 72 DPI |
| **数量** | 1-9张 |

### 5.2 文字设计

| 位置 | 内容 | 字号 | 颜色 |
|------|------|------|------|
| **首图** | 吸引人的标题 | ≥ 48pt | 白色/对比色 |
| **内容图** | 要点/说明 | ≥ 32pt | 深色 |
| **尾图** | CTA/引导 | ≥ 36pt | 品牌色 |

### 5.3 风格指南

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **modern** | 简洁、现代、科技感 | 科技产品、干货分享 |
| **minimal** | 极简、大量留白 | 生活方式、高端品牌 |
| **vibrant** | 活泼、色彩丰富 | 美妆、时尚、生活 |
| **lifestyle** | 真实、生活场景 | 真实分享、种草 |
