# 微信公众号平台规范

> **版本**: v3.1
> **平台**: 微信公众号 (WeChat Official Account)
> **更新日期**: 2026-03-24

---

## 一、平台概述

### 1.1 平台特点

| 特点 | 说明 |
|------|------|
| **用户粘性** | 粉丝关注后更容易看到内容 |
| **深度阅读** | 用户愿意阅读长篇内容 |
| **信任度高** | 公众号内容信任度较高 |
| **变现路径** | 广告、带货、付费订阅 |

### 1.2 内容形式

| 形式 | 字数 | 适用场景 |
|------|------|----------|
| **图文** | 1000-3000字 | 深度内容、干货分享 |
| **纯文字** | 500-1000字 | 短评、观点 |
| **图片** | 9张图 | 视觉内容 |
| **视频** | 1-10分钟 | 教程、演示 |
| **音频** | 3-10分钟 | 知识讲解 |

---

## 二、封面图规范

### 2.1 封面图规格

| 项目 | 规格 |
|------|------|
| **比例** | 2.35:1 (电影比例) |
| **最小尺寸** | 900x383 px |
| **建议尺寸** | 1080x460 px |
| **格式** | JPG/PNG |
| **文件大小** | ≤5MB |
| **DPI** | 72 DPI |

### 2.2 封面图设计

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                                                             │
│                    封面标题文字                              │
│                                                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**设计要点**:
- 标题居中或偏左
- 字号 ≥ 36pt
- 使用对比色
- 背景可略复杂
- 注意手机端显示效果

---

## 三、标题规范

### 3.1 标题长度

| 长度 | 字数 | 效果 |
|------|------|------|
| **最佳** | 14-20字 | 显示完整，有吸引力 |
| **较短** | < 14字 | 可能被折叠 |
| **较长** | > 20字 | 可能显示不全 |

### 3.2 标题类型

| 类型 | 示例 | 点击率 |
|------|------|--------|
| **悬念型** | "没想到，这个方法竟然这么有效..." | ⭐⭐⭐⭐⭐ |
| **数字型** | "2024年，最值得学习的5个技能" | ⭐⭐⭐⭐ |
| **情绪型** | "看完这篇，我终于明白了！" | ⭐⭐⭐⭐ |
| **干货型** | "[领域]完整指南，建议收藏" | ⭐⭐⭐⭐ |
| **热点型** | "[热点事件]深度解析" | ⭐⭐⭐⭐⭐ |

---

## 四、正文规范

### 4.1 内容长度

| 类型 | 字数 | 适用场景 |
|------|------|----------|
| **短文** | 500-1000字 | 快讯、观点 |
| **中等** | 1000-2000字 | 干货分享 |
| **长文** | 2000-5000字 | 深度分析 |
| **超长文** | > 5000字 | 完整教程 |

### 4.2 排版规范

**段落规范**:
- 每段 3-5 行
- 段落之间空一行
- 避免大段文字

**字体规范**:
- 正文字号: 15-16pt
- 标题字号: 18-22pt
- 颜色: 正文 #3a3a3a，标题 #000000

**格式支持**:
```
✅ 支持的格式:
  - 标题 (H1-H3)
  - 粗体、斜体
  - 有序、无序列表
  - 引用
  - 代码块
  - 表格
  - 图片
  - 链接

❌ 不支持的格式:
  - 脚注
  - 任务列表
  - 目录
```

## 五、图片规范

### 5.1 图片规格

| 项目 | 规格 |
|------|------|
| **宽度** | 建议 900-1080px |
| **高度** | 自动 |
| **文件大小** | ≤5MB，建议 ≤2MB |
| **格式** | JPG/PNG |
| **数量** | 无限制，建议 ≤10张 |

### 5.2 图片位置

| 位置 | 用途 | 要求 |
|------|------|------|
| **文章开头** | 引导图 | 高清美观 |
| **文章中间** | 配图 | 说明内容 |
| **文章结尾** | 二维码/引导图 | 清晰可见 |

## 六、标签与分类

### 6.1 标签规范

| 项目 | 规范 |
|------|------|
| **数量** | 最多 3 个 |
| **选择** | 相关性 + 热度 |
| **格式** | 话题标签 |

### 6.2 文章分类

- 根据内容选择合适的公众号分类
- 每个公众号可设置 3 个默认分类

## 七、原创与声明

### 7.1 原创声明

| 项目 | 说明 |
|------|------|
| **开启条件** | 文章为本人原创 |
| **字数要求** | 通常 ≥ 300 字 |
| **好处** | 更多推荐、保护版权 |
| **违规** | 误标原创会被处罚 |

### 7.2 转载规范

| 类型 | 说明 |
|------|------|
| **白名单** | 可申请其他账号白名单 |
| **授权** | 必须获得授权才能转载 |
| **标注** | 必须注明来源和作者 |

| 时间段 | 用户活跃度 | 适合内容 |
|--------|------------|----------|
| 7:00-9:00 | ⭐⭐⭐⭐ | 早安、干货 |

| 类型 | 频率 | 说明 |
|------|------|------|

## 九、合规规范

### 9.1 违规行为

```
❌ 禁止内容:
  - 政治敏感内容
  - 虚假信息
  - 低俗色情
  - 暴力血腥
  - 违法犯罪
  - 侵权盗版

❌ 禁止行为:
  - 诱导关注
  - 诱导分享
  - 虚假宣传
  - 恶意营销
```

### 9.2 广告规范

| 类型 | 规范 |
|------|------|
| **软文广告** | 必须标注"广告" |
| **硬广** | 明确标注 |
| **产品对比** | 不能恶意贬低竞品 |
| **极限词** | 禁止"最"、"第一"等 |

## 十、数据指标

### 10.1 核心指标

| 指标 | 说明 | 优质值 |
|------|------|--------|
| **阅读量** | 文章被阅读次数 | > 粉丝数 10% |
| **在看** | 用户点击"在看" | > 阅读量 3% |
| **留言** | 用户评论数 | > 阅读量 1% |
| **转发** | 用户转发数 | > 阅读量 2% |
| **收藏** | 用户收藏数 | > 阅读量 3% |

### 10.2 优化方向

```
提升阅读量:
  - 优化标题和封面
  - 提高内容质量
  - 选择最佳发布时间

提升在看/收藏:
  - 提供实用价值
  - 内容有获得感
  - 结尾引导收藏

提升留言:
  - 内容引发共鸣
  - 主动提问互动
  - 及时回复留言
```
