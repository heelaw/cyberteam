# 《渠道推广 Skill 体系》

版本：v3.1
更新日期：2026-03-25

---

## 1. 定位

为渠道推广角色提供系统化的渠道选择、效果分析和避坑指南，确保推广资源的高效配置。

---

## 2. 四个子技能

### 2.1 渠道效果分析法

- **功能**：多维度渠道效果评估、投入产出比分析、优化建议输出
- **输出**：渠道效果分析报告
- **路径**：`渠道效果分析法/SKILL.md`

### 2.2 渠道选择决策表

- **功能**：基于多维度权重评估、渠道特性匹配度分析、决策建议生成
- **输出**：渠道选择决策矩阵
- **路径**：`渠道选择决策表/SKILL.md`

### 2.3 第三方渠道类型清单

- **功能**：渠道类型盘点、平台特性分析、适用场景匹配
- **输出**：渠道清单及适用场景说明
- **路径**：`第三方渠道类型清单/SKILL.md`

### 2.4 渠道推广四大避坑点

- **功能**：常见渠道推广错误识别、风险点预警、优化建议
- **输出**：避坑指南及解决方案
- **路径**：`渠道推广四大避坑点/SKILL.md`

---

## 3. 调用方式

```
/channel analyze <推广内容>
/channel select <目标用户> <预算范围>
/channel checklist <渠道名称>
/channel avoid <渠道名称>
```

## 4. 渠道分类

| 类别 | 渠道类型 |
|------|----------|
| 付费渠道 | 搜索引擎广告、信息流广告、KOL合作 |
| 自然渠道 | SEO、社交媒体、内容营销 |
| 私域渠道 | 社群运营、公众号、短信营销 |
| 第三方平台 | 电商平台、垂直平台、App Store |
