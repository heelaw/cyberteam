## 核心Skills

### 1. 社群三门槛分类法
**触发**: 社群门槛怎么设
**功能**: 通过高/中/低三层门槛设计，科学设置社群准入标准

### 2. 用户进群引导三步法
**触发**: 用户进群怎么说、新人引导
**功能**: 通过"告知-说明-确认"三步流程，科学引导新用户加入社群

### 3. 群规三要素模板
**触发**: 群规怎么写
**功能**: 通过"群定位+支持信息+禁止信息"三要素设计完整的社群规范

### 4. 群规执行策略
**触发**: 群规怎么执行
**功能**: 通过"合适时机告知+严格执法"策略，确保群规有效执行

### 5. 早晚资讯三步法
**触发**: 社群价值怎么提供、社群内容怎么发
**功能**: 通过"信息收集→信息整合→信息输出"三步流程，持续为社群提供有价值的内容

### 6. 群内分享六步流程
**触发**: 群内分享怎么做
**功能**: 通过六步流程组织高质量的群内分享

### 7. 社群价值提供三法
**触发**: 社群提供什么价值、社群活跃
**功能**: 通过"早晚资讯+群内分享+打卡监督"三种方法系统化提供社群价值

### 8. 社群变现模式法
**触发**: 社群变现、赚钱
**功能**: 系统化设计社群商业化路径的方法论

### 9. 社群活跃度提升法
**触发**: 提升活跃度、群活跃
**功能**: 系统化提升社群活跃度的方法论

## 使用建议

### 场景匹配

| 场景 | 推荐Skill |
|------|-----------|
| 新建社群 | 社群三门槛分类法 → 用户进群引导三步法 → 群规三要素模板 |
| 提升活跃 | 早晚资讯三步法 + 社群活跃度提升法 |
| 活动组织 | 群内分享六步流程 |
| 商业变现 | 社群变现模式法 + 社群价值提供三法 |
| 群规执行 | 群规执行策略 |

### 组合使用

社群运营是一个系统工程，建议根据实际情况组合使用多个Skills：

1. **冷启动阶段**：社群三门槛分类法 + 用户进群引导三步法 + 群规三要素模板
2. **日常运营阶段**：早晚资讯三步法 + 群内分享六步流程 + 社群价值提供三法
3. **变现阶段**：社群变现模式法 + 社群活跃度提升法
