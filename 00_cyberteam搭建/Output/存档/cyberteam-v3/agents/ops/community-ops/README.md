## 核心Skills

### 1. UGC社区五步搭建法
**触发**: 怎么搭建社区、社区从0到1
**功能**: 帮助用户从0到1搭建UGC社区的完整方法论

### 2. 核心用户维系四法
**触发**: 怎么维护核心用户、KOL维系
**功能**: 通过荣誉、优先、情感、物质四个维度维系核心用户

### 3. 热点话题四渠道发现法
**触发**: 话题哪里找、社区话题
**功能**: 通过四个渠道发现社区热点话题

### 4. 社区活动三要点
**触发**: 社区活动怎么做
**功能**: 社区活动策划与执行的方法论

### 5. 马甲号运用策略
**触发**: 社区冷启动、社区氛围
**功能**: 社区冷启动阶段马甲号运营的方法论

## 使用建议

### 场景匹配

| 场景 | 推荐Skill |
|------|-----------|
| 新社区搭建 | UGC社区五步搭建法 + 马甲号运用策略 |
| 核心用户运营 | 核心用户维系四法 |
| 内容运营 | 热点话题四渠道发现法 |
| 活动策划 | 社区活动三要点 |

### 组合使用

社区运营是一个系统工程，建议根据实际情况组合使用多个Skills：

1. **冷启动阶段**：优先使用马甲号运用策略
2. **成长期**：结合UGC社区五步搭建法和热点话题四渠道发现法
3. **成熟期**：侧重核心用户维系四法和社区活动三要点
