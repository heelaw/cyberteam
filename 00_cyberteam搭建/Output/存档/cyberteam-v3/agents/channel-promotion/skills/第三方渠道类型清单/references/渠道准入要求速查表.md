# 渠道准入要求速查表

## 使用说明

本文档汇总各主流渠道的准入要求，便于快速查阅和准备资质。

---

## 一、付费广告类

### 1.1 广点通（微信/QQ/腾讯视频）

| 要求类型 | 具体要求 |
|----------|----------|
| **资质要求** | |
| 企业资质 | 营业执照、法人身份证、对公账户 |
| 行业资质 | 特殊行业需提供许可证（ICP备案/商标/授权书等） |
| 主体要求 | 不接受个人主体投放 |
| **账户要求** | |
| 首次充值 | 最低5000元 |
| 主体一致性 | 投放主体与资质主体一致 |
| **审核要求** | |
| 审核周期 | 1-3个工作日 |
| 行业限制 | 禁投行业：仿品、赌博、违规药品等 |
| **内容限制** | |
| 落地页 | 必须ICP备案，域名与开户主体一致 |
| 素材规范 | 不能使用绝对化用语、虚假夸大内容 |

### 1.2 巨量引擎（抖音/头条/西瓜）

| 要求类型 | 具体要求 |
|----------|----------|
| **资质要求** | |
| 企业资质 | 营业执照、法人身份证 |
| 行业资质 | 特殊行业需提供相关许可证 |
| 主体要求 | 企业/个体工商户，不接受个人 |
| **账户要求** | |
| 首次充值 | 最低10000元（部分代理5000元） |
| 账户结构 | 需明确广告主名称和投放资质 |
| **审核要求** | |
| 审核周期 | 1-2个工作日 |
| 行业限制 | 医疗、金融、游戏等需特殊资质 |
| **内容限制** | |
| 落地页 | 需ICP备案或链接至平台内页 |
| 素材规范 | 竖版视频优先，时长5-60秒 |

### 1.3 百度SEM

## 二、社交推广类

### 2.1 微博粉丝通

| 要求类型 | 具体要求 |
|----------|----------|
| **资质要求** | |
| 微博账号 | 需蓝V认证或合作代理账户 |
| 企业资质 | 营业执照、认证公函 |
| **账户要求** | |
| 最低投放 | 5000元起 |
| 账户类型 | 标准广告主或特殊行业广告主 |
| **审核要求** | |
| 审核周期 | 1-2个工作日 |
| 内容限制 | 涉及品牌需提供授权 |

### 2.2 抖音/小红书KOL合作

| 要求类型 | 具体要求 |
|----------|----------|
| **资质要求** | |
| 品牌资质 | 商标注册证或授权书 |
| 产品资质 | 特殊行业需提供许可证 |

## 三、自然流量类

### 3.1 SEO/内容分发

| 要求类型 | 具体要求 |
|----------|----------|
| **网站要求** | |
| 域名 | 建议1年以上老域名 |
| ICP备案 | 必须完成ICP备案 |
| 内容要求 | 持续更新高质量原创内容 |
| 技术要求 | 网站打开速度<3秒，移动友好 |
| **平台要求** | |
| 知乎 | 企业号认证（年费600元） |
| 豆瓣 | 小组运营，无特殊门槛 |
| 简书 | 个人即可，无特殊门槛 |
| B站 | 企业号认证，需发布内容积累 |

## 五、特殊行业准入要求

| 行业 | 额外资质要求 |
|------|--------------|
| 医疗/药品 | 医疗机构执业许可证、药品经营许可证 |
| 金融 | 金融业务许可证、ICP证 |
| 教育 | 办学许可证、ICP证 |
| 游戏 | 游戏版号、出版物经营许可证 |
| 食品 | 食品经营许可证 |
| 化妆品 | 化妆品生产许可证/非特备案 |
| 保险 | 保险业务许可证 |
