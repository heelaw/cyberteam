## 六、选择决策表

### 6.1 按预算选择

| 预算范围 | 推荐渠道组合 | 说明 |
|----------|--------------|------|
| <1万/月 | SEO + 内容分发 + 流量联盟 | 低成本渠道组合 |
| 1-5万/月 | 信息流广告测试 + SEO + KOC | 测试为主 |
| 5-20万/月 | 信息流主力 + SEM + KOL | 规模化投放 |
| >20万/月 | 全渠道组合 + KOL主力 | 头部KOL+精准投放 |

### 6.2 按产品选择

| 产品类型 | 推荐渠道 | 辅助渠道 |
|----------|----------|----------|
| 消费品/电商 | 信息流 + KOL + 流量联盟 | 内容分发 |
| APP推广 | 应用商店 + 信息流 | SEM |
| B2B/SaaS | SEM + SEO + 内容分发 | 私域 |
| 在线教育 | 信息流 + KOL + 私域 | 内容分发 |
| 游戏 | 应用商店 + 信息流 + KOL | 异业合作 |

### 6.3 按目标选择

| 推广目标 | 首选渠道 | 辅助渠道 |
|----------|----------|----------|
| 品牌曝光 | 社交媒体广告 + KOL | 信息流 |
| 效果转化 | SEM + 信息流 | KOL |
| 用户增长 | 应用商店 + 信息流 | 渠道联盟 |
| 口碑传播 | KOL + 内容分发 | 私域 |
