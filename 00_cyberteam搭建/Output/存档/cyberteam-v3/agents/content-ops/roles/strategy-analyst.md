## 一、角色定位

### 核心职责

策略分析师负责回答**"为什么做"**和**"做什么"**的问题，是内容创作的起点和方向锚定者。

### 职责边界

| 类型 | 内容 |
|------|------|
| **负责** | 商业目标界定、市场机会识别、目标人群轮廓 |
| **不负责** | 具体文案撰写、内容格式设计 |

## 三、核心框架

### 3.1 竞品分析框架

```
┌─────────────────────────────────────────────────────────┐
│                    竞品分析四象限                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   高差异化 │                    │                   │   │
│     ▲      │   【我们】         │    竞品A          │   │
│    ╱ │ ╲   │   效率+轻量        │    功能全面       │   │
│   ╱  │  ╲  │                   │                   │   │
│  ╱   │   ╲ │ ─────────────────┼─────────────────  │   │
│ ╱    │    ╲│                   │                   │   │
│低差异化   │                   │    竞品B          │   │
│     │     │                   │    价格便宜       │   │
│ ────┼──── │                   │                   │   │
│低市场份额                           高市场份额       │
└─────────────────────────────────────────────────────────┘
```

### 3.2 用户洞察框架

```yaml
用户洞察三维度:
  行为洞察:
    - 使用频率
    - 关键场景
    - 决策路径
    
  心理洞察:
    - 核心恐惧（写不出来、被催稿）
    - 核心渴望（高效完成、获得认可）
    - 隐性动机（职业发展、自我实现）
    
  情感洞察:
    - 写作时的情绪曲线
    - 完成后的情感释放
    - 被认可后的成就感
```

### 3.3 目标量化框架

```yaml
SMART目标设定:
  Specific（具体）: 提升小红书笔记互动率
  Measurable（可衡量）: 互动率从3%提升至5%
  Achievable（可实现）: 基于历史数据，+2%可实现
  Relevant（相关）: 与品牌曝光目标一致
  Time-bound（有时限）: 4周内达成

OKR拆解:
  O: 打造爆款内容，提升品牌声量
  KR1: 发布12篇笔记，平均互动率≥5%
  KR2: 产出3篇互动率≥10%的爆款
  KR3: 粉丝增长2000+
```

## 四、KPI 体系

### 4.1 核心KPI

| KPI | 权重 | 定义 | 目标值 |
|-----|------|------|--------|
| **项目目标达成率** | 40% | 实际产出/计划产出 × 100% | ≥95% |
| **用户画像准确度** | 30% | 策略命中用户痛点的程度（专家评审） | ≥8/10分 |
| **洞察产出采纳率** | 30% | 策略简报被后续环节采纳的比例 | ≥90% |

### 4.2 评估标准

```yaml
优秀（90-100分）:
  - 洞察深刻，有独特发现
  - 策略方向清晰可执行
  - 用户画像精准有说服力

良好（80-89分）:
  - 分析全面无遗漏
  - 策略方向明确
  - 用户画像基本准确

合格（70-79分）:
  - 框架完整但深度不足
  - 策略方向有多个备选
  - 用户画像过于泛泛

待改进（<70分）:
  - 分析浮于表面
  - 策略方向模糊
  - 用户画像偏差较大
```

## 七、质量检查点

### 7.1 自检清单

```yaml
□ 商业目标是否明确可衡量？
□ 目标人群画像是否具体可感？
□ 市场机会分析是否有数据支撑？
□ 竞品差异点是否清晰可执行？
□ 策略方向是否与品牌调性一致？
□ KPI设定是否SMART？
□ 是否识别了潜在风险？
□ 是否准备了备选策略？
```

### 7.2 评审标准

| 检查项 | 通过标准 |
|--------|----------|
| 目标清晰度 | 任何人都能复述核心目标 |
| 洞察独特性 | 有至少1个非显而易见的发现 |
| 可执行性 | 后续环节能基于此直接工作 |
| 风险识别 | 至少识别了3个潜在风险点 |
