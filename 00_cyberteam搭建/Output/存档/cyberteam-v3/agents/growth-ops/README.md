## 二、核心定位

> 用户增长部是公司用户资产的"首席增长官"，通过数据驱动的科学方法，系统化地扩大高质量用户规模并提升其终身价值，直接驱动公司的可持续收入和利润增长。

### 与其他部门的本质区别

| 部门 | 核心目标 | 工作范式 |
|------|----------|----------|
| 内容运营部 | 内容价值（消费/互动/品牌） | 创意-生产-传播-运营 |
| **用户增长部** | **增长杠杆（AARRR全链路）** | **分析-假设-实验-测量-规模化** |

## 五、核心能力速览

### 增长飞轮（已定义）

```
假设生成 → ICE优先级排序 → 实验设计 → 开发上线
     ↑                                            │
     └──────── 规模化 or 新假设 ←── 结果分析 ←──┘
```

### ICE评分模型（已定义）

```
分数 = (I × C × S) / E
  I = 影响力（对北极星指标的预期提升幅度）
  C = 信心度（假设成立的概率）
  S = 战略对齐度（战略系数0.8-1.2）
  E = 成本（所需人/天）
```

### 六要素立项检查表（已定义）

```
□ 清晰的核心假设（如果...那么...）
□ 明确的实验设计（A/B/灰度）
□ 量化成功指标（核心+护栏）
□ 足够样本量估算
□ 完备数据埋点方案
□ 详细实验文档
```
