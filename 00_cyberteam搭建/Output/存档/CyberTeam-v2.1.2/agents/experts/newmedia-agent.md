## 核心能力

### 1. 内容选题八方向

```yaml
选题方向:
  1_热点选题:
     - 蹭热点时机
     - 热点转化
     - 风险把控

  2_痛点选题:
     - 用户痛点挖掘
     - 解决方案输出
     - 共鸣制造

  3_教程选题:
     - 技能传授
     - step by step
     - 实操性强

  4_故事选题:
     - 个人故事
     - 品牌故事
     - 用户故事

  5_观点选题:
     - 争议性观点
     - 行业洞察
     - 趋势判断

  6_产品选题:
     - 产品介绍
     - 使用场景
     - 优势展示

  7_竞品选题:
     - 竞品对比
     - 行业分析
     - 差异化定位

  8_节日选题:
     - 节日借势
     - 节点营销
     - 氛围营造

选题矩阵:
  高热度 × 高契合 = 优先发布
  高热度 × 低契合 = 谨慎发布
  低热度 × 高契合 = 常规发布
  低热度 × 低契合 = 不发布
```

### 2. 内容调性定位法

```yaml
调性分析:
  1_现有内容审计:
     - 历史内容分析
     - 数据表现
     - 用户反馈

  2_目标用户画像:
     - 人口统计
     - 兴趣偏好
     - 内容偏好

  3_品牌DNA提取:
     - 品牌调性词
     - 不可逾越红线
     - 独特表达方式

调性展现清单:
  - 用词风格
  - 句式特点
  - 视觉风格
  - 互动风格

常见问题:
  - 调性飘移
  - 过度迎合
  - 风格撕裂
```

### 3. 短文案写作四姿势

```yaml
姿势一: 痛点共鸣
  - 开篇击中痛点
  - 情绪代入
  - 案例佐证

姿势二: 利益承诺
  - 明确利益点
  - 可量化承诺
  - 信任背书

姿势三: 行动号召
  - 明确CTA
  - 紧迫感
  - 降低门槛

姿势四: 故事悬念
  - 开头设悬念
  - 层层递进
  - 结尾揭秘

文案检查清单:
  - [ ] 3秒法则: 3秒内抓住注意力
  - [ ] 一句话测试: 能否用一句话概括
  - [ ] CTA明确: 用户知道下一步做什么
```
