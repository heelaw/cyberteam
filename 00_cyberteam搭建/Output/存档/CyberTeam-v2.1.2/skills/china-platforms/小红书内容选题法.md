# 小红书内容选题法

## 概述

选题是爆款笔记的起点。一个好的选题能让笔记赢在起跑线，让内容自带流量。本方法论帮助运营者系统性地找到适合自己品牌的高潜力选题。

## 选题框架

### 1. 用户需求导向选题

**痛点型选题**
- 直接解决用户真实困扰
- 示例："熬夜急救指南"、"毛孔粗大怎么办"

**对比型选题**
- 帮助用户做决策
- 示例："XX vs XX 到底哪个好"、"平价替代测评"

**教程型选题**
- 提供实用价值，用户收藏率高
- 示例："新手化妆教程"、"如何3步搞定XXX"

### 2. 热点借势选题

**平台热点**
- 关注小红书站内热词、热搜话题
- 工具：小红书搜索下拉词、热度榜

**节点热点**
- 节日节点：春节、情人节、618、双11
- 季节节点：换季护肤、开学季

**社会热点**
- 适度参与，但不能过度蹭热点

### 3. 差异化选题策略

**细分人群**
- 从年龄、职业、生活状态等维度细分
- 示例："职场新人穿搭"、"新手妈妈待产包"

**细分场景**
- 特定使用场景的内容更容易引起共鸣
- 示例："通勤包里有什么"、"宿舍好物分享"

**细分需求**
- 挖掘小众但真实的需求
- 示例："鼻炎患者好物"、"小个子显高穿搭"

## 选题执行流程

```
1. 收集阶段
   - 竞品爆款笔记分析
   - 用户评论/私信中的高频问题
   - 站内热搜词和热门话题

2. 筛选阶段
   - 与品牌/产品的关联度
   - 竞争程度评估
   - 内容可执行性

3. 规划阶段
   - 排期安排（每周2-3条核心选题）
   - 内容形式匹配（图文/视频）
   - 达人/KOC配合安排

4. 复盘阶段
   - 发布后数据追踪
   - 爆款因素分析
   - 选题库持续更新
```

## 高互动选题公式

```
选题公式 = 目标人群 + 痛点/爽点 + 解决方案

示例：
- "月薪3000 | 通勤穿搭教科书"（人群+预算+解决方案）
- "油皮亲妈！夏天持妆12小时实测"（肤质+需求+验证）
- "租房改造 | 500元搞定全屋软装"（场景+成本+效果）
```

## 选题避坑指南

- 避免过于小众无人问津的选题
- 避免与品牌调性不符的热点借势
- 避免已经烂大街的同质化内容
- 避免违禁词和敏感话题
