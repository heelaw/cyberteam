# 用户生命周期管理

## 概述

用户生命周期管理是私域运营的核心思维。每个阶段的用户需要不同的运营策略。本skill帮助运营者掌握用户全生命周期运营方法。

## 生命周期阶段

### 五阶段模型

```markdown
## 拉新阶段（Acquisition）
- 用户来源：公域/私域各渠道
- 核心目标：添加企业微信
- 关键指标：添加率、成本

## 激活阶段（Activation）
- 用户状态：刚添加企微
- 核心目标：完成首次互动/首单
- 关键指标：激活率、首单转化

## 留存阶段（Retention）
- 用户状态：已有互动/购买
- 核心目标：持续产生互动
- 关键指标：活跃度、复购率

## 变现阶段（Revenue）
- 用户状态：高价值用户
- 核心目标：最大化LTV
- 关键指标：ARPU、复购频次

## 推荐阶段（Referral）
- 用户状态：忠诚用户
- 核心目标：老带新裂变
- 关键指标：裂变系数
```

## 各阶段运营策略

### 拉新阶段

```markdown
## 拉新渠道

1. 公域引流
   - 包裹卡
   - 直播间引导
   - 门店导流
   - 短信触达

2. 私域裂变
   - 老带新奖励
   - 分享邀请

## 拉新话术

"感谢购买！加我微信，领取专属福利
回复'福利'，我单独发给你~"
```

### 激活阶段

```markdown
## 激活策略（0-7天）

Day 0  欢迎语+价值预告
Day 1  新人专属优惠
Day 3  使用指引/教程
Day 7  首单优惠刺激

## 激活话术

"Hi~我是你的专属顾问xxx
感谢信任！这是你的专属权益：
1. 新人券（满99减20）
2. 优先回复
3. 专属客服

有任何问题随时找我~"
```

### 留存阶段

```markdown
## 留存策略

1. 内容价值
   - 定期推送有价值内容
   - 专属知识分享

2. 互动价值
   - 定期互动活动
   - 会员专属福利

3. 情感连接
   - 节日问候
   - 生日祝福
   - 专属关怀

## 留存话术

"嗨~好久没联系了
最近怎么样？有什么需要帮忙的吗？
对了，店里刚到了一批新品，要不要看看？"
```

### 变现阶段

```markdown
## 变现策略

1. 个性化推荐
   - 基于历史购买
   - 基于标签偏好

2. 升级销售
   - 高阶产品推荐
   - 组合套餐

3. 复购提醒
   - 消耗周期提醒
   - 季节性推荐

## 话术示例

"看到你上次买了XXX，用得怎么样？
正好有款升级版在做活动，要了解一下吗？"
```

### 推荐阶段

```markdown
## 推荐策略

1. 激励设计
   - 邀请奖励
   - 阶梯奖励

2. 荣誉体系
   - 荣誉墙
   - 称号认证

3. 专属福利
   - 推荐人专属权益
   - 优先体验新品

## 话术示例

"亲爱的，看你一直在支持我们
有没有朋友也需要我们的产品呀？
推荐一位朋友下单，你可得XX元优惠券
推荐三位，直接送你VIP会员~"
```

## 生命周期自动化

### 触达规则配置

```markdown
## 新客激活触达

触发：添加企微好友
Day 0   欢迎语+新人券
Day 1   产品使用指南
Day 3   邀请入群
Day 7   首单优惠
未首单  1v1私聊激活

## 复购提醒触达

触发：消耗周期前N天
Day -7  使用效果调研
Day -3  复购优惠通知
Day 0   1v1提醒补货
```

## 流失预警

### 流失信号识别

```markdown
## 流失预警信号

1. 互动下降
   - 30天无消息互动
   - 朋友圈互动消失

2. 消费异常
   - 超过平均购买周期未复购
   - 客单价持续下降

3. 行为异常
   - 退群
   - 屏蔽消息
   - 投诉举报
```

### 流失挽回策略

```markdown
## 30天沉默用户

Day 30  朋友圈精准触达
Day 45  专属回归券
Day 60  1v1关怀（非营销）
Day 90  降级处理

## 流失用户话术

"嗨~好久没见你冒泡了
最近还好吗？
给你准备了一份小礼物，专门感谢老客户"
```
