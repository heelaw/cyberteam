# 小红书数据运营分析

## 概述

数据是衡量运营效果的唯一标准。本skill帮助运营者建立完善的数据分析体系，从数据中发现问题、优化策略、提升效果。

## 核心数据指标

### 账号层面

| 指标 | 定义 | 健康值 |
|------|------|--------|
| 粉丝数 | 账号关注者总数 | 持续增长 |
| 获赞与收藏 | 账号累计获得的赞和收藏 | 赞藏比>3:1 |
| 互动率 | (点赞+收藏+评论)/浏览量 | >5% |
| 粉丝增长 | 周期内新增粉丝数 | 月增长>10% |

### 笔记层面

| 指标 | 定义 | 健康值 |
|------|------|--------|
| 浏览量 | 笔记被看到的次数 | 持续提升 |
| 点赞数 | 用户点赞数量 | 互动率>5% |
| 收藏数 | 用户收藏数量 | 收藏率>8% |
| 评论数 | 用户评论数量 | 评论率>1% |
| 分享数 | 用户分享次数 | 分享率>2% |

## 数据分析维度

### 1. 内容表现分析

**单篇笔记分析**
```
- 浏览量：基础曝光
- 点击率：封面和标题吸引力
- 互动率：内容共鸣程度
- 收藏率：实用价值认可
- 搜索曝光：内容SEO效果
```

**内容类型分析**
```
- 图文 vs 视频：哪种形式效果更好
- 不同选题类型：教程 vs 测评 vs 种草
- 不同话题：哪些话题互动更高
```

### 2. 粉丝画像分析

**基础属性**
- 性别比例
- 年龄分布
- 地域分布
- 活跃时间段

**兴趣标签**
- 关注话题
- 互动偏好
- 消费能力

### 3. 竞品分析

**竞品识别**
- 同品类头部账号
- 内容风格相似账号
- 目标人群重叠账号

**竞品监测**
- 发布频率和内容类型
- 爆款笔记特征
- 粉丝增长趋势
- 互动数据表现

## 数据驱动决策

### 高互动笔记特征提取

```
1. 标题特征
   - 包含哪些关键词
   - 标题结构类型
   
2. 封面特征
   - 封面风格
   - 文字大小和位置
   
3. 内容特征
   - 正文字数
   - 内容结构
   - 标签组合
   
4. 发布特征
   - 发布时间
   - 互动引导方式
```

### 迭代优化方向

```
发现问题 → 分析原因 → 制定方案 → 执行测试 → 验证效果

示例：
问题：本周笔记互动率下降
原因：标题缺乏情绪触发词
方案：测试3种不同情绪标题
执行：A/B测试发布
验证：选择互动率更高的标题模板
```
