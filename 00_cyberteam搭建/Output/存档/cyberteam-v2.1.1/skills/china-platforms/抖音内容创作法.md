# 抖音内容创作法

## 平台特性

### 平台画像
- **用户规模**: 7亿+
- **用户特征**: 年轻化、消费能力强、娱乐导向
- **算法逻辑**: 去中心化算法推荐，重视完播率和互动率

### 平台规则
- **内容规范**: 禁止低俗、虚假宣传、侵权内容
- **变现方式**: 直播带货、广告合作、商品橱窗

### Success Metrics (成功指标)

- **完播率**: 视频完整播放比例，>30%为健康
- **互动率**: (点赞+评论+分享)/播放量，>3%为健康
- **GMV**: 直播成交总额
- **ROI**: 广告投入产出比，>2为健康

### Tools & Resources (工具和资源)

- 巨量引擎 - 广告投放平台
- 千川投放 - 直播/电商广告
- 抖音电商罗盘 - 数据分析


## 概述

抖音是一个"注意力经济"平台，用户只有3秒决定是否继续观看。好的内容创作是抖音运营的核心。本skill帮助创作者系统性地掌握抖音内容的创作方法。

## 抖音算法基础

### 流量分发逻辑

```
视频发布 → 初始流量池（200-500曝光）
         ↓
  基础数据检验
  - 完播率 > 30%
  - 互动率 > 3%
  - 关注转化 > 2%
         ↓
  进入更大流量池
  （1000→1万→10万→100万→1000万...）
```

### 数据权重排序

1. **完播率** - 最重要的指标，决定是否进入更大流量池
2. **互动率** - 点赞、评论、分享的综合表现
3. **关注转化** - 新增粉丝/浏览量的比值
4. **主页访问** - 用户点击主页的比例

## 爆款视频结构

### 黄金3秒开头

**冲突型**
```
"千万别买XXX，除非你看完这条"
"我发现了XXX行业不想让你知道的秘密"
```

**利益型**
```
"花XX元解决了困扰我3年的问题"
"学会这3招，XXX效率翻倍"
```

**悬念型**
```
"为什么XXX突然这么火？背后原因..."
"这件XXX，到底值不值得买？"
```

**共鸣型**
```
"是不是每次XXX都特别崩溃？"
"有多少人和我一样，XXX"
```

### 核心内容设计

**信息密度**
- 每5秒至少有一个"钩子"
- 避免无效等待画面
- 语速适中，关键信息重复

**节奏控制**
- 快节奏剪辑保持注意力
- 适当放慢突出关键信息
- 背景音乐卡点增强节奏感

### 结尾钩子设计

```
1. 互动引导
   "你们觉得值不值？评论区告诉我"
   "你们更喜欢哪个？投票告诉我"

2. 系列预告
   "下期教你XXX，先关注别丢了"
   "下期更精彩，错过就太可惜了"

3. 价值总结
   "总结一下：1、2、3"
   "记住这几点，XXX不再难"
```

## 内容类型创作模板

### 知识类内容（30-60秒）

```markdown
## 开头（3秒）
抛出问题或悬念
"你知道XXX吗？今天一次说清楚"

## 正文（45秒）
- 核心知识点1（15秒）
- 核心知识点2（15秒）
- 核心知识点3（15秒）

## 结尾（12秒）
总结+互动引导
"觉得有用的点个赞，下期见"
```

### 产品种草类（15-30秒）

```markdown
## 开头（3秒）
痛点引入或效果展示
"困扰我多年的XXX，终于被它解决了"

## 中间（15秒）
- 产品展示（5秒）
- 核心卖点（5秒）
- 效果对比/使用演示（5秒）

## 结尾（12秒）
价格+购买引导
"链接在评论区，赶紧冲"
```

### 剧情类内容（15-60秒）

```markdown
## 开头（3秒）
制造冲突或悬念
"我今天必须要说清楚这件事"

## 发展（30秒）
- 剧情推进
- 情感铺垫
- 关键转折

## 高潮+结尾（15秒）
情感爆发或反转结局
引导互动或关注
```

## 拍摄与剪辑技巧

### 竖屏拍摄要点

- 构图：主体居中或偏上
- 景别：近景和特写为主
- 光线：正面或侧光，避免逆光
- 稳定：使用稳定器或手机支架

### 剪辑节奏

```
- 切换频率：每2-3秒切换镜头
- 关键信息：重复2-3次加深印象
- BGM选择：匹配内容节奏
- 字幕：必须添加，关键信息放大
```
