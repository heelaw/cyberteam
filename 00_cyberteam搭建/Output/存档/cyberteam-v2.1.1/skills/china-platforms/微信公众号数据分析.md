# 微信公众号数据分析

## 平台特性

### 平台画像
- **用户规模**: 13亿+
- **用户特征**: 全年龄段、社交关系链强
- **算法逻辑**: 社交分发+推荐算法

### 平台规则
- **内容规范**: 内容导向、禁止营销过度
- **变现方式**: 广告分成、内容付费、电商导流

### Success Metrics (成功指标)

- **打开率**: 阅读数/送达数，>30%为健康
- **阅读率**: 实际阅读/粉丝数
- **转发率**: 分享数/阅读数，>2%为健康
- **新增粉丝**: 日增关注数

### Tools & Resources (工具和资源)

- 微信公众平台 - 公众号管理
- 视频号助手 - 视频号管理
- 微信广告 - 广告投放


## 概述

数据是公众号运营的指南针。本skill帮助运营者建立完善的数据分析体系，从数据中发现问题、优化策略、提升效果。

## 核心数据指标

### 用户数据

| 指标 | 定义 | 说明 | 健康值 |
|------|------|------|--------|
| 总关注数 | 累计关注人数 | 基础规模 | 持续增长 |
| 新增关注 | 周期新增关注 | 增长能力 | 月增>10% |
| 取关数 | 周期取关人数 | 流失情况 | <5%/月 |
| 净增关注 | 新增-取关 | 实际增长 | 正增长 |

### 内容数据

| 指标 | 定义 | 说明 | 健康值 |
|------|------|------|--------|
| 图文送达人数 | 消息推送人数 | 触达能力 | - |
| 图文阅读人数 | 实际阅读人数 | 内容吸引力 | - |
| 图文阅读数 | 重复阅读总计 | 内容质量 | - |
| 打开率 | 阅读/送达 | 标题质量 | >30% |
| 原文阅读数 | 点击原文人数 | 转化能力 | >5% |

### 互动数据

| 指标 | 定义 | 健康值 |
|------|------|--------|
| 关注后回复互动率 | 关键词/消息互动 | >5% |
| 菜单点击率 | 菜单栏点击占比 | >20% |
| 留言率 | 精选留言数/阅读 | >1% |
| 分享率 | 分享人数/阅读 | >2% |

## 数据分析方法

### 打开率分析

```markdown
## 打开率影响因素

1. 标题因素
   - 是否有吸引力
   - 是否含关键词
   - 是否制造悬念

2. 发送时间因素
   - 是否在用户活跃时段
   - 周几发送效果更好

3. 发送频率因素
   - 是否过频导致疲劳
   - 是否过少导致遗忘

## 打开率优化
- 测试不同标题风格
- 优化发送时间
- 控制发送频率
```

### 阅读完成率分析

```markdown
## 完成率影响因素

1. 内容质量
   - 开头是否有吸引力
   - 内容是否有价值
   - 长度是否适中

2. 排版因素
   - 是否分段清晰
   - 图片是否适当
   - 是否手机端友好

3. 匹配度
   - 标题与内容是否匹配
   - 是否符合用户预期
```

### 转化漏斗分析

```
粉丝关注
   ↓
收到推送（送达）
   ↓
打开阅读（打开率）
   ↓
阅读完成（完成率）
   ↓
点击原文（转化率）
   ↓
完成目标（注册/购买/咨询）
```
