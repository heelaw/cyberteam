# 京东快车投放

## 平台特性

### 平台画像
- **用户规模**: 5亿+
- **用户特征**: 一二线城市、注重品质、价格不敏感
- **算法逻辑**: 搜索排名+推荐算法

### 平台规则
- **内容规范**: 品质保证、正品保障
- **变现方式**: 商品销售、广告分成

### Success Metrics (成功指标)

- **GMV**: 成交总额
- **转化率**: 下单/访客，>4%为健康
- **客单价**: 平均订单金额，高于行业
- **复购率**: 老客复购比例，>30%为优秀

### Tools & Resources (工具和资源)

- 京东商智 - 数据分析
- 京东快车 - 广告投放
- 京东自营 - 仓储物流


## 概述

京东快车是京东店铺获取流量的核心工具。相比淘宝直通车，京东快车有其独特的算法和投放逻辑。本skill帮助运营者掌握京东快车的投放策略。

## 京东快车基础

### 投放产品

```markdown
## 搜索广告

- 关键词竞价
- 搜索结果展示
- 按点击计费

## 展示广告

- 京选展位
- 首页焦点图
- 竖版展位

## 推荐广告

- 推荐场景
- 精准人群投放
```

### 核心指标

```markdown
## 关键指标

1. CTR（点击率）
   - 衡量创意吸引力
   - 行业均值1-3%

2. CPC（点击单价）
   - 每次点击成本
   - 关键词质量分影响

3. ROI（投入产出比）
   - 核心考核指标
   - 成交金额/广告花费

4. CVR（转化率）
   - 点击到成交比例
   - 影响排名和费用
```

## 投放策略

### 关键词策略

```markdown
## 关键词分类

1. 品牌词
   - 保护品牌流量
   - 高转化

2. 产品词
   - 品类流量
   - 中等竞争

3. 长尾词
   - 精准需求
   - 低竞争
```

### 人群定向

```markdown
## 京东人群包

1. 基础属性
   - 性别/年龄/地域
   - 消费能力

2. 行为人群
   - 浏览/收藏/加购
   - 购买过同类

3. 竞品人群
   - 竞品店铺访客
   - 竞品购买者
```

## ROI优化

### 优化步骤

```markdown
## 步骤1：数据诊断

- 查看各关键词ROI
- 分析人群包效果
- 检查落地页转化

## 步骤2：问题分析

- 关键词是否精准
- 人群是否匹配
- 落地页是否承接

## 步骤3：持续优化

- 保留高ROI词
- 删除低ROI词
- 测试新人群
```
