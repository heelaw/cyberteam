# 京东自营运营

## 平台特性

### 平台画像
- **用户规模**: 5亿+
- **用户特征**: 一二线城市、注重品质、价格不敏感
- **算法逻辑**: 搜索排名+推荐算法

### 平台规则
- **内容规范**: 品质保证、正品保障
- **变现方式**: 商品销售、广告分成

### Success Metrics (成功指标)

- **GMV**: 成交总额
- **转化率**: 下单/访客，>4%为健康
- **客单价**: 平均订单金额，高于行业
- **复购率**: 老客复购比例，>30%为优秀

### Tools & Resources (工具和资源)

- 京东商智 - 数据分析
- 京东快车 - 广告投放
- 京东自营 - 仓储物流


## 概述

京东自营是京东平台的核心模式，由京东负责仓储物流，商家供货。本skill帮助运营者掌握京东自营店铺的运营方法。

## 自营入驻

### 入驻要求

```markdown
## 基础资质

1. 企业资质
   - 营业执照
   - 法人身份证
   - 银行开户证明

2. 品牌资质
   - 商标注册证
   - 授权书

3. 行业资质
   - 食品：食品经营许可证
   - 化妆品：化妆品生产许可证
```

### 入仓流程

```markdown
## 入仓步骤

1. 提交入驻申请
2. 资质审核
3. 签署合同
4. 开通账户
5. 预约入仓
6. 商品入库
7. 正式上线
```

## 库存管理

### 备货策略

```markdown
## 备货原则

1. 安全库存
   - 日均销量 × 备货天数
   - 预留缓冲量

2. 爆款备货
   - 预测爆款
   - 提前1-2周备货

3. 季节性备货
   - 提前规划
   - 分批次入库
```

### 库存监控

```markdown
## 监控指标

1. 库存周转
   - 周转天数
   - 避免积压

2. 现货率
   - 订单满足率
   - 影响店铺评分

3. 滞销处理
   - 定期清理
   - 避免仓储费
```

## 物流时效

### 211限时达

```markdown
## 时效承诺

- 当日11:00前下单，当日送达
- 次日达覆盖更多区域

## 提升方法

1. 就近入仓
   - 全国多仓布局
   - 就近发货

2. 智能分单
   - 系统自动分配
   - 最优配送路径

3. 打包效率
   - 优化打包流程
   - 提升出库速度
```
