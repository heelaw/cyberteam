# 抖音投放策略

## 平台特性

### 平台画像
- **用户规模**: 7亿+
- **用户特征**: 年轻化、消费能力强、娱乐导向
- **算法逻辑**: 去中心化算法推荐，重视完播率和互动率

### 平台规则
- **内容规范**: 禁止低俗、虚假宣传、侵权内容
- **变现方式**: 直播带货、广告合作、商品橱窗

### Success Metrics (成功指标)

- **完播率**: 视频完整播放比例，>30%为健康
- **互动率**: (点赞+评论+分享)/播放量，>3%为健康
- **GMV**: 直播成交总额
- **ROI**: 广告投入产出比，>2为健康

### Tools & Resources (工具和资源)

- 巨量引擎 - 广告投放平台
- 千川投放 - 直播/电商广告
- 抖音电商罗盘 - 数据分析


## 概述

抖音投放是获取精准流量、快速起号的重要工具。DOU+适合内容加热，千川适合直播/商品投放。本skill帮助运营者掌握抖音广告的投放策略。

## 投放产品选择

### DOU+ vs 千川

| 维度 | DOU+ | 千川 |
|------|------|------|
| 适用场景 | 内容加热、账号涨粉 | 直播带货、商品推广 |
| 投放门槛 | 100元起 | 1000元起 |
| 定向能力 | 基础定向 | 精准人群包 |
| 计费方式 | CPM/CPE | OCPM/OCPC |
| 优化目标 | 播放/互动/粉丝 | 直播间进入/下单 |

## DOU+投放策略

### 投放目标选择

```
1. 粉丝量 - 适合账号冷启动期
2. 点赞评论 - 适合内容测试
3. 播放量 - 适合提升曝光
4. 主页浏览 - 适合引流主页
```

### 投放时机选择

```markdown
## 内容发布后投放节奏

1. 发布后1-2小时
   - 观察自然流量表现
   - 基础数据（完播率）达标后开始投放

2. 投放时长选择
   - 24小时：快速放量
   - 2-48小时：稳定放量

3. 分批次投放
   - 先小额测试（100-200元）
   - 效果好再追投
```

### 人群定向

```markdown
## 基础定向设置

1. 性别/年龄：根据目标人群设置
2. 地域：一二线城市/目标市场
3. 兴趣标签：选择与内容相关的兴趣
4. 行为定向：近期活跃用户优先

## 进阶定向
- 相似达人粉丝：投放对标账号的粉丝
- 关键词兴趣：搜索过相关内容的用户
```

## 千川投放策略

### 账户搭建结构

```markdown
## 建议账户结构

账户1 - 短视频带货
- 计划A：商品购买目标
- 计划B：粉丝关注目标
- 计划C：互动目标

账户2 - 直播带货
- 计划A：直播间进入
- 计划B：直播间下单
- 计划C：ROI优化

账户3 - 达人合作
- 投放达人视频加热
```

### 投放策略

**冷启动期（前30场直播）**
```
- 70%付费流量 + 30%自然流量
- 目标：让算法学习目标人群
- 出价：略高于行业均值
```

**成长期（账号稳定后）**
```
- 50%付费 + 50%自然
- 目标：降低付费占比
- 优化：提升内容质量撬动自然流
```

**成熟期（稳定GMV）**
```
- 30%付费 + 70%自然
- 目标：付费用于突破天花板
- 策略：GMV冲刺时加大投放
```

### 出价策略

```markdown
## 出价公式

成交出价 = 目标ROI的1/客单价 × 系数

示例：
- 客单价100元
- 目标ROI 3
- 出价 = 100/3 × 1.1 = 36.7元

## 出价调整原则
- 计划消耗慢：提高出价5-10%
- 成本超预期：检查定向和素材
- 跑量瓶颈：新建计划或扩宽定向
```

## 素材优化

### 高转化素材特征

```markdown
## 口播类素材
- 主播直接对着镜头讲
- 痛点引入+产品介绍+效果展示
- 时长15-60秒

## 产品展示类
- 开箱/试用/效果对比
- 视觉冲击强
- 适合美妆/护肤/食品

## 混剪类素材
- 直播高光片段+字幕+BGM
- 成本低、产量高
- 适合日用/百货品类
```

### 素材迭代节奏

```
1. 每3天测试新素材
2. 数据不好的素材及时替换
3. 爆量素材要准备迭代版本
4. 保持3-5套素材同时在线
```

## ROI优化

### 影响ROI的关键因素

```
1. 素材质量（最重要）
2. 人群定向精准度
3. 落地页/直播间承接能力
4. 商品价格和竞争力
5. 客服和物流能力
```

### ROI优化步骤

```markdown
## 步骤1：数据诊断
- ROI < 1：素材或落地页问题
- ROI 1-2：人群定向可能过窄
- ROI > 2：可适当放量

## 步骤2：分环节排查
- 点击率低：优化封面/标题
- 转化率低：优化落地页/话术
- 客单价低：优化选品/组合

## 步骤3：持续测试
- 测试不同定向组合
- 测试不同素材风格
- 测试不同出价策略
```
