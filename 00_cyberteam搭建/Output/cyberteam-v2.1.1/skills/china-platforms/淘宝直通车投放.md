## 概述

淘宝直通车是淘宝店铺获取精准流量的核心工具。本skill帮助运营者掌握直通车的投放策略，优化广告ROI。

## 直通车基础

### 投放原理

```markdown
## 计费方式

按点击收费（CPC）
- 每次点击的价格
- 最低出价0.05元

## 排名规则

综合排名 = 质量分 × 出价

质量分越高，相同出价排名越靠前
```

### 核心指标

```markdown
## 关键指标

1. 质量分
   - 创意质量
   - 相关性
   - 买家体验

2. 点击率（CTR）
   - 点击量/展现量
   - 越高越好

3. 投入产出比（ROI）
   - 成交金额/广告花费
   - 核心考核指标

4. 点击单价（PPC）
   - 每次点击成本
   - 越低越好
```

### 关键词分类

```markdown
## 核心词
- 大流量词
- 高竞争
- 高花费
- 精准人群

## 长尾词
- 细分需求词
- 低竞争
- 精准转化

## 活动词
- 大促相关词
- 季节性词
```

### 出价策略

```markdown
## 出价原则

1. 新品期
   - 先长尾词出高价测试
   - 找到转化词后提价

2. 成长期
   - 核心词逐渐加入
   - 根据ROI调整出价

3. 成熟期
   - 核心词稳定出价
   - 长尾词补充流量

## 出价公式

目标PPC = 商品利润 × 转化率 × 期望ROI
```

## 人群定向

### 人群包类型

```markdown
## 基础属性

- 性别
- 年龄
- 地域
- 消费水平

## 行为人群

- 店铺人群
  - 访问未购买
  - 收藏加购
  - 购买老客
  
- 类目人群
  - 浏览/收藏/购买过
  - 相似宝贝偏好

## 达摩盘人群
- 自定义标签
- 高阶定向
```

### 人群出价策略

```markdown
## 人群溢价

基础出价 + 人群溢价

## 溢价策略

1. 新品期
   - 低出价，高人群溢价
   - 精准人群优先

2. 成长期
   - 平衡出价和溢价
   - 逐步扩大人群

3. 活动期
   - 高出价，高溢价
   - 抢占精准流量
```
