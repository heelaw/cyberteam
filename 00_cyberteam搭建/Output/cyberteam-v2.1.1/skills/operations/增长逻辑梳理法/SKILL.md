## 核心框架

### 增长飞轮模型

```
增长飞轮 = 驱动力 × 转化率 × 留存率 × 传播系数

驱动力: 产品价值 / 价格优势 / 品牌拉力
转化率: 流量 → 用户 → 付费
留存率: 付费 → 复购 → 忠诚
传播系数: 口碑 / 裂变 / 生态
```

## 执行步骤

### Step 1: 商业模式画布梳理

```
九要素画布:
  □ 价值主张: 解决什么问题
  □ 客户细分: 谁是核心用户
  □ 渠道通路: 如何触达用户
  □ 客户关系: 如何维护关系
  □ 收入来源: 如何赚钱
  □ 核心资源: 关键资产是什么
  □ 关键业务: 必须做什么
  □ 重要合作: 依赖谁
  □ 成本结构: 花在哪里

输出: 商业模式画布图
```

### Step 2: 关键指标拆解

```
北极星指标 (唯一关键指标):
  - 唯一指标
  - 反映用户价值
  - 可驱动增长

辅助指标:
  - 先行指标 (预测北极星)
  -滞后指标 (反映北极星结果)

指标层级:
  北极星指标
    ├── 先行指标A
    │   ├── 指标A1
    │   └── 指标A2
    └── 先行指标B
        ├── 指标B1
        └── 指标B2

示例 (电商):
  北极星: GMV
    ├── 流量
    │   ├── 付费流量
    │   └── 免费流量
    └── 转化
        ├── 访问→加购
        └── 加购→付款
```

### Step 3: 复利增长路径

```
复利公式: 增长 = 基础量 × (1+增长率)^时间

复利来源:
  □ 口碑传播: K因子 = 平均邀请人数 × 邀请转化率
    - K>1 → 自增长
    - K<1 → 需外部输血

  □ 留存复购: 复购率提升 → LTV提升 → 可投入更多获客

  □ 规模效应: 规模增长 → 成本下降 → 价格优势 → 更多用户

  □ 网络效应: 用户增长 → 价值增长 → 更多用户

识别复利点:
  - 找出增长飞轮的"飞轮核"
  - 找到让飞轮转速加快的节点
  - 设计让飞轮自我加速的机制
```

### Step 4: 增长阶段性规划

```
阶段一: 0→1 (Product-Market Fit)
  目标: 找到PMF
  指标: 留存率、净推荐值(NPS)
  重点: 产品打磨、核心用户

阶段二: 1→N (Growth)
  目标: 规模化增长
  指标: 获客成本、增长速率
  重点: 渠道优化、单位经济模型

阶段三: N→品牌 (Defense)
  目标: 建立护城河
  指标: 市场份额、品牌认知
  重点: 品牌建设、生态扩展
```

## 增长逻辑检验清单

```
□ 逻辑自洽: 因果关系清晰
□ 数据可测: 指标可量化
□ 路径可行: 有明确的增长路径
□ 飞轮存在: 有自我强化的机制
□ 壁垒匹配: 增长能建立壁垒
□ 复利可期: 有复利增长的可能
□ 时间合理: 增长所需时间可接受
```

## 使用示例

```
输入: 某 SaaS 产品

Step 1: 画布梳理
  价值主张: 提升团队协作效率
  收入来源: 订阅费
  核心资源: 产品技术

Step 2: 指标拆解
  北极星: MRR (月度经常性收入)
    ├── 新增MRR
    │   ├── 新增客户数
    │   └── 平均客单价
    └── 留存MRR
        ├── 续费率
        └── 扩增率

Step 3: 复利路径
  飞轮: 产品好用 → 口碑传播 → 用户增长 → 数据积累 → 产品更强
  K因子: 0.3 → 目标1.0
  留存: 90% → LTV提升

Step 4: 阶段规划
  PMF: 12个月
  增长: 18个月
  壁垒: 24个月

结论: 增长逻辑清晰，阶段性目标明确
```
