## 核心框架

### 框架一：内容调性三要素模型

```
         ┌─────────────────────────────────────────┐
         │              内容调性                    │
         ├─────────────┬─────────────┬─────────────┤
         │   品牌调性   │   用户调性   │   平台调性   │
         ├─────────────┼─────────────┼─────────────┤
         │ 品牌基因     │ 用户期待     │ 平台特性     │
         │ 核心价值     │ 情感共鸣     │ 表达方式     │
         │ 差异化主张   │ 沟通偏好     │ 流量规则     │
         └─────────────┴─────────────┴─────────────┘
                         ↓
              ┌─────────────────────┐
              │    调性定位文档      │
              │  (风格指南+创作规范)  │
              └─────────────────────┘
```

**三要素关系**：
- 品牌调性：我是谁（核心）
- 用户调性：他们是谁（锚点）
- 平台调性：我在哪（适配）

### 框架二：调性定位四象限

| 维度 | 温暖亲切 | 专业权威 | 年轻活力 | 高端大气 |
|------|----------|----------|----------|----------|
| 语言风格 | 口语化 | 术语化 | 潮流化 | 精致化 |
| 内容视角 | 用户视角 | 专家视角 | 朋友视角 | 精英视角 |
| 视觉调性 | 暖色系 | 冷色系 | 撞色系 | 黑白系 |
| 互动风格 | 平等交流 | 单向输出 | 趣味互动 | 低调互动 |
| 适用品牌 | 消费品/服务 | 专业服务/科技 | 品牌/年轻人 | 奢侈品/高端 |

### 框架三：内容人格五维模型

| 维度 | 选项示例 | 权重 |
|------|----------|------|
| 专业度 | 小白友好 ↔ 专家级 | 20% |
| 温度感 | 冰冷理性 ↔ 温暖感性 | 20% |
| 年龄感 | 成熟稳重 ↔ 年轻活力 | 20% |
| 距离感 | 高高在上 ↔ 平等亲近 | 20% |
| 幽默感 | 严肃正经 ↔ 风趣幽默 | 20% |

## 执行步骤

### Step 1: 梳理品牌属性

**目的**：理解品牌本质，为调性定位提供基础

**操作流程**：

1. **提取品牌核心元素**

   | 元素 | 问题 | 提炼 |
   |------|------|------|
   | 品牌使命 | 品牌存在的意义是什么？ | [一句话使命] |
   | 核心价值 | 品牌最核心的理念是什么？ | [3个关键词] |
   | 品牌个性 | 品牌像一个人，是什么样的人？ | [人格画像] |
   | 差异化 | 与竞品最大的不同是什么？ | [独特卖点] |
   | 用户承诺 | 品牌给用户的承诺是什么？ | [承诺内容] |

2. **品牌调性关键词提取**

   ```
   品牌调性词汇表：
   
   应该有的（加分词）：
   ├─ [温暖/专业/创新/可靠...]
   └─ 来源：品牌使命/价值/差异化
   
   应该没有的（减分词）：
   ├─ [冷漠/刻板/敷衍...]
   └─ 来源：竞品对比/用户反馈
   ```

3. **品牌DNA提炼**

   ```
   品牌DNA公式：
   [品类属性] + [情感属性] + [风格属性] = 品牌调性基调
   
   示例：
   咖啡品牌 = 提神醒脑(功能) + 生活方式(情感) + 文艺小资(风格)
   = "温暖陪伴的文艺生活家"
   
   科技品牌 = 效率提升(功能) + 科技感(情感) + 简约酷炫(风格)
   = "专业极客的效率伙伴"
   ```

---

### Step 2: 分析目标用户

**目的**：理解用户期待，匹配调性方向

**操作流程**：

1. **用户画像整理**

   ```
   目标用户画像要素：
   ├─ 基本属性：年龄/性别/地域/职业
   ├─ 行为特征：触媒习惯/内容偏好/购买行为
   ├─ 心理特征：价值观/兴趣爱好/生活方式
   └─ 内容期待：喜欢什么风格/讨厌什么风格
   ```

2. **用户调性期待分析**

   | 用户类型 | 调性期待 | 内容偏好 |
   |----------|----------|----------|
   | 职场新人 | 鼓励成长 | 干货+情感 |
   | 职场精英 | 效率专业 | 深度+洞察 |
   | 宝妈群体 | 温暖实用 | 轻松+贴心 |
   | Z世代 | 有趣好玩 | 潮流+个性 |

3. **用户语言习惯收集**

   ```
   用户原声收集：
   ├─ 他们常说的词：[收集30+高频词]
   ├─ 他们喜欢的表达：[网络用语/梗]
   ├─ 他们讨厌的表达：[说教/爹味/油腻]
   └─ 转化为自己语言：[品牌化表达]
   ```

---

### Step 3: 匹配平台特性

**目的**：让调性在目标平台有效传达

**操作流程**：

1. **主流平台调性特征**

   | 平台 | 主流调性 | 内容风格 | 互动特点 |
   |------|----------|----------|----------|
   | 公众号 | 深度专业 | 长图文 | 留言互动 |
   | 抖音 | 娱乐有趣 | 短视频 | 短视频互动 |
   | 小红书 | 精致生活 | 图文/短视频 | 种草安利 |
   | B站 | 年轻有趣 | 中长视频 | 弹幕/评论 |
   | 微博 | 热点话题 | 短资讯 | 话题讨论 |
   | 视频号 | 社交传播 | 短视频 | 朋友圈扩散 |

2. **平台调性适配表**

   ```
   基础调性：温暖陪伴的文艺生活家
   
   平台适配：
   ├─ 公众号：温暖讲述+深度干货，文章风格偏文艺
   ├─ 抖音：日常vlog+生活美学，风格轻松治愈
   ├─ 小红书：精致图文+好物种草，风格小资精致
   └─ B站：知识分享+生活方式，风格真实有趣
   ```

3. **平台表达差异对比**

   | 维度 | 公众号 | 小红书 | 抖音 |
   |------|--------|--------|------|
   | 标题 | 悬念/深度 | 痛点/利益点 | 悬念/冲突 |
   | 开头 | 场景引入 | 直接利他 | 黄金3秒 |
   | 配图 | 统一风格 | 精致好看 | 有冲击力 |
   | 结尾 | 总结升华 | 行动引导 | 互动召唤 |

### Step 4: 定义内容调性体系

**目的**：形成可执行的调性规范文档

**操作流程**：

1. **内容人格定义**

   ```
   【内容人格定义模板】
   
   名称：[给这个人格起个名字]
   
   外在表现：
   ├─ 年龄感：[如：25-30岁的朋友]
   ├─ 性别感：[中性/偏女性/偏男性]
   ├─ 身份感：[如：懂行的朋友/专业顾问/同行伙伴]
   └─ 距离感：[如：可以约饭的朋友]
   
   说话风格：
   ├─ 语速：[快/慢/中等]
   ├─ 语气：[亲切/专业/活泼/沉稳]
   ├─ 用词：[专业术语多/通俗易懂/偶尔网络用语]
   └─ 句式：[短句为主/长句为主/长短结合]
   
   性格标签：
   [3-5个性格关键词，如：专业但不说教、有趣但不低俗、
    温暖但不做作、有态度但不极端]
   ```

2. **调性表达清单**

   ```
   【调性表达指南】
   
   我们这样说（✓）：
   ├─ "[口语化表达]"
   ├─ "[共情式表达]"
   └─ "[利他式表达]"
   
   我们不这样说（✗）：
   ├─ "[说教式表达]"
   ├─ "[过于官方]"
   └─ "[负能量/抱怨]"
   ```

3. **视觉调性规范**

   ```
   视觉调性要素：
   ├─ 主色调：[颜色+情绪，如：温暖的米白]
   ├─ 字体风格：[如：现代简约，有设计感]
   ├─ 图片风格：[如：真实生活场景，略微调色]
   ├─ 排版风格：[如：留白多，呼吸感]
   └─ 整体感觉：[如：清爽、干净、精致]
   ```

---

### Step 5: 建立调性执行规范

**目的**：确保团队统一执行调性标准

**操作流程**：

1. **内容审核标准**

   | 检查项 | 合格标准 | 不合格表现 |
   |--------|----------|------------|
   | 语言风格 | 符合设定语气 | 过于官方/说教 |
   | 用词规范 | 使用用户语言 | 生硬术语堆砌 |
   | 情感调性 | 温暖有共鸣 | 冷漠/过度煽情 |
   | 价值观 | 正面积极 | 负面/争议 |
   | 品牌元素 | 适度植入 | 硬广感 |

2. **调性示例库**

   ```
   【正面示例库】
   标题示例：
   1. "别再无效加班了，这个方法帮你准时下班"
   2. "月入3万和3千的人，差的不只是能力"
   3. "作为一个普通人，我是怎么从月薪5000到3万的"
   
   开头示例：
   "你是不是也经常……"
   "最近收到很多私信问……"
   "今天想聊聊我踩过的一个坑……"
   
   结尾示例：
   "希望今天的分享对你有帮助~"
   "你们还有什么问题，评论区告诉我"
   "觉得有用的别忘了点赞+在看"
   ```

3. **调性测试方法**

   ```
   新内容调性自检：
   1. 读出来像谁在说？（是否符合人格定义）
   2. 目标用户会怎么看？（是否符合用户期待）
   3. 竞争对手会怎么说？（是否有差异化）
   4. 换平台能复制吗？（是否保留核心调性）
   ```

## 使用示例

### 完整场景：美妆品牌内容调性定位

**背景**：
- 品牌：国产新锐美妆品牌
- 目标用户：20-28岁年轻女性
- 平台：主攻小红书+抖音

**Step 1: 品牌属性梳理**

```
品牌DNA：
├─ 品类：平价美妆
├─ 情感：自信表达、真实不完美
├─ 风格：年轻活力、真实自然
└─ 调性基点："真实自信的平价好物"

关键词提取：
✓加分词：真实/自然/有趣/平价/好用/接地气
✗减分词：过于精致/高高在上/大牌模仿/爹味说教
```

**Step 2: 用户分析**

```
用户画像：
├─ 典型：大学生/初入职场女生
├─ 痛点：预算有限但想变美、容易被博主种草
├─ 期待：真实测评、平价好物、不踩雷
└─ 喜欢：真实自然、反感过度精修和广告

用户语言：
├─ 高频词：绝绝子、yyds、平价替代、真心好用
├─ 喜欢：真实素人分享、有对比图
└─ 讨厌：广告感强、过度营销、货不对板
```

**Step 3: 调性定位文档**

```
【品牌内容人格定义】

名称：小美（真实闺蜜）

外在表现：
├─ 年龄感：25岁的大学闺蜜
├─ 身份：懂美妆的同龄朋友
├─ 距离感：可以一起逛街聊护肤的闺蜜

说话风格：
├─ 语气：亲切、真实、不端着
├─ 用词：偶尔网络用语、但不过度
├─ 句式：短句为主，像发微信

性格标签：
真实不装 | 有话直说 | 贴心但不碎碎念 | 有审美但不挑剔

【表达清单】

✓ 我们这样说：
├─ "这盘眼影真的绝了，显色度绝绝子"
├─ "作为黄皮亲测，这个色号巨显白"
├─ "学生党冲这个，性价比绝了"
└─ "真心分享，无广放心食用"

✗ 我们不这样说：
├─ "本产品采用XXX专利技术……"
├─ "精致的猪猪女孩必备……"
├─ "女人一定要对自己好一点……"
└─ "这款产品适合所有肤质……"
```
