## 执行步骤

### Step 1: 目标确认

```
激励目标:
  □ 拉新激励 → 降低获客成本
  □ 激活激励 → 提升首活
  □ 留存激励 → 提升留存率
  □ 变现激励 → 提升付费率
  □ 传播激励 → 提升传播系数

选择激励目标:
  核心目标: ________
  次要目标: ________
```

### Step 2: 激励方式选择

```
选择原则:
  □ 目标导向
    - 拉新 → 物质激励为主
    - 留存 → 精神+功能激励
    - 变现 → 利益激励

  □ 用户分层
    - 新用户 → 简单直接激励
    - 活跃用户 → 差异化激励
    - 沉默用户 → 召回激励

  □ 成本控制
    - 边际成本趋零
    - 虚拟价值合理利用
```

### Step 3: 梯度设计

```
激励梯度:
  第一层: 首次行为 (0→1)
    - 极简门槛
    - 立即反馈
    - 建立习惯

  第二层: 持续行为 (1→N)
    - 递进奖励
    - 里程碑激励
    - 保持新鲜感

  第三层: 忠诚行为 (忠诚)
    - 会员权益
    - 专属体验
    - 情感连接
```

### Step 4: ROI测算

```
激励ROI:
  ROI = (激励带来的增量价值) / (激励成本)

测算:
  □ 增量价值 = 提升的留存 × 单用户价值
  □ 激励成本 = 激励发放成本 + 运营成本
  □ ROI >= 1 → 可持续

预算分配:
  - 拉新激励: __%
  - 留存激励: __%
  - 变现激励: __%
  - 传播激励: __%
```

## 激励设计检查清单

```
□ 目标清晰: 激励针对什么行为
□ 方式匹配: 激励方式与目标匹配
□ 梯度合理: 从易到难，循序渐进
□ 即时反馈: 用户能立即感知
□ 成本可控: ROI >= 1
□ 可持续: 不是一次性消耗
□ 无套利: 不会被薅羊毛
```

## 使用示例

```
目标: 提升30日留存率

设计方案:
  第一层 (0→7天):
    - 注册礼: 3元红包
    - 首日礼: 1元
    - 7日礼: 5元 (连续7天登录)
    成本: 人均8元

  第二层 (7→30天):
    - 每日签到: 随机积分
    - 周任务: 3元券
    - 成就激励: 虚拟勋章
    成本: 人均5元/月

  第三层 (30天+):
    - 会员等级权益
    - 专属客服
    - 生日礼包

ROI测算:
  30日留存提升: 10%
  单用户价值: 50元
  增量价值: 5元/人
  总成本: 13元/人
  ROI: 0.38 < 1 → 需要调整

调整方案:
  - 降低物质激励
  - 增加虚拟价值
  - 优化到ROI >= 1
```
