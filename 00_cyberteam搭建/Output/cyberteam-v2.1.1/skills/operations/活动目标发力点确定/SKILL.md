## 核心框架

### 框架一：目标-漏斗-发力点模型

```
                    ┌───────────────┐
                    │   活动目标    │  ← 最终要达成的结果
                    │   (北极星)    │
                    └───────┬───────┘
                            │
                    ┌───────┴───────┐
                    │   转化漏斗    │  ← 达成目标的路径
                    │   (各环节)    │
                    └───────┬───────┘
                            │
                    ┌───────┴───────┐
                    │   发力点      │  ← 需要重点优化的环节
                    │   (杠杆支点)  │
                    └───────────────┘
```

**核心逻辑**：
- 目标拆解为漏斗各环节
- 各环节存在转化率
- 优化转化率最低的环节（瓶颈）= 最高ROI
- 发力点 = 杠杆效应最大的环节

### 框架二：发力点识别四步法

```
Step 1: 结构化目标 → 目标是什么？
    ↓
Step 2: 拆解变量因子 → 受哪些因素影响？
    ↓
Step 3: 分析当前数据 → 哪个环节最薄弱？
    ↓
Step 4: 计算发力价值 → 优化哪里最有效？
```

### 框架三：发力价值评估矩阵

| 环节 | 当前值 | 理论值 | 提升空间 | 优化难度 | 发力优先级 |
|------|--------|--------|----------|----------|------------|
| 曝光 | 100万 | - | - | - | - |
| 点击 | 10万 | 10% | - | - | - |
| 注册 | 3万 | 30% | 高 | 低 | P1 |
| 转化 | 6000 | 20% | 中 | 高 | P2 |

**发力优先级 = 提升空间 / 优化难度**

## 执行步骤

### Step 1: 结构化活动目标

**目的**：将模糊的活动目标转化为可量化、可拆解的结构

**操作流程**：

1. **明确最终目标**

   ```
   目标模板：
   在 [时间周期] 内，通过 [活动名称]，
   达成 [核心KPI] [数值]，环比 [基准] [变化幅度]。
   
   示例：
   在618活动期间（6月15-18日），
   达成GMV 500万，环比日常均值增长50%。
   ```

2. **拆解目标公式**

   ```
   GMV = UV × 转化率 × 客单价
   GMV = 访客数 × 下单率 × 平均订单金额
   
   新付费用户 = 曝光 × 点击率 × 注册率 × 首付率
   ```

3. **列出所有变量**

   | 变量 | 定义 | 基准值 | 目标值 |
   |------|------|--------|--------|
   | UV | 独立访客 | 50万 | 80万 |
   | 点击率 | 点击/曝光 | 15% | 20% |
   | 注册率 | 注册/点击 | 60% | 65% |
   | 首付率 | 首单/注册 | 25% | 30% |
   | 客单价 | GMV/订单 | 200元 | 220元 |

---

### Step 2: 拆解关键影响因子

**目的**：理解每个变量受什么因素影响

**拆解模板**：

```
UV = 站内流量 + 站外流量

站内流量 = 首页流量 + 频道流量 + 搜索流量 + 推送流量
站外流量 = 广告投放 + 社交传播 + 合作渠道 + 私域唤醒

转化率 = f(优惠力度, 商品吸引力, 用户匹配度, 页面体验)

客单价 = f(品类结构, 满减门槛, 推荐精准度, 用户购买力)
```

**关键因子清单**：

| 目标变量 | 直接因子 | 间接因子 |
|----------|----------|----------|
| UV | 渠道数量, 投放预算, 素材质量 | 活动知名度, 时机选择 |
| 点击率 | 入口位置, 素材吸引力, 文案诱惑力 | 推送时机, 用户精准度 |
| 转化率 | 优惠力度, 商品价格, 页面体验 | 用户质量, 流量质量 |
| 客单价 | 品类结构, 满减设计, 搭配推荐 | 用户画像, SKU深度 |

---

### Step 3: 分析当前数据瓶颈

**目的**：识别最薄弱、最需要优化的环节

**操作流程**：

1. **绘制转化漏斗图**

   ```
   曝光(1000万) ════════════════════════════════ 100%
       ↓ CTR 10%
   点击(100万)   ════════════════════════════════ 10%
       ↓ 注册率 50%
   注册(50万)    ════════════════════════════════ 5%
       ↓ 首付率 20%
   首单(10万)    ════════════════════════════════ 1%
       ↓ 客单价 200元
   GMV(2000万)   ════════════════════════════════ 0.2%
   ```

2. **计算各环节转化效率**

   | 环节 | 当前转化 | 行业均值 | 理论上限 | 差距 |
   |------|----------|----------|----------|------|
   | 点击率 | 10% | 12% | 20% | -10% |
   | 注册率 | 50% | 55% | 80% | -5% |
   | 首付率 | 20% | 25% | 40% | -5% |
   | 客单价 | 200元 | 220元 | 300元 | -20元 |

3. **识别瓶颈环节**

   ```
   瓶颈识别原则：
   ├─ 绝对瓶颈：转化率最低的环节
   ├─ 相对瓶颈：与目标差距最大的环节
   └─ 杠杆瓶颈：优化收益/成本比最高的环节
   
   综合判断：
   → 首付率20%（最低）+ 提升空间大（可翻倍）
   → 核心发力点：提升首付转化
   ```

---

### Step 4: 确定发力方向和策略

**目的**：制定针对性的优化方案

**操作流程**：

1. **计算各环节发力价值**

   ```
   发力价值公式：
   价值 = 当前值 × (目标值 - 当前值) × 优化成本效益
   
   示例计算（GMV=1000万基准）：
   ├─ 提升点击率10%→15%：GMV+500万（需优化素材）
   ├─ 提升首付率20%→30%：GMV+500万（需优化转化）
   ├─ 提升客单价200→250：GMV+250万（需优化SKU）
   └─ 三者同步优化：GMV+1250万（协同效应）
   ```

2. **制定发力策略**

   | 发力点 | 当前问题 | 优化策略 | 预期效果 | 资源投入 |
   |--------|----------|----------|----------|----------|
   | 首付率 | 优惠券门槛高 | 降低门槛，增发专属券 | +30%首付 | 低 |
   | 点击率 | 素材创意弱 | A/B测试迭代素材 | +50%点击 | 中 |
   | 客单价 | SKU结构差 | 优化推荐逻辑 | +15%客单 | 高 |

3. **资源配置优先级**

   ```
   资源分配公式：
   最优配置 = 最大化 (发力价值 / 资源投入)
   
   优先级排序：
   1. 首付率优化：价值高 + 投入低 → 优先
   2. 点击率优化：价值高 + 投入中 → 其次
   3. 客单价优化：价值中 + 投入高 → 视情况
   ```

## 使用示例

### 场景：618活动目标发力点分析

**背景**：
- 活动目标：GMV 500万（日常150万的333%）
- 现有资源：预算50万，团队10人

**Step 1: 结构化目标**

```
GMV = UV × 点击率 × 注册率 × 首付率 × 客单价
500万 = UV × 15% × 60% × 20% × 220元

倒推各环节目标：
- UV需达到：500万 / (15%×60%×20%×220) = 126万
- 点击率：维持15%
- 注册率：维持60%
- 首付率：需提升至25%（当前20%）
- 客单价：维持220元
```

**Step 2: 拆解因子**

```
首付率 = f(优惠券力度, 首单商品, 页面体验, 信任度)

关键因子分析：
├─ 优惠券：当前满100减15，门槛偏高
├─ 首商品：缺乏高性价比引流款
├─ 页面体验：首单引导不够突出
└─ 信任度：老用户基础OK
```

**Step 3: 瓶颈识别**

```
漏斗分析：
曝光100万 → 点击15万(15%) → 注册9万(60%) → 首单1.8万(20%) → GMV400万

当前问题：
├─ 首付率20%是最大瓶颈（转化最低）
├─ 低于行业均值25%
└─ 提升空间67%（20%→25%）
```

**Step 4: 确定发力点**

```
发力价值计算：
首付率+5% = GMV+100万（贡献67%差距）
点击率+3% = GMV+20万（贡献13%差距）

发力策略：
1. 核心发力：降低首单门槛
   - 推出99减20专属首单券
   - 选品增加99元以下引流款
   - 预期首付率提升至28%

2. 辅助发力：提升点击率
   - A/B测试3套素材
   - 优化推送时机
   - 预期点击率提升至18%

资源分配：
- 首单券预算：20万
- 素材测试：5万
- 引流款补贴：10万
```
