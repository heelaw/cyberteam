## 思维框架

### 什么是第一性原理

> "我相信第一性原理推理...从物理学的角度看待世界，
> 把事物归结为最基本的事实，然后从那里向上推理。"
> —— Elon Musk

### 三步法

```
1. 识别假设
   - 现在的"常识"是什么？
   - 这些假设有多确定？
   - 谁告诉你这些假设的？

2. 分解到基本事实
   - 什么是无可争议的？
   - 什么是物理上可能的？
   - 什么是经济学上可行的？

3. 从零重构
   - 如果从零开始，你会怎么做？
   - 有没有完全不同的实现路径？
   - 什么技术进步使新方案成为可能？
```

### 第一性 vs 类比思维

| 类比思维 | 第一性原理 |
|---------|----------|
| "别人这么做，我也这么做" | "从物理/经济/技术上，什么是可能的？" |
| 基于经验 | 基于事实 |
| 增量改进 | 颠覆性创新 |
| 风险低 | 风险高但回报高 |

## 注入模板

```
## 第一性原理分析

**识别当前假设**：
当前人们相信的"常识"是什么？

**挑战假设**：
- 这个假设是最基本的吗？
- 如果这个假设错了呢？
- 什么情况下这个假设不成立？

**回归基本事实**：
物理层面：
- 技术上可行吗？
- 物理极限是什么？

经济层面：
- 成本结构是什么？
- 规模效应如何？

价值层面：
- 用户真正需要什么？
- 价值创造的来源是什么？

**从零重构**：
如果今天重新开始，没有任何包袱：
1. 我会怎么设计？
2. 什么阻碍了这种做法？
3. 有没有办法消除这些阻碍？
```

## 使用示例

### 示例1: 电动汽车

```
传统思维: "汽车就应该是烧油的"
假设: 电池太贵、续航不够、充电太慢

第一性原理:
物理事实:
- 电池成本: 2008年$600/kWh → 2024年$100/kWh
- 太阳能发电成本持续下降
- 电机效率>90%，内燃机<30%

从零重构:
- 如果电池足够便宜，电动车成本优势？
- 如果充电基础设施足够密布？
- 马斯克: "重新发明汽车"

结果: 特斯拉
```

### 示例2: 在线教育

```
传统思维: "教育需要教室、老师、时间同步"
假设: 在线教育效果不如线下

第一性原理:
物理事实:
- 互联网带宽持续提升
- AI 辅助学习效率提升

教育本质:
- 知识的传递？
- 技能的习得？
- 学习动机？

从零重构:
- 如果没有教室，教育可以是什么形式？
- 个性化学习 vs 统一教学
- 异步 vs 同步

结果: Coursera, 可汗学院, AI辅导
```

## 决策检查清单

```
□ 当前的主流方案是什么？
□ 这些方案背后的假设是什么？
□ 这些假设是最基本的吗？
□ 如果假设错误呢？
□ 从物理/经济/价值角度，基本事实是什么？
□ 从零开始，你会怎么做？
□ 什么阻碍了从零的方案？
□ 有没有办法消除阻碍？
```
