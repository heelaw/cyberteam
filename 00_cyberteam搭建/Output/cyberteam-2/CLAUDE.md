# CyberTeam 项目指南

## 项目概述

**CyberTeam** 是一款 AI 模拟公司协作系统，核心理念：**一句话交代，整个团队干活**。

用户只需输入一个业务目标，AI 模拟公司即可自动完成：目标分析 → 方案制定 → 部门执行 → 结果交付的全流程。

## 核心组件

### 1. CEO Agent

- **功能**：问题分析 + 任务分发 + 计划审核
- **能力**：
  - 5W1H + MECE 分析
  - 100 思维框架（思考天团）
  - 30+ 运营专家（项目组1+2）
  - 150+ 执行专家（agency-agents）

### 2. 运营 Experts（自研）

| 来源 | Agent数量 | 核心能力 |
|------|-----------|----------|
| 【项目组1】 | ~16个 | 增长、文案、用户、活动、策略 |
| 【项目组2】 | ~30个 | 指标、数据、商业、财务、管理 |

### 3. Skills

| Skill | 功能 |
|-------|------|
| goal-driven | 目标驱动循环，持续工作直到达成 |
| pua-supervisor | 监督 Agent，防止偷懒 |
| office-hours | YC 六问，计划审核 |
| investigate | 根因分析 |

## 使用流程

### 基本流程

```
用户输入 → CEO分析 → 计划审核 → Agent选择 → 执行+PUA → 完成
```

### 详细流程

1. **用户输入业务目标**
2. **CEO分析**（5W1H + MECE）
3. **思维框架选择**（100个思维专家）
4. **运营专家选择**（45+运营Agent）
5. **计划审核**（Office Hours + 专家评审）
6. **任务分发**（ClawTeam协作）
7. **启动执行**（并行处理）
8. **PUA监督**（监控进度，防止偷懒）
9. **Goal-Driven循环**（持续工作直到达成）
10. **输出结果**

## 注意事项

1. **一句话交代**：用户只需描述目标，AI 自动完成全部流程
2. **所有专家参与**：确保相关思维框架和运营专家都参与分析
3. **PUA监督**：在执行过程中监控进度
4. **持续工作**：使用 goal-driven 循环直到目标达成
5. **结果聚合**：汇总所有 Agent 的输出
