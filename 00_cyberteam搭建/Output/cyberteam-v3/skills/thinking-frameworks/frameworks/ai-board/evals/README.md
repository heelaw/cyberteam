# AI私董会专家测试用例

## 正面测试

### 测试1: 创业方向决策
- 输入:
```
【AI私董会请求】
案主问题：是否All-in AI赛道
背景：现有业务稳定
已做决策：暂无
```
- 期望输出:
  - 幕僚提问（每位至少2个）
  - 幕僚反馈（每位都有）
  - 案主承诺
  - 综合总结
- 验证项:
  - [ ] 四位幕僚都有
  - [ ] 人设风格一致
  - [ ] 有行动承诺

### 测试2: 公司战略选择
- 输入:
```
【AI私董会请求】
案主问题：新市场进入时机
背景：当前市场增长放缓
已做决策：考虑多元化
```
- 期望输出:
  - 各幕僚视角
  - 多元建议
  - 案主总结
- 验证项:
  - [ ] 视角多元
  - [ ] 不冲突但有碰撞
  - [ ] 最终有决策

## 负面测试

### 测试1: 人设混乱
- 输入:
```
【AI私董会请求】
案主问题：投资选择
```
- 期望行为: 不能让巴菲特说乔布斯的话
- 验证项:
  - [ ] 巴菲特：简单直接投资视角
  - [ ] 盖茨：数据逻辑视角
  - [ ] 马斯克：第一性原理视角
  - [ ] 乔布斯：简洁产品视角

### 测试2: 无结论
- 输入:
```
【AI私董会请求】
案主问题：复杂战略问题
```
- 期望行为: 不能只有讨论没有决策
- 验证项:
  - [ ] 有案主行动承诺
  - [ ] 有下一步
  - [ ] 不悬而未决
