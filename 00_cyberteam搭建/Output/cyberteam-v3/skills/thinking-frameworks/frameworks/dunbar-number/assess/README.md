# 邓巴数字专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 邓巴数字理论是否正确（150人上限、四层社交圈、认知限制） |
| 可执行性 | 30% | 输出是否可操作（层级诊断、优化策略、边界识别是否明确） |
| 独特性 | 20% | 是否有差异化（与传统社交建议的区分、邓巴数字独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（场景、当前规模、目标、资源限制）
- [ ] 有输出格式（层级诊断、场景匹配度、优化策略、行动清单）
- [ ] 有分析框架（四步法：层级诊断→场景应用→优化策略→边界识别）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（社群运营、组织设计、人脉经营）

## 核心检验点

### 理论准确性
- [ ] 150人邓巴数字正确
- [ ] 四层社交圈划分正确（5/15/50/150）
- [ ] 认知限制概念有效

### 框架完整性
- [ ] 层级诊断有清单
- [ ] 场景应用有对照表
- [ ] 优化策略有具体方法

### 输出有效性
- [ ] 规模建议合理
- [ ] 层级调整可执行
- [ ] 预警信号可识别

- 邓巴数字与组织设计：团队规模优化
- 邓巴数字与人脉经营：个人关系维护
