# 蘑菇管理专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 蘑菇管理理论是否正确（蘑菇法则、成长阶段、管理策略） |
| 可执行性 | 30% | 输出是否可操作（阶段识别、策略选择、行动建议是否明确） |
| 独特性 | 20% | 是否有差异化（与一般职场指导的区分、蘑菇管理独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（当前处境、工作类型、领导风格）
- [ ] 有输出格式（阶段识别、策略建议、行动清单）
- [ ] 有分析框架（蘑菇法则分析、成长阶段判断、管理策略）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（主动领导、向上管理、SWOT分析）

## 核心检验点

### 理论准确性
- [ ] 蘑菇法则正确（被忽视、低谷期）
- [ ] 成长阶段划分合理
- [ ] 管理策略有效

### 框架完整性
- [ ] 阶段识别有判断标准
- [ ] 策略建议有分类
- [ ] 行动清单有具体步骤

### 输出有效性
- [ ] 阶段定位准确
- [ ] 策略可落地
- [ ] 效果可评估

## 相关专家

- 蘑菇管理与向上管理：如何在蘑菇期获得上级支持
- 蘑菇管理与主动领导：主动争取机会而非被动等待
- 蘑菇管理与SWOT分析：分析内外部环境找到突破口
