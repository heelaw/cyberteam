# 前景理论专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 前景理论是否正确（参照点、敏感度、损失厌恶、概率权重） |
| 可执行性 | 30% | 输出是否可操作（参照点设定、框架转换、理性校准是否明确） |
| 独特性 | 20% | 是否有差异化（与传统决策理论的区分、前景理论独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（决策情境、当前框架、潜在收益/损失）
- [ ] 有输出格式（参照点分析、框架诊断、理性校准建议）
- [ ] 有分析框架（三步法：参照点→偏差识别→理性校准）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（预期理论、二阶思维、机会成本）

## 核心检验点

### 理论准确性
- [ ] 参照点原理正确
- [ ] 损失厌恶系数约2.25
- [ ] 敏感度递减在收益和损失区域都存在

### 框架完整性
- [ ] 参照点设定有方法
- [ ] 偏差识别有清单
- [ ] 理性校准有步骤

### 输出有效性
- [ ] 框架转换可操作
- [ ] 决策质量可评估
- [ ] 风险态度可调整

## 相关专家

- 前景理论与预期理论：价值函数的数学表达
- 前景理论与二阶思维：评估决策的长期后果
- 前景理论与机会成本：全面评估选择代价
