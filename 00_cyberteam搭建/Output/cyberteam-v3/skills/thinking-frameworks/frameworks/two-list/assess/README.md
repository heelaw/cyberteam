# 双目标清单专家评估标准

## 评估维度

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 完整性 | 20% | 章节是否齐全（触发词、输入格式、分析框架、输出格式、Critical Rules） |
| 准确性 | 30% | 双目标清单理论是否正确（目标分类、优先级判断、放弃清单） |
| 可执行性 | 30% | 输出是否可操作（目标分类、行动清单、放弃策略是否明确） |
| 独特性 | 20% | 是否有差异化（与传统目标管理的区分、双清单独特价值） |

## 及格线

- 标准模式: >= 85分
- 严格模式: >= 95分

## 快速检验清单

- [ ] 有触发词（5个场景以上）
- [ ] 有输入格式（当前目标、纠结原因、时间资源）
- [ ] 有输出格式（清单A、清档B、行动建议）
- [ ] 有分析框架（四步法：目标收集→分类→优先级→放弃清单）
- [ ] 有Critical Rules（必须遵守4条、禁止行为4条）
- [ ] 有协作配置（OKR、SWOT分析、艾森豪威尔矩阵）

## 核心检验点

### 理论准确性
- [ ] 目标分类标准清晰
- [ ] 放弃清单概念有效
- [ ] 优先级判断合理

### 框架完整性
- [ ] 目标收集完整
- [ ] 分类标准明确
- [ ] 放弃清单有策略

### 输出有效性
- [ ] 清单A和B区分清楚
- [ ] 行动建议可执行
- [ ] 定期复盘有机制

## 相关专家

- 双目标清单与OKR：目标与关键结果的对齐
- 双目标清单与SWOT分析：内外部环境评估辅助目标决策
- 双目标清单与艾森豪威尔矩阵：紧急/重要维度辅助优先级
