# 六标准详解

## 六个筛选标准

### 标准1: 能否反映用户从产品中获得核心价值

**核心问题**: 这个指标能否直接体现用户使用产品后获得了价值？

**评估要点**:
- 用户使用产品的根本目的是什么？
- 这个指标能否直接反映目的达成？
- 是过程指标还是结果指标？

**好指标案例**:
- YouTube: 观看时长 → 用户获得娱乐价值
- 知乎: 回答数 → 用户获得知识价值
- 淘宝: 销售额 → 用户完成交易价值

**坏指标案例**:
- 头条: 文章点击数 → 点击不代表阅读价值
- 公众号: 粉丝数 → 粉丝不代表活跃价值

---

### 标准2: 能否为产品达到长期商业目标奠定基础

**核心问题**: 这个指标与产品的长期盈利模式是否相关？

**评估要点**:
- 产品的长期商业模式是什么？
- 这个指标能否支撑商业模式成立？
- 是否能形成正向的商业闭环？

**好指标案例**:
- YouTube: 观看时长 → 广告变现基础
- Google: 搜索量 → 广告变现基础
- 钉钉: 发送消息数 → B端付费基础

**坏指标案例**:
- 短期营收 → 杀鸡取卵，伤害长期价值
- 单一供给侧指标 → 供需失衡

---

### 标准3: 能否反映用户活跃程度

**核心问题**: 这个指标能否体现用户的真实活跃状态？

**评估要点**:
- 是否是真实用户行为？
- 是否有"刷量"风险？
- 活跃定义是否清晰？

**好指标案例**:
- 日活跃用户 → 每日使用
- 观看时长 → 真实观看行为
- 发送消息数 → 真实互动

**坏指标案例**:
- 总注册用户数 → 历史累积，无活跃含义
- 总粉丝数 → 可能全是僵尸粉

---

### 标准4: 指标变好能否提示整个公司在往好的方向发展

**核心问题**: 指标提升是否意味着公司整体健康增长？

**评估要点**:
- 是否只关注单点而忽略整体？
- 供需双方是否平衡？
- 短期与长期是否平衡？

**好指标案例**:
- 淘宝: 销售额 → 供需两旺
- 美团: 订单数 → 平台活跃

**坏指标案例**:
- 美团: 仅关注商家数 → 供给过剩
- 美团: 仅关注骑手数 → 成本过高

---

### 标准5: 是否简单直观容易获得可拆解

**核心问题**: 这个指标是否易于理解、衡量和分解？

**评估要点**:
- 普通人能否理解？
- 数据是否容易获取？
- 能否拆解到各团队？

**好指标案例**:
- 观看时长 → 直观、易拆解
- 销售额 → 直观、易拆解
- 日活 → 直观、易拆解

**坏指标案例**:
- "过去三个月订房一间以上用户占注册用户比例" → 复杂难懂
- 综合满意度 → 主观难以量化

---

### 标准6: 是先导指标还是滞后指标

**核心问题**: 这个指标能否预测未来还是只能反映过去？

**评估要点**:
- 是结果指标还是过程指标？
- 能否提前预判趋势？
- 是否有滞后性？

**好指标案例**:
- 观看时长 → 先导，反映当下活跃
- 会议完成数 → 先导，实时反映
- 日活 → 先导，预测业务趋势

**坏指标案例**:
- 营收 → 滞后，用户已流失
- 年度品牌美誉度 → 严重滞后
- 净利润 → 滞后，季度或年度

---

## 使用建议

### 评分规则
- 每个标准通过得1分，否决得0分
- 5-6分：优秀指标
- 3-4分：可考虑，需补充
- 0-2分：坏指标，应排除

### 决策原则
- 不要只看总分，要看单项
- 某个标准为0分需特别关注
- 6个标准都重要，无优先级
