## 一、MCP 工具概览

### 1.1 工具分类

| 类别 | 工具 | 用途 | 优先级 |
|------|------|------|--------|
| **MiniMax** | web_search | 网络搜索 | ⭐⭐⭐ 最高 |
| **MiniMax** | understand_image | 图像分析 | ⭐⭐⭐ 最高 |
| **Filesystem** | read_text_file | 读取文件 | ⭐⭐ 高 |
| **Filesystem** | write_file | 写入文件 | ⭐⭐ 高 |
| **Filesystem** | directory_tree | 目录结构 | ⭐ 中 |
| **WebReader** | webReader | 网页读取 | ⭐⭐ 高 |

---

### 1.2 优先级规则

```yaml
优先级顺序 (必须遵守):
  1. MiniMax MCP (必须优先使用)
     - web_search: 搜索信息
     - understand_image: 分析图像

  2. Filesystem MCP
     - read_text_file: 读取文件
     - write_file: 写入文件

  3. WebReader MCP
     - webReader: 读取网页

  4. 内置工具 (Bash, Read, Write)
     - 兜底使用
```

## 四、文件操作最佳实践

### 4.1 read_text_file 使用规范

**函数**: `mcp__filesystem__read_text_file`

**参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| path | string | 是 | 文件路径 |
| head | number | 否 | 只读取前N行 |
| tail | number | 否 | 只读取后N行 |

---

### 4.2 读取策略

```
1. 小文件 (<100KB)
   └── 直接读取全部

2. 大文件 (>100KB)
   ├── 头部: head 100
   ├── 尾部: tail 100
   └── 指定范围: offset + limit

3. 查找特定内容
   └── 先用 Grep 定位，再读取

4. 二进制文件
   └── 不读取，提示用户
```

---

### 4.3 write_file 使用规范

**函数**: `mcp__filesystem__write_file`

**参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| path | string | 是 | 文件路径 |
| content | string | 是 | 文件内容 |

---

### 4.4 写入策略

```
1. 写入前检查
   ├── 检查目录是否存在
   ├── 检查文件是否已存在
   └── 备份重要文件

2. 写入操作
   ├── 使用 Write 工具 (覆盖)
   ├── 使用 Edit 工具 (修改部分)
   └── 避免频繁写入

3. 写入后验证
   └── 读取验证写入成功
```

---

### 4.5 输出标准

```markdown
## 文件操作报告

### 操作类型
- 读取 / 写入

### 文件信息
- 路径: [路径]
- 大小: [大小]
- 编码: [编码]

### 操作结果
| 检查项 | 结果 |
|--------|------|
| 存在性 | ✅/❌ |
| 可读性 | ✅/❌ |
| 权限 | ✅/❌ |
| 操作成功 | ✅/❌ |

### 内容摘要 (读取时)

[记录操作中的问题]
```

## 六、故障排查

### 6.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| **搜索无结果** | 关键词不当 | 更换关键词, 增加同义词 |
| **图像分析失败** | 格式不支持 | 确认支持 JPG/PNG/WebP |
| **文件读取失败** | 路径不存在 | 检查路径, 确认文件存在 |
| **网页读取失败** | 认证/权限 | 检查 URL 是否需要登录 |
| **写入失败** | 权限不足 | 检查目录权限 |

---

### 6.2 错误处理流程

```
1. 识别错误类型
   └── 网络错误 / 文件错误 / 格式错误

2. 尝试替代方案
   ├── 搜索: 更换关键词
   ├── 图像: 转换格式
   ├── 文件: 检查路径
   └── 网页: 使用其他 URL

3. 降级处理
   └── 使用内置工具兜底

4. 记录问题
   └── 记录错误日志
```

## 七、安全规范

### 7.1 数据安全

```
✅ 允许:
  - 读取项目目录内的文件
  - 读取公开网页内容
  - 写入项目目录内的文件

❌ 禁止:
  - 读取敏感配置文件 (含密钥)
  - 写入系统目录
  - 访问需要认证的私人页面
```

---

### 7.2 隐私保护

```
✅ 允许:
  - 搜索公开信息
  - 分析用户授权的图片
  - 读取公开的文档

❌ 禁止:
  - 搜索个人隐私信息
  - 分析未授权的图片
  - 爬取受版权保护的内容
```
