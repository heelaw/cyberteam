## 1. 10种内容结构模型

### 1.1 AIDA模型（直接转化）

**适用目标**: 促使立即购买或行动

**结构公式**:
```
Attention（注意）
    ↓
Interest（兴趣）
    ↓
Desire（欲望）
    ↓
Action（行动）
```

**填充模板**:
| 阶段 | 内容要求 | 字数占比 |
|------|----------|----------|
| Attention | 震撼性事实/惊人数字/引发好奇 | 10% |
| Interest | 与用户相关的问题/有价值信息 | 20% |
| Desire | 产品/服务展示/解决方案/情感共鸣 | 50% |
| Action | 明确CTA/紧迫感/便利性 | 20% |

**适用场景**: 电商详情页、信息流广告、促销文案

**示例框架**:
```
[Attention] "90%的女生都不知道的护肤秘密"
    ↓
[Interest] "皮肤问题反反复复，其实是这一步没做对"
    ↓
[Desire] "XX精华含有XX成分，能从根源解决XX问题"
    ↓
[Action] "现在点击，限时优惠XX元"
```

### 1.2 4P模型（信任构建）

**适用目标**: 建立品牌信任和专业形象

**结构公式**:
```
Problem（问题）
    ↓
Process（过程）
    ↓
Proof（证明）
    ↓
Promise（承诺）
```

**填充模板**:
| 阶段 | 内容要求 | 字数占比 |
|------|----------|----------|
| Problem | 普遍存在的问题/用户困扰 | 20% |
| Process | 解决方案的原理/步骤/方法 | 30% |
| Proof | 数据/案例/证言/对比 | 30% |
| Promise | 效果承诺/品牌保证 | 20% |

**适用场景**: 品牌推文、专业测评、解决方案类内容

**示例框架**:
```
[Problem] "为什么你的减肥总是失败？"
    ↓
[Process] "减肥失败不是因为不努力，而是方法错了..."
    ↓
[Proof] "我们的方法已经帮助10万人成功减重..."
    ↓
[Promise] "按照这个方法，你也能在XX天内看到效果"
```

### 1.3 情景-问题-方案模型（故事化共鸣）

**适用目标**: 引发情感共鸣，建立连接

**结构公式**:
```
Scene（情景）
    ↓
Problem（问题）
    ↓
Solution（方案）
```

**填充模板**:
| 阶段 | 内容要求 | 字数占比 |
|------|----------|----------|
| Scene | 具体场景描述，增强代入感 | 25% |
| Problem | 场景中的痛点/困难 | 35% |
| Solution | 产品/服务如何解决问题 | 40% |

**适用场景**: 种草内容、情感营销、私域引流

**示例框架**:
```
[Scene] "周一早上7点，闹钟响了第3遍..."
    ↓
[Problem] "看着镜子里那张疲惫的脸，今天又要顶着黑眼圈出门了..."
    ↓
[Solution] "直到我用了这个，10分钟搞定倦容妆..."
```

### 1.4 结构化Q&A模型（疑虑打消）

**适用目标**: 解答用户疑问，消除购买顾虑

**结构公式**:
```
Q1（核心问题）
    ↓
A1（详细回答）
    ↓
Q2（延伸问题）
    ↓
A2（详细回答）
    ↓
...
Final CTA（收尾行动）
```

**填充模板**:
| 元素 | 内容要求 | 字数占比 |
|------|----------|----------|
| Q部分 | 用户真实关心的问题 | 20% |
| A部分 | 详细、有据可查的回答 | 60% |
| Final CTA | 引导下一步行动 | 20% |

**适用场景**: FAQ内容、产品答疑、决策辅助文案

**示例框架**:
```
Q: 这个产品真的有效吗？
A: 根据XX第三方检测...（详细解释+证据）

Q: 和XX品牌相比有什么优势？
A: 从XX个维度对比...（具体数据）

Q: 售后有保障吗？
A: 我们承诺...（保障说明）

→ 还在犹豫？点击领取专属优惠
```

### 1.5 故事-启发结论模型（叙事启发）

**适用目标**: 通过故事引发思考，自然引出结论

**结构公式**:
```
Story（故事）
    ↓
Revelation（启示）
    ↓
Conclusion（结论）
```

**填充模板**:
| 阶段 | 内容要求 | 字数占比 |
|------|----------|----------|
| Story | 有戏剧性的真实故事 | 50% |
| Revelation | 从故事中提炼的启示 | 25% |
| Conclusion | 与产品/服务的关联 | 25% |

**适用场景**: 品牌故事、KOL种草、情感营销

**示例框架**:
```
[Story] "我用了3年时间，从月薪3000到年入百万..."
    ↓
[Revelation] "最重要的不是努力，而是选择..."
    ↓
[Conclusion] "就像当初选择XX一样，选择对了，事半功倍"
```

### 1.6 设置悬念-道出真相模型（好奇心驱动）

**适用目标**: 抓住注意力，引发好奇

**结构公式**:
```
Suspense（悬念）
    ↓
Development（展开）
    ↓
Truth（真相）
```

**填充模板**:
| 阶段 | 内容要求 | 字数占比 |
|------|----------|----------|
| Suspense | 反常识/违背直觉的陈述 | 15% |
| Development | 逐步展开，制造紧张感 | 45% |
| Truth | 出人意料又在情理之中 | 40% |

**适用场景**: 爆款内容、社交媒体、知识科普

**示例框架**:
```
[Suspense] "你一直在做的护肤方法，其实是错的"
    ↓
[Development] "90%的人都踩过这个坑，包括我...
    用了各种大牌护肤品，皮肤却越来越差...
    原来问题出在这里..."
    ↓
[Truth] "不是产品不好，而是护肤顺序错了！
    正确的顺序应该是..."
```

### 1.7 众多单点启发-留白模型（拼图式启发）

**适用目标**: 碎片化阅读时代的注意力捕获

**结构公式**:
```
Point 1（单点1）
    ↓
Point 2（单点2）
    ↓
Point 3（单点3）
    ↓
...
Gap（留白）
```

**填充模板**:
| 元素 | 内容要求 | 字数占比 |
|------|----------|----------|
| 单点 | 每个点独立有价值 | 70% |
| Gap | 引发好奇/引导互动 | 30% |

**适用场景**: 微博、朋友圈、短视频文案

**示例框架**:
```
[Point 1] "1. 早起比熬夜更高效"
[Point 2] "2. 番茄工作法真的有效"
[Point 3] "3. 每周运动3次就够了"
...
[Point 10] "10. 独处是最好的增值期"

[Gap] "那第11条呢？评论区告诉你..."
```

### 1.8 归纳型结构模型（金字塔论证）

**适用目标**: 系统性论证，建立权威

**结构公式**:
```
Conclusion（结论先行）
    ↓
Argument 1（论点1）
    ↓
Argument 2（论点2）
    ↓
Argument 3（论点3）
    ↓
Reinforcement（强化结论）
```

**填充模板**:
| 阶段 | 内容要求 | 字数占比 |
|------|----------|----------|
| Conclusion | 核心观点一句话 | 10% |
| Arguments | 3-5个支撑论点 | 60% |
| Reinforcement | 总结强调 | 30% |

**适用场景**: 专业长文、行业分析、深度测评

**示例框架**:
```
[Conclusion] "XX产品是2024年最值得买的护肤品"

[Argument 1] "从成分看：采用XX专利成分..."
[Argument 2] "从功效看：XX用户实测数据..."
[Argument 3] "从口碑看：XX平台好评率..."

[Reinforcement] "综合以上，XX产品实至名归"
```

### 1.9 演绎型结构模型（情节推进）

```

    ↓
Conflict（冲突）
    ↓
Development（发展）
    ↓
Climax（高潮）
    ↓
Resolution（结局）
```

**填充模板**:
| 阶段 | 内容要求 | 字数占比 |
|------|----------|----------|
| Setup | 背景介绍 | 15% |
| Conflict | 困难/挑战/危机 | 20% |
| Development | 解决过程 | 35% |
| Climax | 转折点 | 15% |
| Resolution | 美好结局 | 15% |

**适用场景**: 品牌故事、用户案例、纪录片式内容

**示例框架**:
```
[Setup] "我叫XX，是一个普通的上班族..."

```

### 1.10 对比型结构模型（优劣凸显）

**适用目标**: 突出差异化，建立选择理由

**结构公式**:
```
Before（使用前）
    ↓
After（使用后）
    ↓
Difference（差异点）
    ↓
Reason（原因）
```

**填充模板**:
| 阶段 | 内容要求 | 字数占比 |
|------|----------|----------|
| Before | 问题状态描述 | 20% |
| After | 改变后状态 | 30% |
| Difference | 关键差异点 | 30% |
| Reason | 差异原因解释 | 20% |

**适用场景**: 对比测评、效果展示、选购指南

**示例框架**:
```
[Before] "以前：早上起来脸油腻腻的，上妆特别难"
[After] "现在：皮肤清爽一整天，妆感服帖自然"
[Difference] "差距就在这一步..."
[Reason] "因为含有XX成分，能从根源调节水油平衡"
```

## 2. 内容结构选择决策树

### 2.1 四步决策流程

```
                    ┌─────────────────────┐
                    │ Step 1: 核心目标   │
                    │ 你的内容要达成什么？ │
                    └──────────┬──────────┘
                               │
            ┌──────────────────┼──────────────────┐
            ▼                  ▼                  ▼
     ┌──────────┐        ┌──────────┐        ┌──────────┐
     │ 立即转化  │        │ 建立信任  │        │ 引发思考  │
     │ (购买/行动)│        │ (品牌/专业)│        │ (种草/讨论)│
     └────┬─────┘        └────┬─────┘        └────┬─────┘
          │                   │                   │
          ▼                   ▼                   ▼
     ┌──────────┐        ┌──────────┐        ┌──────────┐
     │ → Step 2 │        │ → Step 2 │        │ → Step 2 │
     └──────────┘        └──────────┘        └──────────┘

                    ┌─────────────────────┐
                    │ Step 2: 用户决策阶段│
                    │ 用户现在在想什么？   │
                    └──────────┬──────────┘
                               │
     ┌──────────────┬──────────┼──────────┬──────────────┐
     ▼              ▼          ▼          ▼              ▼
┌──────────┐  ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│ 我不知道 │  │ 我在比较  │ │ 我在犹豫  │ │ 我在确认  │ │ 我想行动  │
│ 这个需求 │  │ 多个方案  │ │ 担心风险  │ │ 寻找理由  │ │ 只想购买  │
└────┬─────┘  └────┬─────┘ └────┬─────┘ └────┬─────┘ └────┬─────┘
     │              │          │          │              │
     ▼              ▼          ▼          ▼              ▼
┌─────────────────────────────────────────────────────────────────┐
│                        → Step 3: 产品复杂度                      │
└─────────────────────────────────────────────────────────────────┘

                    ┌─────────────────────┐
                    │ Step 3: 产品复杂度   │
                    │ 产品需要解释吗？      │
                    └──────────┬──────────┘
                               │
                    ┌──────────┴──────────┐
                    ▼                      ▼
             ┌──────────┐            ┌──────────┐
             │ 需要解释  │            │ 容易理解  │
             │ (新品/复杂)│            │ (成熟品类)│
             └────┬─────┘            └────┬─────┘
                  │                      │
                  ▼                      ▼
            ┌──────────┐            ┌──────────┐
            │ → Step 4 │            │ → Step 4 │
            └──────────┘            └──────────┘

                    ┌─────────────────────┐
                    │ Step 4: 内容长度     │
                    │ 你能写多长？         │
                    └──────────┬──────────┘
                               │
        ┌──────────────────────┼──────────────────────┐
        ▼                      ▼                      ▼
 ┌──────────┐           ┌──────────┐           ┌──────────┐
 │ 短内容   │           │ 中等长度  │           │ 长内容    │
 │ (<500字) │           │ (500-1500字)│         │ (>1500字) │
 └────┬─────┘           └────┬─────┘           └────┬─────┘
      │                      │                      │
      └──────────────────────┼──────────────────────┘
                             ▼
                    ┌─────────────────────┐
                    │    最终结构推荐       │
                    └─────────────────────┘
```

### 2.2 决策结果对照表

| 目标 | 决策阶段 | 产品复杂度 | 内容长度 | 推荐结构 |
|------|----------|-----------|----------|----------|
| 立即转化 | 知道需求/想行动 | 任何 | 任何 | AIDA / 对比型 |
| 建立信任 | 任何 | 需要解释 | 中等/长 | 4P / 归纳型 |
| 建立信任 | 任何 | 容易理解 | 短 | Q&A / 4P简化版 |
| 引发思考 | 不知道需求 | 任何 | 短/中等 | 悬念型 / 拼图式 |
| 引发思考 | 不知道需求 | 任何 | 长 | 故事-启发型 |
| 引发思考 | 在比较/犹豫 | 任何 | 短/中等 | Q&A / 情景-问题-方案 |
| 种草 | 任何 | 任何 | 短/中等 | 情景-问题-方案 / 故事-启发 |
