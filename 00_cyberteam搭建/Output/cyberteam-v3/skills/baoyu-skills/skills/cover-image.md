# baoyu-skills - 封面图生成

> **版本**: v3.1
> **分类**: 图像生成
> **技能**: baoyu-cover-image
> **触发词**: 封面图, 封面, cover, 生成封面

---

## ⚠️ 铁律

1. **平台适配**: 必须根据平台选择正确的比例和尺寸
2. **文字清晰**: 封面文字必须清晰可读
3. **版权合规**: 确保图像不侵犯任何版权
4. **质量优先**: 生成后必须审核质量

---

## 一、技能说明

### 1.1 功能概述

生成用于各种平台的封面图，包括小红书、公众号、微博等。

### 1.2 支持的平台

| 平台 | 比例 | 尺寸 | 用途 |
|------|------|------|------|
| **小红书** | 3:4 (竖版) | 1242x1660 px | 笔记封面 |
| **公众号** | 2.35:1 | 900x383 px | 文章封面 |
| **微博** | 16:9 / 1:1 | 1080x608 / 1080x1080 | 微博配图 |
| **抖音** | 16:9 | 1920x1080 px | 视频封面 |
| **B站** | 16:9 | 1920x1080 px | 视频封面 |
| **通用** | 自定义 | 自定义 | 其他用途 |

---

## 二、调用参数

### 2.1 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `title` | string | 是 | 封面标题 |
| `--type` | string | 否 | 尺寸类型: portrait/landscape/square |
| `--width` | number | 否 | 自定义宽度 |
| `--height` | number | 否 | 自定义高度 |
| `--style` | string | 否 | 风格: modern/minimal/vibrant/classic |
| `--color` | string | 否 | 主色调: 十六进制色值 |
| `--font` | string | 否 | 字体风格 |
| `--output` | string | 否 | 输出路径 |

### 2.2 参数示例

```bash
# 小红书封面
baoyu-cover-image "AI写作神器，让你的效率提升10倍" --type portrait

# 公众号封面
baoyu-cover-image "2024年最值得学习的技能" --type landscape

# 微博封面
baoyu-cover-image "新品发布" --type square

# 自定义风格
baoyu-cover-image "限时优惠" --type portrait --style vibrant --color #FF6B6B

# 自定义尺寸
baoyu-cover-image "自定义尺寸" --width 1080 --height 1920
```

## 三、执行流程

```
1. 明确封面需求
   └── 确定平台、主题、标题

2. 选择尺寸规格
   └── 根据平台选择比例

3. 确定设计风格
   └── 现代/简约/活泼/经典

4. 生成封面
   └── 调用 baoyu-cover-image

5. 审核质量
   ├── 文字是否清晰
   ├── 色彩是否协调
   ├── 构图是否合理
   └── 是否有版权问题

6. 优化调整
   └── 必要时重新生成
```

## 四、设计规范

### 4.1 小红书封面

**比例**: 3:4 (竖版)
**尺寸**: 1242x1660 px
**分辨率**: 72 DPI

**设计要求**:
- 标题醒目，字号 ≥ 48pt
- 使用对比色突出标题
- 背景简洁，避免过于复杂
- 可添加简单装饰元素

**标题公式**:
```
痛点型: "[数字]个方法解决[痛点]"
数字型: "[数字]个[关键词]，太实用了"
对比型: "用了[产品]，才知道[差距]"
悬念型: "原来[热门话题]的秘密是..."
情绪型: "[情绪词]！[事件]"
```

### 4.2 公众号封面

**比例**: 2.35:1 (电影比例)
**尺寸**: 900x383 px

**设计要求**:
- 标题居中或偏左
- 字号 ≥ 36pt
- 背景可略复杂
- 注意在手机端显示效果

**标题公式**:
```
数字型: "[年份]年必读的[数字]本书"
干货型: "[主题]完整指南（收藏）"
情绪型: "看完这篇，我[情绪反应]了"
悬念型: "[行业秘密]大公开"
```

### 4.3 微博封面

**比例**: 16:9 或 1:1
**尺寸**: 1080x608 或 1080x1080 px

**设计要求**:
- 信息简洁，一眼明了
- 字号 ≥ 36pt
- 适合快速浏览
- 可添加 Logo

## 七、常见问题

### 7.1 文字模糊

**原因**: 字号太小或图片被压缩

**解决方案**:
- 增大字号（≥ 48pt）
- 保存为高质量 PNG
- 避免过度压缩

### 7.2 色彩不协调

**原因**: 颜色对比度不足或过于杂乱

**解决方案**:
- 使用主色+辅色+点缀色（3色原则）
- 背景与文字使用对比色
- 参考色彩搭配工具

### 7.3 平台显示异常

**原因**: 尺寸或比例不符合要求

**解决方案**:
- 根据平台要求选择正确比例
- 使用标准尺寸
- 测试在不同设备上的显示效果
