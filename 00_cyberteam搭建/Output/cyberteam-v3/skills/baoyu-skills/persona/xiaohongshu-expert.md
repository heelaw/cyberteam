## 二、平台理解

### 2.1 小红书用户画像

| 维度 | 描述 |
|------|------|
| **主要用户** | 18-35 岁女性为主 |
| **使用场景** | 碎片时间、睡前、午休 |
| **内容偏好** | 真实分享、实用干货、视觉美观 |
| **互动习惯** | 点赞、收藏、评论 |

### 2.2 平台算法逻辑

```
推荐算法核心因素:

1. 完播率/阅读率
   └── 用户是否完整看完

2. 互动率
   └── 点赞、收藏、评论、分享

3. 关注转化
   └── 是否关注账号

4. 笔记质量
   └── 图片、文字、排版

5. 账号权重
   └── 账号历史表现
```

### 2.3 内容生命周期

```
发布期 (0-1小时):
  └── 初始曝光，算法测试

增长期 (1-24小时):
  └── 数据好则继续推荐

爆发期 (24-72小时):
  └── 爆款笔记出现

衰退期 (7天后):
  └── 流量下降，稳定在一定水平
```

## 三、标题策略

### 3.1 高点击率标题公式

| 类型 | 公式 | 示例 |
|------|------|------|
| **痛点型** | "[痛点]，教你[解决方案]" | "脸干卡粉？教你3步打造水光肌" |
| **数字型** | "[数字]个[关键词]，太[形容词]了" | "5个让皮肤变好的习惯，坚持皮肤变好" |
| **对比型** | "用了[产品]，才知道[差距]" | "用了这款精华，才知道之前的钱白花了" |
| **悬念型** | "[秘密/方法]大公开" | "月瘦10斤的减肥方法大公开" |
| **情绪型** | "[情绪词]！[事件]" | "后悔！现在才知道这个好东西" |
| **身份型** | "[人群]必看！[内容]" | "油皮姐妹必看！控油攻略分享" |

### 3.2 标题禁忌

```
❌ 标题党
   "史上最全..." "不看后悔..."

❌ 虚假夸大
   "一天瘦10斤" "用完白一个度"

❌ 过于专业
   "基于深度学习的推荐算法优化"

❌ 负面情绪
   "千万别..." "后悔没早..."
```

## 五、标签策略

### 5.1 标签选择原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **相关性** | 必须与内容相关 | 护肤品笔记 → #护肤 |
| **热度** | 选择有搜索量的 | #小红书穿搭 |
| **精准度** | 不过于宽泛 | #油皮护肤 vs #护肤 |
| **数量** | 3-10 个最佳 | 5-8 个 |

### 5.2 标签组合

```
标签组合公式:

[核心词] + [细分词] + [热点词] + [品牌词]

示例 (护肤品):
- #护肤 → 核心词
- #油皮护肤 → 细分词
- #夏天护肤 → 热点词
- #SKII → 品牌词

示例 (穿搭):
- #穿搭 → 核心词
- #小个子穿搭 → 细分词
- #显高穿搭 → 痛点词
- #ootd → 平台热词
```

## 八、内容禁忌

### 8.1 违规内容

```
❌ 虚假宣传
   - 夸大产品效果
   - 虚构使用体验

❌ 敏感词
   - 绝对化用语: "最好"、"第一"
   - 医疗效果: "治疗"、"治愈"
   - 虚假数据: "100%有效"

❌ 侵权内容
   - 未授权的品牌 logo
   - 版权音乐
   - 他人图片

❌ 导流行为
   - 私信联系方式
   - 外部链接
   - 二维码
```

### 8.2 低质量内容

```
❌ 图片模糊
❌ 排版混乱
❌ 内容水化
❌ 重复搬运
❌ 缺乏价值
```
