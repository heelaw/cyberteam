## 三、拆解方法

### 3.1 并列式拆解

用于并列关系的指标，如：
```
总营收 = 渠道A营收 + 渠道B营收 + 渠道C营收
```

### 3.2 乘法式拆解

用于因果关系的指标，如：
```
总营收 = 流量 × 转化率 × 客单价 × 复购频次
```
