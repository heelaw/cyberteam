## 四、数据驱动方法（数据驱动Agent融合）

### 4.1 数据驱动决策循环

```
发现问题（数据监控）
    │
    ▼
分析问题（对比/细分/相关分析）
    │
    ▼
提出假设（根因假设）
    │
    ▼
验证假设（A/B测试或数据交叉验证）
    │
    ▼
制定策略（调整计划）
    │
    ▼
执行监控（回到第一步）
```

### 4.2 数据分析方法论

```
描述性分析：
- 发生了什么？发生多少？
- 趋势、同比、环比

诊断性分析：
- 为什么发生？
- 归因分析、维度下钻

预测性分析：
- 将来会发生什么？
- 预测模型、趋势预测

处方性分析：
- 我们应该做什么？
- 优化建议、决策支持
```

## 五、指标管理体系

### 5.1 指标设计（指标拆解Agent + 北极星指标Agent融合）

```
北极星指标选择标准：
- 可衡量的
- 可归因的
- 可理解的
- 可改变的

指标拆解方法：
北极星指标 = 因子1 × 因子2 × 因子3 × ...

示例拆解：
GMV = 流量 × 转化率 × 客单价 × 复购频次
    │
    ├── 流量 = 内容流量 + 投放流量 + 口碑流量
    ├── 转化率 = 注册转化 × 首购转化
    ├── 客单价 = 品类结构 × 促销拉动
    └── 复购频次 = 首购复购率 × 复购周期
```

### 5.2 指标监控（指标监控器融合）

```
监控体系：
1. 核心指标仪表盘
   - 关键业务指标
   - 实时/日/周更新

2. 预警机制
   - 阈值预警
   - 趋势预警
   - 异常检测

3. 分析报告
   - 日报/周报/月报
   - 异常分析
   - 建议输出
```

## 六、财务模型（财务模型Agent融合）

### 6.1 收入模型

```
收入拆解：
收入 = 付费用户数 × 付费转化率 × ARPU × 复购周期

用户价值：
- LTV（生命周期价值）= ARPU × 复购周期
- CAC（获取成本）= 获客投入 ÷ 新用户数
- LTV/CAC > 3 为健康

收入预测：
- 保守/中性/乐观
- 情景分析
```

### 6.2 成本模型

```
成本分类：
- 获客成本（CAC）
- 运营成本
- 人力成本
- 技术成本
- 营销成本

成本优化：
- 固定成本 vs 变动成本
- 规模效应
- 效率提升
```

### 6.3 ROI分析

```
ROI计算：
ROI = (收入 - 成本) / 成本 × 100%

渠道ROI：
- 各渠道ROI对比
- 渠道效率优化

项目ROI：
- 项目投入产出
- 优先级排序
```

## 七、进度追踪（进度追踪器融合）

### 7.1 进度管理方法

```
进度追踪体系：
1. 项目维度
   - 项目总览
   - 里程碑追踪
   - 风险预警

2. 任务维度
   - 任务状态
   - 完成率
   - 延期预警

3. 资源维度
   - 人力投入
   - 预算消耗
```

### 7.2 复盘方法（汇报与复盘顾问融合）

```
复盘四步法：
1. 回顾目标
   - 最初目标是什么？
   - 实际结果是什么？

2. 分析原因
   - 成功因素？
   - 失败原因？

3. 总结规律
   - 经验沉淀
   - 可复制的方法论

4. 指导行动
   - 下一步行动
   - 风险规避

复盘模板：
- 目标回顾
- 结果对比
- 原因分析
- 经验总结
- 行动计划
```
