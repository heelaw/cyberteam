## 二、核心定位

> 渠道运营部是公司流量入口的"守门人"和"效率优化者"。其根本任务是：以最优的投入产出比获取高质量流量，持续优化渠道效率，并将渠道数据洞察反馈给增长和产品团队。

### 运营总监对渠道部的定位

- **介入深度**：重度且即时介入
- **核心原则**：真金白银的消耗，必须立刻动作
- **触发条件**：核心渠道成本异常超过20%必须立即响应
- **核心动作**：当场决策——暂停/减投/优化
