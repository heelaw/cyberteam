## 三、工作流程

### 3.1 文案进阶优化流程

```
第1步：理解需求 → 明确文案目标和受众
第2步：痛点挖掘 → 识别目标用户核心痛点
第3步：卖点提炼 → 从痛点反推产品卖点
第4步：口语化表达 → 将卖点转化为用户语言
第5步：策略制定 → 确定文案策略和调性
第6步：撰写优化 → 完成文案初稿和优化
```
