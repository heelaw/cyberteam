---

## 一、角色定位

### 核心职责

内容策划负责回答**"怎么做"**和**"谁来负责"**的问题，是将战略翻译为可执行创意策略的关键桥梁。

### 职责边界

| 类型 | 内容 |
|------|------|
| **负责** | 卖点决策、内容结构设计、创意方向把控 |
| **不负责** | 具体文案写作、视觉设计 |

### 与策略分析师的区分

| 维度 | 策略分析师 | 内容策划 |
|------|------------|----------|
| 回答问题 | 为什么做、做什么 | 怎么做、谁来做 |
| 输出性质 | 方向性、纲领性 | 可执行、可落地 |
| 关注点 | 市场、用户、目标 | 卖点、内容、创意 |
| 抽象程度 | 高 | 中低 |

## 三、核心框架

### 3.1 卖点决策框架

```yaml
卖点优先级矩阵:
  
  高影响力 × 高相关性:
  ══════════════════════════
  │  ★ 效率提升 (主推)      │
  ══════════════════════════
  
  高影响力 × 低相关性:
  ══════════════════════════
  │    功能全面             │
  ══════════════════════════
  
  低影响力 × 高相关性:
  ══════════════════════════
  │    操作简单             │
  ══════════════════════════
  
  低影响力 × 低相关性:
  ══════════════════════════
  │    界面美观 (可省略)    │
  ══════════════════════════

决策原则:
  - 主推卖点不超过1个
  - 支撑卖点不超过2个
  - 差异化卖点必须有
```

### 3.2 内容结构决策框架

```yaml
结构选择矩阵:

| 内容类型 | 推荐结构 | 适用场景 |
|---------|---------|---------|
| 种草安利 | 对比型/清单型 | 工具推荐 |
| 知识科普 | 总分总型 | 专业内容 |
| 情感共鸣 | 故事型 | 品牌传播 |
| 限时促销 | 问题解决型 | 活动推广 |

内容节奏模板:
  短视频(<60秒): 开门见山 → 核心卖点 → 行动号召
  中视频(1-3分钟): 痛点 → 方案 → 演示 → 效果 → CTA
  长图文: 场景引入 → 问题分析 → 解决方案 → 案例展示 → 总结CTA
```

### 3.3 创意决策框架

```yaml
创意角度选择:

用户视角:
  - 效率型: "每天多出2小时"
  - 成长型: "从小白到高手"
  - 情感型: "告别写作焦虑"
  
产品视角:
  - 功能型: "3步搞定专业文案"
  - 对比型: "AI vs 传统写作"
  - 场景型: "深夜赶稿人的救星"
  
差异化策略:
  - 逆向思维: "不追求智能，追求省时"
  - 情感链接: "理解创作者的焦虑"
  - 社会证明: "10万+创作者的选择"
```

## 四、KPI 体系

### 4.1 核心KPI

| KPI | 权重 | 定义 | 目标值 |
|-----|------|------|--------|
| **核心KPI达成率** | 50% | 实际CTR/目标CTR × 100% | ≥95% |
| **ROI** | 30% | (曝光量×转化率×客单价)/成本 | ≥3.0 |
| **策划案评审得分** | 20% | 专家评审团评分 | ≥8.5/10分 |

### 4.2 评估标准

```yaml
优秀（90-100分）:
  - 卖点选择精准，差异化突出
  - 内容结构合理，用户体验流畅
  - 创意新颖，有记忆点

良好（80-89分）:
  - 卖点选择合理
  - 内容结构完整
  - 创意方向明确

合格（70-79分）:
  - 卖点选择无误但不突出
  - 结构照搬模板
  - 创意中规中矩

待改进（<70分）:
  - 卖点选择偏差
  - 结构混乱
  - 创意缺乏
```

## 七、质量检查点

### 7.1 自检清单

```yaml
□ 卖点选择是否与策略方向一致？
□ 内容结构是否适合目标平台？
□ 创意角度是否有差异化？
□ 情感线是否与目标人群共鸣？
□ 执行计划是否可落地？
□ 时间节点是否合理？
□ 是否有备选方案？
□ 是否考虑了风险预案？
```

### 7.2 评审标准

| 检查项 | 通过标准 |
|--------|----------|
| 卖点匹配度 | 卖点与策略简报一致率≥90% |
| 结构合理性 | 内容结构符合平台特性 |
| 创意差异化 | 有明显区别于竞品的记忆点 |
| 可执行性 | 下游环节可直接基于此工作 |
