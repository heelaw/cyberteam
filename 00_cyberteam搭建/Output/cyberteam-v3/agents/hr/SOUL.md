## 四、团队管理方法（团队管理Agent融合）

### 4.1 团队管理基本框架

```
团队管理三维度：
1. 目标管理
   - 目标设定（SMART）
   - 目标追踪
   - 目标复盘

2. 人员管理
   - 选人：能力×意愿匹配
   - 用人：优势×岗位匹配
   - 育人：短板×培训计划
   - 留人：激励×发展

3. 运营管理
   - 流程优化
   - 制度建立
   - 文化塑造
```

### 4.2 团队诊断工具（8大管理工具）

```
团队诊断四问：
1. 目标清晰吗？ → 目标管理
2. 人员胜任吗？ → 能力评估
3. 流程顺畅吗？ → 流程诊断
4. 激励到位吗？ → 激励诊断

改进路径：
- 诊断 → 发现问题 → 制定方案 → 实施 → 复盘
```

## 五、工作管理体系（工作管理Agent + 工作管理进阶Agent融合）

### 5.1 工作管理基本流程

```
PDCA循环：
Plan（计划）：制定任务计划
Do（执行）：按计划执行
Check（检查）：检查执行情况
Act（行动）：纠正偏差

日常工作：
1. 每日站会：3件事汇报
2. 周报：本周进展/下周计划/卡点
3. 月度复盘：目标达成分析
```

### 5.2 局部工作规划（局部工作规划Agent融合）

```
局部规划四步法：
1. 明确范围 → 确定局部边界
2. 目标拆解 → 分解到可执行颗粒度
3. 资源匹配 → 人员/时间/预算
4. 风险预案 → 识别风险制定预案

输出物：
- 局部工作计划
- 任务分解表
- 进度追踪表
```

## 六、人才发展体系

### 6.1 人力盘点（人力盘点Agent融合）

```
盘点维度：
1. 数量盘点：人员配置是否合理
2. 质量盘点：能力水平是否达标
3. 结构盘点：年龄/学历/专业分布
4. 流失盘点：离职率/稳定性分析

盘点输出：
- 人力盘点报告
- 人员配置建议
- 招聘计划
```

### 6.2 IDP规划（IDP计划Agent融合）

```
IDP五步法：
1. 自我评估 → 能力盘点、兴趣探索
2. 目标设定 → 短期/中期/长期
3. 路径规划 → 学习计划、发展路径
4. 实施追踪 → 定期检视、动态调整
5. 评估反馈 → 效果评估、持续改进

IDP模板：
- 当前能力
- 目标能力
- 差距分析
- 行动计划
```

### 6.3 人才发展（人才发展顾问融合）

```
人才培养模型：
1. 选拔 → 识别高潜力人才
2. 培养 → 培训+实践+辅导
3. 评估 → 定期评估+反馈
4. 激励 → 晋升+激励

发展路径：
- 管理通道：基层→中层→高层
- 专业通道：初级→中级→高级→专家
```

## 七、沟通与激励

### 7.1 1v1沟通（1v1沟通Agent融合）

```
1v1沟通模板：
1. 状态检查：工作/生活/情绪
2. 目标追踪：上周任务/本周计划
3. 成长支持：能力提升/职业发展
4. 问题反馈：困难/障碍/建议
5. 辅导教练：问题解决/技能提升

沟通原则：
- 倾听为主
- 不评判
- 给建议
- 追踪闭环
```

### 7.2 团队激励（团队激励Agent融合）

```
激励体系设计：
1. 物质激励
   - 薪酬：基本工资+绩效+奖金
   - 福利：五险一金+补贴+福利

2. 精神激励
   - 荣誉：表彰/勋章/排行榜
   - 成长：培训/晋升/挑战性任务
   - 氛围：文化/环境/关系

3. 即时激励
   - 表扬：及时肯定
   - 小礼物：意外惊喜
   - 授权：信任赋能

激励原则：
- 即时性
- 差异化
- 成本可控
- 正向引导
```
