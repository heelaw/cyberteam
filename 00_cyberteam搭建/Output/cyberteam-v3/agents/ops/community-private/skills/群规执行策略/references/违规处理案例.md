# 违规处理案例集

## 案例一：发广告处理

```
═══════════════════════════════════════════
【案例背景】
新用户入群第2天，发了一条"扫码领优惠券"的推广链接
═══════════════════════════════════════════

【处理过程】

Step 1: 发现违规
→ 截图保存证据
→ 确认内容：推广链接，属于广告

Step 2: 违规判定
→ 级别：重度违规（发广告）
→ 历史：无历史违规记录（首次）

Step 3: 执行处理
→ 私下@用户：
"你好，[违规行为]违反了我们的群规
群规明确禁止发广告，这是第一次警告
请删除相关内容，下次违规将直接移出"

→ 用户回复："好的不好意思，我不知道，删了"
→ 回复用户："没关系，欢迎多分享[社群主题]相关内容"

Step 4: 群内（如有必要）
→ 不需要群内说明，轻度处理不需要公开

Step 5: 记录存档
【[日期]】[用户昵称] 首次违规 发广告 私下警告 用户配合 已删除

═══════════════════════════════════════════
```

## 案例二：重复违规处理

```
═══════════════════════════════════════════
【案例背景】
老用户入群3个月，上周因发无关链接被警告
今天又发了一条投票链接
═══════════════════════════════════════════

【处理过程】

Step 1: 查看历史记录
→ 【[日期-7]] [用户昵称] 发无关链接 私下提醒 用户配合
→ 确认：本次是第二次违规

Step 2: 违规判定
→ 级别：重度违规（重复违规）
→ 性质：明知故犯

Step 3: 执行处理
→ @用户：
"[用户昵称] 这是你第二次违规
上周已提醒过禁止发无关链接
本次正式警告，下次将直接移出"

→ 用户回复："好的，我注意"
→ 回复："谢谢理解，请仔细阅读群规"

Step 4: 记录存档
【[日期]】[用户昵称] 二次违规 发投票链接 正式警告 用户配合

Step 5: 跟进
→ 3天后观察用户行为，正常

═══════════════════════════════════════════
```

## 案例三：严重违规处理

```
═══════════════════════════════════════════
【案例背景】
用户发了一条"稳赚不赔"的投资链接，声称可以高回报
有其他群友询问具体情况
═══════════════════════════════════════════

【处理过程】

Step 1: 发现违规
→ 立刻注意到这条消息
→ 截图保存证据
→ 确认：疑似诈骗/虚假信息，属于严重违规

Step 2: 违规判定
→ 级别：严重违规（诈骗风险）
→ 风险：可能造成其他成员经济损失

Step 3: 执行处理
→ 直接移出，无需事先警告

→ 私信用户：
"你好，你发布的内容涉嫌[虚假投资信息]
可能对其他成员造成经济损失
根据群规，已被移出社群

如有问题可申诉，但请注意发布信息的真实性"

→ 群内@all：
"@所有人 提醒大家：[用户]因发布不实信息已被移出
请注意辨别各类投资信息，谨防诈骗
如有疑问可私信群主"

Step 4: 记录存档
【[日期]】[用户昵称] 严重违规 疑似诈骗 直接移出

Step 5: 跟进
→ 确认其他群友无跟进询问
→ 移除相关话题影响

═══════════════════════════════════════════
```

## 案例四：私加好友处理

```
═══════════════════════════════════════════
【案例背景】
收到用户举报，有人私加他们好友进行推销
举报人提供了截图证明
═══════════════════════════════════════════

【处理过程】

Step 1: 接收举报
→ 收到举报信息和截图
→ 确认私加行为属实

Step 2: 违规判定
→ 级别：重度违规（私加骚扰）
→ 历史：首次举报该用户

Step 3: 私下沟通
→ @用户：
"[用户昵称] 收到举报，你存在私加群友行为
群规禁止私加好友进行推销/骚扰
这是第一次警告，下次直接移出

请停止该行为，否则将被移出社群"

→ 用户回复："不好意思，我就是想认识一下，没有恶意"
→ 回复："理解你的想法，但群友们不一定这么想
请通过群主介绍等方式，不要私下添加"

Step 4: 记录存档
【[日期]】[用户昵称] 违规 私加好友骚扰 警告 用户承认

Step 5: 跟进举报人
→ 私信举报人："已处理，后续如再发生请告知"
→ 举报人表示感谢

═══════════════════════════════════════════
```

## 案例五：争议话题处理

```
═══════════════════════════════════════════
【案例背景】
两位用户在群里争论某话题，情绪激动
开始出现人身攻击言论
═══════════════════════════════════════════

【处理过程】

Step 1: 发现异常
→ 群内气氛紧张
→ 有用户开始@群主
→ 确认：开始出现人身攻击

Step 2: 介入时机
→ 在人身攻击言论发出后立即介入
→ 越早介入越好

Step 3: 执行处理
→ @所有人：
"请大家冷静一下～

群里讨论[话题]没问题
但请就事论事，不要人身攻击

[用户A]和[用户B]，请各自冷静
如有不同观点，可以各自保留

我们继续讨论[话题]，但请保持尊重～"

Step 4: 后续观察
→ 30分钟后，群内气氛缓和
→ 两位用户不再争论
→ 话题自然转移

Step 5: 私下跟进
→ 私信两位用户：
"感谢配合，群里讨论难免有分歧
但大家都保持理性，群氛围才能好
以后有问题随时找我～"

═══════════════════════════════════════════
```

## 案例处理总结

| 违规类型 | 处理方式 | 关键点 |
|---------|---------|--------|
| 发广告 | 警告→移出 | 保留证据，及时处理 |
| 重复违规 | 升级处理 | 查看历史，一视同仁 |
| 严重违规 | 直接移出 | 快速处理，维护安全 |
| 私加好友 | 警告+说明 | 平衡关系，明确边界 |
| 争论升级 | 及时介入 | 就事论事，不偏袒 |
