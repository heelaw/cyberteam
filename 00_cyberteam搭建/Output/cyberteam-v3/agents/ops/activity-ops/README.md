## 核心Skills

### 1. 活动目标定义四要素
**触发**: 活动目标是什么、拉新/促活/成交
**功能**: 明确活动目标类型

### 2. 不同目标-创意匹配表
**触发**: 拉新用什么活动、促活用什么
**功能**: 选择活动类型

### 3. 八种基础活动创意生成器
**触发**: 做什么活动、活动创意
**功能**: 提供8种基础活动玩法

### 4. 活动创意自检清单
**触发**: 创意好不好、活动评估
**功能**: 筛选活动创意

### 5. 热点蹭法拆解器
**触发**: 蹭热点、热点营销
**功能**: 借势营销方法论

### 6. 活动策划方案七要素模板
**触发**: 活动方案怎么写、活动策划案
**功能**: 标准化活动方案结构

### 7. 活动复盘三步法
**触发**: 活动复盘、活动总结
**功能**: 复盘总结方法

## 工作流程

### 完整活动策划流程

```
第1步：明确目标 → 使用Skill 3（活动目标定义四要素）
第2步：匹配创意 → 使用Skill 7（不同目标-创意匹配表）
第3步：生成创意 → 使用Skill 1（八种基础活动创意生成器）
第4步：自检验证 → 使用Skill 5（活动创意自检清单）
第5步：蹭热点（如需要） → 使用Skill 4（热点蹭法拆解器）
第6步：撰写方案 → 使用Skill 2（活动策划方案七要素模板）
第7步：复盘总结 → 使用Skill 6（活动复盘三步法）
```

## 使用建议

### 完整策划
按工作流程顺序使用7个Skills

### 单项咨询
用户可直接触发对应Skill

### 热点活动
优先考虑Skill 4热点蹭法拆解器

### 创意筛选
生成多个创意后用Skill 5自检
