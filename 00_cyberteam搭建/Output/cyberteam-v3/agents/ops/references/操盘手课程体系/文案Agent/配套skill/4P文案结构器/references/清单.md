# 4P文案 - 检查清单

## Picture检查

### 场景选择
- [ ] 场景有代表性
- [ ] 与目标用户相关
- [ ] 与产品/承诺相关
- [ ] 能引发共鸣

### 细节描写
- [ ] 细节具体
- [ ] 有感官描述
- [ ] 画面感强
- [ ] 代入感强

### 人物塑造
- [ ] 人物真实
- [ ] 有代入感
- [ ] 与目标用户相似
- [ ] 情感真实

## Promise检查

### 承诺质量
- [ ] 承诺具体
- [ ] 承诺可实现
- [ ] 承诺有吸引力
- [ ] 与画面对应

### 收益描述
- [ ] 收益具体
- [ ] 收益可感知
- [ ] 用户视角
- [ ] 与承诺一致

## Prove检查

### 证据充分性
- [ ] 数据真实可信
- [ ] 案例具体
- [ ] 证言真实
- [ ] 背书权威

### 证据相关性
- [ ] 证据支撑承诺
- [ ] 与用户痛点相关
- [ ] 与画面对应
- [ ] 足够支撑承诺

## Push检查

### 行动指引
- [ ] 清晰明确
- [ ] 可执行
- [ ] 与品牌调性一致

### 语气一致性
- [ ] 与前面文案的调性一致
- [ ] 不过于突兀
- [ ] 品牌个性贯穿

### 紧迫感
- [ ] 有时间或数量限制
- [ ] 合情合理
- [ ] 不压迫
- [ ] 适度

## 整体检查

- [ ] 四阶段结构完整
- [ ] 画面感贯穿
- [ ] 情感连接自然
- [ ] 说服逻辑完整
- [ ] 品牌调性一致
