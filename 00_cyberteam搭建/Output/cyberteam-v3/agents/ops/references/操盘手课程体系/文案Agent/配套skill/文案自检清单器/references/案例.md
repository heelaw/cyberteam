# 文案自检 - 问题案例库

## 案例1: 结构问题

### 标题问题
**问题文案**: "我们的产品"
**问题分析**: 标题太平淡，没有吸引力，没有传达用户收益
**正确写法**: "让你每天多2小时的团队协作工具"

### 开头问题
**问题文案**: "随着互联网的发展..."
**问题分析**: 开头太冗长，与用户无关
**正确写法**: "还在为团队文档混乱头疼？3分钟了解解决方案"

### 结尾问题
**问题文案**: "感兴趣的话可以看看"
**问题分析**: CTA太软，没有行动指引
**正确写法**: "免费试用14天，点击立即开始"

## 案例2: 内容问题

### 数据虚假
**问题文案**: "100%的用户都说好"
**问题分析**: 100%这个数据不可信，违反广告法
**正确写法**: "98%的用户表示满意，基于2024年Q4调研"

### 承诺夸大
**问题文案**: "3天学会，月薪翻倍"
**问题分析**: 承诺无法兑现，违反广告法
**正确写法**: "系统学习3个月，提升职场竞争力"

### 表述歧义
**问题文案**: "最多可优惠50%"
**问题分析**: "最多"含义模糊，用户可能误解
**正确写法**: "限时优惠，节省500元，相当于原价50%"

## 案例3: 格式问题

### 错别字
**问题文案**: "竭诚为您服务"
**问题分析**: "竭"应为"竭"，错别字
**正确写法**: "竭诚为您服务"

### 标点错误
**问题文案**: "功能强大，价格实惠，质量可靠"
**问题分析**: 逗号使用不当，应用顿号
**正确写法**: "功能强大、价格实惠、质量可靠"

### 段落过长
**问题文案**: 一段超过500字
**问题分析**: 阅读压力太大
**正确写法**: 每段不超过100字，分5-6段

---

## 案例4: 合规问题

### 违禁词使用
**问题文案**: "全网第一"、"最好用"、"国家级"
**问题分析**: 违反广告法
**正确写法**: "行业领先"、"用户好评"、"高品质"

### 竞品贬低
**问题文案**: "比XX产品好10倍"
**问题分析**: 违反广告法，构成不正当竞争
**正确写法**: "对比行业同类产品，效率提升X%"

### 虚假宣传
**问题文案**: "名人推荐"
**问题分析**: 如无授权，违反广告法和肖像权
**正确写法**: 删除或改为"用户真实评价"
