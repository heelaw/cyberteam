# 常见问题与解答

## Q1：数据太多不知道从何下手？

**问题**：业务有很多数据，不知道分析什么
**解答**：
- 先明确分析目的，不是所有数据都要分析
- 选择最核心的3-5个指标
- 遵循"先宏观后微观"的原则

---

## Q2：找不到根本原因怎么办？

**问题**：分析了很多但找不到真正原因
**解答**：
- 多问"为什么"，逐层深入
- 使用5W2H框架系统分析
- 考虑内部+外部双重因素
- 用数据验证假设，而非主观猜测

---

## Q3：相关性vs因果性如何区分？

**问题**：看到两个数据相关就以为是因果
**解答**：
- 相关性：两个变量同时变化
- 因果性：一个变量导致另一个变化
- 验证方法：控制其他变量，看结果是否变化
- 警惕"虚假相关"（如冰激凌销量和溺水数量）

---

## Q4：分析报告太长没人看怎么办？

**问题**：写了很长的报告但没人愿意看
**解答**：
- 先写执行摘要（一页）
- 结论前置，不要让读者猜
- 用图表可视化数据
- 每部分都有明确的"所以呢"
