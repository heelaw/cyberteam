# 相关概念

## 活动类型与核心指标

| 活动类型 | 核心指标 | 适用场景 |
|----------|---------|---------|
| 拉新活动 | 新增用户数、CAC | 增长阶段 |
| 促生活动 | DAU、活跃率 | 成熟阶段 |
| 转化活动 | 付费率、GMV | 变现阶段 |
| 留存活动 | 留存率、复购率 | 流失预警 |

---

## 活动ROI计算

```
活动ROI = (活动带来的收入 - 活动成本) / 活动成本

收入计算：
- 新用户付费贡献
- 老用户复购贡献
- 品牌溢价贡献（难量化）

成本计算：
- 激励成本（红包、优惠券、奖品）
- 传播成本（广告投放、KOL合作）
- 执行成本（人力、技术、设计）
```

## 活动风险类型

| 风险类型 | 可能问题 | 应对方案 |
|----------|---------|---------|
| 技术风险 | 服务器崩溃 | 压测扩容、降级方案 |
| 运营风险 | 刷单作弊 | 风控规则、IP限制 |
| 物流风险 | 奖品延迟 | 提前备货、替代方案 |
| 舆论风险 | 负面评价 | 舆情监控、快速响应 |
| 财务风险 | 超预算 | 实时监控、动态调整 |
