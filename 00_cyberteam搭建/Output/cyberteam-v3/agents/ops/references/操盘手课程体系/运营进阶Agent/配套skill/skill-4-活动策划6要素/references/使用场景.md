# 使用场景参考

## 场景1：年度大促策划（618/双11）

**输入**：
- 业务目标：618期间GMV提升50%
- 预算：100万
- 参与部门：产品、技术、运营、市场

**执行流程**：
1. 明确目标（销售额、品牌声量）
2. 确定目标用户（老用户召回、新用户拉新）
3. 设计活动玩法（秒杀、满减、红包）
4. 设计激励机制（阶梯优惠、限时抢购）
5. 设计传播机制（站外种草、社媒裂变）
6. 风险评估（服务器、库存、客服）
7. 制定执行时间表

---

## 场景2：产品上线冷启动

**输入**：
- 产品：新款App
- 目标：获取1000种子用户
- 预算：5万

**执行流程**：
1. 明确目标（种子用户质量而非数量）
2. 确定目标用户（早期采纳者）
3. 设计活动玩法（邀请制、内测）
4. 设计激励机制（创始人权限、优先体验）
5. 设计传播机制（KOL推广、社群扩散）
6. 风险评估（用户反馈、产品bug）

---

## 场景3：流失用户召回

**输入**：
- 沉默用户：5000人
- 预算：2万
- 目标：召回500人

**执行流程**：
1. 明确目标（召回率和召回后留存）
2. 分析沉默原因（流失调研）
3. 设计活动内容（专属召回礼包）
4. 设计激励机制（限时优惠、专属权益）
5. 选择触达渠道（短信、Push、邮件）
6. 风险评估（打扰用户、成本控制）
