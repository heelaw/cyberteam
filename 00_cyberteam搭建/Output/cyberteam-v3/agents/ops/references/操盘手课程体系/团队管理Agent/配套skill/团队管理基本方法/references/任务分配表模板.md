# 任务分配表模板

## 使用说明
用于将团队目标分解为具体任务，明确责任人、完成标准和截止时间。

## 模板格式

| 序号 | 任务名称 | 任务描述 | 责任人 | 协作人 | 完成标准 | 截止时间 | 优先级 | 状态 |
|------|----------|----------|--------|--------|----------|----------|--------|------|
| 1 | | | | | | | | |
| 2 | | | | | | | | |
| 3 | | | | | | | | |

### 2. 任务分解结构（WBS）

**总体目标**: [填写目标]**

**一级任务**:
- 任务A
  - 二级任务A1
  - 二级任务A2
- 任务B
  - 二级任务B1
  - 二级任务B2

**责任人分配**:
- 任务A: [姓名]
- 任务B: [姓名]

## 填写指南

1. **任务名称**: 简洁明确，体现交付物
2. **任务描述**: 说明背景和具体要求
3. **完成标准**: 能量化的量化，不能量化的具体化
4. **优先级**: P1=必须完成，P2=尽量完成，P3=有余力完成
5. **状态**: 待开始 / 进行中 / 已完成 / 已延期 / 已取消
