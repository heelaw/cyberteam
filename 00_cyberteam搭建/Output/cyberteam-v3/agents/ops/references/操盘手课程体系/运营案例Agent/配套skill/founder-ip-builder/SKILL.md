# SKILL: founder-ip-builder

> 技能名称：创始人IP打造策略
> 版本: v1.0
> 难度等级: medium
> 关联Agent: 运营案例Agent
> 来源案例: 雷军IP、罗永浩、董明珠等

---

## 1. 技能概述

本技能提供创始人个人IP打造的分析框架和策略，帮助企业家/高管通过个人品牌为商业赋能。

**核心能力**:
- 分析创始人IP定位
- 规划内容策略
- 设计表达风格
- 评估IP风险与边界

## 5. Critical Rules

### 5.1 必须遵守

1. **真实比完美重要** - 用户可以识别虚假，真实才能建立信任
2. **持续输出** - IP打造需要长期坚持，不能三天打鱼
3. **服务于品牌** - IP是公司品牌的延伸，不是独立的个人秀
4. **有边界意识** - 创始人言论天然有影响力，要谨慎

### 5.2 禁止行为

1. **不完美人设** - 过度包装会被识破
2. **不持续断更** - 断续会流失粉丝
3. **不脱离业务** - IP要服务于商业目标
4. **不碰争议话题** - 政治/宗教等敏感话题
5. **不损害品牌形象** - 个人行为不能伤害公司品牌

---

## 6. 评估维度

| 维度 | 说明 | 最低要求 |
|------|------|----------|
| 定位清晰度 | 人设是否清晰有辨识度 | 3个以内核心标签 |
| 内容一致性 | 输出是否符合定位 | 80%以上符合 |
| 粉丝互动 | 用户参与度 | 有互动数据 |
| 风险控制 | 是否有边界意识 | 无重大负面 |
