# 案例库

## 案例1: YouTube（消磨时间产品）

### 背景
- 产品类型: 视频网站（消磨时间）
- 商业目标: 可持续的广告收入
- 用户价值: 发现有趣的视频获得乐趣

### 指标演变

| 时期 | 北极星指标 | 原因 |
|------|-----------|------|
| 2012年前 | 点击率 | 关注视频点击 |
| 2012年后 | 观看时长 | 关注长期价值 |

### 演变原因
- 点击率时代：封面党泛滥，内容质量下降
- 观看时长时代：激励优质内容，闭环健康

### 筛选过程

**备选指标**:
1. 每日总观看时长
2. 每日观看视频的用户数
3. 每周活跃用户数

**六标准评估**:

| 标准 | 观看时长 | 日活用户 | 周活用户 |
|------|:-------:|:-------:|:-------:|
| 用户价值 | ✓ | ✓ | ✗ |
| 商业基础 | ✓ | ✓ | ? |
| 用户活跃 | ✓ | ✓ | ✓ |
| 整体方向 | ✓ | ✓ | ✓ |
| 简单直观 | ✓ | ? | ✓ |
| 先导指标 | ✓ | ✓ | ✓ |
| **总分** | **6** | **5** | **4** |

**结论**: 观看时长最优

## 案例2: Facebook vs MySpace（社交产品）

### 背景
- 时期: 2005-2010年
- 产品: 社交网络

### 指标对比

| 公司 | 北极星指标 | 结果 |
|------|-----------|------|
| Facebook | 月活跃用户数 | 胜出 |
| MySpace | 总注册用户数 | 失败 |

### MySpace失败原因
- 总注册用户数是典型**虚荣指标**
- 数字永远增长，但不反映真实活跃
- 无法指导产品决策

### Facebook成功原因
- 月活是**先导指标**
- 反映真实用户留存
- 指引产品持续优化

## 案例3: 头条（内容产品）

### 背景
- 产品类型: 内容分发平台
- 商业目标: 广告收入
- 用户价值: 获得感兴趣的内容

### 好指标 vs 坏指标

| 指标类型 | 指标 | 说明 |
|----------|------|------|
| **好指标** | 阅读时长 | 用户真正消费内容 |
| **坏指标** | 文章点击数 | 点击不代表阅读 |

### 原因分析
- 点击 ≠ 阅读
- 标题党会获得高点击，但无阅读价值
- 阅读时长才能反映内容价值

## 案例4: 美团（交易平台）

### 背景
- 产品类型: 本地生活平台
- 商业目标: 交易抽成
- 用户价值: 便捷生活服务

### 指标选择

| 指标 | 是否合适 | 原因 |
|------|---------|------|
| 销售额/订单数 | ✓ | 核心交易指标 |
| 商家数 | ✗ | 供给需平衡 |
| 骑手数 | ✗ | 成本需控制 |
| 活跃用户数 | ✗ | 不能反映交易 |

### 核心教训
- 平台类产品不能只看单边
- 供需需要平衡
- 核心是交易，不是流量

## 案例5: Zoom（效率工具）

### 背景
- 产品类型: 视频会议工具
- 商业目标: B端付费
- 用户价值: 高效沟通

### 指标选择

| 指标 | 是否合适 | 原因 |
|------|---------|------|
| 会议完成数 | ✓ | 先导指标，反映使用 |
| 营收 | ✗ | 滞后指标 |
| 用户满意度 | ✗ | 严重滞后 |

### 核心教训
- B端产品也要关注先导指标
- 营收是结果，不是行动指引
- 过程指标比结果指标更有指导价值

## 案例6: 滴滴（出行平台）

### 背景
- 产品类型: 出行平台
- 商业目标: 交易抽成
- 用户价值: 便捷出行

### 指标演变

| 阶段 | 核心指标 | 重点 |
|------|---------|------|
| 早期 | 用户增长 | 扩大市场 |
| 中期 | 订单增长 | 提升交易 |
| 成熟期 | 营收利润 | 商业化 |

### 核心教训
- 不同阶段不同重点
- 指标需随产品生命周期调整
- 早期不要过早商业化

## 虚荣指标清单

| 指标 | 为什么是虚荣 |
|------|-------------|
| 总注册用户数 | 只增不减，不反映活跃 |
| 总粉丝数 | 僵尸粉不计 |
| 总下载量 | 下载不等于使用 |
| 总PV | 刷量容易 |
| 累计GMV | 历史累积，无实时性 |

---

## 先导指标 vs 滞后指标

### 先导指标（预测未来）
- 日活/周活/月活
- 观看时长
- 功能使用次数
- 留存率

### 滞后指标（反映过去）
- 营收
- 利润
- 市场份额
- 年度满意度
