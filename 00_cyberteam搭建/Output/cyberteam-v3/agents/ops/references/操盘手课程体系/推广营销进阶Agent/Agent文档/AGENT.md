## 4个配套Skills

### Skill 1: 转化漏斗三要素

**定位**：转化漏斗诊断器

**用户问题**："不知道转化效果好不好，问题在哪"

**输出**：
- 漏斗三要素分析（曝光量→点击量→转化量）
- 各环节参考值对照表
- 问题诊断报告（卡点定位）

**触发关键词**：
- 转化率低
- 漏斗分析
- 转化效果
- 流量流失
- 转化卡点

---

### Skill 2: 转化流程优化三法

**定位**：转化流程优化顾问

**用户问题**："转化流程太长，用户流失多"

**输出**：
- 问题定位（哪一步流失最多）
- 优化方案选择（简化/前置/激励）
- 预期效果估算

**触发关键词**：
- 转化优化
- 流程太长
- 减少流失
- 提升转化
- 用户流失

---

### Skill 3: 精准定义潜在用户三法

**定位**：用户定义诊断仪

**用户问题**："不知道目标用户是谁，画像模糊"

**输出**：
- 用户定义方法选择（行为/需求/价值）
- 用户画像产出（人口统计+行为特征+需求痛点）
- 验证建议（如何验证画像准确性）

**触发关键词**：
- 用户画像
- 目标用户
- 用户定位
- 精准获客
- 用户需求

---

### Skill 4: 媒介资源投放流程

**定位**：投放流程指南针

**用户问题**："第一次做投放不知道怎么开始"

**输出**：
- 投放步骤清单（6步）
- 各环节注意事项
- 常见坑及避坑指南

**触发关键词**：
- 广告投放
- 媒介投放
- 渠道投放
- 首次投放
- 投放预算

---

## Skills协同关系

```
推广营销进阶Agent
│
├─ 转化漏斗三要素（诊断）
│   └─ 发现问题 → 转化流程优化三法（优化）
│
├─ 精准定义潜在用户三法（定位）
│   └─ 用户画像产出 → 媒介资源投放流程（执行）
│
└─ 媒介资源投放流程（执行）
    └─ 投放后数据 → 转化漏斗三要素（复盘）
```
