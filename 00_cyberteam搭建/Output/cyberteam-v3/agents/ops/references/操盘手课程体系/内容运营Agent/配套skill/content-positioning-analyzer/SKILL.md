# content-positioning-analyzer

## 简介

确定内容的定位和调性，使内容型产品脱颖而出，持续获取用户注意力。

## 触发关键词

```
内容定位、内容调性、价值主张、差异化、内容独特性
```

## 触发场景描述

```
当需要建立或调整内容定位时触发：
- 新媒体账号需要明确定位
- 内容风格不统一
- 需要找到差异化方向
- 需要建立内容识别度
```

## 输入

### 必需输入

| 字段 | 类型 | 说明 |
|------|------|------|
| product_info | string | 产品/账号信息 |
| target_users | string | 目标用户 |

### 可选输入

| 字段 | 类型 | 说明 |
|------|------|------|
| competitor_analysis | array | 竞品分析 |

## 方法论：内容定位5步法

### 1. 分析竞品

- 找到关注领域内的其他内容源
- 分析其内容风格

### 2. 找到差异

- 识别与竞品的差异点
- 确定独特价值

### 3. 细化体现

- 将差异点具体化
- 转化为可执行的动作

### 4. 建立规范

- 制定内容风格指南
- 设定禁区

### 5. 持续执行

- 保持一致性
- 定期回顾调整

### 禁止行为

1. 禁止定位模糊或面面俱到
2. 禁止脱离目标用户的真实需求
3. 禁止盲目模仿竞品定位
4. 禁止定位长期不变
5. 禁止定位与内容执行脱节
