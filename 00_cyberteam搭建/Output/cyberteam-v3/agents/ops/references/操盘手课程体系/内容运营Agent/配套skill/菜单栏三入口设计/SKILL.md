# 菜单栏三入口设计

## 触发关键词

**主关键词**: 菜单栏设计、底部菜单、菜单设置、公号菜单、菜单栏怎么设计

**次关键词**: 公众号菜单、菜单栏内容、菜单配置、自定义菜单、导航栏设计

**场景触发**: 用户新建公众号需要设计菜单栏，或现有菜单栏需要优化升级时自动触发

## 描述PUSHY

菜单栏不是随便放几个链接就完了！好的菜单栏=用户留存+价值展示+转化闭环。90%的账号第一个菜单放"历史文章"，大错特错！跟着走，让菜单栏成为你的24小时自动印钞机。

## 核心定义

通过「功能入口设计 → 内容入口设计 → 转化入口设计」三类菜单的合理配置，让用户在关注后的第一时间找到想要的内容，同时引导用户进入转化路径，最大化菜单栏的商业价值。

## workflow_steps

### Step 1: 功能入口设计

**输入**:
- 账号核心功能
- 用户最高频需求
- 账号定位

**处理**:
1. 确定3个主菜单名称（每个菜单最多5个字）：
   - 菜单1（左侧）：核心功能/最常用功能
   - 菜单2（中）：次要功能/内容分类
   - 菜单3（右侧）：转化入口/联系方式
2. 设计子菜单结构（最多5个）：
   - 跳转网页（需已发布内容）
   - 小程序（需有小程序权限）
   - 发送消息（图文/图片/文字）
3. 确保功能清晰：
   - 用户一眼能看懂菜单含义
   - 优先放用户真正需要的功能
   - 删除长期无人点击的功能

**输出**:
- 三个主菜单命名方案
- 每个菜单的子菜单配置
- 功能优先级排序

**成功标准**:
- 菜单命名简洁明了
- 子菜单数量合理（每个主菜单1-5个）
- 覆盖80%以上用户高频需求

### Step 2: 内容入口设计

**输入**:
- 账号内容分类
- 高价值文章清单
- 用户内容需求

**处理**:
1. 确定内容分类逻辑：
   - 按主题：「职场技能」「职业规划」「自我提升」
   - 按类型：「干货」「故事」「工具」
   - 按难度：「入门」「进阶」「高阶」
2. 设计内容菜单布局：
   - 主菜单名称要具体（如「职场技能」而非「文章」）
   - 子菜单要形成体系（如「入门课」「进阶课」「高阶课」）
3. 植入内容「诱饵」：
   - 精选内容突出展示
   - 设置「限时免费」「热门推荐」等标签
   - 引导用户点击查看

**输出**:
- 内容分类体系
- 内容菜单的详细配置
- 热门内容推荐设置

**成功标准**:
- 内容分类清晰，用户能找到想要的内容
- 有「钩子」内容吸引点击
- 更新机制明确（定期更换推荐内容）

### Step 3: 转化入口设计

**输入**:
- 账号变现路径
- 核心转化目标
- 客服/商务联系方式

**处理**:
1. 设计转化路径：
   - 低门槛转化：免费资料/社群/试听
   - 中门槛转化：低单价产品/体验课
   - 高门槛转化：高价课程/1v1服务
2. 配置转化菜单：
   - 主菜单要有行动号召（如「领取」「加入」「咨询」）
   - 子菜单展示具体价值
   - 落地页要直接解决用户问题
3. 保留联系方式：
   - 至少一种直接联系方式
   - 明确商务合作入口
   - 设置人工客服快捷入口

**输出**:
- 转化路径设计
- 转化菜单配置方案
- 联系方式设置

**成功标准**:
- 转化路径清晰，用户知道下一步做什么
- 菜单有足够的行动号召力
- 有多种转化方式供用户选择

## Critical Rules

1. **禁止菜单命名模糊** - 不能用「更多」「其他」「关于」这种模糊命名，用户看不懂
2. **禁止堆砌链接** - 不是链接越多越好，核心功能放1-3个即可，贪多嚼不烂
3. **禁止没有转化入口** - 必须有至少一个引导转化的菜单，不能全是免费内容
4. **禁止主菜单直接跳转** - 主菜单尽量不要直接跳转（用户手机屏幕有限），用子菜单承接
5. **禁止过期内容** - 跳转的页面要是长期有效的，不能是临时活动页
6. **禁止忽视移动端体验** - 跳转的网页必须在手机端体验良好，不能是PC端页面

## 质量评估标准

### 合格菜单栏特征

- 三个主菜单功能定位清晰
- 用户能在3秒内找到想要的内容
- 有明确的转化引导路径
- 命名简洁有力，不过度营销
- 跳转页面与菜单名称匹配

### 不合格菜单栏特征

- 三个菜单都是「历史文章」「往期内容」等模糊命名
- 子菜单数量过多，用户选择困难
- 没有转化入口，全部是免费内容
- 跳转链接已失效
- 移动端打开速度慢/排版混乱

## 注意事项

1. 菜单栏设置后需重启微信客户端生效（一般24小时内）
2. 跳转的网页需通过ICP备案验证
3. 小程序跳转需提前配置业务域名
4. 建议每月检查一次菜单点击数据，优化低点击菜单
5. 大促/节假日可临时调整菜单，但活动结束后要及时恢复
