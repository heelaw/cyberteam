# exposure-click-conversion-analyzer

## 简介

分析推广营销工作中的三个核心关键词：曝光、点击、转化。通过漏斗分析识别优化点，提升整体推广效率。

## 触发关键词

```
曝光量、点击率、转化率、漏斗分析、优化转化、提升ROI、流量优化、转化路径、流失分析
```

## 触发场景描述

```
当需要解决以下问题时触发本Skill：
- 如何提升推广的曝光/点击/转化
- 转化路径太长如何优化
- 某个环节流失严重如何解决
- 如何设定合理的转化率目标
```

## 输入

### 必需输入

| 字段 | 类型 | 说明 |
|------|------|------|
| funnel_data | object | 各环节数据（曝光/点击/转化量） |
| conversion_path | array | 用户转化路径（各步骤） |

### 可选输入

| 字段 | 类型 | 说明 |
|------|------|------|
| industry_benchmark | string | 行业基准数据 |
| channel_type | string | 渠道类型 |

## 方法论：三个核心关键词

### 曝光

**定义：** 产品展示的信息能被多少人看到

**取决于：**
- 该频道的用户数量
- 推广信息的位置和工作

### 点击

**定义：** 用户点击推广信息，进入下一步

**重要性：** 只有用户点击，才有下一步的转化可能

### 转化

**定义：** 用户完成最终目标行为（购买/注册/下载等）

**特点：** 会有一定比率的漏斗流失

## 漏斗原理

```
曝光 → 点击 → 转化
 ↓      ↓      ↓
N人     M人    K人
```

每增加一个环节，会带来至少20%的流失。

**最佳实践：** 流程最好不要高于5步。

## 三大优化思路

### ① 优化流量

**适用场景：** 曝光不足

**方法：**
- 拓展关键词
- 占据更多展示位置
- 增加投放渠道

### ② 缩短流程

**适用场景：** 转化路径过长

**原则：**
- 每增加一个环节 = 至少20%流失
- 流程不超过5步
- 删除不必要的中间环节

### ③ 提升特定环节转化率

**适用场景：** 某个环节转化率明显低于行业基准

**方法：**
- 分析该环节的用户行为
- 优化该环节的物料/文案
- 降低该环节的用户决策成本

|------|---------------|

## 工作流步骤

### Step 1: 数据收集与漏斗构建

- **输入:** funnel_data, conversion_path
- **处理:** 整理各环节数据，构建漏斗模型
- **输出:** 完整漏斗图和各环节转化率
- **成功标准:** 数据完整，漏斗层级清晰

### Step 2: 对标行业基准

- **输入:** 漏斗数据, industry_benchmark
- **处理:** 将各环节转化率与行业基准对比
- **输出:** 各环节状态评估（normal/warning/issue）
- **成功标准:** 识别出弱于基准的环节

### Step 3: 瓶颈识别

- **输入:** 漏斗分析结果, 转化路径
- **处理:** 分析流失最大的环节和流失原因
- **输出:** 瓶颈排序列表
- **成功标准:** 找出TOP3需要优先优化的环节

### Step 4: 制定优化方案

- **输入:** 瓶颈分析, product_info, budget
- **处理:** 根据瓶颈选择合适的优化策略
- **输出:** 优化策略清单（含预期提升）
- **成功标准:** 每个瓶颈都有对应优化方案

## Critical Rules

### 必须遵守

1. 优先优化影响最大的瓶颈
2. 流程不超过5步
3. 每步都要有数据支撑
4. 先优化流量，再优化转化率

### 禁止行为

1. 禁止不做数据就假设问题点
2. 禁止盲目增加曝光忽视转化
3. 禁止流程超过5步
4. 禁止只看总体转化率忽视各环节
5. 禁止忽视行业基准数据

## 评估清单

### 评估维度

| 维度 | 标准 | 权重 |
|------|------|------|
| 数据完整性 | 各环节数据齐全 | 20% |
| 对标准确性 | 基准数据选择正确 | 15% |
| 瓶颈识别 | 找出的瓶颈有数据支撑 | 25% |
| 策略可行性 | 优化方案可落地执行 | 25% |
| 预期合理性 | 提升预期有依据 | 15% |

### 评估标准

- **90-100分：** 五个维度全部达标，分析精准
- **75-89分：** 四个维度达标，分析合理
- **60-74分：** 三个维度达标，需要补充
- **<60分：** 分析有问题，需要重做
