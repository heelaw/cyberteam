# S4.4：发现推广入口的方法

## 寻找潜在推广入口的四个思考方向

### 思考方向1：联想激发式

**逻辑：** 用户看到A内容，会激起消费B产品/服务的欲望

**应用示例：**
- "帝都十大不容错过的小资饭店" → 餐厅推广
- "史上最全面的产品经理学习资料" → 运营课程推广
- "史上评分最高的100部电影" → 影视平台推广

---

### 思考方向2：问题解决式

**逻辑：** 发生了A事件，用户不得不考虑使用B产品/服务

**应用示例：**
- 屡屡求职受阻 → 参加职业培训
- 线下餐厅排队太可怕 → 使用在线预订服务
- 长期单身缺乏异性缘 → 开始注意自身形象和谈吐

---

### 思考方向3：用户聚集式

**逻辑：** 存在需求B的用户，同时也大量活跃在A平台

**应用示例：**
- 互联网从业者 → 知乎
- 20-35岁女性小资用户 → "新世相"公众号
- 妈妈育儿交流社区用户 → 亲子旅游、教育类产品

**特别提醒：** 线上和线下渠道都需要思考

---

### 思考方向4：搜索行为式

**逻辑：** 当用户产生需求B时，会如何寻找对应的产品和服务

**应用示例：**
- 得病需要动手术 → 百度或知乎搜索治疗方案
- 了解今年好书推荐 → 豆瓣看书榜排名
- 购买智能音响 → 专业测评网站或论坛

---

## 关键建议

> **与其大量铺100个渠道，不如把一个有效渠道玩透。**

当大量用户集中在某一渠道时，一定要深度运营该渠道，投入足够的时间、精力和成本。
