# 渠道选择矩阵

## 矩阵维度

| 维度 | 说明 |
|------|------|
| 用户匹配度 | 渠道用户与目标用户画像的重合度 |
| 流量规模 | 渠道的日均流量水平 |
| 成本效率 | 获取单个用户的成本 |
| 转化周期 | 从曝光到转化的平均时间 |
| 品牌调性 | 渠道风格与产品的匹配度 |

## 选择决策树

```
产品阶段 → 推广目标 → 预算规模 → 渠道选择
```

### 初创期产品
- **目标：** 验证产品、获取种子用户
- **推荐渠道：** 小而精准的渠道
- **优先：** 用户聚集式渠道、搜索行为式渠道

### 成长期产品
- **目标：** 规模化获客
- **推荐渠道：** 可规模化渠道
- **优先：** 信息分发平台、精准效果广告

### 成熟期产品
- **目标：** 品牌+效果双提升
- **推荐渠道：** 组合渠道
- **优先：** 媒介购买+效果广告+品牌传播

## 预算分配建议

| 产品阶段 | 测试期 | 优化期 | 规模化期 |
|----------|--------|--------|----------|
| 预算分配 | 单一渠道 | 2-3渠道 | 3-5渠道 |
| 单渠道占比 | 100% | 60% | 40% |
| 测试预留 | 0% | 20% | 10% |
