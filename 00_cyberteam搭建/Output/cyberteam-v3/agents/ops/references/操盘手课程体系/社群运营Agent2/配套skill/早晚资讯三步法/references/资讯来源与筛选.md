## 内容筛选标准

### 必选标准（至少满足2条）

| 标准 | 说明 |
|-----|------|
| 与社群定位相关 | 内容主题与社群核心价值相关 |
| 对用户有价值 | 内容能帮助用户解决问题或增长知识 |
| 话题性强 | 能引发讨论，有不同观点 |
| 时效性强 | 当天或近期发生的事 |

### 排除标准（满足任一条则不选）

| 标准 | 说明 |
|-----|------|
| 与社群无关 | 内容与社群定位毫不相关 |
| 虚假信息 | 未经核实的信息源 |
| 负面谣言 | 可能引发恐慌的不实信息 |
| 广告推广 | 软广硬广都要排除 |
| 竞品攻击 | 恶意抹黑竞品的内容 |

### 加分标准

| 标准 | 说明 |
|-----|------|
| 独家信息 | 其他媒体没有的 |
| 数据支撑 | 有具体数据引用 |
| 权威来源 | 官方或专家背书 |
| 实操性强 | 用户可以直接应用 |

## 今日话题设计指南

### 话题类型

| 类型 | 特点 | 适用场景 |
|-----|------|---------|
| 选择型 | 二选一或多选 | 快速获得用户观点 |
| 开放型 | 无固定答案 | 引发深度讨论 |
| 经验型 | 分享个人经历 | 增强参与感 |
| 预测型 | 判断未来趋势 | 引发思考 |

### 优质话题特征

1. **与用户相关**: 用户有自己的观点和经验
2. **没有标准答案**: 不会让用户担心答错
3. **便于表达**: 用户可以用简短的话表达
4. **有延伸空间**: 可以引出更多讨论

### 话题池示例

#### 职场类
- 你觉得35岁焦虑是真实的还是被制造的？
- 选择大公司还是创业公司？
- 你愿意降薪30%换一份不加班的工作吗？

#### 电商类
- 你买东西前会看差评还是好评？
- 你被种草后多久会下单？
- 直播带货会影响你的购买决策吗？

#### 兴趣类
- 你跑步时最喜欢听什么？
- 新手跑者最常见的错误是什么？
- 你跑步遇到瓶颈期怎么突破？
