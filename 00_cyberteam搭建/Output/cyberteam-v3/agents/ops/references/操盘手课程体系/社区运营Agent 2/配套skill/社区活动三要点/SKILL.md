## 核心定位
通过"钩子-体验-奖励"三要点方法论，帮助运营者策划高参与率、高满意度的社区活动。从活动目标确定、钩子设计、体验优化、奖励设置四个维度，打造让用户想参与、爱参与、愿意复购的精品活动。

## Critical Rules

### 必须遵守
1. **目标必须清晰单一** - 一个活动聚焦一个核心目标，不能什么都想要
2. **钩子必须有吸引力** - 用户第一眼就被打动，不想划走
3. **体验必须顺畅无阻** - 参与路径要短，门槛要低，不能让用户思考太久
4. **奖励必须有感知价值** - 用户觉得奖励值得，不能是鸡肋
5. **执行必须有SOP** - 每个环节有标准检查清单，不能靠感觉执行

### 禁止行为
1. **禁止目标模糊** - 不能写"提升用户活跃度"这种泛目标，要有具体数字
2. **禁止钩子牵强** - 硬凑的钩子用户一眼就看穿，必须真的有意思
3. **禁止体验复杂** - 不能让用户思考"怎么参与"，要傻瓜式操作
4. **禁止奖励鸡肋** - 用户不想要的奖励等于没有，要调研用户真正想要的
5. **禁止无应急预案** - 必须提前想好各种突发情况的应对方案

## workflow_steps

### Step 1：活动定位与目标确定
**输入**：
- 运营目标（拉新/促活/转化）
- 社区现状
- 可用资源

**处理**：
1. 确定活动核心目标
   - 拉新：注册量、转化率目标
   - 促活：DAU提升、UGC产出、互动率
   - 转化：付费率、GMV、客单价
2. 设定可量化指标
   - 总体目标（核心KPI）
   - 分解目标（各环节转化指标）
   - 及格线（最低可接受标准）
3. 确定目标用户
   - 核心参与人群
   - 期望参与人群
   - 活动门槛与用户匹配
4. 资源盘点
   - 预算
   - 时间
   - 人员
   - 渠道

**输出**：
- 活动定位文档
- 目标指标体系（含数字）
- 目标用户画像
- 资源预算表

**成功标准**：
- 目标可量化到具体数字
- 目标用户精准可描述
- 资源与目标匹配

### Step 2：活动钩子设计
**输入**：
- 活动定位文档
- 目标用户画像
- 市场热点和竞品动态

**处理**：
1. 钩子类型选择
   - 话题钩：蹭热点/引发讨论
   - 利益钩：实实在在的好处
   - 情感钩：引发情感共鸣
   - 荣誉钩：展示自我/获得认可
   - 社交钩：社交货币/引发攀比
2. 核心钩子提炼
   - 一句话描述活动亮点
   - 用户为什么要参与
   - 与其他活动的差异化
3. 钩子呈现设计
   - 活动标题设计（吸引眼球）
   - 首屏展示设计（快速传达价值）
   - 传播素材设计（利于分享）
4. 钩子测试
   - 内部测试：小范围试读
   - A/B测试：如有可能

**输出**：
- 钩子设计方案
- 活动标题（3-5个备选）
- 活动主视觉/文案

**成功标准**：
- 钩子一句话能说清楚
- 用户看到就想参与
- 与竞品有明显差异

### Step 3：活动体验设计
**输入**：
- 活动定位文档
- 钩子设计方案
- 目标用户特征

**处理**：
1. 参与路径设计
   - 最短路径：几步能完成
   - 各路径转化漏斗
   - 路径可视化
2. 关键体验节点设计
   - 入口体验：如何让用户进来
   - 过程体验：参与过程中的峰值体验
   - 结束体验：完成后用户的感受
3. 体验优化
   - 降低参与门槛
   - 减少等待焦虑
   - 增加正向反馈
   - 设置彩蛋惊喜
4. 流程测试
   - 全流程走查
   - 核心路径测试
   - 问题修复

**输出**：
- 活动流程图
- 各节点体验设计
- 操作指引文档

**成功标准**：
- 参与路径≤3步
- 用户能快速理解
- 各节点有正向反馈

### Step 4：活动奖励设计
**输入**：
- 活动定位
- 预算限制
- 目标用户偏好

**处理**：
1. 奖励类型选择
   - 物质奖励：现金/实物/优惠券
   - 精神奖励：徽章/称号/荣誉
   - 权益奖励：会员/特权/优先权
2. 奖励结构设计
   - 参与奖：所有人都能获得
   - 达成奖：完成任务获得
   - 排名奖：排名前列获得
   - 彩蛋奖：随机抽取
3. 奖励稀缺性设计
   - 限量设计
   - 限时设计
   - 条件解锁
4. 奖励感知优化
   - 价值感包装
   - 获得感设计
   - 分享激励

**输出**：
- 奖励体系设计方案
- 各奖励获取条件
- 奖励展示方案

**成功标准**：
- 奖励用户真正想要
- 奖励感知价值高
- 稀缺性设计合理

### Step 5：活动SOP与复盘
**输入**：
- 完整活动方案
- 各环节执行物料
- 目标指标体系

**处理**：
1. 筹备期SOP
   - 倒排时间表（按DDL往前推）
   - 各任务责任人
   - 里程碑检查点
   - 物料清单
2. 执行期SOP
   - 预热执行
   - 活动上线
   - 过程监控
   - 异常处理
3. 复盘模板
   - 数据复盘（实际vs目标）
   - 执行复盘（好的/需改进的）
   - 用户反馈复盘
   - 方法论沉淀

**输出**：
- 活动倒排时间表
- 执行检查清单
- 复盘模板
- 方法论沉淀文档

**成功标准**：
- 时间表责任到人
- 检查清单可执行
- 复盘有方法论沉淀
