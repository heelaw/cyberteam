## 二、AI图像生成工具

### 2.1 核心图像生成引擎

| 工具名称 | 功能描述 | 支持API | 输出格式 |
|---------|---------|---------|---------|
| **baoyu-image-gen** | 多API图像生成引擎 | OpenAI DALL-E<br>Google Imagen<br>通义万象<br>即梦AI<br>Seedream<br>Replicate | PNG/JPG |
| **baoyu-article-illustrator** | 文章配图生成 | 基于 baoyu-image-gen | 配图.jpg |
| **baoyu-cover-image** | 封面图生成 | 基于 baoyu-image-gen | 封面.jpg |
| **baoyu-infographic** | 信息图生成 | 基于 baoyu-image-gen | 信息图.jpg |
| **baoyu-comic** | 知识漫画创作 | 基于 baoyu-image-gen | 连环漫画.jpg |
| **baoyu-slide-deck** | PPT幻灯片生成 | 基于 baoyu-image-gen | 多张幻灯片.jpg |

### 2.2 详细配置

#### baoyu-image-gen (核心引擎)

**前置条件:**
- 选择至少一个图像生成API并配置密钥
- 安装 Bun 运行时

**API配置选项:**

```bash
# 方式1: OpenAI DALL-E (推荐, 国际)
export OPENAI_API_KEY="sk-..."
# 优势: 质量稳定, 风格多样
# 限制: 国内访问可能需要代理

# 方式2: 阿里通义万象 (国内推荐)
export DASHSCOPE_API_KEY="sk-..."
# 优势: 国内访问快, 成本低
# 限制: 风格选择相对较少

# 方式3: 即梦AI
export JIMENG_API_KEY="sk-..."
# 优势: 国内服务, 中文prompt支持好
# 限制: 需要注册

# 方式4: Seedream
export SEEDREAM_API_KEY="sk-..."

# 方式5: Replicate
export REPLICATE_API_TOKEN="r8_..."
```

**调用方式:**
```bash
# 直接调用
/baoyu-image-gen "科技感封面图，蓝色主题，未来风格"

# 指定API
/baoyu-image-gen "prompt" --api openai
/baoyu-image-gen "prompt" --api dashscope
```

#### baoyu-article-illustrator

**功能:** 为文章自动生成配图

**参数选项:**
- **Type (6种)**: infographic, scene, flowchart, comparison, timeline, diagram
- **Style (20+种)**: minimal, colorful, professional, playful, sketch等

**调用方式:**
```bash
/baoyu-article-illustrator path/to/article.md --type infographic --style minimal
```

#### baoyu-cover-image

**功能:** 生成封面图

**5维定制:**
- **类型**: 横版/竖版/方形
- **色板**: 10种预设色板
- **渲染**: 7种渲染风格
- **文字**: 标题/副标题/标签
- **情绪**: 专业/活泼/严肃/温馨

**调用方式:**
```bash
/baoyu-cover-image "标题" --type landscape --palette blue --render modern
```

#### baoyu-infographic

**功能:** 生成信息图

**布局选项 (21种):**
- list, timeline, flowchart, comparison, matrix, pyramid, funnel等

**视觉风格 (20种):**
- minimal, colorful, professional, sketch, 3d, neon等

**调用方式:**
```bash
/baoyu-infographic path/to/content.md --layout timeline --style professional
```

#### baoyu-comic

**功能:** 知识漫画创作

**风格选项 (9种):**
- classic, dramatic, realistic, shoujo, warm, dark, pop-art等

**调用方式:**
```bash
/baoyu-comic "主题内容" --style classic --tone educational
```

#### baoyu-slide-deck

**功能:** PPT幻灯片生成

**调用方式:**
```bash
/baoyu-slide-deck "演讲主题" --slides 10 --theme professional
```

## 三、内容发布工具

### 3.1 社交媒体发布

| 工具名称 | 平台 | 支持格式 | 技术方案 |
|---------|------|---------|---------|
| **baoyu-post-to-weibo** | 微博 | 文字+图片/视频<br>头条文章 | Chrome CDP |
| **baoyu-post-to-wechat** | 微信公众号 | 文章(html/md/text)<br>贴图/图文 | API + Chrome CDP |
| **baoyu-post-to-x** | X (Twitter) | 推文+图片/视频<br>X Articles长文 | Chrome CDP |
| **baoyu-xhs-images** | 小红书 | 连环图组(1-10张) | 11种视觉风格×8种版式 |

### 3.2 详细配置

#### Chrome CDP 通用配置

**前置条件:**
```bash
# 1. 安装 Chrome
brew install --cask google-chrome  # macOS
# 或 Linux: sudo apt install google-chrome-stable

# 2. 创建 Chrome Profile 目录
mkdir -p ~/.baoyu-chrome
export BAOYU_CHROME_PROFILE_DIR=~/.baoyu-chrome

# 3. 首次使用需手动登录各平台账号
# Chrome会使用独立的Profile, 不影响日常使用
```

#### baoyu-post-to-weibo

**功能:** 发布到微博

**支持格式:**
- 普通推文 (文字+图片/视频)
- 头条文章 (长文)

**调用方式:**
```bash
# 发布推文
/baoyu-post-to-weibo "推文内容" --images path/to/image1.jpg,path/to/image2.jpg

# 发布头条文章
/baoyu-post-to-weibo --article path/to/article.md --title "文章标题"
```

**前置步骤:**
1. 首次调用前手动打开 Chrome, 登录微博账号
2. 使用独立 Profile: `~/.baoyu-chrome`

#### baoyu-post-to-wechat

**功能:** 发布到微信公众号

**支持格式:**
- 图文消息 (html/md/text)
- 贴图/图片消息

**调用方式:**
```bash
# 发布图文
/baoyu-post-to-wechat --title "标题" --content path/to/article.html --cover cover.jpg

# 使用 Markdown
/baoyu-post-to-wechat --title "标题" --content path/to/article.md --format markdown
```

**前置步骤:**
1. 配置微信公众号 API 凭证 (可选)
2. 或使用 Chrome CDP 手动登录

#### baoyu-post-to-x

**功能:** 发布到 X (Twitter)

**支持格式:**
- 普通推文
- X Articles 长文

**调用方式:**
```bash
# 发布推文
/baoyu-post-to-x "推文内容" --images path/to/image.jpg

# 发布长文
/baoyu-post-to-x --article path/to/article.md --title "标题"
```

**限制:**
- 需科学上网
- Chrome CDP 可能被检测, 建议少量发布

#### baoyu-xhs-images

**功能:** 小红书图片生成

**视觉风格 (11种):**
- cute, elegant, minimalist, vibrant, vintage, dark等

**版式 (8种):**
- balanced, focus-left, focus-right, grid, full-bleed等

**调用方式:**
```bash
/baoyu-xhs-images path/to/content.md --style cute --layout balanced
```

## 四、内容处理工具

### 4.1 工具清单

| 工具名称 | 功能 | 模式/亮点 |
|---------|------|----------|
| **baoyu-translate** | 翻译 | 快速/普通/精细三种模式<br>支持术语表、图片翻译 |
| **baoyu-url-to-markdown** | 网页抓取 | Chrome CDP → Defuddle管道<br>支持登录态 |
| **baoyu-markdown-to-html** | 格式转换 | md → html<br>微信外链转引用<br>PlantUML支持 |
| **baoyu-youtube-transcript** | 字幕提取 | 多语言、翻译、章节、说话人识别 |
| **baoyu-format-markdown** | 美化 | 纯文本/md → 格式化md<br>frontmatter/标题/摘要 |
| **baoyu-compress-image** | 压缩 | 图片 → WebP/PNG<br>自动选择最优工具 |
| **baoyu-danger-x-to-markdown** | X推文抓取 | X推文 → Markdown |

### 4.2 详细配置

#### baoyu-translate

**三种翻译模式:**

1. **快速模式**: 直接翻译, 速度最快
2. **普通模式**: 基本优化, 平衡速度和质量
3. **精细模式**: 分析→翻译→审校→润色, 质量最高

**调用方式:**
```bash
# 快速翻译
/baoyu-translate path/to/file.md --to en --mode fast

# 精细翻译 (推荐)
/baoyu-translate path/to/file.md --to en --mode fine

# 带术语表
/baoyu-translate path/to/file.md --to en --glossary path/to/glossary.csv

# 图片翻译
/baoyu-translate path/to/image.png --to en
```

**前置条件:**
- 配置翻译 API (OpenAI / DeepL / Google Translate)

#### baoyu-url-to-markdown

**功能:** 网页转 Markdown

**调用方式:**
```bash
/baoyu-url-to-markdown "https://example.com/article" --output article.md
```

**特性:**
- 保留文章结构
- 提取图片
- 支持登录态网页
- 自动清理广告

#### baoyu-markdown-to-html

**功能:** Markdown 转 HTML (适配微信公众号)

**调用方式:**
```bash
/baoyu-markdown-to-html path/to/article.md --output article.html
```

**特性:**
- 代码高亮
- PlantUML 渲染
- 微信外链自动转底部引用

#### baoyu-youtube-transcript

**功能:** YouTube 字幕提取

**调用方式:**
```bash
# 提取字幕
/baoyu-youtube-transcript "https://youtube.com/watch?v=xxx" --output transcript.md

# 翻译字幕
/baoyu-youtube-transcript "https://youtube.com/watch?v=xxx" --translate --to zh
```

**特性:**
- 多语言支持
- 章节识别
- 说话人识别

#### baoyu-format-markdown

**功能:** Markdown 格式美化

**调用方式:**
```bash
/baoyu-format-markdown path/to/file.md --add-frontmatter --add-summary
```

**特性:**
- 添加 frontmatter
- 标题层级修正
- 列表格式化
- 添加摘要

#### baoyu-compress-image

**功能:** 图片压缩

**调用方式:**
```bash
/baoyu-compress-image path/to/image.jpg --output image.webp --quality 80
```

**特性:**
- 自动选择最优工具
- 支持 WebP/PNG 输出
- 批量压缩

**工具名称:** `mcp__MiniMax__web_search`

#### MiniMax Image Analysis

**工具名称:** `mcp__MiniMax__understand_image`

**功能:** 图像分析

**调用方式:**
```python
# 本地图片
mcp__MiniMax__understand_image(
    prompt="描述这张图片的内容",
    image_source="/path/to/image.png"
)

# URL 图片
mcp__MiniMax__understand_image(
    prompt="分析这张图片",
    image_source="https://example.com/image.jpg"
)
```

**限制:**
- 支持 JPEG, PNG, WebP 格式
- 文件路径若有 @ 前缀需去除

## 九、调用方式与限制

### 9.1 调用方式优先级

```
1. Skill 调用 (最高优先级)
   /baoyu-image-gen, /codex, /qa 等

2. MCP 工具调用
   mcp__MiniMax__web_search, mcp__filesystem__* 等

3. Agent 调用
   Activate gsd-executor 等

4. Bash 命令 (最后选择)
   仅用于系统操作, 避免用于有专用工具的任务
```

### 9.2 工具限制

#### baoyu-skills 限制

| 限制项 | 说明 |
|-------|------|
| **API 配额** | 各图像 API 有调用次数限制 |
| **内容审核** | 生成内容需符合平台规范 |
| **Chrome CDP** | 可能被平台检测, 建议少量发布 |
| **文件大小** | 图片压缩有最大文件限制 |

#### MiniMax MCP 限制

| 限制项 | 说明 |
|-------|------|
| **搜索来源** | MiniMax 搜索引擎结果 |
| **图片格式** | 仅支持 JPEG, PNG, WebP |
| **路径格式** | @ 前缀需去除 |

#### Gstack 限制

| 限制项 | 说明 |
|-------|------|
| **/browse** | 必须用于网页浏览, 禁止使用 mcp__claude-in-chrome |
| **并发调用** | 部分 skill 不支持并发 |
