```
❌ 错误方式：物理堆叠 + 映射引用

```

```
✅ 正确方式：能力拆解 + 重写 + 融入
agents/ops/SOUL.md  ← 活动策划能力融入运营部SOUL.md
agents/content-ops/SOUL.md  ← 活动策划能力融入内容运营部SOUL.md
```
