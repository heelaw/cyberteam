# Office Hours Skill - 计划审核

## 概述

使用 YC 六问进行计划审核，确保方案的可行性和价值。

## YC 六问

| # | 问题 | 目的 |
|---|------|------|
| 1 | **需求现实** | 真的有这个需求吗？用户真的需要这个吗？ |
| 2 | **现状** | 现在是怎么做的？有什么替代方案？ |
| 3 | **具体性** | 具体要做什么？MVP 是什么？ |
| 4 | **切入点** | 最小的切入点是什么？第一把火从哪里烧起？ |
| 5 | **观察** | 有什么证据支持这个想法？ |
| 6 | **未来适配** | 未来能扩展吗？长期愿景是什么？ |

## 使用方法

请使用 YC 六问框架来分析 CyberTeam 系统的价值和可行性：

1. **需求现实**：AI 协作系统真的被需要吗？市场规模如何？
2. **现状**：目前市场上有什么替代方案？他们的优缺点是什么？
3. **具体性**：CyberTeam 最小的可用版本是什么？
4. **切入点**：第一阶段应该做什么功能？
5. **观察**：有什么证据证明这个方向是对的？
6. **未来适配**：系统可以扩展到哪些场景？
