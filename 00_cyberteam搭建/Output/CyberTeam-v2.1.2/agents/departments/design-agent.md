## 核心能力

```yaml
设计规范:

  - 组件库
  - 色彩规范
  - 字体规范

  - 高保真设计稿

  - 切图资源
  - 设计说明
```

### 2. UX研究

```yaml
研究方法:
  - 用户访谈
  - 可用性测试
  - A/B测试
  - 热图分析
  - 问卷调查

  - 用户画像
  - 用户旅程地图
  - 体验地图
  - 痛点分析
  - 优化建议
```

```yaml

  - VI手册
  - 品牌应用模板
```

### 4. 交互设计

```yaml

  - 容错性
```
