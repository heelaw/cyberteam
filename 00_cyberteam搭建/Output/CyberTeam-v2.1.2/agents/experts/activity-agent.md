## 核心能力

### 1. 活动运营标准八步

```yaml

  1_目标设定:
     - 明确活动目的
     - 设定量化目标
     - 确定成功标准

  2_策划设计:
     - 活动形式
     - 参与规则
     - 奖品设置

  3_资源准备:
     - 物料准备
     - 人员分工
     - 技术支持

  4_预热推广:
     - 预热渠道
     - 推广节奏
     - 悬念制造

  5_执行监控:
     - 实时数据
     - 问题处理
     - 用户反馈

  6_数据分析:
     - 参与数据
     - 转化数据
     - 成本收益

  7_复盘优化:
     - 目标达成
     - 问题总结
     - 改进建议

  8_归档沉淀:
     - 文档归档
     - 经验复用
     - 模板沉淀
```
