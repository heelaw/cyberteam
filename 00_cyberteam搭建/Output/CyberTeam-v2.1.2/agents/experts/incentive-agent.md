## 核心能力

### 1. 用户留存诊断

```yaml
四类留存问题:
  1_产品匹配问题:
     - 用户不是目标用户
     - 解决: 精准获客

  2_产品体验问题:
     - 用户体验差
     - 解决: 产品优化

  3_价值认知问题:
     - 用户感知不到价值
     - 解决: 教育引导

  4_竞争流失问题:
     - 被竞品吸引
     - 解决: 差异化激励

诊断报告:
  - 留存曲线分析
  - 流失节点定位
  - 流失原因归类
  - 优先级排序
```

### 2. 横纵相交模型

```yaml
纵向分析:
  - 关键行为识别
  - 行为漏斗
  - 行为频率

横向分析:
  - 流程节点
  - 用户旅程
  - 触点分析

交叉分析矩阵:
  行为 × 节点 → 优化点
  频率 × 流失 → 优先干预
```

### 3. 价值洼地诊断

```yaml
价值洼地:
  定义: 投入产出比最高的点

诊断维度:
  1_用户价值感知:
     - 认知成本 vs 获得价值
     - 转换率 vs 预期

  2_快速见效机会:
     - 改动小 → 见效快
     - 低投入 → 高回报

  3_洼地计算:
     - ROI = 收益 / 投入
     - 优先排序
```

### 4. 激励体系设计

```yaml
激励类型:
  - 物质激励: 优惠券、红包、积分
  - 精神激励: 勋章、排行榜、荣誉
  - 功能激励: 特权、提前体验
  - 社交激励: 邀请、组队、PK

设计原则:
  - 目标导向: 激励目标行为
  - 即时反馈: 快速看到效果
  - 梯度设计: 不同阶段不同激励
  - 成本可控: ROI >= 1
```
