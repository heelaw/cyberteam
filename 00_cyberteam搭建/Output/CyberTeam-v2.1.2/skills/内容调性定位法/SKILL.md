## 核心框架

### 框架一：内容调性三要素模型（升级版）

```
         ┌─────────────────────────────────────────┐
         │              内容调性                    │
         ├─────────────┬─────────────┬─────────────┤
         │   品牌调性   │   用户调性   │   平台调性   │
         ├─────────────┼─────────────┼─────────────┤
         │ 品牌基因     │ 用户期待     │ 平台特性     │
         │ 核心价值     │ 情感共鸣     │ 表达方式     │
         │ 差异化主张   │ 沟通偏好     │ 流量规则     │
         │ 生命周期适配 │ AARRR适配    │ 阶段匹配     │
         └─────────────┴─────────────┴─────────────┘
                         ↓
              ┌─────────────────────┐
              │    调性定位文档      │
              │  (风格指南+创作规范)  │
              │  + 生命周期调性策略   │
              └─────────────────────┘
```

**三要素关系**：
- 品牌调性：我是谁（核心）
- 用户调性：他们是谁（锚点）
- 平台调性：我在哪（适配）

**生命周期适配原则**：
| 阶段 | 调性核心任务 | 调性策略重点 |
|------|-------------|--------------|
| 探索期 | 建立认知 | 专业权威+差异化 |
| 增长期 | 快速传播 | 年轻活力+共鸣感 |
| 成熟期 | 维护忠诚 | 温度感+归属感 |
| 衰退期 | 唤醒召回 | 情怀+怀旧 |

### 框架二：生命周期×调性策略（核心创新）

| 阶段 | 调性定位 | 内容重点 | 调性策略 |
|------|----------|----------|----------|
| **探索期** | 专业权威型 | 干货/方法论/创始人故事 | 打造专家人设，强调差异化 |
| **增长期** | 活力共鸣型 | 热点/娱乐/互动/故事 | 降低门槛，提升传播力 |
| **成熟期** | 温度陪伴型 | 品牌故事/用户共创/会员 | 强化情感连接，建立忠诚 |
| **衰退期** | 情怀唤醒型 | 怀旧/召回/限时 | 激活老用户，唤起记忆 |

### 框架三：AARRR×调性匹配

| AARRR阶段 | 调性目标 | 调性配置 | 内容重点 |
|-----------|----------|----------|----------|
| Acquisition获客 | 吸引新用户 | 活力型+共鸣型 | 热点借势、话题制造 |
| Activation激活 | 促进首次转化 | 专业型+利益型 | 干货价值、行动引导 |
| Retention留存 | 维护老用户 | 温度型+陪伴型 | 情感共鸣、持续价值 |
| Revenue变现 | 促进付费 | 信任型+紧迫型 | 限时优惠、会员专享 |
| Referral推荐 | 口碑传播 | 荣誉型+社交型 | 用户故事、分享激励 |

### 框架四：调性定位四象限

| 维度 | 温暖亲切 | 专业权威 | 年轻活力 | 高端大气 |
|------|----------|----------|----------|----------|
| 语言风格 | 口语化 | 术语化 | 潮流化 | 精致化 |
| 内容视角 | 用户视角 | 专家视角 | 朋友视角 | 精英视角 |
| 视觉调性 | 暖色系 | 冷色系 | 撞色系 | 黑白系 |
| 互动风格 | 平等交流 | 单向输出 | 趣味互动 | 低调互动 |
| 生命周期适配 | 成熟期主力 | 探索期主力 | 增长期主力 | 品牌建设期 |
| 适用品牌 | 消费品/服务 | 专业服务/科技 | 品牌/年轻人 | 奢侈品/高端 |

### 框架五：内容人格五维模型（升级版）

| 维度 | 选项示例 | 权重 | 生命周期适配 |
|------|----------|------|-------------|
| 专业度 | 小白友好 ↔ 专家级 | 20% | 探索期偏专家，成熟期偏友好 |
| 温度感 | 冰冷理性 ↔ 温暖感性 | 20% | 成熟期重温度，探索期重专业 |
| 年龄感 | 成熟稳重 ↔ 年轻活力 | 20% | 增长期重活力，衰退期可怀旧 |
| 距离感 | 高高在上 ↔ 平等亲近 | 20% | 增长期重亲近，成熟期可尊贵 |
| 幽默感 | 严肃正经 ↔ 风趣幽默 | 20% | 增长期重幽默，探索期偏严肃 |

## 执行步骤（升级版）

### Step 1: 梳理品牌属性

**目的**：理解品牌本质，为调性定位提供基础

**操作流程**：

1. **提取品牌核心元素**

   | 元素 | 问题 | 提炼 |
   |------|------|------|
   | 品牌使命 | 品牌存在的意义是什么？ | [一句话使命] |
   | 核心价值 | 品牌最核心的理念是什么？ | [3个关键词] |
   | 品牌个性 | 品牌像一个人，是什么样的人？ | [人格画像] |
   | 差异化 | 与竞品最大的不同是什么？ | [独特卖点] |
   | 用户承诺 | 品牌给用户的承诺是什么？ | [承诺内容] |

2. **品牌调性关键词提取**

   ```
   品牌调性词汇表：

   应该有的（加分词）：
   ├─ [温暖/专业/创新/可靠...]
   └─ 来源：品牌使命/价值/差异化

   应该没有的（减分词）：
   ├─ [冷漠/刻板/敷衍...]
   └─ 来源：竞品对比/用户反馈
   ```

3. **品牌DNA提炼**

   ```
   品牌DNA公式：
   [品类属性] + [情感属性] + [风格属性] = 品牌调性基调

   示例：
   咖啡品牌 = 提神醒脑(功能) + 生活方式(情感) + 文艺小资(风格)
   = "温暖陪伴的文艺生活家"

   科技品牌 = 效率提升(功能) + 科技感(情感) + 简约酷炫(风格)
   = "专业极客的效率伙伴"
   ```

4. **生命周期阶段判断**（升级新增）

   ```
   品牌增长阶段诊断：

   探索期特征：
   ├─ 产品刚上线，PMF待验证
   ├─ 用户认知度低
   └─ 调性策略：专业权威，建立信任

   增长期特征：
   ├─ PMF已验证，快速扩张
   ├─ 需要大量获客
   └─ 调性策略：活力共鸣，快速传播

   成熟期特征：
   ├─ 用户基数大，增速放缓
   ├─ 需要维护忠诚度
   └─ 调性策略：温度陪伴，深厚连接

   衰退期特征：
   ├─ 用户流失，增长停滞
   ├─ 需要召回或转型
   └─ 调性策略：情怀唤醒，激活老用户
   ```

### Step 2: 分析目标用户

**目的**：理解用户期待，匹配调性方向

**操作流程**：

1. **用户画像整理**

   ```
   目标用户画像要素：
   ├─ 基本属性：年龄/性别/地域/职业
   ├─ 行为特征：触媒习惯/内容偏好/购买行为
   ├─ 心理特征：价值观/兴趣爱好/生活方式
   └─ 内容期待：喜欢什么风格/讨厌什么风格
   ```

2. **用户调性期待分析**

   | 用户类型 | 调性期待 | 内容偏好 | 生命周期适配 |
   |----------|----------|----------|-------------|
   | 职场新人 | 鼓励成长 | 干货+情感 | 增长期 |
   | 职场精英 | 效率专业 | 深度+洞察 | 探索期 |
   | 宝妈群体 | 温暖实用 | 轻松+贴心 | 成熟期 |
   | Z世代 | 有趣好玩 | 潮流+个性 | 增长期 |

3. **用户语言习惯收集**

   ```
   用户原声收集：
   ├─ 他们常说的词：[收集30+高频词]
   ├─ 他们喜欢的表达：[网络用语/梗]
   ├─ 他们讨厌的表达：[说教/爹味/油腻]
   └─ 转化为自己语言：[品牌化表达]
   ```

4. **AARRR×用户调性匹配**（升级新增）

   ```
   用户生命周期×调性期待：

   Acquisition获客阶段：
   ├─ 用户状态：新用户，对品牌无认知
   ├─ 调性重点：吸引力+差异化
   └─ 匹配：活力型调性，引发好奇

   Activation激活阶段：
   ├─ 用户状态：首次接触，待转化
   ├─ 调性重点：价值感知+行动引导
   └─ 匹配：专业型调性，建立信任

   Retention留存阶段：
   ├─ 用户状态：已转化，待长期维护
   ├─ 调性重点：情感连接+持续价值
   └─ 匹配：温度型调性，陪伴成长

   Referral推荐阶段：
   ├─ 用户状态：忠诚用户，愿分享
   ├─ 调性重点：社交货币+荣誉感
   └─ 匹配：社交型调性，便于传播
   ```

### Step 3: 匹配平台特性

**目的**：让调性在目标平台有效传达

**操作流程**：

1. **主流平台调性特征**

   | 平台 | 主流调性 | 内容风格 | 互动特点 | 生命周期适配 |
   |------|----------|----------|----------|--------------|
   | 公众号 | 深度专业 | 长图文 | 留言互动 | 探索期/成熟期 |
   | 抖音 | 娱乐有趣 | 短视频 | 短视频互动 | 增长期 |
   | 小红书 | 精致生活 | 图文/短视频 | 种草安利 | 增长期/成熟期 |
   | B站 | 年轻有趣 | 中长视频 | 弹幕/评论 | 增长期 |
   | 微博 | 热点话题 | 短资讯 | 话题讨论 | 增长期 |
   | 视频号 | 社交传播 | 短视频 | 朋友圈扩散 | 增长期/成熟期 |

2. **平台调性适配表**

   ```
   基础调性：温暖陪伴的文艺生活家

   平台适配：
   ├─ 公众号：温暖讲述+深度干货，文章风格偏文艺
   ├─ 抖音：日常vlog+生活美学，风格轻松治愈
   ├─ 小红书：精致图文+好物种草，风格小资精致
   └─ B站：知识分享+生活方式，风格真实有趣

   生命周期适配：
   ├─ 探索期：侧重公众号深度内容
   ├─ 增长期：侧重抖音+B站快速传播
   ├─ 成熟期：侧重小红书+视频号深度运营
   └─ 衰退期：侧重公众号+微博情怀召回
   ```

3. **平台表达差异对比**

   | 维度 | 公众号 | 小红书 | 抖音 | 生命周期重点 |
   |------|--------|--------|------|-------------|
   | 标题 | 悬念/深度 | 痛点/利益点 | 悬念/冲突 | 增长期重抖音标题 |
   | 开头 | 场景引入 | 直接利他 | 黄金3秒 | 增长期重3秒法则 |
   | 配图 | 统一风格 | 精致好看 | 有冲击力 | 成熟期重精致 |
   | 结尾 | 总结升华 | 行动引导 | 互动召唤 | 各阶段不同 |

### Step 4: 定义内容调性体系

**目的**：形成可执行的调性规范文档

**操作流程**：

1. **内容人格定义**

   ```
   【内容人格定义模板】

   名称：[给这个人格起个名字]

   外在表现：
   ├─ 年龄感：[如：25-30岁的朋友]
   ├─ 性别感：[中性/偏女性/偏男性]
   ├─ 身份感：[如：懂行的朋友/专业顾问/同行伙伴]
   └─ 距离感：[如：可以约饭的朋友]

   说话风格：
   ├─ 语速：[快/慢/中等]
   ├─ 语气：[亲切/专业/活泼/沉稳]
   ├─ 用词：[专业术语多/通俗易懂/偶尔网络用语]
   └─ 句式：[短句为主/长句为主/长短结合]

   性格标签：
   [3-5个性格关键词，如：专业但不说教、有趣但不低俗、
    温暖但不做作、有态度但不极端]

   生命周期调性调整：[各阶段微调方向]
   ├─ 探索期：偏专业权威
   ├─ 增长期：偏活力共鸣
   ├─ 成熟期：偏温度陪伴
   └─ 衰退期：偏情怀唤醒
   ```

2. **调性表达清单**

   ```
   【调性表达指南】

   我们这样说（✓）：
   ├─ "[口语化表达]"
   ├─ "[共情式表达]"
   └─ "[利他式表达]"

   我们不这样说（✗）：
   ├─ "[说教式表达]"
   ├─ "[过于官方]"
   └─ "[负能量/抱怨]"

   生命周期表达调整：
   ├─ 探索期：多说"我发现/我认为/专业建议"
   ├─ 增长期：多说"你们/我们一起/有趣的是"
   ├─ 成熟期：多说"老朋友/一直陪伴/感谢支持"
   └─ 衰退期：多说"还记得/那些年/老地方"
   ```

3. **视觉调性规范**

   ```
   视觉调性要素：
   ├─ 主色调：[颜色+情绪，如：温暖的米白]
   ├─ 字体风格：[如：现代简约，有设计感]
   ├─ 图片风格：[如：真实生活场景，略微调色]
   ├─ 排版风格：[如：留白多，呼吸感]
   └─ 整体感觉：[如：清爽、干净、精致]

   生命周期视觉调整：
   ├─ 探索期：偏专业冷色调
   ├─ 增长期：偏活力暖色调
   ├─ 成熟期：偏温暖柔和色调
   └─ 衰退期：可怀旧复古色调
   ```

### Step 5: 建立调性执行规范

**目的**：确保团队统一执行调性标准

**操作流程**：

1. **内容审核标准**

   | 检查项 | 合格标准 | 不合格表现 | 生命周期检查 |
   |--------|----------|------------|-------------|
   | 语言风格 | 符合设定语气 | 过于官方/说教 | 探索期禁说教 |
   | 用词规范 | 使用用户语言 | 生硬术语堆砌 | 增长期用网络语 |
   | 情感调性 | 温暖有共鸣 | 冷漠/过度煽情 | 成熟期重温度 |
   | 价值观 | 正面积极 | 负面/争议 | 各阶段一致 |
   | 品牌元素 | 适度植入 | 硬广感 | 增长期可强化 |
   | 生命周期匹配 | 符合当前阶段 | 不适配 | 核心检查项 |

2. **调性示例库**

   ```
   【正面示例库】

   标题示例：
   1. "别再无效加班了，这个方法帮你准时下班"
   2. "月入3万和3千的人，差的不只是能力"
   3. "作为一个普通人，我是怎么从月薪5000到3万的"

   开头示例：
   "你是不是也经常……"
   "最近收到很多私信问……"
   "今天想聊聊我踩过的一个坑……"

   结尾示例：
   "希望今天的分享对你有帮助~"
   "你们还有什么问题，评论区告诉我"
   "觉得有用的别忘了点赞+在看"

   生命周期示例：
   ├─ 探索期："作为专注X年的从业者，我建议……"
   ├─ 增长期："这个问题我也遇到过！一起看看……"
   ├─ 成熟期："老朋友们的反馈来了……"
   └─ 衰退期："还记得当年我们一起……"
   ```

3. **调性测试方法**

   ```
   新内容调性自检：
   1. 读出来像谁在说？（是否符合人格定义）
   2. 目标用户会怎么看？（是否符合用户期待）
   3. 竞争对手会怎么说？（是否有差异化）
   4. 换平台能复制吗？（是否保留核心调性）
   5. 当前生命周期适合吗？（是否符合阶段调性）
   ```

### Step 6: 生命周期调性动态调整（升级新增）

**目的**：根据品牌发展阶段动态调整调性策略

**操作流程**：

1. **阶段判断标准**

   ```
   阶段判断决策树：

   当前指标状态？
   │
   ├── 用户认知度低 + 产品待验证 → 探索期
   │   └→ 调性：专业权威，建立信任
   │
   ├── PMF已验证 + 高速增长 → 增长期
   │   └→ 调性：活力共鸣，快速传播
   │
   ├── 增速放缓 + 用户基数大 → 成熟期
   │   └→ 调性：温度陪伴，维护忠诚
   │
   └── 增长停滞 + 流失上升 → 衰退期
       └→ 调性：情怀召回，激活老用户
   ```

2. **阶段调性转换策略**

   | 转换方向 | 触发条件 | 调性调整重点 | 过渡期 |
   |----------|----------|-------------|--------|
   | 探索→增长 | PMF验证+融资 | 专业→活力 | 3-6个月 |
   | 增长→成熟 | 增速<20% | 活力→温度 | 6-12个月 |
   | 成熟→衰退 | 流失率>10% | 温度→情怀 | 即时 |
   | 衰退→转型 | 新品/新定位 | 情怀→新生 | 重新定义 |

3. **调性一致性维护**

   ```
   阶段转换期调性管理：

   核心原则：调性转变要平滑过渡，避免用户认知断裂

   探索期→增长期过渡：
   ├─ 不要突然从专家变朋友
   ├─ 保留专业底色，增加活力元素
   └─ 过渡期：先增量内容，再逐步调整

   增长期→成熟期过渡：
   ├─ 不要突然从活泼变严肃
   ├─ 保留活力底色，增加温度元素
   └─ 过渡期：先增加情感内容，再减少热点借势

   成熟期→衰退期过渡：
   ├─ 不要突然打情怀牌
   ├─ 逐步增加回顾性内容
   └─ 过渡期：偶尔怀旧试水，观察反应再加强
   ```

## 使用示例（升级版）

### 完整场景：美妆品牌内容调性定位

**背景**：
- 品牌：国产新锐美妆品牌
- 目标用户：20-28岁年轻女性
- 平台：主攻小红书+抖音
- 阶段：增长期（已验证PMF，进入规模化增长）

**Step 1: 品牌属性梳理**

```
品牌DNA：
├─ 品类：平价美妆
├─ 情感：自信表达、真实不完美
├─ 风格：年轻活力、真实自然
└─ 调性基点："真实自信的平价好物"

增长阶段：增长期
├─ 调性策略：活力共鸣，快速传播
├─ 内容配比：热点30%+互动25%+价值20%+娱乐25%
└─ 重点：配合快速增长，K因子提升

关键词提取：
✓加分词：真实/自然/有趣/平价/好用/接地气
✗减分词：过于精致/高高在上/大牌模仿/爹味说教
```

**Step 2: 用户分析**

```
用户画像：
├─ 典型：大学生/初入职场女生
├─ 痛点：预算有限但想变美、容易被博主种草
├─ 期待：真实测评、平价好物、不踩雷
└─ 喜欢：真实自然、反感过度精修和广告

用户生命周期状态：
├─ Acquisition：被博主/朋友种草
├─ Activation：首次购买决策
├─ Retention：复购/持续关注
└─ Referral：分享给闺蜜

AARRR×调性匹配：
├─ 获客期：活力有趣，引发好奇
├─ 激活期：真实测评，帮助决策
├─ 留存期：陪伴成长，持续价值
└─ 推荐期：方便分享，有社交货币
```

**Step 3: 调性定位文档**

```
【品牌内容人格定义】

名称：小美（真实闺蜜）

外在表现：
├─ 年龄感：25岁的大学闺蜜
├─ 身份：懂美妆的同龄朋友
├─ 距离感：可以一起逛街聊护肤的闺蜜

说话风格：
├─ 语气：亲切、真实、不端着
├─ 用词：偶尔网络用语、但不过度
├─ 句式：短句为主，像发微信

性格标签：
真实不装 | 有话直说 | 贴心但不碎碎念 | 有审美但不挑剔

增长期调性调整：
├─ 活力感+：增加趣味表达，紧跟热点
├─ 互动感+：多提问，多回复评论
├─ 传播性+：内容要有分享动机
└─ 保持真实：不硬广，不过度精修

【表达清单】

✓ 我们这样说：
├─ "这盘眼影真的绝了，显色度绝绝子"
├─ "作为黄皮亲测，这个色号巨显白"
├─ "学生党冲这个，性价比绝了"
└─ "真心分享，无广放心食用"

✗ 我们不这样说：
├─ "本产品采用XXX专利技术……"
├─ "精致的猪猪女孩必备……"
├─ "女人一定要对自己好一点……"
└─ "这款产品适合所有肤质……"

增长期表达调整：
├─ 多用："姐妹们/冲/绝绝子/YYDS"
├─ 少用：过于专业的成分分析
└─ 增加：互动提问式结尾
```
