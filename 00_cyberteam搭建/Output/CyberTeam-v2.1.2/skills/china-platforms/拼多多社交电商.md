# 拼多多社交电商

## 概述

拼多多以社交电商起家，拼团、助力、分享是其核心玩法。本skill帮助运营者掌握拼多多的社交电商运营方法。

## 拼团运营

### 拼团类型

```markdown
## 单独拼团

- 单人购买
- 价格固定
- 转化为主

## 多人拼团

- 2-10人不等
- 成团享优惠
- 裂变传播

## 活动拼团

- 限时拼团
- 限量拼团
- 节日拼团
```

### 拼团设计

```markdown
## 拼团价格策略

1. 拼团价 vs 单买价
   - 拼团价低10-30%
   - 制造价格差

2. 成团门槛
   - 2人团：容易成团
   - 3-5人团：社交价值

## 拼团选品

- 适合分享的产品
- 价格敏感产品
- 家庭消费产品
```

## 裂变活动

### 助力免单

```markdown
## 活动设计

1. 邀请好友助力
2. 累计助力人数
3. 完成即免单

## 注意事项

- 控制成本
- 防止刷单
- 合理设置难度
```

### 分享优惠

```markdown
## 分享有礼

- 分享到微信/QQ
- 获取优惠券
- 引导下单

## 分享文案

"帮你砍一刀！"
"0元拿！快来帮我助力"
"差一点就成功了！"
```
