# 百度关键词研究

## 概述

关键词研究是百度SEO的基础。只有找到正确的关键词，才能确保网站获得精准流量。本skill帮助运营者掌握系统的关键词研究方法。

## 关键词分类

### 按搜索意图分类

```markdown
## 信息型关键词
- 用户目的：了解信息
- 示例："如何XXX"、"XXX是什么"
- 适合页面：教程、指南、概念解释

## 导航型关键词
- 用户目的：找到特定网站
- 示例："XXX官网"、"XXX品牌"
- 适合页面：首页、品牌页

## 交易型关键词
- 用户目的：完成购买
- 示例："XXX多少钱"、"XXX哪里买"
- 适合页面：产品页、购买页

## 对比型关键词
- 用户目的：做决策比较
- 示例："XXX和YYY哪个好"
- 适合页面：对比评测页
```

### 按竞争度分类

```markdown
## 高竞争词
- 搜索量大
- 优化难度高
- 适合：品牌词、行业大词

## 中等竞争词
- 有一定搜索量
- 需要持续优化
- 适合：产品词、功能词

## 低竞争词
- 长尾词为主
- 优化难度低
- 适合：细分需求词
```

## 关键词分析维度

### 竞争度分析

```markdown
## 分析指标

1. 搜索结果数量
   - site:关键词 查看收录量
   
2. 首页网站权重
   - 竞争对手是否高权重
   
3. 内容质量
   - 现有内容是否优质
   
4. 外链情况
   - 竞争对手外链数量
```

### 价值分析

```markdown
## 关键词价值评估

1. 搜索量
   - 月均搜索次数
   
2. 点击价格
   - CPC出价（SEM参考）
   
3. 转化潜力
   - 与业务的关联度
   
4. 商业价值
   - 用户购买意向强弱
```

## 关键词布局策略

### 页面关键词分配

```markdown
## 首页
- 品牌词
- 1-2个核心行业词
- 长尾品牌词

## 栏目页
- 中等竞争产品词
- 功能词

## 详情页
- 长尾关键词
- 问答型关键词
```

### 布局原则

```markdown
## 标题标签（Title）

公式：[页面核心词] + [修饰词] + [品牌名]
示例：专业网站建设服务_10年经验_XXX公司

## 描述标签（Description）

包含2-3次核心关键词
自然融入长尾词
有吸引力的描述

## 正文关键词

- 首段出现核心关键词
- 正文自然出现2-3次
- 结尾再次强调
- 使用H标签结构化
```
