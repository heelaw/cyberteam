# 社群精细化运营

## 概述

社群是私域运营的核心场域。精细化运营能让社群持续活跃，提升转化。本skill帮助运营者系统性地运营高价值社群。

## 社群定位与分层

### 社群类型矩阵

```markdown
## 引流群

- 定位：泛流量承接
- 人数：200人上限
- 生命周期：3-6个月
- 转化目标：筛选高价值用户

## 福利群

- 定位：优惠信息发布
- 人数：100-200人
- 生命周期：长期
- 转化目标：日常转化

## 会员群

- 定位：会员专属服务
- 人数：50-100人
- 生命周期：长期
- 转化目标：复购、客单价提升

## VIP群

- 定位：深度服务
- 人数：30-50人
- 生命周期：长期
- 转化目标：LTV最大化
```

```markdown

| 时间 | 栏目 | 内容示例 | 目的 |
|------|------|---------|------|
| 08:30 | 早安问候 | 早安+今日福利预告 | 养成打开习惯 |
| 10:00 | 好物种草 | 单品测评图文 | 价值输出 |
```

## 社群SOP自动化

### 新人入群SOP

```markdown
## 前72小时激活流程

0min    发送欢迎语+群规
30min   @新人，引导自我介绍
2h      私聊发送新人专属券
24h     推送群内精华内容合集
48h     邀请参与当日活动
72h     完成首次互动
```

## 社群转化策略

### 转化时机

```markdown
## 高转化时机

1. 限时限量时
   - 库存紧张
   - 时间倒计时

2. 社会证明时
   - 有人下单截图
   - 晒单好评

3. 情绪高潮时
   - 讨论热烈时
   - 活动高峰期

4. 新人入群时
   - 新鲜感强
   - 期待福利
```

```markdown
## 典型转化路径

社群 → 小程序/链接
     → 1v1私聊
     → 群内接龙
     → 下单转化
```
