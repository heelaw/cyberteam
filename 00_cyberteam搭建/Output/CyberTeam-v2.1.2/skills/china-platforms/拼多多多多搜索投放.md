# 拼多多多多搜索投放

## 概述

拼多多多多搜索是获取精准流量的核心工具。拼多多的算法和用户群体有其独特性，需要专门的投放策略。本skill帮助运营者掌握多多搜索的投放方法。

## 多多搜索基础

### 投放原理

```markdown
## 排名规则

综合排名 = 质量分 × 出价

质量分因素：
- 点击率
- 转化率
- 店铺评分
```

### 与淘宝直通车的差异

```markdown
## 核心差异

1. 用户群体
   - 拼多多：价格敏感
   - 淘宝：品质敏感

2. 竞争程度
   - 拼多多：相对较低
   - 淘宝：竞争激烈

3. 转化特点
   - 拼多多：拼团为主
   - 淘宝：单品为主
```

## 关键词策略

### 关键词选择

```markdown
## 选词渠道

1. 拼多多搜索下拉
   - 用户搜索热词
   - 自动联想

2. 推广中心
   - 关键词推荐
   - 热度分析

3. 竞品分析
   - 竞争对手标题
   - 热销商品标题
```

### 出价策略

```markdown
## 出价公式

目标CPC = 商品利润 × 转化率 × 期望ROI

## 新品期策略

1. 先用长尾词测试
2. 出价略高于市场均值
3. 快速积累数据

## 稳定期策略

1. 核心词稳定出价
2. 长尾词补充流量
3. 持续优化质量分
```
