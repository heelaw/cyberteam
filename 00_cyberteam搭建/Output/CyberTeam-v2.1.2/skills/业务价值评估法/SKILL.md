---

## 评估框架

### 四维评估模型

```
业务价值 = f(市场空间, 竞争优势, 增长潜力, 壁垒)

市场空间: TAM×SOM渗透率 → 可触及规模
竞争优势: 差异化×可防御性 → 市场份额
增长潜力: 复利因子×增长飞轮 → 未来价值
壁垒: 模仿成本×时间 → 护城河
```

---

## 执行步骤

### Step 1: 市场空间分析

```
评估维度:
  □ TAM (Total Addressable Market)
    - 定义边界
    - 计算方法: 数量×单价 or 渗透率×总需求
    - 数据来源: 行业报告、政府数据、公司测算

  □ SAM (Serviceable Addressable Market)
    - 可服务的市场
    - 地理限制
    - 产品能力边界

  □ SOM (Serviceable Obtainable Market)
    - 实际可获得的市场
    - 竞争能力
    - 3-5年可达到的规模

输出格式:
  市场空间: TAM=XXX亿 / SAM=XXX亿 / SOM=XXX亿
```

### Step 2: 竞争优势识别

```
评估维度:
  □ 差异化分析
    - 产品差异化
    - 价格差异化
    - 服务差异化
    - 品牌差异化

  □ 可防御性
    - 网络效应
    - 规模效应
    - 转换成本
    - 独家资源

  □ 竞争优势来源
    - 技术领先
    - 品牌心智
    - 渠道控制
    - 生态锁定

输出格式:
  竞争优势: 强 / 中 / 弱
  核心优势: [来源1, 来源2]
  可防御性: 高 / 中 / 低
```

### Step 3: 增长潜力评估

```
评估维度:
  □ 历史增长
    - 收入增长曲线
    - 用户增长曲线
    - 增长质量 (新增/留存/复购)

  □ 增长飞轮
    - 飞轮节点识别
    - 飞轮转速
    - 自我强化路径

  □ 复利因子
    - 复购率
    - 口碑传播
    - 规模化边际效应

输出格式:
  增长类型: 线性 / 指数 / 平台期
  飞轮成熟度: 成熟 / 搭建中 / 早期
  复利潜力: 高 / 中 / 低
```

### Step 4: 壁垒分析

```
评估维度:
  □ 壁垒类型
    - 技术壁垒 (专利、技术积累)
    - 网络效应 (用户越多越有价值)
    - 转换成本 (用户迁移代价)
    - 规模壁垒 (成本优势)
    - 品牌壁垒 (心智占领)

  □ 壁垒高度
    - 模仿所需时间
    - 模仿所需资金
    - 模仿所需能力

  □ 壁垒持久性
    - 当前壁垒强度
    - 未来壁垒变化趋势

输出格式:
  壁垒类型: [类型1, 类型2]
  壁垒高度: 高 / 中 / 低
  持久性: 3年+ / 1-3年 / <1年
```

## 综合评估

```
┌──────────────────┬─────────────┬──────────────────────────┐
│ 维度              │ 评估等级    │ 关键指标                  │
├──────────────────┼─────────────┼──────────────────────────┤
│ 市场空间          │ A/B/C/D     │ TAM≥100亿/≥10亿/≥1亿/<1亿│
│ 竞争优势          │ A/B/C/D     │ 强差异化+高防御性/...      │
│ 增长潜力          │ A/B/C/D     │ 指数/线性/平台期/下滑     │
│ 壁垒              │ A/B/C/D     │ 高持久性/中/低            │
└──────────────────┴─────────────┴──────────────────────────┘

4A → 顶级业务
3A1B → 优质业务
2A2B → 良好业务
≤1A → 一般/存疑
```

## 使用示例

```
输入: 某在线教育平台

评估:
  市场空间: TAM=4000亿, SAM=800亿, SOM=50亿 → B
  竞争优势: 品牌+师资+技术 → B
  增长潜力: 年增速100%+，飞轮成熟 → A
  壁垒: 师资积累+口碑 → B

结论: 优质业务 (3A1B)
```
