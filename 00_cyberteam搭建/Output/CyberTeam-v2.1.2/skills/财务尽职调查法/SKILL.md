## 执行步骤

### Step 1: 收入核实

```
检查项:
  □ 收入确认原则
    - 是否符合会计准则
    - 是否保守确认
    - 是否有激进确认

  □ 收入质量
    - 经常性收入 vs 一次性收入
    - 现金收入 vs 应收账款
    - 高质量收入占比

  □ 收入增长
    - 同比增长率
    - 环比增长率
    - 增长质量

红 flags:
  - 收入确认过于激进
  - 应收账款异常增长
  - 关联交易虚构收入
```

### Step 2: 成本分析

```
检查项:
  □ 成本结构
    - 固定成本 vs 变动成本
    - 成本占比合理性
    - 成本趋势

  □ 边际成本
    - 每增加一个用户的成本
    - 每增加一元收入的成本
    - 规模效应是否显著

  □ 主要成本项
    - 人力成本
    - 获客成本
    - 研发成本
    - 运营成本

红 flags:
  - 成本异常高于行业
  - 成本分类不清
  - 成本与收入不匹配
```

### Step 3: 现金流评估

```
检查项:
  □ 现金流表分析
    - 经营现金流 (CFO)
    - 投资现金流 (CFI)
    - 融资现金流 (CFF)

  □ 现金消耗
    - 月均现金消耗
    - runway (还能撑多久)
    - burn rate 趋势

  □ 资金效率
    - 每花1元的回报
    - 资金周转天数
    - 现金转换周期

红 flags:
  - 经营现金流持续为负
  - runway < 6个月
  - 融资依赖度过高
```

### Step 4: 风险识别

```
检查项:
  □ 财务风险
    - 债务风险
    - 流动性风险
    - 汇率风险

  □ 运营风险
    - 依赖单一客户
    - 依赖单一供应商
    - 季节性波动

  □ 合规风险
    - 税务合规
    - 会计准则合规
    - 行业监管合规

综合评估:
  - 风险等级: 高 / 中 / 低
  - 核心风险: [风险1, 风险2]
  - 缓解措施: [措施1, 措施2]
```

## 财务尽调清单

```
□ 3年财务报表 (PL/BS/CF)
□ 财务预测及假设
□ 银行流水核查
□ 主要客户/供应商合同
□ 债务合同及条款
□ 税务合规证明
□ 内部控制系统
□ 历史融资情况
```

## 使用示例

```
输入: 某创业公司B轮尽调

财务数据:
  - 年收入: 5000万
  - ARR增速: 150%
  - 毛利率: 70%
  - 经营现金流: -800万/年
  - runway: 18个月

尽调结果:
  收入质量: 良好 (经常性收入>90%)
  成本结构: 健康 (研发占比40%合理)
  现金流: 关注 (burn rate需控制)
  核心风险: 过度依赖大客户
  结论: 可接受，需关注现金流
```
