# 企微SCRM配置

## 平台特性

### 平台画像
- **用户规模**: 取决于企业规模
- **用户特征**: 精准用户、高价值、可控
- **算法逻辑**: 人工运营+自动化工具

### 平台规则
- **内容规范**: 用户导向、价值输出
- **变现方式**: 复购提升、LTV最大化

### Success Metrics (成功指标)

- **添加率**: 公域转私域比例
- **激活率**: 新用户首单转化
- **复购率**: 老客复购比例
- **LTV**: 用户生命周期价值

### Tools & Resources (工具和资源)

- 企业微信 - 私域载体
- SCRM系统 - 自动化运营
- 社群工具 - 群管理


## 概述

企业微信SCRM是私域运营的核心工具。合理的配置能让私域运营事半功倍。本skill帮助运营者系统性地配置企微SCRM系统。

## 渠道活码配置

### 活码类型

```markdown
## 员工活码

- 单个员工二维码
- 适合：1v1深度服务
- 限制：添加好友上限

## 群活码

- 自动创建群聊
- 满人自动换群
- 适合：大规模引流

## 联系我码

- 多人码（轮换）
- 自动标签
- 欢迎语触发
```

### 活码配置要点

```markdown
## 配置要素

1. 员工/群分配
   - 轮换规则
   - 自动分配

2. 自动标签
   - 来源标签
   - 渠道标签

3. 欢迎语
   - 触达时间
   - 内容设计

4. 追踪统计
   - 各渠道数据
   - 添加人数
```

## 标签体系设计

### 多维度标签

```markdown
## 客户来源
- 包裹卡
- 直播间
- 门店
- 短信
- 老客推荐
- 自然搜索

## 消费能力
- 高客单（>500）
- 中客单（200-500）
- 低客单（<200）

## 生命周期
- 新客户
- 活跃客户
- 沉默客户
- 流失预警

## 兴趣偏好
- 产品品类偏好
- 内容偏好
```

### 自动标签规则

```markdown
## 触发规则示例

1. 首次购买完成
   - 添加标签：新客户
   - 添加标签：已消费

2. 30天未互动
   - 添加标签：沉默客户
   - 移除标签：活跃客户

3. 累计消费>2000
   - 添加标签：高价值客户
   - 添加标签：VIP候选
```

## 欢迎语配置

### 欢迎语设计

```markdown
## 欢迎语要素

1. 问候语
   - 感谢关注
   - 品牌问候

2. 价值介绍
   - 你能提供的服务
   - 用户能得到什么

3. 行动引导
   - 回复关键词
   - 引导加群
   - 推荐产品
```

### 欢迎语模板

```markdown
Hi [用户昵称]~欢迎关注[品牌名]！

我是你的专属顾问[顾问名]
这里有：
- [价值1]
- [价值2]
- [价值3]

回复关键词获取你想要的内容：
回复「1」获取产品介绍
回复「2」获取专属优惠
回复「3」加入粉丝群

[附带一张价值感强的海报]
```

## 客户群配置

### 社群类型

```markdown
## 引流福利群

- 入群门槛：无
- 人数上限：200
- 运营目的：泛流量转化

## 会员群

- 入群门槛：消费满xxx
- 人数上限：100
- 运营目的：高价值客户维护

## VIP群

- 入群门槛：消费满xxx或邀请
- 人数上限：50
- 运营目的：深度服务、复购
```

### 入群欢迎语

```markdown
## 欢迎语内容

1. 群规说明
   - 禁止发广告
   - 友好交流

2. 群价值
   - 专属福利
   - 早晚报

3. 运营时间
   - 客服在线时间
```
