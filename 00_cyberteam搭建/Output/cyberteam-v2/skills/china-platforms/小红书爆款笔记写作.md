# 小红书爆款笔记写作

## 平台特性

### 平台画像
- **用户规模**: 2亿+
- **用户特征**: 女性为主、一二线城市、消费能力强
- **算法逻辑**: 推荐算法+搜索分发

### 平台规则
- **内容规范**: 真实分享、禁止硬广、禁止虚假种草
- **变现方式**: 品牌合作、蒲公英平台、直播带货

### Success Metrics (成功指标)

- **笔记爆文率**: 爆款笔记占比，>10%为优秀
- **收藏率**: 收藏数/浏览量，>8%为健康
- **转化率**: 点击购买/浏览量
- **互动率**: (点赞+收藏+评论)/浏览量，>5%为健康

### Tools & Resources (工具和资源)

- 笔记分析 - 创作者中心
- 蒲公英平台 - 品牌合作
- 小红书商城 - 电商工具


## 概述

小红书是一个"看图说话"的平台，笔记的标题、封面、正文共同决定了一条笔记的命运。本skill帮助运营者掌握爆款笔记的核心写作技巧。

## 标题写作技巧

### 高点击率标题公式

**痛点+解决方案**
```
"用了30天 | 终于可以说说XXX的真实感受了"
"别再踩坑了！XXX的正确打开方式"
```

**数字+效果**
```
"3步搞定 | 新手也能学会的XXX"
"坚持28天 | 我的皮肤竟然变成了这样"
```

**对比反差**
```
"从路人到女神 | 只差这一步"
"百元平替vs千元大牌 | 到底差在哪"
```

**情绪触发**
```
"后悔没早点买！这个XXX真的绝了"
"救命！这件XXX也太好看了吧"
```

### 标题写作要点

- 控制在18-22字
- 包含1-2个核心关键词（利于搜索）
- 善用符号和emoji增加视觉吸引力
- 制造悬念或情绪共鸣

## 封面设计原则

### 图片比例与规格
- 推荐竖版3:4比例（获得更大展示面积）
- 主图尺寸建议1080x1440像素

### 封面类型选择

**大字报风格**
- 大字叠加在图片上
- 突出核心卖点或对比效果

**对比图风格**
- 使用前后对比强烈
- 适合护肤、穿搭、减肥类内容

**产品展示风格**
- 简洁干净的产品图
- 突出产品质感和颜值

**真人出镜风格**
- 真实感更强，更容易建立信任
- 适合有IP的账号

## 正文结构模板

### 种草笔记模板（300-600字）

```markdown
## 开头（痛点共鸣）
"有没有姐妹和我一样，XXX困扰了好多年..."

## 产品引入
"最近被闺蜜种草了这款XXX，用了一段时间必须来分享"

## 详细体验
- 外观设计/使用感受
- 核心功效/使用效果
- 和同类产品的对比

## 总结建议
"整体来说，XXX，适合XXX的人群"
"温馨提示：XXX"
```

### 教程笔记模板

```markdown
## 开头（价值预告）
"今天手把手教你XXX，学会了你也能XXX"

## 步骤详解
Step 1: XXX（配图说明）
Step 2: XXX（配图说明）
Step 3: XXX（配图说明）

## 注意事项
"划重点：XXX"

## 效果展示
"看！这就是完成后的效果"
```

## 正文写作要点

1. **代入感**：用第一人称"我"的视角叙述
2. **具体化**：有数据、有细节、有感受
3. **真实感**：适当提缺点反而更可信
4. **行动指引**：结尾有明确的总结或建议

## 标签策略

### 标签组合公式
```
2-3个热门大标签 + 3-5个精准长尾标签 + 1-2个品牌标签

示例：
#好物推荐 #平价好物 #护肤分享 #我的护肤日常 #XXX品牌
```

### 标签选择原则
- 选择热度适中的标签（太热门容易被淹没）
- 精准长尾标签更容易获得精准流量
- 品牌标签利于品牌搜索
