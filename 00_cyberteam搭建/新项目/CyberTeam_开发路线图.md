# CyberTeam 开发路线图

## 阶段 0：准备
- 确定新目录
- 初始化仓库
- 确认技术栈
- 拆分文档
- 明确 MVP 边界

## 阶段 1：可运行骨架
- Electron 主进程
- Next.js 前端
- Claude Code 检测
- SQLite 初始化
- 基础设置页

## 阶段 2：聊天核心
- 会话列表
- 私聊界面
- 流式回复
- 消息存储

## 阶段 3：组织系统
- 公司创建
- 部门管理
- Agent 管理
- 组织树

## 阶段 4：群聊协作
- 创建群聊
- 拉人/拉部门
- @Agent
- Agent 之间讨论
- CEO 汇总
## 阶段 5：Playground
- 会议记录
- 摘要
- 审核
- 导出

## 阶段 6：市场与模板
- Agent 市场
- Skill 市场
- 模板安装
- 模板复制

## 阶段 7：增强
- 更复杂的协作
- 权限
- 高级审核
- 外部集成

## 每阶段验收标准

- 能运行
- 能保存数据
- 能完成一个完整流程
- 用户能感知价值
