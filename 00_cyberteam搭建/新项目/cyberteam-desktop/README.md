# CyberTeam Desktop

CyberTeam 的新项目骨架。

## 结构
- apps/desktop: Electron 主进程
- apps/renderer: Next.js 前端
- packages/core: 核心编排
- packages/claude: Claude Code 接入
- packages/team: 公司/部门/Agent
- packages/chat: 聊天/群聊/消息
- packages/skill: Skill 系统
- packages/playground: 输出与审核
- packages/market: 模板市场
- packages/db: SQLite
- packages/ui: 通用 UI

## 目标
- macOS 桌面端
- 基于用户本地 Claude Code
- AI 军团协作
- 类微信聊天
- Playground 交付
- 为 PUA 等增强模块预留扩展点
