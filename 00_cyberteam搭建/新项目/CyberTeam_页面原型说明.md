# CyberTeam 页面原型说明

## 1. Chat 页面

### 布局
- 左侧：会话列表
- 中间：聊天窗口
- 右侧：成员 / 上下文 / 状态

### 目标
像微信一样自然，但具备 Agent 协作能力。
### 视觉要求
- 类微信聊天体验
- Agent 显示头像、名称、岗位
- 消息流式出现
- 支持 @ 提及

### 必要状态
- 正在思考
- 正在回复
- 已完成
- 错误

## 2. Organization 页面

### 布局
- 左侧：公司和组织树
- 中间：节点编辑画布
- 右侧：节点属性面板

### 目标
让用户像搭一家公司一样搭 AI 团队。
### 视觉要求
- CEO 节点固定在顶部
- 部门分层清晰
- Agent 节点可拖拽
- 节点可绑定多个 Agent

## 3. Playground 页面

### 布局
- 左侧：文档列表
- 中间：内容编辑 / 预览
- 右侧：审核与导出

### 目标
把聊天结果变成交付物。
### 视觉要求
- 像会议成果中心
- 有清晰版本
- 可见审核状态

## 4. Market 页面

### 布局
- 上方：分类切换
- 中间：卡片列表
- 右侧：详情面板

### 视觉要求
- 像应用商店
- 可以筛选 Agent / Skill / 模板

## 5. Settings 页面

### 布局
- 左侧：设置导航
- 右侧：设置表单

### 必须设置项
- Claude Code 检测
- 本地路径
- 外观主题
- 数据目录
- 快捷键

## 6. 设计原则

- 像微信，而不是后台系统
- 像真实公司，而不是玩具
- 像协作软件，而不是单纯聊天框
- 每个 Agent 都要有身份感
