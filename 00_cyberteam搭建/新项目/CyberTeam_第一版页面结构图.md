# CyberTeam 第一版页面结构图

## 1. 页面总览

```text
┌────────────────────────────────────────────────────────────┐
│ Top Bar: 公司名 / 当前状态 / Claude Code 连接状态 / 设置     │
├───────────────┬───────────────────────────────┬────────────┤
│ 左侧导航栏     │ 中间主内容区                    │ 右侧信息栏  │
├───────────────┼───────────────────────────────┼────────────┤
│ Chat          │ 聊天页面                        │ 成员状态    │
│ Organization  │ 组织页面                        │ 节点属性    │
│ Playground    │ 成果页面                        │ 审核信息    │
│ Market        │ 市场页面                        │ 模板详情    │
│ Settings      │ 设置页面                        │ 系统状态    │
└───────────────┴───────────────────────────────┴────────────┘
```

## 2. Chat 页面结构

### 左侧
- 会话列表
- 私聊 / 群聊 / 部门群聊

### 中间
- 消息流
- 输入框
- @ 提示
- 流式输出

### 右侧
- 成员列表
- Agent 状态
- 当前群聊上下文

## 3. Organization 页面结构

### 左侧
- 公司树
- 部门树
- Agent 节点树

### 中间
- 组织画布
- 拖拽编辑
- 节点连接

### 右侧
- 节点属性
- 绑定 Agent
- 编辑名称 / 岗位 / 头像 / 职责

## 4. Playground 页面结构

### 左侧
- 文档列表
- 会话来源
- 历史版本

### 中间
- 文档编辑区
- 预览区

### 右侧
- 审核按钮
- 导出按钮
- 版本记录

## 5. Market 页面结构

### 左侧
- 分类导航

### 中间
- 卡片列表
- Agent / Skill / 模板

### 右侧
- 详情面板
- 安装按钮
- 复制按钮

## 6. Settings 页面结构

### 左侧
- 设置分类

### 中间
- Claude Code 检测
- 路径配置
- 主题配置
- 数据目录
- 快捷键

## 7. 第一版设计原则

- 聊天像微信
- 组织像公司
- Playground 像成果中心
- 市场像应用商店
- 设置像本地桌面软件
