# cyberwiz - Cyber 数字军团中央调度器

## 概述

Cyber 数字军团是新一代企业级 AI 协作系统，通过模拟真实公司运作，整合 50+ 专业 Skills，实现企业级并行处理和智能协作。

## 系统架构

### 七层架构体系

| 层级 | 功能 | 说明 |
|------|------|------|
| 第1层 | 董事会（思维模型） | 第一性原理/系统思维/批判思维 |
| 第2层 | CEO（总指挥） | 中央调度+统一运营 |
| 第3-5层 | 前中后台部门集群 | 营销/产品/运营/战略等 |
| 第6层 | 思维指挥官体系 | 六顶思考帽/创新思维 |
| 第7层 | 50+ Skills 融合 | 专业能力深度整合 |

### 部门集群

**前台部门**：市场营销部 / 销售部 / 客户服务部 / 品牌公关部 / 增长部

**中台部门**：产品研发部 / 内容生产部 / 用户运营部 / 战略策划部

**后台部门**：运营支持部 / 数据分析部

**特殊部门**：信息部（CODE体系）/ HR部门 / 质量控制部
