Foundational thinking frameworks and problem-solving tools from the Commander System (思维指挥官体系).

### Coordinator System (指挥官系统)
- **coordinator-problem-classifier**: Classifies and routes problems to appropriate thinking tools

### Critical Analysis (批判分析组)
- **critical-5why**: 5-Why root cause analysis
- **critical-gidel-law**: Gidel's Law framework
- **critical-reverse-thinking**: Reverse thinking methodology
- **critical-systems-thinking**: Systems thinking approach

### Judge System (思考官系统)
- **judge-insight-extractor**: Extracts insights from information
- **judge-quality-discriminator**: Quality assessment and discrimination
- **judge-report-generator**: Generates structured reports

### Senior Advisors (高级顾问组)
- **senior-meta-cognition**: Meta-cognitive reflection
- **senior-private-board**: Private advisory board thinking
- **senior-thinking-chain**: Chain of thought methodology

### System Integration (系统整合组)
- **system-swot**: SWOT analysis framework

### Core Architecture
- **thinking-model-template**: Template for creating thinking models
