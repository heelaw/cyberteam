## 一、背景与目标

### 1.1 项目背景

- **产品/服务**：位于广州洛溪某居民楼下的社区咖啡厅，主打精品手冲咖啡和简餐
- **目标用户**：
  - 洛溪及周边3公里居民（社区属性强）
  - 25-40岁年轻白领（咖啡消费主力）
  - 社交媒体活跃用户（小红书、抖音重度用户）
- **市场环境**：
  - 周边有2-3家连锁咖啡店竞争
  - 社区店铺，缺乏品牌知名度
  - 主要依赖自然客流，增长受限
- **资源限制**：
  - 预算：约5000元
  - 人力：店主+2名店员
  - 时间：1个月

### 1.2 活动目标

| 核心指标 | 目标值 | 实际值 | 达成率 |
|---------|--------|--------|--------|
| 线上报名人数 | 100人 | - | - |
| 到店核销人数 | 30人 | - | - |
| 到店转化率 | ≥60% | - | - |
| 社交媒体发布数 | 20篇 | - | - |
| 裂变转化人数 | 10人 | - | - |
| ROI | ≥2倍 | - | - |

### 1.3 发力点分析

- **产品生命周期**：探索期（新店，需要建立品牌认知和用户基础）
- **核心问题**：
  - 缺乏线上流量
  - 社区知名度低
  - 需要建立种子用户群
- **发力点选择**：线上引流+到店转化+社交传播
- **选择理由**：
  - 社区属性强，适合打造归属感
  - 咖啡厅自带社交属性，适合内容传播
  - 小预算情况下，社交传播是性价比最高的方式

## 二、活动方案设计

### 2.1 活动主题与形式

**活动主题**：洛溪咖啡日记——寻找身边的咖啡角落

**外在形式**：趣味互动 + 社交媒体打卡 + 线下体验

**核心支撑点**：
- **有奖励**：免单拿铁（价值25元）+ 优惠券
- **有好处**：发现身边好去处，获得社交货币（好看的照片）
- **有价值**：打造社区归属感，成为"洛溪咖啡达人"

### 2.2 主线玩法

**玩法逻辑**：三步到店机制

```
线上报名 ──────▶ 到店打卡 ──────▶ 分享传播
   ↓                ↓                ↓
领取任务卡      免费拿铁       赢取优惠券
```

| 阶段 | 用户动作 | 奖励机制 | 设计意图 |
|-----|---------|---------|---------|
| 线上报名 | 扫码填写表单（姓名+电话+居住区域） | 获得「咖啡任务卡」电子券 | 收集用户信息，建立联系 |
| 到店打卡 | 凭任务卡到店，拍摄咖啡+店铺照片发朋友圈 | 获得免单拿铁1杯 | 核心诱因，产生UGC |
| 二次传播 | 带话题#洛溪咖啡日记#发小红书/抖音，定位店铺 | 获得5张8折券（可送朋友） | 裂变传播 |

### 2.3 支线玩法

| 支线名称 | 解决问题 | 玩法机制 | 秘诀应用 |
|---------|---------|---------|---------|
| 咖啡天使招募 | 前期关注不足 | 前10名到店升级「咖啡天使」，专属杯套+月卡 | 稀缺感+尊崇感 |
| 周末咖啡盲盒 | 活动中热度下降 | 每周六前10名到店，随机获得隐藏特调 | 炫耀猎奇 |
| 早鸟奖励 | 订单集中/羊群效应 | 每日前3名到店，额外赠送小食 | 稀缺感 |
| 最美角落评选 | UGC内容质量低 | 活动结束后评选，赢取月卡+设计权 | 竞争攀比 |

### 2.4 九大秘诀应用

| 秘诀 | 是否应用 | 具体体现 |
|-----|---------|---------|
| 物质激励&概率事件 | ☑是 | 免单拿铁+咖啡盲盒（概率性奖励） |
| 稀缺感 | ☑是 | 前10名咖啡天使、每日前3名早鸟、周末前10名盲盒 |
| 竞争&攀比 | ☑是 | 最美咖啡角落评选、排行榜机制 |
| 炫耀&猎奇 | ☑是 | 拍照打卡、咖啡盲盒隐藏特调 |
| 营造强烈情绪 | ☑是 | 社区归属感、"身边的咖啡角落"情感连接 |
| 尊崇感&重视感 | ☑是 | 咖啡天使专属杯套、店招展示 |
| 超值感 | ☑是 | 免单拿铁+5张优惠券（价值远超一杯咖啡） |
| 低成本&单线程 | ☑是 | 扫码报名→到店→拍照，流程简单 |
| 趣味性反差 | ☑是 | 社区小店vs"寻找咖啡角落"的文艺感 |
