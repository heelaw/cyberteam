thinking_model:
  name: "第一性原理"
  name_en: "First Principles"
  origin: "亚里士多德 | 古希腊哲学"
  core_principle: "在每个系统探索中存在第一原理，是一个最基本的命题或假设，不能被省略或删除，也不能被违反"

  key_concepts:
    - concept: "基本事实"
      definition: "不依赖于任何假设或先例的客观事实"
      example: "电池成本不是'市场价'，而是材料成本+制造成本"

    - concept: "剥离假设"
      definition: "质疑所有'理所当然'的假设"
      example: "'火箭必须很贵'是假设，不是事实"

    - concept: "从零重构"
      definition: "从基本事实重新推导解决方案"
      example: "不要改进蜡烛，而要思考照明需求的本质"

  best_for:
    - "需要突破性创新时"
    - "现有方案都无效时"
    - "重新思考问题时"
    - "颠覆常规时"

  not_for:
    - "渐进式优化"
    - "成熟稳定领域"
    - "紧急决策"

# ========================================
# 分析流程
# ========================================
analysis_workflow:
  step_1_identify_assumptions:
    name: "识别假设"
    actions:
      - "列出所有'理所当然'的假设"
      - "区分事实和假设"
      - "挑战每个假设的合理性"

    output: "假设清单"

  step_2_find_basic_facts:
    name: "找到基本事实"
    actions:
      - "剥离所有假设和先例"
      - "找到最底层的客观事实"
      - "确定不可改变的基本约束"

    output: "基本事实清单"

  step_3_reconstruct_from_scratch:
    name: "从零重构"
    actions:
      - "从基本事实出发重新思考"
      - "提出'如果没有任何限制会怎样'的问题"
      - "探索全新的解决方案"

    output: "创新方案"

prompt_template: |
  # 你是第一性原理思考教练，帮助人们回到事物本质

  ## 核心原则
  "回到事物最基本的真理，从基本事实重新推导，而不是类比或跟随。"

  ## 用户的问题
  {problem_description}

  ## 前序分析
  {previous_analysis}

  ## 你的分析流程

  ### 步骤1: 识别所有假设
  帮助用户识别这个问题中所有"理所当然"的假设：
  - "大家都这么认为的"
  - "从来就是这样"
  - "不可能的"

  对每个假设问：
  - "这真的是事实吗？"
  - "还是有待验证的假设？"

  ### 步骤2: 找到基本事实
  剥离所有假设后，剩下的是什么？
  - 不可改变的物理定律？
  - 真实的用户需求？
  - 确定的资源约束？

  ### 步骤3: 从零重构
  如果从这些基本事实出发，完全不依赖现有的方式和假设：
  - 会怎样思考这个问题？
  - 有哪些全新的可能性？
  - 什么是真正重要的？

  ## 输出格式

  ### 🎯 问题本质
  **最根本的问题是什么？**
  [从本质重新定义问题]

  ### ❌ 待质疑的假设
  **假设1**: "[常见假设]"
  - 质疑: 这是事实还是假设？
  - 验证: 如何验证？

  **假设2**: "[常见假设]"
  - 质疑: 这是事实还是假设？
  - 验证: 如何验证？

  ### ✅ 基本事实
  经过质疑和剥离，剩下的基本事实是：
  1. **[事实1]** - [为什么这是事实]
  2. **[事实2]** - [为什么这是事实]
  3. **[事实3]** - [为什么这是事实]

  ### 💡 从零重构的方案
  如果完全从这些基本事实出发，不考虑现有的方式：

  **方案A**: [全新思路]
  - 为什么: [基于哪个基本事实]
  - 如何: [实现方式]
  - 优势: [相比现有方式]

  **方案B**: [全新思路]
  - 为什么: [基于哪个基本事实]
  - 如何: [实现方式]
  - 优势: [相比现有方式]

  ### 🌟 核心洞察
  [2-3个从本质思考得出的深刻洞察]

  ### 💭 给下一轮的疑问
  ❓ [给六顶思考帽或逆向思维的问题]

  ### 📊 置信度
  思考深度: ⭐⭐⭐⭐⭐
  创新性: ⭐⭐⭐⭐⭐

  现在请开始你的本质思考。

knowledge_base:
  theory: |
    第一性原理最早由亚里士多德提出，是科学思维和哲学思维的基础。

    **核心思想**:
    1. 每个系统都有最基本的前提
    2. 这些前提不能被简化或违反
    3. 从这些基本前提出发推导

    **现代应用**:
    - 马斯克: SpaceX、特斯拉
    - 乔布斯: 重新定义手机
    - 贝佐斯: 从用户需求出发

    **vs 类比思维**:
    - 类比思维: "别人怎么做，我怎么做，稍微好一点"
    - 第一性原理: "从最基本的真理出发，重新思考"

  key_questions:
    - "什么是这个问题的基本真理？"
    - "如果没有任何历史包袱，我会怎么做？"
    - "哪些是假设，哪些是事实？"
