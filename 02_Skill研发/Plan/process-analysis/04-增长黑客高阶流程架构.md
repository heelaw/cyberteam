## 课程定义的流程架构

### 核心框架：AARRR增长模型

```
第1章       第2章         第3章          第4章           第5-6章       第7章
  ↓           ↓             ↓              ↓              ↓            ↓
课程导论    建立增长大局观  上线增长实验   数据驱动内功    用户激活+留存  拉新+变现(选修)
```

### 步骤1：课程导论
- 增长本质理解
- 增长全景图

### 步骤2：建立增长大局观（Week 1）⭐
**Phase 1: 评估产品增长准备度**
- 判断产品是否准备好做大规模增长
- 通过4个问题判断现阶段增长重点

**Phase 2: 确定北极星指标**
- 找到增长方向标：北极星指标
- 理解北极星指标深层含义

**Phase 3: 构建增长模型**
- 基础：构建和应用3种增长模型
- 进阶：构建最精细的全定量增长模型

### 步骤3：上线增长实验（Week 2）⭐
**Phase 1: 产生假设**
- 3步产生高质量实验假设

**Phase 2: 排列优先级**
- 通过ICE模型排列优先级

**Phase 3: 设计实验**
- 制定3类实验指标
- 确定实验受众
- 设计实验版本

**Phase 4: 上线与分析**
- 不懂编程也能快速上线实验
- 分析实验结果，放大实验成效

### 步骤4：数据驱动增长内功（Week 3）⭐
**Phase 1: 分析数据找机会**
- 从整体数据入手
- 用户分群
- 用户行为分析
- 组合运用

**Phase 2: 理解数据**
- 3步产出"用户行为路径图"

**Phase 3: 收集与监测数据**
- 快速制定埋点计划
- 搭建净增用户仪表盘
- 搭建细分指标仪表盘

### 步骤5：新用户激活（Week 4）⭐
**Phase 1: 理解激活目标**
- 以终为始理解激活目标

**Phase 2: 找到Aha时刻**
- 4个步骤找到新用户的Aha时刻

**Phase 3: 寻找激活线索**
- 激活漏斗分析
- 激动指数分析

**Phase 4: 设计激活策略**
- 最佳实践应用

### 步骤6：老用户留存（Week 5）⭐
**Phase 1: 绘制留存曲线**
- 4步法绘制留存曲线
- 区分好留存和坏留存

**Phase 2: 评估留存难度**

**Phase 3: 寻找留存线索**
- 找到输入变量+对比用户群组
- 功能留存矩阵

**Phase 4: 提升留存**
- 提升新用户留存（3种手段）
- Hooked模型
- 用户行为闭环
- 加深用户参与（4种方法）
- 个性化体验+避免流失

### 步骤7：拉新和变现（Week 7，选修）
- 渠道分析
- LTV计算
- 变现金字塔
- 价格敏感度测试

## Agent层与Skill层划分

### Agent层设计

**Agent: 增长大局观构建Agent**
- 输入：产品信息、市场环境
- 决策：PMF评估 → 增长重点 → 北极星指标 → 增长模型
- 输出：增长战略框架

**Agent: 增长实验Agent**
- 输入：业务目标、增长假设
- 决策：假设产生 → ICE排序 → 实验设计 → 分析
- 输出：实验方案与结果

**Agent: 数据分析Agent**
- 输入：业务数据、用户数据
- 决策：选择分析方法 → 产出洞察
- 输出：分析报告、仪表盘

**Agent: 用户激活Agent**
- 输入：新用户数据、行为数据
- 决策：找Aha时刻 → 分析激活漏斗 → 设计策略
- 输出：激活方案

**Agent: 用户留存Agent**
- 输入：用户数据、留存数据
- 决策：绘制曲线 → 找线索 → 设计提升方案
- 输出：留存提升方案

**Agent: 商业化Agent（选修）**
- 输入：用户数据、产品信息
- 决策：渠道分析 → LTV计算 → 变现设计
- 输出：商业化方案

## 数据流向

```
产品+市场 → 增长大局观Agent → PMF → 4问题 → 北极星 → 增长模型
    ↓
增长假设 → 实验Agent → 假设 → ICE → 设计 → 上线 → 分析 → 放大
    ↓
业务+用户数据 → 数据分析Agent → 整体 → 分群 → 行为 → 路径 → 仪表盘
    ↓
新用户数据 → 激活Agent → Aha → 漏斗 → 激动指数 → 激活策略
    ↓
用户+留存数据 → 留存Agent → 曲线 → 线索 → 矩阵 → 提升方案
    ↓
用户+产品 → 商业化Agent → 渠道 → LTV → 变现
```

## 总结

增长黑客高阶的核心理念：
1. **数据驱动** - 一切决策基于数据分析和实验验证
2. **北极星指标** - 用单一核心指标指引增长方向
3. **增长模型** - 构建可量化的增长公式
4. **A/B测试** - 通过实验验证假设的科学方法
5. **AARRR全链路** - 获取→激活→留存→变现→推荐

与运营高阶课的区别：
- 运营高阶：框架式思维，运营规划
- 增长黑客：数据驱动，AARRR全链路增长实验方法
