## 课程定义的流程架构

### 核心框架：精益运营与进阶能力提升

```
底层逻辑 → 文案进阶 → 数据分析 → 活动运营 → 推广营销
    ↓           ↓           ↓          ↓          ↓
通用能力    专业技能    决策依据    执行能力    增长引擎
```

### 步骤1：理解运营底层逻辑（选修）
- 理解运营岗位相通的业务逻辑
- 做符合产品生命周期的运营
- 精益运营的流程及工作方法

### 步骤2：提炼产品卖点（文案进阶上）
- Agent调度以下Skill：
  - 7个问题找方向Skill（卖点7方向思考法）
  - 从用户痛点反推卖点Skill
- 判断：根据产品特点选择卖点提炼方法

### 步骤3：制定文案转化策略（文案进阶上）
- Agent调度以下Skill：
  - 用户分层Skill（文案面向的用户）
  - 需求类型判断Skill
  - 场景判断Skill（转化场景与潜在阅读时间）
- 判断：根据用户特征确定转化策略

### 步骤4：构建文案结构与表达（文案进阶下）
- Agent调度以下Skill：
  - AIDA模型Skill
  - 4P模型Skill
  - 书写原则Skill（讲清楚、讲人话、有感、增强感受）
  - 文案删改Skill
  - 自检清单Skill

### 步骤5：数据分析驱动运营
- 数据分析的价值
- 数据指标拆解思路
- 结合历史数据找增长发力点
- 筛选优质渠道
- 异常数据判定与分析

### 步骤6：活动运营（上下）
**上部分 - 理解与创新**：
- 理解活动运营本质
- 常见活动形式和用户吸引逻辑
- 梳理活动核心流程
- 活动"微创新"方法
- 活动页面设计原则

**下部分 - 完整策划与执行**：
- 6大基本要素
- 活动后端策略设计
- 活动主题策划与前端玩法
- 活动风险评估
- 执行与落地四步走

### 步骤7：推广营销
- 认识推广与营销
- 发现及开拓推广入口
- 影响推广效果的三要素
- 搭建转化场景
- 转化路径和场景设计
- 转化诱因和用户决策
- 推广数据分析和优化

## Agent层与Skill层划分

### Agent层设计

**Agent: 文案转化策略Agent**
- 输入：产品信息、目标用户
- 决策：调用用户分层/需求判断/场景判断Skill
- 输出：转化策略

**Agent: 文案生成Agent**
- 输入：产品卖点、转化策略
- 决策：调用AIDA/4P模型Skill + 书写原则Skill
- 输出：完整文案

**Agent: 活动策划Agent**
- 输入：活动目标、预算
- 决策：判断活动类型 → 调用对应Skill组合
- 输出：完整活动方案

**Agent: 数据分析Agent**
- 输入：业务数据
- 决策：调用指标拆解/增长分析/异常分析Skill
- 输出：分析报告与建议

**Agent: 推广营销Agent**
- 输入：产品、目标用户、预算
- 决策：选择渠道 → 设计转化场景 → 优化策略
- 输出：推广方案

## 数据流向

```
产品信息 → Agent判断 → 选择Skill → Skill执行 → 输出 → Agent优化 → 最终结果
    ↓
用户特征 → 用户分层Skill → 需求判断 → 场景判断 → 转化策略
    ↓
卖点 + 转化策略 → 文案生成Agent → AIDA/4P → 书写原则 → 删改 → 自检 → 最终文案
    ↓
活动目标 → 活动策划Agent → 形式选择 → 6要素 → 风险评估 → 执行方案
    ↓
业务数据 → 数据分析Agent → 指标拆解 → 增长分析 → 异常分析 → 建议
```

## 总结

运营进阶课的核心理念：
1. **精益运营** - 基于数据驱动的精细化运营方法
2. **系统化文案** - 从卖点提炼到转化策略到最终生成的完整流程
3. **活动方法论** - 从理解本质到微创新到完整策划的系统方法
4. **推广营销** - 从入口发现到转化场景搭建的完整链路

这与运营初阶课的区别在于：
- 初阶：掌握基础技能（怎么做）
- 进阶：建立系统方法论（为什么+怎么做+如何优化）
