# 运营入门 - 最小可执行单元分析

## Skill候选汇总

### 高价值独立Skill（可直接使用）
1. **业务模式分析** - 盈利方式→组织结构→运营关系
2. **运营岗位识别与选择** - 岗位类型→工作内容→匹配度
3. **职业规划制定** - 目标→路径→行动
4. **能力层次理解** - 底层→基础→高阶→特定场景
5. **运营价值认知** - 业务价值→个人价值→长期价值

### 需要Agent/MCP支撑的Skill
1. **行业趋势分析** - 需要MCP搜索
2. **岗位前景调研** - 需要MCP搜索

### 需要工具支撑的Skill
1. **数据分析能力** - 需要数据工具
2. **行业数据获取** - 需要数据系统
