# 运营小课 - 最小可执行单元分析

## 课程概述
- **课程类型**: 运营专项小课合集

## Skill候选汇总

### 高价值独立Skill（可直接使用）
1. **AI生产力应用** - 提示词→生成→优化
2. **运营6要素解构** - 工作→6要素→分析
3. **用户路径分析与优化** - 路径→问题→优化
4. **数据漏斗构建与分析** - 漏斗→转化→优化
5. **用户场景与痛点分析** - 场景→痛点→需求
6. **需求验证与优先级排序** - 验证→排序→决策
7. **9大互动秘诀应用** - 心理→机制→互动
8. **转化链路设计** - 行为→链路→要素
9. **案例拆解方法** - 框架→拆解→提炼

### 需要Agent/MCP支撑的Skill
1. **趋势分析** - 需要MCP搜索
2. **关键词分析** - 需要MCP搜索
3. **网络数据挖掘** - 需要MCP搜索

### 需要工具支撑的Skill
1. **GPT使用与调教** - 需要AI工具
2. **数据漏斗构建** - 需要数据系统
3. **百度统计数据获取** - 需要统计工具
4. **用户互动神器使用** - 需要运营工具
