# 运营初阶课 - 最小可执行单元分析

## 课程概述
- **课程类型**: 运营新人入门课程

- **适用对象**: 零基础运营新人

## Skill候选汇总

### 高价值独立Skill（可直接使用）
1. **短文案5种写法** - 恐惧诉求、认知对比、畅销、权威、热销
2. **活动策划方案撰写** - 目的→创意→玩法→时间→预算
3. **社群裂变机制设计** - 诱饵→路径→激励→监控
4. **快速选题策划** - 定位+需求+热点→选题
5. **用户画像分析** - 数据→特征→需求→场景

### 需要Agent/MCP支撑的Skill
1. **竞品分析** - 需要MCP搜索竞品信息
2. **行业素材搜集** - 需要MCP搜索行业信息
3. **投放数据分析** - 需要数据系统支撑

### 需要工具支撑的Skill
1. **任务宝裂变操作** - 需要任务宝工具
2. **SEM投放配置** - 需要投放工具
3. **数据监控** - 需要数据面板
