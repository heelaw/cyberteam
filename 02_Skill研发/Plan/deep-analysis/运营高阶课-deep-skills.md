# 运营高阶课 - 最小可执行单元分析

## 课程概述
- **课程类型**: 运营高阶课程
- **核心内容**: 框架式思维、运营框架理解、行动指导型框架、产品运营规划、用户行为激励、策略执行、数据驱动运营
- **适用对象**: 3年以上运营从业者、运营负责人

## Skill候选汇总

### 高价值独立Skill（可直接使用）
1. **框架式思维训练** - 三种拆解思路
2. **抽象/具体运营问题拆解** - 维度分解
3. **行动指导型框架构建** - 从认知到行动
4. **产品生命周期运营策略** - 阶段匹配
5. **用户激励体系设计** - 积分、等级、勋章
6. **数据驱动决策** - 从数据到行动

### 需要Agent/MCP支撑的Skill
1. **陌生领域快速学习** - 需要MCP搜索
2. **竞品分析** - 需要MCP搜索
3. **行业趋势分析** - 需要MCP搜索

### 需要工具支撑的Skill
1. **数据获取与分析** - 需要数据系统
2. **数据面板搭建** - 需要BI工具
3. **数据埋点设计** - 需要工程支持
4. **推广数据分析** - 需要投放工具
