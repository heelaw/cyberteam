# 运营进阶课 - 最小可执行单元分析

## 课程概述
- **课程类型**: 运营进阶课程
- **核心内容**: 运营底层逻辑、文案进阶、推广营销进阶、工作管理、活动运营、新媒体内容运营
- **适用对象**: 1-3年运营从业者
