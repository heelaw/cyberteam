### 7.1 紧急改进（P0）

**项目组1**: 为所有Skill添加标准Frontmatter

```yaml
---
name: Skill名称
description: 一句话描述
version: 1.0
tags: [tag1, tag2]
triggers:
  - "触发词1"
  - "触发词2"
---
```

**两个项目组**: 补充输入输出JSON Schema定义

```json
{
  "input": {
    "type": "object",
    "required": ["field1"],
    "properties": {
      "field1": {"type": "string", "description": "..."}
    }
  },
  "output": {
    "type": "object",
    "properties": {
      "result": {"type": "string"}
    }
  }
}
```
