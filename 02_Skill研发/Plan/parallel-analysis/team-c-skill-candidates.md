## 优先级A（高优先级，可立即制作）

### 1. 北极星指标制定法
- **来源**: 增长黑客高阶_2.1
- **核心方法论**: 三步法制定北极星指标（明确商业目标与用户价值交集 → 列出备选指标用六标准筛选 → 确认北极星指标）
- **输入参数**: 产品类型、商业目标、用户价值主张
- **输出形态**: 北极星指标描述 + 筛选评估表
- **制作难点**: 六标准评估需要结合具体业务判断
- **优先级评分**: 9/10
- **评估维度**:
  - 颗粒度: 4/5（明确的三个步骤）
  - 可操作性: 9/10（步骤清晰可直接执行）
  - 模板化程度: 8/10（可形成筛选评估表）
  - 场景通用性: 9/10（所有互联网产品适用）
  - Skill可行性: 9/10
