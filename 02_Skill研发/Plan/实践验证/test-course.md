# 北极星指标课程逐字稿

大家好，我是增长黑客课堂，今天给大家讲解北极星指标。

## 什么是北极星指标

北极星指标（North Star Metric）是你产品增长的核心指标。它是唯一重要的指标，所有的增长工作都应该围绕这个指标展开。

北极星指标有三个核心特征：
1. **唯一性**：只能有一个，不能同时有多个
2. **可衡量**：必须是可量化的数据
3. **可行动**：团队可以围绕它制定策略

## 如何选择北极星指标

选择北极星指标有五个标准：
1. 能够反映用户价值
2. 能够被产品团队影响
3. 是一个先行指标而非滞后指标
4. 整个公司都能理解
5. 可以拆解到各个团队

## 北极星指标三步法

### 第一步：定义战略目标

首先明确产品的长期战略目标。比如你的产品是在线教育平台，战略目标可能是"帮助用户高效学习"。

### 第二步：识别关键指标

列出所有可能的关键指标候选。比如：
- 日活跃用户数
- 完课率
- 用户留存率
- 付费转化率
- 用户推荐指数

### 第三步：选择唯一指标

根据五个标准筛选出唯一的北极星指标。

## 案例：某电商产品

某电商产品的北极星指标选择过程：
- 最初有100个指标
- 通过五步筛选法
- 最终确定"成功下单用户数"作为北极星指标

## 总结

选择北极星指标是一个团队对齐的过程，需要大家充分讨论，达成共识。好的北极星指标能够让团队聚焦，让增长更加高效。
